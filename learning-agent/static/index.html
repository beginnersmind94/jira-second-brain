      How Learning Studio works

      Every guide starts from a real training session. Learning Studio reads the transcript, looks up the matching records your team has already approved, writes the guide, and then refuses to hand it over until it has checked that every single claim traces back to a real source — quoted, not invented.

      Every guide flows from a training transcript through researched, cited writing and a four-stage pipeline, then must pass an automated source check. A human reviewer reads it — requesting AI-assisted edits if needed — and approves it before it enters the Library.

      1 Find the right material

      Reads your transcript and pulls up the records your team has already approved for that module — so the guide is built on what you actually shipped, not guesswork.

      2 Decide what to cover

      Works out which tasks and workflows the guide should explain, and how much depth each one needs.

      3 Confirm every fact

      Matches each statement back to its source and notes how strong that evidence is — agreed criteria first, then release notes, then the original description.

      4 Write the guide

      Writes it in plain language and attaches the exact source quote to every claim — quotes are lifted straight from the source, never written by the model.

      Verified, then reviewed

      An automatic source check confirms every claim traces to a real source. Only then can a person read it, request AI-assisted edits if needed, and approve it into the Library.

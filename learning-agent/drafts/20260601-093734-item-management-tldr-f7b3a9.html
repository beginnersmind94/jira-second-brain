Item Management — TLDR

What this module does

Item Management is the master catalog where District Administrators create and configure every food and non-food product — establishing pack sizes, nutrients, allergens, and pricing that Inventory, Menu Planning, and POS all depend on.

Key features

FeatureWhat it does
Food / Non-Food item typeDetermines whether serving sizes, nutrients, and POS fields are available.
Pack sizes (up to 3 tiers)Defines ordered unit hierarchy (e.g., Case → Carton → Each) for Inventory receiving.
Serving sizes & pack linkingLinks serving sizes to packs via Sub-Qty; unlinked food items are Inventory Ineligible.
Base Unit designationMarks the lowest countable inventory unit; can be promoted to the serving size for food items.
CostingEnter cost for one pack tier; system auto-calculates all other tiers from sub-quantities.
Nutrient InfoCaptures label nutrients; (M) flag marks missing values as unknown rather than zero.
Allergen InfoAssigns standard or custom allergen status (Contains / May Contain / Free From) per item.
Contribution InfoRecords meal-pattern credits (Meat/Alternate, Grains, Vegetables, Fluid Milk).
Direct Menu Item (DMI)Promotes a food item to a standalone menu entry visible on production records and POS.
POS PricingSets per-eligibility-group prices (Free, Reduced, Paid) across site types for cashier screens.
HACCP StepsAttaches process type (No Cook / Same Day / Complex) and prep directions to an item.
Custom Allergen configurationAdds non-standard allergens to the district list via Item Management > Configuration > Item Configuration.

Who uses it

District Administrators create and publish items; Menu Planners (Nutritionists) manage nutrient, allergen, contribution, and Direct Menu Item settings.

Common workflows

  • Create a non-food supply item: Item card → select NON-FOOD → add pack sizes → enter costing.
  • Create a food item: Item card → select FOOD / fluid type → add serving size → add pack sizes → link via Sub-Qty → enter costing.
  • Configure nutrients, allergens, and meal-pattern contribution credits on the Nutrient Info tab.
  • Set up a Direct Menu Item with Button Name, Menu Item Category, and POS Pricing grid.
  • Add a custom allergen in Item Configuration, then assign it to items from the allergen panel.

Important gotchas

  • Add all pack sizes before entering costing. Adding a tier after costing is set causes the Missing Costing alert to persist.
  • Base Unit and count mode lock after the first Inventory transaction. Configure both correctly before the item enters active Inventory use — they cannot be changed afterward.
  • Unlinked food items are Inventory Ineligible. Serving sizes and pack sizes must be linked via Sub-Qty or the item cannot participate in full Inventory workflows.
  • Maximum three pack-size tiers. The + PACK button disables after tier three; handle higher-level units (e.g., pallets) at receiving time in Inventory.
  • Custom allergens must be pre-configured before assignment. The item allergen dropdown only lists allergens already created in Item Configuration — the item panel cannot create new ones.

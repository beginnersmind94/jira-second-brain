Item Management — TLDR

What this module does

Item Management is where all food and non-food items are created and maintained in SchoolCafé 2.0; every downstream module — Inventory, Menu Planning, POS, and Family Hub — draws its data from here.

Key features

FeatureWhat it does
Item cardCaptures item name, category, storage category, and valuation group.
Food / Non-Food type selectionSet on the Units card; Non-Food skips serving sizes and the fluid question.
Fluid / Non-Fluid selectionFood items only; determines whether weight is recorded in grams or milliliters.
Pack sizes (up to 3 tiers)Defines purchasing and storage hierarchy for Inventory counts and ordering.
CostingEnter price for one unit; system calculates all other tiers through the Sub-Qty hierarchy.
Serving sizes (food only)Units of measure used for recipe ingredients and menu planning; added before pack sizes.
Serving-to-pack linking (food only)Links serving size to lowest pack unit via Sub-Qty to achieve Inventory eligibility.
Base Unit + counting modeSets the unit used for inventory counts and toggles Whole Numbers vs. Decimals.
Nutrient Info tab (food only)Records required nutrients; Missing Value checkbox flags absent data without implying zero.
Allergen Info card (food only)Assigns Contains / May Contain / Free From status; supports custom allergens.
Direct Menu Item (Menu Info tab)Turns a single item into a schedulable menu entry with POS button name and pricing.
Steps card (HACCP)Assigns HACCP process and critical control points for food-safety compliance.

Who uses it

District Admins, Menu Planners, and Nutritionists create and maintain items; Item Management Super Users manage module configuration and publish items to Inventory.

Common workflows

  • Create a non-food supply item: Item card → Units card → Pack sizes → Costing → optional GTIN.
  • Create a food item: Item card → Units card → Serving sizes → Pack sizes → Link servings → Costing → Base Unit.
  • Enter nutrient and allergen data for a food item.
  • Configure a Direct Menu Item and enable POS display with pricing.
  • Assign HACCP process and critical control points on the Steps card.

Important gotchas

  • Add all pack sizes before entering costing — adding a pack after costing forces full re-entry of all prices.
  • Base Unit and Whole Numbers/Decimals setting lock permanently after the item's first Inventory transaction; incorrect settings require Cybersoft support to fix.
  • Food items display a persistent "Servings and Pack Units must be linked" alert until Sub-Qty is entered on the serving size row.
  • Custom allergens must be created in Item Management → Configuration → Item Configuration → Custom Allergens before they appear in the dropdown on any item.
  • For nutrients absent from an item's label, use the Missing Value checkbox — blank implies unknown; zero implies the item contains none.

Item Management: Creating Food and Non-Food Items — TLDR

Module: Item Management · Platform: SchoolCafé 2.0 · Template: TLDR one-pager · Source: Long-form guide 2026-06-01

What this module does

Item Management is the foundational product catalog for SchoolCafé 2.0 — Implementation Specialists and Nutrition Directors configure every food and non-food item here before it can be used in Inventory, menus, recipes, or POS.

Key features

FeatureWhat it does
Item Card Captures name, category, storage, valuation, and identification fields; must be saved before any other tab activates.
Food vs. Non-Food selection Controls whether Serving Sizes, Nutrient, Menu, and Steps tabs are available.
Non-Fluid / Fluid selection Sets serving size weight unit — grams (Non-Fluid) or milliliters (Fluid); food items only.
Pack Sizes (up to 3 tiers) Defines purchase units with Sub-Qty links; max three tiers, + PACK disabled once reached.
Serving Sizes (food only) Defines the served portion; Amount = 1 enables inventory eligibility and Base Unit promotion.
Costing Enter cost on one unit; system calculates all others proportionally through the Sub-Qty chain.
GTIN/UPC Barcodes Optional barcode per pack level; available once at least one pack size or inventory-able serving exists.
Base Unit & Count Method Designates lowest inventory-countable unit; food items can promote a linked serving size to Base Unit (dual unit).
Nutrient Info tab Records all nutritional data; system auto-calculates Per 100g column and daily-value percentages.
Missing Value flag Marks a nutrient as (M) — Missing — rather than implying zero when label data is unavailable.
Ingredient Info card Sets edible yield factor, sub-ingredients, recipe directions, and Locally Grown / Buy American compliance flags.
Contribution Info card Records meal pattern credit (e.g., oz equivalent Meat/Meat Alternates) for menu planning compliance.
Allergen Info card Assigns standard allergen statuses (Contains, May Contain, Free From, Facility) and displays the district-responsibility disclaimer.
Custom Allergen Configuration Creates district-specific allergens in Item Configuration before they can be assigned to any item's allergen card.
Direct Menu Item (DMI) Converts a food item into a menu-placeable item with Menu Item Name, Button Name, and POS configuration fields.
Menu Serving Sizes & Exceptions Adds alternate portion sizes and restricts specific servings by meal pattern group or grade level.
Steps card (HACCP & Recipe) Records HACCP process (No Cook / Same Day Service / Complex Food), recipe steps, and Critical Control Points.
POS Pricing tab Sets student and adult pricing by eligibility type and site type; includes SET ALL PRICES shortcut, Allow Sale toggles, and taxable flag.
Images & Documents Attaches product images and supporting files (JPEG, PNG, PDF, Word, spreadsheets) with a Use as designation per file. [TO VERIFY: 5 MB per file limit — stated at [11:02], not confirmed in Jira AC]
Inventory Info tab Read-only view of active vendor contracts (vendor, unit, lead time, min/max qty, price) once item is published to Inventory.

Who uses it

Implementation Specialists (onboarding), Nutrition Directors / Item Management Super Users (full create and edit), Nutritionists / Menu Planners (nutrient, allergen, contribution, and HACCP data), and Menu Planning Managers (menu item and POS pricing configuration).

Common workflows

  • Create a non-food supply item — Item Card → Units Card (Non-Food) → Pack Sizes → Costing → Publish to Inventory
  • Create a food item — Item Card → Food/Fluid selection → Serving Sizes → Pack Sizes → Costing → Link Servings to Packs → Base Unit confirmation → Nutrient Info → Contribution Info → Allergen Info
  • Configure a Direct Menu Item for POS — Menu Info tab → CREATE MENU ITEM → enable Show on POS → complete POS Pricing tab
  • Add HACCP data and recipe steps — Steps card → ENABLE RECIPE → select HACCP Process → add steps and CCPs
  • Create and assign a custom allergen — Item Management → Configuration → Item Configuration → Custom Allergens → + NEW → return to item Allergen Info card

Important gotchas

  • Base Unit and count method lock after the first Inventory activity. Once an item appears in any count, receipt, or adjustment, Base Unit and Whole Numbers/Decimals cannot be changed without specialist intervention — configure both before activating the item.
  • Do not address Missing Costing until all pack sizes are added. Entering cost before all tiers exist produces incomplete calculations — add every pack level first, then click the alert.
  • Nutrient Info, Menu Info, and Inventory Info tabs are grayed out until the Units Card is saved. Save the Units Card first; only then do the downstream tabs become editable.
  • Item Category, Storage Category, and Valuation Group are conditionally required. Mandatory only when Inventory is enabled, but skipping them creates labor-intensive retrofitting if Inventory is enabled later.
  • Changing the HACCP Process clears incompatible Control Points. The system warns before removing CCPs not associated with the new process; confirm only if intentional removal is desired.

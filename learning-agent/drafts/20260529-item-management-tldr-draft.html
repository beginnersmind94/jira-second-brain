What this module does

Item Management is where District Admins and Nutrition Managers create and maintain every food and non-food item — including pack sizes, nutrients, allergens, and menu configuration — before the item can be used in Inventory, Recipes, or POS.

Key features

FeatureWhat it does
Item card Create items via +NEW; Item Description required; Tags optional.
Product Info attributes Three required dropdowns unlock Inventory; skippable for POS-only items.
Food / Fluid selection Two header questions control available unit types and serving fields.
Pack sizes Add multiple pack size tiers; Sub-Qty links tiers together.
Unit Identifiers and Base Unit Identifiers column shows per-unit flags; lowest pack size auto-assigned as Base Unit.
Costing Enter cost for one tier; system scales to all others; FMV required.
Inventory Readiness banner Always-visible banner shows whether the item meets Inventory requirements.
Whole Numbers / Decimals Sets whether the Base Unit allows decimal counting in Inventory.
GTIN / UPC Attach 8–14 digit barcodes per pack size; barcode icon appears in Identifiers.
Serving sizes (food items only) Add serving sizes with Amount, Description, and Weight.
Serving-to-pack link Sub-Qty on the serving row connects it to a parent pack size.
Merge / Base Unit promotion Merge prompt combines a matching pack and serving into one dual-use unit.
Yield Factor and Ingredient Info Yield Factor per serving on Units card; Ingredient Info holds recipe text.
Images & Documents Drag-and-drop drawer; up to 10 files; assign a Use As type each.
Nutrient Details Enter required nutrients; Vitamins A and C optional; zero parent auto-zeros children.
Contribution Info Records meal pattern credits (Fruits, MMA, Grains, Milk, Veg); skippable.
Allergen Info Standard allergens in priority order: Contains, Processed, May Contain, Free From.
Custom Allergen configuration Create district-specific allergens at IM > Configuration; duplicates blocked.
Menu Info tab / Direct Menu Item Create a Menu Item on the record; Show on POS unlocks pricing.
Menu Item fields Fields configure Menu Item name, category, marketing details, and carryover rules.
Menu Item Servings and Exceptions Add menu servings via +Menu Serving; mark exceptions per school or group.
Steps / HACCP and Control Points Select a HACCP Process, add prep steps, and add Control Points.
POS Pricing Student and adult pricing in separate tabs; Taxable on Adults tab only.
Inventory Info tab Read-only active vendor contracts preview; icon opens Contract Items in new tab.
Editability locking Base Unit and Whole Numbers/Decimals lock once used in Inventory transactions.

Who uses it

District Admins create and maintain all item records; Nutrition Managers complete nutrient, contribution, allergen, and menu details.

Common workflows

  • Create a non-food item — add Item Description, select Product Info dropdowns, add pack sizes, enter costing.
  • Create a food item — add pack sizes, add serving sizes, link serving to pack via Sub-Qty, confirm Base Unit.
  • Set up nutrients — open Nutrient Details, enter required nutrients, use Missing Value checkbox, save.
  • Configure allergens — record standard allergen forms; add custom allergens first via IM Configuration.
  • Make an item a Menu Item — open Menu Info tab, complete fields, enable Show on POS, configure pricing.
  • Attach documents — drag files onto Images & Documents card, assign Use As type per file.

Important gotchas

  • Add pack sizes before entering costing — system flags costs after all tiers exist.
  • Serving must link to a pack via Sub-Qty — unlinked servings leave item Inventory Ineligible.
  • Base Unit and Whole Numbers/Decimals lock after the first Inventory transaction and cannot be changed.
  • Custom allergens must be created in Configuration before they appear on item records.
  • Discrepancy: presenter placed Yield Factor on Ingredient Info [20:42]; AC says Units card.

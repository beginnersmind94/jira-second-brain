Item Management — Quick Start Guide

Overview

Item Management is the central catalog for every product a district uses in SchoolCafé — from food items served at lunch to the supplies that support service. Every other module reads from the item records built here.

Getting Started

Before creating items, ensure that Item Category, Storage Category, and Valuation Group are configured in district settings. These fields are required when Inventory is enabled; without them, the item cannot achieve Inventory readiness.

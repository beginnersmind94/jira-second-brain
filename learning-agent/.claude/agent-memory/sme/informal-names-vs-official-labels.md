| Presenter term | Official term (source) |
|---|---|
| "promote" (serving to base unit) | "Merge" / "Merge w/ Serving" (NXT-40778 AC) |
| "Base Unit chip" | Not named in AC; referred to as Base Unit step in NXT-37111 AC |
| "Missing Costing alert" | "Missing Costing" flag / prompt (NXT-37568 AC confirms concept, label from Description) |
| "Inventory Ready flag / green flag" | "Inventory Readiness banner" (NXT-53530 AC) |
| "Manage Costing button" | Not named in NXT-37568 AC; label from Description only |
| "Edible Yield Factor" on Ingredient Info | Now on Units card per NXT-47249 AC |
| "FOOD / NON-FOOD buttons" on Units card | "Food item and Fluid item questions" (NXT-35898 AC) |
| "three-tier max" | No AC-level statement; Description references multi-tier in NXT-37110 |
| "Show on Family Hub menus" toggle on Custom Allergen | Not in NXT-24128 AC; unsupported |

**How to apply:** When a presenter uses an informal name, look up the official name here before asserting a mismatch. Many labels are implementation-detail only (Description-tier), not AC-level.

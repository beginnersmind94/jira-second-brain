The prior failure mode for long-form guides is the Writer stopping after 4 of 10 required sections, treating the first four as "the main content" and dropping the rest.

**Why:** The 10-section template structure is long. Without explicit per-section mandates in the content plan, the Writer treats later sections (Exceptions, Troubleshooting, Related content, Sources) as optional or appendix-like.

**How to apply:** Every long-form content plan MUST include an explicit, inventoried list of what goes in each of the 10 sections. Use the phrase "Writer must generate ALL of the following" for the Full workflows subsection list. Add a note in the plan stating that no section may be omitted. The handoff checklist should include a checkbox confirming all 10 sections are explicitly covered.

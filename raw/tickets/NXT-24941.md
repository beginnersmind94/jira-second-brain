# NXT-24941 — PR- Receiving Site Planned Quantity

## Description
As a receiving site Manager, when I enter in my planned counts for my prepared items, I want those counts to reflect for the prep site to know how many servings I need.

## Acceptance Criteria
Only consider planned counts from receiving site when plan is complete. 

When planned counts are entered for prepared items, this information must be available for the prep site to consume along with Menu line, serving size, and plan information.

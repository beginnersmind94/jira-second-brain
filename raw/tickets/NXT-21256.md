# NXT-21256 — View Model Page

## Description
As a Model Build Manager, I want to see the configuration of a model in an easy to read, report-like format so that I can quickly scan through it to find the piece of information I am looking for.

*Application*: Face and Food Detection Website

*Flow:*

* On any given model in the grid, click the View button
* User taken to the Model Configuration page

 

*Assumptions:*

* Dependencies: Assumes you have done the stories for Build Model (v2), Add View Button to Model, and Add Model Configuration to Database, else you won’t be able to do this

 

*Requirements:*

* Create a new page with the title View Model
* It does not need a new menu link
* To get to the page, the users clicks View Model from Model Management page
*  
* All fields are read only, there are no action buttons except grid behaviors
* There are two groupings, a header and a grid below it
*  
* Header Fields above the menu grid
* All these fields come from Model Configuration (See story for Add Model Configuration to Database if need more info)
** Model ID
** Description
** Tags
** District Name
** User Name
** Status
** Menu Items Count
** Date/Time Created
** Start Date/Time
** End Date/Time
** Time to Build
**  
** -Linked Images list:-
*** -Image ID-
*** -Menu Item ID-
*** -link to image (don’t need if can easily build link from Image ID in code later)-
*  
* Grid of menu items that are linked to the model
** Usual sort, paging, filtering, export to excel functionality
** Columns:
*** Menu Item (the name of the food)
*** Metadata
*** Linked Images (a number that shows the count of linked images)
*** View Images – hyperlink button
**** On click, show the Images popup (see below)
*  
* Images Popup
** It is assumed that a popup is the easiest way to do this, but if there is a better way, let’s talk
** It is assumed that the user clicks the View Images hyperlink
** That will load this popup
** On load, get the images linked to that menu item and show them in an image grid similar to what you did for New Images
** Can click on the image and redirect to that Trained Images page with that image loaded
** Show the Menu Item as a title above the images
** Show a close button
** No other functionality necessary for now
*  
* Security
** Not necessary, covered by previous story

 

*Mockup*:

* None

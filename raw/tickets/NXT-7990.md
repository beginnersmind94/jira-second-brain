# NXT-7990 — INV - Orders - Order Detail -  Receipts Tab - Receipt Details (view-only) - Charges & Discounts

## Description
As an operations manager I want a Charges & Discounts section on the Receipt Details screen, so I can view any charges or discounts which were saved on an existing receipt. 

This section displays the total calculated values for the 2 charges and 1 discount which are configured per item on the vendor contract. It also displays the list of additional charges/discounts that are added via the ‘Additional Charges & Discounts’ configuration screen ([NXT-16382|https://primeroedgenext.atlassian.net/browse/NXT-16382]), which allows user to manually type a charge or discount amount and apply it to the receipt.

*Requirements*

* Add a section below the receipt item list, titled ‘Charges & Discounts’
* This section will list any charges/discounts which were applied to the specific receipt
** In this view-only version of the screen, do not display any charges/discounts that did not have a value on the receipt (greater than $0)
** Do not display the section at all if there were no charges or discounts on the receipt
* Charges & Discounts List:
** Provide header groupings to separate Charges from Discounts
** ‘Description’ column - this will display the description of the charge/discount
*** this would either be from the vendor contract (Processing Fee, Delivery Fee, Net Off Invoice)
*** or it would be from the ‘Additional Charges & Discounts’ screen, where users configure their own charges/discounts ([NXT-16382|https://primeroedgenext.atlassian.net/browse/NXT-16382])
** ‘Price’ column - this shows the saved value for the charge/discount as calculated or input on the receipt
* Provide an Edit button (separate story)

No need for ‘Add’ button at this time

## Acceptance Criteria
* Charges & Discounts will be displayed in a section below the receipt item list
* Charges will be grouped separately from Discounts
* Only the charges/discounts that had some value on the receipt will be displayed

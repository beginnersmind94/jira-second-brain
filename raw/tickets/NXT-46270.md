# NXT-46270 — IM - Items - Update Food Item Report & consider filter criteria

## Description
As an IM user I want the Food Items report to consider the filter criteria I apply to the grid search so I can generate the report for only the filtered food items I’m seeing in the results grid.

Implement the same approach from [https://primeroedgenext.atlassian.net/browse/NXT-44775|https://primeroedgenext.atlassian.net/browse/NXT-44775|smart-link]

## Acceptance Criteria
* Update the Food Item report to consider the column searches/filters when generating the report data
* Change the report button to *Food Item Report* (remove ‘All’)
* Change the approach of the report generation to remove the drawer panel
** report will *consider all page and grid filters*
* Make visual report updates
** Change the report title (on PowerBI) to the ‘Food Item Report’
** Move the Data Source label to be a report-level label rather than per item
*** it will either be Local or CNDB
** Remove Form from the header section 
** Change Ingredient Name label to Item Description
** Change Ingredient Code to Item # (or Item ID)
** Display other page-level filters somewhere on the report
*** Item Type, Status
*** Brand/Manufacturer search
*** Ideally show *ALL* applied filters, including grid searches
*** (optional) show quantity of items included in the report

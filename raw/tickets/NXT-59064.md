# NXT-59064 — PEI - Auth - Enable SSO in SC

## Description
As a SchoolCafé (SC) user with Insights access, I would like to be able to login to the new Insights website using my existing login credentials so that I don’t have to relogin

*Notes:*

* Users need ability to do SSO with PrimeroEdge Insights (PEI)

*Requirements:* 

* Write the code to enable SSO from SC to the new website
* The code should pass the following:
** Authentication token and info as necessary so that new site knows the user is authenticated
** District
** Profile information (name, etc.)
** Insights license
*** PEI needs to know that the district has the Insights license
* 
* Update existing Insights page in SC:
* Put a button on the page in top right:
** Label: Standalone Insights
** On click:
*** Do SSO with PEI
*** Redirect user to PEI if successful
** Visibility:
*** Only show to users in Technical Support role
*** Later can turn on for all Insights users

## Acceptance Criteria
* User is able to do SSO login from SC
* SC is able to send the authentication information to the new website
* New website is able to get the district, profile and licenses

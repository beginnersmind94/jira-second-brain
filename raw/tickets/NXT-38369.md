# NXT-38369 — Command Centre - Threshold Changes (ADP) - UI Integration

## Description
Currently ADP is not part of the command center dashboard

## Acceptance Criteria
Navigation: Command Centre - Threshold Centre

The user should be able to define the Average Daily Participation in UI against the following:

# All - a common threshold for all the region
# School Type - a common threshold for a specific school type
# Site Level - a specific threshold for one or more sites

The user should be able to save the defined threshold based on percentage ranges defined

the command Centre should show the threshold bar based on the defined threshold at Region level, Site type level and site level

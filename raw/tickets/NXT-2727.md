# NXT-2727 — Authenticate And Authorize User/Resource (APIs)

## Description
User after getting the successful Jwt token should be authorize to access the api resource based on the claims he allowed to.
Also Figure out the way to use the same Jwt token for multiple api resources for the same client.

## Acceptance Criteria
User should be authenticated and authorize to access the resource (api) or api(s) for the same client.

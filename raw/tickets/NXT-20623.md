# NXT-20623 — MP > Site Configuration

## Description
As a user in a district with a large number of sites, I need a more simplified method of setting up my site configuration. With 80 sites, it will take an entire day to set up the sites with the current format. I need this to be easier and faster, with bulk settings to accommodate me.

## Acceptance Criteria
I will be able to set up my site configuration for meal types and menu lines in record time by using a “bulk” configuration option. 

Example 1:  I have 88 sites that feed students
all 88 sites will feed lunch, menu line main
select site type "All" (result - all sites will be listed with checkboxes to select or deselect, top column will have option to mark a box that selects/deselects all)
select meal type "Lunch" (column labeled lunch will display)
select menu line "Main"
click "apply"

Example 2:  out of my 88 sites, 17 are high schools
14 of those high schools feed lunch at menu line "Grill"
select site type "High School" (result - all sites will be listed with checkboxes to select or deselect, top column will have option to mark a box that selects/deselects all)
select All / Deselect 3
select meal type "Lunch"
select menu line "Grill" 
click "apply"

I will still have the ability to add or remove a menu line when necessary, from a single site.
Example: a Single Elementary will no longer be using the "Grab N Go" menu line, and will instead be using the "Breakfast in Classroom" menu line.

On the “Site Group” window, if a user selects a site group, then a pop up will appear that displays all sites. The sites that are currently assigned to the selected site group will have marks in their checkboxes. The sites that are NOT currently assigned to the selected site group will have empty checkboxes. The user will be able to deselect sites that are marked, and select sites that are not marked, as needed. Then user will be able to click “Save” or “Apply” for the changes to take effect.

On the “Site Configuration” window, all sites will be displayed. A user will be able to select (by single or multiples, in drop downs) by Site Group, Meal Types, Menu Lines… and then with checkboxes for individual site selection… and click “Apply” or “Save” for their changes to take effect.

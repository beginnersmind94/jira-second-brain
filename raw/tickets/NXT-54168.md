# NXT-54168 — Acc - Settings - Charge Limits not displaying blank amounts

## Description
!image-20240823-190726.png|width=786,height=520,alt="image-20240823-190726.png"!

*Current Functionality*

* Charge Limits page does not show blank values
** Database may not have charge limits set, but the page is showing $0.00

*Requirements*

* User should see that charge limits have been left blank.
** Text: No Charge Limit
* POS needs to treat no charge limits existing the same way as a blank value.
** No charge limit if data doesn’t exist.

## Acceptance Criteria
* User is informed if no charge limit exists / blank
** Text: No Charge Limit
* POS updated to treat charge limits not existing as a blank value. (no charge limit)

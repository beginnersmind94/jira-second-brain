# NXT-69433 — Fin - Show data points associated with an account on the GL Accounts page

## Description
As a user, I want to see the data points my account is associated with.

## Acceptance Criteria
* Add a rule action button to the grid on Configuration > General Ledger Configuration > GL Accounts
** Clicking it opens a side drawer showing all the rules the data point it is mapped to if any is involved in
** If that GL account is not involved in any rule, don’t enable the action button. 
* Clicking this should reveal the data point (if any) that GL account is mapped to
** If not mapped to any, display “This GL account is not mapped to any data point. Please go to the Account Mapping page and map it to a data point if you need it.”
* This is view-only

# NXT-19417 — Production- Reports- Menu Costing (Production Record Link)

## Description
As a Kitchen Manager, I would like to be able to see my production cost per item on my menu line for the day to know what my average cost and total cost is. 

This link will appear on the Production Record page only when production status is set to atleast record complete. It will belong with the other report links on this page. 

Column headers: 
Recipe code
Description (Menu Item Name)
Serving Size
Amount Prepared (this is produced Quantity)
Unit Cost
Purchase Cost
Left over Today

Items should display in their respective categories just like they do on the production page. Each category will have its own food cost total per unit and purchase cost.

There also needs to be a grand total for all the categories combined as well.

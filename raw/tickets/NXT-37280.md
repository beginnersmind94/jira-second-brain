# NXT-37280 — System - Management - Periods - Enhancements (*WIP*)

## Description
As a District Admin I want enhancements to the Periods page so I can properly manage my district’s financial and functional periods.

*Requirements*

* Separate the Generate Periods and Manage Periods in some way that allows different permissions
** Generate Periods is a District or higher level page
** Manage Periods is a Site and District level page (depending on district)
** Many Site level users will need access to the Manage Periods without having any access to Generate
* Change the defaults for Create Periods
** Flip the order of available values upside down 
** Monthly first (and default) then more frequent options
* Restore proper ‘All Sites' option for Manage Periods
** There is a default page for managing the periods of all sites at once, but this is not accessible once a single site is selected

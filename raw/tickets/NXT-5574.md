# NXT-5574 — Applications - Pending List - Unidentified Students

## Description
*User Story*

As an applications processor I want to see all pending students that are on an application.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=424768%3A861&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=424768%3A861&scaling=min-zoom|smart-link] 

*Requirements*

* Populate the Pending tab on Applications with a list of pending students.
* Show information in the existing grid for applications that are part of the pending list due to an unidentified student.
* The new grid has the following information on pending students. (Note: this information is from the application, and is not currently matched against any student information)
** Student ID
** First Name
** Last Name
** Site Code
** Grade
** Date of Birth
** Application #
** Application Status
** Date Added
** Matched Status
*** This is a green check-mark for any that have a pending match.
** Look Up
*** This is an action - user can search the student database to try and identify this application info against student info.
** The grid will need search filters for the following:
*** Student ID
*** First Name
*** Last Name
*** Date of Birth
*** Application #
*** Date Added
** Sorting should also be possible on the grid for the columns above.
* Matched Status, Look Up and Edit are handled as part of [https://primeroedgenext.atlassian.net/browse/NXT-5220|https://primeroedgenext.atlassian.net/browse/NXT-5220|smart-link] .

## Acceptance Criteria
User can see all pending students on applications for the current academic year.
User has filters for the specified columns.

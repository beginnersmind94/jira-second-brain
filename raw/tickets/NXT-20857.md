# NXT-20857 — DataQuery Framework - UI Save DataQuery Component

## Description
End-Users would like the ability to save the the columns, the column positions, the specified data filters, and the sort order of the data in their DataQuery-based tables.  This story would result in a UI component that would allow the end-users to save these configurations.

See the following attachment as an example.

!image-20211019-004052.png|width=539,height=300!

## Acceptance Criteria
* Module development teams shall have a UI component to use on their DataQuery web pages that will allow an end-user to save the configurations that they have currently set on their UI.  This includes:
** Which columns are displayed
** The positions of the displayed columns
** The filters that have been applied to the dataset
** The sort order of the data
* Saved configurations shall be persisted as a DataQuery
* Duplicate names shall not be allowed.
* The data entered by the user shall be validated and presented to the user if a violation occurs.
* UX treatment for this component shall be decided by the UX team

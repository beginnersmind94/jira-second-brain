# NXT-52150 — INV - Physical Inventory - Review Completed Inventory - Summary

## Description
As an Inventory Supervisor I want a consolidated screen to review a site’s physical inventory for a period so I can reconcile a single time regardless of how many physical inventories were completed.

## Acceptance Criteria
* Create a new page to review site’s submitted physical inventory quantities and compare against their perpetual quantities
** this will replace/eliminate the current page used for inventories in Pending Reconciliation and Reconciled status
** rather than review individual physical inventories, this page will display all perpetual items and their count for the period, regardless of which inventory it was counted on
* Provide two tabs: Summary & Items
* Make the Summary tab very similar to the Summary tab from Physical Inventory Details
** Progress bar, Storage Category grid, sticky footer
* Change the Progress Bar to consider all physical inventories for the period in the calculations for % physical count
* Change the Storage Category grid for the possibility of multiple inventories being submitted
** change the ‘All Storage Categories’ label to instead be a header for the inventory (ex: ‘June 2024’)
** Display a ‘View Inventory’ text link at the end of the cell for each inventory
* If multiple inventories are present for the period, provide a Totals row (reference mockup)
* Provide expand/collapse icons for the Totals row and for each included physical inventory
** the expand arrow will show the Storage Category breakdown for that record
** only allow one record to be expanded at a time
* Update the grid columns
** display *Status* before Counted Items, to show the status of that inventory
*** for Totals row, reflect the status shown on the Status list page
** display *% of Perpetual* after Total Perpetual Items, to show the % of items on each inventory
* Provide a sticky footer
** left aligned Back button
** right aligned Reconcile Inventory button

# NXT-20460 — Research - Platform - Security - Add Failed Logins Method

## Description
As a School Cafe user, system shall prompt me to reset password after 5 failed login attempts across all authentication methods

*Requirements:

* *When user logins for 5 failed attempts, system prompts for reset password function with a message "Your user account has been locked due to max failed attempts. Please reset your password" 

+*UX Mockup:*+

[https://www.figma.com/file/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=1067%3A14770|https://www.figma.com/file/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=1067%3A14770|smart-link]

## Acceptance Criteria
* System shall prompt user for reset password after failed login attempts
* System shall allow maximum of 5 login attempts
* System shall lock the account after 5 failed login attempts
* User shall see “Your user account has been locked due to max failed attempts. Please reset your password” message after failed attempts

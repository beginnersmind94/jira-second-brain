# NXT-3862 — Add Item - Ingredient - Add Item Info - Common Info

## Description
*User Story*

As a Item Management Super User/Menu Planner/Food Service Director/Nutritionist, I can use the ‘Add New’ button to manually add an item and input the general fields

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1&g=1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1&g=1]

*Requirements*

* *Common Info*
* Data source
** Required
** Auto sourced but editable
** Where the data resides. If local, the data for this is editable. If GS1 or CN database is the source, then other fields are not editable unless a copy is made.
** If a new item added manually, default datasource is ‘Local’
* GTIN/Broken unit GTIN - (14 character no alpha)
** Optional
** Auto sourced from shared datasource, such as CN database (if applicable) but editable
** GTIN describes a family of GS1 (EAN.UCC) global data structures that employ 14 digits and can be encoded into various types of data carriers.
* Product Code
** Optional
** Auto sourced from shared datasource, such as CN database (if applicable) but editable
* Manufacturer Name
** Optional
** Auto sourced from shared datasource, such as CN database (if applicable) but editable
* Brand Name
** Optional
** Auto sourced from shared datasource, such as CN database (if applicable) but editable
* Commodity
** Optional
** Available Options - “Yes” or “No”
* Item Status
** Options
*** Active
*** Inactive
** Default to ‘Active’

## Acceptance Criteria
* Above fields are visible
* Fields that are sourced from a shared data source should be autofilled from the shared datasource
* Default Item status = Active
* Required fields are
** Datasource
** Item Status
* Check for duplicate GTIN and Product Code per data source

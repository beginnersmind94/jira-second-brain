# NXT-36491 — IM - Recipe Rework - Step 0 Enhancements

## Description
As a recipe menu planner I want more information in Step 0 so I can manage all pre-steps information in one place.

*Proposal:* Change behavior of Step 0 to be full-card like the other steps/CCPs, and there will be enough space for all the new fields and possible future ones.

*Requirements*

* Similar to HACCP Process, we need to move additional header fields into the Steps card
** We will be doing similar things for the Serving Size fields with the new combined Units card
* Move Prep Time and Cook Time to Step 0 space
* Add Pre-Preparation Instructions to Step 0 
** Text field, 500 characters
** This is a freetext entry field where users can add comments on required actions prior to recipe preparation/cooking
* Move Prep Time and Cook Time to Step 0 space
* Add ‘Critical Control Point’ label before Critical Limit Text (Report for reference)

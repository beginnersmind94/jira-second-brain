# NXT-19166 — Production Dashboard Simple (V1) Page Set up 

## Description
Creating an interactive dashboard for users to see when entering the production module will provide them with important information about their day to day operations. 

The dashboard must contain the following information. 
1) Meal Count History at District, site group and site level
2) Cost history at District or site group level
3) Deviations from Plan counts to Actual Counts
4) Average Meal Participation % per Meal at the site level
5) Production Status dashboard to give district/site manager quick overview of production operations

# NXT-35143 — INV - Order Detail - Quick Receipts - Provide Re-open Order button

## Description
As an Inventory manager I need the ability to re-open closed orders so I can make adjustments in the case of mistakes or unexpected changes.

*Requirements*

* Provide a Re-open Order button for Closed *Quick Receipt* orders
** this will be controlled by permission eventually; not every user will have access
** rather than a new button, the Re-open Order button can take the place of the disabled Close Order button
* Updating items after Reopening
** Return any remaining Due Quantity back into In-Transit Quantity
* Ensure all existing validations, restrictions, etc. are enforced
** Orders cannot be re-opened in a closed period
** orders cannot be re-opened if a Physical Inventory has been completed for those items since the order was closed

## Acceptance Criteria

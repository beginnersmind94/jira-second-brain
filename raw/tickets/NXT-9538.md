# NXT-9538 — Data Exchange - File Import Job Actions

## Description
*Summary*

As a PrimeroEdge user, Once I preview import data and select Import option, I should be able to take any available actions on File Import jobs displayed on the File Imports table so that I can take appropriate action as needed.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=331631%3A216140&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=331631%3A216140&scaling=scale-down-width|smart-link] 

*Requirements*

* User should be able to go to the detailed import job when they click on “VIEW” action available at each row in the import job table
* View Job details should display AUDIT Log history of the import job based on the Import Type
** Should display “Last Action”, User, Configuration Name and “Log Time” headers
** All actions taken on that job should be listed in a pop up window
** Pagination is enabled for records more than 10 rows.
* Bulk actions are available for user to select multiple jobs and select actions available above the import configuration table
* *Cancel action should be enable only when job is in “In Progress” state & in “Staging” state*
* Re-run action should be available when job is in “Done” or “Error” state
* Refresh action should be available at any time to update latest status/progress.
* Expanders are displayed for each import job to provide the detail import job view
** Detailed progress view based on import type should be displayed
** For ex: Number of students imported successfully and pending

## Acceptance Criteria
* Verify available actions are displayed above the file import jobs table
* Verify bulk actions are enabled based on the status of the job selection
* Verify detailed import job audit log is displayed when user clicks on "View" 
* Verify expander of the import job displays progress/history of import

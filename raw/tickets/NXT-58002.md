# NXT-58002 — PR - Consolidated Tab - Removing Non prepared menu items from menu

## Description
As a production manager, Non prepared items should be removed from the consolidated tab at any point of time if it is removed from the menu or from the plan. Since this is just for informational purposes.

## Acceptance Criteria
* Remove non prepared menu items from the plan if removed from the menu even if the prep site plan has already been consolidated
* If categories are associated and no more menu items are available in the category, remove the category as well 
** Currently we are handling this on the receiving site plan so we need to implement the same thing for the prep site
* Since non prepared items are not really part of the prep site functionality and we are displaying the non prepared items as informational purposes, items can be added or removed to be shown on the consolidated tab for NON prepared items at any time
** If additional items are added to the prep site plan after consolidated, we can still show the new non prepared items on the consolidated tab this should NEVER apply to prepared items
* This should also effect any report that consumes prep site information (Grocery List, Plan Summary  etc)

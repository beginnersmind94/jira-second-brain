# NXT-17959 — MP- Default categories in Menu Planner

## Description
As a user I would like to visually see the items I add on my menu planner by categories.

Mockup:

!image-20210819-150514.png|width=1391,height=505!

 

User needs to be able to visually see what menu item is in which category while on the Menu Planner. 

The page also needs to display some sort of Glossary that indicates what color belong to what category when the user sees it added to the day.

## Acceptance Criteria
Colors match categories as specified from Item Management menu item Category configuration. 

Maintain UI level configuration for color scheme
Meat M/A - #544AB4
Grains - #F08B46
Veg - #3AC191
Fruit  - #EB4E93
Milk  - #459BEA
Uncategorized (White or Grey) - #CCCCCC

Group Menu items by menu item category

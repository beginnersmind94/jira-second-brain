# NXT-18473 — Production- Reports- Items Added to Production Records (UI)

## Description
This report will display to users all the menu items that have been added onto the production records. 

Page Display needs to have this criteria:
Site Code
Site
Site Group
Meal Type
*From* and to *Date*

Criteria for Columns to display information:
Serving Date
Meal Type
Menu Line
Category
Menu Item with ( Recipe#)
Serving Size
Comment ( IF no comment, Display who the item was added by and a timestamp)

Row will display Site  Code and Site.

Items will display in order of Serving date, then Breakfast or Lunch with Breakfast being first.

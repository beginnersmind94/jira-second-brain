# NXT-66910 — INV - Order Details - Consider Distribution QOH Setting on Edit

## Description
As a distribution manager I want the ordering sites to be limited by the QOH setting on Order Details so they cannot circumvent the ordering rules through edit functionality.

!Order Details Edit Quantities for Internal-20250930-132048.png|width=679,alt="Order Details Edit Quantities for Internal-20250930-132048.png"!

## Acceptance Criteria
* Update the Order Details to consider the allow/disallow value of the Exceed Distribution Site QOH setting
* Validate edited quantities against distribution site available quantity when editing
** do not allow edited quantities to exceed available quantity
** show the available quantity in parentheses when editing an item
*** display after the Quantity field; e.g. “(42 available)”

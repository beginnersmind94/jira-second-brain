# NXT-14066 — IM- Catalog Integration - Settings 

## Description
As a Nutrition Department administration user I need to set what School Cafe Item fields can  be synchronized and  updated with Global Catalog information.

## Acceptance Criteria
* -This functionality is role based-
** -This option should be visible to users with this role assigned.-
** -This functionality should be available to users with this role assigned.-
* Upon selecting Global Catalog under Configuration  option from the left navigation pane the following criteria applies:
*  There should an option to select a data source where  the settings are going *to be applied to*
* By Default the “Data Set”  should be set to  “Local”
* There should be an option to select preferred manufactures from a List Box with Multi selection list.
** more that one manufacturer can be selected
** selection should be stored as preferred manufacturers
* There should a section for Item Details Configuration.
** This section should display the fields that can be set for update (see filed list below).
*** Item Name
*** Item Short Name
*** Manufacturer Name
*** Brand Name
*** Product Code
** These fields be can set to  ON or OFF. 
*** ON for update
*** OFF for not update
* There should be a section for Nutrition details configuration.
** This section should display the fields that can be set for update (see list bellow).
*** Serving Measures (tab)
*** Nutrients (tab)
*** Contributions (tab)
*** Allergens (tab)
** These fields could be set to ON or OFF
*** ON for update
*** OFF for not update
* By default ALL fields should be set to *NOT TO ALLOW*  updates.
* I should be able to save the configuration and continue to Item Management landing page
* I should be able to Cancel changes, keep previous and return to Item Management landing  page

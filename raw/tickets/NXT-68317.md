# NXT-68317 — K12 - WS - PE Announcements

## Description
As an Insights role user, I would like to see my PE Announcements  so that I can view my important info on the go.

h2. Notes

* We have integrated PE messages, but need to do Announcements also

h2. User Flow

* User logs in and is redirected to home screen
* User clicks Workspace and is redirected there
* Screen shows Announcements 
* User clicks on Messages and is redirected there
* User is able to view a list of Announcements 
* User is able to click on a Announcement and see the details

h2. Requirements

* Integrate the PE Announcements API developed by Lalita’s team
* If user is from PE, do all these things:



_Home Screen_

* Add a new card for Announcements 
** Label: Announcements 
** Show count of new Announcements 



_Workspace Screen_

* Add a new card for Announcements 
** Label: Announcements 
** Subtext: View your system Announcements 
** Show count of new Announcements 
** Use same Inbox icon
** On click:
*** Open the Announcements screen
*** Do the same way as Messages card does



_Announcements Screen_

* Create a new screen
** Label: Announcements 
* To make it easy, copy code from Messages screen, then:
* Connect it to PE APIs
* On Message click:
** Open the Announcements Details screen



_Announcements Details Screen_

* Create a new screen
** Label: Announcements Details
* To make it easy, copy code from Messages Details screen, then:
* Connect it to PE APIs instead of SC Inbox Details
* On Delete click:
** Call the delete API and remove from the app
* On Read
** Send a read receipt to the PE system and mark it as read

## Acceptance Criteria
* PE users can:
** See count of unread Announcements on home page
** See their list of Announcements 
** See the details of an Announcement
** Delete a Announcements 
* PE is notified of read receipts

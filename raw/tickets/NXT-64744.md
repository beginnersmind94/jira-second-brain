# NXT-64744 — WS - Insights - KPI - Menu Performance

## Description
As a Workspace Insights user, I want to view the menu analysis on a KPI card, to get a quick glance.

Notes

* story copied from [https://dev.azure.com/Cybersoft-Technologies-Inc/PrimeroEdge%20Classic/_workitems/edit/107328|https://dev.azure.com/Cybersoft-Technologies-Inc/PrimeroEdge%20Classic/_workitems/edit/107328|smart-link] 

*Requirements*

* On the Insights page in Workspace:
* Add a new card for  Menu item analysis. 
** Overall Performance 
** Participation Score
** Waste (%)
** Waste ($)

* Menu Item Analysis KPI will not be added to benchmark config
* The card will not have the up/down arrow nor red/green color(ignore the green color border in the below  mockup card ); 
** We will show it grey since there are no benchmarks for it.
* Card's data will not be in the performance grid or trend chart since there are not benchmarks for it (card only)
* This KPI will be present in customizing the dashboard view. 
* Clicking on the card will navigate the user to similar page we will have for PrimeroEdge users . Existing page URL for SchoolCafé is this  [https://qacnworkspaceui.perseusedge.com/menu-planner|https://qacnworkspaceui.perseusedge.com/menu-planner]

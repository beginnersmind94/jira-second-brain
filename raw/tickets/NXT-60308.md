# NXT-60308 — Digitize AI - Document upload page

## Description
As a school manager I want the ability to upload images and PDFs into the Digitize page so that I can have it parse my documents/images for data points that will update various forms and pages within my system.

As a school manager I want the ability to capture (take a photo) from my mobile device (phone, tablet) so that I can the digitize page can parse my images and capture the data to update various forms and pages within my system.

!image-20250324-151422.png|width=1169,height=777,alt="image-20250324-151422.png"!

Uploaded and/or captured images/PDFs will show a progress and processing row.

!image-20250324-151850.png|width=1169,height=777,alt="image-20250324-151850.png"!

See “Document Upload Messaging” for post-import actions.

## Acceptance Criteria
* Document control must allow for upload of images and/or PDFs
* Document control must allow for image capture when users are using mobile devices

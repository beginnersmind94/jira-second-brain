# NXT-3634 — Edit Inspections - Dynamic Table Question Type

## Description
*User Story*
As an Inspections user, I can see Dynamic Table Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Dynamic Table Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Table with all columns, content, and the ability for Inspectors to add a row for each response
*** Rows can be removed if added accidentally
** Whether question is required
** Notes Section
** Upload Media
** Note: Point values are *not* displayed here
* Each segment follows the proper rules for user response
** Text
** List
** Rating
** Date/Time
** Signature
** Location
* Choices are properly saved
** If question is required, Inspection cannot be submitted if no selections are made
* Choices can be changed if Inspection has not been submitted
* If any part has a score associated with it, the score is properly updated according to the selection being made.
** Rating or List follow the same rules as the individual Rating and Dropdown question types
** Because Dynamic tables are fluid, each row adds to the total possible score when it’s added

## Acceptance Criteria
* Dynamic Table question appears in Inspections
* All segments can be selected, saved, and edited
* Rows can be properly added and removed
* Score updates properly if present

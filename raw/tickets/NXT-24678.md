# NXT-24678 — INV - Vendors - Contracts - Edit Delivery Schedule

## Description
As a FSD I want a dedicated screen for editing the Delivery Schedule so I can see many sites at once while managing the schedule.

*Requirements (Ideas)*

+Contract Details+

* remove Delivery Schedule grid from main Contract Details page
* add ‘Delivery Schedule’ label to main Contract details card
** just the contract level schedule for the vendor; not the site level
** *view-only* here; will be managed on dedicated page
* provide an icon/button on the Delivery Schedule label to open up a new screen dedicated entirely to the Delivery Schedule
** this will sub-navigate from the Contract Details

+Delivery Schedule+

* move entire Delivery Schedule grid to the dedicated page
* provide the same search/filter options from the Add Contract functionality ([NXT-23953|https://primeroedgenext.atlassian.net/browse/NXT-23953])
** search and apply, repeat, etc.
* show the contract level delivery schedule (functions as a “parent” schedule to be applied to sites)
+*Messages*+
1.Cancelling on Edit: You have unsaved changes. Are you sure you want to Discard?
a. Just make sure this is the standard message we use everywhere. That way we can update it later because we shouldn't capitalize Discard
 2.Continuing w/o ds: Proceed without configuring Delivery Schedule?
a. This is good
3.Overwriting site entry : De-selected day has some site entry. Proceeding further will loss respective day in the site
a. Change this to "There are sites configured with the deselected day(s). Proceed with overwriting?"

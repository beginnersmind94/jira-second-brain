# NXT-62142 — Digitize AI - Document upload control (embedded)

## Description
As a school manager I want the ability to upload images and PDFs into the Digitize control from various pages within the SchoolCafe application so that I can have it parse my documents/images that I capture/upload.

The upload and processing control should be converted to a reusable control that can be embedded within pages of the application. We can trigger with a “Digitize” button and it should do a slide out panel with the upload/process control for digitize. This allows the ability have the feature feel native to our application and users can perform them where it seems appropriate (instead of having to navigate to a different site).

Ex) Production record page, Item Management page (for onboarding Items/Recipes), etc.

## Acceptance Criteria
Document control must be a framework/control to be embedded in the SC platform

# NXT-43461 — Elig - Eligibility Descriptions - State Income Descriptions

## Description
As Anna the Application Processor I want state income guidelines to use the labels as described by the state instead of generic ‘state free’ and ‘state reduced’ labels.

*requirements*

* Add a State Eligibility labels section to the Eligibility Descriptions page
* User can change the description for State Free and State Reduced
* Values are recorded and are shown back to the user
* Update the tables on the page to be collapsible

# NXT-2789 — View Inspections - Filtering

## Description
*User Story*
As an Inspections user, I have a way to filter the View Inspections screen by Date, Status, or through a dynamic filter.

*Mockup*
[https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1|https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1]

*Requirements*

* Status filter exists in a dropdown
** Options: All, Scheduled, In Progress, Pending Approval, Re-Inspection, Passed, Failed
* Date filter provides a range of dates
** Dates are selected using a calendar widget
** Either or both can be open ended
* Dynamic search allows user to search for any other available parameters
** There is an autofill function that displays possible filter items and what category they fall under
** Numerous categories can be used to filter
*** Inspection Name
*** Template
*** Inspector
*** Site
*** Site Rep
* Show Only Mine - Toggle for Supervisor level that goes between displaying all of the district’s inspections and the user’s personally assigned Inspections
** Not present for users who cannot view Inspections that are not assigned to them

## Acceptance Criteria
* Status filter is present, each status can be selected, and can filter out Inspections by status
* Date filter is present, can be set, and can filter out Inspections by date
* Dynamic filter is functioning and can filter out Inspections by each given parameter
* Show Only Mine toggle is functioning and can toggle between global and personal view

Note: Using dummy data is acceptable to demonstrate completeness

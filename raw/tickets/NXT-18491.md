# NXT-18491 — Production- Reports- Recipe Signage (UI)

## Description
This report will give the user a way to visually see their recipe nutritional information as well as allergens and picture if it has been added to the item. 

Criteria to generate report:
Site Code
Site
Meal Type
Serving Line (Default to all)
Start date and end date (Default to 5 days, which is a typical serving week)

Criteria in report: 
Menu Item Name
Nutritional information based on 1 serving while listing serving size
Allergens
Made with ( pulls this information from Recipe)
Picture of Item ( if available)

# NXT-2065 — Production - Reports - Pre-Production Costing (Details by Site)

## Description
The Pre Production Costing Report gives the district information regarding the cost of foods in relation to the completed Production Plans from their sites. It looks at the quantity of ingredients calculated to be used to generate the servings, and pulls the costing from the Fair Market Value listed with the products associated to the ingredients.

Example:  If my Production Plan states that I plan to serve 23 apples, then the software would calculate the Fair Market Value of those apples, divided by how many broken units contained in the whole unit, and multiply that broken unit cost by those 23 apples.

The district should be able to pull this report with the following details:
Site: Default to All

Site Group: Default to All

Meal Type: Default to All

From Date

To Date

Report Options (First Criteria) : Details by Site, Details by Site Group, Summary Only by Site Group

## Acceptance Criteria
As a user I would like to see my pre-production costing (Production Plan Costing) This report should take all the Menu lines for whatever meal type that is selected and break down the plan and cost.

DETAILS BY SITE

Display the date range for the report as well as the type of report it is (Details by Site)

# Display the following Columns

* Recipe Code
* Menu Item
* Serving Size (if there are multiple serving sizes, display items on separate lines and use meal count for that serving group+serving size combination)
* Unit Cost (Cost per serving size)
* Amount Planned (planned count on the item)
* Purchase Cost (Cost per serving size X planned count)

  2. For Each Menu Line We must display

* Site Name: (Site Name), Meal Type: (Breakfast, Lunch or Supper or Snack) Serving Date: (Whatever date is the menu line from), Menu Line: (Menu Line Name), Meal Count: (Total Planned Meal Serving Counts)

  3. Display All Menu Item Categories separately along with the items that belong to that category

  4. Each category will need to have a line for "Food Cost Per Meal" row that will give the average food cost per meal per category.

EXAMPLE 1

*Milk Category*

*$0.2376 per carton <choco> X 250 planned portions =$59.40*

*$0.2376 per carton <white> X 50 planned portions =   $11.88*

*Total of all "planned" counts (300) with cost = $71.28*

*$71.28 / Total Meals (310) = $0.2299*

EXAMPLE 2

*Fruits Category*

*apple juice $0.1593 each X 250 planned portions =$39.81*

*peach cup   $0.5186 each X 250 planned portions = $129.65*

*Total of all "planned" counts (500) with cost = $169.46*

*That total divided by the total number of planned meals (310)*

*$169.46 / Total Meals (310) = $0.5467*

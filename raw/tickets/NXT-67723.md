# NXT-67723 — GC > GDSN Connect (SC)> Upload items page - Item updates tab:  Updating single item from item slide out panel

## Description
Accept all selected updates button will update the selected fields for local datasource item.

## Acceptance Criteria
Functional flow: Users navigate to upload items > item updates tab

# Item updates tab: select an item by clicking on item hyperlink
# Review the available updates for all sections (item details , nutrients, allergens, meal credits , images etc..)  
# Select the field’s available updates that they want to accept and update their local data source item. 
# Save the selected updates by clicking on Update now  button on the slide out panel 
## Asks for confirmation by showing a message “Are you sure you want to update this item now” yes/no 
### Clicking on no button will cancel the action. 
## clicking on Yes:  system will update the local data source item and display a green success message. 
## once updated , this item will be removed from the main grid until it get another update from GDSN in the scheduled updates
# Users can see the updated values on the selected item when they view it from item management > items page. 

In scope: item Basic Details (including GTINS) + Serving Size + Nutrient + Allergens and Meal credits

 Out of Scope- User cannot update pack sizes, vendors, Vendor Prices, Vendor Item Number

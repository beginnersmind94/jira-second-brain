# NXT-9805 — Data Exchange - File Import: Review Step

## Description
As a PrimeroEdge user, I would like to have an intuitive screen that allows me to verify all of the fields in PrimeroEdge are rightly associated with the fields in my first uploaded file, so that I can confidently begin importing subsequent files.

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A239433&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A239433&scaling=scale-down-width|smart-link] 

*Requirements:*

* When the user has clicked ‘*Next*’ from the *File Upload step*, redirect them to a P*review* step.
* On this step, display a preview that shows a table that contains:
** The Headers of the file.
*** Display the header values as the fields that they will mapped to in PrimeroEdge. i.e. if there is a column on the config file called ‘DOB’ and the user has mapped it to ‘Date of Birth’ in PrimeroEdge, display the header for that column as ‘Date of Birth’ in the preview.
** The first 10 rows of data in the file.
* Beneath the table, display 2 *buttons* are enabled
** +*PREVIOUS*+  button- Allow the user to go back to the File Upload step.
** +*IMPORT FILE*+
**# Save the configuration to the database
**# Display a pop-up with a *checkmark* & message “Your file Import has started” in next line “To see progress, click on Monitoring Page”
**## *Title*: Status of File Import
**## *Message*: Your file Import has started \!
**## *Message line-*2: “To see progress, click on Monitoring Page”
* Display an *‘MONITORING PAGE*’ button that, when selected, takes the user back to  Monitoring page.
* If possible, {color:#ff991f}*HIGHLIGHT*{color} the file import row that user has just performed with “In Progress” status

## Acceptance Criteria
* Verify that when the user has clicked ‘Next’ on the File Upload step (Data Import), user is taken to Preview step.

* Verify that on this step, we display a preview that shows a table that contains:
-The Headers of the file, which displays the header values as the fields that they will mapped to in PrimeroEdge (i.e. if there is a column on the config file called ‘DOB’ and the user has mapped it to ‘Date of Birth’ in PrimeroEdge, display the header for that column as ‘Date of Birth’ in the preview).

-The first 10 rows of data in the file.

* Verify that beneath the table, we display 2 buttons:

-PREVIOUS - Allow the user to go back to File Upload step without any confirmation

-IMPORT FILE - when user clicks, Status import confirmation message (as specified in requirements) should be displayed with "MONITORING PAGE" button

* Verify that clicking on  "MONITORING PAGE" button takes user to "File Import Monitoring" page with highlighted row that user has just started the import

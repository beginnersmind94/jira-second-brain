# NXT-14741 — IM - Edit allergens


## Description
As a User I would like to have Edit Inline Capability for the Allergen Info 

!image-20210712-140821.png|width=100%!

## Acceptance Criteria
Given a User is on the Item Page 

And an item is selected from the search result 

And viewing the Allergen section 

When Edit button is tapped 

Then display the User to edit the below information 

* Standard allergens 
* Contains 
* Processed in facility that also processes 
* May Contain 
* Free Form 

And able to “*Save*” Changes 

And upon “*Cancel*”, Discard the changes made 

And Exit the Edit Mode

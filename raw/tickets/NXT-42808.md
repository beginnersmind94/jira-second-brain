# NXT-42808 — SIS Auto Sync – Guardians – Powerschool - Backend

## Description
As a SchoolCafé admin, I would like to import multiple guardians so that I can have that information on the student record when I need it

*Notes:*

* This story is to do the backend so that the front end can be developed in the next story and relates to doing everything necessary so that the front end team can give the user the ability to import multiple guardians
* Another story exists after this one for the front end
* Then another story for execution of the import and exception handling

 

*Requirements*:

* Write all the backend code to consume Contact/Guardians from PowerSchool  APIs
* This will be needed by front end team for the APIs tab
* Will also be needed for Preview
* Will also be needed for execution of the import

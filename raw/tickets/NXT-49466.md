# NXT-49466 — FO - Templates - Field Updates - Disclaimer Message

## Description
As Dora the Director I want to update the disclaimer message used across my letters.

*requirements*

* User can access a ‘Fields’ page in templates and see available fields to edit.
* User can see the Disclaimer field on the Fields page.
* User can edit the Disclaimer field in a text editor to maintain formatting.
** User can see the disclaimer field for each configured Language in the region.
** Copy the English field over to any new languages that are added.
* User can save the Disclaimer fields.
* Clean up of Disclaimer field storage to remove template dependency.

## Acceptance Criteria
* Fields page added off of System > Templates.
** Fields button on the top right. Can be a full page or a drawer.
* User can see a list of fields and the current English text.
** Grid, columns: Field Name, English Text
* User can only see the Disclaimer field for the time being.
* User has an Edit option for fields.
** User can see each configured language.
** User can edit the field text for each language.
*** User has a text editor for this so they can make use of formatting.
** User can save or discard their changes.
* Clean up of Disclaimer field storage to remove template dependency.
* As part of rollout - need to check which languages are currently configured and if a disclaimer already exists in a language other than English/Spanish.

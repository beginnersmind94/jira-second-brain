# NXT-62059 — INV - Distribution - Running Notes

## Description
As a Distribution Site Manager I want a comments/note section so my personnel can communicate info to others working on the same distribution, or capture info which may be reviewed later.

## Acceptance Criteria
* Connect the Distribution Comments/Notes icon to a comments dialog
** 500 character limit
* Label these as “Distribution Notes”
* These notes will only be displayed within the Distribution and will not be used to populate any order/receipt comments
** this will be always editable (until Distribution is Closed)
** there will be no history or anything
** any user with access can modify the notes

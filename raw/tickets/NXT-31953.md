# NXT-31953 — Create Menu Group for Imports and Export Menu Items

## Description
The objective of this epic is to consolidate the 5 import/export related menu items into one menu group so that it is more intuitive and focussed for an end-user.

## Acceptance Criteria
* A new menu group named “Integration” shall be created at the same level as “Sites & User”, “Management”, and “Configuration”.
* The following menu items shall be moved from the existing “Configuration” menu group to the new “Integration” menu group.
** Import Dashboard
** Export Dashboard
** Imports Monitor
*** This menu item shall be renamed from “Imports Monitor” to “Import Monitor”
** Export Monitor
** API Configuration
*** This menu item shall be renamed to “SIS Auto Sync”

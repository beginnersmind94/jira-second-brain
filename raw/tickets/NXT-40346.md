# NXT-40346 — Custom Reports – Add New Report – Scheduling Tab

## Description
As a SchoolCafé user, I would like to be able to schedule distribution of my custom reports so that others can receive them without having to run reports.

*Notes:*

* We are updating the existing Add New Custom Reports and adding in the scheduling of distribution functionality to it
* This functionality is tied to distribution, so do that first
* Note that the user can do 1 of 3 things:
** Configure only, do not send
** Send once (this is new)
** Send on schedule

*Flow:*

* Open SchoolCafé and browse to Settings > Dashboard > Custom Reports
* View list of reports
* Click + New
* Create the configuration and get to the schedule screen
* User takes action:
** Want to Schedule?
** If yes, ask if Once or Recurring
** If once, show and ask them to pick a date and time
** If recurring, show them the scheduling fields with the toggle and scheduling stuff
* User completes configuration and then saves
* Return to the updated list of custom reports

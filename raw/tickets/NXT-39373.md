# NXT-39373 — POS - Auto-Serve Option - Pause Features

## Description
As Sammy the Site Manager I have a grab-and-go line that requires little oversight and no ala carte sales. I want a way for the POS to automatically serve meals so that the line continues to flow.

*Requirements*

* If Auto-Pause is enabled, add additional options to the POS > Device Info > Admin screen
** Pause Auto-Serve for Second Meal
** Pause Auto-Serve for Restrictions
** Pause Auto-Serve for Visiting Students
** Pause Auto-Serve for Allergens
** Pause Auto-Serve for Notes
*** Options: Yes/No
*** Default: Yes
*** (toggles are fine for these)
* If one of the pause options is enabled:
** Keep the pop-up on the screen.
*** Standard POS flow at that point - 
**** Cashier needs to complete the transaction or void it out.
**** Check settings to see if cashier can take the action or if they need to deny the sale.
** Turn off the Auto-Serve mode.
* Charge Limits need to be checked as part of this
** If the limit will be exceeded for the account, it should pause out of Auto-Serve.

## Acceptance Criteria
* Auto-Serve options added to POS Admin screens
** Pause options
* If Auto-Serve is turned on:
** Pause options, if enabled, keep pop-ups on the screen and turn off Auto-Serve mode
* Charge Limits are respected as a non-optional pause.
* Default menu item loads into the Item list for 2nd meals.

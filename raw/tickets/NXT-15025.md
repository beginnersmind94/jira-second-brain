# NXT-15025 — Verification - Details - Documents / Attachments

## Description
As Vera the Director I want to see and interact with any documents tied directly to a verification response. This will provide an audit trail and help justify my actions.

*requirements*

* Add a Documents area to the Verification App Details page (see: [https://primeroedgenext.atlassian.net/browse/NXT-5273|https://primeroedgenext.atlassian.net/browse/NXT-5273|smart-link] )
* User can see any documents that are tied to the app.
* User can attach additional documents and enter a comment associated with them.
* User can delete documents.
* User can see details on the document:
** Document name, size.
** Time it was added, who added it.
** -An indication if the file was submitted via the family portal. (the small leaf from the Classic screenshot)-

## Acceptance Criteria
* Documents section added to the Verification App Details page.
* Shows current documents tied to that application only.
* New documents and comments can be added.
* Documents can be removed.
* Information on the document is shown:
** Document name + extension, size.
** Who uploaded it and when.
** Any associated comments.
* Documents should be sorted by timestamp, latest first.

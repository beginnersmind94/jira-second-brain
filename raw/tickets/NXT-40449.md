# NXT-40449 — IM - Items - Units - Select Menu Item Serving Size

## Description
As a menu planner I want the ability to select a serving size before configuring the Menu Item tab so I can be sure I am configuring the correct menu item serving.

Update the process of selecting a serving size for Menu Item to match design flow for selecting SIR serving

[https://www.figma.com/proto/eDsxhWPpQikc74wptrgntb/Item-Management-6.4?page-id=769%3A48487&type=design&node-id=1468-64846&viewport=1668%2C-164%2C0.3&scaling=min-zoom&starting-point-node-id=1468%3A64846|https://www.figma.com/proto/eDsxhWPpQikc74wptrgntb/Item-Management-6.4?page-id=769%3A48487&type=design&node-id=1468-64846&viewport=1668%2C-164%2C0.3&scaling=min-zoom&starting-point-node-id=1468%3A64846|smart-link]

## Acceptance Criteria
* Provide a selection flow for creating a Menu Item on the Item Details
** this flow should be extremely similar to the first part of Single Item Recipe serving selection
* Show the Menu Item tab in a temporary disabled state until serving size is provided
* Insert the MI identifier once serving is selected
* Provide an Edit/ability to change
* Inherit Single Item Recipe serving as automatic selection
** (this may change in the future)
** in this case, load the user with the serving already selected (skip the selection)
** If Menu Item is created first, SIR can still be created but must also inherit the MI serving size
** Single Item Recipes cannot be deleted once created (for now)
* Skip selection if only one serving exists
** In this case, show an informational message in the banner bar for clarity:
** “Configuring Menu Item for *1 Bowl* serving size”

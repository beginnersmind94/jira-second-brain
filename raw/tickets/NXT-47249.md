# NXT-47249 — IM - Items - Move Yield Factor to be serving specific

## Description
As a nutritionist I want to be able to capture the different yield factors per serving size so I can use a single item (like apples) in numerous different serving contexts (sliced, cored, etc.).



!Edit Yield Factor v2-20240827-141856.png|width=83.33333333333334%,alt="Edit Yield Factor v2-20240827-141856.png"!

## Acceptance Criteria
* Make Yield Factor configurable on each serving in the Units card
* Provide a dedicated step for editing yield factor
** show only serving sizes (not including menu servings)
** add a dedicated Yield Factor column
*** allow Yield Factor to be edited for each serving size
** add a custom record at the bottom for ‘_Default (all other measures)_’
*** this maps to the existing “main item” yield factor
* Remove Yield Factor from Ingredient Info card (moved to Units)
* Don’t show the percentage in view mode if it’s set to 100%
* Update all Recipe calculations to use the serving specific yield factor
** if applicable

# NXT-4681 — ItemMgmt - Item Contributions - Validation

## Description
*User Story*

As an item management user, I want to be able to have my contributions validated on entry / edit.

*Mockup*

!image-20200729-203320.png|width=567,height=731!

*Requirements*

* This logic will be for ‘Add’ and ‘Edit’
* User cannot have a serving size with no contributions selected for contributions
** If contributions section is expanded, then information needs to be filled out
** At least one contribution has to not be 0 or blank
* Message to be displayed [~accountid:5dc4cd94c424110de525c2b3] 
** If serving size has data that is not 0, and no contribution is selected
*** 'A contribution must be selected if a serving size is entered'
** If a contribution is selected but no serving size is entered
*** ‘A serving size must be entered if a contribution has been selected’

## Acceptance Criteria
* User will only be able to fully complete contribution information and receive a message if something isn't filled out right.

# NXT-40309 — eOrders Configuration – View History

## Description
As a SchoolCafé user, I would like to be able to quickly view the history of an eOrder configuration so that I can see who made what changes in the past.

*Notes:*

* This will be just like the Device Manager History screen and usage
* Copy that code as much as possible to save time

*Flow:*

* Open SchoolCafé and browse to Settings > Configuration > eOrders
* View list of eOrders
* Click View History on a row
* Review the history of that eOrder configuration
* Return to the updated list of eOrders

*Requirements*:

*eOrders Dashboard page*

* Add a button to the rows in the action column
** Title: View History
* On click, Open the History drawer

*History Drawer*

* Add a drawer that opens when clicking on View History button on Dashboard
** Title = History
* On page load, load the grid
* 
* Grid has the following columns:
* Vendor
** The name of the Vendor
* Event
** Values:
** Add
** Edit
** Delete
* Description
** Value:
** The text of the values from the add and edit drawers
** Format: FieldLabel=Value; FieldLabel=Value; FieldLabel=Value; etc for all the fields
* User
** The user account logged in that made the change
* Date/Time
** The date / time stamp of the change from the database
* 
* Grid has paging, sorting and filtering

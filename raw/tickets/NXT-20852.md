# NXT-20852 — INV - Physical Inventory (Incomplete) - Redesign Add Item control

## Description
As a Site Manager I want a redesigned Add Item control on the Physical Inventory screen so I can add items without leaving the physical inventory screen.

Provide the user a better way for adding items to an Incomplete Physical Inventory, perhaps using a slide-out or some similar component.

*Requirements*

* On the Physical Inventory (Incomplete) screen, change the Add Item control
** Replace the current control with a new slide-out
* When user clicks the Add Item button, the slide-out will appear for searching and adding items. 
** The selections provided will be the same as they currently are

# NXT-27717 — Code Mapping : Adding Search utility in Code Mapping

## Description
This story is cover the tasks related to Code Mapping : Adding Search utility in Code Mapping.



Currently in the code mapping utility there is search capability. Lack of search capability creates a lot of confusion and takes longer time to complete the task as the user has to go through each and every available field.

Addition of Search option saves time to perform the mapping task and the user can clearly search for a particular field to complete mapping.

## Acceptance Criteria
Verify and validate that the search option is provided in Code mapping utility.

Verify and Validate that the search option is fully functional (test by searching few fields).

Verify and Validate that the search option does not eliminate the available fields.

# NXT-24942 — PR- Consolidated Production Plan (Prep Site View)

## Description
As a prep site manager, I want to see a consolidated plan for all planned counts for my site as well as the “prepared items” for any receiving site I am prepping for.

## Acceptance Criteria
Prepared items planned counts from receiving sites will show on the consolidated plan.

Only consider Plan Complete for receiving sites

*Prep site and receiving site have same meal type/Menu line: Only show allow Consolidated Tab once Prep site Plan is COMPLETE*

If prep site has the same menu line as receiving site/ sites, show consolidated plan tab when user clicks on specific menu line. Display all items for prep site with planned counts, as well as all receiving site “prepared items” and planned counts.

Columns to Display:

* Preparation Date (default to actual record date for prep) user can change if needed (Date Range to select from Current date- Record Date)
* Menu Item (with Menu Item Help Button)
** Include Strict Batch Information if available and use current validation for counts
* Serving Size
* Prep Site Name (with “Preparation Site” as an indicator)
* Site Name ( Receiving Site/ Sites) (Each site will be in its own column)

Total (Planned Counts for Each item for all sites)



If menu items are not available for specific site, display ---

If planned counts are not available for specific site that need “prepared items” display (Not Available) in red (Plan has not been completed)

*DIsplay “Non Prepared Items” as separate items. These items will be all items that are not marked as Prep site items that the Prep Site is making for themselves*

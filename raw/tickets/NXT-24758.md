# NXT-24758 — Platform - Security - General Permissions CRUD Operations (EDIT)

## Description
As a primero edge admin, I should be able to access resource catalog to add a new resources available for any district with functions resource type along with category so that the resource type can be accessed by any user with the assigned role for that resource.

+*Requirements:*+

* Enable Alias for the Resource field in the resource catalog
* Cleanup all resources that are not needed for Release 4 & 4.1
* Resource Permissions CRUD operations with EDIT Permissions to be enabled and making sure FO & BO teams can utilize these resource permissions via roles & permissions page

## Acceptance Criteria
* Any page type resource with EDIT button enabled shall be able to delete the resource
* Enable Alias for the Resource field in the resource catalog with EDIT option for PAGE only
* Cleanup all resources that are not needed for Release 4 & 4.1
* Resource Permissions CRUD operations with EDIT buttons to be enabled and making sure FO & BO teams can utilize these resource permissions via roles & permissions page

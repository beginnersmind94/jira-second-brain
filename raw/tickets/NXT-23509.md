# NXT-23509 — Production- Production Record- Display red box or warning around boxes that have blank counts

## Description
As a user, when I save my plan, sometimes I miss one of the counts that I am supposed to enter and my status never changes. I would like a quick way to see which counts are missing.

## Acceptance Criteria
Display sign to show count has not been entered once production record is saved

Current logic on how production status changes is not effected. This primarily for user visibility.

## Acceptance Criteria
Navigation: System - Administration - Dynamic Queries

Requirements:

To have a new report in place for all the ingredients which are not used in the Menu assignments for the last 2 years

Menu Assigned Dates - From and To - Configurable in Query

Region = Corresponding Region ID, Including Central Realm
Data Sources in (Local, PDE Shared) - configurable description in query
Item Active = Yes
Recipe Active = Yes
Menu Active = Yes
Recipe Generation Type = All

*Field Required:*
Ingredient Code - Ex: LI-12345
Ingredient Name - Ex: Bread
Assigned to Recipe - Yes or No (based on Ingredient to recipe relationship)
Number of Recipe Assignments - Count of Recipes where ingredient linked to Recipe
Associated Recipes - list of recipes - comma separated in 1 column
Assigned to Menu - Yes or No (based on Recipe to Menu relationship)
Number of Menu Assignments - Count of menu items linked Recipes where Recipe is has that ingredient
Last Menu Assignment Date - Date of Last Menu assigned for the corresponding menu item (linked to recipe, recipe linked to ingredient)
Active District - count of region IDs where the same ingredient code is used in active recipes for corresponding data source

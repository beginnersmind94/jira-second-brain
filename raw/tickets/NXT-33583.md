# NXT-33583 — IM - Reports- Add Sesame as a Standard Allergens on Allergen Chart and Report

## Description
Due to a new law that was passed on April 23, Sesame will now be considered as a standard allergen on January 1, 2023. However, Columbia is now in the process of starting to input their items and would like to have Sesame already configured so they could see their allergens in the allergen chart.

## Acceptance Criteria
* Include Sesame as a Standard Allergen on the Allergen Chart
* Allergen Chart print option should also include Sesame as an additional standard allergen

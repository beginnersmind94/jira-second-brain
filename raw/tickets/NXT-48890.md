# NXT-48890 — MP- Serve with- Enhancements to handle removed menu items on menu, plan, record and rename to Offered With

## Description
As a menu planner, when I remove a menu item that is part of a primary or complimentary link, the system is still keeping the link somehow and it needs to completely remove it. Also, since it is majority offered items, we need to change the verbiage to Offered with.

## Acceptance Criteria
* Change the verbiage where it says “Serve With” to “Offered With”
** Menu
** Plan
** Record
* When a menu item is replaced/ removed from a menu via Find and Replace, remove the link completely between the complimentary and primary item regardless of which item is which
* When a menu item is removed from the plan (add from plan), completely remove the link between primary and complimentary item
** This should also reflect properly on the record page
** This should also be covered in vice virsa scenario (Add Item from Record) and removing
* In Case of carryover scenario, IF menu item was added to the plan as part of carryover, and then was removed from source record, trigger to completely remove the link between primary and complimentary regardless of which item is which
* *Need to correct any existing data that is effected*

*Calculation Notes*

* Max Confidence % - ((Max Confidence % / Number of Included Sites) x Number of sites that aren’t Ready for this area)
* Example - 5 sites, 4 are NOT READY for Sessions (25% MAX)
** 25% - ((25 / 5) x 4)
** 25 - (5 x 4)
** 25 - 20
** 5%
* Let’s try using this method for the Site areas
* Applications method:
** Max Confidence % - (Number of New Applications)
* % cannot go under 0

Brief explanation text for each one to be included:

* Sessions: Number of Sessions that are not in Reconciled/Deposited.
* Edit Checks: Number of unresolved Edit Checks per site.
* Reverse Edit Checks: Number of low meal count instances per site.

Notes for the Notes section:

Sessions: 

* If any sites have counts in the Sessions area: 
** X site(s) still have sessions that may require attention. Impact: Y% confidence reduction.
* If all sites have 0 counts: 
** All sessions have been accounted for. 

Edit Checks:

* If any sites have counts in the Edit Checks area:
** X site(s) still have edit checks that should be resolved. Impact: Y% confidence reduction.
* If all sites have 0 counts:
** All edit checks have been handled or none existed.

Reverse Edit Checks:

* If any sites have counts in the Reverse Edit Checks area:
** X site(s) still have low meal counts that may require attention. Impact: Y% confidence reduction.
* If all sites have 0 counts:
** No instances of low meal counts have been identified.

Serving Exceptions:

* If any sites have counts in the Serving Exceptions area:
** X serving exceptions still need to be resolved. Impact: Y% confidence reduction.
*** X = total number of serving exceptions (duplicate + variance added together)
* If all sites have 0 counts:
** All serving exceptions have been handled or none existed.

Review Reasons

* If any sites have counts in the Review Reasons area:
** X transactions are still marked with review reasons and may require attention. Impact: Y% confidence reduction.
* If all sites have 0 counts:
** No transactions marked with review reasons.

Applications:

* If any pending or received/new applications exist across the district:
** X applications are waiting to be processed. Impact: Y% confidence reduction.
* If all sites have 0 counts:
** No applications are waiting to be processed.

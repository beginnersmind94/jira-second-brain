# NXT-4397 — Item Management - Duplicate Check Button Modification

## Description
*User Story*

As a super user, I need to be able to click ‘Save and Continue’ and create the item if there are no duplicates

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=14&view=default|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=14&view=default]

*Requirements*

* Remove ‘Continue’ after putting in the Item Name
* Have the ‘Save & Continue’ button handle the behavior of duplicate check
** User clicks the button Save and Continue, which then checks for duplicates
* If no duplicates, then the user will just be moved to the next step
* If duplicates, then the layout is the same as today

## Acceptance Criteria
When the user clicks 'Save & Continue' and there are no duplicates, they are directed to the general item message
If there is a duplicate then, the functionality stays as is

# NXT-48174 — PR- Reports- Grocery List- Criteria Filters Enhancements

## Description
As a production manager, I would like the ability to be able to filter to see or not see certain information when viewing the grocery list, such as seeing the vendor associated with the items, seeing a list without entering quantities on the plan and not considering counts, and whether as a user I need to see the weight of the quantity I need or just the pack size.

## Acceptance Criteria
* Allow toggle under the basic criteria for the following
** *Vendor* 
*** Toggled *off* by default
*** `This information is provided from IM under the inventory tab, always prioritize the actual vendor associated with the item but the user also has the option to make a custom vendor (Fake vendor) that is not tied to inventory
**** If none available, use N/A
*** When this is toggled on, allow for a new column to display as the first column
** *Vendor Item #*
*** Toggled *off* by Default
*** Hide toggle and only display to be toggled if Vendor toggle is on, when toggled on allow for new column after Vendor
** *Items without Quantity*
*** Toggled *off* by Default
*** Toggling this on will not consider any counts from production plans and will only display a list of items. Only display Item #, Item Description, Amount needed (Only display the dates and meal service associated with this item EX: [3/15/2024- Lunch})
**** Club all items together that are duplicated and show the list of days on the Days needed column
*** Rename Amount needed column to Days Needed
** *Weight Needed*
*** Toggled *ON* Be Default
*** When toggled on, display weight like we are currently displaying after the pack size information, if it is turned off, hide the weight and only display pack size information
*** This should effect both Total Quantity and Amount needed columns
*** Continue to display weight/ volume if there are no pack sizes

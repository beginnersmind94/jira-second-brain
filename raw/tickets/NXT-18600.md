# NXT-18600 — Column Attributes - Date Transformation

## Description
This story is to cover the tasks related to column attributes. There will be several features in the Column attributes story. The first and the primary one is date format.

In any imports or exports, option to change date format configuration helps the user to change the date from one format to another one like mmddyyyy to ddmmyyyy; mmddyy to ddmmyy; mm/dd/yyyy to mm-dd-yyyy or dd/mm/yyyy to dd-mm-yyyy

 

This feature will be used in the mapping page.



*UX not required*

## Acceptance Criteria
# Verify and Validate that the date format is changed as per the format needed.
# Verify and Validate that the data is other columns are not altered when date configurator is used.
# Verify and Validate that only the intended date column is modified.
# Verify and Validate that the saved date pattern is saved and is in intact when the user saves and continues to next page and comes back to mapping page.
# Verify and Validate that the user can navigate to next page from mapping page only after the page is saved.

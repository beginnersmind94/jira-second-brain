# NXT-40308 — eOrders Configuration – Delete

## Description
As a SchoolCafé user, I would like to be able to delete one of the eOrder Configurations so that I don’t have to see it anymore.

*Flow:*

* Open SchoolCafé and browse to Settings > Configuration > eOrders
* View list of eOrders
* Click Delete on a row
* User confirms their intentions, and the system deletes and records in History table

*Requirements*:

* eOrders Dashboard page updates:
* Add a button to the rows in the action column if it is not there already
** Title: Delete
* 
* On click, do the following:
* Ask if they are sure want to delete
* If confirmed, they want to delete: remove the configuration from the system
* Log the deletion in the history log
* 
* Show a success confirmation
* Refresh the table showing that the eOrder is no longer in it
* 
* Notifications:
** Send a message to the inbox of all users that have Delete rights in Roles & Permissions
** Text: [logged in username] has deleted an eOrder configuration for [Vendor Name] To view, click [link to the eOrder Configuration]

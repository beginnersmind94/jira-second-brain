# NXT-63551 — SC/PE - View an Item (UI) - Common View for any Data Source

## Description
_(none)_

## Acceptance Criteria
Navigation: MP-GC - Item Catalog

Changes Required:
User selects the Data Source dropdown (Cybersoft Catalog)
User should be able to enter the GTIN, Item Description, Manufacturer (to be listed as dropdown, text based search should list the items from dropdown ) 
The Brand, Manufacturer list should be based on Master Data Tables (Grouping Name) 

The search results should be listed in the Grid as shown in the prototype (Item, Manufacturer, Brand, GTIN)

Each item will have Hyper link on Item Name and Eye Icon at the end to view the record

!image-20250617-055016.png|width=1073,height=532,alt="image-20250617-055016.png"!

Upon clicking Eye icon or Hyperlink, the item should be open in a popup window (with a close button to resume to search), the search results should be retained

View Item Wireframe: The user should be able to view all the attributes as similar to USDA Branded Foods

User should be able to insert the item into local DB upon clicking import button

# NXT-16824 — INV - Orders - Order Detail - Receipts tab - Charges & Discounts

## Description
As an inventory user I want a Receipts view on the Order Detail page so I can view my list of items in terms of how many have been received.

This would be the 2nd tab of the Order Detail page. This is where Receipts would be displayed

+Page elements:+

* *Charges/Discounts for Summary* *or for individual receipts*

*Requirements*

* Below the ‘Summary’ grid, show the Charges / Discounts section if applicable
** For the Summary if any Charges or Discounts are applied to any receipt
** For the specific Receipt tabs if any Charges or Discounts are applied to that receipt
* List each applicable Charge and Discount along with the value
* Do not display charges or discounts that had no value

## Acceptance Criteria
* Charges / Discounts grid is displayed on Summary if any receipt for that order had charges or discounts
* Charges / Discounts grid is displayed on specific Receipt tab if that receipt had charges or discounts
* Charges / Discounts are not displayed if they did not have a value on the receipt(s)

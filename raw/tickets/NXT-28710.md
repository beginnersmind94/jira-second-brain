# NXT-28710 — Student Data Scheduled Export

## Description
As district I/T operator, I would like the ability to have a scheduled export for student data.

## Acceptance Criteria
As district I/T operator, I would like the ability to have a daily scheduled export for student data.

I would like:

* A daily export that is generated at 5pm for the timezone in which the district is located.  (Can we make this time configurable in settings?)
* Have the exported student data file available on an FTP site for me to access
* The export shall include the current eligibility code and current balance.

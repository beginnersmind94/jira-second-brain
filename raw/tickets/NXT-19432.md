# NXT-19432 — INV - Order Detail - Receipts - Allow Receipts to be edited after Order is Closed

## Description
As a Site Manager I want to be able to make adjustments to receipts even if the order is closed so I can update quantities until the period is closed or add header level info after.

This story should apply to both regular receipts and quick receipts.

*Requirements*

* Change logic on Receipt Detail page to allow Edit Receipt functionality even if Order is Closed
** Edit Receipt is enabled
** Cancel Receipt is enabled
* Disallow item-level adjustments if
** Physical Inventory has been completed after the receipt date
** *OR* Period is closed containing the receipt date
** In these scenarios:
*** Edit Receipt button would enable only the header-level attributes
**** Order Delivered By
**** Invoice #
**** Invoice Amount
**** Comments
*** *Disable/remove the ‘Add Item’ button
*** Show banner messages (need improvement)
**** “Only header attributes may be changed as this receipt period is closed.”
**** “Only header attributes may be changed as a Physical Inventory has been completed after the receipt date.”
*** Cancel Receipt is disabled

## Acceptance Criteria
* Update both Receipt and Quick Receipt logic to be more flexible after order is Closed
* Enable/disable Receipt and Cancel functionality depending on Physical Inventory status or Period status

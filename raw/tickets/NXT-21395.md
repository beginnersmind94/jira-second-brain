# NXT-21395 — Eligibility - Scanning - Automate year begin scripts and barcode creation

## Description
As a Cybersoft Admin, I would like to be able to have SchoolCafé automatically increment barcodes for use on a district’s applications. 

Currently, Expert Care and others are having to use a barcode creation tool to create a barcode and use MS publisher to add those barcodes to a scanned application, as well as add the values for those barcodes into the database via scripts. They have to do this for every district and every language supported at the district. Can we automate all or at least part this process somehow?

Goal: Automate all or at least part the process required to indicate which scanned surveys are mapped to a specific district and language.

## Acceptance Criteria
Automate all or at least part the process required to indicate which scanned surveys are mapped to a specific district and language.

# NXT-5385 — Acct Mgmt -Transactions - Transaction Details

## Description
*User Story:*

As a user I need to see the details of any transaction created.

*Requirements:*

* The details of a particular transaction are show
* The items sold are shown
* Any payments are shown
* Previous comments are shown
* If the user has edit permission they can make changes to the transaction.

## Acceptance Criteria
Transaction Details is launched by sliding from the right. 
When Edit Role Permission is not true all fields are view only.
For the selected transaction Display Meal Type, Transaction Type, Name, date and time.
Show any items sold with the Menu Item name, Quantity, Price and Extended Price.
Show the total of Items sold, Tax(if applicable), and the total.
Payments will show the Amount Tendered for Cash, The amount deposited and any change returned.
Payments by check will show Check Amount and check #
Payments by card will show Card Amount and last 4 digits as entered by the cashier.
Comments previously entered will be shown
Review Reasons and Comments entered by the cashier will also be shown.
When Save or Closed is selected the transaction is updated and the user is returned to the list.

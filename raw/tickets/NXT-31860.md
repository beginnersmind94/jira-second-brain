# NXT-31860 — IM- Reports- Allergen Report

## Description
As a Registered Dietitian, I would like to be able to see all my menu Items that have allergies and what allergies do they have.

*Requirements*

* Update the Allergen Chart report page to expand it to cover new report requirements
* Provide default filters on pageload
** Report Type - single select
** Standard Allergens - multi-select
** Custom Allergens - multi-select 
* If Report Type is ‘Menu Items on Assigned Menus’ show extra filter selections
** Date Range
*** default to next full week range (Mon-Fri)
** Site
*** default to All
** Meal Type
*** default to All

## Acceptance Criteria
* Allow user to pick what menu items that would like to see based on criteria.
** Either by Marketing Name or by Menu Item Name along with a way to indicate if they would like the ingredient breakdown based on the menu item
**  User can pick between seeing:
***  All Active Menu Items
*** Menu Items on Active Menus
*** Menu Items on Assigned Menus 
**** Allow a From and to Date selection, site (Default to all) and Meal Type (Default to All)
* Allow user to pick what menu items they would like to see based on allergens from the standard allergens (Default to select All) 
** Crustacean Shellfish
** Egg
** Fish
** Gluten
** Milk
** Peanuts
** Soy
** Tree Nuts
** Wheat
* Allow user to pick what menu items they would like to see based on allergens from the custom allergens list (Default to select All) 
* Display a way for user to generate report based on all criteria selected

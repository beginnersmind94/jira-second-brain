# NXT-66387 — Inventory Transfers (between sites or warehouse)

## Description
As a *district inventory manager*,
I want inventory transfers between sites/warehouses to generate ledger entries,
So that movement of stock is tracked without distorting total district-level assets.

## Acceptance Criteria
* Preconditions: Same as usage.
* Posting date = last day of inventory period.
* *Ledger entries*:
** _Warehouse Receives (GLEntryTypeCd: WHIRCV, Multisite)_ → Debit Site Asset, Credit Warehouse Asset.
** _Warehouse Commodity Value (GLEntryTypeCd: None, Multisite)_ → Debit Site Asset, Credit Warehouse Asset.
* Rollbacks reverse entries.
* Entries visible in GL journal/View Ledger, net effect is zero across district but shifts balances between locations.

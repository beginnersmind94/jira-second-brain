# NXT-41562 — IM - Item Onboarding - Review Serving Info

## Description
As an onboarding district admin I want a step to review serving size information so I can ensure correct nutritional basis for my bulk imported items.



!image-20241010-145545.png|width=83.33333333333334%,alt="image-20241010-145545.png"!

## Acceptance Criteria
* Provide a review step on the Bulk Item Import tool for the Serving Size(s)
* Require & confirm all mandatory fields for serving size creation
* Provide Fluid flag for Weight/Volume determination
** Set flag accordingly for any imported data based on gram or milliliter basis
* Highlight any missing data points
* Allow independent editing of all serving size fields
* For Serving Measure Description, use the existing suggestion logic from Common Measures

# NXT-61556 — AI Onboarding - Workspace - Default Landing Page

## Description
As a SchoolCafé user, I would like to set my default screen to the new Workspace instead of the SC home screen so that I can go directly to where I need to go.

*Requirements:*

* On the user profile screen, do the following:
* On the My Preferences screen, do the following:
** Remove the Language field
** Remove the Theme field
** Update the dashboard field as follows:
*** Change the text to: Use Workspace
*** Change the toggle button text to: Yes and No
*** Change the info button text to: Make Workspace your default landing page instead of SchoolCafé Home.
*** Default value is No
* 
* Ensure that the user is redirected to the new workspace via SSO if the user selects Yes when they login
* 
* Remove the code for redirecting the user to the alt workspace on login. 
* Note: Do not remove the alt workspace page from the header. Allow the user to access it still. We are just removing it as a default landing option.
* 
* Security: 
** No changes

## Acceptance Criteria
* User can decide if they want to go to the home page or the new workspace when they login
* System respects their choice and directs accordingly
* The old Alt workspace is no longer accessible as a default landing
* The old Alt workspace landing can still be accessed in the header

## Acceptance Criteria
* Regular View Mode
** All entry to menus will be in view mode
*** Menu List page, Menu Cycle Standard, Plans/ Records
** Display Version Number if applicable 
** Display two clickable chips to indicate Production records that have been complete as well as assigned but not completed
** Provide tool tips on all action icons History, Versions, Copy, Edit Header
** Provide tool tip for Serving Counts as Percentage
** Display analysis and allow user to click on the serving group headers to switch between analysis per serving group
*** Highlight column to indicate which serving group is selected along with the different icon for analysis
** Provide a back to menu list button to allow user to go back to the list page
* Version View Mode
** The view icon on the version slide out will open menu version and provide a back to menu button if opened from current menu 
** If opened from the version slide out on the menu list page, open menu version and provide a Back to Menu List button if opened from the version slide out on the list page
** This page will be in complete view mode if there are no assigned menus associated and only record complete
*** Display the number of production records and assigned menus 
*** If assigned menus are still associated with version, allow version to be edited, and it will become a subversion and affect all assigned menus associated 
** Display analysis and allow user to click on the serving group headers to switch between analysis per serving group
*** Highlight column to indicate which serving group is selected along with the different icon for analysis
** Provide a footer message to indicate which version along with the effective date for this particular version

# NXT-37200 — MP- Menu Planner- Create Scenarios Menu/Menu Cycle should lead to list page (UI ONLY)

## Description
Currently, when a user creates a menu or menu cycle, it leads them to the standard version of that menu or menu cycle. Since we now have the option to edit on the menu planner, there is no need for this anymore and we can lead them to the list page.

## Acceptance Criteria
* When a user clicks save and continue or save and close, lead the user to the list page 
** Menu Creation for 1 or 2 menus should lead to the menu page
** Menu Creation of more than 2 will create a menu cycle and lead the user to the menu cycle page
** Update the button from save and Continue to Save and close ( Same as the edit scenario)

# NXT-59557 — PE - Settings to Enable API Integration

## Description
_(none)_

## Acceptance Criteria
Navigation: Settings- System Settings - Global Catalog

Currently there is no option to restrict the usage of GDSN API Integration



Changes Needed:

Introduce a new INTERNAL  setting as “Enable GDSN Data Exchange”, Default Toggle Value: No, User can change to Yes if needed



If the toggle value is Yes, then +SHOW+ the following options:

*Settings - Administration - Import/ Export* - Export Ingredients (API) Radio Button 

*Settings - Administration - Import/ Export* - Export Ingredients (CSV) Radio Button 

*MP - GC - Pending Upload -* Check for Updates Button

*MP - GC - Pending Upload -* 3rd Party Source(s) in Data Provider Dropdown

*MP - GC - Pending Upload - Upload Button, File Upload Button and Download Exception*



If the toggle value is NO, then +HIDE+ the following:

*Settings - Administration - Import/ Export* - Export Ingredients (API) Radio Button 

*Settings - Administration - Import/ Export* - Export Ingredients (CSV) Radio Button 

*MP - GC - Pending Upload -* Check for Updates Button

*MP - GC - Pending Upload -* 3rd Party Source(s) in Data Provider Dropdown

*MP - GC - Pending Upload - Upload Button, File Upload Button and Download Exception*



This option should be accessible only if primary GDSN permission is enabled, otherwise they should be disabled or hidden

Out of Scope:
No impact to Manual Upload Functionalities

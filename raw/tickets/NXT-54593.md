# NXT-54593 — PEI - Auth - Enable SSO in PE

## Description
As a PE user with Insights access, I would like to be able to login to the new Insights website using my existing login credentials so that I don’t have to relogin

*Requirements:* 

* Write the code to enable SSO from PE to the new website
* We have done this before for the face and food detection project, so the code should already exist, but if it doesn’t then write it
* The code should pass the following:
** Authentication token and info as necessary so that new site knows the user is authenticated
** District
** Profile information (name, etc.)
** that the district has the Insights license
** the resources the user has access to in Insights
*** Ex: they are in a role that has access to Insights > FO > Command Center
*** Note: Needed so that the website can security trim

## Acceptance Criteria
* PE is able to send the authentication information to the new website
* New website is able to get the district, profile and resource access from PE

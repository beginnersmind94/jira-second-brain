# NXT-41273 — IM - Items/Recipes/Menu Items - Images & Documents

## Description
As an Item Management user I want the ability to attach documents to items, recipes and other IM types so I may save things like product formulation statements, CN labels, crediting documents, and other important pieces of information.

[Component Library - File Dropzone|https://devuicomponents.perseusedge.com/?path=/story/components-file-dropzone--with-image-preview]

## Acceptance Criteria
* Replace the edit mode of the Images card with a new Image & Document slideout using the common component
* In the slideout, use the image and file preview component
* Provide a ‘Image & Document Options’ card below with Item Management specific use cases
* For images, provide a ‘Use As’ dropdown for the preferred image types
** Preferred Ingredient Image
** Preferred Inventory Image
** Menu Image (for publishing)
* For documents, provide a ‘Use As’ dropdown for the document file types
** CN Label
** Buy American Exemption Letter
** Product Formulation Statements 
** Meal Credits

# NXT-37052 — IM - Item Master - Convert Serving Measure into a smart free text field

## Description
As an Item Management user I want Serving Measure Description to be a more free text field so I can type measures without adding them to a master list, but I want the system to be smart enough to identify standard measures so I can still scale measures as usual.

*Requirements*

* Convert the Serving Measure Description field into a “smart free text field”
** This is currently a dropdown, and all available values are determined from the Serving Measures page
* Allow users to type any description, but attempt to identify common or standard measures
** Ex: user enters “1 Cup, blended”
*** We do not need to save the “Blended” variant to a table, but we need to recognize that the word “Cup” is present
** These will be validated against the updated Serving Measures page, where the common and standard measures will be maintained
** Provide autofill suggestions

## Acceptance Criteria
* Convert the Serving Measure Description field into a “smart free text field”
* Allow users to type any description
* Suggest common or standard measures by referencing the updated Common Measures page

# NXT-34153 — Team Hub - View or add Attachments

## Description
As a school admin, I want to view, add or delete attachments so that I properly share information in a message.

*Flow:*

·         On the Message Details screen, view attachments

·         On the Send Message screen, view, add & delete attachments

 

*Assumptions:*

·         User has logged in

·         To code this, you will have to have completed the push / pull messages stories and one of the source triggers (Home Screen or Messages Screen)

·         To code this, you will have to set up the Message Details and Send Message screens covered in other stories

 

*Message Details Screen Requirements*:

·         The following apply only to the Message Details Screen

·         Add the following fields to the screen:

·         Attachments – list of hyperlinks

o   Make look like traditional attachments field

o   Shows the hyperlinked list of attachments

o   Visible field is name of the document

o   Click on link and open the attachment

o   It is preferrable to just show links to original documents instead of downloading them to the local device

·          

·         Security – no additional requirements necessary

 

*Send Message Screen Requirements*:

·         The following apply only to the Send Message Screen

·         Add the following fields to the screen:

·         Attachments – list of hyperlinks

o   Make look like traditional attachments field

o   Shows the hyperlinked list of attachments

o   Visible field is name of the document

o   Click on link and open the attachment

o   It is preferrable to just show links to original documents instead of downloading them to the local device

·         Add Attachment – Upload attachment button

o   Use the traditional mobile upload attachment button and user experience

o   After select the attachment, add it to the list of attachments

·         Remove Attachment – Hyperlink button or Image button

o   On click, remove the attachment from the list of attachments

o   Keep it next to or associated with each individual attachment

·         Send button

o   When sending the message, push the attachments as well

·          

·         Biz Logic:

·         In the business logic for this screen, add the following requirements according to the source same as in the Compose a Message story:

·         Forward

o   Add attachment links to attachment section if exists

·         Reply

o   Add attachment links to attachment section if exists

·         Reply All

o   Add attachment links to Attachment section if exists

·          

·         Security – no additional requirements necessary

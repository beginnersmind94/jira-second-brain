# NXT-6795 — INV - Vendor Detail - Info tab

## Description
As an Inventory manager I want a tab based view of the vendor so I can more easily view and manage the different vendor information.

This new vendor template is more consistent with classic design and allows for easier management of various vendor components such as General Info, Address info, Contact info, Contracts, Items and any other categories we add for the vendor.

This tab based approach will allow us to continually build on vendors and add new categories without adding to a vertical card page.

The first tab will be called ‘Info’ and will contain separate cards for the existing vendor information:

* General Info
** this contains the attributes currently defined on General Info card
** This card will contain an Edit icon in the top right corner that will allow the user to enter the edit view to make changes

*Requirements*

* Add the tabs for the new design
* Make the UI changes to the new card
* Show the labels in this order
** Vendor Name
** Vendor #
** Data Source 
** Vendor Type 
** Preferred Ordering Mode 
** Vendor Lead Time
** Include Weekends for Lead Time - Yes if checked, No if unchecked
** Production Order Lead Time 
** Comments 
** **Note*: hide or remove the ‘Strategic Partner’ field
* Edit functionality on [https://primeroedgenext.atlassian.net/browse/NXT-21405|https://primeroedgenext.atlassian.net/browse/NXT-21405|smart-link]

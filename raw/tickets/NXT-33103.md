# NXT-33103 — DataQuery Framework - Reports Column Reordering/Resizing

## Description
This is a spike story to research how to achieve reordering and resizing columns, then persisting them with the Kendo grid state into a custom report.

+*Existing UI*+

!image-20220624-045543.png|width=1631,height=663!

## Acceptance Criteria
* The user shall be able to reorder the columns and have the column order persisted with the report when saved
* The user shall be able to resize the columns and have the column sizes persisted with the report when saved

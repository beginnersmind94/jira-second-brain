# NXT-70074 — FIN - Warning When Closing Period with Open Sites

## Description
*Description:* Prevent users from accidentally closing Financial periods when upstream site periods haven't finished posting, which may require subsequent adjustments.

*Given* I am closing a Financial period
*When* one or more sites have not closed their POS or Inventory periods
*Then* I see a warning modal before the period closes

## Acceptance Criteria
# Modal appears when user clicks Close action and any sites remain open
# Modal title: "Some sites have not finished posting"
# Modal body text: "[X] sites have not closed their POS or Inventory periods. You may need to make adjustments once those sites finish posting."
# X = clickable count linking to site status drilldown (same table as FIN-Sites Drilldown story)
# Clicking count opens embedded table within modal showing Site | POS | Inventory statuses
# Modal actions: [Cancel] [Close Anyway]
# Cancel returns to Periods page without closing period
# Close Anyway proceeds with closing despite open sites
# If all sites are closed, no warning modal appears

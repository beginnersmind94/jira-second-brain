# NXT-3328 — Inspection Template - Pass/Fail Question Type

## Description
*User Story*
As an Inspections user, I can add Pass/Fail question type as a question

*Mockup*


*Requirements*

* Pass/Fail Questions can be added to any template when being created/edited
* Users are given only three options that cannot be edited:
** Pass
** Fail
** N/A
* If the scoring type is Custom Scoring, the user can add point values to each option
** If no points are entered, it defaults to 0
* If scoring type is Pass/Fail, this is the only question type available

## Acceptance Criteria
* Pass/Fail question type is available and can be added to templates
* Pass/Fail questions can be edited
* Pass/Fail questions can have points added in Custom Scoring
* Pass/Fail questions are the only available question type in Pass/Fail scoring

# NXT-54690 — System - Grades - Mapping for Grades into Serving Groups

## Description
As Matt the Product Manager I want to establish a mapping between FO grades and BO serving groups.

*requirements*

* Add a Serving Groups tab to System > Site Configuration
* User can see the serving groups that are available from Back Office Serving Groups (Menu Planning > Config > Meal Patterns > USDA Meal Patterns)
* User can see the pre-mapped grades that are part of each serving group.
* User can add grades to the serving groups.
** Cannot remove the master data mapping.
** If the user deletes a grade we should remove it from serving group mappings.
* We’re establishing seed data as part of this - it needs to be part of all new district creation, and added to existing districts.

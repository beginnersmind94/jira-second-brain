# NXT-45123 — IM - Items - Migrate SIR to DMI or Recipe

## Description
As a Menu Planner I want the Single Item Recipe functionality to be migrated so my former SIRs are now DMIs w/ Recipe *OR* are made into a proper multi item Recipe.

h3. MI only 

!image-20231109-152616.png|width=91%!

h3.

## Acceptance Criteria
* Migrate former Single Item Recipes to support new merged DMI concept
* Items which are both SIR and DMI will become DMIs w/ Recipe Prep
* Items which are SIR but *not* DMI will either become an
** Ingredient: if there are no HACCP configuration on the SIR
** multi item Recipe: if there is HACCP configured on the SIR
* Ensure identifiers are updated correctly

# NXT-62702 — POS - Served Sites - Configuration

## Description
As Dana the Director I have several merged campus schools that serve meals together. POS Terminals log the visiting students but it takes time to hit the server and grab their data. Being able to load them from local storage would save time and speed up the line.

*mockup*

this is not pixel perfect and is just for inspiration

*requirements*

* User can configure sites that serve other sites on the Site Configuration page.
** Section: Served Sites
* User can select feeding sites from a multi-select dropdown
** 1:many relationship, same site should not be in the same dropdown.
* User has a tooltip that explains what served sites are for:
** Text: Configure your campuses that serve meals at one central location as a Served Site. This will allow POS Terminals to download account information for the additional sites and will increase line speed.
* User can see ‘child’ sites for a Satellite/Combined Site Category at the ‘parent’ site. Screenshots below.

# NXT-19186 — Acc. - Add. Income - Sessions

## Description
*Description:*
As a user I should be able to search for additional income transactions so that I can pull additional income related transactions
*Summary:* 

When a user views a session details screen should show the income incurred through additional income transactions along with the detailed view. And the reconciliation screen should also track additional income transactions.

*Requirements:*

*View Sessions Screen* : 

# Total Additional income dollar amount should be shown on the first grid
# Add a section just like Sales and Payments that shows all the detailed transactions for Additional Income

!View Session 1.png|width=972,height=415!

*Reconciliation Screen:*

# Add total Additional income card at the top and total number of Additional Income transactions should be shown under summary 

!Reconciliation 1.png|width=1207,height=656!
# If the session has any additional income transactions it must be shown in the transactions section below 

!Reconciliation 2.png|width=818,height=525!

## Acceptance Criteria
AC1: View session details screen should show the dollar amount for income through AI transactions
AC2: Reconciliation screen should show total AI card on the top and the detailed transactions 
AC3: Summary should show total number of transactions for AI

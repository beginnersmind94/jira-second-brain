# NXT-43553 — PR- Site and Site Group filters based on permissions for Pre and Post Production Costing

## Description
As a District Admin, I need my site managers to only be able to see their site information. Along with that, only the site groups that they have access to sites in should show up in the drop down for them to see.

## Acceptance Criteria
* *Reports-* Site Group filters should display only based on sites permission by user
** EX: I am a highschool manager, I should only see the highschool site group and the highschool site in the drop down
** EX: I am a district supervisor that is over some middle and some highschools, I should see the highschool site group and middle school site group along with the sites I have access to
* Post Production Costing (User should only have access to the reports option “Summary by District” if they have access to all sites)
* Pre Production Costing (User should only have access to the reports option “Details by Site Group” or “Summary by Site Group” if they have access to a full list of sites in a site group)

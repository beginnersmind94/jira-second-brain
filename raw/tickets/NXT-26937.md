# NXT-26937 — MP - Using Holiday Information For Calendar

## Description
As a menu planner, I would like to be able to see the holidays that have been entered for the district on my assign menu calendar page.

## Acceptance Criteria
Once Holidays have been entered on the Holiday Calendar Page, allow this information to move over to the MP calendar on assign menus

Display Days with Holiday on MP calendar with Color 

Display Name of Holiday on the top of the calendar day

If user drops a menu on a holiday, display message “The date selected is marked as a holiday, Are you sure you still want to assign?” (Ok/Cancel)

Consume information per site since there could be holiday added on one site or multiple. 

If user selects Select all option, if any school in the select all has a holiday, display as a holiday on the calendar.

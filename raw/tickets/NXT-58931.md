# NXT-58931 — PE - Linked Stock Item Creation - Inclusion of Meal Credits, Serving Size description and Images

## Description
_(none)_

## Acceptance Criteria
*Navigation: PE - MP - Configuration - Import/Export*

Currently the the import process does not cover the meal credits and Images

*Inclusion of Item Categories:* The GPC categories should be created as category if not modified otherwise

*Inclusion of Meal Credits:*

while importing the system should process the meal credits from the corresponding fields

Rules: The values should accept only the list of values as listed in the dropdown

In case of the value is lower than the smallest acceptable value or higher than the acceptable value, then the value should be replaced with lowest value or highest value as applicable

The decimal values on the sheet should be shown as fractions in UI (Ex: 0.5 should be represented as 1/2 in UI)

No Exception handling is needed as there is no change of string values sourced from Manufacturer on these fields

*Images:*

The images should be created in documents section of the ingredient

Maximum of 6 images supported  (JPG and PDF)



*Inclusion of Serving Size Description: (out of Scope) - future use case*

This value should be used to parsed using the multiple measures end point and populate serving measure fields only if they are blank using upload process

# NXT-5316 — Onboarding - System Task List - Add Tour

## Description
As a new SchoolCafé user, I would like to have the option to view a tutorial that shows me how to understand the System task list, so that we can eventually complete our onboarding of SchoolCafé.

*Summary*

Implement Shepherd.js tour steps on the Task List page.

## Acceptance Criteria
Verify that when a user comes to the Task List screen, they immediately see the initial step allowing them to start the Tour

Verify that the user can immediately Skip the tour

Verify that the user can click Start Tour and be taken through the 7 additional steps

Verify that the UI/UX of the text and button colors for the 7 steps is according to the description shown above

Verify that the user can go backwards or forwards in the tour or end it at any time

Verify that the user can see it again if they navigate back to this page or refresh it (not through finishing)

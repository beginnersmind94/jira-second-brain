# NXT-30037 — IM - Items - Create New Item - Product Info (Step 2)

## Description
As a District Admin I want the option to provide general Product Info about an item I am creating so I can populate important attributes.

*Requirements*

* Upon click of Save/Next on [Initial Info (Step 1)|https://primeroedgenext.atlassian.net/browse/NXT-30036], load the next section with the Product Info attributes
** Instead of just loading the new section, we should load the whole row, which would be 2 cards 
*** Product Info (this story), Unit Info grid ([NXT-30046|https://primeroedgenext.atlassian.net/browse/NXT-30046]), Images ([NXT-30038|https://primeroedgenext.atlassian.net/browse/NXT-30038])
* Display the Product Info attributes in a header section
** Item Category
** Storage Category
** Valuation Group
** -Fair Market Value-  (no, this is moved to Unit Info)
* Provide a Skip button
** this allows the user to skip the entire Product Info and Unit Info sections
*** this will only allow a POS item to be created
* Enable the Unit Info grid once the 3 dropdown values are selected
** Unit Info grid: [https://primeroedgenext.atlassian.net/browse/NXT-30046|https://primeroedgenext.atlassian.net/browse/NXT-30046|smart-link]

# NXT-6827 — INV - Vendor - Update Contract Information (Step 1)

## Description
As a Procurement Specialist I need the ability to edit my saved Contract information so I can easily make changes to the editable attributes of my Contracts.

*Requirements*

* An Edit icon on the top right corner of the Contract card
* Ability to click the edit icon and be presented with an editable view of the Contract information and details
** user will be provided the two-step UI similar to adding contract, but some fields will be disabled/locked.

The follow attributes will be editable 

* Contract Info
** Start Date
*** only if the start date is in the future
*** overlapping date range validations will remain
*** will need to validate against orders (future)
** End Date
*** this is needed to allow users to extend current contracts
*** overlapping date range validations will remain
*** Note: in the future we may need to validate end date changes against potential open orders, etc.
** Contract Category
** Status
*** if changing status to Active (from Inactive or any other status), it should apply the date range validations.
** Comments

* Contract Details
** Contract Amount
** Min Purchase Amount
** Max Purchase Amount
** Delivery Schedule(s)
*** in future we will want to validate against open orders and alert the user if they are removing a date for which there is currently an order. I.e. removing Wednesday from the delivery schedule while an open order is set for a Wednesday. (Note: may want to do the same thing for min/max amounts.)
* Save button

# NXT-4132 — System - Notification Templates - Allow user to add images to Notification Template

## Description
As a district administrator, I would like to be able to easily manage all of the letter templates (for letters that will be going out to my parents and students) in one place, so that I can easily make changes as needed.

* In *Notification Templates* page, when the user selects a letter, allow them to add images to their Notification Templates.
** User should be able to upload from their local machine OR via URL.
* Note to Devs from Matt Smithers: Please reach out to Jignesh Shah before starting work on this story. He is the lead template dev from Classic and has some ideas on using database space more efficiently in 2.0 than we did in the legacy application.

Mock-up: [https://nqyc9v.axshare.com/start.html#p=templates&id=x5t8yr&c=1|https://nqyc9v.axshare.com/start.html#p=templates&id=x5t8yr&c=1]

## Acceptance Criteria
Verify that a user can edit the image details of the letter and have it reflect appropriately.

Verify upload can be done via local upload or URL reference.

# NXT-5350 — Workspace - My Modules

## Description
*User Story*
As a PrimeroEdge User, I can see icons for all PrimeroEdge modules that I have access to under the “My Modules” heading.

*Mockup*
[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down-width&hide-ui=1|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down-width&hide-ui=1]

*Requirements*

* Individual icon for each module that I have access to as a PrimeroEdge User
* Clicking the link takes me to the page for that module
* Section resizes vertically to include additional module icons as needed.

## Acceptance Criteria
- My Modules section is present in the workspace
- Functionality present as described in requirements

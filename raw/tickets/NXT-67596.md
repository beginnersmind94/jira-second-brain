## Acceptance Criteria
* Enable serving size selection on the two Update Nutrients screens
** Adjust columns to fit edit pencil / dropdown
** also consider changes from Missing checkbox ( [https://primeroedgenext.atlassian.net/browse/NXT-68104|https://primeroedgenext.atlassian.net/browse/NXT-68104|smart-link] 
* Consume servings that have been added either through item details or the serving info in the tool
* If user has never entered nutrient information and there are more than 1 serving size on the item, provide a way to select serving size 
** IF this is an onboarding session, land the user with all serving size selections already in edit mode
** If this is an update session, user must individually click edit on any items that do not have a serving size selected or if they wish to change serving size to indicate nutrient values
* Serving Size will be a dropdown on both versions of the Update Nutrients edit modes
** it will NOT be available on the Update Vitamins edit mode

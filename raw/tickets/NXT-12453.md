# NXT-12453 — Data Exchange - File Import Monitoring Page - Date format & Timezone

## Description
Ensure the timezone and format of the “Duration” column meets the below criteria.

## Acceptance Criteria
* The date/time for the “Start Time” and “End Time” columns shall be displayed in the timezone of the district.
* The format for the "Duration" column shall be "H:mm:ss". (That is 24 hour with leading zero suppressed: minutes:seconds)

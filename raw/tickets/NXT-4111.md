# NXT-4111 — AcctMgt - Rosters - Landing Page

## Description
*User Story*

As a district administrator I want a page where rosters can be seen and searched for.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=acm_landing_page&id=ravqth&c=1|https://nqyc9v.axshare.com/start.html#p=acm_landing_page&id=ravqth&c=1]

*Requirements*

* Add a Rosters button to the Account Management landing page per the initial wireframes from Karthik.
* Add a Rosters link to the sidebar on the left. (via the hamburger)
* Add a Rosters search page that both the button and the sidebar link lead to.
* Page should contain a way to search for rosters based off the following criteria:
** Roster Name
** Site
* User can search by roster name, or use the site option to view all rosters at a site.
* Results should load into a split view.
** Cards on the left showing the following information: 
*** Roster Name
*** Site(s)
*** Number of active students in the roster

Discussion 8/13/20 - Test Cases

* We’re going to have to show both active and inactive rosters for the time being.

## Acceptance Criteria
Roster landing page added.
Search options added.
Search results load into a split view.
Search results show appropriate information.
Multi-language support added.

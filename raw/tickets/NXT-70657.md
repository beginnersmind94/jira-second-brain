# NXT-70657 — GC- SC - Display source for Cybersoft items on the slide out (USDA Branded foods, CC_GTIN,CC_PC, CC_VIN etc..)

## Description
Tooltip for displaying the source is added in the  results grid and for Cybersoft catalog items. 

* USDA Branded Foods
* CC_GTIN 
* CC_Product Code
* CC_Vendor Item number

No tooltip is required for USDA branded foods.

Viewing an item 

* Display the Data Source on the right end of the item details row on the slide out as shown in the screenshot  below. 
* Cybersoft catalog item: Data Source:  Cybersoft {(name of the source)}
* USDA branded item : Data Source : USDA Branded Foods

Refinement notes: 

* TABLE NAME: GC_ITEMXCUSTOMERDATA

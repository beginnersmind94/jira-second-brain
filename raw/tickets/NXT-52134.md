# NXT-52134 — Acc - Dynamic Entrees - Menu Grids page update

## Description
As Mary the Menu Grid Maker I want to place the dynamic entrees buttons and category onto a grid so it shows up at the POS.

*mockup*

Not needed - existing screens work fine for this.

!image-20240528-034855.png|width=1498,height=685,alt="image-20240528-034855.png"!

*requirements*

If the DYNENTREE setting is turned on:

* User can see menu grids that have Dynamic Entrees added to them easily from the main grid.
** New column: Dynamic Entrees
** If the flag is true, column text is Yes, otherwise No
** User can filter on this column

*Discussion during mini-demo on 7/12/24*

* Column should be visible if:
** The setting is on
** The flag is true for any menu grid
* If both conditions are false, the column should not be visible

## Acceptance Criteria
If the setting DYNENTREE is on:

* New column added to Menu Grids page
** Name: Dynamic Entrees
** User can filter the column to Yes or No

If the setting DYNENTREE is off but the flag is true for any menu grid:

* User can still see the column

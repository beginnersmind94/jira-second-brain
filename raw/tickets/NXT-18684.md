# NXT-18684 — MP > Menu Cycle > Analysis > Nutrition Report>Add Plan Quantity

## Description
This will be an enhancement to the Menu Analysis Report that has already been created.

USDA is requiring a column on the menu analysis report that gives the Count that’s been entered into the text box from the Menu for each menu item.

The Report has sections at the top containing component and nutrition information for the overall menu, and below that section is a listing of each menu item and the nutritional information.

There needs to be a column between the Menu Item Name and the Calorie information next to EACH menu item, that contains the number entered from the menu.  

I have highlighted the column in red on the screenshot - the report itself does NOT need the red box - it simply needs the data. 

!image-20210902-135024.png|width=1920,height=955!

## Acceptance Criteria
A column for “Plan Qty” will appear on the Nutrition Analysis report for the menu cycle, in line with the nutritional details next to each Menu Item.

# NXT-34237 — IM - Recipes - List page updates

## Description
As an IM user I want the Recipes page to have search and filter options that are applicable to the data points important for Recipes so we can search by the important criteria.

!image-20231107-144644.png|width=100%!

## Acceptance Criteria
* Update the Recipe list page to change header filters
** Status (default to Active)
** Tags search
** Contains Ingredient search (change from Contains Item)
** Menu Items Only (checkbox)
* Change the Recipe list columns
** Recipe #
** Recipe Description
** Serving Size
** # of Servings
*** column header search should be set as numeric value and have numeric filters (Greater Than, Equals, Less Than, etc.)
** Strict Batching (hidden by default)
** HACCP Process (hidden by default)
** Menu Item Enabled
*** Either use text (Yes/No) for the values so they can be search, *OR*
*** Use icons but have a Yes/No filter in the column header
** Allergens (comma separated)
** Tags (comma separated)
** *Audit log fields* (all hidden by default)
*** Created Date, Created By, Updated Date, Updated By

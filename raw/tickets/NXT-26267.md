# NXT-26267 — IM- Single Item Recipe- Do not allow more than 1 item or recipe to be added

## Description
As a user, when I am creating a single item recipe, I should not be able to add more than 1 item to the recipe.

## Acceptance Criteria
Do not allow the add step option when a user has added 1 item or recipe
They may delete and be allowed to only enter in 1 item or recipe
Give an indication as why the user can not enter in another step (This is a single item recipe)

* If user trues to add additional step the button will be sable and a tool tip will be displayed indicating this is a single item recipe
* User shall be able to edit serving measures 
* The add button disappears when the user adds an item to the serving measure and shows the message as shown in the screenshot below.

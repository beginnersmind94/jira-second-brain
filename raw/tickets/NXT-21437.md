# NXT-21437 — INV - Inventory - Additions - Report

## Description
As a FSD I want a report on the Additions list page so I can view all the selected transactions and their item details.

*Note:* print functionality may need to be separately handled. Team will research report effort estimation.

*Requirements*

* On the Inventory Additions page, provide a report button on the top right grid header 
* Generate the HTML report on click of the button based on the list filters
** show only current versions of transactions
** do not show cancelled transactions
* Display header (search) level information in the report
** Report Name: Inventory Additions
** Area (if applicable)
** Site 
*** display ‘ALL’ if all sites were selected
** Date Range
* Show field search in the header if applicable
** do not show if there are no search values
** add a label and the search content for each field search, to the report header
*** Ex: Comments: ‘production’
*** Ex: Actual Transaction Date: ‘11/1'
* Display transaction level information rows
** Transaction #
** Site (Code + Name)
** Transaction Date (Date + Time)
** Reason
** Comments
* Display item level rows under each transaction (indented like a sub-grid)
** Item #
** Item Description
** Quantity Added
** Price
*** Comments

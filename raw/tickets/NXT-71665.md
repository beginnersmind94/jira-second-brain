# NXT-71665 — GC - MDM - Create/View/Edit Cybersoft catalog Items in GC Portal.

## Description
* Enable creation of new items exclusively under the Cybersoft Catalog within the GC_Item and associated mapping tables.
* Implement functionality to view and edit Cybersoft items directly from the GC_Item tables.
* Enhance the existing Create UI by aligning it with the View/Edit interface, organizing it into a tab-based layout for each section.
* Implement validation during item creation to prevent duplicate entries based on GTIN. If a GTIN already exists, restrict insertion and guide the user to  edit the existing item instead.
* Strengthen update logic to prevent duplicates by validating against GTIN, Product Code, or VIN before allowing modifications.
* Replace free-text inputs for Manufacturers, Brands, Vendors, Item Category, Storage Category, and Valuation Group with standardized dropdown selections in both Create and Edit screens.
* Remove Edit functionality for USDA items .
* publish and unpublish toggle to update an item unpublish it and publish it, 
** if an item is unpublished then we shouldnot get it in Primeroedge search results and Schoolcafe search results.

# NXT-3804 — Account Management - Advanced Search - Site Section

## Description
The Site Section is a collection of four subsections, Area, Site Type, Site Code and Site Name. Even though the Site Code and Site Name identify a specific site they are separate because most users either select sites by code or name.

!Screen Shot 2020-04-27 at 2.45.30 PM.png|width=308,height=188!

## Acceptance Criteria
When the Site section is expanded the 4 sub sections are available.

When values are selected from multiple subsections they are 'AND'ed together

When a filter is selected from one of the subsections the filter selection count is increased by 1.

When a filter is removed from one of the subsection the filter selection count is decreased by 1.

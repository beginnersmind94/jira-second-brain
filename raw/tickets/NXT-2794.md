# NXT-2794 — **View Inspections - Inspection Details

## Description
*User Story*
As an Inspections user, I have a way to view the Inspection's details from the View Inspections page

*Mockup*
[https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1|https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1]
Switch to Card View, click on the top middle Inspection

*Requirements*

* Clicking on an Inspection in View Inspections takes the user to the Inspection Details screen for that Inspection
** Any available Inspection can be viewed, regardless of Status
* The following information is displayed at the top of the screen:
** Inspection Name
** Site
** Template
** Date and Time
** Status
** Score
* There is a menu with the same actions available on the View Inspections screen
** See NXT-2792
* The header scrolls with the rest of the page
* Questions are displayed
** Note: For this story, placeholder is fine. More robust implementation will come after template implementation

## Acceptance Criteria
* Inspection Details screen can be accessed from View Inspection screen
* Basic details are properly displayed
* Placeholder in place for questions
* Placeholder in place for Actions

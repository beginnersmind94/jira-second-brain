# NXT-55502 — DMF - Dashboard - Auto Trigger

## Description
As a Customer Support role user, I would like to be able to select if I want to auto trigger the modules I am importing so that I can save time

*Notes:*

* After our first prototype review, Dallas asked that we be able to optionally auto trigger the next import 

*Requirements:*

* On the main grid, make these changes:
* Add a button at the top next to the others
** Make it a toggle or checkbox
** Default is No
** Name: Mode
** Tooltip: Auto start next database
** On click:
*** If yes, then automatically start the next phase or database when the current one is done
*** If no, then user will manage manually
* 
* Update the code to do auto or manual starts per this field

## Acceptance Criteria
* User can control if do auto trigger for the next database or control manually

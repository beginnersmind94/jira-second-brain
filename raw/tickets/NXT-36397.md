# NXT-36397 — Participation - Pipeline for Roll-Up Data (Data Exchange)

## Description
Currently the data is prepared manually based on scripts and data gets curated in azure data folders

A new container by the name saledata will be created in the Azure Data Lake.
In this container, the data for each individual ISD will reside in the folder by their respective regionId.
Further, in the regionId folder, there will be folders by academic year description and in the respective academic year description folder, a parquet file by the name saledata.parquet will be created in the rawdata folder by the ADF pipeline.
This pipeline will run every 1 hour.

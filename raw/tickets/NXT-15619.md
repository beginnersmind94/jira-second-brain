# NXT-15619 — INV - Order Template List - View Existing Templates

## Description
As an inventory Operations Manager I want an Order Templates configuration page so I can view my existing order templates, change items on templates or create new templates.

This functionality will allow users to create their own Order Templates and assign items to these templates.

*Requirements*

* Create new Order Templates page under the Orders category
** Accessible from both District and Site level
* Order Templates page will contain:
** a control to add a new template ([NXT-12985|https://primeroedgenext.atlassian.net/browse/NXT-12985])
** a grid for the list of existing templates
*** use pagination on grid
*** all columns will be sortable/searchable
** Default sort for grid:
*** Status column - list all Active before Inactive
*** Template Name - list alphabetical ascending
** +List columns+
*** Template Name column
**** Alphanumeric, allow special characters, character limit of 50
**** Unique across district
*** Number of Items column
**** this will list the quantity of items assigned to the respective template
**** only displayed at site level
*** Number of Sites Using Template column
**** provides the number of sites which have added some items to the template
**** only displayed at district level
*** Status column - for Active or Inactive
**** List all Active templates before Inactive ones
*** Action column ([NXT-15620|https://primeroedgenext.atlassian.net/browse/NXT-15620])
* Save history on all changes (template renaming, deactivation)
** no need to display history on Template List
** renaming template
** deactivating template
** items assigned
** sites assigned

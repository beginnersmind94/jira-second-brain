# NXT-38477 — IM - Items - Units - Display Pack Size prompt for Serving Sub-Qty

## Description
As an IM user I want the ability to quickly make a Serving into a Pack Size base unit so I can quickly set up my dual unit items.



!Dual Unit Promote-20240917-070505.png|width=83.33333333333334%,alt="Dual Unit Promote-20240917-070505.png"!



!Dual Unit Confirm-20240917-070505.png|width=83.33333333333334%,alt="Dual Unit Confirm-20240917-070505.png"!

## Acceptance Criteria
* If Sub-Qty is entered (and saved) for an inventory-able serving, display a ‘Pack Size?’ button in the Identifiers column 
** while in edit mode only
** use the inverted identifier (bordered) style
* If user clicks the prompt button, display a sub-step prompt in the banner to confirm they want to promote the serving
** Promotion sub-step resembles “merge” sub-step from [https://primeroedgenext.atlassian.net/browse/NXT-40778|https://primeroedgenext.atlassian.net/browse/NXT-40778|smart-link] 
* Do not show the button if there are already 3 pack sizes
* Do not show the button if the pack size is already synced/locked

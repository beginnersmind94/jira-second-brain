# NXT-36948 — PR- Reports- Plan Worksheet Enhancements for multiple and Single day (API ONLY)

## Description
As a Production Manager, I would like to be able to see my Strict Batching information, where my carryovers are coming from (if applicable) and the contribution for the serving size on the menu and not the default serving size contribution.

## Acceptance Criteria
* Display Strict Batch Information under the menu item if applicable
* Display “Previous Carryover: (carryover quantity)” if applicable
* Display the contribution of the serving size on the menu and not the default serving contribution
* This should apply to multiple day reports and the single day from the plan or record page

# NXT-28374 — MP- Menu- Menu Templates

## Description
As a user, I need an opportunity to be able to select the template help me start my menu creation.

Once I create a template, I will need to use my templates to create my menus.

## Acceptance Criteria
* Once Template is created, allow templates to be selected from the menu when adding a new menu only. 
* Display what template was used and save that information on the menu.
** Make the label of the template name that is used as a link in a new tab to the template 
** Ex: Template Used- Milk Template (_Used on 3/2/2023)_
* Only allow the templates that belong to that meal type and site group to display to be selected
* Once template is selected and menu is saved, display the items that were in the template on the menu.
* User will then be allowed to continue working on their menu as normal

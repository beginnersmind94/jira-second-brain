# NXT-71502 — SC - PBIRD - Dashboard - Starred

## Description
As a Director, I want to mark Power BI reports as favorites so that I can quickly access them from my Workspace Dashboard.

----

h2. *Description*

Enable users to mark and unmark Power BI reports as favorites from the Power BI Reports dashboard.

This functionality will behave exactly the same as the existing *Managed View Reports favorites (star)* functionality and will integrate with the *Workspace Dashboard (My Reports → Starred)*.

----

h2. *User Flow*

# User navigates to:
#* *Reports → Power BI Reports*
# User views list of reports
# User clicks the *star icon* in the grid:
#* If not starred → becomes starred
#* If starred → becomes unstarred
# Star state updates immediately
# Starred reports appear in *Workspace Dashboard → My Reports → Starred*

----

h2. *Out of Scope*

* Workspace Dashboard implementation _(separate story)_
* Mobile app behavior _(separate story)_

----

h2. *Requirements*

h3. *Favorite Toggle Behavior*

* User can:
** Click star icon to *mark as favorite*
** Click again to *remove as favorite*
* Behavior must match Managed View Reports exactly

----

h3. *Persistence*

* Favorite selection must be:
** Saved to the database
** *User-specific* (not global)

----

h3. *UI Behavior*

* Star icon reflects current state:
** Filled = Favorited
** Unfilled = Not favorited
* Update should occur:
** Immediately on click (no page refresh required)

----

h3. *Workspace Dashboard Integration*

* When a report is marked as favorite:
** It appears in:
*** *Workspace Dashboard → My Reports → Starred*
* When a report is unmarked:
** It is removed from the starred section

----

h3. *Consistency & Reuse*

* Reuse existing favorite/star logic from Managed View Reports
* Follow same:
** Data model pattern
** UI behavior
** Interaction pattern

## Acceptance Criteria
* Given a user views the Power BI Reports dashboard, when the grid loads, then each report displays its correct favorite state
* Given a user clicks the star icon on a non-favorited report, when the action completes, then the report is marked as a favorite
* Given a user clicks the star icon on a favorited report, when the action completes, then the report is removed from favorites
* Given a report is marked as favorite, when the Workspace Dashboard loads, then the report appears under My Reports → Starred
* Given a report is unmarked as favorite, when the Workspace Dashboard loads, then the report no longer appears under My Reports → Starred

# NXT-61562 — AI Onboarding - Workspace - System Setting

## Description
As a SchoolCafé user, I would like to have a setting that controls if we show AI stuff to users or not so that I don’t see it if my district doesn’t have it.

*Requirements:*

* Do the following in System Settings:
* In System Settings, add a new property
** Setting Name: Enable AI
** Type: Internal
** Code: USEAI
** Value: Yes/No
** Default: No
** Category: Configuration
** Module: System
*  
* *Security*
** Only Customer Support roles can make this change

## Acceptance Criteria
* In System Settings, add a new property with the values defined above
* Only Customer Support roles can make this change

# NXT-23276 — INV - Storage Locations (Bins) - Add Put Away to Receive screen

## Description
As a Site Manager I want a Put Away option on the Receipt screen so I can indicate the destination bin and put away items at the same time I am receiving them.

This should be built after the dedicated Put Away screen story, [https://primeroedgenext.atlassian.net/browse/NXT-1737|https://primeroedgenext.atlassian.net/browse/NXT-1737|smart-link] 

*Requirements*

* Add an optional tool to the Create/Edit Receipt screen to conduct the Put Away process for each item 
* This would only display for items assigned to bins
* This would not display for items which were already Put Away (completed)

## Acceptance Criteria
* Add the Put Away tool to the Receipt screen as an optional ability to perform Put Aways at the same time of receiving

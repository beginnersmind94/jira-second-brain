## Acceptance Criteria
* Reduce grid size and consolidate actions into the grid header
** Right-aligned Count Sheet report button
** to the left of Count Sheet button, the Add Item button
** to the left of the Add Item button, two toggle options (discuss if these require some apply action)
*** Show Item Category - default ON
**** this will show Item Category as a simple column, to the right of Item Description
*** Group by Item Category - default ON
**** this will remove the Item Category column and instead will group items in the grid according to Item Category (how it currently looks) within each Storage Category
* Remove Valuation Group from the grid
* Provide a sticky footer
** a left-aligned Back button to go back to the list page
** a right-aligned Save button (this will always save as Incomplete)
** to the left of the Save button, show the autosave banner message

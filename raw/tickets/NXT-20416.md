# NXT-20416 — MP- Menu Planner- Save and Continue- (Confirmation Dialog/Validation) (UI)

## Description
As a user, I want to be able to save my work from Menu Planner Page and continue to the next part of the process of creating menus.

Mock up:

!image-2021-10-06-12-57-36-263.png|width=200,height=183!

 

There will be two different scenarios for this option. 

Scenario 1:

* As a user adds items onto a specific day, the menu name field is made required for that particular day. 
* Once they click save and continue, the system should recognize that only one menu was created and navigate the user to the menu page for additional modifications to be made if needed. 
* Prompt a confirmation screen for user to accept changes
 
!image-2021-10-06-13-04-18-978.png|width=200,height=183!

Scenario 2:

* As a user adds items onto *more* than one day, the menu name for each day will become required and Menu cycle name will also become required to save.
* Once they click save and continue, the system should recognize that a menu cycle was created and navigate the user to the menu cycle page for additional modifications to be made if needed.
* No matter how many days of information are entered, the default menu cycle created will be 1 week 5 day cycle or 7 if include weekends is enabled. 
* Prompt confirmation screen to accept changes
 
!image-2021-10-06-13-07-50-532.png|width=200,height=183!

If a user adds additional weeks, DO NOT SAVE additional weeks unless there are menu items on additional weeks.

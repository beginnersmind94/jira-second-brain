# NXT-23620 — INV - Orders - Receipt Detail - Receive Report - Items

## Description
As a Site Manager I want an Item list on the Receipt Report so I can see a summary of all the items I received along with any charge/discount or date on product information.

Page + Header: [https://primeroedgenext.atlassian.net/browse/NXT-7994|https://primeroedgenext.atlassian.net/browse/NXT-7994|smart-link]

*Requirements*

* List all receipt items on the report
** list all items in the same order of how receipt page is displaying the items
*** items are grouped by Item Category (this will be compared and evaluated)
** Display the fee and NOI information for each line item
*** display these as additional columns, like the page design (not like PrimeroEdge report)
** Display the Date on Product info for any items where it was provided

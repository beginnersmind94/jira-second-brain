# NXT-2869 — Inspections Dashboard - Calendar Widget

## Description
*User Story*
As an Inspections User, I have a calendar widget available for my dashboard that displays both completed and scheduled inspections.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=inspection_dashboard&id=0rb254&g=1|https://nqyc9v.axshare.com/start.html#p=inspection_dashboard&id=0rb254&g=1]

*Requirements*

* User can select a Calendar widget for their dashboard
** Calendar is one of the default widgets
** Calendar is available for all roles
* By default, calendar displays the current month
** Current date is indicated when viewing the current month
** User can look ahead or in the past by using arrows to increment by 1 or clicking the month to select a different month/year
** Calendar only displays by single month
* There is an indicator on any date that has inspections
** Mousing over the date displays the Inspection and Site in a popup
** User can click on any inspection to view it
** It is indicated whether the Inspection is scheduled or already submitted
** There can be multiple Inspections shown for a single date
* Current day’s Inspections are displayed beneath the calendar
** The current date is displayed as a header
** Inspection name, template, and site are displayed
** For admin view, the Inspector name is displayed as well
** Clicking the Inspection takes the user to the Inspection
** If no Inspections are scheduled, there’s a message that says “No Inspections Scheduled for Today”
* Admins can toggle between their own Inspections and all Inspections using the Show Only Mine toggle
** If toggle is set to off, then all Inspections for their district are shown
* Inspectors will only ever see their own Inspections

## Acceptance Criteria
* Calendar widget present and functioning on Inspections dashboard screen
* Data is properly displaying for Inspections
* User can properly navigate to individual Inspections
* Admin Show Only Mine toggle is functioning properly

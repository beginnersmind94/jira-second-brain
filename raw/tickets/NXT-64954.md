# NXT-64954 — WS - Benchmark - Paging & Sticky Headers Update

## Description
As a Workspace user, on the Benchmark drawer, I want to be able to see the grid better, so that I don’t have to using paging

*Use Case*

* Users will typically just want to see all the schools without paging, so we need to give them the option to see all

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* Add an All option to the paging
** Make All the default value
* 
* Make the column headers sticky so that they continue to show if scrolling down
* 
* Make the first column sticky and add a horizontal scroll bar

## Acceptance Criteria
* Users are able to see all sites by default without paging
* column headers is sticky for vertical scroll
* first column is stick for horizontal scroll

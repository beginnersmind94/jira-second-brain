# NXT-27902 — Add a drop down on the notification configuration screen for displaying templates

## Description
As a system we need the ability let the user specify the template they would like to associate with a business process. We need the templates drop down list to display the templates irrespective of the template's status.

## Acceptance Criteria
Story should be considered done when:

# We select notification type scheduled, select a module and select a business process, select the content type as letter then we see the templates drop down list containing all the templates associated to that business process.

# NXT-10957 — [TaggedDashboard] - Dashboard (Insights) - Accountability - Meal Participation

## Description
As a central office user or cafeteria manager I want the ability to see the reimbursable meal counts and participation percentage in relation to the number of patrons enrolled at that eligibility status for a site within a program, meal type, and eligibility.

*Visualization Needed:*

* Data table as shown below for Breakfast, Lunch, and Snack programs. Note: programs are not configured or the district is not operating for a program, the program can be hidden.
* Data Visualization pie charts display the percentages breakdown for the meal type by eligibility.

!image-20210324-132839.png|width=1190,height=845!

*Values Needed:*

Information collected based on the site and timeframe parameters selected. 

* Free Student Count
* Reduced Student Count
* Paid Student Count
* Free Student Reimbursable Breakfast Meal Count
* Reduced Student Reimbursable Breakfast Meal Count
* Paid Student Reimbursable Breakfast Meal Count
* Total Student Reimbursable Breakfast Meal Count
* Free Student Reimbursable Lunch Meal Count
* Reduced Student Reimbursable Lunch Meal Count
* Paid Student Reimbursable Lunch Meal Count
* Total Student Reimbursable Lunch Meal Count
* Free Student Reimbursable Snack Count
* Reduced Student Reimbursable Snack Count
* Paid Student Reimbursable Snack Count
* Total Student Reimbursable Snack Count

Until site and timeframe selector is available, start with the current month of operation and all sites available to the user.

*Inputs Needed:*

* Site Context - Site Selector for (All Sites, Multi-Site, Single Site). By Default, based on the authenticated user
** if a user is a district-level user, then it should be “All Sites”
** if a user is a site-level user having access to multiple sites, then it should be “All Sites”
** if a user is a site-level user having access to a single site, then it is selected (no dropdown is needed)
* Timeframe - By default, Today is the selection. 
** Today
** Last 7 Days
** This Month
** Year to Date

*Calculations Needed:*

* Percentage Calculations
** Daily = Count / Approved
** Weekly = (Count / Operating Days over past 7 days) / Approved
** Monthly = (Count / Operating Days over current month) / Approved
** YTD = (Count / Operating Days over current academic year) / Approved
* Total Student Reimbursable Breakfast Meal Count = F + R + P for breakfast
* Total Student Reimbursable Lunch Meal Count = F + R + P for breakfast
* Total Student Reimbursable Snack Count = F + R + P for breakfast

## Acceptance Criteria
* Meal counts by eligibility and program are displayed
* Percentages by eligibility and program are displayed
* Totals by program are calculated and displayed
* Visual donut charts of participation percentage by eligibility for each program are displayed
* Data table values and visualization charts are updated/displayed based on parameter selections

# NXT-67634 — IM - Bulk Item - Contributions Section add "None" Option

## Description
As a nutritionist, I would like for a way to indicate which items I will never add contributions for. Currently the system is assuming that at some point contributions will be added for every item. I would also like a way to bulk apply for multiple items at once. 



!IO - Contributions View v2-20251117-231002.png|width=444,alt="IO - Contributions View v2-20251117-231002.png"!

!Bulk Onboard - Food Components-20251117-231313.png|width=444,alt="Bulk Onboard - Food Components-20251117-231313.png"!

## Acceptance Criteria
* Provide a column for None indicated on the component step 
** This can be selected with no serving size provided and will apply to the item
** If any components are selected, None option is disabled
** if None option is selected, clear any selected serving size
* In this step, provide a bulk Apply function that allows for users to mark a bunch of items as None Indicated all at the same time
** If components have been selected for a particular item, components must be unselected to be able to mark an item as none indicated
*** This applies to both bulk apply or individual
*** Disable the ability to select until all components have been removed or user is in the middle of selecting a serving size
* On the Review, varieties and values pages, grey out the row and provide a message that says “ 🚫 No Contributions” in the Serving Size column
** Do not show items with No Contributions on the Varieties and Values screens
* Add ‘No Contributions’ option to Contributions filter 
** Add it right below ‘Not Provided’

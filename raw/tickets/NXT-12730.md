# NXT-12730 — IM - Production Recipes - Recipe within a Recipe - Search

## Description
As a use, I need to be able to search and add a recipe into my recipe. 

Requirements:

* I need to be able to search by recipe name or recipe code in the same place I search for items in the recipe step
* Change Label to
** Item / Recipe Search
* This would just use the same search
** Items over recipes, split half and half, so show ingredients first, then recipes, and then ingredients again
** 5 items, 5 recipes, 5 items, etc
** Prioritize exact match over partial matches

## Acceptance Criteria
User needs to be able to search and select a recipe, as well as enter it in like a food item

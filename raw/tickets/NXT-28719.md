# NXT-28719 — INV - Transfers - Transfer Detail - Edit Transfer

## Description
As a Site Manager, or District admin, I want to be able to Edit a Transfer so I can adjust quantities and comments.

*Requirements*

* Enable the Edit function on Transfer Details (according to the rules from [https://primeroedgenext.atlassian.net/browse/NXT-28647|https://primeroedgenext.atlassian.net/browse/NXT-28647|smart-link] )
* Edit enables 
** Quantities
*** they can only be reduced
** Comments
* Adjust perpetual for both sites upon quantity adjustment
** May add quantity back to *sending* site
* Show Save/Cancel button in place of Edit during Edit mode
* Do not show Edit button if there is a completed physical inventory for the items or the period is closed.

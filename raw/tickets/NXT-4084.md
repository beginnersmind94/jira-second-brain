# NXT-4084 — Item Management - Split View - Search - Filters

## Description
As an Item Management User, I want to be able to filter my results for items while viewing my results.

*Mockups*

[https://nqyc9v.axshare.com/itemmanagement_-_super_user.html|https://nqyc9v.axshare.com/itemmanagement_-_super_user.html] 

!image-20200804-142806.png|width=394,height=647!

[https://material-ui.com/components/autocomplete/#limit-tags|https://material-ui.com/components/autocomplete/#limit-tags]

*Requirements*:

* The filter option will be besides the search box
* The available filters are below and mult-iselectable
** Category
** Datasource
** Contributions (TBD actual selections)
** Allergens (TBD actual selections)
** Tags
** Active / Inactive
** The filter values are going to display the options that are available for all data accessible to the user.
** It will show the categories, the available tags the user can see, the datasources available, the types of contributions, and allergens
* When showing the filters, only show the first 10 results for each filter, and 'show more' if there are more to show BEFORE they leave the search box
* Apply will apply the filters to their current results
* Cancel will cancel the window and not apply the filters
* The user can 'x' out of the tags that are there
* Filter icon will show the number of filters that are active

## Acceptance Criteria
* Available filters need to be visible after clicking the filter icon
* Results for the filters will display the values available to that user
* Allow the user to type in the filter of what they are looking for and show the matches for the filter, which they can then select
* The user will be able to select the filter, and then click 'Apply' to apply the filters
* The user will be able to 'Cancel' which will take the user back to their results without using the filters they selected.

# NXT-23816 — INV - Vendors - Items - Provide orderability flag for Contract Items

## Description
As a District Admin I want an Orderable flag on the Contract Items so I can make items unorderable without removing them from the contract.

*Note:* this means we will also need to add a proper way to filter/display unorderable items, and possibly show the item as (partially) available on the Add Item to Contract screen.

*Requirements*

* Provide a flag on contract items which allows items to be made unorderable 
** this would remove the item from any order screen
** checkbox in the contract side panel
*** only in edit mode; no need for an ‘Orderable’ label in view mode
**** this means the checkbox should not disrupt/alter the appearance or placement of the other fields
*** this would be enabled as part of the edit mode
*** checked by default (obviously)
*** do *not* add the Orderable checkbox to the ‘Add Item to Contract’ side panel
* Alert on Save when making items unorderable
** “This item will no longer be orderable to sites as part of this contract.”
* Do not remove the item from existing orders
** items on existing orders *must* be receivable

# NXT-2793 — View Inspections - Actions - Export to PDF/Export to Excel

## Description
*User Story*
As an Inspections user viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to generate a report of the Inspection in pdf or xlsx.

*Mockup*
[https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1|https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1]

*Requirements*

* There are two actions for reports when viewing an Inspection
** This action is present on any Inspection
** This action is present for any users who can view the Inspection
* Users can generate reports in PDF or XLSX
** Each format has a separate action
*** Export to PDF
*** Export to Excel
* If user chooses to Export PDF or Export Excel, the file is generated and downloaded to the user's machine
* Reports reflect the latest state of the Inspection

## Acceptance Criteria
* Reports can be generated in both file formats from the View Inspections screen in either view (card or list)
* Reports can be generated in both file formats from the Inspection Details screen

# NXT-5513 — System - Periods - Display Period Configuration section (no functionality)

## Description
As a a district financials administrator, I would like to be able to specify my district's fiscal year periods within that year, so that I can view reports and claims for the right date ranges.

Summary:

* Display the Period Configuration section on the Generate Periods page. 
* Drop-downs/menus should contain proper options and be dynamic based on what the user has selected previously. 
* Looking at the screenshot below, everything except the Preview Periods button should work based on the following details and mind-map.

Details:

* On the Generate Periods page, when a user chooses a fiscal year where dates have been set, display a Period Configuration section with 2 parts:
** +Period Type+
*** Drop-down options (*display in this order*):
**** Sales Invoice Periods 
***** default selection
**** POS Periods
**** Inventory Periods
**** Financial Periods
** +View Periods+ (header was formerly called ‘Create Periods’).
*** When Period Type is selected, several options become available. We should follow the following logic:
**** *Frequency* - options:
***** Weekly
***** Biweekly
***** Semimonthly
***** Monthly
**** *End Day* - options
***** _IF Frequency = Weekly OR Biweekly_: 
****** Sunday
****** Monday
****** Tuesday
****** Wednesday
****** Thursday 
****** Friday 
****** Saturday
***** _IF Frequency_=_Semimonthly_:
****** 14th
****** 15th
****** 16th
***** _IF Frequency =_ _Monthly_:
****** None ('End Day' drop-down/field can be hidden in this case) 
**** *Include Date Range in Period Name* - can be checked or unchecked.
* Once these are set, everything should be in place to allow the user to preview and then create/modify their periods. That functionality will be in the next story.

# NXT-35786 — Program Access / Participation - API - Date Filters

## Description
The following improvements should be in UI for better user experience

## Acceptance Criteria
The site code should be enabled to reflect in the UI

The user should be able to apply filters based on the below date ranges:

Possible Date Ranges:

·        Current Week – Start Date of current Week Until end of the current week (Monday to Sunday)

·        Last Week – Start Date of Last Week Until end of the Last week (Monday to Sunday)

·        Current Month: State Date of the Month Until Current Date

·        Last Month: Current Month: State Date of the Month Until Current Date

·        Academic Year: 07/01/2022 (Default to Current Academic Year)

·        Custom Range: Defaulted to Start Date of current Week Until end of the current week (Monday to Sunday), but user can select any date between 07/01/Current Year until 06/30/Next Year

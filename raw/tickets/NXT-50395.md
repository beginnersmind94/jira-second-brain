# NXT-50395 — Imports – Files - Adults - Exceptions

## Description
We need to add our current method of exception handling like we do for Students to Adult Imports

*Notes:*

* Splitting up the stories:
** API is in another one
** Files is in this one

*Requirements*:

* On the System > Imports & Exports > Files tab > Adult Imports, do the following:
* The requirements apply to both delimited and fixed width imports
* Change the way exceptions are handled to be done like we do the Student Imports
* All the existing functionality such as exception types of Mandatory, Optional, Comment should be included
* 
* Add Exceptions tab to Files Adult Imports
* Do it the same way as for Students
* Exceptions appear in the exceptions tab
* Exceptions count are accurate
* Ensure users can download the data
* Ensure users can show/hide the filters row

## Acceptance Criteria
* File Adult Imports track and report on Exceptions
* Exceptions tab is visible in View History drawer
* Users can download the data
* Users can show/hide the filters row

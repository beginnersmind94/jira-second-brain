# NXT-64955 — WS - Benchmark - Site Type Filtering

## Description
As a Workspace user, on the Benchmark drawer, I want to be able to filter the grid by site type, so that I can work on just some sites at a time

*Use Case*

* Users might want to just work on benchmarks by site type group, for example just do all the High Schools at once by hand since they might all have the same targets

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* Add a field to the screen
** Label: Site Type
** Values:
*** All
*** list of site types for the district (same you use for the site types in the Filters on the benchmark page)
** Default: All
** On click: filter the table to just the sites that are in that site type
* 
* Auto Fill works on only the selected sites--new story

## Acceptance Criteria
* Users are able to filter the grid by site type correctly

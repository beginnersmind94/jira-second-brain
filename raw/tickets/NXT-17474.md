# NXT-17474 — INV - Orders - Receipts - Update In Transit quantities

## Description
As an Inventory user I want the system to track Committed and In Transit quantities so I can know that the system has an accurate estimation of which items I have and will receive.

*Requirements*

* Decrease In Transit quantity as receipts are created

Other story (for orders) - [https://primeroedgenext.atlassian.net/browse/NXT-17454|https://primeroedgenext.atlassian.net/browse/NXT-17454|smart-link] 

* Track ‘In Transit’ quantities for items as transactions occur 
* Increase In Transit quantity as orders are created
* Adjust In Transit quantity as orders are edited
* Decrease In Transit quantity as orders are closed (for items not already received)
* Decrease In Transit quantity if orders are cancelled

In Transit = Open order quantity - adjustments - receipts - cancellations/closures

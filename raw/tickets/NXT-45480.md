# NXT-45480 — Acc - Transactions - Serving Exceptions - Processing students with State Eligibility

## Description
As Emma the Eligibility Officer I want to see State Eligibility used for Serving Exceptions so eligible students have free or reduced meal pricing for NSLP and SBP.

*requirements*

Mostly need to ensure that the Serving Exceptions updates check for the presence of State Eligibility, and use the correct pricing.

* If the system setting is turned on:
** Serving Exceptions page should show the state eligibility under Serving Eligibility and Approved Eligibility
*** Format: USDA (State) - Example: Paid (TX-Free), Paid (TX-Reduced)
** Serving Exceptions via Transactions needs to charge eligible students appropriately:
*** Student’s State Eligibility is Reduced, use the Reduced meal price
**** Meals are USDA Paid, but State Reduced
*** Student’s State Eligibility is Free, use the Free meal price
**** Meals are USDA Paid, but State Free
* This applies to student first meals only, and for NSLP (Lunch) and SBP (Breakfast) meal types only.

## Acceptance Criteria
* Use State Eligibility in Serving Exceptions
* Serving and Approved Eligibility shows the state eligibility if present.
* Changes made for NSLP and SBP meal types only.
* Second Meals via Duplicate Meals should use the regular Second Meal price as per usual.
* Must be a reimbursable meal.

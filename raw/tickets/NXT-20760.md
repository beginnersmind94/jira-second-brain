# NXT-20760 — Add Select All to Create Model Page

## Description
As an enhancement, we need to improve the efficiency of adding menu items to the menu list.

*Application*: Face and Food Detection Website

*Background:*

* We accepted the functionality as-is from Amniscient in order to meet the deadline, but now need to add in at least a little bit of functionality that we originally wanted.
* We want to add in a “Select All” functionality so that the user doesn’t have to type in all their food items, which could be in the hundreds
* The core requirement is the ability to select all the menu items at once rather than one at a time
* To do that, we think a “Select All” button that loads all the menu items into the list would be the quickest and easiest way, but we are open minded, so the requirements below reflect this
* If there is great difficulty in doing this, talk to me

 

*Flow:*

* Browse to PrimeroEdge and click the link for Food and Face Detection
* Browse to the Model Management
* Click on Create Model link
* Click on Select All button

*Requirements*:

* Add a “Select All Food Items” button on the Create Model popup
** button can be a hyperlink or icon
** needs to be associated with the Food Items, so place next to or on top off the existing Food Items control
** On click, load all the menu food items into the list
* The rest of the popup functionality should stay the same
*  
* On Model Management, please make these UI updates:
** label = Description
** tags = Tags
** food_items = Menu Items
** createdAt = Build Date
** status = Status
** active = Active.

*Mockup*:

* None

## Acceptance Criteria
* Button is added to the popup
* When clicked, the button loads all the menu items to the list of Food Items
* Grid headers relabeled as per above

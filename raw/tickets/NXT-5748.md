Requirements

* Recipe Name - Type in Recipe Name
* Datasource - Dropdown - Default to 'Local'
* User will type in the Recipe Name (Use the same default length as classic)
** Recipe cannot have the same name as another Recipe at the same datasource
** Minimum of 3 characters
** Recipe can have the same name but a different Short Name
*** Short Name is not a visible field and not user editable at this time (used for importing classic recipes, may be put in the future)
* User will select ‘Save and Continue’ after datasource to check for the name.
** Recipe is created under selected datasource and recipe name at this time if no duplicate.
* If there is a duplicate
** *A Recipe with this name already exists. Please enter in a unique name.*
* If no duplicate, show the rest of the fields in [https://primeroedgenext.atlassian.net/browse/NXT-5183|https://primeroedgenext.atlassian.net/browse/NXT-5183|smart-link]

## Acceptance Criteria
User must be able to add steps to a recipe

# NXT-4689 — Item Management - Table View - Apply Filters

## Description
*User Story*

As a user, I want to be able to filter my results.

*Mockup*

!image-2020-07-29-16-44-57-713.png|width=83.33333333333333%!

 

*Requirements*

* Have a 'FILTER' button available to show a dropdown for available filters
* Available filters are: This will be an AND between the filter selections
** Category - Equal | Or
** Datasource - Equal | Or
** Contributions - Equal | Or
** Allergens - Equal | Or
** Tags - Equal | Or
** Status - Equal | Or
* CANCEL Button on filter popup - Cancel without applying any of the selected filters
* APPLY button on filter popup - Applies the filters to the current results
* Filter button will show the number of active filters selected

## Acceptance Criteria
* The user will be able to filter data with stated fields in the requirements

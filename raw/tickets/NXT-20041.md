# NXT-20041 — Find or display reports

## Description
As a user I need to have a screen to display all the data query reports that were saved by the user. The reports table should allow the user to filter its data.

## Acceptance Criteria
Acceptance criteria will be met when:
1. The screen will be able to mock display the data being sent by the API.
2. The reports grid should be filterable and each row should have action buttons for Edit, Delete, Export and Share. 
3. The reports need to be categorized in the tabs as Private or Shared

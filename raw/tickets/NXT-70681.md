# NXT-70681 — GC - SC- New report to give details on Search Filter page of All, Cybersoft and USDA datasource.

## Description
# New reports to show the details from search by filters for All, cybersoft and USDA datasource. 
# Report information is as shown in the mockup below. 
# New report should follow all reporting standards we have in the product for font, layout , showing generated by and powered by text and district name etc..
# All export option we have should work. 
# Report is available for search by filters options. 
## Search by filters when search by GTIN/PC/VIN codes. 
### in this case there wont be “Matched by ingredient name - multiple matches found - needs review” section. we can either hide that section or show the standard message “No data available” 
### i have updated the column headings to suit both the options in the below mockup. 
## We can differ report when searched by ingredient name if needed 
### i feel, we can still allow printing the report. when printed the report for ingredient name results, then under source item details columns there wont be any matched code column value, and match type can be “Ingredient name” for all and ingredient name column will show the matched ingredient name for the given input.
# Column headers should repeat on all pages if report printing in multiple pages and no blank pages should be printed when printed and exported to PDF.
# Match type is going to be the “Code type” from our catalog. 
# Source ingredient name column will be blank when its generated from search by filters option. 
# Sorting for all sections : Source Ingredient name if exists, followed by matched code column value .

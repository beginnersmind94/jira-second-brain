# NXT-21384 — Add Delete to New Images

## Description
As an enhancement, we need to be able to delete unwanted images in the Food Training website

*Application*: Face and Food Detection Website

*Background:*

* We accepted the functionality as-is from Amniscient in order to meet the deadline, but now need to add in at least a little bit of functionality that we originally wanted to be able to delete unwanted images

 

*Flow:*

* Browse to the Food Detection Settings page and then the New Images Page or the Trained Images page
* On any given image in the grid, click the image
* User taken to the Image training page

 

*Requirements*:

* These requirements apply to both New Images and Trained Images, as it is assumed you are using the same page for both
** If separate pages, then do the same thing for both pages
* In the section on the top right where the save button is, add another button called Delete
* If the image is linked to a Model, then disable the delete button
** See Add model Configuration to Database story to know how it is linked
* If it is not linked, then enable the button
*  
* On button click, do the following
** Assumes delete button is disabled if linked to a model
** Delete the physical image from the system
** Do not delete the database record of the image, but mark its status as Deleted
** Record the date/time stamp of the deletion in the DB record of the image
** Advance the image to the next one in the list simulating the next button
*** if difficult, then show a Deletion Successful message on the screen

 

*Mockup*:

* None

## Acceptance Criteria
* In the section on the top right where the save button is, there is a button called Delete
* If the image is linked to a Model, then the delete button is disabled
* If it is not linked, then the button is enabled
* On button click, it does the following:
** Deletes the physical image from the system
** Does not delete the database record of the image, but mark its status as Deleted
** Records the date/time stamp of the deletion in the DB record of the image
** Advanced the image to the next one or shows a Deletion Successful message on the screen

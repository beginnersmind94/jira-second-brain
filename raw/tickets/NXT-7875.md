# NXT-7875 — Login Screen - Add functionality for 'Remember Me' Checkbox 

## Description
As a PrimeroEdge user, I would like to have the option to have the application always remember my username, so that I can more quickly log in and begin working.

*Summary*:

On the Login/homepage, complete the functionality for the 'remember me' checkbox and have the screen display differently when it is checked vs unchecked.

*Details*:

* On the main homepage, check whether the following person visiting the page is: 
*# a visitor (non-user)
*# a brand-new user who has selected the link in their New User email and been redirected here 
*# a returning user that did not have the ‘Remember Me’ checkbox checked
*# a returning user that *did* have the ‘Remember Me’ checkbox checked
* In cases 1-3 above, display the homepage as it is displaying currently.
* However, in case 4, display the homepage differently (see concept images/mockup below), where only the user’s password is required
** Also, display a button that allows the user to log in with a different username if needed. That button returns the page to the default display (both username and password are now required)
* Whenever the user has modified the checkbox *and logged in afterward*, perform the check above when they return to this homepage
* This story is complete when a user, who has checked the ‘remember me' checkbox and then logged in, does not need to re-enter their username when signing out or otherwise returning to the homepage at a later time (only their password should be required, which should also allow auto-fill from their browser). They should also be able to deselect the checkbox when logging in and then have the homepage display as default again.

## Acceptance Criteria
Verify that on the main homepage, we check whether the following person visiting the page is someone who did not have the ‘Remember Me’ checkbox checked, or a returning user that did have the ‘Remember Me’ checkbox checked

Verify that for non-remembered users, we display the homepage as it is displaying currently: both a username and password are required for login

Verify that for remembered users, we display the homepage differently to where only their password is required for login
-Also that we display a button that allows the user to log in with a different username if needed, in this case. That button returns the page to the default display

Verify that whenever the user has modified the checkbox and logged in afterward, we perform the same check above when they return to this homepage and repeat the steps above

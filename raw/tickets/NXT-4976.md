# NXT-4976 — AcctMgt - Patron - Adult Details - Restrictions

## Description
*User Story*

As a district admin I want to see the information on an adult’s account.

*Mock-up*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1]

*Requirements*

* User should be able to load an adult off of the search page and see their information.
* Adult accounts are a subset of the information we’ve put together for students.
* The fourth panel will be the Restrictions information.
* User should be able to view and edit information, like a student.
** Editing will be a full drawer.
* All information and functionality is identical to the student account, just saved for this adult account instead.

## Acceptance Criteria
An adult's information can be viewed while searching / clicking on a card.
Restriction information can be viewed/edited appropriately.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)

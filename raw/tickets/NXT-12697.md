# NXT-12697 — Platform - Security - View Resource Catalog

## Description
As a primero edge admin, I should be able to access resource catalog to view list of resources available for any district so that I can view, edit, add or remove resources the respective modules under system configurations. 

Persona: CyberSoft Admin

*Mockup*: 

[figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=426993%3A304123&viewport=375%2C529%2C0.10553674399852753&scaling=scale-down-width|http://figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=426993%3A304123&viewport=375%2C529%2C0.10553674399852753&scaling=scale-down-width]

Requirements:

* Resource table has Resource name, Type and module columns along with actions column sorted by default descending Resource Name
* Resource Types can either Page, Section or Field. 
* Roles can be viewed which are mapped to a specific resource
* Available actions column with EDIT, REMOVE buttons displayed
* “Add New” button is enabled

## Acceptance Criteria
# User shall be able to view resources table under Resource Catalog page within System/Management module
# User shall be able to view list of resource available in the tabular fashion
# User shall be able to see EDIT and Remove buttons in action column
# User shall be able to see mapped role for a given resource
# User shall be able to view “add new” button on top right of the Resource Catalog page
# User shall be able to sort any column in the resources table
# User shall be able to filter available at any column

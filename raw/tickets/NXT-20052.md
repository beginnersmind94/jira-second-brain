# NXT-20052 — INV - Inventory - Withdrawals - Withdrawal Details

## Description
As a Site Manager I want a Withdrawal Details page so I can view the details of existing withdrawal transactions.

*Requirements*

* Create a Withdrawal Details page which displays the specific information of a single withdrawal
* This should basically be the same screen as the Confirmation story ([NXT-20797|https://primeroedgenext.atlassian.net/browse/NXT-20797]) with some changes
* In or above the header, display action buttons:
** Adjust ([NXT-20051|https://primeroedgenext.atlassian.net/browse/NXT-20051])
** Cancel ([NXT-20053|https://primeroedgenext.atlassian.net/browse/NXT-20053])
** Withdrawal Ticket ([NXT-20053|https://primeroedgenext.atlassian.net/browse/NXT-20053])
** Price Details ([NXT-20054|https://primeroedgenext.atlassian.net/browse/NXT-20054])
* Display a label header section with the header level withdrawal info
** Site
** Type
** Date
** Reason
** Comments
* Display a list grid containing the list of withdraw items
** Item #
** Item Description
** Withdraw Quantity
** Comments

# NXT-39948 — Student Import – Auto Code Mapping - Languages

## Description
As a SchoolCafé Admin I want the system to auto-map some of the fields so that I can import files faster.

*Notes:*

* This comes from importing Salem
* They would like it if the system would automatically map certain obvious fields from the common text name to our abbreviations, such as Texas = TX

*Requirements*:

* When importing a Student Import, do the following:
* We need to create default mappings that are automatically available when the user wants to import files
* We can do this by adding them to the code mapping list automatically and making available for the user to see and edit
* 
* Language Field:
** Use our list of source languages mapped to codes
* 
* This should be for both manual and automatic imports

## Acceptance Criteria
* System provides the auto mapping without the user intervention
* User can modify the mapping or delete it
* User can continue to add / edit new or existing mappings
* System will automatically convert the import text based upon mappings

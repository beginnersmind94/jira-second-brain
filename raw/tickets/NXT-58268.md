# NXT-58268 — PE - USDA Branded Foods / CNDB - Enable Stock Item Tab while Creating ingredient

## Description
_(none)_

## Acceptance Criteria
Navigation: MP - Global Catalog - USDA Branded Foods

*Applicable for Single Item Import Only:*

User select data source as USDA Branded Foods, Search a desired item

Upon opening the item in Popup, the user should have the stock item tab enabled 

Have the stock item populated from other data sources as follows:

*Item Name / Description - Same as Ingredient Name*

Product Code - Search for Product from alternative data source (GDSN Connect tables), If no match found - Leave as Blank, 

GTIN - Same as Ingredient GTIN

Brand - Same as Ingredient Brand

Manufacturer - Brand Owner of USDA Branded Foods

Item Category - *Food Category* of USDA Branded Foods, also allow user to select one from the list of categories available for the corresponding region

Item Type - Default: Non-Commodity

Storage - Default - Cooler, also allow user to select one from the list of categories available for the corresponding region


*Pack Size Information*

 Purchase/Whole Unit* - Default Value = Case

Broken Units Per Whole Unit - Search for IU per PU value in GDSN Tables and PE or SC and populate if the last 10 characters of GTIN Matches, if not, prompt user to enter

Broken Unit Description - Search for Inventory Unit Type value in GDSN Tables and PE or SC and populate if the last 10 characters of GTIN Matches, if not, prompt user to enter
Broken Unit Weight - Search for Inventory Unit Weight value in GDSN Tables and PE or SC and populate if the last 10 characters of GTIN Matches, if not, leave them blank

Broken Unit Size - Default Value = 1 + Search for Inventory Unit Type value in GDSN Tables and PE or SC and populate if the last 10 characters of GTIN Matches, if not, leave both the field as blank
Whole Units Per Pallet - Default to zero

Orderable in Broken Units - Default Value = No



*Costing and Vendor Info*

 Valuation Group - Default: Non-Commodity

Fair Market Value = Prompt user to enter it

Commodity/ Net Off value = Leave as Blank 

Upon Importing the ingredient, stock item should be created and linked with Ingredient (Buying Guide Reference)



*Check Box Option in Stock Item Tab:*

If the user select the check box , then all the field should be disabled, user can proceed importing the ingredient only (as-is, current scenario)

If the select the check box again, then the fields get enabled and validations are applied as per field definitions provided. (Nice to have - retain the values what user has already entered

*Changes in Ingredient Tab:*
 - Deselect exclude Images by default - this means images are enabled to import if available



*Validation: Messages (On Top of Screen):*

Please enter a valid number for broken units per whole units.
Please enter a broken unit description

please enter item Category

Please enter storage Category
Please enter fair market value/Standard Price

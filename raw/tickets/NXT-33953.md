# NXT-33953 — Settings Page - Show Display Values For Default Column

## Description
Settings Page - Show Display Values For Default Column

+*See below for an example*+

!image-20220823-123501.png|width=1328,height=252!

In this example for the setting “Applications - Effective Date”, the default value displays as the number 2.  The end-user has no way of knowing that this refers to the “Received Date” dropdown option.

## Acceptance Criteria
* As an end-user, I would like to see the Display Values in the “Default” column so that I know what the default value refers to (Currently, the default value field shows ids and values that the user does not understand).

# NXT-66299 — CC - WS - PE Messages

## Description
As an Insights role user, I would like to see my PE messages so that I can view my important info on the go.

h2. Notes

* We have integrated SC Inbox, now we need to integrate PE messages
* This story is for PrimeroEdge (PE) districts
* Basically, we want to mimic the Messages functionality in PE

h2. User Flow

* User logs in and is redirected to home screen
* User clicks Workspace and is redirected there
* Screen hides Inbox and show Messages
* User clicks on Messages and is redirected there
* User is able to view a list of messages
* User is able to click on a message and see the details
* System sends PE read receipt
* User is able to delete a message???

h2. Requirements

* Integrate the PE messages API developed by Lalita’s team
* If user is from PE, do all these things:



_Home Screen_

* Hide the Inbox overview card
* Hide the Tasks overview card
* Add a new card for Messages
** Label: Messages
** Show count of new messages



_Workspace Screen_

* Hide the Inbox quick actions card
* Hide the Tasks quick actions card
* Add a new card for Messages
** Label: Messages
** Subtext: View your system messages
** Show count of new messages
** Use same Inbox icon
** On click:
*** Open the Messages screen
*** Do the same way as Inbox card does



_Messages Screen_

* Create a new screen
** Label: Messages
* To make it easy, copy code from Inbox screen, then:
** Remove Inbox and Sent chips
** Remove + Button
** Remove Star (if there)
** Keep all other functionality
* Connect it to PE APIs instead of SC Inbox
* On Message click:
** Open the Message Details screen



_Message Details Screen_

* Create a new screen
** Label: Message Details
* To make it easy, copy code from Inbox Details screen, then:
** Remove Reply, Forward, Archive buttons
** Keep all other functionality
* Connect it to PE APIs instead of SC Inbox Details
* On Delete click:
** Call the delete API and remove from the app
* On Read
** Send a read receipt to the PE system and mark it as read

## Acceptance Criteria
* PE users can:
** See count of unread messages on home page
** Not see Inbox or Tasks related stuff
** See their list of messages
** See the details of a message
** Delete a message
* PE is notified of read receipts

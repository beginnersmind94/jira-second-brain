# NXT-29567 — INV - Production Ordering - Consider Holiday Calendar

## Description

* Cross-functional; references Holiday Calendar
* Dependent on completion of [https://primeroedgenext.atlassian.net/browse/NXT-25541|https://primeroedgenext.atlassian.net/browse/NXT-25541|smart-link]

*Requirements*

* Prevent Production orders from being generated for holiday calendar dates for the site
* Prevent production orders from being edited to select a delivery date that is a holiday

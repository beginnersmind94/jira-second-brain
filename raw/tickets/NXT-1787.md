# NXT-1787 — INV - Receipts - Enhance Quick Receipt

## Description
As a Site Manager I want Quick Receipt to also allow me to search for an order to quickly create a receipt so I can multiple versions of “quick” receipts.

Context: Quick Receipt could be interpreted as a quick way to receive typical orders. Right now it is only functioning as a way to receive without creating an order, but it could serve both functions.

*Requirements*

* Add a step to the beginning of the Quick Receipt process to allow users to search for an Order #
** Radio options:
*** Search for Order
*** Receive without Order
**** *Issue TBD: ‘*Quick Receipt’ is used to distinguish amongst orders/receipts. May need to relabel

# NXT-50677 — INV/PR- Production Withdrawals- Consuming Prepared Items on Prep Site

## Description
As a Prep Site Manager, when I am pulling a production order for a particular data range, I would like to be able to pull all prepared items from the consolidated plan on the production side to ensure that I am ordering enough items for all the plans that I need to prepare for.

## Acceptance Criteria
* Pull Prepared Menu Items based on Prep Site to receiving site relationship based on prep site configuration or single menu line prep site config
** IF site is its own prep site and does not have any other receiving sites associated with it for particular plan/menu line, treat as normal menu item and pull from record complete (Current Functionality)
** IF site is its own prep site and HAS other receiving sites associated with it on a particular plan/menu line, pull information from the prep site distribution tab once all items are marked as shipped AND prep site record is complete (Combine all menu items ingredients from distribution tab and non prepared menu items ingredients from the prep site record)
*** There will be a true/ false flag for Distribution complete that will trigger whether ingredients are eligible to withdraw
*** Trigger withdrawal to be available in inventory only when prep site record is complete
**  IF prep site is only preparing for receiving sites on a particular plan/menu line, pull information from the prep site distribution tab once all items are marked as shipped ONLY since there will be no record for the prep site
*** There will be a true/ false flag for Distribution complete that will trigger whether ingredients are eligible to withdraw
*** Trigger withdrawal to be available in inventory when all items on prep site distribution tab are marked as shipped
* For receiving sites, *only* pull non prepared menu items and display items needed to withdraw without including prepared items

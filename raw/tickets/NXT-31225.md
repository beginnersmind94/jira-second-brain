# NXT-31225 — PR- Print Production Recipes (Multiple)

## Description
As a Production Manager, I would like an option to print multiple recipes at the same time based on my production plans.

The *multiple* option will give the user criteria to narrow their search.
Site code or Site Name (Can search by either)
Meal Type (Could be all)
*From date* and *To date*

Once user clicks apply, all recipes in the selected criteria should appear. User will be given an option to select specific recipes or all to print or export (PDF). The report should also have two formats a Recipe Report, and a Production recipe report. 

Columns to Display:

*Menu Item*

*Recipe Code*

*Category Description*

*#Servings (Size) (Display number of Servings that the recipe makes and what the serving size is)*

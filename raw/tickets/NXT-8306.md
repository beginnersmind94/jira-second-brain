# NXT-8306 — INV - Physical Inventory - Move affirmative buttons to be the right-most - UI

## Description
As an Inventory user I want all the affirmative/confirmation buttons (i.e. positive actions such as Save, Add, Complete, etc.) to be the right-most button available, so the Inventory module is consistent with the overall product.

This is a recent global design decision and we need to update our pages to be consistent with this decision. All affirmative buttons should be the right-most aligned button. Other buttons, such as Cancel or Reset, should be to the left of the affirmative button.

Pages requiring change:

* Search Physical Inventory
* Add Items (Physical Incomplete screen)

## Acceptance Criteria
Action buttons should be the button on the farthest right on each page, per global design decision.
Pages with the affirmative action button on the left should be changed.

# NXT-50532 — MP- Menu Planner- Allow edit menu/menu cycle to have multi select for Serving Groups or remove

## Description
As a menu planner, I would like to be able to edit menu or menu cycles to add or remove serving groups from the menu planner.

## Acceptance Criteria
* Allow user when they are editing menu/menu cycle to multi select serving groups or remove
* Provide an edit to the serving groups to enable selections of serving groups
** If serving groups were added to only particular menus (from standard), if user decides to edit and those serving groups are applied, add menu items to those serving groups

* Display serving groups available based on meal services provided

* Always provide the apply button, but once in edit mode, only allow it to be clickable if a change in serving groups happens since no other criteria can be changed
** Once selections are made, user must click apply to enable new selections or removals
*** Provide error message if all serving groups are removed “Atleast one Serving group must be selected”
*** Provide a prompt “Changes have been made to the serving groups, are you sure you want to apply changes?” Confirm/ Cancel
* If user removes serving group and applies then decides to come back and add the same serving group but has not performed the MAIN save, then bring back the serving group with the counts associates previously
* If user removes serving group that consists of a serving size while other serving groups are marked as not served, once main save is applied, remove menu item since no serving group is serving this menu item
* Once user confirms, refresh all analysis to reflect based on new serving groups and add all menu items already on the menu/s to the additional serving groups as served based on default serving size or based on serving size exception that is associated with that serving group
** If menu item is marked as not served before additional serving groups are added then user adds serving groups, apply the same logic as above and leave the not served serving group alone if it still as part of the serving group selections
* Default to whatever serving group shows in order (Similar to edit mode when user adds serving groups from standard menu)
** EX: User selects K-5 and 6-8, default to select k-5 once user clicks apply
*** User can still switch serving groups and change serving size from the menu item serving size drop down if needed
** Users if they do not need to serve that item for a particular serving group will need to change the serving size to NOT SERVED

# NXT-68148 — GC - USDA BF: API for reading Allergens from Ingredients information. 

## Description
GC - USDA BF: Logic for reading Allergens from Ingredients information.

## Acceptance Criteria
GC - USDA BF: Logic for reading Allergens from Ingredients information. 

Set the  allergen indicator as “Contains”

 

# Crustacean Shellfish: If the ingredients text includes the words “CRAB” or “LOBSTER” or “SHRIMP”, “SHELLFISH”
# Egg:  If the ingredients text includes the words “EGG” 
# Fish: If the ingredients text includes the words “FISH”
# Gluten:  If the ingredients text includes the words “GLUTEN” or “WHEAT”.
# Milk:  If the ingredients text includes the words “MILK” or “WHEY” or “CASEIN”
# Peanut: If the ingredients text includes the words “PEANUT” 
# Sesame: If the ingredients text includes the words “SESAME”
# Soy:  If the ingredients text includes the words “SOY” or “SOYBEAN” 
# Tree Nuts: If the ingredients text includes the words “TREE NUT” or “ALMOND”, “WALNUT”, or “PECAN”.
# Wheat:  If the ingredients text includes the words “WHEAT” or “FLOUR”.

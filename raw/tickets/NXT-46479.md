# NXT-46479 — FH - Menus - Make a Tray - Saving the tray

## Description
As Sally the Student I want to save my tray and be able to bring it up later so I can see what I ate.



*requirements*

* User can save a tray
** Tray must meet the meal components to be saved
*** Message if not: This meal does not fulfill the ‘Make a Tray’ meal component rules.
** User can set a Meal Name
** User can leave a comment.
** User can stay and make another tray after saving.
* User can view their saved meal trays (under My Favorites > My Trays)
** User can see high level info on the saved meals
*** Items involved
*** When it was saved
*** Calories, Carbs, Fats, Allergens, Additional Allergens
** User can view the full nutritional information for the saved meal.
*** User can see the stacked menu item images.
*** User can see allergens and custom allergens.
** User can edit the name and comment of the tray.
** User can click into the items that make up the meal, to see the info on that menu item.
* User can delete their saved trays
** Confirmation prompt.

## Acceptance Criteria
* User can save a tray if the Make-a-Tray rules are met 
** User can enter a name for the tray and a comment to go with it
** If they’re not met, user is informed that they have not met the nutritional rules.
** User has the option to make another tray
*** If they select this, empty the tray selection area so the user can make another tray.
* User can see their saved meal trays
** User can see high-level info on the saved meal 
*** Menu items, last saved, calories, carbs, fats, allergens, additional allergens
** User can see menu item images
** User can click into the menu items
** User can see the full nutritional information of the tray
*** Reuse the Full Nutritional Info section from the prior story
* User can delete a saved meal tray
** Confirmation prompt

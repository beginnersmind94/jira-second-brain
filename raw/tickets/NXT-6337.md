# NXT-6337 — Workspace - My Tasks - View All

## Description
*User Story*

As a PrimeroEdge user, I have a screen that shows any tasks I have, both manual and automatically generated

*Requirements*

* Add a *Tasks* screen where users can view their tasks.
** Page should be accessible via the Workspace - My Tasks section - View All button
** Styling should match the Tasks widget in Workspace
** Both My Tasks and Recommended Tasks can be viewed here. Each will be presented in a separate tab ({color:#6554c0}covered in a separate story{color})
* Sorting:
** Sort page by Due Date in descending order, followed by Status in descending order
** All columns should be sortable
* Users can create a task on this page.
** Pop up covered by NXT-6347
* -There is a gauge at the top to show progress based on timeframe selection- ({color:#ff5630}Deferred to a future story{color})
* User can view current, future, or past tasks
** If tasks are overdue, display their Due Date value in red. We may add an overdue status in the future.
* Tasks can have their status updated:
** New
** In Progress
** Done (Marked as complete)
* -Tasks can be deleted-
** -Task should be removed from the user’s view, not necessarily the database-  ({color:#ff5630}Deferred to a future story{color})
* Tasks can be edited
** A pop up appears with the same fields when creating it

## Acceptance Criteria
* Tasks screen is created
* Tasks appear correctly and can be created, edited, completed, and deleted
* All complete and incomplete tasks can be viewed
* New tasks can be added if necessary
* Workspace 'My Tasks' section is updated
* User can navigate between this page and the Workspace if needed

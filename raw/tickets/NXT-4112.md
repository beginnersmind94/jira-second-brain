# NXT-4112 — AcctMgt - Rosters - View Roster Details

## Description
*User Story*

As a district admin I want to see who is in a roster and any other details.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=rosters&id=yvgr6n&c=1|https://nqyc9v.axshare.com/start.html#p=rosters&id=yvgr6n&c=1]

*Requirements*

* User can click on a roster from the split view cards and have the information on that homeroom load on the right side.
* Information to display:
** Roster Name
** Roster Description (optional field)
** Roster Site
** Table showing a list of students
*** Columns:
**** Name
**** ID
**** Contact Info
***** Info should be pulled from the Primary Guardian
***** Name, Address, Phone #s, Email, etc.
**** Date of Birth
**** Allergens / Restrictions
***** -Show as chips, if more than 2-3 are tagged then show them as a +X (ie: +5) tag that expands.-
***** Update 8/10/20 - This will be a placeholder column, functionality will be handled later.
**** Chevron present on each line to expand the information shown.
***** Expanding shows the full contact information, along with the full allergens / restrictions if spilling over.
*** If no students are present, indicate that no students have been added and show the Add button.

## Acceptance Criteria
Rosters load off the Split View cards.
Name, description, list of students available for that homeroom.
Students can be expanded to reveal full information.
Multi-language support added.

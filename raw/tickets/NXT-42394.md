# NXT-42394 — IM - Recipe Reports - Add option & column for 'Purchase Quantity'

## Description
As a production manager I want an option for viewing Purchase Quantity on the recipe production reports so I can see the production order quantity in pack size units.

!image-20230818-013732.png|width=1062,height=561!

!image-20230822-145540.png|width=100%!

## Acceptance Criteria
* Add an additional toggle option for the Recipe Report for ‘Purchase Quantity’
* Add a new Purchase Quantity column next to ‘Quantity Needed’ 
** Only include this column if the ‘Purchase Quantity’ toggle is enabled
** Display this to the right of Quantity Needed on the report
** Display the +Production Order+ quantity, which does *not* consider yield factor percentage (as opposed to Quantity Needed)
** Display the purchase quantity rounded up to highest pack size units, regardless of Inventory data/config
** Consider decimal setting in display
* For items without Pack Size units, simply display the same serving as ‘Quantity Needed’ column but with the difference in yield factor percentage

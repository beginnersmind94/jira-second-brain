# NXT-3415 — Edit Inspections - Save

## Description
*User Story*
As an Inspections user, I have a way to save an In Progress Inspection and come back to finish it later.

*Requirements*
* There is a Save button available when editing an Inspection
* When clicked, the Save button saves any current progress on the Inspection and takes the user away from the screen
** User is returned to the View Inspections screen
** All questions are saved as responded to, including any media attachments, and can be viewed
** Status remains In Progress
* When user edits Inspection again, all answers should be maintained
* No validation is run when saving since it is not being submitted
** A form can be saved while blank

## Acceptance Criteria
* Inspections can be saved while in progress

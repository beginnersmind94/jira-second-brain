# NXT-18701 — IM- Adding or Subtracting Recipe Weight based on yield factor -Moisture,Fat & Waste

## Description
USDA is requiring that when calculating recipe weight, there needs to be incorporation of yield factor method. This means that a recipe weight needs to account for moisture, fat, or waste factor (bone, rind, core, etc).

These fields should be placed on the bottom of the recipe for user to enter in additional yield factor information and should effect calculated weight per serving as well.

Currently we are displaying # number of servings and recipe total weight as in the example below. 

Values are given in %



Changes  also apply to nutrient report.

## Acceptance Criteria
Recipe weight needs to account for moisture, fat, or waste factor (bone, rind, core, etc).

These fields should be placed on the bottom of the recipe for user to enter in additional yield factor information and should effect calculated weight per serving as well based on user inputted percentages. Example

Display note that this is based on percentages.
(Yield Factor entered is based on % of total recipe weight)
I.E user inputting 10 in one the fields means 10%. User inputting -10 means -10% of the total weight. 



Calculation for Fields

EX: Recipe weight is 100 Grams

Moisture: 

If user inputs -20, decrease total recipe weight by 20%= 80.00 Grams

If user inputs 20, increase total recipe weight by 20%= 120.00 Grams



Moisture, fat and waste yield factors should be display at the bottom of the recipe see below.

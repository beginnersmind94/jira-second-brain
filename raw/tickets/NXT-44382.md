# NXT-44382 — Imports - Employee - Add Statistics Section

## Description
As a SchoolCafé Admin I want to quickly see the import statistic for Employee import so that I can identify if I have any import issues.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor > Employee import record
* Click on View Log button
* View statistics on top

*Requirements*:

* We want to add in the statistics section like in Adult Import. 
* Copy that code and do everything that it does.
* Use the same UI

+For Summary (Statistics) use this endpoint:+
[https://apim-perseuseedge.azure-api.net/accountmanagement/v1.0/api/Import/Student/ImportHistorySummary?importHistoryId=115181|https://apim-perseuseedge.azure-api.net/accountmanagement/v1.0/api/Import/Student/ImportHistorySummary?importHistoryId=115181]
importHistoryId will be dynamic.

+For the History Log use this endpoint:+
[https://apim-perseuseedge.azure-api.net/accountmanagement/v1.0/api/Import/Student/HistoryDetail?importHistoryId=115181&pageNumber=1&pageSize=10|https://apim-perseuseedge.azure-api.net/accountmanagement/v1.0/api/Import/Student/HistoryDetail?importHistoryId=115181&pageNumber=1&pageSize=10]
importHistoryId will be dynamic.

Currently in Student Import, the UI needs to wait for the above 2 endpoints to return the response and then the UI is rendered.

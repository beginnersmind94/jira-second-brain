# NXT-34619 — Import Monitor - Disable View Log While Job Is Running

## Description
On the Import Monitor, the View Log button shall be disabled while the import is running.

!image-20220922-225020.png|width=1650,height=491!

## Acceptance Criteria
* The “View Log” button shall be disabled while the import is running
* When the import completes with success or error, the “View Log” button shall be enabled.

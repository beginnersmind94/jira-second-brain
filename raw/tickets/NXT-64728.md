# NXT-64728 — WS - Filters - Site Types - SC

## Description
As a Workspace user, I want to be able to select sites more efficiently, so that I can be faster in filtering my data.

*Requirements*

* This story is for the filters on the main page
* It applies only to SC Districts
* In the drop down list for the sites, do the following:
** Be able to select by Site Type
* Show only the site types that apply to that district

## Acceptance Criteria
* Users are able to select from site type in the drop down
* Drop down doesn’t show site types that are not applicable to the district

# NXT-42836 — DataExchange Import – Fixed Width – Field Mapping Validations - UI

## Description
As a SchoolCafé Admin I want to ensure validations are happening on the length field so that I don’t accidently import invalid files

*Requirements*:

* Validations:
* Drag and Drop:
** When the user drags from the source table to the mapped table, validate that they have a number greater than 0 in its length text box
** If not
*** -turn the box red-
*** add a red color border of 1px to the width text box of the selected field.
*** show the error message if it hasn’t been shown before
*** Disable the Next button
** If yes, do nothing
* 
* Remove from Mapped table:
** When the user unamaps a field, check if the length text box is red
** If yes, turn it back to black making it valid again
** If no, do nothing
* 
* Textbox typing:
** When the user enters a value in the length text box:
** Validate is has only numbers
** Validate the number is greater than 0
** If not
*** turn the box red
*** show the error message if it hasn’t been shown before
*** Disable the Next button
** If yes, do nothing
* 
* Error Message
** The first time you trigger a length text box invalidation, show the user a popup error message
*** Text: Length text box requires a number greater than 0
*** Location: top of screen same as in other places
*** Background color: Red
*** Duration: 3-5 seconds, whatever we normally do
*** Let them close the message manually also
** After the first time, do not show the error message anymore. No need to spam them
* 
* A red text box for the length text box means it is not valid
* When the length text box is red, disable the Next button and do not allow the user to proceed
* 
* Update Next button
** On click:
*** Validate each selected field’s length text box has only numbers and the number is greater than 0
*** If not
**** turn the box red
**** show the error message even if it has been shown before
**** Disable the Next button
*** If yes, allow them to proceed

## Acceptance Criteria
* Page forces the user to enter a number greater than zero for each selected field

# NXT-8739 — ExpressPoint - Modifications to Menu Service to use New Menu grid formats

## Description
The menu service in ExpressPoint needs to support the new menu grid formatting and menu button configurations.

*Requirements*

* Menu grids can be configured with 16 rows and 15 columns or as few as 1 row and 1 column.
* The grid fills the whole Menu Grid Area.
* Buttons can be different sizes in a grid.
* Each button can be configured.



2/8/21 - Grid shows/prices not set

## Acceptance Criteria
The new menu grid and button configurations are loaded from Perseus to ExpressPoint.
Verify the appearance of the buttons in ExpressPoint menu service to the appearance in Perseus
Verify the operation of the buttons in ExpressPoint menu service, verify correct Item selection, Price, Descriptions

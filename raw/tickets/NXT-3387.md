# NXT-3387 — Inspections - Create Inspections APIs - Schedule

## Description
*User Story:*

As an Inspections user, I have a Create Inspections slide out that allows me to properly start a new inspection.

*Requirements:*

* Users can schedule an Inspection to begin later
** Requires Scheduling permissions (Supervisor, Admin)
** Scheduling is done if the “Schedule for later” checkbox is enabled
** The following fields are present and required
*** Inspection Name
*** Template
**** Only Active Templates are present
*** Site
*** Inspection Date
*** Inspection Due by (Optional)
*** Inspector
*** Inform Site?
**** Checkboxes - Exact Date, 1 Week Window, Do Not Inform
*** Recurrence - If checked, reoccurs based on user’s settings

## Acceptance Criteria
API developed for Create Inspections slide out for properly scheduling an Inspection

# NXT-30131 — IM - Item Config - Menu Item Categories - Edit default categories

## Description
As a District Admin I want the ability to edit the system default Menu Item Categories so I can at least change the color and the display order.

**Note:* This has potential impacts downstream on MP and Prod; we need to be clear on how we are managing in the context of data sources

* Cross-module communication with MP/Prod team for any impacts

*Requirements*

* Provide a modified Edit ability for the system default Menu Item Categories
** these default categories are the ones in the CNDB Data Source
*** *Important:* these should be in *Cybersoft* data source, not the Child Nutrition Database data source
*** We also must ensure that the category itself is in the Cybersoft data source but the color, display order, etc. are unique per district
* Only allow Color and Display Order to be edited
* Remove ‘Uncategorized’ as a global/default Menu Item Category
** change to a different category for all the affected items
* Rename Meat & Meat Alternates to ‘Entrees’
* Rename Dairy to ‘Milk’
* +UI Improvements:+
** Remove horizontal scroll bar
*** Increase width if needed
** Update component styling

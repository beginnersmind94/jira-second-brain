# NXT-7152 — Data Exchange - Field Mapping - Conditions that allow user to proceed

## Description
*Summary*

Add necessary validations to allow users to proceed to the Review step of the Data Exchange wizard.

*Details*

* In Data Exchange → Field Mapping,  disable the Next button by default
* *Sort all mandatory fields to be  displayed at the top of primeroedge column*
* Enable mandatory marks on required fields
* Enable the button when: 
*# every required PrimeroEdge mapped field (for the selected Import Type) has been linked to a column in the file
* When the Next button is selected, commit the changes to the database

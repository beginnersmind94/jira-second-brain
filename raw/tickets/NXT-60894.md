# NXT-60894 — INV - Distribution - Shipping - Modify Shipments - Reset to Original

## Description
As a warehouse distribution manager I want a tool to make changes to the default shipments loaded on a distribution, so I can customize according to my specific shipping/delivery needs.

h3. Shipping tab - Modify Shipments - Reset to Original

!Shipping - Modify - Reset to Original Rev-20250602-145409.png|width=91.66666666666667%,alt="Shipping - Modify - Reset to Original Rev-20250602-145409.png"!

## Acceptance Criteria
* Open a slideout when the ‘Modify Shipments’ button is clicked
* Display a ‘Modify Shipments By’ heading with 3 options
** Reset to Original (default)
*** disabled when user is on default data setup
** Move Orders
** Combine Shipments
* When *Reset to Original* is selected, provide a simple confirmation message
** display a ‘Confirm’ heading
** below it display the message: “This will override any manual changes and reset all data to the original Distribution shipments.”
** display a ‘Reset Shipments’ button in the footer 
*** display a confirmation prompt on click: “Are you sure you want to reset all shipments to the configuration default?”
** button icon: [https://fonts.google.com/icons?selected=Material+Symbols+Outlined:restart_alt:FILL@1;wght@400;GRAD@0;opsz@24&icon.query=restart&icon.size=24&icon.color=%235f6368|https://fonts.google.com/icons?selected=Material+Symbols+Outlined:restart_alt:FILL@1;wght@400;GRAD@0;opsz@24&icon.query=restart&icon.size=24&icon.color=%235f6368|smart-link] 
* After confirmation, reset all the data on the Shipping tab to the originally built shipments
** reset stop order according to originally loaded stop order
** do *not* reset Driver, Ship Time or Comments

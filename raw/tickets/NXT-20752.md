# NXT-20752 — View Edge Scan Results Messages

## Description
As a Cashier, I want to see key messages from the module so that I can see that the module is working properly.

*Application*: POS

*Flow:*

* While using the POS transaction screen with Food Detection enabled:
* Messages are shown at key points in the transaction
* Results are shown when they are given

*Assumptions:*

* This page exists in the classic ExpressPoint
* You can copy all the code and try to copy as-is
* Talk to Saketh if you have any questions

*Requirements*:

* Copy everything as-is from classic
* There needs to be a control that shows certain information in the POS transaction screen with Food Detection is enabled:
** Don’t show the control when Food Detection is disabled
* Show messages to user when received from key points in the process
* Show the food item results when they are given
* More messages will be shown later
*  
* Fields:
** Food Detection Enabled
*** Show a green icon when turned on
*** Show a red icon when turned off
** Messages
*** RO text field
*** Blank and invisible if doesn’t have message to show

** Leverage existing security for this screen

# NXT-3018 — System - Roles - Allow user to Copy a Role

## Description
As a district or central office administrator, I would like to be able to duplicate a role with its permissions, so that I can more easily create a similar role that only slightly differs.

+Summary+:

Add a button next to each role that allows a user to instantly duplicate that role with its permissions.

+Details+:

* On the Roles screen, in the grid, add a button to the right of each role.
* When selected, display a menu with a Copy button. 
* When the user selects the Copy button, open a pop-up similar to the Add New Role button with the details of the previous role pre-filled.
* Allow the user to modify any data.
** Role Name must be unique, but all other values can be duplicated
** At least one organization level checkbox must be selected and a description must still be entered
* If the user selects Cancel, close the menu and return to the grid.
* If the user selects Save, create the new (cloned) role, close the menu, return the user to the grid, and display a success message.
* The same module permissions should be added to the new role as were on the previous role. (This section may be dependent on other stories to complete, but the API work could be done at least)

## Acceptance Criteria
Verify that a button to copy a role is added to the Roles page next to each role

Verify that Copy button opens a pop-up with the old roles pre-filled

Verify that any value can be modified

Verify that Role Name for the cloned role must be unique

Verify that when the user adds the cloned role, it is saved to the grid with the details chosen in the pop-up

Verify that role permissions of the clone and previous role are matching (API)

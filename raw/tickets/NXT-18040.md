# NXT-18040 — INV - Order Detail - Receipts tab - Provide all possible banner messages

## Description
As an ordering manager I want the Receipts tab banner to display specific messages so I can quickly view the different possibles statuses of my order’s receipts.

*Requirements*

* Provide messages for Summary tab banner
** if no items have been received: “No items have been received as of \[current date]”
** if some items are received but not all: “This order is partially received as of \[current date]”
** if all items are received: “All items have been received as of \[current date]”
** if order is closed and has receipts: “This order was received and closed as of \[date when order was closed]”
* Provide messages for Receipt tab banners

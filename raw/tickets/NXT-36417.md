# NXT-36417 — IM - Recipe Rework - Steps & Ingredients - Edit Step - Ingredient Search card

## Description
As a Nutritionist I want the Recipes page to have a dedicated Steps & Ingredients tab so I can make all the recipe preparation changes in one place.

[Figma - Ingredient Search|https://www.figma.com/proto/OAZ7qTRgdG5bFFGuhDmkoZ/Item-Management-6.0?page-id=3238%3A54123&node-id=3292%3A52739&viewport=411%2C-15%2C0.03&scaling=min-zoom&starting-point-node-id=3282%3A47737] 

*Requirements*

* Put the Ingredients search card into editable/add mode based on actions from the Steps card 
** Draw user’s attention to Ingredients card as needed
** Allow users to search for items/recipes to be added as ingredient
** Use ‘Search Items or Recipes’ as ghost text on search field
* Add ingredients to the Step card as they are entered from Ingredients search
** Serving Sizes should be entered for every ingredient before the step can be saved
* Show thumbnail image for each ingredient returned by search

!image-20230206-194222.png|width=91.66666666666666%!

## Acceptance Criteria
* Allow users to add ingredients to a Step using the side panel ingredient search card
** either from Add Ingredient action or Swap action
* Allow searching of items or recipes 
* Add items back to the Steps card as they are selected in the search

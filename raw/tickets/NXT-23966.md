# NXT-23966 — INV - Reports - Price Details - All transaction types

## Description
As a Site Manager or Food Service Director I want a report to pull the Price Details of every transaction types for a particular Site.

*Note:* This story is dependent on the individual transaction types 

* Order Receipt - [https://primeroedgenext.atlassian.net/browse/NXT-23962|https://primeroedgenext.atlassian.net/browse/NXT-23962|smart-link]
* Addition - [https://primeroedgenext.atlassian.net/browse/NXT-23960|https://primeroedgenext.atlassian.net/browse/NXT-23960|smart-link]
* Withdrawal - [https://primeroedgenext.atlassian.net/browse/NXT-23961|https://primeroedgenext.atlassian.net/browse/NXT-23961|smart-link]
* Reconciliation - [https://primeroedgenext.atlassian.net/browse/NXT-23963|https://primeroedgenext.atlassian.net/browse/NXT-23963|smart-link]
* Transfer Receive - [https://primeroedgenext.atlassian.net/browse/NXT-23964|https://primeroedgenext.atlassian.net/browse/NXT-23964|smart-link]
* Transfer Withdrawal - [https://primeroedgenext.atlassian.net/browse/NXT-23965|https://primeroedgenext.atlassian.net/browse/NXT-23965|smart-link]

*Requirements*

* Generate the Price Details report that is displayed when all transaction types are selected on the [report page|https://primeroedgenext.atlassian.net/browse/NXT-20418] 
* List oldest transactions first, regardless of transaction type

Shows the price and batch details of a transaction 
*Report header shows* : Site name, transaction type , transaction#, Actual Transaction date.
*Item details table shows:* 

* Price detail information is grouped by item# 
* columns are Batch#, Received date, WU Qty, WU Price, Line total. Item total for each item
* Transaction total at the end as a last row.

*We have another option to generate the report for only items with multiple batches involved* : if selected only pull the transactions and item involving more than one batch. 

Sample report is attached from classic.

## Acceptance Criteria
Price details report is available for all transaction types and for a site and date range.

# NXT-10968 — Dashboard (Insights) - Eligibility - Changes Widget

## Description
As a district admin I want to see a log of recent Eligibility changes for students, with a quick way to drill down into that information.

*requirements*

* User has a widget that shows recent Eligibility changes.
* User can see the following information for the changes:
** First Name
** Last Name
** ID #
** Site
** Grade
** Former Eligibility
** New Eligibility
** Reason
** Change Date
** Source
*** This would be something like ‘Application #123’ or ‘Direct Cert File #444’ or ‘Manual Change’
* User can search the results to find a particular student.
* User has a date range they can specify to fuel the results.
** Default: Last 7 days.

## Acceptance Criteria
User has a widget on the Insights dash that shows recent Eligibility changes.
Changelog shows information on the student per the requirements.
User can search the results to find a particular student.
User can select a date range for the results. Default is the last 7 days.

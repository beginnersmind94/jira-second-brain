# NXT-56812 — MP- Site Groups- Display site on current Site group and Future based on end date

## Description
As a District Admin,

When I am making decisions to switch site groups, I would like for me to be able to see which site group I am currently on and what site group it is being changed to with some sort of indicator.

## Acceptance Criteria
* Display end date on Current Site group as the day before the new effective date based on new site group
* Display site on new site group from the day of addition until effective date in Italics
** EX of Current Site Group

||*Site Name*||*Effective Date*||*End Date*||
|Harper High|10/16/2023| |
|Aly High|10/16/2023|2/05/2025|

EX of New Site Group

||*Site Name*||*Effective Date*||*End*||
|_Aly High_|_2/6/2025_| |
| | | |

* Provide a Footer “Sites in _Italics font_ have been configured for this site group but the effective date has not yet taken effect”
* If a site has been switched to another site group or removed from a site group, provide an end date as to when that takes into effective
** IF added to another site group, the end date of the old site group should be the day before the effective date of the new group 
** This should apply if being added from the site group tab or from the site configuration tab
** Clear all menu assignments that are available to be deleted (NOT RECORD COMPLETE OR CONSOLIDATED PLAN COMPLETE) or after effective date from previous site group config (on the configuration date and not when the effective date comes into action)
* Allow all sites to be removed from a particular site group from the site group tab and revert the site group to Not set if there are no sites associated with site group
** All current logic should still remain and this site group can now be made inactive but not deleted since there were sites associated before with data
* Any future effective dates that have not been triggered and get changed should effectively change the previous end date to the day before the new effective date on the new site group
** Any assignments that have been done in the interim that will no longer be part of the new site group based on the change in effective date have to be cleared as well

# NXT-16907 — INV - Orders - Review Order Settings - Add New Rule - Purchase Amount (Step 4)

## Description
As an inventory operations manager I want a Purchase Amount option on the Add New Order Review Rule process so I can potentially configure order amount limits to determine when orders are pending review.

*Requirements*

* This is the fourth step of the Add New Rule process of the Review Order settings
** this step is optional
* Display a small grid to show the minimum or maximum purchase amount limit(s) that have been applied
** columns:
*** Parameter
**** this will display ‘Minimum’ or ‘Maximum’
*** Value
**** this will display the $ value that was entered
**** provide a numeric text field to type a dollar value
**** ensure that minimum value is less than maximum value
**** ensure that maximum value is greater than minimum value
*** Action
**** Edit icon, which will enable the Value field for editing
***** while in Edit state, edit icon will be replaced with the green checkmark and red X icons for save and cancel
***** Edit icon for the other parameter (if row exists) will be hidden until edit is saved
**** Delete icon, which will delete the parameter
* Display a Back to Rules List button in the bottom left corner, to cancel and return to the list page
** Prompt user: “Discard rule configuration and exit setup?” Yes/Cancel
* Display a Previous button in the bottom left corner, to return to the previous step
* Display a Next button in the bottom right corner to advance to the next step
* Display a Complete button in the bottom right corner, to skip the remaining steps and jump ahead to the confirmation step

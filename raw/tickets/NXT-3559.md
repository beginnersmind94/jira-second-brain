# NXT-3559 — Edit Inspections - Multiple Choice Question Type

## Description
*User Story*
As an Inspections user, I can see Multiple Choice Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Multiple Choice Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Selections
** Whether question is required
** Notes
** Media addition (functionality in a different story)
** Note: Point values are *not* displayed here
* User can select a choice
** Only one choice can be selected at any time
* Choice is properly saved
** If question is required, Inspection cannot be submitted if no selection is made
* Choice can be changed if Inspection has not been submitted
* If question has a score associated with it, the score is properly updated according to the selection being made.

## Acceptance Criteria
* Multiple choice question appears in Inspections
* Multiple choice and notes can be selected, saved, and edited
* Score updates properly if present

# NXT-65295 — ACC - Labor Hours Feature - Search and Results (displaying data in the grid)

## Description
Story to contain requirements for how search filters behave on the new Labor hours page.
Both entry methods use the same filters of Multi Site select for precise and single site select for standard (based on user permissions) and Date selector (standard control)

Filters grid below after hitting apply.
On page load, load the first site user has access + today’s date and populate the grid.

Sort main grid by Site description > Date

If setting is changed to change entry method we can completely refresh the page. (requiring filters to be re-entered)

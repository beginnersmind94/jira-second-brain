# NXT-40307 — eOrders Configuration – Edit

## Description
As a SchoolCafé user, I would like to be able to make changes to an eOrder Configuration so that I can make changes without having to create a new one.

*Notes:*

* This overlay is just like the + New except it Updates instead of Saves
* Try to copy that one as much as you can

*Flow:*

* Open SchoolCafé and browse to Settings > Configuration > eOrders
* View list of eOrders
* Click Edit on an existing one
* Make any desired changes to the configuration and save
* Return to the updated list of eOrders

*Requirements*:

*eOrders Dashboard page:*

* Add a button to the rows in the action column if not already there
** Title: Edit
** On click, Open the Edit eOrder overlay
* 
* Use all the same screens, fields and functionality as the + New

*Edit eOrders Overlay:*

* If you can, use the + New and just toggle the name and Save based upon if is add new or update
* If not, then create a new overlay
** Title = Edit eOrder
** For all the fields, load and select saved values
* 
* Save Button
** Update existing fields instead of saving new
** Write to the history log using the Edit function
*** See History Log story for instructions
* 
* Notifications:
** Send a message to the inbox of all users that have Edit rights in Roles & Permissions
** Text: [logged in username] has edited an eOrder configuration for [Vendor Name] To view, click [link to the eOrder Configuration]
* 
* Security
** Don’t show in Roles and permissions CRUD tables
** This overlay’s permissions = Edit permission for the eOrders page, therefore restrict security to those roles with eOrders Edit permissions plus Cybersoft admins

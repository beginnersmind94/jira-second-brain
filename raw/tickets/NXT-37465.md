# NXT-37465 — Student Export - Code Mapping - Spike

## Description
As a SchoolCafé Admin I want to be able to perform code mapping on certain fields so that I can configure my exports to match my destination system.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Exports
* Go through the Pre-export and get to the Field Mapping screen
* Pick one of the fields that has field mapping and click on the code mapping button
* Perform the code mapping as necessary and then finish the export

*Notes:*

* We already do code mapping on the Student Import file and want to duplicate that functionality on the Student Export

*Requirements*:

* When in Student Export on the Field Mapping Screen, do the following:
* On rows with the specified fields below, add a button for code mapping
** Label: Code Mapping
** On click: Open the Code Mapping Drawer
* 
* When exporting:
* Just like on Imports, if they map values, then when doing the export, transpose those values
* If they didn’t map any field values, do nothing to it
* 
* Code Mapping Drawer:
* When click Code Mapping button, open the drawer like on Student Import
* Do just like on imports, except switch the internal and external columns (Internal first)
* 
* Fields that need Code Mapping that are already in the list:
** Grade
*** Use the same values in the Student Import
** Eligibility
*** Get list of available values from system
** Reason
*** Get list of available values from system
* Fields that need Code Mapping that may be in the list:
** Gender
** Ethnicity
** Race
** Language
** For all these, use the same values in the Student Import
* Fields that need Code Mapping that have not been done before:
** SiteCode (School)
** For this, create a new code mapping drawer just like all the others

## Acceptance Criteria
* Code mapping button exists for the fields designated above
* When clicked, the code mapping drawer opens
* Code mapping drawer works like on Student Imports
* If mapped codes, values are transposed on export file
* Values for fields are what we have in the system

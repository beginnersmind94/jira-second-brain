# NXT-3441 — Employee Details - Delete Training Record

## Description
As an admin, I want the option to delete (trashcan icon) a training record from an employee profile. 

* Show delete trashcan in line with the training record
* delete option is instant so we need a popup confirmation 
* Once deleted, the hours associated (if checked 'complete') are removed from the employee stats to maintain accurate CEU counts 
* If cancelling, the line item remains - nothing changes

## Acceptance Criteria
Training record item can be deleted from employee details/profile 
once deleted, the hours associated with it will be removed from the compliance count

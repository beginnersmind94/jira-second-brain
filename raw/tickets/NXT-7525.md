# NXT-7525 — Eligibility - Scanning Tool - Scanning Log

## Description
Add an area to the scanning tool that provides the user with a list of scanned applications for review before sending to SchoolCafe.

* User can see scanned images.
* User can remove images.
* User has feedback for any invalid scans.
** Bad barcode, bad alignment, etc.
* User can approve the ‘batch’ to send over to SC.

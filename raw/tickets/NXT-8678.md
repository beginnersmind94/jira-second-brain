# NXT-8678 — [Not Required] - Needed ExpressPoint Website Create Terminal

## Description
When a user logins to ExpressPoint web a terminal is created. For the site if it does not currently exist.

*Requirements*

* A new terminal/device is created when a user opens ExpressPoint for web use.
* The Device Name will be the Username+SiteCode
* The device will be created the first login by the user for the site
* If a device exists for a Username+SiteCode a new device will not be created.
* The first launch of ExpressPoint requires the user to select the District.
* If a user is District level or has multiple sites for Site level access they are prompted to select a site.
* If a user is Site level and has access to only one site the site is automatically selected.

## Acceptance Criteria
When EP is launched the user is prompted to enter the district
The user is prompted for a User Name and Password
When the user logged in if they have access to multiple sites they are prompted to select the site.
A site is automatically selected if they only have access to one site.
When a site is selected a new device entry is created, if it doesn't exist.
The device is Username+Site Code

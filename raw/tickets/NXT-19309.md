# NXT-19309 — INV - Transfers - Transfer Items - Remove Broken Units

## Description
As an Inventory user I want Broken Units removed from the Create Transfer functionality so I can accurately view items and quantities when transferring.

*Requirements*

+On the first step/screen+

* On the Transfer Items page, remove references to Broken Units
** Remove the Broken Unit quantity input column
** Remove Broken Units from the ‘Quantity on Hand’ column
* Group related items together
** child items should be displayed immediately below a parent item
** display from highest level unit type to lowest level unit type
* Make UI changes
** Change ‘Item Code’ to ‘Item #’
** Change Item Name to ‘Item Description’
** Change ‘Whole Unit’ to ‘Transfer Quantity’

+On the Transfer Confirmation screen+

* On the Transfer Items page, remove references to Broken Units
** Remove the Broken Unit quantity input column
** Remove Broken Units from the ‘Quantity on Hand’ column
* Make UI changes
** Change ‘Item Code’ to ‘Item #’
** Change Item Name to ‘Item Description’
** Change ‘Whole Unit’ to ‘Transfer Quantity’

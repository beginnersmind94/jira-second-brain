# NXT-15944 — IM - Edit Product info 

## Description
As a User I would like to have an ability to Edit my Product Info So that I can make any changes needed to an Item

## Acceptance Criteria
Given the User is on the Items Page 

And a search is applied 

And an item is selected from the Item result 

When Edit option on the Product info is tapped 

Then Provide the user an Inline edit capability to make any changes to the below information 

* Item Category 
* Storage Category 
* Purchase/Whole Units 
* Broken Units per Whole Unit 
* Broken unit Description 
* Broken Unit Weight in Grams 
* Broken Unit Size
* Fair Market Value/Standard Price  
* Valuation group 
** USDA Foods
* Buy American 
** Upload option for Exemption Letter
* Orderable in Broken Units 
** Toggle 
* Internal Vendor 
** Yes/No options to select 

And ability to “Save” any changes made 

And “Cancel” to discard the changes 

And exit the Edit mode

# NXT-52077 — System - Settings / POS Settings - Dynamic Entrees

## Description
As Dana the Director I want to enable Dynamic Entrees for specific sites.

*requirements*

* Add a new system setting
** Name: POS - Dynamic Entrees
** Code: DYNENTREE
** Description: This setting controls the Dynamic Entrées feature that integrates entrees from Production Plans for use in Accountability and POS. If enabled, Dynamic Entrées can be added to Menu Grids and a Category is available for placement. A new POS setting (Site Configuration > POS Configuration) must then be turned on to enable this for specific/all sites. At the POS, the Dynamic Entrées will be replaced with entrees from the Production Plan for that site and date. 
* Add a POS Site Setting
** Name: Dynamic Entrees

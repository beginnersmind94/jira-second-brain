# NXT-196 — Report - Special Accounts

## Description
As a Central Office Admin I want a report that shows me transaction history information for the Special Accounts so I can keep an eye on their usage.

*This is a port of the PrimeroEdge report.*

We can likely bolt this on as an option for the Transaction History page however, instead of having it be its own page.

## Acceptance Criteria
User can run a report to see usage on the Special Accounts.

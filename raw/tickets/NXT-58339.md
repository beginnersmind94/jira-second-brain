# NXT-58339 — POS - Setting to clear old local pouch data

## Description
As Matt the Product Manager I want to start using the POUCHCLEAN setting at POS.

!image-20250123-022720.png|width=1195,height=191,alt="image-20250123-022720.png"!

*requirements*

* Update the setting:
** Name: POS Local Data Storage Cleanup
** Description: This setting controls how many days a local POS machine will hold onto data in PouchDB before deleting it. Data that has not been sent to the server will not be deleted.
** Category: POS
** Setting Values: 3 Days, 7 Days, 14 Days, 21 Days, 28 Days, 42 Days, 56 Days
*** Default Value: 21 Days
** Configuration Type: External
* Update POS to use the setting
** Data older than X Days should be removed from local storage.
** Only remove data once we’ve confirmed it’s been sent to the server.
** Move clean-up to trigger later in the service cycle
*** Potentially part of closing session - need to warn the user to leave the machine open
**** Text: Please leave the POS machine open while we perform a small clean-up.
**** Bug Matt for better text

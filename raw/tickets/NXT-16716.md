# NXT-16716 — Platform - Licensing - Summary Page for Customers

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to view list of products we are enrolled with along with recommended products and related products. 

*Requirements*:

* Site admin logs into System/Licensing Summary page for list of products displayed in “My Products” section
** +My products section table Columns+
*** Product Name
*** License Type
*** Quantity
*** Active
*** Usage
*** Expiry/Renewal 
** Types of licenses
*** Site
*** District
** My products are displayed and sorted in alphabetical ascending order
** Products table must enable search filter by clicking on filter icon
** Name, Type Columns should be sortable and filters enabled
** Each product has option to “ON/OFF” toggle with just indicators (text enabled when hover)
** Each product must have usage indicator with +/- indicator to add/reduce licenses
* Add-on Products section is displayed under “My Products” section
** ADD button next to add-on products should be displayed which brings onboarding config screen (implemented in R4)
* Recommended products section is displayed under “ Add-on Products”
** Recommended product must be displayed in tile format with flip with image hover to show name/description of the product (actual implementation is in R4)
* Some of the products (modules) are expandable with sub features with checkboxes to enable/disable upon expansion of the product row
* “Summary” tab should load by default when user clicks on License Management
* Dashboard, Report and Audit tabs are added but should be greyed out
* Expanding District type license product row should display active 5 sites and enables “More sites” to display in a pop-up

## Acceptance Criteria
* List of products procured by the customers shall be displayed in sorted by ascending order
* Expiry/Renewal date, Name and Type columns are sortable and filters enabled
* Expanding District type license should display active 5 sites and enables “More sites” to display in a pop-up
* Active toggle is rendered for each product displayed in “My Products” section
* ADD/REDUCE license buttons are displayed for each product
* ADD-ON & Recommended products sections are rendered below “My Products” section
* Summary tab is loaded by default when customer launches License Management page.
* Active license count is displayed in Usage column for each product.
* For district type license lets' display “-” for available and assigned columns in the products table.

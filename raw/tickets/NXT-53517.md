# NXT-53517 — VND - Letter Templates - Addition of Security PIN field

## Description
As Dana the Director I want to send out letters that have the Security PIN as a table field.

*Requirements*

* Add a new Table Field to all letter templates:
** {{Security_PIN}}
* Should pull the Security PIN from the account for letter purposes.

Notes:

* We did not implement this in PrimeroEdge.
* VendNovation have provided a Postman collection to help with internal testing.

* New table field added: Security_PIN
* Fills in the security pin off the account when generating letters

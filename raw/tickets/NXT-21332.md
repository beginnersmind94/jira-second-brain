# NXT-21332 — System - Site Configuration - Sites - Combined/Parent + Reimbursement Code

## Description
As Vera the Director I have sites that are separate in the application, but are combined as far as the state is concerned. I need a way to represent this combination via the application.

*requirements*

* User needs a way to indicate if a site is a Satellite or Combined campus. This is optional.
* User needs a list of available sites to select from if one of the above options is selected.
* User needs to be shown if the site is already configured as a parent or combined site, including the site code + site name of the site(s) that are linked.
* User should not be able to daisy chain these links - a parent site cannot become a satellite to another campus. (Site A is the parent for Site B. Site A cannot be selected as a Satellite for another site.)
** Error messaging if the user attempts it: This site cannot be made a satellite or combined campus as it is currently linked to another site.
* User needs an option to input a Reimbursement Claim Export Code that differs from the Site Code.
** Text field, optional.

## Acceptance Criteria
* User can indicate if a site is a Satellite or Combined campus.
* User can select from a list of available sites to select from if one of the above options is selected.
* User needs to be shown if the site is already configured as a parent or combined site, including the site code + site name of the site(s) that are linked.
* User should not be able to daisy chain these links - a parent site cannot become a satellite to another campus. (Site A is the parent for Site B. Site A cannot be selected as a Satellite for another site.)
* User can configure a reimbursement claim export code.
** Same as Site Code selected by default.
* Site sync should happen to microDBs.

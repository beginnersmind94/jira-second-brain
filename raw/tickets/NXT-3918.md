# NXT-3918 — Users Phase 2 - Edit User Profile  - Password section

## Description
As a PrimeroEdge user, I would like the ability to change my password.

+Summary:+

When editing the User Profile, there will be a Security section that will contain a link that, when clicked, will take the user to the common “Change Password” screen.

+Mock-up+: 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=7044%3A899&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=7044%3A899&scaling=scale-down-width|smart-link]

## Acceptance Criteria
The "Edit Profile" screen shall contain a "Security" section with a link to change/reset the user's password.

The "Change Password" link shall take the user to the common "Change Password" UI, when clicked.

The "Change Password" UI shall ensure that the user's new password adheres to the requirements of the password policy.

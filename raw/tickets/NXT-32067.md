# NXT-32067 — INV - Production Ordering - UI updates for Pack Size Info

## Description
As a Site Manager I want the Production Ordering page to automatically calculate my production usage and convert into orderable items so I can ensure the most accurate quantity is being ordered.

API changes on [https://primeroedgenext.atlassian.net/browse/NXT-33372|https://primeroedgenext.atlassian.net/browse/NXT-33372|smart-link] 

*Requirements*

* Update the Production Ordering screen to reflect the new pack size configuration
** Remove the Production Item and Order Item approach/columns
* Show orderable item units on individual rows but indicate as linked together
* Display Quantity on Hand
** Enable QOH Alternate View popover
* Display Usage 
** round up to the orderable unit quantity (2 Cases, 62 Cups)
** only display Usage on the highest unit for an item, if two units are orderable
* Allow checkbox selection per Item # (not unit level)
* -Replace Unit Type with Unit Description-
** Unit Description is displayed after the order quantity field; we just need to make sure it’s completely visible and obvious to the user
*** Put Unit Description in its own column
* Display Item # only once for the highest orderable unit
** Apply checkbox once per entire Item (regardless of units)
** When an item is added to the cart, move the entire hierarchy to the Cart Items tab
*** when orders are generated, only add the specific units which have some quantity greater than zero
* Make sure the images are showing correctly
** this will be changed in the API story

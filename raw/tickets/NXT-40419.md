# NXT-40419 — IM - Menu Items - Variety Menu Item - Nutrition Sync

## Description
As a Menu Planner I want the system to automatically sync changes to my Variety Menu Item nutrition info based on changes to the associated Menu Items.

!VMI Nutrients - Prompt-20230607-140452.png|width=66.66666666666666%!

## Acceptance Criteria
* Build a service to sync system calculated/generated updates to Variety Menu Items based on changes to the associated menu items
* If users are using ‘System Calculated’ option, automatically update their values when changes are made to linked menu items
* Consider independent flags for *Nutrient Details* and *Allergens*
* Recalculate the average Nutrient values
* Recalculate Allergen values

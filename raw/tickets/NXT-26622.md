# NXT-26622 — Platform - Security - Add custom roles to users

## Description
As a SchoolCafe admin, I would like to add new custom role specific to customize the modules and its permissions to various resources.

+*Requirements:*+

* Custom roles are available under users page for admins to assign specific custom roles to the users
* Default VIEW permissions are enabled for all resources under each selected module
*  Cancel and Create buttons must prompt for confirmation prompts for any actions
* Create audit is also added to Audit history

## Acceptance Criteria
* User shall be able to create custom role by using “CREATE NEW ROLE” option
* User shall be able to select modules and name of the new role.
* User shall be able to role with default view permissions enabled for each module
* User shall be able to use CANCEL and CREATE buttons with confirmation prompts
* User shall be able to see created user, date and action (created new role) in Audit history tab
* User shall be able to EDIT/COPY after adding new role
* User shall be able to enable/disable modules from the role

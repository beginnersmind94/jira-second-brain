# NXT-23960 — INV - Reports - Price Details - Inventory Additions

## Description
As a Site Manager or Food Service Director I want a report to pull the Price Details of every Inventory Addition to see the price details for multiple transactions.

*Note:* The API for retrieving batch details can be used regardless of transaction type. However, the APIs for retrieving each transaction type need to be created. 

* This story is only for retrieving Inventory *Addition* transactions

*Requirements*

* Display the Price Details Report ([NXT-20418|https://primeroedgenext.atlassian.net/browse/NXT-20418]) when only ‘Inventory Additions’ is selected for Transaction Type
* Load the Addition transactions for the selected date range
** exclude cancelled transactions
** display oldest transactions first

*Report header shows* : Site name, Transaction Type, Transaction#, Actual Transaction date.
*Item details table shows:* 

* Price detail information is grouped by Item #
* columns are Batch#, Received Date, Quantity, Price, Line total. Item total for each item
* Transaction Total at the end as a last row

*We have another option to generate the report for only items with multiple batches involved* : if selected only pull the transactions and item involving more than one batch. 

* This can be a separate story if it makes sense

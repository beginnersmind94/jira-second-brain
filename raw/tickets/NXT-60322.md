## Acceptance Criteria
* Provide a specialized version of the Add Item tool specifically for adding substitutions to a single item
* Add a Substitute button to the Assignment Details slideout
** only show when an item has insufficient QOH 
* When Substitute is clicked, open a modified version of the Add Item tool
** Change header to ‘Add Substitute Item to Distribution'
** Insert placeholder record with instructional text ‘Search above to add substitution’
** Select and disable the ‘Is a Substitution’ checkbox
** Display the 'Substitute For' sub-row as a label
* Allow users to add up to 2 substitution items
** disable the Add button after 2 are added
* Enable the ‘Add X Substitutions’ button once at least one sub is added
** dynamically populate button with quantity (1 or 2 in this case)
* Return the user to the Assignment Details slideout after adding substitutions

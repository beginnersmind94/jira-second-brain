# NXT-69097 — Fin - Static Data Points

## Description
These data points are always created when Financials user is onboarded, regardless of district configuration.

Full list here: [^Master Data for Data Points.xlsx]

## Acceptance Criteria
*Assets*

* Due from Federal Funds
* Cash in Bank

*Liabilities*

* Prepaid Account
* Refunds
* Returned Checks
* Sales Tax Owed

*Revenue*

* Additional Income
* Bank Fees
* Deposit Variances
* A La Carte - Student - Breakfast
* A La Carte - Student - Lunch
* A La Carte - Student - Snack
* A La Carte - Adult - Breakfast
* A La Carte - Adult - Lunch
* A La Carte - Adult - Snack
* Adult Meals - Breakfast
* Adult Meals - Lunch
* Adult Meals - Snack
* *Program Revenue Accounts*
* Created for each program within program config
* Use the program display name without the parenthetical code
** Examples: School Breakfast Program, National School Lunch Program, After School Snack Program
* *Reimbursement Revenue Accounts*
** Created for each program within program config
** Name format: "Reimbursement - [Program Name]"
** Base type: Revenue
** Examples: Reimbursement - School Breakfast Program, Reimbursement - National School Lunch Program

*Expenditures*

* Account Adjustments
* Bonus
* Salaries
* Wages
* Benefits
* Overtime

*Equity:*

* Fund Balance

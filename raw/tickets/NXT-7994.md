# NXT-7994 — INV - Orders - Receipt Detail - Receive Report - Header

## Description
As a Site Manager I want a Receive Report on the Receipt Details screen so I can generate a report version of the individual receipt.

Items: [https://primeroedgenext.atlassian.net/browse/NXT-23620|https://primeroedgenext.atlassian.net/browse/NXT-23620|smart-link]
Charges & Discounts: [https://primeroedgenext.atlassian.net/browse/NXT-23621|https://primeroedgenext.atlassian.net/browse/NXT-23621|smart-link] 

*Requirements*

* Add a report button to the Receipt Details screen
** also add it next to the View Receipt button on the Receipt tab (saves a click into the receipt details)
* Generate the report on button click
** this does not need to use the report overlay
* Display the Report title ‘Receipt Report'
* Display a header section in the report to display receipt header info
** display all values from the receipt header
*** organize the values similarly (or identically) to the page

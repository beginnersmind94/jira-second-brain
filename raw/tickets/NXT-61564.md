# NXT-61564 — AI Onboarding - Digitize - BO Button

## Description
As a SchoolCafé user, I would like to browse to the new AI Digitize in Workspace so that I can scan a document.

*Requirements:*

* On the production screen, do the following:
* Add a Navigation sub menu item under Production module:
** Label: Digitize AI
** On click:
*** Redirect the user to the new digitize ai screen workspace via SSO
*** Open in a new browser window if you can
*  
* Security:
** Check if the AI Enabled setting is on for the district
*** If no, then hide the menu item
** Check if the user is in a role that has the digitize ai special permission
*** If no, disable the menu item, but show it and change the tooltip to be: Digitize AI - special permission required
*** else, enable the menu item

## Acceptance Criteria
* Menu item is added to navigation
* Menu item is hidden if AI is not enabled for the district
* Menu item is disabled if use is not in special permission
* User is able to click on the menu item, if enabled, and be redirected to the Digitize AI screen directly in the new workspace

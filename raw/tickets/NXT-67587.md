# NXT-67587 — IM - Bulk Item - View Mode for Items in a Session

## Description
As a District Manager, when I have my district updating items within the bulk item tool, I would like there not to be some over lap where users can go into the item details and make changes at the same. Items within a session should only be able to be updated in the session

!image-20251023-043636.png|width=1032,alt="image-20251023-043636.png"!

!image-20251023-043713.png|width=1032,alt="image-20251023-043713.png"!

## Acceptance Criteria
* Once session has been started, the “Bulk Update” button will now have a item counter on it to provide a visual to users that an ongoing update session is available
** All items will be published for the session, no partial publishing
* Display Icon on the list page to show item is currently in a session
* Item is available to be accessed but strictly in view mode (List or Split View)
** This applies to any sub navigations to the item details page (Ingredient list on recipe, F&R etc)
** Use the view permission logic and temporarily disable ability to edit
* Display a banner message “Item is currently being updated in Bulk Update Session (Session)”
* Once items have been published in a session, they are now available to be fully accessed from the item list page

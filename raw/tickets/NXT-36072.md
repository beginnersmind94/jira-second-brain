# NXT-36072 — IM - Recipe Details - Menu Item & POS Info tab

## Description
As a Menu Planner I want all Menu Item related information on a single tab for my recipes so I can see all item and menu item information in a single place.

Cross reference with [https://primeroedgenext.atlassian.net/browse/NXT-39594|https://primeroedgenext.atlassian.net/browse/NXT-39594|smart-link] 

*Requirements*

* Populate the Menu Item & POS Info tab according to the new design
** Insert Max Carryover Days into the allocated column space for the overall Leftover Category field
*** don’t show the field if a non-applicable Leftover Category is chosed
** move Favorite to the header of the card
** align Show in Summary and Prep Site Item to fit into a single column space
* POS Info will be a subset of the Menu Item tab card
* a ‘Show on POS’ toggle will control the presence of the POS attributes and Pricing table
** if enabled, the POS attributes will be inserted into the 2nd row
* Once saved, adjust the card for view mode
** reference mockup arrangement
** only show Favorite icon in view mode if enabled

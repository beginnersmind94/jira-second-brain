# NXT-54050 — Exports - Eligibility - Processed Date

## Description
As an export user, I would like to be able to export the Processed Date field for Eligibility/Balance exports so that I can see the day and time that it was processed

*Requirements*:

* These requirements are for the Balance export and Student Exports only
** Add to both
* Add Processed Date to the payload
* Add Processed Date to the front end list of columns

## Acceptance Criteria
* Processed Date is exportable with the correct value for both Student and Balance exports
* Existing eligibility exports do not blow up or break

# NXT-32116 — Elig. - Direct Cert - Updates to make Must Match work properly

## Description
As a product manager I want the Must Match option to work properly so that it can be utilized by customers.

*mockup*

n/a

*requirements*

* If a Pass is disabled, uncheck the ‘Must Match for Automatic Matching’ box if it was checked.
** Example: DOB is set to ‘Must Match’ = true. DOB is disabled. DOB’s ‘Must Match’ is moved to false when the user saves.
* Fix how the ‘Must Match’ column works
** Example: automatic match is set to 85 points. file is processed and 1 particular match has 90 points.
*** IF ‘Must Match’ is enabled for Last Name, points for a last name match must happen before the automatic match can happen.
**** For this scenario, this would become a Potential match instead. User needs to check it themselves.
*** If the points were <85, it would still be a potential match.
** ‘Must Match’ acts as a blocker for automatic matching

# NXT-66380 — Import Button

## Description
*As a* user

*I want* to import my chart of accounts

*So that* the journal entries reflect my actual accounts

## Acceptance Criteria
* The *Import* button is available on the _Import GL Accounts_ tab.
* When the user clicks *Import*, a file explorer opens so they can select and upload their Chart of Accounts file.
* A *progress bar* is displayed while the file is uploading, showing the status in real time.
* After the upload completes, the system automatically validates the file against the configured account code structure.
* The user sees a *validation results summary* showing:
** Total accounts processed
** Number of accounts/account categories that passed validation
** Number of accounts/account categories that failed validation with specific error details
* *If all accounts pass validation*: The accounts are automatically imported and the user receives a success confirmation.
* *If any accounts fail validation*:
** User sees a message like “Expected XXX-XXX-XXX, got XXX-XXXXX for 77 accounts” customized as per their actual account code structure
** Give users the option to either fix it and upload again and continue anyway
*** If the user decides to continue anyway, display a prominent warning mentioning that this may affect report accuracy and that they should fix those accounts ASAP
** User receives a detailed error report showing exactly which accounts failed
*** They are able to download their file with an additional column indicating which accounts were and were not validated
** If the user uploads the file again
* Users can *download the error report*

# NXT-32174 — DataQuery Framework - Make Data Models Available For Districts

## Description
When a data model is added to an environment, it shall be available for all districts to use.

## Acceptance Criteria
* When a data model is added, it shall be available to use by all districts via “Manage Views” and “Reports”
* A method shall be available that will allow promoting a given data model to a specified environment
** _In other words, as new data models are created over time, there needs to be a way to promote them to QA, UAT, staging, and eventually production._

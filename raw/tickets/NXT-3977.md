# NXT-3977 — System - Notification Templates - Allow user to update template status

## Description
As a district administrator, I would like to be able to easily manage all of the letter templates (for letters that will be going out to my parents and students) in one place, so that I can easily make changes as needed.

*Summary:* Allow the user to toggle a letter template as active or inactive (+without+ having to select the ‘Save’ or ‘Save As’ buttons). Letter editor does not have to be done by this point.

* Make the status toggle functional for each letter
* Save status appropriately
* Active letter should then display at the top of the letters list
* If all templates are inactive, the Standard template will be treated as active
* If the Standard template is active, it cannot be deactivated (because there has to be at least one active template at all times)

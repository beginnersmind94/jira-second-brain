# NXT-63693 — Enable/Disable AI Insights based on system settings Enable Insights AI-INSIGHTSAI

## Description
1)The AI Insights icon on the Insights page is hidden when the Enable Insights AI-(INSIGHTSAI) system setting is ‘No' and made visible when the setting value is 'Yes’.
When hiding the icon, ensure that there is no gap or extra space between the Configure KPIs and Configure Benchmark icons.

!image-20250630-034553.png|width=523,height=85,alt="image-20250630-034553.png"!

2)When the system setting is ‘No' then don’t call the Insight API on the Workspace page and hide the control, and call the API and show the controls when the system setting is 'On'.


!image-20250630-035221.png|width=1616,height=144,alt="image-20250630-035221.png"!

# NXT-3785 — Training Events - Add/Upload documents 

## Description
Ability to upload documents to a training event when adding new and editing 

* Classic version allowed extensions: .txt, .csv, .doc, .docx, .pdf, .jpg, .png, .bmp,.msg,.xls,.xlsx
* Add document name on upload
* preview of document (either popup or small window/thumbnail)
* Edit document name
* remove document
* time stamp, user, document name of upload – in history section
* Be consistent with System module, use the upload tool and allowed extensions.

## Acceptance Criteria
user with permission can add/upload documents to training events 
ability to remove upload and edit document name

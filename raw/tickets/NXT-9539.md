# NXT-9539 — Data Exchange - New Export Configuration screen

## Description
*Summary*

As a PrimeroEdge user,  from Data Exchange screen, I should be able to create new export configuration from export screen so that I can create new configuration and export required data from the system for the given module

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=364360%3A1087&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=364360%3A1087&scaling=min-zoom|smart-link] 

*Requirements*

* User should be able to open new Export configuration page by clicking “New Export” button under   configuration of Data Exchange page
* User should be able to add job details by selecting:
** Export Name
** Module
** Data Category
** Export Type
* User should be able to select the File structure under job details section
** Delimiter
** Qualifier
** Include header checkbox
** Module
* User should be able to add export columns from the available Primeroedge fields in a drop down
* User should be able to add ALL columns from PrimeroEdge fields drop-down
* ““Export” button are displayed when at least 1 column is added to File Structure Column section
* Each column added should enable user to EDIT/Update column name
* Each column should have actions enabled to delete
* “Export” options are displayed when all required data is selected
* Clicking on “Export” should save and export data in CSV format

## Acceptance Criteria
* Verify New Export page is loaded when user clicks on " New Export" button
* Verify Job details can be saved with appropriate name and selections for a given module
* Verify File structure section is displayed with delimiter, qualifier and header option
* Verify add columns drop down is available
* Verify user can add ALL columns in PrimeroEdge fields
* Verify External column editable by user which is same as PrimeroEdge by default
* Verify each column has delete action enabled
* Verify Export button is enabled and saves the export configuration upon clicking
* Verify data file is exported in CSV format when user clicks on "Export"

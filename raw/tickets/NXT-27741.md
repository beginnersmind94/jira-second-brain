# NXT-27741 — Item Details Import - Support for Back Office - OLD STORY

## Description
As Item Manager Admin I need to be able to onboard new Items using Global Catalog Item information  using an excel file in order to speed the process of bringing customers working with SchoolCafe’s Backoffice.

This story will cover the Platform team’s portion of the import of getting the file data into a staging table.  The Back Office team will be responsible for taking the data from the staging table, applying its business rules (if any), and moving the data to its final destination.

## Acceptance Criteria
As an Item Manager Admin, I would like the ability to:

* Import a file with a list of items and its details
* View feedback of the import process to see errors, processing statistics, and other line item related information ({color:#ff5630}Back Office team to provide acceptance criteria for this{color})

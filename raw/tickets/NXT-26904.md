# NXT-26904 — INV - Reports - Inventory Valuation - Period Report - Hide 'Market Value' column for Valuation Group report type

## Description
As a Food Service Director I want the 'Market Value' column to only be displayed when I select 'Valuation Group w/ Market Value' as the report type, so I can generate reports with and without the column.

*Requirements*

* Ensure the Market Value column only appears when the 'Valuation Group w/ Market Value' option is selected for Report Type
** Valuation Group - no market value column on the whole report
** Valuation Group w/ Market Value - market value column on sections 2 and 3
* Update the report to display differently depending on the two Valuation Group formats
** remove sub-headings for non Mkt Value format
*** on Summary by Valuation Group section (section 2)
*** on Valuation Details section (section 3, site breakdown)

# NXT-3189 — Menu Grids - Make site type  and days allowed multi selects   - [Require Shared UI Component)

## Description
*User Story*

As a user it is more intuitive to have multi-select options for Site Type and Days Allowed. The values that can be selected will be determined from the data.

*Requirements*

When the data is loaded the values available for the multi-select options will be determined. Example: If days available has days M, T, Th, F only those would be the days that could be chosen from the multi-select.

*Current Status:*

* We will wait for a common component for multi-select

## Acceptance Criteria
Site type is a multi-select filter option
Days available is a multi-select filter option
Only site types present in the selected data have a multi-select option available
Only days available present in the selected data have a multi-select option available

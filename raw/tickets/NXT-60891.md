# NXT-60891 — INV - Distribution - Shipping tab

## Description
As a warehouse distribution manager I want a step for Shipping within the Distribution, so I can organize deliveries into trucks/routes with stop orders and access Delivery and Shipping reports.

## Acceptance Criteria
* Build the Shipping tab of the Distribution tool
** Update the header with updated Stage chip and Status
* Display a Distribution Shipments grid
** provide a Modify Shipments header button (separate stories for Modify Shipment functionalities)
*** [https://primeroedgenext.atlassian.net/browse/NXT-61369|https://primeroedgenext.atlassian.net/browse/NXT-61369|smart-link] 
* Populate the list of system generated shipments at the time of Start Shipping
** one shipment for each unique combination of Shipping and Delivery group
** set default Stop Order according to Delivery Group config
** calculate the estimated Ship Time by finding the earliest delivery time of the schedule
* Build the main row of the grid as a header for the shipments
** Shipping Group
** Delivery Group
*** Route Name (# of stops)
*** if the shipment is modified, show # of stops in italics and show and info icon
**** info icon tooltip: ‘Shipment modified from original'
** Route Name
*** pre-populated with ‘Shipping Group + Delivery Group’
** Driver
** Ship Time (est.)
** Delivery Comments
** Ready for Shipment (checkbox) 
** Action
* Provide an edit icon in the action column to edit individual shipments
** Route Name
** Driver - text with predictive entry from vendor contacts (Contact tab)
** Ship Time - preload with calculated time
*** earliest delivery time from site shipping schedule
** Delivery Comments - use popup icon
* Provide a ‘Modify Stop Order’ icon in the action column
** [https://primeroedgenext.atlassian.net/browse/NXT-61348|https://primeroedgenext.atlassian.net/browse/NXT-61348|smart-link] 
* Display the orders in an expandable sub-grid
** show Stop # as the primary column (always sort by this column)
** display (partial) next to a site Order # if that order is split across multiple shipments
** Delivery Items - number of items from the order on the shipment
** Delivery Value - value of only the delivered portion of the order
* Update the sticky footer
** Show messaging relating to orders and shipments 
** provide the Delivery Tickets button ( [https://primeroedgenext.atlassian.net/browse/NXT-60892|https://primeroedgenext.atlassian.net/browse/NXT-60892|smart-link] )
** display the Complete Shipping button
*** not enabled until all shipments are marked as Ready to Ship

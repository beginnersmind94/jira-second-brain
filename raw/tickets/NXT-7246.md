# NXT-7246 — INV - View Vendor - Do not display the Contact card if no vendor contacts exist

## Description
If no vendor contacts are added for a vendor, do not display a ‘Contacts’ card/section on the view vendor info page. If the user wishes to add a new contact, they can do so by accessing the Vendor Address section and add a contact through that page.

If the user adds a contact for a vendor, display a card in the vendor page.

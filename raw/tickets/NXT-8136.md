# NXT-8136 — INV - Start New Physical Inventory - add Interim option to Period dropdown

## Description
As an inventory user I want the ability to do an interim physical inventory so I can perform inventory counts without connecting a period.

* Provide ‘Interim’ as an option within the Period dropdown.
** Interim will always show in the Period dropdown and should be the first option 
** Physical Inventory is started and performed as usual
** Interim option will be added to API that populates Period dropdown (no additional UI work needed)
* On change of site selection, get new list of periods for that site



* When saving the inventory, the user is required to enter some information, captured in [https://primeroedgenext.atlassian.net/browse/NXT-7833|https://primeroedgenext.atlassian.net/browse/NXT-7833|smart-link]

## Acceptance Criteria
Interim will be an option in the 'Period' dropdown

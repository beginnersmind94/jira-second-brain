# NXT-29289 — INV - Physical Inventory - Start New Inventory - Filter Period dropdown

## Description
As an Inventory user I want the Period dropdown to be filtered on the Start New Physical Inventory page so I cannot accidentally create inventories for prior/closed periods.

*Requirements*

* Filter the Period dropdown to disallow completed periods
** Remove all periods prior to the most recently completed inventory
** Remove current period if already completed

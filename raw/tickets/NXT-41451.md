# NXT-41451 — IM - Items - Units History

## Description
As an IM user I want history for all important Unit changes so I can see all updates to servings and packs.

## Acceptance Criteria
* Provide Units history for all Unit card changes
** Units added/removed, Cost updated, change in Preferred Measure, changes in Description, Weight/Volume, etc.
** Tab-specific actions: MI Exceptions, Menu Servings, Inventory GTINs, change in Default Measure
** Inventory ready: Publish toggle, Base Unit change, Decimal changes, serving-pack linkage
* Show a singular history across all tabs
* Reference the appropriate tab name for any tab-specific updates (MI Exceptions, Inventory GTINs, etc.)
* Reference any important corresponding attributes required to understand the change (ex: showing the Unit for which the price was entered)
** ex: showing the serving size description if weight changes

# NXT-50042 — MP- Menu- Menu Item Help button showing pending/inactive ingredients

## Description
As a Menu planner, I would like to be able to see on the menu item help button any ingredients that are part of my menu item that are pending deactivation or inactive.

## Acceptance Criteria
* Show pending deactivation icon with tool tip on any ingredient that is part of the menu item
* Show inactive icon with tool tip on any ingredient that is part of the menu item
* Ingredients should only be showing pending deactivation or inactive IF the ingredient portion of the statuses is selected
** This should also apply to the “ingredient” version of the recipe (subrecipe)

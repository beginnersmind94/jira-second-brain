# NXT-4055 — AcctMgt - Preview - Merge Patron

## Description
*User Story*

As a district administrator I want a way to merge two patrons together.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1]

*Requirements*

* In the Preview (vertical …) menu/dropdown, add an option for Merge.
** Small window spawns, asking the user who they want to merge with.
** Window has a lookup feature (the search by ID or Name bar should work)
** User selects who they’re merging with, and then sees basic information on both the current patron and the selected patron for comparison.
*** Photo, Name, ID #s, Site, Grade, Homeroom, Current Eligibility + Eligibility Reason (Free - Income for example), Primary Guardian Name + Address, Balance.
*** Both patrons must be the same type. Both students, both adults, etc.
** User can select who will remain active afterwards.
** User has a confirm / cancel option.
** Upon confirming, the patron that was *not* selected to remain active should be deactivated appropriately.
*** Status changed to Merged.
*** History added about the merger.
**** Who triggered it, when, who was merged together, link to the other account, Status changes.
*** Eligibility/enrollment ended.
** Upon confirming, the patron that was selected to remain active should have changed:
*** History added about the merger.
**** Who triggered it, when, who was merged together, link to the other account, balance changes.
*** Balance updated to be the sum of both accounts.
** User receives a toaster/notification that the merge was successful.

## Acceptance Criteria
Users can merge patrons.
History/Logging enabled for merges.

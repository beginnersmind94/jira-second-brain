# NXT-31175 — Acc. - End of Day - Action - Refresh Data

## Description
As Martha the Site Director I want a way to update the data on demand instead of waiting for the background job.

*mockup*

[https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|smart-link] 

*requirements*

* Add a ‘Refresh’ button that will trigger the background job immediately for the date + site combo.
* Tool-tip for the refresh button should show the last time the background job was run for that date + site.
** Last Updated: DD/MM/YY HH:MM:SS AM/PM



Success messaging:

* Data for this site and date will refresh shortly.

## Acceptance Criteria
Refresh button added.

Tooltip for the button shows the date + time the last refresh was done.

* This information should be relevant to the site + date that are selected by the filters.
* ie: when was the last time the data for Site 503, on 05/01/22, was updated for the End of Day page?

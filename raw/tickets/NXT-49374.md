# NXT-49374 — OpenID - District Authentication - Notify Enabled Method

## Description
As a SchoolCafe admin, I want to be able to tell a user to update their authentication so that they can login using it

*Notes*:

* This story is for how to handle the notification process for when SchoolCafe and OpenID are enabled

*Flow (Happy Path):*

* Admin browses to System > Security page
* Admin sets up OpenID method
* System notifies users of change and provides instructions on how to configure
* User follows instructions, updating their OpenID method in their Profile page



*Requirements*:

* These requirements apply to the System > Security page > Authentication tab:
* 
* When the district enables a method by using the Enable button, do the following:
** After confirmation:
** Send an email to the user when the district has added a new method as described in the New Authentication Added by District Email below



*New Authentication Added by District Email* 

[first name],

A new authentication method for your existing SchoolCafe account has been added for [district name]. 

The login page is [hyperlink to the login page].

Your SchoolCafe username is this email address.

The district has added the following authentication method.

[if SchoolCafe authentication, show this:]

SchoolCafe Authentication:

Website simple authentication using email and password. You will receive another email with instructions on how to set up your password. [if it is the preferred method, insert this text] This is the district’s preferred method.

[if OpenID SSO, show this:]

Single Sign On:

Login using your district username and password with your district’s authentication provider. No additional setup by you is required for this method. [if it is the preferred method, insert this text] This is the district’s preferred method.

## Acceptance Criteria
* Users are notified when a new authentication method is enabled
* Emails that go out have updated text
* Emails are curated based upon settings (shows SSO section if has OpenID, else it doesn’t)

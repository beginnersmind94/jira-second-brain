# NXT-3340 — Edit Inspections - Show All Questions Toggle

## Description
*User Story*

As an Inspections user, I can toggle my view between all questions or view by category.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=inspections_with_explanation_to_each_resaponse&id=dguk73&g=1|https://nqyc9v.axshare.com/start.html#p=inspections_with_explanation_to_each_resaponse&id=dguk73&g=1]

*Requirements:*

* Users can toggle whether all questions are displayed on the Edit Inspections screen
** If Show All Questions is enabled, all questions for the Inspection are displayed
** If Show All Questions is disabled, the Inspection is paginated by category
* Selection is remembered between sessions
* Any responses to questions are remembered when toggling between the two
* Question responses are saved when moving between two categories

## Acceptance Criteria
* Show All Questions toggle functions without loss of data

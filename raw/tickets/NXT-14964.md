# NXT-14964 — Accountability - Transactions - Exceptions - Display summary + Expand/Collapse option

## Description
As Sophie the Site Manager I want to see how many exceptions are present in the date range so I can allot enough time to research them.

*mockup*

[https://www.figma.com/proto/m3QGmo7Fjys7um36XDhwAY/Accountability-5.0?node-id=193%3A8761&scaling=scale-down-width&page-id=174%3A5157&starting-point-node-id=174%3A5158|https://www.figma.com/proto/m3QGmo7Fjys7um36XDhwAY/Accountability-5.0?node-id=193%3A8761&scaling=scale-down-width&page-id=174%3A5157&starting-point-node-id=174%3A5158|smart-link] 

*requirements*

* Show the count of how many exceptions need to be processed after the user performs a search on each respective exceptions tab.
* Add an option to expand or collapse all the offerings in the Exceptions grids.

## Acceptance Criteria
User can see how many exceptions are waiting to be processed for the site(s) and date(s) selected.

Counts are split between the different exception types.

User can expand (or collapse) everything listed in the results.

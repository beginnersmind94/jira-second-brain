# NXT-22117 — INV - Create/Edit Receipt - Move the Date on Product grid to a slide-out (UI)

## Description
As a Site Manager I want the Date on Product / Lot Number information to be provided in a slide-out so I can have a less cluttered receive screen.

This is for the Create/Edit Receipt screen. We are still determining whether Date on Product info should be in a sub-grid for view-only mode.

*Requirements*

* Remove the expandable row from the item grid
** Note: this screen will probably need to have a hierarchy grid soon, so we must remove this expandable row
* Add an icon to the Action column for expanding the Date on Product slide-out
** this icon should be easily visible/accessible (we should not use 3-dot menu for this action; maybe Comments is less important)
* Open a slide-out for the Date on Product info
** List each field vertically in the slideout
*** Date on Product - date picker
*** Date Type - dropdown 
**** (fix labels…)
**** Expiration Date
**** Best if Used By Date
**** Packed Date
*** Lot Number - alphanumeric
*** Received Quantity - numeric
** Display a plus icon to display another date entry
*** *UX* Note: with current design we add rows; how should we handle multiple date types within a slide-out?
** Display X icon to remove a row
** Provide a Close button
* Enforce existing validations regarding lot quantity equalling receipt quantity, etc.

# NXT-60902 — INV - Distribution - Quantities Report

## Description
As a distribution manager I want a Quantity Report so I can see the difference between certain distribution quantities, like what was ordered, what was assigned, and what was picked or the difference in value between them.

h3. Distribution (Shipping in Progress) - Quantities Report button

!Shipping with Quantity Report-20250529-053911.png|width=91.66666666666667%,alt="Shipping with Quantity Report-20250529-053911.png"!

h3. Report - Custom w/ 2 selections

* Discrepancy/Difference checkbox becomes enabled/clickable once 2 columns are selected
* in this example, Sent vs. Received (and Received Quantity) is disabled because no shipments have been made ready (tooltip may not be final)

!Quantity Report Custom 2 selections-20250528-210433.png|width=91.66666666666666%,alt="Quantity Report Custom 2 selections-20250528-210433.png"!

h3. Report - Custom w/ 3 selections

* only 3 checkboxes can be checked
* disable any other checkboxes until one is deselected
* automatically un-check Discrepancy/Difference when a Quantity checkbox is deselected

!Quantity Report Custom 3 selections-20250528-210433.png|width=91.66666666666666%,alt="Quantity Report Custom 3 selections-20250528-210433.png"!

h3. Report - Picking Variance selected

* all checkbox selections disabled and selected on behalf of user
** Assigned, Picked, Difference
* ‘Only show items with discrepancies’ default to Yes

!Quantity Report Variance selected-20250528-210433.png|width=91.67%,alt="Quantity Report Variance selected-20250528-210433.png"!

h3. Report - Sent vs. Received selected

* all checkbox selections disabled and selected on behalf of user
** Picked/Shipped, Received, Difference
* ‘Only show items with discrepancies’ default to Yes

## Acceptance Criteria
* Add a ‘Quantities Report’ button the header of the Distribution details page
** only display after Assignment status or later
* Create a Quantities Report Options slide-out
** open slide-out on click of report button
* Provide ‘Report Format’ radio group section
** Custom (default)
** Picking Variance
** Sent vs. Received
* Provide a ‘Quantity Columns’ checkbox section
** Ordered Quantity 
** Assigned Quantity
** Picked/Shipped Quantity 
*** label as ‘Picked’ until distribution is in Shipping stage
** Received Quantity
* Only enable certain checkboxes after the distribution is in an applicable distribution stage
** Ordered Quantity, Assigned Quantity - Assignment stage 
** Picked Quantity - Picking stage
*** update label to Picked/Shipped Quantity after Shipping stage
** Received Quantity - Shipping stage, or as soon as sites can create receipts
* Provide a ‘Difference’ checkbox option when two quantity columns are checked
** this would display the difference (+/-) between the two selected column values, if applicable
** display hyphen for items which have no difference in value
* Allow up to 3 column selections for Custom option
** including ‘Difference’ column
* Disable and auto-select checkboxes for system report options
** Picking Variance - Assigned, Picked, Discrepancy
*** default switch to ON
** Sent vs. Received - Picked, Received, Discrepancy
*** default switch to ON
* Generate report according to all possible selected combinations
** Example 1: Order Qty, Picked/Shipped Qty, Received Qty
** Example 2: Order Qty, Received Qty, Difference
* Make report available after Assignment is begun
** report is available after distribution is Closed

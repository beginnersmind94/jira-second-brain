# NXT-3029 — Item Management - Recipe - Edit Recipe General Info and HACCP

## Description
*User Story*

As a Item Management User, I want to be able to edit the Recipe general Information

*Mockup*

!image-20200929-022130.png|width=100%!



*Requirements*

* User will be able to edit all the same fields as on create a new recipe
* Recipe Name
** User can change the name, but if they do and move they click off or try to save, it needs to check for a duplicate per datasource
** If a duplicate, display the same message that the ‘Recipe already exists. Please enter a unique name’
* Datasource CANNOT be changed after the recipe is created
* All other information can be edited, including the HACCP process.
* ‘Save’ button needs to save the rearranged recipe steps to the database as well

## Acceptance Criteria
All Item fields can be viewed when selected from the list/grid view

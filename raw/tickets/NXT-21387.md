# NXT-21387 — INV - Vendor Detail - Address tab - Contacts card

## Description
As an Inventory Manager I want a Contacts card so I can create view and manage specific contact persons for the vendor. 

* Address card ([NXT-20060|https://primeroedgenext.atlassian.net/browse/NXT-20060])
** this contains the attributes defined under the vendor address section
** the map section can be done later
* Vendor Contacts cards (this story)
** this contains the attributes defined as Contacts are added for a vendor
** add a new card for each new contact 
** *Remove the limitation on maximum 3 contacts

*Requirements*

* Display a Contacts card below the Address card on the Address tab
* Display the labels in the following order:
** Contact Name
** Contact Position
** Preferred Contact Method
** Phone Number
** Email Address
** Address
*** this is a combined field from the various address fields
*** this will *not* be displayed if the ‘Unique Address' checkbox is unchecked
* On click of Edit icon, enable in-line editing
** Turn all label fields into editable rows
** Contact Name - text field -  *mandatory*
** Contact Position - text field 
** Preferred Contact Method - dropdown
** Phone Number - *mandatory*
*** this should be an actual phone number field, which automatically formats the input to a phone number format like 281-453-8500
** Email Address
** Unique Address - checkbox
*** this will determine whether or not to display the address fields
*** unchecked by default
*** if checked, show the address field; if unchecked, hide the fields
** For Address, split the field into multiple fields which comprise the address
*** Address Line 1 - text field - *mandatory*
*** Address Line 2 - text field 
*** City - text field - *mandatory*
*** State - dropdown - *mandatory*
*** Zip Code - numeric text field (5 digits) - *mandatory*
** Show a Delete icon
*** Prompt: “Are you sure you want to remove this contact person?” Yes/No
** Replace the Edit icon with Save & Cancel buttons
* On click of Save or Cancel
** Turn all fields back into labels
** Consolidate the different Address related fields back into a single address (if applicable)
** Replace the Save & Cancel buttons with the Edit icon

Each of these cards will contain an Edit icon in the top right corner that will allow the user to enter the edit view to make changes, add new contacts, etc.

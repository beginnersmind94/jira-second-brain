# NXT-163 — Accountability - Reports - Site Revenue

## Description
As a district admin I want to see a snapshot of how each site, and the district, has performed as far as sales revenue is concerned.

PrimeroEdge version:

!image-20201204-021136.png|width=1084,height=1333!

Detailed version of the report just breaks it down by date for the selected date range:

!image-20201204-021335.png|width=1076,height=1182!



*Requirements*

* User can select:
** Site(s), date(s)
* User is shown revenue data for the selected criteria.
** debit vs cash sales totals. 
** meal count totals
** sale $$ totals
* Revenue data is broken out by meal type.
* User is also shown cash summary information -
** calculated vs actual, over/short $$ amount

## Acceptance Criteria
User can select a date range.
User can select a site, all sites, or multiple sites.
User can opt for a summary of information, or a daily breakdown of the information.
User is shown revenue data as outlined in the requirements.

# NXT-18578 — IM- CNDB designated units of measure for all items, recipes

## Description
USDA is requiring for all CNDB items in SchoolCafe have an indicator that will let the user know that the serving unit of measure is from the USDA or CN database. For example, every year, the new child nutrition database list comes out with new items, serving measures etc, we can indicate that the serving measure is coming based on that EX: CN24

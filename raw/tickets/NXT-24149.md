# NXT-24149 — IM-None-Indicated Allergen -Item Only-

## Description
As a nutritionist I need to know if an item has non-indicated allergen condition in order to consider the use of an item in a recipe or menu item.



!image-20211224-161553.png|width=1217,height=741!

## Acceptance Criteria
Done when:

This applies to adding, editing  Food-Items Allergens including custom food allergen section (in List and split view mode)

* *Follow current rules for adding and editing allergens.*
* Field “None -Indicated” is visible in the Food-Item Allergen section
* User check or uncheck the “None Indicated” allergen box
* User checks “None Indicated” box and when save button is clicked the allergen information is stored and the check box is retained and visible to the user
* User un-checks “None Indicated” box and when save button is clicked the allergen information is stored and the check mark is remove and visible to the user

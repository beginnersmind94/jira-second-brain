# NXT-49469 — MVR - Dashboard - Edit

## Description
As a SchoolCafé user, I would like to be able to edit my managed view reports so that I can make a change to distribution.

*Notes:*

* Do not hard code assumptions that it is only for Users page or only for System module
* This is part of the Managed Views Framework and will work like a Custom Report
* We are using the Users MVR as the first example, but should work for any managed view

*Requirements:*

* On the Edit button of the grid in Reports > Managed View Reports, do the following:
* Action column updates:
** Edit
*** If button not there yet, add it
*** On click, open the Edit Managed View Report drawer



*Edit Managed View Report drawer*

* Create a new drawer
** Name = Edit Managed View Report
** Size = Full width
** Use the sliding action when opening
** Copy the Edit Custom Reports
* 
* Tabs:
* Selection tab
** Name 
*** Same requirements as the Add New MVR
** Description
*** Same requirements as the Add New MVR
** Owner
*** Same requirements as the Add New MVR
** Module
*** Read-only
** Data Source
*** Read-only
** Contains PII
*** Same requirements as the Add New MVR
** Dynamic Query
*** Don’t show
** Global Templated fields
*** Don’t show
** 
* Config tab
** The Config tab functionality is completely different or MVR than a CR
** So don’t show the graphical query builder from CR
** Instead, show the managed view criteria some way
** It doesn’t need to be fancy, maybe just show the JSON file or something
** 
* Sharing
** Same requirements as the Add New MVR
** 
* Distribution
** Same requirements as the Add New MVR
** 
* Scheduling
** Same requirements as the Add New MVR
** 
* Review
** Same requirements as the Add New MVR
* 
* When you save it
** Update instead of saving as new
* 
* Ensure you save update to the History table
* 
* Security:
** User must have edit rights to Manage Views Report dashboard to use this drawer
** In R&P, don’t show this drawer

## Acceptance Criteria
* Users are able to make changes to the editable fields and they are saved to the DB
* Only users with edit rights can update the settings

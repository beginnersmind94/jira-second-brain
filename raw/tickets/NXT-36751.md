# NXT-36751 — Homepage - Show website Version, 'About' link in footer

## Description
As a SchoolCafé user, I want to be able to see more information about SchoolCafé, including the version of our website, so that I can ensure I am up to date with the product and have all the updates.

*Flow:*

* User browses to login page
* User views the current version of the website
* User can also click to view more about SchoolCafé 

*Requirements*:

* Create a Release Version to maintain in the database (sys_applicationversion needs a new column)
* On the login page, at the bottom of the page where it has the copyright info, fetch and add this version number of the website as well. This will be on a new line above the copyright:
** Format: *Version X.Y.Z | About SchoolCafé* 
*** Font-size: 16px with 10px margins around the pipe
*** Font color: #665AD8
*** Font weight: 500 (or bold)
*** About SchoolCafé URL: [https://schoolcafek12.com/|https://schoolcafek12.com/]
* Modify the copyright line:
** Font weight: 500 (or bold)

# NXT-18544 — QA-DataQuery Framework - Framework design should be data driven.

## Description
The whole design should be data driven and UI should be dynamic in nature. The moment new DataModels are added, it should automatically populate in front end. The objective is to eliminate and/or minimize the need for future coding. QA only story.

## Acceptance Criteria
The moment new DataModels are added, it should automatically get populated in front end for users.

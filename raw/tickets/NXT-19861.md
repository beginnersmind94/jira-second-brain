# NXT-19861 — MP- Menu- Menu item Strict batching

## Description
As a Kitchen Manager, I want my strict batched menu items to be indicated on the menu as they are assigned. 

Once I have selected a recipe to be strict batched and I use that recipe as a menu item on the menu, display the strict batch with the serving size for informational purpose. This should not effect how many servings they plan for on the Menu. 

Possible placement:

!image-2021-09-27-09-25-06-535.png|width=200,height=183!

 Display includes “Strict Batch Enabled”

Serving size, Serving Measure, and # of servings

!image-20211110-171416.png|width=1131,height=254!

## Acceptance Criteria
Show strict batch information on menu items that have it enabled. 

Strict batching on menu planning is only informational and should not impact counts on menu. 

Display and pull information (number of servings and serving measure) on recipe for menu item. 

No restriction on projected counts based on strict batch (ONLY IN MENU PLANNING)

# NXT-20355 — INV - Reports - Site Ordering Guide

## Description
Site Order Guide: Selecting a site and one or all vendors will provide the current list of all items orderable for the selected site and from the vendor(s) with following information:
*When Selected a Site, Vendor (one vendor is selection)*

*  Header section to Show: Site name and Vendor name
*  Items will be Grouped by Item category
*  District item number, Item description, Vendor item number, Contract period, unit price, Lead time, Min order Qty, Max order Qty, Delivery fee, processing fee, NOI.

*When selected a site and multiple vendors are selected :*

*  Header section to Show: Site name and Vendor name
*  Page break for each vendor
*  Items will be grouped by Item category
*  District item number, Item description, Vendor item number, Contract period, Unit price, Lead time, Min order Qty, Max order Qty, Delivery fee, processing fee, NOI. (how about showing the par level Qty if configured * for the item? )

# NXT-49471 — MVR - Dashboard - View Config

## Description
As a SchoolCafé user, I would like to be able to view the configuration of my managed view reports so that I can quickly make decisions.

*Notes:*

* Copy the View config from Custom Reports as much as possible and the requirements included here assume you will do that

*Requirements:*

* On the Edit button of the grid in Reports > Managed View Reports, do the following:
* Action column updates:
** View Configuration
*** If button not there yet, add it
*** On click, open the View Managed View Report Configuration drawer



*View Managed View Report Configuration drawer*

* Create a new drawer
** Name = View Managed View Report Configuration
** Size = 3/4
** Use the sliding action when opening
** Copy the View Configuration from Custom Reports
* 
* Show the MVR configuration same way show CR configuration
** On config tab, show the criteria the same way you show the SQL statement
* 
* Security:
** User must have view rights to Manage Views Report dashboard to use this drawer
** In R&P, don’t show this drawer

## Acceptance Criteria
* Users are able to view the configuration of the MVR
* Only users with view rights can views the settings

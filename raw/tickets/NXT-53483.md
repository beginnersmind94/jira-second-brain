# NXT-53483 — PE - GDSN Connect - Consider Item Name Inventory as Menu Item Name for Stock Items only

## Description
_(none)_

## Acceptance Criteria
Manu Navigation:

PE: MP- GC - Pending Upload Screen

Currently system do not process the records for stock items if the menu item name is missing 



Change:

Applicable to Stock Items Only (where Menu Item Names are blank)

Upon uploading the data in Pending Upload Screen , the system shall user Item Name inventory for stock items (instead of Item Name Menu- as Ingredient names are not needed to be considered for Stock Item)

As a result, all the stock item only records processed for updates (instead of rejecting them as exception)  

Regression - If the Item Name inventory is duplicate- those will still be recorded as duplicate (as-is)

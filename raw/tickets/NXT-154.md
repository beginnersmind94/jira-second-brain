# NXT-154 — Configuration - Special Assistance Programs

## Description
As a Central Office Admin I want a way to configure which sites are operating under a Special Assistance Program (CEP, Provision 2), their duration, and their claiming percentages if applicable.

*This is a port of the existing pages, however they could be simplified somewhat.*

The main functionality here is designating what program a site is participating in (CEP/P2), the start and end dates, and any special claiming percentages for each month.

## Acceptance Criteria
User can set a site or a group of sites to be part of a special assistance program, the type of program, the start and end dates, and claiming percentages for each month between those dates.

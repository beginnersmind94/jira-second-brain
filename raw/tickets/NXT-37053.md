# NXT-37053 — IM - Item Config - Serving Measures - Convert page to common measures

## Description
As a Nutrition Manager I want the Serving Measures page to be a list of common reusable descriptions instead of specific saved values (mostly from CNDB) so I can see how many items have these measure descriptions in them.

*Requirements*

* Change the nature of the Serving Measure page to accommodate the change of Serving Measure from a dropdown to a smart text field
** [https://primeroedgenext.atlassian.net/browse/NXT-37052|https://primeroedgenext.atlassian.net/browse/NXT-37052|smart-link] 
* Remove all CN measures from the list
** [https://primeroedgenext.atlassian.net/browse/NXT-37217|https://primeroedgenext.atlassian.net/browse/NXT-37217|smart-link] 
* Create a flag for Common Measure vs. Standard Measure
** Identify Standard Measures (Cup, Tablespoon, #10 Scoop, etc.)
*** Identify *scalable (weight)* standard measures: Gram, Ounce, Pound
**** *Volume* measures ([NXT-37054|https://primeroedgenext.atlassian.net/browse/NXT-37054]): Milliliter, Fluid Ounce, Quart, Gallon, etc.
** Identify Common Measures 
*** this will be a growing list of mostly Cybersoft defined common values
**** we could potentially run some logic on the CN items to identify initial common measures
*** Example: Slice, Wedge, Carton, Piece, etc.
*** Include some pack size type units in the Common Measures
**** Case, Package, etc.
* Rename page to Common Measures (title)
** Rename ‘Serving Measure’ to ‘Measure Description’ (in column and slideout)
* Allow users to add their own common measures
** These are always in *Local* data source (hide the field from the user)
* Create logic to allow these common and standard measures to be referenced by Items
** We will not require users to use exactly these measures, but we can provide them as suggestions
** Identify common measures even when the item has additional description text (Ex: 1 Cup, blended)
* -Potentially create a column to list the number of items which have a measure which includes that common measure descriptions-
** [https://primeroedgenext.atlassian.net/browse/NXT-37216|https://primeroedgenext.atlassian.net/browse/NXT-37216|smart-link]

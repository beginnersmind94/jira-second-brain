# NXT-11774 — Edit- Notification - Setup

## Description
As a district admin or super user (platform level user), I would like to edit earlier setup Notification for a module and business process. 

Allow a user to edit notification based on the following attributes.

*Attributes*  *Possible Values*
*Channel*	- Email, Print, SMS, App Notification ( we will support Email as first priority and Print is time permits)
*Content Type* - Inline, Letter 
*Supported Delivery*	- Real-time, Scheduled

Most of the above attribute selection is mandatory for saving the notification. 

Able to show the save the notification.
Able to cancel not to save the notification.

By clicking the edit on the list in Notification list displays this.

*Acceptance Criteria*

* Save the notification.
* Appears in the list of notifications

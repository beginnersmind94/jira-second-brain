# NXT-5398 — Data Exchange - SIS Individual Student import via API

## Description
*User Story*
As a PrimeroEdge User with access to the POS module, I can click the Refresh Student Data button on a Student record to initiate an API call that updates the information on the specified student on demand.

*Mockup*
[https://ymiyqp.axshare.com/#id=30o2b5&p=page_1|https://ymiyqp.axshare.com/#id=30o2b5&p=page_1] (mockup done in Classic)

*Requirements*

* SIS data imported via API must be enabled and configured on the System → Configuration → SIS Data Import page
* Refresh Student Data button
* Toast notifications on import initiation and completion

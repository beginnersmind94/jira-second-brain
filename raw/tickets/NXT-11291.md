# NXT-11291 — Notification - Setup

## Description
As any user, I would like setup the Notification for a module and business process. This setup is for each district.

Allow a user to define notification based on the following attributes.

*Attributes*  *Possible Values*
*Channel*	- Email, Print, SMS, App Notification ( we will support Email as first priority and Print is time permits)
*Content Type* - Inline, Letter 
*Module* - Student Eligibility, POS, Account Management, Item Management, Inventory, Menu Analysis
*Business Process*	- Based on the modules you can have Business Process. Example: Student eligibility - Module, Application Approval / Disapproval - Business Process
*Supported Delivery*	- Real-time, Scheduled

Most of the above attribute selection is mandatory for saving the notification. 

Able to show the save the notification.
Able to cancel not to save the notification.

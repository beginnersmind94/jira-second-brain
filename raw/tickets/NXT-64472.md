# NXT-64472 — INV - Orders - Receipt Details - Enhanced Icons for Changes

## Description
As a Site Manager I want the Receipt Details to more clearly indicate what changes were made during distribution fulfillment so I can compare against my original order.

!image-20250718-190607.png|width=75%,alt="image-20250718-190607.png"!

## Acceptance Criteria
* Add click event on Substitution icon to show the original item which was substituted
* Do not show the +/- cart icon for substitutions unless the quantity is actually different
** compare against original order quantity (orig: 3, shipped: 3; therefore no + icon)
* Add tooltip or click event on Added Item icon to inform user 
* Add tooltip to on +/- cart icon to show difference between original order quantity and the shipped quantity

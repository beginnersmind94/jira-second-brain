# NXT-2711 — System - Users - Set user status

## Description
As a sponsor administrator I want the ability to set the status of Active or Deactivated for a user so that I can manage staff access to the system.

Create a “Deactivate” button on the user details page. If the user is already inactive, we should create an “Activate” button. 

By deactivating, the user will not be able to sign in and/or have access to any parts of the system.

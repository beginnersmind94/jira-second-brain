# NXT-12556 — AcctMgt - Account Details - Correction Letter

## Description
As a district admin I made a mistake while processing an application - I went ahead and changed the student directly to a different eligibility determination, so I need to send a correction letter.

*Pre-Requirements (merged from NXT-15743):*

* Move the 'Reminder’ templates from Eligibility to Account Management in the System > Templates page.
* Re-enable or add the 'Correction Letter' template (as we showed it in Classic). It should also be displayed in the Account Management of the Templates page.

!image-2021-06-21-15-23-48-339.png|width=200,height=183!

[^CorrectionLetterHTML.txt]

 

*Requirements*

* User has a button to create and send the Correction Letter template *for this scenario below*:
** Student has had an eligibility change in the same academic year and there has been a change in outcome (multiple eligibility records in the same year with different eligibilities or reasons).
** Example: Processed as Free from an Application, then manually edited to Reduced within the same academic year.
** This will only come in to place if the student gets a +*worse*+ status.
* The button should be available on the Eligibility section, and when selected it calls the Correction Letter template and populates it accordingly.
** Only display the button when this scenario is met; otherwise it should be hidden.
** When this ‘Correction Letter’ button is selected, open the letter viewer window for this student and display their correction letter.

## Acceptance Criteria
Verify that the ‘Correction Letter' template type can be found in System > Templates > Account Management. 

Verify that the 'Reminder’ templates have all moved to Account Management section as well.

Verify that the standard letter template (in any languages available) is the same content as in Classic.

Verify that:

* User has a button called ‘Correction Letter’ available to them for the scenario of a student’s eligibility being changed to a worse status during the same academic year.
* The button should only be visible if this scenario is valid.
* The button should be available in the Account Details, within the Eligibility panel (see mock-up).
* When the button is selected, the letter viewer window for this student is opened and their correction letter is viewable.
* A copy of the notification is saved and can be reviewed on the student account in the Correspondences section (can be deferred until the Correspondence story is complete-NXT-14761).

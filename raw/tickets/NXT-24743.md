# NXT-24743 — Acc. - Sessions - Recalculate dynamic meals when editing

## Description
As Sophie the site manager I need Reimbursable Dynamic Meals to be recalculated if I edit a transaction.

Using the business logic that is in the POS we need to recalculate if a reimbursable meal is formed, what items make up the meal, and what items if any are alacarte. The amount charged will be updated according as well as any account balances. The production counts are also updated as needed.

## Acceptance Criteria
Reimbursable meals are calculated based on the business logic used for POS.

This should only be used if the site is currently configured to use dynamic menus.

Amount due/Charged is updated

Account balances are updated based on the changes

Production counts are updated.

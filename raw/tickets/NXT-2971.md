# NXT-2971 — System - Organization - Sites - Site Details - History (audit/logs)

## Description
As an administrator I want to view the history of changes to a site so that I can review who/when/what changes were made for audit purposes.

Should display old and new values.

PrimeroEdge Classic Reference: System > Sites and Users > Sites > Select a Site > History (tab)

Site Characteristics has not been completed yet so we don’t need to worry about displaying it for now until it is complete.

## Acceptance Criteria
1. User can see a list of history records and expand/collapse details
2. User can see the attribute that was changed and the old/new values
3. User can see who (Full Name + (username)) made the change and when (date/time stamp)

# NXT-32135 — INV - Vendors - Items tab - Update for Pack Size Config

## Description
As a District Admin I want the Contract Item configuration to reflect the pack size units of my items so I can configure the correct unit for ordering.

*Requirements*

* Update the Vendor Items tab screen to reflect the various units of each item
* Display each unit on its own row, according to the items added to the contract
** Preserve item relationships for paging
*** “child units” should not be separated by page from their parent units
* Display Item # only once for the highest unit
* Replace Unit Type with Unit Description
* Show conversion (sub) quantity 
* +Business Rules+
** Related units are displayed together, but individual Vendor Items are selectable and expandable in the side panel
* +UI Ideas+
** Remove some header search options
*** Weight, FMV, Unit Description
*** Optional: Item Category, Storage Category (since there are dedicated dropdowns)
** Move all unit-specific attributes to the right-most columns
*** but ensure they are always visible, by reducing width first from Category

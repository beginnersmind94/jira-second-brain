# NXT-30025 — PR- Reports- Post Production Costing (Summary by District)

## Description
The Post Production Costing Report gives the district information regarding the cost of foods in relation to the completed Production Plans from their sites. It looks at the quantity of ingredients calculated to be used to generate the servings, and pulls the costing from the Fair Market Value listed with the products associated to the ingredients.

Example: If my Production Record states that I produced 23 apples, then the software would calculate the Fair Market Value of those apples, divided by how many broken units contained in the whole unit, and multiply that broken unit cost by those 23 apples.

The district should be able to pull this report with the following details:

Site: Default to All

Meal Type: Default to All

From Date

To Date

Report Options (Second Criteria) : Details by Site, Summary By Site , Summary by District

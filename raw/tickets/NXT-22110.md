# NXT-22110 — INV - Additions - Filter cancelled transactions

## Description
As a FSD I want a Show Cancelled checkbox so I can view cancelled transactions when I need to.

*Requirements*

* On the Additions List page
** by default hide all Cancelled transactions
** add a ‘Show Cancelled’ checkbox
*** when checked, this will display the cancelled transactions *_in addition_* to the other transactions
*** unchecked by default
** For cancelled transactions, display ‘(Cancelled)’ next to the Transaction # in the list
*** test visual indicators (red text, strikethrough, colored label, etc.)

Mockup Note: Button title will be ‘Reasons’ (plural)

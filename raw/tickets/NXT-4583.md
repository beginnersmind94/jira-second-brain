# NXT-4583 — AcctMgt - Accounts - Allow user to View Special Account Details

## Description
As a district admin, I would like to be able to view information regarding Special Accounts for any site within my district, so that I can view their account details and transaction history, and make certain changes needed as well. 

*Summary*

* SchoolCafé has several special accounts - Visitors, New Staff, New Students, Additional Income - that are available at the POS and via the Website. However, there is no way to configure those accounts. So, let’s allow a user to view Special Account details and exercise limited administration over them using a few of the same tools that are accessible when configuring Students and Adults.

## Acceptance Criteria
This story is complete when a user can go to the Accounts page, search for a Special Account using the filters, select that account, and be able to view the *_limited_* details above as well as perform *_some of_* the same account actions that they can for Students and Adults.

Verify that the shortcut menu is accurate as well.

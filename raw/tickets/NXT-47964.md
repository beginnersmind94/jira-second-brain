*requirements*

* Add actions to the grid:
** View
*** Drawer showing the user who has a file as part of the batch job
**** Templates: list the letter templates used as part of this
***** TemplateName (LanguageCode), TemplateName (LanguageCode)
***** Ex: Reminder Letters (ENG), Spanish Reminders (SPA), Reminders (FRA)
**** Columns: ID #, Name, Method
**** Action: Download
** Download
*** Serve up the single file for download, add to History tracking
**** Who downloaded it and when
**** File naming:
***** Zip file - Reminders_Notifications_YYYY_MM_DD - example: Reminders_Notifications_2024_05_13
****** File names inside the zip: Reminders_# - example: Reminders_100.pdf
** Delete
*** Remove download file, update status to Deleted, add to History tracking.
**** Who deleted it and when
** History
*** Show history for the batch job:
**** Who initiated the print and when
**** Who downloaded it and when
**** Who deleted it and when
***** Files should automatically be removed after 30 days -- this should also add to History-

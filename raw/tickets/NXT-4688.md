# NXT-4688 — Item Management - Table View - Add and Remove Columns

## Description
*User Story*

As a user, I want to be able to add and remove columns from my results (hide and unhide)

*Mockup*

!image-20200730-132912.png|width=100%!

 

*Requirements*

* Remove default sorting on Table view
* When clicking on the gear, it will show all the columns that are currently selected with check boxes
* Other available columns will be shown without boxes checked
* If user wants to select a checkbox, when they select it, the column is now visible on the results
** The item will already have the attributes, they just will be hidden
* When unchecking a box, the column will hide
* Default list
** Name (Can’t be unchecked)
** Short Name
** Item Code
** Product Code
** GTIN
** Datasource
** Status
* Additional fields unchecked
** Manufacturer Name
** Brand Name
* If no data is available for that column for ANY of the results, still display the column but there will be no data shown in that column

## Acceptance Criteria
* The user will be able to hide and unhide columns for available data

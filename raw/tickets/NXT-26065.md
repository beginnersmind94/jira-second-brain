# NXT-26065 — Adding Missing Allergen

## Description
The current story is to cover the tasks related to adding a missing allergen in code mapping page in Item Management page.

When an missing allergen is noticed in Code Mapping page, the user need to add the missing allergen by visiting the Item Management, item Configuration, selecting data source (local), Custom Allergen Configuration (configuration options).

Note: When an allergen is added as stated above, the newly added is Allergen only available to the concerned school district only and not any other district in the state.

## Acceptance Criteria
Verify and Validate that the newly added Allergen is reflecting in the custom Allergens.

Verify and Validate that the newly added Allergen is reflecting in the code mapping @ Data Exchange Platform immediately.

Verify and Validate that the newly added Allergen is reflecting only in the concerned school district to which thr user belongs and not any other school district.

# NXT-1654 — INV - Inventory - Withdrawals - Header

## Description
As a Site Manager I want a Withdrawals page so I can perform inventory withdrawals and configure reasons, templates, etc.

*Note:* Please use same components developed for the corresponding Additions page [https://primeroedgenext.atlassian.net/browse/NXT-1779|https://primeroedgenext.atlassian.net/browse/NXT-1779|smart-link] (check with INV2)

*Requirements*

* Add a page to the Inventory category titled Withdrawals
** Also rename the Transfers/Withdrawals category to just ‘Transfers’ (since Withdrawals are in the Inventory category)
* This page will be a list page by default
** this will list the existing Withdrawal transactions
* From this page there will be the ability to:
** Create a new withdrawal ([NXT-1740|https://primeroedgenext.atlassian.net/browse/NXT-1740])
** Configure Withdrawal Reasons ([NXT-1674|https://primeroedgenext.atlassian.net/browse/NXT-1674])
* Display a search header for the list page
** Site selection
*** Use standard control (Areas if they are configured)
** Date Range picker
*** default to the past week
** Reason dropdown
*** this will allow the user to filter the list according to specific withdrawal reasons
*** this should be multi-select
* Display a list of the withdrawal transactions based on search criteria ([NXT-20048|https://primeroedgenext.atlassian.net/browse/NXT-20048])

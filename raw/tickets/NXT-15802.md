# NXT-15802 — IM - Warning sign for Missing item information 

## Description
As a User I would like to see a Warning Indicator on an Item So that It’s easy to understand if an Item is missing information

## Acceptance Criteria
Given the user is on the Items landing Page 

And a Search is already been applied 

When the list of items are available in the Search result 

Then Show warning Indicator for the items that are missing any of the blow information 

And also show the missing indicator with a rolling message

* Item Details 
* Nutrient 
* Contribution info 
*  Allergens
* Product Info 
* POS

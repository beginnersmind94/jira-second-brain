# NXT-8035 — {Menu} Analysis - Calculate/display nutrient results

## Description
As a Menu Planner
I need to be able to build Menus
so that I can show that they are compliant.

Calculate (all) and display (only required) the average nutrients for a Menu.

Please refer US-8117 for Mockup. Please see document “PrimeroEdge Menu and Menu Cycle Analysis” posted to Menu Planning Team group Dec 3, 2020 for high level nutrient calculation process. In a nutshell, it is a weighted average of all Items. Basically, for each Item you multiply the number of servings time the nutrient value, sum them up, and then divide by the number of meals.

## Acceptance Criteria
Average Nutrients are calculated and displayed according to documented process.

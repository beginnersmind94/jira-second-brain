# NXT-29065 — IM -POS Flag (Items)

## Description
As user with permissions to see/edit Item Management Items I would like to quickly identify items that are POS items

## Acceptance Criteria
*Entry Point*

* Item Management → Items



* User shall have the option to filter items which are set as POS Items using a check box \[POS Items Only]
* POS Item filed (flag) should be displayed as a column on the result grid
* POS Item filed (flag) should be available as a column under  “Manage View” options

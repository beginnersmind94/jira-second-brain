# NXT-28376 — MP- Menu- Menu Item Quick List display and search

## Description
As a user, I would like to be able to use the quick lists I created on my menu. 

I would like to be able to use multiple quick lists if I would like but can only add one at a time.

## Acceptance Criteria
* I will need the ability, when creating or editing menus, to select one of these lists and have all of the items on the selected quick list (or lists) appear on the menu I’m currently working with.  I should be able to apply multiple quick lists to a menu, but only selecting and applying the lists one list at a time. 
* Be able to view all my items that are part of the quick list (Show same as Add Menu Items Screen)
* If a menu item from the list is already on the menu that I’m working with, DO NOT duplicate
* Allow user to add some items from a quicklist by having a checkbox (multi select option) (Default to select all)
* Display how many items have been used and how many items are in the quick list
* Once user selects the quick list and chooses to add to menu, display all the items on the menu in their respective categories. '
* Once Menu items have been added to the menu, display that menu item as “on menu” from “Add menu item” function
* Status for quicklists:
#items used = 0 means not used
#items used >1 means partially
#items used = #items on list means Fully

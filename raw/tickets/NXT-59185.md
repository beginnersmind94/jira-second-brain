# NXT-59185 — Imports - Download Exceptions - (FD 265800 & 274069)

## Description
As a Central Office user, I want to download a CSV file of all exceptions from an import so that I can review errors offline and take corrective actions.

*Notes:*

* We want to do the download like we do for Source Data tab

*Requirements:*

* All requirements are for the Exceptions tab when click View History on the Import Monitor
* All requirements are for any import that has an exceptions tab, but especially need to do for Student Import
* A Download button is displayed on the exception log screen at the top right of the grid in the body.
* On Click:
** Generates and downloads a CSV file containing all exceptions for the selected import.
** Shows the user a confirmation button
* The CSV file includes the following columns:
** Row Number
** Field Name
** Error Description
* The downloaded file is named using the format consistent with Source Data download
* The file format follows CSV standards with proper escaping of special characters (e.g., commas, quotes).
* The feature works across supported browsers and devices.

## Acceptance Criteria
* User is able to download all exceptions for the import

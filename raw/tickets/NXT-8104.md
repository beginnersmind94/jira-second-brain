# NXT-8104 — INV - Physical Inventory Status - Site List

## Description
As an inventory supervisor I want a Physical Inventory Status page so I can view which sites have not performed a physical inventory yet.

After Search is clicked, load list of sites 

*Requirements*

Display a grid with a list of sites returned by the search criteria:

* Site Code
** searchable/sortable column header
* Site Name
** searchable/sortable column header
* Status
** Not Started
*** no physical inventory record
** Not Completed
*** physical inventory incomplete
** Not Reconciled
*** physical inventory pending reconciliation
* default sort order:
** first sorted by status
*** Not Started
*** Not Completed
*** Not Reconciled
** alphabetically ascending by Site Name

## Acceptance Criteria
User can view the list of sites

# NXT-66965 — Add Financial Oversight Module to License Management

## Description
As a system administrator, I want to enable the Financial Oversight module through License Management so that my team can access financial reporting features.

## Acceptance Criteria
* "Financial Oversight" appears as a product option in System > Management > License Management
* Module can be activated/deactivated via toggle (like other modules)
* License type is "District" (district-wide access)
* Module respects license renewal dates
* Users without active license cannot access Financial Oversight module

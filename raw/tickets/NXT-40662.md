# NXT-40662 — MP- Menu-Hide template if not used to create menu (UI ONLY)

## Description
As a menu planner, if I created a menu and did not use a template, I do not need to see the blank section with “template used”.

## Acceptance Criteria
* Hide Template used section if no template was used to create menu 
** !image-20230614-171915.png|width=1626,height=455!
* IF template was used to create a menu, once the menu is now in edit mode, it can just be a view mode and does not be within the drop down box
** !image-20230614-172012.png|width=1607,height=495!
* When hovering over the template name, display the hand cursor so the user knows they can click

# NXT-49644 — 2FA - District - Delete

## Description
As a Customer Support user, I want to be able to Delete a district’s 2-Factor authentication so that they don’t have to do it anymore

*Notes*:

* This story is for deleting an authentication method that uses 2FA 
* It covers delete for all authentication methods that use 2FA

*Flow:*

* User browses to System > Security > Authentication tab
* User views if an authentication type has 2FA or not
* User clicks delete button

*Requirements*:

* These requirements apply to the System > Security page > Authentication tab:
* 
* On the grid, update the Delete button as follows:
** If the authentication method is used as a 2FA method for another row:
*** Disable the delete button
*** Put tooltip: Cannot delete. Used as 2FA for another method.

## Acceptance Criteria
* User cannot delete a method used as 2FA method for another authentication type

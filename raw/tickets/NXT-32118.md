## Description
As a district admin I want a clearer understanding of how the DC Configuration works to avoid confusion and the need for additional training.

* DC Files that are in progress should have their action buttons disabled.
** Tool-tip: This file is still in progress. Actions will be available after completion.

** Column: Method

** Change text: Add Files → New File

** Add some instructional text and a warning under the Certification File section.

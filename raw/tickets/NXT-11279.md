# NXT-11279 — IM - Items - CN Database Items not editable

## Description
As a user, I shouldn’t be able to add/edit a CN database item (ie, datasource = Child Nutrition Database) at this time (Another story will be in place to identify the fields that CAN be editable)

*NOTHING can be editable at this time.*

Requirements:

* User cannot edit the following sections of a CN Item
** Nutrition Information
*** Serving Measures
*** Nutrients
* User CAN edit
** General Item Information
*** Can’t change the name or short name
** Contribution Information
** Allergen Information
** Product Information
** POS Information

## Acceptance Criteria
User cannot change nutritional information for a Child Nutrition database item, but can change values that are not specific for the CN database

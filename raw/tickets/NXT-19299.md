# NXT-19299 — System - Management - Templates - Add 'Income Survey Notice' template

## Description
As a district admin, I would like to be able to view and edit a canned notice for income survey responses, so that I can notify specific households that we have received their survey.

*No UX Required*

Requirements

* In System → Management → Templates, under Eligibility, add an *Income Survey Notice* template.
** Add templates for any active languages (though the template itself can be in English for each)
* Template should have a default version that can be edited like any other template (HTML provided below).
* It should be easy to retrieve this template for later use when we implement the ability to notify a household of a completed survey, with the correct field data pulling through.

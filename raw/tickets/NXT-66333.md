# NXT-66333 — ACC - Attendance Feature - Monthly Entry Updates and Saving

## Description
For updating and saving the Monthly Entry method for attendance we will need to do the following.

-Update data only after user clicks save.
-Provide a cancel button that simply clears inputs (shows if the page detects a change in any single textbox)
-Temporarily retain data if user pages sites or months. (only after changing months or pagination for sites, not expected if a new site filter is applied, or entry method change)

*Bulk Entry Tool.*

-Provide a bulk entry tool that takes 2 values and applies those to ALL respective columns for every month after clicking an “Apply”. 

* Provide a warning message before applying) - “Are you sure you wish to set the Average Daily Attendance/Attendance Factor for ALL months with the entered values?” (Yes/No)

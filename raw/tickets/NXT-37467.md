# NXT-37467 — Student Export – Change Edit & Delete to toggle button - Dev only

## Description
As a SchoolCafé Admin I want to be able to more intuitively add/remove fields from export so that I can figure out what to do without asking anyone.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Exports
* Go through the Pre-export and get to the Field Mapping screen
* Pick which fields you will export and then finish the export

*Notes:*

* User feedback during initial setup of new customers is that the edit and delete buttons are not intuitive or easily understood how they work and that a toggle button would be better

*Requirements*:

* When in Student Export on the Field Mapping Screen, do the following:
* Where there is an edit button and a delete button, replace them with a toggle button
** On means include in the export
** Off means exclude from the export
* Remove the edit and delete icons

## Acceptance Criteria
* Toggle button added
** When Yes, field is included in export
** When No, field is not included in export
* Edit and Delete icons removed

# NXT-14130 — [CANCELLED] Platform - SSO - Authorization in Perseus Dashboard

## Description
*Persona*: Perseus User/admin

*Access level:* School District/Site

*Description*:

As a Perseus user, I should be able to launch Perseus dashboard using my registered account using my ISD’s domain configuration

*Requirements*:

* User logs into their account with respective email domain and validates the session
* Upon successful login, user is redirected to Perseus login
** Retrieves roles/permissions of that specific user from the authenticated session
* If registered email is not found, prompt for invalid email account and redirect them login page 
* If registered email is not set for a district, launch manual login page

## Acceptance Criteria
* User login is validated with new session established
* User shall be redirected to Perseus dashboard
* User roles/permissions are retrieved
* User shall be prompted to contact administrator if authentication/authorization fails
* User shall be redirected to manual login page if email account is not setup with district email domain.

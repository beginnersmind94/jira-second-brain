# NXT-3782 — Create date range selection

## Description
The date range selection is an extension of the range selection. It allows the user to select two dates and returns a list of dates between and including the dates entered.

## Acceptance Criteria
The user can enter an upper and a lower date, using the previously created date function.

The date range selection returns a list of data that matches the dates between and including the dates entered.

Reference the 'as of date' function, if only one date is entered.

When dates are selected they are treated as an AND with other filters.

# NXT-6333 — Messages - Inbox Actions

## Description
*User Story*

As a PrimeroEdge user, I have actions available on the Inbox screen for any selected messages in my Inbox

*Mockup*

[https://react-material-kit.devias.io/app/mail/all|https://react-material-kit.devias.io/app/mail/all]

The options are available in the dropdown highlighted below (not functional in the example linked above)

!image-20201006-153859.png|width=1607,height=822!

*Requirements*

* There are several options available for selected messages present in the dropdown menu
** If no messages are selected, no actions are taken
* Options
** Mark As Read
** Mark As Unread
** Reply
*** Single Selection Only
** Reply All
*** Single Selection Only
** Forward
*** Single Selection Only
** Delete
* For single selection items, if multiple messages are selected, the actions are not present
** Reply
** Reply All
** Forward

## Acceptance Criteria
* Actions are present in the Inbox
* All actions function correctly

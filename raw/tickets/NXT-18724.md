# NXT-18724 — Menu Calendar Report > Daily

## Description
Once the options have been selected on the page, there are multiple versions of the Daily Calendar Report. 

The different “versions” are based on the options chosen, and are impacted by the selection(s) of:  

# Show Summary Items Only
# Show Menu Name
# Include Nutrients

If “Show Summary Items Only” is selected - then there will be only the items displayed that are marked as “Show in Summary” on the item’s “Item Management” page.

If “Show Menu Name” is selected, then the calendar will display the menu items  AND the Menu Name.

If “Include Nutrients” is selected, then the carbs and calories will appear on this daily report next to the menu items.

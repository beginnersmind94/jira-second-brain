# NXT-66136 — Add New Entry button

## Description
This button is used to consolidate features into a single page. On clicking it, the user will see a popup that will then let them choose between manual and ad hoc entry and make that entry.

## Acceptance Criteria
* Modal is added in top right corner of the page
* On clicking it, user sees the option to select between manual entry and ad hoc entry
* It will then see the functionality associated with each try type based on the selection which is mentioned in the linked stories.

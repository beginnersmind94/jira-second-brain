# NXT-3702 — Add Item - Ingredient - Add Contribution Information 

## Description
*User Story*

As a Item Management User, when creating an Ingredient, I want to add the Contribution Information for a serving size.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1]

*Requirements*

* Header
** Add Contributions
* Serving Size
* Food Component
** Fruits
** Fluid Milk
** Meat / Meat Alternatives
** Grains
*** Whole Grain-Rich
*** Non Whole Grain-Rich
** Vegetables
*** Dark Green
*** Red / Orange
*** Beans/Peas (Legumes)
*** Starchy
*** Others
* Each Food component would have the measurement size and allow the user to put in the contribution

## Acceptance Criteria
For a serving measure (which probably will be the same as the serving measure to enter nutrients but may differ):
Each Food Component can be selected in the appropriate fractions (either 1/8 or 1/4).
'Meat / Meat Alternative' and 'Grains' are only allowed as 1/4 fractions (NOT 1/8)

(FUTURE) This information will be used to suggest contributions for each available Menu Item serving size.

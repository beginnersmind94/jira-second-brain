# NXT-30930 — PR- Prep Site Set one site to one prep site

## Description
As a District manager, I would like to indicate which site belongs to which prep site

I would like to have a way to configure 1 site to 1 prep site relationship



*Navigation:*

Menu Planning> Site Groups configuration> Site Configuration (Tab)

## Acceptance Criteria
* Default all sites to themselves as the prep site on initial entry
** EX: Site: Hundred Highschool > Prep Site: Hundred Highschool 
* A site (receiving) can only have one prep site 
* Display Effective date to show when Prep site was created on initial entry (Column Name: Prep Site Effective Date)
** Display NEW effective date to show when NEW Prep site will take effect (This will effect all future production plans and records) *(NEED CRON JOB)*
* Display History for each site to display changes done for prep site or any other changes (IE: Site Group)
* Any changes done on these configuration pages should reflect on the individual production plan and record. Effect only production plans that are not complete. (These changes will be handled story ([NXT-30931|https://primeroedgenext.atlassian.net/browse/NXT-30931])
** User may still go back to the individual production plan and change the prep site and save. This will only effect that particular production plan and record. 
*** EX: If user manually changes production plan prep site but does not change the status to plan complete and is still in plan incomplete, if the configuration is changed again, it will effect that production plan and record.
* Information on this page should reflect on Prep Site Configuration Tab ([NXT 24943|https://primeroedgenext.atlassian.net/browse/NXT-24943])

# NXT-35061 — MP- Menu Planner Menu Cycle Scenarios- Serving groups not available on 1 or multiple menus in a cycle

## Description
If there are multiple menus in the cycle and have different serving groups, we need to handle the display of how it will show depending on what serving group you select on the drop down on the Menu Planner. > 

Example

Menu -1 has PK, K-5, 6-8

Menu –2 has only K-5

Menu – 3 has only 9-12

If the serving group is not part of this menu, Add a Click to add “+” display so that the user can know the the serving group is not currently part of this menu but can be added if needed.

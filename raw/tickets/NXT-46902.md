# NXT-46902 — Package.json.lock generate at the pipeline

## Description
Currently whenever there is an update happened in the package.json the developer needs to build package.json.lock and commit that as well. 

The recommend approach is to generate this lock file in the pipeline itself so that each module not needed to update the components version everytime. 



Note: This will involve coordination with devops team

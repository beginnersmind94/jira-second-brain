# NXT-37109 — IM - Items - Units - Unit Identifiers

## Description
As an Item Management user I want a column which explains the different use cases of my various units so I can better understand the complex consumption of the serving and pack size units.

*Unit-level Identifiers*

* Base Unit
** only if the base serving is also a pack size 
* +Serving Size identifiers (existing)+
* Cluster within existing singular icon
** Nutrition assigned 
** Contributions assigned
** Used on a recipe
* Preferred Measure
** Show a star/favorite icon for preferred measure
** this can be clicked in edit mode to indicate a preferred serving size
* +Serving Size (++*new*++)+
** Used for Menu Item (MI)
** Used for Single Item Recipe (SIR)
* Possible others
** $ icon on the unit which had the manual cost entry
*** since costs are scaled for all units in view mode, we want to show the unit for which the price was actually entered by the user

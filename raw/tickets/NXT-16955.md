# NXT-16955 — Verification - Edit Sites - Option to use Cause applications when a new site is added to sample

## Description
As Vera the Director I want to shift some of the Cause applications over to be Sampled when the site they’re from is added to the sample pool.

*requirements*

* If the user edits sites and has any added any new sites, let’s offer the choice to shift any Cause applications over to Sampled.
* Scenario:
** User has 3 Cause applications at Site Z.
** Site Z is not part of the official Sample.
** User uses ‘Edit Sites’ and adds Site Z.
** Ask the user to select from the 3 applications as potential apps to move from Cause to Sampled if the recalculation requires more apps.
** We should perform the recalculate option and see if more applications are needed. If so, use the selected applications before moving on to find more apps if needed.

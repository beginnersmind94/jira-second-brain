# NXT-29785 — POS - Detect and Assign input devices to a serving side

## Description
Users need to be able to configure multiple input devices being used at a POS Terminal. The devices need to be able to be associated with a side left/right typically with an arrow pointing to a side. There can be multiple devices per side as an example a PinPad and a Barcode scanner on the left, and a Fingerprint and RFID scanner on the right. The configuration can be saved so that when the POS is launched the device configuration is used and doesn’t have to be re-entered. 

See the link below for web based USB detection, and HID device management APIs

[https://web.dev/devices/|https://web.dev/devices/|smart-link]

## Acceptance Criteria
There is a permission to allow the user to choose Plug-N-Play or Device Configuration.

The user can select to use Plug-N-Play or Device Configuration 

When Device Configuration is selected the user can designate a recognized device as being on the left or right side.

The User can select to add more than one device

The User can save the configuration

The User can update a saved the configuration

The Saved configuration is loaded when the terminal is started by any user.

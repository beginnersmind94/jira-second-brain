# NXT-33576 — Data Synchronization - UI Tool to Monitor and Manage Synchronizations

## Description
A UI tool is needed to monitor and manage the synchronization processes.

## Acceptance Criteria
* A UI shall be built that will allow for the support team to monitor the status of the synchronization processes.
** Failures, successes, warnings shall be made available
** Log shall be made visible
* The ability to trigger the processes shall be made available to the support user

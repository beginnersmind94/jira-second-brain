# NXT-27518 — IM - Variety Menu Item Contribution Edit

## Description
When editing contribution information on a Variety menu item the user shall be reminded of the effects and his approval shall be required 

!image-20220224-110141.png|width=1394,height=418!

## Acceptance Criteria
Entry point

Item Management → Menu Items → Selected Variety Menu Item → Contribution Information \[EDIT]

* When the user click or press the EDIT on the Menu Item Contribution section the user shall be prompted with pop-up window with a notification indicating the effects of of editing the contribution information and requesting user confirmation to continue.
* If user confirm (continue) 
** Then contents of  contributions fields are clear and the user is allow to enter new values
** Menu Items link are remove and user needs to select new menu item
* If user does not confirm 
**  The return to the Variety Menu Item

# NXT-18451 — INV - Orders - Edit Receipt Details - Charges & Discounts - Enhancement

## Description
As an operations manager I want the ability to edit or add values for charges and discounts on a receipt so I can add any fees or discounts associated with the receipt.

This is a continuation enhancement story for [https://primeroedgenext.atlassian.net/browse/NXT-16998|https://primeroedgenext.atlassian.net/browse/NXT-16998|smart-link] 

*Requirements*

* When user clicks the Edit button, all relevant fields will be editable
** This will display all charges and discounts, even if they previously had no value when receipt was saved
* The calculated columns will not have a textbox to enter value
** Delivery Fee
** Processing Fee
** Net Off Invoice
* This section will always be visible in edit mode

Mockup Note: contract-level charges/discounts are *not* editable. Labels only.

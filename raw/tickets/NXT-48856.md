# NXT-48856 — Users Tab - Filters

## Description
As a SchoolCafé users, I would like to be able to see more information about users than what is in the default list using Managed Views, so that I can see other information.

*Requirements:*

* On System > Users, do the following:
* Remove the Show Inactive Only toggle button from the UI
* 
* Add a Filters section at the top
** Do the usual collapse icon
* Add the following filters:
** Status
*** Values: List of status names from our DB
*** Default select Active 
*** Ensure Inactive is a value so that we can get rid of the “Show Inactive Only” toggle
** Created
*** list of date ranges
**** not a date picker
*** Use the date time list in Custom Reports (this week, this month, etc.) like we have for custom parameters
*** Action is on the date/time created for a user so that we can report on new users
** Modified
*** list of date ranges, same as Created
*** Action is on the date/time modified for a user so that we can report on changes to users
** Clear
** Apply
** 
* Note: 
** Example use case: show me a report on users made inactive in the last week
** Example use case: show me a list of new users this month

## Acceptance Criteria
* Show Only Inactive Users removed
* Show Only Inactive Users functionality moved to filters with Status field
* Filters section at top filters grid
* Filters are included in the saved View criteria

# NXT-27075 — System - Cleanup Configuration Settings

## Description
The settings within System | Configuration need to be cleaned-up such that the settings which are no longer used need to be identified and removed.

## Acceptance Criteria
* System shall take a backup of current system settings
* Lookup for the system setting code within SchoolCafe APIs and remove any setting that is not being used
* Settings that are not used in SchoolCafe shall be removed from System | Configuration
* Settings that are not relevant to SchoolCafe shall be removed

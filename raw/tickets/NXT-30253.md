# NXT-30253 — IM - Items - Food Info - View/Edit Nutrients

## Description
As a District Admin I want to be able to view my food info section so I can see at a glance information for nutrients, allergens, etc.

*Requirements*

* Display and populate the Food Info Tab for items which are defined as a Food Item
** If the item was not defined as a food item, disable the tab and color it gray
** Incomplete: flagged as Food item but no Serving Measures saved
** Partial: Serving Measures saved but no Nutrients
** Complete: Nutrient Details saved
* Display all the cards that belong to the food info tab (Serving Measures, Ingredient Info, Nutrient Details,  Contribution Info, and Allergen Information) with all card information as labels in view- only mode
** Make sure the Ingredient Info card is also in view mode on pageload
* Display the checkboxes/flags and comment icon accordingly
** Do not allow them to be changed in this mode but only viewed
* Allow Edit function for Nutrients card
* Note: CNDB Items handled on [https://primeroedgenext.atlassian.net/browse/NXT-31284|https://primeroedgenext.atlassian.net/browse/NXT-31284|smart-link]

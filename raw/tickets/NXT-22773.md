# NXT-22773 — MP- Site Configuration- Ability to configure multiple sites to site group

## Description
As a user, I would like the ability to configure multiple sites in my district to a site group at 1 time without having to configure each one at a time.

## Acceptance Criteria
Allow user to select multiple sites to be configured to a particular site group in one action. 
DO NOT allow multiple site groups to site configuration.

Need Validation for sites that have already been configured to a different site group. 

Effective date disabled if no sites are selected (Site addition not required) 

Only Data source need to be selected (default to local)

Display tool tip to show what site group belongs to what site in the drop down. 

Site group and data source required

Display site group to site in drop down if already configured to another site group.

Disregard filter on site name (Mockup)

# NXT-42756 — Elig - Income Guidelines - State Income Setup

## Description
As Dora the Director I want to setup State Income Eligibility Guidelines to stay in compliance.

*requirements*

* Add a system setting:
** Enable State Income Guidelines (Code: STATEEIG)
*** Options: Yes/No
*** Default: No
* If the setting is set to YES, new options are available on the Eligibility > Income Guidelines page.
** User can see State Ranges for State Free Meals and State Reduced Price Meals
** While editing, user can input percentages for State Free and State Reduced.
*** User can leave it blank or put a 0 in and this should be treated as ‘we aren’t using this option' - no need to show calculated incomes in this scenario.
** Grid should update as the user changes the percentages, like the current grid.

## Acceptance Criteria
* System Setting added.
* If enabled, State Income Guidelines become available on the Income Guidelines page.
* User can edit the State Free and State Reduced percentages.
** Grid should calculate amounts on the fly like it currently does.
* These values should be recorded for that region/district only.
* Special permission added for use with the edit buttons on the page: Income Guidelines - Edit Federal and State IEGs
* We should also remove the Edit option that’s available on the View/Add/Edit/Delete checkbox for this page if we’re doing it this way.

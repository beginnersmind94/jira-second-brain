# NXT-25457 — MP > Menu > SFSP Meal Type needs Adult added as serving option

## Description
Note: This has been a district request multiple times

Users would like the ability to set an "Adult" column when building their SFSP menus, so that they have the ability to Plan in Production for the non-reimbursable food items that will be sold.

## Acceptance Criteria
User will have the ability to select both "All Ages" (for reimbursable production planning purposes) and "Adults" (for non-reimbursable production planning purposes) for their menus.

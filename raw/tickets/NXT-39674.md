# NXT-39674 — Imports – File Mapping – Add New File Field - Dev

## Description
As a SchoolCafé Admin I want to be able to add a field to the File Fields so that I can avoid having to create a new import file when I make a change.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Imports
* Click View on a Student Import record to view the import details
* Go to the File Mapping tab
* Click Add New button and add a record to the list of fields

*Requirements*:

* At a minimum, we need to do this for student import, but would like to do it for all the imports
* 
* On the File Mapping tab of a student import file, do the following:
* Add a button to the File Fields table
** Title: + New
* On click:
** Open a drawer
*** Title: Add New File Field
*** Fields:
*** Name
**** Textbox
**** Required
*** Save
*** Cancel
** If save, do the following:
** Append a row to the bottom of the list of the File Fields table with label from the Name textbox of the drawer
* 
* After add a new row, everything continues to work as normal, so can drag it and associate it with a Mapped Field

## Acceptance Criteria
* Can add a new label to the File Fields
* Can drag and drop it to a mapped field
* Files with this field are able to import into the database if they are linked to a mapped field

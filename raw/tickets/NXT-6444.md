# NXT-6444 — Application Entry - Guided Process (1. Students) - Household Prompt

## Description
*Story* 

As an admin, when adding a student who is part of a household with other students, i want to be notified so i can quickly add other members of the household to application if i desire.

*Mockup* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1296%3A25854&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1296%3A25854&scaling=min-zoom]

*Requirements*

* After a matched student is added to the application, check for other students in the household. If other students are linked to the household: 
** Household Prompt for matched students
*** _It looks like there are other students in this household. Would you like to add any of them to the application?_
**** Shows a grid with other students from the household (same grid in student page - Name, ID, Enrollment Site, Grade, Date of Birth) 
**** Suggestion shows members of the household NOT already added to application (the same student doesnt show in both grids)
*** Action: 
**** Dismiss/'x' to remove the suggestion
***** Suggestions should be persistent until dismissed. User can navigate to another step and back and see the household suggestions.
*** Add Button
**** Triggers the student popup with the student information populated so the admin can answer the categorical and income question for the student 
***** save > student is added to the application (top grid) 
***** cancel > returns to the display screen

## Acceptance Criteria
- Multi-language, ADA compliant.
- show household prompt with options above when a matched student is added to an application (on 'add a student' save - 6022) 
- adding adds them to the application 
- close/done exits the popup

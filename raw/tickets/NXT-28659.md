# NXT-28659 — IM - Permissions (VIEW)

## Description
As item Manager user I shall be able to see Item Management site pages according to user' s permissions (VIEW)

## Acceptance Criteria
* *View* *Permission* - 
** If enabled 
*** If the he user has view permissions to the page the user shall be able to VIEW page content 
**  Else hide page

*Entry Points*

* Item Management → Menu Items 
* Item Management → Item Catalog (List and Split views)
* Item Management → Sync History 
* Item Management → Global Catalog
* Item Management → Item Configuration
* Item Management → Reports
* Item Management → Reports → Allergen Chart
* Item Management → Reports → Ingredient Needing Allergen Review Report
* Item Management → Reports → Locally Grown Usage Report
* Item Management → Reports → Missing Exemption Letters Report
* Item Management → Reports → Recipe Costing Report
* Item Management → Reports → Recipes Materials Report
* Item Management → Reports → Items Aging Report
* Item Management → Reports → Recipe Book Report

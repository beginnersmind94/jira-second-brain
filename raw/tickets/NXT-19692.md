# NXT-19692 — IM - Item Details - Hierarchy - Link Items - Smart Search

## Description
As a FSD I want to use a “smart search” item tool so I can find related items and add them to their correct item hierarchy.

*Requirements*

* When using the Link Items ([NXT-18837|https://primeroedgenext.atlassian.net/browse/NXT-18837]) function on Item Hierarchy card, use a Smart Search to search and preview items
** this smart search is a search box but with dropdown-type results (reference IM > Recipes > Recipe Steps > Add Step)
** label this field ‘Item Search’
** Provide ability to search for related items by
*** Item Number
*** Item Description
*** GTIN 
** display a list of items returned by the search (limit to first 10? items)
*** *Q for Dev:* which rules do we require for eligible items? I.e. do they need to have Product Info populated?
** Limit the item results to only the unit types that are compatible
*** for example, if I am trying to add a *Child* item to a *Case* item type, do not show Pallet, Mixed Pallet, Display shipper, since these cannot be child items to a Case. They can only be parent items
*** *do* return results of items that do not have a Unit Type (user can provide it afterward)

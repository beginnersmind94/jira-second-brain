Workflow: Add New - Manually → type ‘chicken’ → results will show up as pictured above → Click ‘ADD’ on the *2nd item*




Requirements

* Header - Items (This will change when defining how we want to handle recipes. For now, just going with Items
* When typing in the minimum of 3 characters to start searching, start showing the results that come up
*# Prioritize exact match over partial match
*# Prioritize local datasource, shared datasources, and then global datasources
*# Prioritize Name, Item Code
* Search results will be infinite scroll / lazy loading
* Can be Food and non-food items
* Search for:
** Item Name
** Item Code
** Brand Name
* View results will show
** Item Name
** Item Code
** Brand Name
** Datasource
* ADD button will add the result to the Item list
** The search box will disappear and the item will be added
** After adding, clear the search box
* When searching again, the item will still show the ADD button but it will be greyed out and not selectable.

## Acceptance Criteria
User must be able to search and add items to a recipe

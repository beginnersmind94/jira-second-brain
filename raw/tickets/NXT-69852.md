# NXT-69852 — ACC -  Credit Card Terminals Page > Add and Edit Functions

## Description
[https://developer.squareup.com/docs/oauth-api/overview|https://developer.squareup.com/docs/oauth-api/overview|smart-link] 
For implementing support for Square(TM) terminal we will need to create a configuration page for it all to begin.

The configuration page will be where we connect Square APi’s to ExpressPoint devices so that transaction can properly be tracked at the correct POS terminals.

This story covers the Add and Edit Functions of the Main Grid.

* Remove ‘ID’ - no need to show internal terminalid
* 3 min timeout on polling square for linking/syncing updates

* Page permission should be given to the two support roles, Central Office, and Director.
* Page needs to tell the user if the Card setting (CREDCADPAY) is turned off.
** Alert Text: This feature will not work unless the setting for credit cards (CODE: CREDCADPAY) is turned on.
* Confirmation prompt when user disables the Auto Renew reminder
** Text: Warnings will not be sent if you disable the reminder. This may lead to credit card devices expiring at the POS and payments not being accepted.
** Are you sure you want to disable this notification?  
* Success message/toast on saving the Auto Renew

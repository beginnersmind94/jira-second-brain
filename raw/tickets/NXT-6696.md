# NXT-6696 — Sessions - Cards

## Description
*User Story*

As a site manager I want to view a list of sessions tied to the sites I have access to so that I can ensure sessions are balanced.

As a district admin I want to view all sessions across the district to ensure that sessions are balanced and accounted for.

*Requirements*

POS Module > Session takes us to the Sessions list page.

The list of sessions is determined by the Search Criteria entered. 

Cards show the count of sessions and a total meals sold, Free, Reduced, and Paid meal counts and total sales

## Acceptance Criteria
User has access to a page that shows sessions.
User can select a list of sessions based on search criteria
User can filter sessions in the list based on filter selections
User can only see sessions for sites they have access to.
For the list of sessions the total count of sessions
For the list of sessions the total number of meals sold
Total Free, Reduced, and Paid meal counts
Total $amount sold.

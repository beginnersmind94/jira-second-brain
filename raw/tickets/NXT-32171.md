# NXT-32171 — Workspace - New Broadcast Message UI - Usability Improvements

## Description
The “MAILING GROUPS” tab needs to be removed from the popup that is displayed when the user clicks on the “To:” box.

!image-20220623-214124.png|width=694,height=761!

## Acceptance Criteria
* The “MAILING GROUPS” tab shall be removed from the popup that is displayed when the user clicks on the “To:” box.
* The text of the “To” box on the “NEW BROADCAST” UI shall be “To:”
* In the “To:” box, when the user list popup is displayed as a result of a search, only users within the district that show on the Users page shall be displayed.

# NXT-1927 — MP - Reports - Acceptability Factor

## Description
1.) This report should display header information regarding the selections made on the report page - these include:
   a.) Meal Type (can be one, any, or all)
   b.) Site Group (can be one, any, or all)
   c.) Menu Line (can be one, any, or all)
   d.) Start & End dates

2.) The data in the report is to include the following information:
   a.) Menu Item Category
   b.) Menu Item Name
   c.) # of schools that served this menu item on each date (which is separated out in the report)
   d.) Total Count Served (produced quantity minus carryover/waste/return to stock/ala carte counts - taken from the production records of the meals/sites/menu lines/dates selected)
   e.) Total Meals Served (taken from the production records of the meals/sites/menu lines/dates selected)
   f.) Percent (This is the percentage of the total meals served that the menu item comprises - i.e. if there were 100 meals served, and 85 of the menu item served, then it would be 85%)
   g.) Cost per serving (based on calculated cost per serving from recipe/menu item)
   h.) Cost per serving w/NOI (based on costing information from Inventory)
   i.) Total Cost (Total served count multiplied by cost per serving)
   j.) Total Cost w/NOI (cost per serving w/NOI multiplied by total meals served)
   k.) Weighted Cost (Cost per serving multiplied by percentage of total meals served, as in "f" above)
   l.) Weight Cost w/NOI (Cost per serving w/NOI multiplied by percentage of total meals served, as in "f" above)

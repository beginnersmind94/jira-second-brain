# NXT-3001 — Edit Inspections - Create Screen and Heading

## Description
*User Story*
As an Inspections user, I have a screen for a created Inspection with a header that details any pertinent information about the Inspection.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=inspections_approved_layout&id=f1misp&g=1|https://nqyc9v.axshare.com/start.html#p=inspections_approved_layout&id=f1misp&g=1]

*Requirements*

* When an Inspection is created, the user is taken to the Edit Inspection screen
** Can also be accessed if user accesses it while assigned to them and in an In Progress status
* There is a header that contains information based on the information the Inspection was created from
** Site
** Template
** Date
** Status
** Points
* Info from the header is placed along the bottom once the user scrolls past the header
* There are two settings for the user with regard to displaying questions, dependent on the Show All Questions toggle
** Enabled - The entire Inspection is displayed on a single page
** Disabled - Questions appear one at a time with a progress bar on the header
** The actual functionality is covered in a separate story - for the purposes of this story, the entire Inspection should be shown

## Acceptance Criteria
* Edit Inspections screen created and displaying proper information based on the Inspection
* Edit Inspections screen can be accessed by creating a new Inspection
* Edit Inspections screen can be accessed by clicking an In Progress Inspection

# NXT-4049 — IM - Edit - Item Information

## Description
*User Story*

As a Item Management Super User I can edit general item information. 

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7]

!image-20200526-125811.png|width=1085,height=863!



*Requirements*

* Header
** Option to ‘Edit’ 
* Slides the category from the right to left as a ‘drawer’ showing the information that can be edited and added / removed
** If clicking out of the popup and information is entered and not saved, show message
*** You have unsaved information, do you want to 'save' or ‘continue without saving’?
**** Save - Save the information entered and close the record
**** Continue without saving - Close the item
* If Short Name is entered, short name will be checked and shown
* Item Name can be changed, however it needs to be unique
** Duplicate check must be performed if Item Name is changed
** If a duplicate exists display
*** “Item Name” already exists. Please enter a unique name.
**** Item Name replaced by the actual item name
** This happens when the value is entered, not upon save.
* Allow the user to modify all information *EXCEPT* datasource.

-Next and back navigate between the steps and ‘Save and close’ will save and close the record and return to the view-

* Save & Close - Save the information entered and close the record
* Cancel - Close the item without saving

## Acceptance Criteria
* Item information from creation needs to show up here
* If you click 'save' the information is saved and closing the drawer
* If cancel, then revert back to the state before user clicked 'Edit'

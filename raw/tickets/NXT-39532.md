# NXT-39532 — IM - Menu Items - Menu Item Ready flag

## Description
As a Menu Planner I want a Menu Item Ready or Production Ready flag so I can save all various Menu and POS items without selecting Production attributes for every one of them.

*Requirements*

* Create a flag for “Menu Item Ready” which considers the following fields to be “ready”
** Menu Item Category
** Leftover Category

## Acceptance Criteria
* Create a flag for “Menu Item Ready”

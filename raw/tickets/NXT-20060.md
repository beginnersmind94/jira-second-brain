# NXT-20060 — INV - Vendor Detail - Address tab - Address card

## Description
As an Inventory manager I want a Vendor Address tab so I can manage contact information for the vendor.

The second tab will be the *Address* tab and will contain:

* Address card (this story)
** this contains the attributes defined under the vendor address section
** the map section can be done later
* Vendor Contacts cards ([NXT-21387|https://primeroedgenext.atlassian.net/browse/NXT-21387])
** this contains the attributes defined as Contacts are added for a vendor
** add a new card for each new contact 
** *Remove the limitation on maximum 3 contacts

*Requirements*

* Display a Vendor Address card at the top of the Address tab
** **Note:* the Vendor Map may not be done; expand the Address card to full-width
* Display the labels in the following order
** Phone Number - *mandatory*
** Email Address
** Contact Name
** Website
** Address
* On click of Edit icon, enable in-line editing by 
** Phone Number
*** this should be an actual phone number field, which automatically formats the input to a phone number format like 281-453-8500
** Email Address
*** Provide an icon to add a second email field
**** If pressed, display a second email field under the first
**** relabel the first Email Address as ‘Primary Email Address’
**** display the second email as ‘Secondary’ Email Address'
**** display a delete icon next to the secondary email address as long as it is empty
*** Contact Name
*** Website
*** Address 
**** Address is *optional*, but if user enters anything into one of the address fields, then we need to ensure a full address is provided and therefore the below fields are mandatory
**** For Address, split the field into multiple fields which comprise the address
***** Address Line 1 - *mandatory*
***** Address Line 2
***** City - *mandatory*
***** State - *mandatory*
***** Zip Code - *mandatory*
** Replace the Edit icon with Save & Cancel buttons
* On click of Save or Cancel
** Turn all fields back into labels
** Consolidate the different Address related fields back into a single address
** Replace the Save & Cancel buttons with the Edit icon

# NXT-3410 — Template Hub - Actions - Update History (Global Templates)

## Description
*User Story*
As an Inspections user, I have an update history slide out that tracks updates for global templates so I can know that any official inspections are up to date.

*Requirements*
* There is an Update History action available for all Global Templates
** This is only available for Global Templates in the Global Templates screen, not for Global Templates that have been pulled into the district's Templates
* Clicking Update History brings up a slide out
** Behaves similar to Activity
* The Global Template's name, any labels, and description are in the slide out's header
* There is a list of all updates made to the Global Template
** Each line has the following:
*** Date - The date the update was made in the system
**** System Generated
*** Reason - States why the update was made
**** Set by Cybersoft when the update is made
*** Description - Lists out what was changed at a high level
**** Set by Cybersoft when the update is made
* Users cannot interact with the Update History - all parts are read-only

## Acceptance Criteria
Update History is present for Global Templates and properly displaying the expected information

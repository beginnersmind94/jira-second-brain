# NXT-52746 — DC Import – Additional Code Mapping (FD - 250922)

## Description
As a SchoolCafé Admin I want to be able to do code mapping for more fields for DC Import so that I can map values to our codes

*Notes:*

* We already do code mapping for DC Import’s DC Type field
* We need to add it to more fields

*Requirements*:

* For Add New and for Edit screens, when a user selects DC Import, do the following:
* Add a code mapping screen for the following fields:
** SNAP
** TANF
** DCMA
** DCMAR
** FDPIR
** FOSTER
** Note: DC Type is already done
* Use the internal codes from the database for Internal values for each one

## Acceptance Criteria
* For each field, user is able to do code mapping

# NXT-4267 — AcctMgt - Add/Edit Patrons - In-line Validations

## Description
*User Story*

As a district admin I want to be sure there are no duplicates for ID numbers (aside from certain circumstances) to avoid issues, and that emails are in a proper format.

*Requirements*

* Add in-line validation for the following fields:
** Add Patron (New Student)
*** ID Number
**** ID Number must be unique per:
***** Patron Type (Student/Adult)
***** District
***** Example: ID #12345 can only exist 1 time for a Student in district ABC. An adult with ID #12345 could also exist 1 time in the district.
**** State ID Number
***** Same as ID Number.
**** PIN #
***** Unique across all patron types, per district.
** Editing General Info Panel
*** Same criteria as the Add Patron (New Student)
** Adding/Editing Contact Information
*** Email must be in a valid x@y.z format.

ADDITION 7/21/20 AFTER DISCUSSION

* Validation for:
** Special characters should also be validated against.
** ID #s are Alphanumeric only.
** Appropriate error messaging about an invalid ID number.

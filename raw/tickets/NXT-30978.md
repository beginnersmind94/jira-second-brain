# NXT-30978 — IM - Variety Menu Item Nutrition Disclaimer

## Description
As a Nutritionist, when I add Menu Items to my Variety Menu item, the system is taking all the menu item nutrients and giving an average.

*Note:* we only average the nutrients; not the contributions

*Requirement:* 

* Show a disclaimer “Nutrient display on Variety Menu Items use an average of nutritional values of all menu items.”

## Acceptance Criteria
Display disclaimer on all variety menu items

# NXT-2961 — System - Organization - Regions - Manage Regions [Duplicate]

## Description
As an administrator I want the ability to setup areas and sub-regions within my organization (school district) so that I can structure how access is granted and information/reporting is displayed.

Each school district should have the parent district region, but have the ability to create areas and sub-regions that consist of groups of sites.

Order of organization levels: Region > Area > Sub-Region

The existing functionality can be accessed at: System > Sites and Users > Regions. See attached document for screen references.

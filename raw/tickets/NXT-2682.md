# NXT-2682 — Employee List Report 

## Description
The Employee List is a report of compiled employee details. 

* ID Number, Name, Site, Hire Date, CEU Level, Job Title, Phone, Email 
* Filter by District (at central realm), area, site code, site, or hire date 
* download/print report options

## Acceptance Criteria
Employee list report shows emp details based on criteria selected 
with the option to download in chosen format or print

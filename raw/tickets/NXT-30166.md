# NXT-30166 — IM - Items - View Item - Product Details tab - Item Substitution 

## Description
As a District Admin I want a Product Details tab so I can configure expanded item details such as Substitutions.

*Requirements*

* Display and populate the Product Details tab for all created items
** This tab is always displayed after final item creation (after toggles are saved)
** This tab will never be disabled
** Display this tab at the end
* Provide the Substitutions card
** Separate story for updating Substitutions to consider updated unit info configuration

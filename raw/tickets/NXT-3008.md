# NXT-3008 — Reports - Custom Report Builder

## Description
As a TW user, I want to choose data fields, criteria, and layout of the report for compliance auditing, event tracking, and other essential administrative operations. 

* Provide ‘Templates’ of current reports - Training Progress Template will have the current columns in which the user can rearrange, add to, delete, to build their own report 
* Save and name report ‘templates’
* Choose data attributes, style  
* number of columns, rows 
* ‘Save As’ templates allows user to name the report and reference for quick review 
* Preview report, download, print options

!image-20200116-174045.png|width=699,height=431!

* Each report has a Report Name, ‘Generated on’ timestamp (Date and time), by who, criteria selected

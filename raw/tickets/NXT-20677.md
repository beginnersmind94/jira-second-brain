# NXT-20677 — INV - Inventory - Additions - Create Addition - Items (Step 2) - Items List

## Description
As a Site Manager I want an items list so I can view the items I am adding to inventory and provide additional input.

Continued from [https://primeroedgenext.atlassian.net/browse/NXT-20676|https://primeroedgenext.atlassian.net/browse/NXT-20676|smart-link] 

*Requirements*

* As items are added to the list via the Add Items tool they will load in the Item List
* There are 2 main sections on this screen:
** Add Item control ([NXT-20676|https://primeroedgenext.atlassian.net/browse/NXT-20676])
*** the tool which is used to find and select items to add them to the list
** Item list (this story)
*** the list that contains the items which have been added to the list
*** this will also include the *Lot Number sub-grid* for each item in the list (the same thing used on Receipts)
* Item List section
** This will be the lower section on the page
** Display a standard grid which is unpopulated by default
** Display the following columns:
*** Item #
*** Item Description
*** Quantity
**** editable
**** contains the value which was entered when the item was added
*** Price
**** editable
**** loads with current price
****# Last Received Price
****# Standard Price (if no LRP exists)
*** Action
**** Comments icon + popup
**** Delete icon
***** prompt on delete: “Are you sure you want to remove this item?”
** Display the Lot Number / Date on Product sub-grid for each item in the list
*** this is the same control used on Receipts
**** Date on Product (date picker)
**** Date Type (dropdown)
**** Lot Number (textbox)
**** Quantity
*** use the same validations
**** inner quantity cannot exceed outer quantity
* Display the stepper Next button once at least one item is added

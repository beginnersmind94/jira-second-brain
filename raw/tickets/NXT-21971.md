# NXT-21971 — Add Multi-Delete to New Images

## Description
As an enhancement, we need to be able to delete multiple unwanted images in the Food Training website

*Application*: Face and Food Detection Website

*Background:*

* We are adding in the functionality to delete an image in story Add Delete to New Images
We also need to be able to delete multiple images at one time
After you can delete one image, then do this story to delete multiple

 

*Flow:*

* Browse to the Food Detection Settings page and then the New Images Page or the Trained Images page
On any given image in the grid, click the checkbox next to the image
Click on the Delete Selected Images button

 

*Requirements*:

* These requirements apply to both New Images and Trained Images, as it is assumed you are using the same page for both
Next to each image, add a checkbox
In the section on the top right where you are normally putting buttons, add a button called Delete Selected
 
* If click the delete button, do the delete functionality for each image selected
** See Delete Images story to know how to delete
Do it here exactly as you do it there, but for many images, but repeated here:
Delete the physical image from the system
Do not delete the database record of the image, but mark its status as Deleted
Record the date/time stamp of the deletion in the DB record of the image
* After it is done deleting, ensure the image no longer shows up in the grid
* Validator
** If they didn’t select anything before clicking, show a message “Select an Image”

 

*Mockup*:

* None

## Acceptance Criteria
* Next to each image, added a checkbox
In the section on the top right where you are normally putting buttons, added a button called Delete Selected
If the image is linked to a Model, then the check box is disabled (applies to Trained Images only)
* On button click, it does the following:
** Deletes the physical image from the system
Does not delete the database record of the image, but mark its status as Deleted
Records the date/time stamp of the deletion in the DB record of the image
If they didn’t select anything before clicking, show a message “Select an Image”

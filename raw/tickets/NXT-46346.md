# NXT-46346 — IM - Menu Items - Menu Item List Export

## Description
As a district administrator or menu planner I want a basic Menu Items list export so I can generate a list report of all the menu items I have filtered on the list page.

!Menu Items List w Reports-20240111-134638.png|width=83.33333333333334%!

## Acceptance Criteria
* Add the ‘Menu Item List Export’ to the bottom of the Reports dropdown on the Menu Items list page
* Upon generating the Menu Item List Export, an excel file will be generated with the full list of menu item contained in the grid
* The export should consider filters applied to the header and any grid header searches
* The excel export should always include every grid column (regardless of Manage Views selections)
** match the column sequence of the page grid

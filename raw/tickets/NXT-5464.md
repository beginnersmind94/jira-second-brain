# NXT-5464 — Data Exchange - Scheduler

## Description
*User Story*
As a PrimeroEdge developer, I need to make system changes that facilitate end user scheduling of both imports and exports (both file based and api based.)

*Requirements*

* Schedule frequency options
** Daily
** Every _ Hours
* logs for every successful run must be displayed.
* If the scheduler is still running, the status must be displayed so that the user cannot trigger it again or any new changes to the scheduler, must be not implemented immediately  but must have an option to run after the current one is successfully completed.

## Acceptance Criteria
* Import runs as scheduled
* data is successfully imported.
* data is successfully exported.

# NXT-6917 — INV - Vendor - Add Contract Details (2nd step) - Delivery Schedule

## Description
As a Procurement Specialist I want to be able to add contract details to a vendor contract so that I can specify the item level requirements and attributes before adding items.

Second step of Add Contract stepper 

!image-20201027-124133.png|width=960,height=699!

Continuation of [Story 6425|https://primeroedgenext.atlassian.net/browse/NXT-6425]

*Requirements*:

* Delivery Schedules (this will be removed from the main vendor configuration and instead defined here)
** Main vendor level delivery schedule for selecting the days of the week that the vendor delivers as part of the contract.
*** Require at least one day to be selected
**** “Please select at least one delivery day.”
** An optional field for inputting a Time for delivery HH:MM AM/PM
*** Use inline time picker  
*** A toggle for enabling site level delivery schedules 
*** disabled by default
**** if this toggle is not enabled, there will be no site specific delivery schedules (all sites will use the main vendor delivery schedule)
*** when ‘Site Level Configuration’ toggle is selected, a site list will be displayed below with all sites
** If ‘Site Level Delivery Schedule’ is enabled, display a ‘Apply to All Sites’ button
*** Apply to All Sites will copy the main vendor delivery schedule to every site
*** If there are site delivery schedules defined and the user once again clicks ‘Apply to All Sites’, provide a prompt to the user to confirm that they are overwriting the existing delivery schedules
**** “This will overwrite the existing site level delivery schedules. Do you want to proceed?”
** A list of all sites within the region/district with days of the week they will receive from the vendor
*** Site list columns:
**** Site Code
**** Site Name
**** Delivery Schedule
**** Delivery time
***** Use time picker control
**** *Sort by Site Name, alphabetically ascending
*** disable the selection of days that are not selected as part of the main vendor delivery schedule
*** **Note: would display separated by Site Type, upon confirmation of Site Type configuration

Complete button is provided on this step to save the contract.

## Acceptance Criteria
User should be able to save the vendor contract after all mandatory fields are entered.

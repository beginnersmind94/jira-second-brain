# NXT-2062 — INV - Orders - Production Ordering - Confirm Orders 

## Description
As an Ordering Manager I want an Production Order Confirmation screen to review the Production Orders I have created.

*Requirements*

* Load the Confirm Orders screen based on the orders generated from Production Quantities
** after click of ‘Place Orders’ from Cart
* Orders should load according to typical order creation logic
* Add Item functionality should work as usual
** this will add non-production items to the production order
* Set Entry Method upon order generation
** all these orders should have ‘Production’ as Entry Method type
*** orders which are created using ‘Standard Ordering’ page have ‘Standard’ entry method
* Enable Back button (in [NXT-27114|https://primeroedgenext.atlassian.net/browse/NXT-27114])
* Properly note non-production items on the production order

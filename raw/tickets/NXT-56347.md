# NXT-56347 — FH - Menus - Handling for SFSP Meal Patterns

## Description
Summer is coming, and SFSP menu boards will likely be posted. Let’s handle them as well before we get tickets about it.

*requirements*

* Show all items with a serving size since it’s for ‘all ages’

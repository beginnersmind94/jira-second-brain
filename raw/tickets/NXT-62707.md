# NXT-62707 — INV - Distribution - Assignment - Unassign Distribution

## Description
As a Distribution Manager I would like the ability to un-assign a Distribution so I can open that day up for more orders without having to manually approve all of them.

!Assign - Un-Assign-20250527-220858.png|width=91.66666666666666%,alt="Assign - Un-Assign-20250527-220858.png"!

## Acceptance Criteria
* Provide an Unassign functionality for distributions in Pending Assignment status
** Add a back icon button in the Quantity Ordered column (SVG attached)
*** prompt user on click: “This would discard the current assignment and revert this distribution to Unassigned. Are you sure?”
* Upon Unassign, revert the distribution transaction
* Provide a Clear Assignment button in the Quantity Assigned column (pending item from earlier Assignment mockups)
** prompt user on click: “Are you sure you want to clear all quantities and reset the assignment?”

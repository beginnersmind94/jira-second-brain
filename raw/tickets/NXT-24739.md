# NXT-24739 — POS - Dynamic Menus - Dropped Tray review reason?

## Description
As a user I need a way to show that a student had a dropped tray.

* Add a Review Reason ‘Dropped Tray’
* When selected the transaction will be marked as dropped, 
* Deferred - The Production Counts will be updated for the items in the dropped tray

## Acceptance Criteria
User has a way to indicate that a previously processed transaction is now a dropped tray. 

Users can select ‘Dropped Tray’ as the reason from the Transaction Screen in Menu Service

Users can select ‘Dropped Tray’ as the reason from the Transaction List

Deferred -  When 'Dropped Tray is selected the Production Counts are set as a dropped tray and the served counts for the items are adjusted.

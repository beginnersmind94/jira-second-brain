*User Story*

As a Item Management Super User, I can images and documents while adding a new item

*Requirements*

* A section of the page that allows the user to drag and drop an image or a document, or browse for their document
** “Drop your images here, or browse”
** 'Supports JPG, PNG, & PDF Documents

## Acceptance Criteria
*Create as a non functional field, new stories will create functionality
* Allow the user to drag and drop their files from their computer to the browser window
* When clicking 'browse', allow the user to use the file system to find the file
* Allow multiple file selection
* Support JPG, PNG, and PDF files in thumbnail preview view initially

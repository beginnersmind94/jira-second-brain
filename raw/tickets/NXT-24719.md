# NXT-24719 — Acc. - End of Day - Add page with search

## Description
As Agatha the Site Manager I want to see how my site is performing - have we completed our daily tasks, or are some still pending? This will help us focus our time on making sure the essential tasks are completed on a daily basis.

*mockup*

[https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|smart-link] 

*requirements*

* Add the ‘End of Day’ page.
* Add role permissions to view/edit the End of Day page.
* Add a search/filter area to the End of Day page:
** Site (dropdown, single select, site code + site name, only shows the sites the user has access to)
** Date (single date picker, default today’s date)
** Status (dropdown, options are All, Not Started, Not Done, Complete)
* User can search
* User can reset their search options

## Acceptance Criteria
* End of Day page added.

** View, Edit
* Search/filter section added to the page.
** Site

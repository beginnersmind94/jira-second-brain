# NXT-3787 — AcctMgt - Patron Detailed - Rosters

## Description
*User Story*

As a district admin I want to see what roster(s) a patron is on, and interact with those selections.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1]

*Requirements*

* Add the Rosters to the General Information panel of a Patron.
** Show the name of each roster that the account is currently added to.
* While editing:
** User can remove the student from any of their rosters.
*** Dismiss the chip and save.
** User can add the student to a roster.
*** New chips can be added, auto-complete/suggest while typing.
*** -The available rosters should be constrained to the site the student is enrolled at.-
**** -Example: Student is at Site 123, only Rosters that include Site 123 should be available for selection.-
* Save functionality on the Edit drawer should update the roster membership accordingly, be it an add or delete.
* General Info panel should update to show changes in roster.

## Acceptance Criteria
Roster information shows in the Patron > General Info panel.
Patron can be added/removed to rosters via the General Panel's Edit drawer.
Multi-language support added.

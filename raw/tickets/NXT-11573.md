# NXT-11573 —  Integration - Enable support for fixed-width file import

## Description
Currently, Perseus supports import of “csv” files. There is a need to enable support for the import of fixed-width files.

## Acceptance Criteria
* Customers shall have the ability to import fixed-width files

* For a fixed-width file upload, the user will follow the same steps as a CSV file up until the point where the user selects "FixedWidth" from the "Delimiter" dropdown on the "File Upload" step.
* By default, a “File Layout” step shall be present and disabled.  The “File Layout” step shall be positioned between “File Upload” and “Field “Mapping”
* If the user selects "FixedWidth" from the "Delimiter" dropdown, a “File Layout” stepper shall be enabled and the user, upon choosing Next, shall be presented with an area where the layout of the fixed-width file can be defined.  The area shall allow the user to define the name of a field and the size of the field.  This area will have the appearance defined by the UX team in the mockup linked to this story.
* If the user selects anything other than “Fixed Width”, the “File Layout” step shall be disabled and the user shall skip the “File Layout” step and be brought to the step after it upon choosing “Next”.
* The fields shall be sequenced in the order in which they are added. For each field, a display-only "Start Index" field shall be displayed to help the user.  The first field added shall have a start index of 1.  All other fields shall have their "Start Index" calculated based on the sizes of the fields that preceded them.
* Once the user has completed defining the layout of the file and presses "Next" on the "File Upload" step, the user shall be taken to the "Field Mapping" step and the field names from the fixed-width file shall be populated in the "Source" column.
* Once the "Source" column has been populated, the "Field Mapping" shall behave as it does for all other file types and the customer shall have the ability to map each field of the source file to a field in Perseus
* At import time, the fixed-width file layout definition shall be used to import the file into its destination datastore in Perseus.

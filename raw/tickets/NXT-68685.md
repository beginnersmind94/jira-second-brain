# NXT-68685 — UI Components Restructure - Package Upgrade

## Description
*Comprehensive UI Component Library Upgrade – React 19, MUI v7, Kendo v13, TS 5.4 & Storybook 9 Migration and etc*

h3. *Ticket Description (10 lines):*

# Upgrade entire UI component library to latest React 19 with JSX transform and removal of legacy imports.
# Migrate Material-UI v4 to MUI v7 including theme restructure, removal of {{makeStyles}}, and icon path updates.
# Upgrade React Router from v5 to v6 with new {{<Routes>}}, {{element}} API, and {{useBlocker}} for navigation guards.
# Update Kendo UI packages to v13 with new import paths, Grid/TreeList API changes, and TypeScript improvements.
# Upgrade TypeScript from 3.9 → 5.4 with strict mode updates and new compiler settings.
# Storybook migration from v6 → v9 including new {{main.ts}}, {{preview.ts}}, and CSF3 story format.
# Update additional dependencies: Chart.js v4, Axios 1.7, Formik 2.4, Yup 1.3, React-PDF v8, date-fns v3, Dropzone v14.
# Migrate styling approach to {{sx}} and {{styled()}} across 50+ components replacing deprecated {{makeStyles}}.
# Apply Node 24, Dart Sass, Babel config updates, and modern build pipeline enhancements.
# Perform full regression testing: routing, grids, theming, forms, exports, performance benchmarks, rollback planning.

Update all the packages of UI components to latest version

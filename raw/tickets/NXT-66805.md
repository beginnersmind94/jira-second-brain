# NXT-66805 — GC - Portal - Usage of Setup Values / Validations during CRUD operations

## Description
_(none)_

## Acceptance Criteria
Navigation: GC Portal - Search

Changes Required:

The user should have following options when they update the record:

USDA BF:

Item Name - check if the name already exists in BF_consolidated table - if yes, show the message as “Item Name already exists”, if not, allow saving the record 

Item Category - List it as searchable dropdown so that users can pick from the existing list of categories
Brand - List it as searchable dropdown list (select * from Brands where datasourceid=2)

Manufacturer - List it as searchable dropdown list (select * from Manufacturer where datasourceid=2)

Storage Category - Static List (Dry, Cooler, Freezer)

Valuation Category - USDA Foods, Commodity, Non-Commodity

Serving Size - if either of the value is NULL, show the message as “Missing Serving Size”, do not allow saving the record, if details are available, then save the record

Nutrient - of either of the mandatory nutrient information is blank - show the message as “Missing Mandatory Nutrient Information” - do not allow saving. If all the mandatory information is available, then allow saving the record



Cybersoft Item:

Item Name - check if the name already exists in BF_consolidated table - if yes, show the message as “Item Name already exists”, if not, allow saving the record 

Item Category - List it as searchable dropdown so that users can pick from the existing list of categories
Brand - List it as searchable dropdown list (same as in Search Parameter)

Manufacturer - List it as searchable dropdown list (same as in Search Parameter)

Storage Category - Static List (Dry, Cooler, Freezer)

Valuation Category - USDA Foods, Commodity, Non-Commodity

Serving Size - if either of the value is NULL, show the message as “Missing Serving Size”, do not allow saving the record, if details are available, then save the record

Nutrient - of either of the mandatory nutrient information is blank - show the message as “Missing Mandatory Nutrient Information” - do not allow saving. If all the mandatory information is available, then allow saving the record

Meal Credit - List the allowed values as standard dropdown, user can select one of the values , null is allowed 
Allergen - merge Crustacean/Shell Fish as one row - they mean same allergen

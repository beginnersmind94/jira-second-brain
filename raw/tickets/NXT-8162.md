# NXT-8162 — Direct Certification - Extensions - Block extensions leading to extensions

## Description
As a product owner I want our software to comply with regulations and performing a DC extension off of an extension is not allowed.

N/A

*requirements*

* Prevent extended students being used to extend on to other students.
** Scenario: Student A is DC, and is extended to Student B. Student B should not extend on to anyone.

## Acceptance Criteria
System will not allow an Extended student to be used as a source of approval for another student. See the scenario above.

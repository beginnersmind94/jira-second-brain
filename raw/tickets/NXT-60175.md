# NXT-60175 — IM - Items - Units - Disabling historical Menu Servings

## Description
As a Menu Planner I want to be able to disable certain menu servings even if I have used them in the past, so that I can control which servings are assigned in the future.

## Acceptance Criteria
* Allow menu servings to be disabled on the Update Menu Servings sub-step of the Units card
* Manage the historical data requirements of Menu Planning/Production data
** ensure that archived/historical versions of menu servings are accessible for older data

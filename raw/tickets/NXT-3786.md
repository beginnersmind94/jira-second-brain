# NXT-3786 — AcctMgt - Patron Detailed - Correspondence

## Description
*User Story*

As a district admin I want to see and send correspondence to a particular patron account.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=14|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=14]

*Requirements*

* Add a Correspondence panel to the Patron Detailed view.
** We want to use this page to track all outgoing correspondence on the account - any letters generated by the system will need to be linked to an account, and viewable from here.
** Show the top 3 pieces of correspondence for the current academic year in the panel. Grid information outlined below.
*** Grab all letter types for the current academic year, sort by send date descending, show the top 3.
** Additional correspondence is buried in the View More drawer.
** We need filters on the View More drawer to sort correspondence by:
*** Academic Year
**** Default: Current academic year
*** Letter Type
**** Default: --SELECT--
** Add an Info icon to the panel outlining that the panel is only showing the current year’s correspondence.
*** Text for the icon: Correspondence for prior academic years is available under View More.
** Correspondence Grid Columns:
*** Letter Name
**** The Letter Type. We may want to show the Template Name as well.
*** Notification Type
**** Whether the notification was dispatched via email / mail.
*** Generated Date
**** The date the record was created.
*** Generated By
**** The user who created the record.
*** View
**** Placeholder icon for now.
**** Use a document icon. 
* API is available in Administration Module.
* [http://primero20qa.centralus.cloudapp.azure.com/administrationservice/swagger/index.html|http://primero20qa.centralus.cloudapp.azure.com/administrationservice/swagger/index.html]
* [http://primero20dev.centralus.cloudapp.azure.com/administrationservice/swagger/index.html|http://primero20dev.centralus.cloudapp.azure.com/administrationservice/swagger/index.html]

## Acceptance Criteria
Correspondence panel added.
Panel shows top 3 letters for the current year.
View More drawer available.
Drawer has filters for academic year and letter type, grid loads appropriately based on selections.
User can see all prior correspondence in the grid.

# NXT-28382 — INV - Withdrawals - Create Production Withdrawal - Confirm (Step 3)

## Description
As an Inventory Manager I want the confirmation screen for the Production Withdrawal to show me an overview of all items from all menu lines I selected so I can create a single withdrawal transaction.

*Requirements*

* Update the Confirm step of the Create Withdrawal to handle multiple production records on the screen
* Provide header selections for Step 3
** Transaction Date & Time
*** defaults to the start of meal serving time for the latest selected menu line
** Withdrawal Reason
*** defaults to Usage (need to create Usage as standard reason)
** Comments
* On step 3, load all the items from all selected menu lines from step 2
** Do not show the menu lines in the UI for step 3
* List all items with consolidated quantities for all selected menu lines
** Show Quantity on Hand so user can see total vs. QOH
* Provide the Comments icon for items on this step (instead of Step 2)
* Provide a single Confirm button
** this will create a single Withdrawal transaction
* Show error if withdrawal quantity exceeds QOH
** highlight the affected items 
** show an error 'Withdrawal Quantity exceeds available quantity for highlighted items. Please return to Step 2 for corrections.”
** Enhancement: when user returns to step 2, continue highlighting the affected items
** Reload QOH when user returns to step 2

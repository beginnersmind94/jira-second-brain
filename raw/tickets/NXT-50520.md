# NXT-50520 — INV - Physical Inventory - Persistence Behavior

## Description
As an Inventory user I want the list pages to remember my filter criteria and results paging/sorting when I navigate back to the page from a sub-page, so I don’t have to reset my filter criteria each time over and over.



!image-20240514-194222.png|width=75%,alt="image-20240514-194222.png"!

## Acceptance Criteria
* Add persistence to the Physical Inventory list page (Search tab) when navigating back from Physical Inventory Details
** most Inventory pages already have some form of a dedicated back button
* Make sure every page has necessary confirmation prompt for unsaved changes when navigating back or away from page
** reproduce the Orders & Receipts approach on other pages as needed
* Change the Physical Inventory Details to *not* open in a new tab like it does currently

# NXT-48685 — OpenID - Profile Settings

## Description
As a SchoolCafe user, I would like to be able to change my password in my profile only if I am using SchoolCafe Authentication so that I don’t get confused trying to set a password that will never be used.

*Notes*:

* Authentication Type available to the user are determined on the District's Security tab
* A user can only choose the type of authentication available to them by their home district

## Acceptance Criteria
* Edit buttons only affect the tabs they are on
* Change Password tab name changed
* Edit and View modes added to Authentication tab
* Authentication Method field shows the type selected by the user in view mode and defaults to this selection in Edit mode
* Password field has asterisks in View mode and defaults to password in Edit mode
* User is able to change their password type to any and only ones chosen by the district
* If the user chooses SchoolCafe Auth, they must set a valid password
* If they select any other auth method, they cannot set passwords
* The login screen uses this type of authentication selected by the user when they try to login

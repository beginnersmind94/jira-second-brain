# NXT-1966 — MP - Multi-Line Analysis (UI)

## Description
When users assign multiple menu lines to a single site for a single meal service, there is a need to analyse the nutrient and component information between the two lines.

This ability should be provided to the district to be able to analyse a full week at one time, as in cycle menus.

The need will be to search/filter for and select multiple cycles to get analysed together.
a.) I expect a list of menu cycles to appear on the page, with a method to select the menu cycles that I want.
b.) I expect to see those selected cycles listed together with a button or icon to select to perform the analysis.
c.) I expect the resulting analysis page to give me the quick view and the report options as in the standard menu cycle analysis.
d.) The analysis results should be a combination of the selected menus, with the serving group and menu item counts, category options, etc., taken into account.

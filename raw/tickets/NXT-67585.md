# NXT-67585 — IM - Bulk Item - Images and Docs Section

## Description
As a District Manager handling adding all my Images and Docs for my items, I would for a way to be able to upload them through the item bulk tool and just go item by item and update the images and docs that belong to each item.

## Acceptance Criteria
* Add an Images & Docs column, at the end of the grid (Onboarding and Update)
** This also includes edit sections 
* Show numeric chips to indicate how many images/docs are uploaded for that item (Onboarding and Update)
* Clicking the numerical chip will open the Images & Documents slide-out for that item (Onboarding and Update)
* Provide slide out for images and docs (Similar to Item details page)
** Open the slideout from the overview page
** Allow all image and docs uploads as well as designations to be done within the images docs utility
** Any updates should be reflected on the overview with the new count based on what was added
* Percentages will be calculated based on items that have any images or docs greater than 0

# NXT-36322 — Error Reporting - Send Error Email Data Exchange Team

## Description
As a Cybersoft employee working in development, I want to be emailed all the errors so that I can quickly see if I need to address a problem.

*Flow:*

* An error occurs
* An email is sent with the error details

 

*Notes:*

* This story is part of the stories to expose errors to support staff and follows the story to create an error handling email account

*Assumptions:*

* It is assumed that the story to create the error reporting email has been completed
* It is assumed that we have a common way of handling error reporting

 

*Requirements*:

* Everywhere we have code to save an error to the ADF Application Insights , add additional code to email the error the new error handling email address created in the previous story
*  Email should include the relevant details:
** What environment
** What District
** What module (if you can)
** -What page-
** Error Type (Web, API, etc.)
** Error Code
** Long description (the specific error)
** Date / Time
** User account logged in (if exists)
** Anything else that would be useful
* Email title should be like this:
** Environment district name error type
** Example: Prod – Mercer County – Web Error
* Permissions: no additional requirements necessary

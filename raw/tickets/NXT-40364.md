# NXT-40364 — Custom Reports – Log History 

## Description
We need to create a history table to track changes to the Custom Reports so that end users can monitor changes over time to their reports.

*Notes:*

* No front end on this story
* This is the backend story to support the View Report History story
* This table will be updated when a user does one of the following: Adds, edits or deletes a Custom Report
* This table will be read from on the View Report History drawer and also from the Custom Reports Add New Configuration screen in the future

*Requirements*:

* Create a History table for Custom Reports
* 
* At a minimum, the table needs the following columns:
* CustomReportID
** linked to Custom Reports table unique ID so that you know which report these changes are for
* Event Type
** This is which action caused the record to be inserted
** Values:
** Add
** Edit
** Delete
* Description
** Value:
** The text of the values from the Add and Edit are coded like this:
** Format: FieldLabel=Value; FieldLabel=Value; FieldLabel=Value
** Example text:
** Share=No; Email=Yes; Format=PDF; etc.
** Note: this can get long if they add a bunch of email addresses and users, so make sure table column can handle a lot of text
** 
** The text of the Delete comes from Description field directly by user on that screen
* User
** The user account logged in that made the change
* Date/Time
** The date / time stamp of the change
* 
* *APIs Needed*
* At a minimum, you will need the following APIs / Stored Procedures:
* Receive the data from Add New, Edit and Delete Report and insert it
* Don’t need an update for this table
* Provide all the rows with all the columns for that given Report for a given report id
** Used on View Report History drawer
* Provide all the rows with all the columns for all Reports
** Used on Add New Configuration screen when creating a custom report for Custom Report History

## Acceptance Criteria
* All changes are logged to the history table including add, edit and delete functions
* Changes are visible in the History tab

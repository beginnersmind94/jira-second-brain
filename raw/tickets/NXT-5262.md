# NXT-5262 — Direct Certification - Manual Entry - Documents

## Description
*User Story*

As a district admin I want to add the supporting / reference documents that show why I’m manually importing a list of students for direct certification.

*Requirements*

* Add the following functionality to the direct certification ‘Add’ page.
* User can enter a comment.
** Comments are 250 characters max.
* User can see and view these comment inside of DC File Details later in the Documents section.



*note: 3/9/11*

* Reqs for supporting document upload have been removed from this story.

## Acceptance Criteria
User can add a comment also - 250 character max.
User can see comments when viewing the details of a DC File.

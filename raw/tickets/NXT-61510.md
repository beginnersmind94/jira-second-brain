# NXT-61510 — INV - Distribution - Shipping - Revert Shipping

## Description
As a Distribution Manager I want the ability to restart shipping entirely so I can reset my distribution Shipping data to the current contract Shipping Group & Delivery Group configuration.

h3. Shipping with Revert Shipping button

!Shipping - Landing-20250602-144755.png|width=91.66666666666667%,alt="Shipping - Landing-20250602-144755.png"!

## Acceptance Criteria
* Provide a Revert Shipping function
** only in Shipping in Progress status
** Please use color B70000 on button
*** use this red for all Restart/Rollback buttons
* Prompt user: “This will revert the Distribution to Picking and will clear the current Shipping data. Are you sure you want to revert shipping?”
* Reset all Shipping data to re-build shipments according to current configuration
** Revert from Shipping In Progress to Picking Complete
** on click of Start Shipping, look at current config data to build shipments
* All Restart/Rollback buttons will be locked to a functional permission

# NXT-29736 — IM - Variety Menu Item - Menu Item Nutrients average of all menu items - API Only

## Description
As a user, When I add a menu item to variety menu item, I would like to see the average nutrition of all my menu items on the nutrition tab. 

Do not allow menu items to be removed once added and saved onto the variety menu item. If user needs to make changes to the menu items, they must copy the variety menu item and can make the changes accordingly.

## Acceptance Criteria
Menu Items used in the creation of the Variety Menu Item will be used and averaged. (i.e. if there are 3 different cereals for a Variety Cereal Menu Item, and those calories are 90, 120, and 150 - the nutrient analysis will display 120 calories … 90 + 120 + 150 = 360 / 3 = 120)

Same applies to all other nutrients available. 

Do not allow menu items to be removed once added and saved onto the variety menu item. If user needs to make changes to the menu items, they must copy the variety menu item and can make the changes accordingly.

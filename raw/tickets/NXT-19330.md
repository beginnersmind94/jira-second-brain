# NXT-19330 — INV - Apply Period check to transactions - Physical Inventory

## Description
As an Inventory user I want transactional checks for Period Status so the system can alert me if I’m trying to make adjustments to transactions in a closed period.

*Requirements*

* Enforce period rules on transactions
** whenever a period is closed, all transactions should be dated _after_ the end date of the most recently closed period
*** ex: August 2021 is closed, and is the most recent closed period. Any further receipts should be dated in September or later
*** ex: August 2021 is *Open*, but September 2021 is closed. Still, all transactions should be dated October or later
* Message: “The Period for the selected date is Closed. Please select a valid date.”
* Transaction List:
** Completing Physical Inventory
** Modifying Physical Inventory

## Acceptance Criteria
* Apply period check to the list of transactions
* Prevent these actions if the period is closed

# NXT-18048 — INV - Orders - Manage Orders - Receipts - Quick Receipt - Save as Incomplete

## Description
As a site manager I want the ability to save or complete a Quick Receipt so I can either received the product into my inventory or save the Quick Receipt for later.

This story is related to how the new ‘Quick Receipts’ are saved and presented throughout the rest of the Orders category.

*Requirements*

* On the Receive step of the Quick Receipt functionality, provide the Save as Incomplete button
** this will function differently that our current version of ‘Incomplete’
** if a Quick Receipt is saved as Incomplete, it will appear in the Manage Orders list with the Incomplete status, however when opened it will navigate back to the Quick Receipt screen
* When user clicks Save as Incomplete, assign an Order #
** Keep user on this screen
** Show success message: “This Quick Receipt is saved as Incomplete with corresponding Order # 12345.”
* Update the Status and ‘Entry Method’
** Order Status will be ‘Incomplete’
** Entry Method will be ‘Quick Receipt’

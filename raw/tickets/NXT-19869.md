# NXT-19869 — Production- Carryover- Display on Plan and Record Page

## Description
As a Kitchen Manager, I would like to be able to see that an item was carried over from another day and how much of it was carried over. 

Once a user carries an item over, display the information on the production plan and record page indicating *how much* was carried over, *when* it was carried over from and *who* carried it over.
 
On Production plan page, display:
carryovers to serve count along with (serving size) along with menu line and date it was carried over from

On Record Page, Display only what date and line the menu item is being carried over to.

## Acceptance Criteria
*how much* was carried over, *when* it was carried over from and *who* carried it over.

On Production plan page, display:
carryovers to serve count along with (serving size) along with menu line and date it was carried over from

Take in consideration the amount carried over as part the “planned total”

If more items than tool tip size, utilize a scroller to display additional items.

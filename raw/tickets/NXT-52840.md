# NXT-52840 — MP- Assign Menus- Lock all menu assignments after consolidated plan complete on prep and receiving site

## Description
As a menu planner, once consolidated plans have been completed on menus that I have assigned to prep sites and receiving sites, I should not be allowed to delete the menus from the assigned dates as now the prep site has already consolidated and is preparing to make the food for the receiving sites.

## Acceptance Criteria
* Once prep site saves the consolidated plan and the consolidated plan is now complete, do not allow menus from assign menus page to be deleted or removed from calendar
** This applies to both the prep site menu and the receiving sites menus
* IF consolidated plan gets deleted, unlock menus and they can be deleted/removed or replaced with another menu
* Add component to skip months and use (use component or something similar to what is used on the production calendar)
** Allow for Year, and month ONLY ( NO DATE)
** IF time allows, adjust production calendar component to match the new selector with no date and only year and month
* For menu line selection, always display menu lines based on configured Display order

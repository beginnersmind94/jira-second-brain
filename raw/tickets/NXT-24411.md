# NXT-24411 — Platform - Security - General Permissions CRUD Operations Enhancements (PAGE - ONLY)

## Description
As a primero edge admin, I should be able to access resources to add a new resources available for any district with functions resource type for any module with options to select which ACLs (ADD/DELETE/EDIT)

+*Requirements:*+

* Enable resource ACLs to be made available
** ADD
** EDIT
** DELETE
* All resources available under each specific modules will have selected ACL options available for any given role

## Acceptance Criteria
* CRUD ACLs (ADD/EDIT/DELETE) options are available for users under roles & permissions page
* User shall be able to select CRUD ACLs while adding resources
* User shall be able to update the available CRUD options under resource catalog
* Users shall be able to view all selectable options under each module in roles & permissions page
* Resource Permissions CRUD operations to be enabled and making sure FO & BO teams can utilize these resource permissions via roles & permissions page

# NXT-14810 — INV - Order Template Details - Items tab - View Items

## Description
As an inventory Operations Manager I want an Items tab on the Order Template Details page so I can view the specific items which are assigned to the template. 

This screen is accessed by opening the view icon for a specific template via the Order Templates page ([NXT-15619|https://primeroedgenext.atlassian.net/browse/NXT-15619]).

*Requirements*

* This screen will contain a list of items that are currently assigned to the selected template
* Provide an Items tab at the top of the page
* There will be an ‘Add Items’ button to add new items to the selected template ([NXT-14811|https://primeroedgenext.atlassian.net/browse/NXT-14811])
* A list of items grid titled ‘Order Template Item List’
** Use pagination for this grid, default to 50 or 100 items
** Provide the ‘Field Search’ button for toggling search
** Do not need the ‘Columns’ button/option
* The list of items will contain:
** A select column, for selecting items for removal
*** allow checkbox selection for each item
*** have checkbox in the header for selecting all
**** Select All would only select all the items displayed as a result of any search criteria
** Item Number
*** sortable & searchable
** Item Name
*** sortable & searchable
*** default sort order: alphabetical ascending
** Storage Category
*** sortable & searchable
** Item Category
*** sortable & searchable
* Remove button, to save items which have been selected for removal ([NXT-15923|https://primeroedgenext.atlassian.net/browse/NXT-15923])
* Back button, to return user to the Order Template List
** Prompt user with “unsaved changes” prompt if there are items selected when user clicks Back button

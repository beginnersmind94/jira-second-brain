# NXT-61563 — AI Onboarding - Digitize - Special Permission

## Description
As a Customer Support role user, I would like to be able to turn on the Digitize AI special permission for certain roles so that those users can click the digitize buttons

*Notes:*

* This permission is needed to control the visibility of the Digitize buttons in BO

*Requirements:*

* Add a new Special Permission on the special permissions tab for roles and permissions
** Name: AI Digitize
** Description: Can Use AI Digitize
* Ensure default value is unselected for all roles except Director role upon district creation
* When launch the change, make sure unselected is the value for all preexisting districts and preexisting roles

## Acceptance Criteria
* Special permission is added to all environments and default is false
* For new districts, it is enabled for Director role by default

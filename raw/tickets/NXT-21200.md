# NXT-21200 — Menu Calendar Reports - Filter Options

## Description
As a user, if I have recorded "0" (zero) as a nutrient value for one of my items, I expect the display of the 100 gram USDA Conversion to also show those zeros. This should be on ALL pages and reports that contain nutritional data. Currently, these spaces are being left blank on the display of the information.

## Acceptance Criteria
All nutritional values in the 100 gram column will be reflective of the same type of information coming from the data entered and/or calculated. 
If a nutritient has a number, then the 100 gram will have a number.
If a nutritient has a zero, then the 100 gram will have a number.
If a nutrient is missing (M), then the 100 gram will have (M).
If a nutrient has a number AND an (M) from calculations of the ingredients, then the 100gram will also have a number and an (M).

# NXT-2863 — Template Hub - Actions

## Description
*User Story*
As an Inspections user, I have actions I can take on individual templates from the Template Hub.

*Mockup*
https://m5uloa.axshare.com/#id=7r5wnv&p=template&view=default&g=1

*Requirements*
* Actions are available for each template
** Actions are dependent on the template's state
* Common Actions:
** Copy - Creates a copy of the Template and sets it as Inactive
** Delete - Deletes the Template
*** Requires role permission
** Email - Allows the user to send a version of the template to any address or email group
** Export to PDF - Allows the user to save a PDF form of the Template
* Status-Specific Actions
** Activate - Moves Template to Active
*** Must be Inactive
** Deactivate - Moves Template to Inactive
*** Must be Active
** Edit - Takes user to Template Edit Screen
*** Must be Inactive

## Acceptance Criteria
* Actions are available and functioning in the Template Hub Screen

Note: Export/Email dependent on reporting being available

# NXT-21393 — Eligibility - Scanning Tool - Zone Mapping

## Description
Note: This “page” may be consolidated to within the current tool, to get around the need for MS Silverlight. PO can update if needed.

----

As a Cybersoft Admin, I would like users to have a tool that allows them to easily map the zones on their current year’s application to get them ready for SchoolCafé to recognize the correct data on their scanned paper forms, so that I do not have to manually do this for them before every back-to-school season each year.

Summary
Add an Application Mapping Tool page that allows users to scan and map the zones on their paper application for a given academic year and language, so that as little Cybersoft intervention is required as necessary.

Additional details - The user should be able to scan/map both portrait and landscape.

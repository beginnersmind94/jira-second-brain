# NXT-8865 — {Menu Cycle} Edit & Adjust Menu Cycle Weeks

## Description
As a Menu Planner
I need to be able to build Menu Cycles
so that I can work with Menus on a macro basis for 1-9 weeks at a time.

Add the Menu Cycle Week section (display).

Allowing Menus to be selected to be placed on the Menu Cycle for 1 or more days/weeks. 

Menus can be removed by clicking the icon.

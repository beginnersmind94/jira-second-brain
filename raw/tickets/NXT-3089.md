# NXT-3089 — System - Organization - Site Types (List) - Add/Edit 

## Description
As an administrator, I would like to be able to create a list of site types to which all of my sites belong, so that I can properly organize my sites according to the grades they support as well as as other non-school functions.

Classic PrimeroEdge path: _System > Sites and Users > Site Types_



Functionality-wise, this (editing site type data) can be a straight port of the existing application. The main changes should be in terms of UI/UX improvements.

## Acceptance Criteria
1. User should be able to edit existing site types by adding a new grade onto an existing site type when site type is "School Site Type"
2. User should be able to select an existing type and rename it
3. User should be able to change an existing site type type

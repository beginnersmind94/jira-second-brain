# NXT-43921 — POS - Preferred Name at POS

## Description
As Cali the California Director I want to see the use of preferred name at POS so that students are addressed properly.

*requirements*

* If Preferred Name is filled out, use it at the POS.
** Anywhere we use Firstname, go with Firstname (Preferred)
** Example: Smithers, Matthew (Matty)

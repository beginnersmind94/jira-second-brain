# NXT-3820 — Account Management - Advanced Search - Restrictions Section - Allergens section

## Description
The allergens that can be selected for a district are displayed in the Allergens subsection. When an allergen is selected it is added to the filter criteria window and shown. When an item is selected it is ‘OR’ed with other allergens.

!Screen Shot 2020-04-29 at 11.48.15 AM.png|width=281,height=324!

## Acceptance Criteria
The filter values are populated from the DB table

There is a search field that allows the user to search for an allergen(optional)

The search field shows partial matches.

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Allergen section they are 'or'ed together.

When a filter value is selected the Allergen Subsection count is increased by 1 and the Restriction Section count is increased by 1.

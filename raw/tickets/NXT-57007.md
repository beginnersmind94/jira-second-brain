# NXT-57007 — DMF - View Log-Show the tab based on the DB or Framework Script that is currently in-progress

## Description
When the user clicks the View Log button on the District Migration page, the user is navigated to the Framework SQL Scripts tab on the View Log page.
This is irrespective of which DB or Framework SQL Script is currently running.
With this story, we need the user to land on the System DB Source tab when the System DB Source migration is running, and the same should be true for the rest of the database and Framework SQL Script.
When the migration is in a canceled or error state, take the user to the tab of canceled or error state Source or Destination or Framework SQL Script.
When the migration is done, take the user to the Framework SQL Script. tab.

## Acceptance Criteria
Show the tab based on the DB or Framework Script that is currently in-progress.

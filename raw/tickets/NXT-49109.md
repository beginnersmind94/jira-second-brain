# NXT-49109 — DMF - View Logs

## Description
As a Customer Support role person, I would like to be able to migrate data from PrimeroEdge to SchoolCafe using an intuitive dashboard so that I can more quickly resolve the issues

*Note:*

* This story covers the Monitor tab and another story covers the Migration tab
* Note: we are basically copying Import Monitor so copy all the code from that as you can
** See the attachment I gave to Pavan that shows what to keep and get rid of from Import Monitor to understand the similarities

*Requirements:*

* On the System > Data Migration > Monitor tab, do the following:
* Add a Filters section on top
** Copy from Import Monitor
** Change Import Type to Database
*** Values: the 4 database names from the Migration tab
* 
* Add a grid
** Copy from Import Monitor
** Include paging, sorting, filtering
** Include refresh button
** No managed views
** Remove Import Name column
** Remove Import Type column
** Remove Processing Method
** Add Database column
*** Possible values:
**** System
**** Item Recipe Management
**** Eligibility
**** SchoolCafe Family Hub
** Add Location
*** Values: Source, Destination
*** If the migration is from the Source column, say Source
**** Else it is Destination
** 
** Action column:
*** Show View Log icon
**** On click, show the log for that migration activity (Another story)
*** Do not show any other icon or action except View Log
** 
* 
* Security and resources:
** Inherit from Data Migration page

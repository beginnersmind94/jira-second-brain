# NXT-26467 — Acc & AccMgnt - Display Dynamic Menu Reimbursable Meal and Transaction Details on Receipt

## Description
The transaction details for dynamic menus needs to be displayed when showing a Receipt 

When a reimbursable meal is formed the meal is shown with the the price set to the eligibility price of the reimbursable meal.

The items that makeup the reimbursable meal are listed below and indented from the Reimbursable meal entry.

Additional a-la-carte items are shown with the appropriate price.

## Acceptance Criteria
The transaction detail shows the Reimbursable meal formed by DM and the items included to create the meal.

Receipt shows the DM details via the site and via POS.

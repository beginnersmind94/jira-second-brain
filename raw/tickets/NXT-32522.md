# NXT-32522 — INV - Orders - Standard Ordering - Change Par Level for Pack Sizes

## Description
As a Site Manager I want the Site Par Levels page to have a Unit Description dropdown for each item so that I can set the par level quantity in terms of whichever pack size unit I would like to use.

(!) +*-Refinement 9/15/22 Notes-*+

* -Paging presents issues with autofilling, by leaving auto-filled items on a different page and therefore not adding to cart-
* -Two solutions:-
** -1. Only affect items on the current page (we could default to 100 page size)-
** -2. Affect all items across all pages, but at a performance cost-

*Requirements*

* Update the Par Calculated Quantity on the Standard Ordering screen to reflect Pack Size changes
* Remove the Par Calculated Quantity column
** remove popup and fill icon 
* Add an icon or something to note the items with some par calculated quantity
** put this on/in/near the order quantity column
** Display both icons in the Order Quantity column to the right of the order quantity box
*** Show the icon for Par Level first
*** Next show the alert icon for exceeding contract limits 
* Provide a header *button* to -‘Fill from Par Levels’- for Order Quantity
** *Button name* (updated): Autofill
*** Hover tooltip: ‘Autofill from Par Levels’ (follow exact capitalization)
** this would populate the Order Quantity field with the calculated par quantities for all applicable items
** Do not overwrite manual quantity entries for par level items
** this may cause API changes ([~accountid:603839d29b3ef700694b5a1b] [~accountid:5e97d4243a8b910c0859e4e8] )
* Provide a ‘Show Par Level Items’ *toggle*
** this would filter the order list to show only items with some par calculated quantity
*** Suggested order quantity must be greater than zero; we are not showing all items with some par level configured
*** may need to be careful with hiding other items if they have a quantity already 
* Provide a popup/report to repurpose the existing Available Quantity popup to show the same info but for all applicable items

# NXT-8101 — Data Exchange - Enable Schedule Import Option with Data File

## Description
*User Story*

As a PrimeroEdge user, I would like to schedule my data file import when I finish the mappings & configurations so that I can update relevant data files as needed.

*Summary*

Allow a user to open a pop-up that lets them schedule the data upload for various import types on a specific time/date selected by user.

*Requirements*

* In Data Exchange Import Preview Screen→ Once all validations are complete, display “Schedule Import” button for user which saves configurations implicitly. 
* When the Schedule option has been clicked, open a modal with below fields to be configured for scheduled import job
** Schedule Date
** Schedule Times with TZ
** Remind Me check box to receive reminder notification
* When the Schedule option has been clicked, open a modal with “Save Schedule” and Cancel buttons, to allow the user to schedule the import
* Confirmation prompt is displayed with message “Your import is scheduled at MM/DD/YYYY  10:10 AM CST” with Confirm & Cancel options
** Cancel should return to scheduler window
* When user “Confirm” configuration has be preserved and import job should be scheduled.
* Add this import job under “Monitoring” page along with previous import jobs and display status as “Scheduled”
* When scheduled import job is created, email & in-app notification to be sent to the user with schedule details.
* Once Import is complete, user gets email & in-app notifications on the status reflecting from monitoring page

*Out of Scope:*

* Retry for X times
* Recurring Import
* Cancellation/Reschedule of import job
* (NEW) Remind-Me notifications (Planned in new EPIC for R2)

## Acceptance Criteria
1. Validate "Schedule Import" option starts by saving configurations implicitly
2. Validate reminder notifications are sent X hrs prior to scheduled import time
3. Validate scheduler date/time selection is based on future date
4. Validate scheduler date/time is selected based on browser TZ
5. Validate data file scheduled for import is retained and validated prior to scheduling import
6. Validate scheduled import job is added with "Scheduled" status in monitoring page

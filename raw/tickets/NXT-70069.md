# NXT-70069 — FIN - Automatic Posting Awareness Banner

## Description
As a CN Director who posts manually, I want a clear nudge toward automatic posting so I understand the time savings, without being blocked or annoyed.

## Acceptance Criteria
* A standalone banner appears between the summary cards and the Ready to Post section when automatic posting is not enabled
* Banner text: "You're posting manually. Districts with automatic posting enabled only visit this page when something needs attention."
* Right side shows "Enable automatic posting →" link and an X dismiss button
* Dismissing hides the banner for the session (persists via local storage)
* Banner never renders if automatic posting is already enabled
* Banner is visually distinct (light indigo background, left accent bar) but not alarming — it's a nudge, not an alert
* Page subtitle updated to: "Your posting jobs, tracked and audit-ready."

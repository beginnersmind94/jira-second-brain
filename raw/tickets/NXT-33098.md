# NXT-33098 — INV - Management - Contract Items - Update for Pack Size Config

## Description
As a District Admin I want the Contract Items page to reflect the pack size units of my items so I can easily mass update them correctly.

*Requirements*

* Update the Contracts Item Page to consider item pack size configuration
* Display each unit on its own row, with a checkbox available for each unit
** Show item units on individual rows but indicate items which are linked together
*** Replace Unit Type with Unit Description
* Display Item # only once for the highest unit
* Show conversion (sub) quantity

Display in the same nested UI format as the Vendors > Items tab

## Acceptance Criteria
* Update Contract Items to consider item pack size configuration
* Display a row and checkbox for each unit in the pack size config
* Items are managed individually at the Vendor Item # level
* If multiple units of a single item are on the contract, display with each other

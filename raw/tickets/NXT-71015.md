# NXT-71015 — SCTV+ - Logout Timer / Refresh

## Description
As Matt the Product Manager I want users to be logged out automatically after 1 hr. If they return to the same browser window on the next day, they should be taken back to the login screen.

*requirements*

* Implement a 60m logout timer.
* User has a warning/confirmation prompt for logging out which they can dismiss to restart their timer.
* Users returning to the website later are taken back to the login screen.
** Scenario: I left a tab open yesterday and shut my PC down. I relaunch today. The tab should take me back to the login screen instead of sitting on the main dashboard.

## Acceptance Criteria
* 60m timer with warning prompt implemented.
* Users are taken back to login if they relaunch their browser the next day and have the same SCTV+ tabs open.

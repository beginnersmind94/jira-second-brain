*requirements*

We need to add an audit trail / history option for each program.

Track the following information, group them if possible by when they were performed as the user needs to hit Save.

* Performed By
** First + Last Name
* Date
** 09/21/21 12:34:55 AM/PM
* Changes
** Configured
*** Old → New Value
**** On → Off
**** Off → On
** Program Dates
*** Old → New where applicable.
*** Also need to track 2nd date ranges getting added and removed.
**** 07/01/21 - 06/30/22 → 07/01/21 - 07/15/21
**** Added: 06/15/22 - 06/30/22
**** Removed: 06/15/22 - 06/30/22
** Classification
*** Old → New
**** No Breakfast → Severe Need Breakfast
** Reimbursable
*** Old → New
**** Off → On

Tracking is done at the global Program level.

## Acceptance Criteria
History / audit trail added.

Tracks changes being made to program, by program, by site.

# NXT-45479 — AcctMgt - Accounts - Update Date for Eligibility and Enrollment, Sorting

## Description
As Dana the Director I want to see the update dates on Eligibility records so I can make sense of the sequence of events for any changes.

*requirements*

* Show the Update Date column for both Eligibility and Enrollment drawers within Account Details
** Column placement: between Source and Comments
* Change the sorting method for both to be by the Start Date descending
** Active records to show before Inactive records if they have the same start date
* Text change for the column ‘Source By’
** Source By → Source
* Edit should be available on the top active record only

Example:

08/01/2023 - 6/30/2024 (Active record)
08/01/2023 - 6/30/2024 (Inactive record)
07/01/2023 - 07/31/2023 (Active record)

* Active record is shown before the Inactive record with 8/1 startdate

## Acceptance Criteria
Update Date available as a column for both drawers.

Sorting method for both sections changed to be by Start Date descending

Text change of Source By → Source

Edit option available on the top active records only

# NXT-22748 — INV - Reports - Expected Receipts - Details page (Report)

## Description
As a Site Manager I want a Expected Receipts details report so I can see all the items being delivered for a specific date.

*Note:* show the items which have *not* been received for the selected date.

* List only ‘Standard’ orders (no Quick Receipts) and orders in Open status

*Requirements*

* After the Summary page of the report ([NXT-22747|https://primeroedgenext.atlassian.net/browse/NXT-22747]), display the Details section 

Display *details* in the report after the Summary page

* Print the consolidated items and the Qty of that item from all orders (if it ordered in more than one order for the same date. )
* group the items by Storage and Item Categories (if storage category is removed, then only by item category. )
* add Unit Type as a column
* *Note: Vendor breakdown is not needed, and delivery location is not needed

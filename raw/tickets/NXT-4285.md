# NXT-4285 — Item Management - Recipe - Save General Info and Tag with HACCP

## Description
*User Story*

As a user, I want to be able to save my steps at any time while adding or editing a recipe

*Mockup*

!image-20200929-142204.png|width=100%!



*Requirements*

* Have a CANCEL and SAVE & CLOSE button at the bottom of the screen.
* Cancel will cancel the changes and disregard any changes that have been made. We want to use the same disregard message that we’ve used to remain consistent
* If the user clicks ‘SAVE & CLOSE’ then the information will be saved to the database and the user will be returned to the previous screen
* Prep Time / Cook Time - Remove AM / PM from the selection

## Acceptance Criteria
User will be able to save the General Item information and the HACCP information

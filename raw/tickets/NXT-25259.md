# NXT-25259 — Family Hub - School Menus - Display item pricing from SchoolCafé (Research)

## Description
As a Family Hub user, I would like to be able to not only see the nutritional information of an item, but it’s pricing also, so that I can make even better decisions for myself or those in my household as to what to purchase when going through the lunch line.

*Summary*

Unlike in Classic, SchoolCafé now associates menu item pricing directly with items in Menu Planning. Let's get this information and figure out a way to display in the School Menus page so that parents can directly view that info along with the usual calendar/nutritional info.

*Details*

* In Family Hub → School Menus, display the ‘A la Carte’ (Paid) price for the item and meal in the following places: 
*# when loading the menu grid for a school, beneath the name and allergen info, get the price of the menu item and display the price there.
*# also show the price in the Menu Item pop-up itself, when opening a menu item
*# Finally, display the meal price in the Make-a-Tray panel when the user successfully makes a reimbursable tray.
* Both the menu item and meal price should depend on the school selected.
** So, if the user pulls up the item in another page beside School Menus (when school cannot be selected), we don’t need to show the pricing there.

## Acceptance Criteria
Verify that in School Menus, the user can view a menu item’s (Paid) price in the menu item pop-up

Verify that in School Menus, the user can also view a meal’s (Paid) price in the menu item pop-up

Verify that this data can change based on the site selected by the user

Verify that the other pages where a menu item can be viewed remain the same

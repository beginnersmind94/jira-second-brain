# NXT-2530 — Inspections - Consolidated Report

## Description
*User Story:*
As an Inspections admin, I have a report that can show the full results all inspections by sites on a particular template over a date range.

*Current Version:*

[https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports|https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports]

*Requirements:*

* Consolidated Report can be run using existing Inspections and available parameters
** User must have Reports role permission to run report
** Report can only be created in Excel
* The following parameters are used (all are required):
** Site - Can select 1 or many
** Template - Can select 1
** Start Date
** End Date
* All Inspections  that fit within the chosen parameters appear in the report, regardless of status
* The following data is displayed on the report:
** Inspection Name
** Date of Inspection
** Inspector
** Score (If Applicable)
** Responses for each question for each template
*** Failed Questions are highlighted in red
* There is a second table that lists links to each attachment per Inspection and Question

## Acceptance Criteria
* Consolidated Report can be created with available data across all parameters in all formats

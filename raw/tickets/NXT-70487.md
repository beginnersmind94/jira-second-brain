# NXT-70487 — MP - Menu - Edit Menu Sorting

## Description
As a menu planner I would like the ability to be able to resort my menu items from one category to another as well as within my category if  I want it in a specific order 

!image-20260311-064011.png|width=1020,alt="image-20260311-064011.png"!

## Acceptance Criteria
* Provide a mode to Sort menu items
** This can happen from category to another category
* Allow user to sort within a category if needing a specific order
** This will be available in MP and will only be in inserted into the records, any additional sorting will not reflect on assigned menus
* Store sort to be available to publish to FH

# NXT-63386 — AI Onboarding - Recipe & Digitize - System Setting

## Description
As a SchoolCafé user, I would like to have a setting that controls if we show Recipe AI button so that I don’t see it if my district doesn’t have it.

*Requirements:*

* Do the following in System Settings:
* In System Settings, add a new property
** Setting Name: Enable Recipe AI
** Type: Internal
** Code: RecipeAI
** Value: Yes/No
** Default: No
** Category: Configuration
** Module: System
* 
* In System Settings, add a new property
** Setting Name: Enable Digitize AI
** Type: Internal
** Code: DigitizeAI
** Value: Yes/No
** Default: No
** Category: Configuration
** Module: System
* 
* *Security*
** Only Customer Support roles can make this change

## Acceptance Criteria
* In System Settings, add a new properties with the values defined above
* Only Customer Support roles can make this change

# NXT-37126 — PR- Record- Temperature input per menu item based on CCP process and page redesign

## Description
As a Production Manager, I would like for the record page to have a way to input my temperatures for my CCP during my menu item cooking or creation.

## Acceptance Criteria
* Consume the CCP information from the recipe that the menu item is tied to
* Display what type of CCP is needed and a temperature box to be able to enter the temperature for that respective CCP
** IF there happens to be multiple CCP for a particular menu item, then show multiple boxes along with a what CCP they are.
* If the CCP changes, update the CCP on the production record page 
** This does not effect production record complete
* Display Quantity Prepared in a accordion that is defaulted to being closed
* Temperature field input are required/mandatory if CCP is available on the menu item

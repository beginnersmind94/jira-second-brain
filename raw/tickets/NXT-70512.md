# NXT-70512 — SCTV+ - MP Widget - Update existing widgets to 5 items max if another MP widget is added

## Description
*requirements*

* Existing MP widgets on a slide should have their max number reduced to 5 (if greater than 5) if another horizontal widget is added.
* User should be informed that this update was applied.
* Scenario:
** User has 1 slide with 1 MP widget.
** User sets max items to 10.
** User adds a 2nd MP widget.
** The new logic restricts the 2nd MP widget to 5 items max.
** It should also update the existing MP widget to 5 max and notify the user.

*Mini-Demo* {{2026-03-23}}

* Add in another clause to check for number of columns
** If just 1 column, allow user to set Max Number up to 10 for Big Card, 8 for Overlay.
* Try to get images to always use 50% of the available card for now.

## Acceptance Criteria
* Max number of items for MP widget logic updated to edit existing MP widgets on a slide when another widget is added, if the existing MP widget had more than 5 items.
* User is informed that the max number was reduced to 5.

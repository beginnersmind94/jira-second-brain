# NXT-37463 — Imports (only Back office Imports) – Auto-Link Matched Columns - Spike

## Description
As a SchoolCafé Admin I want to quickly move the rows across from Source to Mapped so that I can more quickly complete the task.

*Notes:*

* We already can drag and drop, but we want to be able to even more quickly move fields across

*Requirements*:

* When importing a Student Import on the Field Mapping Screen, do the following:
* When the page loads the first, automatically link fields from Source to Mapped this way:
** For every column in Source that maps directly to a column in Mapped according to column label, automatically associate/link the two the same way as if it was drag and dropped
** Columns map to each other if the text header is the same

## Acceptance Criteria
* All the Source fields are linked to their related Mapped fields when page loads

# NXT-28226 — Enhance the notification configuration to allow more than one schedule per business process.

## Description
As a system we need the notification configuration screen to allow more than one schedule to be associated to a business process.

## Acceptance Criteria
Story should be considered done when:
*  We can associate more than one schedule per business process.
*  We get a warning message only when we are associating more than one schedule per business process per 
   template.
*  We should be able to save one schedule per template per business process per module.

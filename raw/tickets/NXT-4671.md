# NXT-4671 — AcctMgt - General Info - Panel Updates - Preferred Name, Additional IDs, Layout

## Description
*User Story*

As a district admin I want to clearly see what name I should be using for a patron, and additional information that is important.

*Mockup*

!image-20200729-145000.png|width=337,height=120!

!image-20200824-152332.png|width=660,height=423!



*Requirements*

* Preferred Name
** Show the preferred name clearly in the General Info panel.
*** -Preferred Name should show above the full name, with a tag/chip that says ‘Preferred’ next to it.-
*** -If no Preferred Name is entered, the full name + ID + status should be the ‘top line’ on the panel. The Preferred Name pushes that line down.-
*** (Update 8/24/20 - Zeplin updated after the story changed)
**** If a preferred name is available, show it under the patron picture.
**** Show a ‘Preferred Name’ chip under the preferred name.
** Show the preferred name on the search cards.
*** Same criteria - if it’s present, show the preferred name and the chip.
*** If it isn’t, display as per normal.
* State ID / Pin Number
** Show the State ID and PIN Number in the General Info panel as shown in the mockup.
* Balance
** Change the position of the Current Balance to fall under the photo.
* General Info Layout
** Slight re-ordering of the general info further down.
*** Gender and Birth Date first, then Ethnicity and Race.

## Acceptance Criteria
Preferred Name is clearly visible and somewhat prominent on the Panel and the Cards.
State ID and Pin # are visible on the General Info panel.
Layout of the panel re-structured.
Multi-language support where applicable.

# NXT-9223 — Edit Check List - Make status radio buttons sticky

## Description
*Story*

As a user, I would like the ability to choose an edit status filter (All/Unresolved/Resolved) and have that filter remain after I modify and re-apply the “Search Edit Checks” criteria.

Currently, when I am working on Edit Checks, the edit status filter gets reset to “Unresolved” every time I change the “Search Edit Checks” criteria and perform a search. This causes me to lose focus on my data and requires unnecessary clicking of my mouse.

## Acceptance Criteria
On initial page load, the default selection criteria for Edit Check Status shall be "Unresolved" (This is existing functionality)
When a search is performed, the radio button that is selected shall remain selected and will be applied to the search.
Add pagination

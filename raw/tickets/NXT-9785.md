# NXT-9785 — Data Exchange - File Import Dashboard: NEW FILE IMPORT SETUP

## Description
As a PrimeroEdge user, I would like to have an intuitive screen that allows me to view all file import setup configurations that are available for me with an option to create NEW FILE IMPORT SETUP so that I can view existing configurations, use existing config to import data into the system and create entirely new file import configurations.

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A238848&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A238848&scaling=scale-down-width|smart-link] 

*Requirements:*

* When the user has clicks on “*File Import Dashboard*” under configuration, file import dashboard is loaded to view, create and import configurations.
* Header is displayed for the user to clearly to highlight the *Dashboard*.
* File Imports configurations table is displayed with all file configurations available for that user (districts)
** Import Name - File Import Name given by the user
** Module - module name
** Import Type - Import type of the file config
** Schedule - if scheduled, date/time of the schedule
** Last Run At - last run timestamp
** Last Run Status - COMPLETE, IN-PROGRESS and FAILED
** Added On - create date/timestamp
** Actions columns are displayed
* Each row has “IMPORT” and “SCHEDULE” buttons available
* Each file import setup row has expanded actions with DELETE, EDIT options
* Pagination is enabled with only 10 rows displayed by default.

## Acceptance Criteria
* Verify that when the user has clicks on “File Import Dashboard” under configuration, loads the dashboard page

* Verify that File Imports configurations table is loaded properly with all columns described in the requirements

* Verify each File Import Setup row has IMPORT & SCHEDULE buttons enabled

* Verify each File Import Setup row has DELETE * EDIT actions available in ellipsis format.

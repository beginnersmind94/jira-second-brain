# NXT-36068 — IM - Recipe Rework - Remove Recipe Category

## Description
As a District Admin I don’t need a dedicated Recipe Category because I will use Menu Item Category when it’s a Menu Item or regular Item Category when it’s a centrally produced item.

*Requirements*

* Remove (or hide/disable) Recipe Category from the Configuration page
* Identify and remove, or plan to remove, references of Recipe Category or any dependencies on the field throughout the module
** Items > Single Item Recipe tab
** Reports
** Recipe split view search filters
** Recipe list view search filters
* *Note: Recipe Category will be removed from the Recipe screen in the redesign stories

# NXT-65480 — Income Statement Pt.2 Reimbursements

## Description
As a district administrator,
I need to see total earned government reimbursements for the current and prior periods,
So that I can track all program reimbursements in the correct accounting period, regardless of when claim payments are received.

## Acceptance Criteria
# *Data Source*

* Pull reimbursable meal counts from the *Meal Count Report*, ensuring it includes all active federal and state meal programs (e.g., *SNP*, *CACFP*, *SFSP/Summer Meals*, *Afterschool Snack*, and any other district-participating programs).
* Use the *program-specific USDA/state reimbursement rates* in effect for the given period.
* Include separate line items by program in the backend for transparency, but aggregate to a *Total Government Reimbursements* figure in the main summary.

# *Calculation*

* For each program:
{{Earned Reimbursement = Σ (Meal Count × Applicable Rate)}}
* Total Government Reimbursements = Sum of all programs’ earned reimbursement amounts.

# *Accounting Treatment*

* Recognize revenue in the *current service period* (accrual method):
** Debit: _Due from Federal Funds (Asset)_
** Credit: _Reimbursement Program System Accounts (Revenue)_
* When payment is received:
** Debit: _Cash in Bank_
** Credit: _Due from Federal Funds_

# *Output Columns*


Match “Meal Sales Revenue” section’s columns for consistency:

* Current Period
* Prior Period
* Current CPM
* Prior CPM
* Current % of Revenue
* % Change from Prior Period
* Year-to-Date Total
* YTD CPM
* Item % of Category

# *Business Rules*

* Financial periods must be open from posting date to current date.
* All POS periods in the selected financial period’s range must be closed.
* Program meal counts must exclude non-reimbursable meals.
* Handle mid-year reimbursement rate changes by applying rates valid for the service date of each meal.

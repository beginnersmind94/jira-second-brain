# NXT-32799 — PR- Pre/Post Production Analysis- Not including 0 count or not served menu items on the analysis (API ONLY)

## Description
As a production manager, when I 0 out an menu item, that means I am either not planning to plan for it or I did not produce it depending on where I am in my production process. That item should not be accounted for in the component analysis

Currently, a 0 on a menu item is still considered as an input and we are still considering the menu item as an item that will be used/ served. 

USDA requires that menu items that are not truly being planned or produced should not be included in the nutrition or component analysis

## Acceptance Criteria
Do not include 0 count or not served menu items as part of the component analysis 

Continue to display the menu items as part of the nutrition summary report in the menu item section

This should be applied to:

*Pre Production:*

Analysis Slide out

Nutrition Report

Food Component Report

*Post Production:*

Analysis Slide out

Nutrition Report

Food Component Report

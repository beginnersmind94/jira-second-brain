# NXT-43542 — MP- Standard Menu- Disable Analysis until ready to run (UI ONLY)

## Description
This issue has been happening often where when a change is made on the menu, and then even though the menu is saved, it is still not processing the analysis as needed and leaves the analysis blank.

## Acceptance Criteria
* When saving a standard menu and reloading the page, the analysis will display no actual results
** When clicking out and clicking analysis again, then the results will load
** *Disable* the analysis button from being able to be clicked until the analysis is ready with data
** This should apply for the standard menu and the menu cycle page 
*** Menu cycle analysis also does not display results and needs to be disabled until available

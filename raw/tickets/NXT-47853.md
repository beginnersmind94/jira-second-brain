# NXT-47853 — Users - Add & Edit - UI Enhancements

## Description
We need to make a simple enhancement to the lists on the Users Add New and Edit screens. 

*Notes:*

* This is related to the managed views story and the show filters row story, so do at the same time

*Requirements*:

* For the System > Users, do the following:
* Update the Actions column per the Figma
** Get rid of the divider
** Left align the icons
** etc.
* Remove the toggle button if not already done
* 
* For the System > Users > Add New and Edit in Edit mode screens, do the following:
** Sort sites list alphabetically
** Add a Select All to Sites list
** Add a Select All to Roles list
** Put Status and Broadcast User on the same row to save vertical space (edit screen only)
** Check the font consistency for Password and the Status and other fields as they look off
** Add the collapse button to the 3 sections on the page
** Move Cancel button over to the left like other pages
** 
** Make Roles required to have at least one
* 
* For the System > Users > Edit User > View mode screen, do the following:
** Move Cancel button over to the left like other pages
** Add the collapse button to the 3 sections on the page
** Remove the Organizational Level row from the grid
** Remove the Area(s) row from the grid
** Change Region(s) label to Districts
** Change Site(s) label to Sites
** Change Role to Roles
** 
** Put all the Districts the user belongs to, not just their principal district
** Put an asterisk or some kind of identifier for the principal district
* 
* For the System > Users > Edit in Edit mode screens, do the following:
** If the user does not have a picture, don’t show the Remove picture button

## Acceptance Criteria
* Sits sorted alphabetically
* Sites and Roles lists allow to Select All

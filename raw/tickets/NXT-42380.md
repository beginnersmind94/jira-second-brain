# NXT-42380 — IM - Menu Items - Change Serving & Menu Serving identifiers

## Description
As a menu planner I want the Serving & Menu Serving identifiers to be distinct so I can quickly understand from the Units card which are original item servings and which were created only for the menu.

!image-20230817-125759.png|width=75%!

## Acceptance Criteria
* Change the 'S' identifier to only apply to servings added via the Nutrient Info tab / main item
** in view and edit modes
* Change the tooltip on MI identifier to ‘Menu Item Serving’
* Show +only+ the MI identifier on the menu servings
** in view and edit modes

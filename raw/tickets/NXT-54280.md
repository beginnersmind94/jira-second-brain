# NXT-54280 — AcctMgt - ID Cards - Search Options, Patron Type options for Students or Adults, Multi-Select (FD 255714, 263391)

## Description
Freshdesk ticket: [https://primeroedge.freshdesk.com/a/tickets/255714|https://primeroedge.freshdesk.com/a/tickets/255714]

[https://primeroedge.freshdesk.com/a/tickets/263391|https://primeroedge.freshdesk.com/a/tickets/263391]

!image-20241031-202010.png|width=1218,height=811,alt="image-20241031-202010.png"!

*Current Functionality*

* Currently the user has no way to bulk print adult ID Cards
* User also cannot find accounts very easily and select them for printing - have to go page by page to find them
* Very wasteful as they have to print 1 ID card at a time with current workflow

*Requirements*

* User can pick between printing a single ID card or printing in bulk
** Radio buttons, text: Single Card, Bulk Print
** Instructional text added to the top of the page: 
*** {noformat}You can print ID Cards one at a time or in bulk with the options below. Templates determine the placement of text/images on each card. You can edit these on the Templates page. The Layout dropdown contains commonly used card stock options.{noformat}
**** Link to the Templates page on the word Templates.
* Single Card
** User can search by ID or Name
* Bulk Print
** User has the other print options + the Type dropdown below
** User can search for students or adults, together or separate
*** Add the Type dropdown from Accounts Advanced Search 
**** All, Students, Adults
***** No special accounts should be included under All
*** Grade / Homeroom can be hidden if Adults is selected
* New grid column: Account Type
** Adult, Student
* Add search filters to grid columns:
** Patron ID, First Name, Last Name



Filters are split between single (red) and bulk (blue)

!image-20241105-151124.png|width=1017,height=469,alt="image-20241105-151124.png"!

## Acceptance Criteria
* Patron Criteria selector added, impacts search results
* New column added: Account Type, returns Adult or Student
* Search filters added to: Patron ID, First Name, Last Name
* User can bulk print adult ID Cards.
* User can search results grid and select multiple accounts for printing

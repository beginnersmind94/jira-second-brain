# NXT-5279 — System - Authentication - Add Authentication Management page

## Description
As a district system administrator, I would like to be able to manage PrimeroEdge authentication methods, so that I will be able to control how our employees can log into PrimeroEdge securely.

* In the _Configuration_ category of the System module, add the *Authentication* page
* The page can be carried over as-is, with the overall look and functionality being the same as in PrimeroEdge Classic:
** The user (having permission to access the page) should be able to view a list of authentication methods for their district
** The user (having permission to edit on the page) should be able to change the display order and ‘displayable’ attribute for a method.

## Acceptance Criteria
Verify that an 'Authentication' page has been added to the Configuration category

Verify that users with permission can see the list of authentication methods for their district

Verify that users with permission can edit the attributes of individual authentication methods

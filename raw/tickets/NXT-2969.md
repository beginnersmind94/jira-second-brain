# NXT-2969 — System - Roles - View & Save Miscellaneous Permissions for System module

## Description
As a central office administrator I want to view the details of a role to see its permissions, so that I can make changes to it that reflect the needs of my organization.

+Summary+:

When a user selects a role, allow them to view the ‘Miscellaneous Permissions’ for the System module and make changes for each permission for that role.

+Details+:

* When a user selects a role from the Roles grid, open the drawer for that role
* The System module should be selected by default
* With the System module chosen, display all of the miscellaneous permissions with a checkbox. Just like page permissions, this list will be expanded over time:
** Update Role with all Permissions each release
** View Support Chat
* When a user checks the box of a permission and selects 'Save’, update the permissions for the role in the database.
* If there are ever no miscellaneous permissions for a module, display a ‘No miscellaneous permissions for this module.’
* Behavior/application of each permission will be in a separate story.

# NXT-5185 — Device Management – Delete POS Device 

## Description
_Phase 2, System Step 3_



As a SchoolCafé admin, I want to be able to delete an existing POS device so that I can get rid of unwanted devices.

*Flow:*

* User logs in and browses to System > Management > Device Manager
* User clicks the delete button on a record
* Delete confirmation box opens
* User clicks yes
* System calls FO Delete POS Device API

*Notes:*

* This is the continuation of the Device Manager stories.
** 1^st^, we created the APIs in Front Office
** 2^nd^, we updated the device manager screen
** 3^rd^, we created the Add New POS drawer and are able to add new devices
** Now, we will do the delete functionality for a POS device
* Note: this feature requires the work of both System and FO teams.
** By the time we start this story, FO team must have written the relative APIs for button
** And the Add New functionality must have been completed

*Requirements*:

* *Delete POS Device Button*
* All of these requirements are for the Delete POS Device button on the Device Manager page
* Delete button:
** Show a confirmation popup
** If they confirm, then do the following:
** Call the Delete POS Device API from FO team
** When call API:
*** add User to it for logging purposes
** 
** Show a message to the user that the POS Device is being deleted
*** Wait a few seconds for the success response and
*** then show them a message that it deleted successfully for a second
*** then close the notification and update the table with the device removed
** The delete button should only be usable if the status is Suspended (In other words, they must suspend and then delete)
* 
* Security:
** Roles & Permissions actions:
** Only those with delete rights to Device Manager & Cybersoft admins can use this button

## Acceptance Criteria
* Delete button:
** Show a confirmation popup
** If they confirm, then do the following:
** Provide the Delete POS Device message to RabbitMQ
** When send message, add User to it for logging purposes
** Show a message in a modal screen to the user that the POS Device is being deleted and that it may take a few minutes to changes show in the list of devices and show an OK button
* Only those with delete rights to Device Manager & Cybersoft admins can use this button

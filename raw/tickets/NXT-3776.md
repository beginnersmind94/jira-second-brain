# NXT-3776 — Create Advanced Search base window

## Description
The AS base window contains a Selected Filter WIndow, a Clear All filters button and the Sections defined in the AStable for this AS window.

!Screen Shot 2020-04-24 at 1.40.47 PM.png|width=301,height=492!

## Acceptance Criteria
The AS base window is displayed when the Advanced Search button is selected

The AS base window contains a Clear all Filters button that clears all previously selected filters.

The AS base window contains a selected filters window that shows all the previously selected filters.

The AS base window shows the sections that filters can be selected from

The sections are defined in a table containing the section name and the DB Column name to be searched, or other subsections. 

The sections are displayed based on role

# NXT-44795 — Local Ingredient - Create Ingredient and Stock Item using GTIN (real time)

## Description
Currently, the user is forced to created ingredient and stock item separately and establish the relationship.  By doing this change, the user can create ingredient and stock items together and link them automatically.

## Acceptance Criteria
Creation of ingredients using GTIN:

If the GTIN is valid - the system should show the confirmation message similar to Ingredients and populate the details on the screen without prompting for any additional fields

Upon identifying valid GTIN, system should also show corresponding stock item tab and images tab as applicable (Ex: Each - Case, Pack - Case, Case - Pallet or vice versa)

Upon clicking update, the system should create both ingredient and stock item

If the GTIN is invalid, show the i icon similar to ingredients and prompt for other fields as-is

Show the i icon if the GTIN is invalid

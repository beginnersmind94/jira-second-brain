# NXT-52726 — MP - Menu - Full history of all changes made on Menu from Menu Planner

## Description
As a menu Planner, I would like to be able to see a history of all the changes that have been made on the menu that I am working on or any menu in general. This will give me the ability to see why there are any out of the ordinary things or changes I do not recognize.

## Acceptance Criteria
* All Menu Planner History changes whether create or edit will be accessible through the history on the standard menu page
** All changes on either standard or menu planner should be reflected on the history (It would be nice if we could show if the changes were made from the standard or menu planner just to give better context to the user)

* History should display:
** All new data in the create process 
*** Meal Pattern
*** Meal Type
*** Site Group
*** Menu Name (This is the only one that can be updated so show the updated values as well if there are any)
*** Serving Group Values (Create or Edit) (Default 100 from Menu Planner)
**** Group these together if there are multiple serving groups added at the same time
*** Menu Item Categories (Create or Edit)
**** Group these together into one entry if there are multiple added at the same time 
*** Add Menu Items (Add or Delete)
**** Group these together into one entry if there are multiple added at the same time 
**** Include serving size added per serving group 
** Menu Item Counts (Per serving group)
*** Group together if multiple entries are saved at the same time (default calculations when saved)
** Serving Size (Per serving group)
*** Group together if multiple entries are saved at the same time 
** All changes made from any entry point needed to be specified as such on the history 
*** EX: Menu Item Added ( Standard) 
*** EX: Menu Item Added (Menu Planner)
*** EX: Menu Item Added (Find and Replace)
*** EX: Menu Item Removed (Find and Replace)

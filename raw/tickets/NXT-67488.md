# NXT-67488 — Menu Analysis - Menu Item Performance - Add Cost per serving and Column level filters

## Description
As a district manager, When I am looking at the menu item performance, I would like a way to quickly see what my cost per serving of each menu item serving as as well as a way to filter my columns to look at specific metrics.

## Acceptance Criteria
* Add a column for Avg Cost Per Serving ($)
* Provide filters on columns to allow users to filter per metric by descending and ascending order
* On the dashboard, add Avg Cost/Serving ($) on all metrics cards under the waste information
* In the menu item drill down change the Label of Cost to Avg Cost IF there are multiple servings present

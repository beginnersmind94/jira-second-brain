# NXT-6691 — Application Entry - Guided Process (Stepper)

## Description
*User Story* 

As an admin, I have the ability to  jump around through the application entry process to pages i’ve already been through. 

*Mock Ups* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1306%3A27462&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1306%3A27462&scaling=min-zoom]

*Requirements* 

* Stepper 
** Each step is a link to that page 
** I can visit previous pages by clicking the steps in the stepper 
** I can jump to any step that’s already been completed (steps that are NOT greyed out)
* Same validations as Next button
** saving when changes are made in the page

# NXT-66960 — WS - Insights - LM - Integrate Site License from PE

## Description
As an Insights user, I want to be able to see only the sites that I have applied a license to so that I can ignore the extra noise.

*Notes*

* Currently only District level license is applied
* Now sales team wants to sale by site or district
* If they have a district, it equals All Sites

*Requirements*

* Get the updated API from Lalita that includes site level licenses
* Update the token to include site level licenses as well as district
* Everywhere you do a license check for district, add in the license check for Insights Sites
* 
* KPI Screen:
* Update sites filter:
** List of sites filtered for the sites using site licensees
** Update the site types grouping so that if a site type doesn’t have a licenses site, it doesn’t show that site type
** Put an info icon with a hover over message
*** logic: Show if filtering by sites, hide if not filtering
*** Message: List filtered to licensed sites only
* 
* Cards, data grid and trend chart:
** All Cards and visualizations show data only for sites with licenses
* 
* APIs:
** Data is trimmed for only the sites licensed
* 
* Schoolie:
** AI Summary and Daily Recap data is trimmed for only sites licensed
* 
* Benchmark Screen:
** If you have implemented the sites filter list, then apply the same update as on the KPI Screen
* 
* Benchmark Configuration Grid:
** For the big grid where you enter the benchmarks for the sites, do not filter out sites from this list
** We want to let them configure all sites even if they aren’t licensed so as to encourage them to buy more licenses

## Acceptance Criteria
* Code is able to see what sites the district has applied an Insights license to
* Filters are filtered for the sites with licenses if they are using site licensees
* All Cards and visualizations show only for sites with licenses
* Data is trimmed for only the sites licensed
* AI Summary and Daily Recap data is trimmed for only sites licensed

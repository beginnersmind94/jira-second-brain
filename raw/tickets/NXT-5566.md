# NXT-5566 — SIS Import Student data via API

## Description
*User Story*
As a PrimeroEdge User with System Configuration privileges, I can configure import from my district’s SIS.

*Mockup*
[https://j7zek4.axshare.com/#id=30o2b5&p=page_1|https://j7zek4.axshare.com/#id=30o2b5&p=page_1] (mockup done in Classic)

*Requirements*

* Under Data Exchange page
* Select type of import as “Student Information System”
* Enable SIS Data Import checkbox
* Selection for API vs Flat file
** When API is selected, dropdown to select SIS system.
* Text field to enter URL
* Text field to enter Key
* Text field to enter Secret
* Schedule frequency options
** Daily
** Every _ Hours
** Schedule performs import using Hangfire
* Save button to save changes
** Visible when any options have changed
** Saves changes to db
* Cancel button to cancel changes
** Visible when any options have changed
** Reverts changes
* Test button to verify that URL, Key, and Secret values are correct and working
* Import Now button to immediately perform the import rather than waiting for a scheduled import.
* Import tracks time of import and allows for auditing of changes

* *Checkbox for Student data*

## Acceptance Criteria
- SIS data successfully imported and showing in PrimeroEdge

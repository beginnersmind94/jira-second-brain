# NXT-29698 — POS - Reclaim / reuse a Device Name

## Description
If a device name has been used before, give an option to ‘reclaim' it and continue using it.

* User might enter a device name of ‘POS1’ but we need to check with the site number attached to see if that already exists.
** If it does, we show the last activity date for that device and the user has the option to use it.

If a MAC address / device has been used before, show a list of terminals associated with it, and the user can re-use one of those terminal names.

User can see all terminals that have been setup for that site and a last activity date for each of them.

User can select from this list and re-use that device name.



Mini-demo notes 05/20/22:

* Search functionality won’t be needed - just show any machine that had that name previously.
** We’re working on the assumption that customer will check for their old device name via Device Manager, or by checking it in Reconciliation.
* Cannot re-use a terminal if it was last used on the same day.
** Last active date = current date, case not allowed.
* Show the site name in the existing devices list.
* last active = Last Active
* use full timestamp for the Last Active - DD/MM/YY HH:MM:SS AM/PM
* remove the ‘Existing Devices List’
* only show the list of devices if they try to enter one that already existed 

Strings

* If the terminal name already exists:
** A device with that name already exists. Select it from the list below.
* If the terminal cannot be re-selected because it was last used today.
** You cannot use $DeviceName because it is in use today. The device name will be available on $LastUsedDate+1Day.
*** Example: default-402 was last used on 5/20/22.
*** String: You cannot use default-402 because it is in use today. The device name will be available on 05/21/22.
* Confirmation prompt for when the user is selecting to re-use an active terminal that IS available.
** Are you sure you want to reuse that device name? Reusing the device name will deactivate any other devices with the same name.

## Acceptance Criteria
User can reclaim a device name if it’s been previously used.

User is notified if their device / MAC address has previously been used for a device, and the user has an option to re-use that device name.

User can still make a completely new device if they want to.

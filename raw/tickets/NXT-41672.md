# NXT-41672 — SIS Auto Sync – Edit Configuration - Dev

## Description
As a SchoolCafé admin, I would like to edit the SIS Auto Sync configuration fields so that I can make update an SIS import configuration.

*Notes:*

* We will use the same drawer and settings as the add new drawer

*Requirements*:

* On the System > Imports & Exports > SIS Auto Sync tab > SIS Configuration drawer, do the following:
* 
* Update the Provider drawer:
** When the user selects a provider, check if it has been saved
** If not, treat as an Add New like you already do
** If they have saved (configured) the provider, then:
*** Populate the fields on the screen with the existing information
*** Change the text of the save button to Update
*** Show the delete button
** If the user again selects another provider that has been saved while on the same screen, remove the values from the fields and update the values with the new provider
** If the user then selects another provider that has not been saved, remove the values from the fields, change the Save button text back to Save and hide the delete button
* 
* Update button
** On click, validate required fields like Add New and test the connection
** If successful, update the values in the database
* 
* Delete button
** Covered in another story
** For now, just show it and have it do no action (action covered in the other story)

## Acceptance Criteria
* When the user selects a provider that has been saved, it populates the fields on the screen with the existing information
* When the user selects a provider that has been saved, changes the text of the save button to Update
* When the user selects a provider that has been saved, it shows the delete button
* It updates the database rather than saves new

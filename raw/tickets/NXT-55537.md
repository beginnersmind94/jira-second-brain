# NXT-55537 — Family Hub - Agreements - Settings - New Edit Payment Guidelines layout

## Description
As a district administrator, I would like to be able to more easily view and select payment guidelines that make sense for my district before I submit an agreement, so that I will be able to choose the configuration that makes the most sense for parents in my district.

*Summary*

When an Agreements user goes to the Settings step, replace the Convenience Fee section with a similar section to the one which was added in the Super Admin - Payment Guidelines page.

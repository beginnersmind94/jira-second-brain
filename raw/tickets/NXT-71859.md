# NXT-71859 — SchoolCafé - Forms - (Part 3) Family Hub Forms - App Submission

## Description
As a Family Hub parent/applicant, I would like to be able to apply for Summer EBT (as well as any other non-traditional form) applications with minimal steps, so that I do not have to use an external system just for this purpose.

*Summary*

* When a form application is submitted via the Fill out a Form page of the Family Hub (when the Submit Application button is clicked on the Submit step): 
** transmit the application data to SchoolCafé 
** Show the user the confirmation page, which should contain the specific form app image template visible (to show them how their completed form looks)
*** The form image will also be viewable in the Eligibility Info page upon completing [https://primeroedgenext.atlassian.net/browse/NXT-71764|https://primeroedgenext.atlassian.net/browse/NXT-71764|smart-link]

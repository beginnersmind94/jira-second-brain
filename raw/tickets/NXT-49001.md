# NXT-49001 — AM - Reminders - Scheduler - Edit

## Description
As Dana the Director I want a way to edit what I previously setup as things change over time.

*requirements*

* User can edit the saved criteria of a recurring send.

## Acceptance Criteria
* User can edit the saved criteria of the recurring send.
** Edit option off the grid.
** Shoulder showing the saved criteria.
** User can change any of the saved criteria - Patron Criteria, Balance Criteria, Recurrence, Name + Description, Templates, Single vs Household letter grouping
** User can save their changes or cancel out.
** Saves are tracked for History.
** User can make a recurring send inactive.
** User can make a recurring send active.

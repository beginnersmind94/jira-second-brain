# NXT-42772 — Acc - Reports - Edit Check Worksheet - Updates for State Eligibility

## Description
As Anna the Application Processor I want to see State Eligibility on reports so I can verify information and counts as needed.

*requirements*

Edit Check Worksheet report

* If the state eligibility setting is turned on, user can see the State Eligibility columns in the report.
** Only show the columns if they are both configured - for example, if a district does not put an Income Guidelines percentage in State-Reduced, do not show it
** Use the local descriptions if possible

* Add a State Eligibility Counts section after the between Total Served and Comments
** Show how many State Free / Reduced meals were sold, and how many eligible students there are.
** Do not subtract these counts from Paid - we will add the note below to explain why we are doing this.
* Add a note to the report if the setting is on:
** Note: Paid totals include State Eligible meals. State Eligibility columns highlight how many meals were served to those students.

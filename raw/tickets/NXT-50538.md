# NXT-50538 — PR- Reports- Grocery List- Site Group and Site multi Select

## Description
As a production manager, sometimes I am responsible for getting the grocery list for multiple sites at the same time and would like to make one order based on my operations.

## Acceptance Criteria
* Provide a multi select for site group and an all option
** Once site groups are selected filter to show all sites that belong within the particular site groups selected
* Provide a multi select for sites and an all option
** Once sites are selected, display all items for selected sites and group together if same items are needed along with total quantity and days needed as current logic
** All meal types grouped together along with all counts for all sites for a particular day
* Logic involving prep site items should also remain and only display it once if prep site is selected along with receiving sites and never display the items for the receiving sites if they are part of a prepared items that the receiving site is not preparing for themselves
* Report must also show filter criteria so that user can know what site groups or sites were selected for the data

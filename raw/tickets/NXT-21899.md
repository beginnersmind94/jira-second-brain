# NXT-21899 — INV - Vendors - Vendor Detail - Enhanced Contracts tab

## Description
As an Inventory manager I want a tab based view of the vendor so I can more easily view and manage the different vendor information.

This tab will contain the list of Contracts which are configured for the vendor.

*Requirements*

* Change the Action column
** View Items

** View Ordering Conditions

** Edit icon

* Show the Add Contract button
** in the top right grid header
* Show the Status as a label
** Active - light green
** Inactive - light orange
** Marked for Deletion - light red
** Suspended - gray

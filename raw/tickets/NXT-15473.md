# NXT-15473 — Platform - Periods - Option to close the periods

## Description
After verifying the transactions for that period, the district needs an option to ensure no additional changes can be made. If changes are made after they submit their Reimbursement Claims this may cause a district many issues.

!A123 - Modifying POS Periods.mp4|thumbnail!

## Acceptance Criteria
* User shall be able to close period from actions available on the period with 'X' mark.
* User shall be able to see “Status” as a new column to reflect closed status
* User shall be prompted with a confirmation prompt for every action
* User shall not be able to edit/update Closed periods which is one-time operation

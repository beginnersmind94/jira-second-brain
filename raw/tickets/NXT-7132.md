# NXT-7132 — Menu Grid Button config

## Description
*Story*

As a user I need to be able to create menu grids for use on ExpressPoint. There are two types of menu grids, static which have menu items that don’t change and dynamic which has items that change based on menu planning. This story handles Static menu grids. 

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=29545%3A1331&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=29545%3A1331&scaling=scale-down-width]

*Requirements*

* Button size can be changed
* Button Position can be changed on the grid
* Details of the button can be viewed and edited [https://primeroedgenext.atlassian.net/browse/NXT-7197|https://primeroedgenext.atlassian.net/browse/NXT-7197|smart-link]

## Acceptance Criteria
Clicking on the Bottom right corner of the button allows the button to be resized by dragging the corner.
Clicking the cross in the bottom center allows the user to reposition the button on the grid.
Clicking on the 3 Dots on the left opens a selection to edit or delete the button.
Edit opens the Button Details
Delete removes the button from the grid.
When a Button is selected it has a Highlight box around it.
When the 3 Dots are selected they are highlighted
When the Cross is selected it is highlighted
When the Resize Icon is selected it is highlighted

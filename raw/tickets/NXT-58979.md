# NXT-58979 — SchoolCafé - Super Admin - Payment Guidelines - Small adjustment in 'Mixed' scenario

## Description
Per Jay and Shelly, let's make one small change: when a 'Mixed' option is chosen in the '_Who pays the convenience fee on each transaction?_' question, change the _Minimum payable amount for payments_ line from a fixed value to a customizable field. 

Currently the field is a fixed $20.00.

Notes:

* This change is for Super Admins only (this minimum payment field should still be disabled/fixed for admins filling out an agreement. It will display whatever has been entered by the Super Admin here)
* Value for the field should be >= 0.01.
* No changes for other options (User, District, Flat, Percentage) are required

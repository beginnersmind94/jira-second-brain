# NXT-16221 — API - MP > Menus > Assign Menus - allow assignment of Menu Cycle, with Preview and list of Sites

## Description
As a Menu Planner
I need to be able to assign Menu Cycles
so that I can let my site staff know what to serve.

Assign Menus allows selection of a single Menu to assign and puts it on the calendar.

!image-20210302-232932.png|width=100%!

Requirements

* When dragging and dropping menu cycles, show a ‘preview’ of the menus from the menu cycle in the calendar
* When clicking ‘Review’ will then navigate to the ‘Review screen’
* !image-20210709-134228.png|width=77.91308500477554%!

## Acceptance Criteria
Screen allows single Menu assignment. User can click and drag the menu to the specific date after selecting the site group.

## Acceptance Criteria
The system should send a notification to one or more group of Cybersoft users upon the following scenario:

# When Service is Down
# when certificate due for renewal (come as one of the XML files) as notification
# When one or more files are not processed
# When maximum process storage is reached
# If no new xml or updates are received from 1Ws for specific number of days (EX: 30 Days)

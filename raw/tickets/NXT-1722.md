## Acceptance Criteria
* Provide a slideout panel when viewing the details of a specific E-order
** when user clicks the view icon for the e-order
* Display a list of the orders contained in the e-order in a vertical arrangement on the slideout panel
** *Note:* the number of orders can potentially be in the hundreds; we may need to use pagination or something similar
* Provide a Site dropdown to filter the orders
** default to All
** dropdown should consider user access (to sites)
** allow single selection (no multi select)
* Display the following order attributes in the orders grid
** Order #
** Site
** Delivery Date
** Order Amount
** Action
*** Provide a view icon for each order to generate and display the HTML orders report

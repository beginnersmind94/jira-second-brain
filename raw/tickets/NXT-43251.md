# NXT-43251 — AcctMgt - Accounts - Option to soft Delete an account

## Description
As Dana the Director I made a few mistakes during data entry and implementation so I’d like an option to delete some accounts out.

*requirements*

* Add a Delete option to Account Details
** Add a role permission tied to this, available to Central Office admin and Director by default, off for other roles
* Take the user to a small guided workflow
** Step 1: Renaming
*** IDNumber → del_IDNumber
*** PIN → del_PIN
** Step 2: Balance Funds
*** Ask the user what they want to do with the funds
**** Options: Zero Out, Transfer
***** Zero Out → set balance to $0.00
***** Transfer → ask them what account to move funds to, find it, show it, confirm it
** Step 3: FamilyHub Account
*** If the account is on a FamilyHub Account, let the user know that.
**** Give the user an option to notify the household about the deletion.
***** This should trigger a notice in the FamilyHub app about the student being removed.
** Step 4: Confirmation
*** Summary of the changes
**** ID Number → del_IDNumber
**** PIN → del_PIN
**** Balance → Transferred to #12345 / Zeroed out
**** FamilyHub Notification → Will be sent / Will not be sent
*** Confirmation prompt
* If confirmed:
** Update account status to Deleted
** Perform account detail renames for IDNumber, PIN
** Add a Transaction Adjustment or a Funds Transfer
** Notify the FamilyHub Account if enabled
** Remove homeroom, rosters
* Add a ‘Restore Account’ button for accounts with a Deleted status
** Confirmation prompt
** Set account to Active

## Acceptance Criteria
* New phone fields added
* Email doesn’t repeat
* Export changes made
* Search filter added or info icon removed for phone number column

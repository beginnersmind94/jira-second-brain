# NXT-36164 — Student Balance Export - Create Scheduler

## Description
As a SchoolCafé admin, I want to be able to schedule an export of the student balance export so that I can routinely export the data to 3^rd^ party payment systems.

*Flow:*

* User logs in and browses to System > Imports & Exports > Exports
* User clicks on + New button and is redirected to the Add New Export page
* User completes the form including the schedule information and saves the new job

*Notes:*

* Note that we combined the Student Balance Export with the Student Export, so your testing will be on the Student Export
* This story assumes that the story to schedule exports has been completed and we can leverage existing screens and functionality

*Requirements*:

* *Add New Export page*
* When the user clicks the + New button on the Exports tab, do the following on the Add New Export page:
** Ensure you can select Student Balance Export
** Ensure all the correct fields are there
** Ensure you schedule the export
** Ensure when save, it creates the job schedule
* 
* *Exports Tab on Imports & Exports*
* On the Exports tab, make the following changes:
** Ensure IsScheduled is true when scheduled
* Export button
** Ensure when clicked, that it manually runs the export
* Delete button
** Do the same as for non-scheduled jobs
* Edit button
** Ensure when you make changes, they are saved and executed accordingly
* 
* *Export Monitor Tab on Imports & Exports*
* On the Export Monitor tab, make the following changes:
** Ensure that the scheduled jobs show in the export monitor grid when they run and that all the column values are correct and completed
* 
* View Log button
** Do the same as for non-scheduled jobs on the page it opens
* 
* Permissions: no updates required

## Acceptance Criteria
*Add New Export page*

* Can select Student Balance Export
* All the correct fields are there
* Can schedule the export
* When save, it creates the job schedule

*Exports Tab on Imports & Exports*

* IsScheduled is true when scheduled
* Export button - when clicked, that it manually runs the export
* Delete button - Does the same as for non-scheduled jobs
* Edit button - Ensures when you make changes, they are saved and executed accordingly  

*Export Monitor Tab on Imports & Exports*

* Scheduled jobs show in the export monitor grid when they run and all the column values are correct and completed
* View Log button - Does the same as for non-scheduled jobs on the page it opens

# NXT-45142 — USDA Branded Foods - Converting Nutrition to Serving Measure (Instead of 100 Gm by default)

## Description
_(none)_

## Acceptance Criteria
Currently the nutrition values are shown for 100 Gram instead of actual serving size weight (ex: 30 Gram), hence the need of scaling the nutrition)

Calculation: (Serving Size Gram * Quantity Contained Value) / 100

 The UI should represent all the nutrient values according to this formula

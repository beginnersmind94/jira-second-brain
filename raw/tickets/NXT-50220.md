# NXT-50220 — LMA - Subscription - District Licenses

## Description
As a Central Office user, I would like to subscribe to License Management events so that I can stay aware of what’s happening.

*Notes:*

* We will leverage the event subscription framework for this story
* If you have done the License Key subscriptions story, copy that to start with, as most things are the same
* This story is for district users

*Flow:*

* User browses to System > License Management > District Licenses
* User clicks on Subscribe button
* License Key Subscriptions drawer opens
* Selects criteria under which scenario to be notified
* Optionally deletes the notification
* User clicks Save
* Later, if one of the criterion are met, the System triggers the notification process
* System notifies by sending message to user according to notification criteria and timing
* User browses to inbox and reviews the notification

# NXT-3012 — ****View Inspections - Actions - Reassign

## Description
*User Story*
As an Inspections user viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to reassign the Inspection to another user.

*Requirements*

* Reassign is an action that can be applied to any Inspection
** This action is present on all Inspections
** This action is only available to users with the proper permissions
*** Supervisor and Admin personas
* When clicked, the user receives a slideout
** Header: Reassign Inspection
** Information:
*** Inspection Name
*** Site
*** Current Inspector
*** Date
** User has a dropdown list of users that the Inspection can be assigned to
** There is a submit button and a cancel button
* User who the Inspection is reassigned to is notified

## Acceptance Criteria
* User can reassign for any available Inspection from the View Inspections screen in either view (card or list)
* User can reassign on Inspection Details screen

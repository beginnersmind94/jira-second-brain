# NXT-17486 — PNA AI/ML Model Technical Metrics – Performance and Report

## Description
As a system, AI Insight will need PNA technical metrics of the ML model so that we can evaluate performance of the forecasting prediction.

*AI/ML Model metrics:*

* Classification accuracy
* Mean absolute error (MAE)
* Root mean square error (RMSE)
* Sensitivity and Specificity average
* Accuracy percentage
* Other – will be added as per model definition

*Dependency:* AI/ML metrics are dependent on model selection and development

## Acceptance Criteria
*_For PNA eligibility recommendation and forecasting:_* 

# The AI/ML solution shall specify the problems it is solving with defined use cases related to defined hypothesis
# The AI/ML solution shall specify PNA forecasting for the departments and users supported (Districts/Schools – all sites, multiple sites, single site, districts in region or geo)
# The AI/ML solution shall specify the total number of models deployed for PNA 
# The AI/ML solution shall specify the volume of data analyzed for PNA
# The AI/ML model shall record PNA forecasting accuracy metrics with precision and recall scores
# The AI/ML solution shall specify the PNA KPIs impacted with the provided solution (before AI and after AI)
# The AI/ML solution shall specify the usage of PNA eligibility forecasting model to further use and forecast PNA revenue forecasting
# The AI/ML solution shall provide PNA data analysis dashboards with different use cases and data sets (including demographic, geographic and school data)
# The AI/ML solution shall provide dashboards and Viz with PNA pre model and post model analytics – Key AI objectives achieved report (for district admins)

# NXT-68097 — SC - MT - Licenses Update

## Description
As a Central Realm Administrator, I want to purchase licenses for my districts so that they can use the software without having to purchase their own license

*Notes*

* A central realm will buy licenses like normal
* If they want to provide Inventory for their 10 children districts, they will buy 11 Inventory licenes
* They then select with districts the license is for, creating a link in the DB
* The children then “inherit” the license

*Requirements*

* For [https://qa.perseusedge.com/system/District-Licenses|https://qa.perseusedge.com/system/District-Licenses],  Districts works like Sites
* if a realm does not have children, do nothing
* If a realm has children, then need to be able to assign licenses to children as follows:
** Show a View Districts button just like the View sites button on every row
** Check their own license count. if they have only one district license, hide this button
* On click
** Change text to Hide Districts
** Show the Districts panel just like the Sites panel
** Districts panel has a + New button like Sites
** On click, open the + New Districts popup like Sites
** User selects districts just like they do the Sites
*** Can only select children districts
* Once selected districts, they are added to the Districts list in the Districts panel just like in the Sites panel
* Can remove districts
* Site panel tracks count of assigned districts like sites panel
* need a visual indicator if a license is purchased by Central or Self
* If a license is purchased by a parent, district can still enable/disable it for the toggle
* A district can purchase additional licenses if needed
* Children districts inherit the licenses in an “Additive” way, meaning that if parent doesn’t purchase something, they can still purchase more
* 
* Update the authentication token so that a child district can inherit the parent
* 
* Update the Add New so that can add more than 1 District license (Currently always one)

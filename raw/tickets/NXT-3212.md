# NXT-3212 — Inspections Reports - Inspection Status Report

## Description
*User Story*

As an Inspections supervisor, I have a report that allows me to see Inspections status by Site across a pre-set period of time (e.g. Monthly, Quarterly, etc.) to ensure that all Sites have received the appropriate Inspections as needed.

*Requirements*

* Request: [https://primeroedge.freshdesk.com/a/tickets/118725|https://primeroedge.freshdesk.com/a/tickets/118725]
* There is a report called the Inspection Status Report
* When selected, the user is present with the following options:
** Time Frame - Week, Month, Quarter, Semester, Year
** Sites - By default, all sites are selected. Users can select individual sites to customize the report
** Template - By default, all templates are selected. Users can select individual templates to customize the report
* When the report is run, the following information is displayed in this order:
** Site
*** Template
**** Inspection(s) Run - Name, Inspector, Score
* The report can be exported in PDF, exported in Excel, or emailed

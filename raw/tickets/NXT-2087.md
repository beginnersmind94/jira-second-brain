# NXT-2087 — Production - Production Plan - Auto Fill Plan (UI Only)

## Description
As a site manager, I see the same or very similar menus get assigned to my school on a regular basis. I want to be able to simply copy the numbers from a previous date, and apply them to future dates, instead of always entering every single number. Example: We serve hamburgers and cheeseburgers every other Thursday. It has the same side dishes and milk, but sometimes the fruit item might change. I would like to be able to select the date on the calendar the last time the hamburgers/cheeseburgers were served, and copy the numbers from that date, to the future date that I’m planning. This would save me a lot of time. I would then only need to input numbers on the fruit items that did not match between the two menus. (or if the menus were completely identical, then ALL of the numbers would copy for me.)

Another option for Auto Fill, would be to have the ability to enter in my planned meal counts for the serving groups assigned to my site, and have that meal count populate down the page, distributed evenly between the menu items in each category. (I would still have the option to change those numbers, if needed.)

There should also be an option to copy the meal and item counts from the originally created menu… using either the actual counts, or the percentages based on my student counts. (If the original menu shows that 70% of students take entree #1, and 30% take entree #2, then I should be able to enter that my site will serve 200 meals, and 140 would go into entree #1, and 60 would be entree #2.)

## Acceptance Criteria
There will be a button in the production plan page that allows me to choose a previous calendar date, with an identifying feature that shows if previous menus were identical to the menu that I’m on, or if it’s a copy of the menu that I’m on, or if it’s just a completed plan but not identical nor a copy. 

I would be able to select one of those previous dates, and any menu items that are the same between the two will automatically receive the numbers from the date I selected.

# NXT-7270 — INV - Physical Inventory - Count Sheet

## Description
As a site manager I want to be able to view or print a count sheet so I can prepare to count my items.

*Requirements*

Count Sheet can be generated

* On the Physical Inventory page itself, once it’s been created or in progress

Count Sheet - a list of items to be counted

* Header info
** Report Name: Physical Inventory Count Sheet
** Site
** Period
*** if interim, display “Interim (MM/DD/YYYY)”
** Storage Category (if applicable)
*** determined by selection on Summary tab for loading Items tab (select a specific storage category or not)
*** hide this if they didn’t select a specific Storage Category
**** if this is not feasible, we can just show ‘ALL’ for the value
** Date/Time Counted (if applicable)
** Counted By (user)
** Started Date
** Started By (user)
* Attributes/fields:
** Item #
** Item Description
** Unit Type
** Usable Quantity
*** empty field for user to enter value
*** display the saved value if one exists
** Damaged Quantity
*** empty field for user to enter value
*** display the saved value if one exists
** A comments/notes field
*** empty field for user to enter value
*** shows the current system comments if any are entered
** *Sort Items* 
**# by Storage Category first
**# by Item Category second
**# by Item Description third (alphabetically ascending)
**# **Note:* this is temporary for R4. Eventually we will change so that the report always matches the screen.

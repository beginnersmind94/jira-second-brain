# NXT-7268 — INV - Transfers - Transfer List

## Description
As an Inventory user I want a Transfer List page so I have a central page to view existing Transfers and create new ones.

_Inventory > Transfers_

This is a new page that will be accessed by clicking the ‘Transfers' page

This will be a list page that provides a list of transfers and a button for creating a new transfer

+Action buttons:+

buttons are provided in top right corner

* Create button 
** this will open the Create Transfer process in [https://primeroedgenext.atlassian.net/browse/NXT-7267|https://primeroedgenext.atlassian.net/browse/NXT-7267|smart-link]
* Request button (site only)
** this will open the Transfer Request process [https://primeroedgenext.atlassian.net/browse/NXT-9411|https://primeroedgenext.atlassian.net/browse/NXT-9411|smart-link]

+Search/Filter section:+

Site selection

* Site selection - from user control
** Area
** Site dropdown (code & name)
*** only active sites or sites with some transfers
*** provide All option
*** Toggle for Sent or Received will be considered when applying site selection
* Date range
** From 
** To
** default is past week
* Status dropdown (multi-select)
** Requested
** Approved
** Completed
* Toggle
** Transfers Sent
** Transfers Received
* Reset button
** resets to default selections
*** All sites
*** Last week for date range
*** All statuses
* Apply button
** applies selections to the grid

+Transfer List+

* A table listing the existing transfers
* Sorted by default as date descending
* User can click to resort columns
* Use pagination for grid
* +Table columns:+
** Transfer #
*** link to open the transfer detail (covered in [NXT-9441|https://primeroedgenext.atlassian.net/browse/NXT-9441])
** From Site 
** To Site
** Status
** Date
*** actual transaction date that user provides on the Create Transfer (not when user entered into system)
** Comments

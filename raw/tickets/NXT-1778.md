# NXT-1778 — INV - Inventory - Additions - Create Addition - Selections (Step 1)

## Description
As a Site Manager I want a function to create a new Inventory Addition transaction so I can add items to my inventory which were not ordered.

This is the first step in the Add to Inventory process

*Requirements*

* Provide a Create button on the Additions list page
** This will open the Add to Inventory stepper
* Use a stepper component for this process
** utilize the horizontal stepper
** Title the first step ‘Selections’
* Provide the following inputs on this first step
** Site selection dropdown(s)
*** this will be the standard site selection control which includes Area if they are configured for the district
** Transaction Date and Time pickers
*** auto-populated with the current date and time
** Reason dropdown 
*** this will list the reasons which are configured in the Addition Reasons page ([NXT-20675|https://primeroedgenext.atlassian.net/browse/NXT-20675])
*** provide ‘Not Specified’ as the default option
**** this will be displayed if there are no other reasons configured
**** this basically makes the selection optional 
** Comments textbox
*** *optional
*** allow up to 200 characters
*** display character limit
* Provide a Next button via the stepper tool 
** this will progress to the second step ([NXT-20676|https://primeroedgenext.atlassian.net/browse/NXT-20676])

*Mockup Note:* Area is always before Site

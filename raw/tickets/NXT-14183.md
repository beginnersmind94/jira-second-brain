# NXT-14183 — Direct Certification - DC Matching Pass Changes

## Description
As a district admin I want to see clearer matching “passes” to avoid confusion and increase confidence. 

*mockup*

tbd

*requirements*

We want to streamline some of the DC Match Passes to be a bit more clear. This will mean introducing new match fields, removing some, and adjusting the DC match process to accommodate.

* Remove the following:
** Line 1
** Line 2
** City
** State
** ZipCode
** County Code
** FullName
** (FirstName+LastName)-MiddleName
** LastName+MiddleName-FirstName
** FirstName+MiddleName-LastName
** FirstName-(LastName+MiddleName)
** LastName-(FirstName+MiddleName)
* Add the following:
** First Name
** Last Name
** Middle Name
** DOB Twist (DD/MM)
* Update the DC Process to take the changes in fields into account. Perform matching on first name, last name, middle name.
* Don’t record anything under the Discard threshold. Actually discard it.

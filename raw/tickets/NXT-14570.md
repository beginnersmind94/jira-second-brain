# NXT-14570 — INV - Create Orders - Shopping Cart

## Description
As an ordering manager I want a shopping cart on the Create Orders process so I can add items to the cart and view everything I’ve added.

*Requirement*

* Cart will be a slide-out
** ***Note:* check with Platform Team about the slide-out control. Currently there is no control for the collapsed view of the cart, where only the images are shown with the expand button.
* Provide a quick search at the top
** single search field for Item # or Item Name
** quick filter buttons like: Food, Non-Food, etc.
* will contain a list of items which have been added
** the section with items will be scrollable
* Item information will be provided:
** Item Number
** Item Name
** Image
*** will be placeholders until images are fully supported/implemented in Item Management
* Item Quantity will be shown
** Whole Unit Quantity
*** can be updated
** Broken Unit Quantity
*** can be updated
* Delete icon to remove item from the cart and order
* A fixed section at the bottom containing Next button
** *Next* button to proceed to Order Confirmation screen [https://primeroedgenext.atlassian.net/browse/NXT-5714|https://primeroedgenext.atlassian.net/browse/NXT-5714|smart-link]



* API to get images from Item Management
** check with Dan/Guarav about images

## Acceptance Criteria
* Cart will be collapsed by default
* Images will be pulled from Item Management
* Quantities can be updated in the cart
* WU/BU fields will be enabled/disabled according to the item configuration, matching the way the item is displayed on the page before being added to the cart
* The section containing the items will be vertically scrollable
* The section containing the search at the top and the Next button at the bottom will be fixed and not scroll
* Next button will advance the user to the Confirm Orders screen

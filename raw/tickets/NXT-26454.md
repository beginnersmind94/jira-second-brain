*requirements*

* Add the Serving Exceptions task and populate it.

*Suspicious Transactions Task*

* Task Name: Serving Exceptions
* Summary:
** Show the number of each serving exception.
** Examples:
*** 3 Eligibility Variances Found
*** 5 Duplicate Meals Found
* Details:
** Show how many have been resolved.
** Example:
*** 2 Eligibility Variances Resolved
*** 0 Duplicate Meals Resolved
* Status
** Status is based on them completing them all.
** 0% resolved = Not Started
** 100% resolved = Done
** Anything in between = Not Done
* Actions
** Done / Not Done
** History

## Acceptance Criteria
* Serving Exceptions task created.
* This task is only shown if it is Active.
* Summary populates based on whether any serving exceptions exist for that site + date.
** No serving exceptions, text = No Serving Exceptions Found
** If serving exceptions exist, break them out by type:
*** X Eligibility Variances Found
*** X Duplicate Meals Found
* Details populates how many have been resolved.
** If none exist, leave blank.
* Status is based on how many have been resolved.
** Exceptions exist but 0 resolved, status = Not Started
** Exceptions exist but >0 resolved, status = Not Done
** Exceptions exist but all are resolved, status = Done
** 0 Exceptions exist, status = Done

## Acceptance Criteria
Navigation: GC Portal - Search - Select Any Data Source

Changes Required:
User selects the Data Source dropdown
User should be able to enter the GTIN, Item Description, Manufacturer (to be listed as dropdown, text based search should list the items from dropdown ) 
The Brand, Manufacturer list should be based on Master Data Tables (Grouping Name) 

The search results should be listed in the Grid as shown in the prototype (Item, Manufacturer, Brand, GTIN)

Each item will have Hyper link on Item Name and Eye Icon at the end to view the record

Upon clicking Eye icon or Hyperlink, the item should be open in a new page (with a back button to resume to search), the search results should be retained

Allergen - match and render based on GTIN in Cybersoft Catalog, if not found, *look for external data sources: Future Scope: (Open Foods, UPC Item DB, Bar Code Look Up etc)*
Meal Credits - match and render based on GTIN in Cybersoft Catalog

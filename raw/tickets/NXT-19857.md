# NXT-19857 — IM- Recipe- Strict Batching

## Description
As a District Manager, I want to have control of the batch sizes my kitchens can produce and have a way to only allow them to make recipes based on strict batching. 

When a recipe is created and the user provides the number of servings, we need to give them a way to mark the item for "Strict Batching". This will be an indicator for other functionalities that will be using the recipe information. 

There also needs to be a display that strict batching is enabled.

The user should able to add the flag during recipe creation process as well as to edit. This should apply to both scenarios (split/List view).

The scale recipe process must enforce full batch use.

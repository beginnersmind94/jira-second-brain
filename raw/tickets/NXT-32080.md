# NXT-32080 — INV - Transfers - Transfer Details - Pack Size updates

## Description
As a Site Manager I want the Transfers Details to reflect updated pack size information by presenting the item as one unit when viewing but multiple units when editing so I can manage my transfer quantities exactly as needed.

*Requirements*

* Update the Transfer Details screen to reflect the updated item hierarchy configuration
* Rename From Site to ‘Sending Site’
* Rename To Site to ‘Receiving Site’

*Edit Mode*

* +Update the UI to accommodate multiple unit types+
* Replace Unit Type with Unit Description
** remove the dedicated column, as Unit will be selected after Quantity
* Provide a Unit dropdown after the Transfer Quantity
** Display a label instead of the dropdown if there is only one unit in the pack size configuration
** By default, pick the highest unit for which there is quantity on hand
* Provide an icon to expand the pack size hierarchy
** Only if the item has multiple possible units
*** this adds rows below for each sub-unit in the pack size hierarchy
** Remove the dropdown and lock the row to the specific unit label
** Provide an icon to collapse the pack size hierarchy if the sub-units aren’t being used
* Maintain existing validations for editing/saving
* Cannot increase Transfer Quantity
** can only decrease

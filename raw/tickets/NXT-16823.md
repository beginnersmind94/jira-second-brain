# NXT-16823 — INV - Orders - Order Detail - Receipts tab - Display tab for each receipt

## Description
As an inventory user I want a Receipts view on the Order Detail page so I can view my list of items in terms of how many have been received.

This would be the 2nd tab of the Order Detail page. This is where Receipts would be displayed

In this view, the Item List would have additional or different columns to show the Receipt Quantity.

+Story Scope:+

* Adding a “tab” for each receipt, showing the banner, showing View Receipt button 
* Showing the Charges & Discounts section if the individual receipt had any charges or discounts ([NXT-16824|https://primeroedgenext.atlassian.net/browse/NXT-16824])

*Requirements*

* Add a “tab” button for each individual receipt
** Receipt # is the tab name
** Banner: ‘Receipt # 12345 was created on 1/1/2021 by Justin Miller with a total of $123.45.’
** Provide a View Receipt button to access the Receipt Detail screen
* Display the receipt item details in the summary grid
** Item #
** Item Description
** Ordered Quantity
** Received Quantity

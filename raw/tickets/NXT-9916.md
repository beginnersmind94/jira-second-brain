# NXT-9916 — INV - Transfers - Create Transfer Approval Transaction (Site)

## Description
As an Inventory user I want the transfer transaction to go through an approval process, instead of automatically withdrawing and moving to another site

This is needed for future UI screens in [https://primeroedgenext.atlassian.net/browse/NXT-9571|https://primeroedgenext.atlassian.net/browse/NXT-9571|smart-link] 

‘Auto Approve’ setting to be added to System and default to ‘False’ at this time. This is needed since the auto approval transaction details are handled in [https://primeroedgenext.atlassian.net/browse/NXT-9568|https://primeroedgenext.atlassian.net/browse/NXT-9568|smart-link]

Transaction will need to go to the ‘incomplete transaction’ table until approved, rejected, or cancelled.

No UI work should be needed for this at this time.

## Acceptance Criteria
Transaction for the transfer needs to go to incomplete table until approved, rejected, or cancelled

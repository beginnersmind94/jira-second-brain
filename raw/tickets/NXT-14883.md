# NXT-14883 — Platform - PII - Label PII tag for PII fields

## Description
*Persona*: Perseus User

*Access level:* School District/Site

*Description*:

As a Perseus user, I should be able to identify the PII information and tag them for each resource so that proper access controls can protect the sensitive information from any potential security breach

* *PII Fields:*
** First Name
** Last Name 
** Parents Name
** Address
** Student Number
** Birth Date (DOB)
** Gender
** Ethnicity
** Race
** Eligibility
* *Sensitive Sections: (HIPAA, FERPA compliance)*
** Allergens
** Eligibility
** Household Information

## Acceptance Criteria
* PII flag edit must be recorded in the audit logging/history when modified on the Roles & Permissions page
* PII flag disable should prompt for user confirmation (on Roles & Permissions page)

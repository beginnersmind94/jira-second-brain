# NXT-41673 — SIS Auto Sync – Delete Configuration - Dev

## Description
As a SchoolCafé admin, I would like to delete the SIS Auto Sync configuration fields so that I can make update an SIS import configuration.

*Requirements*:

* On the System > Imports & Exports > SIS Auto Sync tab > SIS Configuration drawer, do the following:
* 
* Update the Provider drawer delete button:
** When the user selects a provider that has been saved:
** Show the delete button
** Check if there are any imports using the connection
*** If yes, then you will delete all the imports associated with this configuration
*** If no, then no extra work required
** On click:
*** Show the usual “are you sure” confirmation box
** Text: This will also delete any imports using this configuration. Are you sure?
*** If Yes, then delete the configuration from the database and delete any imports, regardless of status, that are using the configuration

## Acceptance Criteria
* Button is disabled if the configuration is being used
* Tooltip for button if disabled is shown on hover over
* If button clicked while enabled, it remove the configuration from the database and delete any imports that use the configuration

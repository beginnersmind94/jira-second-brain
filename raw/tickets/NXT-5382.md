# NXT-5382 — Accountability - Claims - Checklist (Phase 1)

## Description
Runs analysis before generation to make sure meals and edit checks are accounted for. If review is needed, we show the ability to edit.

Review for any serving exceptions, duplicate meals, etc to make sure all meals are properly accounted for and tracked. This activity typically occurs on the daily activity checklist.

Review flagged edit checks and give reconciliation comments.

Once all things are good to go, the user can proceed to generating the claim.

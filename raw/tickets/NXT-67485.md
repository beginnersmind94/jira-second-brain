# NXT-67485 — Menu Analysis- Sites Performance - Add Produced Column & Totals for all columns

## Description
As a user, I would like a way to see the produced counts per site as well as the totals for all my metrics for the selected sites. 

!image-20251015-190049.png|width=1032,alt="image-20251015-190049.png"!

## Acceptance Criteria
* Add a Produced Column between Planned and Served
** Rename all columns as Portions not Meals
* Add a Wasted Portions after Served Portions
* Add a Totals row that is a total of all the sites that are filtered and selected 
** And all Totals per site and give a total
** For Percentage, add all percentages and divide by number sites filtered
* Include a “Total” Score and this should be calculated based on totals for all sites selected (Average of all sites)
* Waste Percentage formula (SUM(Waste) * 1.0 / SUM(ServedCount + Waste) * 100) 
* Production Accuracy formula (SUM(ServedCount + Waste) / SUM(Planned)) 
** here we have a case if we dont have planned count then that is 0%
* Add a filter to all for site selection
* Provide filtering for each column header for ascending and descending order, do not include Totals row as part of the filter
* Update dashboard to also include Produced and Wasted and change meals to portions

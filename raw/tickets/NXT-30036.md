# NXT-30036 — IM - Items - Create New Item - Initial Info (Step 1)

## Description
As a District Admin I want the ability to create a new item and provide the identifying information.

*Notes:*

* Use a new entry point; we will replace the existing button once complete
* Only focusing on *List View* for now
* Use new APIs / Components where possible
** to ensure fresh approach

*Requirements*

* Load the new initial item creation screen upon click on +New Item button
* Show a toggle/switch/checkbox for the Active flag 
** defaulted to Active
* Display the identifying information fields
** Item Description (formerly Name) - *Mandatory*
*** allow up to 250 characters to be typed (though the text field does not need to show all 250 characters at once)
** Brand Name (dropdown)
*** *Note:* Brand Name will no longer be dependent on Manufacturer selection
*** keep the ‘Manage Brands’ shortcut 
**** future permission based
** Product Code
** Manufacturer (dropdown)
*** keep the ‘Manage Manufacturer’ shortcut 
**** future permission based
* Display Data Source dropdown
** only Local for now
** we may choose to hide this altogether until we are providing proper Data Source support
* Provide the Tags field component
* Provide the check for duplicate items
** as currently exists
* Provide Save & Continue button which would advance to the next step and load more of the screen
** Upon first Save, the item is officially created and assigned an Item #
*** *Note: at some point we will allow users to type their own Item # and it would be a user-defined field

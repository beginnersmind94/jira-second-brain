# NXT-24590 — MP > Menus > Search > Active vs. Inactive 

## Description
When using Menu Planning in the software, and creating menus, those menus are automatically marked as "Active" in the system.

Only "Active" menus should show up on the Menus page to search.

When I change a menu to "Inactive" - I do not want it to always show up in the list of Menus on the Menus page.  However, I need to have an "option" that will allow me to view the Inactive menus, should I want to.

## Acceptance Criteria
When searching for Menus, only Active menus will be displayed in the results. I will have an option to include "Inactive" menus, should need access to one of those that I may have previously marked as "Not Active"

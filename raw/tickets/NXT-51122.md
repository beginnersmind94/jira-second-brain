# NXT-51122 — POS - Dashboard - Daily Stats section

## Description
As Cassie the Cashier I want to see at a high level what the daily stats are.

*mockup:* [https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=6164%3A56305&node-id=6235-51754&viewport=1029%2C601%2C0.25&t=XHQRhl7YSMh0guzO-1&scaling=scale-down-width|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=6164%3A56305&node-id=6235-51754&viewport=1029%2C601%2C0.25&t=XHQRhl7YSMh0guzO-1&scaling=scale-down-width|smart-link] 

*requirements*

* Add the ‘Today’s Stats' section.
** Totals from all sessions opened with today’s date at that terminal.
** Stats:
*** Sessions
**** Number of sessions that have been opened for today at this terminal
*** Meals
**** Number of student meals sold for today at this terminal
*** Transactions
**** Number of transactions completed for today at this terminal
*** Payments
**** Sum of Payments made for today at this terminal
* We will be adding Site Settings for the control of these under [https://primeroedgenext.atlassian.net/browse/NXT-51124|https://primeroedgenext.atlassian.net/browse/NXT-51124|smart-link]

## Acceptance Criteria
* User can see how many Sessions were opened today at their terminal.
* User can see how many student Meals were sold today at their terminal.
* User can see how many Transactions were completed at their terminal.
** Fully voided transactions should not be included in this count.
* User can see the total sum of Payments made at their terminal.

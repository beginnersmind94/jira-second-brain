# NXT-11065 — Notifications - Family Hub - Application Notification

## Description
(This story will hopefully be testing only)

----

As a SchoolCafé parent/guardian I want to receive an official application results (eligibility determination) notification via SchoolCafé so that I can know the results of the application I submitted.

*mockup*

n/a

*requirements*

Once we have the automatic notifications setting available we can do the following:

* When a notification action is triggered (email and/or print) for an application, we should check if that application has a student(s) that is tied to a Family Hub parent account. If connected, we should send a notice to Family Hub. 
** This would be if the application is entered manually also. Check the guardians for a schoolcafe account - if it exists: 
*** send the notification up to schoolcafe
* This functionality should work regardless of if the notice is triggered via a user OR if it is triggered via an automated process.

## Acceptance Criteria
If a notification is triggered for the application and the submission method is "Online", SchoolCafe applicant can see the application results (eligibility determination) notification letter.

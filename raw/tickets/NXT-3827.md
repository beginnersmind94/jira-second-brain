# NXT-3827 — Account Management - Advanced Search - Demographics Section - Race Subsection

## Description
The Race Subsection will allow the user to filter the patrons list by their Race. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Race values. The user can type in the name or a partial name. The Names are listed in Alphabetical order. As names are selected they are moved to the top of the list. Unselected names are separated from selected names and are shown in alphabetical order.

!Screen Shot 2020-04-29 at 2.32.14 PM.png|width=284,height=377!

## Acceptance Criteria
The filter values are populated from the DB table

There is a search field that allows the user to search for a name

The search field shows partial matches.

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Race section they are 'or'ed together.

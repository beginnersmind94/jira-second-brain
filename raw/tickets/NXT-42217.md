# NXT-42217 — POS - Summary Sales - Closing Balance screen, setting to control A la Carte boxes

## Description
As Sammy the Site Manager I want to enter in funds taken as part of a session I am recording after the fact.

*requirements*

* Add a POS Site Setting that controls the A La Carte boxes being available:
** Name: Summary Sales - Show ‘A la Carte’ fields
** Values: On/Off
** Default: Off
** Effect: If the setting is on, show the boxes in Summary Sale. If not, hide them.
** Code: POSALCSS
** Description: If this setting is enabled, the POS will show a la carte fields in Summary Sales so that the total sales amount for a la carte items can be added as a lump sum.
* At the POS - update the workflow for Summary Sales to include a Closing Balance screen.

## Acceptance Criteria
* POS Site Setting added
* POS updated to use the new Site Setting
* POS updated for Summary Sale to include a Closing Balance screen as part of session closure.
* Closing Balance data sent up and processed accordingly.

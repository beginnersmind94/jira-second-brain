# NXT-28373 — MP- Menu Page- Display "On Calendar" Column

## Description
As a user, I would like to be able to see which of my menus have been assigned on a calendar.

## Acceptance Criteria
Display a column that displays On Calendar

Yes= Is assigned on the calendar

No= Has never been assigned on the calendar 

Any assignment applies no matter how far in the past or the future it is. If the menu has been assigned, display true.

Allow users to search thru their menus by whether its on the calendar or not.

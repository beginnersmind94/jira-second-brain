# NXT-20490 — Production- Reports- Technicians Worksheet (Report)

## Description
This report is a way for users to print all of their items and enter the numbers with pen and paper to use to enter production records. This report also will be able to be generated from the Production Record screen as well. 

Page Display needs to have this criteria:
Site Code
Site
Site Group
Meal Type
*From* and to *Date*

Current Primeroedge Display:

!image-2021-08-26-15-03-11-436.png|width=200,height=183!

 

*Report must contain the following information*:
Serving groups with their Planned Counts and a space for them to write actual counts
A total row for planned counts and actual counts
Plan comments box
Record Comments Box

*Columns must include*
Serving size and if they have a portioning tool display what it is 
Planned
Produced
Carryovers
Return to Stock
Waste
Adult/A La Carte ( Set as blank)

Items with Recipe code display in Category order within the category and what their Choose is and nutrition contribution
Example:
Category: Entrees: May choose 1
Cereal Cinnamon Toast Crunch (SY21-22) (LR 1031) 2 WGR

This item also needs to have food required associated with it which should pull based on Planned Counts.

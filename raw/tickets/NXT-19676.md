# NXT-19676 — Platform - BO - Onboarding Settings Summary Widget

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view system modules summary widgets with its overall progress and take necessary actions

*Mockup:*

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|smart-link] 

*Requirements*:

* Site admin logs into Onboarding Summary Dashboard page for Settings widgets displayed in “Dashboard” tab
* *Settings* tab displays all Settings that has been configured for given district
* Settings table displays STATUS of Settings  imports
* Settings table includes actions available for each module to view or EDIT the setting
* Settings widget has VIEW DETAILS button which opens Settings tab in the onboarding page.
* Settings widget has “ADD NEW” setting which brings up the pop-up to add setting per selected module

## Acceptance Criteria
* User shall be able to see the Settings widget in Onboarding dashboard
* User shall be able to see VIEW ALL button to open Settings tab under onboarding page.
* User shall be able to click on “ADD NEW” button to add new Settings from the available settings drop-down per module
* User shall be able to see number Settings configured for each module
* User shall be able to VIEW each Setting to provide an expanded view of Settings detail

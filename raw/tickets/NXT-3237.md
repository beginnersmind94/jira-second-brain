# NXT-3237 — ** System - Organization - Sites - Add Site

## Description
As an administrator I want to add new sites my district, along with their details, characteristics, and settings, so that each site is configured to properly work with each module within the PrimeroEdge application.

Mock-up - [https://nqyc9v.axshare.com/#id=xajrt8&p=sites&g=1|https://nqyc9v.axshare.com/#id=xajrt8&p=sites&g=1]

Note: Module settings will be addressed in a separate story and can be ignored for now.

## Acceptance Criteria
1. Users with permissions for adding a site should be able to see/click "Add site"
2. Users should be able to modify fields and save changes so that a new site can be added
3. Users should be able to cancel changes and return to main Sites list/view
4. Existing required fields and logic should be maintained from PrimeroEdge Classic. (i.e. Site Name and Description should be still unique)
5. Site is active by default
6. Site Name, Site Description and Site Code should be unique
7. Site Types should correspond to selected Region

# NXT-13342 — Direct Certification - Matching - Track exact points allocated at time of matching

## Description
As a district admin I have made recent changes to the points allocated for matching criteria, and I want historical scorecards to remain accurate.

*requirements*

* User can open a historical scorecard and see the exact points given for a matching pass as of that date.
** Example: 
*** Full Name was 60 points for a match done 1 year ago.
*** User changes Full Name criteria to 40 points.
*** Opening the scorecard for the match from 1 year ago should still show 60 pts were given.



Discussion 6/21/22

* So when a match is processed we’ll record the points that were involved and for what pass they were in.
* Add a note to explain that.
** Note: Match Points are from the time the match was generated/processed.

## Acceptance Criteria
User can open a historical scorecard and see the exact points given for a matching pass as of that date when the match was processed.

# NXT-40066 — Student Export – Update UI - Table Functionality 

## Description
As a SchoolCafé Admin I want to see the Field Mapping Screen more consistent with our site UI so that I can not be distracted with changes.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Exports
* Go through the Pre-export and get to the Field Mapping screen
* Pick which fields you will export and then finish the export

*Notes:*

*Requirements*:

* This story is for the SchoolCafé Settings > Imports & Exports > Exports > Field Mapping tab
* This story applies to the Student Export
* When in Student Export on the Field Mapping Screen, do the following:
* Make the tab look like the Figma:
** Change Color Scheme to match
** Change header column height to match
** Combine into one table
** Use the toggle instead of the edit and delete icons
** Use the Code mapping on the right side instead of the ellipsis next to the name

## Acceptance Criteria
* Tab looks like Figma
* Combined into one table
* Uses the toggle instead of the edit and delete icons
* Uses the Code mapping on the right side instead of the ellipsis next to the name

# NXT-17876 — MP- Add Weekends on Menu Planner

## Description
As a user, there are times where I need to create a 7 day Menu Cycle as I am serving kids on the weekends. Many RCCI's use this process.

Mock up

!image-2021-08-17-12-49-35-438.png|width=200,height=183!

 

Proposed option would be a on/off toggle to include weekends. Menu Planner Screen will default to not include weekends but will have the option for a user to turn on. This will not be included in what the save button does when saving original menu criteria.

Scenario- If user starts building menus and decide to turn on weekends, system should recognize the items already included and save before refreshing page to include weekends. 

Order of days should be:

If weekends selected:   Sunday - Monday - Tuesday - Wednesday - Thursday - Friday - Saturday 

If NO weekends: Monday - Tuesday - Wednesday - Thursday - Friday

## Acceptance Criteria
User is given an option to "turn" on or off to include weekends. All days of the week should appear when option is turned on. 

Page should autosave when user decides to include weekends if items are already added onto other weekdays.

Order of days should be:

If weekends selected:   Sunday - Monday - Tuesday - Wednesday - Thursday - Friday - Saturday 

If NO weekends: Monday - Tuesday - Wednesday - Thursday - Friday

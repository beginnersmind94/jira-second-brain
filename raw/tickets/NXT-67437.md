# NXT-67437 — ACC - Attendance Feature - System Setting for Attendance Tie-ins

## Description
New system setting: 

* Name: Calculate Attendance Factor from Daily Attendance Records
* Description: This setting impacts Edit Checks, Edit Check reports, and the Meal Count report. When set to no, the entered Attendance Factor (from Monthly) will be used. When set to yes, Attendance Factor will be calculated (using Daily Attendance records).
* Configuration Type: Common
* Code: CALCAFEC
* Category: Configuration
* Module: Accountability
* Values: No (Default) / Yes


Will also need to affect.

Meal count report (with optional override default checked when setting is on)

workspace dashboard

# NXT-43253 — Production API for Insights - Return to Stock - Region, Site Type and Site Level

## Description
_(none)_

## Acceptance Criteria
{color:#bf2600}+*Note*+ {color}- all the fields outlined in the story is already available, the change is only to include the Return to Stock Percentage as part of the existing end point

*Requirements:*

·        API - To include Return to Stock Percentage as part of the existing end point

*Input :*

·        Region ID (Ex: -9999 Represents all site)

·        Start Date (Ex: 2022-07-01, represents the Serving date)

·        End Date (Ex: 2022-07-01, represents the Serving date)

 

*API Output:*

For the Production Date Range Defined, the API should return the following fields at Site Level, this data shall be used in AI/ML to show site level performance

|*Field Name*|*Definition*|
|Region / District|District ID|
|Site Type (School Type)|Type of the school (Ex: Elementary)|
|Site Code|Site Code - Ex: 0001|
|Site Description|Site Name - Ex: Harper High School|
|Total Planned Count|Planned Count of Production Record (Sum of all the Record Totals)|
|Total Produced Quantity|Produced Quantity of the Menu (Sum of all the Record Totals)|
|Carry Over|Total Number of Menu items entered as Carry Over (Sum of all the Record Totals)|
|Returned to Stock|Total Number of Menu items entered as Returned to Stock (Sum of all the Record Totals)|
|Wastage|Total Number of Menu entered as Waste in Production Record (Sum of all the Record Totals)|
|Total Left Over|Sum of all the Record Totals entered as Left Over (Sum of Carry over + Returned to Stock + Wastage)|
|Total Served|Total Number of Menu Items Served (Sum of all the Record Totals)|
|Production Accuracy %|Percentage of Production Accuracy (Total Produced / Total Planned) *100|
|Served %|Percentage of Served (Total Served / Total Produced) *100|
|Waste %|Percentage of Waste (Total Wastage / Total Produced) *100|
|{color:#ff991f}*Returned to Stock %*{color}|{color:#ff991f}Percentage of Stock items Returned (Total Returned to Stock/ Total Produced) *100{color}|
|Carry Over %|Percentage of Carry Over (Total Carry Over / Total Produced) *100|
|Left Over %|Percentage of Left Over ((Carry Over + Returned to Stock) / Total Produced) *100)|

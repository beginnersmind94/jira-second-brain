# NXT-19174 — Production- Dashboard- Meal Cost View

## Description
As a user, on my dashboard I would like to have a view where I could see my average/total Meal Cost per meal type.

There will be different views for this depending on user permissions as well as what type of view they select. User permissions will determine what view user will have access to. For example, a site manager in most circumstances will not have access to the district or site group view. A regional manager will have access to site group and site view but will most likely not have access to the district view. 

*ALL DASHBOARD VIEWS WILL HAVE THE OPTION FOR YTD ( YEAR TO DATE), Last 30 DAYS, PAST 7 DAYS, TODAY*

District View (All): Selecting this view will allow user to view average meal cost per meal type based on meal count for the district broken down by site groups. Information should be displayed in a bar graph to make it easy for the user to interpret. 

Site Group View: Selecting this view will allow user to view average meal cost per meal type based on meal count per site. This could be broken down further by allowing the user to select a particular menu line to look at cost based on average meal count. They will be given two options to look at the top 10 highest average meal costs, as well as the lowest 10 average meal costs. Information should be displayed in a bar graph to make it easy for  the user to interpret.

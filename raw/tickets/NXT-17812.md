# NXT-17812 — INV - Settings - Delivery Locations - New Site Delivery Location

## Description
As a Food Service Director I want a tool to add new site delivery locations so I can configure all the locations for my sites.

Continuation of [https://primeroedgenext.atlassian.net/browse/NXT-16355|https://primeroedgenext.atlassian.net/browse/NXT-16355|smart-link] 

*Requirements*

* Open a new ‘Site Delivery Location Details’ screen when user clicks the Add button from the Delivery Locations page
** This page will allow the user to add new delivery locations for a particular site
* Provide a small header section to select a Site
** Provide a Site dropdown, showing both Code and Name
*** this is required in order to start adding delivery locations
*** single selection (no ‘All’ option)
** Apply button
*** enable once Site is selected
* Provide another section for the user to add delivery locations
** This section is loaded once a Site has been selected 
* Provide the fields for adding a new location
** Delivery Location Description textbox
*** Alphanumeric field to type the delivery location description
*** Mandatory
*** Unique within Site
**** “Description must be unique.”
**** Display underneath the textbox
*** Up to 50 characters
** Delivery Location Code textbox
*** Numeric field to provide a code for the delivery location
*** Mandatory
*** Unique within Site
**** “Code must be unique.”
**** Display underneath the textbox
*** up to 10 characters
** Add button
*** this will add the location to a list below containing all locations for the site
** Reset these fields each time a location is successfully added 
* Display the list of added delivery locations
** Delivery Location Description
** Delivery Location Code
** Status column
*** This would show Active or Inactive
** Action column (covered in [NXT-17813|https://primeroedgenext.atlassian.net/browse/NXT-17813])
*** Edit icon
*** Delete icon
* Provide a Save button, to officially save the new site configuration
* Provide a Back button, to return to the list page ([NXT-16355|https://primeroedgenext.atlassian.net/browse/NXT-16355])
** Provide unsaved changes prompt if user clicks Back before saving

* Update the existing Contract Items page with the UI changes from the updated Vendor > Items tab

** Move the action buttons to the sticky footer

* Display alert icons (Missing Mandatory Attributes, Unorderable, Pending Deactivation, etc.)
* Update the header to match the Vendor Items
** remove Item Category dropdown
** add ‘Items missing mandatory attributes’ toggle
* Fix missing icon for Pricing slideout
* Use the restyled ‘Update’ button
* Add Manage Views
* Determine whether to use pagination or not
** Ideally use the same approach as the Vendor Items

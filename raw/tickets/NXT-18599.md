# NXT-18599 — Production- Reports- Print Production Recipes (Planned Recipes)

## Description
As a user, I want to be able to see a page that will give me all my servings for a date range so that I can get an idea of how many servings I will be serving of a particular Menu Item per menu line on a certain day. 
EX: 

!image-2021-08-31-10-41-11-268.png|width=200,height=183!

 

This should be an interactive page to allow user to search for a particular menu item, recipe or category once they enter in site, site code and date range. 

Columns that should display information:

*Menu Item*

*Recipe Code*

*Category Description*

*#Servings (Size) (Display number of  Total Planned Servings and  what the serving size is)*

The information that populates could also be printed in a report format either selecting all the items or just a particular few items.

## Acceptance Criteria
This should be an interactive page to allow user to search for a particular menu item, recipe or category once they enter in site, site code and date range. 

Columns that should display information:
Menu Item
Recipe #
Category

Servings (size of serving) 

If there are multiple serving sizes, display on separate lines along with the # of servings that belong to each serving size

Each Meal Type, Serving Date and Menu Line Combination would be displayed in a separate accordion along with the items that belong to that day

Order by Date, then meal type (Breakfast, Lunch, Snack, Supper) , then menu line (alpha)

Allow Field Search on Grid for all columns except # of Servings

When generating report, only the items selected will display in the report with *scaled production recipes* to produce those items.

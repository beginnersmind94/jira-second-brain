# NXT-16286 — INV - Orders - Orders List - View Receipts - Receipt List

## Description
As an Inventory user I want a Receipt List on the Receipts tab of the Manage Orders page so I can view a list of all receipts, view relevant receipt info, and open specific receipts.

*Requirements*

* Provide a grid containing a list of receipts returned by the header selections
* This grid will be populated according to default criteria when page is opened (i.e. don’t require click of Apply button)
* Provide the ‘Columns’ button containing additional optional columns to be selected
* Grid columns:
** Order #
*** searchable
*** Link to the Order Detail 
** Site
** Vendor
** +Receipt Attributes+ - Orders can have multiple receipts but we would want to keep Order # as the primary ID. This may mean listing an Order # multiple times for each Receipt
*** Receipt #
**** searchable
*** Invoice #
**** searchable
*** Receipt Amount
**** searchable
*** Invoice Amount
**** searchable
*** Discrepancy Amount
**** searchable
*** Receipt Date
** Action column
*** View icon to open receipt details [https://primeroedgenext.atlassian.net/browse/NXT-17009|https://primeroedgenext.atlassian.net/browse/NXT-17009|smart-link] 
** Delivery Date 
** Age (of Receipt)
*** number of days

# NXT-8119 — Messages - Allow user to view & download attachments in message

## Description
*User Story*
As a PrimeroEdge user sending a message, I would like the ability to view and download attachments that I receive along with my messages, so that I can view their full details and/or save them for later.

*Summary*

When a user opens a message that has attachments, allow them to view and download each of the attachments.

*Requirements*

* When opening a message from the Workspace Inbox summary or full inbox, if the message includes attachments, display the following attachment summary information for each attachment:
** File name
** File type
** File size
** Download icon
* When the user selects the Download icon, open a Save File window that allows the user to save the file to their machine locally.

## Acceptance Criteria
Verify that when opening a message from the Workspace Inbox summary, if the message includes attachments, the message displays the following attachment summary information for each attachment:
-File name
-File type
-File size
-Download icon

Verify the same behavior remains when the message is accessed from the user's full Inbox.

Verify that when the user selects the Download icon, open a Save File window that allows the user to save the file to their machine locally.

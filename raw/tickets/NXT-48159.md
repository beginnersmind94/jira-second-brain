# NXT-48159 — POS - Picture Mode - Hide Selected

## Description
As Carmen the Cashier I want to hide selected cards in Picture Mode so I can quickly select from remaining students without scrolling.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=1943%3A16680&type=design&node-id=1943-16947&viewport=294%2C783%2C0.13&t=QeAQF7Nf7Z0gbycZ-1&scaling=scale-down-width|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=1943%3A16680&type=design&node-id=1943-16947&viewport=294%2C783%2C0.13&t=QeAQF7Nf7Z0gbycZ-1&scaling=scale-down-width|smart-link] 

*requirements*

* User has an option to ‘Hide Selected’
** Add sorting options to the option menu added with [https://primeroedgenext.atlassian.net/browse/NXT-47956|https://primeroedgenext.atlassian.net/browse/NXT-47956|smart-link]  
** Option name: Hide Selected
** Options: On/Off (default)
* User’s option is saved and used as the default on load.
* If turned on, selected cards should disappear and the remaining cards should move up to fill the gap.

## Acceptance Criteria
* Option to hide selected added
* Option is saved as as a default
* If on, Selected cards disappear and remaining cards move up

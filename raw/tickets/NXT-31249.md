# NXT-31249 — Settings Page - Improve User Experience for Searching

## Description
Per feedback from SMEs, users would like the ability to search for settings throughout “Common”, “Internal”, “Modules”, and “Category”.

With the existing design, users have to step through each Category for every module in order to perform a search.



See the image below for the existing design which is very cumbersome.

!image-20220525-110200.png|width=1083,height=261!

## Acceptance Criteria
* As a user, I would like the ability to perform a single search that would search across all of common, internal, modules, and categories
* Per the attached Figma, the “Configuration Type”, Module, and Category dropdowns shall each include an “All” item.  A user’s search shall be performed within the set of configurations that match those dropdown selections.

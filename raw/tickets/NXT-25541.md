# NXT-25541 — MP - Creating a Holiday Calendar for District

## Description
As a FSD (Food Service Director), I would like a way to input my school holidays and off days that we have across the district or at individual sites.

This is used across multiple modules in Menu Planning, Production and Inventory. 

Currently Display in Classic:

!da88a1bc-a8f7-48f9-b970-19ab40368428-20220216-193529.png|width=83.33333333333333%!



!865476d8-cc47-423d-8e81-4f4ec03e7bae-20220216-193601.png|width=83.33333333333333%!

 

IN Classic however there was no option to assign holidays or days off on a site level.

## Acceptance Criteria
When creating a new Holiday
1. Allow user to select the Academic Year
2. Allow Site Selection with check boxes, (Default to all)
3. Allow user to add Holiday with a Starting and End date with date range picker
4. Allow free text description field (Holiday or day off name)
5. Check mark for Allow deliveries that will be used for inventory
6. Give the option to repeat weekly, monthly, or yearly  
For options:
Weekly- Display an option to Repeat for how many weeks (Display every record on a separate line on the grid)
Monthly- Display two options to repeat on same date for how many months and another option for day of the week  and display which day (based on Date originally selected) (first, second, third, or fourth) for how many months (Display every record on a separate line on the grid)
Yearly- Display an option to Repeat every how many years on specified date (Display every record on a separate line on the grid)

Once created:
Display Columns
Each Date with Holiday on it
Description 
Sites with Specific Holiday (Delivery option per site)
Added on (when it was added)
Allow deliveries (display whether yes or no)
Edit and Delete options ( Allow removing of a site or multiple sites from the Specific date or Holiday)

Atleast 1 site needs to be available on both the add and edit option. 

Only display this disclaimer if no sites are selected “One or more sites need to be selected to be able to save”

If all sites are removed or no site is added, disable the save button

Pop up message confirmation to delete ( Are you sure you want to delete this record?)

Place In System Module Page

Allow data to be consumed and placed in Que for modules to take information. 

*UPDATE 3/4/2022*

*Change from To Date to End Date.*

*Remove all End date from all reoccurrences.* 

*Any reoccurrence that is selected will continue on for the remaining academic years that are available.*

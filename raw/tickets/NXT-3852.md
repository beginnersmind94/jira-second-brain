# NXT-3852 — AcctMgt - Patron Split View - Sticky Bar

## Description
*User Story*

As a district administrator I want an easy way to jump between panels in the Detailed View.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1]

* Scroll down a bit and you’ll see the bar.

!image-20200824-151317.png|width=1290,height=58!



*Requirements*

* Add a bar to the top of the patron details that allows the user to jump between panels.
* Bar/panel should only be visible when we’re not at the top of the screen and have scrolled down.
* Bar should display the Name + ID + Status (Active/Inactive) of the loaded patron.
* Bar should have links to each section.
** Clicking on one should navigate down to that particular section.
* If too many sections / panels exist, bury the remainder under a More… dropdown.
* While scrolling down, the sticky bar should update to highlight which section the user is on.
** Example: they scroll to Enrollment, the text for Enrollment should update to indicate that that’s the section they’re on. Text color change, some kind of line highlight.

## Acceptance Criteria
Sticky Bar added.
If too many present, bury them in a dropdown.
Clicking on a section on the bar jumps the screen to that panel.
Multi-language support added.

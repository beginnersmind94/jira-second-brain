# NXT-31396 — IM - Items - Item Details - Enable Toggles for Tabs

## Description
As a District Admin I want to be able to enable Food, POS or Recipe/MI tab so I can change items even if I did not enable them at the time of item creation.

*Requirements*

* Show Enable toggles for the three main tabs after item is created if toggles were set to ‘No’
* Prompt on Enable flag for Food Info
** “Are you sure you want to convert this to a Food Item?” Yes/Cancel
** Load Serving Measures and Ingredient Info cards (same as creation flow)
* No prompt is required for enabling POS Info, Single Item Recipe, or Menu Item

# NXT-39157 — IM - Items - Units - Restrict Standard Measures from being pack size units

## Description
As an IM user I want the Standard Measures to be excluded from Pack Size units so the system doesn’t allow me to create improper pack size configurations.

*List of restricted Pack Size measures*

* all Volume measures
* all Weight measures
* all numbered scoops (*not* just ‘Scoop’)

!image-20230502-044428.png|width=83.33333333333334%!

## Acceptance Criteria
* Do not recommend standard measures for Pack Size unit descriptions
* Do *not* allow a Serving to be linked via sub-quantity if it is using a standard measure
** Also cannot become a Dual unit; ignore in Base Unit confirmation

# NXT-6829 — INV - Vendor - View Vendor Items - Item List

## Description
As an Inventory user I need an items list so I can view all the items that a vendor is supplying to my district.

*Requirements*

* A Contract dropdown for user to select a specific contract
** this dropdown will list only *active* contracts
*** no ‘All’ option for now
** selecting a contract from the dropdown will change/limit the items to only the items assigned to that contract
* Provide a table list with all the items added to that contract.
** Columns in the items table (all with searchable column headers):
*** Item # (this is intended to be District Item Number)
*** Name
*** Vendor Item # - contract specific attribute
*** Item Category
*** Storage Category
*** Purchase Price (current) - contract specific attribute
**** dollar value
**** ex: $4.5226
** table needs pagination
** sort alphabetically ascending by Name
* This list will display all the items that have been added to that vendor as part of any active vendor contract

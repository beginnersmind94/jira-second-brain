# NXT-35166 — Command Centre - Data Source Changes

## Description
Currently the data source is extracted via CSV dumps and gets processed to populate the FO sections

## Acceptance Criteria
The following sections in Command Centre should be using the same API as its used in FO Dashboards
Program Access (Approved Count in FO Accountability Dashboard)
Breakfast Participation 
Lunch Participation
 MEQ – Meals Equivalent

   Ex:  Breakfast – Student BF+ Adult BF; Lunch: Student Lunch + Adult Lunch;

Total Revenue (Reimbursable / Non-reimbursable in Command dashboard should be in Sync with FO Report

!image-20221115-223108.png|width=1975,height=662!

*School Type* Section should consume the same API but the data should be grouped by School Type and displayed as-Is

*School Section* should consume the same API but the data should be displayed at site level



Production Accuracy and Wastage is out of scope

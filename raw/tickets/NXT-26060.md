# NXT-26060 — Auto Mapping - Allergen Import

## Description
This story is to cover the tasks related to auto mapping of fields populated by reading the allergies in Source file and auto map them with the ones available in the Data Exchange Platform exchange database @ code mapping.

Note : If there are any allergens in the source file and are not present in code look up, a warning label at the bottom must be present that the missing items must be added in Item configuration via Item Management and selecting Custom Allergen configuration in configuration Options.

Please refer to the UX for more details

## Acceptance Criteria
Verify and Validate that the fields from source file maps with the ones in the data exchange platform database.


Verify and Validate that fields present in source file does not map if they are not present in data exchange platform database.



Verify and Validate that the steps to add missing allergens in red at the bottom of the page.

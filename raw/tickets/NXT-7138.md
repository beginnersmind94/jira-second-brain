# NXT-7138 — Direct Certification - Summary - Matched Students

## Description
*User Story*

I want to see a list of students who were directly impacted by the direct certification file I’ve submitted / processed / approved.

*Mockup*

TBD

*Requirements*

Inside of the Summary, show a list of Matched Students.

* Grid showing student information for those that received an eligibility update as a result of this file. 
** ID #
** Name (First + Last + Suffix)
** Site (Sitecode + Sitename)
** Grade
** File Date
** Match Date
** Effective Date
** Match Criteria
*** ‘Manual Match’ if done by Manual Entry
*** ‘Automatic Match’ if done by the future automated matching process
*** ‘Household Extension’ if done by the future household extension process
** Case # (if present)
** Approval Type
*** SNAP if the file was SNAP, Homeless if Homeless, etc.
** Contact Information
*** Primary Guardian Name
*** Address
*** Phone Numbers
*** Email Address
* User should be able to issue a letter of approval (a notification) from the student shown in the grid, or for all students in the grid.
** *These will have to be placeholders for the moment - we can skip putting them in for this story, but the design should keep their placement in mind.*

## Acceptance Criteria
Student list with appropriate information is available inside the summary of a DC file.

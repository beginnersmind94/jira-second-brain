# NXT-19675 — Platform - BO - Onboarding System Modules Summary Widget

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view system modules summary widgets with its overall progress and take necessary actions

*Mockup:*

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|smart-link] 

*Requirements*:

* Site admin logs into Onboarding Summary Dashboard page for Modules widgets displayed in “Dashboard” tab
* Modules tab displays all modules that has been licensed for given district
* Modules table displays number of licenses 
* Modules table includes actions available for each module to view or remove the module
* Modules widget has VIEW DETAILS button which opens System Modules tab in the onboarding page.

## Acceptance Criteria
* User shall be able to see the Modules widget in Onboarding dashboard
* User shall be able to see VIEW ALL button to open Modules tab under onboarding page.
* User shall be able to click on “ADD NEW” button to add new module from the available products drop-down
* User shall be able to see number modules licensed for each module
* User shall be able to view each module to provide an expanded view of module license details

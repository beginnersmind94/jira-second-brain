# NXT-3009 — ***View Inspections - Actions - Activity

## Description
*User Story*
As an Inspections user viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to view a list of all activity that Inspection has had.

*Requirements*

* Activity is an action that can be applied to any Inspection
** This action is present on all Inspections
** This action is available to all users who can view the Inspection
* When clicked, the user receives a slideout from the right of the screen with a list of activities
** There is a header in the popup with the Inspection name
** There is a table displaying the actions with these fields:
*** Action
*** Status
*** User
*** Date
*** Notes
** By default, actions are ordered in date and time descending
** Users can sort by any column
* The following actions trigger a new line:
** Creation
** Status Change
*** New status is displayed in the column
** Reassigned
** Rescheduled

## Acceptance Criteria
* User can view activity for any available Inspection from the View Inspections screen in either view (card or list)
* User can view activity on Inspection Details screen

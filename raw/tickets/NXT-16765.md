# NXT-16765 — Imports - Adding fields 

## Description
This story is to cover the task of user trying to add new field/fields by editing the existing configuration instead of creating a brand-new configuration.

This feature is already present in Classic version and allows the customer to save a lot of time and the whole process of creating brand new configuration and reduce confusion.

## Acceptance Criteria
Verify and validate that the new control “Add New field” is on the mapping page.

Verify and validate that the new fields can be renamed.

Verify and validate that the new fields are saved (both cases before rename and after rename)

Verify and validate that the new fields are only on the source side.

Verify and validate that the new fields can be edited and deleted incase the user needs from the actions menu.

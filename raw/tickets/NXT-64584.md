## Description
_(none)_

## Acceptance Criteria
Navigation: GC Portal - Search - Cybersoft Catalog

Changes Required:
User selects the Data Source dropdown
User should be able to enter the GTIN, Item Description, Manufacturer, Item Category (to be listed as dropdown, text based search should list the items from dropdown ) 

The search results should be listed in the Grid as shown in the prototype (Item, Manufacturer, Brand, GTIN, Category)

Upon viewing the item, user should be able to see all the fields in UI

The end point shall facilitate updates at each record except the record ID (Ex: GL_Key)

Upon saving the record, the user should be able to view the revised record in the next search onwards
The saved record will show in new searches in PE and SC (Regression)

Unpublishing an Item - upon saving the status as unpublished, the user is no longer see that item in Search Results

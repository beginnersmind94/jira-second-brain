# NXT-14179 — IM - Catalog Integration - Import Items 

## Description
As a User I would like to have an ability to Import the Items selected

## Acceptance Criteria
Given the User has already Reviewed the Selected items 

And made necessary changes to the selected list 

When Import items button is tapped 

Then Import the items to the local data source and show a Message 

If the item already exist in the data source then display message: 

* Duplicate item

If it fails for any other reason then display:

“The Import was Failed”



If there was no problems importing the item then display:

As “The Import was Successful

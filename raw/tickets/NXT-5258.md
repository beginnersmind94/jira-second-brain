# NXT-5258 — Direct Certification - Unmatched - Scorecard

## Description
As a district admin I want to see what the matching criteria for a DC Match was, along with any point values that were applied towards the percentage total.

*requirements*

* We’re going to show this scorecard in a few places. 
** DC Matched - Info View
** DC Unmatched List
** Eligibility panel for a patron account (inside Account Management)
* The scorecard should show information about the student against information from the DC file.
* The information shown for both sides should be the exact same information that was used to create this potential match.
** Updates to student information should not update the information shown on this card.
* The rows to show relate to the iterative passes outlined in [https://primeroedgenext.atlassian.net/browse/NXT-7681|https://primeroedgenext.atlassian.net/browse/NXT-7681|smart-link] 
** Show the name of the pass, the student data, the DC file data
** If this was a successful data match - highlight the row and show the points awarded.
* A total percentage should be shown at the bottom.

## Acceptance Criteria
Scorecard is available for viewing for any DC match, be it potential, automatic, reviewed, etc.
Information shown on the card should be accurate as of the date the match was created in the system.
Point totals should not update if configuration values are changed.

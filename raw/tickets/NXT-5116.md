# NXT-5116 — Data Exchange - Import Framework

## Description
*User Story*

As a PrimeroEdge user, I have the ability to import data using a CSV or Excel file

*Requirements*

Set up the API framework for importing sets of data from the backend using a properly formatted CSV or Excel file.  File format is column-based, with the first line representing the name of each column and each row below representing a single entry.

This story only covers the backend; any import UI will be covered in future stories.

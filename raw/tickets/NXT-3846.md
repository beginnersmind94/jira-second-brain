# NXT-3846 — Dashboards

## Description
*User Story*
As an Insights user, I can view and create Dashboards using various enabled Measures.

*Mockup*


*Requirements*

* Dashboards are more interactive than reports and can include data from multiple modules to provide a user a higher level of insight into their data. 
** Dropdown filters and date sliders
** Visualizations can filter/adapt based on selections in other visualizations.
* Users can view a list of all existing Dashboards.
** PrimeroEdge created Dashboards
** User custom Dashboards
* User can create new dashboards
** Users can add Measures
*** Default KPIs
*** Custom KPIs
*** Metrics
** Users can select how Measures are visualized
*** Graphs/charts
*** Tables
*** Limited date ranges

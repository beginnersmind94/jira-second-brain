# NXT-61904 — IM - Bulk Item - Pack Size Info - Header filters

## Description
As an onboarding district admin I want filters within the Pack Size & Inventory Info section so I can more granularly select and manage items as I populate the pack size hierarchies and ready items for inventory publishing.

!image-20251021-193622.png|width=1032,alt="image-20251021-193622.png"!

## Acceptance Criteria
* Add filters to the Pack Size Info section
* Provide a Filter for Inventory Status
** Reference the Items > Advanced tab filters for some reference 
* Provide a custom ‘Is Missing’ dropdown on the Pack Size section review screen (Not Multi Select)
** Units & Sub-Quantities
** Serving-Pack Link 
** Base Unit Weight 
*** This is for items that have a standard measure or serving size not a 1
*** If user has this filter on and goes to update Serving-Pack Link, lead user to the weight section
** Cost/Value
* Provide a GTIN/UPC Search
* Provide additional filters on specific edit sub-steps
** GTIN/UPC: provide a simple toggle for ‘Items without any GTIN/UPC’
** Cost/Value: provide a simple toggle for ‘Only items missing Cost/Value’

# NXT-54914 — POS - Peripheral Device - POS Arrows via Hardware Reprogramming

## Description
Freshdesk ticket: [https://primeroedge.freshdesk.com/a/tickets/258419|https://primeroedge.freshdesk.com/a/tickets/258419]

As Sally the Site Manager I have 4 peripheral devices plugged into each terminal - two barcode scanners, two pinpads - and I need the POS software to identify each device separately and keep them separated.

*requirements*

Accepting the reprogramming method for pinpads at POS

* Updating PIN entry fields to accept alpha and special characters
* Translate this into what side the input came from and turn it back into numbers
* Show the left/right side arrow
* Find the student

The Titan Method of encapsulating/encrypting the ID and 12345 → ‘abcde’ → 12345 left

* left side pinpad → ‘abcde’ → (left) id: 12345
* right side pinpad → ,hijkl, → (right) id: 12345

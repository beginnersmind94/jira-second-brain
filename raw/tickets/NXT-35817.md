# NXT-35817 — INV - Config - Delivery Locations - CRUD permissions

## Description
As a district admin I want permissions for the various Configuration pages so I can make sure only the right users have access.

*Requirements*

* Map CRUD level permissions for the Delivery Locations config page
* View - access to page; visibility in navigation
** No other permissions specific to list page
** Reference sub-page View permission for visibility of View icon
*** i.e. hide the View icon if user doesn’t have View permission for DL Details
*** hide the entire Action column, ideally
** *+New button challenge*
*** Typical/current behavior: reference the list page ‘Add’ permission
**** *Issue:* this would require users to have both Add permissions
*** Ideal behavior: reference the sub-page (Details) Add permission
* Map permissions for sub-page Delivery Location Details
** Edit - access to the Edit pencil icon to change existing locations
*** Comments icon should also be connected to this
** Add - ability to add new location for a site
*** Can’t hide the entire card like Templates, because of Site dropdown
*** Must hide the DL Name, DL Code and Add button if permission isn’t assigned
** Delete - visibility of delete icon, to remove existing locations
*** all existing business rules must be maintained

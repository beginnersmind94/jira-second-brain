# NXT-61401 — INV - Distribution - Shipping - Modify Shipments - Move Orders to New Shipment

## Description
As a warehouse distribution manager I want the option to move orders to a new shipment, so I can create emergency or special shipments when circumstances require.

h3. Shipping tab - Modify Shipments - Move Orders to New Shipment

* New Shipment is an option at the bottom of the dropdown for the destination shipment

!Shipping - Modify - Move Orders to New Rev-20250529-195011.png|width=91.66666666666666%,alt="Shipping - Modify - Move Orders to New Rev-20250529-195011.png"!

h3. Shipping tab - after New Shipment

* Carry forward Shipping Group
* Carry forward Delivery Group
** append Alt
** if multiple groups, display X Groups
* Use ‘Custom Route’ for Route Name

!Shipping - After New Shipment-20250529-195111.png|width=91.66666666666666%,alt="Shipping - After New Shipment-20250529-195111.png"!

## Acceptance Criteria
* Provide ‘New Shipment’ as an option in the destination shipment dropdown of the Modify Shipments > Move Orders function
** this will create an entirely new shipment 
* Insert the new shipment into the Shipping tab
** carry forward Shipping Group from the source shipment
** carry forward Delivery Group from the source shipment
*** append Alt
** Use ‘Custom Route’ for Route Name
*** if multiple custom routes exist, increment e.g. Custom Route 2
* Disable the Move Orders button if all sites are selected to be moved to a new shipment

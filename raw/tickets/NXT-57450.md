# NXT-57450 — MVR - Users Grid - R&P

## Description
We need to add roles and permissions to the distribute button on the managed view drop down Distribute button so that users who do not have rights to MVR cannot attempt to create one

*Notes:*

* The underlying issue is that a user may not have rights to the MVR page and its functionality but does have rights to a grid
* If they try to do an MVR from the grid, they will error out and we don’t want this 
* So, we will use the MVR page’s CRUD permissions and apply them to the button in order to match permissions



*Requirements:*

* On the managed view button of any grid that has MVR functionality, do the following:
** Specifically do on the Users grid
* Apply the MVR page’s CRUD permissions to the Distribute button
** Use the Add permission
** If they have Add rights to the MVR page, they can click the Distribute button
** If they don’t, then hide the button
* Also ensure anyone with CS role or Tech Support role can do it

## Acceptance Criteria
* Only users with MVR page Add permissions or in CS Role can use the Distribute button

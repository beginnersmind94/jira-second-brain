## Acceptance Criteria
Navigation:

MP - Configuration - Import. Export

User selects Import Radio Button Option and click Next

Default Data Source is selected as Local

user uploads the template and click and proceed with import

*Field Mapping* for Stock Items: should be done for *Manufacturer and Brand, and Product Code Fields* (Regression - to be checked for Ingredients)

The following Exception validations are needed

*Missing Mandatory Information* - Name (Menu Name or Inventory Name), Calorie, Fat, Carbohydrate, Saturated Fat, and Protein - DO NOT create these items , just log them and show them in the summary

Exception Messages:

*Nutrient Information* - If non mandatory elements are left blank - the missing check box should be enabled

*Contributions* - if the value available is outside of the range , we shall fit that value to the lowest / highest acceptable value

*Data Type* - If the values are not corresponding data type - mark it as exception (Ex: Nutrient, Meal Credits, Serving Size Quantity or Serving Size Unit Weight, and Pack size fields

|Net_Weight_Lbs|Servings_Per_Purchase_Unit|Inventory_Unit_Type|Servings_Per_Inventory_Unit|IUs_Per_PU|Servings_Per_LB|

Create items only if all the mandatory criteria's are met (*Missing Mandatory Information*), otherwise park the count under “Items Not created”, create the warning messages otherwise

# NXT-37093 — PR- Calendar- Allow Refresh on the calendar to update latest information for month

## Description
As a Production Manager, I would like to be able to refresh my display of the calendar with the latest information that is assigned to my site when the menu planner changes things on the Menu Planner side.

## Acceptance Criteria
* Provide a refresh button that is embedded in the calendar component
** Rearrange the (Today Selector to be on the left side and use the space for the Refresh button)
* The refresh should refresh the information of the calendar based on the site and month the user is on only
* Refresh should only be available after loading of the calendar, disable when the calendar is still loading

!image-20230227-231141.png|width=1299,height=756!

* If user does not refresh and they select a particular plan that is no longer available, do not lead them to a non existent plan/record page, refresh the Calendar page
** Ex: The menu planner reassigns the menu or deletes the menu and the plan is still showing up, this would be the scenario 
** Display a message at the top when the page is refreshed “Plan/Record selected is no longer available”

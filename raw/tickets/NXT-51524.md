# NXT-51524 — GDSN Connect - Item Upload - UI Changes

## Description
Currently customers do not get an opportunity to review the translated data

## Acceptance Criteria
Navigation: IM - items - Upload Icon

User should have two radio buttons  as shown in the wireframe

User can click only one radio button at a time

By selecting GDSN Template, the user should be able to upload the current base GDSN Template As-is (sample attached)

The existing API shall be called to push the data into GC staging table

Once the data is pushed the parsing shall begin and complete the process - As-Is

Once processed, the data shall NOT be moved to SC table automatically (status: Processing)

Regression:

File Type validations, Field Mismatch Validations etc..

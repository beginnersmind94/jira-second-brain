# NXT-40837 — IM - Menu Items - Default Serving Measure

## Description
As a Menu Planner I want the ability to view and change the default Menu Item serving size in the Units card so I can easily swap the serving size that will be default when menu items are added to menus.

* Show Default Measure identifier on Menu Item Units card
** similar to Preferred Measure (same icon)
** show appropriate tooltips
* The selected serving for MI creation automatically becomes the first Default Measure
* Allow default measure to be changed to other menu item servings
** using row level edit

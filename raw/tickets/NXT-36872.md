# NXT-36872 — IM - Recipe Rework - Copy Recipe 

## Description
As an IM administrator I want the ability to copy Recipes so I can create additional variants of mostly similar recipes without recreating each one.

*Requirements*

* Update the Recipe copy function to accommodate the updated design approach
** Change ‘General Info’ to ‘Summary Info’
*** header level info (Description, Serving Size, # of Servings, Cook/Prep Time)
** Give an explicit checkbox for Steps & Ingredients
*** default to checked
** Check all Nutrition related checkboxes by default
*** auto-check on select of Steps & Ingredients checkbox
** Add explicit Yield Factor checkbox
*** copy all or nothing from the Yield Factor card
* Hide Data Source dropdown
* Hide/ignore Menu Item for now

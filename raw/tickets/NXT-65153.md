# NXT-65153 — TH - Insights Widgets Toggles needs to be saved along with ordering

## Description
h2. 

* As a user, when I am using the insights widgets, I want to have the option to show or hide them so that I can customize my screen according to my needs.
* As a user, when I reorder the insights widgets, I want my preferences to be saved so that I do not have to reorder them every time I log in.

h2. Context

The issue involves providing users with the ability to show or hide insights widgets on their screens. Additionally, users should be able to order these widgets, and their preferences should be saved. This functionality should be integrated with an API.

## Acceptance Criteria
* Users can show or hide insights widgets on the screen.
* Users can reorder the insights widgets.
* The ordering of widgets is saved for future sessions.
* The feature is integrated with the API.

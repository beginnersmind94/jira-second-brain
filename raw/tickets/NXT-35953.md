# NXT-35953 — Acct Mgmt - Account Info - Add Back button to return to Accounts page, open links in new tabs

## Description
The Account Info page needs a 'Back' button (in the top-right corner of the page) for two reasons:

# To keep the design in line with other pages. 
# This might better align content on the page as well. The card on the right side of the page is not in line with the main section.

Also, since it takes several clicks to get to an Account Info page (and it cannot be easily refreshed/saved/bookmarked since the URL doesn’t capture the specific account ID, among other reasons), we should open the following links in new tabs instead of using the same tab:

# An App/File# in the Eligibility section
# The History button in the Transactions section

I think those are the only 2 where we redirect the user, but it would be great to do this in other sections where there is a redirect as well if possible.

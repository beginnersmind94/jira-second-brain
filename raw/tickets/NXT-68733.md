# NXT-68733 — IM - Bulk Item - Updates Publish to Local

## Description
As a District Manager, once I have made all the updates in the Bulk tool, I would like for all those updates to now be reflected onto the items in my local database.

## Acceptance Criteria
* Clicking on “Update Items” on the Pending Updates section should push updates to the local DB
* Once items have been updated/Published, release/unlock items
* Store items updated in the “Published Items” slide out 
** Use the same Slide out and button as “Published Items” in Onboarding version
* If no more items are available to be updated, complete the session on the users behalf
** This will start the process over for them to pick new items to start a new update session
** Store Session in Onboarding Slideout with differentiation that it is an update session EX: Update Session #6

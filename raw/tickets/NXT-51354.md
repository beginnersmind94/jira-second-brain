# NXT-51354 — PR- Plan/Record- Consider MMA as grain as display on pages (API ONLY)

## Description
As a menu planner, when I decide to use my MMA as a grain for breakfast, that should reflect on the production plan/record for my production managers to see that information and to know that they are serving that item as a grain for the day.

## Acceptance Criteria
* Menu Planner using the MMA as grain toggle for breakfast should reflect on Production plan and record
* Replace MMA with Grains if menu planner has used the toggle and display on Plan and Record
** EX: 2 M/MA will now just show as 2 Grains on production
* If change is made again on the menu, it should update and reflect on both the plan and record
* Same rules still apply when it comes to record complete menus

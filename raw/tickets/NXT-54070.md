# NXT-54070 — Imports – View Config - Show Mapped Values

## Description
As a SchoolCafé admin, I would like to see the actual mapped values for code fields when looking at the config screen so that I don’t have to go to the configuration process to see them.

*Requirements*:

* On the System > Imports & Exports > API tab > Rows, do the following:
* When click View Configuration, make the updates:
* For fields that have code mapping, show the mapped values
** Right now, it just shows the mapped field name
* 
* Do this for all file and API imports that have code mapping

## Acceptance Criteria
* User can see the mapped values for code mapping fields when click on View Configuration

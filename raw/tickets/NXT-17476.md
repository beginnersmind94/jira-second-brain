# NXT-17476 — AI Data Insights – PNA Data requirements

## Description
As a system, AI insights need PNA data  and external(influence) data to achieve forecasting of potential eligibility. 

*Requirement:* Data requirement includes both internal and external influence/alternate data points.

+*Paid Not Applied (PNA) – data groups*+

h4. Internal data set for PNA forecasting rate:

# Site id info
# Student info
# School type
# Person data – Eligibility identification
# Academic Year
# Application process information
# Eligibility method

h4. Student subcategory for PNA

# Paid Student breakfast
# Paid Student lunches
# Paid Student supper

h4. Applied category for PNA

# Applied status on file (with application/without application)
# Applied and rejected status on file
# Rejected to apply status on file

h4. External data set:

# Paid student Affordability (applied for SNAP, other)
# Paid student Demographics data (age, gender, occupation, education level, ethnicity)
# Paid student Geographic's data (location (state, city, zip code), language, density (urban, rural) etc.
# Paid student Household size 
# Paid student Household/Family income

*Initial data group set - variable and metadata set attached.

# NXT-47956 — POS - Picture Mode - Size Options

## Description
As Carmen the Cashier I want to adjust the number of picture cards I see so they’re easier to interact with on my small screen.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=1471%3A906&type=design&node-id=1471-1180&viewport=858%2C1476%2C0.5&t=553BkxmzmsfpvQ2n-1&scaling=scale-down-width|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=1471%3A906&type=design&node-id=1471-1180&viewport=858%2C1476%2C0.5&t=553BkxmzmsfpvQ2n-1&scaling=scale-down-width|smart-link] 

*requirements*

* Briefly explore if we can adaptively adjust for the user’s screen size regarding the number of accounts on screen.
* User has an option to choose the number of accounts that are shown:
** Text: Number of Accounts to Show
** Options: 4 x 6 (default), 3 x 5, 2 x 4
* User’s option is saved for them and used whenever they load into picture mode.
* The selected option will change how many cards are shown - 3 rows, 2 rows. 4, 5, 6 columns.
** Cards should scale accordingly to make full use of the screen.

## Acceptance Criteria
* Option to change number of cards added
** 4 x 6, 3 x 5, 2 x 4
* Option is saved as as a default
* Screen should update immediately after the option is saved
* Cards scale accordingly to fit the screen

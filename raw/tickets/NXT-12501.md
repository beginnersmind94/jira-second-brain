# NXT-12501 — Eligibility - Common - Date and Date/Time Format Consistency

## Description
Please list all pages and fields that are currently using date or date/time.

Dates should appear as “MM/DD/YYYY hh:mmaa” (ex - 03/10/2021 3:14pm). Be sure that the district timezone setting.

## Acceptance Criteria
* Date only should appear as “MM/DD/YYYY” (ex - 03/10/2021)
* Date only should appear as “MM/DD/YYYY hh:mmaa” (ex - 03/10/2021 3:14pm)
* District timezone setting should be applied in all cases 
Setting Codes to be used: DTTIMEZONE, DAYLGTSATE

# NXT-17414 — Mock up Screens @ Scheduler for API Call

## Description
This story is to cover the mock up screens needed for scheduler.

This screen should reflect the settings needed for setting up scheduler on the page where API configuration can be edited.

Provision to enable and disable the scheduler must be included.

Provision to set the scheduler as given below must be made.
1. All 7 days of the week.
2. Start Time and End Time.
3. Schedule @ Hourly, Minute and seconds basis.

Provision must be made on the API configuration landing page to reflect the scheduler status like enabled or disabled.

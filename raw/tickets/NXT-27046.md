# NXT-27046 — IM - Direct Search Box (Item List View)

## Description
As a user I need a way to quickly search for an item if I already know its name, code or GTIN 

[https://www.figma.com/proto/7c2q9K3FgHMT1g571JJOEH/Item-Management-5.0?node-id=1635%3A32982&scaling=contain&page-id=1045%3A27776&starting-point-node-id=1635%3A32982|https://www.figma.com/proto/7c2q9K3FgHMT1g571JJOEH/Item-Management-5.0?node-id=1635%3A32982&scaling=contain&page-id=1045%3A27776&starting-point-node-id=1635%3A32982|smart-link]

## Acceptance Criteria
*Entry Point*

* Item Management → Items Landing Page
** Smart Search box empty by default 
** Field Search opened by default



* User shall be able to enter the Item name, code or GTIN on smart drop down search box located as per image attached
* If the item exist and there is a perfect match of any of the entered fields (Item name, code or GTIN ) the system should go directly to the  Item View page for the item
* If multiple items are found user shall be able to select item from the drop down list and upon selection go directly to the item’s view page
* If Item does not exist display message “No Matching Records Found”

Dependencies:

Platform: Bar search box control

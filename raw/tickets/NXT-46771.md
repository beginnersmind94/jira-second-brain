# NXT-46771 — FreshDesk Support Chat - Roles & Permissions

## Description
We need to add FreshDesk Support Chat to the roles and permissions and apply it to the chat button

Notes:

Requirements:

* On System > Roles & Permissions screen, do the following:
* Add a permission in the general section where the PII permission is (see screenshot)
** Text: Can access FreshDesk Support Chat
** Action: Toggle button like the PII row
** Default to Off for all roles except Central Office
** Default to On for Central Office
* On the FreshDesk Help screen, do the following:
** Add in the logic to hide / show the Support Chat button based upon this new permission



!image-20240123-211152.png|width=1614,height=638!

## Acceptance Criteria
* Permission is added to the R&P general section
* Central Office is the only role with it on by default
* Users with this role can use the support chat functionality
* Users without the role cannot see the support chat button

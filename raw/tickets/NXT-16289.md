# NXT-16289 — INV - Orders - Orders List - Opening Incomplete Orders

## Description
As an operations manager I want the View Order icon link for Incomplete orders to open the Confirm Order screen so I can finish any orders that are still in progress.

*Requirements*

* On the Manage Orders screen, clicking the view icon for an order with an *Incomplete* status will navigate to the Create Orders confirmation screen
* Multiple Incomplete orders could link to the same page

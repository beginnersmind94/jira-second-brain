# NXT-60309 — Digitize AI - Document upload messaging

## Description
As a user I want to see feedback on the documents/images that I’ve uploaded so I have visual confirmation (success or failure) if the document is able to be used.

Case 1: Support document, but document is blank or information is not able to be extracted (unreadable).

* We can identify the context information and display it as the summary
* We will show a message that “The information on this form is either missing or unable to be extracted”

!image-20250324-152122.png|width=814,height=384,alt="image-20250324-152122.png"!

Case 2: Document type and/or form type is not supported.

* Context is unable to be extracted with this case, so the summary is not required
* Display message that “This form is not recognized or supported.”
* We can display a distinct message if the document type is not supported as “This document type [doc type] is not supported. Please choose/capture an image or upload a PDF.”

!image-20250324-152334.png|width=814,height=181,alt="image-20250324-152334.png"!

Case 3: Supported document, successful parsing

* We can identify the context information and display it as the summary
* Display successful message that “This form was successfully imported and information was able to be extracted.”
* Actions and details preview are covered in other stories

!image-20250324-162456.png|width=806,height=466,alt="image-20250324-162456.png"!

## Acceptance Criteria
* Document control must provide feedback on successful document upload
* Document control must provide feedback on failed document upload due to unsupported doc type or form type
* Document control must provide feedback on failed document upload due to inability to parse or when it is an empty document

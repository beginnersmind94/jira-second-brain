# NXT-46678 — IM - Items - Incomplete status for Menu Items

## Description
As a Menu Planner I want the ability to leverage the Incomplete status for menu items so I can pre-enable them without fully configuring them, allowing me to more easily find them later.

## Acceptance Criteria
* Provide a means to progress the Menu Item status from Disabled to Incomplete
** for Items only (no valid use case for Recipes at this time)
* Show the appropriate status icon change when status advances
* Set the Incomplete icon on the tab bar
* Advance status from Incomplete to Active when mandatory fields are saved
* Add values to dropdowns on all 3 list page Menu Item Status 
** show the Incomplete status chip in the grid (like Pending Deactivation and Inactive)
** show italicized values for Menu Item Name and Serving Size (all other MI columns will be blank)
* Insert the Item Description as a temporary placeholder text for the Menu Item Name
* Hold placeholder values until Menu Info is edited
** until Create Menu Item flow is enabled ([story link|https://primeroedgenext.atlassian.net/browse/NXT-53795])

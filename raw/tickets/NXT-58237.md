# NXT-58237 — FOH Dash - Claim Readiness - Page and Permissions

## Description
As Dana the Director I want the system to tell me if I am “good” to make my claim, and if I am not, where I should put my attention. This will help me focus on what’s important and save me operational hours.

*requirements*

* User has a ‘Claim Readiness’ page 
** Access points: 
*** Side-nav on the FOH Dashboard
*** Button on the top of the Claims page
** Role permission → Page access
*** Text if user does not have access: You do not have permissions to access this page. Check Roles & Permissions if this is in error.
* User can select a POS Period / Month and a Program.
** POS Period: show the periods from the prior academic year, and the current academic year up until the current month.
*** Example: If I ran this for today, I would see February 2024 as the default/top selection, and can select back through July 2023.
** Program dropdown: check Program Configuration to see which programs the district has enabled.
** Dropdown order: SNP, SFSP, CACFP
** Defaults: 
*** Period: Current Month
*** Program: SNP
* Explanation for the top of the page:
** This page can be used to help determine if all of your sites are ready the reimbursement claim. Areas of readiness can be configured depending on your district’s needs at the top right. The page highlights potential areas of concern under Site Readiness that need to be investigated, and generates an overall Claim Confidence value that depends on these areas being address. You will need to perform a new search (Apply) to refresh data on the page.

## Acceptance Criteria
* Claim Readiness option added to the Workspace DOH Dashboard
* User can pick a POS Period and a Program to focus on.
** POS Period → Default to the same month they had selected on the main FOH Dashboard.
*** Example: User had 1/28/25 selected, load the POS Period that that falls into → January 2025.
** If no POS Periods have been generated, tell the user and provide a link to the Periods page.
*** Text: POS Periods have not been configured. They can be configured on the +*Periods*+ page.
*** This can be a warning note on the page.
** Program → Dropdown should only show service programs that have been configured for the district.
*** SNP should be default (this includes SBP, NSLP, ASP), but check to see if district has SFSP or CACFP configured.
* Role Permission → Page/View access
** Add to the following roles as default: Director, Central Office, SchoolCafe Administrator

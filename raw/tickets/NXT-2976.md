# NXT-2976 — Inspections Template - Dynamic Table Question Type

## Description
*User Story*
As an Inspections user, I can add a series of questions organized in a dynamic table to a new or existing template.

*Requirements*

* A series of questions can be organized into a table in a template
** The number of columns is defined by the number of questions the user adds to the table
*** There is a soft limit, to be determined
** Rows are defined by the Inspector during the Inspection
*** User adds rows for each response
*** There is a soft limit, to be determined
* Each column requires a Column Header, Column Type, and any relevant information dependent on Column Type
** Column Header is a text box
** Column Type is a dropdown
** Each column can individually be mandatory or optional
* There are several types of questions that can be included as part of Column Type
** Text - A short text box
*** No additional options
** List
*** User sets list options and score
*** Score only available for Custom Scoring
** Rating
*** User sets range and labels (if desired)
*** Can be scored in Custom Scoring
** Date/Time
*** User sets which date and time options are available
** Signature
*** No additional options
** Location
*** No additional options

## Acceptance Criteria
* Dynamic table question type can be added when creating or editing a template

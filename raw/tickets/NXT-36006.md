# NXT-36006 — MP- Menu Planner- Display Serving Size on the Menu Item 

## Description
As a Menu Planner, I would like to be able to see my serving size on the menu item when I drag and drop it onto the menu planner. This will help me able to make sure the right serving size is selected for a particular menu item for a particular serving group.

## Acceptance Criteria
* Display serving size on the menu item when dropping the menu item on the menu (Always give option for Not Served)
* Display default serving size on the menu item if there are multiple serving size
* Display Serving size exception serving size if one is established in IM for that menu item to a particular serving group
* Display a selector to select a serving size if there are multiples to choose from 
* Once serving size is selected, it needs to dynamically change the analysis if needed (Nutrition for now)
* Spacing/Padding needs to be handled for the category

# NXT-5256 — Direct Certification - Unmatched - Reviewed

## Description
*User Story*

As a district admin I want a way to review what actions have been taken by staff regarding DC extensions, and to correct them if required.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=324052%3A161&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=324052%3A161&scaling=scale-down-width|smart-link] 

*Requirements*

* Add a ‘Reviewed’ tab to the page where the user can see all potential matches that have had an action taken.
** Show the same information as the Search & Review, but show the action taken as a Status column.
* User can filter for dismissed / matched.
* Grid should have contains filters, date picker, etc.

## Acceptance Criteria
User can see all potential matches that have been acted on.

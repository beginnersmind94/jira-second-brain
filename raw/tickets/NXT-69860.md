# NXT-69860 — Insights- Waste KPI - Food Waste % calculation & Card Enhancement

## Description
As a production manager, I want to make sure when I am looking at the KPIs that the appropriate counts are taken into consideration when showing waste percentages.

## Acceptance Criteria
* Below screenshot shows how current waste percentage is calculated and how it should be corrected.  
!image-20260202-214041.png|width=647,alt="image-20260202-214041.png"!
** Revised Formula for % of waste = Total Waste/(Produced+Carryover to this meal service)*100 (OR) Revised Formula  for % of waste = Total Waste/(Carryover to next meal service +RTS+Waste+Served)*100 
* Carried over servings to next meal service + RTS+ Waste +Served = Produced. 
* For prepsite items , make sure Received count of a menu item is taken as produced count menu item analysis page waste % columns
* On waste card slide out, Update the card to include “percentage of” to indiciate where the percentage is in reference to
** Only served will have 2 references to reference percentage of planned and produced
* Add new row-s for carryover- and Return to Stock along with percentage of produced

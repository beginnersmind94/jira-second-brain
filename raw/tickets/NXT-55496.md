# NXT-55496 — Acc/POS - Charge Limits - Options to ignore limits for meal types

## Description
As Dana the Director our district only has charge limits for particular meals like lunch - breakfast should not have one. I need a way to do this to stay in compliance with my district policies.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=10686%3A3958&node-id=10686-17610&node-type=frame&viewport=442%2C353%2C0.25&t=87k3qc0KgPsLZmw3-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=10686%3A17610|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=10686%3A3958&node-id=10686-17610&node-type=frame&viewport=442%2C353%2C0.25&t=87k3qc0KgPsLZmw3-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=10686%3A17610|smart-link] 

*requirements*

* User can select which meal types should ignore charge limits for meals
* User can see which meal types are ignoring charge limits on the Charge Limits tab
* POS updated to ignore charge limits for configured meal types
** Meal sales only, not ALC



Text strings:

* Added to the main page - couple examples as well.
** {noformat}The following meal service(s) are ignoring charge limits for students: A, B, C.
The following meal service(s) are ignoring charge limits for students: None.
The following meal service(s) are ignoring charge limits for students: Breakfast, Lunch.{noformat}
* For the drawer:
** {noformat}You can choose which meal service(s) will <b>ignore</b> charge limits for student meals by enabling the respective toggles. This will only apply to students, and for meals. A la carte is not impacted.{noformat}

## Acceptance Criteria
* User can configure meal types to ignore charge limits for student meals
* POS updated to use the configured data and ignore charge limits for student meals

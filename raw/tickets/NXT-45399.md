# NXT-45399 — Imports - GDSN Connect - Manual Import Frontend

## Description
As a SchoolCafe admin, I would like to be able to import the GDSN Connect Items Export from InTeam so that I can include them in our Global Catalogue.

*Notes:*

* This story is a front end part of the GDSN Connect Items export
* The back end part is in another story
* This story is for the Items only and not the Recipes
* This story covers Manual import only, not sFTP or API
* A lot of these requirements reference the InTeam specifications which are attached to this story

*Requirements:*

* For Add New and for Edit Imports
* *Pre-Upload tab:*
* Add GDSN Connect Items Import as an Import Type
* For file template download, use the attached file called GDSNConnectItemsTemplate
* Do not show the file format fields as it is always CSV
* 
* *File Upload Tab:*
* Do standard upload UI like for Student Import
*  
* *File Mapping tab:*
* Use the ITEM Data v4.0.0 table on pages 19-35 of the attached requisites document for references
* File Fields table:
* Do like Student Import 
** Include Add New
** Include Delete
* 
* Mapped table:
* Use the Field Name column from doc for the column name
* There are no code mapping for this import
* 
* SchoolCafe Fields table:
* Use the Field Name column from doc for the column name
* Use Step 2 (Transfer) from doc to determine if the field is required
** use M only for Required
* 
* *Review tab:*
* Do like Student Import

## Acceptance Criteria
* Tables exist to support file mapping
* Tables exist to store data on import
* APIs exist so front end can consume
* All the fields exist with correct datatypes

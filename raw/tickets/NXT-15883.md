# NXT-15883 — INV - Create Orders - Create Generated Orders

## Description
*User Story*
As an Inventory site manager I want the system to save all my eligible orders as an official order in Open status so I can finalize the create orders process.

Story scope: back end story to save all eligible orders as Open; heavy on API side; no UI work

*Requirements*

* When user clicks the Create button on the Order Review screen ([NXT-5714|https://primeroedgenext.atlassian.net/browse/NXT-5714]), all eligible orders will be officially created and saved as ‘Open’ status.
* Only the orders on the page without any order validation issues will be saved as Open
** Orders with some order validation issue will remain in Incomplete status 
* After creating orders, page will update ([NXT-5647|https://primeroedgenext.atlassian.net/browse/NXT-5647])



!5714.png|width=100%!

## Acceptance Criteria
* Upon click of Create button, scan the page for all orders without any validation issue
* Save all eligible orders in Open status, officially creating the order
* Do not create orders which have some order validation error

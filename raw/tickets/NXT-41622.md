# NXT-41622 — Imports – Remove SchoolCafé Fields Table - Spike

## Description
As a SchoolCafé Admin I want to have less screen clutter on the Fields Mapping tab so that the screen is more intuitive to me.

*Notes:*

* See the story for reordering the fields in the table as it has a dependency on this

*Requirements*:

* When importing a Student Import on the Field Mapping Screen, do the following:
* Investigate how we can remove the SchoolCafé Fields table (the one on the right)
* It seems that it’s primary purpose is to identify which columns are required
* We would like to integrate that (required fields) into the middle table, Mapped, and remove the third table so the screen is cleaner
* 
* Therefore, investigate and find a way to show which fields are required in the middle table so that you can remove the third table
* Do this in a way that will not blow up the importing, preferably in a way that is front end only and requires no back end updates
*

## Acceptance Criteria
* Make recommendation to accomplish the requirements stated above

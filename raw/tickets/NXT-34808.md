# NXT-34808 — UX - Tag Management UI - Add Columns to Tag Detail UI

## Description
As a user, I would like additional tag details.

## Acceptance Criteria
* As a user, I would like to see the details of which entities have tags assigned to them
** This table shall include (in this order from left to right):
*** “ID”  (student id, item id, etc…)
*** “Description/Name”. (Name of the student, description of the item, …)
*** “Entity Type” (Patron, Recipe, …)
*** The module the tag is associated with
*** The tags that are assigned to the entity

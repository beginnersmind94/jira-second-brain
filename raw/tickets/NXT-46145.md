# NXT-46145 — Onboarding - User - Add onboarding experience for all new users (Part 1)

## Description
As a Cybersoft or district admin, I would like for users to be able to have an onboarding experience of their own which helps them easily transition to using SchoolCafé in a personalized way, so that is easier for them to use right away and results in fewer support tickets as a result. 

*Summary*

Onboarding of individual users to help them get setup within their profile (details and preferences), workspace, understanding overall usability (take a look around), and then reviewing their role/responsibility assigned to them (central office, nutritionist, site manager, etc.). This requires two changes.



*Part 1:*

*Users in districts where onboarding is still incomplete:* Have a single page where the user views their assigned profile details (Primary district, roles, access, Password Reset, Expand Side Navigation setting). Also, explain that the district is still completing their set up process, give them a reminder that they can change these settings later on, and tell them they can click '?' to get Help/view whatever new user guide we have created. Finally, provide a logout option with all other options disabled.

* When a new user is created for a district where onboarding is not yet complete, do the following:
** After they have clicked the link the ‘Welcome’ email and log in, redirect them to their Profile.
*** Previously I think we disabled this Profile button in onboarding. If it needs to be reenabled in this case, please mention that in the comments or to Daimien.
*** Experience should only allow the user to update their profile, click '?', or log out.
** Display a ShepherdJs tour in the center of the page that display immediately. Details:
*** Title: *Welcome to SchoolCafé! Please read this.*
*** Body: Hello! Your district is still completing their SchoolCafé implementation. While you wait, you can use this screen to update your profile details and select the '?' button to view helpful guides to prepare you for using SchoolCafé. 
*** Footer: *Finish Tour* button
**** When clicked, this should still create a completed task in Workspace once district onboarding is completed.
*** Text should be editable in Content Management.

# NXT-14737 — Platform - AI/ML - EDIT Device from Device Catalog

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to setup authentication token for new EdgeBox with UUID identifier and assign to a school district. 

*Mockup:*

[https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|smart-link] 

*Requirements*:

* Admin logs into PrimeroEdge portal, to EDIT recognition device with *Serial Number*  of the EdgeBox that belongs to specific *SITE*
** +First 4 digits of the Serial Number will identify ONLOGIX Edge devices+
* Serial Number, Model are non-editable
* Status to be retrieved in real-time and can be updated manually.
* RESET token functionality should NOT be enabled for devices in ONLINE status

## Acceptance Criteria
* User shall be able to add new Edge device from the device catalog screen
* Users shall be able to see list of Edge Devices with School, Serial Number, Status and Last Connected timestamp
* User shall be able to see attached cameras with status & ID that are connected with any edge device
* User shall be able to update/replace and remove device actions available in the device catalogue

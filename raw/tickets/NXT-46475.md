# NXT-46475 — FH - Menus - Make a Tray - Building the tray

## Description
As Nancy the Nutritionist I want households to be able to make a tray and see the tray fill up as components are met.

*Note:* SOME functionality already exists, but not all of it.

*requirements*

* Update the ‘Choose X’ pill to the View Nutrition Info color format
** !image-20240116-222537.png|width=711,height=110!
* User can select menu items off the menu and see the impact on the tray itself:
** As meal contribution rules are met, fill in the portion of the tray.
** When all rules are met, inform the user:
*** Text: This tray has met all nutritional requirements to be a complete <MealType> meal!

!image-20240116-221120.png|width=863,height=701!

* User can see Additional Allergens under the Allergens section.
** !image-20240116-213827.png|width=16.666666666666664%!
* User can see preferred menu item images in the ‘My Tray’ area
* View Full Nutritional Information button + popup
** User can see a sum of nutritional info
** User can see stacked menu item images
** !image-20240116-223127.png|width=1291,height=757!



Make-a-Tray Rules: 

* Breakfast/Lunch/Supper
** Min of 3 colors
** 1 Protein (Purple)
** 1 Grain (Orange)
** 1 Milk (Blue)
** Must select at least 1 Fruit (red) or Vegetable (Green)
* Snack
** Min of 2 colors
** 1 Protein or Grain or Milk (Purple, Orange, Blue)
** 1 Fruit or Vegetable (Red, Green)





!image-20240116-222406.png|width=333,height=194!

!image-20240116-222420.png|width=374,height=241!

## Acceptance Criteria
* Pill format for ‘Choose X’ updated
* Make-a-Tray Rules need to be checked to see if the selected menu items meet the criteria.
* User can ‘build a tray’ and see the impact:
** User is informed if the Make-a-Tray Rules are used to determine if the tray is a complete meal
** User can see the tray sections fill in as meal contributions are added
** User can view Full Nutritional Information
** User can see menu item images in the My Tray section and inside the Full Nutritional Information section
** User can see Allergens and Additional Allergens

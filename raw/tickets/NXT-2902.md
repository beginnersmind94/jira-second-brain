# NXT-2902 — Authentication - Logout with Inactivity

## Description
*User Story*

As a PrimeroEdge user, I am automatically logged out if I am inactive for the length of time set in the password policy.

*Requirements*

* Inactivity is defined by the Password Policy (already complete)
** Inactivity _warning_ is also defined by the Password Policy
* A *warning* is presented to the user before they are logged out (it is displayed *x* minutes before the logout occurs, as defined by the Password Policy - ‘Inactivity Timeout’ setting value)
** Warning displays over the screen
** Wording: “Due to inactivity, you will soon be logged out for security purposes. If you wish to remain logged in, please click Continue.”
** There is a *Continue* button
** If setting is set to None, then no warning appears and the user is automatically logged out
* Warning counts down *y* time left until the logout ('Inactivity Warning' setting value)
** Warning disappears and timer is reset if user clicks the Continue button, which closes the pop-up and lets them continue browsing the application
* Once the timer reaches the end, the user is logged out
** Timer length is dependent on Password Policy setting
** User is logged out and sent to the login screen.
*** If necessary, we can display another pop-up with an ‘OK’ button, saying: “Due to inactivity, you have been logged out.”
** User is required to log in again once logged out
** When logging in, user is taken back to the screen they were last on
* The user should be logged out when their browser (not tab) is closed. The timer can continue when it is just the tab that has been closed. (Can be deferred to release 2 if needed)

## Acceptance Criteria
* Warning banner appears properly and counts down
* User is properly logged out after inactivity
* Banner and logout timing properly follow settings
* Timer properly resets with activity or selecting 'Continue' from the warning pop-up
* User is logged out when browser is closed, but timer continues when only tab is closed (can be handled in release 2 if necessary)

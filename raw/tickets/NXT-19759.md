# NXT-19759 — Import-Photo

## Description
This story is to cover the tasks related to Photo’s Import.

As an end user, School districts want to import student photos to identity during the meal program. However, student profiles do not have any photos by default. They have to be imported and mapped to individual students profile.

Photos are named per the naming convention “StudentID“.jpeg

Here in this, we will be configuring / modifying the existing import platform to import the student photos and map them to the student profiles both individually and and as bulk using the names of the pics to the student ID of the profiles..

Current limitations are

# Each Pic/Photo has to be no more than 5MB (size configurable from internal settings) in size
# Photo format is .JPEG or .PNG
# Max Pics/Photos that can uploaded in bulk depends on technical limitations (architecture based) .

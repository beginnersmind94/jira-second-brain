# NXT-2810 — Inspection Template - Location Question Type

## Description
*User Story*
As an Inspections user, I have the ability to add a question that takes the Inspector's location.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=date_time_question_type&id=1rjsn1|https://nqyc9v.axshare.com/start.html#p=date_time_question_type&id=1rjsn1]

*Requirements*

* Location can be selected as a question type when creating or editing a Template
** Input is done using the device's GPS system
* User can determine whether to allow only one or multiple location entries
* User can require a timestamp
* There is a toggle for whether the location can be edited

## Acceptance Criteria
* Location can be added to a template as a question type

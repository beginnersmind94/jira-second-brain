# NXT-4083 — IM - View - Basic Search - Item Name and ID

## Description
As an Item Management User, I want to be able to search for items while already viewing my items.

*Mockups*

[https://nqyc9v.axshare.com/itemmanagement_-_super_user.html|https://nqyc9v.axshare.com/itemmanagement_-_super_user.html]

!image-20200526-125604.png|width=389,height=235!

 

_Requirements:_

* The Search Box must be able to search by Item Name, Short Name, Item Code, GTIN, and Product Code
* Exact Match is prioritized over partial match
** Prioritize datasource in order (local, shared, and global in this order)
** Exact Match shows before partial even if datasource is not local for example
* Text inside the box (Search by item name or ID)
* If no match is found, display text 'No Results Found'

## Acceptance Criteria
* Search box must be able to hit enter or click the magnifying glass to search
* Only match on Item Name, Short Name, Item Code, GTIN, or Product Code. Prioritize in this order for the search

Moving to future stories, removing from this one
* Suggestions will only show suggestions of 10 or less 
* Exact match is prioritized. If no exact match, then just display the best matching fields

# NXT-65371 — PR - Pre & Post Analysis - Consider Product Based Added Sugars (CACFP)

## Description
As per the new USDA requirements, particular Foods now need to be flagged and analyzed for their added sugar content to make sure that they are within the Compliance Standards

## Acceptance Criteria
* Check menu/menus to find any menu items that have “Product Based Added Sugar” Flag
* Based on what type of Product, analyze to ensure compliance is met
** Breakfast cereals may have no more than 6 grams of added sugars per dry ounce
** Yogurt may have no more than 12 grams of added sugars per 6 ounces (2 grams of added sugars per ounce).
* Flag will be static and only affect records that are plan incomplete or plan complete
* Pass/Fail Added Sugar and provide a alert for user to check what menu item is causing the fail
** Only Display Added Sugar as the nutrient value as required IF Product based flag is available for any item on the menu 
*** Remove % of calories column for all nutrient values as there is no requirement to calculate % of calories
**** This applies for both menu and menu cycle analysis
* Add additional changes as part of added sugar requirement into analysis documentation
* Update all reports that include nutrients as well as the slide out
** Nutrient Report
** Nutrient Summary (Weekly)
* Provide a disclaimer on all appropriate pages “Product based sugar compliance for CACFP has been implemented as of Oct 1, 2025 for breakfast cereals & yogurt”

# NXT-28227 — Ability to associate a template to a schedule in notification configuration.

## Description
As a system we need a way to associate a template with a notification schedule. So that, when scheduled notification runs a specific template will be used while the notification is sent.

## Acceptance Criteria
Story should be considered done when:

* We can save a template with the notification schedule when the content type is Letter.

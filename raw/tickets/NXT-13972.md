# NXT-13972 — POS - Peripherals - Cash Drawer Research and Implementation (TESTING)

## Description
As a POS Cashier I need the POS to support my cash drawer so that I can safely store and retrieve cash or checks during meal service.

Cash Drawers are triggered to open when …

* Open Session is selected
* Close Session is selected
* Payment is made using Cash
* Payment is made using Check
* Sale is made using cash (PAY button)

If a cash drawer is detected, these actions should apply. If a cash drawer is not detected, we should not have any negative impact for the user.

## Acceptance Criteria
* Cash drawer opens when Open Session is selected
* Cash drawer opens when Close Session is selected
* Cash drawer opens when Payment is made using Cash
* Cash drawer opens when Payment is made using Check
* Cash drawer opens when Sale is made using cash (PAY button)

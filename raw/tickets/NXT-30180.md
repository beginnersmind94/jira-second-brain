# NXT-30180 — Ability to schedule a notification for more broader frequencies

## Description
As a user, I would like the ability to schedule a notification for more broader frequencies, such as monthly/every X months and yearly/every X years. 

Example 1: I would like the ability to schedule a specific notification to go out on the last Friday in April every year, and be able to set up another schedule for the same type of template type (but maybe a different template) to go out every 3 months/quarterly on the 12th day of the month.

Example 2: If for the Outreach Letter process (template type) I already have a schedule setup for Outreach Letter 1 actively running, I would like to be able to set up another concurrent schedule for Outreach Letter 2 as well, and have it running at the same time as the other schedule. Even though they are of the same template type (Outreach Letter type).

## Acceptance Criteria
The story will be consider as done when the user is able to schedule multiple notifications of the same template type.

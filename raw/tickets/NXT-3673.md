# NXT-3673 — Inspections Dashboard - Media Widget

## Description
*User Story*
As an Inspections user, I have a widget available for my dashboard that allows me to quickly view any media attached to submitted Inspections and quickly navigate to the appropriate Inspection.

*Requirements*
* Media Widget is available for users to add to their dashboard
** All available media from completed Inspections can be displayed on this widget
*** Images
*** Videos
*** Audio
*** Documents
** Completed Inspections includes Pending Approvals, Re-Inspections, and Approved
** Media is displayed directly in the widget
*** Images are displayed
*** Videos can be played
**** A still is shown with a play button that can be clicked to play
*** Audio can be played
**** An audio image is displayed with a play button that can be clicked to play
*** Documents can be viewed
**** A document image is displayed that downloads the document when clicked
* Media can be filtered by date, template, and inspector
** By default, displays past 2 weeks, all templates, all inspectors (for admins)
** Inspectors can only view their own media
* Media can be scrolled through using arrow buttons on either side
** There is no autoscroll feature (background: that used to be a thing and people didn't like it)
** Data for the Inspection is present for the media
*** Inspection Name (links to the Inspection)
*** Inspector
*** Template
*** Question
*** Site
*** File Name
** There is a button to save the file
*** The file name is formatted: <Inspection Name> - <Inspection Date> - File Name
*** Images that are saved have the above information included in the saved file

## Acceptance Criteria
* Media Widget is available and can access and filter available media
* Inspections can be quickly accessed from the widget

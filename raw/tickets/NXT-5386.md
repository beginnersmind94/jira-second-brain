# NXT-5386 — Acct Mgmt - Transactions - Adjustment

## Description
*User Story*

As a user I need to be able to edit a transaction to session.

*Mock-up*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=7297%3A1746&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=7297%3A1746&scaling=min-zoom]

*Requirements*

* The user can edit a transaction if the role permission is true.
* The user can change the meal type ?
* The user can add/edit or remove menu items served [https://primeroedgenext.atlassian.net/browse/NXT-5389|https://primeroedgenext.atlassian.net/browse/NXT-5389|smart-link] 
* The user can add/edit or remove a payment
* A comment must be entered when editing a transaction.  [https://primeroedgenext.atlassian.net/browse/NXT-5387|https://primeroedgenext.atlassian.net/browse/NXT-5387|smart-link]
* Food modifications are recorded on the day of the original transaction.
* Dollar modifications are recorded on the day the change is made. (this preserves the running balance).

*** Original PMO as Done for sale/item transaction adjustment only. Payment adjustment is still pending. ***

## Acceptance Criteria
+For the selected transaction Display Meal Type, Transaction Type, Name, date and time.
+Show any items sold with the Menu Item name, Quantity, Price and Extended Price.
+Show the total of Items sold, Tax(if applicable), and the total.
/Payments will show the Amount Tendered for Cash,  -The amount deposited and any change returned.
+Payments by check will show Check Amount and check #
+Payments by card will show Card Amount and last 4 digits as entered by the cashier.
+Comments previously entered will be shown
+Review Reasons and Comments entered by the cashier will also be shown.
/Items can be completely removed by clicking the trashcan.  -if only 1 item not removed
+The quantity can be edited. 
-If the Quantity = 0 the item is removed.
+The extended price is calculated Qty * Price
+When the items sold is changed a new amount is entered in the New Sale Amount under Items, Tax is also calculated if needed. (Tax is only calculated on Taxable Items for Adults)
+A new total is calculated.
+New Payment values can be entered.
-The deposit amount and Change returned are calculated based on the new tendered amount. 
-The Amount Deposited is calculated based on Amount Paid - Total Sale Amount.
-Change Returned is calculated based on Amount Paid - Total Sale amount - Amount Deposited.
-The amount deposited can be set to 0
-If the Payment selection is Check or Card the Change Returned is always 0 and any over payment is deposited
+When any part of a transaction is changed a comment must be entered.
+All comments are saved and shown in the comments section. Comments have a Date/Time stamp, User name, and the entered comment.

There is a reversal of the original transaction, including item sales, payments, and deposits, and a new transaction with the updated item sales, payment amounts and deposits recorded.

# NXT-59340 — Family Portal - School Menus - Allow user to filter menu item tags (Stretch Goal)

## Description
In Family Hub, let’s make use of the tagged menu items as configured in SchoolCafé by doing the following:

# In School Menus (Daily, Weekly, and Monthly views), add a new button that opens a pop-up with +multi-select+ checkboxes (similar to Allergens) labelled as *Tags*. Details of this button:
## Displays all tags that have ever been published to the selected district
## When one or more tags have been selected, refreshes the list of menu items to *only* show the items with those particular tags (opposite of allergens)
## If a district has no published menu items with Tags (as is current behavior), button is *hidden* from the UI

!image-20250218-210546.png|width=86.83769282360831%,alt="image-20250218-210546.png"!

# In the printable calendar tag, add Tags to the list of filters in the side menu.

## Acceptance Criteria
Verify that published tags are visible in a new ‘Tags’ pop-up menu in all 3 views (Daily, Weekly, and Monthly).

Verify that if no Tag info has been published for the district, the Tags button is hidden from the UI

Verify that ‘Tags’ filter is added to the printable calendar view as well

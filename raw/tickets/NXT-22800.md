# NXT-22800 — IM - Add Costing Information to Recipe

## Description
As a nutrition user I need to see the recipe material cost and cost per serving when creating , viewing or editing a recipe.

## Acceptance Criteria
* For Recipe Material and serving cost refer to NXT 13231

!image-20211130-110335.png|width=1382,height=746!

* This values are calculated
* This values cannot be edited. Are presented to the user on read only mode
* These values are recalculated when:
**  Number of Servings is changed/modified
** New Items are aded to the recipe
** Items are removed from the recipe
** Item price changes
* It should be visible when:
** Viewing a recipe
** After Creating a recipe
** After saving and editing a recipe

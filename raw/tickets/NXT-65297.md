# NXT-65297 — ACC - Labor Hours Feature - EoD Labor Task

## Description
Story to contain labor task for Labor Entry for both entry methods


!image-20250812-191815.png|width=1263,height=739,alt="image-20250812-191815.png"!



As Sally the Site Manager I want to enter my labor hours daily via the End of Day page as it is my command center for the Accountability module. This will encourage routine entry for me and be an efficiency streamliner.

*requirements*

* If Labor Hour Entry Method is set to Precise, show the Labor Hours task as an option for the End of Day page.
** User can turn the option on via the Settings cog - On by default.
** User can click the Daily Task to go to the Labor Hours page.
*** Preload the site + date.
** User can see Summary information
** User can see Detailed information
** Status = Complete when All active Category hour counts contain a value, Started if at least 1 category contains an entered value. 

----

If Precise Entry is on

If no data available just show blank text boxes.
Daily Task description - “Labor Hours”

Summary: “Number of labor hours entered”
Detailed: “4 of 4 Categories have entered labor hours.” - # of categories contain a value
Status Complete when All active Category hour counts contain a value, Started if at least 1 category contains an entered value. Not Started when no values detected.

Action Button - “Enter Labor” - when clicked present pop up with main grid filtered only to the end of day page filter. Only by one day

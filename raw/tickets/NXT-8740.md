# NXT-8740 — Menu Grid - Allow one item on the Menu Grid to be set as the Default Item.

## Description
We need to be able to select one item to be served as a default item.

!Screen Shot 2021-02-25 at 6.17.41 PM.png|width=843,height=588!



*Requirement:*

* On a Static Menu grid one menu item can be selected to be a default menu item
* The default item is set from the edit popup for the item
* If a default item is already set and the user attempts to set a new item, display a prompt that “A default item is already set for this menu. Do you want to set this item instead?” Yes, clears other item as default and sets newly selected item. No, closes prompt and does not set newly selected item.

## Acceptance Criteria
* Item can be selected to be default served menu item
* Edit Popup had a toggle for setting as Default Menu Item
* On the Edit popup for an item you can set the item to be the default.
* Only one item on a menu grid can be set as a default.
* Default item can be unselected "removed" as the default.
* When performing a session within ExpressPoint, if the selected menu has a default item, it should be populated in the sale list for the loaded student by default

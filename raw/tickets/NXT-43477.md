# NXT-43477 — MP- Menu Templates- Format Menu Templates UI to match Standard Menu page

## Description
As we continue to bring uniformity to the module, we made changes to the the standard menu UI and would like to bring that behavior to the menu templates page.

## Acceptance Criteria
* Reformat the Menu Templates Page to match the standard menu page with the component redesign
** This should apply to the create and edit versions of the page
* Allow drag and drop for items between categories and store those categories
** The categories that the menu items are added on the template should apply when adding template onto a menu 
* Remove the ability to serve with for VMI on templates
** Display new icon for VMI with tooltip for the “Child Menu Items”
* Allow Tags to be editable always whether in create or edit mode 
* Always Allow user to able to use projected servings count as a percentage if deemed necessary
* When in edit mode, site groups do not appear (This is a bug but we can fix as part of this story)

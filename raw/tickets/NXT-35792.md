# NXT-35792 — MP- Menu- Menu List page edit option should allow user to click into standard or menu planner with record complete

## Description
Current implementation is when a user clicks on the edit button for the menu, if the menu is already been used on a production record in complete status we are displaying the message on the top of the list page and not allowing the user to go into the menu. 

!image-20230105-195122.png|width=1596,height=895!



Proposed change: 

Allow user to go into the menu and display the banner on both the standard and menu planner as we currently displaying when a user goes into a menu cycle.

## Acceptance Criteria
Allow user to be able to access the menu from the edit option even if the menu has been used in a record complete

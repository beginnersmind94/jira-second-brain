*Requirements:*

* The user will have an ‘Interview’ type of configuration.
* Based on the answers to questions the ability to set additional configuration will be determined.

## Acceptance Criteria
From the Left Panel, Under Point of Service, below Deposits. there is a function Orders. When Orders is Expanded there is an Order Configuration, Order Review, Order Preparation, and Order Fulfillment.

This page is launched by selecting Order Configuration.
 
The user will answer questions to determine what and how to configure ordering.

Meal Types: Each meal type will have a cutoff time. 
   options are Day Of or Day Before. Default is Day Of
   Default Time is 4 hours before meal service time.

Sites: Select the available Meal Types, Orderable Items,  Group/Persons that can place an order, and Fulfillment types.

Zones: Set locations and fulfillment types for food 
delivery.

Periods: Configure meal delivery/pickup times by meal type, multiple periods per meal type are allowed.

Pickup: Pickup times and locations can be set. Site can also be designated as an alternate pickup site.

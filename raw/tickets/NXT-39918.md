# NXT-39918 — IM - Bulk Item - Select Local Items

## Description
As a district admin I want the ability to make attribute updates to many items at once for a variety of reasons such as updating item categories, publishing inventory items, assigning component contributions, assigning allergens, etc.

!image-20251028-151307.png|width=278,alt="image-20251028-151307.png"!

## Acceptance Criteria
* Provide a button to enable the ability to select items from the local database to add to a session
* As items are added, “Add Items” in the footer will be enabled with the count of how many items have been selected
* Once user clicks on “Add Items”, they will now be added to a repository of selected items that an update session can now be started with
** Note: Multiple update sessions will not be supported as this time
* If user decides to change filters and there are items still in the “Add Items” list, prompt a message when clicking apply on the filter “There are X Items selected, do you wish to add items to session? Yes/Discard
**  If Discard, apply filters and the items previously selected but not added will be discarded
* Items with active status can be added to an update session
** When in bulk update mode, disable Main item status and only pre select active
* Cancel will trigger a message “ Do you want to cancel all item selections?”

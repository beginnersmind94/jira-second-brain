# NXT-5371 — Deposits - List

## Description
*User Story*

As a user I need to see a list of deposits made for a single day or date range for a site or selected sites. I also need to see any adjusted deposits. 

*Mock-up*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5825%3A0&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5825%3A0&scaling=scale-down-width]

*Requirement*

* Users can select a site or sites to show in the list
* Users can select a date or date range
* Users can select the deposit slip by number
* Headers are customizable.
** Deposit Slip #
** Reference #
** Site
** Deposit Date
** Deposit Amount
** Modified Date
** Deposited By
* Drag and Drop grouping with headers
* Users can create a new deposit slip [https://primeroedgenext.atlassian.net/browse/NXT-6769|https://primeroedgenext.atlassian.net/browse/NXT-6769|smart-link] 
* Nest edited deposit slips together. PrimeroEdge does new line entries whenever a deposit slip is edited, with a negation of the previous deposit slip to bring the daily total to $0.00 first. Put all these together in an expandable grid for easier viewing.

## Acceptance Criteria
Users can select a site or sites to show in the list
Users can select a date or date range
Users can select the deposit slip by number
Users can create a new deposit slip
Users can select by reference #
columns are sortable/filterable
a single deposit slip can be selected for display
Grouping is controlled by Drag and Drop headers
Headers are customizable 
From and to date are todays date.
No Mandatory Search Fields.

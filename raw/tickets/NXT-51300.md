# NXT-51300 — Imports - Import Monitor - Dates

## Description
As a Customer Support Role user (has to code irrespective of user role) , I would like to be able to see import data for older imports so that I can diagnose issues from the past

*Requirements*:

* On Import Monitor tab, do the following:
* Add custom dates to the date range field
* Should have a start date and an end date
* Put a validator on it so that the user cannot view imports greater than 1 year
* User can use the predefined date ranges that we have or they can use custom dates in the past
* Front end code will pass the custom dates to the API and filter the data
* 
* Add the Filters accordion section like in the Figma
** Note: we do this in the History Log, so see code there

## Acceptance Criteria
* User can set custom dates for the search filter
* User can continue to use predefined dates
* Filters accordion is added to the filters section

# NXT-19924 — Production- Dashboard- Production Records Incomplete

## Description
As an Operations Manager, I would like to be able to see a quick overview of all incomplete production records from around the district, site group or by site to make sure that all Kitchen/ Production Managers are handling their tasks in a timely fashion. 

Display on dashboard widget (Second Tab) (Past)

Display in District View, Site Group View, Site View and change count of incomplete production records.

EX:
Warning! Please complete the following items! (Possibly in yellow or red)

Incomplete Production Records- 50

If there are no Production Records incomplete, display " You are all caught up, keep up the good work!" (Possibly in green)

Give link to "View More Details" which will take them to the production status page.

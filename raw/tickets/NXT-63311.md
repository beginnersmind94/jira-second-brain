## Acceptance Criteria
Navigation: GC Portal - Search - All Data Sources

Changes Required:
User selects the Data Source dropdown
User should be able to enter the GTIN, Item Description, Manufacturer, Item Category (to be listed as dropdown, text based search should list the items from dropdown ) 

The search results should be listed in the Grid as shown in the prototype (Item, Manufacturer, Brand, GTIN, Category)

The user should be able to select one or more records, then click on Excel Icon on the top right of the grid - with 2 drop down options - PrimeroEdge, SchoolCafe

Upon clicking the Export, the user should be able to download the file with corresponding records listed

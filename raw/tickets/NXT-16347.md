# NXT-16347 — INV - Orders - Order Detail - Edit Order - Add Items to Order

## Description
As an inventory user I want an Add Items button on the Order Detail page so I can add additional items to existing orders.

*Requirements*

* On the Order Detail screen in the item list, provide a button/icon to add items
* When user clicks Add Items, navigate to the Add Items to Order screen
* On the Enter Item Quantities screen, the Site, Order Date, Delivery Date and Vendor fields would all be selected according to the order being edited.
* Site, Order Date, Delivery Date, and Vendor should be disabled so the user cannot change them.
* Order Templates field would be editable and user could apply templates if desired.
** when initially loading the screen, no Order Template will be selected by default
* Provide a *Save* button so the user can return to the Order Detail screen after adding any items.
** do not need the Shopping Cart for this. The Save button will return the user directly to the Confirm screen.

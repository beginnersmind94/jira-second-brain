# NXT-22085 — Add Enable Face and Food in Settings

## Description
As a site administrator, I want to be able to toggle Face and Food availability for my district so that I can show or hide relevant functionality on SchoolCafé and POS.

*Application*: SchoolCafé Website

*Flow:*

* Browse to SchoolCafé and click the link for System in left menu and then Settings link
Click to toggle on or off Enable Face and Food Detection

 

*Background:*

* This already exists in classic
** System > System Settings > POS tab > Category:ExpressPoint > Enable Food & Face Detection field (see screenshot)
* You need to do the same thing for 2.0 that is done for Classic
Talk to Saketh if you have any questions
Just copy the code from classic
Also understand that this story is really about enabling functionality for other stories that will depend on this story

 

*Requirements:*

* In System Settings > Internal Tab > Accountability Module > ExpressPoint Cagetory
** Add a field
*** Title - Enable Food & Face Detection
Code – ENAFOFCD
Default Value - No
* Field should be a yes / no toggle or equivalent, consistent with the rest of the page
* Copy from Classic and ensure the values are the same as there and are stored in the same way as there
** A lot of code in the rest of the site will look for this field’s value

* None

## Acceptance Criteria
* Toggle exists and produces the desired value
Value is callable by other Food Detection code
Is visually consistent with the rest of the page

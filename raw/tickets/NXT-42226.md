# NXT-42226 — Homepage - Add hyperlink to view latest release notes

## Description
As a SchoolCafé user I would like to be able to easily view the latest SchoolCafé release notes, so that I can quickly reference them and discover new features at the same I’m using the website.

*Summary*

Modify the website version at the footer of the homepage so that it opens an overlay that links to the latest release notes in PDF form. If we can add options to view previous release notes, that would be a bonus.

*Details*

* On the Homepage, point the ‘Version X.X’ label to [+this URL+|https://cybersofttechnologies.sharepoint.com/Shared%20Documents/Forms/AllItems.aspx?csf=1&web=1&e=Oh7RDH&cid=3a56a85f%2Da34b%2D4dfb%2D9d3d%2D537743cb7866&FolderCTID=0x01200012B4B553563E0E45949B9B0FB30330A2&id=%2FShared%20Documents%2FRelease%20Notes%2FSchoolCafe%20Release%20Notes%2FLatest%20External%2F6%2E4%20Release%20Notes%2Epdf&viewid=85b14966%2D02c6%2D4e70%2Da484%2De785f7d89971&parent=%2FShared%20Documents%2FRelease%20Notes%2FSchoolCafe%20Release%20Notes%2FLatest%20External]: 
** This directory will always point to the *latest* SchoolCafé Release Notes in the form of a PDF.
* When the user clicks on the new link, open an overlay or 3/4-to-full drawer and display the content of the PDF within the frame.
** Content can be the PDF exactly, in which case we may need to provide a Download PDF button to save, or an iframe which points to the PDF and allows the user to take advantage of the native PDF browsing tools through SharePoint.
* User should be able to view, download from, and close this release notes window at any time in the future.
* Inspiration can be viewed on [https://www.schoolcafe.com|https://www.schoolcafe.com], where we display the release notes in a similar window but with hard-coded text.

## Acceptance Criteria
Verify the Version # section of the homepage has become a clickable link which opens an overlay/drawer/frame and point to the URL mentioned

Verify that the contents of this window display the Release Notes in the form of a PDF 

Verify that user is able to view, download from, and close this release notes window at any time

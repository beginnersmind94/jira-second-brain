# NXT-9676 — System - Regions - Add tab for SFAs

## Description
As a district/SFA admin, I would like to be able to enter the less-commonly known information for our region, so that proper information can be fed into the codes of various Letter Templates that will go out to families or other staff.

*Summary:*

* In Edit Regions, for any region with an Organization Level of Region, in the Region Details tab, add the remaining fields that show up in the SFAs page: County, Correspondence Address, and Tax Status. 

*Mock-up:*

TBA

*Details:*

* In System → Regions → Edit Region, in the Region Details tab, add/modify sections for the following (in addition to the existing content) +*displaying in this order*+:
** Physical Address (only State field required)
*** Should contain the current address fields that already exist
** Correspondence Address (all fields optional)
*** Can be duplicate of the existing address fields.
*** We can display a checkbox, like ‘Same as Physical Address’, that when checked, auto-fills the same data as the Physical Address fields and hides the Correspondence Address section
** County
*** Based on the State field selected in the Physical Address, display the counties in that state and allow the user to add/edit/delete them. We can carry over the same fields as Classic.
** Tax Status
*** Various sub-options that are basically the Site Characteristics section from Site Details, but on a whole region-level. PO can provide details if needed.

# NXT-16730 — Triggering Mechanism @ API Calls-Aeries SIS

## Description
This story is to cover the triggering mechanism of the API Calls.

As a end user, I would like to have a way to trigger (IMPORT DATA) the API calls. 

Once the API's are set up (in the current cases, inbound), there must be a triggering mechanism (button) to trigger the Inbound API and this trigger must be controlled. 

There is a pretty good chance that the user might click multiple times and so there must be warning messages tool tips.

Also, the user must be prevented for clicking more than once (deprecation of the click)until the initial trigger is completed. 

The process can be represented with a hour glass.

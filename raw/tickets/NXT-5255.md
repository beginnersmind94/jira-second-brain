# NXT-5255 — Eligibility - Settings - DC Precedence - Add Reduced programs section

## Description
As a district admin I have reduced direct certifications that should naturally be considered lowest in the precedence order, so I would like to be able to specify that precedence in order to avoid confusion regarding the numbering scheme.

*Requirements*

* In the Eligibility → Settings → DC Precedence page, add a 3rd grid to the page titled *Reduced Eligibility Programs*
* Move DC Medicaid (Reduced) from its existing section into this new section.
* Reactivate the Pre-K (Reduced) reason and add it to this section.
** If this 'Pre-K (Reduced) reason doesn’t already exist from copying Edge, we’ll need to add it to the database.
** Rename the existing Pre-K to _Pre-K (Free)_.
* For Edit mode, change the existing close button icon to the X icon instead.

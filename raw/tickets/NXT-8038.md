# NXT-8038 — {Menu} Analysis - Calculate/display Nutrient results (% of Calories)

## Description
As a Menu Planner
I need to be able to build Menus
so that I can show that they are compliant.

For requirements that are % of calories or nutrients that contribute calories, display %.

Fats (total, sat, and trans) contributes 9 calories per gram, while carbs (incl total and added sugars) and protein contributes 4 calories per gram.

Example with 377.22 Calories:

!image-20201211-182408.png|width=496,height=387!

Total Fat is 5.68 grams, so (5.68 * 9 / 377.22) * 100 \[to get in %] = 13.56%

Carbohydrate is 72.35 grams, so 72.35 * 4 / 377.22 * 100 = 76.72 %

## Acceptance Criteria
% of Calories is displayed for contributing  nutrients a described above

# NXT-3699 — Item Management - Add and Edit Food Item - Skip and Clear Allergens

## Description
*User Story*

As a Item Management User, I can skip or clear Allergens, both standard and custom

*Mockup*

!image-20200813-160951.png|width=100%!



*Requirements*

* 'Clear' button must clear out the information in the Header section section 
** If there is information populated, then have a popup showing a confirmation
*** “Are you sure you want to clear Allergen selections?
**** Confirm or Cancel
*** The background behind loses focus
* User must be able to select the ‘Skip for now’ checkbox if nothing is selected
** If they un-select the ‘skip for now’ checkbox, then the data should return as it was before
* If ‘No allergens indicated’ then hide the ‘clear’ and ‘skip for now’ buttons
* If there are selections, then disable the ‘skip for now’ button
* Custom Allergens
** Allow the user to delete after a custom allergen has been added by clicking the trash can
** When clicking the trash can, it will delete the row, but won’t save until the final save

## Acceptance Criteria
User can skip or clear a section when adding or editing
Allow the user to delete a custom allergen

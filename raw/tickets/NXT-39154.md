# NXT-39154 — IM - Items - Item Creation Issues/Improvements

## Description
Make various fixes and improvements as necessary to properly accommodate item creation flow for user experience.

*Issues*

* When leaving item without finishing pack/serving and have not answered questions, we are loading nutrient tab and then it goes away
* When adding images and saving, images not saving for weird reasons, image card stretching images when in expanded mode 
* Cost not scaling upwards if added on base unit (seemingly fixed)
* Costs for existing data is not porting correctly
* Sort order for Serving sizes is not sorting descending by Weight/Volume 
* Need better serving size selection. Should not be able to edit nutrient or contributions before saving serving size (also mentioned on [Nutrient Info story|https://primeroedgenext.atlassian.net/browse/NXT-37882?focusedCommentId=98348]) 
* Nutrient info stuck in Partial (requiring Contributions or Allergens)
** Only Serving and Nutrients required for tab ‘Complete’
* Info icon for serving showing incorrect info for use on contribution ( No contribution added)
* Can not add serving size or serving size weight, clearing and not allowing to save

*Improvements*

* Don't show the question bar until top level Unit card save
* Default inventory ready publish to on  (also mentioned on [banner enhancement story|https://primeroedgenext.atlassian.net/browse/NXT-38091]) 
* When confirming BU on save, yes or no does not make sense for BU prompt (also mentioned on [banner enhancement story|https://primeroedgenext.atlassian.net/browse/NXT-38091]) 
* When deleting "dual" it will remove the pack version only and keep the serving size with no Sub-qty
** Need a better way to manage this use case
* For pack serving link help step, fix styling for boxes, border too light gray

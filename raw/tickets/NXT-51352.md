# NXT-51352 — MP- Menu Planner- Consider Toggle from Standard Menu for MMA as Grain for Breakfast (UI ONLY)

## Description
As a menu Planner, If toggle on my MMA component item as a grain, this should reflect correctly on the menu planner and provide me with the same analysis as the standard menu or menu cycle pages.

## Acceptance Criteria
* Once user saves toggle MMA as grain, it should update the menu planner to consider the MMA as grain as part of the analysis
* This should affect both daily menu component analysis as well as menu cycle analysis
* Currently user can only change toggle from the standard as far as using MMA as grain but all updates removing or adding the toggle should reflect on menu planner

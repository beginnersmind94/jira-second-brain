# NXT-3534 — System - Users - Add/Edit User - Modify form layout

## Description
The goal of this story is to finalize the layout of the add/edit user form while accommodating for future development (add roles and password validations). 

Please see the mock-ups here: [https://nqyc9v.axshare.com/start.html#p=users&id=yfr8gz|https://nqyc9v.axshare.com/start.html#p=users&id=yfr8gz]



+*Goal:*+

All fields except role, password and user image/avatar should be validated. Default option should be add/selected for various drop-down fields.



+Details+:

# Based on the mock-ups, rearrange fields.

# Remove the ‘Theme’ field. This just takes up valuable real estate and can be moved to the user profile page because it isn’t important.

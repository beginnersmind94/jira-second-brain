# NXT-2960 — **View Inspections - List View

## Description
*User Story*
As an Inspections user, I can view Inspections on the View Inspections screen in either List View or Card View without any loss of information.

*Mockup*
[https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1|https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1]

*Requirements*

* Users can toggle between card view and list view when viewing Inspections
** Toggle is done through the buttons on the top right
** List View is the default
*** Last selection is remembered between sessions
** Sorting and filtering is remembered when toggling between the two views
*** Resets when session is refreshed
* List View
** Inspections are stacked on top of each other in a list
** Default sorting is by date, from most to least recent
** Sorting can be changed by clicking on the individual column headers
*** Can only be sorted by one column at a time - clicking a new one overrides the sort
* Default fields:
** Name
** Status
** Template
** Site
** Site Rep
** Date/Time
** Score
* Actions button is available, but should only have placeholders

## Acceptance Criteria
* List view is present and can be sorted and filtered
* User can switch between the two seamlessly without any loss in data

Note: Using dummy data is acceptable to demonstrate completeness

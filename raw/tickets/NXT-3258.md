# NXT-3258 — Edit Employee Details - Record Training Inline Add

## Description
Admin*** can Record training events for an individual employee within the Employee Details page.

* Select Add New Record
* Choose date of training 
* Choose training event name (populated from training event configuration)
** Event Details displayed - Event Name, Description, Topic, Subtopic, Method, Source, Attachment(s)*** icon
* Training event hours are shown based on event selection (populated from training event configuration)
* Check box to mark “Completed” - can only mark completed if the date is in the past

** if not checked, that training ‘planned’ but not completed
* Check for duplicates, show the date of possible duplicate [~accountid:5da678a69810cf0c3ce3a86e] Do we want to show only the events that are not already added for the current year or we would show everything and let the FE take the decision/logic check to have the duplicate value check?

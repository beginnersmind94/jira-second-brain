# NXT-48999 — AM - Reminders - Scheduler - Page

## Description
As Dana the Director I want a page where I can see and manage scheduled AcctMgt notification sending.

*requirements*

* User has a page where they can see the scheduled events.
* User can see a grid containing the saved/scheduled sends.

## Acceptance Criteria
* User has a page where they can see the scheduled events.
** Add a tab to the Notifications page called Recurring Notices.
* User can see a grid containing the saved/scheduled sends.
** Columns:
*** Name
**** The name of the scheduled recurrence.
*** Description
**** The description for it.
*** Type
**** The letter 
**** Reminder Letters is the only type for now - we’ll roll this option into other pages later.
*** Frequency
**** Recurrence, Days, From StartDate + StartTime until EndDate + EndTime
*** Last Sent
**** Last time the system sent the blast out
*** Next Send
**** Next time the system will trigger the event
*** Status
**** Active/Inactive/Deleted
*** Update Date
**** Last time it was edited (or created)

# NXT-7649 — INV - Physical Inventory - View Physical Inventories

## Description
As a site or district inventory user I want a list for opening existing physical inventories so I can resume in progress counts, review completed inventories, or generate reports.

[Figma|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=88882%3A148612&viewport=325%2C-4288%2C0.08251962065696716&scaling=scale-down]

Once the search filters have been applied, load the results in the Physical Inventory List

List of Physical Inventories (grid)
Table columns:

* Site Name
** add Code with the name
** ex: 101 - Elementary School
* Status
* Started By (user first and last name)
* Last Modified By (user first and last name)
* Last Modified Date
* Counted At (date + time)
** uses last day of the period if inventory is done by period
** this is the date on which the perpetual will be updated
* ‘View’ link/icon to open the selected Physical Inventory, story [https://primeroedgenext.atlassian.net/browse/NXT-7785|https://primeroedgenext.atlassian.net/browse/NXT-7785|smart-link]

Technical Notes:

* These values are pulled from what is saved on the completed or reconciled physical inventory. 
* The Started By, Modified By users will be saved to the Physical Inventory
* The dates and times (modified, counted, etc.) will also be saved to the Physical Inventory.
* If these calls are made as part of this story, effort is 8
* if these are made elsewhere and just displayed here, effort is closer to a 5

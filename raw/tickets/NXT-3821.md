# NXT-3821 — Account Management - Advanced Search - Restrictions Section - Payment Restrictions Section

## Description
The Payment restrictions section has currently has two restrictions ‘No Charge’ and ‘No Checks’. Like all other checkboxes they can be ‘or’ed. 

!Screen Shot 2020-04-29 at 12.12.47 PM.png|width=288,height=111!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Payment Restrictions section they are 'or'ed together.

When a filter value is selected the Payment Restrictions Subsection count is increased by 1 and the Restriction Section count is increased by 1.

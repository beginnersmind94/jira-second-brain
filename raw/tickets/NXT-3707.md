# NXT-3707 — Add New - Import - Template and Upload Step 2 - Validation

## Description
*User Story*

As a Item Management User, I need to be able to create a new item via import and want to validate it

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=import_item&id=o4qto3|https://nqyc9v.axshare.com/start.html#p=import_item&id=o4qto3]

*Requirements*

* After selecting the file and clicking ‘Next’ from Step 1
** Validate the file based off of meeting required fields
** Allow in line editing in the UI
** If missing information, flag the information
* Columns included
** Item Name
** ID
** Vendor
** Datasource
** Product Code
** GTIN
* If validated successfully, display message that it’s Validated and allow the use to click next
* If not validated and the user tries to click next, display warning “Missing information in the file. Please delete the row or correct the information.
* After clicking Next and if it is validated, move to Step 3

## Acceptance Criteria
The file will be validated in the system based off the template

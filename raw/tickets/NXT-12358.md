# NXT-12358 — Platform - AI/ML - Register New Recognition Device

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to setup authentication token for new EdgeBox with a serial number and assign to a school district so that the device is pre-configured

*Mockup:*

[https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|smart-link] 

*Requirements*:

* Admin logs into PrimeroEdge portal, to setup up new recognition device with *Serial Number and Model*  of the EdgeBox that belongs to specific *SITE*
** +First 4 digits of the Serial Number will identify ONLOGIX Edge devices+
* -_Each EdgeBox should have 2 cameras attached to it based on UUIDs of the devices_-
** {color:#00b8d9}*-_Camera device must display the Model, UUID and SiteID assigned_-*{color}
* Each EdgeBox is configured with authentication token in encrypted fashion and stored in the device
* Each EdgeBox must be assigned to a SITE with Site Name, Serial Number (UUID), Authentication Token, POS Name and description of the device
* Each EdgeBox should be in “NEW” status after initial registration

## Acceptance Criteria
* User shall be able to add new Edge device from the device catalog screen from register new device button from device catalog screen
* User shall be able to validate Model & Serial number (first 6 digits) while adding the new device into Perseus
* Status should be set to “NEW” during after registration step
* User shall assign only one SITE per device
* User shall be able to generate device key during device registration

# NXT-56748 — PR- Carryovers- Carryover for Record Complete

## Description
As a production manager, there are times were I finish my record and then I forgot that we had carried over some items from a previous day and I would like to be able to add that to my record. Currently, if the record is already complete, it will not allow me to.

## Acceptance Criteria
* Allow record complete menu lines to show on the carryover calendar
* Display a warning message under the calendar “The day selected is in Record Complete Status, adding carryovers to this day may requires changes to the record”



!image-20241112-211711.png|width=517,height=920,alt="image-20241112-211711.png"!



* When users add a carryover, automatically add 0 for produced to preserve the status and the total served should just be the carryover amount (If menu Item is Can Serve)
* When users add a carryover to a Menu Item Planned, Add the carryover count to the served total (Produced+ Carryover)

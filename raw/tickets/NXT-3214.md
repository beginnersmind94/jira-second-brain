# NXT-3214 — *Inspection Template - Active Status

## Description
*User Story*
As an Inspections user, I have an Active state for Templates that are available to be used to create a new Inspection.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1|https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1]

*Requirements*

* Templates have an Active status
** Active templates appear in the Active portion of the Template Hub screen
* Active templates cannot be edited
* Active templates can be moved to Inactive
** There is an action available in both Card and List view to do this
** The action is present as a button in Card View

## Acceptance Criteria
* Templates can be moved from Inactive to Active and back
* Active Templates appear in the Active section of the Template Hub screen

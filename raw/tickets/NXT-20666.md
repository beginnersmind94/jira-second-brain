# NXT-20666 — Production- Reports- Technicians Worksheet in Spanish (Production Plan and Record)

## Description
This report is a way for users to print all of their items and enter the numbers with pen and paper to use to enter production records. This reportwill be able to be generated from the Production Plan and Production Record screen as well.

## Acceptance Criteria
Must translate entire report in Spanish
Report must contain the following information:
Serving groups with their Planned Counts and a space for them to write actual counts
A total row for planned counts and actual counts
Plan comments box
Record Comments Box

Columns must include:
Serving size and if they have a portioning tool display what it is
Planned
Produced
Carryovers
Return to StockWaste
Adult/A La Carte

Items with Recipe code display in Category order within the category and what their Choose is and nutrition contribution
Example:Category: Entrees: May choose 1Cereal Cinnamon Toast Crunch (SY21-22) (LR 1031) 2 WGR

This item also needs to have food required associated with it which should pull based on Planned Counts.

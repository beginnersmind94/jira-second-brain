# NXT-7127 — Data Exchange - Field Mapping Screen - Allow user to configure fields

## Description
*User Story*

As a PrimeroEdge user, I would like to have an intuitive screen that allows me to configure fields in PrimeroEdge to be formatted based on the fields in my uploaded file, so that there is no conflicting or bad data.

*Summary*

Allow a user to open a pop-up that lets them configure various formatting options for the linked fields.

When a user opens a pop-up and hits the Save button, indicate that the field has been configured.

*Requirements*

* In Data Exchange → Field Mapping, when a user has dragged and dropped a configurable field (handled via back-end or separate story), display a link to ‘configure’ it in the related Configure Field ('Drop' location) cell.
* When the link has been selected, open a modal with Save and Cancel buttons, to allow the user to edit that field’s structure.
** We can carry forward the old structure from Classic. i.e. for the purposes of this story, we can use Student Import’s configurable fields - i.e. Grade, Date of Birth, Active, etc.
* When the Save button is selected, close the modal and change ‘Configure’ to ‘Configured’ and the font color to green. 
* TBD - Field Mappings for Grade, Gender for data transformation purpose

Verify that when the link has been selected, we open a modal with Save and Cancel buttons, to allow the user to edit that field’s structure.

Verify that when the Save button is selected, close the modal and change ‘Configure’ to ‘Configured’ and the font color to green for that particular field.

# NXT-14630 — Integration - File Configuration - Conditions

## Description
Each file configuration should have the ability for conditions (settings, etc) rather than using overall System settings. For example, for SIS, we should allow the user to setup a threshold for withdrawal percentage. This gives flexibility to establish conditions for each file import and/or export.

Needs discussion within the team.

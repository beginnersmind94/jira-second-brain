# NXT-54206 — IM - Items - Inventory Info tab - Contracts card - Show historic contracts

## Description
As a Food Service Director I want the Inventory Info tab Contracts card to show inactive contracts the item was on, so I can know contract history for any item.

## Acceptance Criteria
* Update the Contracts card to show all contracts of which the item has been associated, including those not in Active status
** *DO* show all historic instances of the contracts associated (expired, inactive, suspended, etc.)
** *do not* show contracts from which the item was +deleted+ (as opposed to made inactive or unorderable)
* Change the title of the card to just ‘Inventory Item Contracts’
* Change the message that is displayed when the item is not on any contracts
** "This item is not assigned to any Inventory vendor contracts."
* Add Contract status to the card
** just show as a chip next to Contract description (including the Active status)
* Sequence:
** Custom entry contract (if added)
** Active contracts
** End Date descending (Inactive + Suspended) 
*** most recent first

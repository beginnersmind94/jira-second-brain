# NXT-3027 — Item Management - Add and Edit Food Item - Skip and Clear Serving Measures

## Description
*User Story*

As a Item Management User, I can skip or clear Serving Measures

*Mockup*

!image-20200812-203757.png|width=100%!

*Requirements*

* Clear button must clear out the information in the section
** If there is information populated, then have a popup showing a confirmation
*** “Are you sure you want to clear Serving Measures”?
**** Cancel then Confirm
*** The background behind loses focus
*** When clearing serving measures, it deletes the rows in the database after confirmation
* User must be able to select the ‘Skip for now’ checkbox
** When selected, minimize the section and ‘cache’ the data
** If they un-select the ‘skip for now’ checkbox, then the data should return as it was before
* User cannot ‘clear’ the information if the serving measure is already in use

## Acceptance Criteria
User can skip or clear a section when adding or editing

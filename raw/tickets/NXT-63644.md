# NXT-63644 — INV - Vendors/Contract Items - Only show Cancel when applicable

## Description
As a food service admin I want the Vendor and Contract Items page to show a Cancel button only if I have unsaved changes, so I can confidently know when the Back button will just take me back.

!image-20250326-231447.png|width=91.66666666666666%,alt="image-20250326-231447.png"!

## Acceptance Criteria
* Change all Update screens to dynamically modify the Back/Cancel button based on unsaved changes
** if the Save All button is enabled, change Back button to Cancel
* If there are no changes on screen (no dirty data) then only show ‘Back’ button

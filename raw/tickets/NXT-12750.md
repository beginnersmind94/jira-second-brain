# NXT-12750 — IM - Recipe - View Ingredients

## Description
As an item management user, I need to be able to see all my ingredients and items in a recipe

Mockup

View Items

!image-2021-03-24-08-43-48-478.png|width=100%!

 

Requirements

* We need a summary of all the items in a recipe on the view page for the items
* For example if we had 2 cups of milk in step 1 and 2 tablespoons of milk in step 2, then we need the total quantity, which would be 2 cups 2 tablespoons, or the necessary conversion
* Higher measure is the primary measure
** If secondary is a lower measure, then we need to add it up to the higher measure
* Would like to see the direct link to the item / recipe when clicking on the view open in a new tab (Team is going to check on the effort)

## Acceptance Criteria
As a user, I need to be able to see the summary of the items in the recipe steps

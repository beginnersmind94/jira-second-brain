# NXT-23301 — INV - Reports - Print Orders - Consolidated Quantity by Vendor

## Description
As a Procurement Specialist I want a Print Orders report so I can generate reports for all my orders to send to vendors.

Users can print the list of orders for specific sites, for specific vendors or specific statuses for a given delivery date range. 

Each report type (radio option) will be its own story. The UI for generating the different types will be the same, but each report option will have its own UI (report) and API.

*Requirements*

* Create a new report page titled Print Orders
** this will be listed in the new Reports landing page ([NXT-22621|https://primeroedgenext.atlassian.net/browse/NXT-22621])
* Provide filters to define the dataset selections
** Site control (Area + Site)
*** multi-select
*** default to All
** Vendor dropdown
*** multi-select
*** default to All
** Status dropdown
*** multi-select
*** default to ‘Open’
** Date Range selection
*** default to today’s date 
** Date toggle for Delivery Date or Order Date
*** default to Delivery Date
* *Note:* Report Type section will be handled later.
** For now we will start with ‘One Order per Page’ as default report

!NXT-7987 1.png|width=100%!



Report options are: 

 *Print all orders one order per page for the selected report criteria.* 

*  Sorted by site , vendor and delivery date, order status. 
*  Header: site name, vendor name, order#, Delivery date, order status , Total order Amount, order comments, Vendor address ,vendor phone number. 
* Details: District item number, Item description, Vendor item number, Ordered WU Qty, Price, Total Price calculated for the order Qty, item Comments.

!image-20211006-150329.png|width=867,height=614!

*Print the consolidated Order Qty of ordered item Quantities from multiple orders of one or all site , vendor , for a delivery date and for selected order status(es)* 

*  Sorted by vendor and delivery date, order status 
*  Header: vendor name, Delivery date, order status, order comments, Vendor address , phone number. 
*  Details: District item number, Item description, Vendor item number, Total Ordered WU Qty, Price, Total Price calculated for the total order Qty, order# (comma separated) 
*  When one site is selected header can show the site name, but if all sites selected then the item Qty is consolidated amount for all sites from a specific vendor.

!image-20211006-151844.png|width=875,height=578!

*Print orders by item*: apart from the above selection criteria for site, vendor , delivery date range , order statues etc.. we need to give an option to select one or more items. 

when generated by this option , report information is grouped by item.

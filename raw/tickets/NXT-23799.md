# NXT-23799 — Front Office Reports - Eligibility - Add button to export report to Excel

## Description
*Updated title and cloned story to make sure reports in both modules are reviewed.*

----

As Vera the Director I want to download a copy of my reports to Excel, so that I can sort and filter the data to better suit my district’s needs.

*Mockup*
[https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=5416%3A53347&scaling=scale-down-width&page-id=5416%3A53078&starting-point-node-id=5416%3A53082|https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=5416%3A53347&scaling=scale-down-width&page-id=5416%3A53078&starting-point-node-id=5416%3A53082|smart-link] 



*Requirements*

* For the Eligibility → Reports, -Accountability → Reports pages,- *-and-* -Verification Collection Sheet/Application Processing reports-): when a report is generated, add a button to download the report to Excel also (in addition to the PDF button). There’s a big need for this to manipulate numbers for other reporting needs outside of our system.
* If there are any reports that cannot be translated in Excel, please let the PO know so that a decision can be made. Likely, since the PDF option should already exist for all reports, we would just hide the Excel button in that case.

## Acceptance Criteria
* All Eligibility reports (or as many as possible) have a new download button that exports the report to Excel.
* Behavior/content of reports should be otherwise unchanged

# NXT-32804 — DataQuery Framework - Show All Reports on Page Load

## Description
DataQuery Framework - Reports page should show all reports for all modules on page load

## Acceptance Criteria
* A “Module” column shall be added to the grid that will be filterable.
* All reports for all modules (owned by or shared with the user) shall be displayed by default.
* See “Reports List On Page Load.pdf” for a UI markup containing more acceptance criteria.  The complete set of acceptance criteria shall be a union of the above text and the notes in these attachments.
* The value that the user selects for “Items per page” shall persist until the user changes it (persisting in browser storage is acceptable)

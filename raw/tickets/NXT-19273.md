# NXT-19273 — INV - Create Quick Receipt - Remove Broken Units

## Description
As a Site Manager I want the Quick Receipt screen to not include broken units so I can see accurate item information when viewing receipts.

*Requirements*

+On the Order step+

* Remove Broken Unit Quantity column
** remove all references to broken units
* Rename ‘Whole Unit Quantity’ column to ‘Quantity’
* Change Item Number to ‘Item #’
* Change Item Name to Item Description
* Change Vendor Item Number to ‘Vendor Item #’

+On the Receipt step:+

* Remove broken unit textbox from Receive Quantity column
* Remove the “(WU/BU)” from the Receive Quantity column header label

## Acceptance Criteria
* Remove Broken Units from Quick Receipt process

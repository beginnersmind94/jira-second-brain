# NXT-5323 — Recipe - New Recipe - Add Ingredients - Qty and Units

## Description
As a item management user, I want to be able to add a recipe through Item Management

Mockups
[https://nqyc9v.axshare.com/start.html#p=view_recipes&id=gehfhr&view=default|https://nqyc9v.axshare.com/start.html#p=view_recipes&id=gehfhr&view=default]
Add New - Manually

!image-20200910-132043.png|width=100%!





Workflow: Add New → Manually → Click ‘SCALE’ to view the items

Requirements

* Item Name
* Item Code
* Quantity & Units
** Primary Quantity & Units
*** Numerical Value (user input field, keyboard friendly)
*** Measurement (List of measurements based off the item)
** Secondary  Quantity & Units
*** Numerical Value (user input field, keyboard friendly)
*** Measurement (Convertable measurement based off the primary unit of measure)
* Comment - Optional
** Freeform text field (Max 60 characters)
* 'X' to remove the item
** This will remove the Step that was entered
** Confirmation when clicking X
*** Are you sure you want to remove this Step?
**** Cancel or Confirm
** User is unable to delete the item if it is being used in a recipe step
*** If the item is used in a step, then display the message
**** This item is being used in Step \[# of the step]. Please remove this step before removing this item.
* For non-food item
** Purchase Whole unit measurement
** Broken Unit Description

## Acceptance Criteria
User must be able to search and add items to a recipe

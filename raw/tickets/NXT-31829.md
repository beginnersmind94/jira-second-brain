# NXT-31829 — MP- Menu Planner- UI Improvement to show all weeks together

## Description
As a Menu Planner, I would like to have a way to see all my weeks at the same time. Having to go to multiple tabs does not give me a clear view of being to see how my menu cycle is set up.

## Acceptance Criteria
Option to Display all weeks together (Same logic applies, 3 days must be created for the previous week before next week can be created/used)

Option to Display certain weeks together ( EX: I want to compare week 1 and 4) 

Default should always show all weeks selected with the option for the user to pick what weeks they want to hide

Rearrange styling to allow user to see menu item name as clearly as possible ( category and cross should be rearranged to fit better on the chip)

# NXT-73304 — FIN — Reverse Import: Payroll Import

## Description
Adds Reverse import support for **Payroll Import** on the Import & Export Log page (NXT-73286).

Payroll Import follows the ledger-posting import reverse pattern. Batches for non-overlapping PostedDate ranges are independently reversible.

**Gates:** Year-End Close (hard block), overwrite per NXT-72753 (hard block), closed period (prerequisite — reopen to unblock).

All UI patterns (confirmation dialog, mandatory comment, Cancel / Reverse import buttons, Status = Reversed, audit trail) follow NXT-73286.

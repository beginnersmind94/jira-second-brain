# NXT-3028 — Create the Search/Filter/Lazy Loading View for the UI

## Description
*User Story*

As a Item Management User, I search and filter for my information, be it an ingredient or stock item

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1]

*Requirements*

* Search at the top of the list
** Allow to search by any of the fields, primarily item name or ID
* Click ‘Filter’ Icon to access filters
** Search by tags
** Search by Vendors
* Select by Category - drop down
** Food
** Non Food
* Dynamically update the list when searching
* Show 20 items at a time
* Allow for lazy scrolling for the list on the left

## Acceptance Criteria
List screen reacts to search filters and sorting

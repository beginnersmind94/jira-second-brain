# NXT-19674 — Platform - BO - Onboarding Imports Summary Widget

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view system Imports summary widgets with its overall progress and take necessary actions

*Mockup:*

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|smart-link] 

*Requirements*:

* Site admin logs into Onboarding Summary Dashboard page for Imports widgets displayed in “Dashboard” tab
* Imports tab displays all Imports that has to be configured for given district
* Imports table displays all imports progress % for each import along with actions
* Imports table includes actions available for each import with START or RESUME the import job based on the import type
* Imports widget has VIEW DETAILS button which opens Imports tab in the onboarding page.

## Acceptance Criteria
* User shall be able to see the Imports widget in Onboarding dashboard
* User shall be able to see VIEW ALL button to open Imports tab under onboarding page.
* User shall be able to see status of the imports for each import type
* User shall be able to resume the import progress of each import row available in the table
* User shall be able to click on “Start/Resume” button to start if the import has not started

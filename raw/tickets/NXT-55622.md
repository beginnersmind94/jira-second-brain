# NXT-55622 — DMF - Dashboard - Duration

## Description
As a Customer Support role user, I would like to be able to view the duration of the module imports I am migrating so that I can quickly see how long each step is taking

*Requirements:*

* On the data migration dashboard, do the following:
* For each database row, do the following:
** Add Start Time
** Add End Time
** Add Duration
* 
* Create an API endpoint to provide this data per database

## Acceptance Criteria
* User is able to see the start and end times and total duration for each database migration

# NXT-32521 — Import - Vendor Contract Schedules - Platform Portion Only - 3 of 3

## Description
IM - Onboarding Items - Allergen Information-

As an Item Management Super User / Administrator, I need to be able to onboard new Item Allergen Information using an excel file in order to speed the process of bringing customers working with SchoolCafe’s Backoffice.

## Acceptance Criteria
*Step: Pre-Upload*

* A new “Import Type” shall be used for this import named “Inventory - Vendor Contract Schedules”.
* A dropdown parameter named “Data Source” shall be present containing one entry named “Local”.
** The “Local” Data Source entry shall be selected by default
** There is lack of clarity from the Back Office request on additional values within the “Data Source” dropdown.  Until there is, we will treat it as if there are no additional values.
* The user shall be prompted to download a template using the standard verbiage
** The “No” radio button shall be set by default
** The file template to be used shall be attached to and provided by the Back Office team in story [https://primeroedgenext.atlassian.net/browse/NXT-30257|https://primeroedgenext.atlassian.net/browse/NXT-30257|smart-link] .  The fields marked in red font in the attachment are mandatory.

*Step: File Upload*

* No changes from existing behavior

*Step: Field Mapping*

* No changes from existing behavior
* No code mapping required

*Step: Review*

* No changes from existing behavior



*Business Rules and Error Log Rules*

* The “Business Rules” and “Error Log Rules” as stated in [https://primeroedgenext.atlassian.net/browse/NXT-30257|https://primeroedgenext.atlassian.net/browse/NXT-30257|smart-link] shall be the responsibility of the Back Office team.

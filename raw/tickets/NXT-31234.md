# NXT-31234 — Accountability - POS allow homeroom/roster lookup to display as a picture grid or list

## Description
As Sylvia the Cashier I want the Advanced Lookup Homeroom/Roster to display as a picture grid rather than the current list to allow serving to go faster because I can avoid scrolling.

!image-20220525-144054.png|width=1280,height=984!

!image-20220525-144218.png|width=1280,height=984!

## Acceptance Criteria
There is a new slider that defaults to ON Expanded (Shows in the current style with all the details)

The slider stays set for the user until changed by the user

When the slider is off a grid display of the students shown (New view)

The Grid view shows the Picture and the name. The name may be truncated

The Sort By selector changes the order displayed and stays set until changed by the user.

The Show All Slider works as currently implemented, When OFF shows only students not served a Reimbursable Meal for the selected meal type, When ON shows all patrons in Homeroom or Special Roster.

Touching or Clicking the Patron loads them to be served.

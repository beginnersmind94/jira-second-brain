* User can add additional item features as part of a sale
** We’re combining the additional items + adding additional items into one overlay with a tab
** If no items have been added yet show the following string in the A La Carte Items tab:
*** No additional items have been added. Please use ‘Add Items’ to add items.

## Acceptance Criteria
* User can add additional items 
* User can see the additional items that have been added
* User can remove them
* They are sold correctly when the user clicks Record Sales

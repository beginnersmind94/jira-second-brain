# NXT-65332 — ACC - Attendance Feature - ADA - ADP Report

## Description
Another common metric that districts find useful is the Average Daily Participation (ADP). With the implementation of attendance tracking, we can utilize the entered data in this stand alone report.

Add New page report in Accountability > Reports

Columns
Site Name
Meal Type - shows only configured meal types (SNP only)
Total Attendance/ OR / Total Approved Counts - (when use attendance counts is selected on the report option shows the sum of entered counts from attendance ELSE shows total Enrollment sum per site selected in the report for the selected date range. STUDENTS ONLY)
”# of Serving Days [DATE RANGE 00/00/00 - 00/00/00]” - Selected Date range in the report where at least 1 meal was served
Average Daily Attendance - [Total Attendance / # of Serving Days where a meal was served]
Total Meals Served [Total meals served for selected date range]
Average Meals Served Per Day - [Total meals served for selected date range / # of Serving Days where a meal was served]
Average Daily Participation - [Average Daily Meals / Average Daily Attendance]

Report Filters
Site (Multi Select)
Date Range (standard control)
Meal Type (Multi Select)
Use Attendance Counts for ADP calculations (check box Default unchecked)
When check = Use entered attendance Count
When uncheck = Use Approved Count

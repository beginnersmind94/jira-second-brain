# NXT-30217 — IM - Items - Item Details - Edit Global Info

## Description
As a District Admin I want a header section so I can view the main/global info for an item.

*Requirements*

* Provide an Edit icon for the top-level header info of Item Details
** this Edit must be separate than other header sections
* In Edit mode
** Enable the Active flag/toggle/checkbox
*** More info TBD
** Enable identifying info
*** Item Description
**** *Mandatory, Unique*
**** Alert user if Description is not unique: “This exact description is assigned to another item. Please enter a unique description.”
*** Brand Name (dropdown)
**** Brand Name will *not* be interdependent with Manufacturer
**** keep the ‘Manage Brands’ shortcut 
***** future permission based
*** Product Code
*** Manufacturer (dropdown)
**** keep the ‘Manage Manufacturers’ shortcut 
***** future permission based
** Allow Tags to be edited
** *Do not* allow Data Source to be edited 
* Provide Cancel/Save buttons

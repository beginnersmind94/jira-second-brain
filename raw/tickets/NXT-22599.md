# NXT-22599 — INV - Display Time (Zone) Consistently

## Description
As an Inventory user I want everything displayed in my local district time so I don't have to convert from UTC time.

*Requirements*

* Convert every user-displayed time into district time zone
** System Settings: District Time Zone (pick Time Zone)
** System Setting: Observe Daylight Saving Time (Yes/No)
** All dates stored in UTC
** When retrieving the dates, shown in local district time zone
* Display consistently across all module pages and reports, including history

## Acceptance Criteria
* Time should be displayed consistently within Inventory on all pages, reports, etc.
* All times should be displayed consistently across every module

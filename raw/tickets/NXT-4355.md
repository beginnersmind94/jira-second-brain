# NXT-4355 — IM - Add item - Serving Measures - Preferred Measurement Box

## Description
As a user, I want to add / remove serving measures from the 'Add item' screen when checking and unchecking the preferred measurement box.

Mockups
https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&view=default

Requirements
* When adding an Item and navigating to the Nutrient - Serving Measures section, after adding a serving measure and showing possible measurements, allow the user to check and uncheck the box
* When checking the box in the possible measurement, add the row to the serving measure table
* When unchecking a box in the possible measurements, delete the row from the serving measure table
* User can remove the row by clicking the trash can and confirm if they want to delete it
** If this action happens, then it would uncheck the box in the possible measurement area

## Acceptance Criteria
Possible measurements need to be added and removed when the preferred measurement checkbox is selected and unselected

# NXT-53406 — MP- Menu Configuration- Combining current configuration pages into list and drop down (UI ONLY)

## Description
As a district admin or a menu planner, I would for all my configuration pages to be consolidated into 1 place and I can make the necessary adjustments to them all from a drop down that changes what configurations I am working on.



!image-20240731-133239.png|width=1748,height=674,alt="image-20240731-133239.png"!

## Acceptance Criteria
* Create a New Page (Menu Configuration)
* Provide a drop down at the top of the page that will give all the configuration options
** Sequence of drop down is
*** Meal Patterns
*** Menu Lines
*** Menu Quick list
*** Menu Templates
* When accessing Menu Configuration Page, land user on meal Patterns by default
** All configuration pages function the same and the meal patterns page is still a view only page
* If user refreshes, keep the selection that was selected before

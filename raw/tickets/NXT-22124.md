# NXT-22124 — IM-> Items-> Spinner or loading should be shown at the time of searching item on field search

## Description
Go to schoolcafe site
Click on item management tab
Click on items
Click on field search
search for any item .. for eg. HTIR909

Expected Result-.> While searching for item user should be able see loading sign (Spinner) so that user get to know system is searching for item name
Actual Result ->    user is not getting any sign that system is loading to search item

## Acceptance Criteria
While searching item user should be able to view the spinner , so that user knows system is loading search data

# NXT-25550 — INV - Vendors - Contract Items - Allow an item to be removed from Contract

## Description
I am unable to remove an item from the vendor's contract.  The only option available is to make it not orderable, but this does not remove it from the contract.

*Note:* check with Raju/Michael to determine if items should be fully deleted or if we should save removed item history (efforting without this consideration)

*Requirements*

* Add a ‘Remove from Contract’ button (previously had this but was removed during redesign) 
* *Prevent* users from removing items from the contract when…
** the item is on *Open* orders
** “This item is present on open orders and cannot be removed until these orders are closed: #12345, #42424.”
* Do not remove the item from any existing transactions

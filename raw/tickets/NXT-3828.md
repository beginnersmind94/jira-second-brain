# NXT-3828 — Account Management - Advanced Search - Demographics Section - Gender Subsection

## Description
The Gender Subsection will allow the user to filter the patrons list by their Gender. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Gender values. The Genders are listed in Alphabetical order. As Genders are selected they are moved to the top of the list. Unselected Genders are separated from selected Genders and are shown in alphabetical order.

!Screen Shot 2020-04-29 at 2.53.47 PM.png|width=284,height=179!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Gender section they are 'or'ed together.

# NXT-25731 — PR- Unsaved changes Pop up when leaving Production Record

## Description
As a user, When I enter data on my production record there needs to be a pop check for unsaved changes.

## Acceptance Criteria
Display Pop asking user what they would like to do. 
Display message "There are unsaved changes on this page, would you like to save your changes before leaving the page? Any unsaved changes will be lost. (Yes or No)

Do a status check after changes are saved if user selects yes. Update the status as necessary.

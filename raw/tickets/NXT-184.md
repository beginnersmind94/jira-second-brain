# NXT-184 — Item Management - Meal Pricing

## Description
*Story*

As a user when a menu item is created I need to be able to assign prices to the item so it can be sold. This should be part of Item Management. We should also be able to set a default prices for all reimbursable meals.

*Mockup*

Needs final review and approval

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=4288%3A69517&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=4288%3A69517&scaling=scale-down-width]

*Requirements*

* Individual Item Pricing for Students: Free/Reduced/Paid by Site Type
* Individual Item Pricing for Students: Free/Reduced/Paid by Grade Group
* Individual Item Pricing for non-students Staff/Visitors/Program Adults by Site Type
* A default prices for all Meals can be set.
* All prices for an item can be set the same
* Item can be marked as Taxable, Default is NOT TAXABLE
* POS Item information is considered Local Data.

## Acceptance Criteria
Depending on the system setting for the type of pricing, Site Type or Grade Group, the prices are set 
All prices can be set with one entry.
all prices for an eligibility can be set with one entry.
all prices for a site type or grade group can be set with one entry
The price can be set from a default price
Price types are determined from the price types of the district
Site Types are determined from the serving site types of the district
When 'Use Default Meal Prices' is enabled the user will select the Item to use as the price model from previously set default items. The prices for the item will be copied from the selected default item.
Prices can be set to 0.00
IF no prices are set when save is selected a message "All prices are being set to zero. Do you want to Continue Y/N'
No negative values

If there are no Default Reimbursable Items then a message is displayed when the 'Use Default Reimbursable Meal Prices' toggle is selected, Message = "No default meals defined, please create a default meal"
When the Role permission 'POS Menu Items' is disabled the user can only view POS Information.
    All fields in POS Information are viewable.

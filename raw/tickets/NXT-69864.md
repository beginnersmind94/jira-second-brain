# NXT-69864 — Menu Analysis -Dashboard - Item Performance - Provide Filter sliders for Days Served and Schools Served

## Description
As a district manager looking at my menu items, I would like to have a way to filter items out based on the number of days served and the number of sites it was served to in a particular time frame.

!image-20260203-044845.png|width=916,alt="image-20260203-044845.png"!

Reference for Slider:[https://mui.com/material-ui/react-slider/|https://mui.com/material-ui/react-slider/|smart-link]

## Acceptance Criteria
* On Item Performance Dashboard, provide a filter that allows users to filter by min and max Days Served and Schools Served
* Both filters will be affected by the Date range filter, Schools Filter and Categories filter
* Update the filter sliders to include what the min and max are based on the other filters selected
* Allow user to use slider to select minimum or maximum number of days served
** Provide a text box to allow them to type/select if user does not want to use slider
* Allow user to use slider to select minimum and maximum number of schools served
** Provide a text box to allow them to type/select if user does not want to use slider
* Default filters to the minimum for both 
* Hold filters if they return to the page from the Item Performance list page and return to the dashboard
* The Schools Served and Days Served filter apply to all metrics Overall Performance, Participation Score, Waste Cost and Waste Percentage
* Allow menu items in the graph top/bottom to be clickable and lead them to the menu item detail page

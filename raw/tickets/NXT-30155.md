# NXT-30155 — IM - Items - Create New Item - POS Info

## Description
As a District Admin I want a POS Info tab so I can enable it to be sold on the POS, configure POS attributes and set the pricing. 

*Requirements*

* Enable and load the POS Info tab if the user specifies the item as a POS Item during creation, per [https://primeroedgenext.atlassian.net/browse/NXT-30128|https://primeroedgenext.atlassian.net/browse/NXT-30128|smart-link] 
** color the tab as red to indicate missing mandatory attributes
** Incomplete: flagged as POS but no mandatory attributes
** Partial: mandatory attributes but no pricing
** Complete: mandatory attributes and pricing for at least some groups
* Display the main attributes in the header section
** Button Name (English) - *Mandatory*, freetext
** Button Name (Spanish) - freetext
** Meal Type - dropdown
*** Optional, but required if using Generic Item Pricing
** POS Item Category - dropdown
** Flags (checkboxes)
*** Meal
*** Vendable
*** Generic Item
* Display the Pricing section
** Provide the flag attributes
*** Taxable
*** DM Reimbursable Meal
*** Use Generic Item Pricing
**** this shows another dropdown if the user checks this
**** Generic Items - dropdown, *Mandatory*
**** This requires a Meal Type to be selected
** Display the Pricing grid
*** Separate Students and Adults with a tab
*** Provide the ‘Set All Prices’ option which can be used on one tab at a time (in Edit only)

## Acceptance Criteria
* Enable the POS Info tab if the user specifies the item as a POS Item

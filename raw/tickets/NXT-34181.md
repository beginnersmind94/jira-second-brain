# NXT-34181 — MP- Menus/ Menu Cycle- Auto Sync updates to menu item contributions, nutrition, to show on analysis (API ONLY)

## Description
As a menu planner, when I change contributions for a menu item or nutrition for a menu item as well as a name, it will update on the menu and sync properly, but in order for those changes to show on the analysis, I currently need to save the menu again for those changes to reflect. 

This also effects when I am trying to run the menu cycle analysis since each menu is independent and needs to be saved to display correct information on the menu cycle analysis. I also should not have to resave the menu cycle after I have made changes to menus to be able to update the analysis.

## Acceptance Criteria
* Auto sync menu item, name, nutrition and contribution information to the analysis without having to resave the menu
** This should effect the menu cycle page as well to where you do not have to resave all the menus to reflect the changes on the menu cycle analysis

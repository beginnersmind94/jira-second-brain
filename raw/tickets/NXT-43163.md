# NXT-43163 — IM - All Types - Images & Documents header card

## Description
As an Item Management user I want the ability to preview the images and documents I have assigned to an item, recipe, etc. from the main Details page so I can use the card to access the management slide-out.

## Acceptance Criteria
* Update the header level Images & Documents card to accommodate the new management tool
* Rename the card to “Images & Docs”
** Increase width of card to accommodate new title label
** Rename ‘Menu Item Image’ to ‘Preferred Menu Image’
* Make each image clickable (magnifier) to expand image and view full size (with max size limit; i.e. don’t show too large of an image on screen)
* Stack images in vertically scrollable arrangement (current behavior)
** maintain image aspect radio (do not stretch)
* Add a record at the bottom of the card which displays the number of uploaded +documents+ (non-image uploads)
** Provide an action for user to click on this and open the Image & Doc slideout

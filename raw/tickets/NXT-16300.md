# NXT-16300 — INV - Create Orders - Confirm Screen - Merge Orders

## Description
As an inventory manager I want a Merge Order prompt when creating a new order so I am able to merge my new order with an existing order if they have the same Site, Vendor, Delivery Date and Delivery Location.

*Requirements:*

* On the Merge Orders popup ([NXT-16298|https://primeroedgenext.atlassian.net/browse/NXT-16298]), provide a Merge button
* Assuming orders can be merged, on click of Merge button
** Merge the order into the existing order (shown on the right)
** Close the popup
** Change status to ‘Merged’ in DB
* Update the UI on the Confirm Screen
** Show a custom status label for any merged orders: ‘Merged into Order #12345’
*** If not enough space, show only ‘Merged’ as the status, and provide a tooltip displaying full details “Merged into Order #12345”
** Remove the Merge icon for the order that were merged
** The View icon will navigate to the newly merged order’s Order Detail page

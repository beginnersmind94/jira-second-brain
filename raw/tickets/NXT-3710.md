# NXT-3710 — Item Management - Recipe - Edit Contributions

## Description
*User Story*

As a Item Management User, I want to be able to edit the contributions for a recipe

*Mockup*

!image-20200929-025139.png|width=100%!



*Requirements*

* When clicking the pencil for contributions to edit, bring up the edit screen from the drawer on the right
* Serving size and measurements will be plain text (not editable)
* These contributions will have auto calculated on save of the recipe. This information can be manually edited *ONLY if the serving size and number of servings is present*
* The fields should show all contributions that are brought over from the Items
* User can override the fields and click the ‘SAVE’ button
* Information is then saved into the database
* If the items are updated, but the user has manually changed information here, we will NOT update the information

## Acceptance Criteria
User will be able to edit and save contributions

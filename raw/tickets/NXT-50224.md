# NXT-50224 — LMA - Subscription - Users & Roles

## Description
As a SchoolCafe user, I would like the system to automatically subscribe me to District License Management event subscriptions so that I will be notified when my subscriptions expire.

*Notes:*

* You should do the subscription stories for License Keys and District Licenses first

*Flow:*

* User creates a new account or edits an existing one
* User selects a role that has access to License Management screen
* System automatically subscribes them to the District License Management screen
* Optionally, if a user is added to a role in R&P, it does the same
* Optionally, if a role is granted permission to the screen, it does the same for the users

*Requirements:*

* On System > Users > Add or Edit, do the following:
* If a user is added to a role that has access to District License Management screen:
** Automatically subscribe them to the District License Management events
*** Subscribe to all of the events on the district license management screen
*** Type = Email and Inbox
*** Timing = End of Day
* 
* On System > Roles and Permissions > Role drawer, do the following:
** If a user is added to a role that has access to District License Management screen, do the same thing
** If a role is granted new access to the License Management screen, do the same thing

## Acceptance Criteria
* System automatically subscribes them to the District License Management events when they are added to a role that has it
* System automatically subscribes them to the District License Management events when a role they are in is given access to the screen

# NXT-3731 — Inspections Scoring - Scale Questions

## Description
*User Story*
As an Inspections user, I can create Scale questions with scoring, run inspections with those questions, and receive accurate score results based on the custom scores to evaluate my inspections.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=category&id=d7trj8|https://nqyc9v.axshare.com/start.html#p=category&id=d7trj8]

*Requirements*

* Each Scale question type can be set to apply a score when creating a Template
** Can only be done if Template's Scoring is set to Custom Scoring
* If a Template's Scoring is set to Custom Scoring, Scale questions have a toggle for applying the rating as a score
** By default, set to false
** Toggle can be edited when Template is edited
** Whether labels are included or not does not affect the toggle
** Addition to total Template score is determined by the highest value available on the scale
*** E.g. if the scale is from 1 to 5, 5 points are added to the total possible score
* When an Inspection is run, the points associated with the selection are applied toward the Inspection's final score
** Points are determined based on the score given in the rating
*** E.g. if the user selects a 4, 4 points are added to the Inspection score
** Score is properly updated if selection is changed
* If toggled to off, no scoring is applied from the particular question
* All scale scores are included in the final score when the Inspection is submitted

## Acceptance Criteria
* Score can be applied to all Scale type questions when creating a Custom Scoring Template, saved, and edited
* Score is properly applied as set in the Template when running and submitting an Inspection

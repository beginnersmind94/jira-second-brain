# NXT-69272 — Fin - Data Point History Tracking

## Description
As a finance administrator, I want to view the history of changes made to each data point so that I can audit who made changes and when.

## Acceptance Criteria
* History section displays upon clicking the “History” action button in the main grid and is a side drawer
* Section header shows "History" 
* History displays as a table with five columns: Performed By, Date, Performed Action, New Value, Previous Value
* Table shows most recent events at top
* Performed By column displays the user name who performed the action
* Date column displays timestamp in MM/DD/YY HH:MM:SS AM/PM format
* Performed Action column displays the action type (e.g., "GL Account Mapped", "GL Account Changed", "Data Point Created", "Status Changed", "Name Changed")
** Include before and after values in grid for consistency 
* -System-generated events show "System" as the Performed By value-
** Only actions performed by the user 
* User-initiated events show the user's name as the Performed By value
* New Value and Previous Value show the respective values
* Empty state shows "No history available" message
* Entries are made as soon as changes are made to the data points via the listed performed actions

# NXT-19187 — Acc. - Add. Income - Update reports to include Add. Income figures

## Description
*Description:* 

As a user I should be able to see additional income on the reports so that additional income can be tracked individually on the reports 

*Summary:*

Additional income should be shown on the following reports in accountability

*  Activity Report
* Cash Collection Report
*  Additional Income Report

*Requirements*

# When the user pulls a report from activity, additional income should be shown separately. Currently additional income is not shown on this report.

!report 1.png|width=652,height=439!

2. Cash Collection report should track the income incurred through “Additional Income” and the dollar amount for the selected period should be shown.

3. Add a new report on the reports page reading “Additional Income” with the following dropdowns
a.  *Select a Site(s)*

* Mandatory Filed
* Default to All

b. *Date as of*

* Mandatory Filed
* Default to Today
* Standard variable set for “Date as of”

c. *Report Type - Radio button* 

* Summary 
* Detailed 

The report should have the following details 

*Summary Report*:

# Site Code 
# Site
# Amount 
# Income Date 
# Entered By 
# Comments

!image (1).png|width=1207,height=717!

*Detailed Report*

Should mimic the below report

!Detailed Report.png|width=1065,height=835!

## Acceptance Criteria
AC1 : Activity Report should reflect additional income value for the selected period by the user 
AC2 : Cash Collection report should reflect additional income dollar amount for the selected period by the user
AC3 : A new report needs to be added “Additional Income”

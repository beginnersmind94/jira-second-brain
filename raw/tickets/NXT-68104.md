# NXT-68104 — IM - Bulk Item - Nutrients - Provide Missing checkbox

## Description
As an Item Management user I want the Bulk Update tool to have dedicated checkboxes for nutrients with missing values so I can differentiate between missing a value or pending an input.

## Acceptance Criteria
* Update the view mode of the Bulk Nutrients Review screen
** Condense column widths
** Use ? for missing values
** Use (M) for values with the Missing checkbox checked
* Provide a dedicated checkbox for Missing Value on the applicable nutrients
** 1st screen: Trans Fat, Cholesterol, Sodium
** 2nd screen: Fiber, Sugar, Added Sugar
** Vitamins: all values
* Adjust column sizing to fit all attributes
** reduce the width of the text fields as necessary (expected/common value: 123.45)
** on the Edit Nutrient screens, this may also include editable Serving Sizes ([https://primeroedgenext.atlassian.net/browse/NXT-67596|https://primeroedgenext.atlassian.net/browse/NXT-67596|smart-link] )
* Differentiate logic for Onboarding vs. Update
** Onboarding: automatically assign all null fields as Missing (when publishing)
** Bulk Update: do not automatically set any fields as Missing. Require user input

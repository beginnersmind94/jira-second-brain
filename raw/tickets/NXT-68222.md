# NXT-68222 — SC-PE-Insights-Multi Select UI-Add Apply and Clear buttons

## Description
Add Apply and Clear buttons to the Multi Select control.

When the user selects the required Site Type(s) or Schools and clicks the Apply button, the UI needs to load the data based on the selection.
Clicking the Clear button will unselect all the selections of Site Type and Schools, select All Schools, and load the UI.

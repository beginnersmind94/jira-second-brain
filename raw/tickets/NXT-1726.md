# NXT-1726 — INV - Orders - Order Detail - Receipt Tab - Close Order 

## Description
As an ordering manager I want a Close Order button on the order details so I can move an order into Closed status once I have received it.

*Requirements*

* Provide a Close Order button in the action bar of the Order Detail page
* Close Order button will advanced the order status to ‘Closed’
* Once an order is Closed, it can no longer be edited and no additional receipts can be created for it.
* Adjust In Transit quantity accordingly
** Covered in [https://primeroedgenext.atlassian.net/browse/NXT-17474|https://primeroedgenext.atlassian.net/browse/NXT-17474|smart-link] and [https://primeroedgenext.atlassian.net/browse/NXT-17454|https://primeroedgenext.atlassian.net/browse/NXT-17454|smart-link] 
* Once the order has been closed:
** Update the +*Details*+ tab accordingly
*** Update the status and display it as ‘Closed'
*** Update the banner to ‘Order #12345 is Closed’
**** This should be a blue banner
*** Remove action buttons (Cancel Order)
*** Do not show any Edit icons (Delivery Date, Delivery Location)
**** this includes the Edit icon in the Items list grid
*** Change Comments popup to view-only mode
**** this includes the Comments in the Items list 
*** Remove Add Item button from header
*** Remove Save button from footer
*** Keep the Back button
** Update the +*Receipts*+ tab accordingly
*** Remove action buttons (Create New Receipt)
*** Change Comments popup to view-only mode
**** this includes the Comments in the Items list 
** Put the +*Receipt Detail*+ screen into view-only mode
*** Remove/hide the Edit button
*** Change Comments popup to view-only mode
**** this includes the Comments in the Items list

# NXT-40322 — eOrders – View eOrder Orders

## Description
As a SchoolCafé user, I would like to view the orders in an eOrder file I generated so that I can verify my data.

*Notes:*

* This is a continuation of the eOrders – Dashboard story

*Flow:*

* Open SchoolCafé and browse to Inventory > Orders > eOrders
* View list of eOrders
* User clicks View Orders button and the View Orders drawer opens for them
* They can see the orders and optionally click on one to see it

*Requirements*:

* These requirements are for the eOrders Dashboard page
* 
* In the grid there are rows with an action column
* If not already there, add the View Orders button to the action column
* On click, open the Orders drawer
* 
* Create a drawer to be linked to the View Orders button
** Title = Orders
* 
* The drawer has a grid of orders linked to the eOrder
** Note: Orders are linked in the Generate eOrder screen
* 
* Grid columns:
* Order #
* Site
* Delivery Date
* Total
** Dollar amount of the order
* Action(s)
** View button
*** On click, open a new window in the browser and load the Order details page with the order clicked on
*** Page: [https://uat.perseusedge.com/inventory/Order-Details|https://uat.perseusedge.com/inventory/Order-Details|smart-link] 
* 
* Grid has paging
* Grid has sorting
* Grid has filtering
* 
* Security - No additional requirements

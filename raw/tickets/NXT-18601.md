# NXT-18601 — Production-Reports-Production Record (UI and Report)

## Description
This report will give the districts a way to print their production record. Many district managers would like to be able to print multiple days of production records at the same time. 

Page Criteria for Report: 
Site and Site Code ( Smart Search needed)

Site Group
Meal Type 
From and To (Date) ( allow 60 day range max )

Report should display very similarly in format to the production record page with the following criteria
Display on report
Site code with Site Name
Serving Date
Meal Service
Menu Line
Preparation Site (Site under the Prep site drop down, do not hard code to current site, but code to whatever site is selected in drop down)
Offer VS Serve (True or False)
Meal Pattern
A text box showing:
Serving Group
Planned Count 
Actual Count

There also needs to be a display of any plan or record comments that users have added on UI side.
Columns to display information are:
Locally Grown - dummy column
Menu item (Recipe #) and a small size font underneath for serving size with (portioning tool)
Serving Group
Adult - dummy column
Total Planned
Servings Produced
Total Served
Adult/ A la Carte - dummy column
Reimbursable - dummy column
Leftover
Comments

Items also need to display in same order of category as on the record.

# NXT-3014 — View Inspections - Actions - Approve

## Description
*User Story*
As an Inspections Supervisor or Admin viewing a submitted Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to approve the Inspection.

*Requirements*

* Approve is an action that can be applied to an Inspection
** This action is present on any Inspection with a Pending Approval status
** This action is present only to users with Approve permissions in both their role and the template the Inspection was created from
* When clicked, the user receives a popup
** Header: Approve Inspection
** Information
*** Inspection Name
*** Site
*** Inspector
*** Score
** There is a field for notes from the approver
** There are two buttons:
*** Approve - Approves the Inspection
*** Cancel - Closes popup
* Once approved, the Inspection is sent to the History section and the status is set to Approved
** Inspection no longer appears in the View Inspections screen

## Acceptance Criteria
* User with proper permissions can approve any submitted Inspection from the View Inspections screen in either view (card or list)
* User with proper permissions can approve any submitted Inspection on the Inspection Details screen

# NXT-14656 — Accountability - Menu Grids - Migration for Classic Data

## Description
As a user I want my previous menus that I’ve configured within my Classic PrimeroEdge system to flow into SchoolCafé so that there is no disruption and I can automatically view and use within the Menu Grid configuration and POS.

This is a story to capture the need for data migration of menu grids.

## Acceptance Criteria
* Previous menu grids from PrimeroEdge should be available within Accountability > Settings > Menu Grids
* Previous menu grids from PrimeroEdge should be available within the POS

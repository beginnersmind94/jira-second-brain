# NXT-16889 — INV - Orders - Review Order Settings - Add New Rule - View Items + Edit Order Qty Limits (Step 5)

## Description
As an inventory operations manager I want a Items option on the Add New Order Review Rule process so I can potentially configure order reviews for orders containing certain items.

*Requirements*

* This is the fifth step of the Add New Rule process of the Review Order settings
** this step is optional
** the screen for this step will initially be a list of items that are already assigned to the rule
*** for the first time there will obviously be no items added to the rule
** the user will need to press an ‘Add’ button to open the new screen that 
* Provide an ‘Add’ button so users can add items to the list [https://primeroedgenext.atlassian.net/browse/NXT-16891|https://primeroedgenext.atlassian.net/browse/NXT-16891|smart-link] 
* Display items in an Item List on the main screen of this step
** this will display the list of all items currently added to this rule
** this is where the user can configure optional minimum and maximum order quantities
** provide the ‘Field Search’ button for this grid
** table columns:
*** Select column
**** this will have a checkbox for each item
**** these checkboxes will enable the ‘Remove’ button, to remove these items from the rule
*** Item Number
*** Item Description
*** Storage Category
*** Item Category
*** Minimum Order Quantity 
**** this is an optional text field in which the user can type a number for minimum order quantity 
**** validate that this is less than the maximum for this item, if a maximum exists
*** Maximum Order Quantity 
**** this is an optional text field in which the user can type a number for maximum order quantity 
**** validate that this is less than the minimum for this item, if a minimum exists
** Provide a Save button in this grid
*** this is only for the item list to save any minimum or maximum order quantity entries
*** disable the button until any changes have been made to the min/max entries 
** Provide a Remove button in this grid
*** disable the button until any items have been selected (checked)
*** this will remove any selected items from the rule
**** this means those items will again be present on the Add Items screen 
* Display a Cancel button in the bottom left corner, to cancel and return to the list page
** Prompt user: “Discard rule configuration and exit setup?” Yes/Cancel
* Display a Previous button in the bottom left corner, to return to the previous step
* Display a Next button in the bottom right corner to advance to the next step
* Display a Complete button in the bottom right corner, to skip the remaining steps and jump ahead to the confirmation step

* List all items which have been added to the rule
* Provide Edit button to enable optional Order Quantity Min/Max

* User can remove items by selecting checkboxes
* User can add items by clicking Add button (NXT-16891)

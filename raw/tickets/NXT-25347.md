# NXT-25347 — Allergen Import- Exception Log(UI)

## Description
Currently there is not Exception log related to the import of students / Allergens.

Similar to that of History log in student Imports, there must be exception log. In Exception log tab, the details related to student ID, Exception Message (like failed), Failed Allergens if any and Processed Date and time, Processed by must be reflected.

## Acceptance Criteria
Validate and verify that the Exception Log is present in the logs

Validate and verify that the Exception log contains date and time the file is processed

Validate and verify that the Exception log contain student ID, allergens and the file processed by

Validate and verify that the Exception log contains the same no of students from file (include History if any)

# NXT-40085 — FTP – View Details

## Description
As a SchoolCafé user, I would like to be able to quickly view the details of an FTP execution so that I can verify its settings during debugging.

*Notes:*

* This page is read only content, no actions

*Flow:*

* Open SchoolCafé and browse to Settings > Integration > FTP Monitor
* View list of FTP executions
* Click View Details on a row
* Review the details in a drawer
* Return to the FTP Monitor page

*Requirements*:

* On FTP Monitor page, do the following:
* Add a button to the rows in the action column if it is not already there
** Title: View Details
** On click, Open the View Details drawer
* 
* Create a drawer that opens when click on View Details from the FTP Monitor page
** Title = FTP Details
* 
* On load, show the html version of the label and value pairs like we have done in Details drawers for other recent work
* Show all the fields from the FTP record and their values
* At a minimum, these fields should include:
** ID
** Name
** Description
** Source
** Host
** Port
** Folder
** Username
** Password
*** Mask this field
** FTP Type
** -ConfigID-
** -User- Distributed By
** -Date/Time Created-
** -Date/Time Executed- Distributed On
** Status
** Hyperlink to the file that was sent
* 
* Close button

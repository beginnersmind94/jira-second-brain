# NXT-5355 — Navigation - My Account (Profile) - Preferences

## Description
*User Story*
As a PrimeroEdge User, I would like to be able to quickly manage preferences and accessibility options from the any page using the Personalization header/horizontal navigation, so that I can change my preferences on the fly without having to go directly to My Profile.

*Mockup*
[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down|smart-link] 

*Requirements*

* On the horizontal navigation pane, add buttons to allow a user to update some quick preferences or go directly to the My Profile page.
** Preferences subheader/tab
*** Theme
**** Allow user to choose between Dark and Default
** Accessibility subheader/tab
*** Preferred Language
**** Allow user to choose between Dark and Default
*** -Font Size- ({color:#ff5630}Deferred{color})
** My Profile
*** Redirect user to My Profile page
* Functionality above is merely shortcuts to pages or features that have already been developed in My Profile.

## Acceptance Criteria
- Personalization section is present in horizontal navigation
- Functionality present as described in requirements
- Funcitonality is the same as has been implemented in My Profile

# NXT-70463 — MP - Menu - Menu List page filters 

## Description
As a menu planner, I would like for a way to be able to filter menus and have just the appropriate columns needed 

!image-20260310-163101.png|width=1020,alt="image-20260310-163101.png"!

## Acceptance Criteria
* Default Columns : Name (Rename to Menu Name), Site Group, Meal Type, Meal Pattern and action Column
** Remove Columns On Calendar, Creation Method, and Status
** Lock Menu Name and Action
** Created Date and Updated date need to be left aligned
* Provide filters for Status, Meal Type, Meal Pattern and Site Group
** Default to only show Active menus
** Show icon for inactive
* Remove the ability to go into edit mode for a menu 
** There will only be a view icon, users will go into a menu in view mode and will be able to edit from specific sections
* Action column will include the View icon and also an icon for versions
** Dock Action column
* Change the landing header to Menus in navigation and on the breadcrumbs

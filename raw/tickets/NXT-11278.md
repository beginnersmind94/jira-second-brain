*requirements*

## Acceptance Criteria
Application Processing has the updated look.
Action panel is floating and on the top right of the screen. Buttons are moved.

User has an option to refuse benefits on the application based on the system setting being set to yes.
- Refused benefit applications are marked as Denied due to Refused Benefits

User can see a count of any added comments.
User can view comments in a side-anchored window.
User can still add comments while processing.

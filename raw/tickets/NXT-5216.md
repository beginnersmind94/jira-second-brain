# NXT-5216 — Application Details - Verification

## Description
*User Story* 

As a district admin I want to see verification information about an application.

*Requirements* 

* This will be a new section/panel in the Application Details.
* Panel should show the following information:
** Milestone Dates
*** Eligibility + Basis
*** 1st Notice
*** Follow-up Notice
*** Change Date
**** Date the new eligibility took effect - this should keep adverse action days in mind.
*** Completion Date
*** Completion Notice
** Verification Results
*** Result
**** No Change, Free to Reduced, Free to Paid, Reduced to Free, Reduced to Paid, etc.
*** Reason for Change
**** Income, Directly Verified, etc.
**** Blank if Result = ‘No Change’
* Include a link to take the user over to the Verification Details page for that app.
** Also add a link to the Verification Details to take the user to Application Details.

## Acceptance Criteria
Verification panel is available for applications that are part of a verification sample.
Panel shows information as outlined in the story.

User has a direct link to Verification Details for that particular app (and back again would be good).

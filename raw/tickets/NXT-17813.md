# NXT-17813 — INV - Settings - Delivery Locations - Site Delivery Location Details - View/Edit

## Description
As a Food Service Director I want a Delivery Location Details page so I can view and possibly edit the current configuration for a particular school.

Continued from [https://primeroedgenext.atlassian.net/browse/NXT-16355|https://primeroedgenext.atlassian.net/browse/NXT-16355|smart-link] 

*Requirements*

* Open the details page when the user clicks the View icon for a particular site
** This is the same page from the New Config story ([NXT-17812|https://primeroedgenext.atlassian.net/browse/NXT-17812])
* Pre-select and disable the Site dropdown
** In the dropdown, show the Site as already selected
** Disable the dropdown
*** This essentially functions as a label for which site is being viewed/edited
* *The rest of the screen is the same as* [https://primeroedgenext.atlassian.net/browse/NXT-17812|https://primeroedgenext.atlassian.net/browse/NXT-17812|smart-link] 
** The Edit + Delete functionality is covered in this story
* In the Delivery Locations list, provide Edit and Delete icon in the Action column
** Edit icon
*** this will turn both fields into editable textboxes
**** Validate uniqueness when editing for both Code and Description
***** Display underneath the textbox
***** “Description must be unique.”
***** “Code must be unique.”
*** this will turn the Status column into a checkbox for Active/Inactive
**** user can uncheck a location to make it Inactive
**** prevent user from making a location Inactive if the location is used on any Incomplete or Open orders
***** “This delivery location is being used on some Incomplete or Open orders and cannot be deactivated at this time.”
*** this will provide the green checkmark for saving and red X for cancelling
** Delete icon
*** Hide the icon if the location has already been used on an order
* Provide a Save button, to officially save the new site configuration
* Provide a Back button, to return to the list page ([NXT-16355|https://primeroedgenext.atlassian.net/browse/NXT-16355])

# NXT-69523 — IM - Bulk Updates - Inventory Units & Sub-Qty

## Description
As an IM user I want the ability to bulk update unit descriptions and sub-quantities for items with various inventory statuses.

!INV Info - Units Sub-Qty-20260119-084209.png|width=1137,alt="INV Info - Units Sub-Qty-20260119-084209.png"!

## Acceptance Criteria
Business Rule Updates

* Consume all inventory statuses for items added to Bulk Update sessions
** including Synced and Locked
* Update the Units & Sub-Qty sub-step to lock down relevant selections based on the inventory status
** the radio option cannot be modified for Locked and Synced items 
** Sub-Qty cannot be modified for Locked items
** Descriptions can always be modified, except for dual units (must be edited on Servings)
* Consume alert for duplicate pack size descriptions
** require user to provide unique description
* Consume dual units on this screen
** Display as label-only in Base Unit column
** Show ‘Dual’ chip next to Dual Unit servings

UX Updates

* Use single Item column with *(LI-1234) Item Name* formatting used on other sections
* Add Inventory status column at end of grid

# NXT-51123 — POS - Dashboard - Cashier Impact Stats - Phase 2 - Discuss at ANC

## Description
As Cassie the Cashier I want to see the impact I’ve had on my local community as a cashier running a POS line.

*mockup:* [https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=6164%3A56305&node-id=6235-51754&viewport=1029%2C601%2C0.25&t=XHQRhl7YSMh0guzO-1&scaling=scale-down-width|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=6164%3A56305&node-id=6235-51754&viewport=1029%2C601%2C0.25&t=XHQRhl7YSMh0guzO-1&scaling=scale-down-width|smart-link] 

*requirements*

* Add the ‘Your Impact This Year' section.
** Totals for the user for the current academic year.
** Stats:
*** Day Service
**** Number of days the cashier has opened a session on for the current academic year.
*** Transactions
**** Number of transactions the cashier has completed for the current academic year.
*** Meals Served
**** Number of student meals the cashier has rung up for the current academic year.
**** Both Summary and Detailed for this count.
* We will be adding Site Settings for the control of these under [https://primeroedgenext.atlassian.net/browse/NXT-51124|https://primeroedgenext.atlassian.net/browse/NXT-51124|smart-link]

## Acceptance Criteria
* User can see their community impact.
** User can see how many days they’ve created a session for the current academic year.
** User can see how many transactions they’ve completed for the current academic year.
** User can see how many student meals they’ve rung up for the current academic year.
*** Both Summary and Detailed for this count

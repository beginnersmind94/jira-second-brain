# NXT-59849 — IM - All Weight/Volume Displays - Round and scale to usable units

## Description
As an Item Management user I want all my weight/volume data to round and scale to the highest usable units, so I can view my serving and nutrient data in simple whole numbers instead of 3 decimal places.

## Acceptance Criteria
* Establish a standard for scaling weights and volumes into higher units (e.g. 56 grams > “2 Ounce”)
* Establish a standard for stripping decimal values from numeric displays (e.g. 12.2 Grams of Fat > “12g Fat”)
* Convert scaled weight & volume values to the appropriate higher unit when thresholds are reached
* show two units if appropriate (e.g. 4 Pounds, 2 Ounces)
* Always retain user entered values for edit modes
** i.e. if user entered “grams” then use grams for edit

# NXT-19962 — Production- Production Status- Link to Production Plan/Record

## Description
As a user, I would like to be able to get to my production plan or record from the production status page.

When I am looking at my production status, and I would like to quickly go to a particular production plan/ record, there should be a link to where I can access the page in a new tab. This will give great usability to the user to access the information and then act upon that information with one click. 

 !image-2021-09-28-14-33-20-889.png|thumbnail!

## Acceptance Criteria
User must be able to access production plan/ record from the production status page. 

When clicking link, must allow for link to navigate to new tab and allow the user to still have access to the production status page and still look at the new production plan/ record they have opened.

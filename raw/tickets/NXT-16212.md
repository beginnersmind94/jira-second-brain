# NXT-16212 — Menu Planner - Delete Items

## Description
As a user, I need to be able to ‘Delete’ menus from the menu planner

Mockup

!image-20210702-022738.png|width=91.66666666666666%!

Requirements:

* Have an 'X' by each menu item
* When clicking the X, have a popup asking to delete the item
* Delete Item?
** Are you sure you want to delete ‘Menu Item’?
** Yes or No
* If yes, delete the menu item
* If no, return with no changes

## Acceptance Criteria
The user will be able to delete the item if needed

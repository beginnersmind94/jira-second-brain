# NXT-53314 — IM - Find & Replace - Resources & Permissions

## Description
As a District Admin I want permissions for Find & Replace so I can control who can make bulk recipe ingredient updates and bulk menu item updates.

## Acceptance Criteria
* Define resource and permission structure for the Find & Replace page
* Use the following permission structure:
** +Find & Replace+ - Page
*** +Ingredients (Find)+ - Tab
**** +Replace Ingredients+ - Action
**** +Remove Ingredients+ - Action
*** +Menu Items (Find)+ - Tab
**** +Replace Menu Items+ - Action
**** +Remove Menu Item+ - Action

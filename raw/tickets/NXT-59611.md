## Acceptance Criteria
* Add the truck icon to the new Shipping Groups column of the Contracts tab ([59073|https://primeroedgenext.atlassian.net/browse/NXT-59073])
* Clicking the truck icon opens the Shipping Groups slideout
** model this similar to Withdrawal Reasons
* Have a system default Shipping Group titled ‘Default’
** all items will by default be associated with this group
* Allow new Shipping Groups to be added with unique descriptions
* Allow group descriptions to be edited
* Allow groups to be deleted if no items are associated with it
** tooltip when disabled: “Shipping Group cannot be deleted if items are assigned to it.”

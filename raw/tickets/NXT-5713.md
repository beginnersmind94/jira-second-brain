# NXT-5713 — INV - Create Orders - Populate Item List (Step 2)

## Description
*User Story*
As an Inventory user I want my items to load based on the order selections so I can see only the items which I’ve selected.

This story is for the logic involved in populating the item list based on the selections made on Step 1 ([NXT-5713|https://primeroedgenext.atlassian.net/browse/NXT-5713]), once the *Apply* button is clicked.

*Requirements*

After the user has made order selections and clicked Apply, populate the item list based on the selection criteria.

Items will populate based on the order selections:

* Site - only load items which the selected site can order
** all items on any active vendor contract
* Order Date - order date will be used in combination with delivery date and lead time to load the item list
* Delivery Date - delivery date, if provided, will be used to populate items based on the item lead times
* Order Template - if one or more order templates are selected, load only the items associated with those templates
* Vendor - if one or more vendors are selected, load only the items which these vendors supply to the selected site

The item list populates according to *all* of the selections made. If the user selects an Order Template and they also select a Vendor, we would only show items that are *both* on the template and provided by the vendor.

If none of the optional selections are made, load all the items which the site can order.

+*Examples from Lalita:*+

* Items list: no selection for Vendor, Delivery Date or Template
** All items with active site item ordering condition
* Items list: when vendor(s), a delivery date and template(s) selected:
** All items with active site item ordering condition
** and from the active contracts of selected vendor(s)
** and part of the selected templates
** and the selected delivery date day as part of the vendor contract delivery schedule.  
** considering the lead time of item or vendor
* Items list: ALL vendors, a delivery date and one more order template(s) selected:
** All items with active site item ordering condition
** and from the active contracts of selected vendor(s)
** and part of the selected templates
** and the selected delivery date day as part of the vendor contract delivery schedule.  
** considering the lead time of item or vendor
* Items list: All vendors, No delivery date and All template selected:
** All items with active site item ordering condition
** from all active contracts of all vendors
** and in the all templates of site
** considering the lead time of item or vendor

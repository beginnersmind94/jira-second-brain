# NXT-16625 — INV - Withdrawals - Withdrawal Details - History tab - Transaction Details

## Description
As a Site Manager I want a transaction details page so I can view the specific details of prior versions of my Inventory Withdrawal.

*Requirement*

* After navigating from the History tab ([NXT-20800|https://primeroedgenext.atlassian.net/browse/NXT-20800]), load the historical transaction details page
** this will look just like the main Withdrawal Details screen
* The reports will still be provided, and will generate the data for the historical transaction (separate story)
* Display a message to indicate that the user is viewing a historical transaction
** “This is a historical transaction. View the current transaction for accurate reporting.”
* Show the header info
* Show the item grid
* Display the changed values in red text
** For any fields which were changed, display in red text
** Compare this transaction with the immediately prior transaction
** Attributes which could change:
*** Header
**** Reason
**** Comments
***** show a tooltip with previous comments
*** Item grid
**** Quantity
**** Comments
***** show a tooltip with previous comments
***** change icon color ?
* Display a ‘Back to History' button

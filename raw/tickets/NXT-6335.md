# NXT-6335 — Workspace - Customization

## Description
*User Story*

As a PrimeroEdge user, I have the ability to customize the layout and widgets in my workspace.

*Requirements*

* Widgets can be added and removed
** There is an “x” on each widget that hides it
** Widgets in the column beneath it are moved up to fill the space
* Widgets can be moved around using drag and drop
* Workspace layout can be changed based on provided layouts
* Widget colors can be changed
** There is a set of colors provided
* Any changes are immediately saved

## Acceptance Criteria
* Workspace can be updated as described in the requirements
* Changes are successfully saved and persist between logins

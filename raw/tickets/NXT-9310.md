# NXT-9310 — Verification - Tracking - Deselected Applications

## Description
*User Story*

As a district admin I want to see the applications which were initially part of the verification sample but have been removed for whatever reason.

*Requirements*

Add a 4th tab to the Tracking page.

* Fourth tab: Deselected Applications
** User can select by academic year, default is current academic year.
** Grid columns, typical search and filter options:
*** Application #
*** Eligibility
*** Basis
*** Type
**** This will be either: Sampled or Cause
*** Confirmation
**** This is a placeholder for future functionality.
*** Selection Date
**** Date the application was added to a sample.
*** 1st Notice
**** Date/time for when the 1st Notice was successfully created.
*** Follow Up Notice
**** Date/time for when the 2nd Notice was successfully created.
*** Completion Notice
**** Date/time for when the Completion Notice was successfully created.
*** Inactive
**** This is an indicator for whether the application has any students who are actively gaining benefits from it.
***** If no students are still gaining benefits from this application, then it is considered inactive.
*** Replaced
**** This is an indicator for whether the application was replaced with another application. If it was, list the application #.
*** Additional Columns (should be hidden by default, but can be made visible)
**** Student Names
***** single cell containing the name(s) of all students on the application
**** Applicant
***** applicant first + last name
**** Contact Information
***** contact information provided by the applicant
***** street address, phone numbers, email
* Selection boxes and the Notify button will not be required on the grid.

## Acceptance Criteria
Deselected Applications tab added.
Grid shows information by academic year, current AY by default.
Grid shows applications that are currently incomplete for verification.
Grid shows information for selected applications per the requirements.
Actions aside from History are only available for current academic year.

# NXT-8150 — Orders - Order Review

## Description
As a user I need to be able to review orders placed by students, parents, and teachers and accept, modify, or reject the order.

*Requirements:*

* The user will be able to come onto the review page and see a list of all the orders placed.
* When analyzed is selected the orders will be reviewed by the software to determine if there are any issues with fulfilling the order, Charge Limits, Restricted Items
* The user can select an Order, by clicking the order # and remove individual items from the order.
* The user can Accept, or Cancel and order or leave it in the previous state.
* The user can Accept all orders or accept them individually.
* The user can filter what orders they want to see listed, All, Accepted, Not Accepted, Reviewed.
* The list of orders can be exported
* Individual Orders can be Bundled (Grouped), Bundled orders are identified by a Bundle # and are grouped together.

## Acceptance Criteria
From the Left Panel, Under Point of Service, below Deposits. there is a function Orders. When Orders is Expanded there is an Order Configuration, Order Review, Order Preparation, and Order Fulfillment.

This page is launched by selecting Order Review

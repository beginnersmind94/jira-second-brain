# NXT-2827 — Inspections Dashboard - Dashboard Widget Functionality

## Description
*User Story*
As an Inspections user, I have dashboard widgets that can be rearranged and added/removed.

*Mockup*
[https://m5uloa.axshare.com/#id=0rb254&p=inspection_dashboard&view=default&g=1|https://m5uloa.axshare.com/#id=0rb254&p=inspection_dashboard&view=default&g=1]

*Requirements*

* Dashboard information is presented in widgets
* Widgets can be dragged and dropped to rearrange
* Widgets can be resized
* Widgets can be added and removed
** There is a menu where users can select the widget to be added
** Removal is done through a menu on the widget

## Acceptance Criteria
* Dashboard widgets present with ability to move around and add/remove

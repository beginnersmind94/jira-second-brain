# NXT-18682 — IM - Items - Product Info - Remove Broken Units - Split View - Edit mode

## Description
As an Item Management user I want the Broken Unit fields removed from Item details screen so I can view only the info for this specific item.

*Requirements*

* Make changes to the Item screens in Item Management to remove the concept of Broken Units
** This will only occur in Item Management
** Inventory references will not be updated until later
* Remove ‘Broken Unit GTIN’ from the main item info section (top of the page)
** *Note:* Item Management team will be doing this
* Remove the following attributes from the Product Info section
** Purchase / Whole Unit (dropdown)
** Broken Units per Whole
** Broken Unit Size
** Orderable in Broken Units
** Orderable in Broken Units from Internal Vendors
* Rename the following attributes
** Broken Unit Description
*** *Unit Description*
**** this will be a free text field
** Broken Unit Weight
*** *Unit Weight*
**** this should apply to the weight of the specific item being viewed
* List the Product Info attributes in the following order (left to right, then new row, repeat)
** Item Category
** Storage Category
** Valuation Group
** Unit Description
** Unit Weight
** Fair Market Value / Standard Price
** Buy American

# NXT-50381 — INV - E-Orders - Test E-Order file using Email option

## Description
As an Inventory District Administrator I want the option to submit E-Orders via Email service so I can send orders to vendors who do not support FTP/EDI configuration.

** *Subject*: SchoolCafé electronic order #12345 from Willis County Schools
** *Body*: Attached is the file containing E-Order #12345 from Willis County Schools, generated on 5/7/2024 at 12:23 PM

* Test with non Cybersoft emails

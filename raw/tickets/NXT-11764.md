# NXT-11764 — Eligibility - Application Details - Loading Animation

## Description
As a product owner I want our product to show a loading animation when there would otherwise be a delay in populating data on the screen.

*requirements*

* User should be shown a loading screen when entering into Application Details, with no sections appearing until the page is fully ready for review.
* Replace the bar loader present on pages (Configuration pages in particular) with the circular loading animation.

## Acceptance Criteria
User should be shown a loading screen when entering into Application Details, with no sections appearing until the page is fully ready for review.
Existing bar loading animations replaced with spinning animations.

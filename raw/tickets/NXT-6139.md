## Description

## Acceptance Criteria
User has a By Direct Certification option on the Eligibility reports page.

User can select a date to see information as of that particular date. Default is current date.

User can see a count of students approved by direct certification as of the selected date.
Count is broken down by eligibility (free/reduced), including household extensions, and by type (SNAP, TANF, Homeless, etc.) 

User can toggle whether they are seeing counts or percentages.

User can export the information on the page out to Excel or PDF.

# NXT-3708 — Add New - Import - Template and Upload Step 3 - Confirmation

## Description
*User Story*

As a Item Management User, I need to be able to create a new item via import and want to validate and publish it

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=import_item&id=o4qto3|https://nqyc9v.axshare.com/start.html#p=import_item&id=o4qto3]

*Requirements*

* After selecting the file and clicking ‘Next’ from Step 2
** Show that the file has been validated
*** “Data Validated successfully”
** Allow the user to download the validated file that was edited in the UI
*** Downloads the validated file
** Click ‘Back’ to go back to Step 2
** Click ‘Publish' to finish the import and return the user to previous screen
** Display the list of items in a notification at the top showing all the items imported

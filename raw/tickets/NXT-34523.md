# NXT-34523 — Import and Export Monitor - Update Column Headers

## Description
Update column headers on the Import Monitor page

It will need a lot of effort coding-wise to implement the first acceptance criteria.
The second best approach, we will try is to possibly reduce any white spaces for any grid column and try to increase the width of any of the grid columns where the text is getting truncated.
We will see grid column data getting truncated when we view the page in different screen sizes.

## Acceptance Criteria
* -Make column headers resizable and sticky (browser storage).-
** -Sticky = If user resizes the columns, persist the column sizes in browser storage and re-apply those column sizes the next time the user visits the page.  This will allow the user to not have to resize the columns every time they use Import/Export Monitor.-
* Change “Duration (H:mm:ss)” column header to correct time format

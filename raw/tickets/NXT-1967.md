# NXT-1967 — (Overall Requirement story) Menu Planning - Reports - Menu Calendar

## Description
This report has a dependency on the assign menus functionality.

This is a much used report, and part of USDA requirement.

Multiple variations of this report exist (week, month, day), with & without nutrient details. Make sure we have other user stories OR we will split this story into those different needs.

Explore better customization for this report. Ability to add graphics, logos, watermarks, etc. “Refrigerator-Friendly” ideas should be explored.

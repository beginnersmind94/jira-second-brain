# NXT-69849 — Menu Analysis - Site Performance - Produced Accuracy Metric & Slide out

## Description
As a district manager taking a look at my site performance on workspace, I would like to be able to see how well my managers Produced vs what was actually served.

!image-20260130-222915.png|width=1019,alt="image-20260130-222915.png"!

## Acceptance Criteria
* On Site Performance page, Rename Production Accuracy to Produced Accuracy 
** This will be after the Planned Accuracy Column
* The calculation for Produced Accuracy is (Served/Produced)*100= Produced Accuracy
* Provide an icon to allow user to open a slideout from the column to show calculations
* When user clicks on a particular row on the slideout, update the calculation on the bottom of the slide out to reflect the numbers and show math “work”
* Allow sorting on the slideout
* Freeze Calculation at the bottom and formula at the top, also freeze header row and allow scrolling for the sites

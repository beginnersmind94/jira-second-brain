# NXT-25243 — INV - Orders - Production Ordering - View Items - One Order Per Vendor

## Description
As an Inventory Manager I want a ‘One Order per Vendor’ option so I can consolidate vendor order quantities into a single delivery date.

*No API required

*Requirements*

* Provide a ‘One Order per Vendor’ checkbox next to the Vendor selection and the View Items button
** if checked, on View Items click, reload the items consolidated into first available delivery date

# NXT-3830 — Account Management - Advanced Search - Demographics Section - Birthday Subsection

## Description
The Birthday Subsection allows the usr to select a single date, or date range for list patrons with birthdays within the range. The typical selection will be either a month, or month and day

!Screen Shot 2020-04-29 at 3.00.49 PM.png|width=284,height=83!

## Acceptance Criteria
The user can select a single month

The user can select a single month and day

The user can select a From month and To month

The user can select a From month/day and a To month/day

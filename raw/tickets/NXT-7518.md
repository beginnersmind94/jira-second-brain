# NXT-7518 — Application Processing - (Action) Delete

## Description
*User Story* 

As an admin, I have the ability to delete an application.

*Mock Ups* 

*Requirements* 

* Delete action
** Popup message > _Please select a reason and enter a comment to delete this application._
*** Reasons: 
**** All students are directly certified
**** Students approved on previous application
**** All students are at CEP sites
**** Duplicate application
**** Other (please specify in comments)
*** Comments box
*** Cancel > pick up where they left off
*** Delete > Application Status - Deleted 

* After deleting the application, “Restore” button is available in the application details. 
* Status of the application is “Deleted”

## Acceptance Criteria
ADA, Multi-language
delete confirmation popup appears on click

cancel closes the popup and no info is lost

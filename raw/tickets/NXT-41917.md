# NXT-41917 — DataExchange Import – Fixed Width – Pre-Upload Update

## Description
As a SchoolCafé Admin I want to import a fixed width file so that I can use my state’s existing format.

*Flow:*

* User logs in and browses to System > Imports & Exports > Imports
* User clicks + New Import button
* User selects DC Import on Pre-Upload screen
* User selects the Configuration Type they want to use
* Use selects Fixed Width as a file type
* User clicks the Next button and continues process

*Notes:*

* The following tabs will have additional logic to handle fixed width file 

*Requirements*:

* For the Pre-Upload tab when click + New or click the edit icon, do the following for all import types:
* 
* Add another select box below the existing fields
** 
** Field Name: File Format
** Values: Delimited, Fixed Width
** Auto Select Delimited
** Field is required if visible
* 
* Note: the file format value will drive functionality on the next 2 tabs, so be sure to save the value so that it can be used

## Acceptance Criteria
* File Format shows below Configuration Type
* Delimited is auto selected by default and is required

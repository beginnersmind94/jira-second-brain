# NXT-8270 — INV - Physical Inventory - UI changes

## Description
As an inventory user I want the buttons at the top of the page (without a bounding box) so the Physical Incomplete screen looks similar to the other screens.

*Requirements*

+Main Screen (Physical Incomplete)+

* Move the buttons to the top of the screen
** on Summary tab
*** Save as Complete
*** Save as Incomplete
*** Print Physical Count
*** Cancel Inventory 
** on Items tab
*** Save as Incomplete
*** Print Physical Count
*** **Remove Cancel button
* Left align the buttons, on both tabs
* Add Site Name to header (as it is on Review screen)
* Add Inventory period/date to breadcrumb path
** Example: Home > Inventory > Physical Inventory > Site Details - Physical Incomplete - December 2020
*** use Start Date for Interim inventories

+Review Screen (Pending Reconciliation, Cancelled, Reconciled)+

* Add Inventory period/date to breadcrumb path
** Example: Home > Inventory > Physical Inventory > Site Details - Pending Reconciliation - Interim (12/18/2020)
*** use Start Date for Interim inventories
* Hide the action buttons on Items tab (Modify, Reconcile, Cancel)
* Remove additional tabs

Change Status name for ‘Physical Complete & Pending Reconciliation’ to ‘Pending Reconciliation’

# NXT-66388 — Inventory Fees (warehouse-related costs)

## Description
As a *district inventory manager*,
I want warehouse fees (delivery, processing) to create ledger entries,
So that costs are correctly captured as expenses.

## Acceptance Criteria
* Preconditions: Same as usage.
* Posting date = last day of inventory period.
* *Ledger entries*:
** _Warehouse Receives Commodity Fees (GLEntryTypeCd: None, Multisite)_ → Debit Site Expense, Credit Warehouse Expense.
* Rollbacks reverse entries.
* Entries flow into food cost/operational expense reporting.

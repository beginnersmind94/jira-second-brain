# NXT-19248 — Accountability - Reports - Advanced Cash Collection Report view/options

## Description
Add the following to the Cash Collection report, which is left over from NXT-10909:

# A ‘Summary’ view option. Currently, we are displaying the Detailed view only.
# Report options: ‘Exclude Non-Payment Days', ‘Page Break by Site’, or ‘Include Satellite Schools’.
## *Exclude Non-Payment Days*: Only show those rows where there is actual data
## *Page Break by Site*: Each site represented in the report should be on a separate page
# The header showing report should show report generation time/datestamp info (Generated By X user on X date)
# Adjust the ‘Year to Date’ option in ‘Date as of’. Currently it is showing data from exactly one year ago (i.e. 9/14/2020 - 9/14/2021), but the same option elsewhere shows it as of the beginning of the current academic year. So in this case it should show data from 7/1/2021 - 9/14/2021 for consistency.

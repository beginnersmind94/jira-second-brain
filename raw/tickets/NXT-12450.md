# NXT-12450 — IM - POS User Permissions

## Description
As a POS user, I can only add an item with the general item information, Allergen, and POS information. I cannot add an item with the other values.

I can view the other values if they are there, but I need to view the POS information first and foremost.

## Acceptance Criteria
POS user must only be able to add / edit POS, Allergens, and general item information. Other information should be read only.

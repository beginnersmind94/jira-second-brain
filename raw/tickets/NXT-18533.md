# NXT-18533 — DataQuery Framework - Interactive filters for each datatype

## Description
Filters for datatype should be interactive based on columns selected:

* In Search box, default value or recent value should be populated, plus user can lookup for it.
* In dropdown, user can multiselect the values plus user can search it and filter data while typing the same.
* In date, need to have options like Today, Tomorrow, Yesterday, This week, Past week, This Month, Last month, This Year, Last year and Custom dates

## Acceptance Criteria
Interactive filters for each datatype should be made available to end users:

* In Search box, default value or recent value should be populated, plus user can lookup for it.
* In dropdown, user can multiselect the values plus user can search it and filter data while typing the same.
* In date, need to have options like Today, Tomorrow, Yesterday, This week, Past week, This Month, Last month, This Year, Last year and Custom dates

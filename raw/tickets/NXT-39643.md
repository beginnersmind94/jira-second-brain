# NXT-39643 — IM - MI Consolidation - Variety Menu Item - Add Menu Items

## Description
As a Menu Planner I want the ability to create link menu items to my Variety Menu Item so I can leverage the flexibility for menu planning and production.

## Acceptance Criteria
* Provide a search for menu items which can be added to the VMI
* Return only Menu Item results which have a Serving Size with Contributions that are equivalent or greater than the VMI Contributions
* If a Menu Item has multiple serving sizes (each with Contributions), select the serving which has the *minimum, closest value* to the VMI Contributions
** Ex: if VMI is 1oz, but Menu Item has 1oz and 2oz, auto select the 1oz
* Only allow menu items with minimum mandatory contributions be added to the VMI 
* Display the added menu items in a list format with images, components, etc. all visible
* Provide links to open the menu items in view mode

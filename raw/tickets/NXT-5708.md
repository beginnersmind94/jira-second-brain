# NXT-5708 — INV - Update Vendor General Info

## Description
*User Story*
As a Procurement Specialist, I want an edit option for vendor general info so I can make updates to my saved vendor attributes.

*Requirements*

In the view vendor info screen, user can click the edit icon on the top right corner of each card to open that particular section for editing.

When the edit icon for a section is clicked, it will open that section for editing.

User will be able to edit that section alone and save it.

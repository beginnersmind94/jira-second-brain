# NXT-32086 — Embed Google Analytics Widget

## Description
As a SchoolCafe team member, I would like the ability to see user and application usage statistics via Google Analytics.

## Acceptance Criteria
* As a SchoolCafe team member, I would like to have a single Google Analytics widget per application environment so that I can observe usage for the entire SchoolCafe application.
* As a SchoolCafe Admin, I would like to have a single internal setting that allows me to specify the Google Analytics Tracking ID.
** Setting Name: *Google Analytics Tracking Code*
** Default value: Null
** Code: GLEANLYTCO
** Category: Configuration
** Module: System
** Type: Internal
* The code snippet for the widget shall be:
** <\!-- Global site tag (gtag.js) - Google Analytics -->
<script async src="[+https://www.googletagmanager.com/gtag/js?id=G-CDBHT74024"></script+|https://www.googletagmanager.com/gtag/js?id=G-CDBHT74024%22%3E%3C/script]>
<script>
  window.dataLayer = window.dataLayer || \[];
  function gtag()\{dataLayer.push(arguments);}
  gtag('js', new Date());
 
  gtag('config', 'G-CDBHT74024');
</script>
* The value of the *Google Analytics Tracking Code* setting shall be dynamically assigned/applied in the above script at runtime and not hardcoded.

# NXT-14088 — MP - Menus - Menu Analysis - Use the Menu Item Serving Size

## Description
As a Menu Planning user, I want to make sure the nutrients are calculating to use the correct serving size specified for each serving group. 

When going to Menus -> Menu Analysis, the nutrients need to calculate based off the menu item serving size for the age group, not just for the default recipe serving size.

This is due to multiple serving sizes being available in Menu Planning.

# NXT-17457 — INV - Orders - Merge Orders - Enhanced Merge Orders Popup

## Description
As an ordering manager I want an enhanced Merge Orders popup so I can see all eligible mergeable orders instead of only the one that is most recent.

*Requirements*

* Modify the Merge Orders popup to replace the left-side of the Merge Orders popup with a new list of potentially mergeable orders
* Modify the message at the top of the banner
** “The following orders have the same criteria as the selected order. To merge this order into any of the eligible orders, select the Order # to preview and use the Merge button.”
* On the left side of the popup, display a list of possible orders into which the current order can be merged
** display this as a small table of Orders Numbers which can be clicked
** this will be a single column table, with the Order # as the only column
* As Order Numbers are selected, load their information into the right side of the popup 
** on the right side panel, add ‘Total Order Amount’ as a label
* Each time an order is selected, check the contract validations against the potentially newly merged order
** if the merged order would result in a contract validation issue, disable the Merge button
** show the message details in the bottom of the popup (details in [NXT-16298|https://primeroedgenext.atlassian.net/browse/NXT-16298])
* Use updated logic for finding orders to merge into
** Orders can be merged if they have the exact same value for the following attributes:
*** *Site*
*** *Vendor*
*** *Status*
*** *Delivery Date*
*** *Delivery Location*
** Identify all possible orders for merging, not just the most recent one

## Acceptance Criteria
* Modify the Merge Orders logic to identify all eligible orders for merging
* Modify the Merge Orders popup to list on one side all the eligible mergeable orders
* Allow user to select orders and preview the merge
* Disable the Merge button if any newly merged orders would result in a contract validation issue (Max Purchase Amount, etc.)
* Show any relevant messages in the bottom of the popup, regarding the contract validations
* Add ‘Total Order Amount’ to the right side panel

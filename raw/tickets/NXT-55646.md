# NXT-55646 — Imports – Connecticut Language Codes Auto Mapping

## Description
As a SchoolCafé admin, I would like to subscribe to notifications of imports so that I can be informed if there is an issue with the import.

*Note:*

* We need to update our language codes to include the Connecticut PowerSchool language codes
* This story is for API and File imports both so that we maintain consistency



*Requirements*:

* On the System > Imports & Exports > API tab or Files tab, do the following:
* Do for both Files and PowerSchool API Student Imports
** Not necessary for Aeries or Skyward
* Do for both Add New and Edit screens
* On the Languages Code Mapping tab, do the following:
** Add the code mapping in the excel doc to the default code mapping

## Acceptance Criteria
* Codes are used by the Language code mapping screen and mapped if present
* Existing codes are not affected

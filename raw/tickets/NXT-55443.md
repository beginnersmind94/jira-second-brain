# NXT-55443 — MP- Menu- Final check on save for Record Complete Menus

## Description
As a Menu Planner, I would like for the system to check when I am saving a menu and not allow changes to that I am making to a menu if the menu is now associated with a record complete production record.

## Acceptance Criteria
* Do Final Check on *Save* to check if menu has a record complete record associated 
** This is for any changes to serving groups, serving sizes, or category counts
* Do a final check on Add Menu item save to check if Menu has a record complete record associated
* For both scenarios, provide a banner error message “Menu could not be saved! One or more Production Records have been completed associated with this menu.”

# NXT-37096 — PR- Calendar- Allow user to be able to pick another month or year through a month/year picker (UI ONLY)

## Description
As a Production Manager, I would like the ability to go to another month on the calendar that may not be specifically the following month or the previous month. Currently, I am only allowed to browse one month at a time in consecutive order.

## Acceptance Criteria
* Allow the Month/Year Picker of the component only
* User can click on the current month display on the calendar header to access the picker component
** Display tool tip when user hovers over the display “Click to pick a different month/year”
* The component has a Today picker (Can we hide it?) if not then use that and had the current today picker

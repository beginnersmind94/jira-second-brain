# NXT-50535 — MP- Edit Menu Planner- Adding Menu item on particular serving group

## Description
As a menu planner, when I am adding a menu item on a particular serving group, it should apply to all serving groups available on the menu or menu cycle

## Acceptance Criteria
* Add a menu item to a menu with multiple serving groups will add to all serving groups on the menu
* Default serving size should always be applied  to all serving groups unless there are serving size exceptions that are available for serving groups available

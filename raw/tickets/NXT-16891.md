# NXT-16891 — INV - Orders - Review Order Settings - Add New Rule - Add Items (Part of Step 5)

## Description
As an inventory operations manager I want an Add Items option on the Items step of the Add New Order Review Rule process so I can select items to add to the rule.

*Requirements*

* This is part of fifth step - ‘Items’ - of the Add New Rule process of the Review Order settings
* Once the user has clicked the Add button on the Items step ([NXT-16889|https://primeroedgenext.atlassian.net/browse/NXT-16889]), display this new screen so users can search for items to add
* Provide a control to select items or item categories and add them to the order review rule
** this occurs when the user clicks the Add button on the main screen
** this Add Items screen should live under a different tab/view
* Display a header containing search filters for the items
** Storage Category dropdown
*** default to ‘All’
** Item Category dropdown
*** default to ‘All’
** Data Source dropdown
*** default to ‘All’
** Apply button, to load the item list according to the selection
* Display an Item List table
** this will load the items returned by the search criteria
** provide the ‘Field Search’ button for this table
** table columns:
*** Selection column
**** this will contain a checkbox for each item
*** Item Number
*** Item Description
*** Storage Category
*** Item Category
* Provide an ‘Add’ button
** this will add any selected items to the rule
** disable the button until at least one item is selected
** this will remove the items from the list
** the items selected here will be added to the Item List on the main screen of this step
* Provide a ‘Done’ button
** this will exit the Add Items screen and return to the main screen of the Items step ([NXT-16889|https://primeroedgenext.atlassian.net/browse/NXT-16889])

# NXT-26061 — Sync Data Exchange Platform Database

## Description
This story is cover the tasks related to sync of Data Exchange Platform Database.

Currently when an allergen is added to Item Configuration via Item Management, the allergen is not in sync with the Data Exchange Platform Database. We need to have the sync mechanism so that when ever an allergen is added, it is immediately reflected @ Code mapping in Data Exchange Platform Database mapping page.

## Acceptance Criteria
Verify and Validate that when an allergen is added, allergen is immediately synced and reflected in code mapping @ Data Exchange Platform Mapping.

Verify and Validate that when an allergen is added, it is only for Local Data Source.

Verify and Validate that when an allergen is added, the newly synced allergen is only to the corresponding School Dist.

# NXT-20662 — Production- Production Calendar Table- Handling Incomplete Status

## Description
As a Production Manager, when I am on my production adjustment page, I would like to be able to see my menu items that are in records that are in incomplete status. 

Currently, we are only looking at menu items that are “touched” as incomplete or in Plan Complete Status.

## Acceptance Criteria
As soon as a menu is added on MP for a particular menu line, this information should be available in the production Calendar Table

Always display menu items that are from plans that incomplete, no matter if they have been opened or not (Production Adjustment)

This should also reflect on the production status page to include any and all statuses regardless if they can be “touched” or not

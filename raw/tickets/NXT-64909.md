# NXT-64909 — WS - Filters - Site Groups - PE & SC

## Description
As a Workspace user, I want to be able to select sites more efficiently, so that I can be faster in filtering my data.

*Notes*

* This story is for the Sites filter on the main page
* Use Case: Users want to be able to select all schools of a certain type and then unselect specific ones efficiently.

*Requirements*

* In the drop down list for the sites, do the following:
** When select All Sites, or a Site Type:
*** Check the sites that it applies to (ex: if select Elementary, select all the elementary schools in the list)
** When select a site:
*** Unselect the All Sites and Site Type that it applies to
**** Ex: user selects All High Schools and All Middle Schools. Then user unselects one of the High Schools, so the All High Schools unselects

* When All Schools is selected in the All Schools dropdown:
** Check all the sites and site types.
*** If an elementary school in the school list is unchecked, then uncheck ‘Elementary Schools' in the Site Type list, and then do the same for 'Middle School' and 'High School’ when a middle or high school is unchecked.

## Acceptance Criteria
* System selects the sites in the menu when selecting all or a site type group.

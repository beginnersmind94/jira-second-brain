# NXT-15136 — Eligibility - Applications - Image for the Original Online Submission

## Description
As a district admin I want to see the originally submitted information on the application image, and updated images every time the app is processed. This will provide a visual audit trail for information changing over time.

*requirements*

* Add a toggle to the image pop-up to let the user flip between original online application data and the latest.
** Toggles: Latest Application / Original Submission
** Default: Latest Application
* Latest Application is what we’re showing right now.
* For Original Submission, show the data that was originally submitted via the parent portal.
* Hide the ‘Student?’ column in the Step 1 section.

## Acceptance Criteria
* User can pull up an image containing the original app data.
* User can still see the latest application data.
* ‘Application Image’ button should by default show the latest image.
** User can toggle between images and export appropriately.
* Editing an application should not update the original application image.

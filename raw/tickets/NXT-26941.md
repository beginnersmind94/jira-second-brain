# NXT-26941 — Custom Views - Enable Out-of-the-box View

## Description
The objective of this epic is to provide an out-of-the-box view for first-time users and for a user to be able to “Reset” their view to original out-of-the-box view.

## Acceptance Criteria
* Change the existing text “Set As Default” to “Set As Favorite”
* Change the pencil tooltip from “Edit” to “Rename View”
* A “Default” view shall be available out-of-the-box
** The “Default” view will have the name “Default”
** This new “Default” view will always be at the top of the view box
** This new “Default” view cannot be renamed
** This new “Default” view cannot be shared
** “Default” views will be those that are defined at implementation time and available out-of-the-box
* Modules shall be responsible for defining the “Default” view for each of their UIs and providing these to the Platform dev team for inclusion into deployment scripts for all customer installations.
* Rename the UI component “COLUMNS AND VIEWS” to “Manage Views”
* Change the tooltip for “COLUMNS AND VIEWS” from “ColumnFiler” to “Manage View”
* The ability for the user to easily see which view is currently being used shall be available.
* If an existing view is “dirtied”, provide some visual indication so that the user knows that there are changes pending to be saved.

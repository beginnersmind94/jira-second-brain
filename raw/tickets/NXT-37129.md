# NXT-37129 — PR- Reports- Production Record Report Single and Multiple adding CCP temperature 

## Description
As a production Manager, I would like to have my production record report to reflect the CCP process and give me a place to put the temperature.

## Acceptance Criteria
* Display CCP process in the current temperature 
** IF there happens to be more than 1 CCP, then display with enough room for the user to hand enter the temperature
* IF there are none, display N/A
* Display user inputted temperature if already entered on the UI
* If user has the enable setting off for temperatures, remove the temperature from being available on the report
* Currently, the non reimbursable column and reimbursable columns are displaying regardless of the system setting to turn it on or off, this needs to be take in consideration the setting and remove those columns if they are not on
* Only display the CCP that is available on the production record
* This will effect single records and multiple days

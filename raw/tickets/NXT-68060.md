# NXT-68060 — *GC- PE- Match Status counts needs to be corrected as GTIN search supports search with Product code and Vendor item number fields. 

## Description
_(none)_

## Acceptance Criteria
Match Status counts needs to be corrected as GTIN search supports search with Product code and Vendor item number fields. 



Example: Given GTIN search string: 00005035,10027000422011,10041500906239,10037323125421,00003,00014800582420,811408001182,000001266,23700047441,10269760928,7039014

!image-20251103-154725.png|width=1082,alt="image-20251103-154725.png"!

When Cybersoft catalog or USDA BF is selected: Display the matching count status as shown below. 

*Total Input Values: {Count of input values given in the comma separated string/uploaded file} |  Total Matched: {Count of input values with at least one item match} | No Match Found: {Total Input Values - Total Matched } |Total Item Count: {Total Number of Items displayed in the grid} (Matched on GTIN: {# of items matched on GTIN values from given input string/uploaded file}  | Matched on Product Code: {# of items matched on Product Codes from given input string/uploaded file} | Matched on VIN: {# of items matched on VIN from given input string/uploaded file})*  

Example :*Total Input Values: 11| Total Matched : 8| No match found: 3| Total Item Count: 15* (*Matched on GTIN: 4| Matched on Product Code: 8| Matched on VIN: 3)*

*(OR)*

*Total Input Values: 11| Total Matched : 8| No match found: 3| Total Item Count: 17*(*Matched on GTIN: 5| Matched on Product Code: 8| Matched on VIN: 4)*

!image-20251110-223637.png|width=1085,alt="image-20251110-223637.png"!

 When ALL data providers is selected: currently its showing *Total GTINs:8 | Total Matched:18 (Branded Foods:0 | CNDB:0 | Cybersoft:18 | No Match Found:0*

Remove the text “(Choose which Item(s) you would like to import)”

Display the matched item status  : 

*Total GTINs: {Count of input values given in the comma separated string/uploaded file} |  Total Matched: {Count of input values with at least one item match} (Branded Foods: XX | Cybersoft: XX)* *| No match found: {Count of values from Input string/uploaded file with no match} |Total Item Count: {Total Number of Items displayed in the grid} (Matched on GTIN: {# of items matched on GTIN values from given input string/uploaded file}  | Matched on Product Code: {# of items matched on Product Codes from given input string/uploaded file} | Matched on VIN: {# of items matched on VIN from given input string/uploaded file})*  

Example: *Total GTINs: 300 |  Matched items: 250(Branded Foods: 50| Cybersoft: 200) | No match found: 50 | Total Item Count : 300 (Matched on GTIN: 200  | Matched on Product Code: 75| Matched on VIN: 25)*

# NXT-44877 — Imports - Balance Import - Add Statistics Section

## Description
As a SchoolCafé Admin I want to quickly see the import statistic for Balance Import so that I can identify if I have any import issues.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor > Balance Import record
* Click on View Log button
* View statistics on top

*Requirements*:

* We want to add in the statistics section like in Adult Import. 
* Copy that code and do everything that it does.
* Use the same UI

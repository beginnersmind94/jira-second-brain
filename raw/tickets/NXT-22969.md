# NXT-22969 — Option to delete the image on item creation page  

## Description
As a user I an option to delete the image on item creation page which is not present right now

## Acceptance Criteria
* User should be able to add an image on Item creation page (up to 6 images similar to Recipe)
* User should be able to select a preferred image and that should be shown in Spilt View (small icon) and full image when an item is loaded in the screen
* User should be able to select one or more images and be able to delete them (similar to Recipe)
* The user should be able create an item without having an image (as-is, no change needed)
* This applies for all item creations (Ex: FOOD, NON-FOOD, POS and Menu  Items flags)
* Upload image should be visible and be  able to navigate thru the images
* Images should be visible in the split view
* Upload images format (similar as is currently in recipe)

# NXT-60310 — Digitize AI - Document upload success, parsing, summary, and details preview

## Description
As a user I want to see additional information for successful uploads/captures of documents within Digitize AI so that I can be sure that the information is correct and I can determine if I want to save or discard.

* Context is displayed, showing the form type, date, user, school, district, etc..
* Form information details displays the parsed data
* Action options are displayed to Discard or Save (we can be specific with each form type - ex. “Send to Production” is the save action for Production Record forms). This functionality is covered in another user story.

Example shown above is for the form type “Production Record”. Other forms may contain some different “context” (summary) values that are applicable to that form. These context and form information details will need to be specified per form type.

## Acceptance Criteria
* Display document context summary: Form type, date, user, school, etc
* Display details expansion of captured digital information

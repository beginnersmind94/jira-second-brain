# NXT-21457 — INV - New Physical Inventory - Provide Count Sheet when starting Inventory

## Description
As a site manager I want to be able to view or print a count sheet so I can prepare to count my items.

*Defect:* on New Physical Inventory screen, the Periods dropdown should not show periods which were previously complete.

!image-20220308-205051.png|width=91.66666666666666%!



----

Count Sheet can be generated in several ways.

* at the time the Physical Inventory is created
** provide the count sheet on the same screen that the inventory criteria is selected
** somehow maintain a link between the Physical Inventory and the Count Sheet so that the user can simply start the inventory based on the selections made for the count sheet
*** ex: user generates a count sheet for the Cooler (storage category) and Frozen (item category). Save that information so that when the user returns to start a PI, they can easily start the PI for those same items.
** Provide the same selection options that are available for creating new PI (use the same control):
*** Item Rules
**** All Items
**** Only Items with Positive/Negative QOH
**** Blank - add items on the fly
*** Storage Category (area)
**** start by ‘Cooler’ for example
**** add areas/categories on the fly
*** Item Categories
**** filtered based on the Storage Category selection

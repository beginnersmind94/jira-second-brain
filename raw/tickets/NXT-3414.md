# NXT-3414 — Edit Inspections - Submit

## Description
*User Story*
As an Inspections user, I can submit a completed Inspection for review.

*Requirements*

* There is a Submit button for any In Progress Inspection
* There is validation to ensure all required questions have been answered
** System checks to ensure that all required questions have been answered
** If not, the Submit button is disabled until all required questions have been responded to
*** There’s a tooltip that says “Please answer all required questions to Submit”
** If all required questions have been responded to, the Inspection can be submitted
* A submitted Inspection goes into a Pending Approval status
* Once submitted, the Inspector can no longer edit the Inspection

## Acceptance Criteria
* Inspections can be submitted and are moved into Pending Approval
* Validation occurs when user attempts to submit an Inspection - until all required questions are answered, the Submit button is disabled

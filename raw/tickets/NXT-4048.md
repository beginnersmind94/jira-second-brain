# NXT-4048 — IM - View - General Item Information

## Description
*User Story*

As a Item Management Super User I can view general item information.

*Requirements*

* Header
** Item Name | Item Code | Status (Active/Inactive) | Sourced Locally
** Options to ‘Edit’ and ‘Copy Ingredient’ 
** Ability to Export the document to a PDF
* Image / Document placeholder
* Ingredient Form
* Common Info
** Datasource
** GTIN
** Broken Unit GTIN
** Product Code
** Manufacturer Name
** Brand Name
* Tags
* Sub Ingredients
** Show more button to expand
* Standard Recipe Directions (Mockup needs to be updated with this text)
** Show more button to expand

## Acceptance Criteria
* Item information from creation needs to show up here
* General Item information should be stored by 'Section' for permission purposes. IE This section should always be at the top of the view page itself
* Thumbnail URL,Image URL, Maximum URL for Images

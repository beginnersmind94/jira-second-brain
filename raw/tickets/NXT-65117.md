# NXT-65117 — SCTV - Meal Type Filter for accurate menus

## Description
*As* the Product Owner,
*I want* the system to filter menu items by meal type when calling the API,
*So that* the menus displayed are accurate and relevant to the selected meal.

## Acceptance Criteria
* *Section A*: When linking, users should see only the serving lines that correspond to the selected meal type in the dropdowns.
* *Section B*:Add a new filter in the designer above the serving line with the hierarchy: Meal Type > Serving Line > Categories, displaying accurate data for the selections.
* *Section C:* Also add this to “Temporarily Load Different Menus” feature and preview.

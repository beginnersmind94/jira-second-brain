# NXT-50046 — PR- Plan/Record- Carryover deletion

## Description
As a production manager, I do not want the ability to delete the carryover or delete the item that was added via a carryover. I must remove the carryover from the day it came from.

## Acceptance Criteria
* Do not allow carryover to be removed from the record page ( Same as plan)
* IF menu item is added to the plan or record via carryover, grey out the delete button and provide a tool tip “Added by Carryover”
* IF menu item is carried over to a menu item that already exists on the plan/record. grey out the delete button and provide a tool tip “Carryovers added to this Menu Item”
* This applies to single or multiple serving size scenarios

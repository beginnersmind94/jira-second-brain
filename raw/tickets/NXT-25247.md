# NXT-25247 — INV - Production Withdrawals - Send Withdrawal data to Production

## Description
As a Production Manager I want production withdrawal data and status info from Inventory so I can see my production records as Withdrawal Complete.

*Requirements*

* Send the status of ‘Withdrawal Complete’ to Production for each Menu Line that is withdrawn
** Reference which menu lines were included in various completed Withdrawal transactions (since the transaction can include one or many menu lines)

## Acceptance Criteria
Send all completed production withdrawal info to Production per Menu Line (Production Record)

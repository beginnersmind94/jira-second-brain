# NXT-2059 — Production - Production Record - Carry Over Items

## Description
As a site manager, I am required to enter in the details from the days service for each meal that my site serves. In with this process, is notating if there is food being carried over to another meal service. (This could be the same meal service, or a different meal service… it could also be on the same day, or a different day. For example, if I have cantaloupe chunks leftover from breakfast, I could use the carryovers function to tell the software that I’m carrying this item from breakfast today, to lunch or supper today… or I could choose to carry it over for tomorrows breakfast. 

Each menu item is configured with a maximum number of days that it can be carried over… IF the carryover function is set on the menu item. (They are not required to have carry over as an option.)

## Acceptance Criteria
The software will have a place that allows me to record the fact that I’m carrying over a menu item to a different meal service and/or date. It will also put an alert on the selected meal service/date to remind me that I have the carry overs available to use on that date.

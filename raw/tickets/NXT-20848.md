# NXT-20848 — INV - Orders - Confirm Orders - Cleanup

## Description
The success message that is displayed when orders are created on the Confirm Orders screen seems to be hardcoded to say the status is "Open." Since we now have orders which can go into Pending Review, we need to modify the message to potentially display both statuses, or we should exclude status from the message.

*Requirements*

On the Confirm Orders screen (Create)

* Display the success message on-screen for a longer period of time (or don’t hide it at all)
* Display a message which references the correct order statuses
** **Need to determine which message to show if orders are in both Open and Pending Review statuses after creation
** or *remove the status from the message*: “Success\! 5 orders have been created successfully.”
* Ensure the Merge icon + popup functionality is still working
** ensure the logic is accurate
*** orders can only be merged with the same status (and vendor, site, date)

On the Order Detail > Detail tab

* Fix the banner width to always stretch to the right
* Ensure the buttons are right-aligned

*Also* should make sure the view icon is present for Pending Review orders (*Done*)

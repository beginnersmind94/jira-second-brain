# NXT-26108 — Platform - Permissions - Enable Module level CRUD operations

## Description
As a Primeroedge admin, I should be able to view list of permissions available to view, edit and add to respective modules, pages and functions who can access that specific resource.

## Acceptance Criteria
* Any Module type resource with V/E/A/D button enabled shall be able to delete the resource
* Enable Alias for the Resource field in the resource catalog with V/E/A/D option for Module only
* Resource Permissions CRUD operations with V/E/A/D  buttons to be enabled and making sure FO & BO teams can utilize these resource permissions via roles & permissions page

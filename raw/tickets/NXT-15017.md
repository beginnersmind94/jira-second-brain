# NXT-15017 — System - Templates - Add category for Outreach Letter template types, move existing letters

## Description
As a central office admin, I would like to be able to craft and manage a specific letter for marketing to households in my district, so that I do not have to re-purpose existing letter types to do my marketing.

*Summary*

In order to allow district admins to create and manage letters that are not the typical food-service related type, let’s add a new category and letter type in System that can be accessed in the Outreach Center. Additionally, let’s move the existing Household Letter to this category.

*Details*

* In System → Templates, add a new module: *Account Management*
* In this module’s corresponding Letter Type list, do two things:
** Move the _Letter to Household_ type over from Eligibility. It should now display as a letter type in Account Management, not Eligibility.
*** We can also move the 5 Reminder Letters templates and the ID Cards templates over, since they are also found in Account Management. Bills can remain in Accountability.
** Add the existing letter type: *Outreach Letter*
*** We can use the same template as the _General Letter_ from Classic for now (but rename it to ‘Outreach Letter’)
** The Outreach Letter should also have all of the functionality of other letter types, but it will be accessible in the Outreach Center tabs once complete (just like the Household Letter).
* The same changes should be made in the System Notifications page as well
* This story is just for creating/providing the templates to be used in the Outreach Center. Actual implementation of that will be in another story.
* *No UX Required*

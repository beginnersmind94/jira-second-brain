# NXT-5707 — INV - Ordering Conditions - Summary Page - Item Site Detail slide-out

## Description
As a District Supervisor I want a detailed item ordering conditions slide-out view on the Summary page so I can see the site specific ordering conditions for an item.

_Inventory > Orders > Ordering Conditions_

*Requirements*

On the Ordering Conditions Summary page ([NXT-5706|https://primeroedgenext.atlassian.net/browse/NXT-5706]), provide a button to view the Item Site Details so the user can see the specific site ordering condition breakdown for a single item.

When the user clicks the button, a right-side slide-out will appear displaying the list of sites and the vendor from which they are configured to order the item.

+Site List:+

* sorted by *Site Type*
** Checkbox for each Site Type
*** if unchecked, dropdown is disabled
** Site Types are collapsible/expandable 
*** Site Types are all collapsed by default
*** only one Site Type at a time can be expanded
** display Vendor dropdown per Site Type
*** if all of the Sites in a particular Site Type order from the same vendor, display that vendor
*** if sites within a particular Site Type order from *multiple different* vendors, display ‘Multiple’
**** Multiple is not a valid dropdown selection, but rather an informational value to indicate that the child sites do not all order from the same vendor
*** user can change the vendor for an entire Site Type
**** if user changes the dropdown for an entire Site Type, the selected vendor will apply to only the selected (checked) child sites
**** if the dropdown displays ‘Multiple’ and the user changes it to a single vendor, provide a warning that all checked sites will be configured to the same vendor
***** “Sites within this Site Type order from more than one vendor. Changing this selection will apply this vendor to all the selected (checked) sites.”
****** OK/Cancel
* *Site*
** Code + Name (one column)
** Checkbox for each Site
*** checkboxes are available all the time
*** if unchecked, Site is not ordering from anyone
**** if a Site has a vendor selection and is unchecked, remove the vendor
**** if unchecked, Vendor column is blank
*** if a Site is checked, inherit the current Site Type vendor selection, as long as it is not Multiple
**** Ex: Site Type Elementary orders from ABC Vendor
***** new site is checked
***** by default, new site is configured for ABC Vendor
* Vendor Info 
** this is the vendor for which the site is configured to order the item
** this would display as blank if the site is not configured to order the item (unchecked)
* an Edit button
** this allows the user to change the selected vendor for that site (one site at a time)
** if a Site is unchecked and the Edit icon is clicked, automatically check the Site
** this would provide a Vendor dropdown, which would display all vendors with an active contract containing that item
** if the item is only provided by one vendor, the Edit icon is disabled
* a Save button to save the changes made to the sites.
* a Cancel button to discard changes made
** if user has made changes and clicks Cancel, prompt the user with an unsaved changes alert

## Acceptance Criteria
On the Ordering Conditions page, an option to view the site level details for a selected item
The site details will be displayed in a right-side slide-out view
Sites will be grouped by Site Type
User can change the Vendor for any Site Type or Site

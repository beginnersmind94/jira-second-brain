# NXT-28551 — INV - Orders - View Permissions 

## Description
As a district admin I want permissions for Orders so I can control which users are able to access various pages and subpages throughout the Orders category.

*Requirements*

* Implement the page level view permissions for the nav level Orders category pages
** View Orders
** Standard Ordering 
** Production Ordering
** Quick Receipt
* Disable CRUD operations for the Resources of these pages
** No need for E/A/D
* Identify “hidden” pages, like Quick Receipt Add, where we may have issues if permissions are not grouped

# NXT-65612 — INV - Internal Vendors - Items - Emphasize Default group when no Schedule exists

## Description
As an Inventory Distribution Manager I want the internal vendor Items tab to indicate when an item has the Default shipping group if that group is unscheduled, so I can know to move the item to a different shipping group.

h3. Vendor Items - Default group w/ missing Default schedule

!Vendor Items Default-20250819-141434.png|width=91.66666666666667%,alt="Vendor Items Default-20250819-141434.png"!

## Acceptance Criteria
* Update the Items page to emphasize items with the Default shipping group *IF* the Default shipping schedule is blank
** put them at the top
** emphasize the text for Default
** show alert icon next to shipping group value
** If the Default group has a schedule configured, do not show the alert icon
** Show tooltip on alert icon:
*** “Schedule does not exist for the Default shipping group. Please configure schedule or move item to a different shipping group.”
* Also show icon on ANY shipping group that does not have a schedule
** show alert icon next to shipping group value
** Show tooltip on alert icon:
*** “Schedule does not exist for the [shipping group name] shipping group. Please configure schedule or move item to a different shipping group.”

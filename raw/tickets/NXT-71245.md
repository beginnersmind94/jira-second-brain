# NXT-71245 — SchoolCafé - Forms - (Part 2) Family Hub-related changes for Forms, Fill out a Form Page/Submission

## Description
As a Family Hub parent/applicant, I would like to be able to apply for Summer EBT (as well as any other non-traditional form) applications with minimal steps, so that I do not have to use an external system just for this purpose.

*Summary*

* Story 1:
** Update School Café Family Hub operations and licensing to reflect the new Forms changes.
** Add a new SchoolCafé 'Forms Template' template and application/wizard to reflect the new Forms changes. Effort: 8
* Story 2:
** Add a new SchoolCafé 'Fill out a Form' step to reflect the new Forms changes. The actual logic/ submission of the form application can be added later on if required. Effort: 8

*Details*

# Add a new ‘Fill out a Form’ page which functions like the ‘Apply for Summer EBT’ page, displays all active Forms and allows them to be filled out by a parent (Guest not required). It can have the same steps as that Summer EBT page, except one additional step described below.
# Within the SchoolCafé 'Fill out a Form' page (and template), do the following:
#* Add a new step before the _‘What would you like to apply for today?’_ step:
#** Default Page title: *Confirm your school(s)*
#** Default Body:
#*** To determine whether any students in your account may qualify for other programs, please select all sites to which they attend.
#*** (Display site drop-down)
#* Both title and body should be editable/translatable (provide default translation) in Form Template
#* In the site drop-down, display only those sites which are toggled on.
#* When the user selects their sites, only then should we allow them to proceed to the ‘What would you like to apply for today?’ step.
#* The rest of the form should proceed as normal based on the Form selected.
#** Only Forms where those sites are listed in the 'Site(s) Associated' list

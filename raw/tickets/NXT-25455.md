# NXT-25455 — INV - Vendors - Contract Items - Price Conditions Slideout Enhancements

## Description
As a District Admin I want an option for retaining existing charges and discounts so they are not removed when I add a new price condition.

Improve the Price Conditions slide-out to allow for price editing and charge/discount carryover.

*Requirements*

* Allow users to edit the *current* price condition
** only the Price, Del Fee, Proc Fee, NOI
* Allow users to carry over existing charges and discounts
** on the Add New Price, add a new option called ‘Use existing charges and discounts’
*** this will prepopulate the 3 charge/discount fields with existing values
*** this will be set to Enabled when we are inserting prices via the new Contract Items screen for Update Price ([NXT-23956|https://primeroedgenext.atlassian.net/browse/NXT-23956])

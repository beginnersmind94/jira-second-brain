# NXT-3560 — Edit Inspections - Dropdown Question Type

## Description
*User Story*
As an Inspections user, I can see Dropdown Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Dropdown Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Dropdown selections
** Whether question is required
** Notes section
** Media adding (placeholder, future story for actual uploading)
** Note: Point values are *not* displayed here
* User can select a choice from the dropdown
** Only one choice can be selected at any time
* Choice is properly saved
** If question is required, Inspection cannot be submitted if no selection is made
* Choice can be changed if Inspection has not been submitted
* If question has a score associated with it, the score is properly updated according to the selection being made.

## Acceptance Criteria
* Dropdown question appears in Inspections
* Dropdown and notes can be selected, saved, and edited
* Score updates properly if present

# NXT-5231 — Application Processing - Dynamic Results Panel

## Description
*User Story* 

The application determination shows the current state of the application and updates as the data is manipulated.

*Requirements* 

Show the determination on the right panel - Designation and Basis

## Acceptance Criteria
- Multi-language, ADA compliant
- user can see application determination
- ‘Eligibility’ designation, ‘Basis’ of determination based on info provided
- determination is dynamic and updates as changes are made

# NXT-64666 — WS - Benchmark - Value Type Indicators

## Description
As a Workspace user, on the Benchmark drawer, I want to be able to know what type of value to put in the column header, so that I can know what to put.

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* In the grid, indicate the expected type of value for each column
** (%) for the percentage values
** ($) for the money values
** (#) for the number values
* Rearrange the grid column order: Put Eco Dis first, then Breakfast, Lunch, Snack, Supper, Revenue, Waste Meals and MEQs
* Don’t show School column label in first header

## Acceptance Criteria
* Users are able to see what type of value each column has in the benchmark config screen

# NXT-2383 — INV - Reports - Transaction History

## Description
As a FSD I want a Transaction History report so I can see everything that occurred during a particular period of time.

*Requirements*

* Add new Transaction History report to the Reports page
** Description: Lists all inventory transactions for a selected date range.
* Display Site selections
* Provide Date Range selections
** default past month
* Provide a checkbox option for ‘Show only backdated transactions’
** this should show only transactions which were recorded past the actual transaction date 
** Recorded Transaction Date > Actual Transaction Date
** Do not consider Time
* Provide a toggle to *sort* using Actual Transaction Date or Recorded Transaction Date
* Generate the report on click of Generate button
** Display report in overlay
** List all transactions for the date range
** Columns:
*** Site Code
*** Site Name
*** Transaction Type
*** Transaction #
*** Transaction Date (recorded)
*** Actual Transaction Date
*** Status
**** either status or custom comment

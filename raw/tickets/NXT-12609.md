# NXT-12609 — Item Management - POS Info Changes

## Description
Various changes needed for POS Info within Item Management.

* Modify “Is Meal” to “Meal”
* Modify “Is Vendable” to “Vendable”
* Modify “Is Orderable” to “Orderable”
* Modify “Is Default Reimbursable Meal” to read as “Generic Item”
* Modify “Use Default Reimbursable Meal Prices” to “Use Generic Item Pricing”
* Only items set as ‘Generic Item’ should be in the drop down list for “Use Generic Item Pricing”. Position of  the dropdown should be below the “Use Generic Item Pricing” toggle.
* Remove linkage between “Meal” and “Generic Item”. These are not required to be linked as you can have a generic item for both meal and a la carte items. Currently, when the “Use Default Reimbursable Meal Prices” is enabled it will also enable the “Is Meal”.
* Confirmation message when save is selected for the item that is used as the ‘Generic Item’ that says “You are changing pricing for a generic item. If you continue, all items linked to this item will have their prices changed. Do you want to continue?” Yes/No

## Acceptance Criteria
* Verify label change of “Is Meal” to “Meal”
* Verify label change of “Is Vendable” to “Vendable”
* Verify label change of “Is Orderable” to “Orderable”
* Verify label change of “Is Default Reimbursable Meal” to read as “Generic Item”
* Verify label change of “Use Default Reimbursable Meal Prices” to “Use Generic Item Pricing”
* Verify only items set as ‘Generic Item’ should be in the drop down items list for “Use Generic Item Pricing”
* Verify linkage between “Meal” and “Generic Item” are removed. These are not required to be linked as you can have a generic item for both meal and a la carte items. Currently, when the “Use Default Reimbursable Meal Prices” is enabled it will also enable the “Is Meal”.
* A confirmation message is displayed when an item with 'Generic Item' set has a the prices edited. Options for Yes and No. If Yes, item price is modified. If No, item price is not modified and dialog is closed.

# NXT-10703 — Scanning Tool - Allow user to Scan paper form(s)

## Description
As a central office user I want the ability to scan paper free-and-reduced meal applications into\[ the system, so that I can use them as a reference image when processing to apply benefits to the student(s) within the family.



Summary

Add a ‘Scan’ option to the existing Application Scanning Tool that allows users to scan an application (in addition to its existing features).



Mock-up 

N/A



Details

* In the standalone scanning tool, add a new screen/menu option: Scan Form
* Within the menu, allow the user to do the following:
* Initiate a scan of an Application using their attached scanner
* Use OCR or similar used to copy writing from set fields on the paper form to appropriate table fields 
* Generate unique application ID, assign an application number for that region/district

  * Allow for review of scanned applications with ability to reject selected applications

* Add application to Applications page and assign application entry method as Scanned
* There will be a follow-up story to view that scanned application in SchoolCafé. 



Equivalent guide from Classic:

[http://docs.primeroedge.com/Content/Student+Eligibility/SE_Applications_Scan+Quick+Step.pdf|http://docs.primeroedge.com/Content/Student+Eligibility/SE_Applications_Scan+Quick+Step.pdf]

## Acceptance Criteria
Verify that the new Scan screen within the scanning tool does the following:

* Initiate scan of attached scanner
* Generate unique application ID, assign an application number for that region/district
* Assign application entry method as Scanned
* Allow for review of scanned applications with ability to reject selected applications
* OCR or similar used to copy writing from set fields on the paper form to appropriate table fields 

Application should be at a point where it can be viewed in SchoolCafé at a subsequent point.

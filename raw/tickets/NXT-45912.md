# NXT-45912 — IM/Platform - Manage Views - Global Views

## Description
As an Item Management user I want the Manage Views tool to be defined by the module itself rather than simply allowing custom user views, so that the system can provide very tailored, useful pre-loaded views out of the box.

## Acceptance Criteria
* Need the ability to insert pre-loaded Saved Views (defined by us internally) that users can select using the Manage Views tool (Platform will create “Global Views”)
** These would not be based on user, but would be provided to the entire district/environment
** Need the ability to support these pre-loaded views while *disallowing* users to edit those views, and perhaps to prevent them from creating their own (depending on the page)
* The ‘Show Filters’ and ‘Manage Views’ buttons should be converted to small icon buttons so as to not interfere with module-specific actions placed in that grid location
** basically make them smaller (mockup attached)

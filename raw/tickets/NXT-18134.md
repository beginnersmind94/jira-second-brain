# NXT-18134 — INV - Manage Orders - Order Detail - Update order on Save button (Cleanup)

## Description
As an Inventory user I want orders to be updated only on click of Save button so I can view all order updates made in a single save.

*Requirements*

* For Order Adjusted/Update, capture a history record only on click of Save button
* Even though the Edit actions are independent for some fields (Delivery Date, Delivery Location), require user to click Save button to officially save these updates
** Accordingly, provide ‘Unsaved Changes’ prompt to further communicate this to the user
** Perhaps disable the Save button until any changes have been made
* Update API calls to reflect single save update
* Provide Unsaved Changes prompt on click of Approve/Reject buttons
* Provide Unsaved Changes prompt on click of Resubmit button

+Potential Triggers for Adjustment:+

* quantities changed
* Item added
* Delivery Date changed
** reminder: this will be controlled by special permission
* Delivery Location changed
* Comments entered/changed

*Below are challenges with testRigor.*

# The recorded tests does not work right away, need to update elements manually most of the times– None of the test case worked straight way after recording.
# Cannot update locators at one time as the test cases are mostly using locators based on positions. The locations vary from test to test. This leads to high maintenance.
# For each set of data, we need to create a different test (Ex: Suppose if I need to test login functionality with different roles then I need to create a separate test for each role)
# We cannot have a reference of same test in multiple test suites (Ex: If we a sample test in smoke and regression suite. If there is a need to update the test we should update the test in both places.)
# Cannot debug and check where exactly the issue is present

*Below are features that cannot be automated currently or testRigor team said they will look in to it.*

# We cannot verify if data is sorted
# We cannot perform Cross Browser Testing (Per testRigor team: This feature is being released right now. Should be there no later than Oct 15th)
# Unable to scroll down and click on desired element
# Issues in identifying elements when same page has multiple elements with same description (Ex: Unable to identify serving size + or – icons for contribution section) Per testRigor team they are working on improvement
# Cannot verify tooltip – (Per testRigor team : the tooltip feature would only work with OCR enabled and the OCR have some issues right now)
# Upload images (does not work in all cases. I was able to upload a file in item management -Item information. But unable to upload image for users in Settings)
# Verify data in Card View (Ex: In card view cannot verify data) Per testRigor team they said they will work on double anchoring
# Unable to Verify messages which stay for couple of secs – testRigor team said they will look in to it

*Advantages:*

Note: This is possible for simple tests. If testRigor team fixes above challenges then we can think of considering it.

*Recomendations:*

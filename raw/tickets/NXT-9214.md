# NXT-9214 — {Menu} Analysis - display overall Pass/Fail for Milk Types

## Description
As a Menu Planner
I need to be able to build Menus
so that I can show that they are compliant.

Display “Pass” as regular black text or Fail Low (highlight values yellow) for daily requirements. Note that exceed only applies to Menu Cycle week for M/MA and Grains - is included in title for design purposes only. There are currently no conditions that would cause Fail High for a single Menu as Nutrient requirements are for a school week (see the footnote on the analysis drawer). These are included here for easier extension for Menu Cycle week only.

Includes pass/fail for Milk-Variety check (including at least 1 variety of unflavored for 14+ meal types). An invalid MMA acting as Grain could also cause daily failure for 14+ breakfast.

Indicate overall Pass/Fail for the day.

## Acceptance Criteria
Failing all checks causes a lot of yellow and red overall
Passing checks displayed in black
Failing checks are displayed accordingly
Milk-V failure works as expected (pass/fail)
Invalid MMA acting as Grain causes check to Fail
If any check fails, overall is Fail
If all checks pass, overall is Pass

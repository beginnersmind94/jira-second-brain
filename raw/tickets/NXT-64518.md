# NXT-64518 — Acc - CACFP Reports - Participation

## Description
As Dana the Director I want to run the Participation report for my CACFP program so I can see who participated in detail, and in summary.

Currently the Participation report does not provide support for CACFP.

*Requirements*

* User can select from a Programs dropdown.
** Options: SNP (Default), SFSP, CACFP
* User can select CACFP meal types via the Meal Type dropdown.
** Nice to have: filter the Meal Type dropdown by the Programs dropdown.
* User can run the Participation report, both Summary and Detailed version.
** If filtered for CACFP, then we need to pull back the expected data.
** Summary: Meal counts by Eligibility and Reason, with grand totals.
** Detailed: Specifics on who had the meal, pricing, etc.

## Acceptance Criteria
* User can run the Participation Report by Program.
** SNP (default), SFSP, CACFP.
** Single select dropdown.
* User can select CACFP Meal Types from the Meal Type dropdown.
* User can run the report for CACFP Meal Types
* User can see the PowerBI reports for CACFP if filtered accordingly.
* Report Selections updated to include Program selection.

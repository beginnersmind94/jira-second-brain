# NXT-4553 — (OBSOLETE) AcctMgt - Tags - Configuration

## Description
*User Story*

As a district admin I want a way to see and interact with all account (Student, Adult, Special Account) tags.

*Mock-up (OLD)*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_configuration&id=tklkea&g=1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_configuration&id=tklkea&g=1]

(Password: 443PE2.0Team)

*Requirements*

* On the Tag Management page in System, confirm the Account Management module is available.
* If the AM module is selected, the grid should show all Tags associated with the module.
* Existing interactions should work - 
** User can edit the existing.
*** User can activate / deactivate.
*** User can change the name.
** User can delete.
** User can add new.
** User can see active / inactive.
* This story is for managing Tags themselves.
** A story to apply Tag(s) to an individual account will be handled separately: [https://primeroedgenext.atlassian.net/browse/NXT-4596|https://primeroedgenext.atlassian.net/browse/NXT-4596|smart-link].

## Acceptance Criteria
User can view all AM tags.
User can interact with tags.

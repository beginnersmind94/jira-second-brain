# NXT-37983 — TechDebt-ItemMaster- Remove MP_RecipeImageLocation

## Description
Maintain a Single Source for Image Locations data for both Item and Recipe.

*Code Changes:*  Use ITMM_ItemImageLocation for saving/fetching images for both Item and Recipe

*Schema Changes:* Add a new column Origin(Varchar(20)) to identify the image is for Item/Recipe/menuItem

*PreDeployment:* Take back up of ITMM_ItemImageLocation & MP_RecipeImageLocation

*Post Deployment:* Copy the existing image locations of the Recipe from ITMM_RecipeImageLocation to ITMM_ItemImageLocation.

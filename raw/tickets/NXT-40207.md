# NXT-40207 — Email - View Details

## Description
As a SchoolCafé user, I would like to be able to quickly view the details of an Email sent so that I can verify what was sent and who it was sent to.

*Notes:*

* This page is read only content, no actions

*Flow:*

* Open SchoolCafé and browse to Settings > Support > Monitoring Dashboard
* Click on Email Monitor
* View list of Emails sent
* Click View Details on a row
* Review the details in a drawer
* Return to the page

*Requirements*:

* On Email Monitor page, do the following:
* Add a button to the rows in the action column if it is not already there
** Title: View Details
** On click, Open the View Details drawer
* 
* Create a drawer that opens when click on View Details from the Email Monitor page
** Title = Email Details
* 
* There are 2 horizontal tabs
* Tab 1:
** Title = Details
* On load, show the html version of the label and value pairs like we have done in Details drawers for other recent work
* Show all the fields from the Email record and their values except the user fields
* At a minimum, these fields should include:
** Source
** From
** Roles
*** List of roles selected for distribution
*** If none, then None
** Subject
** Body
** Attachments
*** Show a list of attachments hyperlinked to the files
*** Use full name of the file
** Date / Time Created
** Status
* 
* Close button
* 
* Tab 2:
** Title = Recipients
* On load, load the grid of all SchoolCafé users that received the email and where in the individual users list or roles list
* 
* Grid columns:
* Name
** If external email, leave blank
* Email Address
* Selection
** Values:
** User
*** Matches if the user was selected in the users list
** Name of the role
*** Matches if the user was in a role that received the email
** External
*** Matches to if the email comes from the External list
** Example: if the person was sent the email b/c they were in the list of Users, then show User
** Example: if the email configuration had a Role selected of Cashier and if the person was sent the email b/c they were found in the Cashier role, then show Cashier

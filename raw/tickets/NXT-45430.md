# NXT-45430 — Imports - Long Running Processes - Kill Process

## Description
As a Customer Support User, I would like to kill long running processes so that I can free up system resources.

*Requirements:*

* This functionality is for the Import Monitor tab on Imports & Exports
* Backend needs to write an API that when called, will kill the import process
** When it kills it, it should set the status to Terminated
* 
* In filters section:
* Update Status values to include Terminated
* 
* On each row in Import Monitor, do the following:
* Add a button to the Actions column
** Name: Kill or use symbol 
** Visible only to Customer Support role
** If click:
*** Show the usual confirmation box
*** If confirmed,
*** Call the API to kill the process
*** Show the waiting circle until get confirmation that it is killed or get an error
*** Show the usual Success or Failure message to the user
*** Refresh the grid
* 
* Everything else will remain the same

Modal Popup message when the Cancel button is clicked:
Are you sure you want to cancel the DC Import-Direct Certification (Import-6492)
for Lewis County Schools?
Note: The following values will come from variables.
DC Import-Direct Certification
Import-6492
Lewis County Schools

## Acceptance Criteria
* Customer Support role user can kill an import process
* When the do, it sets the status to Terminated

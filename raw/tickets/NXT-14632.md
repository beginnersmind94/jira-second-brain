# NXT-14632 — MP - View a Calendar-like screen or schedule and the ability to assign menu items to a day

## Description
As a user, I need to be able to view the interactive menu planner calendar to create a ‘new’ menu with the tool

Mockup:

!image-20210618-020322.png|width=100%!

Requirements

* Entrée Search box on the left hand side of the screen
* Be able to add or remove a week at the top of the ‘calendar’
* Have each week show up in an individual column
* If clicking the ‘Next’ button, it will navigate to the ‘next’ page, which is the sides page
* ‘Back’ will take you back to the ‘Home’ page

## Acceptance Criteria
* See individual weeks at the top and can add / remove a week
* Be able to move between the weeks in order to see a different ‘calendar’

# NXT-4736 — AcctMgt - Add New Patron - UI Updates

## Description
*User Story*

As a product owner I want to see a few missed pieces between the wireframes and the product to-date filled in so that the user experience is a bit more seamless.

*Requirements*

* After a patron is added successfully via the Wizard/Stepper, load them immediately via the Split View.
** New patron should appear as a Card, with a ‘New’ clip along the top similar to the below. (Like we’ve done with Roster)
** Any existing search query should be removed.
** This would be the only card shown in the Split View.
*** Just imagine ‘Updated’ says ‘New’ instead. I’ll update this screenshot later.

## Acceptance Criteria
Patron is loaded immediately into Split View after creation is successful.
Any existing search query is removed as part of this.
Patron's card has a little 'New' clip on it.

# NXT-24953 — PR- System setting- Allow Auxiliary Serving Groups on meal counts

## Description
As a production manager, there needs to be a way to add auxiliary serving groups to track the meals planned or served to those particular serving groups

Currently there is a system setting available but is not functional

!image-20220427-151329.png|width=1742,height=80!

## Acceptance Criteria
Allow system setting Enable Meal Counts for Auxiliary Serving Groups

Auxiliary serving groups are:

Program Adult

Second Meal

Drop Tray

Sample Tray

NO OVS selection for these serving groups

ONLY Show Serving Groups on the Meal Count Portion of Plan/Record and reports

!image-20220525-134041.png|width=761,height=472!



Once set,  add all auxiliary serving groups on plan and record pages

Only effect records that are NOT in Record Complete Status

Any production plan complete status records will be effected with counts as 0 for the selected auxiliary serving groups so it does not affect the status

Provide Tool Tip on setting “These changes will affect all production records that are not COMPLETE” 

If Turned off, only effect Plans/Records that are not Complete

Verify all reports that include serving groups that the auxiliary serving groups

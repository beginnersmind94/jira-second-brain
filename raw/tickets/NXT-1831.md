# NXT-1831 — INV - Reports - Batch Detail Report - Aging

## Description
As a Site Operations Manager or FSD I want an Inventory Aging option on the Batch Details Report so I can see how long my batches have been in my perpetual inventory.

*2/7/2022 - Report page will not load (JM)*

*Requirements*

* Modify the Batch Detail Report to accommodate the requirements for Inventory Aging Report
** Batch Details and Aging are very similar reports
** We will add options to make a single report for both 
* Add additional options to the Date Criteria section
** ‘Show All’ option
*** show all currently on hand 
** By Received Date
** By Date on Product
** If either ‘Date’ option is selected, show the search options
*** Provide Month and Year dropdowns as option other than date range search
*** Provide Date Range search
**** include time - for searching Received Date 
**** optional: disable the Time field when user has selected ‘By Date on Product’ (because bin lines do not have time)
**** default: next month
* Add ‘Age in Days’ column to the report, after each Received Date column
** Today’s date - received date = age in days

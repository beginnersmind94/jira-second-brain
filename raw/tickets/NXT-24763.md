# NXT-24763 — MP > Menu > Menu Editing/Creating Page > Menu Item Categories need to match config in IM

## Description
In Item Management, there is a "configuration" page for setting the order of Menu Item Categories.
The Menu pages need to follow the "Display Order" that is configured.

     1.) The Regular Menu pages that are created through Menu Planning > Menu

     2.) The Menu Planner page, when adding menu items

These are currently simply displaying in alphabetical order on the page, and are not following the configured "Display Order." 

The “Display Order” should be used on both the pages for editing/creating menus, and also in the Menu Planner tool.

## Acceptance Criteria
In Item Management, there is a "configuration" page for setting the order of Menu Item Categories.
Both Menu pages need to follow the "Display Order" that is configured.

1.) The Regular Menu pages that are created through Menu Planning > Menu

2.) The Menu Planner page, when adding menu items

NOTE: Menu Items should always appear in the category that they have been assigned from within Item Management. 

Sort menu items within the category based on alphabetical order. (If same name, look for next available different letter to get alphabetical order)

Also display on Menu Planner the colors that are chosen on the IM menu item category configuration page. If No color configurated, display grey color for that category.

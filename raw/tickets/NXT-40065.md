# NXT-40065 — Student Export – Update UI - Drag and Drop Functionality - Dev

## Description
As a SchoolCafé Admin I want to see the Field Mapping Screen more consistent with our site UI so that I can not be distracted with changes.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Exports
* Go through the Pre-export and get to the Field Mapping screen
* Pick which fields you will export and then finish the export
* The table update story needs to be done first

*Notes:*

*Requirements*:

* This story is for the SchoolCafé Settings > Imports & Exports > Exports > Field Mapping tab
* This story applies to the Student Export
* When in Student Export on the Field Mapping Screen, do the following:
** Add drag the rows up and down to configure the order of the fields

## Acceptance Criteria
* Allows to drag the rows up and down to configure the order of the fields

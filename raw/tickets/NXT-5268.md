# NXT-5268 — Verification - Notification - Follow-up Notice Automation

## Description
As Vera the Director I want the system to send notifications automatically so that I can increase my likelihood of receiving a response.

*requirements*

* User needs a way to setup automation for follow-up letters being sent.
** Send a follow-up notice for incomplete applications every X days until the end of verification.
** User can configure the X to be any number from 3-30.
** This would apply only for applications with an email address available.
** If the setting is off, Day selection is disabled.

Label: Automatic Follow-up Notices

Brief toast message on changing, if needed:

* On: Automatic follow-up notices have been turned on.
* Off: Automatic follow-up notices have been turned off.
* Days: Automatic follow-up notices will be sent every X days.

Only needed for the following tracking tabs:

* Incomplete
* No Response

## Acceptance Criteria
* Every X day after the Selection Letter is sent, the system should automatically send the follow-up notice.
* User has a place to configure the number of days between automatic reminders, and can turn them off if desired.
* Default value for days: 4 days.
* Default status for automation: off.
* Emails are sent.

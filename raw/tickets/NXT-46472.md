# NXT-46472 — PR- Consolidated Tab Inactive and Pending Deactivation Menu Items

## Description
As a production Manager, I would like to be able to see what menu items are pending Deactivation on my record page so I can decide if I should use them or not or make a decision to use a different menu item.

## Acceptance Criteria
* Capture menu items that are in active, inactive or pending deactivation status
* Display a warning sign on any pending deactivation menu items
* Provide a tool tip on the same warning, “Pending Deactivation”

# NXT-63285 — SchoolCafé - Production - Calendar - Add button to quickly publish current calendar to Family Portal

## Description
As a production planner, I would like to be able to immediately publish the details of my current plan to the Family Portal, so that I don’t have to navigate to the Publish Menus page or wait on auto-publishing if I’m making a last minute change.

*Summary*

On the Production calendar, add a Publish Plan button which immediately publishes the current production plan calendar details on the page to the Family Portal.

*Details*

* In the Production Calendar, add a new button: *Publish Plan*
** When this button is clicked, immediately publish the details of the current (1) site(s) and (2) production calendar details to the Family Portal.
*** If necessary, we can just publish all details immediately, even for sites/dates which have not changed.
** Display a success message: “Calendar details have been published to the Family Portal successfully!”

# NXT-37170 — PE - Searching USDA Foods in PE (API+ UI)

## Description
_(none)_

## Acceptance Criteria
The user should have an option to select “Data Provider” - Defaulted to 1WorldSync. user can select additional option as “USDA - Branded Foods”

User can only select only option at a time

The user should be able to search the ingredient based on the existing fiends such as Item Name, Description, GTIN, Item category etc.

The items should be searched based on latest available date (no duplication listing allowed in search results)

The items shall list as single record in the grid as there is no hierarchy defined to it

User should be able to click on the hyperlink (As-is) and be able to view the ingredient info tab with the attributes as applicable

Item Name

Item Short Name

Product Code - as NULL

GTIN

Brand

Manufacturer

Ingredient Form - As Purchased (default)

Ingredient Statement

Standard Recipe Directions

Serving Size (based on latest available date) - NULL allowed as some of the entries does not have serving measures defined by USDA

Nutrition Section

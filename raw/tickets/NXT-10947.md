# NXT-10947 — Eligibility - Applications - Application Image (Digital)

## Description
As a central office user I want to view the digital application image so that I can see what information was submitted by the household member as reference.

* New link added in the sub navigation of the application details for “Application Image”.
* If the application is an entry method of “Online” or “Manual”, then we will show the digital image in the main viewer window.
* The application image is not a typical “document” that is associate with the application as it is the application itself. We will have a separate section for documents associated with the application.

This image is built using HTML. It will be embedded into the viewing window when the sub-nav link is selected.

Users will also need a way to export and print the images.

## Acceptance Criteria
User has the option to view the Application Image for an online / manual application while viewing application information/details.
Application Image loads in a preview window with information per the application submission.
User can export the application image to PDF or send to print.
- Any updates / changes to data should not update the actual image.

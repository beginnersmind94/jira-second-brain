## Acceptance Criteria
* Redesign Nutrient Details to more effectively use the card space
** in edit mode, rows should be less vertically stretched out, card should be shorter
** Size the input fields to eliminate unnecessary width (i.e. we know the max length of values)
** Change card to 50% page width
*** consider expanding to 2/3 when editing (optional)
** Improve error handling
*** do not repeat red messages for each row
*** simply highlight the missing fields and provide a singular message
* Automatically set nested nutrients to zero if the parent nutrient is set to 0
** e.g. if ‘Total Fat’ is 0 then both Sat and Trans Fat will be 0
** also for Carbohydrates and the nested Fiber/Sugar rows
* Update business rules:
** Vitamin A, Vitamin C, Moisture, Ash should be changed to optional
** Moisture and Ash do not require a “Missing” checkbox as they are never present on nutrition labels
* Consider 0% as valid input
** Currently, vitamin % is not saving zeroes

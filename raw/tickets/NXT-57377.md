# NXT-57377 — IM - Item Onboarding - Update Contributions - Select Components (Step 1)

## Description
As an onboarding district admin I want a step to bulk select food components so I can configure my item contributions in a bulk method.



!IO - Contributions Components-20241211-154908.png|width=100%,alt="IO - Contributions Components-20241211-154908.png"!

## Acceptance Criteria
* Provide an edit mode for ‘Select Components’ which uses the same bulk Contributions grid but provides selectable icons for each of the 5 components
* List each of the 5 components as a column
* Provide a deselected, grayed version of each food component icon in each cell
** these icons come from Menu Planner and are used on the newly redesign Contributions card 
** Opacity: 50%
* Allow components to be selected by clicking the icon
** upon enabling, change the icon to be full color using the MP colors
* If Fruit is enabled, display the ‘Juice’ checkbox right after the icon
* if Vegetables is enabled, provide a multi-select dropdown containing each of the vegetable sub-groups
** also display the Juice checkbox like Fruit column
** *UI Note:* we should try to change the multi-select dropdown to display “4 sub-groups selected” instead of showing the chips like “(Red/Orange) +3”
*** _Low priority side goal (may require platform involvement)_
* for Grains, show 3 different icons
** WGR - this one is first and should be larger
** Non-WGR - uses gray outline when enabled, with 70% opacity orange, and is smaller than the WGR icon
** GBD - uses orange outline with white fill, new cookie icon, and can only be clicked if WGR or Non-WGR is enabled
* Provide a Save button which returns to the review/view mode
* Provide a Proceed button which moves to the [Contribution Values step|https://primeroedgenext.atlassian.net/browse/NXT-57382]

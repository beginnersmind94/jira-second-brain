# NXT-18516 — Acc. - Program Configuration - Copy from/to another year

## Description
As Vera the Director I want to copy my selected Program Configuration options over to another year to save time during the hectic start of the school year.

*mockup*

[https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=471%3A15615&scaling=scale-down-width&page-id=430%3A2228&starting-point-node-id=471%3A15615|https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=471%3A15615&scaling=scale-down-width&page-id=430%3A2228&starting-point-node-id=471%3A15615|smart-link] 

*requirements*

* Add an option to let a user copy Program Configuration from 1 academic year to another.
** User selects the year they want to copy from.
*** Options: any prior and current academic year.
** User selects the year they’re copying to.
*** Options: current academic year and the next academic year.
* User has a confirmation prompt when they go to activate the copy.
** Please be aware that this will update the configuration for 123 sites. These changes cannot be undone. Are you sure you want to proceed?
* Executing the copy will copy all program configuration options over to the destination and replace any existing ones.

Success message: Program Configuration copy completed.
Error Message: Programs are not configured for the selected year.

## Acceptance Criteria
User can select a year they want to copy from, and a year they can copy to.

* Year restrictions for each dropdown:
** Copy From
*** Any prior + current academic year.
** Copy To
*** Current + next academic year

User has a confirmation prompt before anything is replaced.

Program Configuration updates appropriately.

* Programs, sites, dates, Feeding Site, Reimbursable, everything from the page.

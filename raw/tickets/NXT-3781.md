# NXT-3781 — Create as of date selection

## Description
There is the need to have a returned value or list based on a ‘As of date’. An example is we need all student balances as of 02/02/2020. The list of all student balances on that date will be returned.

## Acceptance Criteria
When a single date is listed it is a 'As of' date filter

When a date is selected they are treated as an AND with other filters.

# NXT-5400 — System - Configuration - POS - Attendance - SIS Integration

## Description
As a FSD I want the SIS system to give attendance figures to my system.

The user will be able to collect data for each site indicating the attendance for each grade for the selected date range.

Totals will be maintained by day/week/month and year for each site

*Mock Up:*

*Requirements:*

* User can input attendance data by site for the selected day/week/month/year
* Data can be input by a file that indicates Site, Date, and count.

## Acceptance Criteria
The user will be able to enter attendance data collected from the customers SIS system.
The user will be able configure Import to accept and process a user supplied file

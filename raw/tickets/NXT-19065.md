# NXT-19065 — DataExchange-Action(s)  functionality changes

## Description
Currently, when we click on 3 dots under the Action(s) column, we get a Context Menu and from the Context Menu, we need to select either Edit or Delete.
This story is to replace the 3 dots under the Action(s) column with Edit and Delete icons which are used across the UI.
This way the user needs to do only one click either Edit or Delete to perform the required action.
Also, rename the column name Action(s) with Action.

Changes need to be made on the following pages:
1)Import Dashboard
2)Export Dashboard
3)API Configuration

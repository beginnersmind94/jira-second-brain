# NXT-5223 — Application Processing - Content & Navigation

## Description
*User Story* 

As an admin, I am able to see the sections completed in a submitted application for processing so that i have ability to edit the sections.

*Mock Ups* 

*Requirements* 

*_GIVEN THAT_* I am a user on application details page

*_WHEN_* I clicked on the Process CTA 

*_THEN_* 

*_Scenario 1-  Navigation_* 

* I should navigate to Application processing page and Path at the top shows “Processing Application”

*_Scenario 2 - Content_* 

* Application Number at the top 
* -Righthand panel view shows determination, preferences,  actions, documents and comments --- separate stories.-
** Application content is populated from Manual Entry or submitted Online application (SchoolCafé) 
* Sections are based on Manual Entry/Review step
** Students
** Benefits
** Household Members
** Contact Info
** Details
* Each of the above sections has the ability to edit (edit pencil) which takes the admin back to the corresponding (guided process UI) page to make changes 
* -Application processing UI should have Save and Cancel options- 
** -Save > Saves changes made and brings them back to the processing details page-
** -Cancel > Discards changes And brings them back to the processing details page-
* -*All changes made are tracked and shown as part of the Application details History section-

## Acceptance Criteria
- Multi-language, ADA compliant
- content shows based on submitted application data
- edit pencil takes the admin back to the page to make changes with cancel and save options 
- admin can cancel changes to discard or save to update

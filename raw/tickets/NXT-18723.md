# NXT-18723 — Menu Calendar Report > Weekly

## Description
Once the options have been selected on the page, there are multiple versions of the Weekly Calendar Report. 

The different “versions” are based on the options chosen, and are impacted by the selection(s) of:  

# Show Summary Items Only
# Include Weekends
# Show Menu Name
# Include Nutrients

If “Show Summary Items Only” is selected - then there will be only the items displayed that are marked as “Show in Summary” on the item’s “Item Management” page.

If “Include Weekends” is selected, then Sunday and Saturday will appear on the page.

If “Show Menu Name” is selected, then the calendar will display the menu items on each date, AND the Menu Name.

If “Include Nutrients” is selected, then the carbs will show up on the weekly & monthly list, and the carbs and calories will appear on the daily report.

If there is a date with no menu, that date will simply have no information in it other than the date.

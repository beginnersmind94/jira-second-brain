# NXT-2795 — Inspection Template - Template Settings

## Description
*User Story*
As an Inspections user,  determine basic settings when creating or editing a template and save that template so it can be used for future Inspections.

*Mockup*
[https://m5uloa.axshare.com/#id=usu4ze&p=add_new_questionnaire&view=default&g=1|https://m5uloa.axshare.com/#id=usu4ze&p=add_new_questionnaire&view=default&g=1]

*Requirements*

* User can set scoring parameters
** Custom Scoring
*** Select from Lower Bound to Upper Bound (Default) or Upper Bound to Lower Bound
*** Add a custom status with Status, Lower Bound, Upper Bound, and color
**** Lower Bound and Upper Bound can be left blank to indicate it is the lowest or highest score, respectively
**** Color is selected from a selection of colors
***** Colors to be determined by UI/UX
*** There is no hard limit to number of statuses a user can set
** Pass/Fail Scoring
*** User sets the percentage of passes required for a Pass (positive whole number)
** None
*** No scoring settings present
* User can set some toggle options
** Require Site Rep Signature (enabled by default)
** Show scoring by category
* Note: For the purposes of this story, we only need to save the value of the toggle options, no actual functionality for any of them outside of this
* User has a save button that creates the template or saves changes to an existing template

## Acceptance Criteria
* Settings can be selected/filled out when creating or editing a Template
* Template settings are properly saved when creating or editing

NOTE: Color wheel can be done at a later time

# NXT-8886 — System - Create System Homepage

## Description
As a PrimeroEdge user, I would like to have a homepage within the System module that briefly describes the purpose of the module and links me to helpful information, so that I can more easily begin learning the module.

*Summary*

Add a System ‘Homepage’ that will be the default landing page when coming from the My Modules page as well as the landing page for when the ‘System’ breadcrumbs link is selected.

*Details*

* Within the System module, when a user navigates to /system from any location, redirect them to a *System homepage*.
* On the page, display: 
*# a short list of helpful information regarding the purpose of the module
*# a button that redirects the user to the System module user guides/FAQs/walkthrough videos/etc. (functionality will be handled in a future story after such a library has been built in phase 2)
* When a user selects the ‘System’ link in their breadcrumbs, redirect them back to this page as well.

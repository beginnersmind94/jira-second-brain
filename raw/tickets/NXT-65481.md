# NXT-65481 — Income Statement Pt.3 Cost of Goods Sold

## Description
*As a* district finance admin
*I need* a monthly Cost of Sales report by inventory valuation group (e.g., Main Dish, Dairy)
*So that* I can see Beginning Inventory, Purchases, Transfers, Ending Inventory, and *COGS* and reconcile to the GL

!image-20250811-210058.png|width=682,height=262,alt="image-20250811-210058.png"!

## Acceptance Criteria
*Columns to Display*

* Beginning Inventory
* Purchases
* Transfers (net in/out)
* Ending Inventory
* Cost of Goods Sold (COGS)

*Calculation*

* COGS = Beginning Inventory + Purchases + Transfers − Ending Inventory

*Category Breakdown*

* Show results by inventory category (e.g., Main Dish, Dairy, Bread, Supplies).
* Provide a total line for the selected scope (district, area, or site).

*Period Controls*

* Values reflect the fiscal month and year selected in the UI.
* Report results respect the user’s scope selection (District or Site).

*Functional Notes*

* COGS values also flow through to the Profit & Loss report under the *Expenses section* as *Cost of Sales*.
* This report reconciles to the General Ledger, since it uses the same ledger postings that drive financial statements.
* If the selected parameters return no data, generate an empty report with the columns

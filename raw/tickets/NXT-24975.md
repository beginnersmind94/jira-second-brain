# NXT-24975 —  IM- Edit a Menu Item

## Description
As a Menu Planner, I would like to be able to Edit a new menu Item from Menu Item Management Page

## Acceptance Criteria
*Entry Point*

* Menu Item → Menu Items Landing Page → Click on Menu Item→ Edit Button

*Edit Menu Item*

* A page with the following fields in is provided to the user (field content is blank)
* Menu Item creation should consist of:
** Menu Item Image (area for Image)
** Select Menu Item Image 
** Delete Menu Item Image 
** Menu Item Name (Free Text Box)( #Mandatory)
** Marketing Name  (Free Text Box)
** Marketing Description  (Free Text Box)
** Menu Item Category (Drop Down)( #Mandatory)
** Leftover Category (Default to Unspecified) 
** Maximum Carryover Days ( # Mandatory if a Left Over Category other than Unspecified is selected)
** Serving Size (Box) ( #Mandatory ) 
** Serving Measure (Box) ( #Mandatory )
** Preparation Site Item (Check box)
** Show in Summary (Check box)
** Favorite (Check box)
** *Menu Item - Recipes (This section is not mandatory but if Menu Item recipe is added then fields with asterisk/mandatory  need to be populated)*
*** ADD NEW Menu Item - Recipe Link
*** Recipes/Menu Items (Smart Search field) (Search by Recipe Name, Code, Menu Item Name, Item Flag as Menu Item )
*** Primary quantity (Numeric field) (similar to Serving measures on Item) (# Mandatory if Recipe is added) (Read Only)
*** Primary Unit (Drop down selection box) (similar to Serving measures on Item) (# Mandatory if Recipe is added) (Read Only)
*** Secondary quantity (Numeric field) (similar to Serving measures on Item) (Read Only)
*** Secondary Unit (Drop down selection box) (similar to Serving measures on Item) (Read Only)
*** Directions (Directions taken from recipes)
*** Save Menu Item Recipe Link
** 
* *Actions*
** Select Menu Item Image
*** Location/File selection option should be provided.  Only 1 image can be loaded
*** Not mandatory
** Delete Menu Item Image 
*** Delete Menu Item Image
** ADD NEW Menu Item - Recipe Link
*** Add a row so a new Menu Item - Recipe link can be entered 
** Save Menu Item Recipe Link
*** Save Menu Item Recipe Link
*** Displays the newly add Menu Item - Recipe Link information
** Delete Menu Item Recipe Link
*** Deletes Menu Item Recipe Link
*** Removes selected  Menu Item Recipe link from display
** Save
*** When pressed or clicked the Menu Item is save/created
** Cancel 
*** When press/click then all creation is disregarded (No menu item is created)
*** Page closes and returns to Menu Item Landing Page.



*General:*

* Menu Item Created will be assigned a Recipe Code as master code
* Menu Item will be flag as Menu Item
* Only Local data source is used

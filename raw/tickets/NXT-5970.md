# NXT-5970 — Lazy Loading in the Grid

## Description
*NEW Requirements:*

* Implement lazy loading in the P2Grid Shared Component as optional feature.
* Scrollbars should be inside grid and always in the viewport.
* Zeplin to be updated to support this View.
* Implement Lazy Loading in Patrons.
* *Mockup:* [https://app.axure.cloud/app/project/nqyc9v/preview/tj8d55|https://app.axure.cloud/app/project/nqyc9v/preview/tj8d55|smart-link] 
* *Zeplin:* TBD

*Original Bug reported:*
Steps to reproduce:
1. Login to Primeroedge >ACM > Patrons
2. Enter search string (Say 'test') - Results are displayed in split view
3. Navigate to table view
4. Try to scroll horizontally to see the data in columns towards right.

Expected result:
Horizontal scroll bar should freeze at the bottom of the page and should allow user to scroll horizontally at any instant

Actual result:
Horizontal scroll bar present and displayed below the last record only. 
For the pages before last page, there is no way to scroll horizontally and view the data in the columns towards right except removing the columns from settings.

# NXT-7262 — INV - Physical Inventory - Supervisor Switch Option - UI

## Description
As a district manager or supervisor I want a Physical Inventory summary view page so I can easily switch through site physical inventories and view them.

Users need a simpler way of switching through multiple site inventories without returning to the search/list page and opening each inventory.

*Requirements*

* Based on the results of the ‘View/Search Physical Inventories’ page, when the user opens a selected inventory, provide Previous and Next buttons that will allow the user to switch through the site inventories. 
* At the top of the screen where the current inventory’s site name is displayed, provide a “Previous” and “Next” button -that will also display the Site Name, based on the sites that were loaded on the PI search screen.-
* This should be added to each status screen on the Summary tab.
** Previous/Next button will be displayed only on the Summary tab of each screen
* This should be tied to a permission.

If the user navigates to a ‘Physical Incomplete’ screen by selecting an existing inventory from the ‘New’ tab, do *not* display the previous/next buttons.

## Acceptance Criteria
Users with corresponding permission will see Previous/Next buttons on the selected Physical Inventory screen
Previous/Next buttons will appear based on the results of the Search Physical Inventory screen
Previous/Next buttons will appear only on the Summary tab of each screen
Previous/Next buttons will not appear if the Physical Incomplete screen is opened via the 'New Physical Inventory' tab

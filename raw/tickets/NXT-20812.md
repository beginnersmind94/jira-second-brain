# NXT-20812 — INV - Orders - Receipts - Provide option to receive other pack size units

## Description
As a Site Manager I want the ability to receive units in an item that I may not be configured to order, so I can receive items exactly as they are delivered, even in the case of damage or mistakes.

*Requirements*

* Allow other Pack Size units to be inserted on the receive screen 
** Ideally there should be a setting

## Acceptance Criteria
* Sites should be able to receive items in other pack size units than the ones they are configured to order
* District should have a setting to control

# NXT-5286 — [TaggedReport] - Eligibility - Reports - Eligibility Summary

## Description
As a central office user I want to see an eligibility summary for my school district to better understand how eligibility is distributed among my student population.

*Summary View*

This view displays a grouping of eligibility status by reason with an aggregation of the totals.

## Acceptance Criteria
User has a Reports category in the side menus.
User has a Eligibility page available.
User has a Summary option on the page.

User can select a date to pull student eligibility information for.

User can see an accurate count of active students - their student population.
Count is broken down by site.
Count aggregates totals for each column + row.

User can toggle whether they are seeing counts or percentages.

Counts can be broken down by additional options:
- Eligibility (Free/Reduced/Paid) by default.
- Eligibility Reason.

User can export the results to PDF or Excel.

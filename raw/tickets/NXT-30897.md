# NXT-30897 — PII - Protect Sensitive Fields on Exports

## Description
When a user exports files, the export process places PII data into the exports which ultimately are visible to a user.  As such, the PII data within these exports need to be protected.

## Acceptance Criteria
* PII data within the exports shall be visible only to those who are a member of a role with PII visibility enabled.
* The values for these fields shall appear as “*”.

# NXT-16354 — INV - Orders - Order Detail - Edit Order - Delivery Location and Comments

## Description
*Requirements*

* Add Delivery Location as a field in the header
** display under Site
** only if the site has delivery locations configured/enabled (not all sites use Delivery Locations)
* Provide an Edit icon next to the Delivery Location field
** only if user has Edit Order permission, AND
** only if order status is Open or Pending Review
* When Edit icon is clicked:
** enable Delivery Location dropdown 
*** dropdown will contain all locations configured for that site
** replace Edit icon with green checkmark (save) and red X (cancel)
* Add Comments icon to the header
** Comments icon will open the popup containing any order comments
** comments can be updated and saved as long as Order is not Closed

## Acceptance Criteria
* Delivery Location is displayed in the header as a label
* Delivery Location can be edited with appropriate status and user permission
* Comments icon is displayed in the header

# NXT-22304 — Sync Photo Import data

## Description
Currently during photo imports, we are uploading photos. However there is no way that these photos are also added to Eligibility DB.

This story is to cover the tasks related to syncing of the photos in Eligibility DB with the photos imported via photo import. The photos will be added to Eligibility DB by sharing the path of photos added already (in photo import).

## Acceptance Criteria
1. Verify and Validate by viewing the uploaded image @ Account Management-Student Profile.

2. Verify and Validate that the image uploaded via photo import and the photo is Account Management-Student Profile is same.

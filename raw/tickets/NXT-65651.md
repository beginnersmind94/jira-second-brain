# NXT-65651 — GC Portal - Settings in SC

## Description
_(none)_

## Acceptance Criteria
Navigation: SC - System - Configuration - Settings

Changes Required:

Config Type: Internal

Module: Item Management

Category: Configuration

A new Internal Configuration should be added as “Enable GC Portal”, Default Value as 0. 

Upon Toggling as Enabled, the “LOGON - GC Portal” button should be visible to user. Upon clicking the button, the GC Portal should be launched.

Upon Toggling as Disabled, the “LOGON - GC Portal” button should NOT be visible to user.

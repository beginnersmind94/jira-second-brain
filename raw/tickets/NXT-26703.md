# NXT-26703 — Platform - Generate Periods - Enable Fiscal Year Definition

## Description
In PrimeroEdge, users have the ability to define the start date and end dates for a fiscal year.  We would like the same to be available in SchoolCafe.

See below for how the UI appears in PrimeroEdge.

!image-20220209-225526.png|width=883,height=141!

## Acceptance Criteria
* As a user, I would like the ability to define the start and end dates for a fiscal year.
* User shall be able to select the start date and end date for any academic year available in the drop-down
* User shall be able to set the dates and click on “Set Fiscal Year Schedule” button
* System shall generates periods within the range of the fiscal year dates configured for that specific academic year.
* Validation - Date range shall fall between 06/01/XXXX - 05/31/XXXX
* System shall not be able to change the dates after generating periods for any fiscal year

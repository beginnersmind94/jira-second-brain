# NXT-4988 — AcctMgt - Add Patrons - Students - Wizard Step 5 - Allergens

## Description
*User Story*

As a district admin I want to fill out allergen information for a student patron via a wizard.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418434%3A238826&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418434%3A238826&scaling=scale-down-width|smart-link] 

*Requirements*

* User needs a way to add allergens while adding a student.
** User should be able to select from any of the big allergens, and from any of the additional allergens that have been added.
** Show the big allergens first if we’re doing a dropdown for this.
** Accounts can have multiple allergens.
* User needs a way to dismiss a selected allergen.
* Allergens should be saved appropriately if/when the account is created.
* Either the user needs to add one Standard or Custom Allergen *OR* they need to check the ‘No allergens indicated’ box in order to continue. If we need to display an error, it can say, ‘Please add at least one allergen, or indicate that no allergens have been made known for this patron to continue.’

## Acceptance Criteria
Allergen information can be entered.
Information is saved while navigating the wizard.
Information is saved appropriately.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)
Accessibility / ADA Compliance programmed appropriately (full KB nav possible, etc.)

User must select the 'No allergens indicated' checkbox or add at least one allergen to continue.

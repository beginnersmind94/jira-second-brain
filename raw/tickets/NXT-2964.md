# NXT-2964 — ** System - Organization - Sites - EDIT Site Details

## Description
As an administrator I want to edit site details, characteristics, and settings so that each site is configured to properly work with each module within the PrimeroEdge application.

PrimeroEdge Classic Reference: System > Sites and Users > Sites > Select a Site

When user selects the edit button/icon, the View mode should change to the Edit mode and all “readable” displays should change to editable controls.

Mockup: [https://nqyc9v.axshare.com/#id=xajrt8&p=sites&g=1|https://nqyc9v.axshare.com/#id=xajrt8&p=sites&g=1]

Note: Module settings will be addressed in a separate story and can be ignored for now.

## Acceptance Criteria
1. Users with permissions for editing should be able to see/click "Edit"
2. Users should be able to modify fields and save changes
3. Users should be able to cancel changes
3a. when the user cancels out a confirmation message needs to be displayed whether to cancel or not " Daimien / Karthik to provide the message before the sprint begins"
3b. if the user chooses to not cancel bring them back to the edit page
3c. if the user chooses to cancel - exit out without saving any changes
4. Form should be in 'View mode' when the drawer is initially opened. Edit pencils should be selected individual in order to modify data in any of the sections.

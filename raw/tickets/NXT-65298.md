# NXT-65298 — ACC - Labor Hours Feature - MPLH Report

## Description
When the MPLH button is clicked a report is ran for the selected date range and site filters.
For Date control
For Both Standard and Precise
user can only select academic year and period


 Columns:
{color:#4c9aff}*Site code,* {color}

{color:#4c9aff}*Site Name,* {color}

{color:#4c9aff}*Breakfast Meal count*{color}

{color:#4c9aff}*Lunch meal count*{color}

{color:#4c9aff}*Snack Meal count*{color}

{color:#4c9aff}*Adult meal count*{color}

{color:#4c9aff}*A La Carte Sales total*{color}

{color:#4c9aff}*Meal Equivalents*{color} = {color:#ff5630}SUM{color} of

  {color:#ff991f}Student Breakfast Equivalents {color}=Student Breakfast Count {color:#ff5630}***{color} Breakfast Conversion Factor
     {color:#ff5630}*+*{color}

{color:#ff991f}  Student Lunch Equivalents{color} = Student Lunch Count {color:#ff5630}***{color} Lunch Conversion Factor
     {color:#ff5630}*+*{color}

{color:#ff991f}  Student Snack Equivalents{color} = Student Snack count {color:#ff5630}***{color} After School Snack Conversion Factor
     {color:#ff5630}*+*{color}

{color:#ff991f}  Adult Meal Equivalents{color} = {color:#6554c0}({color}Adult Breakfast Count {color:#ff5630}***{color} Adult Breakfast Conversion Factor{color:#6554c0}){color} {color:#ff5630}*+*{color} {color:#6554c0}({color}Adult Lunch Count {color:#ff5630}***{color} Adult Lunch Conversion Factor{color:#6554c0}){color}
     {color:#ff5630}*+*{color}
  {color:#ff991f}Non Program Foods Meal Equivalents {color}= (A La Carte $Total {color:#ff5630}*/*{color} Non Program Food Sales Conversion Factor)

{color:#4c9aff}Labor Hours{color} = Sum of All Labor Hours entered in the Period

{color:#4c9aff}MPLH(Actual){color} = Meal Equivalents / Labor Hours (000.00)

# NXT-20228 — Edit an existing data query report

## Description
As a user I need a way to edit an existing data query report and save it.

## Acceptance Criteria
Acceptance criteria will be met when:
1. When I am on the find reports screen I should be able to click on the me ellipses button(…) and select edit option from there to on open the selected report in edit mode.
2.  I should be able to go though the various steps on the stepper and make any necessary updates. 

3.  Finally when I click the save button, I should be sent to the find reports screen with a success notification.

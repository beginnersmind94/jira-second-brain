# NXT-3979 — AcctMgt - Patron Split View - Guardians

## Description
*User Story*

As a district administrator I want to see a patron’s guardian information.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=1]

*Requirements*

* Add a ‘Guardian and Contacts’ panel to the Patron Details section. There will be multiple guardians per account, and the primary guardian should be the first shown.
** Sorting order: Primary first, then secondary guardians by Last Name, First Name.
* Show the following data in the list:
** Name (Prefix/Title, First, Last, Middle, Suffix)
** Phone #
*** Home Phone
*** Work Phone
*** Cell Phone
**** Use symbols to indicate each.
**** We need alt-text for the symbols, and a mouseover tooltip for ADA compliance.
** Full Address
*** Street Address
*** Apt./Suite
*** Zip, ZIP+4
*** State
*** City
** Indicate the preferred communication method with a green tick. Alt-text and mouseover on that.
*** ‘Preferred Communication’
** Email (2 entries possible)
** Primary Guardian chip
** Relationship (Parent, Guardian, Mother/Father, Relative, Self, Other)
** SchoolCafe user/account (Placeholder column)
** Contact info from SchoolCafe (Placeholder column)
* Add an ‘Edit’ mode for the panel.
** Add the ability to edit or delete the guardian information in-line for each entry on the Guardian Information grid.
*** User should be able to make changes to any field, including whether they are the primary or secondary guardian.
*** Just 1 primary contact is possible.
*** If the user is deleting the primary guardian, block the user.
**** ‘You cannot delete the primary contact. Please assign another primary contact before removing this one.’
*** If the user is deleting a secondary guardian, confirmation box.
**** Remove this guardian?
***** Options: Cancel / Remove
** Add a ‘Add +’ button to the Guardian Information panel.
*** This should allow the user to add a guardian with all fields available.
*** Only 1 Primary guardian is possible. If this is selected and a guardian is added, any existing primary guardian should be moved to a secondary guardian.
**** Let’s have a confirmation box if the user is replacing the primary guardian.
***** Replace primary guardian?
****** Options: Cancel / Replace

* -We need to track all changes that are made.-
** -Timestamp of the change, field, previous value, new value, who did it.-

## Acceptance Criteria
Guardian panel added.
Edit mode for the panel added.
Can add/edit/delete guardian info.

REMOVE: History of changes tracked.

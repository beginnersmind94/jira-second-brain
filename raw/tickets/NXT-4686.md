# NXT-4686 — Item Management - Table View Layout

## Description
*User Story*

As a user, I want to be able to see a list of standard fields here with the same results as split view

*Mockup*

 

!image-20200730-132841.png|width=100%!



*Requirements*

* Fields in this order
** Name
** Short Name
** Item Code
** Product Code
** GTIN
** Datasource
** Status
* The results will be alphabetically sortable from either A-Z or Z-A by clicking on the column header
* Default will be results of your search, however prioritize name from A-Z
* If numbers and letters, sort by string
* This sorting will just be in the Table view only, sorting may come in the future for split view

## Acceptance Criteria
* The user will be able to view columns in Table view with the same results as in split view

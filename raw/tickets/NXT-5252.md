## Acceptance Criteria
User has a By Application option on the Eligibility reports page.

User can select a date to see information as of that particular date. Default is current date.

User can see a count of applications that have come in for the selected year.
Count is broken down by eligibility determination (free/reduced/denied) and reason (income, categorical, etc.)

User can toggle between seeing counts of applications and counts of students.

User can toggle whether they are seeing counts or percentages.

User can export the results to PDF or Excel.



Application Totals should look for the following:

* Apps approved with a reason of Income, and the individual categorical reasons: SNAP, TANF, FDPIR, Foster.
* When looking at the student eligibility totals from applications, we’ll want to check for the following:
** Income
** Categorical (split into: SNAP, TANF, FDPIR)
** Homeless
** Migrant
** Runaway
** Foster
** Head Start
** Even Start

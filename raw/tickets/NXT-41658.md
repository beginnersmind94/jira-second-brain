# NXT-41658 — IM - Item Onboarding - Review Contributions

## Description
As an onboarding district admin I want a step to review Contributions so I can pre-configure important menu planning information at the time of bulk importing items.



!IO - Contributions View-20241211-154803.png|width=91.66666666666666%,alt="IO - Contributions View-20241211-154803.png"!

## Acceptance Criteria
* Provide a review step on the Bulk tool for the Contributions 
* In view mode, display the food component icon for each configured contribution for each item
** next to the icon display the value and any extra configurations like Juice
* For Grains, follow the new icon conventions for WGR, Non-WGR, and GBD
** WGR - this one is first and should be larger
** Non-WGR - uses gray outline when enabled, with 70% opacity orange, and is smaller than the WGR icon
** GBD - uses orange outline with white fill, new cookie icon,
* For Milk, show the variety underneath the value
** use shorthand values in some cases, like “FF” instead of “fat free” (check with POs)
* For Vegetables, add a row for each applicable sub-group
** this may mean increasing row height for these records only
* Display an alert icon in the Item # column if any enabled components are missing a contribution value
** Display ‘{color:#ff5630}_Missing_{color}_’_ next to each component with a missing contribution value
** Items with missing contribution values cannot be published (or at least cannot be published with the component)
* Provide a ‘[Select Components|https://primeroedgenext.atlassian.net/browse/NXT-57377]’ button to navigate to the 1st of the 2-step edit mode
* Provide an ‘[Enter Contribution Values|https://primeroedgenext.atlassian.net/browse/NXT-57382]' for the 2nd step button

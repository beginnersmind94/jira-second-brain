# NXT-19259 — INV - Order Detail - Receipts tab - Remove Broken Units

## Description
As a Site Manager I want the Receipts tab on Order Detail screen to not include broken units so I can see accurate item details when looking at the receipt summary or details.

*Requirements*

* +On Order Detail screen - Receipts tab+
* Remove Broken Units from the 3 quantity fields (Ordered, Received, Due)
** remove label text in parentheses ‘(Whole, Broken)’
** Update to remove any references to Broken Units, as Broken Units will no longer exist
** for the unit label (e.g. Basket, Bag, etc.), make sure this is referencing the new ‘Unit Description’ free text value (not the ‘Whole / Purchase Unit’ dropdown value which INV1 is removing)
* Add a ‘Product Type’ column
** This will display one of 6 options: Pallet, Mixed Pallet, Display Shipper, *Case, Package, Each* (bolded ones are most common)
*** This will be pulled from the details of the item in Item Management (See image below)
** display this column after Item Description column, before Vendor
* Group related items together in the grid
** Items which are linked together from Item Hierarchy in Item Management should be displayed as consecutive rows
** Sort the *related* items by Product type descending (Pallet > Case > Package > Each)
* Remove ‘Storage Category’  column
* Change ‘Item Name’ column to ‘Item Description’

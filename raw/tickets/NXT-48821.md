# NXT-48821 — Temp User - User Profile Settings

## Description
As a SchoolCafe user, I would like to be able to see my user type info in my profile so that I can see how long it has before expiration

*Requirements*:

* On the Workspace > My Profile page, do the following:

Changes to the Basic Info tab:

* In the security section:
** Add Account Type
** Show Start Date if Account Type is Temporary
** Show End Date if Account Type is Temporary



Changes to the My Preferences tab:

* On the Authentication tab, show these fields:
** Note: users cannot change their own account type or dates in User Profile
* Fields:
** Account Type
*** Read-only
*** Values: Permanent, Temporary
** Start Date
*** Read-only
*** Value: Temporary Start Date field
*** Visible if Temporary
** End Date
*** Read-only
*** Value: Temporary End Date field
*** Visible if Temporary

## Acceptance Criteria
* Account Type field is shown
* Start and end date show only of account type = Temporary

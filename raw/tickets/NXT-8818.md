# NXT-8818 — INV - Contract Items - Ordering Conditions - View Item Site Detail

## Description
As a District Supervisor I want a detailed item ordering conditions slide-out view on the Contract Items page so I can see the site specific ordering conditions for an item.

_Inventory > Vendors > Contracts_

*Requirements*

On the Contract Items Ordering Conditions tab ([NXT-5648|https://primeroedgenext.atlassian.net/browse/NXT-5648]) provide a button on each item row to view the Item Site Details so the user can see the specific site ordering condition breakdown for the selected item.

When the user clicks the button, a right-side slide-out will appear displaying the list of sites and which are configured to order the item.

+Site List:+

* sorted by Site Type
** a checkbox for each Site Type which will allow the user to bulk select all the Sites within that Site Type
** Site Types are collapsible/expandable 
*** Site Types are all collapsed by default
*** only one Site Type at a time can be expanded
* Site Code - Site Name
** a checkbox for each Site, which will allow the user to select or deselect that Site for the selected item
** checking the Site checkbox will indicate that the Site will be ordering the item from the selected contract vendor
** if a Site is currently configured to order that item from a different vendor, provide a warning icon 
*** provide an alert to the user when saving that the Site’s ordering condition configured will be overwritten 
**** display a (!) icon next to each site which is currently ordering the item from a different vendor contract
*** Provide a message (as a label) at the bottom of the slideout
**** “(!) These sites are currently configured to order this item from a different vendor. Selecting these sites will overwrite existing ordering conditions.”
*** This check would only check against contracts that overlap with the selected contract’s time range
* a Save button to save the changes made to the sites.
** saving logic as part of [https://primeroedgenext.atlassian.net/browse/NXT-9477|https://primeroedgenext.atlassian.net/browse/NXT-9477|smart-link]
* a Cancel button to discard changes made

(mockup difference: only one Site Type expanded at a time)

## Acceptance Criteria
On the Contract Items s page, an option to view the site level details for a selected item
The site details will be displayed in a right-side slide-out view
Sites will be grouped by Site Type
User can select (check) or deselect (uncheck) any Site or Site Type
User will be provided with an alert if any existing Ordering Conditions will be overwritten

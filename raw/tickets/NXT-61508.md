## Acceptance Criteria
* Modify the assignment details slideout for non-substitution items added to the distribution
** Ensure all sites assigned to the distribution are always displayed in the slide-out, regardless of Order Quantity
** Put the sites with zero order quantity below the ones who ordered
* Provide a Delete icon next to the Item label 
** Prompt if no assigned quantity: “Are you sure you want to delete this item from the distribution?”
** Prompt if some assigned quantity: “This added item has assigned quantity. Are you sure you want to delete it from the distribution?”
* Remove the Quantity Ordered column
* As sites are assigned quantities, select an Order ID to which these items will be appended on the receipt
** In the event that the new item is assigned to a site which has multiple orders on the Distribution, select an order to which the new item will be added to the receipt

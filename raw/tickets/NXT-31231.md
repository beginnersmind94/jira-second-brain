# NXT-31231 — Special Permission Tab - Update Column Headers and Permissions

## Description
The “Special Permissions” tab needs to be updated per the items mentioned in the acceptance criteria.

!image-20220525-120310.png|width=759,height=477!

## Acceptance Criteria
* The existing “Resource” column header shall be changed to “Permission”
* The existing “Resource Description” column header shall be “Description”
* The tooltip on the “Action” sliders shall be “True” if the slider is enabled.
* The tooltip on the “Action” sliders shall be “False” if the slider is disabled.
* The existing permission named “Device Keys Generate” shall be changed to:
** *Permission*:  “Generate Device Keys”
** *Description:*  “Can generate new device keys for devices being registered on the Device Management page.“
* The existing permission named “Device Suspend” shall be changed to:
** *Permission*:  “Suspend Devices”
** *Description:*  “Can suspend active devices that have been registered on the Device Management page.“
* The existing permission named “Export Execute” shall be changed to:
** *Permission*:  “Perform Exports”
** *Description:*  “Can perform exports of data on the Export Dashboard page.“
* The existing permission named “Import Execute” shall be changed to:
** *Permission*:  “Perform Imports”
** *Description:*  “Can perform imports of data on the Import Dashboard page.“

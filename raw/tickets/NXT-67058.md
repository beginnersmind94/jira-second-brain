# NXT-67058 — Align Category Checkboxes Across All Meal Types During Device Linking

## Description
As a district administrator setting up a new device, I want the category checkboxes to be visually aligned across Breakfast, Lunch, and Supper sections, so that the interface appears professional and is easier to scan.

!image-20250926-184643.png|width=250,alt="image-20250926-184643.png"!

## Acceptance Criteria
* *Visual Alignment*: All category checkboxes across Breakfast, Lunch, and Supper sections align vertically along the same left edge with consistent spacing from the container edge
* *Responsive & Functional*: Alignment is maintained across all screen sizes and checkbox selection functionality remains unchanged

# NXT-16956 — Eligibility - Verification - Add option to shift an app from Sampled to Cause or vice-versa

## Description
As Vera the Director I want to make some edits to apps that have been selected via Sample, but I still want the household to complete the process.

*Mockup*

[https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=120%3A6820&scaling=scale-down-width&page-id=120%3A6271&starting-point-node-id=120%3A6574|https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=120%3A6820&scaling=scale-down-width&page-id=120%3A6271&starting-point-node-id=120%3A6574|smart-link] 

*Requirements*

* In Eligibility → Verification → Tracking, for the Pre-Verification Application Information section in Verification Details:

!image-20220414-163603.png|width=1230,height=580!

* Show the current selection basis and the date it was added to verification.
** Selection Method: Sampled (DD/MM/YY)
** Selection Method: For Cause (DD/MM/YY)
* Add an option inside of Verification Details to let the user shift an app from Sampled → Cause or from Cause → Sampled.
** Option should *only* be visible for Incomplete apps.
** Button text:
*** Switch to Sampled
*** Switch to Cause
* In either switch case, display a Confirmation prompt and require a comment.
** In the prompt, because ‘Sampled’ requires a certain number of applications, in the case where they are switching to Cause OR Sampled warn them: “Please note: After hitting Save, the Sample will be recalculated.” 
* Upon saving, display a success message and add the comment/historical information to the History of the app in Verification. And then, recalculate the Sample.
* Add special role permission for this button: ‘Switch Verification selection method’

Additionally, move the Back button to top right and change to secondary color.

## Acceptance Criteria
Add the following:

* Selection basis visible with the selection date.
* Add an option inside of Verification Details to let the user shift an app from Sampled → Cause.
** Or from Cause → Sampled.
* Confirmation prompt and a comment required. Add this information to the Comment history.
* Sample gets recalculated after clicking Save.
* Add the switch information to the History of the app in Verification.
* Add role permissions for this button.
* Back button moved and style changed.

# NXT-40933 — Imports – File Mapping – Delete a File Field - QA

## Description
As a SchoolCafé Admin I want to be able to delete a field in the File Fields so that I can avoid having to create a new import file when I make a change.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Imports
* Click View on a Student Import record to view the import details
* Go to the File Mapping tab
* Click delete button on a record in the list of fields
* After confirming, the system deletes the field from the list

*Requirements*:

* On the File Mapping tab of a student import file, do the following:
* Add a button in the rows of the File Fields table
** Title: Delete
* On click:
** Ask for the usual confirmation
** If Yes, remove the field from the File Fields table
** If it is linked to a mapped field, unlink it
* 
* After deleting, everything continues to work as normal and that field is not included in the mapped fields and is not imported

## Acceptance Criteria
* Can deleted a row from the File Fields
* If it is linked to a mapped field, it unlinks
* Files with this field are not able to import into the database

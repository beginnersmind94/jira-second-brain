# NXT-4690 — Item Management - Search Recipes

## Description
*User Story*

As a user, I want to be able to search for recipes from the view screen

*Requirements*

* Add a simple search to search by ‘Recipe Name’ and ‘Recipe ID’ which is the Recipe Code
* Prioritize exact match over a partial match
* Prioritize local, shared, then global datasources
* Sorting default A-Z - Recipe Name
* Sorting Order for Allergens
** Contains A-Z
** Processed in the facility… A-Z
** May contain A-Z

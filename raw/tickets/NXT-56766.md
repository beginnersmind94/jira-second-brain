# NXT-56766 — MP- Menu/Menu Template- Do not allow OVS checkbox to check if No count for Serving

## Description
As a menu Planner,  I should not be able to add OVS to a serving group that I have not given a count for.

## Acceptance Criteria
* If not count given, Grey checkbox until count is given
* If count is given, then check box is clicked, then count is removed, disregard the checkbox entry
* Apply on Standard Menu and Menu Template

# NXT-43063 — AcctMgt/POS - Handling of Homerooms / Rosters with 0 students

## Description
As Cassie the Cashier I want to only see active homerooms/rosters (Groups) at the POS to save time during summary/bulk sales.

*requirements*

* POS
** Don’t show rosters/homerooms (groups) with 0 students in them.
** Don’t show inactive rosters/homerooms.
** If a group has students in it, but they are all inactive, only show the roster if the POS Setting ‘Serve Inactive Students’ is set to Yes.
*  System
** Add a system setting to handle inactivating homerooms that have 0 students in them automatically.
*** Name: Inactivate Homerooms with 0 students
*** Description: This setting allows the system to automatically set homerooms to Inactive if there are 0 students in them.  
*** Options: Yes, No (Default)
** Add a nightly job to check for homerooms with 0 students and make those homerooms inactive.
* Imports
** If a student is added/imported into an inactive homeroom, make that homeroom active again.

## Acceptance Criteria
POS

* Groups with 0 students in them are not available for selection.
* Inactive groups are not available for selection.
* Groups with inactive students only are available if the POS Setting ‘Serve Inactive Students’ is set to Yes for that site.

System

* System setting added.
* Nightly job deactivates homerooms if the setting is set to yes.

Imports

* Homerooms are reactivated if a student is imported/moved into an inactive homeroom.

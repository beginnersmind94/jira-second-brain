# NXT-25782 — System - Periods - AUDIT actions for Manage Periods

## Description
As a food service director or site manager I want a page for Inventory Periods so I can close and manage periods in accordance with our operations.

*Requirements*

* NEW AUDIT action is enabled within Actions ellipsis. 
* When user clicks on AUDIT, periods audit page is displayed with timestamps, user and Description and NOTES Column
* All notes for Re-open and Close periods needs to be captured

## Acceptance Criteria
* User shall be able to click on AUDIT action button to view the AUDIT history of the period
* If the periods audit is empty, deactivate the AUDIT button so user shall not be redirected unnecessarily
* User shall include Notes tab.

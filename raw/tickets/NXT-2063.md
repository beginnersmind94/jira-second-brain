# NXT-2063 — Production - Production Record - Meal Counts From POS (SPIKE)

## Description
Production Record should pull meal counts (reimbursable) from the POS systems that have completed meal service. Should do a check that all sessions are closed out for the day regarding meal service for accurate counts.

Note: Inventory will pull the counts for A La Carte (non-reimbursable) items from the POS systems that have completed meal service. 

Discussion/Brainstorming: Make a determination if the POS system should trigger (send) this information to update Production and Inventory, if the process should be automated, or if it should be a pull from the other modules. We may do a combination approach, but first preference is that the POS push the information after completion as this is typically the workflow.  (POS is designed to push the data to PrimeroEdge as it is… so it’s just a matter of making the data accessible from the Production Record screen.)

## Acceptance Criteria
From the page of the Production Record, there should be a method of accessing the reimbursable meal counts from the POS. If the site has multiple POS machines, the Site Manager should be able to select one, multiples, or all of the POS machine counts to apply to the Production Record that they are looking at. (Many sites have more than one POS machine to ring up meals, and may also have multiple production records to handle different menu lines - so they need to select which POS machines might have contributed the Meal Counts for the Production Record that they are currently working with.)

Once the user has selected the POS Machine(s) that apply to the Production Record page that they’re on, there should also be a way for the person to choose to have those selected counts populate into the meal count area on the Production Record page.

In the event that the user has already entered some of the numbers onto the Production Record Page, there is a checkbox that they must mark in order to tell the software to overwrite what they’ve entered. They must mark that box for the button to “Add POS Counts” to become available.

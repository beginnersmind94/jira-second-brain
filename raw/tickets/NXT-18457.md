# NXT-18457 — Production- Reports- Carryovers (UI)

## Description
This report is to display all carry overs that occur in the production record that users enter at the end of the day.

Page Display needs to have this criteria:
Site Code
Site
Site Group
Meal Type
*From* and to *Date*

Current Primeroedge Display: 
 !image-2021-08-26-12-46-10-031.png|thumbnail! 

Information that needs to display on the report:
Category
Menu Item Name (Recipe #)
Amount (Serving Size)
Days
On menu (whether it is on the Menu)
and display any comments they put on the production record

The report also needs to separate different from and to dates as well as different meal types and site group 

For Example: SchoolCafe Highschool - From Date 8/25/2021- to Date 8/26/2021 - Lunch- Highschool would display all items going to that day before creating a separate line for the next day From Date 8/25/2021- to Date 8/27/2021 - Lunch- Highschool

# NXT-20443 — Platform - Security - Create Custom Role Implementation

## Description
As a SchoolCafe admin, I would like to add new custom role specific to customize the modules and its permissions to various resources.

+*Requirements:*+

* Add “+ADD NEW ROLE” button on top right position in Roles & Permissions page
* New custom role button triggers “Copy” option of *Cybersoft* [*Admin*|https://primeroedgenext.atlassian.net/browse/NXT-9677] role.
* New custom role option has option to enable/disable “Modules” to be enabled for this role
* New Custom Role page must have Close and Save buttons
* Default VIEW permissions are enabled for all resources under each selected module
* Create buttons must prompt for confirmation prompts for any actions

## Acceptance Criteria
* User shall be able to create custom role by using “+ADD” option
* User shall be able to select modules and name of the new role.
* User shall be able to role with default view permissions enabled for each module
* User shall be able to use CREATE buttons with confirmation prompts
* User shall be able to see created user, date and action (created new role) in Audit history tab

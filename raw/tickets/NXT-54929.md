# NXT-54929 — MP- Find and Replace- Do not allow locked/checked out menus to be effected

## Description
As a menu Planner, If I am using the Find and Replace feature, do not allow me to accidently make changes to menu/menus that are currently locked/checked out by another user.

## Acceptance Criteria
* Do not allow Menus that are locked/checked out to be affected by find and replace
* Display a lock symbol next to the menu name (Just like we show the warning symbol for record complete menus) and gray out the line and check box
** Display tool tip on the lock “Menu is checked out by User Name”

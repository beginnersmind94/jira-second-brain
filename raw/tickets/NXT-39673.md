## Acceptance Criteria
* Update VMI nutrition calculation to consider Item Master structure (API effort)
* Calculate an average of all added menu items to determine the values for VMI Nutrient Info tab
** Calculate values per 1 gram so the VMI serving size can be averaged
* Scale the nutritional calculated values according to the gram weight of the VMI serving size
* Assign a System Calculated label to the *Nutrient Details* card
** this label is displayed as long as the Nutrients is using the system generated values
** Assign a Manually Entered label for any user provided values which override the system calculated values
** Make the ‘Manually Entered’ label clickable
*** in view mode as well as edit
** Upon clicking the Manually Entered label, display a prompt which allows the user to replace the values with the system calculated
* Assign the same System Calculated label to the header space of the *Allergens* card
** Same logic as Nutrient Details card
** Make the Manually Entered label clickable
** Show the prompt to user to replace manual values with system calculated
*** Perhaps use an in-line UX approach here instead of popup confirmation prompt

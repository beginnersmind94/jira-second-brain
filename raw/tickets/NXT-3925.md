# NXT-3925 — System - Users - User List - Show optional column for last login date

## Description
As an administrator, I would like to be able to view a list of all users along with their last login date, so that I can see those users who are active or have never logged in and can act accordingly.



Add a column to this page that displays the user’s last login date. May be one of the optional filters.

## Acceptance Criteria
Verify that when going to the Users grid, a column can be displayed that shows all users' last login date.

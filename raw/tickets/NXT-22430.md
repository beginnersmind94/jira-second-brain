# NXT-22430 — INV - Additions - Addition Details - History tab - Transaction Details - Item Sub Grid Color

## Description
As a Site Manager I want a transaction details page so I can view the specific details of prior versions of my Inventory Addition.

*Requirement*

* Display the changed values in red text
** For any fields which were changed, display in red text
** Compare this transaction with the immediately prior transaction
** Attributes which could change:
*** Lot Number sub-grid
**** Date on Product
**** Date
**** Lot Number
**** Quantity
** Highlight the values which changed in the transaction
*** red text/highlight for value changes

# NXT-30751 — User Import - Sync MicroDBs after User Import

## Description
The updates and inserts that occur as a result of the import need to be synchronized with the micro DBs.

## Acceptance Criteria
* After users are created or updated as a result of an import, the changes shall be synchronized across all micro dbs.

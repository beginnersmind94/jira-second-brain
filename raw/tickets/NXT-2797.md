# NXT-2797 — Inspection Template - Add Question Category

## Description
*User Story*
As an Inspections user, I have the ability to add questions and question categories when creating a template.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=template_approved_layout&id=hk2qvp&g=1|https://nqyc9v.axshare.com/start.html#p=template_approved_layout&id=hk2qvp&g=1]

*Requirements*

* Question Categories can be added to any template when being created/edited
* Question Categories are used to divide the questions into different sections
** Categories can be added using a button
** Categories are added at the end of the template
** Once created, Categories can be edited
** Users can set the following with categories;
*** Category Name - Text Field
**** Must be filled out
**** No default value, only instructional text
*** Category Description - Long Text Field
** Categories do not have to be present for a template to be created
** A Template can be saved if a category has no questions
** Category name and description can be edited by clicking an Edit icon
** All category names must be unique within a template
* Questions can be created outside of a category if categories exist

## Acceptance Criteria
* Question Categories can be added to Templates
* Available fields and options can be edited

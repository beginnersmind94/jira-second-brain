# NXT-34656 — Manage Views - Disable Sharing with other districts

## Description
Currently within the “Manage Views” | “Saved Views” tab, other districts are showing up as “Groups” which a user can select to share with.  This should not be allowed.

## Acceptance Criteria
* Users shall not be able to share views with other districts
* Districts shall not display in the “Type Name” dropdown

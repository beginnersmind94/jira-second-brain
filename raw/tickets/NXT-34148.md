# NXT-34148 — Team Hub - View All Messages

## Description
As a school admin, I want to be able to see all inbound and outbound messages and optionally click to see message details so that I can stay up to date with the app and coworkers.

*Flow:*

·         After logging in, user redirected to Home Screen

·         On Home Screen, click the View All Messages Icon and redirected to Messages Screen

·         View all inbound and outbound messages

·         Optionally click on a message in the inbound or the outbound messages list

·         Optionally click to reload all messages

·         Optionally click to create a new message

 

*Assumptions:*

·         User has logged in

·         To complete Messages control, you will have to have completed the push / pull messages stories

 

*Requirements*:

·         Fields:

·         Screen Name: Messages

·         Back – Image button

o   Use same back button as rest of site

o   On click, redirect user to Home Screen

·         Reload – Image button

o   Use a traditional reload icon

o   On click, execute a full push / pull

o   Show a circling or progress graphic to the user to indicate to them that the app is trying to execute the request

o   Show a complete graphic when it is finished for a short period

o   Then refresh the inbound and outbound list boxes

o   If receive an error during the process, show an Reload Failed error message to the user and let them click OK to move past it

·         Compose – Image button

o   Use a traditional writing, composing icon

o   On click, redirect the user to the Send Message screen

·         Inbox – List Box

o   Just like the Messages list box on the Home Screen, just reuse that control, with the only difference being this is only for inbound messages

o   Shows list of inbound messages for the user

o   Show From, date / time, subject and beginning text of message like Outlook and Gmail apps do

§  Text is concatenated to fit on the screen

o   Can scroll through

o   Indicates if read or not read by some kind of visual indicator using State field (Read or Unread)

o   Can click on the message and see entire message by being redirected to the Message Details screen which then loads the selected message

·         Sent – List Box

o   Exactly the same as the Inbound list box, but for outbound messages, just reuse that control and show Sent messages

o   Put Outbound box below Inbound box

·          

·         Inbox and Sent list boxes updated by the Messages Timer

o   The exact same way you update the messages control on the home page, do that here for both the inbox and sent list boxes

·          

·         Security

o   Copy all the security on the Home Screen and do the same for this screen

o   If user not logged in, redirects the user to the login screen

o   User can’t see messages for another user

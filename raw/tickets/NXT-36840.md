# NXT-36840 — IM - Recipe Details - Update Recipe Report (SP)

## Description
As an Item Management user I want the Recipe Report to be updated so I can see the recipe on the report the same way it is on the page.

*Requirements*

* Update the Recipe Report to use PowerBI reporting tool
* Update the report generation slideout to accommodate all 3 report variants
** Add a ‘Include Nutrition’ option that will eliminate the Production variant of report (default: ON)
** Provide a ‘Yield Factor’ toggle as well (default: ON)
** Include a ‘CCP Corrective Actions’ toggle (default: OFF)
** Provide the scaling option on the main report so the separate report can be removed (default: OFF)
*** when enabled, load the Serving Size and # of Servings with default configured values
*** Show a warning about Strict Batching for recipes which have it enabled
**** but allow users to scale however they want for the report
* Update the design of the report to correctly match recipe structure and the updated design

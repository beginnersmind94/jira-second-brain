# NXT-5229 — Application Processing - (Edit) Details

## Description
*User Story* 

Processing application data, I can easily add, edit, remove optional details.

*Mock Ups* 

*Requirements* 

* Details section > Edit Pencil > Takes user to the Details page
** Same functionality as the guided process 
*** Ability to edit details 
**** Ethnicity, racial identify, other benefits, consent
*** Details shown based on configuration/region settings 
* Save and Cancel options 
** Save > Saves changes made and brings them back to the processing page
** Cancel > Discards changes And brings them back to the processing page
* -all saved changes are tracked in the application details \{history} section-

!image-20201201-193228.png|width=567,height=757!

## Acceptance Criteria
- Multi-language, ADA compliant
- edit pencil takes the admin back to the page to make changes 
- admin can cancel changes to discard or save to update 
- details shown based on configurations (other benefits) and settings

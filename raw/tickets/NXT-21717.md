# NXT-21717 — Site Usage Report

## Description
As an administrator, I want to be able to see info regarding the status of terminals for a site so that I can see which need to be updated and which ones might be having troubles

*Application*: Face and Food Detection Website

*Flow:*

* Browse to PrimeroEdge and click the link for Food and Face Detection
* Browse to the Reports
* Browse to Site Usage Report
* View data

 

*Assumptions:*

* It is assumed that you have access to the status of the POS FFD Admin values
** These are the values on the Admin screen: Food Enabled, Face Enabled, Send Results to Cart
** If there is not a setting in a table, then you can get it from the Log table
* It is assumed you are tracking the Model to the sites / terminals
** There are previous stories for this
* It is assumed you have access to the device configuration’s field that shows Learning or Detection

 

*Requirements:*

* Create a new reports page with the title Site Usage
* In the menu navigation, under Reports:
** Create a new menu link under Reports named Site Usage for the new page
*  
* All fields are read only, there are no action buttons except grid behaviors and hyperlinks
*  
* Grid of terminals
** Dataset – show all the terminals linked to an Edge device
*** can get this from device management
** Usual sort, paging, filtering, export to excel functionality
*** default sort site alphabetically, then Terminal alphabetically
** Columns:
*** Site – name
*** Terminal – same as Name / Alias in device management
*** Mode – the value in Device Management that shows Learning or Detection
*** Model – the description field of the model published to the Edge device
*** Model Published – the date the model was published to the Edge device
*** Face Enabled – Yes if enabled, blank if not enabled, comes from POS Admin screen
*** Food Enabled – Yes if enabled, blank if not enabled, comes from POS Admin screen
*** Cart Enabled – Yes if enabled, blank if not enabled, comes from POS Admin screen
*** Date Last Detection – Date (not time) the terminal used the Edge device in detection mode
**** probably can get a max date from the transaction table where has a detection value
*  
* Filters for the grid
** Site
** Terminal
** Mode
** Face Enabled
** Food Enabled
** Cart Enabled
*  
* Security and roles:

o    Copy from Accuracy Report, do exactly as there

* Unauthenticated users cannot view screen

 

*Mockup*:

* Should look the same as Accuracy Report

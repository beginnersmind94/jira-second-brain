# NXT-4639 — [AcctMgt] - ID Cards/Manage Templates - Add support for Pictures + Final ID Card design

## Description
*User Story*

As a District Admin I want to include a student’s photo on their ID card for easier identification.

*Mock-up*

* {color:#6554c0}Not Required{color}

*Requirements*

* Add an option to the ID Card Template to insert the patron photo.
* Pull in the field for Student/Patron Photo (already exists in the database).
* User should be able to view the photo when generating a card, just like the other fields.
** -This should be something they can configure the location on within the template borders.-
*** -User may to position it to the left, the right, etc.-
*** -Maybe a movable box to mark out the area the photo will populate?- (Deferred)
* (Added Aug.11th): Additionally, let’s use this story to:
** Fix alignment issues while printing ID Cards
** Implement final ID card design
** Resolve any other UI/UX needs.

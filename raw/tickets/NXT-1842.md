# NXT-1842 — INV - Reports - Item History

## Description
As an Inventory Manager I want an Item History report so I can see what has happened with one or many items during a period of time.

*Requirements*

* Add a new report to the Reports page titled ‘Item History’
** Description: View the inventory transaction details of one or many items for a period of time.
* Provide Site selections
** default to ‘All’
* Provide Item selections
** Item Category - default to ‘All’
** Item - default to ‘All’
* Provide Item search
** by Item #
** by Item Description
** Exact match option
* Provide Date Range section
** allow to generate by Month/Period
** allow to specify a custom date range

+Report+

* Display a section for each Item and Site
** Sort first by Site, then by Item
** alphabetically ascending
* Display a list of transactions for each item
** oldest to most recent
** show starting balance at the top (as of the start date selected)
* Show relevant columns for each transaction
** Actual Transaction Date
** Transaction Date 
** Transaction Type
** Transaction #
*** Ex: Withdrawal #19012
*** Ex: Order #12345, Receipt #288101
** Adjustment Quantity
*** the quantity associated with the transaction
*** the impact on perpetual, e.g. -12 
** Perpetual Quantity
*** the site perpetual quantity for the item after the transaction
** Comments
*** from the transaction, if applicable

# NXT-21338 — Acc. - Claims - Merge satellite/combined sites option Claim Creation

## Description
As Vera the Director I want the reimbursement claim to merge any sites that are marked as satellite or combined to save manual work later.

*mockup*
n/a

*requirements*

* Add a System Setting to merge sites together for claim creation.
* Check for the setting during claim creation and move meals/eligibility counts accordingly.

System Setting

* Name: Claims - Merge Combined/Satellite Sites
* Description: This option will merge sites that are configured as combined or satellite together, moving meals and any approved counts to the primary site.
* Options: Keep Separate (Default), Combine Sites

## Acceptance Criteria
System setting added.
Claim creation updated to move meals based on the setting.

We should be able to see the change in both the preview and export.

Tested with multiple state claim formats.

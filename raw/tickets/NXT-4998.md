# NXT-4998 — Data Exchange - Modify Data Screen

## Description
*User Story*

As a PrimeroEdge user, I have a screen that allows me to quickly fix any issues with the data before import, as well as schedule the import for later and save the configuration.

*Mockup*

To be built

*Requirements*

* The Modify Data screen is present after the Field Mapping screen
** Any lines with errors are presented to the user in an editable format
** Errors are listed
** User is able to make updates - If updates are not made, the lines are omitted from the import
* User has the option to schedule the import for later
** There is a checkbox that says “Schedule for Later?”
** This scheduler is for one-time imports - recurring is handled from the hub screen
** If checked, the following options appear
*** Time/Date
**** Time/Date can be set at any time or date in the future
***** Date can be the current day as long as the time is after current time
*** Notification Method
**** Users can be notified through email, in-product messaging system, both, or none
**** User is notified for success, for success with errors, or for failure
** Once saved, the import should run at the selected time with the selected settings
* User has the option to save the configuration
** There is a checkbox that says “Save Configuration”
** If checked, a text box appears - “Configuration Name”
** Once saved, the configuration can be run on future imports
* There are three buttons available
** Cancel - User exits the data exchange wizard - A warning box appears when clicked
*** “Are you sure you wish to cancel? Any changes will be discarded.”
** Go Back - User is taken back to the Field Mapping screen
** Contextual Button - Save if scheduling, Import Now if not scheduling
*** Import is conducted if Import Now
*** Import is scheduled and runs as expected if Save

## Acceptance Criteria
* Users are able to review and update errors
* Users are able to schedule imports
* Users are able to save configurations
* Users are able to continue onto import from this screen

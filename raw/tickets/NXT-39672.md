# NXT-39672 — IM - MI Consolidation - (Custom) Menu Item - Nutrient Info tab

## Description
As a Menu Planner I want a Nutrient Info tab on the unlinked/custom Menu Item details so I can see the nutrients and allergens being calculated or generated as I assign more items or recipes.



!Custom MI - System Calculated-20230607-144307.png|width=83.33333333333333%!

## Acceptance Criteria
* Build the Nutrient Info tab for _Custom_ Menu Items (created first as a menu item through the Menu Items page)
** Map the existing Nutrient card data into the new card design (from Items, other screens)
* Add the nutritional value of all added items/recipes to populate the Nutrient Info tab’s values
** Update the calculation to include both Recipes and Items (Items are added in [https://primeroedgenext.atlassian.net/browse/NXT-39671|https://primeroedgenext.atlassian.net/browse/NXT-39671|smart-link] 
** Logic is matched similarly to multi-item recipe calculation logic
* Assign a System Calculated label to the *Nutrient Details* card
** this label is displayed as long as the Nutrients is using the system generated values
** Assign a Manually Entered label for any user provided values which override the system calculated values
** Make the ‘Manually Entered’ label clickable
*** in view mode as well as edit
** Upon clicking the Manually Entered label, display a prompt which allows the user to replace the values with the system calculated
* Assign the same System Calculated label to the header space of the *Allergens* card
** Same logic as Nutrient Details card
** Make the Manually Entered label clickable
** Show the prompt to user to replace manual values with system calculated
*** Perhaps use an in-line UX approach here instead of popup confirmation prompt

# NXT-58340 — Acc - Sessions - Reconciliation - Mark as Deposited

## Description
As Sally the Site Manager I need to move a session back into being Deposited without actually adjusting a Deposit Slip. The system offers me no way to do this and I have sessions stuck in the wrong status.

*requirements*

* User can change a Reconciled session to Deposited via Reconciliation.
** Add a new action: Mark as Deposited 
*** Action is only available if the session status is Reconciled.
*** Tooltip for when the action is disabled:
**** Only Reconciled sessions can be moved to the Deposited status manually.
** Confirmation prompt:
*** {noformat}This will manually update this session to Deposited without creating or adjusting any Deposit Slips. Please consider creating or editing the Deposit Slip first. 

Are you sure you want to proceed?{noformat}
*** CANCEL / PROCEED
** If the user confirms, change the status to Deposited.
*** Update Session History as well with the following note: Session manually moved to Deposited.

## Acceptance Criteria
* User can move a Reconciled session to Deposited.
** Action added

** Session Status updates

** Rollups behave accordingly

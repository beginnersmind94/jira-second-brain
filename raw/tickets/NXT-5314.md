# NXT-5314 — Onboarding - System Task List - Sites - Display Sites Tour

## Description
As a new SchoolCafé user, I would like to have a basic walkthrough on the steps required to set up my district’s sites, so that I don’t have to spend time guessing what to do next.

*Summary*

When the user comes to the Sites task, display a simply tour pop-up (with a Finish button), that explains what the user must do to ‘complete’ this task.

*Mockup*

!image-20230509-180532.png|width=406,height=331!

*Details*

In Onboarding → System → Tasks, when the user selects Sites and is redirected, do the following:

* Display a tour in the center of the screen:
** Text:
*** Title: 
**** *How to Complete This Task*
*** Body:
***# Add all feeding and non-feeding sites in your district.
***# Verify that the site types and grade groups, with their associated grades, are accurate.
***# Quickly look over some special settings that may be useful for your POS machines. You can complete this part of the task later if necessary.
* Display a ‘Finish Tour’ button in the footer (no Skip, Next, or Previous are needed). Text color can be changed back to +white+ if we haven’t done so already.
** When the user selects this button, close the pop-up. This should prevent the tour from executing when the user logs out and returns to this page.

## Acceptance Criteria
Verify that when the user comes to this task, they are immediately presented with a tour step in the center of the screen

Verify that the text and button placement of the tour match the description above

Verify that when the user selects ‘Finish Tour’, the pop-up closes and the user can proceed

Verify that the user will not see the tour again when they are logged out and return to the page

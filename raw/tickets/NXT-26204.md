# NXT-26204 — Import/Export Setup - Reduce Field Mapping Screen to 2 Columns

## Description
The 3 “columns” (Source, Mapped, Target) in the field mapping UI looks cluttered and does not provide any additional value that cannot be accomplished in 2 columns.  The idea is to make the UI simpler and more intuitive.

!image-20220219-192843.png|width=1324,height=292!

## Acceptance Criteria
* The Field Mapping screen shall contain 2 columns “Source” and “Target”.
* The “Target” column shall be removed
* The “Mapped” column shall be renamed to “Target”
* The new “Target” column shall contain the asterisks indicating a field is mandatory.

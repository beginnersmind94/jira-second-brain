# NXT-59209 — Acc - Deposits - Retain search filters when coming out of Deposit Slip Details, grid filters, date picker, site picker updates, excel download

## Description
As Sally the Site Manager I want my search to remain intact when I move out of Deposit Slip Details and back to the main Slips page.

*requirements*

* User search criteria retained when nav’ing back out of Deposit Slip Details
** MVP: Just the filters
** NTH: Search results as well, pagination as a real stretch
* Update site and date picker
* Add filters to grid columns
** Bank Deposit Slip #
** Reference #
** Deposit Total
** Deposited By
** Comments
** Create Date
** Modified Date
** Deposit Variance
* Update grid download to only pull selected columns
* Do not show the Results grid on page load

## Acceptance Criteria
* User search criteria retained when nav’ing back out of Deposit Slip Details
* Update site and date picker
* Add filters to grid columns
* Grid download only pulls selected columns
* Results grid not shown on initial page load

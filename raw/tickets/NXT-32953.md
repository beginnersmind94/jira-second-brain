# NXT-32953 — INV - Additions - Addition Details - Pack Size Updates

## Description
As a Site Manager I want the Additions Details page to reflect updated Pack Size and QOH logic so I can know exactly which items I’m adding to inventory.

*Requirements*

* Update the Addition Details screen to reflect the updated item hierarchy configuration
** Items and their pack size units should be displayed according to how they were entered on the Addition ([NXT-32081|https://primeroedgenext.atlassian.net/browse/NXT-32081])
* Display Quantity on Hand
** Present the full string
** Enable Alternate QOH popover ([NXT-26951|https://primeroedgenext.atlassian.net/browse/NXT-26951])
* +Update the UI to accommodate multiple unit types+
* Display units on their separate rows if 
** there is a unique price value for each 
** the units have different Bin Line dates
* Display a single row if
** there is only one unit 
** the sub-unit price is perfectly scalable to the higher unit
** all units have the same Bin Lin info
* Create batches according to same rules above
** Single batch if all units have the same per-unit price and the same Bin Line
** Different batches for each unique
*** Price (per unit)
*** Date on Product

## Acceptance Criteria
* Update Addition Details to reflect updated Pack Size configuration
* Add Quantity On Hand and new Alternate QOH feature
* Display units on separate rows if they have unique price or dates on product
* Display a single row per item for single-unit items or items which would all be in the same batch

# NXT-40319 — eOrders – Generate Part 1

## Description
As a SchoolCafé user, I would like to generate an eOrder so that I can electronically order food.

*Notes:*

* We are porting over this page: [https://test.primeroedge.co/Inventory/EOrderDetails.aspx|https://test.primeroedge.co/Inventory/EOrderDetails.aspx] from Classic
* We need to ensure that all that functionality is captured

*Flow:*

* Open SchoolCafé and browse to Inventory > Orders > eOrders
* User clicks Generate eOrder button and the overlay opens for them
* User views list of orders, select the ones they want
* User clicks Generate eOrder button
* User redirected to dashboard screen and grid of eOrders updates with new one

*Requirements*:

* These requirements are for the eOrders Dashboard page
* 
* If not already there, add the Generate eOrder button to the page
* On click, open the Generate eOrder Overlay
* 
* Create an overlay to be linked to the Generate eOrders button
** Title = Generate eOrder
* 
* Do as a stepper
* The overlay has 2 tabs:
** Selection
** Completion
* 
* Security
** Don’t show this overlay in Roles and permissions CRUD tables
** This overlay’s permissions = Add permission for eOrders Dashboard page, therefore restrict security to those roles with eOrders Dashboard Add permissions plus Cybersoft admins
* 
* The rest of the requirements for the tabs will be done in other stories.

# NXT-21188 — Eligibility - Income Surveys - Add Income Survey Status History/Notifications History to Survey Details

## Description
As a district admin, I would like to be able to track the history of changes to an Income Survey’s status and notifications, so that I can more effectively audit them as needed.

*Summary*

* Add two sections to the Survey Details page: Income Survey Status and Notification History, with their respective grids. The shortcut menu on the right side of the page should also include these 2 links to jump directly to them on the page.

*Requirements*

* In the Survey Details page, add two sections:



*Income Survey Status* 

* This grid should display a list of records which updates every time the user changes that the status of the survey (i.e. Mark as Pending, Processed, Notified, Deleted, etc.) with associated datetime/user data.
* Shortcut menu label: History

!image-20211101-181715.png|width=1341,height=448!

*Notifications History*

* This grid should display a list of records which updates every time the user sends (prints/emails) a Survey Notice,  with associated datetime/user data.
* The grid also should allow the user to view that historical notice.
* Shortcut menu label: Notifications

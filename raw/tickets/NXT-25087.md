# NXT-25087 — Platform - BO - Onboarding MVP Enhancements

## Description
As a district admin, I should be able to setup my onboarding configurations after logging into SchoolCafe for first time which presents a preview tour and selection of activities

* Welcome preview screen with banner message & onboarding tour video
* Welcome screen presented only for specific onboarding role
* Other users who are not part of onboarding role for that specific district should not be able to login until onboarding is complete
* Selection questionnaire for activities that customer are presented with based on district modules licenses
* Customer sets the target go-live date & readiness date
* Post selection of questionnaire shall bring up the stepper functionality for customer to complete the activities one step after step (*sequential)*. 
* Once all stepper activities are complete, Onboarding Summary Dashboard is loaded

*Content for UX:*

* System:
** Setup District: Quickly create districts for your region which takes about ~3 mins
** Create Roles: Select activity to for creating various in your district - ~1 min per role
** Create Users: Select this activity if you need to create users and assign them to specific roles - ~ 1 min per user
** Import Settings: Select this activity to load settings by default or custom - ~ 4 mins
* Back Office:
** Item Imports: Select this activity if you like to import menu items for your inventory module
** Add Menus: Select this activity for creating & managing menus for your menu planning
** Add Recipes: Select this activity to design your recipe
** Update Stock Info: Select this activity to update your inventory stock
* Front Office:
** Student Import: Select this activity to import students data from your source file or system - ~ 10 mins
** Eligibility Imports: Select this activity to import eligibility info of students - 10 -15 mins
** Balance Imports: Select this activity to import student account balances - ~ 10-15 mins 
* Add-ons:
** Financials: Select this activity to configure reports related to financials module
** Catering: Select this activity to configure catering module
** Inspection: Select this activity to setup auditing for inspection management.

## Acceptance Criteria
Customer shall be able to:

* Preview welcome screen when district admin with Onboarding role logs in for the first time and no setup has been started for that specific district
* No other users from that specific district/region shall be able to login
* Select List of activities under specific modules that the district has purchased. (Scope is only System for R4)
* "Lets get Started" button is available to browse through activities that they selected for onboarding & activate each activity
* Video option is enabled anytime for customer to quickly reivew
* Customer shall be able to set “Target Go-Live Date” in questionnaire screen
* Customer shall be able to set “Readiness Date” in questionnaire screen

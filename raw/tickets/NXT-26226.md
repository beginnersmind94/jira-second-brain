# NXT-26226 — Platform - Permissions - Resource List based on Resource Type - Function.

## Description
As a Primeroedge admin, I should be able to generate a unique function code each resource created of function resource type.

*Requirements:*

* Each Resource to be identified by Unique code with a separate display name
* Based on selections below, unique function code to be generated with uniqueness
* Function ID attribute to be added for each function

## Acceptance Criteria
* System shall generate unique function code for each resource created of function resource type
* {color:#ff5630}Deferred{color} - User shall be able to see tool-tip/helper next to the function code

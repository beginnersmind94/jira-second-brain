# NXT-56611 — RevTrak - Adult Balance Export (FD 266119)

## Description
As a SchoolCafé Admin I want to be able to export adult balances to RevTrak in the format it needs so that I can get our payments back in their system

*Notes:*

* We already do this for Student balance export
* We need to do exactly the same for adults, so just copy that code

*Requirements*:

*Add New Export drawer*

* For each export type except Claim Export, do the following:
* For *Add New* screen, do the following:
* *Select tab:*
* Add Adult Balance Export to the list values of Export Type
* Do all the same things for the other fields as Student Balance Export 
** Same for delimiter, has headers and scheduler
* Do the same for Field Mapping tab as Student Balance Export
* Do the same for Review tab as Student Balance Export



*Edit Drawer*

* Make the same changes to the Edit drawer that you did for the Add New drawer



*Other*

* Ensure the Add New changes are saved in the History log and viewable in the View History drawer
* Ensure the Edit changes are saved in the History log and viewable in the View History drawer
* Ensure the View Config drawer shows the configuration
* User can clone this export type

## Acceptance Criteria
* User can select select if they want to have headers when doing this export type
* User can select if they want to do a pipped delimited export when doing this export type
* System will generate a pipped delimited txt file with no headers if that is what the user selected
* User can still do comma delimited exports with headers
* Edit screen allows user to change these values and they are saved
* User can see the Add New and Edit changes in View History
* User can see the configuration in View Config drawer
* User can clone this export type

# NXT-8157 — INV - Physical Inventory - add setting for Inventory Periods

## Description
As an district inventory user I want a setting for whether or not I will use inventory periods, so I can accurately capture how I do physical inventories.

*Requirements*

* Add a setting for enabling Inventory Periods
* Display or hide the Period dropdown based on the setting selection:
** if periods are enabled, the Period dropdown will be provided on the Start New Physical Inventory page
*** as mention on NXT-8136, Interim will be provided as option
** If periods are disabled, the Period dropdown will not be displayed, and every inventory will be Interim
* Setting will apply to the entire district
* Create temporary setting values tables

## Acceptance Criteria
This setting will control whether or not the Period dropdown is available on New Physical Inventory page
If setting is disabled, all new physical inventories will be 'interim'

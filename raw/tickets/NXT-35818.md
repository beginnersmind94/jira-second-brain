# NXT-35818 — INV - Reports - View permissions for Reports

## Description
As a district admin I want permissions for the various Inventory Reports so I can control the reports to which managers have access.

*Requirements*

* Map view permissions for all Reports 
** Edit/Add/Delete do not apply to Reports
*** Disable Edit/Add/Delete checkboxes on the Resources page 
* Dynamically update Report page based on reports to which the user has access
** Hide reports without view permissions

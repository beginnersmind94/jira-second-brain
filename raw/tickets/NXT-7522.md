# NXT-7522 — Application Processing - (Action) Pending

## Description
*Story* 

As an application processor, I can put an application in pending status if it’s incomplete or needs more info for processing. 

*Mockups*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421304%3A268450&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421304%3A268450&scaling=scale-down-width|smart-link] 

*Requirements* 

* While processing applications, user has a way to park an application as pending.
* User can select from a list of reasons to put application in *Pending* status 
** Missing PFD information
** Restored application
** Incomplete
** Incorrect income information
** Incorrect categorical information
** Incorrect household member(s) information
** No signature
** SSN information missing/incorrect
** Possible duplicate information
** Reapplying after verification
** Other (please specify in comments)

* User has a comment box available, 250 character limit.
* If there are ANY hard stop reason(s) in the pending application reasons section, do NOT apply the eligibility to the student(s) associated with the application. 
** Application is in Pending state for the reason(s) indicated 
** Reasons show in the Application Details
* Eligibility Panel 
** If pending reasons (hard stops) exist, we will not show the panel > show message in that section 
*** _Due to the pending reasons, we cannot reach an eligibility determination._
* Benefits not extended to students in the application since the application has not been processed due to pending reasons.



*Req added 2/25 after daily:*

* User should be taken to Application Details for the application after parking a single application as pending. App is created, status is pending, user loads into Details for the app.
** In Processing Mode, they would instead go on to the next application.

## Acceptance Criteria
- admin can mark the application as pending 
- admin can choose from the listed reasons and enter comments
- indication of a reason is necessary, whether it be a selection and/or comment
- marking a pending application does not extend benefits to matched students
- Multi-language, ADA compliant.

# NXT-16998 — INV - Orders - Order Detail - Receipts tab - Edit Receipt Details - Charges & Discounts

## Description
As an operations manager I want the ability to edit the charges and discounts on a receipt so I can add any fees or discounts associated with the receipt.

This is for only loading the editable Charges & Discounts section of a receipt. There are separate stories for loading the receipt details and editing the header ([NXT-15379|https://primeroedgenext.atlassian.net/browse/NXT-15379]), items ([NXT-16997|https://primeroedgenext.atlassian.net/browse/NXT-16997]), and saving the receipt ([NXT-17418|https://primeroedgenext.atlassian.net/browse/NXT-17418])

Note: this functionality is dependent on the user having the ‘Edit Receipt’ permission (separate story)

**Notes:* 

* no API for this story (covered in NXT-15379)
* Additional requirements will be handled in [https://primeroedgenext.atlassian.net/browse/NXT-18451|https://primeroedgenext.atlassian.net/browse/NXT-18451|smart-link] 

*Requirements*

* When user clicks the Edit button, all relevant fields will be editable
* This section is only displayed if at least one charge or discount is applicable to the receipt
** if there is at least one active configured charge/discount on the Charges & Discounts settings page, *OR*
** if there is at least one item with a contract-level charge or discount (Processing Fee, Delivery Fee, NOI)
* In Edit state:
** Enable the editable fields for the non-contract charges/discounts
*** the textbox where the dollar amount is entered
*** this would be only the *Active* ones configured on the settings page
** show the read-only calculations of the contract charges/discount

Mockup Note: contract-level charges/discounts are *not* editable. Labels only.

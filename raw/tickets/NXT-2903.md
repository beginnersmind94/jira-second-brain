# NXT-2903 — Record Training - Rollcall

## Description
Rollcall is a day-of training admin activity. 

* The rollcall list shows all employees who have RSVP for the event 
* the trainer can then ‘call roll’ and simply record training for those individuals 
* option to add an employee to the list (if they didnt RSVP) 
** the employee can also login from phone and RSVP while there 
** the employee can log in via web and RSVP 
* option to deselect employees who RSVP but did not show up
* when the training is recorded, it will show on each attendees account, reports, etc.

## Acceptance Criteria
- Employee can RSVP for a training event via phone or web, 
- admin/trainer can use RSVP list to record training
- rollcall list has a select/deselect option
- list has an option to add manually 
- employees can also access event via phone/web to RSVP day of event

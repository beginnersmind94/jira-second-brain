# NXT-48726 — MP- Menu/ Menu Cycles/ Menu Templates- Allow complete uniqueness of name not considering spaces

## Description
As a menu planner, sometimes I accidently create a menu/ menu cycle or template with the same name but add an extra space on accident. The system still allows me to create it but it is stored the same. This causes confusion on the list pages as they look the same but are not.

## Acceptance Criteria
* Do not consider extra spaces as a cause for uniqueness
* Trim the trailing and leading spaces of the description, as well as handle multiple consecutive spaces between words by treating them as a single space

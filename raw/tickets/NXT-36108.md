# NXT-36108 — IM - Recipe Rework - Nutrition Info - Allergens

## Description
As a Nutritionist I want the Recipes page to have a dedicated Nutrition Info tab so I can recipe nutritionals without having to scroll past all the steps and ingredients.

*Requirements*

* Redesign the Recipe details page to appear similarly to the Item overhaul design
* Create a tab based design to separate various Recipe sections
* Create a Nutrition Info tab to house all the calculated recipe nutrition info in the Item design format
* Update the *Allergens* card to match the Item design
** Ensure that the logic for manually overriding remains
** Provide the *System Generated* button to reset to system derived allergens
** Show the label for System Generated vs. Manual Entry

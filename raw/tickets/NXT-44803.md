# NXT-44803 — GDSN Connect Updates (SC) - Item Sync History (UI changes)

## Description
_(none)_

## Acceptance Criteria
Navigation: IM - Global Catalog - Sync History

The user applies the filter parameters as applicable list the items

the system list the list of items that got updated with its count of attributes

Upon clicking the attribute count, the system should show the old value and new value based on the History table details



Search Parameters:

Data Source: Local (Default value); Data Provider: 3rd Party Source(s) - Default Value

Default Date Filter - Past 3 Months

From Date and To Date - auto populated (calculated system date - corresponding number of days) 

Imported By (User Name), First Name and Last Name

Default Status - Active Items , user can filter for inactive items on demand - in such cases show inactive items only 

Manufacturer & Brand - Default Value - All, more than one selection is possible

Upon selecting multiple parameters, system should populate based on AND condition 



The grid should show the following fields:

Manufacturer, Item Name, Item Code, Count of Attributes, Imported on, Imported By (User)

Upon selecting a Item Name (Hyperlink), the slide out should show all the entries of audit log that matched

The user can click the audit log entries and see the old value and new value changed

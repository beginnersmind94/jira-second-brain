## Acceptance Criteria
The change is applicable for following user scenarios:

# Create an Item (Food Item / Non-Food Item)
# Edit an Item (Food Item / Non-Food Item)

User should be able to select an Manufacturer and Brand from pre-defined list of values - as attached

# User selects the Manufacturer - the system shows the possible matches as they type

*Brand & Manufacturer Rules:*

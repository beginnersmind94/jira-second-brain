# NXT-64911 — Reuse-WS - Filters - Site Groups - SC

## Description
As a Workspace user, I want to be able to select sites more efficiently, so that I can be faster in filtering my data.

*Requirements*

* This story is for the filters on the main page
* It applies only to SC Districts
* In the drop down list for the sites, do the following:
** Be able to select by Site Group
* Show only the site groups that apply to that district

## Acceptance Criteria
* Users are able to select from site group in the drop down
* Drop down doesn’t show site groups that are not applicable to the district

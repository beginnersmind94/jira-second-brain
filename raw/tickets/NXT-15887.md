# NXT-15887 — INV - Create Orders - Add Comments

## Description
As an operations manager I want Comments icons on my orders and items so I can add important information in the comments as part of the order process.

*Requirements*

* User can add Order level comments by clicking the Comments icon for the Order
* User can add item-specific comments by clicking the Comments icon for a specific item
* In both cases, the user will be provided a popup to enter their comments, with a Save button, or a Cancel button to discard comments
** For Order level comments, show the Order # in the header (dynamic)
** For Item-specific comments, show the Order # and Item # in the header (dynamic)
* Comments can be entered on any step as long as the comments icon is available, during the Create Orders process
** Comments are not editable once orders are created (saved as Open)
* 200 character limit on comments
** Display character limit label/indicator

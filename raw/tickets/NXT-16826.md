# NXT-16826 — INV - Orders - Order Detail - Receipts Tab - Receipt Details (view-only) - Items

## Description
As an inventory user I want a Receipt Details screen so I can view the details of a receipt without editing.

*Requirements*

* Provide a version of the Receipt Details screen (Create Receipt, Edit Receipt) that is not editable by default, showing the saved values as labels.
* Header is covered in [NXT-16188|https://primeroedgenext.atlassian.net/browse/NXT-16188]
* The item list values will be displayed as labels not in editable state
* Edit buttons are provided for the item list (Edit functionality is [NXT-15379|https://primeroedgenext.atlassian.net/browse/NXT-15379])
* Add Item and Charges & Discounts are in separate stories

Note: Confirm column will *not* be present in view-only state

* Receipt Details item list will be in view-only state

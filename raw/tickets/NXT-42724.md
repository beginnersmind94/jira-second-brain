# NXT-42724 — IM - Items - Ingredient Info redesign

## Description
As a district admin I want the Ingredient Info to contain relevant info about status and recipes so I can manage my item as an ingredient from this single card.

## Acceptance Criteria
* Change the UI styling of the Ingredient Info card
** make card 2 columns, with 3 rows
* Add an ‘Assigned to Recipes’ label
** Provide the number of +active+ recipes on which the ingredient is currently assigned
** Provide a view icon to open (in a new tab) the Recipes list page with the ‘Contains Item’ dropdown pre-populated with the ingredient
* Add an Active chip to the card header
** clicking the chip will open the Item Type slideout
** in the future, slideout will manage Ingredient active/inactive status

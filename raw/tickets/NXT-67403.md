# NXT-67403 — LMA - Settings - Toggle

## Description
As a Customer Support (CS) role member, I would like to select which interface to use for License Management so that I can control how the new LM is rolled out

*Requirements:*

* Add a System Setting to switch between old LMA Interface and New LMA Interface
* In System Settings, add a new property
** Setting Name: Enable New LMA
** Type: Internal
** Code: NEWLMA
** Value: Yes/No
** Default: No
** Category: Configuration
** Module: System
* 
* Add in the navigation system the check to hide/show the new license pages per this system setting
* Add in the navigation system the check to hide/show the old license page per this system setting

## Acceptance Criteria
* Users can only see the appropriate license management pages per this setting

# NXT-27831 — INV - Production Ordering - Cart Items

## Description
As a Production Manager I want a list of my ordered items so I can review the list of items I am about to order.

Add a way to add items to the ‘cart’ but the cart will be a version of the main item grid, so all item attributes can be preserved.

*Requirements*

* Add a tab/chip/group for the items being ordered for each vendor
** this will move items from the ‘unordered’ list to the order list, just like the cart
** *Note: more tab/chip separation will be required in the future (ex: [Par Level Items|https://primeroedgenext.atlassian.net/browse/NXT-26973])
*** going tab
** tabs: Production Items and Cart Items
* Change the Vendor dropdown to ‘All’ option when clicking on the Cart Items tab
* The items added to this section will be displayed similarly to the regular item list, but with some slightly different columns
** do not display the Production Item columns
** do not display checkboxes (all items will be ordered)
** Order Item #
** Order Item Description
** Vendor 
** Vendor Item #
** Order Quantity
** Price
** Total Price
* Implement sticky footer for Add to Cart and Generate Orders (3/11/22 JM)
* Show Edit/Delete for each cart item
* Maintain Vendor and Delivery Date grouping on this view
* Need to handle Min and Max amount and Min and Max order qty with regards to the item and the vendor and show validations or alerts as it is done in the Standard Ordering screen.

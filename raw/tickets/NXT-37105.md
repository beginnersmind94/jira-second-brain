# NXT-37105 — Error Reporting – Save Error in Application Insights - Dev only

## Description
We need to save all our data exchange pipeline errors in application insights so that we can run error reports off them.

*Flow:*

* An error occurs in a data exchange pipeline
* The error is logged in application insights

*Notes:*

* This story is part of the stories to expose errors to support staff and follows the story to create an error handling email account
* We already updated the email address on where errors are to be sent, so in all those places, we want to add code to save in Application Insights

*Requirements*:

* Everywhere we have code to send an error email, add additional code to save the error to application insights
* The content of the error message must contain at a minimum all the fields in the error email plus whatever else you need from a dev side
* 
* Permissions: no additional requirements necessary

## Acceptance Criteria
* Everywhere we have code to send an error email, add additional code to save the error to application insights
* The content of the error message must contain all the fields in the error email

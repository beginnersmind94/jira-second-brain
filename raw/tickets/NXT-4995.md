# NXT-4995 — Data Exchange - Add Field Mapping Screen & File Table

## Description
As a PrimeroEdge user, I would like to have an intuitive screen that allows me to easily associate fields in PrimeroEdge with the fields in my uploaded file, so that I am not overwhelmed by the amount of information on the screen.

*Summary:*

# Add a Field Mapping step to the Data Exchange wizard
# When a user lands on the Field Mapping step of Data Exchange, display the file details table. *Table and Previous button only--all other items and actions on the page covered in separate stories.*

*Details*

* In Data Exchange, add a *Field Mapping* screen. 
** This screen shows the details of a successfully uploaded file and allows the user to both map the fields and configure their special properties (separate stories)
* Display a table that shows the details of the uploaded file. Columns:
** Field Header
*** In each row, display the value in the first row of each column in the uploaded file.
**** We will adjust this so that the user can indicate on the previous step whether the file has a header row or not. But for now, we will implement this for when a header column exists in the file.
***  Order of items should match the column. i.e. column A-1 should display in the first row, followed by B-1, etc.
** File Value
*** In each row, display the value in the second row of each column in the uploaded file, in the same order as above.
**** Again, this is subject to the file having a header row.
** User should still be able to go back to the previous screen via a Previous button.

## Acceptance Criteria
* Verify user can proceed to this step from File Upload

* Verify a file details table is displayed with each column displaying based on the description above

* Verify that Field Header column and row values are properly depicted in each row based on the file

* Verify that File Value column and row values are properly depicted in each row based on the file

* Verify user can go back to File Upload step if needed and retain config changes

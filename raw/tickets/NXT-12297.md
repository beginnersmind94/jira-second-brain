# NXT-12297 — Platform - AI/ML - Update Facial Signature

## Description
*Persona*: Student/Parent

*Access level:* School District/Site

*Description*:

As a parent after registration to facial recognition, I should be able to use the model training application to update my son/daughter’s video or image so that system can learn/train latest facial signature which is due on yearly basis.

## Acceptance Criteria
1. Parent should presented with option to capture multiple image or video stream based Facial signature
2. User should be able to replace current facial signature with Photo or Video option
3. User should be able to confirm the liveliness after initial facial signature
4. System should preserve the latest facial signature consumed for pattern detection in Live
5. User should be able to discard and re-capture facial signature
6. User should be given 30 secs for each facial signature for both primary and secondary signatures

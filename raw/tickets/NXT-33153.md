# NXT-33153 — SFTP Import - Enable Student Photo Imports

## Description
Enable the ability to import student photos via sFTP.

SFTP ADF:

!image-20220719-220216.png|width=718,height=777!

## Acceptance Criteria
* Districts shall have the ability to FTP student photos as a means of importing them into the system
* The processing for each photo shall begin within 5 seconds after the photo is persisted to blob storage
* Progress of the import shall be made available within the Imports Monitor UI
* Exceptions to the import process shall be made available within the Imports Monitor UI

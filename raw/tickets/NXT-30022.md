# NXT-30022 — PR- Reports- Post Production Costing (Summary by Site)

## Description
The Post Production Costing Report gives the district information regarding the cost of foods in relation to the completed Production Plans from their sites. It looks at the quantity of ingredients calculated to be used to generate the servings, and pulls the costing from the Fair Market Value listed with the products associated to the ingredients.

Example: If my Production Record states that I produced 23 apples, then the software would calculate the Fair Market Value of those apples, divided by how many broken units contained in the whole unit, and multiply that broken unit cost by those 23 apples.

The district should be able to pull this report with the following details:

Site: Default to All

Meal Type: Default to All

From Date

To Date

Report Options (Second Criteria) : Details by Site, Summary By Site , Summary by District

## Acceptance Criteria
As a user I would like to see my Post production costing (Production Record Costing) This report should take all the Menu lines for whatever meal type that is selected and break down the plan and cost.

SUMMARY BY SITE

Display the date range for the report as well as the type of report (Summary By Site), Meal Type, and Site

Display the following columns:

* Site Code
* Site Name
* Serving Date
* Menu Line
* Actual Meal Count
* Total Food Cost ($)
* Food Cost per meal ($)

Display the first row with the site group and meal type (Site group in alpha followed by Meal Type in order of Breakfast, Lunch, Snack, Supper)

EX:

Site Group: Early Childhood; Meal Type: Breakfast

Display all the sites under that site group and meal type combination in the date range before moving on to the next meal type. Once all meal types are done, move on to the next site group and repeat the process.

*All calculations are based on the Details by Site report* 

Total Food Cost column = Total Purchase Cost for all items on the menu line

Food Cost Per Meal column = Total Food Cost for all items on the menu line

*SubTotal/Average: Site Name row* = Total Meal Counts for all menu lines and all serving dates, Total Food Cost for all menu lines and all serving dates , Food Cost Per Meal = the total food cost for all serving dates/  Actual Meal count.

EX: 

Total Meals: 518

Total Food Cost: $545.7067

Food Cost Per Meal: $545.7067/518= $1.0535

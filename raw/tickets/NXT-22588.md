# NXT-22588 — INV - Inventory - Perpetual Inventory - Enhanced Item Grid

## Description
As a Site Manager I want a better item grid for Perpetual Inventory so I can understand my on-hand products and how they relate.

*Note:* UI Components team has added new TreeList grid component. This grid has indentation and expand/collapse properties that we will use to show item hierarchies more effectively.

*Requirements*

* Update the item list grid on Perpetual Inventory page
* List columns:
** Item #
*** this will be the indented column that will indent for each child item in a hierarchy
** Item Description
** Unit Type
*** show as a very small column
** Storage Category
** Item Category
** Quantity On Hand
*** QOH enhancement in future ([NXT-27833|https://primeroedgenext.atlassian.net/browse/NXT-27833])
* Apply hierarchy sorting
** Child items should be displayed within the same hierarchy grouping
** the Item # should be indented for each child item in the grouping
* Enable expand/collapse
* The default expand/collapse will be determined by the selection of ‘Show All’ vs. ‘Show Only On Hand’
** If ‘Show All’ is selected
*** display all items with a site-item record
*** all child items will be expanded
** If ‘Show Only On Hand’ is selected
*** display all items with on-hand quantity
*** expand all child items which also have on-hand quantity
* Enable typical sorting
** sorting only applies to top-level items
* Enable Field Search
** if search results return a child item, it will be displayed in the context of its hierarchy, up to the parent item

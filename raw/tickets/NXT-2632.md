# NXT-2632 — System - Users - Complete remaining user avatar functionality

## Description
To give users full visibility of and functionality related to user avatars, please do the following in System - Sites & Users - Users:

# In the Card view, add the image icon within each card where an avatar has been uploaded for that user (first and last initial should only display in the circle if an image has not yet been uploaded).
# In User Details, when a user attempts to upload an image, we are not showing a preview, so they will not see anything until they click the Save button and return to that user. Instead, let’s add a preview of the avatar that displays as soon as they upload the image. However, continue to not commit the change until the user clicks the Save button. 
# Also in User Details, when an avatar has already been uploaded, add a delete functionality within the image. When the delete button is selected, remove the avatar from the profile. Again, we can show a preview of the blank image/user initials and only commit the change when the user clicks the Save button.

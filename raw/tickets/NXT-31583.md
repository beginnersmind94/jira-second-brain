# NXT-31583 — Create Edit Food Labeling Page

## Description
As a site administrator, I want to edit a trained image so that I can correct any mistakes I have made.

*Application*: SchoolCafé

*Flow:*

* Browse to SchoolCafé > Accountability > Food Detection > Trained Images > Food Labeling Page (in new tab)
* Select boundary on the left side
* Draw a boundary around an image
* Label the image
* Repeat for other food items
* Click Save
* Close tab

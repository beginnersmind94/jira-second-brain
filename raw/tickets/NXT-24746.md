# NXT-24746 — INV - Management - Contract Items - Update Orderability

## Description
As a District Admin I want the Orderable flag to be manageable via the Contract Items page so I can make many items orderable or unorderable at once.

*Requirements*

* Add ‘Orderable’ flag value to the main Contract Items list grid, when items are populated
* Add a new update method to update the Orderable flag
* Update mode would simply show the orderable checkbox for each selected item
** there would be uncheck/check all options
** there would be individual checkboxes and saves
** there will be a Save All option
* Provide prompt on save/confirm for making items unorderable if they are being used
** For individual saves, show the confirmation message from the vendor contract items page (This item will no longer be orderable to sites as part of this contract.)
** For mass save, show a generic message: 'The selected items will no longer be orderable to sites as part of the contract.”

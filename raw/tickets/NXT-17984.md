# NXT-17984 — MP- Assign Menu Criteria

## Description
As a user, I would like to assign my menus with the criteria that I select. 

Mock up:

 !image-2021-08-19-10-36-48-703.png|thumbnail! 

Assign menus criteria needs to be in the order on the mock up. Additional information below

Meal Type -> Site Group-> organizational Level -> Menu Line 

All criteria should be dependent on previous selection. I.E can not select site group unless a meal type is selected, or a menu line before an organizational level.

## Acceptance Criteria
Assign menus criteria in order of Meal Type -> Site Group-> organizational Level -> Menu Line. 

Selections are dependent on the previous selection and can not be selected otherwise.

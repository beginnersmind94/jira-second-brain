# NXT-4113 — AcctMgt - Rosters - Details - Edit/Add/Delete

## Description
*User Story*

As a district admin I want to edit rosters and their contents.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=rosters&id=yvgr6n&c=1|https://nqyc9v.axshare.com/start.html#p=rosters&id=yvgr6n&c=1]

*Requirements*

* On the Roster Information panel…
** Add an Edit button in the corner.
*** Clicking will load the side-drawer where some information can be edited and saved.
*** Editable:
**** Roster Name
**** Roster Description (optional)
***** Character count ‘X/250’ also available and counts as the user types.
**** Site Name
*** Add a validation check when saving - the Roster Name must be unique across the district + site.
**** Example: 'Brown A' can only exist 1 time at District Cabell for Site A. It can also exist at Site B. 
**** Check with Sravani Netturu (Tech Writer) for appropriate error messaging.
** Add a way to deactivate a roster.
*** Active/Inactive toggle.
*** Status will need to update appropriately.
** -Add a way to delete a roster.-
*** -Confirmation prompt for the user.-
**** -‘Are you sure you want to delete this roster?’-
*** -Cancel / Confirm buttons.-
*** -Check with Sravani Netturu (Tech Writer) for appropriate prompt messaging.-
*** Update 8/10/20 - Delete functionality will be handled in another story.
** Add a Delete action button next to each student in the table.
*** This will remove the student from the roster.
*** This will require a confirmation prompt.
**** 'Are you sure you want to remove FirstName LastName (ID #12345) from Roster Brown B? 
**** Cancel / Confirm buttons.
** Add a way to add additional Students to the roster.
*** Add a ‘+ Add Student’ button that launches an  anchored side bar.
*** Search available inside the panel to identify students. 
*** User can quickly search for a student by any combination of ID, Name, Site, Grade.
*** Students already in this roster should be available as search results, but be shown to already be a part of the roster.
*** Clarification 8/13/20: The site selector while adding a student should only contain sites that are part of the roster.
**** Example: Site A + B are both assigned to Roster 123. While adding a student, the site dropdown should only contain Site A + B.
*** -Have an ‘Add +’ button next to the results for quick adds.-

## Acceptance Criteria
Edit mode available that loads the side drawer.
Duplicate roster names are not possible for a district + site.
Students can be added / removed.
Roster can be deactivated / activated.
Multi-language support added.

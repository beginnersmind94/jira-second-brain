# NXT-66944 — INV - Config - Ordering Conditions - Handle seasonal contracts for internal vendors

## Description
As a district admin I want the Configuration > Ordering Conditions page to handle seasonal contracts so I do not have to re-configure my ordering conditions when an internal vendor switches to a seasonal contract.

## Acceptance Criteria
* Update the Config > Ordering Conditions page to consider Vendor instead of Vendor Contract when assigning vendors to site-item conditions
* Ensure seasonal contracts retain the ordering conditions of the main contract

# NXT-71616 — MP - Menu - Copy From Menu

## Description
As a menu planner, I want to ensure that the new flow of menu creation and editing also accounts for copying menus.

## Acceptance Criteria

* All functions remain exactly the same
* Load new menu once copy is made
** Show chip “Menu Copy” that is clickable to that menu as well as a tool tip to give the name

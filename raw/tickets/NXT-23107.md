# NXT-23107 — Production- Reports - Pre Production Analysis- Nutrition Report

## Description
As a user, I need to have a pre production analysis Nutrition report similar to my weekly menu analysis based on the days I select.

## Acceptance Criteria
Report should match the weekly nutrient report while using the planned production numbers.
Use weekly menu cycle analysis for reference.
Display selected dates nutrient summary with all break downs for all nutrient values with the standard (USDA requirements), Actual, and percent of calories. 

Display selected dates food component summary with all break downs for all nutrient values with the standard (USDA requirements), Actual, and percent total. 

Display selected dates break down for each day nutrients along with planned quantities. 

There needs to be a printable option for this report with All, expanded and simplified versions based on the same criteria in menu planning weekly analysis as well as display on UI. 

Display Date range selected in the pre- production criteria page for user to know where the information is populating from.

Criteria for Page display (MOCKUP is for menu planning)
Meal Type
Site Group
Site
Menu Line
Start Date- End Date

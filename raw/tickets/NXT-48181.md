# NXT-48181 — Grades - Display Order

## Description
As a SchoolCafé Custom Support role user, I would to be able to set display/sort order of the grades so that I can see them in the correct order in the list

*Notes:*

* We missed adding display order when we did the UI, so adding it now

*Requirements:*

* These requirements apply to the System > Sites & Users > Site Configuration > Grades tab
* On the Grid, add a new column for Display Order



* -On System > Sites & Users > Site Configuration > Grades > Add New and Edit drawers:-
* -Add a new field:-
* -Display Order-
** -Required-
** -Must be a number-
** -Can be negative- 
* -When save the grade, also validate and save the Display Order-
* 
* Ensure display order is included in the auto sync

## Acceptance Criteria
* Users can set the sort order value for a grade
* -Save button validates there is a numeric sort order value-
* -Can save negative numbers-
* Display order value shows in the grid

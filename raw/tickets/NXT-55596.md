# NXT-55596 — DMF - Migrate API Configuration

## Description
As a Customer Support role user, I would like to be able to migrate my SIS API Configuration from PE so that I can save time

*Requirements:*

* Include in the data migration scripts to copy the API configuration
** Note: they may not have one, so skip if they don’t
* PE location: NF_DT_RegionProviderInfo
* Only migrate it if it is Active
* After migration, the SIS API Provider config screen should have these values in it and we should be able to test it successfully
* If for some reason they have more than one active config, insert just one
** If you can chose, insert the last one

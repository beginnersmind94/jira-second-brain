# NXT-12700 — Platform - Security - Delete a Resource

## Description
As a primero edge admin, I should be able to access resource catalog to DELETE a resource available for any district so that the resource cannot be accessed by any user in Perseus application.

## Acceptance Criteria
# User shall be able to click on DELETE button
# User shall be able confirm the DELETE from confirmation prompt
# User shall NOT be able to view the deleted resource in resource table

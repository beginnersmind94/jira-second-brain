# NXT-45472 — IM - Menu Items - Variety Menu Items - Display inactive menu items

## Description
As a Menu Planner I want the ability to deactivate menu items so I can exclude them from possibly appearing as available in Menu Planning.

h2. Variety Menu Item - Inactive child Menu Item example

!image-20231122-153425.png|width=91%!

## Acceptance Criteria
* Update all Variety Menu Items containing inactive menu items to highlight the inactive item(s)
* Show the menu item in a disabled styling 
** this styling will be used as a common styling for all inactive scenarios (should be reused in MP and Production as well)
* Provide a tooltip error alert icon to indicate the menu item is inactive
** “This menu item is inactive and will be removed upon resaving”
* Show the inactive menu items until the Menu Items card is resaved
** allow users to access the Comments/Directions for inactive menu items

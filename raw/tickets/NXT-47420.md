# NXT-47420 — IM - Find & Replace - Ingredients - Remove

## Description
As a Nutritionist or Menu Planner, I would like to be able to quickly remove all instances of an ingredient from recipes, so I can remove an ingredient without replacing it with something else.

## Acceptance Criteria
* Once recipes are loaded by the Find action, display the Replace With section
* Until a replacement ingredient is selected, disable the Replace button and display the *Remove* button
** once an ingredient is selected, hide the Remove button and show enabled [Replace button|https://primeroedgenext.atlassian.net/browse/NXT-41624]
* If user clicks Remove button, enable the recipe grid for selection (checkboxes)
** For each selected recipe, show warning in the action column (TBD)
* Enable Remove icon in sticky footer pending selection of at least one recipe
* Show counter in sticky footer
** ‘Removing from X recipes’

# NXT-3695 — Training Events - 'Comments' 

## Description
Admin can add comments to the Training Event section for Employee Details. 

* UI - renaming ‘Notes’ to ‘Comments’ in the Training section and Education section for consistency 
** “Notes & Documents' will still be the last section
* add comments on initial entry in the text field
* comments are saved on initial entry and edit
* user can add comments during edit by typing in the text field 
* clicking the numerical hyperlink will show popup of comments entered

## Acceptance Criteria
Comments are added on initial training event entry and/or when the user clicks edit pencil and saves

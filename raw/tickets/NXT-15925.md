# NXT-15925 — INV - Order Template Details - Sites tab - Update Assigned Sites

## Description
As an Operations Manager I want the ability to assign sites to an Order Template so I can choose which sites or site types will use certain templates.

Continuation of [https://primeroedgenext.atlassian.net/browse/NXT-15986|https://primeroedgenext.atlassian.net/browse/NXT-15986|smart-link]

*Requirements*

* In the Order Template Site List, a selection column is provided for each Site and Site Type (per [NXT-15986|https://primeroedgenext.atlassian.net/browse/NXT-15896])
** All sites that are checked should be assigned to the template
** Checkboxes should reflect existing site configuration when page is opened
** User can change site configuration by selecting/deselecting checkboxes
* Provide a Save button
** Save button should only be enabled if one or more site selections are changed
** Save button will update the template with selected sites
** Provide success message after saving changes
*** Message: “Successfully saved template assigned sites.”
* Provide a ‘Cancel’ button
** Cancel button should only be *visible* if one or more site selections are changed
** Cancel button should be hidden if user reverts site selections to the currently saved selections
* Prompt on unsaved changes on tab exit
** If the user clicks the ‘Items’ tab and they have unsaved site selection changes, provide an unsaved changes prompt (default/generic)

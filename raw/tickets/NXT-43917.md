# NXT-43917 — FO - Notify - Show active letter templates

## Description
As Anna the Application Processor I want to be reminded of the active letter templates so I don’t send out the wrong letter by mistake.

*requirements*

* On the Notifications tabs, tell the user what the currently active template names are when they click on the Notify action.
** Find these from the Templates page:
*** Applications → Approval/Denial Notice → Template Name
*** Direct Cert → Direct Certification → Template Name
*** Carryover → Carryover Expiration → Template Name
** Text to use on the respective tabs: Active Template: TemplateName - LanguageCode. Example: 23-24 Approval Notice - ENG

## Acceptance Criteria
* Current active templates shown in the Notify frame.

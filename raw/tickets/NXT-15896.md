# NXT-15896 — INV - Order Template Details - Sites tab - View Assigned Sites

## Description
As an Operations Manager I want a Sites tab on the Order Template Details screen so I can view the list of sites that are currently assigned to the template I am viewing.

This is a second tab on the main Order Template Details screen (started as part of [NXT-14810|https://primeroedgenext.atlassian.net/browse/NXT-14810])

*Requirements*

* Add a second tab on the Order Template Details screen titled ‘Sites’
** this tab will be to the right of the ‘Items’ tab
* On the new Sites tab page, provide a grid containing the list of sites within the district, titled ‘Order Template Site List’
* The ‘Order Template Site List’ grid will display all sites in the district
** Sites will be grouped by Site Type (e.g. Elementary Schools, High Schools, etc.)
*** Sites should be indented within their Site Type groupings
** There will be a selection column
*** a checkbox will be provided for each individual Site
*** a checkbox will be provided for each Site Type 
**** selecting (checking) a Site Type will select all sites within the Site Type
** Site Code column - displaying the numeric ‘Site Code’ value for each site
*** This will be a searchable/sortable column
** Site Name column - displaying the name of each site
*** This will be a searchable/sortable column
** In the right corner of the grid, display a ‘Number of Sites Selected’ label
* Provide ‘Save’ and ‘Back’ buttons ([NXT-15925|https://primeroedgenext.atlassian.net/browse/NXT-15925])

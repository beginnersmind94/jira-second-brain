# NXT-53995 — FH - Apply - Text editing for apps being turned off

## Description
As Cynthia the CEP District Admin I want a way to edit the text on the Apply page when FRE Apps are turned off so households are better informed and do not call us.

!image-20240815-171654.png|width=1687,height=432,alt="image-20240815-171654.png"!

!image-20240815-171933.png|width=1457,height=607,alt="image-20240815-171933.png"!

*requirements*

* Add a Step where the user can see and edit the Apps Offline text - last step in the dropdown.
** Nice to have: Only show this step if the ‘Override SchoolCafé Online App acceptance settings (1-No, 2-Allow submissions, 3-Prevent submissions)’ is set to 3.
** Step name: Prevent Application Submission
*** Default Message: Your district is currently not accepting online applications. For any questions, including applying for next school year, please contact your school district. Their contact information is on the Support page of your SchoolCafé account.
* This text is only shown if the override setting is set to 3 (Prevent Submission)
** Otherwise it’s the standard workflow.
* Tooltip update for the settings:
** If set to ‘Allow submissions’, this allows parents to submit meal applications (or surveys, if applicable) even if today’s date does not fall within the “First day” and “Last day” window to submit an application. The opposite goes for ‘Prevent Submissions’ and you can edit the messaging for this on the Templates page. Remember to set this back to ‘No’ when you no longer want to override the SchoolCafé setting!



!image-20241113-151946.png|width=1274,height=368,alt="image-20241113-151946.png"!

## Acceptance Criteria
* Prevent Applications Submission step added to the template wizard
** Nice to have: Only show this step if the Override setting is set to 3.
* User can view and edit the text with the template wizard
* Edited text is used on the Apply page if the Override setting is set to 3.
* Setting tooltip updated

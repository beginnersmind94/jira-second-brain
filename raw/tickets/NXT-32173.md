# NXT-32173 — DataQuery Framework - Data Model Creation Process Document

## Description
A data model creation process definition/document and scripts is needed for module teams to know how to efficiently and successfully create data models for their modules.

## Acceptance Criteria

** What is a DataQuery Framework Data Model
** How to efficiently and successfully create a data model
** How to make a Data Model available for use in DataQuery Framework Custom Views and Reports

# NXT-3629 — Edit Inspections - Scale Question Type

## Description
*User Story*
As an Inspections user, I can see Scale Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Scale Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Scale with proper number of selections
** Labels, if present
** Whether question is required
** Notes Section
** Upload Media
** Note: Point values are *not* displayed here
* User can only select one option from the scale
* Choices are properly saved
** If question is required, Inspection cannot be submitted if no selections are made
* Choices can be changed if Inspection has not been submitted
* If question has a score associated with it, the score is properly updated according to the selection being made.
** Score is added based on the rating given (for example, giving an 8/10 adds 8 points to the total score out of 10 possible)

## Acceptance Criteria
* Scale question appears in Inspections
* Scale and notes can be selected, saved, and edited
* Score updates properly if present

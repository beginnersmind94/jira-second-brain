# NXT-39905 — IM - Items - Single Item Recipe - Serving Size & Yield Factor

## Description
As a Nutritionist I want the Single Item Recipe configuration to allow me to create a “prepared” version of my serving so that it may reflect the serving size after the Yield Factor calculations are made.

*Requirements*

* Create an additional variant of the Single Item Recipe serving size if Yield Factor values are applied
** this variant would represent the serving with the adjusted Weight/Volume according to the Yield calculations
** Pre-fill description with a default value that is identical to the selected serving but with ‘(Prepared)' after 
*** Ex: default serving is *4 oz*, so the prepared serving would be *4 oz (Prepared)*
* If a sub-serving size is being created, do not create an additional (3rd) variant of the serving
** Instead, simply apply the adjusted weight/volume to the sub-serving size
** still apply the ‘Prepared’ label 
* Create the additional serving size as a regular serving size available on the Nutrients Info tab and available as an ingredient on recipes

# NXT-17983 — INV - Settings - Delivery Locations - Cleanup / Update Orders

## Description
Update the usage of Delivery Locations throughout the Orders process to accommodate the functionality of Delivery Locations as specified in the new stories: [Delivery Location List|https://primeroedgenext.atlassian.net/browse/NXT-16355], [New Delivery Location Configuration|https://primeroedgenext.atlassian.net/browse/NXT-17812], [View/Edit Existing Delivery Location Configuration|https://primeroedgenext.atlassian.net/browse/NXT-17813]

*Requirements*

* Modify existing API
* Retain Delivery Location selection on orders
** If a location is made inactive but the location is used on an order, retain that selection unless user attempts to change the location
* Display a label instead of dropdown if a site only has one location configured
** on Create Orders - display a label instead of dropdown
** on Order Detail - hide the edit icon

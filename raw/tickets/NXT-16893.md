# NXT-16893 — INV - Orders - Review Order Settings - Add New Rule - Confirm and Create (Step 6)

## Description
As an inventory operations manager I want a Confirmation step on the Add New Order Review Rule process so I can assign a date range to the rule and complete configuring my rule.

*Requirements*

* This is the sixth and final step of the Add New Rule process of the Review Order settings
* Display a summary section of the rule configuration
** this should be a vertical list that summarizes the configuration of each step
* Within the last row of the grid, provide a Start Date selection
** this is mandatory
** use date picker control
** default to current date
* Provide an End Date selection
** this is optional and disabled by default
** use a switch/toggle to enable the End Date date picker
* Provide an Active flag
** Yes/No radio buttons
** Yes by default
** user can switch to Inactive option to make a rule inactive
* Display a Cancel button in the bottom left corner, to cancel and return to the list page
** Prompt user: “Discard rule configuration and exit setup?” Yes/Cancel
* Display a Previous button in the bottom left corner, to return to the previous step
* Display a Create button, to officially create and save the rule
** do not enable this button until at least a Start Date has been selected
** exit the stepper and return to the list page ([NXT-5716|https://primeroedgenext.atlassian.net/browse/NXT-5716])

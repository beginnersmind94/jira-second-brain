# NXT-3213 — Inspection Template - Inactive Status

## Description
*User Story*
As an Inspections user, I have an Inactive state for Templates that are not available to be used to create a new Inspection.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1|https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1]

*Requirements*

* Templates have an Inactive status
** Inactive templates appear in the Inactive portion of the Template Hub screen
* Inactive templates can be edited
* Inactive templates can be moved to Active
** There is an action available in both Card and List view to do this
** The action is present as a button in Card View

## Acceptance Criteria
* Templates can be moved from Active to Inactive and back
* Inactive Templates appear in the Inactive section of the Template Hub screen

# NXT-15668 — IM - Items Search box/Search result display 

## Description
As a User I would like to have a Search option on the Items page So that I can quickly search for the items

## Acceptance Criteria
Given User is on the Items Page 

When Search bar accessed with (GTIN, Item Name, Item code, Data Source, Item Short name)

Then present the Scrollable items list based on the search result

Top 50 items to be displayed 

And as the User scrolls down 

Then load more Items as the scroll continues

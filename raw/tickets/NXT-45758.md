# NXT-45758 — IM - Items/Recipes/VMI - Simplified POS Pricing

## Description
As a District Admin I want the POS Pricing to be simplified for standard menu items since these are almost always A La Carte items with only one price.

h3.

## Acceptance Criteria
* Implement a simplified control for enabling POS sales to various eligibility groups
* Implement a simplified, singular price input field on the Menu Info card which can be used by default
* Provide an icon/button to access the “Advanced Pricing” as an alternative to the singular price input
* for Advanced Pricing, load the full POS Pricing grid in a large slide-out drawer 
** Remove the POS Pricing tab for Items and Recipes (maybe VMI too… TBD)
* If multiple different prices are provided in the Advanced Pricing slide-out, for either eligibility groups or site types, disable the primary single price field
** show a ghost text of ‘_Multiple_’ in place of the usual dollar value

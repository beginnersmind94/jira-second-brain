## Acceptance Criteria
* Restructure the Contracts tab of the Vendor page
** Remove the individual View Contract screen
*** Change the view contract icon to an edit pencil and directly open the sub-screen in edit mode
*** Move the Save action to the sticky footer
** Remove Contract Type (it’s only visible here)
** Merge the status into the Contract Number column
** Use an expandable sub-grid to show the other contract attributes
** Add a column for Shipping Groups for internal vendors ([59611|https://primeroedgenext.atlassian.net/browse/NXT-59611]) 
* Remove some fields (from both internal and external vendors)
** Contract Category 
** Contract Amount
* Add ‘Batch Price’ as a new value to the Receipt Item Price Condition attribute
** for internal vendors, force this selection and do not allow edits (for now)
* Add new option(s) for internal ordering
** Allow Sites to see QOH
*** default to *NO*
** Use Actual QOH or Projected QOH
*** this option is only available if the Allow option is enabled
*** default to *Actual*

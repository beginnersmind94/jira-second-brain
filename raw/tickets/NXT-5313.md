# NXT-5313 — Onboarding - System Task List - Users - Display Users Tour

## Description
As a new SchoolCafé user, I would like to have a basic walkthrough on the steps required to set up my district’s users, so that I don’t have to spend time guessing what to do next.

*Summary*

When the user comes to the Users task, display a simply tour pop-up (with a Finish button), that explains what the user must do to ‘complete’ this task.

*Mockup*

!image-20230523-213627.png|width=448,height=343!

*Details*

In Onboarding → System → Tasks, when the user selects Users and is redirected, do the following:

* Display a tour in the center of the screen:
** Text:
*** Title:
**** *How to Complete This Task*
*** Body:
***# Add at least one user. We've already added you, but you can also make changes if necessary. 
***# Note that users will get an email to verify their account. To prevent that from happening before you are ready, you can skip those users and come back to this screen later (or after completion of onboarding).
* Display a ‘Finish Tour’ button in the footer.
** When the user selects this button, close the pop-up. This should prevent the tour from executing when the user logs out and returns to this page.

## Acceptance Criteria
Verify that when the user comes to this task, they are immediately presented with a tour step in the center of the screen

Verify that the text and button placement of the tour match the description above

Verify that when the user selects ‘Finish Tour’, the pop-up closes and the user can proceed

Verify that the user will not see the tour again when they are logged out and return tot he page

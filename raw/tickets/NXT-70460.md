## Acceptance Criteria
* Allow Menu to be created using template
* After header section has been created and saved, provide a drop down for templates
** Only templates matching the menu criteria should be available
* Once user selects a template and applies the template, prepopulate with the information from the template
** Menu Item Category Choose Counts
** Serving Group Counts
** Preview of the menu items that are being added as part of the template
* Once confirmed, add a chip for template used and make it clickable to allow user to navigate to that template if needed in a new tab
** Add tooltip for the name of template used

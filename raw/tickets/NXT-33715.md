# NXT-33715 — INV - Receipt Detail Report - Update for Pack Sizes

## Description
As a Site Manager I want the Receipt Details report to reflect the way items were entered on the receipt so I can see all unit and price details.

*Requirements*

* Update the Receipt Details Report to display pack size units as they are displayed on the Receipt screen
* UI Corrections
** Move Item #, Vendor Item # and Item Description into their own header row (technically missing labels)
** Make the table itself dedicated to the Quantity and Price fields (and Unit Description of course)
** Make the Date on Product sub-grid only appear if the user entered values for it and size it so that it doesn't span unnecessary width
** Do the same thing for the Comments section, and share the same space with the Date on Product sub-grid

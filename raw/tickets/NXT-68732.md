# NXT-68732 — IM - Bulk Item - Update Use a File

## Description
As a district manager using the Bulk Update tool, I would like to be able to have the system give me an excel file to take a look at the data already provided and to make changes and reupload

## Acceptance Criteria
* Provide only the ability to Download Template with Data
* User can only edit items and can not add new items 
** Since we can not add that type of condition in excel, disregard any new items added on excel

## Acceptance Criteria
* Provide the Add Item button for distributions in Pending Assignment status
** Use a slide-out panel to search for items to be added
* Provide a smart search at the top of the panel
** search the contract items by Item #, Vendor Item # or Description
** Once an item is selected, allow it to be added to a preview grid below
* Display a ‘View Contract’ button to open the Contract Items page in a new tab
** pre-select the contract in the dropdown
* Display a preview grid where added items will be displayed 
** Item #
** Item Description
** Is a Substitution column
*** checkbox option
** Action column
*** Delete icon
* When ‘Is a Substitution’ is checked, display a row below
** display ‘Substitute For’ label
** provide a dropdown to select an item from the distribution to associate as a substitution 
* Provide a footer in the slideout
** Cancel button
** Add X Items button
*** only enabled if all substitutions are selected 
* Display added items uniquely in the item list once added
** Display a dark orange plus (+) icon next to Item # for added items
** Display a regular orange swap 🔁  icon next to Item # for substituted items
** Display hyphens (--) for Quantity Ordered
* Substitutions covered more in [https://primeroedgenext.atlassian.net/browse/NXT-60322|https://primeroedgenext.atlassian.net/browse/NXT-60322|smart-link]

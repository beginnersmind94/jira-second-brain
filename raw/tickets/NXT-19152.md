# NXT-19152 — Adjust Menu Item Serving Size for each Serving Group in a Menu(UI)

## Description
Once I've created a menu, there may be different serving measures of food that I use for different serving groups. 

Older students might receive a larger portion size of an item than younger students. An elementary student (K-5) might receive 3/4 cup of corn, while a PK student might only receive 1/2 cup of that same corn.

I am able to create the different serving sizes on the menu items, but currently, there's no method of selection when creating the menu. Specified servings can be defaulted from recipe management but need to be editable on the menu page. 

!2.0_MP_ServingMeasure.png|width=200,height=183!

## Acceptance Criteria
When I create a menu, or add a menu item to a menu, I will be able to edit the serving size that I want to give to any of my serving groups, when I have created multiple serving sizes on the menu item. 



Specified servings can be defaulted from recipe management but need to be editable on the menu page.

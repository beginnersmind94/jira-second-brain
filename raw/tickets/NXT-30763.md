# NXT-30763 — IM - Config - Menu Item Categories - Merge POS Item Categories

## Description
As a District Admin I would like to be able to configure POS item categories as an extension of Menu Item categories so I do not have to create redundant categories for each area I use items.

Coordination required with Front Office team, for existing POS Categories (under Accountability)

*Requirements*

* Add new options to the Menu Item Category configuration slide-out
** Menu Item checkbox
*** this would mean the category is available for assignment to Menu Items 
** POS checkbox
*** this would make the category available for assignment to POS items
** default checkbox values accordingly for migration of existing data
** Otherwise, any categories would be available for both Menu Items and POS items
* Add a Type column
** Menu & POS
** Menu Item Only
** POS Only
* Update the Menu Item & POS Item Category dropdown on all IM screens which reference
** Menu Item categories should be on all instances of updated Menu Item 
** on the ‘POS Only’ screen, show only the POS categories (excluding Menu Item categories)

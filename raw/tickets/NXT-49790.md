# NXT-49790 — Retrieve Users count from DB Tables on Roles & Permissions Page

## Description
Currently on Roles & Permissions page the users count is being retrieved from keycloak for each role. In background there were numerous number of calls being called to keycloak to retrieve this data.  In sprint 116, there is a story that was created to insert UserXRole, UserXSite tables which includes mappings between users and roles.  We can utilize these tables to fetch the data about users count for each role to achieve the best performance and quick load times

# NXT-19301 — IM -UI -  Item Details - Hierarchy Details - Navigate to related item

## Description
As a user I want a link for each related item on the hierarchy so I can navigate to that item without returning to the Item List and searching.

*Requirements*

* Per story [NXT-19300|https://primeroedgenext.atlassian.net/browse/NXT-19300], we are providing a View icon on the Hierarchy Details card for the related items
** add a tooltip to the view icon: “View item details.”
* When the user clicks this icon, we should open the Item Details page of that selected item
** preferably do this in a new tab
** **Note:* this may only work in List View. What can we do for Split View?
* When the new Item Details page is loaded, we should navigate the user to the hierarchy section
** *Note: this may not be possible with card view

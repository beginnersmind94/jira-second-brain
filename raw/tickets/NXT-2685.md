# NXT-2685 — Deploy Identity Server and loginService on IIS (local)

## Description
Deploy Identity Server and loginService on local IIS Server and make it available for UI team so that they can get authorization token for ResourceOwnerPassword flow.

## Acceptance Criteria
On hitting the api/auth/token service end point with username, password, client_id, client_secret. user should be able to get the access token.

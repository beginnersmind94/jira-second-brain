# NXT-9536 — Data Exchange - File Import Enhancements

## Description
As a PrimeroEdge user, I should be able configure import jobs with specific module and import type with a specific name so I can save and select import configuration whenever I want. 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=4772%3A75788&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=4772%3A75788&scaling=min-zoom|smart-link] 

*Summary*

# In the Data Exchange Pre-Upload, user should be able to provide a name as “Import config name” after import type selection tagged for each configuration that user creates
# Each Import configuration should  has to have unique name in the system
# After pre-upload screen, user should be able to select any configuration to load from a drop down with list of existing import configurations
# If user select existing configuration, it should retain the configuration name
# File Imports Monitoring page  lists all import jobs includes File Import Configuration name

## Acceptance Criteria
* Verify that Configuration name is displayed in Pre-upload screen for any given import type

* Verify Import configuration name is captured and listed in existing configuration detection

* Verify File Import Monitoring page includes Configuration Name column in the table

# NXT-18732 — INV - Manage Orders - Receipt Detail - Track Receipt History

## Description
As an Inventory user I want the system to track history on receipts so I can view historical receipt transactions.

This is a back-end story. Receipt history will be viewable per [https://primeroedgenext.atlassian.net/browse/NXT-18615|https://primeroedgenext.atlassian.net/browse/NXT-18615|smart-link]

*Requirements*

* Track history for receipt transactions
* Historical transactions will exist for the following actions:
** Receipt created - original transaction
** Receipt adjustment
** Receipt cancellation

# NXT-54657 — PEI - Auth - Enforce Insights License

## Description
We need to ensure that PrimerEdge Insights (PEI) is only usable by districts with a valid license so that we don’t have unauthorized usage

*Requirements:*

* Ensure that district the user is coming from has a valid PEI license
* If they do not:
** Log them out if they are logged in
** Redirect them to the log out screen
** On the logout screen, show this message:
*** Insight license is required to view this site. Contact Customer Support for further help.

## Acceptance Criteria
* Site checks if the user’s district has a valid license
* A user cannot use the site if their district does not have a valid license

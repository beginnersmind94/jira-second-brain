# NXT-2750 — IdentityServer4: Revoke Token Feature for Access and Refresh Token

## Description
Implement revoke token function so that issued access and refresh token for clients can be revoked in case of misuse.

## Acceptance Criteria
By given the refresh token or access token, System should be able to revoke. And user should not be able to access the resource API(s) for that client.

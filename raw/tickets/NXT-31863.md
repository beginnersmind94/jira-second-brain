# NXT-31863 — IM- Reports- Allergen Report (Menu Items on Assigned Menus Option)

## Description
As a Registered Dietitian, I would like to be able to see all my menu Items that have allergies and what allergies do they have.

## Acceptance Criteria
* Allow user to pick what menu items that would like to see based on criteria.
** Either by Marketing Name or by Menu Item Name along with a way to indicate if they would like the ingredient breakdown based on the menu item
** If User Picks Report Option “Menu Items on Assigned Menus”, Allow a From and to Date selection, site (Default to all) and Meal Type (Default to All)
** Display on report:
*** Which Report Option they picked and Date Range Selected
*** Site (Only show Site Name if Site selection in criteria is All)
*** Menu Date (What date is this Menu Item Assigned within the date Range)
**** Display every time the menu item is assigned within the date range 
**** Display Menu Items from the start of the earliest date to latest in order of Breakfast, Lunch, Snack, Supper and Site 
**** EX:
*****  Aldine Highschool 
****** 5/1 Breakfast
****** 5/2 Breakfast
****** 5/3 Breakfast
****** 5/1 Lunch
****** 5/2 Lunch
***** Black Middle School
****** 5/1 Breakfast
****** 5/2 Breakfast
****** 5/3 Breakfast
****** 5/1 Lunch
****** 5/2 Lunch
*** Meal Type (Only Show Meal Type if Meal Type Selection in Criteria is All)
*** Data Source
*** Recipe Code
*** Menu Item (Menu Item Name)
*** Allergen(s)
* Allow user to pick what menu items they would like to see based on allergens from the standard allergens (Default to select All) 
** Crustacean Shellfish
** Egg
** Fish
** Gluten
** Milk
** Peanuts
** Soy
** Tree Nuts
** Wheat
* Allow user to pick what menu items they would like to see based on allergens from the custom allergens list (Default to select All) 
* Display a way for user to generate report based on all criteria selected

# NXT-21157 — Display additional information on Item Catalog Landing Page

## Description
As a user I need an option to see/look for additional information in in the data grid of the Item Catalog Landing Page



* On landing to Global Catalog -> Item Page
**  Default  Fields To Display When Landing
*** MANUFACTURER
*** GTIN
*** PRODUCT CODE
*** ITEM NAME
*** BRAND NAME
*** ITEM CATEGORY
* Always on data grid (not removable)
** MANUFACTURER
** GTIN
** ITEM NAME
* Optional Fields on Data Grid:
** PRODUCT DESCRIPTION
** SHORT NAME
** CHILD GTIN
** PARENT GTIN
** PURCHASE UNIT
** IS FOOD
** IS COMMODITY
** STATUS
** GC ITEM CATGORY
** GC STORAGE CATEGORY
** IS BUY AMERICAN
** VALUATION GROUP NAME
** VENDOR NUMBER
** VENDOR NAME
** VENDOR TYPE
** UPC 
** When user click on reset button the system should return save view  if created or default view if not on a save view
* Ability to check fields to be displayed from the drawer
* Ability to uncheck fields to be display from the drawer
* Ability to reset fields selected to default selection
* System should load the saved fields when opening the screen again until it is reset to original fields.

## Acceptance Criteria
The user should be able view “Manufacturers” configuration button as shown in UI

Upon clicking the button the system shows the slide out as shown in UI



*Adding a New Manufacturer:*

User should be able to enter the manufacturer name, upon clicking the ADD button - it should be added to configuration

In case of duplicate entry - system should alert the user with error message (“Manufacturer already exists, please enter a different Manufacturer”)



*Editing a Manufacturer:*

User should be able to click on Edit icon and update the description as needed

The unique validation should be applied in case of duplication again



*Deactivating a Manufacturer:*
The user should be able to click on the icon and de-activate the entry only when the entry is not used in any Item
the deactivated Manufacturer are not listed in the dropdown while creating or editing an item

Deactivate item should be disabled  for manufacturer which are used

*General Rules:*

The system should store the values with a flag that user has created the configuration
Only user added entries should be listed in this configuration

The manufacturer listed during item  creation/edit should consolidated list from catalog + CND

# NXT-14725 — IM - Catalog Integration - Add Global catalog Menu option 

## Description
As a User I would like to access Global Catalog (GC) and navigate thru its functionalities by selecting them (GC functionalities ) via a  menu bar located at the top of the screen so that I can easily navigate Item Management (IM) - GC import functionalities.

Under the Menu Name “Item Management”  there must be a Menu Item option named : “Global Catalog”. 

Once the Menu Item “Global Catalog” is selected  a sub-menu of item Names should appear offering the following options:

* Item Catalog
* My Items
* Item Synch History
* Pending Updates
* Settings

Once a sub-menu option is selected/clicked on then the application should direct me to the specified page (where desired functions are located).



NOTE: This menu should always appear at the top of the screen.

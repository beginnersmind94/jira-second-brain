# NXT-17099 — Platform - Licensing - Add/Reduce Licenses for a Product

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to view list of products we are enrolled and add/delete number of licenses in order to expand/reduce sites.

*Mockup:*

[https://www.figma.com/proto/nXdAgweY5FV1A3z87rX1wo/Add-Reduce-Licenses?node-id=1%3A40&scaling=min-zoom&page-id=0%3A1&starting-point-node-id=1%3A40|https://www.figma.com/proto/nXdAgweY5FV1A3z87rX1wo/Add-Reduce-Licenses?node-id=1%3A40&scaling=min-zoom&page-id=0%3A1&starting-point-node-id=1%3A40|smart-link] 

*Requirements*:

* Each Product must have a “ADD” icon to add/reduce licenses
** Clicking on add/reduce, should bring up the sites with active licenses being used
* For each site, ADD action dis displayed
** Existing active license usage should displayed for all active sites
** Cancel button should be enabled for user to go back to products table
* Once user submits the form, a confirmation should be displayed with message “approval request for license addition has been sent to central office” 
* Under REDUCE/DELETE action
** Cancel button should be enabled for user to go back to products table
** Once user submits the form, a confirmation should be displayed with message “approval request for license updates has been sent to central office” 
** Notification is sent to user & central office team to review the request via Workspace tasks
* User shall be able to filter All sites, sites with licenses and sites without licenses

## Acceptance Criteria
* User shall be able to use ADD/DELETE license button to see list of active licenses being used
* User shall be able to ADD licenses from ADD action at site level
* Confirmation is prompted for any changes in number of licenses
* User shall be able to request maximum 49 licenses per request per district
* User shall be able to select validity of the license during the request
* User shall be able to filter All sites, sites with licenses and sites without licenses

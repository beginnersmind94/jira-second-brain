# NXT-47314 — remove - old files view history

## Description
As a SchoolCafé admin, I would like to be able to View the audit history of File import configuration changes so that I can see who made a change and when.

*Note:*

* We already do audit / history tracking a bunch in System team, so please copy what they are doing
* Do not reinvent this, use the framework they have created
* Do this the same way you do for the API History

*Requirements*:

* For File Imports tab’s grid, do the following:
* In the ellipsis, add a button to the actions:
** Title: View History
** On click, open the History drawer
* 
* Create a new drawer
** Title: File Import History
** Size: same as in other system screens
** This is a rea-only screen, no edit
* 
* Use the components created by System team to display the audit history
* On page load, show the history of all Add & Edit actions for that specific File import

## Acceptance Criteria
* User can see the add and edit audit history for any given File import configuration
* The drawer uses the existing history component created by System team

# NXT-1816 — Inventory - Orders - Back Orders (Report)

## Description
As a Food Service Director I want a Report to show the list of orders for a given site and date range from one or all vendors. 

*Requirements*

* Add a new report page 
** this is accessed via the new report landing page
* Provide Site control selections
** Area (if applicable) + Site
** default to ‘ALL’
* Provide a Vendor dropdown
** multi-select
** default to ‘ALL’
* Provide an Order status dropdown
** Open, Closed, All (both) are the options
** Open is the default
* Provide a Date Range filter
** this applies to the order’s Delivery Date
** default to the past 7 days 
* Generate the report on click of Generate button
** List each site on its own page 
** List all orders and item Quantity if the difference between order Qty and Received is > 0 and in Open/closed status. 
**  Header section shows: site name if all sites selected  then show “ALL sites'“ , date range
** Group by Vendor name 
** Item details shows: 
*** District item # 
*** Item Description 
*** Vendor item #
*** Order #
*** Status
*** Delivery Date 
*** Ordered Qty
*** Received Qty (received Qty is sum of all Quantity received for that order from all receipts. )
*** Back order Qty (the difference between order and total received Qty)
**** If Received Qty is greater than Order Qty, that will be a negative number and the items should not be displayed on the report

Mockup note: use separate date pickers for now

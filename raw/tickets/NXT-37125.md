# NXT-37125 — PR- Reports- Production Record and Production Form Serving Groups and Temperature Details

## Description
As a user, I would like to be able to see all the serving groups that are being served on this Production Record. Provide serving group list within one “Serving Group” Column. Temperature tracking should also be hidden if the temperature tracking setting is turned off.

## Acceptance Criteria
* Provide Serving groups that are being served within a particular production record as so 
** This is an example:

||*Menu Item*||*Serving Size*||*Serving Groups (Count)*|| ||
|Mac and Cheese|2 Each|6-8 ( 100)
9-12 (150)| |
|Mac and Cheese|1 Each|K-5 ( 100)| |

All remaining columns will remain the same as previous implementation

* Temperature display should consider settings as well as CCP process associated with the particular recipe
** Provide adequate space for user to hand write the temperature (Only truly applies for Production Form)
** Only Display Underscore with a place to enter temperature if it is not already provided electronically
** Display Values if provided electronically

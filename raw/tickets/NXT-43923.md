# NXT-43923 — IM - Recipe - Consume Preferred Image flag

## Description

** First, use Preferred Menu image
** Second, use Preferred Ingredient Image
** Otherwise, use any image
*** Ignore images with a Document Type assigned (CN label, etc.)
* Do not consider Document type options (CN label, etc.)
* Update the Recipe Steps card and Ingredient Search card to consume the Preferred Ingredient Image when adding as a sub-recipe/ingredient
** Update the Custom Menu Item “ingredients” card as well

# NXT-18641 — INV - Orders - Manage Orders - Quick Receipt - Save as Incomplete in Edit state

## Description
As a site manager I want the ability to save an existing Quick Receipt order in Incomplete status so I can make updates to a Quick Receipt that is still in progress.

*Requirements*

* After the user opens an existing Incomplete Quick Receipt ([NXT-18050|https://primeroedgenext.atlassian.net/browse/NXT-18050])
** User can once again Save as Incomplete
*** This will update previously saved values
*** This will *not* change or create an Order #

## Acceptance Criteria
* Incomplete Quick Receipts can be again saved as Incomplete
* Previously saved values will be updated

# NXT-31686 — PR- Post Production Analysis- Adding indication of "Offered" on all reports (UI Story)

## Description
Based on USDA feedback, “The software must allow the user to create the weighted analysis report on the offered (prepared) menu. The user must have a way to designate the menu analyzed as the offered menu.”

a.      All Post-Production reports need to be labeled “Offered” (can be in parenthesis); this label should be on each report and also indicated under the “reports” menu

## Acceptance Criteria
On Production Reports Page, Add the word offered to the report name and reflect it on the criteria page once clicked on.

Once User Clicks on Analysis, also change the verbiage on the slideout header

Current: Post-Production Analysis 

Proposed: Post Production (Offered) Analysis

On Nutrition, Nutrition Summary and Food Component Report, there must be an indication that “All calculations are based on meals offered” 

Changes will reflect on the print options as well as UI

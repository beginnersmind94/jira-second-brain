# NXT-15171 — Platform - System - Improve Roles & Permissions hierarchy representation

## Description
This EPIC handles all the efforts required to improve UX for Roles & Permissions page so that the page navigation simple and user friendly for site admins/users

*Requirements:*

* Update Roles & Permissions screen to enable consolidated view of mappings between role and permissions instead of navigating multiple levels

*Latest Mockups:*

[https://www.figma.com/file/lFlockasIudWFoWBiitWS4/Roles%26permissions?node-id=425%3A44|https://www.figma.com/file/lFlockasIudWFoWBiitWS4/Roles%26permissions?node-id=425%3A44|smart-link]

## Acceptance Criteria
* Role & Permissions shall be displayed with single expandable panes.
* Modules shall be displayed in a tabular fashion based on user permissions.
* Changes in checkboxes shall be saved automatically
* Roles are sorted by name by default
* Columns alignment after expanding shall be consistent

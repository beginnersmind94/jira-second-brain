# NXT-20762 — Trained Menu Items Report

## Description
As an administrator, I want to see how many pictures I have for my menu items and be able to see any given image so that I can determine which menu items need more training.

*Application*: Face and Food Detection Website

*Flow:*

* Browse to PrimeroEdge and click the link for Food and Face Detection
* Browse to the Reports
* Browse to Trained Menu Items Report
* View data

 

*Assumptions:*

* If you have done the View Model Page story, you have already developed 90% of this page
* Copy and paste from there as much as you can
* If you are doing this one first, then copy this code to that one when you get to it

 

*Requirements:*

* Create a new reports page with the title Trained Menu Items
* In the menu navigation, create a header called Reports
** Ensure the header is consistent with EdgeCafé website menu headers HTML
* Create a new menu link under Reports named Trained Menu Items for the new page
* Move the current menu link for Accuracy Report under the Reports header and rename it Accuracy Report
*  
* All fields are read only, there are no action buttons except grid behaviors and hyperlinks
*  
* Grid of menu items that are linked to the model
** Usual sort, paging, filtering, export to excel functionality
** Columns:
*** Menu Item (the name of the food)
*** Metadata (if it exists in the menu API)
*** Linked Images (a number that shows the count of linked images)
*** View Images – hyperlink button
**** On click, show the Images popup (see below)
* Filters for the grid
** Menu Item (the name of the food)
** Metadata (if it exists in the menu API)
*  
* Images Popup
** Do this the exact same way as View Model Page
** On load, get the images linked to that menu item and show them in an image grid similar to what you did for New Images
** Can click on the image and redirect to that Trained Images page with that image loaded
** Show the Menu Item as a title above the images
** Show a close button
** No other functionality necessary for now
*  
* Security and roles:
** Copy from Accuracy Report, do exactly as there, but repeated below just in case
** Each of these roles can view the results of this control:
*** Cybersoft FFD Admin
*** Cybersoft FFD Viewer
*** Food Detection Model Manager
*** Food Detection Model Viewer
*** Food Detection Report Viewer
** Unauthenticated users cannot view screen

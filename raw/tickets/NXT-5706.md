# NXT-5706 — INV - Ordering Conditions - Summary Page - Search

## Description
As a District Supervisor I want an Ordering Conditions summary page so I can view the ordering conditions I have created.

This will be a new page:
_Inventory > Orders > Ordering Conditions_

This will be the landing page when Ordering Conditions are opened. This page will contain dropdowns to filter the items listed on the page.

*Requirements*

Provide an Add button to create new Ordering Conditions ([NXT-5705|https://primeroedgenext.atlassian.net/browse/NXT-5705])

+Dropdown filters:+

* Storage Category
** display only Storage Categories which correspond to the item list
* Item Category
** display only Item Categories which correspond to the item list
** filter the Item Category dropdown based on Storage Category dropdown selection
* Data Source

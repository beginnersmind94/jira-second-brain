# NXT-1766 — INV - Reports - Purchases

## Description
List of all receipts created by one specific site (or all sites) from one specific vendor or all vendors for all items in a given receipt date range and with or without Additional charges. 

!image-20210928-222602.png|width=827,height=272!

!NXT-1766 1 (3a469b4c-cee7-4d66-999a-c4eaecef2d98).png|width=100%!

!NXT-1766 2-20220128-145619.png|width=100%!

 First page of the report: Shows the summary of selected Site(s) from selected vendor(s): table shows sites as rows , vendors as columns , last column being the total for each site from all vendors and Last row shows the total of all sites from a vendor. 

!image-20210928-222707.png|width=786,height=383!



 Second page onwards: Items details (I think it will sum the Qty if an item from a vendor is received more than once ) 

*  Sorted by Site name Vendor name , item details are sorted by purchase date, item description. 
*  Grouped by Site and then Grouped by Vendor 
*  columns Purchase date, district item#, item description, WU Qty, WU price, Total Amount, total Value according to fair market value of the items from that vendor. 
*  Charges and discounts section will be displayed if the check box is selected. 
*  Row to display the total of all additional charges. 
*  Last row display the total of Receipt amount from all receipts of that site from that vendor (grouped site and vendor. )
* if more than one receipt for an item on a day from a vendor then the Qty /charges and discounts will be added from all receipts. 
* !image-20210928-223236.png|width=830,height=593!

If a single item is selected ([~accountid:61703ed1b9c549006f78e230] )

* Display only the transactions containing that item
* show only the item itself in the transaction(s)
* show only the charges and discounts for the item (not receipt-level ones)

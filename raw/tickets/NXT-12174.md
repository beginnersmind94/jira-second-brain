# NXT-12174 — MP - Menu Cycle Reports - Nutrition Options

## Description
As a user, I need to be able to select Nutrition Options for Menu Cycle Reports

 !image-2021-03-11-14-07-05-330.png|thumbnail! 

Requirements:
* All (Default)
* Expanded
* Simplified

## Acceptance Criteria
User must be able to select nutrition options for menu cycles reports

The “All / Simplified / Expanded” options are specifically for the report. It impacts the Nutrient Summary on the report, and the menu items, by showing only specific nutrients and not “All” nutrients (without making the selection of “all”)

# NXT-55589 — Acc/AcctMgt - Transaction History - Couch dependency removal

## Description
With Couch being phased out / removed, need to update how we are handling the following pages re: data and delivery of requested downloads:

* Accountability > Transactions > Transaction History
* Accountability > Transactions > Search
* Account Management > Accounts > Transaction History 
* Reconciliation > Transactions



*requirements*

* Update the pages so it knows where to get this data from if it is no longer in Couch
* Print/Export functionality updated to go out via Document Center for large print jobs
** 500+ transactions will trigger this workflow
* Immediate file download for smaller print jobs (<500 transactions)
* Info icon added with hover/click text:
** Text: There will be a delay in exporting if more than 500 transactions are included.



Discussion:

* Account Balance HTML report needs to be removed (separate story for removing all HTML buttons from report pages in FO)

## Acceptance Criteria
* Following pages moved away from couch dependency:
** Accountability > Transactions > Transaction History
** Accountability > Transactions > Search
** Accountability > Sessions > Reconciliation > Transactions
* User able to access data after dependency removal.
* User able to download data after dependency removal.
** File download should be served up immediately for small prints (<500 records)
** Document Center for larger prints (500+)
* Info icon + click/hover text added to the pages

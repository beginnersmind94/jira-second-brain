# NXT-14622 — Platform - SSO - Enable session validation with timeout

## Description
*Persona*: Perseus User/admin

*Access level:* School District/Site

*Description*:

As a Perseus user, I should be able to login to Perseus using my registered account when there is a valid session established (15 mins )

*Requirements*:

* User with registered email account launches Perseus dashboard
* User is automatically redirected to dashboard without log-in prompt if the valid session exists
* If the user session is expired or not available, user is taken to SSO login page
*

## Acceptance Criteria
* User with registered email account launches dashboard without login prompt
* User shall be redirected to login page if the session is expired (>15 mins)
* {color:#ff5630}*User session to be restored with preserved data*{color}

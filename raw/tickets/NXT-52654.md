# NXT-52654 — Acc - Sessions - Role permission to ignore max over/under amount

## Description
!image-20240714-161404.png|width=995,height=289,alt="image-20240714-161404.png"!

*Requirements*

* Add a new role permission
** Name: {{Adjust Session Beyond Max Amount}}
** Description: {{Allows user with to adjust a session beyond the value set for setting 'Reconciliation - Maximum Over/Under Amount Allowed'}}
* If the user has this new role permission, they can reconcile a session even if it exceeds the max over/under amount from system settings MAXAMTRC and MAXAMTAURC 
** User needs to enter a comment in this scenario.



User has access to adjust sessions in the following locations:

* Sessions grid (Auto-Reconcile option)
* Inside a Session (Reconciliation → Adjusting over/under)
* End of Day page (Auto-Reconcile for Sessions task)

## Acceptance Criteria
* New role permission added
** On by default for all Director and Central Office admin roles
* If user has the permission, they can adjust & reconcile a session, ignoring system settings MAXAMTRC and MAXATMAURC
** User must enter a comment when their role permission allows them to ignore the MAX amounts.

# NXT-43485 — MP- Menu Planner- Consider Sodium T1A for Sodium calculations daily and weekly analysis (UI ONLY)

## Description
Since the new Sodium Target 1A requirement date has now passed, we need to introduce considering it as part of the menu planner analysis for both the daily and weekly analysis

## Acceptance Criteria
* Consider Sodium Target 1 A for applicable meal type and Display “Sodium T1A” instead of “Sodium T1”
** K-5, K-8 6-8, 9-12 Lunch for 14+ Meal Pattern
* On Daily Analysis, mark Sodium as exceeded if it is passed the max sodium allowed, this applies to both Target 1 and Target 1A (Confirm for QA)
* On weekly analysis, mark sodium as exceeded if it is passed the max sodium allowed, this applies to both Target 1 and Target 1A  (Confirm for QA)
** Fail the analysis on the sodium exceeding weekly amount
*** Correct verbiage on the Fail pop up based on what type of Sodium we are checking against

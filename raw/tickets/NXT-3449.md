# NXT-3449 — Configuration - Edit/Delete Topic, Subtopics

## Description
With permission, users with permission have the ability to edit Topics. Subtopics cannot be edited 

* Edit pencil at topic level opens popup - user can edit status, subtopics display and create new subtopics for that topic 
* When making a topic (required field) inactive, they will remain in assigned events but not be available for future training events added/edited 
* when the topic status is changed to inactive, the subtopics under it become ‘frozen’/read only
* Once a topic/subtopic is made inactive, user has the option for soft delete (BE sends to a deletion table for support to retrieve if necessary)
* Topics/subtopic names cannot be edited 
* User cannot edit the topics and subtopics preloaded in the stystem provided by USDA

## Acceptance Criteria
Admin (with permission) can edit topic (status, subtopic display, create new subtopics), Subtopic status is all that can be edited
once topics/subtopics are made inactive, only then does the soft delete trashcan appear

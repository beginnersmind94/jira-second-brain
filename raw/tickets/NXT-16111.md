# NXT-16111 — INV - Order Template Details - Items tab - View Items to Add - Header

## Description
As an inventory Operations Manager I want filters on the Add Items to Template functionality so I can easily filter the list of eligible items.

This screen is accessed via the ‘Add Items’ button that is presented on the Order Template Details screen ([NXT-14810|https://primeroedgenext.atlassian.net/browse/NXT-14810])

*Requirements*

* on the Add Items, provide a header section containing filter dropdowns
** The filter selections will be:
*** Storage Category
*** Item Category
*** Data Source
** All of these are optional filters
** Default value for each dropdown is ‘All’
** These dropdowns can be independently selected
** Dropdowns should be linked together for the values
*** If user selects a Storage Category, the Item Category dropdown should only provide values that have items assigned to the selected Storage Category
** Provide action buttons in the header section
*** Reset button - to reset any dropdown selections
*** Apply button - to apply the selections and update the item list below

## Acceptance Criteria
* Add header to the Add Items to Template page
* Provide 3 dropdowns

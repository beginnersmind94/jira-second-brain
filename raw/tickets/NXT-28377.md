# NXT-28377 — MP- Menu Item Quick List Edit from Menu Page (UI ONLY)

## Description
As a user, When I select a menu item Quick list from the menu page and would like to make changes to it, I would like a way to edit it without having to navigate back through the navigation to get to the menu item quick list page.

## Acceptance Criteria
Allow user to edit the quick list from the menu page

If user removes menu items that have already been added to the menu, it does not effect anything. Menu items will continue to stay on the menu as normal.

Allow users to edit quicklist in overlay, they can make changes, add/remove items, and save and close

Once save and close, all changes should reflect on the quicklist slide out with new changes and information

Status for quicklists:

#items used = 0 means not used

#items used >1 means partially

#items used = #items on list means Fully

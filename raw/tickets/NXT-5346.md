# NXT-5346 — AcctMgt - Patron Search - Even More New Filters

## Description
*User Story*

As a district admin I want to refine my search to better create custom lists for exporting.

*Mock-up*

TBD

*Requirements*

* Add the following filters for Patron Search (Split View/Table View). These would be considered additional filters, only available via the Split / Table views.
** Homeroom
*** Default is blank, aka All.
*** All homerooms should be available for multi-select.
*** Available selections are fueled by selected site(s).
*** Hide Rosters and Homerooms if the patron type is set to Adult only.
** Roster
*** Default is blank, aka All.
*** All rosters should be available for multi-select.
*** Available selections are fueled by selected site(s).
*** Hide Rosters and Homerooms if the patron type is set to Adult only.
** Status (All/Active/Inactive)
*** This will apply a filter for searching by patron status.
*** Default is Active.
*** -Multi-select possible.-
** Eligibility
*** Default is blank.
*** Multi-select.
*** Options are dependent on whether we’re looking at students or an adult (the patron type filter):
**** Students: Free, Reduced, Paid
**** Adults: Staff, Program Adult, Visitor

## Acceptance Criteria
New filters available for fine-tuning search.
Filter selections do not need to have an impact on predictive results for now.
Hide Rosters and Homerooms if the patron type is set to Adult only.
Status is not multi-select, but contains an All option per discussion notes.

# NXT-9408 — INV - Transfers - Transfer Request - Confirmation

## Description
As a site user I want a Transfer Request Confirmation screen so I can do a final review of my request, make any changes or save the request.

Once user has created their Transfer Request ([NXT-9411|https://primeroedgenext.atlassian.net/browse/NXT-9411]) or Site Requests ([NXT-9509|https://primeroedgenext.atlassian.net/browse/NXT-9509]) they will click Next and land on the Transfer Request Confirmation screen.

This screen will provide a read-only summary of the transfer details and provide report options.

After the user has created a transfer and clicked Next, they will be brought to the Transfer Confirmation screen.

+Header section:+

* To Site
* Transaction Date
* Comments

+Transfer Item List:+

* Item #
* Item Description
* Whole Unit Request Qty
* WU Description
* Broken Unit Request Qty
* BU Description
* From Site
** this will only appear if the user had the Other Sites QOH permission when they created the transfer
** if the user didn’t have the Other Sites QOH detail, this column would not be applicable, and it would only show the total quantity requested for the item
* Comments

+Action buttons:+

* Edit Transfer Request
** this will take the user back to the Transfer Request screen to adjust the items & quantities
* Confirm Transfer Request
** this will confirm and save the transfer request
** once this is done, no more edits are allowed
*** remove the Edit Transfer Request and Confirm Transfer Request button
** after the Transfer is confirmed, the page will provide additional actions:
*** Transfer Ticket
*** Price Details
** This should create the Transfer Number(s)

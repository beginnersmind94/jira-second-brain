# NXT-56379 — IM - Items - Nutrient Details - Nutrition Label View

## Description
As a menu planner I want a view of the nutrient details that resembles a nutrition label for convenience of reading and for potential printing and labeling.



!Nutrient Details Label View-20241212-185813.png|width=91.66666666666667%,alt="Nutrient Details Label View-20241212-185813.png"!

## Acceptance Criteria
* Add a label print icon to the Nutrient Details card
** for all Items, Recipes and VMI
* Open the Print Nutrition Label slideout when the icon is clicked
* Provide a Label Options section
** Serving Size selection dropdown
*** default to the serving that is Nutrition Linked
** Include %DV (default YES)
** Round decimals to whole numbers (default YES)
** # of labels per page (default 6)
* Provide a label preview section
** display a version of the nutrients card which closely resembles the official FDA approved nutrient label format
** Use formatting guidelines from this [website resource|https://foodlabelmaker.com/regulatory-hub/fda/nutrition-facts-font/#:~:text=What%20font%20should%20you%20use,benchmark%20style%20for%20food%20labels.]
** Use the [FDA website|https://www.fda.gov/food/nutrition-education-resources-materials/nutrition-facts-label] for general information about labelling
* Do not include optional vitamins if there is no value
** Vitamin A
** Vitamin C
* Never include Moisture or Ash
* Provide a button to print the labels on a paper sheet
** Use the # of labels per page field to populate labels on printer sheet preview
* Provide the ability to download the label as a (high res) image (not in a PDF or report viewer)

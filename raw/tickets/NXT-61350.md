# NXT-61350 — IM - Item Onboarding- Pack Size Info- Inventory Publish

## Description
As a district admin using the onboarding tool, I would like a way to be able to indicate which items I want to publish/not publish to inventory once they are in the system.

!image-20250429-073253.png|width=1362,height=723,alt="image-20250429-073253.png"!

## Acceptance Criteria
* Provide an “Inventory Publish” option under Inventory options on the Pack Size Review screen
* Keep the user on the Review Pack Size Screen and enable the Inventory status column with toggles for the items that can be published/unpublish
** Default all items that are eligible to be publish
* Provide a toggle “Show Only Items Can Publish” that will only display the items that can even be inventory publishable
* Disable any of the update options from being selected when in this mode
* Do not show any items that have no inventory information for (Marked as None/Decide Later)

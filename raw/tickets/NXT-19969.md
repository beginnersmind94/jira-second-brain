# NXT-19969 — UI-Editing all import parameters, delimiter and qualifier for all Import types.

## Description
As an end user, giving the option to modify the existing saved configuration is recommended. This will save a lot of time and reduce the changes of mapping errors. Currently, once the configuration is saved other import parameters like delimiter, qualifier, file formats cannot be altered. The only way to change them is to recreate a new configuration and delete the existing one.

## Acceptance Criteria
# Verify and Validate that the import configurations can be edited without the need to delete and recreate.
# Verify and Validate that the originally saved file format can be modified and the newly selected format is saved.
# Verify and Validate that both delimiter and qualifier can be altered and saved without the need to recreate the complete configuration  and save time.
# Verify and validate that various types of file formats along with various types of delimiters and qualifiers (all permutations available) are tested and saved rather than creating a new configuration every time.

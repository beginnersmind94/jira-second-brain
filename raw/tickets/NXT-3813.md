# NXT-3813 — Account Management - Advanced Search - Site Section - Site type subsection

## Description
The site types defined in the district will be listed here. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Site Type values. Standard Site Types are Elementary, Middle, and High schools there may be others.

!Screen Shot 2020-04-27 at 3.33.08 PM.png|width=305,height=459!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Site Type section they are 'or'ed together.

When a filter value is selected the Site Type Subsection count is increased by 1 and the Site Section count is increased by 1.

# NXT-34144 — Team Hub - Perform Timed Push & Pull of Messages

## Description
Per a certain timed schedule, it will need to automatically perform an incremental push / pull and reconciliation of messages with the server so that the app stays in sync with the server.

 

*Flow:*

·         Upon timer trigger

·         Incremental Push / Pull is called, and app messages are reconciled to server’s messages

 

*Background:*

·         This code is related to the Incremental Push / Pull story

·         This code covers the timer and the other one what to do

 

*Requirements*:

·         Fields: None, backend code only

·          

·         Need a timer to run in the background

·         When time is up, call the Incremental Push / Pull

·         When that is complete, reset the timer

·         Timer value is in seconds and defaults to 120

·         Timer value should be easily configurable, not hard coded so can change without a code push

·          

Security – already covered in previous stories

## Acceptance Criteria
·         There is a timer to run in the background

·         Messages are updated according to the timer

·         When timer completes, resets the timer and starts again

# NXT-29874 — PR- Production Record- Display Quantity Prepared

## Description
As a Production Manager, I would like to be able to see all the quantity it took to produce each menu item on my production record based on my produced counts.

## Acceptance Criteria
Display Quantity Prepared on the Production Record based on the items that are used within a menu item (Same Logic as Food Required on production Plan) Display the total units, and weight that are needed per each item as well as display the recipe/ item Name and Code for each portion of the menu item. IE: 25 Each OR 1 pound 3.25 ounces (Peeled Cucumber- LI123456) Calculate the total amount needed based on produced counts If there are multiple serving sizes, take into account each serving size with that count and treat them as separate items when calculating total quantity prepared When user clicks save with a different produced count, recalculate the quantity prepared and update the new amount needed.

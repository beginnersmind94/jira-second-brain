# NXT-67633 — IM - Bulk Item - Update Contributions (Individual Item Slideout)

## Description
As a district manager, going through my items in the Bulk Update tool can be harder to look at with multiple items. I would for a way to still go through the list and just update an item at a time with the contribution slide out that I am used to on the Item details page.

* Provide icon to open slide out for individual update of contributions per item

** Add a “No Contributions Indicated” flag that will allow users to mark the item for no contributions
** This can be done with a serving size selected or not but will apply for the whole item
** Need Separate story for item details?
* Update the contributions review and all sections with the selections made on the individual edit

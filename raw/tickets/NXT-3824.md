# NXT-3824 — Account Management - Advanced Search - Balance Section

## Description
The Balance section allows the user to enter value(s) to allow the user to Filter on balances above a value, below a value, or between two values. It will also allow the user to select an ‘as of’ date for when a particular balance(s) occurred in the past.

!Screen Shot 2020-04-29 at 1.29.16 PM.png|width=277,height=116!

## Acceptance Criteria
A upper and lower range can be set.

When only a lower range is set any value greater than or equal to the lower range is a match.

When only an upper range is set any value less than or equal to the upper range is a match.

When both a lower and upper range is set only values between or equal to the range values is a match.

A range value is a dollar figure

The list shown is for the current date unless the 'As of' filter is set.

If a range is given in the incorrect order an error is displayed saying 'Lower range value is greater than higher range value, please correct'

When the As of date is selected, the accounts listed are for balances that are in the range for that date.

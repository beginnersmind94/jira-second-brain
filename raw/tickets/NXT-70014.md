## Acceptance Criteria
If the district is configured for Square, and:

When Payment Type Card is selected from Filters:

Display new Check Radio button section with options: “Card Transaction Type: All, Certified Only, Manual Only”
When All show all Credit Card Transactions (Including Normal)
Updated/edit transactions are still considered “Certified”
When Certified Only show only Square Transactions that have been completed through a Square Terminal.
When Manual Only show only Square Transactions that have been added or appended to an edited transactions In reconciliation.

Tool tips
Add tool tip for Certified Only “Transaction that have been completed through 3rd party processing”
Add tool tip for Manual Only “Transactions that have been added through reconcilliation”
Add tool tip for All “Includes standard Credit Card transactions”

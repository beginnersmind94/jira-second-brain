# NXT-26599 — Platform - Security - Implement Page CRUD operations for Backoffice team

## Description
As a primero edge admin, I should be able to access resource catalog to add a new resources available for any district with page resource type along with available CRUD operations for all front office pages.

+*Requirements:*+

* Resource Permissions CRUD operations with CRUD Permissions to be enabled and making sure FO pages can utilize these resource permissions via roles & permissions page
* Permissions to be implemented for below pages
** Item Management: Items page [https://qa.perseusedge.com/item-management/items|https://qa.perseusedge.com/item-management/items]
** Inventory module: Manage Orders page  [https://qa.perseusedge.com/inventory/Manage-Orders|https://qa.perseusedge.com/inventory/Manage-Orders|smart-link] 
** Menu Planning: Manage Menu page [https://qa.perseusedge.com/menu-planning/menu|https://qa.perseusedge.com/menu-planning/menu|smart-link] 
** Production: Grocery List page [https://qa.perseusedge.com/production/groceryList|https://qa.perseusedge.com/production/groceryList|smart-link]

## Acceptance Criteria
* Any page type resource with CRUD operations enabled shall be able to CRUD permissions for page resource
* User shall be able to view the CRUD operations to any page that has updated permissions in roles & permissions page

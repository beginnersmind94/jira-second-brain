# NXT-3565 — Edit Inspections - Checkbox Question Type

## Description
*User Story*
As an Inspections user, I can see Checkbox Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Checkbox Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Selections
** Whether question is required
** Notes Section
** Upload Media
** Note: Point values are *not* displayed here
* User can select as many choices as they want
* Choices are properly saved
** If question is required, Inspection cannot be submitted if no selections are made
* Choices can be changed if Inspection has not been submitted
* If question has a score associated with it, the score is properly updated according to the selection being made.
** Score is added based on all selections made

## Acceptance Criteria
* Checkbox question appears in Inspections
* Checkboxes and notes can be selected, saved, and edited
* Score updates properly if present

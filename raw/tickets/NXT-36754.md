# NXT-36754 — MP/PR- Update Link for Menu Item Help button and Recipe Link on Add menu Item (Single Item Recipe) (UI ONLY)

## Description
When we have created Single Item Recipe in IM, we never took that into consideration in the Menu Planning and Production side. When looking at the menu item help link to the menu item, we are taking the user to the recipe page. We should not lead them to the recipe as there should not be a recipe displayed on the UI. We need to lead them back to the Item Page and land on the Recipe/ Menu item Tab.

## Acceptance Criteria
* Lead users to the Item Page and land on the Recipe/Menu Item Info tab when it is a Single Item Recipe
* Apply changes to the following sections
** Menu Item Help Button (MP)
** Add Menu Item Recipe Link (MP)
** Menu Item Help Button (PR- Plan)
** Menu Item Help Button (PR- Consolidated Plan)
** Menu Item Help Button (PR- Prep Site Dist)
** Menu Item Help Button (PR- Record)

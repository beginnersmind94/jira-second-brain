# NXT-39508 — SIS Auto Sync – Add Grades Import to Import Type

## Description
As a SchoolCafé admin, I would like to be able to schedule Grades Imports via SIS Auto Sync so that I can automatically import my data.

*Notes:*

* This request comes from Dallas and is needed to support the SIS Auto Sync for new district onboarding

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports
* Click SIS Auto Syc tab
* Click + New button
* In Import Type field, select Grades Import
* Finish the rest of the process
* Once a configuration is completed, the system will execute the imports per the configured schedule

*Requirements*:

*Add New SIS Auto-Sync screen*

* All these requirements are for the Add New SIS Auto-Sync screen:
* In the Import Type field:
** Add Grades Import to the list of values
** All the business logic of Student Import apply to this screen to such as saving values, setting Status

*Job Scheduler*

* All these requirements are for the Job Scheduler:
* Ensure that the SIS Auto-Sync process runs per the schedule set

*SIS Dashboard tab*

* All these requirements are for the SIS Dashboard tab:
* Rows are inserted in the grid for this import configurations after they are created
* Import Type column values are Grades Import
* Status can be enabled/disabled like with Student Import
* IsScheduled can be Active or Inactive like Student Import
* Actions:
* Delete
** Can delete the configuration
** Handle the same way as Student Import
* Edit
** Can edit the configuration the same way as Student Import

## Acceptance Criteria
* Add Grades Import to the list of values to the Import Type field
* All the business logic of Student Import apply to this screen to such as saving values, setting Status
* Ensure that the SIS Auto-Sync process runs per the schedule set
* Rows are inserted in the grid for this import configurations after they are created
* Import Type column values are Grades Import
* Status can be enabled/disabled like with Student Import
* IsScheduled can be Active or Inactive like Student Import
* Can delete the configuration the same way as Student Import
* Can edit the configuration the same way as Student Import

# NXT-11281 — IM - Menu Items - Add New Serving Size



## Description
As a user i would like to have an ability to add/edit new serving sizes so that they can manage the food contribution

The user must be able to add and edit a default serving size as well as the corresponding food contribution.

Requirement: 

* As a user, I need to be able to add a user serving size based off the serving size of the recipe
* I need to be able to set the serving size as a default
* I need a free form test field for ‘Portioning Tool’
* When editing, allow the user to ‘clear' or ‘edit’
* Cannot clear a serving size if it’s being used in a menu



* Once a menu item has been added to a recipe, you cannot change the recipe serving size unit of measure. If this is needed, they need to copy the recipe without the menu details
*

## Acceptance Criteria
User must be able to add and edit a serving size for a menu item

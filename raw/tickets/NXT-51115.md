# NXT-51115 — Acc - Dynamic Entrees - Addition of Dynamic Entrees buttons and category to Menu Grids

## Description
As Mary the Menu Grid Maker I want to place the dynamic entrees buttons and category onto a grid so it shows up at the POS.

*mockup*

Not needed - existing screens work fine for this.

!image-20240528-034855.png|width=1498,height=685,alt="image-20240528-034855.png"!

*requirements*

If the DYNENTREE setting is turned on:

* User can add a category called ‘Dynamic Entrees’ to the menu grids.
** User can style the category as per usual: size on the grid, colors, fonts.
** User can rename the category.
* User can drag a menu item button called ‘Dynamic Entree’ onto the grid.
** User can perform this action repeatedly, and each time it should add the button to the grid.
** The button, when dropped, should have an increment added to it. Ex: Dynamic Entree 1, Dynamic Entree 2.
*** First time the user drops it it’s Dynamic Entree 1, second time it’s Dynamic Entree 2, etc.
** User can colorize the individual buttons but cannot rename them.
* New flag to track which menu grids have DE configured.

*Discussion during mini-demo on 7/12/24*

* Additional requirement: User can see that the menu grid they’re editing has a dynamic entree on it.
** If the setting is on, show a text label next to the Template toggle.
** Text: Dynamic Entrees: {Status}
*** {Status} = Yes if the menu grid has either the category or a DE item on it, No if neither condition is true

## Acceptance Criteria
If the setting DYNENTREE is on:

* ‘Dynamic Entrees’ category added as a system default category.
** Category can be placed on menu grids and edited for style - fonts, size, colors.
* ‘Dynamic Entree’ button added and available for use.
** User can drop the button onto the grid multiple times.
** A counter is added to the button text: Dynamic Entree 1, Dynamic Entree 2, etc.
** User can edit button for style - fonts, size, colors.
* New flag for Menu Grids to track DEs
* Flag is visible as a text label to the user while editing the Menu Grid layout

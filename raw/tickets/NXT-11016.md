# NXT-11016 — Dashboard (Insights) - Accountability - Menu Item Served Widget

## Description
As a central office user or cafeteria manager I want the ability to see a list of the top 5 menu items sold, and the count for a Site or District

See the Menu Item sales report, Item Summary for data.

[^MenuItemSalesByItemReport.pdf]

## Acceptance Criteria
A graph or pie chart is produced for the top 5 menu items sold.
Users can select the date, date range to be displayed.
Users can select the meal type to be selected or all.

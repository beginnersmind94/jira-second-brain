# NXT-15074 — POS - Default Item - Disable, re-enable, or pause

## Description
As a cashier I want the ability to disable, pause, or re-enable default item selection during meal service so that I have better flexibility during meal service and do not have to re-configure my menu grid to do so.

* Disable/enable toggle needed to turn on/off the default item from displaying
* Pause button to temporarily stop the default item during the service session. Will be enabled again when a new session is started.
* Need a permission that allows a user to perform this action

## Acceptance Criteria
* When enabled, default items should be loaded in the sale
* When disabled, default items should not be loaded in the sale
* When disabled, it should remain set until the user re-enables
* When paused, default items should not be loaded in the sale
* When un-paused, default items should be loaded in the sale
* When paused, it is to remain set only for the current session and should be un-paused (reset) when a new session is opened

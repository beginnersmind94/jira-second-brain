# NXT-10517 — Special Assistance Programs - P2 Claiming Percentages - Need to select sites

## Description
As a user I need to be able to select the sites that I choose to participate in the P2 program.

*Requirements:* 

* The user can select which sites are participating.
* If the site is not chosen then the percentages are zeroed out and nothing is entered in the table for the site.
* There is an option to select ALL sites.

## Acceptance Criteria
There is an Included Check like in CEP.
All sites can be selected with one check
If the site is not selected the values are zeroed
If the site is reselected the values are restored.
When the site percentage for Free or Reduced is changed the Paid percent is recalculated.

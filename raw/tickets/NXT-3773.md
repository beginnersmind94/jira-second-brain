# NXT-3773 — AcctMgt - Add Patrons - Add New Student - Student Information

## Description
*User Story*

As a district admin I want to add patrons via the UI for scenarios where I am entirely managing a patron via this product.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=14|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5&g=14]

*Requirements*

* Add a ‘+ Add’ button to the Patrons page.
* Clicking this brings up a little dropdown with options:
** Student
** Adult
** Import
* Selecting Student will bring in a drawer called ‘New Student’
** Page / pop-up has a Save and Cancel button.
* User is returned to the Patrons page if they click Cancel / x out.
* Save button should save to appropriate tables for recorded fields.
** User should receive a toast message about the student being added.
** ‘Patron FirstName LastName (ID # ) has been successfully created.’
* Add the Student Information section per the wireframe, minus the ‘Add Photo’ section.

* Fields to capture from the Student Information section:
** Patron ID Number
** First Name
** Last Name
** Middle Name
** Preferred Name (optional)
** Suffix (optional - dropdown)
** Date of Birth (date selector)
*** No future dates.
** Ethnicity (optional - dropdown)
** Racial Identity (optional - dropdown - multi-select)
** State ID Number (optional)
** PIN Number (optional)
** Primary Language (dropdown)
** Gender (optional - dropdown)
** Eligibility (dropdown)
** Reason (dropdown)
*** Each reason is tied to an Eligibility.
*** Eligibility should be chosen first.
** Enrollment Site
** Grade
*** Each grade is tied to a particular site.
*** Ideally, site should be chosen first, but not always.
*** Available list should dynamically change when a site is selected.
** Homeroom
*** Dropdown list, tied to the selected site.

Note: Default value for dropdowns is the label itself like the wireframes suggest. Example below. If selected, we’d not actually be recording anything for this value as no true value has been selected.

# NXT-54040 — Acc - Sessions - Restore Session feature

## Description
As Dana the Director I deleted a session by accident and I need a way to restore it.

*requirements*

* User has a way to restore deleted sessions.
** All transactions within the session should be returned.
*** Sales, payments, etc. Summary or Detailed session, doesn’t matter.
*** Opening/Closing with Denominations/Checks/Cards/etc also need to come back.
** Opening/Closing details should be returned.
** Confirmation prompt for the user:
*** User should enter a comment for this.
*** Text: This will restore the session to its previous status. You may need to review the following in case they need changes. Report Rollups, Deposit Slips, Reimbursement Claims. 
** Tracked as a Session Adjustment.
** Prior Transaction Adjustments as part of the Deletion should be removed.
** Session History note added for Restoring the session.
*** Note: Session Restored: {UserComment}

*Discussion Notes - Refinement (9/16/24)*

* Use the last Session Status before the session was deleted
* Role Permission → Delete Session
* Time Restrictions:
** POS Period must be open
*** Button should be disabled, tooltip → POS Period is closed. It must be opened before the session can be restored.
** Same academic year

## Acceptance Criteria
* User can restore a deleted session.
* Action is tracked as a Session Adjustment. (Transactions > Adjustments)
* Remove the Transaction Adjustments that were added when the session was deleted
* Session History updated with the user’s comment.

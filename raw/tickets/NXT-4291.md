# NXT-4291 — IM - Item Code Generation

## Description
As a user, I want the item code to be generated automatically.



1.	Have the NextItemCodeNumber per datasource stored in the database. @Nandish Shivaprasad, please review the option of a separate table to store these as this new table will have equal number of reads and writes.
2.	Have MinimumCodeLength column per DataSourceType to enable shorter codes for local items compared to global/shared.
3.	At the time of creating the item, start a transaction, get the NextItemCodeNumber by DataSourceId, adjust the length based on MinimumCodeLength and finally add the prefix.
4.	Check for a possible duplicate code before inserting the Item or catch any unique index errors and retry the process. I would suggest the second option as inserting item is not as frequent as say, placing orders or making payments. Also, two persons managing the same datasource attempting to add an item at exactly the same time is a remote possibility.
5.	Before ending the transaction, upon successful item creation, update the NextItemCodeNumber to NextItemCodeNumber + 1.

## Acceptance Criteria
Item Code is generated automatically per datasource.

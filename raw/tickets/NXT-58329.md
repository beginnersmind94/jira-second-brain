# NXT-58329 — PR- Plan/Record- Serving Size Capture for IM

## Description
As a nutritionist that is adding information into Item Management, the system should not allow me to remove serving sizes that I currently using in Menu Planning as to not create bad data

## Acceptance Criteria
* Capture any serving size that is being used in Production
** Plan Add Menu Item
** Record Add Menu Item
* This information should be used for Item Management to not allow users to remove serving sizes that are currently in use
* This will need to be further expanded when versioning comes where we must store all serving sizes active/inactive

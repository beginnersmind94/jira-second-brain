# NXT-42809 — SIS Auto Sync – Guardians – Front End

## Description
As a SchoolCafé admin, I would like to import multiple guardians so that I can have that information on the student record when I need it

*Flow:*

* Browse to SIS Auto Sync
* Click on Add New
* Overlay opens
* User selects Guardians as Import Type
* User configures field mapping and distribution screens accordingly
* User completes the stepper and saves
* System runs the import per the schedule
* User can view the history of the imports on the Import monitor
* Optionally, the user can edit or delete the import
* Optionally, the user can view the source data
* Optionally, the user can view the exceptions and their count
* Optionally, the user can preview the data

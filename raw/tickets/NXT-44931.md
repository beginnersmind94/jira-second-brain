# NXT-44931 — Custom Reports – ReportDistribution - NotificationSync For Custom Reports - Modifications

## Description
This ticket is to extend the report distribution package to respect the BlobMetadata as param (or FileStream) based on the option it needs to upload/skip the file stream.

Below are the actions that needs to be completed as part of this ticket



* Modify DataQueryService to upload the file directly to blob and return back the BlobMetaData
* Modify DistributionPackage to take BlobMetaData and skip the upload file stream if blobmetadata already exsits
* Modify NotificationSyncAgent to remove attachments (for Inbox & Email) and send as a link to report.

## Acceptance Criteria
* Emails should be received with the link to blob file and able to download the file once the user clicks on the link
* Inbox message should contain the link to file but not attachment. Clicking on the link should download the file without any errors

# NXT-3816 — Account Management - Advanced Search - Patron Section

## Description
The patron section will allow the user to select a specific or a list of matching patrons. There are two sections to the Patron section one is checkbox section of person types. This could be done as a subsection. The second section is a list of text boxes that allow the user to input Specific filter information, IDs, Name, or Household ID. When a patron type is selected the typed information will search in that patron type only.

!Screen Shot 2020-04-28 at 11.33.20 AM.png|width=287,height=380!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Site Type section they are 'or'ed together.

When a filter value is selected the Site Type Subsection count is increased by 1 and the Site Section count is increased by 1.

This should check all possible IDs for matching - Patron ID #, State ID #, PIN #, First Name, Last Name, Middle Name

# NXT-63851 — INV - Orders - Order Details & History improvements

## Description
As a Site Manager I want the Order History to have a compliant UI so I can clearly visualize historical changes.

h3. Current Order History Details UI

* red font is enlarged for some reason
* green font is using incorrect green, insufficient contrast

!image-20250708-172713.png|width=83.33333333333334%,alt="image-20250708-172713.png"!

## Acceptance Criteria
UI improvements within the Order History Details 

* Banner should be moved to sticky footer
* Make it clear the user is looking at the historical transaction
** keep the History tab selected; not the Details one
** append ‘Historical Transaction’ after Order Details heading
* No action buttons should be available on history (Cancel, Add Item, etc.)
* Green text should not show on items for the initial transaction
** it should show only if an item was added to the already existing order
* Red text should not show for things which did not change
** why are Delivery Date and Status showing as red changes?
* Red text should not be enlarged
* Green text should be darker, compliant green

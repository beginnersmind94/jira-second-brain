# NXT-9562 — INV - Transfers (Site) - Create Transfer (Site-to-Site) - Items

## Description
As a site level Inventory user I want a Create Transfer functionality so I can transfer product from my site to another site when I know what the other site needs.

This is accessed by the user clicking the ‘Create’ button on the Transfer List page.

Based on selections from [https://primeroedgenext.atlassian.net/browse/NXT-7267|https://primeroedgenext.atlassian.net/browse/NXT-7267|smart-link]

Once the Apply button is clicked after Site selections are made, load a list of items which have some QOH at the From Site

+Item List:+

* Organize by Storage Category & Item Category
* Do not use pagination for now
** user inputs can be lost
* Only load items with QOH > 0
* +Item List+
* Item Number
** searchable
* Item Description
** searchable
** Include pack size information
*** Ex: “1234 Milk \[1 carton = 12 bottles]”
* QOH - label
** ‘From Site’ quantity on hand
** Ex: “100 cases, 12.00 packs”
* Whole Unit Transfer Qty
** numeric *input* field
** include Whole Unit description (either in the cell or a dedicated column)
** cannot exceed QOH
* Broken Unit Transfer Qty
** numeric *input* field
** include Broken Unit description (either in the cell or a dedicated column)
** cannot exceed QOH
* Comments
** user can enter comments for the specific item
* Delete icon
** only enabled if user has entered some Transfer Quantity for the item
** this will allow the user to clear the Transfer Quantities which have been entered
* **Highlight the items which have some Transfer Quantity entered

*Next* button

* Next button will only be enabled if user has entered some transfer quantity for at least one item
* On click Next, calculation will convert extra broken units into whole units
* User has to enter some Transfer Qty (WU or BU) for at least one item
* Validate any items exceeding QOH
** Message: “Transfer quantity exceeds available quantity on hand for item #12345 - Item Name.”
*** If there are multiple items exceeding QOH list them after the message, comma separated: “Transfer quantity exceeds available quantity on hand for items #12345 - Mozzarella Cheese, #98765 - Baked Bread, #19283 - Cheese Pizza”

Technical Details:

* For now, only allow whole numbers for WU and BU input
** future setting will be added to allow for partial broken unit entries
* For now, allow sites to transfer products to sites that may not be configured to receive the item
** future setting will determine whether items are restricted by receiving site configuration

Mockup differences:

* no item checkboxes needed
** highlight the row when any Transfer Qty is entered
* no From Site in the grid
* WU *Transfer* Qty instead of ‘Request’ Qty
* BU *Transfer* Qty instead of ‘Request’ Qty
* Delete icon will be added to far right side
** delete icon will clear the entered transfer quantities

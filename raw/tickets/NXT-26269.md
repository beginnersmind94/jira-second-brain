# NXT-26269 — INV - Management - Period Valuation

## Description
As an Operations Manager I want a Period Valuation page so I can view the periods for my school and generate the valuation.

*Requirements*

* Create a new Period Valuation page
** the data loaded on this page should be connected to the rollup required for Valuation Report [https://primeroedgenext.atlassian.net/browse/NXT-26251|https://primeroedgenext.atlassian.net/browse/NXT-26251|smart-link] 
* Provide a Site selection dropdown (use common control)
** single site must be selected
*** this will usually be a site-level page, so users will not typically select
* Load the periods for the current school year, as created according to System Generate Periods page
** provide ability to switch through years (UI component needed)
* List each period along with Start and End Date columns
* Show the Inventory Value if applicable
* Show the period’s valuation status
** Generated
** Not Generated
** Needs Regeneration
*** highlight in red
* Provide a Generate/Regenerate button ([NXT-26251|https://primeroedgenext.atlassian.net/browse/NXT-26251])
** this will apply to all periods requiring generation or regeneration

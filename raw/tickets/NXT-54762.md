# NXT-54762 — SC - Pending Updates - Single Image Updates

## Description
_(none)_

## Acceptance Criteria
Navigation: 

Item Management - Items - Item Upload Button - Pending Updates Tab

Changes Required:

*Single Item Updates:*

By default all the images sourced will be selected (tile image view with check box enabled)

The user should be able to deselect or select the desired image they can take

PDF Files - can show the PDF Icon, no preview is needed, clicking on PDF link will download the file to user’s download folder. Accepted PDF files shall be visible in item’s local images & documents section

The check boxes should be disabled for images that are already updated

The update now button should be enabled if there are any new images updated

The grid should not list the entry once all the images are accepted, until the new images are uploaded



*Regression:*

GDSN Export - to be available in CSV

Ability to review images at item level

Audit History update when the images are getting updated

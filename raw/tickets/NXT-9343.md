# NXT-9343 — INV - Orders - Order Detail - Edit Order Items

## Description
As an inventory user I want a Edit/Modify view on the Order Detail page so I can make changes to existing orders.

Some related back end changes are handled in [https://primeroedgenext.atlassian.net/browse/NXT-16350|https://primeroedgenext.atlassian.net/browse/NXT-16350|smart-link]

*Requirements*

* In the lower grid containing the list of items, provide an Edit icon to each item row
** Orders can only be edited on this screen if the status is Open or Pending Review
** Edit icon only available if user has Edit Order permission
* Edit icon
** will enable the quantity fields of that item for editing
** will provide a delete icon, to remove the item from the order
*** Provide a prompt when user attempts to delete an item: “Are you sure you want to remove this item from the order?”
** will be replaced with the green checkmark (save) and red X (cancel) icons
** will disable the Edit icon for all other items
* Validate Order Quantities 
** check any quantity edits against the contract limits
** provide the alert (!) icon will be displayed if the item is violating any contract order quantity limits
*** the icon will be displayed next to the quantity field
*** Tooltip messages:
**** if quantity does not meet minimum: “Increase order quantity by X to meet minimum order quantity.”
**** if quantity exceeds maximum: “Decrease order quantity by X to meet maximum order quantity.”
* Add ‘Total Order Amount’ to the header section
** this will be the total dollar amount of all items on the order
** display the alert (!) icon if the order is violating any contract purchase limits after editing
*** the icon will be displayed next to the value for Total Order Amount
**** ex: $565.00 (!)
*** Tooltip messages:
**** if Order Total does not meet contract minimum purchase amount: “Increase Order Total by $XXX.XX to meet minimum purchase amount.”
**** if Order Total exceeds contract maximum purchase amount: “Decrease Order Total by $XXX.XX to meet maximum purchase amount.”
* If the order is in Open status, do not allow any invalid quantities to be saved
** Disable the Save button if any order or item validation issues are present
* Display all items with a validation issue at the top of the item list

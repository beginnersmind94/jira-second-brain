# NXT-35057 — Acct Mgmt - Account Info - Transactions - Add the Custom Views framework

## Description
As Vera the Director, I would like to save the layout of columns for an account’s Transaction history section, so that I can save time by not having to constantly scroll over to view the very-important balance column for each account.

*Mockup*
N/A

*Requirements*

* In the Accounts → Account Info → Transactions section, add the custom views framework so that the user can save custom views for this individual section.
* This is especially needed now that we’ve made all columns display in this section by default.



Reference email from Matt Payne:

{quote}_Andrea at the Central Office asked me to pass along an enhancement request that she has.  They would like to be able to save a default view for their user session in the Transactions portion of the Account Management page.  Currently she must reset her view to be able to see the account balance without scrolling each time she opens the page._{quote}

## Acceptance Criteria
* Custom View framework added to the Account Info Transactions section.
* Views can be saved, including filters, columns and column ordering.
* Views can be loaded.

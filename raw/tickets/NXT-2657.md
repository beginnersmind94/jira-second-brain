# NXT-2657 — System - Configuration - Routines

## Description
Routines is a coordination of recurring activities for an individual, persona, group, or organization. Improves efficiency, consistency, and coordination. Streamlines those activities that occur on a daily, weekly, monthly, quarterly, or annual basis. Allow the system to work for you.

* Assign to user, persona, or organization
* Associate page and/or task
* Ability to trigger or initiate certain automated tasks and/or processes (ex - reminders)
* Reminders

# NXT-6384 — Application Processing - Documents Drawer - [RTG]

## Description
As an admin, I am able to see the original application copy, along with any supporting documentation.  I can also upload more documents if needed.

*Mock Ups* 

*Requirements* 

Documents 

* Documents section is on the right action panel of the Processing window.
** Show a count of how many documents are attached to this application.
* Clicking the link/icon/section opens a drawer from the right showing the user existing documents.
** This includes the original copy of the submitted application. No delete option on this item.
** User has options to: view, upload (add), print, delete docs
** User can see any comments that were entered to go with any documents.

## Acceptance Criteria
Documents area added to the action section of application processing.
Number of documents is shown to the user.

User can click into this information and be shown all attached documents for this particular application.

User can upload multiple documents.

Files are stored appropriately so they can be retrieved later.

Files are marked as containing personal information.

User can remove files they've uploaded.

Files adhere to standard for accepted formats and sizes.
User is informed appropriately if their file is wrong size or format.

User can rename the document after uploading it.

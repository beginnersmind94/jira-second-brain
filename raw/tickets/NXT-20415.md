# NXT-20415 — INV - Reports - Batch Details Report

## Description
*Batch details Report* : Report to show the the batch level details of existing items on-hand along with the date on product information available.

Report criteria: selection of one or all sites, selection of on or  all item categories and one or all items

!image-20211006-175423.png|width=840,height=318!

!NXT-20415 1.png|width=100%!



*  Group the information by item # and Item Description and show all the batches received for that item at the site and it’s transaction details.
*  Report header section displays Site name (page break for each site information)
*  Report to show the following columns: Batch number, Transaction Number (could be order# (Receipt number), Transfer number, add to inventory number) WU Qty,  Received date, Lot number, Date on Product, Date type, Wu Price.

Attached the sample report from classic.

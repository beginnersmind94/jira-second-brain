# NXT-46742 — Elig - SIGs - Carryover

## Description
As Dana the Director I want to comply with state direction for carryover on state eligibility.

*requirements*

* Add a System Setting for the control of Carryover for State Eligibility Students
** Setting Name: State Eligibility - Provide Carryover for Students
** Description: This setting controls whether students with state eligibility will receive a carryover eligibility. If set to yes, it will add the Carryover eligibility for these students per the dates on the Carryover page during the 7/1 end of year updates. If set to no, students will move to Paid (Default) instead.
** Options: Toggle - Yes (default), No
** Setting should only be available if the overall State Income Guidelines setting is turned on.
* Update the “Year Begin” scripts to check the settings
** If the setting is on, give State Eligibility students a carryover status with their respective site dates.
*** State Free → Paid (Default) / State Free
*** State Reduced → Paid (Default) / State Reduced
*** Comment → Year Begin (State Eligibility)
** This check needs to happen prior to any CEP Carryover updates in Year Begin.
*** If the student is at a CEP site, they should receive a new eligibility line with the comment above instead of a CEP Carryover comment.

*** Example: Script executes for the 24-25 school year on 6/30/24.
**** It should also insert the 25-26 school year.

*** Keep students as USDA paid.

*** Update nightly jobs to check for expiring State eligibility and change the student to State N/A if required.

# NXT-67247 — GC>Item Catalog in PE> Viewing/exporting/importing matched items after searching for items by uploading the file with preferred name

## Description
Viewing/exporting/importing  matched items after searching by uploading the file: display the ingredient name same as the preferred ingredient name. 

PE - 3

## Acceptance Criteria
Ingredient name field should display the preferred ingredient name from the uploaded file in the results grid and in the popup in PE 

Review items popup will display the preferred name from uploaded file

downloaded excel files  will show the given preferred name from file

Importing items will import the items with the given preferred ingredient name.

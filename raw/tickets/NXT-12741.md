# NXT-12741 — Platform - AI/ML - Remove a Recognition Device

## Description
*Persona*: Student/Parent

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to remove any EdgeBox with UUID identifier or all devices under a SITE. 

*Mockup:*

[https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|smart-link] 



*Requirements*:

* Admin should be able to search for any EdgeBox assigned to a specific school district to remove the device from the ISD
* Edge Device must be in *OFFLINE | SUSPEND | DE-ACTIVATED* status to enable *Remove* action on that device.
* Authentication Token must be revoked from the device

## Acceptance Criteria
* User shall be able to search and remove the Edge device from any ISD from Device Catalog page
* Authentication token shall be revoked from IVL APIs

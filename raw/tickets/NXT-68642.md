# NXT-68642 — SC - MT - Change District Update

## Description
As a Customer Support role user, I want to have my own central realm roles so that my users can manage children districts.

*Requirements*

* On the change district screen, update it to account for Central Realm:
* If the user is in a Central Realm, filter the list to it’s children and itself
** Order by realm type (Central first, followed by Districts alphabetically)
* If the user selects another district, automatically log them into that district without forcing them to relogin

## Acceptance Criteria
* Central realm users can log into children districts and itself only
* Users are not forced to relogin

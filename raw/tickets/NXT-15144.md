# NXT-15144 — [AcctMgt] - ID Cards - Successfully view and print one or more ID cards

## Description
As a district admin, I would like to be able to view and print one or more ID cards for any student or adult in my district.

*(No UX Required)*

+*Summary*+:

* Allow a user to go end-to-end within ID cards and successfully:
** Search for an individual Account ID, OR
** Search from a list of filters
** Select a card orientation
** Select custom page margins
** Select a custom template, depending on orientation
** Generate a list of results
** (*NEW*) Open the ID Card viewer
*** Allow user to view and print a single ID card
*** Allow user to print multiple ID cards at once

## Acceptance Criteria
Verify that a user can go end-to-end within ID cards and successfully:

* Search for an individual Account ID, OR
* Search from a list of filters
* Select a card orientation
* Select custom page margins
* Select a custom template, depending on orientation
* Generate a list of results
* (NEW) Open the ID Card viewer
** Allow user to view and print a single ID card
** Allow user to print multiple ID cards at once

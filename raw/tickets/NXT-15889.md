# NXT-15889 — IM - Edit/Delete Inline for Serving Measures 

## Description
As a User I would like to have an Inline Edit/Delete Capability So that I can make changes to the Serving measures as needed

## Acceptance Criteria
Given the User is on the Nutrient and Contribution section 

And Edit is applied for the Nutrient and Contribution section

When Inline Edit option is selected 

Then Provide the User to Edit the below 

* Serving Measures
** Serving size
*** A check box with strike 
*** Ability to Enter the Serving Size in number digits 
*** Ability to Select Serving Measure 
** Weight in Grams
*** A text filed to enter the number value 
** Preferred Measurement
*** As Yes or No radio options 
** Possible Measurement
* And Upon hitting the check mark should save the Inline changes 
* And “Delete” should delete the Added Inline

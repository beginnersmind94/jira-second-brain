# NXT-49641 — 2FA - District - Dashboard

## Description
As a Customer Support user, I want to be able to view a district’s 2-Factor authentication so that they can use a code sent to them to complete login

*Notes*:

* This story is for adding the 2FA concept to the authentication grid
* Each authentication type will have a separate story to add 2FA to the add and edit screens

*Flow:*

* User browses to System > Security > Authentication tab
* User views if an authentication type has 2FA or not

*Requirements*:

* These requirements apply to the System > Security page > Authentication tab:
* 
* On the grid, 
** Add 2FA as a column
** If the auth type does not have 2FA, use the text “No” as a value
** If the auth type does have 2FA, use the list of selected 2FAs
*** Ex: if 2nd factor is SMS and Email, show “SMS, Email” as the value
** Default value is No
** Get value from Add and Edit screens of each auth type

## Acceptance Criteria
* User can see if an authentication type has 2FA enabled
* User can see which authentication type is enabled

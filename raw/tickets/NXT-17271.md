# NXT-17271 — Notify Central Office for License Updates via Workspace Task

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Central office user, I should be able to see and review the requests submitted by site/district admins for their license updates.

*Mockup:*

TBD

*Requirements*:

* Each License request update should create a new task under “My Tasks” section of Workspace for the central office team for that particular district. 
* Notification is received by the user when a new task is added into their “My Tasks” section
* Each Task should have due date of 5 business days with status as “NEW” and subject in below format
** “<ISD/Site Name> - License Update Request for <ADD/REDUCE> by <# of licenses>”
* User shall be to process the license and set the task as “Done”

## Acceptance Criteria
* New Task items shall be added for each request sent by Site/ISD.
* Each request will create a notification for the user when a new task is added into their “My Tasks” section
* User shall be to process the license and set the task as “Done”

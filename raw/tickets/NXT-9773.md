# NXT-9773 — Data Exchange - File Import Setup: Pre-Upload Step

## Description
*User Story*

As a PrimeroEdge user, I would like to have a screen where for a selected district (ISD) I can import file types and have the ability to download the appropriate template when needed, so that I can easily begin the process of uploading as much of my organization’s data as possible.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=3024%3A51698&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=3024%3A51698&scaling=scale-down-width|smart-link] 

*Requirements*

* User should see Configuration category, add a new page: *File Import Setup* under *System> Configuration*
* A Vertical stepper function is displayed on left with steps and current focus step highlighted.
* Each user selection behaves in stepper format are enabled
* User should be able to multi-select *DISTRICTS* from “Search” drop down
** User should be able to search for district names which should filter and display results in drop down selection
** user should be able to remove any of the district selected
** {color:#ff5630}If user is entitled to ONLY ONE ISD (independent school district), Districts search field should NOT be displayed to user but captured/stored in database/backend. {color}
** If user has *ONLY ONE* district, {color:#ff5630}*DO NOT*{color} display for user with the *Districts* selection
* District tooltip is displayed with a user friendly message - “_Please select a district to setup file import configuration”_
* District selection enables *Import Type*
* Display an Import Type drop-down menu that includes the following options (for the different types of imports available):
** Import Type selection is mandatory
**# Student Import
**# DC Import
**# Inventory
**# Item Management Data
**## When this option is selected, display a Data Source drop-down menu where the user has to select a data source.
**## Once the data source has been selected, proceed to displaying the Download Template option as normal.
**# Production
**# Marketplace
**# Student Eligibility
**# POS Balance
* Depending on the option selected, display a prompt whether user would like to download a template of sample configuration file. 
** Only display the button after an Import Type has been selected
* If user selects YES, 
** Template download option in .csv OR .xlsx file for the option associated with it.
*** The user prompt should disappear once user selects
*** The template will include all possible column headers in the file, but the rows will be blank for the user to fill out.
*** Tool tip is displayed with template download option
*** “*NEXT*” button is displayed for user to be go to next step: File Upload
* If user selects NO, system should automatically takes user to the next step: File Upload

## Acceptance Criteria
* Verify that within the Configuration category of the System module, we have added a File Import Setup page is added

* Verify Districts Search Filter drop-down is enabled as required field with ability to remove selected districts

* Verify tooltip for district is displayed

* Verify each selection in the pre-upload screen brings next function in stepper format

* Verify all options are selected: Selecting Import Type and downloading a template file

* Verify that the Import Type drop-down contains at least the 8 options above 

* Verify template download prompt is displayed when user selects import type and disappears after user selection

* Verify File Upload step is loaded when user select NO to download template prompt

* Verify Download template is displayed only when user selects Import Type and YES to download template prompt

* Verify that download template button does not appear until an Import Type has been selected

* Verify that download template button allows the user to download an .xls or .csv file (can be blank for now if needed, as long as file download is possible) for every Import Type

*Verify that, once any one of import selections indicated in the requirements are selected, the user can click on "NEXT" button which takes them to "File Upload" step

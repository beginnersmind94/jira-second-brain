# NXT-57666 — DMF - Migrate Application History data from SQL to Azure Table Storage

## Description
Migrate Application History data from PrimeroEdge SQL DB to SchoolCafe Azure Table Storage in AuditTrail table.
We had used this stored procedure [dbo].[{{Export_To_CouchbaseBucket_SE_ApplicationHistory_Migration}}] previously to migrate data to Couchbase.
As we are going away from Couchbase, we need to migrate to Azure Table Storage.
Source DataSource will be SQL Server and Target will be Azure Table Storage.
This story is similar to Person History story [https://primeroedgenext.atlassian.net/browse/NXT-57313|https://primeroedgenext.atlassian.net/browse/NXT-57313|smart-link]

# NXT-31557 — Add Training Button to Camera Alignment Screen

## Description
As a cashier, I want to be able to manually trigger an image upload into the cloud so that I can test the process or train images.

*Application*: POS

*Flow:*

* Open POS
* Go to Camera Alignment screen
* View video of tray in landing zone
* Click the Scan button

*Background:*

* This was completed in classic and you can copy the code from it
* Get with Saketh on the classic code
* We already have the upload functionality on the cart screen as a “Rescan” button, so you can copy from that
* Take the button and do what that does, uploading images into the cloud
* Need to be able to take tons of images in one session so that we can train images with it.

*Requirements*:

* On the camera alignment screen, add a button to trigger an upload
** Can copy from the cart screen
** label it “Train Image”
* When button is clicked, do all the things you do on the rescan button on the cart screen
** The end result is that the Edge device takes a pic and uploads it into the cloud the same way it does when in learning mode on the cart screen
*  
* Security – no updates required

* In Progress

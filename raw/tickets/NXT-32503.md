# NXT-32503 — Application - Root Cause Why Application Frequently Stops Responding

## Description
Users frequently need to log out and then log back to “resolve” the issue of the application no longer responding.  This not only creates a bad impression, but it could lead to many frustrated users who lose their work while encountering this issue.  

This objective of this story is to find the root cause of why the application stops responding.

## Acceptance Criteria
* A root cause and solution shall be identified for why the application stops responding/working during sessions.

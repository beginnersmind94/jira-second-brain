# NXT-39409 — Scheduler – View Job Scheduler Details

## Description
As a SchoolCafé user, I would like to be able to quickly view the details of an Job Scheduler execution so that I can verify its settings during debugging.

*Notes:*

* This page is read only content, no actions

*Flow:*

* Open SchoolCafé and browse to Settings > Monitor Dashboard > Job Scheduler
* View list of Job Scheduler executions
* Click View Details on a row
* Review the details in a drawer
* Return to the Job Scheduler page

*Requirements*:

* On Job Scheduler page, do the following:
* Add a button to the rows in the action column if it is not already there
** Title: View Details
** On click, Open the View Details drawer
* 
* Create a drawer that opens when click on View Details 
** Title = Job Scheduler Details
* 
* On load, show the html version of the label and value pairs like we have done in Details drawers for other recent work
* Show all the fields from the Job Scheduler record and their values
* 
* Close button

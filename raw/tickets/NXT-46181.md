# NXT-46181 — MP- Menu- Add and Remove Menu Items with inactive and pending deactivation status

## Description
As a Menu Planner, I would like to be able to make changes to my menus to make sure that my menus are up to date and always should have only active menu items.

## Acceptance Criteria
* Any Menu Item Marked as Inactive should not display on the Add Menu Item option
* Any menu item Marked as Pending Deactivation will display on the Add Menu Item option but with the warning, Ideally users should not be adding those menu items but we will allow it for now
** Display Warning with tool tip next to the menu item name in the menu item column “Pending Deactivation”

!image-20240103-141650.png|width=1231,height=438!

* If Menu Item on menu is marked as pending Deactivation, and it is the last menu containing this item, once this item is removed, it should trigger the inactive status and the item will no longer be able to be added back
** This should also effect Family Hub ([~accountid:600ee74adc29b4006a399e3d] [~accountid:603839d29b3ef700694b5a1b] )
* Do a final check of status before the final trigger of the action
** (*Menu Item*) being added has now been made inactive, Menu item can not be added”

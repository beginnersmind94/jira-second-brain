## Acceptance Criteria
+Requirements:+

* Display updated banner chips in the banner section of the Units card
** Make font, UI, icon, size, etc. changes
* Update banner prompts and messages (when expanded)
* Have -3-  2 versions of each banner chip
** icon only
** icon with shorthand status (Incomplete) 
** -icon with expanded status description (Inventory Incomplete)-
* Update style of Summary card banner to match any Units card changes
** show Summary card banners independent of Units card progress
* Provide a “Quick Edit” button on some expanded banner prompts

+Banners outlined:+

* Inventory Readiness chip - always present 
** Ineligible - “Add Pack Size units to enable item for Inventory.”
*** *REMOVE DISABLED for Inventory sub-status*
** Incomplete (orange) - 3 possible banners:
*** 1st. “This item is Inventory Incomplete due to unlinked Pack-Serving relationship.” 
**** *Quick Edit*: take the user directly to the Pack-Serving link edit mode
*** 2nd “This item is Inventory Incomplete due to missing Cost/Value.”
*** 3rd “This item is Inventory Incomplete due to missing Category information.”
** Unpublished (gray) - “This X-tier item is {color:#ff5630}*Inventory Unpublished*{color} with *[base unit description]* as the Base Unit.” 
*** *Quick Edit*: take the user directly to Inventory Options sub-step
** Ready (green) - “This X-tier item is {color:#36b37e}*Inventory* {color}[{color:#36b37e}*Published / Synced / Locked*{color}] with *[base unit description]* as the Base Unit.”
* Menu Item chip - always present (flow is covered in [https://primeroedgenext.atlassian.net/browse/NXT-53795|https://primeroedgenext.atlassian.net/browse/NXT-53795|smart-link] )
** Disabled (gray) - “This item is menu item disabled. Enable it as a Direct Menu Item?” 
** Incomplete (orange) - “This menu item is {color:#ff991f}*Incomplete*{color}. Continue enabling this Direct Menu Item?”
** Ready (green) - “This item is {color:#36b37e}*Menu Item Active*{color}.” 
** _in the future we will try to show Menu Item Synced with more information about active menu assignments, production records, etc._
* Missing Cost/Value - _use the current message and Quick Edit behavior_

# NXT-9008 — INV - Ordering Conditions - Summary Page - List

## Description
As a District Supervisor I want an Ordering Conditions summary page so I can view the ordering conditions I have created.

_Inventory > Orders > Ordering Conditions_

+Item List+

* Displays only items with some ordering conditions currently configured
* Sorted by Storage Category then Item Category
* Table:
** Item Number
*** searchable
** Item Description
*** searchable
** Site Type 1 (ex: Elementary)
*** Site Types will be pulled from System module Site Type configuration
**** Columns will be named according to Site Type names from System module
**** ***research for dynamic column names*
*** this will display the vendor from which the majority of these sites are ordering this item
*** this will also display in parenthesis how many of the sites are ordering from the vendor
*** ex: ‘Sysco Foods (20/24)’
*** searchable, by vendor name
** Site Type 2 (ex: Middle Schools)
*** same as Site Type 1
** Site Type 3 (ex: High Schools)
*** same as Site Type 1
** Site Type 4 (ex: Alternative)
*** same as Site Type 1
** Additional Site Types (using Kendo grid with selectable columns)
*** Display top 4 Site Types by default, with additional Site Types selectable via Kendo additional grid columns
** *Pagination used for the table
* A button to open the Item Detail slide-out (covered in [NXT-5707|https://primeroedgenext.atlassian.net/browse/NXT-5707])

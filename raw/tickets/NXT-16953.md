# NXT-16953 — Applications - Block processing of applications that have gone through Verification

## Description
As Vera the Director I want to make sure my staff do not adjust any apps that have exited Verification. They need to advise the household to submit a new application, or enter a new one themselves.

n/a

*requirements*

* Any applications that are in a verification sample should not be adjusted.
** Do not display More Actions under Application Details.
*** Show a comment for why the actions are blocked.
**** 'Applications that are part of a verification sample cannot be adjusted. The household must submit a new application after verification.'
* The default action for a Verified application needs to be updated to Notify instead of Process.

The only action available for a Verified application is Notify.

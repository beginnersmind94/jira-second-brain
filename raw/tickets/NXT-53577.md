# NXT-53577 — SC - GDSN Import - Removing - Item Name Already Exists validation at GC End

## Description
_(none)_

## Acceptance Criteria
Currently, the items with duplicate names are getting validated at GC End, this is causing issues at initial upload where records are getting removed. 



In order to enable the upload further, this validation needs to be removed.

As a result, the duplicate names shall be validated at SC end, not at GC end.

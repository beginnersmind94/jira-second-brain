## Acceptance Criteria
* Create the Pick Ticket reports
** see attached examples
* Open a report slideout when the Pick Tickets button is clicked
* Provide a Report Format option - radio selection
** Consolidated by Item (*default*)
** Separated by Site Order
*** if this is selected, display a Site dropdown
**** select All sites by default
**** allow individual sites to be selected (multi-select is ideal, but single site could be acceptable in the short term)
* Provide a Display Option section (stretch goal)
** Group by Storage Category (*checked by default*)
** Group by Item Category (*NOT checked by default*)
* Generate the report according to the selections
** Consolidated By Item
*** Show all items with the consolidated pick quantities
** Separated by Site Order
*** Print a page for each order
*** add site/order details to the header
** Include pack size conversions; decide on format
*** after Item Description
*** underneath Item Description
*** in a dedicated Pack Sizes column (like Count Sheet)

# NXT-21346 — IM Report Landing Page

## Description
As a Nutrition department user I need a landing page where all IM reports available are listed for my selection.

## Acceptance Criteria
This story is done when:

* I click on the report option on the left navigation bar and the system displays the reports page
* All reports are listed on the report page grouped by category
** Allergen Report
** Allergen Chart
** Ingredient needing allergen review
** Locally grown usage report
** Missing exception letters
** Recipe Costing Report
** Added ingredient recipe
** Recipe Material Report
** Nutrient standard report
** Menu Item Nutrients report
** Acceptability Factor
** Usage Report
* Placement of the report is a place holder where by the report name could be replace by another report name
* When I select a report the system generates the selected report
* List on Alphabetical order
* Display report description

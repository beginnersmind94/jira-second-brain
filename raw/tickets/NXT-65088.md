# NXT-65088 — MP - Menu/ Menu Cycle Analysis - Update and Combining Grains and MMA for Daily and Weekly (SBP 14+)

## Description
Based on new requirements, analysis for breakfast should be combining the Fruit and Veg values when making sure a district is compliant with USDA standards.

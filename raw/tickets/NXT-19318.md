# NXT-19318 — UI API Integration - INV - Physical Inventory (Complete status) - Items tab - Remove Broken Units

## Description
As a FSD I want Broken Units removed from the Items list so I can accurately review items on completed physical inventories.

*Requirements*

* Remove Broken Units
** from Physical Quantity column
** from Perpetual Quantity column
** from Damaged Quantity column
** from Discrepancy column
* Remove Whole Unit sub-heading
** use only ‘Physical Quantity’ column
** use only ‘Perpetual Quantity’ column
** use only ‘Damaged Quantity’ column
** for Discrepancy header
*** use only ‘Quantity’ as sub-header (not ‘Whole Unit’)
* Remove ‘Supervisor Search’ header section heading
** No heading is needed. “Supervisor” was only used to help convey the purpose when first developing
* Group related items together in sorting
** Show items in descending Unit Type order (Example: Pallet > Case > Package > Each)
** Example: for Rice Krispies which are related as parents or children, show next to each other in the list when initially loading the page

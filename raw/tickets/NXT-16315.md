Requirements

* Recipe Code
* Menu Item
** Allow this to be clickable so the user can navigate directly to the menu item (new tab)
* Serving Site
* Meal Contribution
* Number of Servings (User editable field)
** Pulls from Production
** If no data available, put 0
*** This is common if there isn’t enough information on the recipe that is tied to the menu item
* Materials Needed
** Calculates the total number of ‘items’ needed for the recipe. Calculates the serving size by the number of servings, then uses that to add up the total quantity of every item in that recipe
* Cost per serving ($)
** Shows the total cost per serving
*** This is not quite implemented yet in Item Management
*** Technically the price in Item Management is not a mandatory field and lives under the ‘Product’ section
* Total Cost ($)
** Takes the cost per serving times the number of servings
* Trash / Delete Icon
** Are you sure you want to delete this Menu Item from the Forecast?

Paginated list

Total number of items in the bottom right

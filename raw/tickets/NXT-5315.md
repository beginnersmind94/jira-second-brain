# NXT-5315 — Onboarding - System Task List - Roles - Display Roles Tour

## Description
As a new SchoolCafé user, I would like to have a basic walkthrough on the steps required to set up my district’s roles, so that I don’t have to spend time guessing what to do next.

*Summary*

When the user comes to the Roles task, display a simply tour pop-up (with a Finish button), that explains what the user must do to ‘complete’ this task.

*Mockup*

!image-20230523-214528.png|width=420,height=359!

*Details*

In Onboarding → System → Tasks, when the user selects Roles and is redirected, do the following:

* Display a tour in the center of the screen:
** Text:
*** Title:
**** *How to Complete This Task*
*** Body:
***# This step is completely optional, as we have added standard roles for you. However, you can also select a role, select a module and then modify its permissions in order to customize it.
***# It is recommended that you select the dotted icon and copy a Standard role before customizing it. This will allow you to go back to using a role with default permissions more quickly. You can also select '+New' to add a permission from scratch.
* Display a ‘Finish Tour’ button in the footer.
** When the user selects this button, close the pop-up. This should prevent the tour from executing when the user logs out and returns to this page.

## Acceptance Criteria
Verify that when the user comes to this task, they are immediately presented with a tour step in the center of the screen

Verify that the text and button placement of the tour match the description above

Verify that when the user selects ‘Finish Tour’, the pop-up closes and the user can proceed

Verify that the user will not see the tour again when they are logged out and return tot he page

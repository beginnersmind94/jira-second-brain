# NXT-56981 — System - Management - Year Begin - Add initial set of Year Begin tasks, View Assigned tasks (STRETCH)

## Description
*This is a stretch goal for the Year Begin feature*

As a Cybersoft or district admin, I would like to be able to have a central location that explains every task which is necessary to have a complete year begin process, so that users will have smooth transitions between academic years.

*Summary*

# Check-in the tasks Matt will have created in Texas ISD (QA, active tasks only, there’s 27 of them) as part of “master data” for this feature for new district creation
# Allow users to view the full text of the description in the tasks grid
# Allow users who have been assigned tasks to view those assigned tasks in their Workspace, under My Tasks
## Users should only be able to navigate to a page if they have permission or are were assigned that task (if we can make exceptions for users who may not have access to a page but have been assigned to complete a task on that page.)
# -When a Cybersoft Support (Customer Support) role/user adds a new Standard task, distribute it to every active district on the server.-
## -Likewise, when they update a Standard task, update it for every active district on the server.-
## -Also, when they delete a Standard task, remove it from every active district on the server.-

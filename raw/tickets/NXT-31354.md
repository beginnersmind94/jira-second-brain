# NXT-31354 — IM - Items - Allow Single Item Menu Items without Recipe

## Description
As a District Admin I want the ability to create a Menu Item out of a single item without being required to also create a single item recipe, so that I may create menu items for purchased finished goods.

*Note:* 

* the scope of this story is only for *single item* recipe and menu items
** dedicated Menu Items / Recipes will be handled later
* R&D required first
* this story may have downstream impacts on Menu Items and Recipes and may require additional stories for cleanup.

*Requirements*

* Allow Menu Items to be created from a single item independently of recipe logic
** Serving Measure - Provide a Serving Measure selection
*** only for direct Menu Items (no Single Item Recipe)
** Calculate Menu Item cost according to Fair Market Value 
*** convert unit into serving size 
* Save recipe-specific fields for “invisible” recipe
** In the backend, we will still *link* these ‘non-recipe menu items’ to a recipe, but this recipe will not be visible to the user
** # of Servings - default as 1
* Require a Recipe if preparation steps are required

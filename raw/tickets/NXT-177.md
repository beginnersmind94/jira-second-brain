# NXT-177 — POS - Additional Income

## Description
As a cashier I should be able to accept payments for transactions that are not associated with Lunch program so that I can categorize these into additional income

*Summary:*
Cashiers should be able to accept payments for purchases that are not associated with lunch programs. For instance, sales that are made on POS which are not associated with Lunch program such as Fees, sport fees, fines etc., 
PS: The items sold in additional income will be handled in a different story

*Requirements:*

# Cashier should be able to accept payments within an open service session by clicking “Additional Income”
# The option should appear in the breadcrumb menu reading “Additional Income” preferably below payments option.

## Acceptance Criteria
AC1 : Additional Income option should be added to the breadcrumb 
AC2 : Additional Income needs to be its own transaction type
AC3 : Cashier should be able to accept multiple payments for additional income in a session

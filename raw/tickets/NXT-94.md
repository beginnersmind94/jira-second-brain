# NXT-94 — Account Management - Debt Management - Repayment Plans

## Description
As a Central Office Admin I want a way to manage and control the amount of debt that student balances have at the district. 

*The basic functionality here is the same as the PrimeroEdge Reminders page.*

Requirements:

* User can search for students based on balance or debt criteria, and produce a list of students / households that letters can be sent to. (Example: All students / households that owe between X and Y dollars.)
* User can create letter templates that will be printed with data from the account.
* User can create and manage multiple levels of reminders based on set criteria. Reminder 1 may be anyone owing $10 or less. Reminder 2 may be anyone owing $20 or less, and anyone who was already sent Reminder 1.
* User can set a particular default letter template to be used for each level.
* User can configure debt repayment plans for accounts.
** User can access who has met their repayment plan, and who has missed it.
** User can send reminders for missed payments.

# NXT-39592 — IM - MI Consolidation - New Menu Item - Create from Item/Recipe

## Description
As a Menu Planner I want an option to create menu items for existing items or recipes, so I can create all menu items in one place regardless of which type.

*Requirements*

* On the +New Menu Item button dropdown, provide an option to ‘Create from Item/Recipe’
* Provide a popup, slideout, etc. to search for items and recipes
** only show results of items and recipes which do not already have a direct menu item
* Allow users to select a single item or recipe, provide a confirmation step
* Once confirmed, load the item or recipe on its corresponding details page, with the Menu Item & POS Info tab pre-loaded

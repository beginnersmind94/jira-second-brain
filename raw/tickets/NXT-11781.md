# NXT-11781 — Generation of Letter based on the template

## Description
As a platform user, I would like a library that can generate letters based on the definition of the letter templates.

*Input :* List of business objects (Applications, Students etc), Letter template id

*Output :* Generated letter content each business object or Map of business id as key and content as value. 

Mashing up the variables in the letter template with the data that will be passed in the list of business objects. 

*Example :* 

# Replacing the Student Id field defined in the template with actual value.
# Replacing the Address field defined in the template with the actual value.

Values are passed with in the list of business objects as input.

*Mockup*

None as this is a library to generate letter.

## Acceptance Criteria
Letter content as HTML aligned with proper values from business objects.

for example, For student eligibility module and approval / disapproval letter type, there should be approval or disapproval letter exactly as defined in the template with proper values.

# NXT-5405 — Data Exchange - File Storage

## Description
*User Story*

As a PrimeroEdge developer, I have a solution in place for file storage that I can hook into while developing any feature that requires it.

*Requirements*

Develop a solution for storing any type of files when uploaded by a user. Create any APIs and document to ensure any developers working on PrimeroEdge can hook into the solution.

## Acceptance Criteria
* Files able to be uploaded and stored in a user's instance
* API available for other developers to use
* API documentation created as needed

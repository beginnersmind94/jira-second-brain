# NXT-35066 — PR- Prep site Distribution- Page Redesign (UI ONLY)

## Description
This story is to handle page redesign to give a better user experience without the idea of horizonal scrolling

We will be displaying the Distribution Tab in a way that we will show all the sites at once

## Acceptance Criteria
* The header will only contain the sites along with the Planned/ Shipped for receiving sites and Planned/ Prepared ( For Prep Site)  and the total column (Must accommodate max 8, after that we can look into horizontal scrolling) 
* Menu item row will contain Menu item, menu item help button,  serving size, Shipping calendar and Food required
* User will be able to enter the counts per menu item for each site under the menu item row
* Display non prepared items without all the sites since they are all menu items that belong to the prep site. 
** On the total Column It should be Planned/ Prepared
** Do not display non prepared items if there is no prep site on the particular menu line

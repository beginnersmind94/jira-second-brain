# NXT-15763 — Tagging  - Log Assignments/Unassignments 

## Description
Platform needs to provide a means to store the audit for any changes for the tagging assignments. Any assignment or unassignment of a tag shall be logged.

*Mock up*
None. This is for backend work only.

## Acceptance Criteria
* All assignment or unassignment of a tag shall be logged.
* The data elements to be logged shall be those supported by the existing audit log mechanism plus the tag name and the tagged entity.  This includes the user making the change, the timestamp of the change, the tag name, and the tagged entity.

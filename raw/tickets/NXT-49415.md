# NXT-49415 — IM - Find & Replace - Improvements for New Ingredient Quantity

## Description
As a menu planner I want the New Ingredient Quantity to provide the gram weight of available measures, as well as hiding the convertible measures by default, so that I can easily ensure the correct replacement quantity is provided.



!image-20240416-170642.png|width=83.33333333333333%,alt="image-20240416-170642.png"!

!image-20240416-183229.png|width=33.33333333333333%,alt="image-20240416-183229.png"!

## Acceptance Criteria
* Change the dropdown to the one with the ‘Show Convertible Measures’ button
** this is on the individual recipe Steps & Ingredients
* Show Gram Weight (or Milliliter volume) for the replacement ingredient 
** Display an Info icon with a tooltip of the Gram Weight or Milliliter Volume
* If necessary, change the behavior of the Secondary Measure field to display below the primary measure field, instead of to the right of it
** Ideally this would be determined dynamically off page sizing
* Remove clear icon from ingredient quantity dropdowns
** creating more space for dropdown values

# NXT-26110 — Platform - Permissions - Resource List based on Resource Type

## Description
As a Primeroedge admin, I should be able to view list of permissions available to view, edit and add to respective modules, pages and functions based on the resource type

+*Mockups:*+
[https://www.figma.com/proto/vEncfcQPT3DYGVKOVAagis/System-%26-Platforms-5.0?node-id=1363%3A105397&scaling=scale-down-width&page-id=1363%3A105299&starting-point-node-id=1363%3A105397|https://www.figma.com/proto/vEncfcQPT3DYGVKOVAagis/System-%26-Platforms-5.0?node-id=1363%3A105397&scaling=scale-down-width&page-id=1363%3A105299&starting-point-node-id=1363%3A105397|smart-link] 

*Requirements:*

* When user selects a Module, list of pages under specific resource type to be displayed respectively
* Based on the resource type, resource table list to be populated for the given hierarchy
* Updating the resource type shall reflect the resource list
* All pages, sections and fields are pre-defined by developers and are not editable (name) for the district admins

## Acceptance Criteria
* User shall be able to see list of resources based on the resource type selection
* System shall display list of pages available (not used) under specific selected module
* User shall be able to update the resource type in which system shall update the resource list table
* User shall be able to select the resource that is applicable for the resource hierarchy

# NXT-2709 — System - Users - Set user image

## Description
As a sponsor administrator I want the ability to set a user’s profile image so that I have a personalized view throughout the application when working with my staff. 

We should also give the ability for individual users (if they have the permission) to modify their profile picture. 

This functionality is not provided in PrimeroEdge Classic.

# NXT-2708 — System - Users - Configure user roles

## Description
As a sponsor administrator I want to configure a single or multiple role permissions to a user.

Note: Roles are defined within the role management pages. Depending on the organization access (district, region, site) these roles may vary.

New: Roles typically consist of access to a particular page and/or feature. We need the ability to segment CRUD actions per role as well for the general/overall framework.

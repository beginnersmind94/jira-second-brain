# NXT-2824 — Template Hub - Global Template Section

## Description
*User Story*
As an Inspections user, I have Global Templates provided by Cybersoft that I can add to my own instance and receive updates for.

*Mockup*
[https://m5uloa.axshare.com/#id=7r5wnv&p=template&view=default&g=1|https://m5uloa.axshare.com/#id=7r5wnv&p=template&view=default&g=1]

*Requirements*

* There is a Global Templates section available on the Template Hub screen
* All available global templates are present when viewed by the user
** Sorted alphabetically by default
** Any restrictions placed on Global Template through the settings are applied
*** For example, a Global Template only for users in Texas do not appear for other users
*** Restrictions covered by [https://primeroedgenext.atlassian.net/browse/NXT-2812|https://primeroedgenext.atlassian.net/browse/NXT-2812|smart-link]
* Global Templates are created and provided by Cybersoft
** Users cannot directly edit Global Templates
** Users can view Global Templates like any other template
** User can email and export Global Templates like any other template
*** Actions are available, functionality covered by respective stories
* Users can add Global Templates to their instance
** This is done through an action on the individual template card/list item
*** The action is replaced by a grayed out “Already Added” once it has been added
** A copy is made that the user can edit
** Templates are placed in the Inactive category and can be used or edited immediately
** The template is labeled as a Global Template, as well as any tags associated with it
* Global Templates that have been added by the user are indicated and grayed out
** UI mockup required for this
** There is an indication if the Global Template has been updated since it has been added
*** UI mockup required for this
** There is a button that allows user to update their current template to the new version
*** Functionality covered in another story

## Acceptance Criteria
* Global Template section accessible in the Template Hub
* Global Templates able to be copied into user's district's templates

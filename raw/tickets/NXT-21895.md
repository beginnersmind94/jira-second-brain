# NXT-21895 — INV - Vendors - Vendor Detail - Items tab

## Description
As an Inventory manager I want an Items tab on the Vendor Detail page so I can see all the items for my vendor regardless of Contract.

*Requirements*

* Add an Items tab after the Contracts tab
* Include Add Item button on the Contract Items grid
** *only* when a single contract is selected/displayed
** this opens the the Add Item screen
* Display the item list from the Contract Items page, but for all active contracts
* Provide a Contract dropdown 
** Multi-checkbox dropdown
*** by default, *all active* contracts will be selected and displayed
** List all contacts
*** All Active contracts first (ascending by Contract End Date)
*** Inactive/expired contracts second (sort by Contract End Date - soonest first)
** Show a label for Active/Inactive next to each contract
* Provide an Apply button 
** on click of Apply, if any unsaved changes for an item, show the unsaved changes prompt

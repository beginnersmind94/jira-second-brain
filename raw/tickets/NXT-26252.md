# NXT-26252 — System - Periods - AUDIT actions for Generate Periods - API

## Description
As a SchoolCafe admin, I should be able to see the audit trail of the periods when it was created and managed.

*Requirements*

* NEW AUDIT/History action is enabled within Actions column. 
* When user clicks on AUDIT, periods audit page is displayed with timestamps, user and Description and NOTES Column
* History of when a period has been created along with timestamp & user information is added
* SPLIT function on a period is recorded for all spitted periods
* EDITs on the periods is also recorded

## Acceptance Criteria
* User shall be able to click on AUDIT action button to view the AUDIT history of the period
* If the periods audit is empty, deactivate the AUDIT button so user shall not be redirected unnecessarily
* User shall be see the history when, who and what time the periods has been created by any given site selected
* EDIT and SPLIT function changes to the periods are also recorded in AUDIT tab

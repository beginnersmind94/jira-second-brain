# NXT-15056 — Acct.Mgmt - Outreach - Quick Broadcast -Add page and filters

## Description
As a central office admin, I would like to be able to instantly broadcast an outreach letter to specific students and adults within my district, so that I can use SchoolCafé to  market a variety of district programs to a subset of our population.

*Summary*

In the Quick Broadcast tab of the Outreach Center, let’s fill out the tab's content with filters -and a template editor-, and a Broadcast button, allowing users to select their audience and immediately send them the Outreach letter via email.

*Details*

* In Account Management → Outreach Center → Quick Broadcast, display a list of filters. All of them can default to ‘All’ except for Template, and should cascade as needed, such as how Site determines the list of available grades, etc. Fields:
** Template - In this field, 
*** display all templates of the Outreach Letter type, both active and inactive (similar to how we show all active and inactive ID Card templates in the ID Cards page). 
*** The user should be able to select a single template.
*** We can have the active template be the default option.
** Site Code /Site Name
** Patron Type - Student, Adult, All (both)
** Account Status - Active, Inactive, All
** Eligibility
** Roster
** Homeroom
** Grade
** Age
** Birthday Month
** Tags
** ZIP Code (of Primary Guardian, if applicable)

Below the filters, display the following:

* -Subject field - Allow the user to type a string of text into this field. When the user clicks the Broadcast Message button to send the email, use this string as the subject line of the email.-
* -Template Editor - Display the HTML from the selected Template. Also, allow the user  to edit the template as needed using a rich text editor.-
* *Also include* an Edit Template button that links the user to the System → Templates → Edit Outreach Letter. 
** When it redirects the user, we should open the editor for whichever Template was selected in the filters list of Quick Broadcast, so that the user can edit it. 
** We should also display an additional ‘Save’ button, which when clicked allows the user to get back to the previous Quick Broadcast page with their saved still filters intact.
* Apply Button (not pictured in mock-up) - When clicked, perform the necessary API calls and display:
** Footer message - text: "This broadcast will be sent to X accounts", where X (which should be in bold) is the number of accounts who will be on the mailing list for the message.
* Broadcast Message button - When this button is selected: 
** send the email out to the mailing list (for student, only the Primary Guardian, if they have an email address), 
** reset the filters. 
** display a toast message: "Broadcast sent successfully\!"

## Acceptance Criteria
Verify that the Quick Broadcast contains a set of related filters as shown above

Verify that within that list of filters, the user can select any active or inactive Outreach Center template

Verify that there is a footer message showing the number of accounts who will receive the email

Verify that when the Broadcast Message button is selected, the letter emailed out to the list, and the content on the page is reset.

Verify that there is a shortcut button that allows the user to go and edit the template and then return to this page

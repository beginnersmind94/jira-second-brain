# NXT-40320 — eOrders – Delete

## Description
As a SchoolCafé user, I would like to delete the eOrder I generated so that I can avoid sending a mistake.

*Notes:*

* This is a continuation of the eOrders – Dashboard story

*Flow:*

* Open SchoolCafé and browse to Inventory > Orders > eOrders
* View list of eOrders
* User clicks Delete button and confirms their intention
* System Updates status of the eOrder to deleted and removes it from the scheduler

*Requirements*:

* These requirements are for the eOrders Dashboard page
* 
* In the grid there are rows with an action column
* If not already there, add the Delete button to the action column
* The Delete button can only be shown if Status = Open
** Else hide or disable it
* 
* On click:
** Ask if they are sure want to delete
** If confirmed, they want to delete, then proceed
** Remove the eOrder from the job scheduler
** Update the status of the eOrder to deleted
** Refresh the grid list
* 
* Security - No additional requirements

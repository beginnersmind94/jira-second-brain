# NXT-16886 — INV - Orders - Review Order Settings - Add New Rule - Vendors (Step 3)

## Description
As an inventory operations manager I want a Vendor option on the Add New Order Review Rule process so I can potentially configure order reviews for orders going to a specific vendor or vendors.

*Requirements*

* This is the third step of the Add New Rule process of the Review Order settings
** this step is optional
* Provide a control to add vendors to the rule
** provide a multi-select, searchable dropdown containing all active vendors
** Add button to add the selected vendors to the rule
* Display a list of the vendors which have been added to the rule
** display Vendor Name
** Action column
*** for each vendor, provide a link to open the corresponding vendor page
*** provide a Delete icon to remove the vendor from the rule
*** no need for checkboxes
* Display a Back to Rules List button in the bottom left corner, to cancel and return to the list page
** Prompt user: “Discard rule configuration and exit setup?” Yes/Cancel
* Display a Previous button in the bottom left corner, to return to the previous step
* Display a Next button in the bottom right corner to advance to the next step
* Display a Complete button in the bottom right corner, to skip the remaining steps and jump ahead to the confirmation step

## Acceptance Criteria
* This step is optional/skippable
* Can select one or more vendors from vendor dropdown

* Can open vendor detail (in a new tab) by clicking Vendor view icon

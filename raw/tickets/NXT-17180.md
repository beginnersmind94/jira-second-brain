# NXT-17180 — Export Monitor-Mock up

## Description
This story is to cover the task related to Mock up’s @ Export Monitor

Exports Monitor dashboard will be accessed through system-->ConfigurationExports Monitor and it will be accessed from the tool bar menu beside Imports monitor. 

This will be similar to that of Imports Monitor dashboard.

There must be filters based on below criteria.

# Export Type
# Date Range
# Status

The dashboard should reflect the Status of the job (error, successful or running). Provision must be provided to more detail logs if needed.

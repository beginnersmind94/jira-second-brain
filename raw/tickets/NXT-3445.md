# NXT-3445 — Training Events - List View Details 

## Description
View training event details Topics, Subtopics, Training sources, Training methods. 

* ability to sort/filter events for a convenient find
* Filter for District Events, State Events, All 
* Card View 
* Shows above fields and (with permission) ability to edit, make active/inactive, delete
* Add Created Date (auto populate) column
* Add Event End Date (calendar picker) column
* Filters by annual year, by source (state, district, ICN, SNA), also column category sort/filters
* Per USDA, there are 4 key areas of focus for training topics (see screen shot)
** Nutrition, Operations, Administration, Communication/Marketing

## Acceptance Criteria
The view details card view shows high level information about training events. 

The USDA required fields are completed to save the event.

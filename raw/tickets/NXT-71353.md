# NXT-71353 — GC - PE - Item Catalog  - Do not require item selections for Exporting ingredients and Exporting ingredients and stock items  

## Description
GC - PE - Item Catalog  - Do not require item selection for Exporting ingredients only and Exporting ingredients and stock items  

# Export ingredients only should export all items listed in the search results
# if any column filters are applied , then export items should consider the column filters . 
## Example: if given input (it can be a uploaded file or search text in search filters ) resulted both Cybersoft and USDA items in the results, and if i applied filter for getting only Cybersoft items, then exporting items should only export cybersoft items . So, if user filtered the search results by any column, then that column filter should be applied for exported items.
# item selection is not required for both “export ingredient only” button and “export ingredients and stock items” button.
# Item selection is still required and review items and import items functionality  should work from review items popup.

# NXT-3734 — Fix 1900 value in Expiration Date and End Date

## Description
* Remove the 1900 value from Education > Degree > Expiration Date 
** There is no expiration date for degrees so no entry is needed 
** See mockups for add new employee
* Remove the 1900 value from End date 
** this is an option field and can be left blank when a user creates an employee record 
** the start and end date determine whether the employee is active or inactive.

## Acceptance Criteria
1900 values removed from Degree > Expiration Date and End Date

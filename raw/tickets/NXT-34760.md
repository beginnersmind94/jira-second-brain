# NXT-34760 — Site Code UI Component - Smart Population

## Description
An intelligent UI component that modules can incorporate on their pages.  The desire is to have a UI component that a consumer can easily incorporate by setting values for a UserID and District.  The component will retrieve and populate the appropriate sites that the user has access to based on these inputs.

## Acceptance Criteria
* A UI Component Site dropdown shall exist that will populate itself based on property values set by the consumer of the component

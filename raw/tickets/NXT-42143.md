# NXT-42143 — GDSNConnect - Pending Update - Modal Popup Changes (Allergen, Meal Credits)

## Description
_(none)_

## Acceptance Criteria
*Problem Statement:* 

Currently the user is unable to see any allergen or meal credit changes in UI

*Changes required:*

Enable the field mapping as scrollable view on modal dialog

*Allergen Fields in UI:*

The user should be able to view allergen changes in UI as follows:

Check for each allergen in source data and show the applicable value in UI, In case of UNDECLARED - show them as “None Indicated” in UI, the individual list in modal dialog need not be shown 

Enable check box where user can opt out during import

*Meal Credit Fields in UI:*

should be updated only when serving size section is selected; If serving Size is not selected, show them as disabled view, import should Not update the values on target ingredient

show serving size as additional label in the header section if one or more Meal Credits exist at source

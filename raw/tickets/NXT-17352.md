# NXT-17352 — INV - Orders - Receipt Detail - Move Charges/Discounts section into header

## Description
As an ordering manager I want Charges & Discounts to be displayed in the header so I can view all charges or discounts without scrolling to the bottom.

*Requirements*

* Move the Charges / Discounts grid to the top of the page, aligned with the header
** This will be a third “column” in the header section, independent of the main header sections (Receipt/Invoice fields)
* Fix the height of the Charges/Discounts grid, to ensure it is the same height as the header section
* Provide in-line scrolling for the grid if the list of charges and discounts exceeds the fixed height
* Do not display the Charges & Discounts grid if *no* charges/discounts apply to the receipt
** this would mean there are no charges or discounts configured on the settings page, AND
** there are no items on the order/receipt with any contract charges/discounts (Processing Fee, Delivery Fee, NOI)
* This change would apply to every version of the Receipt Details screen (Create, View, Edit)

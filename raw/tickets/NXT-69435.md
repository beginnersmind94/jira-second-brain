# NXT-69435 — Fin - General Ledger Accounts – Add to Import Types

## Description
As a **district finance/admin user**, I want to **import my GL account categories and GL accounts from an Excel template in one step**, so that I can **quickly populate and maintain my General Ledger configuration without manual data entry**.

## Acceptance Criteria
* On the Add New of the Imports tab of the Import & Export screen when you click Add New, do the following:
  * On the preload tab, add 'GL Accounts Import'' Import to the list of values
  * File Format = **.xlsx/xls/csv** files
  * In the tooltip there, provide the following information
  1. ask your finance team for a **chart of accounts export** (excel/csv) — **don't guess** account codes
  2. enter **account categories** + **gl accounts**, and select a **base type** (assets, revenue, expenditures, liabilities, fund balance)
  3. keep account codes as **text** (preserves leading zeros) and follow your district's **segment structure**; financials adds a **site segment** for site-level reporting (example: `…-025`)
* Required mapped columns:
  * Account Code
  * Account Description
  * Account Category
  * Account Base Type
* ~~Auto-map columns by header match where possible, with the ability for the user to adjust mappings~~
  * Removed as of 2026-02-13
* Prevent the user from proceeding unless all required fields are mapped for each sheet
* Show an import progress indicator stating the import may take a few minutes and the user will be notified when complete
* Treat Account Code as a string during import to preserve leading zeros and formatting exactly as provided
* On completion, show an Import Results (Imports & Exports > Import Monitor > 'General Ledger Accounts' import record >Click on View Log button)
  * Add a statistics section on top
    * ~~Account Categories: total rows, imported/updated count, needs fixing count~~
    * GL Accounts: total rows, imported/updated count, needs fixing count
  * Add the 'Activity Monitor', 'Source Data', 'History Log' and 'Exceptions' tabs

# NXT-48150 — Subscriptions - View Details

## Description
As a SchoolCafe user, I would like to be able to see what subscriptions there are so that I can know who is being notified of what.

*Notes:*

* This is the follow up story to the View Subscriptions tab when a user clicks on the View Details Icon

*Requirements:*

* For the Event Subscriptions page > Subscriptions Tab:
* When a user clicks on the View Details icon, it should open up the View Subscription Details drawer
* 
* Create a new drawer called View Subscription Details
** Size: Half page
** Show an html only version of the label / value pairs like we do in other places
** Be sure to show the following:
*** Module
*** Event
**** Show the user friendly name with spaces
*** Users
**** List all the users assigned to the event
*** Roles
**** List all the roles assigned to the event
*** Criteria
**** Show the criteria, even if it is in xml or whatever format
*** Timing
**** Show timing value selected
*** Created
**** Show datetime it was created
*** Created By
**** Show who created it
** Close button

## Acceptance Criteria
* User is able to see a read only list of the labels and values for the event subscription

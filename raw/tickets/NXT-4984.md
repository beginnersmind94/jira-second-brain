# NXT-4984 — AcctMgt - Add Patrons - Students - Wizard Step 2 - Guardian and Contacts - Add/Edit/Delete

## Description
*User Story*

As a district admin I want to fill out contact information for a student patron.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default]

*Requirements*

* Populate Step 2 of the wizard added via NXT-4982.
* Step 2 will lead with asking for information on the primary contact, which is similar to how the Contact Information section in the current Add Student.
* After adding the primary, information is shown in a table/grid similar to the Contacts panel.
** Edit and Delete options. Confirmation dialog on Delete.
* User can add additional contacts here as well via an ‘+Add Contact’ button, which will reveal a new set of fields.
** -There is a ‘Same address as primary contact’ option in there, which should disable the address fields. We may want to pre-fill them at that point with whatever the primary’s contact address is. (Removed: 9/17/20)-
** User can Cancel out of this additional contact if they change their mind on adding one.
* Validations for zip code, email format, need to fire on this page appropriately.
* 'SAVE & CLOSE' button this step, which will save any information but close out of the wizard.
** User should receive a success confirmation toast about the completed task.
* ‘NEXT’ button available which will save, and take the user to Step 2.
* ‘CANCEL’ button should provide a confirmation prompt for the user.
** ‘Are you sure you want to cancel? Changes from this step will be discarded.’
* ‘BACK’ button is available, acts as a Save.
* Stepper (top right component) can be clicked to allow navigation to a future step.
** This would be the same as the user clicking on BACK/NEXT. Validations + Save must happen.

## Acceptance Criteria
Contact information can be entered.
Multiple contacts can be added.
Information is retained while navigating the wizard.
Information is saved appropriately.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)

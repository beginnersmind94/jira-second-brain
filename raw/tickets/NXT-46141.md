# NXT-46141 — Elig - Applications/DC - SIGs - Updates based on other Application/DC actions

## Description
As Anna the Application Processor I want to perform other actions on State Eligibility applications and I need the system to behave appropriately.

*requirements*

This story is to handle workflow updates for State Income Guidelines (SIGs) and application actions. These scenarios only apply if the setting is turned on.

* Action: Delete
** If an SIG application is deleted, we should roll back any students involved like we do a regular application.
** State Eligibility data on the student should also be rolled back.
* Action: Refuse Benefits
** If an application is re-processed and benefits are refused, students should be updated appropriately.
** State Eligibility should be removed as part of the update process, if applicable.
* Action: Remove student from application and reprocess
** Same as above - we need to roll the student back.
* Action: Pending Student
** If a pending student is matched up and the application processed, State Eligibility should be applied.
* Automatic Processing
** If auto-processing is turned on, State Eligibility should be approved if it falls into the bracket. 
** Accounts should be updated with State Eligibility.
* Online Applications
** Check to see if State Eligibility is being removed / added properly as an application is processed.
* Actions: Direct Cert File removed, DC Student Unmatched, DC Extension Unmatched
** During rollback of student eligibility we need to check to see if State Eligibility needs to be reapplied and update the account.
* Action: Direct Cert Match
** If the student moves to DC, any active State Eligibility should be removed from the account re: person records.

## Acceptance Criteria
If the System Setting is/was on when the application was processed:

* Student State Eligibility rolled back appropriately for actions:
** Delete Application
** Refuse Benefits
** Student removed from app + application reprocessed.
** DC File removed
** DC Unmatched
** DC Extension Unmatched
** Online app approval
* Student State Eligibility updated for actions:
** Pending Student
** DC Match
** Online app approval

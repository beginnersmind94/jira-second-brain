# NXT-68061 — GC -PE - Vendors display on item popup if vendor information is available for an item.

## Description
Display vendor names on the item details popup as shown in the below screenshot. 

Comma separated names of all vendors is displayed. 

Import the AllVendors column data for all cybersoft CC items from Release DB server, Menu_Catalog_Global_Release.dbo.GC_Client_Item table

Currently, item and vendor are 1 to 1 now we need to allow one  to many for item and vendors. 

vendor item number we have in gc_item table is  supposed to be the VIN for the item vendor that is linked to the district item .

* for cybersoft items based on  CC data you can get it from gc_client_item table from release db and Menu_Catalog_Global_Release
* for cybersoft catalog items from USDA data : you can get it from QA database Menu_Catalog_Global_Release.dbo.USDA_CustomerData_MatchingItems

search by filters and selecting vendors should work with any of these vendors to bring up this item in search results.

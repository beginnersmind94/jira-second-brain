# NXT-60896 — INV - Distribution - Assign & Pick - Reset Assignment

## Description
As a Distribution Manager I want the option to rollback my distribution from Picking status to Assignment status, so I can make adjustments/corrections before I start picking again.



!Distribution - Complete Picking-20250415-130503.png|width=91.66666666666666%,alt="Distribution - Complete Picking-20250415-130503.png"!

## Acceptance Criteria
* Provide a reset icon next to the Quantity Assigned column heading (or find space for another button?)
** This icon will be available once the distribution is Assignment Complete or Picking Started

h4. Assignment Complete rules:

* Upon click, prompt user “This will revert the distribution from Assignment Complete. Are you sure?
* Revert the status to Pending Assignment and return the assignment controls

h4. Picking Started rules:

* Upon click, prompt user “This will revert the distribution from Picking to Assignment. Are you sure?”
* Revert the status to Pending Assignment and return the assignment controls
** Release any reserved quantities

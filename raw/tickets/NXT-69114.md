# NXT-69114 — Fin - Complete Mapping on Account Mapping Page

## Description
complete mapping side drawer from mapping page in gl config

## Acceptance Criteria
h2. COMPLETE MAPPING SIDE DRAWER

* Opens when clicking "Complete Mapping" button in progress banner
* Drawer slides in from right side
* Shows overlay behind drawer
* Header displays: "Complete Mapping" with X close button
* Content shows only unmapped data points
* Section title: "[X] Unmapped Accounts"
* Each unmapped account displays:
* Data Point name
* Group name (subtitle)
* Typeahead input for GL Account selection
• Selecting a GL account in drawer:
* Saves mapping
* Removes account from drawer list
* Updates main table
* Updates progress banner
• When all accounts mapped:
* Drawer shows success state with checkmark icon
* Text: "All accounts mapped!"
* Subtext: "No unmapped accounts remaining"
• Drawer closes when clicking X button, clicking overlay, or pressing Escape key

* *When all system data points are mapped, the progress banner transforms to a success state displaying "All system data points mapped!" with a green checkmark, hiding the "Complete Mapping" button and the bar*
* System only uses active data points so if a needed data point is made inactive, it is removed from the mapping progress count.

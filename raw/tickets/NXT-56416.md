# NXT-56416 — DMF - ADF-KeyCloak Integration

## Description
Support the district migration framework with its integration needs into KeyCloak

Data exchange has started the project to migrate the legacy clients to SchoolCafe 2.0. As part of this process they are migrating all the data from Classic database to SchoolCafe. All the SQL related stuff will be handled by DataExchange Team.



We need to handle below items

* Create a District in Keycloak
* Create all the Sites that were migrated from Classic in Keyclaok
* Create all the users that were migrated from Classic in Keycloak (Assume that all the users will have email associated. Any user without email will be skipped)
* Update SQL Mappings tables(NF_UserXRole etc..) that are bridge between Keycloak & SQL



We can utilize the existing CreateDistrictAsync method to create the district and necessary things.

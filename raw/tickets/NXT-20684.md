# NXT-20684 — INV - Inventory - Additions - Addition Details - Cancel

## Description
As a Site Manager I want a Cancel function on Inventory Additions so I can make changes if I have the permission.

*Requirements*

* On the Addition Details page ([NXT-20680|https://primeroedgenext.atlassian.net/browse/NXT-20680]) if user clicks the Cancel button
* Prevent cancellation if… (on click of Cancel button)
** any of the quantity from the Addition batch(es) have been used
*** message: “This Addition cannot be cancelled because some quantity has been used.”
** the items are part of a completed Physical Inventory which has a ‘Counted On’ date that is later than the Addition transaction date
*** message: “This Addition cannot be cancelled because it has one or more items on a completed Physical Inventory.”
** -the period has been closed- (we will do this later on the Period stories)
* Display a popup requiring Comments to be entered (similar to Physical Inventory)
* Once the comments are entered, reload the page
** Hide all buttons
* Display a success message: “Addition transaction #123456 has been cancelled.”

# NXT-12296 — Platform - AI/ML - Student Opt-out Facial Recognition

## Description
*Persona*: Student/Parent

*Access level:* School District/Site

*Description*:

As a parent, I should be able to use the Perseus application to de-register my son/daughter’s profile  for Facial recognition.

## Acceptance Criteria
1. Parent should be able to access facial recognition section in profile page from students/parents portal. 
2. User should be able to disable facial recognition with reason for disabling drop-down
3. Parents should be able to confirm the student account update to Opt-out for all services or specific services
4. Email confirmation is sent for parents about facial recognition Opt-out with link to re-enable with benefits in detail along with privacy confidence factors.

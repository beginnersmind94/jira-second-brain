# NXT-8153 — Orders - Label Configuration Page and Label Dimensions

## Description
As a user I need to be able to create labels or tickets to indicate who order is intended for, what is in the order, and how it is to be delivered.

see [www.labelgrid.net|http://www.labelgrid.net]

*Requirements:*

* The user will be able to create a label template for use by the whole district
* The user can define page configuration, width, height, left margin, top margin, number of rows and number of columns. The measurements are in Inches.
* The user can define the label configuration, width, height, horizontal space, vertical space, all in Inches.
* The templates can be saved.

## Acceptance Criteria
Under Extra a function Order Label Configuration can be selected.
The user can select a template or select NEW
When a template is selected a representation of the template is displayed
When NEW is selected a blank page template is loaded, Page size 8.5x11, 0 margins, 1 row, 1 col
Label dimensions are  8.5x11 and 0,0 for the vertical space, and horizontal space
When new dimensions are saved the updated template is displayed.

# NXT-3203 — Accountability - Sessions - Reconciliation - Bulk Adjustments

## Description
*User Story*

As a user I need to be able to make the same change to multiple meals in a session or to the session details.

*Mock-up*

PrimeroEdge version for Reference

!Screen Shot 2020-09-04 at 4.49.27 PM.png|width=807,height=489!



*Requirements*

* Adjustments to the session can be made
* Adjustments to the transactions can be made.
* A summary of what was changed is shown
* A comment and user date/time tag are required.

## Acceptance Criteria
Adjustments to the session can be made
The Terminal name can be changed for a session
The Date can be changed for a session
When the session date is changed the date for all the transactions are changed.
Adjustments to the transactions can be made.
A different Menu Item can be selected.
all transaction matching the selected criteria update.
A comment and user date/time tag are required.
Meal prices are based on the patrons eligibility

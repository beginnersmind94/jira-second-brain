# NXT-18073 — INV - Orders - Order Detail - Receipt tab - Receipt Details - Add Items to Receipt

## Description
As an operations manager I want the ability to add items to a receipt so I can add any substituted items to the receipt.

Scope:

* Selecting items
* Adding items to receipt
* Loading added items on receipt details screen

*Requirements*

* Once the Add Items to Receipt screen is loaded (per [NXT-16383|https://primeroedgenext.atlassian.net/browse/NXT-16383])
** User can select multiple items by checking them
** Once items are selected, user can click Add Items 
* Item List columns:
** Select column
*** this will provide a checkbox for each item
** Item #
** Item Description
** Vendor Item #
** Storage Category
** Item Category
* Below the grid, display a Back button that will exit the Add Item process and return to the Receipt Details
* Below the grid, display an Add button that will add the selected items to the receipt item list
** Do not enable the Add button unless at least one item has been selected
** Return to the Receipt Details screen with the items added to the item list
* Once items are added to the receipt:
** add items with zero Order Quantity
** no pre-populated Received Quantity
** load current price
** populate any contract charges/discounts
** **Note:* items are not added to the order; only the receipt

# NXT-18389 — API and QA - UI- Production - Production Plan- Add Menu Item

## Description
As a site manager, who is planning service for the day… there are occasionally times when there needs to be additional menu items added to my production plan that were not originally on the menu assigned. This could happen because an item has gotten popular or possibly close to an expiration date and needs to be used.

When this occurs, I need to be able to select and add a different menu item to my Production Plan.

## Acceptance Criteria
The user will be able to select a menu item from the software to add to their production plan.

The user will be able to add the quantity planned.

The user will be able to also remove that item from the page, in the event that they select an incorrect item to add. (Note that the user will not be able to remove any of the items originally assigned to their menu.)

The food required and withdrawal details will also come through with that item on the production record, as the user enters the details.

# NXT-6055 — INV - View Vendors - Table View

## Description
User Story

As an inventory user, I want to be able to see all of my vendors in a list so I can quickly view and access my vendor information.

Mockups

!image-20201013-153511.png|width=100%!



*Requirements*

* +Vendor list table+
** Table columns:
*** Vendor Name
**** this is a link that will open the [view vendor info screen |https://primeroedgenext.atlassian.net/browse/NXT-6049]
*** Vendor #
*** Phone Number
*** Vendor Type
** Default table sort order:
**# Vendor Name - Alphabetically ascending
**# Vendor # - Numerically ascending
**# Vendor Type - Alphabetically ascending
** column headers have a search box (as part of [Table Header Search story|https://primeroedgenext.atlassian.net/browse/NXT-6591])
** Active/Inactive tab at the top to switch table view from active vendors to inactive vendors.
*** This is a toggle/switch to show either Active or Inactive vendors
*** page is otherwise exactly the same
* Add button in top right corner to add a new vendor
* Results on the page will be a pagination control
** this control allows has a selector for how many results per page are shown

Ignore for now:

* ‘View Items’ link next to vendor name

## Acceptance Criteria
User will be able to view all vendors in a table.
Clicking the vendor name link will open the specific vendor view info page.
Clicking the Add button in top right corner will open the Add Vendor screen.
Pagination control will be used for table.

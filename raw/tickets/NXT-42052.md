# NXT-42052 — Vendor Imports – POS Items Import – Auto-Link Matched Columns

## Description
As a SchoolCafé Admin I want to quickly move the rows across from Source to Mapped so that I can more quickly complete the task.

*Notes:*

* We already can drag and drop, but we want to be able to even more quickly move fields across

*Requirements*:

* When importing an POS Items Import on the Field Mapping Screen, do the following:
* When the page loads the first time, automatically link fields from Source to Mapped as if it was drag and dropped this way:
** Columns with the exact same name
** Columns that match with special characters removed
*** Strip spaces, _, ., - and other special characters and try to match
** Column names used in our import file
*** We know our import file and what they map to, so make sure those columns map too

## Acceptance Criteria
* All the Source fields are linked to their related Mapped fields when page loads

# NXT-68635 — Fin - Reports Landing Page – Report Selection

## Description
As a *District Finance User*, I need a Reports page that clearly presents each financial report I can generate (General Ledger, Journal, Revenue & Expenditure (P&L)) so I can quickly produce required financial statements.

## Acceptance Criteria
h2. *Report Cards Section*

h3. *Reports Displayed*

The page must display *three* report tiles:

# *General Ledger*
_Description:_ “Export detailed debits and credits with per-account worksheets.”
# *Journal*
_Description:_ “Export a list of all financial postings in transaction order for a selected date range.”
# *Revenue & Expenditure (P&L)*
_Description:_ “View revenue and expense activity summarized by category for a selected period.”

h3. *Tile Requirements*

Each tile must include:

* Report name
* The description stated above
* Entire tile is clickable and opens the report’s configuration page
* Sorted A-Z

Visual behavior:

* Hover state uses elevated shadow
* Click leads to the specific report’s configuration screen

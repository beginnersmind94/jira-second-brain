# NXT-35827 — System - Users - Allow user to sort/filter by User Status column

## Description
As a central office user who manages user accounts, I want the ability to manually sort all users by their status, so that I can quickly find the inactive accounts and reach out to those users to activate them or do so on my own (in case they have trouble getting the new user email). 

*Requirements*:

* Enhancement request originated from Columbia Public Schools with ticket [#200918|https://primeroedge.freshdesk.com/a/tickets/200918].
* In System → Sites & Users → Users, make changes to the User Status column:
** Allow the user sort it
** Allow the user to filter it with text strings like the other columns
* The user should be able to do both (preferably), but at least use one to easily find Active or Inactive users. We may eventually want to allow them to export this for their IT as well, but that can be a future enhancement.

## Acceptance Criteria
Verify that users can sort and/or filter the User Status column in the Users page.

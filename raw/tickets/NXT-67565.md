# NXT-67565 — IM - Bulk Item - Update Nutrients (Individual Slideout)

## Description
As a Nutritionist inputting information into the system on the Bulk Item Tool, I would for a way to individually enter information for my nutrients similar to how the item details page is. This allows for me to easily make sure all the information is entered in one go.

h3. Bulk Nutrients - View - Updated

* Fat and Carbs condensed to one column with multi labels (view mode only)
* Action column with view icon added (view mode only)

!Bulk Nutrients - View-20251105-034520.png|width=512,alt="Bulk Nutrients - View-20251105-034520.png"!

h3. Bulk Nutrients - single item slide-out (view mode)

* Take the Nutrients card from Item Details and put in a slide-out
* Load the nutrients slide-out in view mode
* Replace the top-level edit icon with a footer-level edit button

!Bulk Nutrients - Slideout View-20251105-034030.png|width=512,alt="Bulk Nutrients - Slideout View-20251105-034030.png"!

h3. Bulk Nutrients - single item slide-out (view mode)

* In edit mode, change footer buttons to Cancel and Save
** only enable Save if changes are made
* Allow Serving Size edit according to the exact same rules as the Item Details page

!Bulk Nutrients - Slideout Edit-20251105-034030.png|width=512,alt="Bulk Nutrients - Slideout Edit-20251105-034030.png"!

## Acceptance Criteria
* Update the view mode of the Bulk Nutrients Review screen
** Condense column widths
** Use ? for missing values
** Use (M) for values with the Missing checkbox checked
* Combine all 3 Fat columns into a single Fat column with comma separated values
* Combine all 4 Carb columns into a single Carbs column with comma separated values
* Provide a Details column with a view icon to allow for users to individually edit nutrients (Onboarding and Update)
* Trigger a slide out for the user to view or modify nutrient information for a particular item
** This will look like the nutrient info card on the item details page
** Move edit pencil at top to an edit button in the slide-out footer
* All validations for entering nutrients on “Item Details” page will be enforced on this slide out
** Do not allow save to be enabled if any required nutrient information is missing
** Allow Serving Size edit according to the exact same rules as the Item Details page

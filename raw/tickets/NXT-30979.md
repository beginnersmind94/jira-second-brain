# NXT-30979 — MP- Menu- Variety Menu Item Nutrition Disclaimer (Menu/ Menu Planner)

## Description
As a menu planner, when I add a Variety Menu item to my Menu, the system is taking all the menu item nutrients and giving an average.

Requirement: 

# Show a disclaimer “ Nutrient display on Variety Menu Items uses an average of nutritional values of all menu items”
# Show a Disclaimer “ Meal Component Display on Variety Menu Items (VMI) uses the meal component set by user on VMI creation”

## Acceptance Criteria
* Display both disclaimers if variety menu item is *added* to the menu on the menu analysis screen
** Display on Analysis Slide out
** Display on Nutrition Report
* Display Disclaimer on the Menu Cycle if a variety Menu item has been *added* to *any day*
** Display both on Analysis Slide out 
** Display both on Nutrition Report
** Display (1) disclaimer on Nutrition Summary Report
** Display (2) disclaimer on Food Component Report
* Display both disclaimers if variety menu item is *added* to the menu planner in the disclaimer section
** Currently Disclaimer is static once criteria is loaded, will need to switch to dynamic once VMI is added and show appropriate additional disclaimers

# NXT-41203 — ICC - Back Office - Production Accuracy - UI Integration - Site Level (Data Grid)

## Description
_(none)_

## Acceptance Criteria
Navigation: Insights → Command Centre

User should be able to see 2 menu for command center as Front Office and Back Office

Upon clicking Front Office all the KPIs built based FO data shall be visible to users (as-is)

Upon Clicking Back Office, the following KPIs shall be listed at site level

Production Accuracy %

Serving Percentage %

Upon clicking on percentage, the following data should be displayed in slide out:

Planned Quantity

Produced

Total Served

Site Level cards

The threshold bar should reflect in UI based on the values in defined threshold settings

Export - user should be able to export the data in Excel

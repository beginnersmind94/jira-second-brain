# NXT-16910 — INV - Orders - Review Order Settings - Edit Existing Rule

## Description
As an operations manager I want an edit option for my order review rules so I can change the parameters of an existing rule.

*Requirements*

* When user clicks the view icon on the Review Order settings list page, open the stepper for that rule
* Retain existing saved values
* Open the rule on the final step, Confirm 
* User can click through the steps to make changes
* Changes are temporarily saved in the session
* As user makes changes to the rule, those are indicated as ‘Draft’ or ‘Unsaved’ changes, which are indicated on the final step
** Display an alert icon next to the label of any steps which had changes
** Display a hover tooltip on the alert icon: “This step has changes which have not been saved to the active rule.”
* ‘Create’ button is replaced with a ‘Save’ button
** Save is required to officially save any changes to the active rule
* If user leaves the stepper process without saving the rule, changes will be lost
** prompt user with unsaved changes prompt

# NXT-13231 — IM - Recipe Costing Report 

## Description
As a User i would like generate a Recipe cost report so that It’s easy for me understand the recipe cost and the cost per serving

## Acceptance Criteria
*Report*

* Entry point is IM page → Reports → Recipe Costing Report
* Report fields should populate as per below report sample
* Report should use pagination similar to items “All Food Items Report”

!image-20211130-080102.png|width=1125,height=793!

* User should be able to print report
* User should be able to export report to PDF format
* Upon exciting the system should return to reports landing page



*Calculations (See Attached excel file)*



*Recipe Material Cost Calculation*

!image-20211130-071053.png|width=1197,height=629!

The calculation sample is attached in excel sheet

*Step #1  Calculate as Purchase Amount  (Amount needed to prepare recipe)*

*Step #1  Calculate Cost* 

* 2.1 Get Item Cost                                                                      - E.g.  1 LB  / $ 3.00

!image-20211130-072428.png|width=938,height=460!

 

* *Step #3  Convert units as required*                                                 

          1 LB = 16 oz; 16oz = $3

* *Step #4  Calculate Individual Item Cost*

         Individual Item Cost = Amount needed to prepare recipe  *  Cost per Unit  

          E.g. (3oz *  $3)/16oz  = $ 0.5625 



* *Step # 5 Repeat steps 1 - 4 until all items forming the recipe steps are included*



* *Step # 6  Add up all individual item cost  in recipe steps to obtain Total* *_Recipe Cost (=Recipe Material Cost)_*



*Cost Per Serving Calculation*

Cost Per Servings = Total Recipe Material Cost / Number Of Servings

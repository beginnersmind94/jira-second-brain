# NXT-16375 — Show description for Code Mapping-Internal Code

## Description
Currently, we are only showing internal codes which are not very intuitive to users and they may not what this code means.
To improve the useability, we need to also provide the description for each internal code.
The description will be provided in the Internal Code dropdown.

e.g. For the Gender field, we are showing 3 options F, M, and N on the code mapping page.
will add descriptions next to code  such as F (Female), M (Male), N  (Not Provided)
If a code is self-explanatory then it will not have any description next to the code.
Increase the width of the Internal Code dropdown so that the description will be visible to the user.

## Acceptance Criteria
Internal code description if available needs to be displayed along with the internal code in the mentioned format - Internal Code (Description).
Internal Code and the description should be legible.

# NXT-33518 — IM/INV - Items - Units - Option for counting Base Unit in decimals

## Description
As a District Admin I want the ability to decide whether an item’s base unit should be able to be counted in decimals in Inventory so I can ensure we are counting items according to their real pack size details.

Example: a milk carton will never be counted in decimals, but a bag of flour would be.

*Requirements*

* Provide an option in the context of the new Units card to determine whether the Base Unit can be counted in decimals
** TBD when this option is locked 
* If base unit does not allow decimals, the item will always be restricted to being counted in whole numbers
* -Update Inventory to consume the decimal flag to enable or disable the presence of decimals throughout the module for that item-
** Separate future story

# NXT-43545 — MP- Menu Templates- Apply serving size exceptions

## Description
As a menu planner, when I designate a serving size exception in IM, I am purposely specifying that a particular serving group gets a specific serving size or a serving group does not get served a particular serving size.

## Acceptance Criteria
* Apply serving size exceptions based on configuration in IM when adding menu items

!image-20231003-071725.png|width=1607,height=733!

* When adding menu items, continue to show the default serving size on the add menu item grid, however once the menu item is added display the correct serving size associated with the serving group 
** User still has the ability to change serving size but its just the default that will be displayed for them once added
* If user removes menu item and adds it back again, display the serving size exception again even if it was changed the first time after it was added

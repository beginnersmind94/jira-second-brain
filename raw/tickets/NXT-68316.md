# NXT-68316 — DE - SFTP > Student Import>Validate the import file before importing it

## Description
Validate the SFTP Student Import file; if the validation fails, don't start the import process via ADF.
Validate the file for the following validations:
1)The file is empty.
2)The file has only header rows.
3)The file has no header rows.
4)Column names in the file match the DE configuration. Column names are case sensitive 
For example, the file has StudentId as the column name, and the DE configuration has StudentId, then it is a valid match, and if the DE configuration has studentId or Studentid, then it is not a valid match.
5)Column order. The order of columns in the file and that in the DE configuration should match.

There needs to be a line item in the Import Monitor grid when the validation fails for an Adult Import.
An error-friendly message needs to be displayed when the mouse hovers over the Error label.
Fixed-width and Excel files will not be validated.
For a file without headers, check if the file is empty or not.

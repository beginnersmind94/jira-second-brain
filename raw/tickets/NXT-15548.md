# NXT-15548 — System - Periods - Disable the ability to edit past periods

## Description
If the period is in the past or currently taking place, there should not be an option to edit the dates.

*Requirements:*

* Periods that in OPEN state with past dates shall NOT be able to EDIT

## Acceptance Criteria
* As a user, I shall be able to choose a starting and ending month in an intuitive manner when defining reimbursement rates.
* As a user, I shall be able to intuitively understand what the starting and ending months are when viewing reimbursement rates.

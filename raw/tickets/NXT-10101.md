# NXT-10101 — Verification - Dashboard call-to-actions

## Description
As a district admin I want the dashboard to give me useful information about verification so that I can take quick actions.

*requirements*

If Verification is happening, we should make use of the call-to-action insight buttons at the top of the dashboard to call out pertinent information.

* User is aware of how many days until verification starts.
** This can check the system setting for the final sample pool date
** This will be a countdown in days until the final verification sample is pulled.
** Title: Days until Verification Begins
** Link: Take the user to Verification > Sampling
** This should only be shown until 10/2.
** Note:
*** If the user has scheduled a final sample, use that date. Otherwise use the system setting date.
* User is aware of how many days until verification ends.
** This will become a countdown until the verification end date system setting mentioned below.
** Should only be available after verification has begun.
** Title: Days until Verification Ends
** Link: Take the user to Verification > Tracking
** This can be shown until 14 days after the system setting date below.
** This will need to use the system setting below. ‘Verification Response End Date’
* Add the following system setting:
** ‘Verification Response End Date’
*** Date, default: 11/15/XX 11:59:59 PM

## Acceptance Criteria
CTA available for:

* Days until Verification Begins
* Days until Verification Ends

CTAs only show on the dashboard when relevant.

* Begins only shows until the Final Sample date + 1.
* Ends only shows until the Verification End Date + 14.

# NXT-7568 — INV - Physical Inventory (Complete) - Physical Inventory Report - UI Only

## Description
As a district admin I want a Physical Inventory report so I can review the items and entered counts for a site inventory.

*Requirements*

* Add another report to the Physical Inventory review screen
** “review screen” = Pending Reconciliation or Reconciled status
** currently there is the Discrepancy Report and Valuation Report
* Provide the simply ‘Physical Inventory Report’
** this is basically just a printable version of the item list
** should be the last report for this screen
* Display header level info (from Summary tab)
* Provide item details (simple version only)
** Item #
** Item Description
** Quantity
* Group items
** if possible, consider page grouping from [https://primeroedgenext.atlassian.net/browse/NXT-28580|https://primeroedgenext.atlassian.net/browse/NXT-28580|smart-link] 
** alternately, display Storage + Item Category grouping

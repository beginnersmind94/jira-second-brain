# NXT-71 — System - Management - Add Program Configuration page

## Description
As a Central Office Administrator I want to manage the programs in which my district is participating, so that I can properly configure site participation data.

*Summary*: Add a Program Configuration page in the navigation and redirect the user to the page.

Requirements:

* In the *Management* category, add a *Program Configuration* page
* On the page, add 2 tabs: 
** Programs: This tab is used for configuring what Programs that the district participates in and how, by site
** Sites: This tab is used for configuring what Site programs should be reimbursable, and their meal type timings
* The Programs tab should be the landing tab of the page
* All other details and page behavior will be addressed in [https://primeroedgenext.atlassian.net/browse/NXT-2703|https://primeroedgenext.atlassian.net/browse/NXT-2703|smart-link] and [https://primeroedgenext.atlassian.net/browse/NXT-3555|https://primeroedgenext.atlassian.net/browse/NXT-3555|smart-link]

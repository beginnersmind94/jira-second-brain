# NXT-31644 — PR- Pre Production Analysis Changes for WGR Percentages

## Description
Based on New USDA regulations effective July 1, 2022 WGR offering must be >=80.00% for all Breakfast and Lunch Meals for students grade K through 12.

Currently in the software and based on previous requirements, we were checking against 50.00% 

!image-20220606-213103.png|width=823,height=644!

## Acceptance Criteria
Change the WGR check for weekly analysis to be against >= 80.00% for all grade groups EXCEPT PK

Apply logic to fail WGR percentage if under 80% 

This change applies to:

*Pre Production Analysis Slideout*

*Food Component Summary Card (Nutrition Report) and Print Options*

*Food Component Report and Print Options*

Apply change to *Disclaimer* on the bottom of the slide out 

Current: 

4.PrimeroEdge will flag based on USDA requirements starting School Year 2019-2020 for WGR and 1% flavored milk, for prior years, please consult your State Agency for that School Year requirements and waiver status.

Proposed: 4. Schoolcafe will flag based on USDA requirements started School Year 2022- 2023 for WGR and 1% flavored Milk.

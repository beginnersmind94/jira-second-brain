# NXT-19745 — Import - File Scheduler

## Description
This story is to cover the development of a scheduling platform. This platform is to trigger the file transfer which helps in the moving of files within local directories of a system or the from secured-FTP systems. 

The designed scheduling platform can be used to trigger both Imports and Exports as needed.

The scheduler must be able to run at-least every minute and must be able to accommodate requests of 24x7 transfers and to run any day of the week

User must have the option of selecting serial or parallel processing. In case of serial processing, if a file is taking longer time for processing and the next scheduler comes in to effect, the next one has to be skipped since it is already running.

Incase of errors or a scheduling is taking. more than 10-15 minutes (threshold value), email notifications must be sent to the support team so that they can look into the issue.

Multiple schedulers can be set up for a single file transfer.

# NXT-6216 — Item Management - Recipe - Edit Nutrient Info

## Description
*User Story*

As a Item Management User, I want to be able to edit the nutrients for a recipe

*Mockup*

!image-20200929-144424.png|width=100%!



*Requirements*

* When clicking the pencil for nutrients to edit, bring up the edit screen from the drawer on the right
* Display serving size and measurement as plain text rather than being editable
* These nutrient information can be manually edited
** This information can only be edited IF there is a serving size and number of servings
*** If there are no number of servings AND serving size, then the pencil for edit is hidden
* The fields should show all nutrients from Item Management
* User can override the fields and click the ‘SAVE’ button
* Information is then saved into the database
* If the items are updated, but the user has manually changed information here, we will NOT update the information

## Acceptance Criteria
User will be able to edit and save nutrients

# NXT-33759 — Workspace - Distinguish Between Read/Unread Messages

## Description
The messages in Workspace need a visual indicator so that a user knows if the message has been read or not.

## Acceptance Criteria
* Messages shall have a visual indicator so that a user knows if the message has been read or not
* An action is needed for a user to mark a message as read and unread

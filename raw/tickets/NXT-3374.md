# NXT-3374 — Inspection Template - Template Creation

## Description
*User Story*
As an Inspections user, I can create/edit and save a template using the most basic settings.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=add_new_questionnaire&id=usu4ze&g=1|https://nqyc9v.axshare.com/start.html#p=add_new_questionnaire&id=usu4ze&g=1]

*Requirements*

* Create Template screen can be accessed from any screen using the Create button
** Existing Template can utilize the same screen to be edited
* Template name can be filled out (Required)
** 50 characters max
** Special characters allowed (make sure to sanitize for potential SQL injections)
* User can select scoring from None, Pass/Fail, or Custom Scoring
* User has a save button that creates the template or saves changes to an existing template

## Acceptance Criteria
* New Template can be created
* Existing Template can be edited
* Name and Scoring Type can be edited
* Template can be saved and appear in Template Hub

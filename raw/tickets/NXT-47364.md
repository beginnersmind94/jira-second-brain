# NXT-47364 — IM/MP - Find & Replace - Menu Items - Replace

## Description
As a Nutritionist or Menu Planner, I want the Find & Replace page to have both Ingredients and Menu Items so I can replace both on the same page.

## Acceptance Criteria
* Provide a replacement step for replacing menu items on menus
** Further filter the menus that DO NOT Contain both menu items ( “Find” Item and “Replace” Item) – Currently handled from UI but needs to be filtered from API
* Upon clicking the Replace action, enable a selectable mode of the Menu Items grid
* Adjust the header to a view-only mode where action buttons are removed
** Show both menu items with an icon to open in a new tab
* Provide checkboxes for each menu item 
* Display a sticky footer 
** Cancel button to reset the entire process
** banner displaying the count of selected menu items
** Replace button to confirm selections and validate data entry
* Upon click of sticky footer Replace button, advance to the Confirm step
** Replace Cancel with “Back to Menu Selection” to allow user to go back to the selection grid

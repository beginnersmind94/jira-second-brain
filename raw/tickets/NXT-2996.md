# NXT-2996 — Employee Details - Profile Image

## Description
Ability to capture or upload an image to an employee profile. Visible in grid and card view. 

* UI shows capture/upload capability 
* use same upload tool as system module 
* Admin can add/upload photo

## Acceptance Criteria
Demo that the admin (and employee - phase II) can capture or upload profile picture. That picture is visible in the card and grid views.

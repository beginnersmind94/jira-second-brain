# NXT-51301 — Imports - Import Monitor - Dates - Activity Monitor

## Description
As a Customer Support Role user, I would like to be able to see import data for older imports so that I can diagnose issues from the past

Notes:

* This story clones the enhanced date range story
* Because of that, there is now the possibility to view data older than 28 days
* Azure limits the Activity Monitor data to only 28 days, so if we look at an older import, there will be no data to display
* This story encapsulates how to handle no Activity Monitor data

*Requirements*:

* On Import Monitor tab when click View Log button, do the following:
* On Activity Monitor tab, do the following:
* If there is no data to show because it has been purged:
** Do not show an error
** Do not try to show the grid data
** Show this message in the body: Activity Monitor data purged after 30 days

## Acceptance Criteria
* User sees a friendly message when there is no activity monitor data

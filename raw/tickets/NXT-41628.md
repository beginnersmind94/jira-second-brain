# NXT-41628 — IM/MP - Find & Replace - Menu Items - Find & View

## Description
As a Nutritionist or Menu Planner, I would like a page that will give me the capabilities to be able to find items or recipes that are within recipes, and be able to just look at those recipes and replace an item or recipe with another within recipes.

## Acceptance Criteria
* Repurpose the existing MP Find & Replace page
** merge with new IM Ingredients page
** separate Ingredients and Menu Items with tabs (or something else!)
*** Accessing from the IM Find & Replace - Land on Ingredient tab
*** Accessing from MP Find & Replace- Land on Menu Item tab
**** Rename nav path in Menu Planning to Find & Replace
** redirect navigation to IM page
* Update the Find UI to use the new UX from Ingredients F&R page
* Provide a Menu Item search which will allow the user to select a menu item to remove/replace from menus
* pre-filter this list by actual menu items on Active menus
* for performance quality, consider requiring minimum character count for ingredient search (3 Characters)
* Provide filter options
** Menu Item Type (multiple checkboxes): Direct Menu Item, Recipe, Variety Menu Item (Check all by default)
** Pending Deactivation Menu Items Only: this would filter to show only menu items which are in Pending Deactivation status (*Already Done*)
* In the dropdown, show an icon for Pending Deactivation menu items (Done)
** consider aligning with IM icons
* Upon clicking the Find button, load and display all active menus containing the selected menu item
* Once menu items are loaded by the Find action, display the Replace With section
** show the menu item being replaced in a *view only* mode at the top of the card
** provide an option to open the menu item in a new tab
*** Land User on Menu Info Tab when loading Menu Item
** display the icon if the menu item is Pending Deactivation
* Display the Menu grid with overview info
** Menu
** Site Group
** Meal Type
** Next Date Assigned
** Action column with view icon
*** to open the menu item in a new tab

# NXT-52110 — PR- Autofill- All autofill calendar options default to serving date of the plan (UI ONLY)

## Description
As a production manager, when using the autofill functionality, currently it is setting the calendar to todays date and not the date of service that I am working on.

## Acceptance Criteria
* Default calendar display to always land on serving date
** If user closes the slide out and opens up autofill up again, always land on the serving date
*** This will ensure that the calendar will always be showing the most relevant information in terms of days that can be used to autofill based in relationship to the serving date.

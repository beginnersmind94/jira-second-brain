# NXT-62722 — PR - Pre & Post Analysis - Consider Conversion for Veg-BP to MMA Component

## Description
Per USDA guidelines, Veg- BP is not always a 1 to 1 conversion with MMA in terms of using Veg- BP as an MMA on a menu.

## Acceptance Criteria
* IF user selects “Use as MMA” for Veg-BP, check the conversion set in IM to ensure proper component analysis
* All logic remains the same for the weekly as far as using the offered Veg- BP but now the change will be in considering the amount of MMA to calculate
* All logic for daily analysis will always consider the conversion from Veg-BP to MMA
* Do not affect any record complete records, consider conversion at the time of record complete status change
* For carryovers that are being inserted into a plan (Not on the menu), use default Veg- BP
** For carryovers that are being Added to a plan (Menu item is on menu), use flag that is set from the menu

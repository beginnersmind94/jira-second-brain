# NXT-66310 — Translated Data Download - Field Level Changes

## Description
_(none)_

## Acceptance Criteria
Navigation: DB Enabler

Changes Required in following areas:

*  Search API
*  View Item API
*  View Log API
* Item Import Batch Creation (SC)
* Translated Data Sheet
* Exception Download Sheet


*USDA Branded Foods Source:*

Item Category value should map the field value of “branded_food_category”


Brand Name value should map the field value of “brand_name”


Is Food - Populate it as Yes  if calorie is not NULL (zero is valid calorie) , if not, populate it as No

Lowest GTIN - is populating GTIN_UPC without leading zeros - should be fixed to reflect the GTIN_UPC value as-is

Unit 2 GTIN is not applicable as we only have one GTIN in USDA - so leave this as blank

*Cybersoft Source:*


Item Category value should map the field value of “Item Category Group Name”

Storage Category value should map the field value of “Storage Category Group Name”



Valuation Group value should map the field value of “Valuation Group Name”



Brand Name value should map the field value of “Brand Group Name”


Manufacturer Name value should map the field value of “Manufacturer Group Name”


Product Code - check if either of the Product Code or Stock Item Product Code is found - include the 1st match that is found



Is Food - Populate it as Yes  if calorie is not NULL (zero is valid calorie) , if not, populate it as No



Lowest GTIN - should be populating GTIN with leading zeros, populate either of the GTIN or Broken Unit GTIN if Not NULL based on the 1st matched value

Unit 2 GTIN - should map the value of GTINCode field

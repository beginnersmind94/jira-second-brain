# NXT-9807 — Data Exchange - File Import Dashboard: File Import Setup EDIT

## Description
As a PrimeroEdge user, I would like to have an intuitive screen that allows me to view all file import setup configurations that are available for me with an option to EDIT file Import setup configurations so that I can edit/update configurations relevant to the latest business process

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A238848&scaling=scale-down|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A238848&scaling=scale-down|smart-link] 

!image-20210112-143456.png|width=734,height=234!

*Requirements:*

* When the user has clicks on “*File Import Dashboard*” under configuration, file import dashboard is loaded to view, create and import configurations.
* File Imports configurations table is displayed with all file configurations available for that user (districts)
* Each row has “*EDIT*” buttons available when user clicks on ellipsis icon
* when user clicks on *EDIT*, system should:
** retrieves File Import Setup configuration from the system and loads the *FIELD MAPPING* screen
** Once user completes the field mapping updates, system should load the “*Review Step*”
* In *“Review Step” ,* updated mapping table is displayed with *CURRENT FILE NAME* labelled field.
* System should update the file name *incrementor by +1*. 
** for ex: If the current file name was “*ISD_Student_Import_Config_1*” before the EDIT action, after user navigates through mapping, system should update the name as “ISD_Student_Import_Config_2” automatically
** At this step user should be able to *EDIT* the file import name and proceed to next step by clicking on “*SAVE*” button
* System should update the File Import Setup configuration row with *latest* *name* in the table that is displayed in File Import *Dashboard*.

## Acceptance Criteria
* Verify that when the user has clicks on “File Import Dashboard” under configuration, loads the dashboard page with EDIT action available in ellipsis icon for each row of the file import configuration table. 

* Verify when user clicks on EDIT action, user is taken to "Field Mapping" screen

* Verify that once user completes Field Mapping configurations, Review screen is loaded with updated "Current File Name" labelled field.

* Verify that system reflects the latest name of the File Import Name in the File Import configurations table in File Import Dashboard

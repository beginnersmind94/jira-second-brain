# NXT-63976 — Acc - Reports - Participation - Page / Report Updates

## Description
As Dana the Director I want finer control over selections on the Participation reports so that I’m not having to run the report multiple times. This will be an efficiency driver.

!image-20250714-192739.png|width=1019,height=633,alt="image-20250714-192739.png"!



*requirements*

For both Participation and Non-Participation:

* Update dropdowns to be multi-select
** Site Code / Site Name
** Homeroom
** Grade
** Meal Type
** Menu Item
* Date Selector swapped out for the new one
* Reports updated to take and display the parameters accordingly

## Acceptance Criteria
For both Participation and Non-Participation reports:

* Selected dropdowns are converted into multi-select
* Date selector swapped out
* Reports accept parameters and display accordingly
** Report Criteria also updated accordingly

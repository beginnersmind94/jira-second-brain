# NXT-24730 — Acc. - Dynamic Menus - Option to enable/disable by site, by terminal

## Description
As a user of Dynamic Menu I need to be able to select what sites have Dynamic Menus used.

Add a Dynamic Menu on/off to POS Configuration in the System Site Configuration page if the Dynamic Menu district setting is enabled.

Default is off.

Page for reference

!Screen Shot 2022-01-19 at 5.27.52 PM.png|width=1413,height=644!

The District setting for enabling Dynamic Menus is no longer used.

POS Is updated to use the site setting only.

## Acceptance Criteria
If DM is enabled for the District then a selection for the site is available.

The user can enable DM for the site

DM is disabled for the site by default

The District setting for enabling Dynamic Menus is no longer used.

POS Is updated to use the site setting only.

# NXT-28383 — INV - Orders - Change Delivery Date R&D

## Description
As a Site Manager I want to be able to change the Delivery Date for an order without having special permissions, so I can select a different valid vendor delivery date.

*Note: this is different than the existing edit Delivery Date functionality, which is for *overriding* the delivery schedule. By contrast, this is the need to simply switch to another valid delivery date for that vendor, typically a _later_ date. Right now the system will choose the earliest available (another thing we need to enhance). 

*Component Note:* 

* Currently we are using the Material Date Picker and old Kendo time picker 
* There is a new Date Picker and Time Picker not being used anywhere in Inventory
** [Date Picker|https://devuicomponents.perseusedge.com/?path=/docs/components-date-time-kendo-date-picker--default]
** [Time Picker|https://devuicomponents.perseusedge.com/?path=/docs/components-date-time-kendo-time-picker--default]
** We can update to the new components if they will help accomplish this story’s requirement

*Requirements*

* Identify an approach and proper component for the following requirements
** Add the ability to modify the Delivery Date on the Confirm Orders screen
*** this could appear like the existing edit icon, but would have a different function for these users
*** or we could use a different icon, since this could be a common function (depending on how much we change the delivery date selection on the Standard Ordering page)
** Allow users to select valid vendor delivery dates
*** perhaps use a dropdown to show the next X number of valid dates, according to vendor delivery schedule
*** If user has ‘Override Delivery Schedule’ permission, they can select any date
**** if the date is not valid, they will still see the alert (but no hard stop)
** Implement sticky footer for Create button and Back button
*** could possibly put the Combined Order Total here
*** could possibly add other reports or future additions in this as well

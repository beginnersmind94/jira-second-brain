# NXT-30288 — PR- System- Plan- Allow changes to X days in the future (Plan Only)

## Description
As a Food Service Director, I would like my managers to only be able to plan production for a certain amount of days in the future. 

Currently, the system allows a user to edit or change production plans however in the future they would like.

## Acceptance Criteria
Use the setting placeholder currently in the system> settings >production> plan and record to control this setting

*Plan - Allow Changes X Days in the Future (Plan Only)*

This setting will affect all production plans and records (reminder records are controlled by the internal setting to not allow changes to future records)

The input for this setting is numerical, default to 100 days

Based on default of 100, user will be able to plan for 100 days in the future

User has the ability to *view* the plans but can not make edits or save any new information (if more than 100 days in future)

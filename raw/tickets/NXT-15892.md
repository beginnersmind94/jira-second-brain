# NXT-15892 — IM - Clear the Serving Measures 

## Description
As a User I would like to have an Ability to Clear my Serving Measures So that it’s easy to clear all the existing Serving measures and re-enter to make changes 

Requirements

* Serving measures that are actively being used by a recipe / menu item and is attached to a menu, we cannot delete the serving size.

!image-20210712-144907.png|width=91.66666666666666%!

* Disable the checkbox to select and then ‘clear’ if the measurement is being used in a recipe / menu
* [~accountid:5d56d6b9a1f3290c16e66c64] to check if the measurement can be ‘edited’
** Also check if it is in use in nutrients / contributions
* Confirmation when clicking ‘Clear’: “Are you sure you want to clear the selected serving measures?”

## Acceptance Criteria
Given the User is already on the Edit Mode for Nutrient and Contribution Information 

And already made changes or appears with existing information 

When Clear button is tapped 

Then Clear all the data for the below fields 

Serving Measures

* Serving size
** A check box with strike
** Ability to Enter the Serving Size in number digits
** Ability to Select Serving Measure
* Weight in Grams
** A text filed to enter the number value
* Preferred Measurement

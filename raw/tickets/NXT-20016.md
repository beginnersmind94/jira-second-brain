# NXT-20016 — INV - Receipts - Allow $0 Invoice Amount (UI)

## Description
As an ordering manager I want to be able to create receipts with a $0 value for Invoice Amount so I can create receipts when the vendor sends me free product.

This applies to both regular receipts and quick receipts.

*Requirements*

* Remove the pre-population from the Invoice Amount field
* Allow $0 to be saved in the Invoice Amount field

Scenarios:

* Creating regular receipt
* Creating Quick Receipt
* Editing any receipt

## Acceptance Criteria
* Allow receipts and quick receipts to be created with $0 value for Invoice Amount

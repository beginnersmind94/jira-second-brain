# NXT-3440 — Employee Grid - Active/Inactive Toggle Filter

## Description
As a TW user, I want to have a quick search toggle to easily view active and all (including inactive) employees.

* Active/Inactive toggle for quick search on grid and card view
* flipped toggle shows all, other shows only active employees

## Acceptance Criteria
Flipping the toggle shows all employees 
default is active employees 
grid and card view have toggle

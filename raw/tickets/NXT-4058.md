# NXT-4058 — Security - Admin Login

## Description
*User Story*
As a PrimeroEdge admin, I have the option for an extra login being required for admin pages to provide extra security for sensitive information and settings.

*Requirements*
* There is a setting available that requires a second login when attempting to access sensitive information or system settings
** Information includes any PII or other private information
** System settings include any user or site screens, or payment information
** This provides an extra level of security should an admin leave a screen open
* Setting can be toggled on or off
** When on, user has to reenter password when moving from non-admin page to admin page
*** There is a timeout where the user can go between without this occurring
** When off, user can access screens without any additional credentials

## Acceptance Criteria
* Setting is available
* When on, password needs to be entered
* When off, user can navigate without additional credentials

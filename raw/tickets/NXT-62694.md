# NXT-62694 — POS - Local Pictures - Controls (Download, Update, Delete)

## Description
As Cassandra the Cashier I see that photos are out of date and want to trigger a manual update.

*requirements*

* Add controls to POS Admin Settings to let users control this portion of local storage.
** User can trigger a download of photos
** User can trigger an update
** User can delete local photos
* Actions at the POS:
** -Update Account Information → should also check for picture updates-
** Photos → Bulk Download, Delete 
*** Bulk Download
**** Should download all account photos for that site
**** Confirmation prompt for the user
***** Text: This will download account photos to local storage for this serving site. It may take ~5 minutes to complete. Are you sure you want to start this now?
**** Cancel button so the user can stop the process
*** Delete
**** Confirmation prompt for the user
***** Text: This will remove all account photos from local storage. Are you sure you want to proceed?
**** Should delete all local photos if confirmed



Discussion (Refinement - 5/28/25)

* Changing POS Sites should also remove all local photos

## Acceptance Criteria
* New Photos options in Admin Settings:
** User can trigger a download of local photos
** User can delete local photos
** User can update local photos (via Update Account Information)

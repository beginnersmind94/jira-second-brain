# NXT-3836 — Edit Custom KPIs

## Description
*User Story*
As an Insights user, I can edit an existing custom Goal using existing Measures.

*Mockup*
[https://qvv7ze.axshare.com/#id=70kqkv&p=custom_kpi_1|https://qvv7ze.axshare.com/#id=70kqkv&p=custom_kpi_1] (Mockup created before having visibility into existing 2.0 mockups)

*Requirements*

* Edit page auto-populates all values related to the existing Custom KPI.
** Name
** Description
** Calculation
** Levels
* Users can edit the values of any field on this page.
* Only enabled Measures will show up
* Name, Description, and Calculation must be entered to Save a Custom KPI.  Calculation must be Validated.
* Users are prompted when attempting to navigate away from this page with unsaved changes.

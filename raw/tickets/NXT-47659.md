# NXT-47659 — Elig - Applications - Better Benefits processing for State Eligibility (SIGs)

## Description
As Anna the Application Processor I want to see State Eligibility updated when processing an application, even if the USDA eligibility does not change.

*requirements*



Scenario:

|| ||*USDA*||*State*||*Actual*||*Expected*||
|App 1|Paid (Income)|Reduced|--|--|
|App 2|Paid (Income)|Free|State stays as Reduced|State should update to Free|



* State Eligibility should update for students even if the USDA eligibility does not change.
** Students should receive a new Eligibility line.
* Need to use the Better Benefits setting as part of this.
** USDA > State
*** Any update that gives the student USDA Free or USDA Reduced would also give State N/A
** State Free > State Reduced
* Reprocessing a paid application is entering a new eligibility line with an Income pricetype start reason instead of Denied.

## Acceptance Criteria
* State Eligibility updates on accounts if a change has happened, even if USDA status is the same.

## Acceptance Criteria
* Provide an Update mode in which the values for the *vitamins* (micro nutrients) can be entered or modified
* Display Serving Size in read only for context of nutrient values
* Provide a radio option to enter either by Weight or by % of Daily Value
** whichever one is selected, show the other option as a read-only label
* Require & confirm all mandatory fields for vitamins
* Highlight any missing mandatory data points
* If any non-mandatory values are skipped, assign them the ‘Is Missing’ attribute (Vitamin A and C)
* Do not include Moisture or Ash

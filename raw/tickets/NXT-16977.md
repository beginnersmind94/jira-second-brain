# NXT-16977 — MP - Menu Planner - Summary

## Description
As a user, I want to be able to add Menu Lines to the menu planner

Mockup

!image-20210728-131312.png|width=100%!

Requirements

* When clicking on the Add Menu Line, it will allow the user to ‘search and select’ or free type to create a new menu line on the local datasource
* Then the user can click and drag an item to the menu line
* When dragging another item to the second 
* Example
** Menu Line
*** Parent Item
**** As many sub items as needed
** Search for Menu Lines, or add as free text
*** Not saving a ‘new’ menu line
** NO SAVE in the DB, just in the browser instance (in memory in the UI but not API / DB)
* Show the most recently used items
* Only keep up to 10 recently used items

## Acceptance Criteria
The user needs to be able to able to add menu line to the menu planner

# NXT-7543 — Item Management - POS Information - COPY Information

## Description
When Copying an Item there should be a section that can be checked allowing the user to copy the POS Information

Selecting the check box copies the POS Information to the new item.

!image (3).png|width=1366,height=768!

## Acceptance Criteria
A checkbox is added to allow POS Info to be copied.
When Selected the POS Information is copied to the new item
When not selected the POS Information is not copied to the new item

# NXT-63480 — INV - Internal Vendors - Contracts - Delivery Groups - Provide an 'Unassigned' group & Update Schedule

## Description

## Acceptance Criteria
* Add an ‘Unassigned’ group to the *Delivery Groups slide-out*
** this will be a system group like the Default one
** put the Unassigned group at the bottom, below any user-created groups 
** do not provide the Edit Stop Order or Delete buttons
* Allow sites to be moved to the Unassigned group
** In edit mode, show a warning message about the schedule being cleared for sites in the Unassigned group.
*** “{color:#bf2600}Shipping Schedules will be cleared for all Unassigned sites.{color}”
* Create Unassigned group for existing data
** add to all existing internal vendor contracts
* Update the *Shipping Schedule* to consider the Unassigned delivery group
** do not allow schedule configuration for sites in the Unassigned group
** display Unassigned as the bottom grouping in view mode
*** but collapse it by default
** do not display the view icon for Unassigned sites
** in place of the schedule tool, display a message in light italic font: _To configure site, assign to a Delivery Group_
** do not display the Unassigned group or sites in any edit mode

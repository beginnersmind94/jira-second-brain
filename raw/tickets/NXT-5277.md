# NXT-5277 — Verification - Collection Report (FNS-742)

## Description
As a district admin I want the system to generate the information for a FNS-742 for me, so that I can export it out or transcribe the information to report to my state department.

*mockup*

[https://www.figma.com/proto/Y3tlwN2yvRc67304adzyyB/Eligibility-3.0?node-id=4222%3A27391&scaling=scale-down-width&page-id=1%3A254608&starting-point-node-id=1%3A258308|https://www.figma.com/proto/Y3tlwN2yvRc67304adzyyB/Eligibility-3.0?node-id=4222%3A27391&scaling=scale-down-width&page-id=1%3A254608&starting-point-node-id=1%3A258308|smart-link] 

*reqs*

[https://fns-prod.azureedge.net/sites/default/files/FNS-742.pdf|https://fns-prod.azureedge.net/sites/default/files/FNS-742.pdf]

We need to add a page that shows the user a facsimile of the report above.

* Under Verification, add a Reports page. Add a Collection Worksheet (FNS-742) tab for this particular report.
* Page needs an academic year selector. Default current year.
* The FNS-742 Collection Worksheet should be available on the page for the user to view and export.
** *Instructions are in the link above on the last page.*
** California has a special section that must be available if the district is in California.
*** !image-20210125-234109.png|width=746,height=96!
* User needs an Export option for this page which will generate a PDF of the report.

## Acceptance Criteria
Verification > Reports page added.
Academic year selector present and working.
Information on the page is in a shape similar to the actual FNS-742 report.
Information is generated/collected per the instructions from the PDF.
California section is also available.
User can export to PDF

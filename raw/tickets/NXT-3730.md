# NXT-3730 — Inspections Scoring - Multiple Choice Questions

## Description
*User Story*
As an Inspections user, I can create Multiple Choice questions (Dropdown, Radio, Checkbox) with custom scores, run inspections with those questions, and receive accurate score results based on the custom scores to evaluate my inspections.

To add

*Requirements*

* Each multiple choice question type can have scores added to the responses when creating a Template
** Occurs for all three multiple choice questions types: Radio, Dropdown, Checkbox
** Can only be done if Template's Scoring is set to Custom Scoring
* If a Template's Scoring is set to Custom Scoring, Multiple Choice questions have a points field next to each possible response
** This field determines how many points a response provides when selected during an Inspection
** Field can only have positive whole numbers (0 or higher)
*** Max 99 points
* If field is left empty, it defaults to 0
* Field can be edited when Template is edited
* Calculating the total possible points for the Inspection depends on the type of question
** Radio - Highest selection is applied to total possible score
** Dropdown - Highest selection is applied to total possible score
** Checkbox - Sum of all selections is applied to total possible score
* When an Inspection is run, the points associated with the selection are applied toward the Inspection's final score
** How this is applied depends on the question type
*** Radio - Selection's point value is applied to the Inspection's score
*** Dropdown - Selection's point value is applied to the Inspection's score
*** Checkbox - Sum of all selection point values is applied to the score
** If nothing is selected, no value is applied to the Inspection's score
** Score is properly updated if selection is changed
* All multiple choice scores are included in the final score when the Inspection is submitted

## Acceptance Criteria
* Score can be applied to all Multiple Choice question types (Radio, Dropdown, Checkbox) when creating a Custom Scoring Template, saved, and edited
* Score is properly applied as set in the Template when running and submitting an Inspection

# NXT-13311 — Accountability & Acc. - Transactions - Allow users to view/print Receipts

## Description
As a cashier or manager, I need the ability to print out receipts for patrons, so that they (and we) have a financial record and paper trail for  any transaction.

** Add transaction date to the receipt underneath transaction type 
** Rename Current Balance on the receipt to Previous Balance 



Changes from 12/3/2021

if paymentType is Check, show full check number

Show Bonus field with bonus amount if available. 

If no paymentType is available, show default as Cash

Remove Previous balance

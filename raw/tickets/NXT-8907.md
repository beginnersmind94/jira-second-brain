# NXT-8907 — INV - Physical Inventory - Supervisor View - Items tab - Add new columns and table type

## Description
As an Inventory District Supervisor I want additional item attributes available in the Items list table.

Add additional columns to the table which can be selected or deselected, using [Kendo grid control|https://devuicomponents.perseusedge.com/?path=/story/components-grid-kendo--with-all-options].

Columns will be searchable.

* Text columns will use ‘Contains,' ‘Does not contain’ search filter
* Numeric columns will use ‘Greater Than,’ ‘Equal To’ search filter
** Absolute value for:
*** all discrepancy columns
*** valuation columns

Existing columns:

* Item Number
** disable checkbox
* Item Name
* Usable Quantity
* Damaged Quantity

New columns:

* Perpetual Quantity
** Whole Unit
*** Perpetual WU
** Broken Unit
*** Perpetual BU
* Physical Quantity
** Whole Unit
*** Physical WU
** Broken Unit
*** Physical BU
* Discrepancy
** Whole Unit
** Broken Unit
** Value
** Percentage
* Valuation
** Physical value
** Perpetual value

## Acceptance Criteria
Grid columns will change when user selects Discrepancy view radio button

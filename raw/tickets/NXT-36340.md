# NXT-36340 — IM- Items- Autosync changes to nutrients, allergens, contributions to single item recipes and Direct Menu Items

## Description
As a user, when I make updates to nutrients, allergens or contributions, I need those changes to reflect on the *single item recipe* or *Direct menu item* associated with this item. 

Currently user needs to resave the recipe or the menu item to reflect the changes. Without these changes, MP and Production can not get the most updated data.

## Acceptance Criteria
* Sync *nutrients*, *contributions* and *allergens* automatically to the SIR and DMI associated with the item when changes are made.
* Redirect users that access a SIR from the recipe list page to the item details recipe/menu item info tab (Same as Direct Menu Item flow )

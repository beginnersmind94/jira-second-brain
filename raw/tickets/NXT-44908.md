# NXT-44908 — API – Custom – Add New - Field Mapping Tab

## Description
As a SchoolCafé admin, I would like to treat the SIS Auto Sync the same way as manual imports by adding the field mapping tab along with all its functionality so that I can have more control over the import.

*Requirements*:

* On the System > Imports & Exports > SIS Auto Sync tab > Add New and Edit, do the following:
* 
* Field Mapping
** Copy over from the manual upload
** Ensure it has all the same functionality as the manual upload
*** This includes the code mapping especially
** The Source Data fields will be the field names from the API call(s) gotten in the API tab

## Acceptance Criteria
* The tab can do everything that it does in Manual upload
* Source field names are from the pre-config data or api calls

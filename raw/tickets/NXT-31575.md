# NXT-31575 — Create Summary Accuracy Report

## Description
As a site administrator, I want to view a report that summarizes the accuracy and counts of tray detection matching so that I can use track and report on our progress.

*Application*: SchoolCafé

*Flow:*

* Browse to SchoolCafé > Accountability > Food Detection > Accuracy By Day
* View Report and optionally export

*Background:*

* This was completed in FFD website and matches to the [https://foodtrainingprod.azurewebsites.net/reports|https://foodtrainingprod.azurewebsites.net/reports|smart-link]  page
* Copy the page as it is and don’t make any changes
* Get with Saketh if you need help with the code or understanding any technical challenges
* This page depends on the main page being done first

*Requirements:*

* Location: SchoolCafé > Accountability > Food Detection > Accuracy By Day
*  
* Page is visible only if the main page is visible
*  
* Copy all the code from the FFD website page called Accuracy By Day
* Make your page look the same or as UX team says if they want changes
*  
* Ensure that the report grid has sorting, paging, filtering
* Ensure it exports the grid to excel
* Ensure filtering works (currently broken on FFD website)
* Default sort by Date, most recent first, then school
* Default paging 10 records at a time (currently wrong on FFD website)
*  
* Additionally, correct the following which is broken on the website:
** This report should only show schools with Edge devices
** You can get this from the device registration screen
** Automatically filter out all the non-Edge schools
**  
** It should also filter out all terminals not linked to an edge device so that the results show only the true results
**  
** It should also filter out all non-detection transactions
** Transactions can ignore food detection, or they can be in Learning Mode, or in Detection mode
** Count only the ones in detection mode
** To know this, the transaction record has fields that are completed when in detection mode and ignored otherwise, Saketh can show you which ones

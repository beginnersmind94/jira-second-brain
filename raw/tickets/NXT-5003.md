# NXT-5003 — Data Exchange - File Import Monitoring Page

## Description
*Summary*

As a PrimeroEdge user, Once I preview import data and select Import option, I should be able to see the progress along with status of the File Import in a separate page or on the confirmation screen.

As a PrimeroEdge user, I can minimize the Progress Screen and do other work in PrimeroEdge while the import occurs in the background, with a notification when it’s complete

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=319483%3A1&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=319483%3A1&scaling=scale-down-width|smart-link] 

[https://primeroedgenext.atlassian.net/browse/NXT-8892|https://primeroedgenext.atlassian.net/browse/NXT-8892|smart-link] 

*Requirements*

* A progress bar is present to show the import’s progress/status along with 
** File Number/Reference #, Summary, Import Type, start time progress, end time and status
** An estimated time remaining for import completion
** Expander with number of lines imported and total lines are displayed
* The progress bar can be minimized (if displayed on the confirmation screen)
** A toast message appears when the import is complete
*** Clicking the toast message brings the progress bar back up
** User can continue working in PE on any screen while minimized
* Once complete, user is present with a continue button
* There is a button for canceling the import
** Canceling takes user back to the Field Mapping screen
* User should be able to go to monitoring page under System>Config and sees list of import jobs
* Columns to be displayed for File Number/Reference #, Summary, Import Type, start time progress, end time, state and status
* User should be able to search filters & sort on any the above columns & last 24 hrs, 7 days and last month
* User should be able to refresh the monitoring page
* Monitoring page should include pagination
* User should be able to expand any line item for File Imports to show detailed descriptions and progress of the Import job
** # of rows imported successful
** # of row failed to import
** # rows remaining
* Status:
** Done
** Scheduled
** Running
* Filters:
** TIME: 24 hours, Last 7 days, Last 30 days
** Status: DONE, SCHEDULED, RUNNING
** ERROR: True/FALSE
** USER: search by user email ID/login ID

ADF API:

[https://docs.microsoft.com/en-us/rest/api/datafactory/pipelineruns/cancel|https://docs.microsoft.com/en-us/rest/api/datafactory/pipelineruns/cancel]

## Acceptance Criteria
* Verify File Import job is added to monitoring or confirmation page with required columns to display
* Verify status and number of rows imported are accurate
* Verify user can work on different screen and return to this page/confirmation whenever he likes
* Verify notification includes link to the file import job #
* Verify user can select any file import job & perform required action
* The format for the "Duration" column shall be "H:mm:ss". (That is 24 hour with leading zero suppressed: minutes:seconds)

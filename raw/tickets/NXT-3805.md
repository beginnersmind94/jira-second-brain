# NXT-3805 — Account Management - Advanced Search - Site Section - Area subsection

## Description
When a district has defined areas those areas will be listed here. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Site Area values

!Screen Shot 2020-04-27 at 3.09.50 PM.png|width=305,height=202!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

When a filter value is selected the Site Area Subsection count is increased by 1 and the Site Section count is increased by 1.

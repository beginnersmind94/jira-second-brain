# NXT-50380 — INV - E-Orders - Test different EDI settings for EDI 850

## Description
As an Inventory District Administrator I want the EDI settings to update the way files are generated so I can submit EDI 850 files to the vendors in whichever configurations they require.



!image-20240509-140150.png|width=657,height=630,alt="image-20240509-140150.png"!

## Acceptance Criteria
* Test different scenarios for EDI 850 E-Order configuration settings
* Document different EDI 850 options and translate into Inventory terms
** Example: N9 and MSG segments correspond to Order Comments
* Ensure segment delimiters are changed/populated according to config
** delimiter is supposed to be a dropdown, not freetext
* Change radio options on configuration and confirm file is updated accordingly
** we require knowledge documentation about these radio options and how they affect the files
* Test ‘One Order per File’ option
** Determine how to present these in the UI
* Test file creation with very large quantity of orders
** 100+ orders
** consider using script/automation to create new orders in bulk 
*** very helpful in the long term

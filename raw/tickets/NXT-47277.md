# NXT-47277 — Acc - Claims - Oregon SNP Export (with State Eligibility)

## Description
As Opal the Oregonian Director I want to export my reimbursement claim in a specific format so I can upload it into my state website. This will save us time and frustration with having to copy numbers out.

*requirements*

* Add a default Oregon claim for the SNP program
** CSV export, 44 fields points.
** Has a header row and then there’s 1 line per site.
** File name format will be the program name (SNP) followed by the month and year followed with a CSV extension.  
** There’s several fields in there to pull out Oregon State Eligibility counts - we need to do that as part of this, the claim export from PE is incorrect in how this was handled.



Claim format file from the OR DOE website has been attached, along with the PrimeroEdge script to generate the claim. Recommend we start with the script and update to SC then adjust if required.

## Acceptance Criteria
* SNP claim for OR added
* Users can add it via the ‘State Default’ option while configuring a SNP claim export

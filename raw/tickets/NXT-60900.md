# NXT-60900 — INV - Distribution - Assign & Pick - Reset/rollback Picking

## Description
As a Distribution Manager I want the ability to restart picking if I have not shipped any orders, so I can make item adjustments and then re-ship.

!Distribution - Picking Completed-20250423-222102.png|width=91.66666666666666%,alt="Distribution - Picking Completed-20250423-222102.png"!

## Acceptance Criteria
* Provide a ‘Restart Picking’ button if no orders have been shipped
** this would appear in Picking Complete or Shipping in Progress
** hide/remove the button when any trucks are Shipped
* Prompt user: “This will revert the distribution and restart Picking. Are you sure?”
* Revert the distribution if user confirms prompt
* Reverse the Withdrawal transaction and move the batches back to the reserved quantity

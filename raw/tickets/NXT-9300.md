# NXT-9300 — Data Exchange - SIS Import - View history of student additions/modifications via the import process

## Description
As an Application Admin i would like to View student history so that i can see any additions or modifications made via the SiS process

## Acceptance Criteria

And applied the required filters 
And the records are available 
When View Option is selected in any record 
Then show Import history which includes the below
Logged Messages
log Date 
Type of import 
And Start date and End Date

# NXT-8363 — System - Configuration - Special Assistance Programs - P2 Student Analysis

## Description
Configure special assistance for the Provision 2. Allows for determination of sites that can offer meals at no cost to students. Claims are calculated by percentage for Free or Paid meals.

The user will be stepped through the process and can either Import the data (ver 2 dependent on platform), or enter by hand.

Reference Provision 2 Guidance for details of implementation

*Requirements:*

* User can select P2 and set configuration options
* User can enter claimed meal counts for a year or by month 
* User enters claimed meal counts for each meal type being claimed
* Claiming percentages are calculated from the claimed meal counts entered.
* Claiming percentages can be enabled for a site

## Acceptance Criteria
Provision 2 can be selected
Delayed Implementation can be selected.
Base Year can be set
Configuration can be set to be Annual or Monthly
Configuration can be set by District or Site
Each meal type can be enabled Breakfast must be implemented for SFSP/SSO breakfast to be enabled, Lunch must be enabled for SFSP/SSO lunch to be enabled.
The base year for the claim data is set.
Claimed Meal Counts for free/reduced/paid can be entered. (based on settings)

see page 54-55 of the attached Provision2 Guidance document.
Claimed percentages are calculated for Free/Reduced/Paid 
Total Meals Claimed cnt = Free Claimed Cnt + Reduced Claim Cnt + Paid Claim Cnt
Free % = Free Claimed cnt/Total Meals Claimed cnt
Reduced % = Reduced Claimed cnt/Total Meals Claimed cnt
Paid % = Paid Claimed cnt/Total Meals Claimed cnt
Percents can be enabled or not.
Counts and % are saved.

Note: If delayed implementation is approved by the State agency, the first claiming period of the school year is not considered to be a part of the base year. While a standard categorical count is taken at the point of service (or approved alternate) for each claiming period of the school year, a monthly claiming percentage is not calculated for the delayed implementation period. For the claiming period of each non-base year corresponding to the delayed implementation period of the base year, an annual claiming percentage must be calculated, derived from all months of the base year except the first claiming period of the school year. Monthly percentages may be used for the remaining months.

# NXT-61060 — AI Onboarding - Digital Recipe - Submit Recipe

## Description
As a customer support person, I would like to use AI to digitize a recipe document so that I can speed up onboarding

*Notes:*

* This story covers what happens when a user edits a recipe and clicks submit
* It is a continuation of the Edit Recipe story, but broken out because of complexity

*User Flow:*

* User browses to Item Management and then Recipes
* User clicks on the AI button and it opens the AI Recipe drawer
* User ensures the New chip is selected
* User selects format and a file to upload and clicks upload
* AI analyses the file and returns the results
* System saves the results, loads the File Analysis section
* System automatically selects the first recipe and loads it into the edit recipe section
* User makes any changes they want and hits submit
* The changes are committed to the database as a new Recipe
* Users can see the new recipe in their recipe list

*Requirements:*

* On the Report AI screen, do the following:
* 
* *Action buttons at the bottom:*
* Select what to import checkboxes
** We want to give the user the option to save only Basic and Servings, or optionally Ingredients and Steps
** They must save the info from Basic and Servings no matter what
** If the checkboxes for the other 2 are not selected, don’t attempt to add them to DB
* Submit
** Ask to confirm
** If no, do nothing
** If yes:
*** Validate all required fields
*** Add the recipe to the Recipe database
*** If they want to save steps and ingredients, add those to the Recipe database in the correct steps
*** If they want to save allergens, add those to the allergens field in the Recipe database
*** Mark the status as Complete and ensure the File Analysis section is updated accordingly
*** Ensure a recipe with Complete status is not editable
**

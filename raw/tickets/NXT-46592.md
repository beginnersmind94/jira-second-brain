# NXT-46592 — Imports - Files & Exports & API Tabs - Show Additional Columns

## Description
As a SchoolCafé user I want to be able to see additional columns on the Files tab and Exports tab so that I can see additional information not shown in the default view

*Notes:*

* We have this functionality already in other places, so please just copy it from there, don’t rewrite it from scratch

*Requirements:*

* This functionality applies to Files tab and Exports tab
* Add a columns drop down to the grid
* Show the additional fields not shown in the grid as options
* Be able to select one or more and have them show in the grid
* Be able to remove them from the grid
* Do not allow the user to hide the Status, Import, Export or Actions columns

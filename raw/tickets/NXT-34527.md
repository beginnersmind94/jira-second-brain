# NXT-34527 — Photo Import Summary Details UI

## Description
Need to show the summary details, the way we are showing for other imports.
Details should include:
Number of images uploaded.
Number of images processed successfully.
Error-prone files.

## Acceptance Criteria
Fields shall include:

* Number of images uploaded.  Label should be: “*Number of photos selected for import*”
* Number of images processed successfully.  Label should be “*Number of photos imported successfully*”
* Error-prone files. Label should be “*Number of photos not imported due to errors*”

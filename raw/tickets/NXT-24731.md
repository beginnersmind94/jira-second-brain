# NXT-24731 — POS - Dynamic Menus - Select a service line

## Description
As the cashier I want to be able to select the serving line for the terminal I’m operating.

* User can select what service line they're serving based on data available from production plans for that site, for that day and Meal Type
The Menu Grids are listed by Meal Type
A Serving line can be saved as the default and A Static Menu can be saved as the default for the Meal Type
A single default menu can be set for a meal type
If the set Default Menu is not available when the meal service is launched a message is displayed “Default menu not available, please select a menu grid”

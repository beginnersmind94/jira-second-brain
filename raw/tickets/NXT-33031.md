# NXT-33031 — Import - Recipe - Platform Portion Only (3 of 3)

## Description
{color:#ff5630}This story is intended to cover the Platform team’s portion of the import.  The module team will have to complete their piece for the import to work end-to-end.{color}

There are 3 steps involved in an import:

# Platform uploads the file to a staging repository
# The module team exposes an API to be notified that step 1 is complete
# The Platform notifies the module mechanism via the API identified in step 2.

This story is for step 3.

{color:#ff5630}UX is not needed.{color}

## Acceptance Criteria
* The Platform import mechanism shall notify the module that the upload to the import staging repository has been completed
** Once notified, the module mechanism will complete their portion of the import and apply their business rules

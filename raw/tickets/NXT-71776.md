# NXT-71776 — SchoolCafé - Forms Framework - Connect Application Result to Applications

## Description
*Summary*

* In the forms framework, configure how an app gets 'Qualified' or not, and add a note to Form Configuration explaining this logic to the user.
* In the Other Forms page, update the Form Applications' _Results_ column data based on these rules. Display this data wherever the result is displaying within SchoolCafé.
** Family Portal will be addressed in a separate story.

*Details*

* In the background, create rules to determine how a form application is given a determined ‘Result’ (which is currently based on Income):
** When *Income Collection Method = Individual* or *Household*, if the total household income us set at or below the Free/Reduced threshold then the result = ‘{color:#36b37e}*Qualified*{color}', otherwise the result will be '{color:#ff5630}*Not Qualified*{color}’. 
*** We can copy this logic from our classic Summer EBT:
{noformat}For Each Student
    If Student Eligibility == Free or Reduced
        Then Student S-EBT Eligibility Status = Qualified
                Student S-EBT Reason = Student Eligibility Reason
        Else Student S-EBT Eligibility = Not Qualified                 
                Student S-EBT Reason = Student Eligibility Reason{noformat}
** When *Income Collection Method = None*, set the result of any app to '{color:#97a0af}*N/A*{color}'

* In _Eligibility > Form Configuration,_ when adding or updating a form, insert a note beneath the question related to Income collection method to explain this logic to the user. Here is how it looks in Classic/PE:
!image-20260407-201652.png|width=1059,alt="image-20260407-201652.png"!

* When applications are processed, update the determination for Qualified vs. Not Qualified (or N/A for 'None' income) based on this question.
* In _Other Forms_, display the grid results correctly when filtering based on Form Application Result.
* We will handle this in the family portal in a separate story: [https://primeroedgenext.atlassian.net/browse/NXT-71764|https://primeroedgenext.atlassian.net/browse/NXT-71764|smart-link]

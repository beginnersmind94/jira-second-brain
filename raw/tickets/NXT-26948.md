# NXT-26948 — INV - Reports - Provide Withdrawals Report ( UI )

## Description
As a Food Service Director I want to be able to access my Withdrawals List Report from the Reports page like any other report.

As part of additional Withdrawal Report stories, we will have 4 reports which are not accessible through the main Reports page. We should take advantage of the existing filters and logic to simply reproduce this page on the Reports landing page.

*Requirements*

* Add Withdrawals List Report to the Reports page

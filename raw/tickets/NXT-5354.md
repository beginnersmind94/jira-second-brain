# NXT-5354 — Workspace - My Account - Add My Profile page

## Description
*User Story*
As a PrimeroEdge User, I can manage (including editing) my picture, name, and basic details from the Workspace landing page under the Profile header.

*Mockup*
[https://react-material-kit.devias.io/app/account|https://react-material-kit.devias.io/app/account]

[https://demos.telerik.com/kendo-ui/upload/index|https://demos.telerik.com/kendo-ui/upload/index]

*Requirements*

* Button on Workspace or navigation pane to go to separate Profile page.
* Edit button allows user to update profile items
** Option to upload a profile picture
** Name is displayed (not editable)
** Job title displayed (not editable)
** Email address (not editable)
** Phone number(s) (editable)
** Option to change password
** Switch between light and dark mode
* When Edit is selected, the button changes to show Save and Cancel buttons
* When a sidebar item is clicked, it should display only the one selected

## Acceptance Criteria
- Profile section is present in the workspace
- Functionality present as described in requirements

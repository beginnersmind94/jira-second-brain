# NXT-63099 — SC - Learning - AI Bot 2Q25

## Description
As a Workspace user, I want to be able to use an AI chat bot on a page to query documentation so that I can get help information in an AI chatbot format

*Notes:*

* This story is not going to go in great detail for all the requirements. It will cover the latest requirements since the last development
* Since Workspace only has PE login at the moment, phase 1 requirements are for PE data only
* The Learning AI chatbot is called Schoolie

*Phase 1 Requirements:*

* Backend:
* AI bot can read text data from any standard document type
** Images will be handled in a later release
* Source Data includes documents for:
** Help Manuals
** Training Materials
** FAQs
** Product team will give Dev team links to folders that contain these documents
* Documents should be able to be tagged with metadata:
** Platform
*** PrimeroEdge
*** SchoolCafé
*** Workspace
*** Family Hub
*** Needed for filters
** Source
*** Actual document name
*** Link to document
** Type
*** Manual
*** Training Material
*** FAQ
*** etc.
*** Needed for filters
** Module
*** Module name related to the material (eg. POS, Accountability, etc.)
*** Needed for filters
* Rate limiting
** limit how many questions a user can ask in a period of time
** I propose 10 in 24 hours
* 
* Front End:
* Update the AI Assistant screen to work with the Learning AI APIs
* Verify the user’s district has AI system setting and  Learning AI system setting set to true for both
** If not, hide the the AI Assistant screen



*-Phase 2 Requirements:-*

* -Include SchoolCafé materials-
** -And potentially Family Hub, etc.-
* -Enforce platform filtering of data-
** -If user has PE, only PE documents-
** -If user has SC, only SC documents-
** -etc.-
* -Add filters to the screen to allow the user to narrow their search:-
** -Platform-
** -Type-
** -Module-
* -In results, show source of information with link to document-
* -User able to provide feedback with voting (up/down)-
* -Save chat history-
* -Clear button-
* -“What can I ask?” guide-
*

## Acceptance Criteria
* AI Assistant screen can perform queries against the learning training model
* Users can only use if AI and Learning AI are enabled for their district

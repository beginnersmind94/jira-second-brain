# NXT-45550 — IM/INV - Deactivate Inventory Item - Check Orders

## Description
As a district admin I want the ability to make an Item inactive if it is not being used anywhere in the system, so I can delete any unused or accidentally created/imported items.

h3. Pending Deactivation message

!image-20231207-144411.png|width=91%!

h3.

## Acceptance Criteria
* After disabling on the contracts, check item for presence on Open or Pending Review orders
** If the item is present on Open/Pending Review orders, move the status to Pending Deactivation
** if the item is not on any orders, move directly to Inactive status
* Once all orders containing the item are Closed, the item can be made Inactive
** Item cannot be placed onto new orders
** Incomplete orders containing the item cannot be submitted (until the item is removed)
** Quick Receipts cannot contain the item
** Item can be received from existing orders without interruption 
* Update all Order UIs to indicate Inactive or Pending Inactive items with an alert icon and a tooltip

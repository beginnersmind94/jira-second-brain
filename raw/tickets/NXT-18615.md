# NXT-18615 — INV - Manage Orders - Receipt Detail - History tab

## Description
As an Inventory user I want a History tab on the Receipt Details so I can view past version transactions of a specific Receipt.

*Requirements*

* Add a History tab on the Receipt Detail
* History tab will display a ‘Transaction History’ table
** this table will list an entry for each time the receipt was updated/changed
*** this is simply each time the receipt was saved
* This history table will have the following columns
** Receipt Transaction #
*** this needs to be some sort of unique number to indicate the specific transaction
** Transaction Details 
*** this will explain the details of the change for that transaction
** Modified Date
*** this is the date and time that the receipt was updated
** Comments
*** this will display any Receipt Comments that were saved at the time that the order was changed
** Modified By
*** the user that saved the updated order
** Action column
*** this will contain a View icon that will load the historical Receipt Detail page for the selected transaction ([NXT-18128|https://primeroedgenext.atlassian.net/browse/NXT-18128])
* Historical transactions will exist for the following actions (history tracking is done in [NXT-18732|https://primeroedgenext.atlassian.net/browse/NXT-18732])
** Receipt created - original transaction
** Receipt adjustment
** Receipt cancellation

# NXT-3247 — Employee Details - Add/Upload Documents

## Description
Ability to upload documents to an employee record. 

* Classic version allowed extensions: .txt, .csv, .doc, .docx, .pdf, .jpg, .png, .bmp,.msg,.xls,.xlsx
* Add document name on upload - ‘File Name’ field
* preview of document (either popup or small window/thumbnail) 
* Edit document name – ‘File Name’ field 
* remove document 
* time stamp, user, document name of upload – in history section
* Be consistent with System module, use the upload tool and allowed extensions.

## Acceptance Criteria
documents can be uploaded to the profile 
time stamped with user who uploaded -- in history section
ability to remove upload and edit document name (File Name)

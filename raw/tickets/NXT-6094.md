# NXT-6094 — Application Entry - Guided Process (8. Summary)

## Description
*User Story* 

After submitting my application, an online version of the app is displayed for me to print or download. 

*Mock Ups* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1306%3A27072&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1306%3A27072&scaling=min-zoom]

*Requirements* 

* After submission, application info is populated in an online version template with message
** _Application \[number] successfully submitted with the following information:_
*** Application number matches number on app
**** unique application number is generated
**** each district has unique app numbers
*** Status > *Processed (*or *Pending P2)*
*** Eligibility Determination
**** ‘Eligibility’ designation
***** Free, Reduced, Denied 
**** ‘Basis’ of determination
***** Income
***** Categorical
****** Assistance Programs 
******* (Configuration for local name) - SNAP, TANF, FDPIR
****** Other Source 
******* Homeless
******* Migrant
******* Head Start
******* Runaway 
******* Foster
***** Refused Benefits
**** ‘Start Date’ of eligibility 
* Image of application with populated text from application - 7041 (based on district/state configuration)
* option to start processing another app or View Applications (takes them to the applications list page)
** ADD ANOTHER APPLICATION 
** I’M DONE FOR NOW

## Acceptance Criteria
- ability to start new app for student(s)
- unique app number generated at submission
- Multi-language, ADA compliant.

# NXT-8177 — INV - Physical Inventory - Supervisor View - Search/Filter/Views Section 

## Description
As an inventory supervisor I want a section for search and filter tools so I can refine the list of items and perform evaluations.

Provide a section at the top of the Items tab that allows for filtering of the item list.

+*Organization*+ - radio button (Selection A in mockup)

Provide an option for the user to switch the item list organization from Storage & Item Category to Valuation Groups. [https://primeroedgenext.atlassian.net/browse/NXT-8178|https://primeroedgenext.atlassian.net/browse/NXT-8178|smart-link]

+*Views*+ - radio button (Selection B in mockup)

* Default View - Usable Counts: 
* Discrepancy View - Perpetual, Discrepancies [https://primeroedgenext.atlassian.net/browse/NXT-7548|https://primeroedgenext.atlassian.net/browse/NXT-7548|smart-link]
* Valuation View - Perpetual, Valuations [https://primeroedgenext.atlassian.net/browse/NXT-8618|https://primeroedgenext.atlassian.net/browse/NXT-8618|smart-link]

+*Filter*+ - dropdowns (Selection C in mockup)

* When the list is in ‘Storage & Item Category’ organization
** Storage Category - dropdown
*** if user navigates to the Items tab by selecting a Storage Category on the Summary tab, that category will be selected in the dropdown
*** Only display Storage Categories that are on the inventory
** Item Category - dropdown
*** Only display Item Categories that are on the inventory
* When the list is in ‘Valuation Group' organization
** Valuation Groups - dropdown
* Reset button
* Apply button

The Item list will be filtered by the selections made in the dropdowns.

## Acceptance Criteria
New header section
Organization option (radio button) for Item + Storage Category or Valuation Groups
View option (radio button) for Default, Discrepancy, Valuation item list views
Filter dropdowns depending on Organization selection
Dropdowns will only contain values that exist on the current Physical Inventory

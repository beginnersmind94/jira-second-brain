# NXT-53696 — Sys/POS - Automatic Logout Setting - Rename, default value, usage

## Description
|Consider Terminal Offline After X Minutes|TEROFFA|System Setting|

*requirements*

* Updates to the system setting:
** Rename: Automatically Logout from POS in X Minutes if Idle
** Description: This setting controls how long the POS can remain idle before the user is logged out. Minimum timeout at POS is 5 minutes, with a maximum of 120 minutes.
** Change from Internal to Common
** Default value: 60 minutes
* POS should be using this to log the user out after the setting value in minutes
* Also move the DYNMENU setting to Internal

|Dynamic Menus and Itemized Meal Service|DYNMENU|

## Acceptance Criteria
* System Setting updated
** Name, description, Internal → Common, default value
* Verified working at the POS for initiating the automatic logout
* DYNMENU moved to Internal

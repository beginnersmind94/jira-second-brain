# NXT-4025 — AcctMgt - Add Patrons - Photo

## Description
*User Story*

As a district admin I want to upload a photo while adding a patron. This will save time.

*Mockup (May change slightly due to technical limitations)*

!Capture 2021-02-12 at 4.13.46 PM.png|width=703,height=428!

*Requirements*

* User needs a way to upload a photo when adding an account (student or adult) manually.
* Flat circle with initials if no photo is uploaded.
** Upload Photo option.
** Show user info on acceptable file formats. 
*** Supported file types:  .jpg .jpeg, .png, .gif.
** Provide feedback if the user selects the wrong type of file or if the file is too large (larger than 5 MB).
* Photo available for viewing once added.
* User can remove the photo or replace it with another image.
* Display a note for this scenario: when a photo is added, but the user navigates forward and then previous (landing on this step again), the photo no longer displays: show the note below-
** “Photos that are uploaded but do not display here are saved in the background. After adding this patron, their photo can be edited or removed on the patron's details screen.”

## Acceptance Criteria
Add Photo option added to the add patron wizards.
Photo displays in real-time once uploaded.
User can add a photo.
User can remove photo.
Multi-language support added.

# NXT-3818 — Account Management - Advanced Search - Enrollment Status section

## Description
The Enrollment Status section is used to select if a patron is active, inactive or merged. Merged are treated as Inactive.

!Screen Shot 2020-04-28 at 2.54.04 PM.png|width=283,height=135!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Enrollment Status section they are 'or'ed together.

When a filter value is selected the Enrollment Status count is increased by 1

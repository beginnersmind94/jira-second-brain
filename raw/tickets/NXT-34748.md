# NXT-34748 — Roles and Permissions -  Protect SchoolCafe-only pages from Roles that are not SchoolCafe Admins

## Description
I, with non-SchoolCafe roles, am able to set “View” permission (and other permissions) for any role to a SchoolCafe-only UI page (Districts, Resource Catalog).

[https://www.loom.com/share/9da2fbf9bd0a4e619e077e126c9781e3|https://www.loom.com/share/9da2fbf9bd0a4e619e077e126c9781e3]

## Acceptance Criteria
* SchoolCafe Admin pages shall not be accessible by roles that are not SchoolCafe Admins
* SchoolCafe Admin pages shall not be viewable on the Roles and Permissions page by roles that are not SchoolCafe Admins
* SchoolCafe Admin-Only pages are:
** Districts
** Resource Catalog

# NXT-57294 — USDA Branded Foods - Insert the ingredient with with modified fields 

## Description
_(none)_

## Acceptance Criteria
MP - GC - Item Catalog:

USDA Branded Foods Source:

When the details getting populated from other data source, the user should have option for the following:

Ability to insert the Ingredient with modified fields as follows: Ingredient Name
Ingredient Short Name, Ingredient Form, Ingredient Statement/Sub Ingredient, Standard Recipe Directions, Serving Size, Allergens 

Ground Rule - even if other fields are modified, the ingredient name has to be unique 

Ability to ignore the images (Deselection of Check Box), introduce the check box (“Exclude Image(s)” and select it by default

Ability to delete the allergens populated

Upon deselection of images or deletion of allergens, item should be imported without factoring them

Scenario is applicable for one item import only

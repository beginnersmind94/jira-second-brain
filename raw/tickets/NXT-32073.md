# NXT-32073 — INV - Vendors - Ordering Conditions - Update for Pack Size Config

## Description
As a District Admin I want the Vendor Item Ordering Conditions configuration to reflect the pack size units of my items so I can configure sites to order the correct item unit.

*Requirements*

* Update the Vendor’s Ordering Conditions tab to consider item pack size configuration
* Display each unit on its own row, with a checkbox available for each unit
** Show item units on individual rows but indicate items which are linked together
*** Replace Unit Type with Unit Description
* Display Item # only once for the highest unit
* Show conversion (sub) quantity 
* Add Vendor Item # (missing in mockup)
* +Business Rules+
** Ordering Conditions are still defined for each Unit 
* +UI Improvements+
** Resize slideout width
** Move notice to the footer
** Expand available view space for Site list
** Adjust expand/collapse button (proximity to checkbox)
** any other improvements

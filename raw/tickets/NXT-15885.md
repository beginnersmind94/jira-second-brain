# NXT-15885 — INV - Create Orders - Edit Items on Order 

## Description
As an inventory user creating an order I need an edit screen so I can modify the orders that have been generated through the Create Orders process.

Edit icon will be provided for each individual item. 

*Requirements*

+Edit Item logic:+

if the user clicks the Edit icon for a specific item within an order, the following will happen:

* the quantity fields (both WU and BU) will become editable fields
* a delete icon will be presented next to the item
* the Edit icon will change into a Save icon

+Tooltips (for Item level):+

* Item Validations:
** Show the minimum/maximum order quantity (per contract), so user can know what the order quantity should be
** Example: “Item exceeds maximum order quantity of 5 cases.”

## Acceptance Criteria
When user clicks Edit:

* turn the quantity labels into editable numeric fields

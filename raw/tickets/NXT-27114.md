# NXT-27114 — INV - Production Ordering - Confirm Orders - Back to Production screen

## Description
As an Ordering Manager I want a Back button on the Confirm Orders screen when I am placing production orders so I can return to the main Production Ordering screen.

*Requirements*

* Enable the Back button on Confirm Orders screen ([NXT-2062|https://primeroedgenext.atlassian.net/browse/NXT-2062])
* Navigate back to the Production Ordering screen
** retain prior selections
** disable the editing of prior initial selections (Site, Date Range)
** retain items in carts
*** show non-production items at the end with ‘Non Production’ label in Production Item column(s)
**** show an icon next to the item (not the alert icon; something else)
**** show a tooltip on the icon: Non-Production Item
** show non-cart items on page as expected
* If all production orders are deleted, return to the Production Ordering page

# NXT-20050 — INV - Withdrawals - Create Withdrawal - Confirm (Step 3)

## Description
As a Site Manager I want a Confirmation step so I can view and confirm the details of my Inventory Withdrawal.

*Requirements*

* This is step 3 of the Withdrawal process, after user has navigated from the Items step.
** display the ‘Previous' stepper button
* Display a header containing the main Addition info
** Site (label)
** Withdrawal Type
** Transaction Date and Time
** Reason 
** Comments 
** *Everything except ‘Site’ is still editable and will contain the value specified on Step 1
* Display a read-only version of the Item List ([NXT-20796|https://primeroedgenext.atlassian.net/browse/NXT-20796])
** Item #
** Item Description
** Quantity on Hand
** Withdrawal Quantity
** Price
*** average price of the selected batches
** Comments
*** still editable via the popup
* Provide a Confirm button to save the transaction ([NX-20797|https://primeroedgenext.atlassian.net/browse/NXT-20797])

*Mockup Note:* Withdrawal Type is not editable, Reason is a dropdown, no Delete icon needed on this step

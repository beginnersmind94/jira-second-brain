# NXT-30123 — IM - Items - View Item - Product Info card

## Description
As a District Admin I want a second row to the header section so I can view Product Info, Unit Info and any uploaded images

*Requirements*

* Display a card for the Product Info 
** Attributes
*** Item Category
*** Storage Category
*** Valuation Group
** Provide an Edit button
** If the Product Info section was skipped, display some sort of indicator that the section is incomplete
*** *Note:* exact component/style TBD 
* Display the grid for Unit Info
** Show only view-only columns:
*** Unit Description
*** Quantity
*** Fair Market Value
*** Weight
** Will likely need to use the [TreeList Grid component|https://devuicomponents.perseusedge.com/?path=/docs/components-treelist--default]
** Show only the selected rows from create/edit mode
** Do not provide an Edit button (this will be done on Product Details tab)

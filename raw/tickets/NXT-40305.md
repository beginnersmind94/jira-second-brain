# NXT-40305 — eOrders Configuration – Dashboard

## Description
As a SchoolCafé user, I would like to be able to see and action `eOrder Configurations so that I can add, edit, run or view the configurations.

*Notes:*

* This page does not exist in classic
* We are following the same design as we do for Data Query and Monitors

*Flow:*

* Open SchoolCafé and browse to Settings > Configuration > eOrders
* View list of eOrders
* Optionally add a new one
* Optionally edit an existing one
* Optionally delete an existing one

*Requirements*:

*Menu Changes:*

* Add a page to under System > Configuration
** Title = eOrders
** On click, opens the eOrders Configuration page

 

*eOrders Configuration Dashboard Page:*

* Create a new page
** Title = eOrders Configuration Dashboard
* 
* Add a button to the top right
** Title = + New
** Onclick, open the Add New drawer
* 
* Create a grid:
* Columns:
* Vendor
** Vendor field from Add New
* Import Type
** Import Type field from Add New
* Last Updated On
** Date and time it was last updated
* Last Updated By
** Name of user that last updated it
* Status
** Enabled / Disabled
* Actions
** Enable / Disable
*** If click Enable, the configuration is usable by sites
*** If click Disable, the configuration is not usable by sites
*** To do this, update the Status field accordingly
** Edit
*** Requirements covered in the Edit eOrders story
** Delete
*** Requirements covered in the Delete eOrders story
** History
*** Requirements covered in the Delete History story
* 
* Grid has paging
* Grid has sorting
* Grid does not need filters as there are not that many vendors
* Grid has export to Excel or CSV
* 
* Security
** Add page to resources
** Security is managed via the resource in Roles & Permissions (R&P)
** R&P Edit permission applied to the Edit button
** R&P Delete permission applied to the Delete button
** R&P Add Permissions applied to the + New button
** R&P View Permissions applied to the page in general

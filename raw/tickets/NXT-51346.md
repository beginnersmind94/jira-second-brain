# NXT-51346 — IM- Contribution Info- Allow only 1 Milk Variety Per Item

## Description
As a Nutritionist, When I select a Fluid Milk selection, I should only be allowed to select 1 type of variety of milk type. There will never be an instance where a milk will have multiple varieties.

## Acceptance Criteria
* When fluid of Milk quantities are selected allow only 1 variety of milk to be selected
**  Provide a drop down for the different varieties of Milk as a sub selection of the Fluid Milk component category
** Logic still applies where varieties can only be selected once a component number has been given to the fluid milk
** Allow user to unselect variety of milk in the future if they so choose
** Only display the variety of milk code once a particular selection is made 
!image-20240604-193036.png|width=1176,height=822,alt="image-20240604-193036.png"!
* Correct data in lower environments to only have 1 variety of milk selected
* Query production data to ensure there are no items that have multiple variety of milk 
* Resave menus and production plan/record to ensure all analysis is updated based on new requirements
** Maintain this code so that it can be reused in the future for other enhancements are done that require resaving menus/ plans/ records
* Ensure changes are made to all contribution screens if there happens to be multiple (Assumption is there is only 1)

# NXT-4114 — AcctMgt  - Rosters - Add

## Description
*User Story*

As a district administrator I want to add a roster.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=rosters&id=yvgr6n&c=1|https://nqyc9v.axshare.com/start.html#p=rosters&id=yvgr6n&c=1]

*Requirements*

* Add an ‘Add +’ button to the Rosters split view.
* Clicking this will load a sidebar where the user can set the Roster Name, Description and Site(s).
** Blank fields for Roster Name and Roster Description.
*** Roster Description is optional, 250 character limit.
** Site(s) is a drop-down of active sites at the district.
*** Multiple sites can be selected.
** Roster Name must be validated - each roster name is unique per district + site level.
** Save / Cancel buttons.
*** Success message if saved successfully.
*** Search results, if any, should be removed so that the newly created roster is visible in the search results.
*** The new roster will be the top result.
*** The new roster  will have a little chip/tag at the top which says ‘NEW’ per the wireframe.
**** This chip/tag should not display on any other roster, only the freshly created one.
*** Right-side of the split view should load with the freshly created roster.

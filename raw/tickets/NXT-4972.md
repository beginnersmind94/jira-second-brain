# NXT-4972 — AggtMgt - Add Patrons - Adult - Wizard Step 3 - Restrictions

## Description
*User Story*

As a district admin I want to fill out general information for an adult patron.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default]

*Requirements*

* Populate Step 3 of the wizard added via NXT-4969.
* Restrictions are identical to how they’re handled for a student. Same options as the edit drawer right now.
* 'SAVE & CLOSE' button for this step, which will save any information but close out of the wizard.
** User should receive a success confirmation toast about the completed task.
* ‘NEXT’ button available which will save, and take the user to Step 4.
* ‘CANCEL’ button should provide a confirmation prompt for the user.
** ‘Are you sure you want to cancel? Changes from this step will be discarded.’
* ‘BACK’ button is available, acts as a Save and takes the user back to Step 2.
* -Stepper (top right component) can be clicked to allow navigation to a future step.-
** -This would be the same as the user clicking on BACK/NEXT. Validations + Save must happen.-
* Add Steps 4 and 5 as placeholders.
** Step 4 - Allergens
** Step 5 - Documents & Comments

## Acceptance Criteria
Restriction information can be entered.
Information is retained while navigating the wizard.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)

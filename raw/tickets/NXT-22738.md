# NXT-22738 — INV - Additions - Create Permission Resources

## Description
As a FSD I want resources for Additions so I can assign users with the appropriate permissions.

*Requirements*

Map permissions for the Additions functionality

+Additions list page+

* *View* - access to the list page/view in navigation
** Report is included; no report permissions required (yet)
* *Add* - visibility/access to New button

+Addition Reasons slide-out+

* *View*  - enables the Reasons button on the Additions list page
* *Add* - enables the Add button on the Addition Reasons slide-out
** also needs to show/hide the Reason Description field
* *Edit* - enables the Edit icon in the grid on the Addition Reasons slide-out
* *Delete* (Cancel) - enables the Delete icon in the grid on the Addition Reasons slide-out

+Addition Details page+

* *View* - visibility of View icon on list page, access to Details page
* *Edit* - enables the Edit button on the Addition Details
* *Delete* (Cancel) - enables the Cancel button on the Addition Details

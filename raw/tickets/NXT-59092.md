# NXT-59092 — LMA - Sites - Inventory Distribution

## Description
As a Customer Support person, when I want to apply a distribution license to my sites so that we can sell this technology to customers.

h2. Context

This story aims to add a check on the inventory distribution licenses in School Cafe by not allowing the user to use more distribution licenses than they have purchased. Once completed, the BO team will utilize this license type in its code as part of the distribution project.

h2. User Flow

* District buys 1 Inventory Distribution license for ABC Elementary
* CS adds license to License Management Screen
* CS goes to the ABC Elementary in System > Sites and clicks edit
* CS changes the Site Type to Inventory Distribution
* System checks if they have an Inventory Distribution license
** If yes:
*** System gets count of licenses and verifies there is one unused
*** If yes, system allows the use to select Inventory Distribution license
*** If no:
**** System gives user error message: All licenses used
**** System does not allow the user to select Inventory Distribution
** If no:
*** System gives user error message: Inventory Distribution license is required
*** System does not allow the user to select Inventory Distribution
* User saves the configuration

h2. Requirements

* In the System > Sites, for both Add New and Edit drawers, do the following:
* Add logic to Site Types drop down:
** Ensure the name is changed from Warehouse to Inventory Distribution
** If user selects Inventory Distribution, do the following:
** Verify there is at least one unused Inventory Distribution license
*** If yes, allow them to select Inventory Distribution license
*** If no:
**** Show error message: All licenses used
**** Do not allow the user to select Inventory Distribution
* If no:
** Show error message: Inventory Distribution license is required
** Do not allow the user to select Inventory Distribution
*

## Acceptance Criteria
* Users are able to select Inventory Distribution license type if they have the license and there is at least one unused
* User are not able to select it otherwise
* Warehouse is changed to Inventory Distribution in Site Types

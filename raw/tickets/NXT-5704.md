# NXT-5704 — INV - Add New Ordering Conditions - Select Sites (Step 3)

## Description
As a District Supervisor I want an Add Ordering Conditions site selection page so I can select the sites and assign them to the items I previously selected.

This is the second part of the Add New Ordering Conditions page which allows users to select Sites and Site Types and create ordering conditions based on the items selected in the first step (which is covered in [NXT-5705|https://primeroedgenext.atlassian.net/browse/NXT-5705])

*Requirements*

Provide an additional right side panel view on the Add New Ordering Conditions page which displays the sites to the right of the selected items.

+Site List+

* Display list of sites 
** Grouped by Site Type
*** checkbox for site type
*** sort Site Types by which have the most Sites
** Sort alphabetically ascending by Site Name (within each Site Type grouping)
** Site Types are collapsible/expandable
* checkbox for each site
* Site table columns:
** Site Code
*** searchable
** Site Name
*** searchable
** if possible, provide one single Site column which contains both Code and Name
*** only if user can search for code and/or name
* Provide total number of sites selected in the header
* Next button to indicate the sites have been selected
** Once Next is clicked and site selections are saved, Create button will become available (NXT-9186)
* Back button to return to the item selection step
** clicking Back button will disable the Site selection and re-enable the Item selection (per [NXT-5705|https://primeroedgenext.atlassian.net/browse/NXT-5705])
* Continued on [https://primeroedgenext.atlassian.net/browse/NXT-9186|https://primeroedgenext.atlassian.net/browse/NXT-9186|smart-link]

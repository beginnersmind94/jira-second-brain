# NXT-61925 — IM - Bulk Item Update - Cybersoft Catalog integration

## Description
As a district admin I want the bulk item onboarding/update tool to have access to the Cybersoft Catalog, so I can view items from the curated catalog and use global, regional, or neighboring district items to help onboard or update my local item catalog.

## Acceptance Criteria
* Integrate the Bulk Item tool with our Cybersoft Catalog to assist with data input/entry/update
** Cybersoft curated item catalog

# NXT-49533 — MVR - Dashboard - Clone

## Description
As a SchoolCafé user, I would like to be able to clone a managed view report so that I don’t have to take the time to create a new one from scratch.

*Notes:*

* Copy the Clone from Custom Reports as much as possible and the requirements included here assume you will do that

*Requirements:*

* On the grid in Reports > Managed View Reports, do the following:
* Action column updates:
** Clone 
*** If button not there yet, add it
*** On click:
**** Do Clone just like you do for CR, same way and rules
**** Copy the clone business rules
* 
* Ensure the clone is logged in History log
* 
* Security:
** User must have view rights to Manage Views Report dashboard to use this drawer
** In R&P, don’t show this drawer

## Acceptance Criteria
* Users are able to clone an MVR

# NXT-66824 — SC (GDSNConnect) - Update Item UI 

## Description
_(none)_

## Acceptance Criteria
Navigation: SC - Items - item Upload - Item Updates Tab 

Changes Required:



User should be able to view one record at a time

The view item dialog will show a update now button to user 

By default all the check boxes are selected (the user can unselect the ones they do not want)
Upon updating the record, the system should update the local item
Upon updating the local record, the audit log should also be triggered

In scope: item Basic Details (including GTINS) + Serving Size + Nutrient + Allergens and Meal credits



Out of Scope- User cannot update pack sizes, vendors, Vendor Prices, Vendor Item Numbers

# NXT-71831 — GC - MDM Portal - Review Queue page - Add dropdown selection to search for items that are not approved , approved and search online options.

## Description
GC - MDM Portal - Review Queue page - Add dropdown selection to search for items that are not approved , approved and search online options.

Scope of this story is to add dropdown and get the search results for the selected option and given search text for GTIN and Ingredient name.

# Currently review queue is fetching the items from staging table (processed data mart)
# we need to add a dropdown “Search In” for searching from items that are 

* Not reviewed yet - staging table
* Approved items - GC tables
** supports search by GTIN  or search by ingredient name only or both GTIN and ingredient name.
** lists all items matching the given input 
** if both GTIN and Ingredient name is entered - search using AND condition between GTIN and Ingredient name, 
* Search online - online search and get the items for the given  GTIN or Ingredient name.
** supports search by GTIN  or search by ingredient name only or both GTIN and ingredient name.
** Display only items with GTIN information in search results - don't get item results without GTIN value. - 
** if both GTIN and Ingredient name is entered - search using AND condition between GTIN and Ingredient name, 

# search by GTIN should work and list the matching items from the selected option 
# Search by ingredient name should be a Like search and list all items exactly matching or partially matching with the given name, 
# Search by GTIN should allow multiple GTINs comma separated, - discussed in refinement meeting

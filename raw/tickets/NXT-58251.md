# NXT-58251 — Acc - Exceptions - Search Filters, Time Difference Search

## Description
As Dana the Director I want to quickly handle duplicate meals within a few minutes of each other to save time and enhanced search filtering will help me with that.

*requirements*

* Both Duplicate Meals & Eligibility Variance: 
** Update the Site Picker (multi-select)
** Update the Date Range filter (if possible - Today, This Week, Last Week, This Month, Date Range)
* Duplicate Meals
** User can specify a Time Difference filter for number of minutes between first and duplicate meal.
*** Greater Than X Minutes
*** Less Than X Minutes
** If used, this should check the sale time between first and duplicate meal and filter accordingly based on the difference.

*Discussion Notes at Refinement (1/27/25)*

* User cannot enter 0 or negative numbers for the Time Difference filter

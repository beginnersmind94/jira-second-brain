# NXT-5232 — Application Processing - (Action) Process

## Description
*User Story* 

As an application admin, i can process applications to extend benefits to matched students on the application. 

*Mock Ups* 

*Requirements* 

* Process > extends benefits to matched students in app and change application eligibility according to the new changes 
* And deleted students eligibility should get restored to prior eligibility
* And a comment is required if the application has already been processed and is now being re-processed
* if the user has hit the Process without entering the comments 
** Then  show a dialog as “Please enter comments before you proceed to process the Application”

# NXT-28530 — IM - Recipe Costing Report (Consider IYF)

## Description
As a User i would like generate a Recipe cost report so that It’s easy for me understand the recipe cost and the cost per serving

## Acceptance Criteria
*Report*

These calculations applies to Recipe Cost Report / Recipe Cost (Screen) / Recipe Material Cost / Scale Recipes / Scale Production Recipes Entry point is IM page → Reports → Recipe Costing Report

* Report fields should populate as per below report sample
* Report should use pagination similar to items “All Food Items Report”
* C & D = Item Primary Qty and Unit
* Edible Yield Factor = Item Editable Field Factor
* I-Cost = Item Fair Market Value / Standard Price
* I-Unit = Item Unit Description

!image-20211130-080102.png|width=1125,height=793!

* User should be able to print report
* User should be able to export report to PDF format
* Upon exciting the system should return to reports landing page

*Recipe Material Cost Calculation*

The calculation sample is attached in excel sheet

*Step #1*   Convert unit J to be same as unit in  L

*Step #1*  Calculate R-Cost*.* (R-Cost = I/H)

*Step #3*  Calculate Item Cost (I-Cost = R-Cost * F)                                               

*Step #4  Repeat steps 1-3 until all recipe items have been converted and column “I” ,  “K” and  “M”  are totaled in.*

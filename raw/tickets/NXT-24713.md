# NXT-24713 — UX Mock for Allergen Import - Patron

## Description
This task is to cover the task related to creating Mack up for Allergen Import- Patron.

This import is pretty straight forward similar to that of Adult Import - Patron.

The major change is the availability of Allergen Import option in Import Type and selecting the option @ File Type as Patron (in later stages we will add Item as an additional Parameter).

There are only two fields for Mapping, Student ID and Allergens. Student ID is mandatory.

The rest of the flow is similar to that of Patron Import. Please refer to Patron Import Epic 19109, Story 19813-UX mock up.

* Verify and Validate that the option "Allergen Import" is available in Import type.

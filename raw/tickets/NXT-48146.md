# NXT-48146 — Subscriptions - Dashboard

## Description
As a SchoolCafe user, I would like to be able to see what subscriptions there are so that I can know who is being notified of what.

*Notes:*

* This is the dashboard story and first part of the front end for subscriptions
* It will cover the Monitoring Dashboard and the Subscriptions tab of the Event Subscriptions page
* The History tab will be covered in a separate story
* The subscription creation will be created in a separate story
* This story relies on the backend framework story to create the list of events and list of subscriptions
* The objective of this story is to be able to view a list of subscriptions that have been created

*Requirements:*

* Add a card to the system section to link to the Event Subscriptions page
** Title: Event Subscriptions
** Description: View the list of subscriptions and their distributions

## Acceptance Criteria
* Card is added to Monitoring Dashboard and links to the Event Subscriptions page
* Event Subscriptions page exists with right name and 2 correct tabs 
* Page has a back button
* Tab has a filter section to the top with list of modules and events
* Grid has no Add New, but can edit and delete
* Grid shows modules, events, users and Roles
* Users and Roles are truncated and show full list in tooltip
* User can click on View Details icon to view the subscription details
* User can click on History to view the History of the event
* When editing a row, module and name are read-only. and can edit and select multiple Users and Roles
* Grid has the usual paging
* Grid has sorting

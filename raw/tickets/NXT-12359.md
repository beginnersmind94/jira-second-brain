# NXT-12359 — [Cancelled] Platform - AI/ML - Generate Authentication Token

## Description
*Persona*: Administrator

*Access level:* School District/Site

*Description*:

As a site admin, I should be able to generate and configure device key during the registration of the  EdgeBox with valid serial number

*Mockup:*

[https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|smart-link] 

*Requirements*:

* Right after registration of the device, System shall generate a token and call Amnisicent API to configure in the device
* 
* Same Authentication token shall be used for camera devices along attached to the EdgeBox. 
* System admin shall be able to RESET the authentication token and update the EdgeBox with new token

## Acceptance Criteria
* Site admin/User shall be able to authenticate/validate Edge device token when POS is initiating the Edge device
* Authentication of Edge Devices are based on Authentication Token, SchoolID and POS ID as unique key
* System shall notify site admin for any failed authentication or RESET authentication request

## Acceptance Criteria
* Update overview to provide more statuses to give more visibility into truly required information
* Combine Item # and Description into a single column
** left-align alert icons 
** plan to do this on every sub-screen
* Change all edit pencils to a view icon
** move the view icon below the column heading, next to Item Count 
** Use a button instead of icon for the Item column
* Condense columns as small as possible
** Make some label changes to support this
*** Serving Info → Servings
*** Pack Size Info → Inventory Info
* Add an Images & Docs column, at the end of the grid
** show numeric chips to indicate how many images/docs are uploaded for that item
** clicking the chip will open the Images & Documents slide-out for that item
* Update the icon system
** refer the Section Status legend
** Disabled - 20% Black icon on 5% Black background
*** this is used for sections which are completely inapplicable to a particular item (ex: food columns for non-food item, or Menu Item column for non-Menu Item)
** Pending - Orange question mark icon
*** this is used for any section which is in scope of the session but is not yet populated

** Pending but Not Ready - Gray question mark icon
*** this is used for sections which are in scope of the session but not yet accessible (ex: Nutrient section when Servings have not been provided)
*** [https://fonts.google.com/icons?selected=Material+Symbols+Outlined:help:FILL@1;wght@400;GRAD@0;opsz@24&icon.query=question&icon.size=24&icon.color=%231f1f1f|https://fonts.google.com/icons?selected=Material+Symbols+Outlined:help:FILL@1;wght@400;GRAD@0;opsz@24&icon.query=question&icon.size=24&icon.color=%231f1f1f|smart-link] 
** Partial/Incomplete - Orange 3-dot icon
*** used for sections with some data but not all mandatory data points populated
** Complete w/ some missing - Green outlined checkmark
*** only applies to Nutrients section
** Fully Complete - Green filled checkmark icon
* Use Inventory Readiness icons for inventory info column
** Disabled - Ineligible icon
** Pending - new icon (clipboard with question mark) SVG attached
** Incomplete - Incomplete icon
** Ready - Published icon
** Unready - Unpublished Icon
* Use the Menu Item Readiness icons for Menu Info column

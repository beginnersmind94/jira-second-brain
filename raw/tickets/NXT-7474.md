## Acceptance Criteria
Menu Cycle can be created (Active by default)
After creation, user is given the edit screen for the same Menu Cycle seamlessly
Menu Cycle can be edited (Data Source, Meal Pattern, Meal Type, Site Group are locked, similar to Menu)
Save success message = Menu Cycle has been saved successfully.
Menu Cycle name is unique, failure message = Menu Cycle name must be unique.
Number of weeks 1-9 (default = 1)
Days per week 3-7 (default = 5)

# NXT-49525 — MVR - Dashboard - View History

## Description
As a SchoolCafé user, I would like to be able to view the history of my managed view reports so that I can quickly make decisions.

*Notes:*

* Copy the View history from Custom Reports as much as possible and the requirements included here assume you will do that

*Requirements:*

* For the Reports > Managed View Reports grid, do the following:
* In the ellipsis, add a button to the actions if not already there:
** Title: View History
** On click, open the History drawer



*View Managed View Report Configuration drawer*

* Create a new drawer
** Name = View Managed View Report History
** Size = same as CR
** Use the sliding action when opening
** Copy the View History from Custom Reports
** Use the same History component as usual
* 
* Ensure you show all the add, edit, delete updates for MVRs
* 
* Security:
** User must have view rights to Manage Views Report dashboard to use this drawer
** In R&P, don’t show this drawer

## Acceptance Criteria
* Users are able to see all add, edit, delete actions for a given MVR
* Only users with view rights to MVR dashboard can see this page

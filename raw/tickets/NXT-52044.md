# NXT-52044 — File – Validate Data - Student Import - Part 2

## Description
As a SchoolCafé admin, I would like to be notified when I validate the data of a File Import so that I can see what issues I will have before implementing the import.

*Note:*

* When doing the original story, we had issues with the notification part of the story so we separated it out as a part 2; this story covers that part

*Requirements*:

* On the System > Imports & Exports > File tab > Rows, do the following:
* 
* Validate File overlay:
* Once validation is done:
** Notify the user in a toast success message if they are still on the screen
** 
** Send the user a notification message via their preferences
*** The user will receive a notification animation when it is done
*** Work with System team to implement
*** User can browse to it and open it from Workspace Inbox
*** Message - Exceptions download complete. Go to the {ConfigName} import configuration and click Validate to view the file.

## Acceptance Criteria
* User is notified in Inbox when the file is ready to view

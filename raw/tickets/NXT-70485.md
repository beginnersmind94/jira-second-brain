## Acceptance Criteria
* Allow Edit ability for the header level section of the menu
** Everything should stay in label mode except for Menu Name and Tags
** Any inactive menu will not allow for editing unless switched to active or draft mode (Separate Story)
* Gray out all other sections and only allow the user to save or cancel 
* Provide an edit for selecting serving groups and indicating counts in a slide out along with ability to indicate projected meal servings in percentage
** Updating the counts or adding serving groups/ovs should reflect on the main page
** There must be atleast 1 serving group available at all times
* Provide an edit for selecting menu item categories and choose counts
** Updating the categories and or choose counts should relfect on the main page
** User can not remove the counts on a category if it already contains menu items but can change the choose count
* Trigger historical entry on every save point

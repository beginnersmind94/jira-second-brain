# NXT-57367 — MP- Site Configuration- Full Capture of History

## Description
As a district admin, I would like to have a full history of everything that happened to particular sites regarding configurations such as change in site groups, meal types, menu lines etc.

## Acceptance Criteria
Currently History on the Site configuration only shows any prep site configuration (Should work from either tab)

* History should also include
** Site Group change whether individual and edit site configuration or from the site groups tab
*** Capture Effective and end date as well as new effective date if site group is again changed
** Menu Lines either individual or edit site configuration 
** Meal Services (This should be triggered based on Menu lines configured on particular meal type)

# NXT-3567 — Item Management - Add and Edit Food Item - Skip and Clear Nutrient Information

## Description
*User Story*

As a Item Management User, I can skip or clear Nutrient Information

*Mockup*

!image-20200812-204636.png|width=100%!

*Requirements*

* Clear button must clear out the information in the section
** If there is information populated, then have a popup showing a confirmation
*** “Are you sure you want to clear Nutrient Information”?
**** Confirm or Cancel
*** The background behind loses focus
*** When clicking the ‘clear’ button the data, it removes the data from the database upon save & close (edit and add) or navigating ‘back’ or ‘next’ on add a new item
* User must be able to select the ‘Skip for now’ checkbox
** When selected, minimize the section and ‘cache’ the data
** If they un-select the ‘skip for now’ checkbox, then the data should return as it was before

* The section when collapsed will have the ‘skip for now’ enabled in the database which also means ‘missing’
* When expanded, the buttons show up, and the default of ‘skip for now’ is unchecked and the clear button appears
* When information is entered at all, the ‘skip for now’ button is disabled until information is cleared or user manually deletes values from the fields

## Acceptance Criteria
User can skip or clear a section when adding or editing

# NXT-14427 — Platform - Security - Add Resource Hierarchy to View Resources page

## Description
As a primero edge admin, I should be able to access resource catalog to add a new resources available for any district so that the resource can be accessed by any user with the assigned role for that resource.

# User shall be able to click on “Add New” button
# User shall be able to add new resource from drawer pop-up
# User shall be able to the newly added resource in resource table

# NXT-7658 — System - Set Password screen- Display page when link is expired

## Description
As a PrimeroEdge user or administrator, I would like to know when a link in a password-reset email has expired, so that a new one can be requested and set out as needed.

*Requirements*

* When a New User OR Password Reset email is sent, it contains a link that expires in a certain time period.
* If a user selects this expired link, instead of directing them to the Set Password page, redirect them to a page that says:
** *Title/Header*: This link has expired
** *Body message - Paragraph 1*: Sorry, this link has expired. Please request a new link from your administrator. You can contact them using the information below:
** *Paragraph 2/Info section*:
*** <DistrictName>
*** <Contact PhoneNumber>
*** <Contact emailAddress>
**** Display the email address as a ‘mailto’ hyperlink that opens up the user’s email client and adds the address in the ‘To’ field.
** *Button*: Return to Homepage
* When the ‘Return to Homepage’ button has been selected, redirect the user to the homepage for that tenant.

## Acceptance Criteria
Verify that when a New User OR Password Reset email is sent and the link has expired, if a user selects this expired link, instead of directing them to the Set Password page, we redirect them to a page that shows a message with support info for the tenant.

Verify message content and fields are accurate for the related tenant

Verify that we display the tenant's email address as a ‘mailto’ hyperlink that opens up the user’s email client and adds the address in the ‘To’ field.

Verify that when the ‘Return to Homepage’ button has been selected, we redirect the user to the homepage for that tenant.

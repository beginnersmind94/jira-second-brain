# NXT-60312 — Digitize AI - Production Record - View documents

## Description
As a user I want to see a button where I can access my digitized documents within the Production Record so I can preview and decide if I want to use them to update my production record.

* New button called “Digitized Documents” added to the Production Record tab. This appears if the setting/license is available.

!image-20250324-164104.png|width=1421,height=342,alt="image-20250324-164104.png"!

Selecting the button displays a popup with the document preview, digitized summary and details, and action buttons.

!image-20250324-164241.png|width=1426,height=711,alt="image-20250324-164241.png"!

## Acceptance Criteria
* Provide button for “Digitized Documents”
* Provide document preview with scrolling and/or paging when multiple pages/images
* Provide digitized information grid with parsed data

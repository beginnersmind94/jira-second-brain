# NXT-8191 — INV - Create Control for Site selector

## Description
As an inventory user I want a control for the Site selection so every page that uses the Site selections will look and operate the same.

* Area
** areas are either enabled or disabled for the entire district, so Area dropdown will be hidden if they are disabled
** if no Areas are defined, Area dropdown will be hidden
** if user has access to only one site, Area dropdown should be hidden
* Site 
** one dropdown for both code and name
** ‘All’ option for site (for use on some pages)
** if user has access to only one site, they will see only that site and will not see ‘All’ option

This control should be used on every page that has site selections so that there is consistency throughout the module.

## Acceptance Criteria
Site selection control will be a reusable component
Logic for displaying Area will be handled within the control
Logic for displaying 'All' option or not will be handled within the control
Site dropdown will contain Site Code and Site Name

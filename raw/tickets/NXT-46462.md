# NXT-46462 — Data - PowerQuery - POC

## Description
As a SchoolCafé user, I want to be able to run PowerSchool PowerQueries so that I can retrieve specific data that I need to see.

*Notes:*

* This is POC / spike story to investigate how to connect to PowerSchool API and run a PowerQuery
* If successful, we will do another story to run this from the front end
* The primary objective is to prove that we can run a PowerQuery

*Requirements:*

* Be able to connect to PowerSchool API
* Run any valid PowerQuery 
* Retrieve the results
* Document what is necessary to implement PowerQuery in the UI/UX
** Do we need to expose any fields to the user? Which ones?
** Do we need to save any config info in the DB?
** etc.
*

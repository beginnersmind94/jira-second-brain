# NXT-4317 — System - Users - Add/Edit User - Add functionality to upload user avatar/image

## Description
As a user, I would like to be able to upload my own image or the images or other users in my district, so that we will be easier to identify throughout PrimeroEdge.



* In the Users - Add User drawer, when the logged-in user selects the image icon and the document upload appears, allow them to select an image to be the user avatar.
* When the image is selected (jpeg,png, gif), update that user’s image.
** If user does not enter correct file type, display an error
** Create a reasonable size limit. If user uploads an image beyond that size, displays an error
* Perform necessary resizing, compressing, etc. so that the image fits in the circle icon.
* Changes can be committed immediately or when the user clicks the Save button.
* Do the same for Edit User.

## Acceptance Criteria
Verify that when adding a new user, their image can uploaded

Verify that when editing an existing user, their image can uploaded

Verify that if user does not enter correct file type, we display an error

Verify that if user uploads an image beyond the size limit we set, we display an error telling them that the file exceeds X size

Verify that we perform necessary resizing, compressing, etc. to fit image inside circle

Verify the changes are saved (immediately or when user clicks Save button)

Verify that the avatar is visible when the logged-in user navigates back to this selected user account

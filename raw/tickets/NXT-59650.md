# NXT-59650 — IM - Item Onboarding- Review Pack Size Info

## Description
As an onboarding district admin, I want a a step to be able to review important information about my pack units and the configurations associated.

With Inventory Options

## Acceptance Criteria
* Provide a review step on the Bulk tool for Pack size
* Provide a Pack Size Overview that illustrates the pack units and relationships
** This will originally be populated based on the selections made on the *initial info* section of the tool
*** Display Placeholder units for items that have not had their pack sizes configured 
*** Items that have been marked as None/Decide Later will be filtered out with a toggle by default
** Display Dual Unit and higher pack relationships
** DO not Display “Inventoriable” serving relationship with the rest of the pack units on the Review Pack Size (Will show where applicable)
* Provide a column for “Count Base Unit in” to display whether the base unit is being counted in decimals or whole numbers
** Display none for any items that are still in “placeholder” mode
* Provide an column for Fair Market Value and always show based on the cost of the highest unit
* Provide an inventory status column that will contain 4 “statuses” to display what will happen to the inventory information once the item is published to the system
** Ineligible: Missing Pack unit information (Sub Qty, serving/pack size link, etc), all missing categories (Refer to Summary info), Missing Cost
** Published: This is the default for all items that are eligible and have all mandatory information
** Unpublished: User must toggle this option off in the inventory options section (This should not be used often)
** Incomplete: Items with partial pack size information, unit cost indicated but no cost, some categories missing
* Provide alerts for missing mandatory information, if multiple alerts display with a line break (This will be enhanced later as part of clickable messaging on the inventory status)
** Tool tip for missing pack unit information “Missing mandatory pack unit information”
** Tool tip for missing cost “Missing cost”
** Tool tip for missing categories “Missing categories”
** Tool tip for missing Pack- serving link “Missing pack-serving link”
* Provide Edit Options that will be the different steps of entering full pack size information on each column
** Inventory Options
*** Units & Sub-Qty - Providing number of units, Descriptions of Units and Sub Qty relationship
*** Pack & Serving Link- Providing a link between serving size and base unit either via sub-qty (Inventorable serving), weight or linking as a dual unit
*** Inventory Publish- Allowing users to mark which items to not publish to inventory
** Cost/Value- Adding or updating Fair Market Value per item
** GTIN/UPC- Add or Updating GTIN/UPC per Pack Unit

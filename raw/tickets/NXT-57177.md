# NXT-57177 — IM - Items - Units - Limit Volume Measurements 

## Description
As a menu planner I need the system to contain only valid volume measurements so I can see my fluid item units in either Milliliter, Fluid Ounce or Gallon measurements.

h4. Context:

!image-20241127-181032.png|width=91.66666666666666%,alt="image-20241127-181032.png"!

## Acceptance Criteria
* Volume measurements should be limited to *Milliliter, Fluid Ounce,* and *Gallon* when configuring units for fluid items
* The system should still autofill the Volume value when a volume-based serving measure is entered in the Serving Measure Description field of a fluid item (because we know the corresponding ML value)
** Cup, Quart, Pint, Tablespoon, Teaspoon
** however these measures should +not+ be present in the actual Volume dropdown

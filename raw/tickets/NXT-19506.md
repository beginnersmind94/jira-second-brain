# NXT-19506 — Production- Production Record- Page display based on Status

## Description
As a Kitchen Manager, I should not be able to have access to my production record page until my production plan status has changed from production incomplete to production complete.

Once production plan has been marked as complete, the production record page should be able to be accessed.

## Acceptance Criteria
Once production plan has been marked as complete, the production record page should be able to be accessed.

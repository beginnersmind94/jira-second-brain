# NXT-18137 — INV - Manage Orders - Order Detail - Save history record when order changes

## Description
As an Inventory user I want history records to be created when orders are updated or changed so I can view the historical progress of an order.

*Requirements*

* Save a historical entry each time an order is updated/changed according to the list of changes below

+List of potential updates:+

* Original Transaction - order first saved on Order Detail (Open, Pending Review)
** at time of ‘Create’ button on Create Orders > Confirm Orders screen
* Order Adjusted/Updated 
** quantities changed
** Item added
** Delivery Date changed
** Delivery Location changed
** Comments entered/changed
* Order Merged
** Merged into another order
** another order merged into this order
* Order Approved
* Order Rejected
* Order Resubmitted
* Order Closed
* Order Cancelled
* Receipt Created

## Acceptance Criteria
* History record will be saved when one of the listed changes occurs with an order

# NXT-16896 — INV - Withdrawals - Create Permission Resources

## Description
As a FSD I want resources for Withdrawals so I can assign users with the appropriate permissions.

*Requirements*

Map permissions for the Withdrawals functionality

+Withdrawals list page+

* *View* - access to the list page/view in navigation
** Reports are included; no WD report permissions required (yet)
* *Add* - visibility/access to New button

+Withdrawal Reasons slide-out+

* *View*  - enables the Reasons button on the Withdrawals list page
* *Add* - enables the Add button on the Withdrawals Reasons slide-out
** also needs to show/hide the Reason Description field
* *Edit* - enables the Edit icon in the grid on the Withdrawals Reasons slide-out
* *Delete* (Cancel) - enables the Delete icon in the grid on the Withdrawal Reasons slide-out

+Withdrawal Details page+

* *View* - visibility of View icon on list page, access to Details page
** Reports are included; no report permissions needed
* *Edit* - enables the Edit button on the Withdrawal Details
* *Delete* (Cancel) - enables the Cancel button on the Withdrawal Details

# NXT-37498 — ICC - Performance Improvement - FO APIs - UI Integration

## Description
The following UI changes should be factored

Current Benchmark:

Data Loading during first load: 50+ seconds

Data Load on Filters - 45 to 55 seconds

Threshold Refresh - 25 to 35 seconds

## Acceptance Criteria
The performance of the API should be brought to between 3 to 5 seconds to refresh to data when the user change the date criteria or Threshold criteria

Pre-requisites - The APIs to be split with FO team

The new API changes to be incorporated in UI

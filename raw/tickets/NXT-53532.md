## Acceptance Criteria
* Update the recipe Servings (Units) card to have a single version across all tabs and item types
* The same functions should be available across all tabs (depending on status)
* The banner area should only be used for banner chips, statuses, instructions, etc.
* Edit mode:
** provide an edit pencil to edit the main serving info (current edit mode)
** display the menu item banner chip in the header, in collapsed form 
** display the footer with Menu Serving and Exception buttons, *once menu item is enabled*
* When the Menu Info tab is selected but the recipe is not a menu item…
** show the menu item banner chip in the header, in expanded form
* To enable a recipe as a menu item, click the chip to expand the prompt (similar to items)
** Provide a second click by using the footer to confirm (just like items)
** Put the menu item into Incomplete status and fill the placeholder Menu Item Description
* Make sure all list pages support Incomplete status for menu items

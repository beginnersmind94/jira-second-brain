## Acceptance Criteria
Complete when:

When creating an item I can associate/define an item as a substitute
Entering this information is not mandatory
The substitute item is selected from a smart dropdown list
The smart dropdown list will list

# if the item is a NON-FOOD ITEM then only list active NON-FOOD ITEMS (no duplicates) and in the same Item Type as the item
# if the Item is a FOOD ITEM them  then only list active  FOOD ITEMS  (no duplicates) and in the same Item Type as the item

General:

* This requirement applies to list and Split view
* This also applies when editing an item
*  Only user created Item Name/Code should be display
* Only Active Items Should be display
* Only one item can be selected
* This field is not mandatory
* CN Items can be use as a substitute on a manually created item
* For CN do not display Substitute field

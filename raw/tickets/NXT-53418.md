# NXT-53418 — FH - Menus - Show Menu Description in pop-up windows

## Description
As Nancy the Nutritionist I put a cool marketing description together for my items and I want households to see it to encourage participation.

*requirements*



!image-20240730-220759.png|width=1253,height=827,alt="image-20240730-220759.png"!

!image-20240730-221025.png|width=1885,height=912,alt="image-20240730-221025.png"!



* Add a Family Hub admin setting:
** Show Marketing Description in Menus
** Options: Yes/No
*** Default: No
* If the setting is yes:
** Show the Marketing Description between the photo and the Ingredients list.
** Also show it in the Make a Tray windows including Saved Trays.
* Double check that we’re using the Marketing Name if it exists for menus
** If we are not, implement.
** If it does not exist, use the Menu Item Name instead.
* Rename the existing setting:
** Current: Show Marketing Description in Menus
** New name: Use Marketing Name for Menus
** Tooltip/Functionality: If enabled this will use the Marketing Name field for menus instead of the menu item name.

## Acceptance Criteria
* Setting added to Family Hub District Admin
* Marketing Description added to menu item pop-up windows if setting is set to Yes
** Use it in the Trays as well
* Use the Marketing Name if it exists - verify if we are using it already first
** If it does not exist, use the Menu Item Name - believe we are already doing this

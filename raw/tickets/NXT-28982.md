# NXT-28982 — System - User History

## Description
As a SchoolCafe admin, I should be able to view the audit history of the user to view the creation and updates on any user information.

## Acceptance Criteria
* User shall be able to see the audit history from users info page.
* System shall record the user creation time, created by attributes to audit history tab.
* User shall be able to see when user is created with timestamp and user/process that created
* User shall be able to see the user updates with the changes

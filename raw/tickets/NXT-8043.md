# NXT-8043 — {Menu Cycle} Analysis - Calculate/display nutrient results

## Description
As a Menu Planner
I need to be able to build Menus
so that I can show that they are compliant.

Display the average nutrients as described in external document. Basically, this is a simple average for each day (after those calculations are completed as described in previous story 8035) - meaning that the number of meals per day does NOT affect the average. These should be displayed in a manner consistent with that for a single Menu using mockups in a previous story (8117).

Simple example for Calories:

Day 1 = 500, Day 2 = 550, Day 3 = 600, Day 4 = 650, Day 5 = 700. Average: (500+550+600+650+700)/5 = 600.

Now suppose the number of days = 6 and the last day is blank or no Menu Items. Average: (500+550+600+650+700+0)/6 = 500.

## Acceptance Criteria
Nutrients for a school week are calculated as described and displayed.
Any missing Menus, or Menus with no items count as 0 and affect the average in that way.

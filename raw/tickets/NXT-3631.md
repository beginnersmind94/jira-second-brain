# NXT-3631 — Inspections Template - Allow Media

## Description
*User Story*
As an Inspections user creating a Template, I have the ability to enable or disable the ability to add media on individual questions.

*Mockup*
To be added here:
https://nqyc9v.axshare.com/start.html#p=template_approved_layout&id=hk2qvp&g=1

*Requirements*
* There is a toggle on each question when creating a template called "Allow Media"
** The toggle is present for all question types
* When enabled, flag is set to allow Inspector to attach media to the question when running an Inspection using the Template
** When disabled, the flag is set to not allow attaching media
** By default, the setting is enabled

## Acceptance Criteria
* Allow Media option is present for all questions on all templates, enabled by default
* Flag is properly set when the option is enabled/disabled

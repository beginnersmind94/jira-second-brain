## Acceptance Criteria
* Provide an Ingredient search which will allow the user to select an item or (sub)recipe present on active recipes
** pre-filter this list by actual ingredients
** for performance quality, consider requiring minimum character count for ingredient search
* Provide filter options
** Ingredient Type: Items, Recipes, *Both (default)*
** Pending Deactivation Ingredients Only: this would filter to show only ingredients or sub-recipes which are in Pending Deactivation status
* In the dropdown, show an icon for Pending Deactivation ingredients
* Upon clicking the Find button, load and display all active recipes containing the selected ingredient
* Once recipes are loaded by the Find action, display the Replace With section
** show the ingredient being replaced in a *view only* mode at the top of the card
** provide an option to open the ingredient in a new tab
** display the icon if the ingredient is Pending Deactivation
* Display the Recipe grid with overview info
** Recipe #
** Recipe Description
** Step #  (the step which contains the ingredient)
** Ingredient Quantity 
** Action column with view icon
*** to open the recipe in a new tab
* If a recipe appears twice in the list, show them together sorted by earliest step
** do not repeat Recipe # and Description

# NXT-26855 — PR- Production Record- Non Reimbursable Serving Column

## Description
As a user, I would like to indicate counts for Non Reimbursable counts for my menu items on the Production Record.

This may not apply to everybody so we need to have a system setting to Turn this feature on or off.

## Acceptance Criteria
Display “User Specified” column and “Reimbursable” Column. FOr Example: User Specified in this case is “Ala Carte

The Reimbursable Column will be calculated as so

Total Produced- (Carryover+RTS+Waste)= Total Served

Total Served- (User Specified) = Reimbursable

Set a system setting Plan Plan/Record Page (Production- Plan and record Common setting) - Non-reimbursable Servings Column Header Show Text Box. Display text that is entered by user as the column header on production. IE “Ala Carte” Only Alpha.

Set a system setting Plan/Record Page (Production- Plan and record Common setting) - Enable Non-reimbursable Servings Column (Yes or NO) or toggle or off 

If setting is set to NO for Enable Non- reimbursable servings column, hide both columns. Only effect non completed production records. Once a production record is complete, do not change the setting value for that particular record. 

For Non-reimbursable Servings Column Header Show Text Box, if user changes the name, only apply it to non completed production records. Once a production record is complete, do not change the setting value for that particular record.

# NXT-9934 — INV - Transfers - Transfer Request - Search & Display Search Items - Step 2

## Description
•	Displaying the list of all items according to selection criteria and giving filter options on the results grid columns

* A search section will be provided to search items to be added to the Item Request List
* Only items which have some inventory site-item record should be searchable
* Item Search and list from: [https://primeroedgenext.atlassian.net/browse/NXT-9933|https://primeroedgenext.atlassian.net/browse/NXT-9933|smart-link] 
* Item # / Item Name - Search
** When searching in Item Management, be able to search for the Item # and Item Name
** Prioritize exact match before like matches
** If possible, show results while typing
** Minimum of 3 characters needed to search
** Error: if trying to search with less than three characters
*** “Search must contain a minimum of 3 characters before searching”
* Transfer Item List
** Item # - Filterable and searchable
** Item Name - Filterable and searchable
** Item Description - Filterable and searchable
** List will be paginated
** Default of 20 records on a page
** Show all results and total number of pages in the bottom right

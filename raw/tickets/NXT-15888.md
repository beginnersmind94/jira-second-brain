# NXT-15888 — IM - Edit/Cancel/Save Serving Measures

## Description
As a User I would like to have an ability to Edit/Add/Save/Cancel options So that I can make  make any changes to the Serving measures

## Acceptance Criteria
Given the User is On the Nutrient and Contribution Info sections 

When Edit button on the Nutrient and Contribution is selected 

Then Enable to User to Edit and 

And “Add New Row” with the Below 

* Serving Measures
** Serving size
** Weight in Grams
** Preferred Measurement
** Possible Measurement
* And ability to “Save” the changes 
* And “Cancel” Should discard the changes 
* And Exit from the Edit Mode

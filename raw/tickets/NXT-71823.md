# NXT-71823 — Eligibility - Configuration - FRE Application Image - Add Date Range to bind an image to a subset of Apps

## Description
As a Cybersoft admin, I would like to be able to modify the HTML image template for my FRE apps and have it be for a certain time period, so that I may make them more closely aligned to my SchoolCafé (and paper) application.

*Summary*

* Add a new page to allow users to designate a date range for an application image template for the current FRE Application.
* When opening an application, open the correct application image based on the provided date range.
* This means that the ‘Active’ image will only apply to new applications.

# NXT-3341 — Employee Dashboard - Results Count/Lazy Load 

## Description
We will use lazy load for the grid/card views will loading employees. The style/format will be based on the uniform design for the entire system. 

* use lazy load to view records 
* a count of the results is updated when user searches 
* on initial load, the total number of records
* common component

## Acceptance Criteria
Employee card view and grid view use lazy load
user sees count of results when searching 
the count is updated when a new search is triggered

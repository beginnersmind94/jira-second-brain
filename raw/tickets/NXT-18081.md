# NXT-18081 — INV - Orders - Manage Orders - Receipts - Quick Receipt - Receive Step - Back to Order Step

## Description
As a Site Manager creating a Quick Receipt, I want the ability to go back to the Order step so I can make changes to the items or restart the Quick Receipt process.

*Requirements*

* On the Receipt step of the Quick Receipt process, provide a ‘Back’ button in the bottom left corner
** Back button would navigate back to the Order step
* If the Quick Receipt has *not* yet been saved (as incomplete):
** Back button will navigate back to Order step
** All fields will be editable, including header selections
** Item quantities will be populated from previously entered quantities
** If user changes header selections (Site, Order Date, Vendor) and clicks Apply, this will clear all the items and restart the process
*** Provide a prompt: “This will reset the Quick Receipt process and clear all selections. Do you wish to proceed?” Cancel/Yes
* If the Quick Receipt has been saved as Incomplete:
** Back button will navigate back to Order step
** Header selections will be disabled
*** Site
*** Order Date
*** Vendor
** Item quantities will be populated from previously entered quantities
* Next button will proceed to the Receipt step once again

## Acceptance Criteria
* Back button on Receive step will navigate to Order step
* If Quick Receipt is saved as Incomplete, do not allow header changes
* If Quick Receipt is *not* saved, allow header changes but provide prompt when resetting
* Next button will proceed back to Receive step

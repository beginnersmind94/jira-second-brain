# NXT-20414 — MP > Menu Planner > Save Button & Clear Filters Button (name change)

## Description
As a user, I want to be able to save my work on the Menu Planner Page.

!image-2021-10-06-12-44-55-217.png|width=200,height=183!

 

A user starts working on the Menu Planner and will not be able to finish creating the menu all at once, but wants to be able to save their work. Clicking the save button will save the menu name if only one day is created. If there are multiple days, it will display a pop up that will allow the user to enter a menu cycle name. 

If only one day of the menu planner has information, it will create a single days menu. If there are 3 days with information, it will create a 5 day menu cycle (or 7 day if "include weekends" is selected).

Menus with no items on days ( for this example Tuesday and Thursday). Proper behavior would be to not save a menu for that day but still include the day on the menu cycle. 

Once a user clicks save, a link (or links) should be displayed that will allow the user to navigate to the menus or menu cycle pages that have been created.

!image-2021-10-06-12-44-45-083.png|width=200,height=183!

 

* User can not add additional weeks unless atleast 3 days are filled on the first week.

If a user adds additional weeks, DO NOT SAVE additional weeks unless there are menu items on additional weeks.

When pop up with menu cycle link appears, user can click “confirm” to get link to the Menu Cycle active on the pop up, then click the red X to drop back onto the Menu Planner page to begin a fresh build of a menu. (Blank screen is expected).

**Currently, there’s a button with the text “Clear Filters” - we would like this to be made a little more clear to users by changing the text to “Reset”

## Acceptance Criteria
Save button will allow user to stay on menu planner, saves menu or menu cycle with names entered by user.
Menus within menu cycle save with menu cycle name with additional Week and day associated with it if no individual names are entered.

There will be no editing after save, all information will be locked on menu planner, links provided to menu or menu cycle for user to click on and open in new tab make additional changes.

* User can not add additional weeks unless atleast 3 days are filled on the first week.

Menu cycle can only be saved with minimum of 3 days.

Menu Name should be unique.

# NXT-30774 — Acc. - POS - Block session creation if POS Period is closed

## Description
As Vera the Director I don’t want anyone to be able to add/edit sessions if the POS Period is closed.

## Acceptance Criteria
When the POS Period is closed the following actions are blocked at the POS:

* Creating a New Session

The buttons for these actions would be disabled.

User is informed why via a tool-tip: This option is disabled because the POS Period is closed.

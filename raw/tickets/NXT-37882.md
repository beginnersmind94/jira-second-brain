# NXT-37882 — IM - Items - Nutrient Info tab Interactions (Select Serving Size)

## Description
As a nutritionist I want the Nutrient Info tab to interact with the serving sizes in the header so I can associate my inputs with the corresponding serving size.

*Goal:* Provide appropriate interactions between the Nutrient Info tab cards and the Serving Sizes in the Units header card, for the purpose of serving size selection or nutrient info data entry.

*Requirements*: 

* Move the Nutrient Details card to the top of the Nutrient Info tab (now with Serving Measures removed)
* Reorganize the attribute presentation in the Ingredient Info card (UX)
** provide a toggle to show/hide Ingredient Info card if all values are blank
*** this would be a tab-specific toggle option in the Item Type slideout
*** only visible when opening the slideout while on the Nutrient Info tab
* Update the Nutrient Details and Contributions cards to interact with the Serving Sizes in the Units header card
** in *view mode* the Unit Identifiers will correspond to the serving size for which they were entered
*** e.g. show the Nutrients identifier on the serving size for which the nutrients were assigned
**** show the Contributions identifier on the corresponding serving size as well
** in *edit/create mode*, enable the selection of a single, specific serving size for capturing entry in either card
*** Nutrients and Contributions must each be assigned to a specific serving size, but they do not have to use the same serving size
*** *Update*: Just provide a selection dropdown at the top of the Nutrient Details and Contribution cards before data input
**** This would populate the identifier in the corresponding serving size in the Units card
**** no dropdown necessary if only one serving size exists
* Allergens does not require any change
** perhaps more width can be made for the right side cards and a little less for the Nutrients? (UX)

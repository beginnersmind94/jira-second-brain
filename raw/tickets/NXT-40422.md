# NXT-40422 — IM - (Multi-Item) Recipes & Custom Menu Item - Nutrition Sync

## Description
As a Menu Planner I want the system to automatically sync changes to my multi-item Recipe nutrition info based on changes to the associated ingredients.

## Acceptance Criteria
* Build a service to sync system calculated/generated updates to multi-item Recipes based on changes to the associated ingredients
* If users are using ‘System Calculated’ option, automatically update their values when changes are made to linked ingredients
* Consider independent flags for *Nutrient Details* and *Allergens*
* Recalculate the total recipe Nutrient values
* Recalculate Allergen values

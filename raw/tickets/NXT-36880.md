# NXT-36880 — Imports – View Log – Add Filters & Sorting

## Description
As a SchoolCafé Admin I want to be able to filter and sort the grid columns on the view log tabs so that I can better use the grid.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* Click on View Log button
* Click on History, Source Data or Exception tab
* Click column to sort and use the filter section

*Requirements*:

* Would be great if it was done for every import type, but do at least for Student Import
* 
* On the grid, do the following:
* Add sorting to each column
* Add filters to any column missing it

## Acceptance Criteria
* Able to sort and filter the grid columns

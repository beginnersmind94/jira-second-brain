_(none)_

## Acceptance Criteria
Navigation: DB  / API Enabler

Changes Required:

The serving size parsing should be revised based on the attached scenarios:

Serving Size - Skeleton

{ 

*Scenario: All Blank*

   Serving Description = Null -  No need of parsing

  Quantity and Measure = NULL

   Unit Weight and UOM = NULL

    mark it as exception

}

{

*Scenario: Non Standard / Parsed ||  Standard / Parsed*

   Serving Description = Not Null

   Quantity and Measure = NOT NULL

   Unit Weight and UOM = NOT NULL

   Use Parsing

+*_If Quantity is or Measure is NULL_*+ – use the Quantity and Measure from original file as-is

  Check measure mapping table – If measure found with conversion exists – Quantity * Measure = Unit Weight, Use corresponding UOM from Mapping table (Set 2) ; Ex: 1 Ounce (28 Gram)

If Measure is not found – leave Weight and UOM from original file ; Ex: 1 packet (5g)

*_If Quantity is Not NULL and Measure is NOT NULL_*

    Quantity and Measure – Parsed Variables (Set 1)

     Check measure mapping table – If measure found with conversion exists – Quantity * Measure = Unit Weight, Use corresponding UOM from Mapping table (Set 2) ; Ex: 1 Oz (28 Gm)

If Measure is not found –

  Check UOM - If measure found with Equivalent Measure – If found use the corresponding value (Ex Gm – Gram)

     If UOM is not found then leave Weight and UOM from original file; Ex: 1 Slice

}

{

*Scenario: Only Set 1 present / Standard ||  Only Set 1 present / Non Standard*

   Serving Description = Null

Quantity and Measure = NOT NULL

   Unit Weight and UOM = NULL

   Use Parsing

    Quantity and Measure – Parsed Variables (Set 1)

     Check measure mapping table – If measure found with conversion exists – Quantity * Measure = Unit Weight, Use corresponding UOM from Mapping table (Set 2) ; Ex: 1 Oz (28 Gm)

            If Measure is not found – leave Weight and UOM from original file ; Ex: 1 packet (5g)

 }

 {

*Scenario: Only Set 2 present / Standard || Only Set 2 present / Non Standard*

   Serving Description = Null

* No need of parsing

  Quantity and Measure = NULL

   Unit Weight and UOM = Not NULL

Check UOM in measure mapping table – If measure found with conversion exists – Unit Weight * Measure = Unit Weight, Use corresponding UOM from Mapping table (Set 2); Replicate converted Unit Weight and UOM as Set 1 (Quantity and Measure)

    If OUM is not found – use Weight and UOM from original file (Ex: 50 Gm)

    Replicate Unit Weight and UOM as Set 1 – this will go as exception in SC (Ex: 1 slice)

}

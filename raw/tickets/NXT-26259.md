# NXT-26259 — AcctMgt - Accounts - Principal Account Quick Preview

## Description
As a district admin, I would like to be able to preview certain Special Accounts, so that I can make payments and transfers for any principal account within my district. 

*Summary*

* Let’s allow a user to make payments and transfers from the Principal Account Special Accounts by providing them with these 2 functions in a Quick Preview on the Accounts page.

*Mock-up:*

N/A

*Details*

* From the Accounts page, in the search grid, display a Quick Preview button for Principal Accounts to:
** Allow user to make a payment for the Principal Account (primary action)
** Transfer from the account (secondary) to a student
** Transferring from student back to principal will be covered in a demo
* No other functionality is needed, all should work as it does for other existing account types already.

## Acceptance Criteria
This story is complete when a user can go to the Accounts page, search for a Special Account using the filters, select that account, and be able to pay and transfer from a Principal Account.

# NXT-7955 — System - Management - Create & Manage Periods

## Description
As a food service director or site manager I want a page for Inventory Periods so I can close and manage periods in accordance with our operations.

*Mockup:*

[https://www.figma.com/file/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=4750%3A47823|https://www.figma.com/file/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=4750%3A47823|smart-link] 

*Requirements*

* Show a ‘Period Type’ dropdown for each Period Type
** not all users will see all options; if user only has Inventory access they will only have Inventory option
** this may be used to show (mirror) the page in the respective modules
* Provide a search header with the following dropdowns:
** Fiscal Year
*** mandatory single selection
*** default to the current fiscal year
** Site
*** single dropdown for both Code + Name
*** provide an ‘All’ option
*** default to ‘All’
** Status
*** Open
*** Closed
*** Reopened
* Provide a ‘Periods’ button (or Apply) to load the list of periods
* Display a grid below the header which contains the list of periods for the selected Fiscal Year
** Display the following columns in the grid:
*** Period
*** Start Date
*** End Date
*** Status
*** Action
**** this column will display either a ‘Close’ button to close the period or a ‘Reopen’ button 
* Validations/prompts when closing a period
** When closing a period, alert the user if any prior periods are still open
*** “August 2021 period is still open. Are you sure you want to close September 2021?”
*** alert only; do not stop user from closing

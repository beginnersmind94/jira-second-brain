# NXT-49612 — Data - Dashboard - Edit & Delete

## Description
As a SchoolCafé admin, I would like to be able to edit or delete source data configs so that I can manage my list.

*Requirements*:

* On the System > Imports & Exports > Data tab, do the following:
* On the Grid Actions buttons
** Delete
*** Validate has Delete rights to Imports & Exports
*** On click:
**** Show the usual confirmation popup
**** If confirmed:
***** Delete from DB
***** Show green success or red failure message
***** Refresh the grid with the row leaving it
** Edit
*** If button doesn’t exist, create it
*** Validate has Edit rights to Imports & Exports
*** On click:
**** Show the PowerQuery Edit drawer if it is a PowerQuery row
**** Show the Data Source Edit drawer if it any other type
* 
* Security
** Parent’s edit permission required to click on Edit button
** Parent’s delete permission required to click on delete permission
* 
* Edit drawers
** Do the same as the Add New for both types except update instead of save
** Ensure IsPII on the preview button
** Don’t allow them to change the Source Data selection by making it read only

## Acceptance Criteria
* User can delete one if they have rights
* User can edit one if they have rights
* Source Data field on Edit drawers is read only

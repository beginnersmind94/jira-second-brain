# NXT-9882 — Direct Certification - File Search - Actions

## Description
As a district admin I have found the DC file entry I want to use and want to match it against a particular student.

*mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421832%3A1615&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421832%3A1615&scaling=scale-down-width|smart-link] 

*requirements*

* User can view a match scorecard for any result that is already matched.
* User can perform a DC match manually for an entry in the table against a selected student.
** User can select an entry in the results grid.
** User has a Student Lookup option available.
** User can search for students per usual lookup parameters.
** User can review this manual match, seeing data from the DC file vs student information.
** User can confirm this match and must enter a comment.
** User can view the history for anything that is already Matched.
* The manual matching option is only available for the current academic year.

## Acceptance Criteria
User can see a match scorecards for any entry that has already been matched.

User can perform a DC match manually by selecting an entry in the grid and then locating a student to match against.

User is forced to review their selection and must confirm the match.
User must enter a comment for why this match is happening.

Student is updated appropriately for eligibility.

# NXT-12535 — Button Component for Email notification

## Description
We would like to build a button that will be used across other functional module to achieve the email notification. 

* It will be Button that will have below configuration
** Display Name - Notify by Default ( Should be configurable by the developers who will implement)
** Module - Id or name. example - Student Eligibility 
** Business Process - Id or name. example - Approval / Disapproval
** Channel - name. example -  Email, Print etc
**  Business objects and Email- Map of \{email, objects}. example - Student applications / student objects

API call is set the common API that is built in the backend. Check the links below for the story.

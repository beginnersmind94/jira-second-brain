# NXT-47772 — Roles & Permissions - New Screen - Part 3 (Clone)

## Description
As a SchoolCafé admin, I want a more intuitive way to set permissions for given roles so that I can control who has access to what.

*Notes:*

* This is a continuation to Part 1 & 2
* In this part, we are doing Clone functionality

*Requirements*:

Updates to R&P Dashboard:

* For the System > Roles & Permissions landing page, do the following:
* For each row, add a Clone button
** Title: Clone
** On click: 
*** Note: UI / UX is like the + New Role drawer
*** Open a drawer:
*** Title: Clone [Role Name]
*** Fields:
**** Role Name
***** Same rules and requirements as + New
***** Auto Insert value: Clone from [source role name]
***** Enforce unique name for the role for the district
**** Description
***** Same rules and requirements as + New
***** Auto insert the description from the source role it is cloning from
**** Save
***** Same requirements as + New screen
***** Additionally, copy all the users, CRUD values and anything else from the source role
***** Do not copy the name and description of the cloned role, use the values in this screen
**** Cancel

## Acceptance Criteria
* A user is able to clone a role with all its permissions and users
* Crud values, linked users are all copied
* Name cannot be a duplicate

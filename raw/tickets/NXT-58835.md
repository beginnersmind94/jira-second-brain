# NXT-58835 — License Management

## Description
As a school administrator using the SchoolCafe dashboard, I want to see my current license usage and quickly access license management, So that I can monitor and manage my school's device licenses efficiently.

*Functional Requirements:*

h3. Display

# *Show Current License Usage*
#* Format: {{XX/YY}} where:
#** {{XX}}: Number of licenses currently in use (cannot exceed {{YY}}).
#** {{YY}}: Total licenses available.
#* *Location*: Below "SCHOOLCAFE TEST" in the left sidebar.
#* *Default Color*: {{#3970b0}} (matches existing link color).
#* *Font Size*: {{12px}}.
#* *Text*: Should read {{Licenses: XX/YY}}.

----

h3. Color States

# *Normal State*
#* Usage: Less than 90%.
#* Color: {{#3970b0}} (existing link blue).
# *Warning State*
#* Usage: 90% to 100%.
#* Color: {{#FFA500}} (orange).
# *Hover State*
#* Color: Same as the current state (blue or orange) but with an *underline*.

----

h3. Navigation

# *Clickable License Text/Count*
#* The entire license text/count should be clickable.
#* *Click Behavior*: Navigates to the License Management page on family hub.
#* *Hover Behavior*: Same as other sidebar links, with an *underline* to visually indicate it is clickable.

----

h3. Real-time Updates

# *License Count Updates Automatically*
#* Triggers for updates:
#** When new devices are *activated*.
#** When devices are *deactivated*.
#** When the license quota *changes*.

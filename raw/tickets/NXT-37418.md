# NXT-37418 — Data Exchange - File Export - Eligibility code mapping and Balance Date field

## Description
As a user I want to see the eligibility codes that are in the export file as numeric values instead of the written status so that I can support what is required to import back into my Student Information System.

* Code mappings needed are Free = 1, Reduced = 2, and Paid = 0

As a user I want to see a column for the AccountBalanceDate  so that I can support what is required to import back into my Student Information System.

* Account Balance Date should simply be the date that the file is generated. IE - GetDate()
* This field should be available as an option to configure in the Student Export dataset

# NXT-39436 — Student Import – Language Codes – Allow Special Characters

## Description
As a SchoolCafé admin, I want to ensure we allow special characters in the languages I am importing so that we can properly map them to our internal codes.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Imports
* Click Add New or click Edit for a Student Import
* On field mapping screen, map languages and click the ellipsis
* Be able to see special characters in the external source field

 

*Requirements*:

* The following changes are for Student Import, related to Language mapping drawer
* Specifically, we need to allow space and round braces in the External Code text box of Coder Mapping, like the way we did for '-'.
* 
* -Allow special characters in the External Codes field-

## Acceptance Criteria
* -Special characters are visible in the field-
* allows space and round braces in the External Code text box

# NXT-18472 — Production- Reports- Non-Reimbursable Items (UI and API Report)

## Description
This report will give user information about all the Non- Reimbursable Items served in a day or date range
Information does not break it down by day but over the date range selected. 

Page Display needs to have this criteria:
Site Code
Site
Site Group
Meal Type
*From* and to *Date*

Criteria needed to display in columns:
Recipe Number
Menu Item
Serving Size
Quantity Sold (Non reimburseable column count)
Unit Cost
Purchase Cost

When displaying quantity sold, it will compile all amounts sold for an item over a time period and not broken one day at a time.

## Acceptance Criteria
Page Display needs to have this criteria:
Site Code
Site
Site Group
Meal Type
*From* and to *Date*

Criteria needed to display in columns:
Recipe Number
Menu Item
Serving Size
Quantity Sold ( Non Reimbursable Column )
Unit Cost
Purchase Cost ( Non reimbursable count X Unit cost)

When displaying quantity sold, it will compile all amounts sold for an item over a time period and not broken one day at a time. 

Only recognize and display items that have a non reimbursable user specified count. If blank, do not display menu item on report.

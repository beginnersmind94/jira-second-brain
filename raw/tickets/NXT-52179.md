# NXT-52179 — INV - Orders - Implement E-Order Sent status into Order status

## Description
As a Site Manager I want to know when an order has been successfully sent on an E-Order, so I can know when the order can no longer be updated.

## Acceptance Criteria
* In conjunction with the [banner message|https://primeroedgenext.atlassian.net/browse/NXT-51033], restrict order editing when the order has been successfully sent on an E-Order
* Update status from Open to Sent 
** consider ‘Sent’ as a sub-status of Open

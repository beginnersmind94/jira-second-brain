# NXT-19395 — INV - Vendor - Items - Contract Item Changes

## Description
As a FSD I want the Contract Items to be fully functional so I can manage all aspects of contract items.

Contract Items have been moved to the Items tab at the vendor level, with a dropdown to switch contracts.

Other additions:

* Addition Item Details ([NXT-7252|https://primeroedgenext.atlassian.net/browse/NXT-7252])

*Requirements*

* Make the Contract dropdown single selection (for now)
** show Contract #, Date Range
* Update the Contract Items tab components to ensure full functionality
** Grid styling should match other recent Inventory components
** Grid sizing should match standard Inventory grid appearance
* Remove Storage Category column
* Size the columns appropriately
** Item # should be roughly 12-14 characters wide
** Item Description should have more width
** Unit Type should be no bigger than the width of ‘Display Shipper’
*** with the rarity of ‘Display Shipper’ and ‘Mixed Pallet’ we could just size this column for ‘Package’ and wrap the text for DS and MX
** Purchase Price should be no bigger than 12 or so characters (ex: $1,088.4705)
* Remove the white background on top of the main page background
* -Make the side panel a fixed component- ([NXT-22560|https://primeroedgenext.atlassian.net/browse/NXT-22560])
* Resolve existing issues

+Issues with current functionality:+

* Row highlighting should work
** when selecting an item, there is barely any visual indicator of the item that is being edited or added
** there is supposed to be a color change highlight for the row of the item being selected
* Purchase Price is not being displayed as a currency, i.e. $25.00 or 25.4997

# NXT-26834 — System - Periods - Manage Periods Shortcut

## Description
As a food service director or site manager I want a page for Inventory & POS Periods so I can close and manage periods in accordance with our operations

*Mockup:*

[https://www.figma.com/file/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=4750%3A47823|https://www.figma.com/file/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=4750%3A47823|smart-link] 

*Requirements*

* Rename page & URL from “Manage Periods page to “Periods”
* Filter the site and period type based on the shortcut navigation of the user
** If user has access to only one site, site drop down should default and disabled
If user is navigating the periods shortcut from Inventory module, period type shall be defaulted to “INV”

## Acceptance Criteria
* Users shall view the “Periods” page under management category with Generate and Manage periods tabs
* Site user that has access to only one site, site drop down should default and disabled
* User navigating the periods shortcut from Inventory module, period type shall be defaulted to “INV”
* 2 shortcuts will be created for POS and INV

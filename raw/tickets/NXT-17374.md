# NXT-17374 — INV - Orders - Order Detail - Receipt Tab - Receipt Detail - Cancel Receipt

## Description
As an ordering manager I want a Cancel Receipt button on the receipt details so I can completely cancel a specific receipt

**Note:* there is some overlap in logic between this story and the Editing Receipt in [https://primeroedgenext.atlassian.net/browse/NXT-17418|https://primeroedgenext.atlassian.net/browse/NXT-17418|smart-link] 

*Requirements*

* Provide a Cancel Receipt button on the Receipt Detail
** this would be in the top row next to the Edit Receipt button
* Check against batch for any usage
** Do not allow Cancel if any product in the batch has been used
** Prevent Cancellation if batch has been used
*** “Sorry, this receipt cannot be cancelled because some of the product has already been used.”
** For now we are assuming FIFO is always enforced
* Prompt on Cancellation
** “Are you sure you want to cancel this receipt?” No/Yes
* Display the Cancelled Receipt on the Order Detail for historical value
* Once a Receipt has been cancelled:
** Change the banner to ‘Receipt #142 was *cancelled* on 8/10/2021 by Justin Miller.’
*** show as a blue or red banner
** Remove the Edit Receipt button
** Change the text on the Receipt tab for that receipt to -strikethrough-
** Show cancelled receipts as the last receipt tab

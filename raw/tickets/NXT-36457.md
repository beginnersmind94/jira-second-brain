# NXT-36457 — POS - Warn user if they try to close out before sync completes (UI ONLY)

## Description
While the user is closing their session, check to see if data syncs have finished.

* While they’re completing, show a progress bar.
** Sending data…
* Once they’re finished, complete the progress bar and notify the user.
** Data sent successfully!

## Acceptance Criteria
User is stalled in closing the POS site if sync is in progress.

User is given the message prompt.

User is not given the message prompt if sync is already finished.

# NXT-3413 — Template Hub - Actions - Copy

## Description
*User Story*
As an Inspections user, I have the ability to create a copy of a template to build a new template from it.

*Requirements*
* There is a copy action available for Templates
** This action is not present for Global Templates in the Global Template screen
** This action is present for Global Templates that have been added to the district's templates
* When clicked, a confirmation prompt appears
** Text: Create a copy of <Template Name>?
** Yes/No
* A copy is created of the template when confirmed
** Copy is set to inactive
** Template name is "Copy of <Template Name>"
** All other settings should match the original template exactly

## Acceptance Criteria
* Copy action is present and can be used to create a copy of a template

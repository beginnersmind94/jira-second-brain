# NXT-58356 — IM - Recipes - Menu Info - Remove 'Return to Stock' from recipe Leftover Category options

## Description
As a menu planner I want the Menu Info card to prevent me from selecting Return to Stock for the Leftover Category on recipe-based menu items, so I don’t try to “return” recipes to inventory stock like DMIs.

!image-20250123-135547.png|width=83.33333333333334%,alt="image-20250123-135547.png"!

## Acceptance Criteria
* Only allow Return to Stock as a Leftover Category selection on DMIs
* Remove RTS from Recipe based Leftover Category options
** Only allow Return to Stock as a Leftover Category selection on DMIs
* Migrate current RMI with Return to Stock to instead be switched to Unspecified

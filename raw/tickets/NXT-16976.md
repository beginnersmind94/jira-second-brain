# NXT-16976 — MP - Assign Menu Cycles Summary

## Description
As a user, I need to be able to view a summary of the Menu Cycle for the specific week, showing the different menus per sites and the effective dates

Mockup

!image-20210728-125031.png|width=1133,height=634!

Requirements:

* Starting Date
** Display the date. If clicking on the calendar, open as a date picker to select another date (This would then refresh the data below)
* Confirm: (Menu Cycle Name)
* Skip Holidays (Default selected)
** When selected, it will skip the menu for a holiday
* Skip Weekends (Default Selected)
** When selected, it will skip weekends
* Use all menus (Default NOT selected)
** Override if there are menus already there
* Menu | Day | Serving Date | Existing Menu
** Menu Name
** Day of the Menu
** Date of the menu
** Existing Menu - If there is already a menu for that day
* Site Name
** Site Name
** Site Group Name
** Effective date (date when the site was placed in site group)
** End date (date when the site will no longer be in a site group
* Assign
** Assigns the menu to the dates above

## Acceptance Criteria
User must be able to assign a menu cycle and show the summary of all information above

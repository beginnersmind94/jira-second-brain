# NXT-52109 — INV - Physical Inventory - Review tab updates

## Description
As an Inventory Supervisor I want a consolidated screen to review a site’s physical inventory for a period so I can reconcile a single time regardless of how many physical inventories were completed.

## Acceptance Criteria
* Enhance the Physical Inventory Status tab to provide a more comprehensive overview of a site’s physical inventory
* Remove the ‘By Dates’ radio option and provide only a Period dropdown
** default the Period dropdown to the current period
** Consider Interim inventories as part of the period in which they were started
* Update (and redefine) the possible values of the Status filter
** Not Started - no physical inventories for the selected period
** In Progress - at least one physical inventory for the selected period
*** also if there are multiple inventories but they are all incomplete (in progress)
** Partially Complete - if there are multiple inventories for the period and some are complete but one or more are still incomplete
** Pending Review - all physical inventories for the period have been saved as complete
** Reconciled - the site’s perpetual has been reconciled to the physical counts for the selected period
* Update the Site grid
** Provide the show/hide filters icon
*** when filters are expanded, provide a column header search for % of Perpetual Counted and for Last Reconciled Inventory
*** ensure the numeric filter options are available for the % of Perpetual Counted column search
**** Greater than, Less than, Equal to, etc.
**** Apply only to the percentage value (not item counts)
** display a single Site column (To discuss: how to show site code)
** populate the Status column with the appropriate updated value
** display a new ‘% of Perpetual Counted’ column
*** display the percentage of that site’s total perpetual items that were counted across all inventories for the period
*** in parentheses, display the count of the items and the total count of the site perpetual. Ex: *52% (111 of 214 items)*
*** if multiple inventories are present for the period, show a ‘Multiple Inventories’ text after the item count
** display a ‘Last Reconciled Inventory’ column
*** display the last time that site had their perpetual reconciled
** Provide an Action column with a view icon
*** the view icon will open the new ‘Review Completed Inventories’ sub-page

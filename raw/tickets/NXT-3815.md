# NXT-3815 — Account Management - Advanced Search - Site Section - Site Name subsection

## Description
The site names defined in the district will be listed here. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Site name values. The user can type in the name or a partial name. The Names are listed in Alphabetical order. As names are selected they are moved to the top of the list. Unselected names are separated from selected names and are shown in alphabetical order.

!Screen Shot 2020-04-28 at 9.31.18 AM.png|width=287,height=341!

## Acceptance Criteria
The filter values are populated from the DB table

There is a search field that allows the user to search for a name

The search field shows partial matches.

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Site Type section they are 'or'ed together.

When a filter value is selected the Site Type Subsection count is increased by 1 and the Site Section count is increased by 1.

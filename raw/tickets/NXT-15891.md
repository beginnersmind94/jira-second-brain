# NXT-15891 — IM - Adding Possible Measurement 

## Description
As a User I would like to have an ability to Add Possible measurements for an Item So that I can understand an Items Measurable sizes

## Acceptance Criteria
Given a User is already in the Nutrient and Contribution Edit mode 

And Already selected the Inline Edit option 

When Add button is tapped under the Possible Measurement 

Then Display a Dropdown with the Possible Measurement for 

* Serving Size 
* Weight in grams 
* Preferred Measurement 

And ability to select the measurement through checkbox 

And “Save” the selected measures

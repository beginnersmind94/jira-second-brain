# NXT-55078 — SC - USDA Foods API - Conversions Function for all the Nutrient Fields 

## Description
_(none)_

## Acceptance Criteria
Navigation: API  Backend Story

Currently the USDA API does not provide Vitamin related values against serving size weight, they are available as 100 Gm Only

This function shall cover the following data elements

All the Nutrient Elements including, Vitamin A, Vitamin C

and Vitamin D

Formula To Convert against Serving Size : (Nutrient Value of 100 Gm * Nutrient Value of Serving Size Weight ) / 100

The function should be consume as common in API endpoints for both PE & SC

# NXT-22616 — IM - UI Improvements - Items - Upload Exemption Letter

## Description
_(none)_

## Acceptance Criteria
The user should be able to upload a letter and be able to store against an item

The letter should support PDF/JPG file types (what is the file size allowed)

*Label:* Exemption Letter (the UI represents in CAPS, but it has to be in Sentence Case)

*Upload Button* - User should be able to upload either an PDF or an Image file

*Delete Option* - user should be able to click on X icon to delete an existing letter

*Buy American Flag* - The exemption letter section should be hidden when user check this box

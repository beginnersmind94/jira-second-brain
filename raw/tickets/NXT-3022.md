# NXT-3022 — View Inspections - Actions - Delete

## Description
*User Story*
As an Inspections user with proper permissions viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to delete an Inspection.

*Requirements*
* Delete is an action that can be conducted on Inspections
** This action is present on all Inspections unless they've been Approved
** This action is available to users with the delete permissions
*** Admins
* Clicking delete will permanently remove the Inspection
* When clicked, the user receives a popup
** Header: Delete Inspection
** Information
*** Inspection Name
*** Site
*** Template
*** Inspector
*** Date
** Text: Permanently delete this Inspection
** Two Buttons:
*** Delete
*** Cancel
* When user presses delete button, another prompt appears
** Are you sure you want to delete this inspection? This action cannot be undone
** User has the option to proceed with deletion or cancel

## Acceptance Criteria
* User with proper permissions can delete any available Inspection from the View Inspections screen in either view (card or list)
* User with proper permissions can delete any available Inspection from the Inspection Details screen

# NXT-23956 — INV - Management - Contract Items - Update Price

## Description
As a District Admin I want an Update Price process so I can quickly update price conditions for many contract items at once.

*Requirements*

* Provide an action on the Contract Items page ([NXT-22306|https://primeroedgenext.atlassian.net/browse/NXT-22306]) to update the pricing for all the selected items
* Lock down the dropdowns 
* Display the item list, but change the list/columns to a dedicated screen for prices
** Column list:
*** Item #
*** Item Description
*** Vendor Item # 
*** Prices
**** show the current price condition, ex: 1/1/2022 - 12/31/2022 : $24.50
**** if a future condition exists, display it as well, ex: 1/1/2023 - 6/30/2023 : $26.00
**** Provide a way to view current price conditions ([NXT-24679|https://primeroedgenext.atlassian.net/browse/NXT-24679])
*** *Effective* Date
**** Provide a date picker in the header which can be used to apply the same date to all items in the list
**** allow each individual item date to be modified (but perhaps don’t use a date picker on every one – like PrimeroEdge)
**** disallow dates in the past 
*** New Price
**** a field to enter a new price for each item
*** Show the Pricing Condition icon  
**** open the Price Conditions slideout for the selected item on click
*** Save button
**** to save individual items
** Provide a Save All button for mass updates
* Insert a new pricing condition according to the New Date and New Price values
** automatically adjust existing condition End Date accordingly
** ignore items with empty fields 
* Provide a ‘Complete’ action to return to the searchable main version of the Contract Items page

*Mockup Notes:*

* hide Action column in view-only

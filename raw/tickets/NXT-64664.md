# NXT-64664 — WS - Benchmark - Previous Years - PE

## Description
As a Workspace user, I want to be able to view the benchmark data for previous years, so that I can track what I used to vs what I am doing now.

*Requirements*

* This story is for the benchmark configuration page
* It applies only to PrimeroEdge Districts (SC districts are in a related story)
* In the drop down list for the academic year, show previous years
** Show all the academic years in the system for the district
** Allow users to edit the values for previous years

## Acceptance Criteria
* Users are able to see all their previous academic years in the benchmark config screen
* Users are able to edit them

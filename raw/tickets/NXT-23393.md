# NXT-23393 — INV - Config - Templates - CRUD permissions

## Description
As a district admin I want permissions for the various Configuration pages so I can make sure only the right users have access.

*Requirements*

* Map CRUD level permissions for the Templates config page
* Template List
** View - access to page; visibility in navigation
*** sub-page has distinct CRUD operations
** Edit - visibility of edit pencil, to change Template Name
** Add - ability to add a new template
*** Hide the entire ‘Add’ card from the page if the user doesn’t have Add permission
*** Move the list card up on the page
** Delete - visibility of delete icon, to remove existing templates
*** all existing business rules must be maintained
* Template Details
** View - access to view icon on list page
** Edit - access to checkboxes on Sites tab
** Add - access to Add button and add item subpage
*** no need for individual permissions on the sub-page
*** we can “hide” the sub-page resource and simply associate it with the Add permission on the Template Details resource
** Delete - access to checkboxes and Remove button on Template Items tab

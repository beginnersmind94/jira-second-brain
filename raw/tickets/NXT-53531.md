# NXT-53531 — IM - Items - Units - Common Units card for all tabs

## Description
As a district admin or menu planner I want the Units card to behave the same across all tabs so I don’t have to memorize the UI logic to make important updates.

## Acceptance Criteria
* Update the Units card to have a single version for all tabs
* The same functions should be available across all tabs
* All banners, messages, tooltips, etc. should be the same across all tabs
* The banner area should only be used for banner chips, statuses, instructions, etc.
* The edit mode should behave the same across all tabs (and types) and should be very clearly in edit mode to the user (UX R&D)
** the expanding size of the card should perhaps be more exaggerated
** the background blurring/graying effect could also be exaggerated 
** we should always be able to make enough space for the columns
*** Summary card and Images & Docs can be greatly compressed as necessary
** collapse/shrink other page elements as necessary
* Use the new consolidated Add and Update buttons in the footer
** Add: Serving, Pack, Menu Serving
** Update: Cost/Value, Inventory Options (new), Serving Exceptions, Yield Factor (new), UPC/GTIN
* All existing items should work perfectly with the new design

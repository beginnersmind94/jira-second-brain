# NXT-71115 — IM - Recipe - Not allowing Standard Measure units

## Description
As a menu planner, because recipes are always inheriting weight per serving, it never makes sense to allow me to enter in my unit description in terms of standard weights (gram, ounce, Pound).

## Acceptance Criteria
* Do not allow standard measures to be added as recipe servings
** Gram
** Ounce 
** Pound
* Provide an alert for the user “Standard Measures are not allowed”
* All current recipes that have a standard measure as a serving must be corrected 
** Gram → Grams
** Ounce → Ounces
** Pound → Pounds

# NXT-5967 — Item Management - Disregard Changes - Product Info

## Description
As a user, I want to have a confirmation message if I click the cancel button on Product Info

Mockups

!image-2020-08-18-08-22-00-681.png|width=66.66666666666667%!

 

Requirements

* This is specifically for product info on the Add screen and the edit screen.
* This is a clone of [https://primeroedgenext.atlassian.net/browse/NXT-5046|https://primeroedgenext.atlassian.net/browse/NXT-5046|smart-link] and will use the same logic
* When clicking cancel on 'Edit' or 'Add New' a warning has to appear
* Message: "You have unsaved changes. Are you sure you want to disregard?"
** Have the option to 'CANCEL' or 'DISREGARD'
* If clicking cancel, then return them to the add or edit screen with the unsaved changes
* If clicking disregard, then navigate out of the record or return to the previous state
*

## Acceptance Criteria
The user must be warned if they want to disregard their unsaved changes or not.

# NXT-35600 — System - License Management - Allow SC admins to set license counts

## Description
*Description*:

As a SchoolCafé admin, I would like to be able to enter the amount of site licenses that have been purchased by a district, so that all they have to do is assign those licenses to their preferred sites.

!image-20221215-202156.png|width=938,height=924!

*Requirements*:

* In System → Management → Licensing Management, do the following:
** In the _Actions_ column of the grid, add an Edit pencil to each product row
*** This pencil should be tied to the Edit role permission, and that permission should be added to SchoolCafé Administrator roles
*** When this edit pencil is selected, open a drawer that allows a user with permission to edit the following for the selected product, as well as Save and X buttons:
**** Licenses (corresponds to the *Available* column)
***** Must be a positive numeric value (0 is acceptable. If a maximum is needed, it can be 999)
**** Renewal Date (corresponds to the *Renewal Date* column)
***** Should not be a past date
*** If the user clicks X in the drawer that opens, do not commit the changes.
*** If the user clicks Save in the drawer, commit the changes by updating the Available and Renewal Date fields. 
* Any changes should update both the District Account onboarding dashboard (if available) and the Onboarding Wizard, wherever we show products and their associated license count.

## Acceptance Criteria
Verify that an Edit pencil has been added to each product row within the Licensing Management page

Verify that the Edit pencil is tied to the Edit permission of this page in Roles and Permissions

Verify that this permission is automatically granted to any SchoolCafeAdmin-type roles

Verify that the Edit pencil opens a drawer when selected, that allows user to view and update the license count and renewal date

Verify that license count must be a number greater than 0

Verify that past Renewal Date values are not allowed

Verify that changes not committed will not update the Available and Renewal Date values in the grid for that product

Verify that committed changes will update the Available and Renewal Date values in the grid for that product

Verify that changes are reflected in both the Onboarding Dashboard and Wizard

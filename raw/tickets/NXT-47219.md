# NXT-47219 — Imports – Sites - Auto Sync to Eligibility

## Description
As a SchoolCafé admin, I would like to ensure that the Eligibility database is automatically updated with site information when System database is updated via import so that Student Import works.

*Note:*

* Currently, when System DB is updated with a site import, it does not ensure the data is synced to the Eligibility DB
* The data is needed to sync to the Eligibility DB immediately b/c if the user tries to do a Student Import, it will fail w/o the proper site data in Eligibility DB

*Requirements*:

* Ensure the Site data is pushed to the Eligibility DB when the System Sites data is updated via an Import

## Acceptance Criteria
* Site data is pushed to the Eligibility DB when the System Sites data is updated via an Import

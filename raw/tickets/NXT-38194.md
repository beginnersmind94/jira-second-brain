# NXT-38194 — POS - Auto-Serve Option - Service Mode

## Description
As Sammy the Site Manager I have a grab-and-go line that requires little oversight and no ala carte sales. I want a way for the POS to automatically serve meals so that the line continues to flow.

*Requirements*

* Add an option to the POS > Device Info > Admin 
** Enable Auto-Serve
*** Options: Yes/No
*** Default: No
*** (toggles are fine for these)
** Auto-Serve Delay Before Charging
*** Dropdown options: 0.5s, 1s, 2s, 3s, 4s, 5s.
* If enabled, this adds an option to the POS service as seen in the screenshot below.
* The service mode can then be toggled on or off.
** If on, the button needs to be green and text shows as Auto-Serve: ON
** If off, the button needs to be purple and the text shows as Auto-Serve: OFF

!AUTO-SERVE ON.png|width=1920,height=1083!

!AUTO-SERVE OFF.png|width=1920,height=1083!

* If the service mode is turned on, POS needs to check to see if the selected menu grid has a default meal item.
** If it does, proceed and enable the service mode.
** If it does not, inform the user:
*** {noformat}This menu does not have a default meal item. Auto-Serve cannot be used. 
Please select a different menu grid or configure the default meal & reload POS.{noformat}
* If the service mode is enabled:
** Transactions should complete as ‘Charge’ after waiting the amount of time set in the setting.
** First meal, second meal, doesn’t matter - Charge. 
*** No payments are accepted in this scenario, no ala cartes are added.
* Any on-screen pop-up (allergens, special instructions, second meal, etc.) will be dismissed after the wait time, then the Charge happens.

## Acceptance Criteria
* Auto-Serve mode added to POS Service for sites that are not using Dynamic Menus
* Auto-Serve options added to POS Admin screens
** Main Auto-Serve option
** Auto-Serve Delay Before Charging
* User can turn Auto-Serve on if the option is turned on
** User is warned if no default menu item is available for the grid, and auto-serve does not turn on
* If Auto-Serve is turned on:
** Default menu item is charged after a 0.5 second delay

# NXT-27120 — Update Tag Management UI columns and Design

## Description
Update Tag Management UI and Design as per updated Figma

UI Changes needed:

* Right half of the screen should have columns as Entity, Entity Type, Modules and Assigned Tags
* Under Assigned Tags there will be Views link for previewing the Tags which are assigned to that particular entity
* When you click on Views, the drawer should open.
** At the top the header should state “Tags assigned for this Entity”
** Grid should have 2 columns as Entity, Entity Type.
** Under that, all the tags assigned to that particular Entity should be displayed.
** Users can associate/disassociated the particular tag and save the changes

*Dependency on other Modules*:

Every Module has to publish a message to indicate association happened, along with tag association details.

* EntityId: Id representing the entity, it could be PersonId, ItemId or could be anything that uniquely identifies an entity.
* ModuleId: Module id in which tag is being associated.
* Entity: Could be Patron’s name, Item, or anything to which tag is being applied.
* EntityType: Could be the screen name where the Tag is being assigned.
* ModuleName: Module’s name in which tag is being applied.
* TagId: List of guids representing the tag’s ids which are being associated to an entity.

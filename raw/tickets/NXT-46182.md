# NXT-46182 — MP- Menu Planner- Search field, Frequently used, recently used and Favorites display for Pending Deactivation Menu Items

## Description
As  as Menu Planner, I would like to be able to clearly see on the menu planner on my side bar sections which items are marked as pending deactivation including the search bar.

## Acceptance Criteria
* Always display the correct status of the item on the side panel on the menu planner for all the different types

!image-20240103-142407.png|width=809,height=486!

* Allow the search bar to show pending deactivation menu items with the warning, they should not be regularly adding these items but we will allow for now
* Do not display inactive menu items for any reason
* If Pending Deactivation Menu item is removed from menu and it is on the last menu, it should become inactive they will not be able to add it back if they save

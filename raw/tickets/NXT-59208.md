# NXT-59208 — IM - Recipes - Steps - Card Updates

## Description
As a menu planner I want the Recipe Steps & Ingredients to have a dedicated Delete mode so I don’t accidentally delete ingredients during usual management of my recipe steps and ingredients.

h3. Recipe Steps - Edit Step

* ‘Delete…’ button to open Delete Mode
* (ignore Lock functionality for this story)

!Recipe Steps - Single Step Edit-20250218-160236.png|width=91.66666666666666%,alt="Recipe Steps - Single Step Edit-20250218-160236.png"!



h3. Recipe Steps - Edit Step - Delete Mode

* dedicated Delete Step button
* all edit functionality/buttons disabled
* Action column shows only Delete icon
* Cancel button returns to Edit Step 
* Save button enabled (and required) after ingredients deleted

!Recipe Steps - Delete Mode-20250218-161044.png|width=91.66666666666666%,alt="Recipe Steps - Delete Mode-20250218-161044.png"!

## Acceptance Criteria
* Update the Recipe > Steps & Ingredients tab 
** flip the left/right alignment of the Ingredient and Steps cards 
* Modify the Edit Step mode to provide a dedicated Delete mode sub-step
** change the existing Delete Step button to a ‘Delete…’ button
** remove ingredient delete icons from regular edit mode
* When Delete… button is clicked, open a Delete Mode sub-step
** disable all other edit functionality
** display Delete Step button in card header
*** use the existing prompt to confirm deletion of entire step
** display Cancel button to exit out of Delete Mode
*** this would return to the regular Edit Step mode
** change the ingredient action column to +only+ have delete icons
** enable the Save button after ingredients are deleted
*** require the Save button, otherwise ingredients are not deleted

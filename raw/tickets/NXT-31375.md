# NXT-31375 — Users Page - Change Mouse Pointer

## Description
The mouse pointer on the Users page shall be a pointer except when hovering over the icons in the “Actions” column.

## Acceptance Criteria
* The mouse pointer on the Users page shall be a pointer except when hovering over the icons in the “Actions” column in which case the mouse pointer shall be a hand.

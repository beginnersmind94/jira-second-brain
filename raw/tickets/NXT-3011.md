# NXT-3011 — View Inspections - Actions - Start Inspection

## Description
*User Story*
As an Inspections user viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to start an Inspection that is assigned to me.

*Requirements*

* Start Inspection is an action that can be applied to Inspections that are ready to be conducted
** The action is only present on Inspections in Scheduled or Re-Inspection status
** The action is only present if the Inspection is assigned to the user
* The action can be accessed from both the card/list view menu and from a button on the Inspection Details screen
* The action is present and can be clicked even if not on the scheduled date
* Once clicked, the user is taken to the Edit Inspection screen
** The Inspection's status is set to "In Progress"
** Exception:
*** If an Inspection is Scheduled, and the date is not correct, the user receives a confirmation prompt
**** Today is not the scheduled date for this Inspection. Proceed? Yes/No
***** Yes: Inspection moves to In Progress and user can conduct the Inspection
***** No: Inspection remains in Scheduled/Re-Inspection, user cannot conduct inspection

## Acceptance Criteria
* User can Start Inspection for any assigned Inspection from the View Inspections screen in either view (card or list)
* User can Start Inspection for any assigned Inspection Inspection Details screen

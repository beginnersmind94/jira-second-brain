# NXT-45127 — GDSN Connect Updates (SC)- Internal Configuration - UI Integration

## Description
_(none)_

## Acceptance Criteria
Navigation: IM - Global Catalog - Pending Updates

If the Configuration Type = No, the Pending Updates screen should be hidden from the left side Menu 

Based on the configuration settings (Type=Common) ( NXT-39287), the user should be able to select or deselect the desired attribute for updates  

Upon selecting Yes - allow users to choose data provider dropdown; Pending Updates Screen - The upload and refresh button should not be visible in Pending Updates Page

Upon selecting No - allow users to continue to take pending updates only from 1WS

Yes – upon setting the value as Yes – the user should be allowed to select or deselect the corresponding attribute in pending update screen

No – upon setting the value as No – the system disable the check box (meaning – the user is not allowed to overwrite that attribute) 

Screens to be Impacted: Pending Updates, Item Sync History

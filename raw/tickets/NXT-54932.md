# NXT-54932 — PR- Plan/Record- Add Menu item ability to add multiple items 

## Description
As a production manager, I would like the ability to be able to add multiple items at the same time.

## Acceptance Criteria
* Allow the ability to add multiple menu items from the add+ function on both plan/record page
* Display a selector for the users to select multiple menu items and display 10 items at a time with pagination (Same as Menu Planning add menu items)
** Provide a Menu item Selection counter to let the user know how many items they are adding
* Provide instructions at the top of the pop up “Select Menu Items and Click Add Item(s) to confirm all menu items selected”

!image-20240918-070722.png|width=1223,height=527,alt="image-20240918-070722.png"!

* Once user clicks add Item(s), provide a second page that will show the menu items selected and the information the users will need to fill out (previously below the single selection)
* Provide how many items have been selected and all the information that is needed including category, serving sizes for each serving group and planned count (Produced count for adding from record)
** Current logic for strict batching still applies



!image-20240918-071106.png|width=1231,height=683,alt="image-20240918-071106.png"!

* Users have the opportunity to go back and make changes to the menu items they have selected but if they have already entered counts or changed categories or serving sizes, preserve their selections for the menu items
* Once all items are added and confirmed, add all menu items to the production plan or record and all current functionality will remain the same 
* *Improve Add Menu Item screen Search Get call for better performance (API ONLY)*

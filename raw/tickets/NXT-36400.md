# NXT-36400 — MP- Assign Menus- Allow skip day option on assign menu screen for menu cycles (UI ONLY)

## Description
As a Menu Planner, there are random days where we might not be serving a particular menu on a date and would like to quickly be able to disregard it when assigning menus.

!image-20230201-210748.png|width=720,height=708!

!image-20230201-210903.png|width=719,height=711!

## Acceptance Criteria
* Allow user to be able to skip one or more days while assigning menus
* Enabling Skip Date (s) will allow users to select what days to skip
** If user disables skip Date (s) again, clear all selected dates that were previously selected
* Use strike through for the dates that will be skipped
** Use all menu toggle will come into play, if on, then slide the remaining menus down the dates until all menus are fulfilled
* DO NOT ASSIGN MENUS TO DAYS SKIPPED
* REMINDER: Left Check boxes with “Skip” Header, also look at the new design of the toggles and warning message
* !image-20230207-154902.png|width=1382,height=882!
Provide Collapse for both Menus and for the Sites Affected
** Default to expand Menus and collapse Sites Affected

# NXT-52716 — MP- Menu-  Full history of all changes made on Menu from Standard 

## Description
As a menu Planner, I would like to be able to see a history of all the changes that have been made on the menu that I am working on or any menu in general. This will give me the ability to see why there are any out of the ordinary things or changes I do not recognize.

## Acceptance Criteria
* Provide history option on the menu
** Top right hand corner of the menu



!image-20240717-032635.png|width=1022,height=392,alt="image-20240717-032635.png"!

* History should display:
** All new data in the create process 
*** Meal Pattern
*** Meal Type
*** Site Group
*** Menu Name (This is the only one that can be updated so show the updated values as well if there are any)
*** Serving Group Values (Create or Edit)
**** Group these together if there are multiple serving groups added at the same time
**** OVS 
***** Group these together if there are multiple serving groups added at the same time 
*** Menu Item Categories (Create or Edit)
**** Group these together into one entry if there are multiple added at the same time 
*** Add Menu Items (Add or Delete)
**** Group these together into one entry if there are multiple added at the same time 
**** Include serving size added per serving group 
*** Add Menu Items from Quicklist (Add or Delete)
**** Track these menu items as a separate entry and not as a regular add menu item 
***** EX: Muhammad Aly 7/19 Menu Item(s) Added/ Updated

** Serving Size (Per serving group)

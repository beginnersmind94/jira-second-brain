## Description
As a item management user, I want to be able to view the steps of a recipe

Requirements

Header

* Recipe Steps
* Display HACCP (Toggle On/Off)
** When this is On, the HACCP will show in a grid for each step
*** CCP | Critical Limit | Corrective Action
** When this is Off the HACCP is hidden
* Display Pictures (Toggle On/Off)
** When this is On, the pictures will be visible
** When this is Off, hide the pictures
* Step Number
* Text in the steps. This can contain special characters as well
* Pictures (This hasn’t been finalized yet, so placeholders can be used for now, to simulate a picture)
** If no picture is assigned to that step, then it will appear as if the pictures were hidden

## Acceptance Criteria
User must be able to view the ingredients in a recipe

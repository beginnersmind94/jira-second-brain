# NXT-40803 — IM - Recipes - Hide Yield Factor card (SIR + Multi Item)

## Description
As a Menu Planner I want the Yield Factor removed so I’m not allowing Prepared Servings to alter my nutritional analysis when USDA does not require it.

## Acceptance Criteria
* Hide the Yield Factor from the multi item recipe screen
** Move up the Allergen and Contribution cards
* Hide the Yield Factor from the Item SIR tab
** Show only Description
* Hide Yield Factor option from Recipe Report generation slideout (toggle)
* Hide Yield Factor section from Recipe Report
* Prevent ‘Prepared Servings’ from being created (for now)
* (Enhancement) Add the Recipe Report to the SIR Summary card More Actions (started in [NXT-40418|https://primeroedgenext.atlassian.net/browse/NXT-40418])

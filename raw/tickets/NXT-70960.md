# NXT-70960 — Insights - KPI - PNA - Schoolie

## Description
*I am a* district user *that I want to* have Paid Not Applied (PNA) included in Schoolie AI analysis
*so that I* can receive insights that reflect all relevant KPIs shown on the dashboard.

----

h2. *Notes*

* This story applies to *Schoolie AI integration within the Insights dashboard*.
* Schoolie is triggered from the dashboard and performs analysis on the *visible KPI cards* for the selected:
** Timeframe
** Site filters
* PNA (Paid Not Applied) is a *new front office KPI* that must be included in AI analysis when applicable.

----

h2. *Out of Scope*

* Any KPI other than PNA

----

h2. *User Flow*

# A user logs into the system and navigates to the *Insights dashboard*.
# The user configures:
#* Timeframe
#* Site filters
#* Visible KPI cards
# The user clicks the *Schoolie AI button*.
# The system sends a prompt to the AI engine including the visible KPI data.
# The AI returns an analysis.
# The user views insights, including PNA if it is part of the visible KPIs.

----

h2. *Requirements*

h3. *1. Include PNA in AI Prompt*

* The backend must include *PNA data* in the prompt sent to the AI engine.
* PNA data must reflect:
** Current timeframe
** Selected site(s)
** Current KPI values

----

h3. *2. Conditional Inclusion Based on Visibility*

* PNA must only be included in the AI prompt if:
** The PNA KPI card is *visible on the dashboard*
* If PNA is not visible:
** It must *not be included* in the AI prompt

----

h3. *3. AI Response Inclusion*

* When PNA is included in the prompt:
** The AI response must include analysis related to PNA
* When PNA is not included:
** The AI response must not reference PNA

----

h3. *4. Consistency with Existing KPI Handling*

* PNA must follow the same prompt structure and handling as existing KPIs
* No custom or separate logic should be introduced beyond inclusion

## Acceptance Criteria
h3. *1. PNA Included When Visible*

* When PNA card is visible:
** PNA data is included in the AI prompt
** AI response includes PNA analysis

----

h3. *2. PNA Excluded When Not Visible*

* When PNA card is hidden:
** PNA is not included in the AI prompt
** AI response does not include PNA

----

h3. *3. Data Accuracy*

* PNA values included in the prompt match the dashboard values for:
** Timeframe
** Site filters

----

h3. *4. No Regression*

* Existing KPI analysis behavior remains unchanged
* Schoolie continues to function for all other KPIs

## Acceptance Criteria
*Entry Point*

*Item Management → Menu Items → Add → Variety Menu Item → Variety Menu Item Accordion* 

*Item Management → Menu Items → Selected Variety Menu Item → Variety Menu Item Accordion* 

* User shall be able to link similar menu items to a variety menu item link. (drop down box) similar as when linking recipes when creating menu items
* Display all Menu Items in search but gray out items that do not match Variety Menu Item Contributions at any serving size level  (Smart search dropdown box)
* Allow user to remove Menu Items if added incorrectly or user comes back later to make changes
* Units (Use standard from contribution)
* Values for Quantity and Unit shall be taken from selected Menu Item
* Quantity and Unit value  fields shall be for display informational  purposes (the text boxes shall be gray out

# NXT-20798 — INV - Inventory - Withdrawals - Withdrawal Details - Cancel

## Description
As a Site Manager I want a Cancel function on Inventory Withdrawals so I can make changes if I have the permission.

*Requirements*

* On the Withdrawal Details page ([NXT-20052|https://primeroedgenext.atlassian.net/browse/NXT-20052]) if user clicks the Cancel button
* Display a popup requiring Comments to be entered (similar to Physical Inventory)
* Once the comments are entered, reload the page
** Hide all buttons
* Display a success message: “Withdrawal transaction #123456 has been cancelled.”

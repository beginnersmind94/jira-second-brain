# NXT-66226 — PR - Record - Calculation update for Reimbursable menu items

## Description
Currently, We are not considering carryovers as part of our calculations for reimbursable menu item counts.

## Acceptance Criteria
* Consider carryovers as part of the counts of the reimbursable calculations
** Produced + Carryover - (RTS + Waste) = Served
** Served - Non Reimbursable field = Reimbursable count
* Update all reports that include this information

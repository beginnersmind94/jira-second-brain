# NXT-32219 — Import - Photos - Enable Support For More File Formats

## Description
The objective of this story is to ensure that the photo import supports more file formats.

!image-20220624-155527.png|width=840,height=259!

## Acceptance Criteria
* The photo import shall support the following file formats:
** BMP
** jfif
** JPEG
** JPG
** PNG
** tif
** TIFF
* The “Accepted file types:…” displayed at the bottom left of the file selector shall be updated to reflect the above list.
** These extensions shall be displayed in lowercase.

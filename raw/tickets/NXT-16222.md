# NXT-16222 — Import / Export Monitor - Trigger By shall be user who executes the job

## Description
Currently, when a user executes a file import job, the “Triggered By” column in both  “Import Monitor” and “Export Monitor“ is showing the username who created the job.  It should show the username who executed the job.



{panel:bgColor=#deebff}
See related issue [https://primeroedgenext.atlassian.net/browse/PCS-392|https://primeroedgenext.atlassian.net/browse/PCS-392|smart-link] for details on the issue.
{panel}

## Acceptance Criteria
* The “Triggered By” column in the “Import Monitor”  and “Export Monitor“ shall be populated with the name of the user who imported the job.

# NXT-31839 — MP- Menu cycle- Standard/Menu Planner view and edit option (UI Only)

## Description
As a Menu Planner, I would like to have the ability to see my menu cycle in the menu planner view. I would like the ability to edit my menus within my menu cycle from the menu planner as well.

## Acceptance Criteria
* Remove link from Text to go to the menu cycle
* Give a view and a view and edit option from menu cycle page
* Give action column that will have both a view and an edit icon 
* Each icon will have a standard (Edit Menu Cycle Page) and Menu Planner (Menu Planner Page)
** The actions column needs to be “Locked” for the user to always see the ability to view or edit no matter what columns are selected
* View: User can only view the menu cycle from either option but will not be able to make any changes or edits. Everything is in view mode ([NXT-33758|https://primeroedgenext.atlassian.net/browse/NXT-33758])
** Default view for menu planner will show all weeks together if there are more than 1 week for the cycle
* Edit: User can be able to make all the changes on either page, Everything is editable as usual ([NXT-31838|https://primeroedgenext.atlassian.net/browse/NXT-31838])
** Default view for menu planner will show all weeks together if there are more than 1 week for the cycle
* Freeze the Actions column to always show as the last column in the grid, user can not remove it.

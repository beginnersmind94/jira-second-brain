# NXT-63517 — Set up Azure Blob Storage for handling media uploads in SCTV+

## Description
We're now at the point where SCTV+ for 2.0 needs a reliable backend to store images uploaded through the Image Library. The UI for uploads is already complete. This story covers the backend setup: provisioning Azure Blob Storage, wiring up the backend to handle uploads, and returning public URLs for rendering on client devices.

## Acceptance Criteria
* File successfully uploads and returns a URL for the right server depending on the environment (QA, Staging, Production)
* File appears in Azure Storage under the correct district path

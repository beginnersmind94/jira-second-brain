# NXT-18517 — Production- Reports- Variance Reports (UI and Report)

## Description
This report will give the user an excel exported report to look at food cost variances that happen between their plan and record.

Criteria to run report:
Meal Type (Default to All)
Site Group (Default to All)
Organization Level (Default to District)
Menu Line (Default to All)
*From* and *to* Date

There will be two different versions of this report. 
*Food Cost Per Meal Variance*
Columns will include:
District
Site Code
Site Name
Site Group
Meal Service
Serving Date
Menu Line
Planned Meal Count
Actual Meal Count
Serving Variance ((Actual Meal Count-Planned Meal Count)/Planned Meal Count)
Plan Food Cost (All food Cost For Menu line)
Actual Food Cost ( All food cost for Menu Line)
Food Cost Variance ((Actual Food Cost-Plan Food Cost)/Plan Food Cost)
Planned FCPM (Food Cost Per Meal)
Actual FCPM (Food Cost Per Meal)
FCPM Variance ((Actual FCPM-Planned FCPM)/Planned FCPM)

*Food Cost Variance Detail*
Columns will include:
District
Site Group
Site Code
Site Name
Meal Service
Serving Date
Serving Line
Menu Item Category
Menu Item
Recipe Code
Serving Size
Food Cost Per Serving
Planned Servings
Actual Servings
Serving Variance
Planned Food Cost
Actual Food Cost
Food Cost Variance
Leftover Servings

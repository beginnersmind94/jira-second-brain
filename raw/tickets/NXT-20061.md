# NXT-20061 — INV - Vendors - Vendor Detail - Contracts tab

## Description
As an Inventory manager I want a tab based view of the vendor so I can more easily view and manage the different vendor information.

This tab will contain the list of Contracts which are configured for the vendor.

*Requirements*

* Add a Contracts tab after the Address tab
* Show a Contracts list as a card on this tab
* Add an Action column 
** View icon
*** this opens the Contract Items page for that contract
** Edit icon
*** this opens the contract itself for editing the info

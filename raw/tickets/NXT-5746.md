# NXT-5746 — Recipe - New Recipe - Recipe Steps - HACCP

## Description
As a item management user, I want to be able to add a recipe through Item Management

Add New - Manually

Click on recipe step

Workflow: Add New - Manually → HACCP Process → Entire Recipe

!image-20200915-014424.png|width=100%!

Workflow: Add New → Manually → HACCP Process → For each Step

Requirements

* When HACCP Process → 'For each step' is selected
** Display the @mention item information from story [https://primeroedgenext.atlassian.net/browse/NXT-5324|https://primeroedgenext.atlassian.net/browse/NXT-5324|smart-link] but have this below the step, between step 1 and 2.
* Checkbox for HACCP in the top left if they need it or not.
** Acts as a ‘Select all’ for the HACCP steps below it
** The row is greyed out and can’t be edited until the user checks the box
** Default of ‘Checked’
* 'CCP' displays the CCP laid out in [https://primeroedgenext.atlassian.net/browse/NXT-5184|https://primeroedgenext.atlassian.net/browse/NXT-5184|smart-link]
** Tooltip when hovering over CCP - 'Critical Control Point'
* ‘Critical Limit’ displays the Critical limit as laid out in [https://primeroedgenext.atlassian.net/browse/NXT-5184|https://primeroedgenext.atlassian.net/browse/NXT-5184|smart-link]
* 'Corrective Action' - User defined free form text box. Allows for max of characters same as classic for now.
** Text will wrap for this
* There is no ‘save' for HACCP. The information will save when the user hits the checkbox or the X

## Acceptance Criteria
User must be able to add steps to a recipe to show the HACCP

# NXT-27206 — Acct. Mgmt - Outreach - History - Add drawer/section to view history of sent broadcasts

## Description
As a district admin, I would like to be able to view the detailed history of broadcasts I sent out, so that I can easily track if something went out that should not have, or how often I have already marketed to a subset of our district population.

*Summary*

# Begin tracking history of messages sent through the Outreach Center
# Add a History tab with a grid that allows a user to see the details of previously sent broadcasts.

*Details*

* In Account Management → Outreach, when a user sends out a Quick Broadcast (or a Campaign message is sent out automatically), track this instance.
* In the same page, add a History button that, when selected, opens a drawer that allows the user to view:
** Template Name
** Account Type
*** If the message went out to Students, Adults, or both
** Recipients
*** Number of users who received the message at the time it was sent
** Sent On
*** Day and time the message was sent
* The user should be able to both field search (as many columns as possible) and show/hide this additional column:
** Campaign Name (will be blank if the message was sent out as a Quick Broadcast, but will display if it was sent out as part of a Campaign)
** User Name - The first+last name of the user who sent the broadcast. Depending on how we have designed the scheduled notifications which will be sent out automatically, I am fine with either leaving those blank or having a generic ‘System’ name.

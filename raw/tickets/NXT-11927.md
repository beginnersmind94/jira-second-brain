# NXT-11927 — DataExchange-Create CI-CD pipeline for Azure DataFactory pipelines

## Description
Create CI-CD pipeline so that ADF code need not be moved to QA subscription manually.

Develop branch will be the collaboration in Dev subscription and master branch in QA.

All development work will be done from Dev subscription by creating a feature branch from develop branch.

Changes will be merged from feature to develop and from develop to master. Release build will be triggered 

when a PR will be merged from develop-master.

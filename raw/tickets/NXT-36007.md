# NXT-36007 — District Onboarding - Feature Selection - Allow Front Office onboarding

## Description
As an Onboarding user, I need an intuitive wizard that allows me to go through the steps required to set up all SchoolCafé Front Office modules, so that my district can begin utilizing the Front Office suite as soon as possible.

# NXT-15539 — System Notifications - Add Notification Search filters, Improve Business Process dropdown UX

## Description
As a user, I would like the ability to filter notifications by Notification Type and Content Type.

## Acceptance Criteria
* As a user, I shall be able to filter the notifications by Notification Type.
* As a user, I shall be able to filter the notifications by Content Type.
* As a user, when “All” is selected within the Module dropdown, I would like all business processes displayed within the “Business Process” dropdown.

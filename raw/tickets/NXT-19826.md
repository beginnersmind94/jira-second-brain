# NXT-19826 — MP- Get Menu Line information

## Description
As a user, I would like that when I select site group in my assign menu criteria or menu calendar report, it should pull the information for all sites within site group and all menu lines associated with each individual site.

## Acceptance Criteria
Site group selection will trigger site and menu line selections. Menu lines should be saved by site and not grouped under site group. MenulineiD should be saved under siteid and not sitegroupid.

# NXT-3348 — Template Hub - Session Handling

## Description
*User Story*
As an Inspections user, I have consistent session handling for filtering and sorting settings when viewing the Template Hub.

*Requirements*

* Filtering
** Any filters set are remembered within a session
** Any filters set are reset if a session is refreshed
* View (List/Card)
** By default, user sees List view
** If user switches to card view, it is remembered, even if the session is reset
** The setting is updated if the user changes it
* Sorting
** Sorting is remembered within a session
** Sorting is reset if a session is refreshed

## Acceptance Criteria
* User's settings on the View Inspections screen are properly remembered or reset as detailed in the requirements

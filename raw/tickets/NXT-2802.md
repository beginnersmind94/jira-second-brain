# NXT-2802 — ** Inspection Template - Date/Time Question Type

## Description
*User Story*
As an Inspections user, I can create a question where the date and/or time is entered.

*Mockup*
[https://nqyc9v.axshare.com/#id=1rjsn1&p=date_time_question_type&g=1|https://nqyc9v.axshare.com/#id=1rjsn1&p=date_time_question_type&g=1]

*Requirements*

* Date/Time can be selected as a question type
** Not available for Pass/Fail scoring type
* Users can select any or all aspects to include
** Date
** Year
** Time (24 hour or 12 hour)
* There are no scoring options for Date/Time

## Acceptance Criteria
* Date/Time question type can be added to templates

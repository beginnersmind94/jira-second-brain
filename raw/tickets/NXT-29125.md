# NXT-29125 — Acc./POS - Menu Grids - Don't show inactive items, remove inactive items from menu grids

## Description
As Martha the Site Manager I don't want the cashiers to serve inactive items to save time having to change the transactions later.

*requirements*

* Don't show inactive menu items on the POS Menu Grids, and in Menu Item Category lists.
* Deleted menu grids should also not be available at the POS.

## Acceptance Criteria
At the POS only:

* Don't show inactive menu items on the POS Menu Grids, and in Menu Item Category lists.
* Deleted menu grids should not be available at the POS.

User can still see the inactive item on the menu grid via the site.

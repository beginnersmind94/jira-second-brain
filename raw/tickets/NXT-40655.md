# NXT-40655 — Exports – Clone

## Description
As a SchoolCafé Admin I want to be able to clone an import or export so that I can avoid having to create a new import file from scratch.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Imports
* Click clone on an Import record
* System asks for name and then creates a copy of the import
* Optionally do the same thing for an Export

*Requirements*:

* On the Exports tab of the Import & Export screen, do the following:
* Add a button in the rows of the grid
** Title: Clone
* On click:
** Open a drawer
** Title: Clone
** Fields:
*** Import Name
**** Textbox
**** Required
*** Enabled
**** Toggle just like when doing Add New
*** Save
*** Cancel
** If save, do the following:
*** Create an exact copy of the export, except for the new name, in the database
*** Ensure that you setup the job scheduler if there is one
** Upon drawer close, add the export to the grid

## Acceptance Criteria
* Can clone an import or export
* If clone, it will work just as well as the manually created imports and exports with no new bugs
* If cloning an export, it will clone any job scheduler stuff and the jobs will work

# NXT-19960 — Merge 2.0 Platform UICode to System UI code to avoid routing issues.

## Description
This story is to cover the tasks related to avoid routing issues by merging the 2.0 Platform UI code to System UI code.

There there are two issues faced.

# URL does not reflect the navigation panel modules instead, shows different module and reroutes unnecessarily. 
# Also, While navigating through various modules, complete Page refreshing occurs instead of  refreshing at module levels.

Above two issues can be resolved if the Platform code is merged into System Code.

## Acceptance Criteria
# Verify and Validate that the URL represents the module we are currently navigating.
# Verify and Validate that there is no routing while navigating through modules.
# Verify and Validate that the Complete page refreshing is not happening while navigating thorugh various modules instead the module pages are refreshed.

# NXT-28286 — Allergen Import - Unrecognized Allergen UX Improvement

## Description
When a user is importing allergens and is required to “jump to” Item Management from the import mapping UI in order to add an unrecognized allergen, the user is not returned back to the mapping UI after adding the allergen.  Instead, the user is required to start over.  This lack of flow is a disruption to the user.

The objective of this story is to provide the user the ability to add an unrecognized allergen when in the “Allergen Mapping” UI while maintaining the user’s context and focus of setting-up the allergen import.

## Acceptance Criteria
* As a user setting-up an allergen import, I would like the ability to easily add unrecognized allergens and remain within the context of setting up the import while doing so.

* Show the site code and site name together.

** User can drill down and set a different start/end date for a particular site under each program.
** User has an ‘Add’ action to add a 2nd date.

## Acceptance Criteria
Update the Site column to have Site Code and Site Name together.

Program Configuration updated to have an edit mode.

Role permission created for view vs edit.

User can edit program start and end dates.

* Dates must fall within the academic year.
* Program dates can overlap with one another. (example: SNP dates can overlap with CACFP, Breakfast can overlap Lunch, etc.)

User can edit site start and end dates under each program. Multiple for all programs to accommodate academic year overlap.

SSO added.

SFSP/SSO option removed from SFSP.

User can filter the list to search for a particular site.

Feeding Site info + toggles removed.

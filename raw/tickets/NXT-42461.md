# NXT-42461 — IM - Item & Recipe Sync - Prevent Serving deletion if serving is used

## Description
As a District Admin I want the system to prevent deletion of serving measures if those measures are being actively used on recipes so I do not indirectly create invalid recipes.

## Acceptance Criteria
* Prevent serving measures from being deleted if they are being used on active recipes
* Prevent serving measures from being deleted if they are being used on active custom menu items
* Prevent serving deletion on both Items and Sub-Recipes
* Continue to allow serving size *_edit_* according to base business rules (weight/volume can be edited)

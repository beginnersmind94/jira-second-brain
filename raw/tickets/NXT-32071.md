# NXT-32071 — INV - Orders - Create/Edit Receipt - Update for Pack Size Units

## Description
As a Site Manager I want to see related items on the Create Receipt screen so I can know which items have multiple receivable units according to the order.

*Requirements*

* Update the Create/Edit Receipt to reflect the new pack size configuration
* This page will still only display items which were explicitly added to the order
** multiple item units will not be displayed unless those units are specifically on Contracts and Ordering Conditions
*** *Note:* Separate story for receiving units which were not orderable
* Show item units on individual rows but indicate items which are linked together
* Display Item # only once for the highest orderable unit
* Show conversion (sub) quantity 
* +UI Improvements:+
** Remove the highlight effect
* +UX Points (TBD later)+
** Reevaluate the columns for Delivery Fee, Processing Fee, and Net Off Invoice discount
** Consider sticky footer

# NXT-20660 — Production- Production Record- Variety Menu Item

## Description
As a production Manager, I would like a way to pick which items from my Variety Menu Item I am going to serve if I did not make any selections at the production plan.

## Acceptance Criteria
Display Variety Menu Item with a drop down checkbox selection to be able to choose which menu items they would like to actually Serve
Once saved, Display menu items in a stepper or tree grid under the variety menu item to indicate where these items are coming from. All items stay within the same category that belongs to the variety menu item.
Display all menu item information and give box for each menu item to enter in produced counts.
Allow items to be unchecked and removed from being served.

Do not allow user to turn toggle off if the menu item has been added from production plan, user must enter in zero produced count

If user turns toggle on from the record, system should place 0 planned counts and allow user to enter in produced amount

If there are multiple servings display all under 1 toggle.

If menu item from variety is carried over, follow normal carryover process (if it is carried over to another day that has a variety menu item., turn toggle on and follow process mentioned in [https://primeroedgenext.atlassian.net/browse/NXT-20659|https://primeroedgenext.atlassian.net/browse/NXT-20659|smart-link]

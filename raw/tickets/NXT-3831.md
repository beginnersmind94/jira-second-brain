# NXT-3831 — Account Management - Advanced Search - Notification Section

## Description
The Notifications Section allows the user to select a single or multiple notifications and returns a list of patrons matching the filter criteria.

!Screen Shot 2020-04-29 at 3.28.30 PM.png|width=281,height=265!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Notification Status section they are 'or'ed together.

When a filter value is selected the Notification Status count is increased by 1

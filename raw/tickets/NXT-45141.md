# NXT-45141 — 1WS (PE) - Creation of Non Food Items

## Description
_(none)_

## Acceptance Criteria
Navigation: MP - Item Catalog - Search

Search for an non food item (Ex: Plate - GTIN #00727875064349)

Check if the given GTIN or child GTIN has missing Serving Size and Nutrition Information

If yes- allow that item created as stock item only (ingredient cannot be created for non food items)

Scenarios:

Single Hierarchy: A Case GTIN which does not have child GTIN and no serving size or nutrition information

Dual Hierarchy: A Case GTIN which has Child GTIN (Ex: Each), yet does not have serving size or nutrition information

Upon successful import, the user should be able to search the item in Stock Items 

In Item Catalog Screen - the corresponding stock item ID should also be populated

# NXT-31682 — Acc. - End of Day - Settings - Special Permission

## Description
As Vera the Director I want to configure which tasks are required for my site managers so they focus on just the important ones.

*mockup*

[https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320] 

*requirements*

* Add a Special Permission setting (under Platform → Roles and Permissions → Accountability) for the End of Day Settings button.
** Permission: End of Day - Settings
** Description: Provides access to the Settings option on the End of Day page.
* Tie the Settings button on the End of Day page to this special permission.
** If you have the permission, you can see the Settings button and make use of it.
** If you don’t have the permission, the Settings button is not available and you cannot use it.
*** The Refresh button should be in its place. There should not be a blank space where the Settings button is located.

## Acceptance Criteria
Special Permission added / programmed against the Settings button on the End of Day page.

If you have the permission - you can see the button, and use it.

If you don’t have the permission - you cannot see the button, and cannot use it. 

* There should not be a blank space where the Settings button is otherwise located

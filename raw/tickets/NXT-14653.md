# NXT-14653 — Accountability - Menu Grids - Configuration for expanded grid size (16x15)

## Description
As a user I want to configure my menu grid for up to 16x15 sizing so that I can fit more items within my grid on the POS for those campuses that serve a large amount of items.

Previously had this set as a goal, but had to remove due to inability of the chosen library to support it. Functionality needs to go in tandem with the POS capability to display whatever is configured.

## Acceptance Criteria
* Ability to configure up to 16 (columns) x 15 (rows)

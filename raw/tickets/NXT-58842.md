# NXT-58842 — Showcase Widget

## Description
As a cafeteria manager, I want to use the Showcase widget to highlight a single menu item with visually engaging elements on a digital menu board so that it draws attention without requiring interactivity.

Acceptance Criteria:

* *Widget Creation*:
** Add a new widget type called *"Showcase"* to the menu planning widgets.
** Selecting the widget prompts three dropdowns for configuration:
**# *Serving Line Dropdown*: Users can select a serving line (e.g., breakfast, lunch).
**# *Menu Item Category Dropdown*: Users can filter menu items by category (e.g., Entrees, Sides).
**# *Item Dropdown*: Users can select a single item from the category to showcase.
* *Calorie Display (Orange Circular Element)*:
** The calorie count is displayed in a *circular container* positioned near the bottom-right of the item image.
** *Styling*:
*** The container is *slightly rustic* with a gradient effect for depth (e.g., radial gradient from light orange to darker orange).
*** Bold, white text is centered inside the container to display the calorie count (e.g., "300 Cal").
*** The element is sized to stand out but remain proportional to the overall widget (e.g., 60px diameter for standard screens).
*** User can change the color of the container. 
** The container blends seamlessly with the item image and background, with no harsh borders or shadows.
* *Text Display*:
** Provide a text box above the menu item name where the user can enter some text like “Today’s Special” (default)
** *Styling*:
*** The container is *slightly rustic* with a gradient effect for depth (e.g., radial gradient from light orange to darker orange).
*** The font elements for this should be same as the components in other widgets. 
** The text blends seamlessly with the background, with no harsh borders or shadows.
* *Widget Display*:
** The widget dynamically displays the following details:
*** *Item Name*: Large, bold text at the top of the widget.
*** *Serving Size*: Smaller text displayed below the item name (e.g., "300g").
*** *Food Components*: Represented by small icons arranged in a row below the serving size.
*** *Allergens*: Represented by industry-standard icons (e.g., peanuts for nuts, wheat for gluten).
*** *Item Image*: Displayed prominently in the center of the widget, with no opacity or transparency adjustments, ensuring it appears flat and natural against the background.
*** *Calories*: Displayed in the orange circular container near the bottom-right of the item image.
* *Seamless Integration*:
** The widget’s design blends into the existing background:
*** The item image appears flat with no transparency or shadows.
*** Text elements and icons are styled to complement the menu board aesthetic (e.g., clean, modern fonts and colors).
*** The calorie container is styled to stand out while maintaining harmony with the overall design.

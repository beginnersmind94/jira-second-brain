# NXT-5389 — Acct Mgmt -Transactions - Items

## Description
*User Story:*

As a manager I need to be able to see, add, delete or modify the items sold to a patron.

*Requirements:*

* List Items sold
* List items available to sell
* Show price of items
* Able to add a new item
* Able to remove an item
* Able to change the item qty sold
* Calculate extended price
* Calculate total tax, total sold
* Modifications to menu items are recorded on the original date of the transaction.
* Cost or price changes are recorded on the date of the modification. 
* Transaction Detail types

## Acceptance Criteria
The quantity can be edited.
If the Quantity = 0 the item is removed.
The extended price is calculated Qty * Price
New Sale Amount is the Total of ALL Extended Prices.
An Item can be deleted by checking the trashcan next to the item
A new item can be added by selecting the Add button.
***If a reimbursable meal has already been sold and the item being added is a meal then it is entered as a 2nd Meal and charged at the 2nd meal price.***
When the add button is selected a popup is shown. (See attached for format)
The contents of the popup is Menu Item Category dropdown (Default = ALL), a selectable/scrollable alphabetical list of active items 
When a Menu Item Category is selected only the active items in the selected category are listed
When an Item is selected the Price is loaded, based on price type and person type.
The quantity is initially set to 1 and the Extended price calculate,
When saved the item is added to the transaction list and control returned to the transaction detail page. (The item is saved locally) 
When the items sold is changed a new amount is entered in the New Sale Amount under Items, Tax is also calculated if needed. (Tax is only calculated on Taxable Items for Adults)
A new total is calculated.

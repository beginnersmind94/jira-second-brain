# NXT-14811 — INV - Order Template Details - Items tab - View Items to Add

## Description
As an inventory Operations Manager I want a page containing all eligible items that could be added to a template so I could search and find new items to be added to my Order Template.

This screen is accessed via the ‘Add Items’ button that is presented on the Order Template Details screen ([NXT-14810|https://primeroedgenext.atlassian.net/browse/NXT-14810])

*Requirements*

* Provide new screen within the Items tab of the Order Template List page, after user clicks ‘Add Items’
* Provide a header section containing filter dropdowns ([NXT-16111|https://primeroedgenext.atlassian.net/browse/NXT-16111], in next sprint)
* Provide a searchable grid containing a list of items
** only populate items which belong to the site (have site-item record)
** do not show items that are already assigned to the _selected_ template (items assigned to other templates can still be shown)
* Searchable/sortable item list will contain the following attributes:
** Item list should use paging control
** List columns:
*** a select column with a checkbox for each item
*** Item Number
*** Item Name
*** Storage Category
*** Item Category
* Action buttons ([NXT-15924|https://primeroedgenext.atlassian.net/browse/NXT-15924])

This story’s scope: Loading items and applying sorting/filtering

## Acceptance Criteria
* Add Items screen is accessed by clicking the Add Items button on the Template Details screen
* User can use header dropdowns to filter the list of items
* User can search for items that are configured as orderable for their site
* User can search items via searchable column headers

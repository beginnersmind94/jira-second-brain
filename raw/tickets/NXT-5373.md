# NXT-5373 — Deposits - Deposit Details - Checks

## Description
*User Story*

The user needs to be able to see and edit the checks entered on the deposit slip. 

*Mock-up*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5834%3A730&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5834%3A730&scaling=min-zoom]

*Requirement*

* All the checks that are in reconciled sessions for a site for the date of the deposit slip are automatically entered.
* A user can add a check
* A user can delete a check
* A user can edit a check
* The total of all checks is entered in the Checks entry in the Summary Section and added to the Deposit Total

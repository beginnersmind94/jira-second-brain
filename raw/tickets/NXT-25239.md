# NXT-25239 — INV - Production Ordering - Create Order - Include secondary prep sites

## Description
Raymond is a site Operations Manager who needs to make sure that his production orders will also include his items for the secondary school he will be prepping for.

**Note:* this remains in the parking lot until Production module completes Prep Sites.

## Acceptance Criteria
Production orders take into consideration prep site functionality

Prep site will have a consolidated planned/shipped count to use for production orders

A total count for each menu item is given (planned/shipped), there it is broken down into how much of each item in the menu item is needed to be ordered in the same fashion a normal production order will.

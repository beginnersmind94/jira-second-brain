# NXT-84 — POS Daily Checklist - Area Manager / Central Office Admin

## Description
As a Central Office Admin / Area Manager I want to see how all sites are performing with regards to completing their daily checklists.

This is tied to [https://primeroedgenext.atlassian.net/browse/NXT-2748|https://primeroedgenext.atlassian.net/browse/NXT-2748|smart-link] as a higher level view.

*Requirements*

Provide a page where:

* User is able to see all sites they have access to.
* User can see how far along each site is towards completing their daily goal.
* User sees a green check mark if a site has completed their daily goal.
* User can click onto any site to navigate to that site’s daily checklist.
* User is able to say that they’re ‘all done’ once all sites under them are completed.
* User can change to a weekly or monthly view to see how all sites under them have performed.
** May be something as ‘simple’ as a grid showing each site on the vertical axis, and dates on the horizontal, and green check marks or red X’s for each day.

## Acceptance Criteria
User has a page where they can see all sites they have access to, and each site's progress towards completing the daily checklist.

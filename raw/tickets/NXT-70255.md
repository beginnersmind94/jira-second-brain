# NXT-70255 — SCTV+ - Automated Schedule Playback

## Description
As a Cafeteria Manager,
I want Schoolcafe TV+ to automatically display the correct template whenever the TV is turned on and a schedule is active,
So that the screens operate perpetually, day-in and day-out, without my staff ever needing to interact with the hardware or manually reload the application.

## Acceptance Criteria
h3. Seamless Auto-Launch Handoff: 

  - Upon device power-on, the application must auto-launch and immediately check the schedule and display the active content.

  - It must not land on a screensaver or idle page if a schedule is active and menus are published for that day/meal.

h3. Persistent State Management: 

  - The application must remain in the “Playing” state as long as a schedule is active, regardless of how long the device has been running (e.g., 24/7 operation or daily power cycles).

h3. Automated Recovery: 

  - If the application encounters an error or network interruption, it should play the last available data where possible.

  - It must attempt to retry and resume the scheduled content automatically, without requiring a manual refresh.

h3. No Manual Reloads: 

  - The “Reload Menu” button is for manual interventions only.

  - Standard operation must never require a user to press it to resume playback.

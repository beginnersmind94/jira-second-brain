# NXT-37589 — IM - Items - Create Item - Item Type tabs

## Description
As an IM user creating or managing items I want a better way to manage all the various item sub-types, so I can only manage the use cases I need depending on the specific item, recipe, etc.

*Requirements*

* +Create Flow+
** Replace the “question bar” with a tab bar that has selection options/circles
*** these selection circles would turn into the status icons after the toggles are saved
** Update to remove ‘Food Item’ question (this is moved to Units card)
** Capture existing item sub type flags (in this order)
*** Single Item Recipe
*** Menu Item
*** POS Item
** some tabs (POS, Menu Item) are determined by user input/selection, while some tabs like Inventory Info (renamed from Product Info) are determined by Inventory readiness
* +Edit/Update+
** Update tab bar to enable dynamic tab population
*** only present the tabs which are applicable/enabled for the item
*** Update tabs and names
**** Rename Food Info to ‘Nutrient Info’ (just like recent Recipe redesign)
**** Single Item Recipe will be on its own tab now (not shared with Menu Item)
***** [https://primeroedgenext.atlassian.net/browse/NXT-37086|https://primeroedgenext.atlassian.net/browse/NXT-37086|smart-link] 
**** Menu Item and POS Info will be combined into a dynamic hybrid tab
***** if both are applicable, it’s a combined tab ‘Menu & POS Info’
***** if only one is applicable, it’s only that item type for the tab (POS Info)
***** Menu Items will be able to have POS enabled easily on the same tab (as part of the *future* Menu Item consolidation epic)
**** Rename Product Details to ‘Inventory Info’
***** Note: this tab is still not usable yet, until Substitutions is finished or additional functionality is built
** Provide a menu icon on the tab bar to open the new Item Type slideout 
*** Show enable toggles in the Item Type slideout menu
*** Include additional item controls here as needed
**** *Note: this section and these toggles will be used in the future to also deactivate/disable item sub-types or features
** Replace tab status banner with status icon 
*** Red/Yellow/Green concept remains (Incomplete, Partial, Complete)
*** use tooltips to convey status details
* Order the tabs for view mode
** Nutrient Info
** Single Item Recipe
** Menu & POS Info
** Inventory Info

# NXT-37346 — Student Import – Exceptions Tab

## Description
As a SchoolCafé Admin I want view the exceptions so that I can quickly find any import failures.

*Flow:*

* User logs in and browses to System > Imports & Exports > Import Monitor
* User clicks view log and taken to view log screen
* User clicks the Exceptions tab
* User views exceptions

*Notes:*

* We completed the story for marking rows to have exceptions so now we want to display the exceptions as a part of the history of the import

*Requirements*:

* For the import details page where we see the log and history, do the following:
* Add a tab next to History tab
** Title: Exceptions
* Add a grid to the body of the tab
* Make it look consistent with the other grids
* Columns:
** Student ID
** Exception Message
** Any other field we have that would be helpful
* Grid should have pagination, column sorting & export to excel

## Acceptance Criteria
* Tab exists with right title and UX
* Grid shows all the exceptions and nothing else with the required columns
* Grid has pagination, sorting and export to excel

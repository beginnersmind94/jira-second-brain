# NXT-60311 — Digitize AI - Document upload action options

## Description
As a user I want to have the option to save or discard my Digitize AI documents/images so that I can be ensure this action is intentional and not updated automatically.

* Discard option is available for the user to remove the uploaded/captured document/image. The discard will simply remove the document from the list that is uploaded.
* Save option is available for the user to “accept” the uploaded/captured document/image. To be more specific, we can be specify the button name for each form type (ex. “Send to Production” is the save action for Production Record forms). If we do not want to make this specific, it is also acceptable to use “Accept” as the text for the button.

## Acceptance Criteria
* Provide option to Discard a successful upload/image capture
* Provide option to Save/Accept a successful upload/image capture

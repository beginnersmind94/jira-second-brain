# NXT-3224 — Template Hub - Card View UI

## Description
*User Story*
As an Inspections user, I can view Templates on the Template Hub screen in Card View and switch between List View and Card View without any loss of information.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1|https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1]

*Requirements*

* User can toggle between Card View and List View when viewing Templates
* Toggle is done through the buttons on the top right
* List View is the default
* Card View
** Templates are visible in rows of three cards
** Each card has a fixed display that cannot be edited
* Fields
** Inspection Name
** Questions
** Last Updated
** Last Updated By
** Actions
** Scoring
* Actions are available but are placeholder for the purposes of this story.
** Copy for Active and Inactive
** Edit for Inactive
* Default view is Active
* Default sorting is by Template Name

## Acceptance Criteria
* Card view is present in a visual format

# NXT-70661 — GC- PE -  Allergens column in search results grid should consider the flag for  "None Indicated for Allergens"

## Description
GC- Allergens column in search results grid should consider the flag for  "None Indicated for Allergens" 

this story covers PrimeroEdge View, export, and import options for allergens when allergens present or none indicated for allergens.

SC story

GC portal:

## Acceptance Criteria
Scenario:

if Menu planning > ingredients> ingredients page> ingredient details:Allergen tab: check box “None indicated “ is selected for an ingredient 

that means user has reviewed the item and it’s allergens and ingredient has no allergens so, we need to say “None indicated” flag selection a valid Allergen information. So, we need to display as “Yes” in allergens column of item catalog grid results.

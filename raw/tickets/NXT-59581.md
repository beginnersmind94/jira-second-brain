# NXT-59581 — IM - Menu Items - VMI - Ability to save no contributions 

## Description
As a menu planner, there are times where I would like to use the VMI functionality for menu items like condiments where I would like the options available but they do not have contributions.

## Acceptance Criteria
* Allow user to either not select contributions, or give an options for “No contributions indicated”
* Filter menu items to add to VMI only by menu items that do not have contributions indicated

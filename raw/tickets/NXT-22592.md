# NXT-22592 — INV - Templates - Add Item to Template - Enhanced Item Grid

## Description
As a FSD I want an enhanced item grid so I can see item hierarchies when adding items to templates.

*Note:* UI Components team has added new [TreeList grid component|https://devuicomponents.perseusedge.com/?path=/docs/components-treelist--default]. This grid has indentation and expand/collapse properties that we will use to show item hierarchies more effectively.

*Requirements*

* Update the item list grid on the Template - Add Item screen (for both Order and Withdrawal templates)
** *Note:* some grid functionality may not be fully available, such as checkbox option
* Implement the indentation for the Item # column
** this will will indent for each child item in a hierarchy
* Apply hierarchy sorting
** Child items should be displayed within the same hierarchy grouping
** the Item # should be indented for each child item in the grouping
* Minor UI changes
** Item Name > Item Description
** add Unit Type after Item Description
* Enable expand/collapse
* Enable typical sorting
** sorting only applies to top-level items
* Enable Field Search
** if search results return a child item, it will be displayed in the context of its hierarchy, up to the parent item

# NXT-67401 — LMA - License Keys - View License

## Description
As a Customer Support (CS) role member, I would like to view the license info at any time so that I can see what was included regardless of status.

*Flow:*

* CS role user browses to System > License Management
* User sees a list of licenses generated for the software
* User clicks View License on a row
* View License drawer opens and views sees history



*Requirements:*

* On License tab grid, do the following:
* Grid Actions column:
** View License
*** On click:
**** Open the View License drawer
**** Enabled for all statuses
*** Customer Support role and License Management role can view this drawer

 

*View License drawer*

* Just reuse the Edit drawer to save coding effort
** Title: View License
* If coming from View License click, disable all fields

## Acceptance Criteria
* Users can view the license key regardless of status but cannot edit anything

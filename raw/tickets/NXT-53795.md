## Acceptance Criteria
* Provide a banner for Menu Item readiness, much like the Inventory Readiness banner
** the banner should match Menu Item sub-type status
** set the banner (and status) to Incomplete if the user selected DMI Yes on the first step
** Show the new Menu icon with relevant status icon
** reset the Menu Item to ‘Disabled’ status if the user Cancels editing without populating the required fields
* Clicking the banner will provide a creation prompt
** “Enable this item as a Direct Menu Item?” with X and Continue buttons
** X will close the prompt
** Continue will proceed into Menu Serving selection 
* Provide a Menu Serving selection mode
** if only one serving exists, auto select it
** Provide a clickable Menu Serving identifier button
** Allow multiple servings to be selected
* Display a confirmation line in the footer with the Confirm button
** adjust message to reflect whether single or multiple servings are selected
* Upon MI creation, show the Menu Serving identifiers

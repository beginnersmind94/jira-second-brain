# NXT-24846 — INV - Reports - Add Descriptions for all Reports  - UI Only

## Description
As an Inventory user I want a description for every report on the Reports page so I know which report to open.

*Requirements*

* Add descriptions for every developed report 
** Expected Receipts - Lists items to be delivered by a selected vendor on a specified date
** Price Details - Provides the batch level pricing details for items on each transaction
** Back Orders - Lists all ordered items that have not been received from external vendors
** Print Orders - Provides a list of open orders in report format 
** Ordering Guide - Lists all orderable items for a site
** Purchases - Details the on-hand counts and values of items for each vendor for a selected date range
** Batch Detail - Provides batch level information for on-hand inventory items
* Remove reports which have not yet been developed
** Note: we may want to change the display order of the reports after we remove all invalid ones

## Acceptance Criteria
* Provide descriptions for every valid report
* Remove invalid reports from the page

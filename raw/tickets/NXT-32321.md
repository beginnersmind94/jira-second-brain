# NXT-32321 — Insights Data and Execution pipelines - Process definition

## Description
{color:#0747a6}Record all steps required to process Insights pipelines for Dev and QA environments.{color}

*_Pipeline execution time:_* Execution time shall be no greater than 10 minutes. for initial loads and larger files- 60 minutes shall be the maximum lead time.

*_Edge use cases:_* Process shall cover edge use cases - for example certificate failure/recovery, API refresh failure/recovery, file conversions failure/recovery, and other as needed.

## Acceptance Criteria
*_Acceptance criteria:_* Once data files (Initial/Delta) placed in source / landing folders, automated pipelines shall execute and complete. Insights UI shall display updated data.

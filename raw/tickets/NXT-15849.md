# NXT-15849 — IM - Display Product Info 

## Description
As a user I would like to view the Product info of an Item So that I can view and Edit the information

## Acceptance Criteria
Given the User is on the Items Page 

And a search is applied 

And list of items are available in the Search result 

When an Item is selected from the search result 

Then display the Product info of the item 



* Item Category 
* Storage Category 
* Purchase/Whole Units 
* Broken Units per Whole Unit 
* Broken unit Description 
* Broken Unit Weight in Grams 
* Broken Unit Size
* Fair Market Value/Standard Price  
* Valuation group 
** USDA Foods
* Buy American 
** Upload option for Exemption Letter
* Orderable in Broken Units 
** Toggle 
* Internal Vendor 
** Yes/No options to select 

And ability to “Save” any changes made 

And “Cancel” to discard the changes 

And exit the Edit mode

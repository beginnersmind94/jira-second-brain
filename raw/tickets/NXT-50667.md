## Acceptance Criteria
* Review the CN27 files for any changes and impact based on those changes
* Action items based on USDA guidance
** Continue to identify and tag discontinued CNDB items using the discontinued date in the Status field. With CN27, the Status field in the Food Description (FDES) table will change to a date format and be renamed "Discontinued Date". Items with a discontinued date populated will remain in the software for one year and then be removed to be consistent with our prior method of marking items “discontinued”. Continue to use the same process for removing CNDB items from your software. The Discontinued Date field (formerly Status field) will appear only in the FDES table starting with CN27. The Status field will be removed from the other CN27 tables.
** Adjust your software, where appropriate, to accommodate the new Source Code fields for the Nutrient Values (NUTVAL) and Weights (WGHT) tables. This field will be added to these tables in CN27. The software is not required to display the Source Code to users but may show this information as text or a numerical code (with a key).

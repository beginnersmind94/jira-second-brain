# NXT-4269 — AcctMgt - Add Patrons - Add Student Long Form

## Description
*User Story*

As a district admin I want to add all the information on a student at one time without having to follow-through into editing the patron.

*Requirements*

* We’re going to add additional sections to the Add Student drawer so that a user can accomplish the entire setup in 1 page.
* A lot of this is taking the ‘edit panel’ sections from the Patron View and adding them on as collapsible sections for data entry.
* Sections:
** Change the Contact Information to Guardian and Contacts.
*** User should be able to add additional guardians similar to the Edit on the panel.
** Add a Household Information section.
*** Household Pin is available for entry.
*** Additional household members can be added with look-up functionality.
** Add a Benefits section
*** Placeholder for now. Collapsible.
** Add a Documents & Notes section.
*** User can upload/remove docs, add/remove comments,
** Allergies & Medical
*** User can set allergen information.
*** User can add/remove Medical documents, medical comments.
** Restrictions
*** User can set account restrictions per the Restrictions panel.
*** Spending Limits
*** Menu Item Restrictions

## Acceptance Criteria
User can fill out the entirety of student information in 1 form.

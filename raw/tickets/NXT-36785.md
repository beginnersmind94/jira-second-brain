# NXT-36785 — PR- Plan- Add menu item pop up menu item type and variety menu item preview  (API AND UI) 

## Description
When trying to add menu items to the production plan page, the user does not know which menu item is a variety menu item vs a regular menu item, They also would like to see the meal contributions, serving size and the allergens associated with a particular item.

## Acceptance Criteria
* Add a Menu Item Type Column to display the menu item types
** User should be able to search between the types
* Change the Menu Item Column to say Menu Item Description
* Add Columns for Default Serving Size, Meal Contribution, Allergens
* Display order of the columns should be Category, Menu Item Description and combine Recipe Code with link, Default Serving Size, Meal Contribution, Allergens, Menu Item Type
** Link to Recipe code should take a user to a direct menu item to the item details page and land on the recipe/menu item tab
* Display a preview for the user to know what items their are in a variety menu item (Same as the add menu Item in menu planning)
* Keep the grid the same and allow user to continue adding counts and change serving sizes

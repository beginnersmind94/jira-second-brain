# NXT-18131 — Platform - AI/ML - Audit Device History

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to see detailed audit trail made on specific device starting from adding new device till device deletion.

*Mockup:*

[https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|https://www.figma.com/proto/tpwyiu5Ynk6Bh0cX0tsG7Q/Systems-3.0?node-id=7%3A3203&scaling=scale-down-width&page-id=0%3A1|smart-link] 

*Requirements*:

* Admin logs into PrimeroEdge portal under device management, each device should contain “History” tab when they click either on HISTORY button from the device catalog page
* History should display all events occurred on the device loads in a page (not a drawer) with pagination 
* Audit table should contain below columns
** Event Name
*** <List of events>
** Type
*** ADD
*** DELETE
*** UPDATE
*** AUTHENTICATION
** Updated By
** Timestamp
* All events should be recorded with detailed with description of event
** Ex:  change of Serial number,  Device suspended, Device has been re-assigned to new site
* Delete device should also be recorded and persisted in the database

[https://www.figma.com/proto/DUQFqou21xSX4dT0VAZwhA/DeviceManagement_-NXT-18131_JS?node-id=1%3A380&scaling=min-zoom&page-id=0%3A1&starting-point-node-id=1%3A380|https://www.figma.com/proto/DUQFqou21xSX4dT0VAZwhA/DeviceManagement_-NXT-18131_JS?node-id=1%3A380&scaling=min-zoom&page-id=0%3A1&starting-point-node-id=1%3A380|smart-link]

## Acceptance Criteria
* User shall be able to see detailed audit trails of the device based on event type
* User shall be able to see event name of specific event type add/delete/update persisted in database for reporting purpose
* User shall be able to see the device history tab when they click on “history” button from device catalog page
* User shall be able to monitor authentication changes to trigger notifications to admins (future)

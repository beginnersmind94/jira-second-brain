# NXT-45937 — Imports - Persist Filter Values on Import Monitor After View Log

## Description
As a SchoolCafé user I would like it if the Import Monitor would persist the filter selections and grid view when I click on View Log and come back so that I don’t have to redo my filters

*Notes:*

* When a user clicks on View Log, it opens that overlay and then when they click back, the Import Monitor has reset the filter values and grid
* We want it to persist the selected values and grid filtering

*Requirements:*

* This functionality applies to Import Monitor
* When user clicks View Log and then clicks back, retain all the filter selections and grid the way it was
* Refresh button and the tool tip control should not be visible when the Import/Export status is Completed or Cancelled or Error.

## Acceptance Criteria
* When user clicks View Log and then clicks back, it retains all the filter selections and grid the way it was

# NXT-29108 — IM > Method to filter "All Food Item"

## Description
Requirement: Provide a method to request/filter the “All Food Item” Report to produce results based on Descriptions, Item #'s, Item Codes, or Category entry (specifically, if I type in the word soups and then hit enter or tab out of the current “Tags” search, it does not actually select the tag, thus giving no results on the search.) 

The user must be able to include multiple food items in this report.

## Acceptance Criteria
Requirement: Provide a method to request/filter the “All Food Item” Report to produce results based on Descriptions, Item #'s, Item Codes, or Category entry (specifically, if I type in the word soups and then hit enter or tab out of the current “Tags” search, it does not actually select the tag, thus giving no results on the search.) 

The user must be able to include multiple food items in this report.

* Use OR condition for multiple selections
* Item Description shall show tool tip with full Item name
* Criteria is optional
* Search shall work for Item Code and Item Name
* Item Code - Item Name shall be presented under Search criteria.
* Item Criteria and Tags shall be expanded to accommodate the higher number of entries possible
* If not selection the generate report the ALL print
* Include Active & Inactive items in the search

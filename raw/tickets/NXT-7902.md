# NXT-7902 — INV - Physical Inventory - Review Screen - Cancel Inventory (Pending Reconciliation status) 

## Description
As an inventory user I want an action bar on my physical inventory summary/review screen to complete the necessary actions on my inventory.

This action bar should be accessible from the Summary tab/section on the Review Screen.

Provide Access to:

* Actions (permission based)
** Modify Physical Inventory - different story
** *Cancel Physical Inventory*
*** this will prompt the user to cancel the inventory and invalidate all of the item counts
*** this changes the status to Cancelled
*** if user clicks Cancel, prompt user to enter comments
*** user remains on the same page but header changes to Cancelled information
** Reconcile Inventory - different story

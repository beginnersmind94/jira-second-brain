# NXT-65345 — WS - Benchmark - Sum count and dollar daily values-SC

## Description
As a Workspace user, I want to be able to see a dynamically calculated value for count and dollar benchmark amounts, so that I can get correct amounts for all time frames.

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* Update the benchmark calculations for count and dollar
** the benchmark screen should show the daily expected value
** the cards should show the dynamic value using the daily value
*** Currently shows the same numeric value no matter what time frame
** Ensure uses only operating days
* -Put a message on the benchmark screen:-
** -For number (#) and dollar ($) fields, enter the daily value expectation (n per operating day or $ per operating day). Insights will calculate the expected benchmark based on the selected timeframe using these daily targets. For example, if you set Revenue to $15 here and you select 1 week as your timeframe on the KPI page, the Revenue card will show as $45 benchmark for that week#-
* For the Revenue, Waste, Meals and MEQs columns, put a hover over tool tip: Enter the daily expected value like you just did for Revenue

## Acceptance Criteria
* Benchmark config screen shows message
* Cards show correct benchmark value dynamically according to the timeframe selected

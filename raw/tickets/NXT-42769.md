# NXT-42769 — POS - Use State Eligibility for Student Meal Pricing

## Description
As Emma the Eligibility Officer I want to see State Eligibility used at the POS so eligible students have free or reduced meal pricing for NSLP and SBP.

*requirements*

* If the system setting is turned on, POS needs to charge eligible students appropriately:
** Student’s State Eligibility is Reduced, use the Reduced meal price
** Student’s State Eligibility is Free, use the Free meal price
* This applies to student meals only, and for NSLP (Lunch) and SBP (Breakfast) meal types only.
** A la carte and second meal sales must use the USDA Eligibility.
* Additional rule needs to be added for all POS sale formats:
** Detailed sales, bulk sales (roster, homeroom)

## Acceptance Criteria
* POS updated to use State Eligibility.
** State Free → Free meal pricing
** State Reduced → Reduced meal pricing
* Changes made for NSLP and SBP meal types only.
* Does not impact a la carte purchases / second meals.
* Must be a reimbursable meal.
* Charge Limits should interact with these updated meal prices accordingly.
* Must work for both detailed sales and bulk sales.

# NXT-3633 — Edit Inspections - Static Table Question Type

## Description
*User Story*
As an Inspections user, I can see Static Table Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Static Table Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Table with all rows, columns, and content
** Whether question is required
** Notes Section
** Upload Media
** Note: Point values are *not* displayed here
* Each segment follows the proper rules for user response
** Text
** List
** Rating
** Date/Time
** Signature
** Location
* Choices are properly saved
** If question is required, Inspection cannot be submitted if no selections are made
* Choices can be changed if Inspection has not been submitted
* If any part has a score associated with it, the score is properly updated according to the selection being made.
** Rating or List follow the same rules as the individual Rating and Dropdown question types

## Acceptance Criteria
* Static Table question appears in Inspections
* All segments can be selected, saved, and edited
* Score updates properly if present

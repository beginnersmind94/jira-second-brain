# NXT-61509 — INV - Distribution - Assign & Pick - Assignment Details for Substitutions

## Description
As a Distribution Manager I want the ability to assign quantities to substitutions alongside the items for which I am substituting, so I can make informed decisions about which site receives which products.

* Single row being edited
* All other actions disabled

## Acceptance Criteria
* Modify the assignment details slideout for items with substitutions added
** ensure slide-out sizing is dynamic and large enough for max 3 item columns
** Ensure all sites assigned to the distribution are always displayed in the slide-out, regardless of Order Quantity
** Put the sites with zero order quantity below the ones who ordered
* Insert a Substitute beneath the Item label for each substitution
** if multiple: Substitute 1, Substitute 2
* Provide a Delete icon next to the Substitute label
** Prompt if no assigned quantity: “Are you sure you want to delete this substitution from the distribution?”
** Prompt if some assigned quantity: “This substitution has assigned quantity. Are you sure you want to delete it from the distribution?”
* Add an extra Quantity column for each substitution (limited to 2)
* Move QOH from the header area to the grid
** display under each Quantity column
*** display the original item under the Assigned Qty column
* As substitutions are assigned quantities, associate them with the original item for the purpose of swapping or adding to the site order receipt

# NXT-29878 — PR- Production Plan- Menu Item on MP serving group set as NOT Served 

## Description
As a menu planner, when I set a menu item serving group as not served on the Menu Planning side, I am expecting that we are not allowed to serve it to that serving group on the production plan side also.

Currently, We are allowing the user to enter in a count for planned counts on production even when item is marked as NOT SERVED on MP.

## Acceptance Criteria
When menu item is marked as NOT SERVED on MP for one or more serving groups, display NOT SERVED on the production plan as well. 

Do not allow the user to enter in a planned count for that particular serving group

If menu is changed on MP, affect all plans that have that menu that are in PLAN INCOMPLETE Status

PLAN COMPLETE STATUS if change happens on MP, enter 0 planned count and serving size associated with item

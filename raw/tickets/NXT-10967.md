# NXT-10967 — [TaggedDashboard] - Dashboard (Insights) - Eligibility - Application Counts

## Description
As a district admin I want a chart to see the total number of FRE applications and their entry methods.

*mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=429018%3A0&scaling=scale-down-width&page-id=84%3A0|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=429018%3A0&scaling=scale-down-width&page-id=84%3A0|smart-link] 



*requirements*

* User has a chart on the Insights page that shows them the total number of applications.
** This is broken down by entry method. A donut chart would be good for this.
* User can also see how many of those applications are still active.
** An application is considered ‘active’ if at least 1 student is still gaining eligibility from it.
* -User has an area where they can see what percentage of Paid students have not yet applied for benefits.-

## Acceptance Criteria
User has an area that shows them the total # of FRE applications received for the current academic year.

User can also see how many of those applications are active - at least 1 student is still gaining eligibility from it as of the current date.

-User has an area where they can see how many paid students have not yet applied for free/reduced benefits.-

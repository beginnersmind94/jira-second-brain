# NXT-14177 — IM - Catalog Integration - Item Selection

## Description
As a User I would like to have an ability to select items from the Catalog list So that I can Review them before I can do the Import

## Acceptance Criteria
Given the User has already searched for Items 

When the list of items are available in the Search results 

Then the User should have the ability to 

* Check and Select the individual or Multiple Items 
* And ability to paginate between multiple pages of items 
* And Ability to change the pagination limit for the item view

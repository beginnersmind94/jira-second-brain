# NXT-54751 — AcctMgt/System - Student Import - Mapping for Preferred Name, Account Updates

## Description
As Dana the Director I want to update preferred name fields via the automatic import process so that students are addressed by name.

*requirements*

* System - Student Imports (Flat file)
** Add a mapping field for Preferred Name.
* AcctMgt - Update the Preferred Name field if it is present on the file.
** Track the update under Account History.

Discussion at Refinement - 9/16/24

* If the field comes in blank:
** Do not update the existing PreferredName.
* User can still change the field (and update it to a blank value) via editing Account Details.

## Acceptance Criteria
* User can see and map the Preferred Name field via Student Imports (flat file as well if possible)
* User can see updates to this field via Account History
* Account Details updated as part of import process.
** If the field is left blank in the import file we will not update the PreferredName field.

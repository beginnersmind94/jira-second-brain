# NXT-51124 — POS - Dashboard - Option to disable display of stats

## Description
As Dana the Director I want to control what stats are shown to cashiers to prevent oversharing and keep the ‘blind close’ intact.

*mockup: n/a*

*requirements*

* Add POS Site Settings (System > Site Settings > POS Site Settings) to control visibility of the new Stats sections.
*# POS Stats - Show Daily Stats
*## This should control whether the entire Daily Stats section is available at the POS. If it’s on, show the section. If it’s off, show nothing in that space.
*## Default: On
*# POS Stats - Show Payments section in Daily Stats
*## This should control whether the Payments stat is shown within the Daily Stats section at the POS. If it’s on, show the section. If it’s off, show nothing in that space.
*## If ‘Show Daily Stats’ is off, then the Payments section should also be considered off.
*## Default: Off
*# -POS Stats - Show Cashier Impact Stats-
*## -This should control whether the Your Impact This Year section is available at the POS. If it’s on, show the section. If it’s off, show nothing in that space.-
*## -Default: On-
* Update POS to use these site settings.

## Acceptance Criteria
* User can control visibility of the following sections at the POS Dashboard via Site Settings > POS Site Settings.
** Daily Stats
** Payments under Daily Stats
** -Your Impact This Year-
* POS checks the new POS Site Settings and displays sections accordingly.

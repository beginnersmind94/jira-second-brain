# NXT-4898 — Authentication - Password Expiration

## Description
*User Story*
As a PrimeroEdge user, my password automatically expires after a certain length of time.

*Requirements*

* Passwords expire after a certain length of time
** Default is dependent on user setting
** Configuration is covered in [NXT-4053] and [NXT-4070]
* An email is sent to notify the user both when there are 3 days before their password expires and when their password has expired.
** Email contains a link to create a new password
** Link expires once the password has been reset
* Once expired, the user is required to reset password after their next login
** Password must be changed for user to be allowed to log into the system

*Notes*
This covers USDA requirement NR62

## Acceptance Criteria
* Passwords can expire after a certain amount of time and user is prompted to change their password
* User cannot log in if they do not reset their password
* User is notified through an email when their password expires

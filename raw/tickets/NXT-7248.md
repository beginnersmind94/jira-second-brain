# NXT-7248 — Menu Grid Menu Items and Categories

## Description
*Story*

As a user I need to be able to create menu grids for use on ExpressPoint. There are two types of menu grids, static which have menu items that don’t change and dynamic which has items that change based on menu planning. This story handles Static menu grids. 

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=29545%3A1331&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=29545%3A1331&scaling=scale-down-width]

*Requirements*

* Menu Items can be selected and dragged to the Menu Grid
* Categories can be selected and dragged to the Menu Grid
* Search text can be entered for Menu Items or Categories
* Menu Items can be filtered by Meal Type and/or Category

## Acceptance Criteria
Clicking "Menu Items" or "Category" switches between the two lists
The Menu Category is initially set to the Menu Category from Menu Info.
You can type a name to find a specific Menu item
You can select a POS Item Category to filter Menu Items shown
Meal Type can be selected to filter Menu Items
Meal Type and Item Category default to ALL
You can filter the Category list
The color displayed for a Menu Item is determined from the Color selected for its POS Item Category
Buttons have 3 colors a Fill Color, Text Color, and Border Color. 
The Fill color for a Menu Item is derived from the POS Item Category, Text Color default is white (There is a calculation that can determine the best contrasting color), Border Color defaults to Fill Color.
The Fill color for a Category by default is White, The Text Color default is Black, and the Border Color is the POS Item Category Color.

Only ACTIVE Menu Items are shown in the Menu Item List

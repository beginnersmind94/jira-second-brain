# NXT-4581 — AcctMgt - Special Accounts - Landing Page

## Description
*User Story*

As a district admin I want a way to search/view all special accounts - the patron accounts used for special occasions / charging.

*Mock-up*

TBD

*Requirements*

* Add another search page off of the Account Management lander called ‘Special Accounts’
** Search fields like the others, by name or by site.
** Blank search should be available.
** Search results should load in the split view as cards.
** Cards should show the following information:
*** Special Account Name
*** Site(s) it’s linked to.
*** Circle with initials of the Name.
** Text links to the search page should be present off the main Patron lander (another button on the top) and in the sidebar.

## Acceptance Criteria
User has a page dedicated to Special Accounts.
User can search for special accounts.
Search results are in split-view, cards show appropriate information.

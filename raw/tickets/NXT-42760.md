# NXT-42760 — Elig - Applications - Add State Eligibility to Applications Grid, Processing Sessions, Application Export

## Description
As Anna the Application Processor I want to see the State Eligibility status of applications for ease of use.

*requirements*

Using this newly added setting: Enable State Income Guidelines (Code: STATEEIG) and if it is set to YES:

* Forms > Applications Grid
** New column: State Eligibility
*** Expected values: N/A, State Free, State Reduced
** User can filter the column.
* Forms > Applications > Details > Export
** Add to Eligibility section
* Forms > Processing > Session Summary
** Add State Eligibility column with results
* Forms > Processing > Session Summary > Application Approval List
** Add State Eligibility column with results
** Print/Export should also contain values.

## Acceptance Criteria
If the System Setting is on:

* State Eligibility column + results added to:
** Forms > Applications Grid
** Forms > Applications > Details > Export
** Forms > Processing > Session Summary 
** Forms > Processing > Session Summary > Application Approval List
* Added to all Exports

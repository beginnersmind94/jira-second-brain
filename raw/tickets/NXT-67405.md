# NXT-67405 — Acc - Edit Checks - Updates for CACFP and CEP

## Description
As Dana the Director I want to process valid edit checks - and the way CEP is being handled creates additional work for my team as they are not valid. Also, I need to run these for CACFP.

Currently, Edit Checks do support CACFP meal types, but we may need to address how we are flagging these.

*Requirements*

* Edit Checks Logic Check:
** Only flag an edit check if it is >, not >=
*** Example: 100 meals, 100 eligible students, AF 95
**** We should only flag this for the Attendance Factor exceeding as 100 meals > 95 AF
**** We do not need to flag this for overclaiming (100 meals = 100 eligible students) - 100% participation is fine.
* Edit Checks for CEP/P2 Sites
** Only flag these for the Total meal counts, not for F/R/P
* Edit Checks for CACFP Sites
** Flag for all eligibilities, meal types, attendance, attendance factor, totals

## Acceptance Criteria
* All Edit Checks
** Only flag when the comparison number greater than, not greater than+equal
* CEP/P2 Sites
** Only flag on the Totals - total meal count, total attendance, attendance factor, etc.
* CACFP
** Same as SNP, but also the Totals

# NXT-70963 — PE - Insights - KPI - MPLH - Schoolie

## Description
*I am a* district user
*that I want to* have Meals Per Labor Hour (MPLH) included in Schoolie AI analysis
*so that I* can receive insights that reflect all relevant KPIs shown on the dashboard.

----

h2. *Notes*

* This story applies to *Schoolie AI integration within the Insights dashboard*.
* Schoolie is triggered from the dashboard and performs analysis on the *visible KPI cards* for the selected:
** Timeframe
** Site filters
* MPLH (Meals Per Labor Hour) is a *new front office KPI* that must be included in AI analysis when applicable.

----

h2. *Out of Scope*

* Any KPI other than MPLH

----

h2. *User Flow*

# A user logs into the system and navigates to the *Insights dashboard*.
# The user configures:
#* Timeframe
#* Site filters
#* Visible KPI cards
# The user clicks the *Schoolie AI button*.
# The system sends a prompt to the AI engine including the visible KPI data.
# The AI returns an analysis.
# The user views insights, including MPLH if it is part of the visible KPIs.

----

h2. *Requirements*

h3. *1. Include MPLH in AI Prompt*

* The backend must include *MPLH data* in the prompt sent to the AI engine.
* MPLH data must reflect:
** Current timeframe
** Selected site(s)
** Current KPI values

----

h3. *2. Conditional Inclusion Based on Visibility*

* MPLH must only be included in the AI prompt if:
** The MPLH KPI card is *visible on the dashboard*
* If MPLH is not visible:
** It must *not be included* in the AI prompt

----

h3. *3. AI Response Inclusion*

* When MPLH is included in the prompt:
** The AI response must include analysis related to MPLH
* When MPLH is not included:
** The AI response must not reference MPLH

----

h3. *4. Consistency with Existing KPI Handling*

* MPLH must follow the same prompt structure and handling as existing KPIs
* No custom or separate logic should be introduced beyond inclusion

## Acceptance Criteria
h3. *1. MPLH Included When Visible*

* When MPLH card is visible:
** MPLH data is included in the AI prompt
** AI response includes MPLH analysis

----

h3. *2. MPLH Excluded When Not Visible*

* When MPLH card is hidden:
** MPLH is not included in the AI prompt
** AI response does not include MPLH

----

h3. *3. Data Accuracy*

* MPLH values included in the prompt match the dashboard values for:
** Timeframe
** Site filters

----

h3. *4. No Regression*

* Existing KPI analysis behavior remains unchanged
* Schoolie continues to function for all other KPIs

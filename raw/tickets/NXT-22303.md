# NXT-22303 — Remove Fixed Width delimiter-Import Configuration

## Description
Currently in the Import File settings, we have options to import various formats of files. 

Requirement is to get rid of an existing format (fixed width delimiter).

## Acceptance Criteria
1. Review and Validate that there is no option - fixed width delimiter while importing files @ Import Configuration.

2. Review and Validate that the rest of the options are like excel, CSV and other file formats are still present.

3. 2. Review and Validate that the removal of the format does not impact rest of the file import process.

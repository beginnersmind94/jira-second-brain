# NXT-33232 — DataQuery Framework - Reports List Page and Behavior Improvement - For Shared Tab

## Description
DataQuery Framework - Reports page layout and behavior change

## Acceptance Criteria
* The “Personal” tab name shall be renamed to “My Reports”
* The “Shared” tab shall be renamed to “Reports Shared with Me”
* The “Reports” column header shall be renamed to “Name”
* The “Data Model” column headers hall be renamed to “Data Source”.
* The new “Name” column shall be the leftmost column in the table.
** This column shall be sorted ascending by default
* “Manage Views” shall be removed (The user does not need to modify the UI containing the list of reports.).
* User shall not be able to share a report with himself
* The Mousepointer shall be a hand when hovering over a report row
* See “Reports List UI Markup.pdf” for a UI markup containing more acceptance criteria.  The complete set of acceptance criteria shall be a union of the above text and the notes in these attachments.
* The value that the user selects for “Items per page” shall persist until the user changes it (persisting in browser storage is acceptable)

# NXT-39041 — Command Centre - Tool Tip for Sections (UI)

## Description
_(none)_

## Acceptance Criteria
*Navigation*: Insights - Command Centre 

The user should be able to see the following Tool Tip on mouse over on the headers:

*Program Access:*

“*Program Access %*” represents the percentage of students who has been enrolled into eligibility categories such as Free/Reduced/Paid

The Total % represents the students who are enrolled into Free + Reduced price types.



*Breakfast Participation:*

*Breakfast Participation* *%* represents the overall percentage of students who has participated on Breakfast Meal under different eligibility categories (Free / Reduced / Paid)

It’s calculated based on Total Number of Meals Served ÷ Total Number of Operating Days ÷ Total Number of students enrolled * 100  



*Lunch Participation:*

*Lunch Participation* *%* represents the overall percentage of students who has participated on Lunch Meal under different eligibility categories (Free / Reduced / Paid)

It’s calculated based on Total Number of Meals Served ÷ Total Number of Operating Days ÷ Total Number of students enrolled * 100  



*MEQ:*

Meal equivalents can be used as a benchmark for productivity and financial performance.
It is calculated based on type of meal (Ex: Breakfast) multiplied by the conversion factor (Ex: 0.67) as defined in Accountability Dashboard (Front Office Module)



*Revenue:*

Revenue represents the Total Revenue generated under “Reimbursable Meal” + “Non-Reimbursable Meal” categories



*Threshold Button:*

*Threshold* represents the targets that can be set against a district / site type / site level and check how the district / sites are performing against the defined threshold

The vertical bar (*|*) in KPI section represents that threshold targets set by the user. 



*Legend:*

The legend / color codes represents the performance of the district  / school types / sites against the defined threshold.
{color:#006644}Dark Green{color} represents *Excellent/High*
{color:#36b37e}Light Green{color} represents *Good/Expected*
{color:#ff991f}Amber{color} represents *Fair*
{color:#bf2600}Red {color}represents *Low*



*Show Visualization:*

Enable Pie Chart view to view the KPIs details graphically



*Show areas requiring attention:*

Check this to view the sites that are not meeting 2 or more KPIs (Ex: Lunch Participation)



Approach:

A common i icon that shows a slide out with all the content organized

# NXT-17459 — INV - Orders - Manage Orders - Order Detail - Prompt user on unsaved changes to Items

## Description
As an ordering manager I want to be prompted that I have unsaved changes if I attempt to leave the Order Details screen without saving any added items or updated item quantities.

If the user changes item order quantities (via the in-line editing), or adds items (via the Add Item button), these changes are not automatically saved.
The user is not alerted that their edits are not being saved, and there is no prompt when user navigates away from this screen

*Requirements*

* If the user makes changes to an order without saving the order itself, provide the *unsaved changes prompt* when they attempt to leave the page
* Research: display some indicator near the Save button, or elsewhere on screen, to indicate that there are unsaved changes

## Acceptance Criteria
* User should be alerted to unsaved changes on the Order Detail screen
* Research: visual indicator that there are unsaved changes

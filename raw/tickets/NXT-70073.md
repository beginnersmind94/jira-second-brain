# NXT-70073 — Sites Column Drilldown on Periods Page

## Description
Allow users to view site-level period status breakdown to identify which upstream periods are blocking Financial period closure.

## Acceptance Criteria
*Given* I am on the Periods page viewing Financial periods
*When* I see a period with a Sites column showing "X Open, Y Closed"
*Then* I can click the text to open a popover showing individual site statuses

*Acceptance Criteria:*

# Sites column displays format: "X Open, Y Closed" as clickable text (use link styling)
# Clicking opens a popover positioned below the cell
# Popover contains a table with columns: Site | POS | Inventory
# Each row shows:
#* Site name (left-aligned)
#* POS status: ✓ Closed or ○ Open
#* Inventory status: ✓ Closed or ○ Open
# Site is considered "Closed" only when BOTH POS AND Inventory show ✓ Closed
# Sites sorted alphabetically by name
# Click outside popover or ESC key to close
# Popover max height 400px with vertical scroll if needed

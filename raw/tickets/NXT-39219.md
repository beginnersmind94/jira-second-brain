# NXT-39219 — Imports – History Log – Show Original Import Data - Multi-Filter Capability

## Description
As a SchoolCafé admin, I would like to be able to filter on more than one column when viewing the original import file so that I can use it to diagnose any issues.

*Notes:*

* We already created the ability to view the files, but need to come up with a way to filter on more than one column at a time.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* Click on View Log button
* Click on Source Data tab
* View Data and filter 2 columns

*Requirements*:

* All these requirements are for the View History screen
* All these requirements should be added to every import type
* 
* On the Source Data tab, do the following on the grid:
** Enable the grid to filter on more than one column at a time

## Acceptance Criteria
* Grid to filter on more than one column at a time

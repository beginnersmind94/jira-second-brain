# NXT-7662 — MP > Menus > edit Menu screen - Remove MIC and prohibit removing based on rules

## Description
As a Menu Planner
I need to be able to build Menus
so that I can show that they are compliant.

for lower Menu screen mockup see: [https://primeroedgenext.atlassian.net/browse/NXT-6983|https://primeroedgenext.atlassian.net/browse/NXT-6983|smart-link] 

A Menu Item Category that has Menu Items in it cannot be removed from the Menu, from the grid option, nor by removing the count in the top section.

A Menu Item Category that does not have Menu Items in it can be removed from the Menu, from the grid option, or by removing the count in the top section.

## Acceptance Criteria
A Menu Item Category that has Menu Items in it cannot be removed from the Menu.
A Menu Item Category that does not have Menu Items in it can be removed from the Menu.

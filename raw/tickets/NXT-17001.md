# NXT-17001 — INV - Orders - Orders List - View Receipts - Display Totals row

## Description
As an Inventory user I want a totals row on the Receipts List so I can see the totals for Invoice Amounts, Receipt Amounts and Discrepancy Amounts

*Requirements*

* Display a Totals row above the grid, vertically aligned with the respective grid columns

+*Totals Information:*+

* At the top of the Order List grid, provide a section to display totals for the following info:
** Total Receipt Amount
** Total Invoice Amount
** Total Discrepancy Amount
* Ideally this would be an additional “Totals” row at the top of the grid, but if not possible this could be provided outside of the grid
** Advantage of custom approach: could show the totals for ALL receipts within the search criteria, not just the ones on the screen
** First approach: insert a header in the grid of receipts, which would be a ‘Totals’ row and have only values for the 3 columns mentioned above
*** Similar example: Charges & Discounts section has two headings
** Second approach: add a footer to the grid that would display the totals for those columns
*** see component library for Kendo grid w/ footer
*** Possible limitation: only shows the total for the receipts on that page of the list
**** Max 100 receipts at a time

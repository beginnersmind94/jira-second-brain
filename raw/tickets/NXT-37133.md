# NXT-37133 — MP - Configuration - Menu Templates Create New

## Description
Manage, view, add, and edit menu templates to be used in Menu Planning.

As the user handling the process of creating menus, I would like to have the ability to set up templates of menus that I can use as a “base” for creating a new menu.  One example of a use for a quick list would be to have a template created of a grouping of breakfast items that are used frequently together, so that I can select that template as I begin the “Create Menu” process, and those items will start out on the menu, saving me time by just needing to add or remove specific items to finalize the menu. Or if I have items that I use on my menus every time I serve a particular type of food (such as on Taco Tuesday, the district might always have tacos, the side salad, quesadillas, refried or charro beans, spanish rice, and guacamole). I can start my menu using my Taco Template, and simply make any adjustments to customize it for a specific serving day.

## Acceptance Criteria
* When creating a template, I will need the ability to select multiple meal types and site groups as I may want to use this template for multiple menus across site groups. 
* Once template is created, user can not change the meal types or site groups
* Template Name/Description is required.
* User must enter a projected meal count for atleast one serving group ( same as menu) 
** Logic applies same as menu for choose counts (1 for all categories except veg 2)
* When creating a template, I will need the ability to search for, select, and add menu items to the list.

# NXT-17478 — INV - Orders - Create Orders - Apply more accurate validation tooltips + Cart enhancement

## Description
As an ordering manager I want more specific messaging on the Enter Item Quantity screen so I know exactly by how much I need to increase or decrease my order quantities.

In the Create Order process, we did not implement the more detailed messaging on the first item screen. (See screenshot below.) The more detailed tooltip messaging we are providing elsewhere in the Create Order process needs to be applied here as well.

*Requirements*

* On the ‘Enter Item Quantities’ screen, which is loaded after the user clicks Apply on the Create Orders screen
* In the Shopping Cart, which is expanded and viewed in order to proceed to Confirm Orders screen
* Where the (!) icon is displayed, provide the more detailed messaging in the hover tooltip:
** if quantity does not meet minimum: “Increase order quantity by X to meet minimum order quantity.”
** if quantity exceeds maximum: “Decrease order quantity by X to meet maximum order quantity.”
* Modify the icon on the ‘Expand’ button of the Shopping Cart
** change the arrow/expand icon to the cart icon 🛒 
** display a badge on top of the cart icon, with the number of items in the cart
* Research possibility to display small Item Category groupings in the Shopping Cart

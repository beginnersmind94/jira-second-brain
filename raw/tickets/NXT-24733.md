# NXT-24733 — POS - Dynamic Menus - Swap between dynamic meal service and static Alacarte menus

## Description
User can easily swap between dynamic meal service menu and regular alacarte static menus.

There is a NEW button, next to the Menu Selection, that toggles between the Default Dynamic Menu and Default alacarte Static Menu when selected. 

If Dynamic Menus is.not enabled the button is not shown. 

Disable the Swap button if no alacarte Static Menus are defined for the site.

If the swap button is selected when Dynamic Menus are loaded and a static menu is not shown the Menu Selection popup is shown allowing the user to select a static grid to use for alacarte serving.

## Acceptance Criteria
The Swap Button name is changed from ‘Dynamic’ and ‘Static’ to ‘Meal Service’ and ‘AlaCarte’.

When the Dynamic Menu is displayed selecting the Menu Toggle button switches to and displays the Default Static Menu for the device.

The Swap button is only displayed if Dynamic Menus is enabled.

Swap button is disabled if no static menus are available.

If a dynamic serving line is set and default Static Menu is not set the Menu Select Grid is displayed.

When the Static Menu is displayed selecting the Menu Toggle button switches to and displays the Default Dynamic Serving Line Menu if available.

When the transaction is completed the Dynamic Serving Line menu is displayed .

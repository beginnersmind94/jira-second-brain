# NXT-40332 — eOrders Configuration – Add New Part 5 – Distribution Tab

## Description
This is a continuation of the eOrders Configuration – Add New Part 1 and covers the Distribution Tab

* If the Sites tab is not already created, then create it and do the following on it:
* 
* Flow:
** Ask the user how they want to distribute this eOrder?
** If yes
*** Ask how to distribute
*** If they select email, they enter email addresses
*** If they select FTP, ask them for the FTP information we need in order to send the eOrder via FTP to a location of their choosing
** If no, hide or make read only this section
* 
* Fields:
* Email
** Yes / No
** Optional
** Default value is No
* 
* Fields to show if they select Email:
* Email Addresses
** Textbox
** Required if Email = Yes
** No default value
** Tooltip: Add comma delimited list of email addresses
** Note: The user can type in a list of email addresses separated by commas
** Note: make this a larger textbox and ability to scroll since they may have a lot of email addresses they want to use
* 
* FTP:
** Yes / No
** Optional
** Default value is No
* 
* Host
** Textbox
** Required if FTP = Yes
* Port
** Textbox
** Required if FTP = Yes
** Default = 22
** Validate: Must be a number
* Folder
** Textbox
** Optional
** Note sure if we need this
* Username
** Textbox
** Required if FTP = Yes
* Password
** Textbox
** Required if FTP = Yes
** Use usual hide / show text eye button
* FTP Type
** Single Select List
** Required if FTP = Yes
** Values:
*** sFTP
*** Active FTP
*** Passive FTP
* 
* Previous
** Go back to previous screen
*  Save
** On click, validate required fields and show the user validation error messages until they are all resolved
** If validated, save the eOrder configuration to the database and add data to all the relevant tables such as scheduling table, sharing table, etc.
** 
** Log the save action to the History Log
*** See History Logging story for instructions
** 
* Notifications:
** Send a message to the inbox of all users that have Add rights in Roles & Permissions
** Text: [logged in username] has created an eOrder configuration for [Vendor Name] To view, click [link to the eOrder Configuration]

# NXT-5046 — Item Management - Disregard Changes

## Description
As a user, I want to have a confirmation message if I click the cancel button.

Mockups

 !image-2020-08-18-08-22-00-681.png|thumbnail! 

Requirements
* When clicking cancel on 'Edit' or 'Add New' a warning has to appear
* Message: "You have unsaved changes. Are you sure you want to disregard?"
** Have the option to 'CANCEL' or 'DISREGARD'
* If clicking cancel, then return them to the add or edit screen with the unsaved changes
* If clicking disregard, then navigate out of the record or return to the previous state

## Acceptance Criteria
The user must be warned if they want to disregard their unsaved changes or not.

# NXT-70581 — GC - PE -Item Catalog - Item details popup - Import item with meal contribution.

## Description
Currently, when the Import button is clicked for an item with meal contribution values, the system imports the item by assigning the minimum contribution value to all food components.

This needs to be corrected so that only the food components with actual values in the catalog are imported, and their corresponding values are accurately assigned.

# import only food components with contribution amount >0 and it should be with min and max values. 

!image-20260227-203704.png|width=1239,alt="image-20260227-203704.png"!

## Acceptance Criteria
When the user clicks the Import button for an item with meal contribution values, the system must import only the food components that have defined values in the catalog and assign their actual catalog values. 

The system must not assign default or minimum contribution values to food components that do not have defined values.

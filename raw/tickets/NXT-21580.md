# NXT-21580 — INV - Create Withdrawal - Add Items (Step 2) - Template Logic

## Description
As a site manager I want the ability to add items via Template selections so I can add many items at once.

Only the individual item logic was handled in original story [https://primeroedgenext.atlassian.net/browse/NXT-1675|https://primeroedgenext.atlassian.net/browse/NXT-1675|smart-link] 

*Requirements*

* Change the Add Item section UI to handle the following logic regarding the *Add* button:
** enabled once a single template is selected *OR…*
** enabled once an item is selected and there is a value for the Quantity field
* When user selects a template, provide the Add button and do not require a single item to be selected or quantity to be input
* -Filter the smart search results according to the selected templates- *(Done)*
** this would mean only items contained within the selected templates would be returned by the search
* If user selects an individual item *first,* temporarily disable the Template dropdown until the item is added OR the item selection is cleared
* Add a label by the Withdrawal Templates dropdown to show how many items will be added according to the selected templates

* Allow user to add items by selecting only a Template

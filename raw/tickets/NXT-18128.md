# NXT-18128 — INV - Manage Orders - Receipt Details - History tab - Transaction Details

## Description
As an operations manager I want a Transaction Details view for Receipts so I can view prior transactions of the selected receipt.

*Requirements*

* Once user clicks the view icon for a particular transaction from the Receipt’s History tab, open the historical transaction details
* Display the Receipt Details screen in a non-editable state
* Replace the banner with an informational banner for the history:
** “This receipt transaction was changed on 8/23/2021 by Justin Miller.”
* Remove the buttons
** Edit Receipt
** Print Receipt
** All input icons
*** Confirm icons
*** Add button for Lot Number sub-grid
*** Comments (for Orders or Items)
**** only allow comments which existed before transaction was adjusted/cancelled
** Confirm/Save button
* Also add charge/discount columns (Delivery Fee, Processing Fee, Net Off Invoice) to the view-only state of Receipt Details
** only show columns if the charge/discount exists for one or more items
** this change should be made on Receipt Details regardless of historical transaction (change on the main view as well)

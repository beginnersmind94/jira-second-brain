# NXT-2790 — **View Inspections - Card View

## Description
*User Story*
As an Inspections user, I can view Inspections on the View Inspections screen in Card View.

*Mockup*
[https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1|https://m5uloa.axshare.com/#id=c7gcdx&p=inspections&view=default&g=1]

*Requirements*

* Users can toggle between card view and list view when viewing Inspections
** Toggle is done through the buttons on the top right
** List View is the default
*** Last selection is remembered between sessions
** Sorting and filtering is remembered if set a specific way in List View
*** Resets when session is refreshed
* Card View
** Inspections are visible in rows of three cards
** Each card has a fixed display that cannot be edited
** Cards should display in infinite scrolling
*** 21 at a time - can be adjusted in future for performance needs
** Default filtering is Date descending (most recent to least recent)
** Default date range is 1 week in the future back 60 days
* The following information is present:
** Inspection Name
** Template
** Inspector
** Site
** Site Rep
** Date/Time
*** If scheduled, display the date the Inspection is scheduled for
*** If Inspection was completed within a single day, display the date and time in/time out
*** If Inspection was completed across multiple days, display the date the Inspection was submitted and the number of days the Inspection spanned.
** Score
** Status

## Acceptance Criteria
* Card view is present and can be filtered
* User can switch between the two views seamlessly without any loss in data

Note: Using dummy data is acceptable to demonstrate completeness

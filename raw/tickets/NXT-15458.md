# NXT-15458 — System - Program Configuration and Reimbursement Rate - Standardize program names

## Description
The program names on “Program Configuration” page shall include the program code appended to the name.

!image-20220707-221728.png|width=809,height=616!



!image-20220707-221705.png|width=761,height=701!

## Acceptance Criteria
* The column header for “Program” on the tables in these pages should read “Program” and not “Programs”
* On the “Program Configuration” page,  the program “(XXX)” code, if exists, shall be appended to the program name. Examples are below.
** School Nutrition Program (NSP)
*** School Breakfast Program (SBP)
*** National School Lunch Program (NSLP)
*** After School Snack Program (ASP)
*** Seamless Summer Option (SSO)
** Other Programs
*** Special Milk Program (SMP)
*** Child and Adult Care Feeding Program (CACFP)
*** Fresh Fruit and Vegetable Program (FFVP)
*** Minnesota Kindergarten Milk Program (MKMP)

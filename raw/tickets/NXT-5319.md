# NXT-5319 — Onboarding - System Task List - Manage Settings - Display Settings Tour

## Description
As a new SchoolCafé user, I would like to have a basic walkthrough on the steps required to set up my district’s settings, so that I don’t have to spend time guessing what to do next.

*Summary*

When the user comes to the Manage Settings task, display a simply tour pop-up (with a Finish button), that explains what the user must do to ‘complete’ this task.

*Mockup*

!image-20230523-221246.png|width=412,height=310!

*Details*

In Onboarding → System → Tasks, when the user selects Manage Settings and is redirected, do the following:

* Display a tour in the center of the screen:
** Text:
*** Title:
**** *How to Complete This Task*
*** Body:
***# This step is also completely optional and is only to allow you to view and modify default settings to better suit your district.
***# *Warning:*  It is recommended that you do not modify any setting whose purpose is unclear to you, as it might create an unforeseen issue later. Please work with your Success Mentor first.
* Display a ‘Finish Tour’ button in the footer.
** When the user selects this button, close the pop-up. This should prevent the tour from executing when the user logs out and returns to this page.
* [~accountid:6035e74df8c057007079bd9b], please verify for each of these stories that the tour only displays for users with the Onboarding role. Other users should therefore not see the tour when they navigate to these pages normally.

## Acceptance Criteria
Verify that when the user comes to this task, they are immediately presented with a tour step in the center of the screen

Verify that the text and button placement of the tour match the description above

Verify that when the user selects ‘Finish Tour’, the pop-up closes and the user can proceed

Verify that the user will not see the tour again when they are logged out and return tot he page

# NXT-5926 — SE - Eligibility Guidelines - Setup

## Description
*User Story*

As a district admin I want to edit the income eligibility guidelines as soon as they are available for the upcoming academic year.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5144%3A80238&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5144%3A80238&scaling=min-zoom]

*Requirements*

* Add an Edit option to the IEGs page.
** This should allow the user to edit the following values:
*** Poverty Guideline value for a household size of 1 (min: $0.00, max: $100,000.00)
*** Amount for each additional family member (min: $0.00, max: $100,000.00)
*** Percentage amounts for Free/Reduced (min: 0%, max: 1000%)
**** We should be smart here - the Reduced % should always be more than the Free %.
*** !image-20200921-183112.png|width=839,height=598!
** All income guidelines grid values should fill/update dynamically as poverty guidelines are entered, and if percentages are adjusted.
* Cancel and Save options.
** If IEGs for that Academic Year + State Category already existed, prompt the user for confirmation.
** ‘This will replace the existing guidelines. Are you sure you want to proceed?’
* Values in the grid need to update according to these rules:
** The Federal Poverty Guideline for household size > 1 increments by the additional amount per family member.
*** If the poverty guideline for a household size of 1 is 10000 and the additional amount is 5000:
**** Household size 2 = 15000, 3 = 20000, etc.
** *Round up while calculating the frequency rates below.*
*** Annual rate = federal poverty rate * percentage.
*** Monthly = (federal poverty rate * percentage) / 12
*** Twice per month = (federal poverty rate * percentage) / 24
*** Every two weeks = (federal poverty rate * percentage) / 26
*** Weekly = (federal poverty rate * percentage) / 52



Added 11/10/20 during Planning:

Added 11/19/20 during Daily:

Edit is only available for the current academic year. (Added 11/4/20 after discussion in daily+)
Free % cannot be higher than Reduced %.
User is given a confirmation prompt if IEGs for combo of academic year + state category already existed.
Grid calculates dynamically based on the values inputted.

Remove the following buttons: Get Guidelines, Generate Report, Add, Delete Icon.

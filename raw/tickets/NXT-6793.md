# NXT-6793 — Application Entry - Guided Process (Cancel)

## Description
*User Story* 

As an admin, I have the ability to cancel an application.

*Mock Ups* 

{color:#ff5630}Mockup not required{color}

*Requirements* 

* Cancel button 
** The cancel button forfeits the application and does not save any data for the admin
** Cancellation message > _Are you sure you want to cancel this application? The information will not save once you leave._ 
*** ‘Cancel Application’ > takes them to Application list 
*** ‘Continue Working’ > no information lost, pick up where they left off

## Acceptance Criteria
ADA, Multi-language
cancel confirmation popup appears on click
Canceling the application does not save any data
continue working closes the popup and no info is lost

# NXT-11038 — IM - Recipe Report

## Description
As an Item Managment User, I need to be able to view my Recipe Information in a report

The flow in classic is to be able to go to the Recipe itself and click Recipe Report.

!image-20210215-010447.png|width=100%!

*ACTUAL REPORT - Report Name -* Recipe Report

We need to just copy this report for now

* Show Recipe Name (Recipe Code)
* Display date and time generated
* Show Meal Contribution (Total number of contributions for this recipe
* Show the Recipe steps, and the Ingredient (Item) as well as quantity
** Stock Item #, Stock Item, and Stock Quantity are not required here since we aren’t linking
** Still include the ‘Cost’ as we have the price for the item in the system as of now
* Include Nutritional information
** Total Fat, Sat Fat, Carb, Protein, Total Sugars
* Nutrients per 100g
** (M) if missing nutrients
* Nutrients per serving (Weight per serving)
** (M) if missing nutrients
* Show legend at the bottom
** (M) Indicates Missing Nutrient Values

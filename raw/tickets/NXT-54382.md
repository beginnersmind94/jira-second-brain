# NXT-54382 — IM - Items/Recipes/etc. - Convert scaled weights to higher units (R&D)

## Description
As an Item Management user I want all my unit/serving weight data to be scaled to higher units so I can read it in understandable values.

*Conclusion of R&D:* we will attempt this implementation in the future [https://primeroedgenext.atlassian.net/browse/NXT-59849|https://primeroedgenext.atlassian.net/browse/NXT-59849|smart-link]

## Acceptance Criteria
* Convert scaled weight & volume values to the appropriate higher unit when thresholds are reached
* establish thresholds for how we pick the next higher unit
* show two units if appropriate (e.g. 4 Pounds, 2 Ounces)
* Always retain user entered values for edit modes
** i.e. if user entered “grams” then use grams for edit

# NXT-16409 — MP - Forecast Saving

## Description
As a user, I want to be able to save my forecast

Mockup

Requirements

* When I click the 'Save' button at the top or the bottom, the forecast will save to the database
* Before clicking save, the information will be stored in the UI
* When clicking on the forecast, user can edit the information
* Forecast Description can be changed
* Meal Type and Site Group values can be added, but cannot be deleted
* Users can edit the Number of Servings and can delete menu items from the forecast

## Acceptance Criteria
Forecast saves successfully

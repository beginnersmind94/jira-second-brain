# NXT-20902 — Production - Reports - Production Record - Items Planned But Not Produced (Report)

## Description
Menus are assigned to sites, with the intention for those sites to serve the items as they’re assigned. 

Site Managers put their planned (or intended) counts next to each menu item, in advance of the actual serving date for the items. However, sometimes things occur that cause the site managers to NOT serve some or all of the menu items that they had planned/intended to serve… on those occasions, I will put a “Zero” in the “Quantity Produced” field on the production record.

## Acceptance Criteria
*On UI criteria page, only display in this order Site Group, site, Meal Type, and To/From dates.*

This report should give me the list of menu items from each date and site that were not produced. It will also give me any comments that were entered next to those menu items, so that if a user has put in a reason or information in regards to why something was not produced, it will appear on this report so that I can review it. 

Display Menu Item Category, Menu item, Planned count, and any comment that was entered.

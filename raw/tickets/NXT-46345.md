# NXT-46345 — IM - Recipes - Recipe List Export

## Description
As a district administrator or menu planner I want a Recipes page export so I can generate a simple recipe report with only the recipes I have loaded on the page.

## Acceptance Criteria
* Add the ‘Recipe List Export’ to the bottom of the Reports dropdown on the Recipes list page
* Upon generating the Recipe List Export, an excel file will be generated with the full list of recipes contained in the grid
* The export should consider filters applied to the header and any grid header searches
* The excel export should always include every grid column (regardless of Manage Views selections)
** match the column sequence of the page grid

# NXT-30931 — PR- Prep Site configuration- Changes on Production Pages

## Description
As a district Manager, I want the prep sites I configure to display on the appropriate production plan and record pages with newly configured information

*Navigation:*

Production> Calendar> Production Plan/Record

## Acceptance Criteria
* Any changes done on either 1 to 1 or one to many prep site configuration pages should reflect on the individual production plan and record. Effect only production plans that are not complete. 
** User may still go back to the individual production plan and change the prep site and save. This will only effect that particular production plan and record. 
*** EX: If user manually changes production plan prep site but does not change the status to plan complete and is still in plan incomplete, if the configuration is changed again, it will effect that production plan and record.
* Only consider plans and records that are from the effective date and moving forward

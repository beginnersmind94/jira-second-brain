# NXT-23958 — INV - Management - Contract Items - Update Order Quantity Limits

## Description
As a District Admin I want an Update Order Quantity Limits process so I can quickly update minimum or maximum order quantity limits for many contract items at once.

*Requirements*

* Provide an action on the Contract Items page ([NXT-22306|https://primeroedgenext.atlassian.net/browse/NXT-22306]) to update the Min/Max Order Quantity for all selected items
* Lock down the dropdowns 
* Display the item list, but change the list/columns to a dedicated screen for Min/Max Order Quantity
** Column list:
*** Item #
*** Item Description
*** Vendor Item # 
*** Minimum Order Quantity
**** load each field in editable mode with its current value
*** Maximum Order Quantity
**** load each field in editable mode with its current value
*** Provide a Save button
**** to save individual items
**** only allow Save if some value has changed
** Provide a Save All button for mass updates
* Update the Minimum Order Quantity and Maximum Order Quantity value for each item with an entered value
* Provide a ‘Complete’ action to return to the searchable main version of the Contract Items page

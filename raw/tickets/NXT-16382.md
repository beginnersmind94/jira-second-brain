# NXT-16382 — INV - Settings - Charges & Discounts Configuration

## Description
As an inventory operations manager I want a Charges and Discounts configuration screen so I can add any charges or discounts to be used on receipts.

This screen is for configuring and viewing any charges or discounts that are not item-specific and therefore are not configured on a vendor contract.

*Requirements*

* Create a new ‘Charges & Discounts’ page under the Settings category
** Note: this marks the creation of the ‘Settings’ category for the Inventory module
* Display the system level charges/discounts, which are defined on vendor contract, as un-editable rows in the list
** Processing Fee
** Delivery Fee
** Net Off Invoice
** These charges/discounts should not be editable
*** hide the edit icon
*** do not allow these 3 types to be inactive
* Provide a list of existing charges/discounts already configured
** Columns:
*** Description - alphanumeric description for the added charge/discount
**** sortable column
**** Unique within district
**** 100 character limit
*** Type - either ‘Charge’ or ‘Discount’
**** sortable column
*** Status - Active or Inactive
**** sortable column
*** Action column:
**** Edit icon
***** clicking the edit icon will enable the Description to be edited and will enable a checkbox for Active status in the Status column
***** validate Description uniqueness on save
***** use green checkmark / red x in Action column when editing
***** disable edit icon for other rows
**** Delete icon
***** icon only present if charge/discount is not in use on a receipt
***** Prompt on delete: “Are you sure you want to delete this charge/discount?”
* Provide the ability to add a new Charge/Discount
** Description field - textbox to type alphanumeric description
** Charge or Discount toggle - radio button to select whether it’s a Charge or Discount
** Active checkbox 
*** this would allow the user to potentially create a charge/discount that is inactive from the start, to be activated later
** Add button

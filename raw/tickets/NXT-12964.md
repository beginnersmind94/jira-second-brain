# NXT-12964 — ExpressPoint - Load Default Menu

## Description
As a cashier if a menu is assigned as a default for a particular day/meal type I want that menu to automatically load when opening the terminal and/or changing the day/meal type so that I don't have to select it each time.

## Acceptance Criteria
* Based on configured default menus that apply to the terminal, default menu should be loaded by default on session opening
* Based on configured default menus that apply to the terminal, default menu should be loaded if day and/or meal type is changed by the user

# NXT-52180 — INV - Physical Inventory - Review Completed Inventory - Audit View

## Description
As an Inventory Supervisor I want an Audit View when reviewing physical inventory items so I can see the last time items were reconciled and view their inventory count history.

## Acceptance Criteria
* On the Items tab of the new Review Completed Inventories page, add a new Audit option to the View radio group
* When Audit view is applied, change columns
** Keep Physical and Perpetual Count
** Add a ‘Last Reconciled’ column, which will show the most recent period or physical inventory in which that item was reconciled
** Add a History column
*** Display a history icon where the Count History can be viewed in a slide out panel (instead of current popup)
* Display the Item Count History slideout panel when the history icon is clicked
** Display a historical list of the items that item has been included on a physical inventory for the site
** Date, descending sort order
*** display Period, or Date for interim inventories
** Count - physical count
** Discrepancy - difference between physical and perpetual at the time of inventory completion
** User - counted by
* Use wider panel if needed (up to 50%)

# NXT-5512 — Roles - Global Permissions

## Description
*User Story*

As a PrimeroEdge administrator, I have a section called “Global Permissions” that contains any system-wide permissions that do not fall under any one Module.

*Requirements*

* In addition to module permissions, there is a screen available for Global Permissions
** Each permission only has an on/off toggle
* Global Permissions is selected from the dropdown when viewing a role in the same way any individual module would be
* The following permissions are available:
** Access to Personally Identifiable Information (PII)
*** If activated, allows user to view PII for patrons in any place it appears in the product
** Admin Reset
*** Allows user to reset any user’s password
** Admin Login
*** Allows user to log into any user’s account
* Others should be able to be easily added as needed

## Acceptance Criteria
* Global permissions screen can be accessed
* Permissions can be selected and saved

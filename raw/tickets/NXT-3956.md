# NXT-3956 — Production Plan - allow overriding for Prep Site Items

## Description
Requested by Denver PS and Sodexo at minimum.

Allow overriding by Site and/or Date for which Menu Items are produced at a Prep Site.

[https://primeroedge.freshdesk.com/a/tickets/77107|https://primeroedge.freshdesk.com/a/tickets/77107]

## Acceptance Criteria
A Menu Item can be produced at a Food Prep Site one day and not on the next, or can vary by Site. Defaults should be configured but not totally dependent on Menu Item nor previous value.

Perhaps a checkbox next to the “Site Prep” menu items… it’s an empty checkbox to signify that the other site will prep the item, but the feeding site can pull it “local” by marking the checkbox? This should make it so that the counts from that item do not flow to the prep site for that particular item.

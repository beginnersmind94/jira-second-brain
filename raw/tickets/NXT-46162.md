# NXT-46162 — Elig - Applications - SIGs - Verification template updates

## Description
As Anna the Application Processor I want send Verification notices which include State Eligibility outcomes and I need the system to behave appropriately.

*requirements*

This story is to handle template updates for State Income Guidelines (SIGs) + Verification. These scenarios only apply if the setting is turned on.

* Verification Change Notice
** Add table fields:
*** {{Current State Status}}
*** {{Previous State Status}}
**** Fields are to use the local descriptions for state eligibility if available
** Add letter fields:
*** {{Result State Text}}
**** Should only populate if a State Eligibility was used as part of the verification outcome.
**** Examples: NJ-Free to NJ-Reduced, NJ-Free to N/A, N/A to NJ-Free
* Verification Completion Notice, Verification Notice, Verification Subsequent Notice
** Add table field: {{Current State Status}}
*** Use the local labels if in-use, otherwise State-Free, State-Reduced, N/A

## Acceptance Criteria
New fields available if the System Setting is on:

* All 4 verification templates updated to have the {{Current State Status}} table field
* Verification Change Notice updated to have table fields: {{Current State Status}}, {{Previous State Status}}
* Verification Change Notice updated to have field: {{Result State Text}}

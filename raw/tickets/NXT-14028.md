# NXT-14028 — Eligibility - Applications - Multiple application images

## Description
As a district admin I want to see the originally submitted information on the application image, and updated images every time the app is processed. This will provide a visual audit trail for information changing over time.

*requirements*

* An application image should be stored with the original data.
** This is vital for online submissions, particularly with pending students and editing.
* An additional application image should be created every time an application is processed.
* The ‘Application Image’ button should show the latest application image.
* All application images should be available for the user to browse through.

## Acceptance Criteria
* User can pull up an image containing the original app data.
* User can pull up an image containing the processed app data.
* ‘Application Image’ button should by default show the latest image.
** User can nav between application images and export any of them.
* Editing an application should not update the application image - data must be saved prior to the image being updated.

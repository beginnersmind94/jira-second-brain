# NXT-31652 — MP- Meal Pattern- USDA MEAL PATTERN SY 2014+ Add Additional Sodium Target

## Description
Based on USDA regulations, They are introducing new Sodium Target to be affective July 1, 2023 but would like for the software to include it to start getting users used to it being there. 

Add this additional Sodium Target 1A to the meal pattern pages that apply.

## Acceptance Criteria
Change Current Name of Sodium (mg) to Sodium Target 1 (mg) on all pages (Breakfast, Lunch) Except PK

!image-20220606-230422.png|width=1098,height=220!



Include New Sodium Target 1A (mg) in a new column for Lunch Meal Type

Targets: 

K-5 <=1,110 mg

K-8 <=1,110 mg

6-8 <=1,225 mg

9-12 <= 1,280 mg



Apply change to *Disclaimer* on the bottom of the Meal Pattern Page

Current:

4.PrimeroEdge will flag based on USDA requirements starting School Year 2019-2020 for WGR and 1% flavored milk, for prior years, please consult your State Agency for that School Year requirements and waiver status.

Proposed: 4. Schoolcafe will flag based on USDA requirements started School Year 2022- 2023 for WGR and 1% flavored Milk.

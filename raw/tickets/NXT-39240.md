# NXT-39240 — PR- Carryover Scenarios for Multiple carryovers on 1 Menu Item

## Description
As a Production manager, there may be those times where I am carrying over a menu item from multiple menu lines or days to 1 specific day. There needs to be a way to know where the carryovers are coming from and how to handle

## Acceptance Criteria
* Allow multiple carryovers from different dates to the same serving date
* Once one carryover is added on a “Can Serve” menu line/ meal type day, all other carryovers to that same combination will show “Menu Item Planned” when looking at the carryover calendar
* Display carryovers separately on both the plan and record in separate “bubbles” with all information 
** Display serving size information per bubble EX: 1/2 cup or 1 cup
* Allow “receiving” serving date to have multiple serving sizes carried over to it (IF the menu item is added by carryover)
** We will use the serving size of the carryover that was added FIRST to the future plan/record
*** *-EX: if 1/2 servings were carried over from 5/1 FIRST and 1 cup servings were carried over from 5/2 both to 5/5, show 1/2 servings for the serving groups-* 
*** *-EX: if 1/2 servings were carried over from 5/1 and 1 cup servings were carried over from 5/2 FIRST both to 5/5, show 1 cup servings for the serving groups-* 
*** If the user enters 0 for all the serving groups, still display the menu item on the record as this item was added by carryover
*** Display different rows for the serving sizes that were carried over on the record with a planned count of 0
**** Once user types in a produced number then add the produced+ carryover to get the total served for each serving size 
*** If the user enters counts for some or all serving groups. display the planned counts associated with each serving size on the record in separate rows 
* *Do not remove the menu item from both the plan and or the record unless all carryovers are deleted even if there are counts on the plan or record*
** *We maintain that the menu item would have never been on the plan or record had the carryover not been there and we will continue to maintain that concept*

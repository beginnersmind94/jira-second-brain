# NXT-3777 — Create Section Template

## Description
A section contains either other sections or filter criteria. A section can be expanded to show the contents of the section, or collapsed to hide. Multiple sections can be expanded at a time. A section consists of a Name, a count of the filters chosen within the section, an expand/collapse button, and the subsections or filters that can be selected.

## Acceptance Criteria
A section has a name

A section can contain subsections

A section can contain multiple filter criteria

A section will show the number of filters selected in the sections

A section can be expanded and collapsed

A section has a toggle to switch between an 'Equal to' to a 'Not equal to' for the filter. 

Equal to is if the filter matches the field.

Equal to is the default.

Not equal to is everything but the filter that matches the field.

Filters within a section or subsection are considered to be 'OR' 

Filters between sections or subsections are considered to be 'AND'

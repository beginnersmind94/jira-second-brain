# NXT-68188 — Display Meal-Specific Empty State Message

## Description
As a cafeteria staff member, I want to see the correct meal type referenced in the empty state message so that I know which specific meal period has no menu published.

## Acceptance Criteria
* When no menu is published for the currently scheduled meal period, the message displays "Looks like there is nothing planned for [meal type] today" where [meal type] dynamically reflects the active schedule (breakfast, lunch, or supper)
* The meal type determination is based on the current time matching the Start Time and End Time of the corresponding schedule configuration

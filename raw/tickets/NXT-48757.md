## Description
_(none)_

## Acceptance Criteria
The user should be able to search based on user name , First name and Last Name (based on AND Condition)

if User Name alone is entered - Search by User Name

If user Name & first name is entered - Search records that matched with User name and first name

if all the 3 fields are entered, search records meeting with all the names

All the 3 fields should work based on like condition (%Name%)  

Upon clicking search - the record should only list the items which are accepted by the corresponding user as entered

# NXT-34481 — IM - Items - Internally Produced Items

## Description
As a District Admin I want to be able to create regular items from Menu Items (and Recipes) so I can make the items orderable to my own sites, from a prep site or central kitchen.

*Requirements*

* Provide tab for Item link (To be named)
** Provide a ‘Create as Item’ button
* Create a regular Item when the item link is created
** Use the existing flow, pre-filling some info if possible
** this will exist for Recipes as well, and all 3 will be linked together in most cases
* Pre-fill the Food Info if it’s available from Menu Item (or Recipe)
** Show the Menu Item serving size, nutrition, etc.
* Modify the Menu Item / Recipe Info tab to have a different UI
** instead of allowing the item to be created as a new MI/SIR, this will point back to the source Menu Item or Recipe, establishing a link

## Acceptance Criteria
* Provide the ability to create a regular ‘Item’ from a Menu Item
* Integrate the updated item creation flow into the Menu Item link
* Display a special version of the Menu Item / Recipe Info tab to link with the source MI/Recipe

# NXT-45401 — Users - Managed Views - Reset View

## Description
JAS - the below is from Justin. to accomplish this, we will do the following:

* On the ellipsis next to Managed Views, do the following:
** Add a button
*** Label: Clear View
*** On click, remove the managed view applied to the grid and revert the grid to the default view created by code when there is no managed view applied
* Apply this to System > Users first



Original Text:

As a SchoolCafe user I want the ability to reset the saved view selection so I can view the default grid behavior when there are custom views created.

Currently once at least one custom view is saved to the Manage Views tool, users cannot access the original default grid configuration. We need the ability to either deselect a view and reset to the default grid, or have an option in the slide-out for that default view.

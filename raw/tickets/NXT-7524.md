# NXT-7524 — Application Processing - Received Date and Preferred Language

## Description
*User Story* 

The processing page has the option to updated preferred language and change received date (based on setting). 

*Mock Ups* 

*Requirements* 

Application Received vs. Application Processed (section):

* *Look at ‘Application - Effective Date’ (setting)* 
** If set to ‘Received Date’ > show Date selector > admin can go as far back as the start of the program year to backdate an application 
** If set to ‘Processed Date’ > Section is display only populated with Today’s date 

Language Preference:

* Shows the selected language preference from the submitted application with the ability to change/edit if needed
** Drop down selection for configured languages for the district (system module)
*** default: English
** Application Language > Looks at configured languages for the district (system module)

## Acceptance Criteria
- Multi-language, ADA compliant 
- Ability to see selected language and update if needed
- Application Processed vs Received date shows based on the setting 
- Application processed is view only 
- application received is a date picker with the ability to backdate

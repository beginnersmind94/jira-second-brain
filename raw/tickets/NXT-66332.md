# NXT-66332 — ACC - Attendance Feature - Monthly Entry - Main Grid and Filters

## Description
In order to support the ability to utilize accurate attendance counts within reporting as well as meet NC requirements the attendance page will need several changes.

To accommodate a relaxed entry method of tracking ADA and Attendance Monthly we will be taking the existing monthly entry method on the attendance page and introducing a few quality of life improvements.

When Monthly Entry Method is selected

First we will be implementing separate tracking for SNP, CACFP, and SFSP counts.

Implement individual grid storage tabs for SNP, CACFP, and SFSP
*Top Filters*

Site filter (Allow multi - site select for site control)
Academic Year filter
Months” Filter ( July - December, January - June)

*For the monthly grid view*
- Display 6 months at a time starting from July
- Shrink the size of the individual column and text box sizes to comfortably fit 6 months.(keep existing text box control behavior)
- On Page load Text boxes are immediately editable

*Buttons*
- Provide a simple back and forward arrow buttons to switch between the 6 month increments. (within the same academic year only)
- Provide pagination for sites (default 10)
- Provide a “Highlight Missing Counts” when click stylizes cells with null values (bold colored border and bold highlighted background)
- Provide a grid export button that simply exports current grid view to excel
-Stylize current month to be highlighted with Bright Border around the Major column Cell

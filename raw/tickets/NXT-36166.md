# NXT-36166 — Student Purchase History - Create SPROC

## Description
As a SchoolCafé admin, I want to be able to export the student Purchase History so that I can upload it into another system like My School Buck.

*Note:*

This is the story for FO to create a stored procedure to be consumed by Platform team related to Student Purchase History. Below are the requirements we received from the customer.

*Requirements*:

* Here is what we received from the customer:
** Student Purchase History (SchoolCafé to My School Buck):
** Columns: “SID”, ”LAST NAME”, ”FIRST NAME”, ”ITEM NAME”, ”ITEM PRICE’, ”SCHOOL”, ”DATE/TIME”, ”GUID” 
** Example: "654321", "JONES", "MOLLY", "SSO Breakfast - B", "0.00", "RED HIGH SCHOOL", "2021-04-08 08:18:00", "92E5618E-BEC8-47A9-9E3A-572F97151234"
* We need to map our internal fields to theirs
** Use our names for the headers, just make sure they are in the same order
* Note that they want the header not included in the file

## Acceptance Criteria
Stored Procedure produces the correct data from the fields mentioned in the requirements.

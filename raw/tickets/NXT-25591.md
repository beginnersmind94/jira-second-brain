# NXT-25591 — IM - Item History (ITEM DETAIL)

## Description
As Item Manager administrator I need an option where I can observe the history of a item for example updates made to the item by who and when so that have an audit process

## Acceptance Criteria
* *Entry Point*
** Item Management → Items  → Selected item → additional options icon \[More actions button]

 

* An option to view item history shall be available to the user under the more options icon  \[More actions button]
* User shall be able to see the following item property values:
* System date and Time on which the update took place
* User name which performed the update (who performed the update)
* What fields were updated, preview value(s) and new value(s)
* The information is for display purposes (Shall not be alter by the user)
* Applies to EDIT item

*General*

Page pagination shall be present (as standard way)

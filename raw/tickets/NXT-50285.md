# NXT-50285 — MP- Copy Menu Cycle- Selector for serving groups and option to copy all menus

## Description
As a menu Planner, If I am copying a menu cycle that has menus will multiple serving groups, allow me to select what serving groups to add to the copied menus within the menu cycle.

## Acceptance Criteria
* Provide an option to Copy all menus when copying the menu cycle
** Default it to off 
* Provide a serving group selector only if the copy of all menus toggle is *ON*
* Default Select the serving groups that are available on the menus if all the menus have the same serving groups
** IF ALL have one or more serving groups, check the ones that belong on all 
*** Ex: all have 6-8 but only some have 6-8 and 9-12, only check 6-8
* If different meal pattern is selected, no data should be available and show all serving groups within that meal pattern as available for selection
** Consider Meal Pattern and Meal Type combination for proper display of available serving groups
* If serving groups are different on the menus, provide blank check boxes for the users to select what serving groups to insert into the copied menus
** When serving groups are selected (Keep current counts) or additional serving groups are selected, default the projected meal counts to 100 on each menu
*** If serving groups are removed, pretend like they never existed (DO NOT COPY COUNTS)
* Minimum 1 serving group must be selected to complete the copy process IF using Copy of Menus option

# NXT-5432 — System - Licensing Summary - View Site-Level License Details

## Description
As a Cybersoft or district administrator, I would like the ability to manage the currently-assigned site licenses, so that I can make sure that the district has access to everything within PrimeroEdge that has been purchased.

+Summary+

* Within the Licensing Summary page, for all licensed modules, allow the user to view the site-level module details, as well as add or remove licenses as needed.

+Details+:

* When the user selects a module in the Site-Level Licensing details section, display: 
*# Information on number of licenses in use vs. number of licenses available, for the selected module
*# The sites that are currently licensed to use that module
*# The sites that are *not* currently licensed to use that module
*# The ability to jump to other modules
* Page content/columns can be carried over from existing solution, with new UI/UX controls such as toggles.
** Sites should also now display in alphabetical order.
* An unlicensed site can have a module license added as long as at least 1 license remains available, otherwise we should prevent the user from adding that license and/or display an error. “No licenses are available. Please remove a license first.”
* Removing a license can be done for as many sites as needed.
* Overall license count for the module should adjust as the user adds and removes individual licenses for each site. Values should update in the database.
* The user should be able to search for licenses on the page, and easily filter out licensed vs. unlicensed sites
* When a user jumps to another module, we should display the site-level licensing details for that module and allow the user to perform the operations above
* The page should no longer contain a button to view/modify licensing at a Site Type level, though we may add that feature back in the future.

## Acceptance Criteria
Verify that site-level licensing details can be viewed from within the Licensing Summary page

Verify that when a module is selected, the user is immediately presented with a detailed view of their licenses for each site for that module, including:
-Overall license count
-Licensed sites
-Unlicensed sites

Verify that sites are displayed in alphabetical order

Verify that an unlicensed site can have a module license added as long as at least 1 license remains available

Verify that licenses can be removed from any or all sites as needed

Verify that the overall license count for the module adjusts as the user adds and removes individual licenses for each site. Values should update in the database.

Verify that the user can jump to another module to view site license details and perform the functions above

# NXT-4974 — AcctMgt - Patron - Adult Details - Contact Info

## Description
*User Story*

As a district admin I want to see the contact information on an adult’s account.

*Mock-up*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1]

*Requirements*

* User should be able to load an adult off of the search page and see their information.
* Adult accounts are a subset of the information we’ve put together for students.
* The second panel will be the contact information.
* User should be able to view and edit information, like a student.
* We can mostly copy what was done for a Student here, but there are a couple of changes - 
** No additional contacts can be added. 
** The contact they are adding is the adult themselves, so the following fields are not necessary:
*** Prefix
*** First Name
*** Middle Name
*** Last Name
*** Suffix
*** Relationship

## Acceptance Criteria
An adult's information can be viewed while searching / clicking on a card.
Information can be edited appropriately.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)
Accessibility / ADA Compliance programmed appropriately (full KB nav possible, etc.)

# NXT-22105 — INV - Withdrawals - Withdrawal Report - Item Summary

## Description
As a FSD I want an Item Summary variant of the Withdrawal report so I can  see a list of withdrawn items.

**Note:* there will be four (4) total variants of the Withdrawals report

* Transaction List - new
** This will be a list of the Withdrawal transactions without any of the item details
* Transaction Details - existing SchoolCafe report on Withdrawals page
** a list of all withdrawal transactions including item level details (basically a list of Withdrawal Tickets)
* *Item Summary* (this story) - new
** a list of items that were withdrawn
* Reason Summary - new
** a detailed report which organizes withdrawal items into Reason groupings

*Requirements*

* On the Inventory Withdrawals page, provide a report button on the top right grid header 
* On click of report button, display a prompt to ask which report option
** List
** Details
** *Item Summary*
** Reason Summary
* Once report option is selected and Generate button is clicked
** generate the HTML report on click of the button based on the list filters
** show only current versions of transactions
** do not show cancelled transactions
* Display header (search) level information in the report
** Report Name: Inventory Withdrawals
** Area (if applicable)
** Site 
*** display ‘ALL’ if all sites were selected
** Withdrawal Type
** Date Range
** Reason
* *Ignore field search* for this Summary report
* +Display item level withdrawal transactions+
** Item #
** Item Description
** Quantity (+ description, obviously)
** *Sort by item description/name alphabetically ascending

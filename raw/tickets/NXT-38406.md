# NXT-38406 — Imports – History Log – Show Original Import Data - Dev only API

## Description
As a SchoolCafé admin, I would like to see the original file data that was sent to the import engine so that I can use it to diagnose any issues.

*Notes:*

* This request comes from one of our new customers
* It is most helpful for automated imports

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* Click on View Log button
* Click on Source Data tab
* View Data and optionally export it

*Requirements*:

* All these requirements are for the View History screen
* All these requirements should be added to every import type, but begin with Student and Adult
* 
* Add a new tab to the right of the other tabs
* Title: Source Data
* 
* On the Source Data tab, do the following:
** Add a grid
** This grid is to not be hard coded to any dataset
** Column headers are dynamic and come from the source dataset itself
** The rows come from the source dataset as-is
* Grid should have usual paging, sorting, filtering
*

## Acceptance Criteria
* Source Data tab exists and has grid on it
* Grid is dynamic and shows the source columns and data, whatever they are and however many they are
* Grid has paging, sorting, filtering

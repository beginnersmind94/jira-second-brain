## Acceptance Criteria
Navigation: Item Management - item Upload - item Updates

Default Data Provider - 3rd Party Source(s)

The UI should automatically load the items which has updates compared to local items

Upon clicking on item name (hyper link), the UI should show the comparable page that shows current and available update

By default - all the attributes are selected



The serving measure and nutrition and contribution box should be coupled - if the user deselects serving size related fields, then other section should be deselected as well and vice versa

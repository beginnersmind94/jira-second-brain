# NXT-8910 — INV - Physical Inventory - Show/Hide Supervisor options based on permission

## Description
As an Inventory district manager I want permissions for the Physical Inventory “supervisor” functionality so I can control who has access to the increased functionality on Pending Reconciliation screen.

*Requirements*

Create permission based functions:

* Discrepancy view
** enable the Discrepancy View option and enable the availability of the Discrepancy columns
* Valuation view
** enable the Valuation View option and enable the availability of the Valuation ($) columns
* Storage+Item Category
** enables the Storage and Item Category organization option
** this should be always enabled/default
* Valuation Group organization
** enables the Valuation Group organization option
* Note: these will always be displayed regardless of permission:
** Item Category filter dropdown
** Storage Category filter
** Reset/Apply button

Provide a report generation button/option (all individually controlled by permission)

* Physical Inventory Report 
* Discrepancy Report
* Projected Valuation Report

Adjust the UI accordingly when users do not have certain access

* Example 1: user only has Storage & Item Category organization option and Default view option. These options (Organization, View) could be hidden since they are irrelevant to the user. The best thing for this user is the Storage and Item Category dropdown filters.
* Example 2: user only has access to Valuation view. When this user loads the screen, the grid should already be in Valuation view mode with Valuation columns.

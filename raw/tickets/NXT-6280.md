# NXT-6280 — System - Sites - Add/Edit Site - Update 'Areas' field to display other types of records

## Description
Currently, when adding or modifying a site, there is no way to designate that site as being a part of a particular area or sub-region, as the ‘Area Name’ field only displays districts:

!image-20201001-154949.png|width=625,height=309!

Let’s update this field so that it displays the types of regions below:

* Areas
* Districts
* Regions

The following two are *unnecessary*:

* Districts
* Counties

This change should allow users to select the proper site when using Area filters on other pages.

## Acceptance Criteria
Verify that the 'Areas' field in View/Add Sites it displays the types of regions below:
Areas
Districts
Regions

Verify that the following two are *not* displayed: Districts, Counties

Verify that this change allows users to select the proper site when using Area filters on other pages where the 'Area' filter is located.

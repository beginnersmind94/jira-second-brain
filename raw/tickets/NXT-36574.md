# NXT-36574 — Student Import – SIS Student Imports use SQL Validations - Dev only

## Description
We need to update the validations for the SIS Student Import so that they are consistent with the way the manual imports are done which will save system resources and provide consistency.

*Notes:*

* We already did the updates to the ad-hoc imports
* We already have the SIS auto-imports
* We need to update the SIS auto-imports to use the same validations as the ad-hoc (SQL instead of API validations)

 

*Requirements*:

* For the SIS Student Imports, do the following:
* Turn off API validations
* Enable to use the SQL validations that were created for the manual imports

## Acceptance Criteria
* SIS Student Imports use SQL Validations instead of API validations

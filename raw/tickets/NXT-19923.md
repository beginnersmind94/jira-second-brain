# NXT-19923 — Production- Dashboard- Production Plan and Menus assigned display

## Description
As an Operations Manager, I would like to be able to see a quick overview of my production statuses from around the district, site group or by site to make sure that all Kitchen/ Production Managers are handling their tasks in a timely fashion. 

Display on dashboard widget 

 Total Menus assigned, Plans Incomplete, Plans Complete.

EX: 

District View:
Current Menus assigned - 450
Plans Completed - 125
Plans Incomplete - 375

Site Group View: Highschool
Current Menus assigned - 290
Plans Completed - 90
Plans Incomplete - 200

Site View: School Cafe High
Current Menus assigned - 40
Plans Completed - 16
Plans Incomplete - 24

User can select view based on:
ALL Menus assigned (Look for last possible menu assigned as the end date)
Date Picker ( Can pick any date as end date in the future, all dates in past disabled)
21 Days (Next 21 days)
14 Days (Next 14 days)
7 Days (Next 7 Days)

Give link to "View More Details" which will take them to the *production status page.*

# NXT-14223 — Direct Certification - Trigger Secondary Matching

## Description
As a district admin I want the system to perform automatic DC matching whenever a student is updated. Right now, it does this for new students being added, but not for updates.

*requirements*

* When a student record is updated via the Account Management pages, check their information against uploaded DC file information to see if there are any hits.

## Acceptance Criteria
Manual updates to student information result in a check for potential / automatic DC matches.

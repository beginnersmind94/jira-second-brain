# NXT-5647 — INV - Create Orders - View Created Orders

## Description
*User Story*
As an inventory user creating an order I need a summary view on the Confirmation screen so I can view the order details of the orders which I have confirmed/created.

On the Confirmation screen, once one or more orders have been officially created (saved as Open), the grid for each of the created orders should update and reflect that the order has been created.

Story scope: UI changes after orders have been saved as Open (per [NXT-15883|https://primeroedgenext.atlassian.net/browse/NXT-15883])

*Requirements*

For successfully created orders:

* The status should update from ‘Incomplete’ to ‘Open’
* The Edit icon should be removed (only for *Open* orders successfully created)
* Provide a View icon in place of the Edit icon
** View icon would open the corresponding Order Detail 
* The orders which are in Open status will be collapsed by default
* display success message…
** at the top of the page: “Success\! 3 orders have been created with status of Open”
* Remove the ‘Create’ button if all orders on the screen have been saved as Open
* Back button is removed if any orders have been saved as Open

If there are still orders which have *not* been created, those will remain on this page as usual. 

* List all Incomplete orders before the Open orders
* Incomplete orders will remain editable and will not change
* Create button will only be displayed if any Incomplete orders remain

Mockup note: expand/collapse arrows will still be present

## Acceptance Criteria
* Screen elements will update for orders that are created
* Order # will be provided and will be a link to the Order Detail page
* Success message will be displayed for orders which are created
* Incomplete orders will be listed before the new Open orders

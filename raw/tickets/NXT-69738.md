# NXT-69738 — ACC - Square > Device Configuration Setup Page  Filters, Revoke butt, New Auth button, and Main Grid

## Description
[https://developer.squareup.com/docs/oauth-api/overview|https://developer.squareup.com/docs/oauth-api/overview|smart-link] 
For implementing support for Square(TM) terminal we will need to create a configuration page for it all to begin.

The configuration page will be where we connect Square APi’s to ExpressPoint devices so that transaction can properly be tracked at the correct POS terminals.

We will be creating a new page under System > Management > Credit Card Terminals.

The page will consist of a standard grid with sub header sections to visually distinguish Square Attributes vs SchoolCafé Attributes, Page filters for the main grid, and several button functions for setup and maintenance of Square Devices.

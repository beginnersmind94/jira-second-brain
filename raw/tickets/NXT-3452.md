# NXT-3452 — System - Management - Add Languages page

## Description
As a district administrator, I would like to easily be able to configure which languages my district supports and have them correspond to the letters we send out as well, so that we can more effectively communicate with our households.



* In the System - Configuration category, add a *Languages* page
* This page was formerly in the Student Eligibility module of Classic.
* When the user navigates to this page, display a grid with the same data as on the Classic page, but with the new grid layout.
** The user should be able to see English and Spanish by default, plus any language they’ve added previously.
** The user should be able to edit an existing language’s attributes (IsActive, Show in SchoolCafé)
*** When selecting a trash bin icon respective to a language, we should hide the language from the page  and deactivate it (we cannot delete it from the database). 
** The user should be able to add a new language (from a drop-down list which contains all known global languages)
** When a language is added, we must add a ‘Standard’ letter template in each of the letter template types for that language. However, the actual letter template can be in English by default.
*** Essentially, when a user adds a new language, duplicate all standard letters and make the duplicate associated with the new language.

## Acceptance Criteria
The user should be able to see English and Spanish by default, plus any language they’ve added previously.

The user should be able to edit an existing language’s attributes (IsActive, Show in SchoolCafé)

The user should be able to add a new language (from a drop-down list which contains all known global languages)

The user should be able to hide a language (NOT English or Spanish) by selecting the trash bin icon.

Language updates should be saved in the database.

In Notification Templates, for all existing letter types, a new language should be visible in the list

In Notification Templates, for all existing letter types, a new Standard letter (duplicate of existing English standard letter) should be visible in the list when the new language is selected

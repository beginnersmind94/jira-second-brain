# NXT-26994 — MP- Menu Planner- Nutrient Calculations for daily menu

## Description
The Menu planner is now calculating the nutrients for all menu items that are added to the menu. We are not taking into consideration the choose counts and how the average of the calories will take into effect when calculating nutrients. 

In AC, it will list out all calculations that need to be handled for nutrients.

## Acceptance Criteria
WE are currently defaulting the serving group meal count to 100. With that in consideration, this is how we should be calculating nutrients. 

Each Menu Item Category has a default count of 1 except veg has 2. 

If a user drops 1 menu item for the category, calculate 100% of that menu item for the nutrients values (100 servings)

If a user drops 2 menu items for the category, calculate total nutrients for the menu items divided by 2 (50 servings (Menu Item 1), 50 servings (Menu Item 2))

If a user drops 3 Menu items for the category, calculate total nutrients for the menu items in this fashion by alphabetical

34 servings (Menu Item 1)

33 servings (Menu Item 2)

33 servings (Menu Item 3)

If a user drops 4 Menu items for the category, calculate total nutrients for the menu items divided by 4 (25 servings (Menu Item 1) 25 servings (Menu Item 2), 25 servings (Menu Item 3), 25 servings (Menu Item 4)

If a user drops 5 Menu items for the category, calculate total nutrients for the menu items divided by 5 (20 servings (Menu Item 1) 20 servings (Menu Item 2), 20 servings (Menu Item 3), 20 servings (Menu Item 4), 20 servings (Menu Item 5)

If a user drops 6 Menu items for the category, calculate total nutrients for the menu items in this fashion

100/6= 16.666667 

Calculate the closest whole number to 100 

16x6=96 with 4 left over to get to 100. Use the extra 4 and give each of the first 4 Menu items 1 serving.

16+1=17= 150 calories(Per serving)X17=2550

16+1=17= 175 calories(Per serving)X17=2975

16+1=17= 200 calories(Per serving)X17=3400

16+1=17= 140 calories(Per serving)X17=2380

16= 50 calories(Per serving)X16=800

16=75 calories(Per serving)X16=1200

Total calories= 13305 for all servings

Apply the same method for any number that can not be evenly divided into 100. 

In the case of any category that has choose count as 2 such as veg, double the counts of each menu item and follow the same structure for the amount of items added in the category. 

EX: 

2 items= 100 servings for each

3 items: (34x2)=68, (33x2)=66, (33x2)=66

4 items (25x2)=50, (25x2)=50, (25x2)=50, (25x2)=50

In the case of any category that has choose count as 3 or more. Multiple the counts indicated for choose 1 by the number of chooses to get the number of servings per menu item.

# NXT-69308 — SC - MT - User Profile - Roles and permissions Grid (workspace and system)

## Description
* As a user with access to multiple districts, when user view the roles and permissions, user want to see them displayed in a grid format so that I can easily understand the available districts along with their corresponding roles and permissions.

## Acceptance Criteria
* Roles and permissions should be displayed in a grid format.
* The grid must show available districts corresponding to their roles and permissions.
* Client-side pagination should be implemented for better navigation.

# NXT-5353 — Workspace - Tasks Section - Recommended Tasks

## Description
*User Story*
As a PrimeroEdge User, I can see automated system tasks under the Recommended Tasks heading of the Workspace landing page.

*Mockup*
[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=222364%3A4744&scaling=scale-down-width&hide-ui=1|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=222364%3A4744&scaling=scale-down-width&hide-ui=1|smart-link] 

*Requirements*

* Recommended Tasks is a tab under the Tasks section on the Workspace
** Shared with My Tasks (NXT-5523)
* Generation of tasks in this section is done automatically by the system
** This will be covered by another story
* Option to select time period to display tasks based on due date
** Today, This Week, This Month
* Gauge to show total number of tasks in the selected time period and how many have been marked as completed.
* Resize vertically to include all tasks in the selected time period.
* Display Task Name and whether a task has been completed via check mark icon.
** Clicking Task Name will take user to a popup with a detailed description of the task.
** Clicking on the check mark icon of an uncompleted Task will change the status of the Task to completed and change the icon.
** In This Week and This Month, a due date is shown for each task
* New Task button is present, but any tasks added are added to the My Tasks section
** Functionality should match that in [https://primeroedgenext.atlassian.net/browse/NXT-5523|https://primeroedgenext.atlassian.net/browse/NXT-5523|smart-link]
* *NEW* - When no tasks exist in a particular date range, display a message: “You’re all caught up\!”
* View All Tasks button to navigate to a separate Task page that shows all Tasks assigned to the logged in user.
** Covered in a separate user story

## Acceptance Criteria
- Recommended Tasks section is present in the workspace
- Tasks are appearing properly
- Functionality present as described in requirements
- When no tasks exist in a particular date range, display a message: “You’re all caught up!”

# NXT-25258 — Family Hub - Admin - Reports - Add Menu Interactions Report

## Description
As a Family Hub admin, I would like to be able to view a list of all menu items and their average ratings, so that I can use that data to improve our meal planning.

*Summary*

This report existed in Classic SchoolCafé (before v.2) but was cut. Let's restore it so that an admin can easily view all menu items in their district and the average rating, even showing a breakdown between user type, site, meal type, and category if possible.

*Details*

* In Family Hub admin → School Menus → Reports, add a new report: *Menu Impressions*
* Filters:
** Site
*** Default option: All
*** Single-select
** Menu Item
*** Default option: All
*** Single-select
** Apply/Search button:
*** When selected, display the list of item(s) at the site(s) selected along with:
**** Likes: # of users who have thumbed-up the item 
**** Dislikes: # of users who have thumbed-down the item
**** Favorites: # of users who have favorited the item currently
**** Saved Trays: # of users who have saved the item to a tray
**** Average Rating: Average number based on likes (5.0) and dislikes (1.0) divided by the total of both. (May need a tooltip explaining that ratings are district-wide, unless we’re able to narrow this down to being by site).
* Grid should display the columns above and should have a search field. 
* Columns should also be sortable.
* The report should allow the user to print/export it.
* Like/dislike/etc. data can be lifetime, not necessarily based on a date range.

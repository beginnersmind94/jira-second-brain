# NXT-8116 — INV - Physical Inventory (Pending Reconciliation status) - Items tab

## Description
As an inventory user I want the Pending Reconciliation inventory screen to have a dedicated Items tab so it is more similar to the other inventory status screens.

*Requirements*

*  Move the Item List to a dedicated Items tab
* Provide the same Items List grid from other screens
** Sorted by Storage Category then Item Category
** Item #
*** searchable column header
** Item Description
*** searchable column header
** Usable Quantity
** Damaged Quantity
** Perpetual Quantity
* Provide Modify, Reconcile, Cancel buttons at top 
** these action buttons are available regardless of tab selection

!image-20201202-230522.png|width=91.66666666666666%!

## Acceptance Criteria
Items will be moved to a dedicated Items tab
Items tab will have the Items list
Items list will have a table containing the items in read-only view (uneditable)
Items list will be searchable by Item # and Item Description

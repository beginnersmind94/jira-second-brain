# NXT-5052 — Messages - Set Up Email Service

## Description
*User Story*

As a PrimeroEdge user, I have an email service that allows sending any relevant information from PrimeroEdge to a single email address or multiple email addresses.

As a PrimeroEdge developer, I have a service I can hook into for any email functionality I’m implementing.

*Requirements*

Using [Twilio|https://www.twilio.com/]’s email service, we want to be able to send emails from PrimeroEdge to external addresses. Set up the service, ensure it can be used to send a test email to a work email, and document any information needed for future development.

## Acceptance Criteria
* Email service implemented and able to send test emails that are successfully received
* Any relevant information related to future implementation documented on Confluence

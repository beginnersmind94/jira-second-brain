# NXT-7261 — INV - Physical Inventory - Main Screen - Items tab (Physical Incomplete status)

## Description
As a site manager I want to be able to view or print a count sheet so I can prepare to count my items.

[Figma|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=90479%3A149622&viewport=325%2C-4288%2C0.08251962065696716&scaling=scale-down]

Count Sheet can be generated in several ways.

Once a Physical Inventory has been created and a Storage Category is selected, those items will be loaded on the Items tab.

Items will be grouped by Storage Category then Item Category

* Attributes/fields:
** Checkbox for Item selection
*** all items are checked by default, but user can deselect an item to remove it from the inventory
**** when item is deselected, it is not fully removed but disabled for counting. Add strikethrough text
*** this would be used if this item is not valid or user does not intend to count it
** Item # - pulled from the item (Item Management)
*** searchable column header
** Item Description - pulled from the item (Item Management)
*** searchable column header
** Usable Whole Units - a numeric entry field 
** Usable Broken Units - a numeric entry field 
** Damaged Whole Units - a numeric entry field 
** Damaged Broken Units - a numeric entry field 
** Comments/Notes - a text field (popup, inline, etc.) to enter additional comments related to that item. 
*** Example entry: “3 damaged cases”

Action buttons:

* Save as Incomplete [https://primeroedgenext.atlassian.net/browse/NXT-7864|https://primeroedgenext.atlassian.net/browse/NXT-7864|smart-link]
* Print Physical Count Sheet

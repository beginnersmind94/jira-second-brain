# NXT-9651 — INV - Physical Inventory - allow users to modify a Reconciled inventory - UI & Research

## Description
As a Supervisor I need the ability to modify a Reconciled inventory so I can make necessary changes after the inventory is complete.

*Requirements*

* Create a new permission (district level) that allows user to reopen and modify a Reconciled physical inventory.
** Permission name: Modify Reconciled Inventories
** Permission details: This permission allows user to modify a Reconciled Physical Inventory.
* Provide Modify button on the Reconciled inventory screen if user has the permission
** currently there are no action buttons on the Reconciled screen.
* Provide a prompt/warning to users when modifying the reconciled inventory.
** Use the same Comments popup used for Modify button when Inventory is in Pending Reconciliation status

*Validations*

* Check for any physical inventories after the reconciled one being reopened.
** If Physical Inventory exists _after_ the one being modified, do not allow user to cancel Reconciliation and modify.

*Details*

* Cancel the Reconciliation transaction and allow the user to edit the physical inventory counts. (This will be handled in[ NXT-8267|https://primeroedgenext.atlassian.net/browse/NXT-8267])
** Research in current sprint

## Acceptance Criteria
Create new permission for Modify Reconciled Inventory
Enable Modify button on Reconciled screen if user has the permission
Research reverting Reconciliation transaction

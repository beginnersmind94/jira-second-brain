# NXT-37458 — Imports – View Log – Add Excel Export

## Description
As a SchoolCafé Admin I want to be able to export to excel so that I can download and review the log later in excel for History, Exceptions and Source Data.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* Click on View Log button
* Click on History Log, Source Data or Exceptions tab
* Click Export button

*Requirements*:

* Would be great if it was done for every import type, but do at least for Student Import
* 
* On the grid for the History Log, Source Data or Exceptions tabs, do the following:
* 
* Add an export button to the grid
** Label: Export
** On click: download the data in the grid to Excel

## Acceptance Criteria
* Able to download the data in the grid into excel

# NXT-29044 — INV - Create Withdrawal - Items + Confirmation - Pack Size updates

## Description
As a Site Manager I want the Withdrawal functionality to allow me to submit a withdrawal for any unit in an item’s pack size configuration so I can enter the exact quantities I am withdrawing from my perpetual inventory.

*Requirements*

* Update the Withdrawal screens to reflect the updated item hierarchy configuration
** Items will be searched and managed at the Item # level throughout the Inventory category
*** Ex: I need to withdraw 2 cases and 4 boxes of Item 100 - Orange Juice
* Display Quantity on Hand
** Present the full string
** Provide Alternate QOH popover  ([NXT-26951|https://primeroedgenext.atlassian.net/browse/NXT-26951])
* Other Improvements
** Step 1 selections should be retained when returning from Step 2
* Update behavior for adding items to the list
** Change the Item search to a multi-select dropdown
** Update the Template selection to simply pre-select the template items in the Items dropdown
** Add button logic
*** If multiple items are selected, disable the Quantity field and enable the Add button
**** Could add tooltip/label showing ‘Quantity is disabled since multiple items are selected.’
* +Update the UI to accommodate multiple unit types+
* Replace Unit Type with Unit Description
** remove the dedicated column, as Unit will be selected after Quantity
* Provide a Unit dropdown after the Withdrawal Quantity
** Display a label instead of the dropdown if there is only one unit in the pack size configuration
** By default, pick the highest unit for which there is quantity on hand
* Provide an icon to expand the pack size hierarchy
** Only if the item has multiple possible units
*** this adds rows below for each sub-unit in the pack size hierarchy
** Remove the dropdowns and lock the row to the specific unit label
** Provide an icon to collapse the pack size hierarchy if the sub-units aren’t being used
* Fix styling to be consistent with Additions and Transfers
** Left align header selections 
*** on Step 1 as well
* Change ‘Quantity’ to ‘Withdrawal Quantity’ on column header

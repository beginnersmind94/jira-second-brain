# NXT-28653 — INV - Transfers - Create Transfer - Do not allow negative inventory

## Description
As a Site Manager I want the Create Transfer screen to prevent me from transferring more than is available so I do not create negative inventory quantities.

*Requirements*

* Determine if the Transfer functionality is allowing negatives
** This may be considering the System Setting for True FIFO
*** this setting should be removed or hidden per ([NXT-28654|https://primeroedgenext.atlassian.net/browse/NXT-28654]) because we are not planning to allow FIFO to be disabled 
*** remove the logic to consider this setting
* Prevent transfers from creating negative perpetual values like all other Inventory transactions

# NXT-33430 — INV - Physical Inventory (Incomplete status) - Row based view for Pack Sizes

## Description
As a Site Manager I want a better item grid for Physical Inventory item list so I can visualize the hierarchy and ensure I am counting the correct unit type.

The real use case of this view is that it enables us to handle different unit types having different locations, which will be needed in the future with Storage Locations or especially with Bins.

*Requirements*

* Update the Physical Inventory item screen (Incomplete status) to accommodate updated Pack Size configuration
* Provide two views for managing hierarchy info
** Column-based view - a column for each unit type, somewhat like PrimeroEdge 
** Row-based view - each item on its own row, units shown linked together

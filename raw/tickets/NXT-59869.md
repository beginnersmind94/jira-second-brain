# NXT-59869 — IM - Item Onboarding - Review Nutrients - New Enhancement

## Description
We are not showing (M) for missing values that should have been entered by the user. Follow the same logic we are using for the update process where the fields are not considered as (M) unless the user has entered a value for 1 nutrient value or vitamin value, depending on the section. We can also do a red (M) for the mandatory fields and an orange (M) for the Non Mandatory. DO NOT SHOW (M) if nutrient values have NOT been entered for a particular item for the view nutrients section and if vitamin values have NOT been entered in the view Vitamins section.

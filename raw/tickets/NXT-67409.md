# NXT-67409 — Acc - Claims - Page Updates

## Description
As Dana the Director I want better support for Reimbursement Claims - a way to attach documents, reports, and a rollback feature.

!image-20251013-211025.png|width=570,alt="image-20251013-211025.png"!

*requirements*

* Break the dependency on a DataExchange export existing
** We should only care about DE for export - not for creating the claim itself.
* Update the grid
** Remove Academic Year column
* Add CACFP
** Column to the main grid
** CACFP Report drawer
** Check Program Config to see which program columns should be shown
* Add a button that takes the user to Program Management
* Use the Amount columns to provide context:
** Clickable link if the claim has been generated which leads to the Reports action
*** Example: Click on SNP for August 2025, get the SNP Reports drawer
** User can see if a claim has not been created, or if it is queued or currently generating
* Actions limited to Create / Recreate and leads to the workflow wizard in [https://primeroedgenext.atlassian.net/browse/NXT-67408|https://primeroedgenext.atlassian.net/browse/NXT-67408|smart-link] 
* Additions to the SNP / SFSP / CACFP Report drawers:
** Move the History action drawer from the main grid into a new tab
*** History & Supporting Docs
*** User can see / download the Supporting Docs from [https://primeroedgenext.atlassian.net/browse/NXT-67408|https://primeroedgenext.atlassian.net/browse/NXT-67408|smart-link] 
** Move the Export action drawer from the main grid into a new tab called State Export
* Updates to Reports:
** Summary → Students Approved → Add Paid and Total
** Detailed → Add some lines so it’s easier to see where each sites counts are:
!image-20251013-212120.png|width=876,alt="image-20251013-212120.png"!

** Standard Export → Add explanation text: This is a standard export format that highlights meal counts, approved counts, and attendance figures for each site.
** State Export → If we don’t have a state export available, inform the user:
*** There is no configured export for claims. If a claim format exists for your state for uploading purposes, please send the configuration through to our Support team.

## Acceptance Criteria
* Claims can be created without an Claim Export existing
* Grid updates:
** Academic Year removed
** CACFP column added - tied to Program Config
** Actions moved to Drawer Tabs: History, Export
** SNP Report / SFSP Report / CACFP Report available via the $$ amounts
** User can see if a claim has not been generated, is in progress, or is queued.
* Report updates:
** User can see approved counts for free, reduced, paid, and a total.
** Line separator added for each site in Detailed.
** History shows Supporting Docs
** Export tells the user if no claim format exists for them.
** Explanation text added to Standard Export tab.
* User can delete the claim, and it will roll back to the last one before it.

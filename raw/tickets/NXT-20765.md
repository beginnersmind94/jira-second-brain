# NXT-20765 — Menus API

## Description
There needs to be an API that exposes menu items from districts and sites to the FFD Website that mimics the classic API

*Application*: API

*Flow:*

* Source system needs to know what food items are on the menu for a given district or site
* Calls code to get the items

*Assumptions:*

* This API exists in the FFD website for classic called Get Menu Items
* You need to do the same thing for 2.0 that is done for Classic
* Talk to Saketh if you have any questions

*Requirements:*

* Needs to be able to return menu food items per district or site with the exact same fields, methods, etc. as the current Menu API from classic
*  
* Security – only Cybersoft code should be able to access it

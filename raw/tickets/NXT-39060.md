# NXT-39060 — MP- Menu- Breakfast- Never allow Variety Menu Item to toggle MMA as Grain

## Description
As a menu Planner, Do not allow me to toggle MMA as a Grain for Variety menu items if they happen to have MMA. Since the variety menu item has component criteria that the menu items need to fall under, we should not allow that to be changed on the menu.

## Acceptance Criteria
* If VMI “parent” that is added onto a menu happens to have MMA, do not allow user to be able to toggle it as a Grain
** Option should not be available to even be toggled

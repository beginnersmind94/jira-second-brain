# NXT-30216 — IM - Item Details - POS Info - View/Edit

## Description
As a District Admin I want a POS Info tab so I can view a POS item’s attributes and pricing.

*Requirements*

* Display and populate the POS tab for items which were defined as a POS Item
** if the item was not defined as a POS Item, disable the tab and color it gray
* Display the attributes as labels in view-only mode
** Button Name
** Button Name Spanish
** Meal Type
** POS Item Category
* Display the checkboxes/flags accordingly
** do not allow them to be changed in this mode
* Display the pricing grid in an appropriate view-only mode
** hide the ‘Set All Prices’ field/button
** Display the pricing-specific checkbox fields
*** Show the Generic Item being used, if ‘Use Generic Pricing’ flag is checked
* Provide an Edit button for the entire POS section
** Enable the freetext and dropdown fields for attributes
** Enable checkbox attributes
** Show the ‘Set All Prices’ field/button on the pricing grid

## Acceptance Criteria
* Display the POS Info tab for POS Items

* Disable the POS Info tab for items which are not POS items

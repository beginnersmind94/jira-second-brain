# NXT-50454 — Reports - Sites

## Description
As a Customer Support role member, I would like to view reports on district sites in one page so that I can use that information to manage our customers.

*Notes:*

* The scope of this page is for ALL districts at once, not a single district
** It is like a global dashboard, used for managing all licenses
** Do not restrict visibility to just data for a single district

*Flow:*

* User browses to Reports > Global Dashboard > District Reports Dashboard
* User sees a KPIs, graphs and data points for District sites

*Requirements:*

* Add a new page to the resource catalog
** Title: Site Reports Dashboard
** Put it under Global Dashboard
** Disable it from being managed in the Role and Permissions front end

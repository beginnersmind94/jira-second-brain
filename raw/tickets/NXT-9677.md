# NXT-9677 — System - Program Config - Add tab for Carryover Eligibility (formerly 'Grace Periods')

## Description
As a district admin, I would like to be able to set the Carryover dates for each site in my district, so that the system knows when to have each student’s Free or Reduced meal eligibility to start and end for a given academic year.

*Requirements:*

* In System → Management → Edit Program Configuration, add tab to the page:
** Carryover Eligibility - This tab will contain carryover date info (concept image below)
* In this tab, display the sites who belong to that District, along with some filters:
** Academic Year - Includes last 10 years
** Sites drop-down for that specific district
** Set for All Sites button (optional for the user) - 
*** *Labeled as ‘Apply’ in the mock-up, but should be labelled  ‘Set for All Sites’ instead.*
*** When selected, open a pop-up (may not be reflected in mock-up) with an OK/Cancel: “Carryover period end dates will be updated for all sites.”
*** After user clicks OK, immediately saves the selected Start and End Date for all sites in the organization.
** Start Date & End Date (optional for the user): 
*** Start Date - +*If*+ ‘Set for All Sites’ button is hit, then this field…
**** Cannot be edited (in any scenario) if the date has already passed
**** Must be greater than the End Date of the latest Carryover Period in the previous Academic Year 
**** Must be less than the End Date of the latest Carryover Period in the selected Academic Year
*** End Date - +*If*+ ‘Set for All Sites’ button is hit, then this field…
**** Must be less than the Start Date of the earliest Carryover Period in the next Academic Year 
**** Must be greater than the End Date of the latest Carryover Period in the selected Academic Year
*** Also check grace period/Carryover settings to make sure we are respecting those when allowing the user to make updates.
** Each site should be editable individually, either before or after (or instead of) using the ‘Set for All Sites’ button, so that the user can set Start and End Dates for each site.
** By default, sort the sites by Site Name, in alphabetical order.

## Acceptance Criteria
* User shall be able to see “*Carryover Eligibility*” tab under Program Configuration edit page
* User shall be able to select the date and click on “*Set for all Sites*” button
* User shall be able to see the filters Academic year, Sites
* User shall be able to see all sites of the current district that user is trying to update carry over period
* User shall be able to edit the carryover date after applying to all sites (at district level)
* User shall be able to *Save* or *Cancel* the carryover date edit
* Carryover/Grace period settings should be respected to the best of our ability

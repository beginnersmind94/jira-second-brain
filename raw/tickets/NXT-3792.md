# NXT-3792 — AcctMgt - Patrons - Search Results - Table View

## Description
*User Story*

As a district administrator I want my quick search to show me results in a table view. 

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5]

*Requirements*

* Display the results from the Search button in a table.
* All table columns can be sorted.
* Table columns can be dynamically rearranged with drag and drop.
* -Additional columns can be added. Table should update dynamically when additional columns are added.- 
** -Add a symbol on the corner of the table that reveals a list of the additional columns.-
** -All columns can be selected or deselected - checkbox to show active table columns.-
** -Example: Grade would show as a default column in the table. It would have a checkbox in this list. User can uncheck Grade, and Grade disappears from the table.-
* Default columns for the table:
** Patron ID
** Last Name
** First Name
** Site
** Grade
** Eligibility
** Balance
** Status
* -Additional columns:-
** -Birthday-
** -Patron Type-
** -Status-

## Acceptance Criteria
Search fields/filters.
Table added with results per the search fields/filters.
Table features available.
- Sorting
- Column add/hide.
- Column drag and drop.

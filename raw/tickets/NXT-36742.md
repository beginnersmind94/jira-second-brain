# NXT-36742 — IM - Recipe Rework - Recipe Steps - HACCP - Add/Edit CCP

## Description
As a Nutritionist I want the HACCP options available on recipes so I can easily assign required control points for my recipe preparation.

*Requirements*

* Once HACCP Process is selected, provide the ‘Add Control Point’ sub-option to the Add Steps button
** [https://primeroedgenext.atlassian.net/browse/NXT-36583|https://primeroedgenext.atlassian.net/browse/NXT-36583|smart-link] 
** This would display an add/edit view very similar to Add/Edit Step but would provide other fields in place of Directions
*** Split button example: [https://mui.com/material-ui/react-button-group/#split-button|https://mui.com/material-ui/react-button-group/#split-button|smart-link] 
** e.g. if user selected ‘No Cook’ process, they will only see CCPs associated with No Cook (per the config page)
*** If user selects *No HACCP Process*, disable the Add Control Point button/option
* First, users must select one of the CCPs from the list of CCPs configured for their selected process
** Use the ‘Ingredient Search’ card as a ‘Control Point selection’ card
** Show the list of associated Control Points in the side card
** Show the Description and Critical Limit
** Show the Corrective Action in an accordion expandable area
** Allow a single CCP to be selected, at which point it can be added to the Steps
* Upon selection, the remaining fields can be populated (in the Steps card area)
** Critical Limit 
*** the temperature field(s) should be edited, but exactly how depends on which CCP is selected 
** Corrective Action
*** this will load with the configured value but can be edited
* With the required fields populated (via default or user values), the CCP can be saved
* Insert CCPs into the recipe as steps [https://primeroedgenext.atlassian.net/browse/NXT-36743|https://primeroedgenext.atlassian.net/browse/NXT-36743|smart-link]

# NXT-60890 — WS-PE-Insights-Learning-AI Bot

## Description
As a SchoolCafé user, I want to be able to use an AI chat bot on a page to query documentation so that I can get help information in an AI chatbot format

*Notes:*

* This chatbot should do the exact same thing as in Workspace, we are just including it on another screen inside SchoolCafé 

*Phase 1 Requirements:*

* Move the chatbot button to the top of the screen in the header row where the other buttons are (like favorites)
* Verify the user’s district has AI system setting and  Learning AI system setting set to true for both
** If not, hide the the AI Assistant screen
* Change the event click to open a drawer instead of a modal
* 
* In System Settings, add a new property
** Setting Name: Enable LearningAI
** Type: Internal
** Code: LEARNAI
** Value: Yes/No
** Default: No
** Category: Configuration
** Module: System
** Only Customer Support and tech support roles can make this change

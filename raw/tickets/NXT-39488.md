## Acceptance Criteria
* Build logic to enable the new Physical Inventory Summary Report
** this will be the report button provided when a single site is *not* selected
*** currently there is a report when a single site is selected
** Consider the selections of the page for determining Site, Storage Category & Item Category report filters
* Enable an Action column for each item to generate a limited version of the report
** This will generate the report for only the selected item (subset of main report)
** this will only be enabled if more than one site is selected
* Build the report in PowerBI
** Provide the report selections at the beginning of the report
** List each item on a new page
** Provide Site Type grouping in grid
*** List alphabetically ascending by Site Name within Site Type grouping
*** Display Site Type totals
** Display Item total
** Adjust label text when a subset of all sites is selected (i.e. multiple sites but not all sites)
* Update both single site and Summary reports to include a label for the page radio option setting
** “Include Zero Quantity Items: Yes/No”

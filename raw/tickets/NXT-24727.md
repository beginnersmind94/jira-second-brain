# NXT-24727 — Acc - Daily Checklist - Area Manager / Director View

## Description
This is a 'supervisor' view that lets them see if all sites they have control / access to have taken care of their daily checklists. Any notes that have been added, etc.

Likely want to see percentages, ability to filter who has and hasn't completed, etc.

* Roll this data up to a director view.
* Director has the same page, but a district-wide view that shows all sites and their statuses.
** This could be 100s of sites, so we may want to consider sorting by area (if available), then by site type, then by site name.
** We may just want to show each site name and their completion percentage for that date.
* Director can filter for sites that are done for the day (all tasks complete) and those who are not yet finished.
* Director can click down into any of the sites to see the site manager’s dashboard, and return to the director’s view.

## Acceptance Criteria
User can see statuses for all sites they have control over for the selected date / week.

# NXT-6441 — Messages - New Message -Image & Other Attachments

## Description
*User Story*
As a PrimeroEdge user sending a message, I have the ability to add attachments to my message to deliver files or images.

*Mockup*
https://react-material-kit.devias.io/app/mail/all

*Requirements*
* Attachments can be added when composing a message
** Images are added directly into the message using the image button
** Non-image files are attached to the message
* Attachments are subject to a 5mb file size limit
** This applies to the sum of all attachments
* Users can access attachments when viewing a message
** Images are displayed as embedded within the message
** Non-Image files must be downloaded to be viewed
* Files are included when a message is replied to or forwarded

## Acceptance Criteria
* Files can be attached to a composed message, including embedded images, and are subject to the file size limit
* Files can be viewed and accessed by the user or users who receive the message
* The file size limit message shall be: "Your attachment exceeds the size limit of xyz.“  (Where “xyz” is the actual limit you are checking against.)

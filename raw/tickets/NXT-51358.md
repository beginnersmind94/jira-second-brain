# NXT-51358 — MP- Menu- Logic for using Beans, Peas & Lentils as MMA

## Description
Under the rule, schools may count beans, peas, and lentils offered as meats/meat alternates at lunch toward the weekly beans, peas, and lentils vegetable subgroup requirement. When offering beans, peas, and lentils as meats/meat alternates, schools may only count those items toward one meal component daily: either the meats/meats alternates component or the vegetables component. When schools offer beans, peas, and lentils as meat alternates, they are still required to offer additional vegetables to meet the daily and weekly vegetable meal component requirements.

## Acceptance Criteria
* Allow Veg- BP to be toggled “Use as MMA” for 14+ Lunch
** Use same approach used for MMA use as grains for breakfast currently implemented
* IF toggled to use as MMA, count Veg- BP component towards the daily MMA contribution on daily component analysis
* Do not count Veg-BP towards total veg component offering for menu
** User must still offer additional vegetables for the menu to pass the vegetable daily component offering
** Count Veg- BP as offering for the week to fulfill the weekly analysis (Separate story)

# NXT-65050 — INV - Distribution - Quantities Report - Site Order Details & Item Category grouping

## Description
As a distribution manager I want a Site Details version of Quantity Report so I can see the difference between certain distribution quantities for specific sites on the Distribution.

h3. 

h3. Quantities Report - Assignment

* Detail Options section for Summary or Site Order Details
* Display Options section for Item Category grouping option

## Acceptance Criteria
* Add a Report Detail Options section to the report slide-out for the Quantities Report
* Provide additional two radio options at the bottom 
** Item Summary
** Site Order Details (default to this)
* Generate the Site Details version of the report
** List each site on its own page, with order details (like Delivery Tickets)
* In the report header list the Site and Order #
* Add a Display Options section to the bottom of report slide-out for the Quantities Report
** Provide a single checkbox option to Group by Item Category
** Default the checkbox to *unselected*
* Ensure the Site Order Details version of Quantities report can be generated on Unassigned distributions
** only Ordered Quantity will be available
** This will basically be an Orders Report for all sites who ordered to the distribution

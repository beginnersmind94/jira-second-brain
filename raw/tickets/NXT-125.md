# NXT-125 — Menu Grids

## Description
*Story*

As a user I need to be able to create menu grids for use on ExpressPoint. There are two types of menu grids, static which have menu items that don’t change and dynamic which has items that change based on menu planning. This story handles Static menu grids.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=4831%3A0&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=4831%3A0&scaling=scale-down-width]

*Requirements*

* View a list of grids.
* Create a new grid
* Open a grid for viewing or editing
* Filter Headers



* Version 2 Requirements
** Meal Combo column is in version 2
** Orderable is in version 2
** Grid at the bottom is in Version 2 ?
** When Menu Grid Templates are available, The user will select the Template. For use with Dynamic Menus [https://primeroedgenext.atlassian.net/browse/NXT-5903|https://primeroedgenext.atlassian.net/browse/NXT-5903|smart-link]

## Acceptance Criteria
menu grid list loads at time of opening with All selected for all searchable fields. 
changing search criteria updates the menu grid list based on the search criteria entered 
Days allowed are searched by letter M=Monday,T=Tuesday,W=Wednesday,Th=Thursday,F=Friday,S=Saturday,Su=Sunday
When a letter is entered any menu that can be served on that day that is enabled or disabled is shown.

Headers are filterable and sortable
A new menu can be created by clicking +Add (the build menu is opened)
A menu grid can be enabled or disabled, Disabled is the default.
A disabled menu grid can be deleted. With confirmation.
There is pagination defaults to 20 pages
Add and Edit loads the Menu Grid Particulars page.
Selecting the Enable toggles the state of the menu grid and updates the menu.

Message for disabled menu ‘You have disabled this menu. It will not be available for serving on ExpressPoint’

Message for enabling a menu ‘You have enabled this menu. It will be available for serving on ExpressPoint 

If they try to create a menu with an existing name ‘The menu name is already in use. Please enter a different menu name’

When deleting a menu."You are deleting  a menu. It will be permanently removed. Are you sure?  With a Yes/No Confirmation

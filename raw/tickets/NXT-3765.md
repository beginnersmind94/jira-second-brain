# NXT-3765 — AcctMgt - Patron Details - Photo

## Description
*User Story*

As a district administrator I want to see a patron’s picture and replace it if needed.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418624%3A1&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418624%3A1&scaling=min-zoom|smart-link] 

*Requirements*

* Account photo, if available, should be visible when the account is loaded.
* User needs a way to remove the existing photo.
* User needs a way to upload a new photo.
* User should be given appropriate feedback if they select a file that’s in the wrong format / file extension.

*Requirements*

* Account photo, if available, should be visible when the account is loaded.
* User needs a way to remove the existing photo.
* User needs a way to upload a new photo.
* User should be given appropriate feedback if they select a file that’s in the wrong format / file extension.
* Edit pencil should be added to photo area and be required for a user to add, update, or delete a photo. This will tie into a role permission later like the other edit pencils on the page.

# NXT-3881 — Item Management - Recipes - Save Step Re-arrangement

## Description
*User Story*

As a user, I want the record to save when rearranging or editing the steps

*Requirements*

* When a user is adding or -editing- (Future story - [https://primeroedgenext.atlassian.net/browse/NXT-3029|https://primeroedgenext.atlassian.net/browse/NXT-3029|smart-link] ) a recipe, when the steps are moved the record should be saved to the database
* When a new step is added or removed with the check box or the X, then it is also saved to the database
* The step should also save once the user changes the fields in the step itself, and clicks the confirmation check

## Acceptance Criteria
User should be able to add a recipe and have their  steps save when rearranging the steps and when adding or removing a step

# NXT-9528 — NXT - Transfers (District) - Complete Site Transfer Request

## Description
As a district manager I want a site request completion screen so I can fulfill site transfer requests.

*Requirements*

* Upon clicking into a Transfer Request as a district level personnel, provide a Fulfill button
* When user clicks Fulfill, open a detailed screen where the current on-hand quantities of each item on the request are displayed in a sub-grid
* Allow the district user to decide from which site(s) the request will be fulfilled
* Once the user has input the fulfillment quantities from each site, provide a summary view on the Next screen
* Generate individual transfer transactions for each of these sites, so they may generate any reports and complete the transfers.

## Description
As a site administrator, I want to select my items and build a model so that I can deploy my model to the site for food detection.

*Application*: SchoolCafé

*Flow:*

* Browse to SchoolCafé > Accountability > Food Detection > Model Management > Click Create Model button > Create Model
* Complete header information
* Optionally filter items
* Select which items want in report
* Click Create button

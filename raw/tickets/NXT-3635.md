# NXT-3635 — Edit Inspections - Temperature Grid Question Type

## Description
*User Story*
As an Inspections user, I can see Temperature Grid Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Temperature Grid Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Grid with description, temperature entry, and the ability for Inspectors to add a row for each response
*** Rows can be removed if added accidentally
*** Rows can be edited
*** Temperature units match what is set in the template
** Whether question is required
** Notes Section
** Upload Media
* Entries are properly saved
** If question is required, Inspection cannot be submitted if no selections are made
* Entries can be changed if Inspection has not been submitted
* Question does not provide any score

## Acceptance Criteria
* Temperature Grid question appears in Inspections
* Each field can be selected, saved, and edited
* Rows can be properly added and removed

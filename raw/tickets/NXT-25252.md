# NXT-25252 — INV - Config - Review Order Settings - Update Items for Pack Size config

## Description
As a District Admin I want the item’s pack size units to be available to me when configuring Order Review rules so I can make sure I am setting the right constraints for the rule.

*Requirements*

* Update the Review Order Settings ‘Choose Item’ step to include item units when configuring Minimum or Maximum order quantity limits

# NXT-54825 — POS - Settings - Control via the Website

## Description
As Sally from Support it would be useful to have POS settings turned on remotely instead of having to direct the user to do so themselves. This often comes with a significant communications delay and we lose time to help.

*requirements*

* User can view POS Admin Settings via Device Management for a specific terminal.
* User can enable any POS setting via Device Management.
* User can enable POS Settings in batch via DM
** Select multiple terminals or select by site
* POS is updated to take setting updates from the website.

## Acceptance Criteria
* User can view POS Admin Settings on the website
* User can change Admin Settings via the website
* User can change multiple terminals at the same time
** Batch update by selected terminals or by site
* POS Admin Settings update as changes are made

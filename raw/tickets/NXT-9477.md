# NXT-9477 — INV - Contract Items - Ordering Conditions - Saving Item Site Detail

## Description
Continuation of [https://primeroedgenext.atlassian.net/browse/NXT-8818|https://primeroedgenext.atlassian.net/browse/NXT-8818|smart-link]

As a District Supervisor I want a detailed item ordering conditions slide-out view on the Contract Items page so I can *save* site specific ordering conditions for an item.

_Inventory > Vendors > Contracts_

*Requirements*

On the Contract Items Ordering Conditions tab ([NXT-5648|https://primeroedgenext.atlassian.net/browse/NXT-5648]) provide a button on each item row to view the Item Site Details so the user can see the specific site ordering condition breakdown for the selected item.

When the user clicks the button, a right-side slide-out will appear displaying the list of sites and which are configured to order the item.

+Requirements+

After user selections for site ordering conditions, allow user to Save new configuration

* a Save button to save the changes made to the sites.
** once the configuration is saved, the slide-out will close and page will refresh with updated configuration

(mockup difference: only one Site Type expanded at a time)

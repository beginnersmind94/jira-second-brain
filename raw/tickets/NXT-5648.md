# NXT-5648 — INV - Contract Items - Ordering Conditions tab

## Description
As a District Supervisor I want to be able to select items on the Vendor Contract Items screen so I can configure Ordering Conditions directly through my vendor items.

_Inventory > Vendors > Contracts_

By allowing users to select items from a vendor’s Contract Items list, the user can more quickly create ordering conditions by proceeding directly to the Site Selection step ([NXT-5704|https://primeroedgenext.atlassian.net/browse/NXT-5704?focusedCommentId=17810&page=com.atlassian.jira.plugin.system.issuetabpanels%3Acomment-tabpanel#comment-17810]), based on the contract item selections.

User would be able to select an item from the Contract Items and assign sites, or site types, to that item, therefore creating the ordering conditions.

*Requirements*

* Changes to existing page ([NXT-9480|https://primeroedgenext.atlassian.net/browse/NXT-9480])
** add a ‘Items’ tab which contains the current version of the page (list of contract items)
*** move the ‘Add Item’ button to the top
**** the Add button should only be visible on the Items tab

* Add a new tab to the Contract Items list called ‘Ordering Conditions’
** the Ordering Conditions tab will contain the same list of contract items with different columns
** list of columns:
*** Item Number
**** searchable
*** Item Name
**** searchable
*** Site Type 1 (ex: Elementary)
**** this will display the number of sites ordering the item from the vendor
**** ex: ‘20/24'
*** Site Type 2 (ex: Middle Schools)
*** Site Type 3 (ex: High Schools)
*** Site Type 4 (ex: Alternative)
*** Additional Site Types (using Kendo grid with selectable columns)
*** *Pagination used for the table
** A button to open the Item Detail slideout for individual items (covered in [NXT-8818|https://primeroedgenext.atlassian.net/browse/NXT-8818])
** add checkbox for each item
*** selecting multiple items would allow user to create conditions for multiple items at once
** add ‘Configure’ button ([NXT-9475|https://primeroedgenext.atlassian.net/browse/NXT-9475]) to create multiple ordering conditions at once
*** button is in the top right corner (takes the place of the Add button on the new tab)
*** the button is disabled by default
*** the button becomes enabled after user selects/checks items
*** clicking the button would open the Detail slide-out and allow user to assign site configuration for multiple items at once

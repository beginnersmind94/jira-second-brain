# NXT-28478 — INV - Receipts - Cancel Receipt - Update Available QOH for Commodity Contract Items - API Only

## Description
As a Site Manager I want the system to return available quantity when a Site cancels an order so the quantity is available for other sites to order.

*Requirements*

* Create trigger when cancelling a receipt that contains Commodity Contract items
* Upon cancellation, return previously received quantities to the Commodity Contract item quantities

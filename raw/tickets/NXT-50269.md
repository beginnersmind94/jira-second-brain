# NXT-50269 — LMA - District - History

## Description
As a SchoolCafé Admin role member, I would like to view the history of licenses we have created for any district in one page so that I can manage licenses across all our customers in one place.

*Flow:*

* LMA role user browses to System > District Licenses
* User sees a list of licenses generated for the software
* User clicks View History on a row
* History drawer opens and views sees history



*Requirements:*

* On License tab grid, do the following:
* Grid Actions column:
** History
*** On click:
**** Open the History drawer
**** Enabled for all statuses
*** Customer Support role and License Management role can view this drawer

 

*History drawer*

* If the drawer does not already exist, create it
** Title: History
** Width 1/2
** Use sliding motion

 

*Header*

* Show these fields are read only:
** District
** License Key
** License
* Show these fields as editable and required:
** Search bar
*** Allow user to enter in search words to filter the cards below it
* 
* Table of cards
** Show cards based upon audit log type (Add, Edit, Delete)
** If Add, don’t show before and after, only show the values they entered
** If Edit, show the before and after as appropriate
** Be sure to show who and when did it
*  
* Security
** CS and Tech Support and License Management role can do anything
** View permissions to the District Licenses page can click on the History button

## Acceptance Criteria
* Users can view all add, edit, delete actions for licenses, including linking a license to a site
* Security trimming is observed

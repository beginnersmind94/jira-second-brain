# NXT-9409 — INV - Transfers (Site) - Create Transfer - Confirmation screen

## Description
As a site level Inventory user I want a confirmation screen for my site-to-site transfers so I can view only the items which I am transferring and confirm the transfer.

After the user has created a transfer and clicked Next, they will be brought to the Transfer Confirmation screen.

+Header section:+

* From Site
* To Site
* Transaction Date
* Comments

+Transfer Item List:+

* Item #
* Item Description
* QOH 
* Whole Unit Transfer Qty
* Broken Unit Transfer Qty
* Comments

+Action buttons:+

* Edit Transfer
** this will take the user back to the Create Transfer screen to adjust the items & quantities
* Confirm Transfer
** logic is handled on [https://primeroedgenext.atlassian.net/browse/NXT-9568|https://primeroedgenext.atlassian.net/browse/NXT-9568|smart-link]

Mockup differences:

* Transfer Qty boxes are not editable (label only)
* No From Site column needed

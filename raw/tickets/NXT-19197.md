# NXT-19197 — IM - Recipe Material Report

## Description
As a Nutrition Department user I need a report that provides a dollar cost for the materials (items)
required to produce each recipe. All items and item quantities required to produce a single recipe should be listed.

## Acceptance Criteria
*Selection Criteria*

* User should be able to select data source to use
* Use should able to search for specific recipe (smart search base on recipe code and recipe name)

*Actions*

* Generate Report (Produces the report and displays it on the screen)
* Export Report ( Excel (CVS), word, PDF)

*Report:*

By Recipe display the following fields:

* Item / Recipe Code 
* Recipe Name
* Item Description
* Primary Measure
* Secondary Measure
* Qty Needed (default value 1)
* Line Cost (Recipe Step Level Item cost based on amount used)
* Recipe Cost  (*Recipe cost = Sum of line cost per recipe*)

Refer to Recipe Costing Report for material cost calculations. (NXT-13231  ) and  Attached Excel file 

*Header:*

* User Name
* Report Generation Date/time

*Footer:*

* Criteria used for report generation
* Page number

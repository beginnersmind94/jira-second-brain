# NXT-41624 — IM - Find & Replace - Ingredients - Replace

## Description
As a Nutritionist or Menu Planner I want to be able to replace all instances of an ingredient with another ingredient so I can quickly update recipes.

## Acceptance Criteria
* Upon clicking the Replace action, enable a selectable mode of the Recipes grid
* Adjust the header to a view-only mode where action buttons are removed
** Show both ingredients with an icon to open the ingredient item/recipe in a new tab
* Provide checkboxes for each recipe
* For each selected (checked) recipe, enable the New Ingredient Quantity field
** this should be just like the Add Ingredient selection from Recipes, including secondary measure
* Display a sticky footer 
** Cancel button to reset the entire process
** banner displaying the count of selected recipes
** Replace button to confirm selections and validate data entry
* Upon click of sticky footer Replace button, advance to the [Confirm step|https://primeroedgenext.atlassian.net/browse/NXT-41623]
** require valid measure input for each selected recipe

# NXT-43158 — IM - Items - Deactivate entire Item

## Description
As a district admin I want the ability to make an Item inactive if it is not being used anywhere in the system, so I can delete any unused or accidentally created/imported items.

!image-20231121-024407.png|width=91.66666666666667%!

!image-20231121-032531.png|width=91.66666666666667%!

## Acceptance Criteria
* Provide an ‘Entire Item’ row at the top of the Item Type status panel
* *Remove POS toggle from panel*
* Provide a prompt on click of Deactivate button
** “Do you wish to make this item and all its sub-types Inactive?” [Yes, Deactivate], [No, Cancel]
** On click of ‘Yes, Deactivate’ check for all active sub-types 
* If the sub-types can all be made inactive, make the main item inactive
* If any sub-types go into Pending Deactivation status, assign the Item with the same status
* Once all sub-types are Inactive, make the main item inactive

# NXT-58466 — Acc - Deposits - Delete Option, Update Bank Deposit Slip Number

## Description
[https://primeroedge.freshdesk.com/a/tickets/272154|https://primeroedge.freshdesk.com/a/tickets/272154]

*requirements*

* Add a Delete option inside of Deposit Slip Details
** Confirmation prompt
*** {noformat}Are you sure you want to delete this Deposit Slip?{noformat}
** Update Status to Deleted
** Track who deleted it and when
** Deleted sessions should not be included as part of reports/rollups
** Lock to Delete CRUD permission for the Deposit Slips page
*** Default: Central Office, Director, SchoolCafe Administrator
* User can update the Deposit Slip Bank Number while editing

## Acceptance Criteria
* Delete option added
** Confirmation prompt
** Locked to Delete CRUD permission
*** Rolled out to Central Office, Director, SC Admin roles
* Status available for viewing in both Grid and Details
** Can filter out deleted slips
* History updated → we can use a comment:
** Deposit Slip deleted by {{UserFirst}} + ' ' + {{UserLast}} at {{Full Timestamp}}.
* Deleted Deposit Slips are not included in rollups / reports
* Linked Sessions moved out of Deposited (move to Reconciled)
* User can still see deposit slip details of a Deleted slip
* User can update the Deposit Slip Bank Number while editing

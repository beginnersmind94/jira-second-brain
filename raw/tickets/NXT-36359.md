# NXT-36359 — IM - Recipe Rework - Steps & Ingredients - Edit/Add Step

## Description
As a Nutritionist I want the Recipes page to have a dedicated Steps & Ingredients tab so I can make all the recipe preparation changes in one place.

[Figma - Edit Individual Step|https://www.figma.com/proto/OAZ7qTRgdG5bFFGuhDmkoZ/Item-Management-6.0?page-id=3238%3A54123&node-id=3315%3A68003&viewport=411%2C-15%2C0.03&scaling=min-zoom&starting-point-node-id=3315%3A60028]

*Requirements*

* Go into ‘edit specific step’ mode when a step is being edited or a new step is being added
** Either from the edit step icon or from the Add Step button 
** This step will occupy the entire Steps card, hiding other steps while in this mode
** hide the Add Step button
* In edit mode, load the Directions as editable
** Dynamically increase the heights of the Directions field as text is entered so it’s not hardcoded to be as tall
* Load the Ingredients sub-grid below the Directions
** Provide an Add Ingredient action to enable the Ingredients card
** Change the ‘Primary Serving Measures’ column to ‘Quantity Needed’
** Remove the dedicated column for Secondary Serving Measures
*** Add an icon next to the serving size control for each added ingredient that would allow the user to add a secondary measure if needed
*** the secondary measure could be displayed underneath the primary in the same Quantity Needed column
** Provide a Delete icon
** Provide a Swap icon
*** this would switch to the Ingredient search and would replace the selected ingredient with the one added from the search card
* Add ingredients to the Step card as they are entered from Ingredients search
** Serving Sizes should be entered for every ingredient before the step can be saved
* Provide a Save Step button so the user can return to the higher level edit ([NXT-36358|https://primeroedgenext.atlassian.net/browse/NXT-36358])
** steps can be saved as Directions only

## Acceptance Criteria
* Provide Add/Edit Step functionality for Recipe Steps
* Enable Steps card and Ingredient Search card in Add/Edit mode
** Steps are only edited one at a time
* Allow users to add ingredients to a Step using the side panel ingredient search card
* Require valid serving measures for all added ingredients before Step can be saved

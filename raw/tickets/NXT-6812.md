# NXT-6812 — INV - Vendor - View Contract Card

## Description
As an Inventory user I want to be able to view my saved Contracts on the vendor screen so I can easily view and access my saved Contracts.

*Requirements*

On the Vendor page, provide a card view of saved Contract:

* Contract ID
* Status
** Highlighted in Green for Active
** Highlighted in Red for Inactive, Suspended, Mark for Deletion
* Contract Category
* Date Range
* edit icon next to each Contract to edit contract information
* Add Contract button at the bottom to add a new contract
* link on Contract ID to open contract items (will be added in [Story 6829|https://primeroedgenext.atlassian.net/browse/NXT-6829])

Add a ‘Contracts’ label to the navigation card

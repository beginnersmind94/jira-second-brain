# NXT-7526 — My Profile - Move Security Card to separate tab

## Description
To avoid issues stemming from KeyCloak (see Mahathi/Amar), in order to allow a user to change their password from their profile, let’s do the following: 

* When the user navigates to My Profile and selects the Edit button, do the following:

# Split the Profile into 2 tabs: 
## Basic Info - Should only contain the *Contact Info*, *Roles & Permissions*, and *Accessibility* cards/sections, as well as Save and Cancel
## Change Password - Should contain the *Security* card, plus a separate Save and Cancel
# When the user updates information on the Basic Info tab and selects the Save button, save the data from those 3 sections only
# When the user updates their password on the Change Password tab and selects the Save button, save the new password only

* The summary view of My Profile should remain the same (i.e. Security card showing how many days until the user’s password expires should still be visible).

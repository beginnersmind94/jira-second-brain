# NXT-34754 — User Page and Profile - Password information is based on District selection and not home district

## Description
On the Users page and Profile page, the user’s password information is based on what the currently selected district is and not the user’s “home” district.  The user’s password information should be based on the user’s “home” district.  I also found that if I visited the security page, then returned to the USer’s page, the value of the password expiration would change.

## Acceptance Criteria

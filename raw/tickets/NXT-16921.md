# NXT-16921 — Enhancements in Eligibility Api to associate notification rules with modules

## Description
As a notification scheduler, I need an api that can be called when a particular set or rules met with their triggering conditions. This API end point in eligibility would take a collection of rules and then find their respective module based associations from eligibility. Depending upon what that rule means for a particular module, the actual module based business criteria is checked. Once the check succeeds the notification is sent out.

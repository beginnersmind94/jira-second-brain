# NXT-27926 — Platform - System - New District Setup - Automation - API

## Description
As a Cybersoft support team member, I would like an automated way of setting up new districts so that setup time is reduced and that I eliminate human errors as a result of manual setup & faster deployment (from 15 mins to 15 secs)

+*Requirements:*+

* Enable District creation & districts table
* Map specific role to the new district - SchoolCafe Admin role

## Acceptance Criteria
* All required assets and “base data shall be deployed including, but not limited to:
** Configuration data
** Default Custom Views
** Configuration settings
** Azure-related resources in place
** MicroDBs fully replicated
* All installation/deployment scripts shall be run automatically
* Feedback on the success/failure of the setup shall be provided
* Ideally, this will be triggered and managed via a UI

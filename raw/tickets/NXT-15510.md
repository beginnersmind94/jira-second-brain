# NXT-15510 — UI Control for managing tags

## Description
As a module developer, I would like to have a UI control that I can use to allow my end-users to easily create, assign, and unassign tags to “taggable” entities.

## Acceptance Criteria
* A UI control shall be available that will be populated with tags that have been assigned to the entity.
* The UI control shall allow an end-user to create new tags and assign the new tag to the entity
* The end-user shall have the ability to remove the tags that are displayed within the field
* The UI control will operate as a field when initially rendered and as a field with a dropdown beneath it when the field is clicked on, and shall allow the end-user to select an existing tag to assign to the entity which has context.
* The control shall allow the user to select 1 tag at a time.
* Each tag that is selected shall be assigned to the entity that has the context
* The UI control will behave like the label field within Jira.

# NXT-47709 — PR- Batch processing to update analysis based on find and replace (Plan/Record)

## Description
We need a batch process that will allow analysis to be updated on menu and menu cycle based on changes made on find and replace ingredients in recipes or sub recipes in recipes.

## Acceptance Criteria
Apply a batch process to allow analysis to be saved in batches and not all at once due to the high volume of changes that may occur.



Once any calendar needs an analysis refresh, we need to check the Pre- and post-analysis refresh should be done. Once both are done, we must update that calendar with an Analysis flag with 0 in the DB.

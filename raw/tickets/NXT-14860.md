# NXT-14860 — Onboarding - Integrate SIS Auto Sync (Phase 1)

## Description
As a Cybersoft or District Administrator, I would like for onboarding users to have the ability to immediately begin syncing data from the district’s SIS, so that they don’t have to manually add data through each step of the onboarding process and get value out of SchoolCafé more quickly.

*Summary*

When a user selects their district on the initial onboarding wizard screen and continues, prompt them to select their SIS first, and data they would like to begin syncing second. Run the syncs in the background so that the data is in place by the time they get to the last step of the onboarding (setting up imports and exports).

{color:#403294}*Will likely need to be split in various stories (UI for initial selections, connections to SIS auto-sync, running in background)*{color}

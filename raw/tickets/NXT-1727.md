# NXT-1727 — INV - Orders - Orders List - View Receipts - Header 

## Description
As an Inventory manager I want a Receipts tab on the Orders List so I can review Receipt and Invoice details.

On the Orders page, provide an additional tab for viewing only orders which have at least one Receipt. 
This tab would also be used to reviewing Invoice information (classic page: Invoice Reconciliation)

*Requirements*

* Add a Receipts tab to the Manage Orders list page
* +In the header section, provide the following filter/search selections:+
* Site selection (user control)
** Area, Site dropdowns
** Default to ‘All’ option
* Vendor dropdown
** Default to ‘All’ option
* Date Range search 
** From (date picker)
** To (date picker)
** Radio button to toggle between: (example below)
*** Receipt Date, OR
*** Delivery Date 
** default is past week to one week ahead
* Invoice Discrepancy 
** a toggle for viewing all Receipts or only Receipts with Invoice discrepancies
** ‘Show All Receipts’ or ‘Show Only Receipts with Invoice Descrepancies’
* Partial or Completely Receipted
** provide some option for the user to toggle whether to view orders that have been completely received or orders with partial receipts
** ‘Show Partial Receipts’ checkbox
*** default to checked
* Reset button - to reset the filter selections
* Apply button - to apply the selections to the receipts list below

# NXT-13449 — Acct Mgmt - Account Details - History - Track Adult/Allergen changes made through SIS - Testing Only

## Description
After giving them a new API in [https://primeroedgenext.atlassian.net/browse/NXT-26862|https://primeroedgenext.atlassian.net/browse/NXT-26862|smart-link], the Data Exchange team should be making 2 significant additions that will be impacting Account Management. Let’s test them and confirm that everything is working:

# Adult Import
## Just as with students, we should be able to confirm that the monitor captures +*all*+ changes made to the adult accounts who are updated through this SIS import, such as:
### Enrolled
### Withdrawn
### Reactivated 
### Transferred
# Allergen Import
## New accounts are imported with their allergens added (if possible. May be existing accounts only, check with Platform team)
## Existing accounts are imported with their allergens updated
## History data for accounts is updated with appropriate messaging
## Any remaining cases to consider involving account allergens are considered (Accountability team will want to create a separate story to verify at the POS, but we can test that as well.
## We should be tracking any Allergens added (not removed) to an account on the file 

These should display in both the Import Monitor and the account’s History (the messaging can be the same or look slightly different there).

## Acceptance Criteria
Verify that whenever an adult account is on a file that is imported, their account history page shows a record with any changes made to them based on that file.

Verify that account allergens are added when an allergen import is complete, and that the account history shows those additions as well.

# NXT-55495 — POS - Overlays - Overlay sizes for larger, smaller screens (FD: 261446)

## Description
As Dana the Director we got bigger screens for our staff, but the overlay windows in POS are still scaled pretty small. It would help operations to have bigger windows. Conversely, we still have smaller screens for some schools, and the overlays are barely visible due to scrolling issues.

*requirements*

* Make the POS overlays larger for larger screen sizes.
* Make the POS overlays fit better on smaller screens.

## Acceptance Criteria
* POS Overlays are larger for bigger screen sizes.
* POS Overlays fit better on smaller screens.
** Less scroll. We really need an adaptive window here, may need to check the anchor points as well given the user’s example.
* Scroll bar added to side-nav.
* Responsive text for POS buttons
** May need to see how this looks for long button descriptions first and then see if we need to truncate text.

# NXT-53415 — DMF - Cancel migration

## Description
The user should be able to cancel the migration if needed.
Cancel button/icon should only be available (visible) when the migration status is 'In-Progress'.
It should be invisible when the migration is in Error, Canceled, Queued, and Done status.

Add new property {{ImportKeyCancelADFPipeline }} to the {{DistrictMigrationResponse}} as shown above.
In the /api/DistrictMigration/DistrictMigration code, we select all the ImportKeyId and get the StatusId of it. In the same SQL, get the ImportKey which is the RunId of the ADF pipeline.
Out of the collection find the ImportKey(RunId) that is in-progress.
There will only be one ADF running at a given point in time.
UI must pass the {{ImportKeyCancelADFPipeline}} available in the DistrictMigration payload to the new Cancel endpoint.

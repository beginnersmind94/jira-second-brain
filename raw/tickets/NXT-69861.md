# NXT-69861 — Menu Analysis - Food Waste % calculation Revision (API)

## Description
As a production manager, I want to make sure when I am looking at the KPIs that the appropriate counts are taken into consideration when showing waste percentages.

## Acceptance Criteria

!image-20260202-214041.png|width=647,alt="image-20260202-214041.png"!

* Carried over servings to next meal service + RTS+ Waste +Served = Produced. 
* For prepsite items , make sure Received count of a menu item is taken as produced count menu item analysis page waste % columns
* This calculation should be updated for both Menu item calculations, Site Breakdown, and Insights Waste Card

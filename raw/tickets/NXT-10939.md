## Acceptance Criteria
User can select to see the Meal Counts by Day/Month/Date Range
Summary review only
Options to organize data by Eligibility are are selectable
There are pages per site
A count of the of meals sold by the eligibility type for reimbursable meals is shown for breakfast, lunch, and dinner.
The percentage participation is calculated for each eligibility type. (%Participation = Number of Meals/Total Approved)
The meals counts for Program Adults, Staff, Visitors, and second meals is also displayed. A percentage participation cannot be calculated because there in not an approved count for those meals

# NXT-59550 — IM- Item Onboarding- Keyboard Navigation for all sections

## Description
As a Menu Planner/Nutritionist entering data into the system, I would like to be able to navigate through the bulk onboarding/update tool with the keyboard. This is especially valuable when first entering my item descriptions and answering the initial attributes questions. 

Currently somebody keyboard functionality is available but it is not very intuitive.  Every time a selection is made with the keyboard, user must start tabbing again from the beginning on the row.

!image-20250225-205710.png|width=2133,height=1021,alt="image-20250225-205710.png"!

## Acceptance Criteria
* All pages must be keyboard friendly when moving from column to column in a section
** Allow tab from column to column and keyboard arrows within a particular column along with “enter” to confirm a selection
* Landing page must be keyboard friendly when moving from section in a section or returning back to previous pages

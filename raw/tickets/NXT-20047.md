# NXT-20047 — Share a report - Data Query Framework

## Description
As a user I need a way to share the report I've created with users.

## Acceptance Criteria
Acceptance criteria will be met when:
1. When I click on share button of a particular, I should see a popup containing a search bar where I can search for users that I would like to share my report with.
2. Once I share  a report with a user, I should see an entry for the table that's displaying the users with whom this report has been shared with.

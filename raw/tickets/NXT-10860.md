# NXT-10860 — Data Exchange - File Import Dashboard View Details

## Description
As a PrimeroEdge user, I would like to have an intuitive screen that allows me to view all file import setup configurations that are available for me with an option to create NEW FILE IMPORT SETUP so that I can view the details of the import progress and updates made from a specific import job.

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A238848&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A238848&scaling=scale-down-width|smart-link] 

*Requirements:*

* When the user has clicks on “*File Import Dashboard*” under configuration, file import dashboard is loaded to view, create and import configurations.
* Header is displayed for the user to clearly to highlight the *VIEW DETAILS*.
* {color:#00b8d9}*when user selects the filters from drawer, it should be displayed for user with X marks to remove any time which refreshes the results in the table.*{color}
* View Details table should list below columns:
** Activity
*** New student record added, removed, updated
** Status
** Start Time
** End Time
** Duration
* Pagination is enabled with only 10 rows displayed by default.

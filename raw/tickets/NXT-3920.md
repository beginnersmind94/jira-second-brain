# NXT-3920 — [AcctMgt] - Preview - Basic Details+UI

## Description
*User Story*

As a district administrator I would like an easy way to perform repeated interactions with an account, such as payments and adjustments. 

*Mockup*

[https://www.figma.com/proto/yCRz6zcMFIjX3cY1BxgQI7/Accounts?node-id=481%3A3567&scaling=scale-down-width&page-id=481%3A883&starting-point-node-id=481%3A3567&hide-ui=1|https://www.figma.com/proto/yCRz6zcMFIjX3cY1BxgQI7/Accounts?node-id=481%3A3567&scaling=scale-down-width&page-id=481%3A883&starting-point-node-id=481%3A3567&hide-ui=1|smart-link]

*Requirements*

* Preview (Quick Glance) panel can appear when 
** Include a preview icon that launches the preview slide out.
** This icon should display *in addition to* the existing document icon which already opens the Account Info page.
* Display basic information regarding the patron: 
** Name
*** First Name + Last Name
*** IF there is a PreferredName, display it in the middle (in parentheses)
** Student Photo
** ID 
** Site, Grade 
** Status, Eligibility + Reason
*** Display these in UI as chips
** Account balance.
*** Display an icon next to the balance
*** Display the balance using the same color format as is used in the grid:
**** {color:#36b37e}Positive Balance{color}
**** $0.00 Balance
**** {color:#ff5630}Negative Balance{color}
* Add CTAs (**+*UI only***+) - A user can use these to perform actions from the preview window without navigating elsewhere - these are covered in individual stories:
** Add Payment
** Send Balance Reminder

## Acceptance Criteria
Preview Action available on the accounts grid.

User can see basic information as outlined in the requirements.

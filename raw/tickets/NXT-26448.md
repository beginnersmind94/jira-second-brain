# NXT-26448 — Acc - End of Day - Sessions Task

## Description
As Agatha the Site Manager I want to see information about daily sessions and be made aware of any issues that require attention before I close out for the day.

*Mock-up:*

[https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|smart-link] 

*requirements*

* Add the results grid showing all active tasks.
* Populate the Sessions task.



*Sessions Task*

* Task Name: Sessions
* Summary:
** Show the number of sessions for the site and their current statuses.
** Examples:
*** Sessions Opened: X
*** Sessions Closed: X
*** Sessions Reconciled: X
*** Sessions Deposited: X
* Details:
** Show the total over/under for sessions.
** Example:
*** Total Over/Under: $00.00 / ($00.00)
* Status
** Status is determined by the state the sessions are in.
** Not Started = no sessions created
** Not Done = sessions created, but not all sessions are deposited.
** Complete = sessions created, and all sessions are deposited
* Click-thru
** User can click through the task and be taken to the Sessions page.
** User can open this in a new tab. (This may not have the site + date filtered)
** Sessions page should load with the following already set, and the search underway:
*** Site
*** Date
* Actions
** Done / Not Done
** History

## Acceptance Criteria
Results grid added.

* Complete = sessions created, and all sessions are reconciled/deposited

## Acceptance Criteria
* Provide a review step on the Bulk Item tool for the Allergens 
* Display a column for each allergen form (Contains, etc.)
** make most space for the Item Description and for Contains
* Do not display a column for ‘None Indicated’ in view mode
** display items with ‘None Indicated’ in the Contains column
* Display the allergen forms with corresponding icon 
* Provide an Update Allergens button for the edit mode 
* List allergens in each cell for the corresponding allergen form
** display all standard allergens first, then custom
* Provide the Show/Hide Filters button to search allergens 
* Show the Allergen disclaimer button in the top right of the header card
** this opens the existing Allergen Disclaimer Slideout

# NXT-42141 — IM - Config - Menu Item Categories - Change colors for default categories

## Description
As a Menu Planner I want the five default Menu Item categories to have the same color as their corresponding Menu Planning food component, so I can more easily associate menu items with their intended component.

!image-20230808-225350.png|width=1727,height=809!

## Acceptance Criteria
* The five default *MI&POS* categories should have the same initial color as the matching MP component
** Fruit - A4E9FF 
** Entree - DE6FA1
** Grain - F08B46
** Milk - 459BEA
** Veg - 3AC191
* The default (local) *POS Only* categories should have the following default colors:
** A La Carte - 6F1031
** Drinks - CC0000
** Generic Meals - 8B40D7
* The default (local) *Menu Only* category should be as follows:
** Condiments - 666666
* Users should still be able to change the colors

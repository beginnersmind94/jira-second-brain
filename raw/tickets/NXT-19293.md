# NXT-19293 — Production- Reports- Technicians Worksheet (Production Plan Link)

## Description
This report is a way for users to print all of their items and enter the numbers with pen and paper to use to enter production plans This report also will be able to be generated from the Production Record screen as well. 

This is giving the user an option to view technicians worksheet for the day from the production plan screen. If production plan is completed, the planned numbers should appear under the planned column along with any food required. 

 

*Report must contain the following information*:
Serving groups with their Planned Counts and a space for them to write actual counts
A total row for planned counts and actual counts
Plan comments box
Record Comments Box

*Columns must include*
Serving size and if they have a portioning tool display what it is 
Planned
Produced
Carryovers
Return to Stock
Waste
Adult/A La Carte (Show blank for now)

Items with Recipe code display in Category order within the category and what their Choose is and nutrition contribution
Example:
Category: Entrees: May choose 1
Cereal Cinnamon Toast Crunch (SY21-22) (LR 1031) 2 WGR

This item also needs to have food required associated with it which should pull based on Planned Counts.

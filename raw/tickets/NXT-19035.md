# NXT-19035 — IM- Adding or Subtracting Recipe Weight based on yield factor -Selecting Type of Fat

## Description
This story is complement and complete story NXT- 18701.

When user enter a value for FAT yield factor the the user will be required to enter the type of fat. This will be done by prompting the user with smart search field (alphanumeric) that will allow the user to search for items. The items will be search from item management active items.

The length of the field should be 50 chars.  if name is too long display a help tip that when hoover will display full item number and full name of the item.

## Acceptance Criteria
Fat yield factor should be associated with and display item source of fat.

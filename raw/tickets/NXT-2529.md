# NXT-2529 — Inspections Reports - Pending Approvals

## Description
*User Story:*
As an Inspections admin, I have a report that shows me a list of pending approvals by sites or templates.

*Current Version:*

[https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports|https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports]

*Requirements:*

* Pending Approvals Report can be run using existing Inspections and available parameters
** User must have Reports role permission to run report
** Report can be created in PDF, Excel, or Emailed
* The following parameters are used (all are required):
** Site - Can select 1 or many
** Template - Can select 1 or many
* All Inspections that are in a Submitted status that fit within the chosen parameters appear in the report
* The following data is displayed on the report:
** Header with Report Name and District
** Inspection Name
** Template
** Date
** Inspector
** Approvers
* Inspections are grouped by site

## Acceptance Criteria
* Pending Approvals Report can be created with available data across all parameters in all formats

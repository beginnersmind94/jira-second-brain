# NXT-14757 — Acct Mgmt - Reports - No Charge List

## Description
As a Manager, I would like to generate a list of accounts who are not allowed to charge to their accounts, so that I can provide my cashiers with that information in case one of those patrons come through the cafeteria line. 

*Summary*:

Add a No Charge List report that displays a list of accounts and their basic details, based on some criteria. Basic functionality can mimic classic.

*Details*:

* Allow a user to filter by Area, Site, and Date Range
** Use preset date ranges of Today, Last 7 Days, Last 30 Days, Year to Date, and Custom Date Range
*** Last X Days are calendar days, not business days
*** Year to Date should begin with the first day of the current Academic Year
*** Custom Date Range should display a tooltip: “Please note that custom date ranges cannot exceed one year.”
**** It should only allow the user to enter a date range spanning 1 year.
* Report Criteria should: 
** allow user to enter an outstanding amount with an amount >= 0.
** allow them to filter by any individual No Charge reason, displaying any patrons that have that reason on their account
** allow them to only show active accounts if needed
** allow them to only show accounts with a No Charge reason
* Report Options should allow the user to display different columns as well as group the records as needed (see examples below).

# NXT-11216 — [AcctMgt] - Preview - Reminders

## Description
*User Story*

As a district administrator I want a way to quickly view or send reminders for a patron with a low account balance.

*Mockup*

[https://www.figma.com/proto/yCRz6zcMFIjX3cY1BxgQI7/Accounts?node-id=481%3A3567&scaling=scale-down-width&page-id=481%3A883&starting-point-node-id=481%3A3567&hide-ui=1|https://www.figma.com/proto/yCRz6zcMFIjX3cY1BxgQI7/Accounts?node-id=481%3A3567&scaling=scale-down-width&page-id=481%3A883&starting-point-node-id=481%3A3567&hide-ui=1|smart-link]



*Requirements*

* Add a ‘Balance Reminder’ action to the quick preview.
* This should slide the user to select the options from the Notify drawer.
** We should simplify the approach here.
** User can select:
*** Format: Individual or Household
*** Email or Print or Both
* No sorting options are needed.
* All functionality/test cases should match the existing function found in the Notifications page (based on the limited requirements shown above).

## Acceptance Criteria
Balance Reminder button added to the quick preview.

Parred down Notify options - 

* Format: Individual or Household
* Email / Print / Both

All functionality/test cases should match the existing function found in the Notifications page.

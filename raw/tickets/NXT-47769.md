# NXT-47769 — GDSNConnect - Additional Exception Handing (Non Standard Serving Measures)

## Description
Currently, The serving size entries for created based on the weight given at source

The change is required for standard measures such as Ounce, Pound, Gram

## Acceptance Criteria
On the parsed serving size data, the following changes are required

Scope: 

# Items with Measure description Not in (Ounce or Oz, Pound or Gram) 
# Where Serving Size Unit Weight is Oz or Pound (Gram is represented as Gram anyway)

Reference: Measure Mapping Table

Change, Unit Conversation Factor should be included as additional column and the following should be maintained:

1 Ounce = 28.35 Gram

1 Pound = 453.59237 Gram

1 Gram = 1 Gram



*Where should this be applied?* Upon writing serving size unit weight and unit weight measure in Serving Size table, the quantity should be multiplied by the conversion factor and stored as Unit weight. Unit weight UOM should be written as Grams always  


+Reference: Item Measure Mapping with Conversion Factor defined+

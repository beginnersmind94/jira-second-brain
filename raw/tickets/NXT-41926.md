# NXT-41926 — eOrders Configuration – Status

## Description
As a SchoolCafé user, I would like to be able to update Status one of the eOrder Configurations.

*Flow:*

* Open SchoolCafé and browse to Settings > Configuration > eOrders
* View list of eOrders
* Click Enabled/Disabled on a row
* User confirms their intentions, and the system Update status and records in History table

*Requirements*:

* eOrders Dashboard page updates:
* Add a button to the rows in the action column if it is not there already
** Title: Enabled/Disabled
*  
* On click, do the following:
* Ask if they are sure want to Enabled or Disabled
* If confirmed, they want to Enabled or Disabled the configuration from the system
* Log the Enabled or Disabled in the history log
*  
* Show a success confirmation
* Refresh the table showing that the eOrder is Enabled or Disabled
*  
* -Notifications:-
** -Send a message to the inbox of all users that have Edit rights in Roles & Permissions-
** -Text: [logged in username] has updated an eOrder configuration for [Vendor Name] To view, click [link to the eOrder Configuration]-

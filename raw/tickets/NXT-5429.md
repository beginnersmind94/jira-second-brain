# NXT-5429 — Data Exchange - File Upload Shared UI Component

## Description
As discussed in Backlog Grooming,

This should support not just image but any documents. Component should have the configuration through properties to accept based on type of document ( images, PDF, excel etc).

*Additional Notes After Demo:*

* Remove logging that’s appearing - Customers do not need to see that
* For non-image files, we do not need to preview
* Messaging for drag and drop is fine to be replaced with “Done” after initial file upload

## Acceptance Criteria
* Component available for use across the product for file uploads
* Allowable file extensions: .txt,.csv,.doc,.docx,.pdf,.jpg,.png,.bmp,.msg,.xls,.xlsx
* Max size of each file type is configurable - Set to 5MB for images, 250MB for documents

# NXT-42804 — API – Custom – Add New - Pre-Upload Tab

## Description
As a SchoolCafé admin, I would like to treat the SIS Auto Sync the same way as manual imports by adding the field mapping tab along with all its functionality so that I can have more control over the import.

*Notes:*

* This story assumes you will copy code from the existing SIS add new drawer
* This is part one of three stories for the Add New
* The other tabs will be covered in separate stories

*Requirements*:

* On the System > Imports & Exports > SIS Auto Sync tab > Add New and Edit, do the following:
* 
* Change the drawer to an Overlay
* Add the following tabs per the Figma
** Pre-Upload
** APIs
** Field Mapping
** Schedule
** Review
* 
* Do the following on the Pre-Upload tab
** Put the Provider and Import Type fields on this page per the API
** Include all their existing functionality 
** Provider is read-only like in the drawer

## Acceptance Criteria
* converted to a drawer
* Provider and Import Type fields are on the screen
* these fields work the same as in the current design

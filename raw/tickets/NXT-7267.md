# NXT-7267 — INV - Transfers (Site) - Create Transfer (Site-to-Site) - Transaction

## Description
As a site level Inventory user I want a Create Transfer functionality so I can transfer product from my site to another site when I know what the other site needs.

This is accessed by the user clicking the ‘Create’ button on the Transfer List page.

Page title: Transfer Items

+Site Selection+

* From Site
** only allow specific Site selection
** Site level users will only see their own site
** default is ‘Select’
* To Site
** only allow specific Site selection
** default is 'Select'
* From and To Site cannot be the same
** message: “Sending site cannot be the same as the destination site.”

+Transaction Details+

* Date
** if user is back-dating the transfer, the date cannot precede the most recent reconciled date
** default is current date
* Time
** default is current time
* Comments
**  optional text field to type comments
* *Apply* button
** trigger validations on click
** disable the header section and load the Item List
** Validations:
*** “Please select From Site and To Site”

+Item Selection+

Once the Apply button is clicked after Site selections are made, load a list of items which have some QOH at the From Site

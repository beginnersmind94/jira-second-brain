# NXT-35337 — Acc. - Menu Items - Reimbursable Flag vs Meal Flag

## Description
As Dora the Director I want to differentiate between a Reimbursable Meal and a standard Meal so that adults can eat off their children’s accounts without the child incurring a Second Meal charge.



Short:

* A student can only have 1 reimbursable meal per meal type per day.
* Any subsequent meals should use second meal pricing.
* Any non-reimbursable meal should be allowed, and make use of regular meal pricing.
** How do we solve for the student getting a reim. meal, and then an adult eating off the account? Pricing will be wrong for 2nd meal.



* Introduce a Reimbursable flag for menu items.
* Reimbursable flag needs to be used for:
** Claims
*** When generating

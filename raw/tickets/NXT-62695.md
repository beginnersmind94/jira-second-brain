# NXT-62695 — POS - API Timing Updates

## Description
As Dana the Director I want to see POS line speed given a performance boost. Adjusting some of the server communication timings should help with this.

*requirements*

!image-20250527-183732.png|width=1210,height=547,alt="image-20250527-183732.png"!

* Add Internal settings to control API timings. Names/descriptions below.
* Update POS to use the timings.
* Add visibility to the cashier so they can see the next transaction sync timing in the Transactions window.
** Text: Note: Transactions set to update every [INTERVAL] - Last Updated: 00:00:00 AM/PM | Next Update: 00:00:00 AM/PM

!image-20250527-183922.png|width=1313,height=878,alt="image-20250527-183922.png"!



----

Settings

* Name: Transaction Sync Interval
* Options: 30 secs, 1 min (default), 2 min, 5 min, 10 min, 15 min, 30 min, 60 min
* Code: *POSPSSYNTM*
* Internal
* Category: POS
* Description: Controls the frequency that Transactions, Previous Sales, and Keylogging are sync’d between the terminal and the servers.  

----

* Name: Patron Account Info Sync Interval
* Options: 2 min (default), 5 min, 10 min, 15 min, 30 min, 60 min
* Code: PATSYNINT
* Internal
* Category: POS
* Description: Controls the frequency that patron account information is sync’d between the terminal and the servers.

----

* Name: Terminal Heartbeat Interval
* Options: 1 min, 2 min, 5 min, 10 min, 15 min, 30 min (default), 60 min
* Code: TERHBTINT
* Internal
* Category: POS
* Description: Controls the frequency that the terminal registers itself as ‘alive’ to the server via the background heartbeat API.

----

* Name: Error Logging Sync Interval
* Options: 1 min, 2 min, 5 min, 10 min, 15 min, 30 min, 60 min (default)
* Code: *POSAUDINT*
* Internal
* Category: POS
* Description: Controls the frequency that POS machines send up any errors for later analysis.

## Acceptance Criteria
* Internal system settings added
* POS updates to download + use timings
* Transactions window at POS tells the cashier about transaction status updates

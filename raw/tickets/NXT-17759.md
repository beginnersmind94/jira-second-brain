# NXT-17759 — IM - Catalog Integration -Item Catalog Landing Page -

## Description
This story describes the looks and functionality of Item Catalog landing page

As an authorized user I want a page where I can Search for items in Global Catalog so that I can see, based on a search criteria, a list of available items from manufactures 

The search results area should not display any information until the SEARCH button is pressed/clicked.

The search criteria should default to all options,  but a search on Global Catalog should NOT BE performed until the SEARCH button or Enter is pressed/clicked.

The Search Criteria fields are:

* Manufacturer
* GTIN, Product Code
* Product/Item Name 

This should a smart search.

The filter Criteria fields are:

* Manufacturer
* Brand Name
* Item Name (Description)
* GTIN (Global Trade Item Number)
* Item Category
* Product Code
* Vendor Name (to be use in near future
* I want an option to clear all search criteria / filters

The Search Results  should display the following information: 

* Manufacturer
* Product Code
* Product Type
* Item Name
* Brand Name
* Item Category
* GTIN
* Item #
* Ingredient Code
* Vendor Name (to be use in near future )

On the search results I want the option to also see     the items, that according to a selection criteria, have already been imported to IM. 

I want an option that allows me to review selected item information.

I want to see a count of items selected for review out a total items available to import.

I also want an option to allow me to see stock item information ar ingredient information when I click on and item name (located on the results grid). If item have not been previously imported or the item is of product Case  ingredient code will not be displayed.

If the results list is large I want the option to select the number of items to be displayed in the screen and also be able to navigate forward and backward from page o page.

I also want an option that allows me to choose if i want perform the search on the list of preferred manufacturer or all manufactures

I need a function that allow me to review, prior to importing, the selected items from the results grid

## Acceptance Criteria
Items and its information are displayed according to selection / filter criteria.

# NXT-18575 — Production- Reports- Meal Count Review (UI)

## Description
This page and report will give the user a way to cross check their meal counts based on production meal counts, and their Terminal meal counts. If used correctly, this will be a great tool to allow for users to find discrepancies between the two processes. 

Search Criteria for report:
Site Type
Site Code
Site
Date Selections (Daily, Weekly, or Date Range)
Option for information to display only discrepancies (only show meal types that are reporting a difference between production meal count and terminal meal count)

Report Criteria:
Site code- Site Name
Serving Date
Max Production Status (PI= Plan Incomplete, PC= Plan Complete RC=Record Complete or WC= Withdrawal Complete
Production Meal Count
Terminal Meal Count
All this criteria should display individually per meal Type (Breakfast, Lunch, Snack, Supper, etc)

# NXT-47221 — PR- Reports- Print Production Recipe- UI enhancements

## Description
As a Production manager, I would like some more information about my planned recipes along with a way to quickly go to the recipes in IM.

## Acceptance Criteria
* Display Menu Item name along with Code 
** Column Name: Menu Item Name (MI#)
** Allow user to search by either Name or Code (Similar to add menu item function)
* Display Recipe Name along with Code
** Column Name: Recipe Name (MI#)
** Allow user to search by either Name or Code (Similar to add menu item function)
* Provide an Action Column with the ability to navigate to the Recipe in IM (Open in new Tab)
** Provide Tool tip (Open Recipe) on icon
* Apply on both Recipe Report and Production Recipe Report versions
** On production Recipe Report version, always display serving dates in oldest to newest order
*** IF USER picks ALL for Meal Type: Display all Meal Type and Menu line for a serving date in order before moving to next Meal Type and menu lines (Alpha). Once all meal types are displayed for the serving day, move on to the next serving date and repeat process.
EX: 

Breakfast- Apple Line 3/1/2022

Breakfast- Menu Line 3/1/2022

Breakfast- Taco Line 3 3/1/2022

Lunch- Menu line 1 3/1/2022

Lunch- Menu Line 2 3/1/2022

Lunch- Menu Line 3 3/1/2022

Supper- Menu line 1 3/1/2022

Supper- Menu Line 2 3/1/2022

Supper- Menu Line 3 3/1/2022

* IF there happens to be multiple serving sizes of the same recipe , when user unchecks or checks one of the recipes, treat them as independent and not as the same recipe

!image-20240208-193045.png|width=1289,height=675!

* Display a warning sign on pending deactivation menu items or recipes
** Provide a tool tip on the same warning, “Pending Deactivation”
* Display a warning sign on inactive menu items or recipe
** Provide a tool tip on the same warning, “Menu Item is Inactive”

# NXT-62668 — Planner AI - Site/School Performance Score (District View Only)

## Description
As a district admin, I would like to quickly be able to see a breakdown of all my sites performances when it comes to a combination of their planned vs produced, planned vs served, Enrollment vs served and Menu item waste.

## Acceptance Criteria
* Calculate Site Performance by using multiple metrics in order to rank Sites
** This only appears if user is looking at the data in the District/ All view
*** Hide if looking at a particular site
* Provide a top 5 and bottom 5 rank based on with bar chart
** Overall Performance
** Production Accuracy
** Food Waste %
** Food Waste $
* Provide a list of the remaining sites (Same as Item Performance Page)
** Provide Site Breakdown based on the same information provided above
* Allow user to select performance ranking to see on the list page and list from best to worst or vice versa in a list view (Not just top 5)
* Overall Performance Calculations
** See Attached Files for calculations[^siteperformance_sampledata.xlsx] [^SitePerformacedata.docx] [^siteperformance_sodexodata.xlsx] 
** Use highest Overall Performance score as baseline for scoring out of 100
* Production Accuracy Calculations - (All Menu item *Served Servings* Per Category/Choose Count)/(All Menu item *Planned Servings* Per Category/Choose Count)= Total Count Differential*100
* Food Waste % Calculations - All Menu Item Servings Waste Count/ Produced count *100 = Percentage of Waste (Give an aggregated total of all waste percentages)
* Food Waste $ Calculations - All Menu Item Servings  Waste Count * Cost per Serving = Total Waste Cost (Give an aggregated total of all waste cost)

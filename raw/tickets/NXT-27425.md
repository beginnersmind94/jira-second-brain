# NXT-27425 — System - Periods - Manage Periods Access

## Description
As a district admin, I should be able to access both generate and manage periods tabs under “Periods” page but any other role shall not be able to access “Generate Periods” tab.

*Requirements*

* Generate Periods and Manage Periods tabs are available under “Periods” page for admin roles only
* -Generate and Manage periods are to be added as special permission that can be managed under roles & permissions page-
* Filter the site and period type based on the navigation of the user
** If user has access to only one site, site drop down should default and disabled
If user is navigating the periods shortcut from Inventory module, period type shall be defaulted to “INV”
* -Site level users can only see Manage Periods tab-
* District level user can see both generate and manage periods tabs
* -IF the user has no access to the Generate Periods tab, Manage periods tab should be loaded by default and Generate Periods tab must be disabled.-

## Acceptance Criteria
* Users shall view the “Periods” page under management category with Generate and Manage periods tabs
* Site user that has access to only one site, site drop down should default and disabled
* User navigating the periods shortcut from Inventory module, period type shall be defaulted to “INV”

* -Site level users can only see Manage Periods tab, Generate periods tabs shall be greyed out/disabled-
* District level user can see both generate and manage periods tabs
* For site level users, “Manage Periods” tab to be loaded as default upon clicking on Periods page
* -Special Permissions to be added for both Generate and Manage periods-

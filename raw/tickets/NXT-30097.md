# NXT-30097 — IM - Items - View Item - Global Info

## Description
As a District Admin I want a header section so I can view the main/global info for an item.

*Requirements*

* Show the view-only state of the first item header section
* Display the icon for Active/Inactive somewhere near the Item # 
* Display the identifying information fields
** Item #
** Item Description (Name)
*** This needs to support up to 250 characters, per USDA requirements. This is very rare, however, and most descriptions will be fewer than 100-150 characters.
* Display global attributes if populated
** Brand Name
** Product Code
** Manufacturer
* Show Data Source 
** we may choose to hide this altogether until we are providing proper Data Source support
* Display Tags 
** if the number of tags exceeds the width allotted, display a ‘+more' and show the other tags in a popup
* Provide an Edit button

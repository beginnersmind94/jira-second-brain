# NXT-6440 — INV - Add Vendor Info - remove Delivery Schedule from Add Vendor page

## Description
As an inventory user, I want the ability to add a vendor without specifying a delivery schedule so that I may be able to configure my delivery schedule on the contract and be able to manage non-delivery vendors.

Per the [https://primeroedgenext.atlassian.net/browse/NXT-6425|https://primeroedgenext.atlassian.net/browse/NXT-6425|smart-link]  story, vendor delivery schedules will be configured per vendor contract and not on the main vendor information. We need to move the delivery schedule configuration from Add Vendor to the Contract Details configuration.

## Acceptance Criteria
User should be able to add vendor without entering a delivery schedule.
Complete button should be provided on the 'Add Contact Details' second step.

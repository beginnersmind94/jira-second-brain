*Requirements:*

* Session is selected from the session list
* Information about the session is displayed
* As transactions are recorded the display is updated.

## Acceptance Criteria
The session is selected from the session list.
Sessions can be selected for viewing.
View Only
The Cashier name is displayed
Opening balance and time are displayed
Number of meals served is displayed
Cash taken in is displayed
Refresh on demand

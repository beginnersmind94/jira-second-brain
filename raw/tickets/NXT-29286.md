# NXT-29286 — INV - Reports - Valuation - Provide Specific Date option

## Description
Provide the Specific Date option to the Valuation Report

*Requirements*

* Provide the ability to generate the Valuation Report for a selected date, as an On-Demand variant of the Valuation Report 
* Add another report option titled ‘Specific Date'
** if selected, show a Date and Time selector
* Consider size of data rollup:
** in PrimeroEdge, if ‘ALL sites’ is selected, the report requires time to generate
** If single site is selected, report is immediate
* Generate the data rollup for the selected date and time
** Potentially save these custom rollups for later display, like in PrimeroEdge
** *For ALL sites,* take time to generate the report
*** replace ‘Generate’ button with a ‘Start’ button
*** options: show a ‘Come back later’ type message
* Add all previously generated to the ‘On Demand’ dropdown
** show all previously generated rollups
* *Show different report design* 
** no Beginning vs Ending, just current valuation

# NXT-3796 — AcctMgt - Homerooms - Landing Page

## Description
*User Story*

As a district administrator I want a page where homerooms can be seen and searched for.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=acm_landing_page&id=ravqth&view=default|https://nqyc9v.axshare.com/start.html#p=acm_landing_page&id=ravqth&view=default]

*Requirements*

* Add a Homerooms button to the Account Management landing page per the initial wireframes from Karthik.
* Add a Homerooms link to the sidebar on the left. (via the hamburger)
* Add a Homeroom search page that both the button and the sidebar link lead to.
* Page should contain a way to search for homerooms based off the following criteria:
** Homeroom Name
** Site
* User can search by homeroom name, or use the site option to view all homerooms at a site.
* Results should load into a split view.
** Cards on the left showing the following information: 
*** Homeroom Name
*** Site
*** Grades (Added 8/20/20)
*** Number of active students in the homeroom

## Acceptance Criteria
Homerooms landing page added.
Search options added.
Search results load into a split view.
Search results show appropriate information.
Multi-language support added.

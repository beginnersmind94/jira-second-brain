# NXT-45552 — INV - Inactive Items - Configuration screens

## Description
As a district admin I want the ability to make an Item inactive if it is not being used anywhere in the system, so I can delete any unused or accidentally created/imported items.

## Acceptance Criteria
* Update the Configuration pages to handle inactive items, however easiest
** remove or disable the item as necessary to prevent page or logic errors
** otherwise, show the item with an alert icon and a corresponding error or tooltip message
*** exact implementation is dependent on the specific Inventory screen
* *No change*: Delivery Locations, Charges & Discounts
* *UI change:* Templates, Site Par Levels
** showing item as inactive until user edits, then removing from the record
* *Logic change*: Ordering Conditions, Review Order Settings
** Retaining existing item and values and prompting the user to adjust the record themselves when editing

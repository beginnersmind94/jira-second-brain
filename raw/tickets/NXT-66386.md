# NXT-66386 — Inventory Usage (including Commodity Usage)

## Description
As a *district inventory manager*,
I want inventory usage (withdrawals for production) to create ledger entries,
So that food costs are captured as expenses and inventory balances decrease accordingly.

## Acceptance Criteria
* Preconditions:
** Financial accounts mapped for valuation groups (_Inventory Asset, Inventory Expense_).
** All inventory periods up to posting date closed.
** Financial period open through posting date.
* Posting date = last day of inventory period.
* *Ledger entries per site*:
** _Usage (GLEntryTypeCd: WD)_ → Debit Expense, Credit Asset.
** _Commodity Usage (GLEntryTypeCd: COMMUSG, Value = Standard Price)_ → Debit Expense, Credit Asset.
* Rollbacks: Reverse entries if reposted.
* Entries visible in GL journal/View Ledger, flow to P&L as food expense and Balance Sheet as reduced inventory.
* The availability of Approve, Post, and Reapprove follows the checkbox state matrix (see attached rules).

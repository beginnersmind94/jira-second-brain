# NXT-57277 — RevTrak - Adult Balance Export - Blanks (FD 268315)

## Description
As a SchoolCafé Admin I want to be able to map blank values in the adult balance exports to RevTrak so that the export file can map their needs

*Notes:*

* Note this is for Adult balance export only
** Student balance export can be handled separately
* Note this comes from a production request: [https://primeroedge.freshdesk.com/a/tickets/268315|https://primeroedge.freshdesk.com/a/tickets/268315] 

*Requirements*:

* For Add New and Edit screens for Adult Balance Export, do the following:
* On the mapping screen, do the following:
* Left table should have columns that can be used to map blank values:
** Add 5 rows to the bottom of the list with the names as follows:
*** Blank 1
*** Blank 2
*** Blank 3
*** Blank 4
*** Blank 5
** Rows should be draggable like normal columns to any column in the target columns table
*** Should act just like any other field
* 
* When generating the file, ensure that if a column is mapped to one of the blank fields, the output file should put empty text like ''
* 
* Ensure the blank fields are saved in the History log and viewable in the View History drawer when done by Add New
* Ensure the blank fields are saved in the History log and viewable in the View History drawer when done by Edit screen
* Ensure the View Config drawer shows the blank fields 
* Ensure blank fields are cloned when cloning

## Acceptance Criteria
* User can drag one of the 5 blank columns to a field in the target table
* System will include empty text (zero length) in the export file for that column
* Everything else works the same
* User can see the Add New and Edit blank columns changes in View History
* User can see the blank columns in View Config drawer
* Clone will clone blank columns as well

# NXT-33716 — INV - Transfers - Transfer Ticket - Update report for pack sizes

## Description
As a Site Manager I want the Transfer Ticket report to reflect the way items were entered on the Transfer so I can see exact unit and price details.

*Requirements*

* Update the Transfer Ticket Report to display pack size units as they are displayed on the Transfer Details screen
* Update for History tab version as well
* UI Corrections
** This should not open in another full tab
** The top of the report is cut off
** Not proper width (follow recently clarified standards)
*** 10-15 characters for Item #
*** Max possible width for Item Description
*** Decent width for Quantity column
*** Minimal width for numeric/dollar value columns 
**** values would rarely exceed 4 digits + decimals (e.g. $1,000.00)
*** Comments can use any extra width not needed by Item Description

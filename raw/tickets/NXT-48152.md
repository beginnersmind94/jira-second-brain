# NXT-48152 — Subscriptions - View History

## Description
As a SchoolCafe user, I would like to be able to see what changes to subscriptions there are so that I can know who is being notified of what.

*Notes:*

* This is the follow up story to the View Subscriptions tab when a user clicks on the History Icon

*Requirements:*

* For the Event Subscriptions page > Subscriptions Tab:
* When a user clicks on the History icon, it should open up the Subscription History drawer
* 
* Create a new drawer called Subscription History
** Size: Half page
** Show our standard view history component for add/edit/delete actions for a given subscription
** Close button

## Acceptance Criteria
* User is able to view the audit history for a given subscription

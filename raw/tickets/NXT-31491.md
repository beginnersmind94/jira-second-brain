# NXT-31491 — IM - Onboarding Items - POS Information - View Log 

## Description
IM - Onboarding Items - POS Information

As  an Item Management Super User / Administrator, I need to be able to onboard Items POS Information using an excel file in order to speed the process of bringing customers working with SchoolCafe’s Backoffice.  



The detailed import steps are included in the story # [https://primeroedgenext.atlassian.net/browse/NXT-29782|https://primeroedgenext.atlassian.net/browse/NXT-29782|smart-link]

## Acceptance Criteria
AC:

Once the import is completed, the user should be able to view the log as per UX design (attached below). 

The summary should list the total number of records in Excel, how many are processed, how many are created, how many are updated, how many are exceptions, Start Date/Time, End Date/Time and Status (as Completed)

The grid should list the status of import for reach record along with exception message as shown in UI

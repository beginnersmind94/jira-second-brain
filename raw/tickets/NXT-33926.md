# NXT-33926 — Import - Improved Import Exception Management - Item Details - Integrate UI and API

## Description
When imports are performed, many exceptions can occur preventing numerous records of data from being imported. The objective of this story is to help narrow the importing user’s focus onto the records that could not be imported so that she/he can easily make corrections to the data and complete the import as fast as possible.

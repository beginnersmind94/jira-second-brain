# NXT-28384 — INV - Orders & Receipts - Create Orders - Order Guide report

## Description
As a site ordering manager I want an Order Guide report so I can have a convenient report version of all the items I can order, the vendor from which I can order them, expected prices, expected delivery dates, and other information.

## Acceptance Criteria
* Provide an Order Guide report on the Create Order page 
* Display the list of items for each vendor
* Display the contract attributes

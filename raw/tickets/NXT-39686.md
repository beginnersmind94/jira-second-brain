# NXT-39686 — PE - Item Serving Size Changes (USDA Branded Foods)

## Description
_(none)_

## Acceptance Criteria
Navigation: MP → Global Catalog

User should be able to import a USDA Branded Foods items with following serving sizes:

# Actual Serving Size (As-is, working already)
# Additional Serving Size as 100 Gm or 100 ml based on the serving measure (ServingSizeUnitId)
## Serving Size = 100
## Measure = Gram or ml
## Weight = 100
## Unit = Gram or ml

Upon creating a local item, the user should be able to see 2 entries in the 

# Actual Serving Size (As-is, working already)
# Additional Serving Size as 100 Gm or 100 ml

Upon Navigating to Nutrient Tab in an local item, the user should see 100 gm / 100 ml as a default serving size (because USDA always share the nutrition as per 100 gm or 100 MLT serving measure)

Rule: 100 gm / 100 ml entry shall enable the scaling to work when the item is used in a recipe. This change should be applicable only for USDA Branded items import (not applicable for 1WS)

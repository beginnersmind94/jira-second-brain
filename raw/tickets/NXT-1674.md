# NXT-1674 — INV - Withdrawals - Withdrawal Reasons Configuration

## Description
As a FSD I want the Withdrawal Reasons page so I can configure the reasons that will be associated with a withdrawal.

*Note:* use the same UI as corresponding Additions story: [https://primeroedgenext.atlassian.net/browse/NXT-20675|https://primeroedgenext.atlassian.net/browse/NXT-20675|smart-link] 

*Requirements*

* Display a configuration page titled Withdrawal Reasons
** User navigates to this from the main Withdrawals page (NXT-1654)
* Display a header section that allows the user to create a new Withdrawal Reason
** Provide a textbox titled ‘Withdrawal Reason’
** Display an ‘Add’ button to add to the list
* Display a list section which displays a list of existing Withdrawal Reasons
** Columns:
*** Withdrawal Reason
*** Action column
**** Edit icon
***** this will enable the text field for the Withdrawal Reason so user can edit
***** this will replace Edit icon with Save and Cancel icons 
**** Delete icon
***** this will remove the reason from the list
***** this icon will be hidden if the reason is used on any transaction
** This will contain several pre-populated default reasons
*** Waste
*** Spoilage
*** Usage - Production Withdrawal
*** Usage - POS A La Carte Withdrawal
*** Usage - Standard
*** Usage - Distribution

# NXT-43480 — MP- Menu Item Quick list Create on Menu Template Page (UI ONLY)

## Description
As a user, I would like a way to create a new menu item quick list from the menu template page.

## Acceptance Criteria
Give the user a way to create a menu item quick list from the quick list view page on the Menu template. 

This will open a overlay/ new tab for user to go to quicklist configuration

User will land on the create screen for quicklist

Creation process will be the same as ([https://primeroedgenext.atlassian.net/browse/NXT-1880|https://primeroedgenext.atlassian.net/browse/NXT-1880|smart-link] )

Save will be save and close so that the previous add items from quicklist screen appears

Once quicklist is created, show new quicklist with information on the slideout on the Menu template screen.

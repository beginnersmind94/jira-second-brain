# NXT-32104 — PR- Consolidated Plan Tab- Planned Recipes Report Link

## Description
As a site manager, when I am working on serving my students each day, I need access to the recipes that belong to the menu items. These recipes are most helpful when they’re scaled to my needs. The nutritionist might write the recipe for 50 servings, but I’m going to be serving 275 students. I need the recipe to give me the details for those 275 servings. If the recipe is for a cheeseburger, I want it to tell me that I need 275 buns, 275 patties, and 275 slices of cheese. The quantity of food required that the recipe generates should match and be appropriate for the number of servings that I have entered on my production plan.

## Acceptance Criteria
There will be a button on the Consolidated Plan Page that opens a page or pop up that allows me to select the recipes that I’m going to need. I will be able to make those selections and generate a report that will contain all of the recipes that I’ve selected, scaled to the quantities that are reflected on my consolidated plan tab

Link appears when consolidated plan is saved for the first time

If consolidated plan is deleted, hide the planned recipes link until it is saved again

# NXT-2450 — Inspections - Export Report

## Description
*User Story:*

As an Inspections user, I have the ability to export a template, inspection, or corrective action into a pdf report, print it directly, or send it via email to an individual, individuals, or groups.

*Requirements:* [https://primeroedgenext.atlassian.net/wiki/spaces/INSP/pages/26509313/Inspections+Export+Report|https://primeroedgenext.atlassian.net/wiki/spaces/INSP/pages/26509313/Inspections+Export+Report|smart-link]

## Acceptance Criteria
Templates are able to be exported as .pdf files, printed, or emailed to an individual, individual, or groups without any issues.

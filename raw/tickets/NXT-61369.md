# NXT-61369 — INV - Distribution - Shipping - Modify Shipments - Move Orders

## Description
As a warehouse distribution manager I want a tool to make changes to the default shipments loaded on a distribution, so I can customize according to my specific shipping/delivery needs.

h3. Shipping tab - Modify Shipments - Move Orders

* Reset is default, but is disabled when there is nothing to reset

!Shipping - Modify - Move Orders Rev-20250529-195011.png|width=91.66666666666666%,alt="Shipping - Modify - Move Orders Rev-20250529-195011.png"!

## Acceptance Criteria
* Open a slideout when the ‘Modify Shipments’ button is clicked
* Display a ‘Modify Shipments By’ heading with 3 options
** Reset to Original (default)
*** disabled when user is on default data setup
** Move Orders
** Combine Shipments
* When *Move Orders* is selected, provide dropdown tools to move one or more orders from one shipment to another
** first capture the shipment being transferred *_from_*
** then provide a multi-select dropdown for the orders in that shipment
*** display (Site Code) Site Name - Order #
** lastly, capture the shipment to which the order(s) will be moved
** Display a ‘Move X order(s)’ button in the footer
*** only enabled after all selections are made
*** Display a confirmation prompt on click: “Are you sure you want to move X order(s)?” 
*** Button icon: [https://fonts.google.com/icons?selected=Material+Symbols+Outlined:multiple_stop:FILL@1;wght@400;GRAD@0;opsz@24&icon.query=swap&icon.size=24&icon.color=%235f6368|https://fonts.google.com/icons?selected=Material+Symbols+Outlined:multiple_stop:FILL@1;wght@400;GRAD@0;opsz@24&icon.query=swap&icon.size=24&icon.color=%235f6368|smart-link] 
* Exclude from the dropdowns any shipments which are marked ‘Ready to Ship’
* After moving orders, add the orders to the ending of the destination shipment
* After moving orders, modify the Delivery Group column
** show italicized # of stops
** add info icon, tooltip: Modified Route

# NXT-40753 — PR- Record- Display no data available or hide if there is no production record (UI ONLY)

## Description
When a prep site is purely prepping for another site and not prepping for themselves, there is no reason to show the production record tab or at the very least be able to display no data available ( The way we show on the plan tab)

## Acceptance Criteria
* When the production plan does not have data in a prep site situation, hide the production record tab after the prep site distribution
** IF hiding the tab is not feasible, display the same message we are displaying on the plan tab
* On Plan tab, display warning icon with tool tip (No Plan available), this only should apply to prep sites that do not have their own plan
* Also use this warning when user is already on the plan/record and then the plan gets removed from MP

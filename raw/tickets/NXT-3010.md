# NXT-3010 — **View Inspections - Actions - Reschedule

## Description
*User Story*
As an Inspections user viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to reschedule the Inspection for a different date.

*Requirements*

* Reschedule is an action that can be applied to any Inspection in the Scheduled status
** This action is only present on Inspections in the Scheduled status
** Only users with permission to schedule and view the particular Inspection can see the 
* When clicked, the user receives a slideout
** The header says "Reschedule Inspection"
** The Inspection Name, Site, and Current Scheduled Date are all listed
** User can choose from a date on a calendar widget
** There is a button for submitting and a button for canceling
* Once submitted, the Inspection is updated wherever the scheduled date is displayed
** The Inspection remains in the Scheduled status
* The assigned Inspector receives a notification of the new scheduled date.
* If the site was informed when initially scheduling the Inspection, they are informed of the change in date.

Functionality of  Re-scheduling in UI:
If the Card/List item in the Scheduled status only the Re-schedule action button should be {color:#ff5630}enabled{color}
For other Status should be {color:#ff5630}DISABLED{color}

## Acceptance Criteria
* User can reschedule a Scheduled Inspection from the View Inspections screen in both views (list and card)
* User can reschedule a Scheduled Inspection from the Inspection Details screen

# NXT-23108 — Production- Reports - Post Production Analysis- Nutrition Report

## Description
As a user, I need to have a Post production analysis Nutrition report similar to my weekly menu analysis based on the days I select.

## Acceptance Criteria
Report should match the menu planning weekly nutrient report while using the served production numbers.
Use weekly menu cycle analysis for reference.
Display selected dates nutrient summary with all break downs for all nutrient values with the standard (USDA requirements), Actual, and percent of calories. 

Display selected dates food component summary with all break downs for all nutrient values with the standard (USDA requirements), Actual, and percent total. 

Display selected dates break down for each day nutrients along with served quantities. 

There needs to be a printable option for this report with All, expanded and simplified versions based on the same criteria in menu planning weekly analysis as well as display on UI.

 How to calculate counts since there are no serving groups:

1.) Get the "Produced" count from each menu item on each day
2.) Multiply that number by the nutrient information 
3.) Add each days results together into one number
4.) Divide that result by the total meal count on that day to get the single day average 
 a.) If there are multiple serving groups, add the TOTAL counts of the serving groups together, then determine the percentage of each serving group on each day
 b.) Multiply the results from step 4 by those percentages to get the daily average for each serving group
5.) Take all of the single days averages, add them together, and divide by the total number of days
 a.) Do this for each serving group, if there are multiple

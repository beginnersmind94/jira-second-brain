# NXT-18503 — Acc. - Claims - Site List Report

## Description
As Matt the Product Manager I want the system to inform users on what sites are configured for a particular claim export, and for what dates of that period.

*mockup*

[https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=950%3A25792&scaling=scale-down-width&page-id=537%3A6269&starting-point-node-id=950%3A25792|https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=950%3A25792&scaling=scale-down-width&page-id=537%3A6269&starting-point-node-id=950%3A25792|smart-link] 

*requirements*

* User has a way to view what sites are configured for the selected period, and for what dates.
** In the action dropdown for the Program Reports option, add a ‘Site List’ tab for each active program.
** It should check Program Configuration for the:
*** selected period date range
*** selected program
*** return the sites that are part of both
*** return the dates that they’re part of both
** It should show the:
*** details on period, program, date range that the period covers
*** site type
*** site code + site name
*** active dates the site is participating
*** group by site type
*** order by site type, site code

Discussion on 10/8 during refinement:

* Actual implementation will use a similar layout to Program Configuration instead of the UX re: tables vs accordion placement.

## Acceptance Criteria
User has a small report that shows them

* information on the period (period name, date range)
* sites that are included in the program + date range
* the dates the site is participating

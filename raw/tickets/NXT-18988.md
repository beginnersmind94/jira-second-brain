# NXT-18988 — Production - Production Status 

## Description
The Production Status page can be used by an assortment of users within a district. It allows 

a.) central office to view all of the sites in the district

b.) supervisors to view their area or territory of sites within the district

c.) Site Managers to view their own site for a date range

Those users are able to determine what status the production plans and/or records are in for the range of sites/meals/dates that they’re looking for. It gives an easy “Color Coded” view, that matches the production calendar status colors, so that it can be interpreted “at a glance” by the users.

!image-20210621-163549.png|width=830,height=537!

## Acceptance Criteria
This page should be accessible by district, area, or site - as needed by the district.

It will allow the users to select a meal type (or all meals), a date range of their choosing, and the status level that they want to filter by. 

The results will display in an easy to view manner for the user, and will allow for the user to export the results of the grid into an excel spreadsheet.

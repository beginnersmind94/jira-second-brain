# NXT-71358 — GC - SC - Itemmanagement - Items - Upload items : Add nutrient serving size column in the results grid.  

## Description
Serving size column is added after item name column

Serving size column shows serving amount, serving size unit, serving weight in grams

Serving size column is sortable allows filtering by a value 

serving size column should be available 

* for both search by filters and upload file options. 
* search by filters option and for all data source dropdown options.

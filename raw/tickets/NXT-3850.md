# NXT-3850 — Reports

## Description
*User Story*
As an Insights user, I can view and create Reports using various enabled Measures.

*Mockup*


*Requirements*

* While reports can still have filters, they are less interactive than Dashboards and generally only contain data from a single module.
* Users can view a list of all existing Reports.
** PrimeroEdge created Dashboards
** User custom Dashboards
* User can create new Reports
** Users can add Measures
*** Default KPIs
*** Custom KPIs
*** Metrics
** Users can select how Measures are visualized
*** Graphs/charts
*** Tables
*** Limited date ranges

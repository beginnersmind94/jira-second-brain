# NXT-53450 — IM- Update CNDB to remove items that do not have Total Fat

## Description
USDA has acknowledged having over 700 items in the CNDB that are missing total fat. This is a problem if users are trying to copy these items as there is no way for the system to be able to correct these items later.

## Acceptance Criteria
* Remove all CNDB items from CN27 that do not have Total Fat
** Based on excel filters, there should be 764
* If the item exists on CN26, use CN26 data (with correct fat value)
* If the item is new, don’t create it with the missing fat value

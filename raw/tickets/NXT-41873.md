# NXT-41873 — Imports – File Mapping – Make 2 Columns Height Align

## Description
As a SchoolCafé Admin I want the 2 table heights on Field Mapping to align vertically so that it it easier to drag and drop.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Imports
* Click View on a Student Import record to view the import details
* Go to the File Mapping tab
* View the 2 school café tables with the same height

*Requirements*:

* Apply this to all imports
* On the File Mapping tab, do the following:
* Align the height of the the middle and last tables 
** when adding chips to the middle, it expands and grows larger than the right
** make it so this doesn’t happen
* If you can, get rid of the scroll bar on the left one

## Acceptance Criteria
* Middle and right tables have the same height even when drag chips into the middle

# NXT-5248 — System - Reimbursement Rates - Add Administrative Rates tab

## Description
As a district or central office admin, I would like to be able to view the Administrative sponsorship reimbursement rates pertaining to my organization, so that I can rightly plan our financial goals.

+Summary+:

Allow a user to view the reimbursement rates for sponsorship of day care homes for their selected academic year and state category.

+Details+:

* This page will be for viewing data. For now, no updates will take place here.
* In the Administrative Tab of the Program Configuration page, allow the user to select an academic year and state category filter. Just like previous tabs-
** Academic Year: Current year should be selected by default, and previous years should be sorted in descending order. It can display up to the last 10 years.
** State Category: Contains ‘Contiguous States’, ‘Alaska’, and ‘Hawaii/Puerto Rico’ (in that order). Contiguous States should be the selected option by default.
* Beneath the filters, display a list of day care home sponsorship rates, which should match those from Classic.
* For each line, display associated reimbursement rates for Federal or State categories. This data will come from the government each academic year but can be copied from Classic and stored in the database for now.
* When the user switches between filters, the data should be reflected to the user immediately.

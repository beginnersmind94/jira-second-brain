# NXT-61848 — IM - Bulk Item - Summary Info - Header filters

## Description
As an onboarding district admin I want filters within the Summary Info section so I can select a subset of items for a specific update, or find items with specific data conditions such as missing categories.

!image-20250926-201006.png|width=366,alt="image-20250926-201006.png"!

## Acceptance Criteria
* Add filters to the Summary Info section (all selections should be held as user moves to update either categories or branding)
** Reference the Items > Advanced tab filters for some reference
* Provide category dropdown filters
** Item Category
** Storage Category
** Valuation Category
* Provide Brand / Manufacturer combined smart search
* Provide Tags dropdown
* Ensure Product Code column is searchable with filter options
* Provide an additional “Summary Info Readiness” flag drop down
** All Categories Missing
** Partial Categories Missing
** Branding Missing
** Tags Missing
* Add a toggle to allow for users to see the items that have Pack size info and are required to enter Category information “Show Items with Pack Information”
** These are all items that the user has indicated as having inventory units
** This is default off
* Provide instructions in help flyout (This will be the standard across the tool)

# NXT-12398 — Menu Grid - Default Item - Display icon indicator in ExpressPoint

## Description
As an ExpressPoint user if a default item is configured for the menu I want to see a visual indicator (star as it is configured in Menu Grids) and have the default item to load automatically for an identified patron so that I can speed up line service and avoid having to select the button.

## Acceptance Criteria
* If menu is configured with default item we should display the visual indicator (star) in the top-right corner so that it appears the same as menu grid configuration
* Default item should load into the sale/transaction list automatically once the patron is identified
* Default item should trigger a second meal alert if the patron has already had a reimbursable meal for the day/meal service

# NXT-64973 — SCTV - Remove static data categories for default templates

## Description
As a user, I want default templates to only have my categories

## Acceptance Criteria
* On linking, default templates should include only categories that belong to the user’s menu.
* If the template has more slides than categories, the category selection will be removed instead of static categories or blanc slides in order to improve the UX (As discussed).

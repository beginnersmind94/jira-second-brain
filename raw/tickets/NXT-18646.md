# NXT-18646 — INV - Orders - Manage Orders - Do not allow an Order to be edited if any Receipts exist

## Description
As an operations manager I want the Edit functionality to be disabled if any receipts exist for the order, so I do not accidentally make changes to an order which is completed.

*Requirements*

* Disable/remove the Edit icons on the Order Detail when one or more Receipts exist for an order
** Delivery Location
** Delivery Date
** Items
* Hide the Add Item button as well
* Change Comments popup to view-only mode
** Order level and Item level

## Acceptance Criteria
* Prevent order adjustments if a receipt exists for the order

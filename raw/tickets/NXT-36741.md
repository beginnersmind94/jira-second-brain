# NXT-36741 — IM - Item Config - HACCP Configuration page - Control Points

## Description
As a District Admin I need a HACCP configuration page so I can configure or add any processes unique to my district.

*Requirements*

* Add a HACCP configuration sub-page to the Item Configuration page
* Have a section for the HACCP Processes [https://primeroedgenext.atlassian.net/browse/NXT-36069|https://primeroedgenext.atlassian.net/browse/NXT-36069|smart-link] 
* Have a section for the HACCP CCPs (Critical Control Points)
** Maintain the following fields for each Process
*** CCP Description
**** unique 
*** Critical Limit *
*** Default Corrective Action
* Provide the ability for users to add custom CCPs
** Capture the following fields
*** CCP Description
*** +Critical Limit+ attributes (**Note:* we likely to explore options for how to manage the editable operator to manage this a more cohesive string)
**** Leading Text
**** Operator
***** <= (less than or equal to)
***** = (equal to)
***** >= (greater than or equal to)
**** Default Temperature
**** Trailing Text
*** Default Corrective Action
* Allow *custom* CCPs to be edited or deleted
* Maintain the *History* for all CCPs
** display at header level instead of each CCP row

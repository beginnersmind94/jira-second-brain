# NXT-32226 — Periods Page - Enforce Fiscal Year As Mandatory

## Description
On both the “Generate Periods” and “Manage Periods“ tabs, an end-user should not be able to interact with with any UI control if a fiscal year is not selected or if a fiscal year is deleted from the fiscal year dropdown.

## Acceptance Criteria
* As a user, on both the “Generate Periods” and “Manage Periods” tabs, if the “Fiscal Year” dropdown is empty, I only want to see and be able to interact with the “Fiscal Year Configuration” section.

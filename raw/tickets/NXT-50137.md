# NXT-50137 — LMA - License Keys - Cancel

## Description
As a License Management (LMA) role member, I would like to cancel a license key from a district so that they can’t continue to use the site

*Notes:*

* The use case for the story is a district cancels our software before the license expires and we need to remove their access

*Flow:*

* District buys full suite district licenses
* They then decided they don’t want them and we agree to cancel the licenses
* LMA role user browses to System > License Management
* User sees a list of licenses generated for the software in the grid
* User clicks Cancel License button on a row
* Cancel License drawer opens and enters termination date and saves
* System sets the status to Cancel on the date specified
* License Keys grid updates with the new information
* When district users login, they now no longer have access

*Requirements:*

* On License tab grid, do the following:
* Grid Actions column:
** Cancel License
*** On click:
**** Open the Cancel License Key drawer (covered in another story)
**** Enable only for New and Active status
*** Only License Management role can click this button



*Cancel License drawer*

* If the drawer does not already exist, create it
** Title: Cancel License
** Width 1/2
** Use sliding motion



*Header*

* Show these fields are read only:
** District
** License Key
** Status
** Licenses
*** Comma delimited list of licenses on the key
** Usage message (see bolt code)
* Show these fields as editable and required:
** Termination Date
*** Must be between today and the expiration date
* Submit Cancellation button
** Only LMA roles can click this button
** On click:
*** Validate required fields
*** Show this message: This will permanently cancel the license key on the selected date. Are you sure?
*** If confirmed:
*** Set status to Cancelled on the date they picked
**** Append to the note who cancelled it and when
*** 
** Show the success message, close the drawer and return the user to the License Keys page with the grid updated
** Log in the History table as an Update
* 
* Security
** License Management role only has rights to take actions on this page

## Acceptance Criteria
h3. 1. Access and Navigation

* From the License tab grid, clicking Cancel License opens the Apply License drawer.
* Cancel License button is enabled only for New and Active status licenses.
* Button is disabled for all other roles except License Management (LMA).
* Only LMA role can take actions on this page.

h3. 2. Drawer Layout

* Title: Apply License
* Width: half page
* Opens with a sliding motion.

h3. 3. Header Section

* Read-only fields:
** District
** License Key
** Status
** Licenses (comma-delimited list of all licenses on the key)
** Usage message (from Bolt code)
* Editable and required field:
** Termination Date
*** Must be between today and the expiration date of the license key.

h3. 4. Submit Cancellation Button

* Only LMA role can click.
* On click:
** Validate that Termination Date is provided and within the allowed range.
** Show confirmation message:
_"This will permanently cancel the license key on the selected date. Are you sure?"_
** If confirmed:
*** Set Status = Cancelled on the selected termination date.
*** Append to the Notes field who cancelled it and when.
*** Show success message.
*** Close the drawer and return user to the License Keys page with the updated grid.
*** Log the cancellation in the History table as an Update.

h3. 5. Security

* Only License Management (LMA) role can take actions on this page.

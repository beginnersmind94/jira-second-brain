# NXT-66379 — Import script for chart of accounts (Proposed #3)

## Description
*As a* user

*I want* to import my chart of accounts

*So that* the journal entries reflect my actual accounts

## Acceptance Criteria
* When the user uploads a Chart of Accounts template, the system reviews it against the district’s configured account code structure.
* The system identifies whether each account category and individual account aligns with the configured structure.
* The system produces a list showing which account codes are valid and which are not.
* The results are displayed in a table linked to the customer’s account, with a clear indicator of each account’s status (valid/invalid).
* If changes are made to the uploaded accounts, the table reflects the updates and their current validation status.

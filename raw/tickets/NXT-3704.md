# NXT-3704 — Add Item - Ingredient - Add Nutrient Information - Save & Close

## Description
*User Story*

As a Item Management User, when creating an Ingredient, I want to save the information I have entered for an Ingredient

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1]

*Requirements*

* When clicking ‘Save & Close’ at the end of the process
** If any Required fields are missing, have a popup showing those fields
** If all required fields and logic are correct, then save the item to the database
** Navigate back to where the user clicked ‘Add New'

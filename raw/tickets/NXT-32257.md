# NXT-32257 — Resource Catalog - Audit Log

## Description
Audit all changes to the Resource Catalog.

## Acceptance Criteria
* The following changes to the Resource Catalog shall be audited:
** Additions
** Modifications
** Deletions
* An “audit” icon shall be added to the action column on the Resource Catalog page that when clicked, will display an audit history window.
* The audit history page shall look and behave like the user audit log page

## Acceptance Criteria
* *Update* the common QOH flyout component that is displayed across the Inventory module
* Change the heading to ‘Pack Size Quantity Details’
* Label the first row based on the context of the page
** if page is referencing Actual QOH, display “Quantity Actual”
*** Note: all current pages should be using this version
** if page is referencing projected, available, etc., display ‘Quantity Available’
*** Note: this probably doesn’t apply to any existing pages except for Distribution
* For the pack size conversion rows, change ‘QOH' to just ‘Quantity’
** ex: Highest Unit Quantity, Middle Unit Quantity
* Resize the flyout to be minimal size; reduce empty space

# NXT-1954 — Meal Pattern - Reports - Nutrient/Component Standards (Report) - Printable Option

## Description
Page 54 of the USDA Approval requirements document. Can this be handled with a print of the page? If dedicated report is needed, speak with the Insights team on how reports are generated. May be able to have a dedicated “report” page that displays the information.

The USDA Approval document states “The user must be able to specify the nutrient standards to be included on the nutrient standard report, including multiple nutrient standards or all the nutrient standards, if desired.“ 

Our current version of this particular report does not allow for the user to specify which (if any specific) nutrients they desire to to view, and instead lists them all. 

The right hand column of the current Meal Pattern Details should be printable in the manner that was available prior to the navigation changes. 

Print Icon to be used in the top right hand corner

Serving Group should be included in the printable information.

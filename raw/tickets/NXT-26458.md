# NXT-26458 — Acc. - End of Day - Actions - Day Finished

## Description
As Agatha the Site Manager I want to sign off on a task and have that information tracked so my overall percentage goes higher.

*mockup*

[https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|https://www.figma.com/proto/Yo99YhQm5Xk91Ie7b0PRkD/Front-Office-6.0?node-id=66%3A5775&scaling=scale-down-width&page-id=11%3A1711&starting-point-node-id=44%3A5320|smart-link] 

*requirements*

* Add a ‘Day Complete’ button
** This marks the day as ‘Complete’ and should lock further interactions on the end of day page.
** User can roll the day back to Incomplete
* Add a Comments box. 
** Comments are not mandatory.
** 500 characters.
** Hint text for the box: Document any notes for the day or why any tasks are still open.
* ‘Day Complete’ can be triggered even if all tasks are not in a Done status. 
* Add a History action for the day.
* When clicked, load a drawer containing the following details for the day:
** Site Name, Date
** Day Status (Incomplete / Complete)
** Who clicked it
** Timestamp for the click
** Comments if entered



Success messages on button press:

Day Complete → Day marked as complete.

Day Incomplete → Day marked as incomplete.

## Acceptance Criteria
‘Day Complete’ button programmed

Can toggle between Complete and Incomplete

Button presses are tracked for the history action.

* Site + Date
* Day Status (Complete / Incomplete)
* Who pressed the button
* When did they press the button
* Comments

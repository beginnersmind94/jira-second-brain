# NXT-62438 — INV - Orders & Receipts - Merge Production & Standard Ordering

## Description
As a site manager I want a consolidated ordering screen so I can use a single page to do all order methods, including standard, production and par level ordering.

## Acceptance Criteria
* Remove the Production Ordering page
** (shelf for possible later usage)
* Add functionality to the Standard Ordering screen to search for production plans 
** Allow plans to be selected and quantities added to the order screen
* Link orders to the corresponding production plan

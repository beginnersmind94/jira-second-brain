# NXT-19307 — INV - Config - Site Par Levels - Update for Pack Sizes

## Description
As a Site Manager I want the Site Par Levels page to have a Unit Description dropdown for each item so that I can set the par level quantity in terms of whichever pack size unit I would like to use.

*Requirements*

* Modify the Site Par Levels page to consider pack size changes
* Provide a Unit Description dropdown for each item to specify which unit is being used to set the Par Level quantity
** default to the highest level unit
** only save items with a quantity greater than 0
** *Note:* if there are performance issues related to so many dropdowns, we can only enable the dropdown for items with some quantity
* -Convert Par Level Quantity to other pack size units as needed for ordering-

# NXT-47405 — GDSN Connect Import (SC)-Consumption of End Point to create Items in Schoolcafe- Spike

## Description
GDSN Connect End point for new item creation is available and same can be consumed to create items in SC End

## Acceptance Criteria
The API end points should be consumed at SchoolCafe for creation of Items at Schoolcafe end.

+*Scope*+: Items with Pack sizes 

The item shall be returned in the attached format (very close to Item Details Import, along with additional placeholders for Images)



*Coverage:*

* Creation of Staging table in SC
* Leverage existing Stored Procedure and apply business rules (revision of Images required in SPs)
* Consumption of end point and write the data in SC End

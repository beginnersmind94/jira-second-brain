# NXT-10973 — [TaggedDashboard] - Dashboard (Insights) - Accountability - Average Daily Participation (ADP) Widget

## Description
As a central office administrator or site manager I want to see the breakfast and lunch average daily participation (ADP) percentage and meal count for my district/site so that I can determine my performance.

!image-20210324-125352.png|width=983,height=519!

!image-20210324-130229.png|width=729,height=268!

*Values Needed:*

Information collected based on the site and timeframe parameters selected. 

* Student Reimbursable Breakfast Meals (Count) - For selected month/year
* Student Reimbursable Lunch Meals (Count) - For selected month/year
* Operating Days - Number of days within selected month/year that have meals
* Average Daily Attendance - Taken from System > Attendance > ADA

Until site and timeframe selector is available, start with the current month of operation and all sites available to the user.

*Inputs Needed:*

* Site Context - Site Selector for (All Sites, Multi-Site, Single Site). By Default, based on the authenticated user
** if a user is a district-level user, then it should be “All Sites”
** if a user is a site-level user having access to multiple sites, then it should be “All Sites”
** if a user is a site-level user having access to a single site, then it is selected (no dropdown is needed)
* Timeframe - Month/Year Selector (Current Month, \[Month/Year] descending). By default, the Current Month is the selection. Months are in descending order. Top option will be the Academic Year YTD that combines all values for the year to this point.
** Example values (Month Year)
*** 2020-2021 (YTD)
*** March 2021
*** February 2021
*** January 2021
*** December 2020
*** November 2020
*** October 2020

*Calculations Needed:*

* Breakfast ADP Rate % = (Student Reimbursable Breakfast Meals Count / Operating Days) / Average Daily Attendance
* Breakfast ADP Rate % = (Student Reimbursable Lunch Meals Count / Operating Days) / Average Daily Attendance

## Acceptance Criteria
* A visual representation is created that shows the Average Daily Participation Rate for breakfast (SBP)
* A percentage of breakfast ADP Rate is displayed
* A count of breakfast meals is displayed
* A visual representation is created that shows the Average Daily Participation Rate for lunch (NSLP)
* A percentage of lunch ADP Rate is displayed
* A count of breakfast lunches is displayed
* Operating Days is displayed
* Average Daily Attendance is displayed (taken from System > Attendance > ADA)
* All displayed values/calculations are based upon site and timeframe selections. If site/timeframe selections are not available, by default, we can load the current month of operation and all sites that the user has access to.

# NXT-56745 — AcctMgt/System - ID Card Templates - QR Code instead of Barcode

## Description
As Dana the Director I want to make use of QR codes on ID Cards instead of barcodes as they are easier to scan and less finicky. Plus it’s 2025.

*requirements*

* Add a field for QR Code to ID Card templates
** Essentially the same as the Barcode field but it will turn the Account ID # into a QR code instead.
** User can see how much space the QR code will use on the template
* Add a Standard Template that includes QR Code
** Avery - 30 per
** ID Cards
** Matt to provide layout for both cards

## Acceptance Criteria
* User can add QR Codes to ID Cards
** They should use the Account ID # like the Barcode field
* Prints, scans appropriately
* Standard Template added for QR Code

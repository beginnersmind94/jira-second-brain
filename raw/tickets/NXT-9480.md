# NXT-9480 — INV - Contract Items - Changes to existing page

## Description
As a District Supervisor I need my Contract Items page split into two tabs, Items and Ordering Conditions, so I can manage all contract items.

As a prerequisite to [NXT-5648|https://primeroedgenext.atlassian.net/browse/NXT-5648], we need to make changes to the existing Contract Items page:

*Requirements*

* Changes to existing page:
** add a ‘Items’ tab which contains the current version of the page (list of contract items)
** move the ‘Add Item’ button to the top
*** the Add button should only be visible on the Items tab
** remove the Contract dropdown
* Update the breadcrumb path correctly:
* Inventory > Vendors > \[Vendor Name]
* title of page should be the Contract number
** ex: ‘Contract A-123456’

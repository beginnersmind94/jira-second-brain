# NXT-47429 — API – Auto Field Mapping – Skyward – Adult

## Description
As a SchoolCafé admin, I would like to do the field mapping as fast as possible of the custom API call so that I can have more time to do other things.

*Note:*

* We know the data fields imported from the SIS providers and we know our own fields
* We can make life easier for import people by doing the field mapping for them in one click
* We can automate for them what they do by dragging and dropping
* Also, we can auto map certain code mapping fields, but that should be done already

*Requirements*:

* Backend
* Create a auto-mapping table to use to match SIS values to ours
* For each provider, add mapping rows for our codes and theirs
* 
* On the System > Imports & Exports > API tab > + New and Edit buttons, do the following for Skyward Adult Import:
* On the Field Mapping tab, do the following:
* On page load, do the matching action of the source data to the Mapped table for all the fields that we have pre auto mapped in the DB
* If there is a field that does not match, then ignore it
* Code mapping edit button behavior should behave the same way as if it was done manually dragging and dropping
* The rest of the functionality is unaffected by this

* 
* Add a button
** Title = Select All
** On click, do the same matching as on page load 
* Add a button
** Title = Unselect All
** On click, remove the matching of all fields

## Acceptance Criteria
* User can click the button to select all and map to Field Mapping automatically for them everything we have pre mapped
* On page load, it does the same thing as the auto map button
* User can unselect all field mapping
* The rest of the functionality is unaffected

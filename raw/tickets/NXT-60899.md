# NXT-60899 — INV - Distribution - Assign & Pick - Add item during Picking

## Description
As a Distribution Manager I want the ability to add new items to the distribution during Picking status so I can make item adjustments or substitutions without restarting Assignment.

h3. Picking - Add Item

* Reason sub-row required for each added item
* No substitution option

## Acceptance Criteria
* Provide the ability to add an item to a Distribution during Picking status
** Use the same tool from Assignment ([NXT-60898|https://primeroedgenext.atlassian.net/browse/NXT-60898])
* Remove the Substitution option
* Insert a sub-row for each added item to capture a Reason the item is being added
** require some comments to be added
* Show added items on the Assign & Pick tab with the same (+) icon
** Show Quantity Assigned as hyphens (--)
* Update the Picking Details slideout
** Add a Delete icon next to the Item label
** Do not remove the Quantity Assigned column
** Show hyphens for each site
** show ‘{color:#ff5630}Items Not Assigned{color}’ in assigned column footer
** Ensure all sites assigned to the distribution are always displayed in the slide-out

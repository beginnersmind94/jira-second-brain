# NXT-8181 — INV - Physical Inventory - add setting for Inventory Close Day

## Description
As an inventory user I want a setting for Physical Inventory Close Day so I can control whether or not my inventories reference the day of closing an inventory or the day before.

*Requirements*

* add a setting for Physical Inventory Close Day
* this setting only applies if Periods are not being used 
** references setting from [https://primeroedgenext.atlassian.net/browse/NXT-8157|https://primeroedgenext.atlassian.net/browse/NXT-8157|smart-link]
** Rename the setting: ‘Interim Physical Inventory Close Day’ 
** Provide description: TBD
* options are ‘Current’ and ‘Prior’
** if Current, the save as complete will check on the day the inventory is being saved
** if Prior, the save as complete will check on the day before the day the inventory is being saved
* Setting will apply to the entire district
* Create temporary setting values tables

# NXT-32516 — Roles and Permissions - UI Control to Check/Uncheck All

## Description
On the Roles and permissions page, the user needs the ability to “Check All” or “Uncheck All” for each of the View, Edit, Add, and Delete permissions.

+*Existing UI*+

!image-20220706-204958.png|width=818,height=521!

## Acceptance Criteria
* On the “General Permissions” tab on the “Roles and Permissions” page, there shall be a checkbox underneath the header for each of the View, Edit, Add, and Delete columns that will allow a user to “check all” or “uncheck all”.
** When this checkbox is checked, all resources within that column shall have their checkboxes checked.
** When this checkbox is unchecked, all resources within that column shall have their checkboxes unchecked.

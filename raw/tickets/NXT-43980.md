# NXT-43980 — Custom Reports – Add New Report - Configuration Tab Updates

## Description
We need to make the following enhancements that were missed requirements.

*Requirements*:

* All these requirements are for the + New and the Edit Overlays on the Configuration Tab
* 
* Operator column
** Text fields are missing the EQUALS value
* 
* Auto-Enable
** If the user selects a value in the Operator column for a text or number value (ones with an enabled list), auto select the Enable button for that row
** If the user selects a value in the Value column from an enabled list, auto select the Enable button for that row

 

h3.

## Acceptance Criteria
* User can select that a text fields equals something
* The Enabled button is auto selected when a user selects something from the list values in Operator and Value column

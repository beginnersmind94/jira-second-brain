# NXT-61861 — IM - Item Onboarding - Publish Mode Tabs

## Description
As a district admin using the item onboarding tool, I need a way to be able to at any time publish items that are in my session that are ready to go. I do not want to have to have all my items ready before I can publish.

!image-20250610-154754.png|width=1376,height=894,alt="image-20250610-154754.png"!

!image-20250610-154826.png|width=1381,height=893,alt="image-20250610-154826.png"!

## Acceptance Criteria
* Provide a count and list of items that are determined as ready to be published (Publish Ready Tab)
** If items being published are identified as items with duplicate descriptions (with local items), reload publish ready tab with the remaining “unpublished” items
*** Provide Alert for user to know “Duplicate item description found” 
* Provide a count and a list of items that are NOT ready to be published and need to be reviewed and corrected (Unpublishable Tab)
** User must go back to the overview and correct these in order for them to be able to publish
** Provide alert/ for each unpublish item reason
*** This will include Items that are “publishable” by validation standards, but not by item name uniqueness

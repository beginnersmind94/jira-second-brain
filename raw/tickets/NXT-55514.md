# NXT-55514 — IM - Item Onboarding - New Session

## Description
As an onboarding district admin I want the ability to create a new Item Onboarding session so I can decide which data I am going to create in the scope of the session.

## Acceptance Criteria
* Provide the ability to start a new item onboarding session
* Create a temp table that will store the newly created data until the session is completed
* Provide options to user for data entry method (manual, import, etc.)
* Provide a step in which the user can select which sections are in scope for their session
* Create the schema to store the selected sections
* Provide an ‘Edit Sections’ button that returns the user to the section selection screen
** Allow sections to be added
** Allow sections to be removed, only if no data has been entered for that section
** Once the sections are re-confirmed, navigate to the user back to the tool
*** if there are already items, navigate to the landing page
*** if there are no items yet created, directly navigate to the Initial Item Info step

# NXT-20074 — INV - Vendors - Add Items to Contract - Set 'Highest Units' flag to enabled by default

## Description
As a District Admin I want the ‘Show only highest units’ label to be enabled by default so I don’t have to see many items which are very unlikely to be orderable.

Follow-up enhancement for few additional changes from [https://primeroedgenext.atlassian.net/browse/NXT-32072|https://primeroedgenext.atlassian.net/browse/NXT-32072|smart-link] 

*Requirements*

* Set the ‘Display only highest units’ toggle to be enabled by default
** Since most items are only orderable at the case level, users should not see the other units unless they are specifically looking for them
* Set toggle in the API
** Currently the toggle is only hiding rows in the UI

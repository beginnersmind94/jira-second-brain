# NXT-27888 — IM - Item Substitution (2)

## Description
Asa Item Management Admin user I should be able designate an item substitute to another item so that in the vent original is not available the original Item can be substituted with the designated one.

## Acceptance Criteria
*Entry Point:* 

Item Management → Items → Selected Item → Item Substitution

*Fields*

* Search (As is)
* Substitute Item# (Item Name/code of the selected item as substitute, non-editable)
* Substitute Item Quantity  (As is)
* Substitute Unit Description (Unit of the selected item as substitute, non-editable)
* Item Quantity  (Text Box - Data entry)
* Item Unit (Unit of the current  item, gray out)

*Interchangeable* (As it is currently working)

*Actions* (Current rules apply)

* Filter based on local data source and items that have detail card
* For now CN items cannot be used as substitute

*Requirements:*

Allow the user to set Menus for all the meal types to be used as a default.

Default Menus can be set for a District, Area, Site Type, Site, and Device

Default Menus do not have to be configured for all conditions.

A Meal Type and Menu must be selected

## Acceptance Criteria
Launches from Menu Items, Default Menu Grids
Default Menus that have been set are displayed in the grid list
Clicking Add opens Set Menu Grids Popup
Default(s) can be set for a District
Default(s) can be set for a Area
Default(s) can be set for a Site Type
Default(s) can be set for a Site
Default(s) can be set for a Device
Default Entry for all drop downs is All
User selects a Meal Type and Menu Grid.
Selecting Edit for an existing default menu grid opens the Set Menu Grid popup populated with the saved data
Selecting the trash can for an existing default menu grid deletes the default settings, (it does not delete the grid)
The drop downs serve as a filter AND for setting the selection.
The hierarchy is Area->Site Type->Site->Device
If Area,Site Type,Site,Device is ALL then the menu is a district wide default menu
If the Area is set then only Site Types/Sites/Devices in that Area are displayed in the drop downs
If Site Type is set then only Sites/Devices of the selected Site Type are displayed in the drop downs
If Site is set then only Devices at that site are displayed in the drop down.
Only grids for the Site Type or Site selected are available.
When saved a message is displayed 'Default Menu Grid set successfully'.
If it fails a message ' Default Menu Grid not set. Please try again."

MenuGrids in the Menu Drop Down will display based on the MealType selection only
WeekDays are mandatory ?
Format we need to display the week day names in the Default Menu Grid - M, Tu, W, Th, F, Sa, Su
We need to display machine name in the Device Drop down
If the menu grid is set for default menu grid then the menu grid cannot be deleted. A message is displayed "This Menu Grid is set as a default. Please remove from the Default Menu Grid setting to allow this Menu Grid to be deleted."

The Area dropdown is not shown if there are no Areas.
A default menu cannot be set for the exact same set of selections. Example Selections are All for Area, Site Type, Site, and Device. Meal Type is Breakfast, and Menu Grid is Breakfast 2. The days selected are M,T,W,Th. You cannot set ANOTHER default grid for the same selections. You could set Grid Breakfast 3 for Friday ONLY.

Another Example After setting for Breakfast 2. You could do the following.
Selections are All for Area, Site, and Device. Site Type is Elementary, Meal Type is Breakfast, and Menu Grid is Breakfast Elem. The days selected are M,T,W,Th (note the site type makes the Default Grid more defined) 

Breakfast 2 would be used at High Schools, and Middle Schools on M,T,We,Th
Breakfast Elem would be used at Elementary schools on M,T,We,Th
and Breakfast 3 would be used at High Schools, MiddleSchools, and Elem Schools on F.

Meal Type and Menu Grid are place holders.

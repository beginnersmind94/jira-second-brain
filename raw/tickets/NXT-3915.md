# NXT-3915 — System - Sites & Users - Regions - Add toggle for 'SchoolCafé enabled' (Mock-Needed)

## Description
As an administrator, I would like to be able to easily determine which Regions are accessible in SchoolCafe, so that parents in those districts don’t accidentally register for a Region that is not using SchoolCafe by mistake.

* In the ‘Edit Region’ drawer for a region, add a toggle that allows the user to turn SchoolCafé affiliation on or off.
* Control should be a toggle instead of a checkbox
* Control label should read, ‘SchoolCafé enabled’ and not ‘SchoolCafe enabled’

Classic image:

!image-20200504-190818.png|width=1071,height=640!

## Acceptance Criteria
When a user opens the 'Edit Region' drawer for a region, the 'SchoolCafé enabled' toggle should be visible. 

When a user toggles the 'SchoolCafé enabled' switch ON and selects the 'Save' button, update the database record for the region accordingly.

When a user toggles the 'SchoolCafé enabled' switch OFF and selects the 'Save' button, update the database record for the region accordingly.

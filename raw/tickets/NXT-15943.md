# NXT-15943 — When reply or reply all, have the ability to add or remove people

## Description
When replying or reply all to a message, have the ability to add or remove people from the message. Right now it only displays those who are getting the message but no ability to add or remove.

## Acceptance Criteria
* As a user replying (or replying all) to a message, I would like the ability to add or remove people from the message. _(Right now it only displays those who are getting the message but no ability to add or remove.)_

# NXT-9730 — INV - Create Orders - Confirm Screen - Show Merge Icon

## Description
As an inventory manager I want a Merge Order prompt when creating a new order so I am able to merge my new order with an existing order if they have the same Site, Vendor, Status, Delivery Date and Delivery Location.

*Requirements:*

* Provide a merge icon if an order can be merged into an existing order
** Tooltip: “This order can be merged with an existing order.”
** Merge icon will be only displayed on orders that have already been created/confirmed and are therefore in Open status
* Orders can be merged if they have the exact same value for the following attributes:
** *Site*
** *Vendor*
** *Status*
** *Delivery Date*
** *Delivery Location*
* Open a popup when user clicks the Merge icon ([NXT-16298|https://primeroedgenext.atlassian.net/browse/NXT-16298])
* For orders already merged:
** do not show the Merge icon
** display the status as ‘Merged into Order #12345’
*** note: this is not the saved status, but only shown for display purposes
*** If not enough space, show only ‘Merged’ as the status, and provide a tooltip displaying full details “Merged into Order #12345”
** the View icon would navigate to the merged order

## Acceptance Criteria
* on pageload, scan for any mergeable orders
* Show the Merge icon for any mergeable orders
* Provide a tooltip for Merge icon
* Hide the icon for any merged orders
* Show different status for merged orders

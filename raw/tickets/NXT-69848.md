# NXT-69848 — Menu Analysis - Site Performance - Planned Accuracy Metric & Slide out

## Description
As a district manager taking a look at my site performance on workspace, I would like to be able to see how well my managers planned vs what was actually served.

!image-20260130-222557.png|width=671,alt="image-20260130-222557.png"!

## Acceptance Criteria
* On Site Performance page, provide a new column for Planned Accuracy 
** This will be after the Served Portions Column
* The calculation for Planned Accuracy is (Served/Planned)*100= Planned Accuracy
* Provide an icon to allow user to open a slideout from the column to show calculations
* When user clicks on a particular row on the slideout, update the calculation on the bottom of the slide out to reflect the numbers and show math “work”
* Allow sorting on the slideout
* Freeze Calculation at the bottom and formula at the top, also freeze header row and allow scrolling for the sites
* Freeze main page Site and Totals header
* Provide button at the bottom right for “Go back to the top” to allow users to get to the top of the sites section

# NXT-51721 — PR- Plan Summary- Prep site and receiving site items on report consider only consolidated

## Description
As a prep site, when I am pulling the Plan Summary for all the items I need to prepare for all other sites, they are currently not showing.

## Acceptance Criteria
* *ONLY Consider prep site item counts once consolidated plan is complete*
**  *Until consolidated plan is complete, consider plan counts for prep site only from plan tab and not include consolidated totals*
* *If user saves and all numbers appear on report and then they go back and delete the consolidated plan, roll back the calculations to consider totals without consolidated plan*
* *Once planned counts are added for prepared items for receiving sites, they should never appear on their report regardless if consolidated plan is completed or not*

# NXT-62716 — IM - CNDB 2025.X- Monthly Update Implementation (R&D)

## Description
USDA has now announced that the CNDB will be doing monthly updates instead of yearly. This will allow for more frequently updated information to flow to our CNDB items.

## Acceptance Criteria
* Create Monthly updates to consume CNDB data
* Ensure mechanism is available to drop file or extract to update with new CN information
* Establish documentation for how the updates are happening

# NXT-4194 — Item Management - Add Item Notification

## Description
Story

As a user, I want to be notified when my item has been successfully created and saved

Mockup

!image-20200715-132951.png|width=1470,height=261!

Workflow - when adding a food or non-food item, this success message shows up after the last step of the add item flow

Requirements

Message: ‘Item Name | Item Code' has been created successfully'

This will happen whenever leaving the ‘creation’. For Cancel and Save & Close, this message will display.

Limit the item name to a smaller amount of characters so the message shows up on a single line. Not sure what 

3000 milliseconds to display (This has been used in inspections and professional standards for reference)

## Acceptance Criteria
When a new item is created, the success message shows up on the split view and Table View and displays the item information

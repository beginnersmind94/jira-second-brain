*Requirements:*

* Subtotal the sites
* Provide totals for all sites

## Acceptance Criteria
Depending on the claim type the data for each site is shown in the required format for the claim data created for SFSP.
There is a column for each type of program participation. 
There are 10 possible columns but only 5 are possible for a claim.
The 2 possible columns for each SFSP Program are Seamless Summer, and Traditional SFSP. both are possible on a claim, but only 1 is valid for a site.
The 5 possible columns for SFSP are Lunch, Breakfast, Snack (AM), Snack (PM), and Supper. 
If a site does not participate in a program the column entries are left blank.
Each site has
Meal counts for each Program the site participates in for each meal count.
Each meal count is multiplied times the appropriate rate resulting in a claim amount.
There are subtotals per program participation by site.
There are totals per eligibility type by site

There are grand totals per column.

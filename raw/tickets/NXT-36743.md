# NXT-36743 — IM - Recipe Rework - Recipe Steps - HACCP - View CCPs

## Description
As a Nutritionist I want the HACCP options available on recipes so I can easily assign required control points for my recipe preparation.

*Requirements*

* With the required fields populated (via default or user values), the CCP can be saved [https://primeroedgenext.atlassian.net/browse/NXT-36742|https://primeroedgenext.atlassian.net/browse/NXT-36742|smart-link] 
* Insert CCPs into the recipe as steps
** Consider each added CCP a custom step
*** style the CCP steps differently
** Allow CCPs to be dragged and dropped just like Steps
** Display the CCP in label-only view unless the CCP is being edited
** Allow edit of CCPs in the same behavior as Steps are edited
** Display only the Critical Limit as the main Step description
** Provide the Corrective Action text area in the accordion expand for the CCP Step

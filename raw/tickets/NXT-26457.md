# NXT-26457 — Acc. - End of Day - Deposit Slip Task

## Description
As Agatha the Site Manager I want to see if there was a deposit slip created, and if so, how much was in there and if any variances need to be attended to.

*requirements*

* Add the Deposit Slip task and populate it.

*Suspicious Transactions Task*

* Task Name: Deposit Slip
* Summary:
** Show if a deposit slip has been created.
** Examples:
*** No deposit slip created.
*** Created: 04/27/22 03:22:12 PM
* Details:
** Show the amount for the deposit slip, and if there is a deposit slip variance.
** Example:
*** Deposited Amount: $44.33
*** Deposit Variance: ($1.50)
* Status
** If no deposit slip has been created, status = Not Started
** If a deposit slip has been made, status = Complete
* Actions
** Done / Not Done
** History

## Acceptance Criteria
* Deposit Slip task created.
* Summary populates based on whether the deposit slip has been created.
** No deposit slip, text = No deposit slip created.
** Deposit slip made, text = Created: MM/DD/YY HH:MM:SS AM/PM
* Details populates the Deposited Amount.
* Details populates the Deposit Variance.
* Status is based on whether the deposit slip has been created.
** No deposit slip, status = Not Started
** Deposit slip made, status = Complete

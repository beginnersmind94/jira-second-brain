# NXT-19092 — Income Surveys - Add New Survey - Review Step

## Description
As a district admin, I would like the ability to quickly and easily add a manual income survey on behalf of a household, so that their income data is entered into our system for reference at any time.

*Summary*

Within the ‘Add New Survey’ function, add a step to help add a new survey record for a set of students and review that the information that has been added is correct, allowing them to jump back to those sections and correct something if needed.

*Requirements*

* Display a *Review* step that displays all of the information selected in previous steps (see mock-up for details/formatting)
* Within each section, display an edit pencil that will allow the user to go back to that particular step and make changes as needed. 
** Within the Comments step card, we can just display the file name and extension of any documents the user has uploaded. (i.e. ‘paystub.pdf’) It does not need to be a clickable link. 
* In addition to the normal Cancel and Back buttons, display the following buttons at the bottom of the step:
** Pending
*** Saves this as a pending survey and returns the user to the Income Surveys grid. 
**** Success toaster: “Pending survey #\{surveyNumber} saved successfully.”
** Submit
*** Saves this as a complete ('processed') survey and takes the user to the confirmation/Summary step.

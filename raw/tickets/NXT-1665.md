# NXT-1665 — INV - Create Orders - Available Quantity for Ordering (Commodity Contract)

## Description
As a site manager I want an ‘Available Quantity for Ordering’ popup/link/etc. for commodity vendors so I can view how much product is available for me to order.

*Requirements*

* Add a ‘Available Quantity’ column for items on the Standard Ordering screen
** this is only for items which are on a *Commodity Contract* ([NXT-1839|https://primeroedgenext.atlassian.net/browse/NXT-1839])
** do not show the column if no visible items are from a commodity contract
** populate this column with the available quantity on hand value for that item
*** e.g. ‘28 cases’
** Provide a way to view the calculated details of the Available Quantity value
*** display the values for each of the following in the popup/slideout
**** Quantity on Hand
**** Committed Quantity - items committed to existing orders
**** Total Available Quantity

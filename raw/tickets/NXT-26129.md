# NXT-26129 — SPIKE - User Preference - Show/Hide District Name in Header

## Description
*As a user, I would like the ability via a user preference setting to determine whether or not the current district shows in the header of SchoolCafe.*

## Acceptance Criteria
* User shall be able to see the district name drop-down next to the profile page
* User shall be able to change the district from the drop-down if he/she has access to multiple districts
* Default district should be the first in the available drop-down list.

# NXT-5920 — Messages - Inbox - View All

## Description
*User Story*

As a PrimeroEdge user, I have an Inbox where I can view and manage any messages that have been sent to me.

*Requirements*

* Inbox is present and can be accessed either on its own page through the sidebar
** -Inbox can be accessed from anywhere in the product-(deferred)
** Any internal PE messages will appear in the Inbox
*** System Messages, Person to Person, etc.
** There is a size limit on the Inbox  (tbd)
** Messages have no expiration
* User can view any messages they have received in the Inbox
** Messages are shown with the person who sent them, subject, and a preview of the message’s body
** -Attachments can be viewed as well-(deferred)
** Clicking the message shows the full message
** -Users can access any attachments or images in this view- (separate story)
** Messages are formatted as they were when the user sent them
** User can reply, reply all, -forward-(deferred) or delete the message
*** This should hook into the compose functionality (NXT-5915)
* There are different views available to the user, all of them part of the provided template
** Inbox
** Sent
** Other available folders are not needed
* Compose is available from this screen
** Functionality covered by a separate story, but if done first, this story should include the button
* Users can search for messages

# NXT-27752 — IM - Export to Excel (Menu Item)

## Description
As a Item Manager administrator I need an option to download the data in the result grid to an excel file in order to allow me to do data analysis

## Acceptance Criteria
*Entry Point* 

* IM → Menu Items



* User Shall be able to see a button to download excel \[use standard button image/icon]
* Download icon shall be place on the field search and column & views bar
* The user shall be able to click with the mouse on the export button/icon and also navigate to it with keyboard to download data.
* -User shall be notified when download is completed-
* -User shall be able to open the excel once export is completed-

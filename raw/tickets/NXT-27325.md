# NXT-27325 — IM->Regression->Sprint_62-> Menu item attached to Variety menu item should be disabled for editing(Menu Item)

## Description
Login to SchoolCafe site
Click on IM tab
Click on Menu items option
Click on Variety Menu item add button and create one Variety menu item
Link Menu item to Variety menu item
Now go to menu item attached and do some changes with menu item

Expected Result:  User should not be allowed to do changes with menu item attached to variety menu item
Actual Result:      User is able to do modifications with menu item attached to variety menu item

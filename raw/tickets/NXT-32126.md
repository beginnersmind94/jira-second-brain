# NXT-32126 — sFTP Import - Automate sFTP Resource Creation on New District - Backend Only

## Description
Automate the creation of the Azure resources necessary for sFTP to be fully functional for a district

## Acceptance Criteria
* The creation of the Azure resources that are necessary for a new district to be able to securely FTP files to SchoolCafe shall be automated
* The automation may be triggered and executed as part of the “new district automation” code that the System team has built.
* The district shall be provided with the information necessary to connect to their FTP site and perform transfers.
** Host
** SSH Keys
** *Other*: Jignesh to identify
* Jignesh to identify which data, if any, needs to be collected by the user in order to set this up.  
** A UI will be built to collect this data, if needed.
* We need to check with System team on where this can be triggered and where the needed information can be provided to the district’s I/T user.
** The System team UI is not in scope under this story.

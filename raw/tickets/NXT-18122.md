# NXT-18122 — INV - Manage Orders - Order Detail - History tab - Transaction Details

## Description
As an Inventory user I want a transaction details so I can view the specific Order Details for a historical order transaction.

*Requirements*

* On the History tab of the Order Detail page ([NXT-7989|https://primeroedgenext.atlassian.net/browse/NXT-7989]), once a specific transaction is selected, open the Order Detail page of the historical order transaction
* Only show view icon for actual order adjustments:
** Order Adjusted/Updated 
*** quantities changed
*** Item added
*** Delivery Date changed
*** Delivery Location changed
*** Comments entered/changed
** *Remove* View icon on History tab for transactions that are not adjustments
*** Order Merged
**** Merged into another order
**** another order merged into this order
*** Order Approved
*** Order Rejected
*** Order Resubmitted
*** Order Closed
*** Order Cancelled
*** Receipt Created
* Load the Order Detail of that transaction in view-only mode
* Replace the banner with an informational banner for the history:
** “This transaction was changed on 8/23/2021 at 1:15 P.M. by Justin Miller.”
*** *Add time to the banner
* Display the changed values in red text
** For any fields which were changed, display in red text
** Compare this transaction with the immediately prior transaction
** Attributes which could change:
*** Delivery Location
*** Status
*** Delivery Date
*** Number of Items
*** Comments
*** Items
**** Order Qty: show a tooltip with previous order quantity
**** Deleted items: show with a strikethrough text
**** Added items: show in green/blue(?) bold text to indicate new item
*** Item Comments
**** show a tooltip with previous comments
**** change icon color ?
* Remove the action buttons
* Change the ‘Back’ button to ‘Back to History' button

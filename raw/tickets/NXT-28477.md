# NXT-28477 — INV - Orders - Cancel Order - Update Available QOH for Commodity Contract Items

## Description
As a Site Manager I want the system to return available quantity when a Site cancels an order so the quantity is available for other sites to order.

*Requirements*

* Create trigger when cancelling an order that has Commodity Contract items
* Upon cancellation, return reserved quantities to the Commodity Contract item quantities

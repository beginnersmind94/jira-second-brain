# NXT-21786 — IM- Single Item Recipe- Default to 1 as # of servings (UI Only)

## Description
As a user, when I have a single item recipe, I should only have 1 as the # of servings.

Currently, we are allowing the user to select # of servings for a single item recipe, it needs to be defaulted to 1 but allow them to change if applicable.

!image-20211110-175045.png|width=1076,height=189!

## Acceptance Criteria
Set default # of servings on single item recipe to 1 but allow to be edited if needed. 

This change is applicable when user creates a single item recipe

For multiple item recipes pick the first one on the list

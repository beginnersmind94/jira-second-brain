# NXT-37844 — IM - Recipes - Create Recipe - Updated flow (Summary)

## Description
As a recipe menu planner I want an updated Create Recipe flow similar to Items so I can manage all my serving related info in the consolidated Units card.

*Requirements*

* Redesign the initial Create Recipe flow to accommodated the completed Units card (from Items)
** Update the initial general Recipe Info card to provide only the remaining general info attributes
*** Recipe Description
*** Tags
* Require the same attributes for continuing with Recipe creation
** Description (1st card)
*** Require Recipe Description as only mandatory field in Summary
*** Hold recipe in temporary session until Serving Size is added to Units
* Continue to Units card for Serving Size entry ([https://primeroedgenext.atlassian.net/browse/NXT-37898|https://primeroedgenext.atlassian.net/browse/NXT-37898|smart-link] )
* Accommodate the dynamic Image card on Recipes as well
** Should have adequate space to stay expanded most of the time

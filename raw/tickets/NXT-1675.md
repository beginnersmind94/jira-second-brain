# NXT-1675 — INV - Withdrawals - Create Withdrawal - Add Items (Step 2) - Standard

## Description
As a Site Manager I want an Add Items tool so I can search and add items to my inventory withdrawal.

This is the second step in the Withdrawal process, after user navigates from the first step ([NXT-1740|https://primeroedgenext.atlassian.net/browse/NXT-1740])

*Note:* this is the ‘Standard’ version of Step 2 based on the ‘Withdrawal Type’ selection in Step 1

*Requirements*

* This is the second step of the Inventory Withdrawal stepper tool
** title this step ‘Items’
** display the stepper ‘Previous’ button to navigate back to step 1
* There are 2 main sections on this screen:
** Add Item control (this story)
*** the tool which is used to find and select items to add them to the list
** Item list ([NXT-20796|https://primeroedgenext.atlassian.net/browse/NXT-20796])
*** the list that contains the items which have been added to the list
* Add Item section
** Rather than providing a dedicated screen for searching items, we want to use a lightweight control so we can keep the user on the same screen for both adding items and viewing the added items
** This will be the upper section on the page
** Display the following
*** Template dropdown
**** this will contain all the Withdrawal Templates which are configured for the site 
**** this should be multi-select checkbox (like Create Orders)
**** -Disable the other fields if user selects at least one template-
*** *-Vertical Divider-*
**** -The Template dropdown will work independently of the main Add Item dropdowns.- 
**** -The Add button will enable if the user selects at least one template-
*** Item Category dropdown
**** default to ‘All’
**** allow single selection
**** selections from this dropdown will filter the results in the main Item dropdown
*** Item dropdown (may be Smart Search instead)
**** display Item # and Name 
**** this will list all items with some quantity on hand (greater than zero)
**** this will be searchable 
**** single selection
*** Quantity On Hand (label)
**** shows the current value for the item’s perpetual quantity
*** Quantity textbox
**** user must type some number
**** do not allow user to type more than QOH
*** Add button
**** enabled once a single template is selected *OR…*
**** enabled once an item is selected and there is a value for the Quantity field
** Once an item is added, it will load in the Items List ([NXT-20796|https://primeroedgenext.atlassian.net/browse/NXT-20796])
*** Reset the selections and disable the Add button
*** Remove that item from the Add Item list

# NXT-20659 — Production- Production Plan- Variety Menu Item

## Description
As a Production Manager, I would like a way to pick which items from my Variety Menu Item I am going to plan for.

## Acceptance Criteria
Display Variety Menu Item with a drop down checkbox selection to be able to choose which menu items they would like to actually plan for.
Once saved, Display menu items in a stepper or tree grid under the variety menu item to indicate where these items are coming from. All items stay within the same category that belongs to the variety menu item.
Display all menu item information and give box for each menu item to enter in planned counts.
Allow items to be unchecked and removed from being planned.

When user “turns off” one of the menu items in the variety, keep the counts hidden in the boxes until they save, once saved we will not store the counts. 

The below pointers will be considered as part of story  [https://primeroedgenext.atlassian.net/browse/NXT-20660|https://primeroedgenext.atlassian.net/browse/NXT-20660|smart-link] and get verified over there.

*You are able to make changes to the plan until “record Complete” status*

*If carryover is added to a future day that has a menu item that is turned off, turn on the menu item in the variety to account for the carryover. Total Planned count should include the carryover quantity.* 

*If the toggle is turned on by the system because of a carryover, do not allow toggle to be turned off.*

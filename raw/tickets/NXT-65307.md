# NXT-65307 — ACC - Attendance Feature - End of Day Task

## Description
With attendance entry

When Daily Attendance is Active

ONLY for SNP

Add a new end of day task with a text box that allows entry of selected date/site’s daily attendance count.

Show text box that pulls attendance count for that site and selected date.

Task Name: Attendance Count

Summary: Count Entered / Count Not Entered

Detailed: Count: 12345

Status: Complete if a number exists for site + date combo.

Action: Button text: Record → they can see/update the attendance count for that site+date.

## Acceptance Criteria
* End of Day task is available if:
** Attendance entry method is set to Daily
** At least 1 site is configured for SNP
** Task is toggled off by default
* Other columns:
** Summary → Count: 123456 / Not Recorded
** Detailed → Leave blank
** Status:
*** Not Started if no count has been entered
*** Complete if any count, including 0, has been entered
** Actions:
*** Record button
*** User can record/edit the attendance count for the site+date
* Task count increases if the task is turned on for EOD and FOH Dashboard

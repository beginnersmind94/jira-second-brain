# NXT-5240 — Direct Certification - Matched - List & Search

## Description
*User Story*

As a district admin I want an easy way of identifying and notifying all direct cert (DC) matches that have happened.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=133668%3A0&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=133668%3A0&scaling=scale-down-width|smart-link] 

*requirements*

* Add a Matched page under the Direct Certification menus. Add a link to the left sub-nav as well.
* This page will allow users to see all matched students, and information on their matching.
* User needs some basic filter and search options:
** Academic Year (AY)
*** Default: current AY, no All option.
** Search by Student Details
*** First Name, Last Name, ID #, Status (Active/Inactive)
** Search by File Details
*** Import Date, Process Date, Approval Date
**** Date Pickers for all, date range possible.
*** Case Number
**** This searches for a particular data field in the DC File data, a case number field.
* Show the information in a grid with the following details. Standard grid filtering applies.
** ID #
** First Name
** Last Name
** Site
** Grade
** File #
** Type
*** Which approval type were they issued approval from? SNAP, TANF, etc.
** Case Number
** Approval Date
** Process Date
** Import Date
** Match Method
*** System
*** Extensions
*** Manual
** Notification Date

## Acceptance Criteria
User can view matched students, search and filter accordingly.

# NXT-45473 — IM - Menu Items - Remove Custom Menu Items

## Description
As a Menu Planner I want the Menu Items to only support Items or Recipes so I don’t accidentally create empty menu items.

## Acceptance Criteria
* *R&D:* research production data for presence of existing Custom Menu Items
** Run a query on prod data to see how many CMIs there are 
*** if CMIs have 1 or more ingredients, consider migration
*** otherwise, consider the CMI an invalid entry and do not migrate
** *IF* migration is required, convert CMI into Recipe with Menu Item
* Remove or hide the CMI functionality
** this may be reused in the future as an enhancement for quick Menu Planning
** Ensure CMIs are removed from the back end so they do not interfere with recipe ingredient counter or inactivation requirements
* Remove from +New button on Menu Items
* Remove as a Menu Item Type

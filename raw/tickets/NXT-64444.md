# NXT-64444 — SC - Bulk Import USDA Branded Foods items Via Global Catalog (UI Integration)

## Description
_(none)_

## Acceptance Criteria
Navigation: IM- GC - Item Catalog

Default source - USDA Branded Foods

User should be able to search an item based on Item Description, GTIN, Manufacturer, Brand and Item Category

The user should be able to select multiple items and click on Import Button and be able to import the item to local data source 

Once imported, the user should be able to view the details

The mapping of the fields in UI will based on reference sheet attached

The nutrition details shall be shown based on reference sheet attached

Pack Size - Create a Default Entry as 1 Case along with weight provided

Serving Size - Capture Parsed serving measure and populate along with Unit weight

Pack Size, Meal Credits and Allergen - Out of Scope, need not be created

The user can select master check box and be able to move all the data to import

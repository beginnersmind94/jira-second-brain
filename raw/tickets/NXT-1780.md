# NXT-1780 — IM - Valuation Groups

## Description
As a nutrition Dept. user I need to a manage the valuation groups that are going to  be associated to items in order to properly obtain inventory valuation by groups.

!image-20211209-104941.png|width=961,height=554!

!image-20211209-105018.png|width=961,height=533!

## Acceptance Criteria
Story is done when:

Under Configuration option of Item Management I can select Valuation Groups

* When selection Valuation 
* A  Serving Measure configuration option is added  under Item configuration  area
* Option Item Configuration Configuration
* Data source as dropdown box default to local
* Configuration parameters should list the field name in which the configuration is been added. E.g  Valuation Group

*Actions*

Add button should be on the top right corner

An Apply button should be visible  next to dropdown box at the right of the screen

 

* Clicking ADD 
** Add Local Data Source where the Valuation Group is going be apply to (drop down box) This a mandatory field
** Add a Valuation Group  ( free text box ) This a mandatory field
** -Add a Valuation Group Code  (free text box) This a mandatory field-
** Add a Valuation Method (drop down box) This a mandatory field
*** Options
**** FIFO (FIRST-IN-FIRST-OUT)
**** LIFO (LAST-IN-FIRST-OUT)
**** SP (STANDARD PRICE)
**** MAP (MOVING AVERAGE PRICE)
**** LRP  (LAST RECEIVED PRICE)
**** RPV (REPPLACEMENT VALUE)
** -Set a valuation group as default  for physical inventory-
**  Added Valuation Group information is displayed in the result grid

*Table Filters*

* User should be able to search by selected fields  (AND condition)
* User should be able to sort by selected fields
* Display Toggle bottom “active / inactive” items. by default set to active items

*Edit*

User should be able to edit:

* Valuation Group field
* Valuation Method
* -Default in Inventory-

*Deactivate*

* User should be able to Deactivate a Valuation Group ONLY if it is not used (item/Recipe)
* If  valuation group group is in use show  a tool tip  “Configuration is in use by one or more transactions”

*General Rules:*

Only user created items can be edited / Deactivated

# NXT-22765 — MP > Site Configuration edit Sites and Menu lines (UI only)

## Description
As a user in a district with a large number of sites, I need a more simplified method of setting up my site configuration. With 80 sites, it will take an entire day to set up the sites with the current format. I need this to be easier and faster, with bulk settings to accommodate me.

## Acceptance Criteria
When saving on edit site configuration page, there must be at least 1 site selected for validation to save.

Menu lines do not need to be configured at the time of validation. 

Message: Please select at least 1 site. 

User can diselect already configured sites to a site group or select additional sites

Configured sites for another site group will appear in drop down with tool tip to indicate what site group they belong to ( User can still select this site if needed and override current site group selection)

Hover view on the count  for list of selected sites only

For multiple site configuration :  Click Edit site configuration button

For single site configuration: existing functionality works as is.

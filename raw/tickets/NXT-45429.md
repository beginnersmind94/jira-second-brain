# NXT-45429 — Imports - Long Running Processes - Show All Districts

## Description
As a Customer Support User, I would like to see all the districts at once so that I can search for long running processes.

*Notes:*

* The Import Monitor is normally for single district
* In this story, we are going to enable seeing multiple districts at once for Customer Support users

*Requirements:*

* This functionality is for the Import Monitor tab on Imports & Exports
* Backend needs to write an APIU that is the same as what the grid uses, but fetches all districts instead of just one
* 
* On Import Monitor, do the following:
* Add these fields to the top of the screen:
* View All
** Toggle button
** Default false
** Visible only to Customer Support role
** If click yes:
*** This will change the dataset used to load the grid
*** So fetch the All Districts dataset and load the grid with that data
** If click No:
*** This will change the dataset used to load the grid
*** So fetch the By District ID dataset and load the grid with that data
* 
* Everything else will remain the same

## Acceptance Criteria
* Customer Support role user can view the button and click on it
* Clicking the button shows all districts or only one depending upon selection
* Non customer support role users cannot see the button and cannot see all districts data
* All other functionality remains the same

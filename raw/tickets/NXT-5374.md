# NXT-5374 — Deposits - Deposit Details - Comments

## Description
*User Story*

As a user if a deposit slip is modified or the amount expected by the system is different than the amount entered a comment is required.

*Mock-up*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5834%3A730&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5834%3A730&scaling=min-zoom]

*Requirement*

* The user must enter a comment if the deposit is changed.
* The user must enter a comment if the Deposit total is different than the Expected Deposit total
* The users name and a time/date are recorded with the comment
* Required if Checks are Added, Edited, or Deleted.
* Comments entered in Checks also show in the generic comment section. (removed)

## Acceptance Criteria
The user must enter a comment if a previously saved deposit is changed.
The user must enter a comment if the Deposit total is different than the Actual System total
The users name and a time/date are recorded with the comment
The user must enter a comment if a Check is Added, Edited, or Deleted.
Comments from checks show in the generic comment section. (Removed)

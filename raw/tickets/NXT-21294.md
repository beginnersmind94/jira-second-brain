# NXT-21294 — IM - Add Search by Manufacturer

## Description
As a Nutrition department user I want to be able search for items on Global Catalog by Manufacturer.

## Acceptance Criteria
This story is done when:

*  I have an option to search for item on Global catalog by manufacturer and Brand
* When user select Manufacturer or Brand the list of Manufacturer  associated brands should be populated in the Brand field and vice versa is a Brand is selected the manufacture field should be automatically populated 
* The Fields are smart text box
* When I select a manufacturer or brand on the search section the result grid displays only items from the selected manufacturer or brand

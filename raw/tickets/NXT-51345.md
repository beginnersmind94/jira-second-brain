# NXT-51345 — IM- Contribution Info- Items, Recipes and Menu items Component Name Change

## Description
Per USDA new requirements, Beans/Peas (Legumes) will now be renamed to Beans, Peas & Lentils.

## Acceptance Criteria
* Change Name of the Subcomponent of Veg - Beans/Peas (Legumes) to Beans, Peas & Lentils
** This should effect Items, Recipes and Menu items Contribution Slide out on Nutrition Info Tab
** This should effect the display of the Contribution Info Card

# NXT-50953 — Exports – View History

## Description
As a SchoolCafé admin, I would like to be able to View the audit history of File export configuration changes so that I can see who made a change and when.

*Note:*

* Do the same as you did for File and API View History

*Requirements*:

* For Add New and Edit Export Configurations, do the following:
* When you save the configuration on the review screen, do the following:
* When doing Add New:
* Save all the label / value pairs from the config screens to the history table
* Be sure all fields for all import types are included
* 
* When doing Edit:
* Record in the same format as Add New anything that was added, edited or removed
* Do not record anything that was not modified
* 
* When a user deletes a configuration, record the deletion in the history log as well



* For Exports tab’s grid, do the following:
* In the ellipsis, add a button to the actions:
** Title: View History
** On click, open the History drawer
* 
* Create a new drawer
** Title: Export History
** Size: same as in other system screens
** This is a read-only screen, no edit
* Do the same actions as you do for File View History

## Acceptance Criteria
* All additions, edits and deletes are recorded in the history log
* When a user deletes a configuration, it is recorded in the history log
* User can view the changes in the view log

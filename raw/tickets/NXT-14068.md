# NXT-14068 — Applications - Pending - Selection when multiple students are found

## Description
As a district admin I want to make an informed choice when multiple pending students are identified.

*mockup*

tbd

*requirements*

When we have a pending student with more than 1 potential match available:

* User needs a visual prompt to help them understand that there’s a choice to be made for that particular match.
** The green checkmark can likely be updated to a caution symbol.
* User needs a way to select from the options and confirm their choice.
** The checkbox for processing should not be available until after the user has made a choice.

## Acceptance Criteria
Grid symbol updated to show the user they need to make a decision here.

User can select from the options available in the comparison window and confirm the choice.

‘Matched Status’ should update to the green checkmark after a selection is made.

Checkbox for processing is only available after a choice has been made.

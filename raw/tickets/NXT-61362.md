# NXT-61362 — AI Onboarding - Digital Recipe - File Analysis Section

## Description
As a customer support person, I would like to use AI to digitize a recipe document so that I can speed up onboarding

*Notes:*

* This screen’s content is broken out into various stories
* This story picks up on showing the results of the file analysis
** This content can be shown right after the user uploads a file or if they select to view a previous file

*User Flow:*

* User browses to Item Management and then Recipes
* System checks if the district has enabled AI and shows the AI button if they have
* User clicks on the AI button and it opens the AI Recipe drawer
* User ensures the New chip is selected
* User selects what type of file they are uploading
* User selects a file to upload and clicks upload
* AI engine analyzes the file and saves the results
* This section is shown with the results and allows the user to click through the recipes one at a time

*Requirements:*

* On the Report AI screen, do the following:
* 
* Add a section on the left
** Put below Upload
** Title: File Analysis
** Make expandable / contractible 
* Fields:
* Format 
** Type: read only
** Value: Name of Format selected from upload
* Status 
** Type: read only
** Value: Status of the file process, as follows:
*** New if haven’t completed any recipes
*** In Progress if have completed any recipes, but not all
*** Complete if have completed or discarded status for all recipes
** Use green color and check mark icon if completed
* File Information
** Show the name of the file that was uploaded
** Allow the user to click on the name as a hyperlink to view it
** Show the user who uploaded the file and when they did it
* Recipes Found
** Show read only count of recipes found the AI engine
* Recipe Chips
** No title
** Shows a list of chips 
** Total count should match the recipes found
** Chip label is the number of the recipe (1,2,3, etc.)
** On click, show the recipe information in the edit recipe (covered in another story
** Color:
*** Color is based upon status of the recipe represented by the chip
*** Purple for the one loaded on screen
*** Green if recipe status is complete
*** Yellow if recipe status is In Progress
*** Red if discarded
*** Grey if status is new
* Progress Bar
** Show a % progress bar like what in bolt
** Note: for purposes of % complete, count both Complete and Discarded as completed
** 
* 
* *Usage:*
* Show this section when the user uploads a file and the AI engine has analyzed it
* Also show this section when the user loads a previous file
* Be default, when loading the section the first time, make sure it is expanded and the Upload section is collapsed

# NXT-9782 — Data Exchange - File Import Setup: SAVE Configuration

## Description
As a PrimeroEdge user, I would like to have an intuitive screen that allows me to verify all of the fields in PrimeroEdge are rightly associated with the fields in my first uploaded file, so that I can confidently save the import configurations for an data import later.

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5269%3A82698&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5269%3A82698&scaling=scale-down-width|smart-link] 

*Requirements:*

* When the user has completed the “*REVIEW*” of mapped fields and Current File Name is review system should enable SAVE button for user.
* When user clicks on *SAVE* button, a PROMPT is displayed for user that they have successfully saved the configuration. 
* System should preserve all the File Import configurations
* System should display 2 buttons within Import setup success prompt.
** *NEW IMPORT FILE SETUP:*
*** When user clicks on this button, they should be re-directed to new “Pre-upload” Page for a new file import setup (same activity when user clicks on “NEW FILE IMPORT SETUP” button from File Import Dashboard”
** *FILE IMPORT DASHBOARD:*
*** When user clicks on this button, they should be redirected to “File Import Dashboard” Page
* User should not be able to cancel from this prompt

## Acceptance Criteria
* Verify that when user clicks on "SAVE" button brings up a prompt for user

* Verify that system saves File Import Setup configuration with the file name displayed in the review screen

* Verify that PROMPT has 2 options for user to navigate to:
1. NEW IMPORT FILE SETUP:
2. FILE IMPORT DASHBOARD:

* Verify when user clicks on NEW IMPORT FILE SETUP button, Pre-upload screen is loaded for new file import setup

* Verify when user clicks on FILE IMPORT DASHBOARD button, File imports dashboard page is loaded

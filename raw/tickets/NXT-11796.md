# NXT-11796 — API - Update Recipe Ingredients upon recipe steps save 

## Description
As an item management user, I need to be able to add an item or multiple items and their primary and secondary measures for each step.

Requirements

* User will be able to search for items and click 'add' and then add a primary and secondary quantity and unit of measure
* An item does not need to be assigned to a step
* Item Search will be shown in the item step screen

## Acceptance Criteria
I need to be able to add items and their quantities to a recipe step

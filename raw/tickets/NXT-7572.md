# NXT-7572 — INV - Physical Inventory - Main Screen - Item Validations

## Description
As an inventory user I want enhanced item count validations and alerts so that I can minimize my discrepancies and avoid issues.

Connected to [https://primeroedgenext.atlassian.net/browse/NXT-7573|https://primeroedgenext.atlassian.net/browse/NXT-7573|smart-link] as some of these thresholds or alerts will need to be configured.

Alerts:

* Discrepancy Alerts
** district (or site-specific) configurable thresholds for discrepancies
** based on quantity
*** ex: if discrepancy exceeds 25% of perpetual QOH, alert the user
** based on dollar amount/value
*** percentage
**** ex: if discrepancy amount exceeds 25% of current QOH value
*** value
**** ex: if discrepancy amount exceeds $100, alert the user
* Validations on Whole Unit vs. Broken Unit (extension of above)
** prevent accidental entries into the wrong field
*** ex: perpetual is 20 cases, but user enters 0 cases and 20 broken units
* Zero quantity entries
** if user enters zero for an item that exists in perpetual, alert the user to search for that item

# NXT-35978 — Item Details Export

## Description
As a SchoolCafé admin, I want to be able to export the real data from my database that is imported from Item Details Import in the exact same format so that I can compare what I think I imported with what I actually imported to analyze what might be missing.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Exports
* Click on + New on the right
* Select Item Details in the Export Type
* Hit Next
* Verify Source and Target fields and hit Next
* Review the last tab and make any changes
* Hit Save

 

*Notes:*

* The way it works should be like Student export, same flow

 

*Requirements*:

* The export file format must be exactly the same as the import file format
** Use the existing import file format as the template
* Add a value to the Export Type drop down list
** Label: Item Details
* 
* If select Item Details:
** Show the Field Mapping - Item Details tab when click next
* 
* Field Mapping - Item Details tab
** Section title: Field Mapping - Item Details tab
** SOURCE - SchoolCafé Fields: exact same ones as the item Details Import file
** Target - SchoolCafé Fields: exact same ones as the item Details Import file
** All the same functionality as the Student Export
* 
* Review - Item Details tab
** Section title: Review - Item Details tab
** All the same functionality as the Student Export
** If possible, can we default name the file ItemDetailsExport-nnnn (where n is number you generate)
* 
* Permissions:
** No changes necessary

## Acceptance Criteria
Navigation: item Catalog - Provider - Cybersoft Catalog

Changes Required:

Search Parameters: GTIN, Item Description, Manufacturer, Brand, Vendor and State. When more than one parameter is used, it should be used as AND condition while matching

User selects the Data Provider dropdown and select Cybersoft Catalog
User should be able to enter either of the filed such as a single GTIN, Item Description, Manufacturer (text based search should fine for MVP) 
The Brand, Manufacturer list should be based on Master Data Tables (Grouping Name)

The search results should be listed in the Grid as shown in the prototype (Item, Manufacturer, Brand, GTIN)

Each item will have Hyper link on Item Name and Eye Icon at the end to view the record

GTIN - Multiple GTINs searched in Comma separated values:

In case of multiple GTINs entered in text box, The system should use the multiple GTIN MATCH API and render the details in the grid for matching results

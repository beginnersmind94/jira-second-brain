# NXT-55598 — SC- Pending Updates - Download Exceptions - API

## Description
_(none)_

## Acceptance Criteria
Currently, there is no exception mechanism exists on data upload 

API - Backend Story

As a change, the system should capture the exceptions as follows:

# When target ID is missing in data source - capture the status as Failed with a message as “Item Not found at data source”
# When Images are missing against an item - capture the status as Completed with a message “No Image update available”
# When datatype was wrong or if its not a link- capture the status as Failed with a message as “Invalid Image link”  

The user should be able to download the file in excel format for their review

Rules:

Only latest download option is available for end users

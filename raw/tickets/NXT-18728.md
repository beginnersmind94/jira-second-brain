# NXT-18728 — IM - Upload Picture To Recipe

## Description
As a authorized user I  need a function to allow me to:

* To add one or more pictures to a recipe when creating or editing a recipe.
* It needs to available in list and and split view.
* If more than one picture is loaded then the pictures navigation should be like cycling thru a carrousel
* The screen should also have a option to offer to display the picture or hide the picture.
* The system should have an option to designate a picture as default or favorite for display
* When adding one or more pictures to a recipe the system should:
**  Prompt the user for the location where the picture(s) is/are stored
** The user should be able to drag one or more pictures to picture area
* When adding one or more pictures to a recipe the system should allow for multiple selection

## Acceptance Criteria
* Picture (s) should be loaded and added to the recipe as indicated on the Description section of this story
* *This functionality should be available in list and split view*

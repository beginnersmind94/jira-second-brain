# NXT-33154 — Photo Import - Implement Retry Logic for Failed Uploads

## Description
After the Photo Import is completed, upload photos that have failed.
Upload photos that have failed due to HTTP Status-500 (Internal Server Error) and 504 (Server time-out).
Upload it by reducing the Photo Import Upload Chunk Size (in bytes) and Photo Import Upload Thread Quantity to half.
So for eg, if the Photo Import Upload Chunk Size (in bytes) is set to 50 MB then reduce it to 25 MB, and the same for Photo Import Upload Thread Quantity.

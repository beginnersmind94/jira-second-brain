# NXT-2965 — Organization - Sites - Global Site Database

## Description
As a PrimeroEdge organization I want to manage/maintain a global site database for school districts across the nation so that I can alleviate much of the initial startup process.

Utilize the ELSI data that has been published regarding the public information of school districts and sites to create an initial database of sites. When a new customer starts, we will use this initial list to pre-populate all of the sites and allow the customer to simply manage certain additional attributes and customizations. 

This significantly increases the speed of onboarding a new customer and avoids the need for a manual import of sites.

## Acceptance Criteria
1. Store site information for school districts to be used for onboarding purposes
2. Update global site database if new sites are created locally but not stored already in the GSD

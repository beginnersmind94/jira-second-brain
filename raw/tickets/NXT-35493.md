# NXT-35493 — (Part 1) New District Setup - Add contact fields to Add District page

## Description
As an SchoolCafé Support user, I need to designate the contact person for a new district that we have created, so that this person will automatically get a welcome email and be able to get started with onboarding rather than waiting for us to manually grant them access in KeyCloak.

*Requirements:*

* In System → Add District, add 3 new contact fields:
** Contact Person First Name
** Contact Person Last Name
** Contact Email
* When that info is saved along with the new district details, do the following:
** Add the district
** Automatically create a new user for the district with those basic details
** Automatically assign the user the Onboarding role
** Send them the ‘welcome’ email with instructions on how they can sign in and get started (email template can be polished in the part 2 story).

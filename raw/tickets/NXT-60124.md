# NXT-60124 — IM- Item Onboarding- Update Menu Item Name (Step 1)

## Description
As a district admin using the item onboarding tool, I need a step for me to confirm my menu item selections as well as add my menu item name and marketing information

!image-20250317-203344.png|width=1559,height=926,alt="image-20250317-203344.png"!

## Acceptance Criteria
* Provide an edit mode for “Update Menu Item Name” that includes some of the columns from the review
** Item Description (View Only)
** Serving Size (View Only)
** Is this Direct Menu Item? (Editable)
** Menu Item Name (Editable and Mandatory)
** Marketing Name and Description (Editable)
*** Provide a text box for Name and a comment box for Description with tool tip “Enter Marketing Description”
* Items that are marked as not a menu item do not have any fields available to edit
* Provide a way for users to use ingredient Name as their Menu Item Name
** This must be available to do one at a time or a bulk process 
* If user clicks on Use Ingredient Name button, provide a pop up confirmation “Using this button will update all empty Menu Item Names with the Item Description? (Confirm/Cancel)
** Only populate if field is empty
* If user clicks on the arrow next to the item name, autofill the item description in the menu item name textbox
* Menu Item name must be unique
* Provide an alert next to item # on a row if user goes through the row and does not enter Menu Item Name
** Tool tip on alert “Menu Item Name Missing”
* User is allowed to save this page at any point in time whether Menu item Name is entered or not, highlight textboxes that have not been entered
* Provide a Confirmation “Some Menu Item Names have not been entered, this will prevent additional configuration, Proceed?”

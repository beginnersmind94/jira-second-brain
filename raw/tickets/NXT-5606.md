*Requirements*

As an Item Management Super User I should have access to the search capability by missing information from the Item Management Landing page:

# General Info
# Nutrient Info
# Contribution Info
# Allergen Info
# Product Info

Default to Active and Food item

Add filter for ‘Missing Information' so that when these options are selected, they have an active ‘missing nutrient filter’ for example. This will also include the ‘apply’ filter button on the split view and table view.

## Acceptance Criteria
The user should be redirected to the Item Detailed/Split View where the search results are viewable and editable by the chosen missing information search criteria.

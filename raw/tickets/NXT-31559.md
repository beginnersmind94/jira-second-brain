# NXT-31559 — Add Logic to Edge Reconnection

## Description
As a cashier, I want the system to be able to be able to reconnect to an Edge device automatically without bogging down the system so that I can reconnect and use the Edge device for tray detection and learning.

*Application*: POS

*Background*

This comes from a production bug at NC ISD. The terminal, when it loses its connection to the Edge device, is overwhelming system resources trying to reestablish connection.  We need to put in some business logic to prevent this.  There needs to be some kind of timer or counter that slows it down so that the system is not overwhelmed trying to reconnect.

# NXT-70186 — SC - MT - Multiple Parents

## Description
h3. User story

* As a central user, when I create a custom role, I want to select multiple districts so that I can access them as a custom admin.

h3. Context

The issue involves updating district access for a central user. A new custom role will allow a state to act as a custom admin, enabling access to multiple districts.

h3. Acceptance criteria

* A new custom role can be created for a state.
* There is an option to select districts that this role will manage.
* A dropdown shows all available districts in the state for selection.
* Selected districts will grant custom admin access.

# NXT-5291 — [TaggedDashboard] - Dashboard (Insights) - Eligibility - Eligibility Summary Widget

## Description
As a central office administrator I want to see the eligibility summary breakdown of count and percentage of my students so that I can have a clear picture of my free, reduced, and paid population.

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=429018%3A0&scaling=scale-down-width&page-id=84%3A0|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=429018%3A0&scaling=scale-down-width&page-id=84%3A0|smart-link]

## Acceptance Criteria
User has an area of the Dashboard showing summary details for the active student population.
Counts are broken down by eligibility (free/reduced/paid)
Counts can be further broken down by reason for each eligibility (income, default, dc, etc.)

User can see what the Economically Disadvantaged % for the district is.

User can select this information by site and date.

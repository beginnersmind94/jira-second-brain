# NXT-51121 — POS - New Dashboard Layout

## Description
As Matt the Product Manager I want to offer a more friendly and welcoming dashboard to the POS to help our sales. 

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=6164%3A56305&node-id=6235-51754&viewport=1029%2C601%2C0.25&t=vedxHLyCY3wlpbUY-8&scaling=scale-down-width&hide-ui=1|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=6164%3A56305&node-id=6235-51754&viewport=1029%2C601%2C0.25&t=vedxHLyCY3wlpbUY-8&scaling=scale-down-width&hide-ui=1|smart-link] 

*requirements*

* Begin redesign process to match the mockup
** Keep screen sizes in mind - *1024x768* for tablets (less, really, since we lose the top section), 720p, 1080p
* User sees the site name they are setup to serve at.
* User is greeted differently depending on the time of day:
** hours < 12 - Good Morning, {{FirstName}}!
** hours >= 12 and hours <= 17 - Good Afternoon, {{FirstName}}!
** hours >=17 and hours <= 24 - Good Evening, {{FirstName}}!
** Icon changes depending on greeting.
* Start Service buttons:
** Open Service
*** Switch to Continue if a detailed session is already open
** Summary Sales
** Bulk Sales
** Payments
* Report buttons:
** Transactions
** Item Sale Counts
* Hamburger menu:
** Show the SiteName and Terminal Name
*** Site: {{SiteName}} - ex: Site: Creekside Park Junior High
*** Terminal: {{TerminalName}} - ex: Terminal: CSPJH-POS1
** Order for buttons:
*** Open Service, Summary Sales, Bulk Sales, Payments, Transactions, Item Sale Counts, Device Information, Switch Sites, Sign Out

## Acceptance Criteria
* User has a new dashboard experience.
* User can see the Site Name the terminal is configured for.
* User is greeted differently depending on the time of day.
* User has access to main screen buttons:
** Start Service: Open/Continue Service, Summary Sales, Bulk Sales, Payments
** Reports: Transactions, Item Sale Counts
* User can see site name and terminal name in the side menu.
* User can access all buttons from the side menu:
** Order: Open Service, Summary Sales, Bulk Sales, Payments, Transactions, Item Sale Counts, Device Information, Switch Sites, Sign Out
* Existing functionality for all screens remains intact - we’re only re-designing the main dashboard after login.

# NXT-67708 — WS - Benchmark - Configuration -Show prior year benchmark values for each site and each KPI (PE & SC)

## Description
This story is for both PE and SC:
When the Academic Year 2025-2026 is selected, show the benchmark values of the prior year (2024-2025), and if the Academic Year 2024-2025 is selected, then show the benchmark values from the previous year (2023-2024).
If a benchmark value provided by a user for a KPI and for a site is less than the prior year’s value, then show the prior year’s benchmark value in the color as per the Bolt UI, and if it is greater tha,n show it in red.
For a given year, if there are no benchmark values available, then dont show the Prior Year Benchmark values.
UI should be as per the Bolt UI.

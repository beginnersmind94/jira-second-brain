# NXT-12287 — Platform - AI/ML - Student Account Facial Registration

## Description
*Persona*: Student/Parent

*Access level:* School District/Site

*Description*:

As a parent, I should be able to use the Perseus application to register my son/daughter’s profile activated for Facial recognition.

## Acceptance Criteria
1. Parent should be able to access sign-up page for facial recognition from students/parents portal. 
2. User should be able to enable facial recognition by accepting required disclosures & terms.
3. Parents should be able to confirm the student account recognition 
4. User should be able to tag the profile to enable facial recognition for specific services (for now, only Payments, Identification services are enabled)
5. Email confirmation is sent for parents about facial recognition enablement with link to disable for any privacy reasons.

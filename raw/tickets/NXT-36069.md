# NXT-36069 — IM - Item Config - HACCP Configuration page - Processes

## Description
As a District Admin I need a HACCP configuration page so I can configure or add any processes unique to my district.

*Requirements*

* Add a HACCP configuration sub-page to the Item Configuration page
* Have a section for the HACCP Processes
** there are 3 system defaults: No Cook, Same Day Service, Complex Food
** Maintain the following fields for each Process
*** Process Description
**** this should be unique across all processes
*** Associated CCPs
**** either comma separated or popup or something
** Provide History at the card level
** Provide the ability to add additional processes
*** The only required field is Process Description
*** Require selection of at least one associated CCP
****  use a multi-select dropdown
** Provide the ability to edit custom processes, and delete custom processes (icons)
** Action column will have not have any actions for the *default* processes
** Action column will have 2 icons for *custom* processes: Edit, Delete
* Have a section for the HACCP Critical Control Points [https://primeroedgenext.atlassian.net/browse/NXT-36741|https://primeroedgenext.atlassian.net/browse/NXT-36741|smart-link]

# NXT-40447 — IM - Menu Items - Servings - Menu Servings (scaled)

## Description
As a Menu Planner I want the ability to create additional scaled measures for menu item serving sizes so I can have all the different sizes I serve to students.

## Acceptance Criteria
* Provide the ability to add additional serving sizes for the Menu Item which are scaled from the primary measure (or using weight/volume)
* Separate this “Add Serving” function from the primary Add Serving function, as this function is only adding Menu Item servings
* Within the Menu Item tab specific version of the Units card, provide an Edit mode
* Within the MI Units edit mode, provide a “+Menu Serving” button
* When +Menu Serving is clicked, insert a new serving size row into the grid
** only Amount and Description are editable
** pre-load Description with original Menu Item serving measure
** Make Description a dropdown (not free text) with only the menu item measure and the corresponding weight or volume measures (also inherited from main measure)
** Calculate Weight/Volume by scaling from original measure
** Use row level Save icon to return to primary edit mode
* Provide guided informational messages in the banner bar

# NXT-5524 — Workspace - Inbox (Landing Page Summary View)

## Description
*User Story*
As a PrimeroEdge User, I can see recent unread user messages under the My Messages heading. This functions basically like an inbox. (Shorter version of Messages - Inbox)

*Mockup*
[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down-width&hide-ui=1|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down-width&hide-ui=1]

*Requirements*

* Section expands vertically to include additional messages as needed.
* Messages previews are displayed in this section
** First 140 characters
** “…” at the end of the preview to indicate that there is more to the message.
* “View All Messages” link will take users to a separate messages page.
* Clicking on a message preview will take users to a page where they can view the message in full.
** Inbox Screen (NXT-5920)
* Only the 10 most recent messages are displayed on the My Messages section of the Workspace.
* New Message button brings up a Messages pop-up window that allows users to create a new User Message.
** NXT-5915
* Button to reply to User Messages
** Users can send messages with rich-text formatting using the Compose feature (NXT-5915)

## Acceptance Criteria
- My Messages section is present in the workspace
- User messages are appearing properly
- Functionality present as described in requirements

# NXT-2064 — Production - Reporting - Pre-Production Costing (UI)

## Description
Would like for this to be a page and/or analytical display so that a report is not required to be generated. However, we can have a print mode that can be generated/used.

Ideally, it will give district wide information, with the ability to narrow it down by site type, and/or site. 

It will give a breakdown  based on meal type and menu lines for the current week. (i.e. breakfast totals, lunch totals, etc)

## Acceptance Criteria
This page will display information regarding production plans that have been filled out (completed), in regards to the costing information for those planned counts. 

It should be a page and/or analytical display so that a report is not required to be generated. However, we can have a print mode that can be generated/used.

It will give district wide information, with the ability to narrow it down by site type, and/or site.

It will give a breakdown based on meal type and menu lines for the current week. (i.e. breakfast totals, lunch totals, etc)

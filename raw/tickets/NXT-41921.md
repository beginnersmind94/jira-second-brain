# NXT-41921 — DataExchange Import – Fixed Width – Create and Edit Import configuration

## Description
As a SchoolCafé Admin I want to import a fixed width file so that I can use my state’s existing format.

*Notes:*

* This is the backend story to save the fixed width format to the import configuration in the database / backend
* Currently, we can only store delimited files, so we need to configure a way to save the fixed file format created or used and link it to the import routine instead of doing it with a delimited way
* This is related to the story to create and configure fixed width formats

*Requirements*:

* Use the database you created to save fixed width file formats and link them to the Import routine you create
* This is done in the Save button on the Review tab of the DC Import + New or Edit process
* 
* Save the fixed width configuration in a way that can be used by the import routine instead of the delimited way
* 
* When importing, the import routine will use this saved data

## Acceptance Criteria
* We are able to store fixed width file formats to the database so that the import routine can use them

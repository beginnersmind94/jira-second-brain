# NXT-53530 — IM - Items - Units - Inventory Readiness revised

## Description
As a district admin I want the Inventory Readiness tool to be extremely clear and to guide me in the UI so I know exactly what I need for inventory publishing and when I should publish.



!Inventory Options Step-20240827-125921.png|width=83.33333333333334%,alt="Inventory Options Step-20240827-125921.png"!

## Acceptance Criteria
* Update the logic and UX behavior for Inventory Readiness
* Remove the sequential logic of the inventory requirements
** instead it should operate more like a status
* Always display the inventory readiness banner
** including for placeholder units, when the “# of countable inventory units” selection is 1, 2 or 3
** there will be a status/banner for every scenario, including one for ‘Non-Inventory’ for Ineligible or Disabled
* Provide very clear messaging for each banner version
* Display very user friendly icons/labels for highlighted fields

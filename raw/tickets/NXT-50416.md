# NXT-50416 — Roles & Permissions - New Roles

## Description
We need to add new roles to support SchoolCafe that are applicable to Cybersoft employees only

*Requirements:*

* These requirements apply to the new version of Keycloak
* They apply to all levels of environments
* 
* Create a new role
** Name: Tech Support
** Add in the dev managers for each team
** Add anyone else they want
** Add Johnathon, Dallas, Bhanu
** Role will be used to manage Resource Catalog and any other technical resource we do not want Customer Support messing with
** This role is a super admin role and can do anything, so clone all the rights of the Customer Support role
* 
* Update Resource Catalog
** Apply Tech Support role to the resource Catalog
** Remove access to Resource Catalog from all other roles including Customer Support role
* 
* Create a new role
** Name: License Management
** Add Jay, Tricia, Dallas, Johnathon
** Role will be used to manage License Management in the future, but don’t mess with it yet; we’ll cover it in those stories

## Acceptance Criteria
* Roles are created
* Members are added
* Resource Catalog is updated to be only for Tech Support role

# NXT-2898 — Authentication - Login with Password Reset

## Description
*User Story*

As a user I want to login into PrimeroEdge and set a new password that can be used for future login attempts. 

*Requirements*

* Scenarios for password reset:
** User’s password expires based on the password requirements (90 Days by Default) - [https://primeroedgenext.atlassian.net/browse/NXT-4898|https://primeroedgenext.atlassian.net/browse/NXT-4898|smart-link] 
** User forgets password and manually triggers a password reset. [https://primeroedgenext.atlassian.net/browse/NXT-7890|https://primeroedgenext.atlassian.net/browse/NXT-7890|smart-link] 
* Password reset is covered separately depending on the scenario
** If the user’s password expires or administrator resets a user’s password, the user is presented with a reset screen on their next login (after entering proper credentials/temporary password)
** If the user triggers a password reset themselves, they are required to go through multi-factor authentication before reaching the set password page (this part may be deferred)
* User is required to enter the same password twice
* User is logged in automatically when the password change is successfully completed
* Password settings are covered in [https://primeroedgenext.atlassian.net/browse/NXT-4897|https://primeroedgenext.atlassian.net/browse/NXT-4897|smart-link] and must be followed when resetting password
** Password requirements are displayed
** If user does not meet requirements, an error is shown detailing which requirements are not being met

## Acceptance Criteria
1. User must authenticate prior account login (see scenario 1) in order to enter new password.
2. User must fulfill the password rules in order to change password.
3. New password must be saved in order to move forward with authentication and access into the system
4. An item should be added to the user page history indicating that a password change or password reset email was generated (specific details are not required, just a summary item that the password was changed/email was sent). Optional.

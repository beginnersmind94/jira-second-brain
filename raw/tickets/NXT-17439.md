# NXT-17439 — INV - Create Orders - Confirm Screen - Screen Enhancements

## Description
As an ordering manager I want the item grid on the Confirm Orders screen to be aligned similarly to the order level grid, so I can more easily see the details of my orders.

*Requirements*

* Move both the order level and item level action column to be the right most column
** Order Total should be to the left of the action column
** Try to keep the Action columns vertically aligned as much as possible
*** also try to keep Order Total aligned with Total Price from the items grid
* Change all instances of “Code” (Item Code, Order Code, etc.) to “Number”
** Order Number
** Item Number
** Vendor Item Number
* Change “Item Name” to “Item Description”
* Alert column will remain left-aligned
* If possible, adjust the sizing/width of the item columns to create more space for Item Description
* Overflow points from Sprint 48:
** (from [NXT-16300|https://primeroedgenext.atlassian.net/browse/NXT-16300] - Merge Orders) Once an order has been merged, the View icon should navigate to the newly merged order. The order which was merged *_into_*. Not the order which was just merged and no longer exists.

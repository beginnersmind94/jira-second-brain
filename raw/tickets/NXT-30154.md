# NXT-30154 — IM - Items - Create New Item - Recipe / Menu Item tab

## Description
As a District Admin I want a Menu Item card so I can specify the required Menu Item attributes at the same time I am creating the corresponding Item.

*Requirements*

* Load the Recipe / Menu Item Info tab according to user answers, per [https://primeroedgenext.atlassian.net/browse/NXT-30128|https://primeroedgenext.atlassian.net/browse/NXT-30128|smart-link] 
** if this a *Food Item*, this tab will appear standard with only Recipe/Menu info
** if this is *not* a Food Item, this tab will appear differently and will include a Serving Measures card, menu item info card and an Allergens card
*** [https://primeroedgenext.atlassian.net/browse/NXT-30132|https://primeroedgenext.atlassian.net/browse/NXT-30132|smart-link] 
* Display the Menu Item card if the user specified that the item was a Menu Item 
** The card will remain the same for either food or non food items
* Display the Recipe card if the user specified that the item was a Single Item Recipe
** Display along with the Menu Item Info card and Recipe Costing Card
* Disable the tab and color it gray if the user said *NO* to both Recipe and Menu Item

## Acceptance Criteria
* Load and display the editable Menu Item card if the user specifies a new item as a Menu Item
* Load and display the editable Recipe card if the user specifies a new item as a Single Item Recipe
* Load and display the Non Editable Recipe Cost Card
* Dynamically adjust the width of the cards based on Recipe/Menu Item card enabling
* Disable and color the tab gray if the item is neither a Recipe nor Menu Item
* Non-Food Menu Items are out of scope for this (separate story)

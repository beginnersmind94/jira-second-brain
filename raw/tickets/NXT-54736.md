# NXT-54736 — Roles & Permissions - GDSN Connect Special Permission

## Description
As a Customer Support role user, I would like to be able to turn on the GDSN special permission for certain roles so that those users can click the button

*Notes:*

* This permission is needed to control the visibility of the GDSN button in BO
* BO team will use this value to set visible = true/false for that button

*Requirements:*

* Add a new Special Permission on the special permissions tab for roles and permissions
** Name: Can Use GDSN Connect
* Default unselected for all roles
* Leave unselected for all roles in all existing production districts
* Note: When GDSN is turned on for a district, CS role will have to come and enable it for the roles they want
* Enable Global Connect left side Menu if this permission is enabled

## Acceptance Criteria
* Special permission is added to all environments and default is false

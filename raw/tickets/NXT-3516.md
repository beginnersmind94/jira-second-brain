# NXT-3516 — System - Site Types - Add functionality to create Grade Groups

## Description
As a district or central office admin, I would like to be able to add and manage grade groups for my district, so that I can easily reference them when planning my menus.

+Summary+:

Allow users to add a Grade Group (similar to Site Types) and associate multiple grades with that grade group.

+Details+:

* Transform the Site Types page into a page with 2 tabs (similar to Program Configuration) to accommodate Grade Groups.
** The page title can be renamed to *Site Configuration*, with the Site Types content moving to a Site Types tab. That functionality will not change
** The second tab will be called *Grade Groups*.
** {color:#6554c0}In the future, we will move the Sites page content to this page as a third menu option if possible.{color}
* In the Grade Groups tab, the layout and functionality will be the same as Classic:
** User can add a Grade Group by giving it a description.
** Within a Grade Group, a user can insert one or more grades (multi-select control again would be useful) from PK - 14.
*** Grades are seed data and will not change
*** A grade can only be in one grade group at a time, so it must be removed from one group to be added to another
*** Grade Groups cannot be deleted/removed

# NXT-4069 — Security - IP Whitelist UI

## Description
*User Story*
As a PrimeroEdge admin, I have a screen that allows me to easily add IPs to the whitelist to allow access.

*Requirements*
* Whitelist IP settings can be updated through security policies
* By default, all IPs are allowed through the policy
* There is a toggle to enable/disable the IP whitelisting
** The toggle is grayed out until at least one IP is added
** Once enabled, any sessions are exempted until they expire
*** A warning indicating this is presented when the whitelisting is enabled
* There is a button for adding IPs
** A popup appears for adding IPs
** User has the choice to add the following:
*** My IP (displays the user's current IP address)
*** Static IP - Allows adding a single IP
*** IP Range - Allows adding a range of IPs
* Added IPs and IP ranges are displayed
** IPs can be deleted
** When deleted, the user is required to reenter password
** If removed, any session on the IP are exempted until the session expires

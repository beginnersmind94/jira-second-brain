# NXT-17438 — INV - Orders - Order Detail - Cancel Order

## Description
As an operations manager I want a Cancel button on the Order Detail screen so I can cancel any orders which are not being submitted to the vendor.

*Requirements*

* Add a Cancel Order button to the Order Detail screen on the Details tab 
** this button will not be visible on the Receipts tab
** for Open and Pending Review orders
* Add a banner message in alignment with the row of buttons that provides information about the order
** at this time only show the order status: “Order #12345 is in *Open* status.”
* When user clicks the Cancel Order button
** prompt the user to confirm they want to cancel the order
*** “Are you sure you want to cancel this order?” Yes/No
** after confirming on the prompt, reload and change the status to Cancelled
** perform the check to see if any receipts exist
*** if somehow a receipt is created in between the time the user has opened the order and clicked the Cancel Order button, show this message:
**** “This order cannot be cancelled because at least one receipt has been created.”
* Do not provide the Cancel Order button if…
** the order has any receipts created for it
** the order is Closed 
** the order is already Cancelled
* Once an order is cancelled (8/18/21)
** Remove all action buttons related to the current order (Cancel Order, Print Order)
** Do not show the Receipts tab
** Do not show any Edit icons (Delivery Date, Delivery Location)
*** this includes the Edit icon in the Items list grid
** Change Comments popup to view-only mode
*** this includes the Comments in the Items list 
** Remove Add Item button from header
** Remove Save button from footer
** Keep the Back button

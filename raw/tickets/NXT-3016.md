# NXT-3016 — View Inspections - Actions - Send Email

## Description
*User Story*
As an Inspections user viewing an Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to email a copy of the Inspection to individual emails or email groups.

*Requirements*

* Send Email is an action that can be applied to an Inspection
** This action is present on any Inspection
** This action is present for any users who can view the Inspection
* When clicked, the user receives a popup
** Header: Send Email
** Information
*** Inspection Name
*** Site
*** Inspector
*** Score
** The user is able to input individual emails
*** There is a soft limit (to be determined)
*** There is logic to determine that a properly formatted address has been entered
** The user is able to input email groups
*** Implementation of email groups will be covered in a separate story
** There is a field for adding an additional message
** There are two buttons:
*** Send - Sends email
*** Cancel - Closes popup
* The end user receives an email with a pdf version of the Inspection attached.

## Acceptance Criteria
* User can email any available Inspection from the View Inspections screen in either view (card or list)
* User can email any available Inspection from the Inspection Details screen

# NXT-59651 — IM- Item Onboarding- Pack Size Info- Inventory Options- Units & Sub-Qty

## Description
As an onboarding district admin, I want a step in the bulk tool to be able to confirm the number of units, enter my pack unit descriptions and establish the sub qty relationships between the units.

!image-20250429-070214.png|width=1392,height=707,alt="image-20250429-070214.png"!

## Acceptance Criteria
* This step will be accessed through the Inventory Options (Units & Sub- Qty) on the Review Pack Size page
* Allow confirmation of # of inventory units
* Provide a toggle that hides all items marked as None/ Decide later 
** Default it to hide (On)
** For non food, items can not be marked as None/Decide later based on rules
* Dynamically provide text boxes for each pack unit based on the # of inventory units selected
** This applies to both description and Sub Qty fields
* Provide an alert per item once the user works through the row and does not complete all the fields based on their selection and highlight fields missing
* Allow save at any time and display highlighted text boxes based on what is missing once page is accessed again
** This is only for any rows that have been worked through
* Provide a way for the user to continue moving through the inventory options with a “Update Pack/serving Link” button to move to the next step instead of having to go back to the review screen to access that option

# NXT-5045 — Item Management - Split View Warnings

## Description
As a user of Item Management, I want to be able to see at a glance what the items are 'missing'

Mockup

!image-20200818-132943.png|width=91.66666666666666%!

 

Requirements

* Icon will show up on the split view with a 'warning icon'.
* This Icon will show all of the 'missing information' at a glance
* "Nutrient, Allergen, and Product Info is missing" would be the message displayed
** Another example would be "Allergen information is missing" or "Nutrient and Allergen Info is missing"
* The text would show up when hovering over the icon
* If nothing is 'missing' then the icon will not display

## Acceptance Criteria
* When viewing results in split view, I want to see the icon show up when information is missing and having it give me a very quick summary of what is missing

# NXT-5975 — System - Management - Eligibility Guidelines - Display Error Prone Info

## Description
As a district admin, I would like to be able to view the federal error prone guidelines, so that my organization can reference this information when working with parents and students.

+Summary+:

Allow users to view Error Prone information within the Eligibility Guidelines tab.

+Details+:

* In the Management category of the System module, add an *Eligibility Guidelines* page.
* Within the Eligibility Guidelines page, add a tab/section titled *Error Prone Info*.
* Basic design can mimic Classic:
** Within the tab, utilize the same display filters for Academic Year and State Category.
*** Current academic year should be selected by default and drop-down menu should display last 10 years.
*** State Category should display Contiguous States (default option), Alaska, and Hawaii in that order. 
** Display the current academic year and contiguous states' error prone info for the user by default when they land on the page. Table information can be the same.
** We should maintain tooltip buttons.
* Export to Excel button can be covered in a future story, so it can show as disabled for now.

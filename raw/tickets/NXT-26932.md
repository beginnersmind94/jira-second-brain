# NXT-26932 — IM - SPIKE- Export to Excel File

## Description
When doing a search or displaying the hole content of a search on a grid the user paginates thru the grid according to the selected number of records to display (1-100).  If the results grid throws more  100 records and the user desired to export the whole file then the user has to export page by page. The requirement to be able to export the complete set of result records to an excel file 

[https://devuicomponents.perseusedge.com/?path=/docs/components-grid--with-all-options|https://devuicomponents.perseusedge.com/?path=/docs/components-grid--with-all-options|smart-link]

## Acceptance Criteria
When the user selects data (all or based on filter criteria)  the user should be able export the complete result data set to an excel file

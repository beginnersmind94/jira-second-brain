# NXT-3015 — View Inspections - Actions - Re-Inspection

## Description
*User Story*
As an Inspections Supervisor or Admin viewing a submitted Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to send the Inspection back for Re-Inspection.

*Requirements*

* Re-Inspection is an action that allows redoing part or all of an Inspection
** This action is present on any Inspection with a Submitted status
** This action is present only to users with Approve permissions in both their role and the template the Inspection was created from
*** Supervisor and Admin roles
* When clicked, the user receives a slide out drawer
** Header: Re-Inspection
** Information
*** Inspection Name
*** Site
*** Inspector
*** Score
** The user is able to mark which questions get Re-Inspected
*** Re-Inspect all failed questions (default)
*** Re-Inspect all questions
*** Re-Inspect selected questions
**** If chosen, user goes through and selects individual questions to be Re-Inspected
** There is a field for notes from the user
** There is a field to schedule the Re-Inspection (optional)
** There is a field to assign
*** Marked with original Inspector by default, can be changed
** There are two buttons:
*** Re-Inspect - Sends Inspection to Inspector in Re-Inspection status
*** Cancel - Closes popup
* Once marked for Re-Inspection:
** Status is changed to Re-Inspection
** Assignee is changed to the proper Inspector as needed

## Acceptance Criteria
* User with proper permissions can mark any submitted Inspection for Re-Inspection from the View Inspections screen in either view (card or list)
* User with proper permissions can mark any submitted Inspection for Re-Inspection on the Inspection Details screen

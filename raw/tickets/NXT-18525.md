# NXT-18525 — Production- Reports- Menu Item Nutrients (UI and API Report)

## Description
This report needs to be able to display nutrient information for the user based on the criteria they select and which nutrient information they would like to see.

Page Criteria; 
Site Code
Site
Meal Type
*From* Date and *to* Date
They also will be given an option to check or select which nutrients they would like to see with the default selection is to all. 

Under the criteria, give option as so,

Once report is generated all columns should display what the user selects as the nutrients they would like to see. 
The rows will be divided based on Date, Meal Type and Menu Line. All menu items within that meal type and menu will show in the row in item category order split by item categories.

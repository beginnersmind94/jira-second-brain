# NXT-56234 — PE - GDSN Connect - Select / Deselect all fields in Ingredients and Stock Item Tabs

## Description
_(none)_

## Acceptance Criteria
Navigation: MP - GC - Pending Updates

Currently, the users do not have an option to select or deselect all the fields. 



As a change, the user should be able to have master check box that enables by default which selects all the fields. 
Upon deselecting the master check box, the system should deselect all the fields in the corresponding Tab.

Changes applicable for Ingredients and Stock item Tabs, the check box should be available separately for each tab

Regression: Single attribute update, multiple attribute update, all attributes and no attributes selected in a specific tab or both tab. The button gets disabled when all the attributes are deselected (As-Is)

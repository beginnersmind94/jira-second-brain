# NXT-8106 — Production - Production Record

## Description
As a site manager, the production record page will reflect the menu that has been planned for my site on the calendar. This will contain the menu items that have been assigned to me, as well as the grade or age groups that were assigned in the menu, and any other menu items that I might have 

I must be able to enter in counts that reflect the quantity of servings that I produced of each menu item. In addition, I must also be able to enter a number for the estimated total quantity of meals per age/grade group and/or person type.

The page should also allow access to the following reports:

1.) Technician’s Worksheet

2.) Recipes

3.) Production Form

4.) Production Report

5.) Menu costing report



There should also be a place to enter any comments that might correlate with my Production Record. I should have the ability to confirm or decline “Offer Vs. Serve” on the page. There should also be the ability to add a menu item from this page, the ability to save the data that I’ve input, the information that is displayed in the production plan area and display the status of the record. The Food preparation Site should also be displayed (in the event that my site is configured for preparation at another location)

The page must also contain information regarding the Meal Type, the date of the plan that I’m on, and the name of my site.

h2.

## Acceptance Criteria
h2. Acceptance Criteria

This page will allow/display the following:

1.) Site Name

2.) Date of service

3.) Meal Type

4.) Line Name

5.) Offer vs. Serve (set by site configuration, but should be adjustable on this page)

6.) Food preparation Site should be displayed (in the event that my site is configured to have my food prepared elsewhere)

7.) Serving Groups with text boxes where I can enter planned counts for meals

8.) List of menu items that are planned by the site (with text boxes for each item)

9.) Technician’s Worksheet Report

10.) Production Recipes

11.) The ability to Add Menu Items

12.) The ability to save the data that I’ve input

13.) The status of the record

14.) Production Form

15.) Production Report

16.) Menu costing report

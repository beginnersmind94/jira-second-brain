# NXT-10972 — [TaggedDashboard] - Dashboards (Insights) - Accountability - Meal Equivalents (MEQ) Widget

## Description
As a central office administrator and/or site manager I want to see the calculated meal equivalents KPI table so that I can understand my performance within the district/site.

Reliance on date range and district/site selectors may be needed for full capability of this widget. Until these are available, we can display the single instance using values for the current month of operation for the district (if a district level user) or site (if a site level user).

## Acceptance Criteria
* User can see the MEQ KPI table displaying number of counts/amount by category and calculated values (MEQ)
* User can see the total MEQ calculated as the total for each category MEQ amount
* Timeframe selector available for selecting \[Month Year] context
* Site selector available for setting site context
* Conversion factor values preset with values, but able to be edited

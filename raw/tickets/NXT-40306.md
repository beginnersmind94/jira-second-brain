# NXT-40306 — eOrders Configuration – Add New Part 1

## Description
As a SchoolCafé user, I would like to be able to create eOrder configurations so that I can use them to electronically order food.

*Notes:*

* This story covers when a user is on the dashboard and clicks the + New button and creates a new eOrder Configuration
* We are replicating the work done in Classic to gain parody
* Classic URL: [https://online.primeroedge.com/Inventory/ManageVendorEDISettings.aspx|https://online.primeroedge.com/Inventory/ManageVendorEDISettings.aspx]

*Flow:*

* Open SchoolCafé and browse to Settings > Configuration > eOrders
* Click + New
* Create the configuration and save
* Return to the updated list

*Requirements*:

*eOrders Dashboard page:*

* Add a button if not already there on the top right
** Title: + New
** On click, Open the Add New eOrder overlay

*New Overlay:*

* Create a new overlay, per the Figma
** Title: Add New eOrder
** Overlay will use a Stepper control like on the Reports and on the Imports & Exports

*Stepper:*

* Add a stepper per the Figma
* Add these steps:
** Vendor
** Configuration
** Sites
** Distribution
* Save state on screens
** When using Previous and Next to move back and forth, don’t make the user reselect / redo things, keep state
* 
* Each tab will be treated in a separate story
* 
* Security
** Don’t show this overlay in Roles and permissions CRUD tables
** This overlay’s permissions = Add permission for eOrders Dashboard page, therefore restrict security to those roles with eOrders Dashboard Add permissions plus Cybersoft admins

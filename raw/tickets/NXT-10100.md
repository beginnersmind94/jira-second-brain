# NXT-10100 — Broadcast Message - Allow Broadcast banners to persist throughout the entire website

## Description
+*Summary:*+ Currently, Banners only appear in Workspace pages, but in order to make sure that users see important messages no matte where they are in the application, let’s make the changes we need to have the banners persist on any page, in any module. This change was envisioned a while ago, but requested again by Customer Care to be able to use with Release Notes.

Also, please use this enhancement to work out Role/permission issues with this button, as it is not currently working in QA. 

+*Goal:*+ We should be able to use this button to display a banner at the top of the screen, site-wide, for the current district (or, if the functionality is completed, for multiple districts).

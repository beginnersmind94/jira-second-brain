# NXT-4973 — Acct Mgmt - Add Patrons - Adult - Wizard Step 5 - Documents & Save

## Description
*User Story*

As a district admin I want to fill out general information for an adult patron.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418434%3A236821&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418434%3A236821&scaling=scale-down-width|smart-link] 

*Requirements*

* User needs a way to upload documents via the final step of the ‘Add Adult’ flow.
* User should be able to leave comments as well.
* User should be able to remove documents.
* Documents must be stored appropriately for this exact student.

We need to keep in mind:

* Allowed extensions and size limitations. If these aren’t established by DataExchange, we’ll need to determine them.
** User needs to be informed appropriately if the document breaks any of these rules.
* The user could abandon the ‘Add Adult’ flow - if they do, we shouldn’t keep any uploaded documents on our servers.

*Transaction Description / Receipts*

* Need to be very clear that a refund was issued, and also show if the refund was done via cash or check.
** Update the description to show as ‘Refund Cash’ or ‘Refund Check’ respectively.
** Include in the receipt as the Transaction Type (or a sub-heading)

*Reports*

* Cash Collection report
** Add an option to ‘Include Refunds’
** If enabled, show a Refunds column with the amount of ‘Refund Cash’ + ‘Refund Check’

Discussion Notes (7/31/25 Mini Demo)

* Refund Checks
** Do not show in closing balance or deposit slip
* Over/Under
** Only show in System > Checks as a negative
* Add the Description column to the Transactions grid in Sessions > Reconciliation
** Default column = on for the default view

## Acceptance Criteria
* User can issue a refund via either Cash (Default) or Check.
* User can see whether a refund was done via Cash or Check in Transaction lists
* User can see whether a refund was done via Cash or Check in Receipts
* User can see refund totals in the Cash Collection report

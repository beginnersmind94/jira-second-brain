# NXT-65296 — ACC - Labor Hours Feature - Updates

## Description
Story to contain data entry behavior of main grid for both entry methods.
Standard Method
User can immediately edit values but must click save to commit changes.
User is presented a +- text field that only accepts numbers and can be stepped + or - by 1.
User can click Cancel to discard changes.
Grid updates Displayed month based on selected date.
Data is only for a month at a time.
Totals update after saving.


!image-20250812-184426.png|width=839,height=652,alt="image-20250812-184426.png"!



Precise Method
User can immediately edit values but must click save to commit changes.
User is presented text field that only accepts numbers up to a tenth “00.0”.
User can click Cancel to discard changes recalls last saved data.
Grid updates available text boxes based on selected date and site.
Grid can remember change after pagination.
Grid supports ADA compliant field tabbing.
Provide a checkbox to highlight the background of Cells that have no data (“Highlight Missing Counts”) Remembers even after filter change.
Totals update after saving.

Nice to have (not required)
Highlight any text field where a change is detected.

# NXT-33908 — Settings - Make Internal Configuration Type Visible Only To SchoolCafe Admin

## Description
Settings - Make Internal Configuration Type Visible Only To SchoolCafe Admin

## Acceptance Criteria
* Settings with “Internal” Configuration Types shall be visible to only SchoolCafe Admins
* As a SchoolCafe Admin, I shall be able to see “Internal”, “Common”, and “All” within the Configuration Type dropdown.
** “All” shall be set by default on page load
* As a non-SchoolCafe Admin user, I shall be able to see only “Common” within the Configuration Type dropdown.
** “Common” shall be set by default on page load

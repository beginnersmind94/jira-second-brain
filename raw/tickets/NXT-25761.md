# NXT-25761 — Platform - Security - General Permissions CRUD Operations (DELETE)

## Description
As a primero edge admin, I should be able to access resource catalog to add a new resources available for any district with functions resource type along with category so that the resource type can be accessed by any user with the assigned role for that resource.

+*Requirements:*+

* Enable DELETE buttons for the Resource field in the resource catalog with specific permissions
* Resource Permissions CRUD operations with DELETE Permissions to be enabled and making sure FO & BO teams can utilize these resource permissions via roles & permissions page

## Acceptance Criteria
* Any page type resource with DELETE button enabled shall be able to delete the resource
* Enable Alias for the Resource field in the resource catalog with DELETE option for PAGE only
* Cleanup all resources that are not needed for Release 4 & 4.1
* Resource Permissions CRUD operations with DELETE buttons to be enabled and making sure FO & BO teams can utilize these resource permissions via roles & permissions page

# NXT-34510 — IM - Recipe Rework - Create New Recipe

## Description
As a Nutritionist I want a semi-guided flow for creating a brand new recipe so I can focus on the required tasks in the right order.

This is the list of business rules or UX variances on the new Recipe Details page that will occur for new recipe creation.

[Figma - New Recipe flow|https://www.figma.com/proto/OAZ7qTRgdG5bFFGuhDmkoZ/Item-Management-6.0?page-id=3238%3A54123&node-id=3282%3A47737&viewport=411%2C-15%2C0.03&scaling=min-zoom&starting-point-node-id=3282%3A47737&show-proto-sidebar=1] (click New button)

*Requirements*

* Load the new recipe summary card in full screen mode, hiding all other cards
* Display all the editable fields 
* Indicate the 3 required fields (Recipe Description, Serving Size + Measure, # of Servings)
* Provide a Save & Continue button in the card header
* Check for duplicate Recipe Description
** Ideally before user clicks Save button if possible
* Upon successful save, load the remainder of the page in a restricted mode
** summary card goes into view mode with some exceptions
*** until steps are provided, only edit icon is needed (no copy, reports, etc.)
*** several calculated fields will be null and should be shown as ‘--’ or with some indication that the ingredients need to be added
** tabs are loaded, but only Steps & Ingredients is accessible 
*** Provide a hover tooltip on the disabled Nutrition Info tab ‘Please add recipe ingredients’
*** Load the Steps & Ingredients card in [Edit Steps mode |https://primeroedgenext.atlassian.net/browse/NXT-36358]so user can immediately add a step
** Image card can be available, but should not be in edit by default

## Acceptance Criteria
* Load page elements in a guided way the first time a recipe is being created
* Focus the user on the summary card until initial mandatory attributes are provided
* Load the remaining page but focus user on Steps & Ingredients tab
* Disable other tabs and provide explanatory tooltips focusing on ingredients

# NXT-43548 — PR-Production Adjustment- Apply serving size exceptions

## Description
As a menu planner, when I designate a serving size exception in IM, I am purposely specifying that a particular serving group gets a specific serving size or a serving group does not get served a particular serving size.

## Acceptance Criteria
* Apply serving size exceptions based on configuration in IM when adding menu items

!image-20231003-071725.png|width=1607,height=733!

* When doing a production adjustment for a menu item, insert the serving sizes as part of the exceptions in the same fashion as add menu item logic
** Display the serving size exception if applicable for the serving groups that are present in the date range
** User still has the ability to change serving groups but its just the default serving exception that will be displayed for them once in the grid

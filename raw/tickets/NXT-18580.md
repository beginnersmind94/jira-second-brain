# NXT-18580 — IM- Multiple Recipe Steps with Same Ingredients

## Description
USDA requires that if an item is used in multiple steps of a recipe, they need to calculated separately and not combined together. 

Example: A battered steak with gravy. The batter for the steak requires 2 tablespoons. The gravy requires also 3 tablespoons of flour. These items need to be listed separately as food required and not combined measurement. 

Currently, software will calculate this scenario as 5 tablespoons instead of 2 tablespoons and 3 tablespoons.

## Acceptance Criteria
Steps with items should all be calculated separately and never combined.

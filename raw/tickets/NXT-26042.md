# NXT-26042 — UX - Centralized Tag Management UI

## Description
h4. Description

As a user, I would like the ability to view a summary of tag assignments with the ability to easily manage the tags for the items containing a specific tag.

## Acceptance Criteria
h2. Acceptance Criteria

* As a user, I would like the ability to see a list of tags and a count of items each tag has been assigned to.
** This list shall include the tag name and the count of assignments
** The tag name shall be sortable alphabetically be default
** The tag name shall offer a filter/quick search
** The count shall be a link that when clicked on shall allow the user to view a list of the entities containing that tag
* As a user, I would like the ability to view a list of the individual entities that a given tag has been assigned to.
** This list shall include:
*** The module the tag is associated with
*** The entity type that the tag is associated with
*** The entity id
*** The entity description
*** The tags that are assigned to the entity
**** The tags shall be editable whereby a new tag could be created and assigned, an existing tag assignment could be removed, or an existing tag could be assigned
*** Filters / Quick search shall be available for the following columns:
**** Module
**** Entity type
**** Entity ID
**** Description

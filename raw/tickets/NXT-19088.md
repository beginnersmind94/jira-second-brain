# NXT-19088 — Income Surveys - Add New Survey - Contact Info Step

## Description
As a district admin, I would like the ability to quickly and easily add a manual income survey on behalf of a household, so that their income data is entered into our system for reference at any time.

*Summary*

Within the ‘Add New Survey’ function, add a step to help add a new survey record for a set of students by allowing the household to leave their contact information if they want.

*Requirements*

* Display a *Contact Info* step within the survey, with the following fields:
** First Name
** Last Name
*** if the first and last name matches a student on the survey, warn the user that the are using the name of a student (just as we do on the application)
** Work Phone Number
*** IF the user enters a number, validate it to make sure it has 10-digits and then automatically format the number to display in the (XXX) YYY-ZZZZ format.
** Cell Phone Number
*** IF the user enters a number, validate it to make sure it has 10-digits and then automatically format the number to display in the (XXX) YYY-ZZZZ format.
** Street Address
** City
** State
** ZIP Code
*** IF the user enters a ZIP code, validate it to make sure it has 5-9 digits.
** Email Address
*** IF the user enters an email address, validate it to make sure it is a valid email format.
* All fields should be *optional*, and we should only validate the number fields above if the user enters data into them.

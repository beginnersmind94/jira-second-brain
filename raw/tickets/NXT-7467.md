# NXT-7467 — Misc - Edit Check Resolution

## Description
*Story*

As a user I need to be able to see all edit check that need to be processed so a comment can be entered as to why it occurred. An edit check is created when the percentage of students for an eligibility type exceeds the attendance factor.

*Requirements*

* A list of Edit Checks can be shown, default status is UNRESOLVED [https://primeroedgenext.atlassian.net/browse/NXT-179|https://primeroedgenext.atlassian.net/browse/NXT-179|smart-link] 
* One or more comments can be entered for an Edit Check. 
* The user and timestamp are part of the comment
* When a comment is saved the Edit Check is marked as RESOLVED
* When a resolved EC is opened the comments are visible

## Acceptance Criteria
The user can enter a comment on edit checks that are unresolved.
When a comment is entered and saved the status of the edit check is marked as resolved.

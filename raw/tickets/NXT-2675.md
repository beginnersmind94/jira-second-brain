# NXT-2675 — Configuration - View (Topic, Subtopic, Method, Source, Job Title)

## Description
Ability for admin (with permission) to configure Topics, Subtopics, Training sources, Training methods 

* View Method, Source, Topic, Subtopic populates in the in the field for next event entry
* Trashcan only shows when topics, subtopic, method, source are inactive

## Acceptance Criteria
Administrators can configure training event criteria to use when creating a new training event

# NXT-36784 — PR- Consolidated Plan/Prep site distribution- Remove whole screen loader and provide background loader (UI ONLY)

## Description
When a user goes to the consolidated plan or the prep site distribution tab, it locks the screen from being accessed until the user loads the page completely. This stops the user from being able to switch tabs if needed especially when they are being led into that particular tab (from the calendar or production status page)

## Acceptance Criteria
* Show loader on the consolidated plan to be the same as the plan or record page
* Show loader on the prep site distribution page to be the same as the plan or record page
* User should be allowed to switch between any of the tabs at any moment if they are available whether the page has completely loaded or not

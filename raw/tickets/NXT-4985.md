# NXT-4985 — AcctMgt - Add Patrons - Students - Wizard Step 3 - Household - Add/Delete

## Description
*User Story*

As a district admin I want to fill out household information for a student patron via a wizard.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default]

*Requirements*

* Populate Step 3 of the wizard added via NXT-4982.
* This should be the same functionality as the current edit drawer for Student Details > Household.
** User can perform lookups, can see who will be part of this household after creating the account via a table, etc.
** The confirmation pop-up for students who are already in a household of size >1 should still exist.
** No students should be moved into this household however until a SAVE action is taken on this page.
* 'SAVE & CLOSE' button this step, which will of course save any information but close out of the wizard.
** User should receive a success confirmation toast about the completed task.
* ‘NEXT’ button available which will save, and take the user to Step 2.
* ‘CANCEL’ button should provide a confirmation prompt for the user.
** ‘Are you sure you want to cancel? Changes from this step will be discarded.’
* ‘BACK’ button is available, acts as a Save.
* Stepper (top right component) can be clicked to allow navigation to a future step.
** This would be the same as the user clicking on BACK/NEXT. Validations + Save must happen.

## Acceptance Criteria
Household information can be entered.
Information is saved while navigating the wizard.
Information is saved appropriately.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)
Accessibility / ADA Compliance programmed appropriately (full KB nav possible, etc.)

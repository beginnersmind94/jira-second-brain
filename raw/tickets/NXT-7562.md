# NXT-7562 — INV - Physical Inventory - Main Screen - Add Items (Physical Incomplete status)

## Description
As a site manager I want the ability to add an item to my physical inventory so I can count those items as I encounter them.

[Figma|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=90479%3A149975&viewport=325%2C-4288%2C0.08251962065696716&scaling=scale-down]

On the physical inventory count page, provide the ability to add an item.
This would also be used for the ‘Quick Entry’ / on the fly Physical Inventory approach.

Provide an Add button on the Items tab. (not on Summary tab)
Once the Add button is clicked, load the Add Item panel

Selection criteria to add an item:

* Item Category - dropdown
* Item Description - dropdown
** this would be a list of items that can be added
* Usable Whole Units - numeric entry field
* Usable Broken Units - numeric entry field
* Damaged Whole Units - numeric entry field
** potentially configurable 
* Damaged Broken Units - numeric entry field
** potentially configurable 
* Comments - text field

Validation on add item control:

* only items that have been orderable or on-hand for that site
** item site record exists
* hide items that are already on the Physical Inventory

Add Item button

Once the item is added, return to the Items tab.

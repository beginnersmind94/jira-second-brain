# NXT-8151 — Orders - Order Preparation

## Description
As a user I need to be able to track the preparation of orders and get them ready for distribution and fulfillment

*Requirements:*

* The user can select the criteria of what orders to see for preparation
* The user can see a list and the count of All items ordered (Item summary)
* The Item summary will also show the counts of the items to be delivered to the different locations
* The order list will show the delivery/fulfillment/pickup location
* The status of the order can be updated. 
* Bulk orders show the counts of items to be prepared and to what location they are to be delivered. 
* Users can print or export the pages
* Users can print labels Individually, or all based on the template selected. [https://primeroedgenext.atlassian.net/browse/NXT-8153|https://primeroedgenext.atlassian.net/browse/NXT-8153|smart-link]

## Acceptance Criteria
From the Left Panel, Under Point of Service, below Deposits. there is a function Orders. When Orders is Expanded there is an Order Configuration, Order Review, Order Preparation, and Order Fulfillment.

This page is launched by selecting Order Preparation

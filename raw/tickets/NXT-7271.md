# NXT-7271 — INV - Physical Inventory - Reconcile (UI)

## Description
As an inventory user I want a Reconciliation process so I can save physical inventory counts and update my perpetual inventory.

*Requirements*

+UI work:+

* Provide an alert/message to indicate this will update the perpetual as of the date provided.
** Message: “Your Usable Physical Inventory will modify and become your new Perpetual Inventory. Are you sure you want to reconcile?”
** OK / Cancel

If an item is on its first Physical Inventory, validate that a price exists for that item.

* potentially highlight items without a price, indicating they require review
* double check these items
* provide some insight into the price being used for that item
* if zero for price, validate that it’s not a commodity.


Reconciliation back end captured in [https://primeroedgenext.atlassian.net/browse/NXT-7707|https://primeroedgenext.atlassian.net/browse/NXT-7707|smart-link]

## Acceptance Criteria
User can click Reconcile button
User will see Reconciliation confirmation prompt
Reconciliation will update Perpetual Inventory
Reconciliation will change PI status to Reconciled

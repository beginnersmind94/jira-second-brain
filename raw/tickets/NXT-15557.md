# NXT-15557 — INV - Orders - Orders List - View Orders - Search Results

## Description
As an Inventory user I want an Orders list page so I can view all existing orders and open the specific orders I need to modify or review.

This story is for the results which are loaded according to the search/filter selections made in the header, per [https://primeroedgenext.atlassian.net/browse/NXT-8911|https://primeroedgenext.atlassian.net/browse/NXT-8911|smart-link].

*Requirements*

+Orders List:+

* List - Kendo grid selectable columns
* Note: all columns can be searchable, if ‘Field Search’ button is enabled
** Order #
*** this one must be searchable
*** Fixed column (cannot be hidden)
** Site
** Vendor
** Delivery Date
** Order Date
** Order Amount
** Status
** User (Created By, Modified by)
** Entry Method (possible options for now)
*** Receipt w/o Order
*** Production Order
*** Standard/Manual
** Action column
*** View icon for each order, to open the Order Details for the selected order
**** Placeholder icon until [NXT-7272|https://primeroedgenext.atlassian.net/browse/NXT-7272] is developed
*** Fixed column (cannot be hidden)

## Acceptance Criteria
* Load orders based on initial, default selections
* Use adjustable Kendo grid with additional column selectors
** Order Number and Action column are the fixed columns
** Other columns can be hidden but are shown by default
* Columns will be searchable with ‘Field Search’ button
** Orders column *must* be searchable

# NXT-58834 — Menu Publishing API for SCTV+

## Description
As a product owner, I want to have the APIs ready to connect the menu planning module in SchoolCafe 2.0 with SCTV+ so users can view their menus.

## Acceptance Criteria
* Able to pull menu item details including:
** Serving Lines associated with each site and associated dates
*** Menu Item Categories for each serving line
**** Details for each item within that category including:
***** Item Name
***** Menu Item Image
***** Nutritional information (NEW - more than just Calories)
***** Allergens (NEW)
***** Meal Components



*Refinement Discussion 02/17/25:*

* What nutrients would we be pulling? 
** Calories, Protein, Carbs, Total Sugars, Sodium
* Concerns about how we show this information in the UI - may not have the space etc.
** Expanded nutrient info is more on the back end for now - the UI would receive updates later.
** Districts may get options to control the visibility of nutrients - turn protein on/off for example.

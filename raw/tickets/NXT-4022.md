# NXT-4022 — Authentication - New User Email

## Description
*User Story*
As a new PrimeroEdge user, I am able to receive an automatically generated email that allows me to set my new credentials and log into PrimeroEdge for the first time.

*Requirements*

* When a new user is created, an email is automatically generated
** Email is sent from a no-reply email address
** Email contains a one time use link
** Link expires after a configurable amount of time
*** Default: 2 hours
*** Max: 24 hours
* Email looks professional and clean
** PE Logo
** Text:
*** *Email Subject*: Welcome to PrimeroEdge\!
***Email Text*:
PrimeroEdge logo (centered)
***Header* (centered): Welcome <First Name>\!
<Admin First Name Last Name> from <schoolDistrict> has invited you to start using PrimeroEdge. To get started, click the button below to set up your account:
***Button*: Setup Account
* When the user clicks the button/link, they should:
** be logged in to PrimeroEdge in their web browser 
** be taken to a Set Password screen (covered in NXT-4023)
** have the link expire
** not have access to any other screens
* A resend button will be covered in another story if needed

## Acceptance Criteria
* Email is automatically generated and sent out when a new user is created
* Email has PE logo, configurable district logo, and text set by PO
* Expiring link is present with configurable expiring time
* Link takes user to a placeholder screen

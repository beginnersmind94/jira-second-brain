# NXT-15894 — Verification - Family Portal Integration - Verification Response

## Description
As an applicant I want to see letters regarding verification and send back my response via the Family Portal to save time.

*Mockup*

No additional screens needed. Prior (Classic) screens are below

*Requirements*

If an online application is included for verification:

* Family Portal user should be able to see any letters that are sent to them:
** Selection Notice
** Follow-up Notice
** Determination Notice
* FP user should be able to respond to the request:
** Replicate the questionnaire and document attachment that’s in the classic version of Schoolcafe/PrimeroEdge.
*** User can agree/refuse to respond.
*** User can upload documents tied to each indicated income.
*** If 0 household income was entered, ask for supporting proof.
*** *-NEW-*-: User can enter a comment to help explain their situation, 500 characters.-
*** User can review and then submit the information.
* Documents that are sent down will need to be stored appropriately and tied to the Verification Details for the selected application. (Deferred)
** Track when the documents were sent in.
** Note that they came from the family portal.
* *History should be captured but will be done in another story.*

## Acceptance Criteria
Family Portal updated for users there to:

* be notified of their verification selection.
* can refuse to respond.
* can respond and attach documents and comments.

Response flows back down to the Eligibility module and is available under Verification Details.

# NXT-59091 — LMA - Inventory Distribution - License

## Description
As a Customer Support person, when I want to apply a distribution license to my sites, I want to do so so that we can sell this technology to customers.

h2. Context

This story aims to add an inventory distribution as a license type to School Cafe. The implementation will follow the same approach used for the Vending license. Once completed, the BO team will utilize this license type in its code as part of the distribution project.

h2. Requirements

* All requirements apply to System > License Management.
* The backend must be updated to ensure Inventory Distribution is a valid license type.
* Inventory Distribution should appear in the list of license types under the summary and configuration tabs in license management.
* The license can only be applied to sites.
** It must not be allowed to apply to Districts.
** The license count must be positive numbers only.
* Any changes made must be captured in the History log.
* 
* In the System > Sites < Site Types tab:
** Change the word Warehouse to Inventory Distribution for both name and description
** Note: Ensure BO tests this change thoroughly before moving to prod

## Acceptance Criteria
* Users are able to enable and disable Inventory Distribution licenses
* Users are able to apply to sites only
* Users are able to view changes in the history

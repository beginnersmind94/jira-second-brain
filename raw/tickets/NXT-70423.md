# NXT-70423 — FIN - Add "Select All Accounts" Shortcut and Export Format Choice to the General Ledger Report

## Description
*As a* Child Nutrition Director running a General Ledger Report, *I want* to be able to export all accounts in one action and choose how my export is packaged, *So that* I don't have to manually select every account one by one and can receive my data in the format that works best for my finance team.

## Acceptance Criteria
* The account selection dropdown must include a select all checkbox that instantly selects all active, mapped GL accounts currently visible within the user's search filter.
* The system must display the selected accounts as removable chips alongside a label that accurately counts the total number of selected accounts.
* The asynchronous Document Center generation must successfully process and export the report without timing out, even when the maximum number of district GL accounts are selected.

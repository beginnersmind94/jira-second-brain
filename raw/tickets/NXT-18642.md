# NXT-18642 — INV - Orders - Manage Orders - Quick Receipt - Confirm in Edit state

## Description
As a site manager I want the ability to Confirm an existing Quick Receipt order in Incomplete status so I can open an existing Quick Receipt and complete it.

*Requirements*

* After the user opens an existing Incomplete Quick Receipt ([NXT-18050|https://primeroedgenext.atlassian.net/browse/NXT-18050])
** User can once again Save as Incomplete ([NXT-18641|https://primeroedgenext.atlassian.net/browse/NXT-18641])
** User can Confirm 
*** Note: this is different from Confirm in [https://primeroedgenext.atlassian.net/browse/NXT-18080|https://primeroedgenext.atlassian.net/browse/NXT-18080|smart-link]
*** this will officially close the Quick Receipt
** When user clicks Confirm button
*** This will officially create the receipt and complete the Quick Receipt
*** the order will be saved and immediately put into ‘Closed’ status
*** Receipt # will be assigned
*** The screen will update ([NXT-18083|https://primeroedgenext.atlassian.net/browse/NXT-18083])
*** Update the Status 
**** Order Status will be ‘Closed’
*** Perpetual Inventory will be increased accordingly

## Acceptance Criteria
* Incomplete Quick Receipts can be opened and Confirmed
* Confirm will require same required fields regardless of Quick Receipt status
* Order will be closed
* Receipt # will be assigned
* Perpetual inventory will be increased

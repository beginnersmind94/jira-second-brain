# NXT-42385 — Custom Reports – Add New Report – Distribution Tab - Inbox Only

## Description
As a SchoolCafé user, I would like to be able to distribute my custom reports so that others can receive them without having to run reports.

*Notes:*

* This is the sister story for the Add New Custom Report Distribution tab story
* In that one, we did email and FTP, in this one we are doing Inbox
* 

*Requirements*:

* These requirements are for the Add New Custom Report overlay Distribution tab
* 
* Fields to show if they select Inbox:
* Use Sharing List
** Radio Buttons
** Values – Yes / No
** Required if Email = Yes
** Default = No
** If yes, show or enable the users and role selectors
** If No, hide or disable the users and role selectors
** Note: this field has no impact on the External Emails field, they can complete that fields regardless of this answer
* SchoolCafé Users
** Same as on Share screen
** Multi-select list
** Values – active user accounts for the district
** Required if Sharing List = No
** No default values
* SchoolCafé Roles
** Same as on Share screen
** Multi-select list
** Values – Roles for the district
** Required if Sharing List = No
** No default values
*

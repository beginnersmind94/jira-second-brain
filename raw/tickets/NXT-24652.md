# NXT-24652 — IM > Recipes > Recipe Page > Contributions need to pull from Items + Recommendation feature

## Description
When a user is creating a recipe to use as a menu item, and have entered the component contribution details on the Item/Ingredient initially, that component information should flow through (and calculate) in the recipe so that the user does not have to enter the information again. 

I entered the 1 1/2 WGR component on the item of Rice (LI-16192) 
I then used that item in the recipe for Basmati Rice (LR13270)

When turning that recipe into a menu item, and expanding the component section, there should be an option to use the "calculated" component information (based on data from the ingredients), or manually enter the component contribution information.

* The Header should show a label as “\[System Generated]” if its auto populated
* Provide a column for the Calculated Recommendation
** Provide an icon/button to accept and populate the Component with the recommendation

# NXT-7197 — Menu Grid Button Details

## Description
Button detail allows the user to view and edit the parameters that determine the style, location, and content of a button.

*Requirements:*

* The button representation is shown based on settings.
* Colors for the field and border can be selected
* Font characteristics can be selected.
* Alternate Button Text can be entered.
* The position and size of the button can be changed.
* Category buttons can reference a Menu Grid to display when the Category is selected in ExpressPoint

## Acceptance Criteria
The button is shown

Color for the fill is editable (Default is Category color of Item for items, White is the default fill color for Category)

Color for the border is editable (Default is Category Color for Menu Items, White for Categories)
Border thickness is editable (Default is 0 for Items, and 0 for Categories, Min is 0, Max is 10 in Pixels
Default Button size is 1x1
Position is set when the button is initially dropped.
Font is editable (Default is Arial)
Font size is editable (Default is 16) (Min is 8, Max is 48)
Font style is editable (Default is Regular)
Font color is editable (Default is White for Items, Black for Categories) 
Button Text can be edited (Default is Item Mgmt Button Name) Max characters 30
Button Position is editable (Position is Top Left Corner)
Button Size is editable (Minimum is 1x1, Maximum is defined by Grid Size)
Category Buttons allow the user to select a menu grid to display when the button is selected.
Category Menu is editable (Contains the name of the menu to display when the button is selected from ExpressPoint) Default is NULL

When the button is saved the change is to the current grid only. Updates do NOT populate back to Item Management POS Info.

Color for the fill is editable (Default is Category color of Item for items, White is the default fill color for Category)

Max Button x = grid column size
Max button y = grid row size
Max Position is calculated based on grid size and button size
Max Row = Grid Row Size - Y Button Size + 1
Max Column = Grid Column Size - X Button Size + 1

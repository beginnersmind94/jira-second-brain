# NXT-15514 — Tag Framework

## Description
As a platform, I would like to offer the ability for module owners to be able to assign tags to module entities they identify/define and have these assignments persisted and subsequently searched/returned.

## Acceptance Criteria
* A framework shall exist that will allow the modules (Front Office, Back Office, etc..) to define which of their business entities (Items, Recipes, Accounts, …) they would like to be “taggable”.
* The framework shall be data-driven.  This means that the modules shall be able to identify which business entities they will allow to be tagged at run-time.
** _The objective of this data-driven framework is for the platform to build it once and the usage of the framework by the modules can be done via configuration and/or data changes versus requiring the platform to implement code changes every time the modules want a new entity to be marked as “taggable” or not._

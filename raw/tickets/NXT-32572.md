# NXT-32572 — INV - Orders - Create Quick Receipt - Update for Pack Size (Receive screen)

## Description
As a Site Manager I want to see all the orderable units within an item’s pack size if I ordered multiple units so I can create a Quick Receipt exactly as I received the item. 

*Requirements*

* Update the Quick Receipt receive screen to reflect the new pack size configuration

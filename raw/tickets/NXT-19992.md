# NXT-19992 — MP- Menus- Menu Page unsaved changes Confirmation (UI & QA)

## Description
As a user, when I make a change to ANYTHING on my menu, I would like to make sure the system lets me know that changes have not been saved if I try to navigate away from the page. 

(Specifically for the Menu & Menu Planner Pages)

## Acceptance Criteria
Pop to display for user to ask if they are sure they want to navigate away from page if any changes are made on the page that have not been saved. 

UI component team has a new component for all the application to apply this unsaved change confirmation. (Get with UI component team for component)

Check if implementation can be added on the current pop up for menu planner page.

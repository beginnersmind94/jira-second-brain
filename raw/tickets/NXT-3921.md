# NXT-3921 — [AcctMgt] - Preview - Transaction History

## Description
*User Story*

As a district administrator I want to see the Transaction History of a particular patron without having to dig through individual sessions.

*Mockup*

NEW: TBA

*Requirements*

* Add an interaction to the ‘Transaction History’ of the Preview menu.
** Box spawns showing a grid of recent transactions for the account.
** Grid can be sorted, filtered, can be exported.
** Grid columns
*** Transaction Date
*** Sale Date
*** Site
*** Description
*** Debit
*** Credit
*** Balance
* Add buttons to the grid:
** View Transaction
*** This should show more details on the transaction.
**** Menu Items, payment amounts, adjustments, etc.
** Adjust Transaction
*** This should allow the user to make changes to the transaction.
*** Change payment amount, menu items, etc.
*** Finer details will be covered in another story.

## Acceptance Criteria
Transaction History box spawns when the QI Menu is interacted with.
Shows a grid of transactions if available with the columns above.
Grid results can be exported, sorted, etc.

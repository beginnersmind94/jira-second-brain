# NXT-7171 — Application Details - Documents (View/Upload)

## Description
*Story* 

As an apps processor, i have the ability to view and upload documents as part of an application.

[https://www.figma.com/proto/Y3tlwN2yvRc67304adzyyB/Eligibility-3.0?node-id=7766%3A23666&scaling=scale-down-width&page-id=7766%3A23665&starting-point-node-id=7766%3A23666|https://www.figma.com/proto/Y3tlwN2yvRc67304adzyyB/Eligibility-3.0?node-id=7766%3A23666&scaling=scale-down-width&page-id=7766%3A23665&starting-point-node-id=7766%3A23666|smart-link] 

*Requirements*

Documents section in application details: 

Actions

* Add (Upload)
* Delete
* View History

Add Drawer

* Multiple files can be uploaded
* Comment can be added for each one

Deleted items are still shown in the grid, but only the History action is available.

History

* Show the upload date
* Show the delete date

## Acceptance Criteria
User can see documents tied to that particular application.
User can remove existing documents but will need to enter a comment as to why.

User can upload new documents. 
Documents are stored appropriately.
Uploads adhere to standard for file extensions and sizes.

User can download / view existing documents

# NXT-10941 — Meal Exceptions - Resolve Eligibility Meal Exceptions

## Description
When to make claims accurate when meals are recorded at an eligibility rate and the eligibility for a patron is different than the eligibility of the meal I want to be able to identify and change the eligibility of the meal if needed.

The transaction item is updated to the new eligibility price when the save is selected.

## Acceptance Criteria
Selecting the Check keeps the meal eligibility status and price as recorded.
Selecting the double arrows changes the eligibility status and price of the meal to the approved status and price of the patron.
When a transaction status is changed the students account meal is updated to reflect the change.
When save is selected display "Meal eligibility changes have been applied'
  If it fails ''Selected eligibility variances did not update properly. Please verify the changes.'
Selections can be unselected before Save is selected.

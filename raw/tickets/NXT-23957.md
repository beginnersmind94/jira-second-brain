# NXT-23957 — INV - Management - Contract Items - Update Lead Time

## Description
As a District Admin I want an Update Lead Time process so I can quickly update Product Lead Time for many contract items at once.

*Requirements*

* Provide an action on the Contract Items page ([NXT-22306|https://primeroedgenext.atlassian.net/browse/NXT-22306]) to update the Product Lead Time for all selected items
* Lock down the dropdowns 
* Display the item list, but change the list/columns to a dedicated screen for Lead Time
** Column list:
*** Item #
*** Item Description
*** Vendor Item # 
*** Product Lead Time
**** load each field in editable mode with its current value
*** Save button
**** to save individual items
** Provide a Save All button for mass updates
* Update the Product Lead Time value for each item with an entered value
* Provide a ‘Complete’ action to return to the searchable main version of the Contract Items page

# NXT-7548 — INV - Physical Inventory - Supervisor View - Items tab - Discrepancy list (Pending Reconciliation)

## Description
As an Inventory District Supervisor I want an option to view inventory items by discrepancy quantities so I can review all discrepancies on the inventory.

This will be an optional view on the Items tab of the Pending Reconciliation screen.

When user selects the ‘Discrepancy’ view radio button, the item list will update the displayed columns

!image-20201224-212824.png|width=91.66666666666666%!

Item List:

* Same as regular Items view:
** Storage Category
** Item Category
** Item #
*** searchable header
** Item Description
*** searchable header
* Additions:
** Physical Quantity
*** Whole Units
*** Broken Units
** Perpetual Quantity
*** Whole Units
*** Broken Units
** Discrepancy
*** Whole Units
**** searchable header 
**** also has the filter option to select ‘Greater Than’, ‘Equal To’, etc.
***** this search would need to work by absolute value (i.e. +20 cases or -20 cases would both be returned by the search)
*** Broken Units
**** searchable header 
**** also has the filter option to select ‘Greater Than’, ‘Equal To’, etc.
***** this search would need to work by absolute value (i.e. +20 cases or -20 cases would both be returned by the search)
*** Dollar Amount
**** searchable header 
**** also has the filter option to select ‘Greater Than’, ‘Equal To’, etc.
***** this search would need to work by absolute value (i.e. +20 cases or -20 cases would both be returned by the search)
*** Percentage (Quantity)
**** searchable header

## Acceptance Criteria
Grid columns will change when user selects Discrepancy view radio button
Columns will be searchable with filters
Discrepancy column filters will work according to absolute value (returning a number regardless of positive or negative)

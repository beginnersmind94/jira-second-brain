# NXT-50452 — DC Import – Edit Update

## Description
As a SchoolCafé Admin I want to not be able to change the DC Configuration Type for DC Import so that I can avoid messing up the import configuration.

*Requirements*:

* For Imports & Exports > Files > DC Imports > Edit, do the following:
* Make Configuration Type read-only when editing so that they can’t mess up the configuration
* Only applies to Edit, not Add New
* Applies to any DC Import Configuration Type

## Acceptance Criteria
* User cannot edit Configuration type in DC Imports Edit screen

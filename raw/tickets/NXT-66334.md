# NXT-66334 — ACC - Attendance Feature - Daily Entry - Main Grids and Filters

## Description
To accommodate a more detailed tracking Attendance Daily we will be providing a daily entry method.

* Implement separate tracking tabs for SNP, CACFP, and SFSP counts.

Main Page for daily entry
- Implement individual grid storage tabs for SNP, CACFP, and SFSP
- Provide a Site filter (Allow multi-site select for site control)
- Academic Year filter.
- Date Filter (Standard Date filter) (date range max = 31)
- Provide an apply button to update grid

Shows when entry method “Daily” is selected

For the daily grid view

* Display Main Grid Header that show “Daily [Program] Attendance Entry For [00/00 - 00/00]

* Display columns for Site(combined with site code) and Date for selected range (displaying abbreviated day name)
* Display the current week Starting with Sunday by default.
* Grid “day” column area expands or shrinks depending on date selection (up to 31 days)
* Grid has horizontal scrolling for days if grid gets too large
* Grid supports pagination for sites (default 10)
* Provide a “Highlight missing counts” check box where when checks stylizes the borders of check boxes with null values in a bright red. (default unchecked)
* Provide a “Include weekends” which when checked displays all Saturdays and Sundays contained within selected date range (default unchecked)
* Provide a grid export button that simply exports current grid view to excel
* On Page load Text boxes are immediately editable
* Stylize current day to be highlighted with Bright Border around the Major column Cell(shown in mockup)
* acceptable input (whole numbers only, zero is acceptable, limit to 6 digits)
* {color:#ff991f}New after PO review* Add the visual change indicator same as monthly view (shows red dot when change is detected){color}

# NXT-41959 — IM - Recipes & Menu Items - Contribution recommendation for Menu Servings

## Description
As a menu planner I want the menu item contribution recommendation to consider the primary serving components so I can scale from those values when applying contributions to additional menu servings.

**Note:* UI side is dependent on changes from [https://primeroedgenext.atlassian.net/browse/NXT-41301|https://primeroedgenext.atlassian.net/browse/NXT-41301|smart-link]

## Acceptance Criteria
* Provide a secondary Contributions recommendation formula which calculates based on the primary/default menu serving size instead of the sub-ingredients
** This secondary recommendation would only be used when the primary serving has its own contributions set, either by recommendation or manually
* Calculate the recommended component values by scaling the original values according to serving size weight/volume
* Update the Contributions slide-out Recommendation label to indicate whether the recommendation is from ingredients or from original serving
** *Ingredient:* “This Recommendation is based on the calculated total of all added ingredients.”
** *Default Serving:* “This Recommendation is based on the Default Serving and is scaled according to serving weight.”
* If a Menu Item has only manual contributions assigned to serving sizes, any new serving sizes should recommend based on whichever serving is identified as the *Default Measure*
* Multi-Item recipe should have the same rules and contributions logic as custom menu items

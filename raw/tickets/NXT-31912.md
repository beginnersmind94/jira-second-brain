# NXT-31912 — Roles and Permissions - Allow Row Selection to Work in R5

## Description
In Release 5, while browsing through roles, the user needs to click on very specific locations within a row in order for the permissions to display. This was fixed in R6.1.  This epic is to ensure that the changes that allowed this to work in R6.1 be pushed to R5.

## Acceptance Criteria
* As a user in Release 5, I would like to be able to click on any location within the row of a role in order for the permissions to display.
** This was fixed in R6.1.  As such, this is a request to push the changes to R5.

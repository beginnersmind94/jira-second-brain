# NXT-18424 — On the Menu Planner page when we click on the Recent searched item , it expands in such a way that it hides the other two "Frequently used" and "Favorites" options. This is done on 100 percent window mode

## Description
Go to the menu planner page by clicking menu planner under Menus
now on the left panel of the page we see a search bar and 3 items "Frequently used"
"favorites"
"recently searched"
click on the "Recently searched"

Expected Result:  The item should expand but other options should also be visible to user
Actual Result :  It expands in such a way that it completely hides other two options

Extra Details:
The window is used in 100 percent mode. If we use window on 60 percent mode then we are able to see other options but the space "Recently searched" takes while expanding is way much more

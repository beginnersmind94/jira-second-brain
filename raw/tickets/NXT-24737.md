# NXT-24737 — POS - Dynamic Menus - Build-a-meal graphics and Transaction item

## Description
As the cashier I want to see the progress of a meal being built as I add menu items.

* When a patron is selected the Componet graphic is cleared.
* As menu items are added the graphics fills showing the categories added. 
* If items are removed the graphic updates to show any categories removed.

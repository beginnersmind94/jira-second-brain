# NXT-28846 — Platform - Special Permissions enablement for FO modules

## Description
As a Primeroedge admin, I should be able to view list of special permissions available to enable/disable respective modules from roles & Permissions page

Special permissions take precedence over page level permissions

List of permissions for Front Office modules:

* ELIGIBILITY_PROCRESS_REFUNDS
* VERIFICATION_SCHEDULE_SAMPLE

Both special permissions are just enable/disable buttons on respective screens, no other parts get impacted for this permissions.

## Acceptance Criteria
* User shall be able to see list of “special permissions” available from roles & permissions page under Eligibility & Verifications modules
* User shall be able to see the description of the special permissions list available for the role
* User shall be able to enable/disable special permissions from any given role
* User shall not be able to see list of special permissions if the user do not have admin rights.
* User shall be able to cancel or save the changes of the special permissions with confirmation prompt for cancel and information message for save
* User shall not be able to use functions/buttons (Process Refunds & Schedule samples) from respective module pages.

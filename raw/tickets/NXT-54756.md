# NXT-54756 — PE - GC DB - USDA-Populate Missing details from other Data Sources 

## Description

## Acceptance Criteria
Navigation: MP - Item Catalog
Scope: only Allergens, Meal Credits and Images
Changes Required:

The allergen, Meal Credits and Images should be matched based on last 10 digit (Right end) of the GTIN from USDA Branded Foods API or GDSN Connect Table

If the match is found in GDSN Connect table, fetch the Images, Meal credits in Translated format, and Allergens in translated format

If the match is not Found in GDSN Connect Table, then look for GDSN Connect SC Table as well

Once the match is found, populate the information in Allergen section, load the images in images section

During the import process, include additional elements identified so that it can be imported to local items

Meal Credit information is not shown in UI, but it gets imported and shown in Nutrition Section

Upon creating local items, the details should be reflecting in Meal Credits ( Nutrient , Allergen Tab and Documents (Images)   


Regression:
Search an Item, View an Item, Import an Item, and Audit Logs

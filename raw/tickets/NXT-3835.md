# NXT-3835 — Create new Custom KPIs

## Description
*User Story*
As an Insights user, I can create a new Custom KPI using existing Metrics, Default KPIs, and other existing Custom KPIs.

*Mockup*
[https://qvv7ze.axshare.com/#id=6dmlcp&p=new_kpi|https://qvv7ze.axshare.com/#id=6dmlcp&p=new_kpi] (Mockup created before having visibility into existing 2.0 mockups)

*Requirements*

* Users must enter a Name into a text field
* Users must enter a Description into a text/paragraph field
* Users use provided lists of Operators, Metrics, Default KPIs, and Custom KPIs to build a Calculation.
** Only enabled Measures will show up.
** Manual type in.
** Drag and drop?
** Double click to add at cursor
* Validate button to ensure that the Calculation makes mathematical sense and will not error.
* Checkboxes to define the reporting Level of the report (*who* can be measured by this KPI).
** District
** Site
** Employee
* Users are prompted when attempting to navigate away from this page with unsaved changes.

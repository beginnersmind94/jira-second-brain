# NXT-10964 — [TaggedDashboard] - Dashboard (Insights) - Account Management - Enrollment Activity Widget

## Description
As an account administrator, I would like to be able to easily analyze account activity for a given date range, so that I can immediately correct any obvious issues that may have come up in the data exchange process.

*Summary*:

On the Dashboard, add a widget that allows a user to see the number of enrolled, withdrawn, and transferred students in the district for a set number of date ranges.

*Mockup*:

[Link|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=422818%3A277812&scaling=scale-down-width]

*Details*:

* In the Account Management module Dashboard, add a widget for *Enrollment* *Activity*
** This widget should show the count of all +student+ accounts in the district who have been:
*** Enrolled: Those students who were added to the district (whether manually in Accounts or through an import file)
*** Withdrawn: Those students who were removed from the district (whether manually in Account Details or through an import file)
*** Reenrolled: Those students who were previously deactivated, and then reactivated (whether manually in Accounts or through an import file)
*** Transferred: Those students who were moved from one site in the district to another (whether manually in Account Details or through an import file)
** Note that these widgets do *not* include users or adults, _only_ +student+ accounts.
* Add view options (if not already handled in a prior story): *Today*_(default selection)_*, Last 7 days, Last 30 days, Year to date*. 
** Data should be updated whenever the user chooses a new date range.
** Last 7 days & Last 30 days should include today in their count.
** Academic Year to date = Count of records between academic year start date and today

## Acceptance Criteria
This story is complete when a user can navigate to the Account Management Dashboard and view an Enrollment Activity widget for their district which contains a count of every student in the district that has been enrolled, withdrawn, reenrolled, or transferred, with the count being adjustable based on 4 different date ranges.

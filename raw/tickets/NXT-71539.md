# NXT-71539 — SC - MVR - Sessions

## Description
As a Director, I want to distribute managed view reports from the Sessions page so that I can create and send reports directly from that grid.

----

h2. *Description*

Add the *Distribute* action to the ellipsis (⋮) menu for Managed Views on the *Sessions page* under the Accountability module.

This functionality should behave exactly the same as the existing *Managed View distribution functionality on the Users grid (System module)*.

----

h2. *User Flow*

# User navigates to:
#* *Accountability → Sessions*
# User views Managed View grid
# User clicks the *ellipsis (⋮)* on a managed view row
# User selects *Distribute*
# System follows existing Managed View distribution flow

----

h2. *Out of Scope*

* Any changes to distribution framework
* Any changes to scheduling or delivery logic
* New UI or configuration flows

----

h2. *Requirements*

h3. *Ellipsis Menu Update*

* Add *Distribute* option to the ellipsis menu for Managed Views on the Sessions page

----

h3. *Functionality Reuse*

* Distribute action must:
** Use the *existing Managed View Reports distribution functionality*
** Follow the same behavior as implemented on the *Users grid (System module)*
* Do NOT create new distribution logic

----

h3. *Permissions*

* Permissions must follow the same rules as the Users grid implementation:
** Users with appropriate permissions can access and use Distribute
** Others see it disabled or not available (consistent with existing behavior)

----

h3. *Report Creation & Management*

* When a Managed View report is created from the Sessions page:
** It must appear in the *Managed View Reports dashboard*
* Once created, it must:
** Behave exactly like any other Managed View report
** Support all standard actions (edit, distribute, schedule, etc.)

----

h3. *Consistency*

* UI behavior, labeling, and interaction must match:
** Users grid (System module)
** Other Managed View distribution entry points

## Acceptance Criteria
* Given a user is on the Sessions page, when they open the ellipsis menu, then the Distribute option is available
* Given a user selects Distribute, when the action is triggered, then the existing Managed View distribution flow is executed
* Given a report is created from Sessions, when saved, then it appears in the Managed View Reports dashboard
* Given a report exists, when accessed from the dashboard, then it behaves the same as any other Managed View report

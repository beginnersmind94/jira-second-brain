# NXT-4096 — IM - Edit - Nutrient Information

## Description
*User Story*

As a Item Management Super User I can edit nutrient information from a record already created.

*Requirements*

* Header
** Option to ‘Edit’ 
* Slides the category from the right to left as a ‘drawer’ showing the information that can be edited and added / removed
** If clicking out of the popup and information is entered and not saved, show message
*** You have unsaved information, do you want to 'save' or ‘continue without saving’?
**** Save - Save the information entered and close the record
**** Continue without saving - Close the item
* Allow the user to modify all information on this page
* If clicking cancel, then revert data
* If Nutrient Information is using a serving size, user cannot remove the serving measurement
* Logic to match ‘add new’ for editing the item as well

## Acceptance Criteria
* Nutrient information for Serving Measures, Nutrient Information, and Contributions shows up here
* If Save, close the record after saving the information
* If cancel, then revert back to the state before user clicked 'Edit'

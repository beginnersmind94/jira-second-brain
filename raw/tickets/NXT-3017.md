# NXT-3017 — System - Licensing - View Licensing Summary

## Description
As a Cybersoft or district administrator, I would like the ability to view the currently assigned district and site licenses, so that I can make sure that the district has access to everything within PrimeroEdge that has been purchased.



*Note:* The items below may be/have been already developed by the Platform team, in which we just need to integrate the Licensing APIs with login so that user gets access to the modules that he’s licensed to. Additional stories may be needed.



* Within the Configuration category, add a *Licensing Summary* page
* When the user selects the operation, redirect them to this page:
* Page content can be carried over from existing solution, with slightly more organization (see wireframe below)
* The user should be able to view the State, District, Site, and Device licenses belonging to their district at a glance
** State & District Level section should sort by Module Name in alphabetical order.
** Device and Terminal sections will be combined. For example, the section will include Vending licenses as well as Intelliscan licenses.
* User can see full license details + features within the module and then toggle on/off at a module *or* feature level. (*Separate story*)
* They can also request new modules (see Jira “Discover Applications” page). *Can be ignored for now as this will be a phase 2 item.*
* The page should no longer contain a button to view/modify the actual licensing key for Cybersoft administrators. *The ability to apply a new license will be covered in a separate story.*
* This story is complete when full page is done (including search functionality), 1st tab is complete, and 2nd + 3rd tabs have summary data.

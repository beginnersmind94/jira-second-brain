# NXT-14017 — IM - Recipe - Show Manual or System Generated Label for Allergen info 

## Description
As a User I would like to see a label next to Allergen info as (System generated or Manually entered) so that it’s easy to understand whether the changes made in the View/Edit page are System generated or Manually entered)

## Acceptance Criteria
Given the User is on the View/Edit page for the Allergen Info 

When any changes are made to the Allergen info page 

Then reflect the Label next to Allergen info on details page as

If Manually entered for any Manual changes 

And System Generated for any System generated changes

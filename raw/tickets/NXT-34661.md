# NXT-34661 — Import/Export Monitor - Disable View Log Button When No Data

## Description
Don’t enable the “View Log” button when there is no data to show.  Currently, a user can click into the screen when a job is running, but thee is nothing to show the user.

!image-20220928-010031.png|width=1649,height=508!

## Acceptance Criteria
* The “View Log” button will only be enabled when a job is complete or there is data in the log to view

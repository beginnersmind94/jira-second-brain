# NXT-36800 — IM - Recipe Rework - Split View Search Panel - Disabled State

## Description
As an Item Management user I want the split view search to be disabled when I’m making edits so I don’t accidentally click away from the item/recipe on which I am working.

*Requirements*

* Extension of [https://primeroedgenext.atlassian.net/browse/NXT-36766|https://primeroedgenext.atlassian.net/browse/NXT-36766|smart-link] 
* Maintain different interactive states (fully disabled, or search enabled but item selection disabled, etc.) which can be used when the user is in Add/Edit modes on the details section
** e.g. lock and disable the side panel while user is creating a new item/recipe
** these scenarios will vary and will need to be defined on the item/recipe section business rules
* Disabled state examples:
** Edit Recipe (new flow)
*** Editing summary card
*** Editing Steps & Ingredients
*** Editing Images (less important)

# NXT-60893 — INV - Distribution - Review - Additional orders after Assignment started

## Description
As a Distribution Manager I want to see when sites have placed orders to a specific day after I have begun assignment, so I can decide whether to add the order(s) to the current Distribution or push to a new one.

h3. Review tab after Assignment begins

* Stretch goal: change Action column icon color to gray once the distribution is Assigned

## Acceptance Criteria
* Display an alert icon when an order is placed for a day on which the distribution has already begun Assignment
** display on the calendar (done already)
** display on the Review tab
* Add the order to the Review tab separate from the orders already assigned
** technically could be multiple orders
* Provide available actions regarding the new orders
** Add to Distribution
*** this will reset Assignment
*** prompt user: “This will reset the Assignment process. Are you sure?”
** Create New Distribution
*** this will create a separate Distribution containing these orders
*** prompt user: “This will create a new Unassigned distribution for the new orders. Are you sure?”
*** this new Distribution will be Unassigned
*** these orders will be removed from the current distribution

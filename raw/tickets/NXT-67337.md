# NXT-67337 — GC > Upload Items in SC > Add columns for Nutrients, Allergens , Meal Credits, and Stock Item Info in the results grid and display the Y/N value for each item in the grid.

## Description
*As a* Menu Planner or Inventory Manager, *I want* to view whether each catalog item has corresponding Nutrient Info, Allergen Info, Meal Credits, and Stock Item Info available *So that* I can quickly identify which items have complete data and make informed decisions when selecting items to import into my local item catalog. Sorting and filtering options on the columns headers will help us a lot in picking the items for import.

## Acceptance Criteria
* The Item Catalog search result grid displays four new columns after Item Name:
** Nutrients	
** Allergens	
** Meal Credits	
** Stock Item
** Adjust the columns to include these new columns and avoid the horizontal scrollbar in the grid for PrimeroEdge.
* Each column displays values as:
** “Yes" – if information is available for that item.
** "No" – if information is not available.
** New columns should display value for all search criteria, Search by all data providers, search by on e specific data provider option or when show previously imported items are selected. 
* Sorting & Filtering
** All columns supports sorting (ascending/descending ).
** Filtering by columns headers is added for new columns and for these existing columns Product code, Item name, GTIN, Data source name. 
*  Hide the importable item column. 
* New columns should be available in School Café Views functionality  for add or remove columns functionality. 
* New columns should show up show previously imported items is selected.

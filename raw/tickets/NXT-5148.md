# NXT-5148 — System - Program Configuration - Add Reimbursement Rates tab

## Description
As a district administrator, I would like to be able to manage my district’s Performance Based Incentive as well as their Federal, State, and Administrative rates in one location, so that I can quickly view and update our financial  status for the current or upcoming school year.

Summary: Allow a user to be able to view the ‘Reimbursement Rates' menu within the Program Configuration page, as well as navigate between 4 tabs.

Details:

* Within the Program Configuration page, beneath the ‘Sites’ option, add a new menu option for *Reimbursement Rates*.
* In this page, carry over the same four tabs from the Classic page:
*# Standard (encompasses Federal & State tabs from Classic)
*# Administrative (formerly called ‘Administrative Rates’ but we can shorten it)
*# Performance Based Incentive
* The functionality for those tabs will largely remain the same, but will be handled in individual stories for each tab.

Mock-up: [https://nqyc9v.axshare.com/#id=iwefuj&p=pos_configuration&c=1|https://nqyc9v.axshare.com/#id=iwefuj&p=pos_configuration&c=1]

Classic location: /POS/ReimbursementRates.aspx

(System → Configuration → Reimbursement Rates)

## Acceptance Criteria
Verify that a user is able to view the ‘Reimbursement Rates' menu within the Program Configuration page

Verify that the user can view as well as navigate between the 4 tabs with the names provided above.

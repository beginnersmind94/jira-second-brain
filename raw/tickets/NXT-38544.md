# NXT-38544 — IM - Items - Units - UI adjustments

## Description
As an Item Management user I want the Units card to have clean and cohesive experiences for both view and edit mode so I can see all my serving and pack details in a clear way across all item types.

*Requirements*

* Make the header cards shorter vertically
** only 4 rows of height should be needed (in the summary card)
** Use inline scroll on Units card if needed
** Image card can increase height on edit mode until we move the checkbox and Add Image button
** Edit mode of Summary card will also slightly increase height because of Tags field
** The cards can be adjusted horizontally as well to make more space for Category and Brand fields in Summary card
*** Units card can be much less wide, at least in view mode
* +Units card+
* Move the P/S identifier to the identifiers column, as the first icon
* Provide nesting for sub-qty in view mode
** Sub-Qty value moves to the nesting line (only exists as a column for edit mode
** each child unit is indented to display 
* Fix identifier icons
** icons should not increase row height
** all icons should be same size and horizontal alignment
** Star icon for Preferred Measure should only display on the selected preferred measure in view mode
* Fix font size – should not be smaller than rest of page
* Merge Amount + Description columns in view mode
** hide '1' for sub-qty linked serving units
* *Split View*
** Adjust widths accordingly
** Default Image card to collapsed mode
** Leverage Category/Brand toggle in Summary card

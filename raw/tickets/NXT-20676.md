# NXT-20676 — INV - Inventory - Additions - Create Addition - Items (Step 2) - Add Items

## Description
As a Site Manager I want an Add Items tool so I can search and add items to my inventory addition.

This is the second step in the Add to Inventory process, after user navigates from the first step ([NXT-1778|https://primeroedgenext.atlassian.net/browse/NXT-1778])

*Requirements*

* This is the second step of the Inventory Addition stepper tool
** title this step ‘Items’
** display the stepper ‘Previous’ button to navigate back to step 1
* There are 2 main sections on this screen:
** Add Item control (this story)
*** the tool which is used to find and select items to add them to the list
** Item list ([NXT-20677|https://primeroedgenext.atlassian.net/browse/NXT-20677])
*** the list that contains the items which have been added to the list
*** this will also include the *Lot Number sub-grid* for each item in the list (the same thing used on Receipts)
* Add Item section
** Rather than providing a dedicated screen for searching items, we want to use a lightweight control so we can keep the user on the same screen for both adding items and viewing the added items
** This will be the upper section on the page
** Display the following from left to right
*** Item Category dropdown
**** default to ‘All’
**** allow single selection
**** selections from this dropdown will filter the results in the main Item dropdown
*** Item dropdown (may be Smart Search instead)
**** display Item # and Name 
**** this will list all items with a site-item record
**** this will be searchable 
**** single selection
*** Quantity textbox
**** user must type some number
*** Add button
**** enabled once a single item is selected and there is a value for the Quantity field
** Once an item is added, it will load in the Items List ([NXT-20677|https://primeroedgenext.atlassian.net/browse/NXT-20677])
*** Reset the selections and disable the Add button
*** Remove that item from the Add Item list

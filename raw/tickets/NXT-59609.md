# NXT-59609 — INV - Vendors & Contract Items - Update VI# - Autofill from Item #

## Description
As a Distribution Manager I want the option to skip or autofill Vendor Item # so I can skip unnecessary configuration for internal vendors.

!Vendor Items - Autofill VI-20250301-052343.png|width=66.66666666666667%,alt="Vendor Items - Autofill VI-20250301-052343.png"!

## Acceptance Criteria
* Provide an option to autofill Vendor Item # on the Update Vendor Item # screen
** *ONLY* for internal vendors
* When Autofill button is clicked, prompt the user
** ‘Do you wish to use the Item # for all items missing Vendor Item #?’ *Cancel/Confirm*
** upon Confirm, autofill all *blank* VI# fields with the Item #
*** remove LI prefix
* If multiple pack size units are present, assign a suffix to the lower units
** append ‘BU’ to the end of the VI# for the lowest unit
** append 'M' to the end of the VI# for the middle unit in the (unlikely) case that a 3-tier hierarchy is on the contract
* Examples:
** 1234 (highest unit or only one unit)
** 1234-M (middle unit)
** 1234-BU (base unit)

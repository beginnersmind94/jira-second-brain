# NXT-37580 — Student Import – SiteCode Code Mapping - Dev

## Description
As a SchoolCafé Admin I want to be able to map site codes so that I can avoid import failures.

*Notes:*

* This comes from importing Salem
* They have SiteCode “abbreviations”, such as instead of putting in the full code like 2980052, they would put just 52
* They would like to be able to go in and put 52 maps to 2980052

*Requirements*:

* When importing a Student Import, do the following:
* Make the following changes to the Site Code field:
* Add the elipses showing that the use can do code mapping
** On click, open the code mapping drawer for SiteCode
* In the drawer, show the standard UI for Code Mapping
** The values for Internal Code comes from the list of Site Codes the user should have uploaded already
** If there are no values, then they have not uploaded the list yet
** Therefore, show them an error message saying “School codes have not been uploaded yet. Enter school codes first and then return to this screen.”
* This validation should be for both manual and automatic imports

## Acceptance Criteria
* Users can map site codes against the site code list

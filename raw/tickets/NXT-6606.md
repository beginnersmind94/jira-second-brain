# NXT-6606 — INV - Update Vendor Contact/Address information

## Description
*User Story*
As a Procurement Specialist, I want an edit option for vendor address info so I can make updates to my saved vendor address and contacts. 

*Requirements*

In the view vendor info screen, user can click the edit icon on the top right corner of Vendor Address or Contact Info card to open that particular section for editing.
Since the Address and Contact information are on the same screen, clicking the edit icon on either card will open the same screen.

!image-20201022-140611.png|width=91.66666666666666%!

When the edit icon for either Address or Contact card is clicked, it will open the vendor address/contact screen.

User will be able to edit that section alone and save it.

!image-20201022-140657.png|width=91.66666666666666%!



Upon saving, user will be returned to the [view vendor info card screen.|https://primeroedgenext.atlassian.net/browse/NXT-6049]

Story 2 - Edit Vendor Address / Contact cards
API - 1
UI - 2
QA - 1

## Acceptance Criteria
User can click edit icon for Vendor Address card.
User can click edit icon for Vendor Contacts card.
User will access and edit screen for vendor address/contact attributes.
User can update editable fields or add new contact.
User can Save changes or Cancel.

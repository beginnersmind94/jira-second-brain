# NXT-5327 — Item Management - Add New Food Item - Product Info - Skip for now

## Description
As an Item Management User, I want to have the skip for now functionality added to the Product Info as the last step in the 'Add new item' step for a Food item

Mockup

!image-2020-08-27-07-26-54-375.png|width=100%!

 

Requirements

* Need to have skip for now and clear
* Follow the same skip for now and clear functionality as on the other pages.
* When clicking ‘Clear’ it will ask for a confirmation if there is information filled in. If the user confirms, then clear the information filled in below.
* If there is no information filled in below, the user can select ‘skip for now’
* When checking the box for ‘skip for now’ then collapse the section so the user can’t input any information.

This is NOT for non-food, just for food items.

## Acceptance Criteria
As a user, I should be able to use the skip for now for Product Info

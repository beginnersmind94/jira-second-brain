# NXT-34198 — IM- Items, Recipes, Menu Items - Add Sesame as a Standard Allergen on UI

## Description
Due to a new law that was passed on April 23, Sesame will now be considered as a standard allergen on January 1, 2023. However, Columbia is now in the process of starting to input their items and would like to have Sesame already configured so they could indicate allergens for those items

*Note:* this is part of several related stories all required ([Report story|https://primeroedgenext.atlassian.net/browse/NXT-33583], [Import story|https://primeroedgenext.atlassian.net/browse/NXT-33584])

*Requirements*

* Add Sesame as a Standard Allergen in all relevant places in the UI
* This must be updated on all View, Edit, Add screens for the three main IM types (Items, Recipes, Menu Items)

## Acceptance Criteria
* Include Sesame as a Standard Allergen for users to indicate on *items* in the Item Details page
* Include Sesame as a Standard Allergen for users to indicate on *Recipes* in the Recipe Details page
* Include Sesame as a Standard Allergen for users to indicate on *Menu Items* in the Menu Item Details page
* New row needs to be added in the same fashion as other standard allergen as an extra option

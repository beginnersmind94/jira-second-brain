# NXT-3783 — Create Selected Filter Window

## Description
As filters are entered or removed the Selected Filter window will be populated with the individual filter values. Each filter value can be removed individually without having to go to the section that contains the filter. 

!image-20200427-172215.png|width=289,height=165!

## Acceptance Criteria
The Selected filter window will be positioned at the top of the advanced search frame.

The SF window will expand as filters are added, but will be no larger than 4 lines of buttons visible at one time.

When there are more lines than are visible the SF window is scrollable.

As filters are added a button is added to the SF window that can be selected to remove the filter.

The buttons are variable size.

When a button is selected it is removed from the filter criteria

Range buttons will be shown as  
    '>= Value' if only a lower range is selected
    '<= Value' if only an upper range is selected
    ' lower Value - upper Value' if both range values are selected
    'As of - Value' if an As of date is selected.

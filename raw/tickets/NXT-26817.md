# NXT-26817 — IM - Recipe History (Nutrient Details)

## Description
As Item Manager administrator I need an option where I can observe the history of a Recipe’s Nutrient Details
for example updates made to the item by who and when so that have an audit process

## Acceptance Criteria
*Entry Point*

* Item Management → Recipe → Selected Recipe → Nutrient Details -> History icon
* An option to view Recipe' s Nutrient Details  history shall be available to the user \[ history on ]
* User shall be able to see all Recipe' s Nutrient Details  property values that were updated
* System date and Time on which the update took place
* User name which performed the update (who performed the update)
* What fields that were updated, preview value(s) and new value(s)
* The information is for display purposes (Shall not be alter by the user)
* Applies to ADD and EDIT item

*General*

* Page pagination shall be present (as standard way)

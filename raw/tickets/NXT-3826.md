# NXT-3826 — Account Management - Advanced Search - Demographics Section - Ethnicity Subsection

## Description
The Ethnicity Subsection will allow the user to filter the patrons list by their Ethnicity. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Ethnicity values. The user can type in the name or a partial name. The Names are listed in Alphabetical order. As names are selected they are moved to the top of the list. Unselected names are separated from selected names and are shown in alphabetical order.

!Screen Shot 2020-04-29 at 2.27.19 PM.png|width=279,height=335!

## Acceptance Criteria
The filter values are populated from the DB table

There is a search field that allows the user to search for a name

The search field shows partial matches.

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Ethnicity section they are 'or'ed together.

When a filter value is selected the Ethnicity Subsection count is increased by 1 and the Demographics Section count is increased by 1.

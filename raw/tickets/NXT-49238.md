## Acceptance Criteria

Default Data Provider - 3rd Party Source(s)

User should be able to click on Upload Button and select the updated file (downloaded from GDSN Connect)

The should have basic validation such invalid file format, field mismatch and empty file

Upon successful upload, the data should be uploaded in new table named as GDSNConnect_SC_Updates

The uploaded data should be compared with attributes and show the count of attributes that are different than local data source

The NULL values on the updates should not be counted as updates

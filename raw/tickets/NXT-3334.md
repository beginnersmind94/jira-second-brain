# NXT-3334 — Inspection Template - Collapsible Categories

## Description
*User Story*
As an Inspections user editing a template, I can collapse Categories to make it easier to edit other categories on the same screen.

*Requirements*
* Categories can be collapsed, hiding all questions beneath the Category
* Collapsed Categories are still displayed
* Collapsed categories can be uncollapsed

## Acceptance Criteria
* Categories are able to be collapsed and uncollapsed with no loss in data

# NXT-15923 — INV - Order Template Details - Items tab - Remove Items

## Description
As an inventory Operations Manager I want a checkbox for each item along with a Remove button so I can remove any specific items from a specific Order Template.

This is a continuation of [https://primeroedgenext.atlassian.net/browse/NXT-14810|https://primeroedgenext.atlassian.net/browse/NXT-14810|smart-link]

*Requirements*

* Within the Order Template Item List, a selection column will be the left-most column
* The select column will contain:
** a checkbox for each item
** a checkbox in the header for selecting all items
*** the Select All checkbox would only select all the items displayed as a result of any search criteria
* Provide a Remove button, to save the items which have been selected for removal from the template
** the Remove button should only be enabled if one or more items have been selected
** Provide a confirmation prompt when user clicks the button: “Are you sure you want to remove these items from the order template?”

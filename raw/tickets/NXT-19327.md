# NXT-19327 — System - Management - Generate Periods - Add 'Inventory' as a Period Type option

## Description
As a food service director I need ‘Inventory’ as an option for Period Type so I can generate Inventory periods for a fiscal year.

*Requirements*

* Add Inventory as a Period Type option to the Generate Periods page in the System module
* Update code so Inventory periods are generated by this page
** Note: Inventory periods have already been developed in the back end but do not have a UI yet
* While creating new periods for a given period, user should NOT be able to select multiple period types
** System shall prompt for user confirmation to remove one of the period type.

## Acceptance Criteria
* User shall be able to see under *Generate Periods* page with *Period Type* as ‘Inventory’
* Users should be able to generate Inventory periods from this page
* Users should be able to edit the periods as they can for POS periods
* User shall be able to select period type as a filter and periods result table shall include “Period Type” as another sortable column next to the name.

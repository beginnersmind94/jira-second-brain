# NXT-2966 — Organization - Regions - Global Sponsor Database

## Description
As a PrimeroEdge organization I want to manage/maintain a global sponsor database for school districts across the nation so that I can alleviate much of the initial startup process.

Utilize the ELSI data that has been published regarding the public information of school districts and sites to create an initial database of sponsors. When a new customer starts, we will use this initial list to pre-populate all of the sponsor information and allow the customer to simply manage certain additional attributes and customizations for the district. 

This significantly increases the speed of onboarding a new customer.

## Acceptance Criteria
1. Store sponsor information for school districts to be used for onboarding purposes
2. Update global sponsor database if new sites are created locally but not stored already in the GSD

# NXT-3550 — Configuration - Edit Method, Source

## Description
With permission, ability to edit, methods, sources

Ability for admin (with permission) to configure Training sources, Training methods 

* Editing current Method, Source populates in the in the field for next event entry
* When making a method or source (required fields) inactive, they will remain in assigned events but not be available for future training events added/edited 
* once status is changed to inactive, the delete option is available (soft delete - remove from UI, store on BE)

## Acceptance Criteria
Admin (with permission) can edit method, source

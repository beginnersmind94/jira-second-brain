# NXT-61348 — INV - Distribution - Shipping tab - Edit Stop Order

## Description
As a warehouse distribution manager I want a tool for adjusting the Stop Order of a shipment so I can adjust the shipment to match what the sequence the driver will be following, and print the Delivery Tickets accordingly.

h3. Shipping tab 

* Need to be able to reorder the stops within shipments

h3. Stop Order Drag & Drop - Idea from Delivery Groups

* If we use this for the DG config, perhaps we could use this same slideout on the Shipping tab

!Delivery Groups - Edit Stop Order-20250502-044759.png|width=91.66666666666666%,alt="Delivery Groups - Edit Stop Order-20250502-044759.png"!

## Acceptance Criteria
* Provide a ‘Modify Sort Order’ icon in the action column of the Shipping tab
* Open a slide-out to manage Stop Order
** Use drag & drop to reorder the stops within a single shipment 
** Use the same implementation from Delivery Groups config
* Provide footer Cancel and Save buttons in the slideout
* Reload the Shipping tab data according to the modified stop order

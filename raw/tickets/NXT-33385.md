# NXT-33385 — MP/PR- Find and Replace & Production adjustment- Allow user to view Menu from Find and Replace Page/ rename production adjustment view to icon (UI Only)

## Description
As a user, When I am finding and or replacing a menu item, there are times I would like to be able to quickly look at some of the menus to make sure that the menu item makes sense if I were to replace it.

As a user, we, currently on the production adjustments, have a view link, we need to change that to a icon to view and allow view in an overlay

## Acceptance Criteria
* Give actions column with view option on both production adjustment and find and replace page
* Give hover for action “View Menu” for find and replace and “View Plan” for production adjustment
* Display menu/ plan in overlay and only allow user to close 
* They can not make any edits to the menu or production plan

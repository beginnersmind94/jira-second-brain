# NXT-46290 — INV - Vendors - Contracts - Make contracts Inactive when contract reaches End Date

## Description
As a Procurement Specialist I want the Vendor Contracts to be made automatically Inactive when the End Date occurs so we do not order items from an expired contract.

!image-20240109-155022.png|width=83.33333333333334%!

## Acceptance Criteria
* Automatically switch the Vendor Contract status to ‘Inactive’ when the contract end date is reached
** this will require a job to check each time the date changes (night job)
* Remove the ‘Marked for Deletion’ status from the Contracts Status dropdown
** ‘Suspended’ status will be only manual selection (never auto triggered)

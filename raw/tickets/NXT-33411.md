# NXT-33411 — District Onboarding - SIS API Connectivity Stepper

## Description
A framework is needed to allow districts to perform self-guided onboarding.

## Acceptance Criteria
* Steps shall be enforced to execute in sequential order
* Each step shall trigger a “wizard” stepper that will walk the user through setup. 
* Upon completion of the wizard, the user shall be returned to main page and the button shall change to the green checkmark to indicate completion.
* Web pages that depend upon the data from each setup activity shall be disabled until the dependency is completed.
* The Freshdesk widget shall be available for linking to articles and videos which will be defined by the content team.

# NXT-63464 — INV - Internal Vendors - Contracts - Shipping Groups - show Schedule for groups

## Description
As an Inventory Distribution Manager I want the Shipping Group slide-out to show me which groups have a shipping schedule so I can make sure I’m using the correct groups for my items.

h3. Delivery Group slide-out w/ Schedules

!Shipping Group Slideout Schedules-20250818-204403.png|width=83.33333333333334%,alt="Shipping Group Slideout Schedules-20250818-204403.png"!

## Acceptance Criteria
* Modify the Shipping Group slide-out to put the Default group at the bottom, below any user-created groups
* Expand the width of the SG slide-out and add a column for Shipping Schedule
** display the *overall* shipping schedule (for all sites)
** Add a tooltip to the Delete icon to inform user that groups cannot be deleted with a schedule associated
*** “Shipping Group cannot be deleted if there is a schedule associated with it.”
* Sort the groups in the Contracts card in the same way they are sorted in the slide-outs (default at the end)
** Do not show the Default group in the Contracts card if there are no items associated with it 
* Prevent Shipping Group deletion if any schedule exists for that group
** tooltip: “Shipping Group cannot be deleted if there is a schedule associated with it.”
* for Default group, remove the Edit and Delete icons instead of showing them disabled

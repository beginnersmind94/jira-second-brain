# NXT-16356 — INV - Vendors - Contract - Add setting for Receipt price

## Description
As a district admin I want the ability to define whether to use Delivery Date or Received Date for retrieving receipt price so I can ensure my receipts match my vendor invoices.

Setting will be applied to receipt logic in [https://primeroedgenext.atlassian.net/browse/NXT-23808|https://primeroedgenext.atlassian.net/browse/NXT-23808|smart-link] 

*Requirements*

* Add an additional selection to the Contract configuration to specify whether to use Delivery Date or Receipt Date for determining item receipt price
** Label:  Receipt Item Price Condition
** Options: 
*** Delivery Date (*default)*
*** Receipt Date
** Tooltip: This setting determines which date is used for retrieving item receipt prices, charges and discounts according to the pricing conditions.

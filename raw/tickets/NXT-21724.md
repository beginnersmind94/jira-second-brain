# NXT-21724 — Platform - Security - Resource Type Function-Category Enhancements

## Description
As a primero edge admin, I should be able to access resource catalog to add a new resources available for any district with functions resource type along with category so that the resource type can be accessed by any user with the assigned role for that resource.

+*Requirements:*+

* When user clicks on “Add Resource Catalog”, we need to re-order “Module Name” to be made available right after “Resource Type” selection
* When a users performs Function-Category-Module mapping and page is loaded with correct permissions based on user profile
* Module Filter drop-down is enabled instead of search filter

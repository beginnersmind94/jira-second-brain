# NXT-1757 — INV - Orders - Review Order Settings - Add New Rule - Site (Step 1)

## Description
As an inventory operations manager I want a Site option on the Add New Order Review Rule process so I can potentially configure order reviews for orders from a specific site.

*Requirements*

* Create the first step of a multi-step process that allows users to configure custom rules for flagging orders as Pending Review
** This step will allow users to configure order review rules for orders that are placed by:
*** a specific site
*** a group of sites or a site type
* Use the stepper component to demonstrate the number of steps and highlight that this page is the first step
** Make each step a clickable component so the user can skip to a specific step
** The label for this step should be ‘Sites'
* Provide a control to select sites (optional)
** list all sites within the district
** organize sites by site type
*** provide a checkbox for each Site Type as well
** allow user to select multiple sites
*** checkboxes 
* Display a Back to Rules List button in the bottom left corner, to cancel and return to the list page
** Prompt user: “Discard rule configuration and exit setup?” Yes/Cancel
* Display a Next button in the bottom right corner to advance to the next step
* Display a Complete button in the bottom right corner, to skip the remaining steps and jump ahead to the confirmation step

## Acceptance Criteria
* Add Sites will be the first step in the stepper
* User can search sites by code or name
* User can select individual sites
* User can select Site Types
* User can Save site selections
* User can skip this step

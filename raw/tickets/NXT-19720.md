# NXT-19720 — API Configuration - Add "Updated Date/By" Columns

## Description
Import Dashboard - Include "Modified By" and "Last Modified" in grid

Additional notes:
On Import Dashboard, when an import configuration is created successfully, we are showing the Added on(Created Date)  and Added By(user) details, 
when it is updated, we are only showing the user who updated in Added/Edited by column, Added on is only showing the date when the configuration was created.
The basic idea for this story is to show when the creation and update happened.
And if we add Created On, Updated On, Created By, and Updated By columns the grid will have more columns which will tend to make it illegible.
So to overcome this, we will show Created Date under Added/Edited On column if there is no update date available.
If the updated date is available, we will show the updated date under Added/Edited On column.

Changes needed:
Import and Export Dashboard: Modify Added on column header to Added/Edited on and the APIs need to be changed to provide Updated On date.
API Configuration: Add ‘Added/Edit on' and 'Added/Edited by’ column header and the API need to be modified to provide these needed data.

## Acceptance Criteria
On the “API Configuration” grid, the following columns shall be included:

* “Added On”
* "Added/Edited By"

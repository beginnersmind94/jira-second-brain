# NXT-67554 — Observations in LMA

## Description
1-- Renew license button should be disabled for Expired license, if the license is not assigned to any district*--Fixed*
2--Some modules are by default (say System). Do we still need to show them as part of License Modules. 

   [Ravi]: Remove System Module from the list of Licenses. It needs be added by default no need to show in licenses.--*Fixed*
3--For any cancelled license notes in the view license option should be displayed the reason for cancellation. It needs to be appended to the existing notes section*--InProgress*
4-- for Renew status license when we add licenses , the start date in the add license drawer and start date in the dashboard miss match.*--Changed the behavior of renewed License, It behaves as active*

5-For all schools, the license duration should align with the academic year, which runs from July to June. Regardless of the actual start month, the license end date should automatically be set to June of the following year. For example, if a license starts in August 2025 or any later month, the end date should default to June 2026.
6-The 'Renew' option should have the same functionality as 'New' for adding licenses — the 'Edit' option should be enabled instead of 'Add License *--Fixed*
7-manual cancellation is accepting previous date*--Fixed*
8-In the dashboard, the "View History" option is not visible in the three-dot menu for the last entry.*--That is in corner case, so can be ignored*
9- Only active districts should be listed in generate license districts   dropdown.*--Fixed*
10-Once there is no active ,new district license in the dashboard ,that district  should be listed in the generate license dropdown (Soon after the license is cancelled then it should be displayed in the list). *--Fixed*
11-For a new license ,if we have a district ,on edit it should not allow to edit or remove the district if it is already assigned.*--Fixed*
12- When we remove a license from an active district, the existing license gets cancelled and a new one is created. However, the cancelled license is displaying irrelevant data even though revenue exists.

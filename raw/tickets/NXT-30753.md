# NXT-30753 — Sites Import - Update Existing Sites

## Description
The Sites Import process needs to account for updating sites if the import file contains sites that already exist.

## Acceptance Criteria
* When a site that is imported already exists within SchoolCafe, the SchoolCafe site record shall be updated with the data from the import file.

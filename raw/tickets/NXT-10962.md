# NXT-10962 — Dashboard - Insights Page for Account Management

## Description
As a user I want to see an “Insights” link within the navigation of the Account Management module so that I can view metrics, visualizations, and smart messages.

*Summary*

Rename ‘Dashboard’ to ‘Insights’

*Mock-up* 

Not Required

*Details*

* When a user attempts to navigate to the Account Management module, the page they land on should be the Dashboard, which we should rename ‘Insights’. 
* Insights should be the landing page for the module.
* Page should be available for viewing permission within Roles (Account Management > Insights)

## Acceptance Criteria
This story is complete when the Account Management's Dashboard has been renamed 'Insights' and a permission for it has been added/renamed in the Roles page.

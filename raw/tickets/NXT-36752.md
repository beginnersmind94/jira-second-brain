# NXT-36752 — Exports – Job Scheduler - UI only

## Description
As a SchoolCafé admin, I want to be able to schedule an export job so that I can proactively monitor the application and ensure higher customer satisfaction.

*Flow:*

* User logs in and browses to System > Imports & Exports > Exports
* User clicks on + New button and is redirected to the Add New Export page
* User completes the form including the schedule information and saves the new job

*Notes:*

* This story assumes that we can leverage the existing pages for Exports and “plug in” the scheduling component of it and attempts to just add the necessary pieces to enable that functionality
* This story is based off the SIS Auto Sync and all the work we did with Monitoring Scheduler and is following those flows, layout and functionality
* We are only changing the appropriate fields as necessary, so when in doubt, copy what that page does

*Requirements*:

* *Add New Export page*
* When the user clicks the + New button on the Exports tab, do the following on the Add New Export page:
** Use the SIS Auto Sync + New page as the example
* 
* Add the Scheduler section that is on the SIS Auto Sync + New page and the Add New Monitor page in the same way and location
* 
* Schedule Section:
* Title: Scheduler
* Show the schedule section like on Syst Auto-Sync page just like it is and with all the existing functionality for that page
* Cancel button
* Just like on Syst Auto-Sync page
* On click, forget what you are doing and go back to the Exports page
* Save button
** Just like on Syst Auto-Sync page
** On click, create the scheduled job per the dates and time selection in the schedule section (what to do in detail follows below) and then redirect to Exports tab
** Be sure to set the Last Updated and Last Updated by values so that the exports grid has the value it needs
** Ensure you are able to determine when an export configuration is a scheduled or manual so that it can be used in the Scheduled column on the Export tab’s grid
* 
* *Exports Tab on Imports & Exports*
* On the Exports tab, make the following changes:
* Add a column to the table:
** Title: Scheduled (Not IsScheduled)
** Values: Yes / No
** Mimics the functionality of the IsScheduled column
** Yes if the configuration is for a scheduled job
** No if it is not configured with a schedule
* Export button
** Do the same as for non-scheduled jobs
* Delete button
** Do the same as for non-scheduled jobs
* Edit button
** Do the same as for non-scheduled jobs
* 
* *Export Monitor Tab on Imports & Exports*
* On the Export Monitor tab, make the following changes:
* Ensure that the scheduled jobs show in the export monitor grid when they run and that all the column values are correct and completed
* 
* View Log button
** Do the same as for non-scheduled jobs on the page it opens
* 
* Permissions: no updates required

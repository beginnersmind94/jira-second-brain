# NXT-40448 — IM - Menu Items - Servings - Serving Size Exceptions

## Description
As a Menu Planner I want the ability to define serving size exceptions for the menu item serving sizes so I can ensure the correct portion size is served to the correct grade groups.

## Acceptance Criteria
* Provide a button to edit and add serving size exceptions for the menu item
* In edit mode, replace the Identifiers column with a singular editable multi-select dropdown which contains both *Meal Pattern* and *Serving Group*
* Load all menu item serving sizes with the Meal Pattern and Serving Group editable
* Show all eligible Meal Pattern + Serving Group combinations in the multi-select dropdown
** Remove options which are already assigned to other servings (every serving group is a single selection; mutually exclusive)
* Provide a +Not Served button, to add a custom Not Served record
** Insert a custom row with Not Served label with editable Meal Pattern and Group
* Once the Edit Exceptions mode is saved, return to the primary Edit mode 
** Leverage the Identifiers column to indicate the Serving Exception serving/grade groups
** Consolidate all ‘Not Served’ exception into a single custom Not Served row
*** Use the identifiers to display each serving group

# NXT-45551 — INV - Inactive Items - Update Inventory UI & transactions

## Description
As a district admin I want the ability to make an Item inactive if it is not being used anywhere in the system, so I can delete any unused or accidentally created/imported items.

## Acceptance Criteria
* Update all other Inventory UIs and transactions to manage the Pending or Inactive item
** Item cannot be included on new Inventory Additions
** For a single site, until QOH is depleted, the Item will still be available on
*** Physical Inventory
*** Withdrawals
*** Transfers
* Update all Inventory UIs to show the Pending Deactivation or Inactive status with every instance of the item
** show the item with an alert icon and a corresponding error or tooltip message
** exact implementation is dependent on the specific Inventory screen

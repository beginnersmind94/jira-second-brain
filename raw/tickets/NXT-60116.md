# NXT-60116 — IM- Item Onboarding- Review Menu Item Info

## Description
As a district manager, when using the bulk item onboarding tool, I would love to be able to also indicate and enter information for my items that are also menu items so that I do not have to go configure them one at a time later. 

!image-20250317-203239.png|width=1527,height=908,alt="image-20250317-203239.png"!

## Acceptance Criteria
* Provide a column on the landing page for Menu Item Info
* Provide a review step on the Bulk Item Import tool to update all menu item information
** Include item name and serving size along with the rest of the menu item info
*** Combine Prep Site Item, Show in Summary and Tags into a column called *Menu Item Flags*
** Include a column for “is Menu Item” that is populated based on the initial item info questionnaire
* Display an alert icon in the Item # column if there is missing information for MANDATORY fields/columns
** Display ‘{color:#ff5630}_Missing_{color}_’_ in any Mandatory column
** On Leftover Category, Unspecified or Carryover asks users to enter a max Days, if users select either selection but do not put the max days, display  ‘{color:#ff5630}_Unspecified_{color}_’_  or  ‘{color:#ff5630}_Carryover_{color}_’_ to indicate there is still missing information needed
** Menu Items with missing mandatory information cannot be published (Can publish the item with no DMI)
* Provide a “Update Menu Item Name” button to navigate user to the 1st of the 2-step edit mode
* Provide a “Update Menu Item Options” as the second button

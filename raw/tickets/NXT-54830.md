# NXT-54830 — Workspace - MWD - Acc - Calendar View

## Description
As Dana the Director I want a way to review the entire month and see if all sites have completed their daily tasks from the End of Day page.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=8827%3A5889&node-id=8827-5890&viewport=465%2C491%2C0.13&t=kLhGb1UT9TUh6B9f-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=8853%3A175|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=8827%3A5889&node-id=8827-5890&viewport=465%2C491%2C0.13&t=kLhGb1UT9TUh6B9f-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=8853%3A175|smart-link] 

*requirements*

* User has a Calendar View option on the page.
* User can select sites.
* User can select a month.
* User can see each day of the month.
* User can clearly see what the current day is.
* Prior working dates are highlighted:
** Red if 0% of sites have completed any End of Day tasks. 
** Orange if some End of Day tasks are completed, but no site has reached 100%.
** Yellow if >0 and <100 have completed all End of Day tasks.
** Green if 100% of sites have completed all End of Day tasks.
* -User can mouseover a date to see the % of sites that have completed all Tasks.-
* Configured Holidays can be shaded in blue.
** Still need to check for service tho and color accordingly.
* User can click a date and the regular Focus View should load for the selected sites + specific date.

## Acceptance Criteria
* Calendar View added to the page.
* User can select sites + a month.
* User can see each day of the month like a calendar.
* User can see the current day clearly noted.
* Dates are shaded based on how many sites have completed all End of Day Tasks.
** 100% = Green
** 0% = Red
** >0 and <100 = Yellow
** Holidays = blue
* User can click a date and it loads in Focus View

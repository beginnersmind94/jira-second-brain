# NXT-20747 — Turn On / Off Food Detection Transactions

## Description
As a Cashier, I want to configure POS to enable or disable Food Detection Transaction Integration so that I can hide or show that functionality in the transaction.

*Application*: POS

*Flow:*

* Browse to the Food Detection Settings screen
* Turn on or off Food Detection Transactions during transactions

 *Assumptions:*

* This page exists in the classic ExpressPoint
* You can copy all the code and try to copy as-is
* Talk to Saketh if you have any questions

*Requirements*:

* Be able to browse to page
* Be able to toggle On/Off the Food Detection Transaction Integration field
** This field can only be turned on if Food Detection is enabled
*  
* Fields:
* Food Detection Transaction Integration – On/Off
** On/Off toggle, default to Off
** Auto set to off if Food Detection field is toggled to Off
** If select On:
*** Food Detection will be integrated into the transaction log and work
*** Log Food Detection Integration Enabled
** If select Off
*** Food Detection will not work with the transaction screen
*** Log Food Detection Integration Disabled
* Security
** Anyone with rights to use the Terminal has rights to this screen

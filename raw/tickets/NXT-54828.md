# NXT-54828 — Workspace - MWD - Acc - Summary Stats per Site

## Description
As Dana the Director I want to see how each individual site is performing as the information is useful at a high level to help me focus on sites that need attention.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=8827%3A5889&node-id=8827-5890&viewport=465%2C491%2C0.13&t=kLhGb1UT9TUh6B9f-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=8853%3A175|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=8827%3A5889&node-id=8827-5890&viewport=465%2C491%2C0.13&t=kLhGb1UT9TUh6B9f-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=8853%3A175|smart-link] 

*requirements*

If the user has selected multiple dates, we need to show them a summary/total for the dates selected. If they have only selected 1 date, then the available actions are identical to how [https://primeroedgenext.atlassian.net/browse/NXT-54829|https://primeroedgenext.atlassian.net/browse/NXT-54829|smart-link] operates.



So if multiple dates have been selected: 

* User can see each selected site.
* User can see a summary of the following stats for the date range:
** Meals Served
*** Total number of meals served to students
*** Inclusive of second meals
** Payments
*** Total
** Revenue
*** Total
** Meal Participation %
*** Average over the number of service days
** Tasks
*** Only shown if a single date has been selected - column is hidden otherwise.
*** Show how many tasks from End of Day have been completed for that site + date.
* Standard grid options:
** Sorting columns
*** Default sort: Site Name
** Search filters for all columns
** Pagination → default 10 per page

## Acceptance Criteria
* User can see summary stats for each selected site
* If the user selected a date range, show them the summary totals for that date range:
** Meals Served - to students only
** Payments
** Revenue
** Participation % - average over the number of serving days, not a total
* If the user selected only 1 date, then we are just showing them the totals for that date - same stats + Tasks.
** Tasks - only shown if a single date is selected
* User can click the line to see more information, but that’s covered in [https://primeroedgenext.atlassian.net/browse/NXT-54829|https://primeroedgenext.atlassian.net/browse/NXT-54829|smart-link]

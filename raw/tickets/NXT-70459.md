## Acceptance Criteria
* Provide slideout for versions to be able to see what version have been made for a menu
** Current version should always be on the top and cascade down in order
* Display Number of production records and assigned menus that are associated with each version 
** Provide the effective date for the particular version 
* Provide an icon for a user to view a particular version in an overlay that leads them to the new standard menu page ([NXT-70460|https://primeroedgenext.atlassian.net/browse/NXT-70460])
* Provide an icon that triggers a choice for the user to restore a version or create a new menu ([NXT-70461|https://primeroedgenext.atlassian.net/browse/NXT-70461])
* On the list page, provide options to view all the versions including current 
** Leads them into view mode of the current menu

# NXT-43922 — IM - Items - Consume Preferred Image flag

## Description
As an Item Management user I want the Items page to show Preferred Ingredient and Preferred Stock Item images so I can see these images when using the split view.

## Acceptance Criteria
* Update the Items split view to consume the Preferred Ingredient or Preferred Inventory images as the default image
** First, use Preferred Ingredient image
** Second, use Preferred Inventory Image 
*** this would be the only image flag for non-food items
** Third, use any image
*** Ignore images with a Document Type assigned (CN label, etc.)
* Do not consider Document type options (CN label, etc.)
* Update the Recipe Steps card and Ingredient Search card to consume the Preferred Ingredient flag
** Update the Custom Menu Item “ingredients” card as well

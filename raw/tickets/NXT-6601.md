# NXT-6601 — INV - View Vendor Info - Section Cards

## Description
As an inventory user, I want to be able to view my saved vendor information sections so that I can view saved details in card view.

Mockups
[Figma mockup|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=390%3A19422&scaling=scale-down-width]

!image-20201026-163220.png|width=100%!

*Requirements*

* Saved vendor sections will be displayed in card view. Edit button will enable the edit information view.
** Each card will have an edit icon to open that particular card for editing, as per [Update Vendor Info story|https://primeroedgenext.atlassian.net/browse/NXT-5708]
** General Info card will display all saved general vendor info attributes
** Vendor Address will display all address information
** Vendor Contacts will display a card for each saved contact
** Future vendor sections will have a corresponding card on this screen.

Story 2 - Right side scrolling cards

## Acceptance Criteria
User will be able to view all saved vendor information in card view.
Cards will vertically scroll

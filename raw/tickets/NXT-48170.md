# NXT-48170 — PR- Update Calculations for Fluid Units on Food Required (Consolidated/ Prep site Tab)

## Description
As a production manager, I would like to be able to see the calculations of quantities needed accurately when I look at fluid items such as my juices, milks etc.

## Acceptance Criteria
* Calculate Fluid items to make sure the correct amount needed appears
* If Fluid item has a dual unit and dual unit serving is used, always use the sub qty to calculate how much is needed 
** IF sub Qty is given, but no dual unit is present, always try to use sub qty to derive the amount needed
* If fluid item is used within a recipe, calculate with a 1 to 1 conversion with grams to find the amount needed
* Affect the distribution worksheet food required with new calculations

# NXT-24196 — MP> Menu Planner > Recently Searched to become Recently Used

## Description
When working in the Menu Planner of the Menu Planning area of the software, there’s a section on the left hand side labeled “Recently Searched.” Based on the intended functionality, this section should be labeled as “Recently Used.”

## Acceptance Criteria
There will be a section on the left hand side of the page in the Menu Planner labeled as “Recently Used” instead of “recently searched.”

Store items that were used in current or previous sessions based on user.  

Display most recent 5 items at a time, allow “load more” option below to give user chance to load additional items if applicable. 

Keep previous 5 items and display additional 5 with scroller option. Only allow 5 items to display at any point in time.

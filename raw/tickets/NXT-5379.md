# NXT-5379 — [Accountability] - Integration - Exports - Claim Preview and Export (Texas)

## Description
As a user I want the ability to preview and export my reimbursement claim in my state’s specified format so that I can import it into my state claim website to complete my monthly submission in order to receive my reimbursement. 

*Proposed Design*

Design idea is to create the more actions column/icon within the claim record that allows the user to select more options and choose “Export”.

*Requirements*

* The user can preview the claim for each program, using the active format for their district.
** The preview should show column headers, and the data that will be exported for each cell.
* The user can select a program, and export the claim.
** We’ll need to call the export created via [https://primeroedgenext.atlassian.net/browse/NXT-5380|https://primeroedgenext.atlassian.net/browse/NXT-5380|smart-link] 
** A CSV file is created of the selected claim type using the selected claim export configuration when the export button is selected

## Acceptance Criteria
* When a single claim period is selected the user can select they format for the claim to exported
* Using the Claim Template for the district 
* The file is created when the export button is selected

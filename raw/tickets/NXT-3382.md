# NXT-3382 — Template Hub - Template Details Screen

## Description
*User Story*
As an Inspections user, I can view a Template's details screen when clicking on it to be able to view a read-only version of the Template.

*Requirements*
* Clicking on a Template in the Template Hub opens the Template Details screen for that Template
** Works for both Card and List view
* The Template Details screen should work the same way as the Inspection Details screen
* Basic information about the Template is displayed at the top of the screen
** Template Name
** Number of Questions
** Scoring Type
** Score Settings
*** Custom Scoring - All Statuses with Bindings and Colors
*** Pass/Fail - Percentage for Pass
*** No Scoring - Nothing Displayed
** Last Updated
** Last Updated By
** Total Number of Questions
*** For Custom Scoring, show possible points as well
* All Question Categories and Questions are displayed
** Each Question Type shows up as it would in an Inspection
*** Ex: Multiple Choice Questions show all choices
*** Answers cannot be responded to
* User can access all relevant actions for the Template
** For the purposes of this story, actions are placeholder

## Acceptance Criteria
* Template Details Screen can be accessed for each Template from the Template Hub screen
* All information from individual Templates appears correctly

# NXT-15774 — Inbox and Messages needs a "Back Button"

## Description
When in the inbox for task list or messages there is no back button to go back to main workspace or a close button.  You have to use the back arrow in the browser.

## Acceptance Criteria
* As a user, I will have a button or some other means to return to the main workspace page other than using the browsers back button.

## Acceptance Criteria
* Provide a review step on the Bulk Item Import tool for the update of all attributes from item level Ingredient Info card (except Exemption Letter upload)
* Create both the view and edit mode of this step
** the UI won’t be very different for view and edit
* In edit mode switch the columns to editable versions
** Enable Yield Factor percentage text field for modification (ensure 100% default is loaded, or existing configured value for local items)
** Use UI tools like popup/slide-out with styled icon for fields like Sub Ingredients and Standard Recipe Directions which are very lengthy
** Enable checkboxes for Locally Grown and Buy American
*** Provide an ‘Apply to All’ header checkbox in edit mode
* Display as much of Sub Ingredients and Standard Recipe Directions in view mode
** text will need to be truncated at some point to fit in the cell

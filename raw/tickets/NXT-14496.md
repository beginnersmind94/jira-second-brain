# NXT-14496 — User Import - Step 1 of 2 - Data Exchange piece

## Description
This Story is to cover the Users @ Imports. User imports are scheduled more frequently and some school districts on a daily basics. 

User Import is a straight forward import with couple of filters (parameters).

User imports are usually in csv format and requires two parameters while importing.

# Override Roles Yes / No
## Over ride Role : Here when this option is marked as yes, the roles in the database are replaced with the ones on the file and when marked as NO, the values get appended to the existing roles. All the roles are separated by comma. The Default value is always NO.
# Override Sites. Yes / No
## Override Site : Here when this option is marked as yes, the site info in the database are replaced with the ones on the file and when marked as NO, the values get appended to the existing roles. All the roles are separated by comma. The Default value is always NO.

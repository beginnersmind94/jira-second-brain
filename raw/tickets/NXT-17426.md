# NXT-17426 — AI/ML Model Technical Metrics – Performance and Report

## Description
As a system, AI Insight will need technical metrics of the ML model so that we can evaluate performance and impact analyses.

*Description:* 
AI/ML solution technical metrics will be evaluated to measure model accuracy and performance. These metrics will be compared and analyzed along side Application actionable KPIs and Business goal KPIs and Metrics. 

*AI/ML Model metrics:*

* Classification accuracy
* Mean absolute error (MAE)
* Root mean square error (RMSE)
* Sensitivity, Specificity, Accuracy
* Other – will be added as per model definition

h3. Dependency: 

AI/ML metrics are dependent on model selection and development

## Acceptance Criteria
For Eligibility, Participation recommendation and forecasting: 

# The AI/ML solution shall specify the hypothesis solving
# The AI/ML solution shall specify the problems it is solving with defined use cases
# The AI/ML solution shall specify the departments and users supported (Districts/Schools – all sites, multiple sites, single site, districts in region or geo)
# The AI/ML solution shall specify the total number of models deployed
# The AI/ML solution shall specify the volume of data analyzed
# The AI/ML model shall record accuracy metrics with precision and recall scores
# The AI/ML solution shall specify the SN KPIs impacted with the provided solution (before AI and after AI)
# The AI/ML solution shall specify the possibilities of using this model to other SN areas with specific use cases
# The AI/ML solution shall provide data analysis dashboards with different use cases and data sets
# The AI/ML solution shall provide dashboards with pre model and post model analytics – Key AI objectives achieved report.

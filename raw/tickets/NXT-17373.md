# NXT-17373 — INV - Orders - Order Detail - Merge suggestion

## Description
As an ordering manager I want a Merge Order suggestion/prompt on the Order Details screen so I can merge my order with any identical orders.

*Requirements*

* Provide Merge icon on Order Detail if the order can be merged with another order
** show an informational message in the banner “This order could be merged with other orders.” 
*** this could potentially “stack” with other messages in the banner
** provide a ‘Merge Order’ button right after the banner
*** clicking the Merge Order button would open the merge popup (same as Create Orders)
* Use updated logic for finding orders to merge into
** Orders can be merged if they have the exact same value for the following attributes:
*** *Site*
*** *Vendor*
*** *Status*
*** *Delivery Date*
*** *Delivery Location*
** Identify all possible orders for merging, not just the most recent one
* Use the new, enhanced Merge Popup - Details in [https://primeroedgenext.atlassian.net/browse/NXT-17457|https://primeroedgenext.atlassian.net/browse/NXT-17457|smart-link] 
* Do not allow Merge if any receipts have been created or if the order is Closed
** only *Open* and *Pending Review* orders can be merged
* Once the order has been merged:
** Update the status and display it as ‘Merged into Order #98765’
** Replace the banner message with “This order has been merged into Order #98765.” 
*** This should be a blue banner
** Show a ‘View Order’ button next to the banner, to open the newly merged order
** Remove all action buttons related to the current order (Cancel Order, Print Order)
** Do not show the Receipts tab
** Do not show any Edit icons (Delivery Date, Delivery Location)
*** this includes the Edit icon in the Items list grid
** Change Comments popup to view-only mode
*** this includes the Comments in the Items list 
** Remove Add Item button from header
** Remove Save button from footer
** Keep the Back button

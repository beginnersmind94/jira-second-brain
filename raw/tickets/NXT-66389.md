# NXT-66389 — Inventory Withdrawals / Waste

## Description
As a *district inventory manager*,
I want withdrawals and waste to generate ledger entries,
So that unusable stock is removed from assets and recorded as expense.

## Acceptance Criteria
* Preconditions: Same as usage.
* Posting date = last day of inventory period.
* *Ledger entries*:
** _Withdrawal/Waste (GLEntryTypeCd: WD or WASTE depending on setup)_ → Debit Expense, Credit Asset.
* Rollbacks reverse entries.
* Entries flow into GL and financial statements as waste expense.

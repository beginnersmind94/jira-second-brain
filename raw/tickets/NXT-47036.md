# NXT-47036 — IM - All List Pages - Cohesion Updates

## Description
As an Item Management user I want the list pages to be cohesive and to use UX elements to help me operate the page.

!image-20240528-143450.png|width=91.66666666666666%,alt="image-20240528-143450.png"!

## Acceptance Criteria
* Change all Status filters – both Main and Sub-Type – on all 3 pages to list the values in the below sequence (with relevant statuses for each dropdown)
** Incomplete
** Active
** Pending Deactivation
** Inactive
** Disabled
* Change Inventory Item status to also be (mostly) sequential
** Incomplete
** Unpublished
** Published
** Synced
** Locked
** Ineligible
** Pending Deactivation
** Inactive
** Disabled
* Change all ‘List Export’ report buttons to simply say ‘List Export’
** Ex: Menu Item List Export > List Export
** make sure file and alt labels still have specific page reference
* Show status chip on non-Active items/recipes/menu items
** next to Description
** show chip label for Incomplete, Pending Deactivation and Inactive statuses
** use existing sub-type status icons, colors, and labels

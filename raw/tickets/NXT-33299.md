# NXT-33299 — Platform- Need for import template to include inputs already configured in system

## Description
As a User trying to input data into my Item Details Import, I need a way to see how certain columns data input fields are configured in the system. 

For Example on the Item Details Import, Manufacturer Name, Brand Name, Valuation Group, Item Category, Storage Category and Serving Size Unit Measure are all data points that require you to enter information in the same manner of how it is configured on the UI.

There also needs to be a way for other fields to give a drop down value of Yes or No for flag inputs

AT the moment, imports are not viable for a user to use as a data entry point until these fields are populated for a user to see

## Acceptance Criteria
Import template when generated should populate inputs for fields that have already been created on the UI

Import template should generated yes or no drop down for fields that are flags on the UI

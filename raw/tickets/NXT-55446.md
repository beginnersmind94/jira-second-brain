# NXT-55446 — MP- Find and Replace- Final check on save for Record Complete Menus

## Description
As a Menu Planner, I would like for the system to check when I am saving a menu and not allow changes to that I am making to a menu if the menu is now associated with a record complete production record.

## Acceptance Criteria
* Do Final Check on Find and Replace *Save* to check if menu has a record complete record associated
* This should work the same for both Replace and Remove
* Provide a Banner Error Message  “Menu: Menu Name could not be saved! One or more Production Records have been completed associated with this menu”
** If multiple menus are affected  “Menus: Menu Name, Menu Name could not be saved! One or more Production Records have been completed associated with this menu”
* Grey out the menus that are no longer available and provide the production record warning icon that we are providing currently and keep the current selections of the menus that CAN be saved
** Remove the menus that can not be saved from the counts next to the Replace/Remove

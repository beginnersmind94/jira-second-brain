# NXT-46679 — IM - Menu Items - Allow POS to be independently deactivated

## Description
As a district admin I want the ability to toggle off the POS portion of a POS-enabled Menu Item, so I can remove it from POS menu grids without affecting the menu item status.

!image-20240307-151632.png|width=83.33333333333334%!

## Acceptance Criteria
* Allow the POS section of POS-enabled menu items to be “disabled”
** use the POS chip on the Menu Item card
** on click of POS chip, show prompt “Do you wish to disable this POS item? This will not affect the Menu Item.” 
* Upon disabling the POS item, revert the card to the non-enabled state
** Retain the POS item data/values, but hide the UI fields
* Set the POS status to Inactive when it is disabled
** this ensures the item remains on any POS Menu Grids
* If the POS section is re-enabled, preload the fields with the former data
** this will be in edit mode so validations can be applied again at the time of Save
** this will use the existing ‘Enable POS Item’ toggle 
*** previously named ‘Show on POS’ (please relabel on all pages)

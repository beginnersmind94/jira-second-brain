*Requirements:*

* Select the site
* Select the date range
* Dates can be in the past or the future (up to 2 weeks)
* Select the meals being served.

## Acceptance Criteria
The serving site can be selected
Multiple serving dates can be selected.
There is an option to include weekends for serving (Default is off)
There is a selection for each meal type possible for serving
Each meal type has a menu item that can be selected
Multiple meal types can be selected.
When a meal type is selected a list of menu items of that meal type are available to select.
If you don't select a menu item the meal type is de-selected

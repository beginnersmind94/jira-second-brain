# NXT-7115 — Configuration - POS Item Categories

## Description
As a user I need to create the POS Item Categories that will be used to organize POS Menu Items

*Mockup:*

[https://www.figma.com/file/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1137%3A22723|https://www.figma.com/file/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1137%3A22723|smart-link] 

!POS Item Categories (c8dc92fe-f457-40a7-bf18-a50fce834ee3).png|width=1922,height=1340!



*Requirements:*

* There are 7 default categories Entree, Meat/Meat Alt, Fruit, Vegetable, Grain, Milk and Ala Carte
* There is an area where users can create their own POS Menu Item Categories
* Each Category has a unique name, a Color, and an optional Display Order

## Acceptance Criteria
POS Categories is launched from the Configuration from the left panel
POS Categories is reached through the tabs of the Configuration page.
POS Item Categories will have a Name (default names are Entree, Meat/Meat Alt, Fruit, Vegetable, Grain, Milk, Ala Carte
Additional POS Item Categories can be created by the user
Duplicate Names are not allowed.
POS Item Categories have a color associated with them.
Use the common Color Picker to select the color. (Developed for POS Menu Grids) 
Categories have an optional display order
The Color and Display order of Default POS Item Categories can be changed.

Default Data

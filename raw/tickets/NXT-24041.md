# NXT-24041 — IM-Item Aging Report

## Description
As a Nutrition Department user I need a report that  provides me information on how long has active items been in the system so I can determine if they are outdate and need to be inactivated.

## Acceptance Criteria
Entry Point

!a871c006-b01c-43aa-92df-679558ac470b|width=320,height=522,alt="Pasted Graphic.tiff"!

Item Management -> Reports ->Reports Landing Page -> Items Aging Report




Selection Criteria

Quit Filters (Selectable Drop Down Box)

* A filter by Item Category should be resent
* Options :
** 30 Days  (when selected system  selects  and displays item’s   information  as per mockup for items which have been created  30 days between system date and creation date)
** 60 Days  (when selected system  selects  and displays item’s   information  as per mockup for items which have been created  60 days between system date and creation date)
** 90 Days  (when selected system  selects  and displays item’s   information  as per mockup for items which have been created  90 days between system date and creation date)
** 120 Days (when selected system  selects  and displays item’s   information  as per mockup for items which have been created  120 days between system date and creation date)
** 1 Year      (when selected system  selects  and displays item’s   information  as per mockup for items which have been created  365 days between system date and creation date)
** Date Range  (when selected system  Enable Start and End date fields  and  system  selects  and displays item’s   information  as per mockup for items which have been created  between date range and creation date

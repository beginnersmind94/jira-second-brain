# NXT-6330 — Workspace - Add help text to headers of each section

## Description
As a PrimeroEdge admin, I would like to easily learn the purpose of each section within my Workspace, so that I can more comfortably and effectively begin using each feature.

*Summary*

Add selectable tooltip icons next to the header of all 4 sections of the Workspace.

*Details*

* Within the Workspace, add help text for each of the 4 sections of the Workspace:
** Messages: View and send important messages throughout your organization.
** My Tasks: View and schedule tasks to set up helpful reminders.
** My Favorites: These are shortcuts to all of your favorite screens. Select the star icon to remove one.
*** May be deferred until the implementation of this section in Phase 2.
** Last Visited: These are the screens you have visited most recently. Select the star icon to create a favorite.
*** May be deferred until the implementation of this section in Phase 2.
* Allow the user to view the respective text when either hovering over or tapping a section (for mobile web browsers).

## Acceptance Criteria
Verify that help text is added to the 4 sections of the workspace (final 2 may be deferred until the implementation of those sections in Phase 2)

Verify that the text can be viewed by either tapping on or hovering over a section

Verify the specific text content of each icon matches its description above

# NXT-18050 — INV - Orders - Manage Orders - Opening a 'Quick Receipt' Order - Incomplete Status

## Description
As a site manager I want the ability to open a Quick Receipt order in Incomplete status so I can manage it similarly to how I would manage other orders/receipts.

*Requirements*

* Quick Receipts will be available alongside orders in the Manage Orders list
* For Quick Receipts in Incomplete status, and user clicks the view icon
** navigate to the Receive step of the Quick Receipt screen
* Retrieve and display all previously saved values
** Order # will be available at this point
* All Quick Receipt validations will still apply
* Action buttons will be available as usual
** Save as Incomplete ([NXT-18641|https://primeroedgenext.atlassian.net/browse/NXT-18641])
** Confirm (NXT-18080)

# NXT-12066 — Account Management - Accounts - Update Filter Panel

## Description
As a product I want to provide users with the ability to filter/search information a consistent, organized, and elegant way so that users know what to expect as the interact with filterable data grids throughout the application.

*Summary*

In the Accounts Management *Accounts* page, update the Filters section to make it is easier to select filters for the user and to provide a consistent experience when compared to other screens within the product.

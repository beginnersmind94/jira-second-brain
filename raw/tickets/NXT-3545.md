# NXT-3545 — Edit Inspections - Text Question Type

## Description
*User Story*
As an Inspections user, I can see Text Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Text Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Whether question is required
* There is a text field that the user can enter text into
** Users can enter any text
** There is no hard limit on amount of text that can be entered
* Text can be saved
** If question is required, Inspection cannot be submitted if text box is empty
* Text can be edited

## Acceptance Criteria
* Text question appears in Inspections
* Text can be entered, saved, and edited

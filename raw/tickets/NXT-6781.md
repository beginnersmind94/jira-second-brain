# NXT-6781 — System - Users Phase 2 - Add User  - User Avatar section

## Description
As a District Admin, I would like to be able to easily grant PrimeroEdge access to a new user in my organization, so that they can quickly begin working in the system.

+Summary+:

Utilize the existing breadcrumbs (System → Configuration → Users → Add User button) and update the existing Add User page to use the new UI template *for the User Avatar/Profile Image section only*.

+Details+:

* In the Add User screen, add a *User Avatar/Profile Image* section to the top of the screen.
* This section is optional for adding/editing users.
* This section should display the following:
** A generic user image icon.
** A note regarding supported file types
** An Upload Picture button
*** When the button is selected, open the file explorer/browser to allow the user upload an image from their local environment.
*** When the image is selected (_minimum_ file types to support: .jpeg, .jpg, .png), update that user’s image.
**** Replace the generic image icon with a preview of the image that the user uploaded
**** If user does not enter correct file type, display an error
**** Create a reasonable size limit. If user uploads an image beyond that size, displays an error
*** Perform necessary resizing, compressing, etc. so that the image fits in the circle icon.
*** (optional) User should be allowed to re-position their image within the circle, crop, etc. as well.
*** Changes can be committed immediately or when the user clicks the Save button/adds the new user.

## Acceptance Criteria
Verify that when adding a new user, a user avatar/image section is displayed

Verify that when selecting Upload Picture, the user's file explorer/browse opens to allow them to upload a local image

Verify that at least .jpeg, .jpg and .png are supported file types

Verify that if user does not enter correct file type, we display an error

Verify that if user uploads an image beyond the size limit we set, we display an error telling them that the file exceeds X size

Verify that we perform necessary resizing, compressing, etc. to fit image inside circle

(Optional) Verify that user is allowed to re-position their image within the circle, crop, etc. as well.

Verify the changes are saved (immediately or when user clicks Save button/adds the new user)

Verify that the avatar is visible when the logged-in user navigates back to this selected user account

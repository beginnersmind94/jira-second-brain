# NXT-23579 — Production- Reports- Pre-Production Analysis Page (Actual Value Data)

## Description
This Analysis will mimic the Weekly Menu Analysis with all the same characteristics but instead of using menu counts, it will use Planned Production counts.

## Acceptance Criteria
All the same characteristics of the Weekly Menu Analysis but this function will use the Planned Production counts. 

This will only affect Meal Pattern SY 2014+.

Criteria for Page:
Meal Type
Site Group
Site
Menu Line
Start Date- End Date ( Must select between 3-7 serving days for the analysis to work)

When You select Start Date, the end date should automatically populate a date one week ahead 
EX: if user selects 09/01/2021 the end date should display 9/7/2021.

When making selections for Meal type, Site Group and Site, this information should be saved for future searches if user leaves the page or if they go to the post production analysis page as well. 

It will have all 3 reports same as the weekly analysis. Nutrition report, Nutrition Summary Report, and Food Component Report. 

Analysis must be printable same as weekly analysis layout.

# NXT-9917 — INV - Transfers - Transfer Detail - Rejected / Cancelled

## Description
As an Inventory user I want a Transfer Detail page so I can open a transfer from the Transfer List and view the detailed information.

This is the screen that will be displayed for existing transfers and the page that will be accessed when the user opens a transfer from the Transfer List.

This Transfer Detail screen will look different depending on:

* Type of Transfer
** Site-to-Site transfer
** Transfer Request 
** District-generated Transfer
* User access/permission - Add ‘Auto Approve’ setting to System table. Default to ‘Off’ due to only having Super User access
** District user
** Site User
** Receiving Site
*** Approver
** Sending Site

+*Action buttons*+

* Transfer Ticket (report button) - Placeholder
* Price Details (report button) - Placeholder
* Handled in - [https://primeroedgenext.atlassian.net/browse/NXT-9571|https://primeroedgenext.atlassian.net/browse/NXT-9571|smart-link] 
** -Approve Transfer-
*** -this button only appears for the Receiving Site if the user has the Transfer Approval permission-
*** -Once approved, move the item from incomplete transaction table to complete transaction table to the site that it is going to-
* Reject Transfer
** this button only appears for the Receiving Site if the user has the Transfer Approval permission
** Once rejected, move the item from incomplete transaction table to complete transaction table to the site that it came from
* Cancel Transfer - Only available until ‘Approved’ or ‘Rejected’
** this button only appears for the Sending Site if they wish to cancel the transfer before it has been received/approved
*** If needing to cancel after approval, the user would have to do a new transfer
*** Once Cancel, move the item from incomplete transaction table to complete transaction table to the site that it came from

* Once ‘Cancelled' or 'Rejected’ it moves to ‘Comments’
** Comment box is required
** Character limit of 250 characters
** Show the number of characters left in the bottom left.
* Cancel (Return to previous page)
* Save (Go to ‘Rejected' or 'Cancelled’ Page)
** Show Transfer List Item - Rejected 
** Show Transfer List Item - Cancelled
* Show info ‘This transaction has been cancelled' or ‘This transaction has been rejected’

NOTE: We will only be able to cancel at this point. We won’t be able to reject as permissions are not in place.

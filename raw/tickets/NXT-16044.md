# NXT-16044 — INV - Create Orders - Order Review - Back to Enter Item Quantities

## Description
As an inventory site manager I want a Back button on the Order Review screen so I can return to the Enter Item Quantities screen and change the items I am ordering.

*Requirements*

* On the Order Review screen, provide a Back button
* *Back* - allow user to return to Enter Item Quantities screen 
** only show the button if NO orders have been created (all orders Incomplete)
** hide the button if one or more orders have been created
** Prompt user if this is clicked:
*** “This action will clear the currently created orders and will return to the enter item screen. Do you wish to proceed?”
* Return user to Enter Item Quantities screen
** existing items and quantities will be retained in the Shopping Cart
** retain the selections which were previously applied
*** this includes primarily Site and Order Date
*** also retain Vendor or Delivery Date only if all the orders have a common value for these. 
**** If there are orders to multiple vendors or for multiple delivery dates, do not make a selection for these fields
*** do not make a selection for Order Templates
** clear shopping cart if user changes *Site* or *Order Date*
*** Prompt on click of Apply button: “This will clear all item quantities currently entered. Do you wish to proceed?”
* Regenerate orders when user again clicks on Next in Shopping Cart ([NXT-16288|https://primeroedgenext.atlassian.net/browse/NXT-16288])

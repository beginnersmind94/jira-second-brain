# NXT-20077 — MP- Menus-Serve With Functionality

## Description
As a user, I would like to offer clarity on my menus regarding which menu items are to be served with other menu items. (Example, if I have entrees of Beef Stew, Chef Salad, and Pizza… a Dinner Roll on my menu would be served with the Beef Stew or Chef Salad, but would NOT be intended to be served with the Pizza.)

Clicking on "Serve with" next to a menu item will allow you to select any other menu item or items on the menu to indicate that they will be served with this item.

Once items are selected and user saves, the items selected need to be indicated as "complimentary items" and will no longer be able to be selected as a Primary item.

The "Serve with" item needs to be indicated as a "primary item"  and will continue to have the "serve with" button as an option for user to select additional items to serve with this item. 

Under "primary item", display all "complimentary items" that will be served with along with their meal contributions and how many servings are planned for those items.

Under all one or all "complimentary items", display the primary item along with its meal contributions and how many servings are planned for that item.

## Acceptance Criteria
Allow serve with option on all items.
Allow all items to be served with other items. 
Once serve with has been selected for an item and complimentary items have been added to be served with Primary Item, Complimentary items no longer have the serve with button.
Display meal contributions and serving counts for each Complimentary item or primary item. 
All serve with functionalities can be deleted and reverted back to no "connections".

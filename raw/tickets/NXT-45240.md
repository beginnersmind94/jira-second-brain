# NXT-45240 — Acc - Reconciliation - Use State Eligibility for Bulk Adjustments

## Description
As Emma the Eligibility Officer I want to see State Eligibility used for Bulk Adjustments so eligible students have free or reduced meal pricing for NSLP and SBP.

*requirements*

* If the system setting is turned on, Bulk Adjustments via Reconciliation needs to charge eligible students appropriately:
** Student’s State Eligibility is Reduced, use the Reduced meal price
** Student’s State Eligibility is Free, use the Free meal price
* This applies to student first meals only, and for NSLP (Lunch) and SBP (Breakfast) meal types only.

## Acceptance Criteria
* Use State Eligibility in Bulk Adjustments.
* Changes made for NSLP and SBP meal types only.
* Does not impact a la carte purchases / second meals.
* Must be a reimbursable meal.

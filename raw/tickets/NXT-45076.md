# NXT-45076 — Notifications - Add Toast Notification to Custom Reports

## Description
As a SchoolCafé user, I would like to receive banner notifications when a user add, edits or deletes a Custom Report so that I can know immediately any changes.

*Notes*

* Before doing this story, make sure the story for preferences and banner flyouts are completed

*Flow*

* User logs in and browses to any page
* System sends the user an Inbox notification when a user adds, edits or deletes a Custom Report
* User sees a message banner slide onto their screen for a few seconds
* The banner disappears or the user closes it

*Requirements*

* Add into the code for Custom Reports the ability to notify users when someone adds, edits or deletes a Custom Report
* This code will be added to the Save buttons for Add New and Edit and to the delete button
* When someone performs one of these actions, check the preferences to see if anyone wants to be notified of the change
* If yes, send them a message or banner or both depending on their preference
* Use this setting in User Preferences:
** I would like to be notified of Custom Report [CRUD List] by [Distro List]
** if it is not in the list, then add it
** See Preferences story for full notes on that field
* Inbox message:
** Add a template to the System > Templates
*** Module = System
*** Letter Type = Custom Report Notifications
*** Use same general format as custom report distribution
* Banner Message
** Title: Custom Report [Added or Changed or Deleted depending on action]
** Body: [first name] [last name] [Added or Changed or Deleted depending on action] the [name of custom report] custom report just now.

## Acceptance Criteria
* User is notified when their is an add, edit or delete on the custom reports if they have specified Yes for that setting in their preferences
* User is not notified if not
* User is notified per their selection in the CRUD list
* User is notified via the means specified in the Distro List (Inbox, banner, both, none)
* Inbox template is stored and used in Templates screen

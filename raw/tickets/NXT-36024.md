# NXT-36024 — INV - Transfers - Create Permission Resources

## Description
As a FSD I want resources for Transfers so I can assign users with the appropriate permissions.

*Requirements*

Map permissions for the navigable Transfers functionality

+Transfer list page+

* *View* - access to the list page/view in navigation
* *Add* - visibility/access to New button

+Transfer Details page+

* *View* - visibility of View icon on list page, access to Details page
** Reports are included; no report permissions needed
* *Edit* - enables the Edit button on the Transfer Details
* *Delete* (Cancel) - enables the Cancel button on the Transfer Details

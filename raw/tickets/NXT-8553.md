# NXT-8553 — Direct Certification - Extensions - Reviewed - Actions & History

## Description
*User Story*

As a district admin I want a way to review what actions have been taken by staff regarding DC extensions, and to correct them if required.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=199665%3A186328&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=199665%3A186328&scaling=scale-down-width|smart-link] 

User can see the history of actions taken on the potential matches along with pertinent information on the student / match.

* User should have an option to re-review all potential matches after the fact.
** User can use this to change their decision and perform an extension, or undo an extension.
*** They can choose to Match this if it wasn’t already extended, and enter a comment.
*** They can choose to Unmatch this if it was extended, and enter a comment.

## Acceptance Criteria
User can reverse their decision.
If the user is reversing an extension, then students should return to their prior eligibility accurately.

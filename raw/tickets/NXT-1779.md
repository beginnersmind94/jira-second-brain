# NXT-1779 — INV - Inventory - Additions - Header

## Description
As a Site Manager I want a list page for existing inventory addition transactions so I can view or adjust existing transactions.

*Note:* we will use the same components for corresponding Withdrawals page - [https://primeroedgenext.atlassian.net/browse/NXT-1654|https://primeroedgenext.atlassian.net/browse/NXT-1654|smart-link] 

*Requirements*

* Add a page to the Inventory category titled Additions
* This page will be a list page by default
** this will list the existing Addition transactions
* From this page there will be the ability to:
** Create a new inventory addition ([NXT-1778|https://primeroedgenext.atlassian.net/browse/NXT-1778])
* Display a search header for the list page
** Site selection
*** Use standard control (Areas if they are configured)
*** ‘All’ is default
** Date Range picker
*** default to the past week
** Reason dropdown
*** this will allow the user to filter the list according to specific addition reasons
*** this should be multi-select
*** ‘All’ is default
*** ‘Not Specified’ (which is basically ‘None’) will always appear
**** if no Reasons are configured, ‘Not Specified’ will be the only option
** Reset button - to reset selections to default
** Apply button - to apply changes in selections to the results grid
* Display a list of the addition transactions based on search criteria ([NXT-20588|https://primeroedgenext.atlassian.net/browse/NXT-20588])

# NXT-42761 — Elig - Notifications - Filter options for State Eligibility

## Description
As Anna the Application Processor I want to send a special letter to students with State Eligibility outcomes instead of a regular one and need an easy way to target them.

*requirements*

!image-20230830-231302.png|width=1483,height=440!

* For previously generated notices - add the Generation Date at the top of the frame.
* Update the page path and title
** Eligibility > Notifications
** Notifications

!image-20230830-233215.png|width=430,height=119!

If the Enable State Income Guidelines (Code: STATEEIG) is set to YES:

* Add options to either show only apps with a State Eligibility result, or to exclude them.
** Exclude applications with State Eligibility results
*** If enabled, results grid will not contain apps with a State Eligibility result of free/reduced
** Show only applications with State Eligibility results
*** If enabled, results should only show applications where a State Free or State Reduced outcome was determined.
* Add a column showing State Eligibility with the expected results.
* Use this notification template when notifying an application with a State Eligibility outcome of free/reduced.
* A student that is fully denied will use the standard Approval/Denial letter.

## Acceptance Criteria
* Generation Date added to the Download last generated notifications frame.

** Show the 2 new filters
*** If both filters are unchecked, the State Eligibility result apps should also be included with the Federal Eligibility apps.
** State Eligibility column added to the Notifications > Applications grid.
** Notify workflow updated to use this template for an app with a State Eligibility result.
* Page pathing + name updated

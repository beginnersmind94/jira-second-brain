# NXT-10956 — [TaggedDashboard] - Dashboard - Insights Page for Accountability

## Description
As a user I want to see an “Insights” link within the navigation of the Accountability module so that I can view metrics, visualizations, and smart messages.

Insights should be the landing page for the module.

Dashboard page is ready from the Platform team. Accountability module will not utilize any of the layout aspects of the template provided by the Platform team. For now, we will hide those elements for usage in future releases. 

For Pilot launch, 3 dashboard elements will be utilized - ADP, MEQ, and Meal Participation.

Figma displays the proposed layout. [https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=428210%3A1469&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=428210%3A1469&scaling=min-zoom|smart-link]

## Acceptance Criteria
* User is able to see a page for Insights within the Accountability module
* Insights is the default landing page for the module
* Page uses the standard dashboard layout provided by the Platform team for dashboard page
* Dashboard elements provided by platform team are hidden for Pilot
* Dashboard elements provided by Accountability team are displayed in the Insights page according to their design

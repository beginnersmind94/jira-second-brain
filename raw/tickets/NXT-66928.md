# NXT-66928 — Director, Central Office, SchoolCafe Admin

## Description
As a Director, Central Office user, or SchoolCafe Administrator, I need full administrative access to all Financial Oversight functions so that I can oversee and manage all financial operations without restrictions

## Acceptance Criteria
*Given* I am logged in with one of the following roles: Director, Central Office, or SchoolCafe Administrator
*When* I access the Financial Oversight module
*Then* I should have the following permissions:

*Complete Administrative Access:*

* I have FULL access to ALL modules and features
* I can VIEW, EDIT, ADD, and DELETE across all Financial Oversight functions
* No restrictions apply to any feature or capability
* All three roles (Director, Central Office, SchoolCafe Administrator) have identical permissions

*Dashboard Access:*

* FULL access to Financial Oversight dashboard
* I can VIEW, EDIT, ADD, and DELETE all dashboard elements
* I can configure dashboard layouts and settings

*Configuration Module - FULL ACCESS:*

* FULL access to all Configuration functions
* GL Code Structure: FULL access (VIEW, EDIT, ADD, DELETE)
* Account Mapping: FULL access (VIEW, EDIT, ADD, DELETE)
* Import/Export Templates: FULL access (VIEW, EDIT, ADD, DELETE)
* Version History: FULL access (VIEW, EDIT, ADD, DELETE)

*Transaction Management Module - FULL ACCESS:*

* FULL access to all Transaction Management functions
* Manual Entries: FULL access (VIEW, EDIT, ADD, DELETE)
* Post from POS/Inventory: FULL access (VIEW, EDIT, ADD, DELETE)
* Transaction Search: FULL access (VIEW, EDIT, ADD, DELETE)
* Void/Reverse: FULL access (VIEW, EDIT, ADD, DELETE)

*Reports Module - FULL ACCESS:*

* FULL access to all reporting functions
* Profit & Loss: FULL access (VIEW, EDIT, ADD, DELETE)
* Balance Sheet: FULL access (VIEW, EDIT, ADD, DELETE)
* Site-Level Analysis: FULL access (VIEW, EDIT, ADD, DELETE)
* Export to ERP: FULL access (VIEW, EDIT, ADD, DELETE)

*Period Management - FULL ACCESS:*

* FULL access to period management
* I can VIEW, EDIT, ADD, and DELETE all period settings
* I can open/close financial periods
* I can manage fiscal year configurations

*System Behavior:*

* All permission checkboxes are automatically enabled when any of these three roles is selected
* Master toggle shows "Disable All" by default (since all permissions are enabled)
* Permission changes are logged with appropriate audit trail for each role
* All saved configurations are accessible regardless of who created them
* Role switching between Director, Central Office, and SchoolCafe Administrator maintains full permissions

# NXT-5212 — Application Details - Application Status History

## Description
*User Story* 

As an application processor, I can see a history of statuses for the application.

*Mock Ups* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=87%3A2599&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=87%3A2599&scaling=min-zoom]

*Requirements* 

* Application Status History shows overall status changes (status code, user, date/time). 
* Fields
** Application Status 
*** Show the Eligibility + Basis in the same area when the status = Processed.
** Date 
** Official (who made the change, first + last name)

## Acceptance Criteria
- Multi-language, ADA compliant
- status changes are logged, including the received date
- processed status includes the eligibility + basis

# NXT-46567 — Imports - Import Monitor - Update Refresh

## Description
As a SchoolCafé user I would like it if the Import Monitor and Export Monitor refreshed only when there are rows that need a refresh so that it is less annoying

*Requirements:*

* This functionality applies to Import Monitor and to Export Monitor
* Change the refresh button as per Figma
* Change the auto-refresh to only do when there are running imports or exports
** If there are no rows, don’t refresh
** If there is nothing active, don’t refresh



JAS - New Requirement on 26Jan24:

* Set the time from 10 seconds to 30 seconds on Import Monitor and Export Monitor

## Acceptance Criteria
* Refresh icon is updated on both tabs
* Grid doesn’t auto-refresh is there are no records
* Grid doesn’t auto-refresh if everything is completed



JAS - New acceptance criteria on 26Jan24:

* Timer is 30 seconds on Import Monitor and Export Monitor

# NXT-3778 — Create checkbox selection

## Description
The Checkbox selection is allows the user to select individual data values to filter the DB. Multiple Checkboxes can be selected for filtering. When a checkbox is selected it is added to the Selected Filter window. 

!Screen Shot 2020-04-28 at 9.31.18 AM.png|width=287,height=341!

## Acceptance Criteria
The checkboxes are populated by the data in the DB column in  the AStable 

There is an Optional Search selection 

The Search is a partial search while typing.

The number of items matching the filter is shown on the line with the checkbox

The items in the section are shown in Alphabetical order.

When an Item is selected it moved to the Top of the list and is alphabetical with other selected items.

The unselected items are sorted in Alphabetical order below the selected items

When a checkbox is selected that value is added to the filter list, the filter selection count is increased by 1, and value is added to the Advanced Search filter window.

When a checkbox is selected it is 'OR'ed with other checkboxes within the section

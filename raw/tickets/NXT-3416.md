# NXT-3416 — Template Hub - Actions - Export to PDF

## Description
*User Story*
As an Inspections user, I have an action that allows me to export a template into a pdf file form that can be printed out or sent to others.

*Requirements*
* There is an action in the Template Hub called Export to PDF
** Action appears for all templates, including Global Templates
* When clicked, a pdf version of the Template is downloaded
* PDF represents the most recent saved version of the template
* The exported form displays all questions and possible answers
** For answers that have a possible text response, there is a text box
** For custom scoring, the number of points per response is shown

## Acceptance Criteria
* Export to PDF action is available
* Templates can be exported to PDF

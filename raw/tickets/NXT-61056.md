# NXT-61056 — AI Onboarding - Digital Recipe - New File Upload

## Description
As a customer support person, I would like to use AI to digitize a recipe document so that I can speed up onboarding

*Notes:*

* This screen’s content is broken out into various stories
** This story covers the new file upload process
** Another one will cover the previous upload process
** A third story will cover editing the recipe on the right side
** The AI analysis and saving are covered in other stories

*User Flow:*

* User browses to Item Management and then Recipes
* System checks if the district has enabled AI and shows the AI button if they have
* User clicks on the AI button and it opens the AI Recipe drawer
* User ensures the New chip is selected
* User selects what type of file they are uploading
* User selects a file to upload and clicks upload
* Note: this story goes only this far. What happens when uploading is covered in the AI Analysis story

*Requirements:*

* On the Report AI screen, do the following:
* Divide the screen up into 2 parts
** Left section is for file management
** Right section is for recipe editing
** Make it roughly 1/3 of the screen space, but can be adjusted as makes sense
** Ensure it is responsive
* 
* Add a section on the left for file management
** Title: Upload
** Add chips:
*** Labels: New, Previous
*** Chips act as tabs or screen filters
*** If click New, show the New content (covered below)
** Make expandable / contractible 
*** Expanded by default
*** Contract after upload a file and shows the File Analysis section
* 
* If click Previous, show the Previous content (covered in another story
* 
* In the New content, do the following:
* Add an instruction text per the Bolt design
* Add Format field
** Type: Single select drop down list
** Required
** Values: 
*** Types of competitor’s recipes formats we support, plus Excel, plus Custom
*** Get values from Bolt code
** Default: do not select a default value, make the user select
* Add a file upload dialogue box
** Use Bolt for text
** User must upload a file to enable the Analyze button
* Add a button:
** Label: Analyze
** Placement: bottom right, blow upload box
** Visibility:
*** Disabled until user selects Format and uploads a document
** Tooltip: Analyze File with Recipe AI
** On click: 
*** Do nothing for now
*** In the Analyze AI story, we will cover the actions to take when clicking this button
*

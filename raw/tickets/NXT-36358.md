# NXT-36358 — IM - Recipe Rework - Steps & Ingredients - Edit Recipe Steps

## Description
As a Nutritionist I want the ability to edit the recipe steps so I can add additional steps or reorder steps through drag and drop.

[Figma - Edit Recipe Steps|https://www.figma.com/proto/OAZ7qTRgdG5bFFGuhDmkoZ/Item-Management-6.0?page-id=3238%3A54123&node-id=3315%3A63554&viewport=411%2C-15%2C0.03&scaling=min-zoom&starting-point-node-id=3315%3A60028]

*Requirements*

* On the new Steps & Ingredients tab ([NXT-36071|https://primeroedgenext.atlassian.net/browse/NXT-36071]) provide an Edit Steps button which enables the initial edit mode for the Steps card
* Make the recipe steps reorderable through drag and drop in this edit mode
* Display the Edit icon for the steps
** This will enable the Edit Step mode ([NXT-36359|https://primeroedgenext.atlassian.net/browse/NXT-36359])
* In edit mode, display an ‘Add Step’ button in the card header 
** Add Step will insert a new Step with the Directions editable
** Hide Add Step button while in Add/Edit Step mode
* Use inline scroll on the Steps card to manage recipes with many steps
* Require Save if any changes are made (if steps are reordered) 
** Consider using UI behavior to emphasize that user is in edit mode (TBD)
** Autosave when individual steps are added or edited, so that the user is not dependent on this single main Save Steps button 

!image-20230206-194356.png|width=100%!

## Acceptance Criteria
* Load the Steps card in a basic edit mode when the Edit Steps button is clicked
* Steps will be reorderable through drag & drop
* Add Step button will be displayed
* Edit icons will be displayed for each step

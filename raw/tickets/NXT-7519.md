# NXT-7519 — Application Processing - (Action) Cancel

## Description
*User Story* 

As an admin, I have the ability to cancel out of an application.

*Mock Ups* 

*Requirements* 

* Cancel button 
** The cancel button forfeits the application and does not save any data for the admin
** Cancellation message > _Are you sure you want to cancel the changes? The information will not save once you cancel._
*** “Cancel “ > takes them to Application list 
*** ‘Continue Working’ > no information lost, pick up where they left off 
**

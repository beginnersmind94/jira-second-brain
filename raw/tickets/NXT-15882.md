# NXT-15882 — Item Import--Grouping of Fields

## Description
This story is to capture the work @ grouping of fields in Items Import.This grouping of Items will be leveraged for both file based and API Based.

The grouping of Items will be on Both source and Target Mappings. The grouping will be @ accordian and grouped as given below.

## Acceptance Criteria
# Verify and validate that grouping of fields on all File based imports
# Verify and Validate that grouping of Fields is as per the list presented above
# Verify and Validate that grouping of Fields is as on both Source and Target Mappings
# Verify and Validate that the grouping is done and is @ Accordian format.
# Verify and validate if any new field present (other than in meta-data) should be not in any group.
# Verify and validate if the order of the existing fields is changed, the output must not disturb the grouping of the list.

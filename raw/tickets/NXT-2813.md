# NXT-2813 — Template Hub - Filtering

## Description
*User Story*
As an Inspections user, I have a Template Hub Screen where I can view and filter Templates.

*Mockup*
https://m5uloa.axshare.com/#id=7r5wnv&p=template&view=default&g=1

*Requirements*
* There is a hub screen where users can view Templates
* Users view Templates by several categories
** Active
** Inactive
** Global Templates
** Global Template Staging (Cybersoft Only)
* User have options to filter on each page
** Date selection - No date selected by default, uses a calendar widget to select date the template was last edited
** Dynamic filter - Users can search by a variety of parameters
*** Template Name
*** Editor
*** Scoring Type

## Acceptance Criteria
* Template Hub page is present
* Templates can be filtered using all parameters

Note: Dummy data can be used to validate acceptance criteria

# NXT-3411 — Template Hub - Actions - Publish Global Template

## Description
*User Story*
As a Cybersoft Inspections user, I have an action available to publish changes to a Global Template when they're ready to be published.

*Requirements*
* There is an action called Publish Global Template that is only available for Global Templates
** Action is only available to Cybersoft employees with a specific permission (future story)
** Action is only available on the Global Templates screen
** Action is only present if changes have been saved to the Global Template or if the Global Template is currently not published
* Clicking the action brings up a pop up
** Heading:
*** Publish Global Template? if not published
*** Publish Global Template Changes? if published
** Fields are present that correspond with the Update History that customers see
*** Date - Displays current date (read-only)
*** Reason - Text Box
*** Description - Text Box
** All fields must be filled in to allow submitting
** There are two buttons available
*** Publish - Publishes the Global Template
**** Publish cannot be clicked until all fields have text
*** Cancel - Closes out of the dialogue without saving anything
* Once publish is clicked, a dialogue appears telling the user whether it was successful or not
* Once published, all users with proper permissions can access the updated template

## Acceptance Criteria
* Global Templates and updates to Global Templates are able to be published
* Action is present or not depending on context
* Dialogue boxes function properly

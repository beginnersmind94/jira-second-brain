# NXT-5927 — Eligibility - Income Eligibility Guidelines - Get Guidelines

## Description
*User Story*

As a self-hosted district I want the ability to pull the Income Eligibility Guidelines (IEGs) from the hosted platform down to my local install.

*Mockup*

TBD

*Requirements*

* Add a ‘Get Guidelines’ button to the IEGs page in the Edit mode.
** This should communicate out to our main hosted site.
** If the IEGs for that academic year are available, pull a copy down and populate the grids appropriately.
** If the IEGs for that academic year are not available, show the following message:
*** Income Eligibility Guidelines for that academic year have not been set yet.
* User can save these values.

## Acceptance Criteria
Get Guidelines button added for the IEGs Edit mode.
If the IEGs are available, they're copied down. 
Error message about the rates not being available otherwise.

# NXT-22833 — INV - Config - Charges & Discounts - Full CRUD permissions

## Description
As a district admin I want permissions for the Charges & Discounts configuration screen so I can control who can view, edit, add or delete on the page.

*Note:* there are two Charge and Discount pages in the Permissions list; need to figure out why

*Requirements*

* Assign basic CRUD level permissions for the Charges & Discounts screen
** View - access to the page, visible in navigation
** Edit - visible edit icon for charges/discounts
** Add - access to add new charges/discounts to the list
*** Hide the entire ‘Add’ card from the page if the user doesn’t have Add permission
*** Move the list card up on the page
** Delete - access to the delete icon in the list
* Hide the Action column if both Edit and Delete permissions are missing

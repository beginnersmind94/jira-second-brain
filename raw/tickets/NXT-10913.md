# NXT-10913 — Eligibility - Notifications - Page

## Description
*User Story*

As a central office user I want a central location to view and manage all notifications related to eligibility so that I can efficiently and quickly handle this part of the process.

*Mockup*

!image-20210226-161856.png|width=901,height=327!

*Requirements*

* Page is located at Eligibility > Notifications (see routing link)
* Page should contain 4 horizontal tabs: Applications, Direct Certification, Carryover, and Household
* Applications will consist of all notifications related to free and reduced meal applications
* Direct Certification will consist of all notifications related to Direct Certification that are matched and/or extensions
* Carryover will consist of all notifications related to year begin eligibilities that are carried over with the grace period into the new school year
* Outreach will consist of the letters to the household for general information about the programs offered for eligibility (especially for the households who have not applied for benefits)

See links for each page in related links.

*Future Functionality (Not needed now):* After near-term push, we will create an initial Summary portion of the Notifications page to display overall notification statistics and give smart information about notifications that are available to be sent. We can also include a quick way to print all available applications regardless of type and give a summary of what is to be printed and/or emailed. This will allow them to perform daily actions of notifications if they choose to do so.

## Acceptance Criteria
User can access the Notifications page if role permission is assigned
User cannot access the Notification page if the role permission is not applied
User can see 4 distinct tabs for notification types: Applications, Direct Certification, Carryover, and Outreach

# NXT-5054 — Messages - Set Up SMS Integration

## Description
*User Story*

As a PrimeroEdge user, I have a system in place that allows me to send SMS messages when necessary to users.

As a PrimeroEdge developer, I have an SMS service I can hook up to when implementing any relevant functionality.

*Requirements*

Using [Twilio|https://www.twilio.com/]’s SMS service, we want to be able to send emails from PrimeroEdge to external addresses. Set up the service, ensure it can be used to send a test email to a work email, and document any information needed for future development.

## Acceptance Criteria
* SMS integrated into PrimeroEdge
* Audit trail available on whether a message has successfully reached the recipient or not
* SMS able to send messages to mobile phones
* SMS integration documented for use in future development

# NXT-18650 — Acc. - Claims - Claims generated directly to azure blob

## Description
As Vera the Director I want the system to hold onto all claims that I’ve created, and have access to them, so that I can see how numbers are changing.

* ‘Create Claim’ or ‘Recreate Claim’ → create the claim export file based on the current configuration, if available, and place it into file storage.

Claim Export is created as part of Claim Generation, put into file storage.

Export buttons updated to pull from file storage - latest claim for the period.

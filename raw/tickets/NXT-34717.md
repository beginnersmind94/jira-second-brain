# NXT-34717 — MP- Menu and Menu Cycle Analysis- Analyze components with Null counts

## Description
As a Menu Planner, when I first add menu items to a menu, I would like to at a quick glance be able to run my analysis to make sure that I am offering enough components to pass USDA regulations.

## Acceptance Criteria
* When analyzing menus or menu cycles, include menu items with null counts as part of the component analysis
* Null menu item counts as well as regular count should be considered
* User must specify a menu item as not served to not include in analysis
** If user enters 0 for a menu item count, do not include as part of component analysis
* This will also effect Menu Planner Component analysis

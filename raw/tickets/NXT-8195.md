# NXT-8195 — Claims - Claim Detail - SFSP Summary

## Description
As the FSD I need to see a summary of the reimbursement claim created. The summary is a total of all sites included in the claim

*Mockup:*

[^SFSPSummaryDistrictClaim (1).pdf]

*Requirements:*

* Format the display to show the programs, meal counts and amounts for reimbursement 
* Calculate number of Free and Reduced Eligible students

## Acceptance Criteria
Claim type SFSP the claim data for the district is shown in the format shown.
General Information contains a list of SFSP programs the district participated in
Each row shows the Number of schools participating in the program, number of days in operation for the claiming period, and average daily attendance.
The number of students eligible for Free and Reduced meals
Meal Counts for each classification are shown for each Price Type, State and Federal rates are applied to the counts and totals for each price type are calculated.

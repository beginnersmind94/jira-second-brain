# NXT-4081 — Authentication - Active Directory

## Description
*User Story*
As a PrimeroEdge admin, I have a tool that automatically imports user data from my system into PrimeroEdge to automatically set up and update users to allow for my team members to use the same sign on credentials and have the same data without having to double work.

*Requirements*
* Users have a way to utilize Microsoft Active Directory to bring in user data
* Users are able to download a tool that allows them to setup server and sync settings
** User can setup server settings to allow the AD tool to properly connect
** Users can select what user data is transferred
*** Name
*** Email
*** Password
*** Status (active/inactive)
*** Other settings as needed
* Users can set up automatic sync or manually sync
** Automatic sync is setup with a time to conduct sync and a cadence (every x days)

## Acceptance Criteria
* Active directory sync and tool available and functioning
* Settings able to be configured and applied

# NXT-31884 — All Site Dropdowns and Site Configuration - Restrict list based on user

## Description
Sites should be restricted based on the user’s role.

## Acceptance Criteria
* On the Site Configuration page, only SchoolAdmin and Director roles shall be able to add, edit, and delete new Sites.
* On the Site Configuration page, the list of sites that are visible shall be restricted based on those the user has access to.
* On all Site dropdowns throughout the application, the list of sites visible to a user shall be limited to those the user has been granted access to.

# NXT-4989 — AcctMgt - Add Patrons - Students - Wizard Step 6 - Documents & Comments

## Description
*User Story*

As a district admin I want to attach documents and add comments for a student patron via a wizard.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418434%3A236821&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=418434%3A236821&scaling=scale-down-width|smart-link] 

*Requirements*

* User needs a way to upload documents via the final step of the ‘Add Student’ flow.
* User should be able to leave comments as well.
* User should be able to remove documents.
* Documents must be stored appropriately for this exact student.

We need to keep in mind:

* Allowed extensions: .txt, .csv, .doc, .docx, .pdf, .jpg, .png, .bmp, .msg, .xls, .xlsx
* Max file size: 5 MB
* User needs to be informed appropriately if the document breaks any of these rules.
* The user could abandon the ‘Add Student’ flow - if they do, we shouldn’t keep any uploaded documents on our servers.
* Allowed extensions: .txt,.csv,.doc,.docx,.pdf,.jpg,.png,.bmp,.msg,.xls,.xlsx

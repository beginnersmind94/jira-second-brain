# NXT-36070 — IM - Recipe Rework - Nutrition Info tab - Serving Measures

## Description
As a Nutritionist I want the Recipes page to have a dedicated Nutrition Info tab so I can recipe nutritionals without having to scroll past all the steps and ingredients.

*Requirements*

* Redesign the Recipe details page to appear similarly to the Item overhaul design
* Create a tab based design to separate various Recipe sections
* Create a Nutrition Info tab to house all the calculated recipe nutrition info in the Item design format
* Display *Serving Measures* card first just as on Items 
** *Effort: 5*
** This section is *not* user editable though (at least until some changes for Menu Items)
** Auto insert Serving Size created by Recipe
** Auto insert entire recipe as a serving size in the grid
*** this should be only for viewing purposes; not a real serving size unless the user creates a MI like that
** Hide the Add row button
** Set the height of the card to be shorter
*** either dynamically sized with a maximum or set to the height of the Yield Factor card
** *Remove/Hide* Preferred Measurement and Possible Measurements
** *Add* a Portioning Tool column (in place of one of the removed columns)
** *Allow Edit* only for Portioning Tool value

# NXT-24566 — Blank page is appearing when Guest user trying to access any of the application pages through and no network call recording in backend.- Global issue for all modules

## Description
Blank page is appearing when Guest user trying to access any of the application pages through and no network call recording in backend.- Global issue for all modules


Steps to Reproduce -

1. Launch the application.
2. Hit any internal page URL through Guest user and observe the appilcation response.


Expected Result -
Guest user should redirect on Access Denied page with "Go Back to Login" button when trying to hit any Application internal page URLs.


Video URL of bug - https://www.awesomescreenshot.com/video/6674405?key=2cbb5c997a5a0eaeb0f49cc1dbc9a3e6

## Acceptance Criteria
The user is redirected to the login page when browse to a page and doesn't have an active session

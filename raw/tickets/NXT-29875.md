# NXT-29875 — PR- Production Record- Allow Edit for Quantity Prepared

## Description
As a production Manager, I would like to have the ability to edit my quantity prepared information that is populated. This information is just for my information and to be saved for audit purpose but not effect withdrawals.

## Acceptance Criteria
Give the user a way to edit the quantity prepared information

This will only affect the text and not affect anything related to withdrawals or counts. 

Display that the quantity prepared was modified and who and when it was modified.

IF user enters a new produced count, override what the entered on the edit quantity prepared

User has opportunity to revert back to system generated with radio button

# NXT-4118 — Security - Device Whitelist UI

## Description
*User Story*
As a PrimeroEdge admin, I have a UI screen that allows me to enable and disable device whitelisting, as well as adding and removing whitelisted devices.

*Requirements*
* Device whitelist settings can be updated through security policy
* By default, all devices are allowed to access the site
* There is a toggle to enable/disable the device whitelisting
** The toggle is grayed out until at least one device is added
** Once enabled, any sessions are exempted until they expire
*** A warning indicating this is presented when the whitelisting is enabled
* There is a button for adding devices
** A popup appears for adding devices
** User adds one device at a time
** Devices are added by MAC address
*** There is an option to add device name and user
* Added devices are displayed with MAC address and any other information added
** Devices can be deleted
** When deleted, the user is required to reenter password
** If removed, any session on the device are exempted until the session expires

## Acceptance Criteria
* Device whitelisting is present when creating or editing a security policy
* Devices can be added and removed through the UI and can access the site when whitelisted
* Whitelisting can be enabled and disabled through the UI

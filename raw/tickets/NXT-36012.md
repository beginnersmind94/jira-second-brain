# NXT-36012 — MP- Menu Planner- Update Analysis daily and weekly to consider meal component/ nutrition of serving group

## Description
When making all the changes to the menu planner, we were able to handle the ability to analyze the nutrition based on a different serving sizes on different serving groups. However we need to also be able to analyze the menu item with the serving size and meal contribution associated with it.

## Acceptance Criteria
* Analyze the menu or weekly analysis based on the serving size meal contribution
** Currently, we are analyzing it based on the default serving size meal contribution no matter what serving size is selected per serving group.
* Analyze the menu or weekly analysis based on the serving size nutrition
** Currently we are analyzing based on the default serving size

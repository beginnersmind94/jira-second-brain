## Acceptance Criteria
User can select to see the Meal Counts by Day/Month/Date Range
User can select to show as summary or detailed (detailed is by date)
User can select to show counts by Serving Site or Enrollment Site.
Meal Counts are totaled by meal types, for each eligibility type and site.
When Detailed is selected the list is broken down by day.

# NXT-30132 — IM - Items - Create New Item - Non-Food Item - Menu Item Info

## Description
As a District Admin I want the system to provide a non food option so I can create non food items but use it as a menu item

*Requirements*

* If user specifies the item as *not* a food item AND *is* a menu item based on the questions asked in ([NXT-30131|https://primeroedgenext.atlassian.net/browse/NXT-30131]), load the Serving Measures card, Menu Item Info card, and Allergen Info card
** color the tab as red to indicate missing mandatory attributes
** Display Serving Measures in “Edit/Create” mode as soon as the cards are loaded to indicate that this should be entered first.
** Add a new field on the Menu Item Info card that will display Menu Item Cost
*** Cost calculation is the same as the recipe cost, it will however take the serving measure weight for specified to give Menu Item Cost. Fair Market value (always assume association with highest unit weight) / serving size weight= Cost per serving for Menu Item
* All three cards can be worked on in any order as they do not relate to each other
* Allow creation of a non food item to a menu item without the user having to create a single item recipe
* Once any card is saved, change the tab color to yellow to indicate that some info has been entered but not all

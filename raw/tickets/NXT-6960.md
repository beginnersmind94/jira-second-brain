# NXT-6960 — SE - Eligibility Guidelines - Alaska PFD Amounts

## Description
*User Story*

As an Alaskan district admin I want to see / edit the Alaska Permanent Fund Division (PFD) so I have confidence the program is using the latest published numbers.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=6609%3A42&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=6609%3A42&scaling=min-zoom]

Current page - [http://primero20qa.centralus.cloudapp.azure.com:3002/system/eligibilityGuidelines|http://primero20qa.centralus.cloudapp.azure.com:3002/system/eligibilityGuidelines]

*Business Information*

* Alaska has a yearly payment that goes out to residents - it’s from a deal they made regarding their oil reserves.
* More reading for anyone curious - [https://en.wikipedia.org/wiki/Alaska_Permanent_Fund|https://en.wikipedia.org/wiki/Alaska_Permanent_Fund]
* This amount needs to be considered for free/reduced applications in Alaska.

*Requirements*

* We need to add a PFD tab to the Guidelines page, if the State Category is set to Alaska.
* On this page, we should show the amount for the current and subsequent year.
** Examples - the academic year is 2020 - 2021.
** We need to show an amount for 2019 and 2020.
** AY 21-22, show 2020, 2021.
** AY 22-23, show 2021, 2022.
** AY 23-24, show 2022, 2023.
* This should be editable. Both figures can be edited from any year.
** Consideration: The value is a set amount each year - the 2019 value won’t be different in AY 2019-2020 vs AY 2020-2021.
** Min: $0.00, Max: $100,000.
** Tie this to the Edit permissions added as part of [https://primeroedgenext.atlassian.net/browse/NXT-5926|https://primeroedgenext.atlassian.net/browse/NXT-5926|smart-link] 
** Save/Cancel buttons.
*** If a value already existed for one of the years, prompt the user for confirmation.
**** “A PFD amount for year 2019 already exists. Are you sure you want to replace it?”
**** Options: Cancel / Replace

## Acceptance Criteria
PFD values show for the Alaska State Category.

A year's particular value should be tied to that year regardless of selected academic year - we shouldn't show different values for 2019 in AYs 2019-2020 and 2020-2021.

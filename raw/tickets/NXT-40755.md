# NXT-40755 — MP- Find and Replace- Do not allow Production record complete menus to be updated

## Description
As a menu Planner, for me to stay in compliance for auditing, I must keep the integrity of the menu I used on the production record. Currently, we are not allowed to change menus except from the Find and Replace feature. This needs to also look at production record complete menus and not allow the replacement

## Acceptance Criteria
* Do not allow menu items on Menus that are production record complete to be replaced
** Gray out the selection box so that the menu can not be selected
** Provide warning on the menu to show which menus have record complete productions using them
** Provide tool tip “One or more Production Records have been complete using this menu, menu item can not be replaced”

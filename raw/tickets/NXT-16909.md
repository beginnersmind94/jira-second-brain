# NXT-16909 — INV - Manage Orders - Order Detail - Edit Order - Apply Order Review rule checks to order updates

## Description
As an operations manager I want the Order Review rules to be applied to orders when they’re being edited, so that I can ensure that all applicable orders go into Pending Review status.

*Requirements*

* When user edits an order, check if any Order Review rules apply to the updated order.
** For each rule, check each component of the rule (And condition)
** The date range of the Order Review rules will apply to the order’s *Delivery Date*
* If any review rules apply to the order, put the order in Pending Review status
** on Save

## Acceptance Criteria
* Apply Order Review rules to orders that have been edited
** on Save
* Change order to Pending Review if a rule applies

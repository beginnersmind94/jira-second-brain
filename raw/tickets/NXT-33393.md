# NXT-33393 — MP- Menu Planner- Daily Milk Varieties and Unflavored Check Meal Pattern 14+ (UI Only)

## Description
Currently, when adding a  milk menu item, we are turning the component on which is current as far as passing the fluid milk, however the varieties of milk and unflavored milk offering could still be failing

## Acceptance Criteria
* Once user drops 1 menu item that contains a the minimum daily requirement for milk based on grade group , only HALF fill the milk component icon
* If the user hovers over the milk icon to see what is missing and why the icon is not filled fully
** IF Fluid Milk not met “Daily Fluid Milk not met” >=1
** IF milk varieties is not met “Milk varieties are not met” >=2
** IF no unflavored skim milk “ Unflavored 1%/skim milk offered not met” >=1
*** There can be both of these warnings at the same time
* *Reorder the daily requirement to show Fruit, M/MA, Grain, Milk, Veg*
* This effects All serving groups EXCEPT PK

# NXT-32169 — Workspace - New Message UI Improvements

## Description
The “New Message” UI shown below requires some improvements.

!image-20220714-123522.png|width=669,height=614!

+*Screenshot showing odd users in the user lookup.*+

!image-20220714-125539.png|width=1666,height=938!

## Acceptance Criteria
* The text of the “To” box shall be “To:”
* The “URL tool” popup shall be corrected to completely fit on the UI
* The “Attach Image” icon to the right of the “Send” button shall be made to work.
* The user shall not be able to paste messages into the message body. Currently, messages with images do not reach their recipients' inboxes.
* In the “To:” box, when the user lookup is displayed, only users within the district that show on the Users page shall be displayed. See attached image for the list of odd users showing up in the user lookup.
* Size validation shall be added for attachments.  If the size of an attachment will lead to a message that cannot be sent, the user will be informed, after clicking on “Send”, that the attachment is too large.
** Message: “Attachments larger than \[_X]_ are not supported”
*** Where “\[X]” is the size limit.  1 MB, 500 KB, etc…

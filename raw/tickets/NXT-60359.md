# NXT-60359 — IM- Item Onboarding- Pack Size Info- Cost/Value

## Description
As a district admin using the bulk onboarding tool, I would love a way to bulk add my Fair Market Value for all the items I am trying to create.

!image-20250429-071446.png|width=1366,height=706,alt="image-20250429-071446.png"!



Missing cost on the review screen

!image-20250429-071720.png|width=111,height=252,alt="image-20250429-071720.png"!

## Acceptance Criteria
* The “Cost/Value” button from the Review Pack Size info screen will lead to this step
* Provide a drop down to allow user to select which pack size to add the cost to
** Cost can only be added onto 1 unit will scaled up or down
*** Cost text box only available after selection is made
** Do not prefill with any unit, user must make a selection
* Provide an alert if row is worked through and no cost is entered 
** Tool Tip “Missing Cost”
* On the pack size overview, add a cost identifier to the pack that has cost added
** Dynamically update once cost is added
* Only show items that have full pack size information
* If Cost unit is selected but no cost is entered show Missing Cost in red on the Review Screen
* If Both are missing, show missing cost in grey on the review screen

# NXT-69495 — IM - Bulk Item - Serving Info Review with Identifiers

## Description
As a District manager using the Bulk tool, I would like for a way to see my servings on a review screen and what my servings are associated with whether it be my nutrition, contribution recipes etc. 

!Multi Servings Review-20260119-142431.png|width=536,alt="Multi Servings Review-20260119-142431.png"!

## Acceptance Criteria
* Remove Update Serving Info (Update Session)
* Display new Column called Servings & Identifiers 
** This will replace the Serving size and description and Weight and Measure columns  
* Display 2 servings and identifiers associated with each (If space allocates for more)
** Display priority should be based on serving with most identifiers
*** If all have one, use priority as follows: nutrition, contribution, used in recipe, used on menu
*** If none, show servings by highest gram weight 
*** Any extra servings will be displayed within the View/Edit slide out for each item (In Manage Servings Story) 
**** Use same logic as units card for display order
* Update servings with identifiers when changes are made on the nutrients, contribution or menu item sections

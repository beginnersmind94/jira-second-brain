# NXT-31830 — MP- Menu- Standard/Menu Planner View or Edit Option

## Description
As a Menu Planner, I enjoy the interactive nature of the menu planner and would like to be able to edit my menu after I create it. I would like to be able to come back to the menu Planner screen and view or make my edits from the Menu Planner screen and not just the edit menu page.

## Acceptance Criteria
* Remove link from Text to go to the menu
* Give a view  and edit option from menu page
* Give action column that will have both a view and an edit icon 
* Each icon will have a standard (Edit Menu Page)--Change when in view mode to View Menu and Menu Planner (Menu Planner Page)
* View: User can only view the menu from either option but will not be able to make any changes or edits. Everything is in view mode ([NXT-33757|https://primeroedgenext.atlassian.net/browse/NXT-33757])
* Edit: User can be able to make all the changes on either page, Everything is editable as usual ([NXT-31834|https://primeroedgenext.atlassian.net/browse/NXT-31834])
* Freeze the Actions column to always show as the last column in the grid, user can not remove it.

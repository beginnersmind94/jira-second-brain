# NXT-3021 — View Inspections - Actions - Parent Inspection Report

## Description
*User Story*
As an Inspections user viewing an Inspection that has gone through a Re-Inspection in the View Inspection screen or on the Inspection Details screen, I have an action that allows me to generate a report of the Inspection with its previous results.

*Requirements*
* There is an action for viewing the results of the original Inspection or Inspections
** This action is present on any Inspection that has entered Re-Inspection and any further statuses
** This action is present for any users who can view the Inspection and have report permissions
* The report is available in PDF
** When clicked, the file is generated and downloaded to the user's machine
* If the Re-Inspection has not been completed, the questions marked for Re-Inspection are indicated on the report
* If the Re-Inspection has been completed and submitted, the new and old responses are both displayed, with the old responses marked
** If multiple Re-Inspections have occurred, all responses are included

## Acceptance Criteria
* Parent Inspection Report can be generated in both file formats from the View Inspections screen in either view (card or list)
* Parent Inspection Report can be generated in both file formats from the Inspection Details screen

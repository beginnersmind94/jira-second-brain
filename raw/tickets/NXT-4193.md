# NXT-4193 — Item Management - Items - Non-Food (Phase 1)

## Description
*User Story*

As a super user, I need to be able to add a non food item and put in the necessary information

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=14&view=default|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=14&view=default]

Click Add New - Non Food Item. Then create the item. The 2nd page will have the common information as well as the below information.

*Requirements*

* Item Name
* Item Short Name
* -Item Form-
** -As Purchased-
** -As Served-

*This information would be below the name*

* Item valuation group
* Item type
** USDA Commodity
** Non-Commodity
** Non-Food (Only option when a ‘non food’ item)
* Item category
** Standard list from classic
*** User Configurable
* Purchase / Whole units
** standard list of units (eg. Case)
* Broken units
** Broken Units per Whole Unit
** Broken Unit Description
** Broken Unit Size
** Broken Unit Weight
** Orderable in broken units
*** Yes
*** No
* Storage Category
** Cooler
** Cooler - DSI=7
** Dry
** Freezer
** Freezer - DSID=7
* Fair Market Value / Standard Price
* Net off invoice
* Internal vendors only?
** Yes
** No
* Buy American or not?
** If  not selected, upload exemption letter button
* Bid category (Only if the user has Bid Analysis, so should be hidden at this time)

*Common Info section and attachments / Images on the right*

Save and Close returns the user to split view and viewing the item created.

## Acceptance Criteria
The item is added as a non food item and returns the user to the split view and viewing the item created

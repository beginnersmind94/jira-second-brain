# NXT-70296 — FIN - Add Cost of Foods Used to Profit & Loss Report

## Description
*As a* District Finance Director, *I want* the Profit & Loss (Revenue & Expenditure) Report to include a detailed "Cost of Sales" section that breaks down the components of Cost of Food Used (Beginning Inventory, Purchases, Ending Inventory) based on the mapped System Entry Types, *so that* I can see the calculation of expense, not just the final number.

## Acceptance Criteria
# *Report Structure*

* Add a "Cost of Food Used" subsection in the P&L.
* Group by Valuation Category (Food & Non-Food) as already defined in the Inventory Usage Report.

# *Line Items (sourced from Inventory Usage Report)*

* Line A — Beginning Inventory
* Line B — Purchases
* Line C — Additions
* Line D — Net Transfers (Transfers In − Transfers Out)
* Line E — Ending Inventory
* Line F — Cost of Food Used ({{A + B + C ± D − E = COFU}})

# *Data Dependency*

* All Cost of Sales data is fetched live from the Usage Report. The P&L does not independently calculate these values in columns A - D.

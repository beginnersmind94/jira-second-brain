# NXT-55591 — FO - Background Jobs Monitor - Framework for Monitoring, Alerts

## Description
FO team is to build a small framework for tracking and monitoring of specific background jobs with basic functionality as outlined below.

Order

* Build the framework for monitoring and alerting on failure (this story)
** This includes adding a page to let SchoolCafe Admin roles see job performance for the entire installation, not region specific
** Though there are a handful of region-specific jobs that also need to be monitored for failures.
* Use the framework for Eligibility Jobs + Reminders (this story)
* Gradually roll it out for the remainder of jobs (future stories) 
** Move job history out of Couch if it’s still in there



*requirements*

* Build framework for monitoring + configuring who gets alerts
** Monitoring runtime, history, failures
** -Setup to alert configured users if the job fails to run after X minutes-

Requirements

* Add a monitoring page to track the jobs *across all regions*
** Tracking:
*** Job name
*** Current Status
*** Execution/Run History
* -Ability to setup who is notified (select from users or add emails) if a service triggers an alert:-
** -Fails-
** -Fails to run after X minutes-
*** -Define how long that is-
*** -Setup who receives the alert-
** -Runs for too long-

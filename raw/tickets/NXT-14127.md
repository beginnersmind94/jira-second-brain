# NXT-14127 — SPIKE - Platform - Enable SSO for Google and Microsoft 365

## Description
*Persona*: Perseus User/admin

*Access level:* School District/Site

*Description*:

As a Perseus user, I should be able to login to Perseus using my registered account using my ISD’s Google/MS 365 domain configuration

*Requirements*:

* User logs into their MS or Google account with their registered email account
* If registered email is not found, prompt for invalid email account and redirect them login page
* If registered email is not set for a district, launch manual login page

## Acceptance Criteria
* User shall be re-directed to respective SSO provider based on the registered email address
* User shall be prompted with authentication setup based on the domain preferences including 2FA
* User shall be prompted for manual login if the registered email address is not setup in Perseus system
* User shall be prompted for manual login if the domain provider fails to authenticate

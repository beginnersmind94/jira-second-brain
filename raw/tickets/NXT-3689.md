# NXT-3689 — Authentication - Send password reset email

## Description
As a user I want to receive a randomly-generated password to log in to PrimeroEdge on behalf of my system administrator so that I can use it to log in to PrimeroEdge and set a new password.

*Requirements*

* In the User Details page, Administrators can select a Reset Password button to generate random password and email user credentials”. 
* Users should be able to receive this password via email, since they will need it to set a new password in order to continue with authentication.
* The goal of this story is to trigger the email and send it to the user
* Requirements for the Reset Password email are similar to the New Password Email story:
** Email looks professional and clean
*** PE Logo
*** Text:
**** *Email Subject*: Your password has been reset
****Email Text*:
PrimeroEdge logo (centered)
****Header* (centered): Hi <First Name>,
<Admin First Name Last Name> from <schoolDistrict> has reset your password on your behalf. To continue, click the button below to set up a new password for your account:
****Button*: Change Password
** When the user clicks the button/link, they should:
*** be logged in to PrimeroEdge in their web browser 
*** be taken to a Set Password screen (covered in NXT-4023)
*** have the link expire (it does not need to expire initially like the New User link)
*** not have access to any other screens
** A resend button will be covered in another story if needed

## Acceptance Criteria
In User Details, when an administrator selects the Reset Password button, send an email to the affected user that contains:

Text explaining the reset password (shown above)

A button to immediately log in and set their new password.

Once this is complete, verify that the user's password is updated successfully, allowing them to log in with it moving forward, and that the link in the email has expired and takes the user to the expired link page when selected (NXT 7658)

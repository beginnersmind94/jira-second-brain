# NXT-26911 — IM - Variety Menu Item History (Detail Card)

## Description
As Item Manager administrator I need an option where I can observe the history of Variety Menu Items Details for example  when it is created, updates made to the item by who and when so that have an audit process

## Acceptance Criteria
* *Entry Point*
** Item Management → Menu Items  → Variety Menu Item → History icon

* An option to view  Variety Menu Item details history shall be available to the user under the history icon
* User shall be able to see the following Menu Item property values:
* When it was created
**  (system date/time)
**  Who created it
* Updates
** System date and Time on which the update took place
** Who performed the update (User Name)
** What fields were updated, preview value(s) and new value(s)
* This information is for display purposes (Shall not be alter by the user)

*General*

Page pagination shall be present (as standard way)

No primero color to be use on “More Options” button

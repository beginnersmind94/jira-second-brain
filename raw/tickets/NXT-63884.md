# NXT-63884 — SC - Cybersoft Catalog - UI Changes (Filter, Data Query Framework)

## Description
_(none)_

## Acceptance Criteria
*Navigation*: SC - Items - Item Upload - Item Search

*Changes Required:*

The data filters should be based on dropdown for Brand, Manufacturer, State, Vendor. User should be able to do multiple selection on such parameters

The user should be able to filter and sort the data at column level - enable filter at all columns when the user clicks on Filter Icon

*Import Button* - when the users selects master check box, select all the records in the entire result set instead of just 20 records

*Data Query Framework:*

The user should be able to use the data query framework on the following Fields:

* Item Name (Default Selected, user cannot remove this)
* Manufacturer  (Default Selected, user cannot remove this)
* Brand  (Default Selected, user cannot remove this)
* GTIN
* Item Category
State
* Vendor
Product Code
* Storage Category
* Valuation Group

# NXT-27641 — Platform - Security - Audit Custom Role Updates

## Description
As a SchoolCafe admin, I would like to see all updates on custom role as audit notes with when, who and notes updated. 

+*Requirements:*+

* System need to record the creation of custom role with created user, timestamp and event notes
** System need to record any updates on the custom role with history

## Acceptance Criteria
* User shall be able to see the audit history for custom role creation, updates
* User shall be able to see the audit history icon in actions column which brings up audit drawer
* User shall be able to see the audit history buttons in actions column both create and update custom roles

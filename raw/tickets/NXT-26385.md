# NXT-26385 — INV - Receipt/QR/Addition - Date Slideout - Enhancement for default bin line

## Description
As an Inventory user I want the default, uneditable Date on Product record to be displayed differently so I understand it is an automatic record

*Requirements*

* Show the default, uneditable record differently from other records
** Label the record ‘Default Record’ instead of Record - 1
** add a tooltip: ‘Items will be associated with Received Date if no other date is provided.’
* Retain ghost text but populate fields
** display transaction date as Date on Product
** display ‘Received Date’ as Date Type (this is not a user selectable value)
** display --- for Lot Number
*** or hide the field
* Hide date picker and dropdown icons
** display as labels
* Change logic to require Save button
** do not autosave as currently doing
** show unsaved changes alert on Close

+Overall Slideout Enhancements:+

* add a save button

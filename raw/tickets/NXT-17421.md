# NXT-17421 — Import-Adult

## Description
This story is to cover the tasks related to Patron Import @ SIS Import.



Patron Import consists of two components, student and Adults. Adults are further divided into two as Adults and Employees (Formerly Program Adults).

User while importing the Patron Import file has the option to select adults or employees. The fields to be mapped change based on the options selected (adults or employees) under the Import type = Patron Import.

Patron Import is also a simple and straight forward import similar to that of the student import.

# NXT-66962 — LMA - Insights Site Licenses

## Description
As an Insights user, I want to be able to see only the sites that I have applied a license to so that I can ignore the extra noise.

*Notes*

* Currently SC set Insights to be only District level license
* Now sales team wants to sale by site or district
* So we need to update the code so that Insights can be either District or Sites

*Requirements*

* In the Generate License Screen, update the drop down filtering for Insights to also have Site selection available
* Make the same change everywhere you have license selection such as the Edit and Add screen
* Ensure that everywhere we show a list of site licenses, it shows up there if they have selected Site
* If you have done the screen to apply sites to a license, then make sure it works on that screen

## Acceptance Criteria
* Users can select Site for Insights license
* Everywhere license selection is done, we update that code
* If applying a site to a license is already done, ensure it works for Insights also

# NXT-66301 — Sys - Special Programs - Override % options

## Description
As Dana the Director I have a set of claiming percentages from my state department and just want to enter those numbers into SchoolCafe for accuracy.

*requirements*

* User can go out to 5 decimal places for all percentage numbers in Special Provision programs. (ex: 45.12345)
* User can ‘override’ the claiming percentages as part of adding / editing Special Provision programs.
** Add an Override Percentages button
** User can then edit the Free & Reduced percentages manually
*** User cannot enter any number >100
*** Paid calculate as 100 - (Free + Reduced)
**** User can still edit if necessary
*** Total Claiming (F + R + P) percentage must = 100 or user cannot save

## Acceptance Criteria
* User can manually enter Claiming Percentages for CEP / P2
* Entry added for History to note the percentages were manually set
* Verify the percentages save successfully
* Verify the percentages are used successfully for reports / claims

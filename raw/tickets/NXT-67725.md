# NXT-67725 — GC >GDSN Connect > SC >  Upload items page > Item Updates tab :  Single item updates: Mark as reviewed button functionality

## Description
GC >GDSN Connect > SC > Upload items page > Item Updates tab : Single item updates: Mark as reviewed button functionality

## Acceptance Criteria
Functional flow: Users navigate to upload items > item updates tab

# Item updates tab: select an item by clicking on item hyperlink
# Review the available updates for all sections (item details , nutrients, allergens, meal credits , images etc..)  
# Select the available updates for one or more fields that they want to accept from all tabs 
# click on Mark as Reviewed - 
## Ask for confirmation, “Are you sure you want to change the status” Clicking on NO, will cancel the action. 
## click on Yes: will change the status following the below 2 conditions
###  if at least one update is not selected for overwriting: status changes to “Partially Reviewed” in the grid under status column. 
### if all available updates are selected, then it will turn the status as “Reviewed” 
### Do we have system setttings? just like how we have them in PrimeroEdge?
## Results grid should provide a column filter to allow them select all reviewed items and select bulk update all of them at once.

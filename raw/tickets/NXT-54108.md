# NXT-54108 — Acc - Sessions - Closing Balance - Only add Check/Card After Hitting Save

## Description
!image-20240821-202638.png|width=1027,height=558,alt="image-20240821-202638.png"!

*Current Functionality*

* Hitting Save on this will close the Closing Balance screen & refresh the Reconciliation page
* Closing Balance total is updated even though this session is still Open

*Expected Behavior*

* Hitting Save on Add Check/Card should return the user to the Closing Balance screen
* Nothing should actually save until Save on Closing Balance is clicked

## Acceptance Criteria
* User is returned to the Closing Balance screen after clicking on Save for Add Check/Card
* Session is not updated, orphaned check/card is not added, until the user clicks Save on Closing Balance

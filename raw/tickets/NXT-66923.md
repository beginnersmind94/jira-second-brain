# NXT-66923 — Acc - Menu Item Sales - Terminal dropdown changes, Group By Terminal report change

## Description
As Sally the Site Manager I want to run the Menu Item Sales report to populate my Production Record but the terminal listing makes that difficult for me.

*requirements*

* Terminal Dropdown:
** Default, select none
*** Treat None as All for purposes of the report
** List Active terminals first, then Inactive terminals 
*** Would be nice to have an indicator in the dropdown list for Active Terminals vs the Inactive Terminals, even if it’s just a non-selectable line like this:
**** Terminal ABC
**** ---
**** INACTIVE TERMINALS
**** ---
**** Terminal XYZ
* Report option: Group Type → Terminal
** Change the group by ordering. Right now it’s Menu Item > Site > Terminal for the Daily Summary
*** Change ordering to Site > Terminal > Menu Item
** Add the Terminal Group Type option to the Item Summary report
*** Order: Site > Terminal > Menu Items
* Filter updates:
** Some dropdowns should work as filters
*** No default value selected.
*** Only use them if the user selects a value.
*** If the user selects no values, ignore that filter and proceed with using the other filters.
**** Examples:
***** User selects nothing from Grade → do not filter based on Grade, use all other selected filters.
***** User selects nothing from Menu Item → no filtering happens on Menu Item, so we would return all menu items based on the other criteria.
***** User selects nothing from Homeroom → do not filter results by homeroom.
***** User selects nothing from Terminal → do not filter results by terminal.
*** This is primarily targeted at the Menu Item dropdown not showing inactive menu items - the report does not show them as a result of the selected values.
**** The user needs to be able to see everything that was sold on a date.
** These dropdowns should be changed to work as filters:
*** Grade
*** Homeroom
*** Menu Item
*** Terminal

## Acceptance Criteria
* User can see Active vs Inactive terminals in the Terminals dropdown
* Daily Summary report
** Group By Terminal updated to be higher in the order
* Item Summary report
** Group By Terminal option added
* 4 dropdown filters updated to act as filters
** No default selections
** Only used as part of the query if values are selected

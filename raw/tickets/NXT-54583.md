# NXT-54583 — PEI - Auth - Add Insights Link in PE

## Description
As a PrimeroEdge Insights (PEI) user, I would like to be able to click on a link in PE and be directed straight to the PEI website so that I don’t have to re-login on that website and save time

*Requirements:*

* Add a link in PrimeroEdge main navigation to the new website
* Put between System and SchoolCafe in the module on top (see pic below)
* Button name: Insights
* Only show the button if the district has a valid license for the Insights module
** Note there is [another story|https://primeroedgenext.atlassian.net/browse/NXT-54659] to add Insights to the licensing in PE
* User should be taken straight to the PE Insights website from the button click
** Prefer to open them in another window/tab
* 
* Security Trim based upon role access
** In order to visit the site, the user must be in a role that has rights to the Insights module based upon role membership
** There is [a separate story |https://primeroedgenext.atlassian.net/browse/NXT-54660]to create the module in the roles permission page
** The hard requirement is to not allow a user to go to the new site if they don’t access via the role
** To do that, either hide or disable the button from users that do not have the role or capture the event click and check for role access before redirecting
*** The preferred method is to hide or disable the button
* 
* Link Configuration:
* Store the link to the new website in a way that is configurable per district in case we end up with multiple Insights sites
** Do not hard code the link in code 
* 
* The link needs to pass on the user profile information, authentication, etc. as part of the SSO process, but this is covered in [another story|https://primeroedgenext.atlassian.net/browse/NXT-54593]

## Acceptance Criteria
* If the district has the Insights license, then the button is shown in the menu at top after System
* If the user does not have rights via Role access, they cannot be redirected to the new site
* Clicking on the link takes the user to the new website

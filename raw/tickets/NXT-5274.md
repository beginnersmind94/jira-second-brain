# NXT-5274 — Verification - Tracking - App. Information - Rollback to Incomplete

## Description
As a district admin I need a way to mark an application as incomplete after it’s been completed.

*mockup*

[https://www.figma.com/proto/Y3tlwN2yvRc67304adzyyB/Eligibility-3.0?node-id=1499%3A0&scaling=scale-down-width&page-id=1110%3A0|https://www.figma.com/proto/Y3tlwN2yvRc67304adzyyB/Eligibility-3.0?node-id=1499%3A0&scaling=scale-down-width&page-id=1110%3A0|smart-link] 

*reqs*

* Inside the App. Information of a completed application, add an action.
** ‘Rollback to Incomplete’
*** Confirmation prompt - ‘This will return the application to a status of being incomplete and may impact eligibility. Are you sure you want to proceed?’
*** Comment box.
*** Options: CANCEL / CONFIRM
**** Confirm should do the following:
***** Roll the application back to being incomplete, clear out any completion dates.
***** -Add history for this action having taken + who + when. (will be part of another story NXT-16954)-
***** Change the eligibility of the student(s) involved back to what it was previously.
***** Provide the user a success message.
****** Application #12345 has been rolled back to an incomplete status.
***** The page that they’re on (App. Information) should update to reflect the changes also. Might just need to reload.

## Acceptance Criteria
User has an eraser to return an application to an incomplete status.
History is updated for this.
Eligiblity for students is updated appropriately.

# NXT-2456 — Inspections - Create Inspections APIs - Start Now

## Description
*User Story:*

As an Inspections user, I have a Create Inspections slide out that allows me to properly start a new inspection.

*Requirements:*

* Users can create an Inspection to start immediately
** Requires Create New Inspection permissions (Inspector, Supervisor, Admin)
** Will start immediately if the “Schedule for later?” checkbox is unmarked
** The following fields are present and required:
*** Inspection Name
*** Template
**** Only Active Templates are present
*** Site
** Once the Start Now button is clicked, the Inspection is created in an In Progress status

* The Inspection is automatically assigned to the user who created it.

## Acceptance Criteria
API developed for Create Inspections slide out for properly creating an Inspection using start now

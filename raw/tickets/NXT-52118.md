# NXT-52118 — MP- Menu Item Help Button- Showing ingredient code when item is used as an ingredient as part of a recipe and dmi rules API AND UI)

## Description
When looking at the menu item help button, it would make more sense to show the ingredient code when using an item as an ingredient as part of a recipe rather than the menu item code which is the code used for the finished product.

## Acceptance Criteria
* When item is used as an ingredient as part of a recipe, currently we are showing the menu item code if it is a DMI
** Display ingredient code if used as an ingredient and only show MI code if used as a Menu item which means only in  DMI instance
* When an item is a DMI, Do not display recipe name as it is no longer a valid scenario as well as number of servings
** ONLY Display and show the HACCP label if they selected a HACCP process and always hide if they did not

# NXT-10963 — [TaggedDashboard] - Dashboard (Insights) - Account Management - Active Enrollment Widget

## Description
As a account administrator, I would like to be able to view the number of active accounts belonging to my district over a given period of time, so that I can better prepare and allocate my food service resources when and where they are best needed.

*Summary*:

On the Dashboard, add a widget that allows a user to see the number of active student accounts in the district for a set number of date ranges.

Specifically, this section only:

*Details*:

* In the Account Management module Dashboard, add a widget for *Active Enrollment* 
** This widget should show the count of +all student accounts in the district whose user status is Active+. 
** Note that this does *not* include _users_ or _adults_, only _students._
* This widget is not subject to the view options (if already implemented): Last 7 Days, Last 30 days, etc. It is only the active count as of Today.

## Acceptance Criteria
This story is complete when a user can navigate to the Account Management Dashboard and view an Active Enrollment widget for their district, which contains a count of every active account in the district as of today.

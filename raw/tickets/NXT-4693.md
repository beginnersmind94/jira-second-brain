# NXT-4693 — Item Management - Recipe - Edit Allergens

## Description
*User Story*

As a Item Management User, want to be able to edit my recipe’s allergens

*Mockup*

!image-20200928-021401.png|width=100%!

!image-20200928-021507.png|width=100%!

*Requirements*

* Click on the ‘Edit’ pencil for Allergen Info
* Drawer will open from the right
* User will be able to edit the Allergens, just like as an Item
* Header
** Allergen Info
** Recipe Name | Recipe ID
* Clear and Skip for now functionality should function the same as in item management.
* When items are updated, the Recipe WILL NOT update the allergens at this time (Future Enhancement to be done)
* If user clicks Save, then the record is saved
* If user clicks ‘Cancel’ then have the same message for Disregard changes

## Acceptance Criteria
User must be able to edit Allergen information for Recipes

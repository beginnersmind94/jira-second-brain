# NXT-37131 — PR- System Setting- Enable CCP temperature Tracking on Production Record

## Description
As a district, I would like to have the option to either track my temperatures for CCP or not on my production record.

## Acceptance Criteria
* Provide a system setting that enables Temperature tracking on production record
** Name the setting “Enable Temperature Tracking on Record”
* User can “turn on” or off the tracking and will display temperature input for the user
** Do not effect Records that are already complete
* Setting Code “PRDRECT”

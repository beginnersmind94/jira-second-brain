# NXT-47441 — IM - Recipes - Steps & Ingredients Fixes & Improvements

## Description
Various fixes, improvements and enhancements to the Steps & Ingredients behavior to provide better usability.



!image-20240503-200021.png|width=91.66666666666666%,alt="image-20240503-200021.png"!

## Acceptance Criteria
+UI+

* Correct the alignment of ingredients displayed underneath an expanded step
** the ingredient list should not be indented
* Correct vertical alignment of Swap and Delete icons in action column when editing a step (fixed already?)
* Change tooltip on Add button
** it doesn’t need to mention Recipes, because recipes can be ingredients; just “Please select ingredient”
* Ingredients card title should change when it goes into search mode
** it used to behave this way

+Enhancement+

* Enhance the Directions field to consider line breaks 
* When showing CN items in the search, replace the image thumbnail with the CN icon
** in other words, put the CN identifier in the place of the image thumbnail, because +CN items don’t have images+
** This is true also for the ingredient thumbnail images displayed under a step
* Allow ingredient images to be clicked to be expanded
** like behavior from Images & Documents
** or at least enable viewing in another tab (browser behavior doesn’t currently work)
* Show identifier icons in serving measure dropdown
** star icon for preferred measure
** L icon for local measure
*** ensure the L is not removed from the value after moving away from the field (this currently happens)
* Move Delete Step function to be inside the Edit Step
** remove it as an icon in the action column
** provide dedicated “Delete Step” button with existing warning prompt
** upon confirmation of delete, move back to main Steps card edit mode with the step removed

## Acceptance Criteria
Navigation: GC Portal - Search - USDA Branded Foods

Changes Required:
User selects the Data Source dropdown
User should be able to enter the GTIN, Item Description, Manufacturer, Item Category (to be listed as dropdown, text based search should list the items from dropdown )

Upon saving the record, the user should be able to view the revised record in the next search onwards. The changes should be visible in SC and PE on all Data Source option

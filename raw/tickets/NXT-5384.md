# NXT-5384 — Acct Mgmt -Transactions - List

## Description
*User Story:*

As a user I want to be able to see the transactions entered for a patron and to be able to select them for   editing and printing.

*Requirements:*

* A list of transactions can be displayed using the standard date and date range selections.
* Headers allow filtering and sorting options
* Drag and drop of headers to allow grouping Only Date, Site, Terminal, User, Description can be Grouped.
* A balance is calculated for each transaction.
* when a transaction is selected the transaction details are shown.
* When a transaction has been modified it is italicized? (we need to be able to identify modified transactions)
* Only un-modified transactions can be edited
* Un-modified transactions will have an Edit option on the row. If the Role Permission is enabled. 
* The Description Shows Meal Type/Payment Type/Edited, 
** Meal Type or Payment Type must be included
** The Meal Type line will also show the number of items purchased.
* The Transaction list is integrated to the Account Management page for a patron.

## Acceptance Criteria
Is a Section On Account Management page,
Clicking the Current in the Overview section takes the user to the Transactions Section.
The user can scroll to find the Transactions Section
The user can use the top navigation (displayed when the user starts scrolling to select the Transactions section.

A list of transactions can be displayed using the standard date and date range selections.
Headers allow filtering and sorting options
Drag and drop headers to allow grouping Only Date, Site, Terminal, User, Description can be Grouped.
A balance is calculated for each transaction.
when a transaction detail icon is selected the transaction details are shown.
When a transaction has been modified it is italicized? (we need to be able to identify modified transactions, the modified transaction won't have the detail icon for viewing)
Only un-modified transactions can be edited or viewed
Un-modified transactions will have a detail icon on the row. 
The Description Shows Meal Type/Payment Type/Edited, 
Meal Type or Payment Type must be included
Payment types are (Cash, Check, Card, Charge, Other. Other is a future enhancement for coupons/discounts)
The Meal Type line will also show the number of items purchased.
Start Date defaults to 30 day from the current data
End Date is today
No limit on the date range.
Transactions that are in a Closed Period cannot be modified

# NXT-10946 — Eligibility - Application Details - Application Image (Scanned)

## Description
As a central office user I want to view the scanned application image so that I can see what information was submitted by the household member as reference.

* If the application is an entry method of “Scanned”, then we will show the scanned image in the main viewer window.
* The application image is not a typical “document” that is associated with the application as it is the application itself. We will have a separate section for documents associated with the application.

This is needed for those school districts in our pilot program that have a high volume of scanning applications.

This is also dependent on scanning functionality being available.

----

Future options - Explore the possibility when going to the “Application Image” to upload/attach an image taken from a phone or a local image file available on the user’s system.

## Acceptance Criteria
User can see the scanned application image for entry method of ‘Scanned’ while clicking the Application Image button.

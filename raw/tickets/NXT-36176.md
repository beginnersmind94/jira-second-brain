# NXT-36176 — MP- Assign Menu- Menu cycle Pop Up- Use all Menus toggle (UI ONLY)

## Description
As a Menu Planner, I would like an option to be able to when I have holidays to indicate whether I would like to use all menus or not. 

This will allow the user to pick whether to force the system to use all menus regardless of how many days were skipped because of holidays. 

!image-20230201-192257.png|width=716,height=695!

## Acceptance Criteria
* Give user a toggle to use all menus
** Default the toggle to off
* When toggle is off, use up to the intended number of days the cycle is set for (10 days)
** EX: I assign a 5 day 2 week menu cycle, there is a holiday on both Mondays, if user skips holidays it will assign the SECOND menu of the week on Tuesday and continue on for the rest of the week, It will disregard the Monday menus for both weeks
* When toggle is on, continue to assign to days that are available until all menus are used 
** EX: I assign a 5 day 2 week menu cycle, there is a holiday on both Mondays, if user skips holidays it will assign the FIRST menu of the week on Tuesday and continue on for the rest of the week, leading to day five being assigned on the following Monday
* Use all menus toggle only works when paired with skipping holidays

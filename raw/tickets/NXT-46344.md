# NXT-46344 — IM - Items - Item List Export 

## Description
As a district administrator I want a basic Excel export for the Items page so I can generate a file containing all the info captured by the list page columns.

!Local Items - Reports-20240111-134216.png|width=83.33333333333334%!

## Acceptance Criteria
* Add a ‘Reports’ button to the Items page with a dropdown like Recipes
** put Food Item Report as first dropdown option
* Make the Reports button available on CN Items grid as well
* Add the ‘Item List Export’ to the bottom of the Reports dropdown
* Upon generating the Item List Export, an excel file will be generated with the full list of items contained in the grid
* The export should consider filters applied to the header and any grid header searches
* The excel export should always include every grid column (regardless of Manage Views selections)
** match the column sequence of the page grid

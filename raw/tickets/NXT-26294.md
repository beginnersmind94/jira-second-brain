# NXT-26294 — Scheduled Exports

## Description
As district I/T operator, I would like the ability to schedule exports so that the district SIS systems can consume them.

## Acceptance Criteria
As district I/T operator, I would like the ability to schedule exports so that the district SIS systems can consume them. 

I would like the ability to:

* Define a schedule for when I want the export to automatically occur (with the ability for it to be recurring)
* Define the filename of the export
* Define a destination location for the export from which the SIS system can have access to as a filesystem (Fileshare?) for the purpose of consuming the file
* Provide for a mechanism to communicate to the consuming SIS system team the information they need to access the location of the exported files.

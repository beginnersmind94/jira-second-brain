# NXT-36940 — PR- Reports- Display No records Available on reports that have no information

## Description
As a user, I would like to know if the criteria I am using to look at a report has information or not. Currently, when looking at criteria and I am doing a report search, I have no idea if it is doing anything or if there are any record to Display.

## Acceptance Criteria
* Display “No Records Available” on all reports if there is no criteria to display if applicable
** Do not display this for Pre and Post Production Analysis as they have their own validations
** Plan worksheet for single and multiple will have PowerBI Preview, display “No record available” on the preview (Requires API change) 
** This should effect Production Form, Production Record, Menu Costing reports on the Plan or record pages (Corner Case if menu is deleted)

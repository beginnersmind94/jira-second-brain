# NXT-11340 — Authentication - Add User - Regions Enhancement (Hierarchy)

## Description
The existing UI for adding a user could use some enhancements to improve the user experience.  The goal being to make the UX more intuitive and reduce the number of clicks and typing a user would required to apply in order to add a user.

## Acceptance Criteria
The UI/UX will follow the designs per the UX team (See attached)
State and Region will be derived the current district. (There shall be no need to ask the user for data that can be derived.)
An "All" dropdown entry will be available for a user to select.
The dropdown entries shall be alphabetized
Users shall be able to be added at the district level
Users shall be able to be added at the region level
Users shall be able to be added at the site level

# NXT-46179 — MP- Menu- Inactive and Pending Deactivation Menu Items

## Description
As a Menu Planner, I would like to be able to see which menu items on my menus are either inactive or pending deactivation and should be paid attention to possibly removing them from the menus they are on.

## Acceptance Criteria
* Capture menu items that are in active, inactive or pending deactivation status
* Display a warning sign on any pending deactivation menu items
* Provide a tool tip on the same warning, “Pending Deactivation”

# NXT-23533 — Production- Production Record- Information to flow from plan to record and status change

## Description
As a user, I want to be able to put my production record into record complete status after filling in all the produced counts.

## Acceptance Criteria
All actual counts and produced counts entered will *change* status to *record complete*.
IF there are carryovers, carryover date *must* be entered to change status to *record complete*. Carryover can not exceed produced count. 
IF menu items are added on production record with no counts, *change* status to *plan complete* and show planned counts as 0.
IF menu items are added on production record with counts, *keep* status as *record complete* and show planned counts as 0. 
IF menu items are added on production plan with no counts, *change* status to *plan complete* and show planned counts as 0.
IF any count is removed from production plan while in record complete, *change* status to *plan incomplete*
IF any count is removed from production record while in record complete. *change* status to *plan complete*.

IF any count is changed from production plan, display new count on the production record.

Display menu items added on production plan on production record.
Display all planned counts from production plan on production record but do not allow changes to planned counts on page.

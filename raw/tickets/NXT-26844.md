# NXT-26844 — PR- Grey Out the textboxes that are not available Carryover, Return to Stock

## Description
As a user, it is not clear to me when I can not enter information into a text box if it is not available.

## Acceptance Criteria
Gray out the text boxes to display to the user that they can not enter in counts.

Also do not display 0 as default for text boxes. It seems like we are doing that for the Planned Counts, Actual Counts and the counts for Carryover, Return to Stock, Waste. Display these text boxes as BLANK.

When menu item is added on production plan/ record, and does not enter counts, also show counts as blank on the production plan/record page. Blank count on plan (status is plan incomplete), Blank count on record ( status is plan complete)

*Waste should always be available to enter information in.*

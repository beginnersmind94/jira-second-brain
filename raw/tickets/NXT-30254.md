# NXT-30254 — IM - Items - Food Info - View/Edit Contributions

## Description
As a District Admin I want to be able to edit the Contributions on my Food Info tab so I can manage Contributions for my Serving Measures.

*Requirements*

* Display and populate the Contributions card on the Food Info tab based on creation/edit logic
* Contributions information is dependent on Serving Measures but otherwise is edited independently
** The Serving Size for Contributions should be the size selected on the Serving Measures card 
* Contributions are edited in the slideout
** *Note:* Contributions now allowing multiple serving measures to be configured results in possible downstream impact for Menu Items, when selecting serving measure for MI

!image-20220517-092755.png|width=100%!

## Acceptance Criteria
* Disable the Food Info tab for items that are non food
* Ensure Contributions card displays appropriately
* Use the Serving Size selected on Serving Measures
* Allow Edit function 
* Edit is performed in the slide-out component
* Manage the multiple Serving Measures

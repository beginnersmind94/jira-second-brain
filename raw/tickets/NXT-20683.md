# NXT-20683 — INV - Inventory - Additions - Addition Details - Edit

## Description
As a Site Manager I want an Edit function on Inventory Additions so I can make changes if I have the permission.

*Requirements*

* On the Addition Details page ([NXT-20680|https://primeroedgenext.atlassian.net/browse/NXT-20680]) if user clicks the Edit button
* Enable some header fields for editing
** Reason dropdown
** Comments area
*** required for editing
* Enable some item list fields for editing
** Quantity
** Price
*** this will update the item price on any of the transactions
** Comments
* Provide Save and Cancel buttons
** require Comments on Save
* Once saved, load the view-only page 
* Enforce validations
** check perpetual quantity 
*** user cannot reduce the Quantity less than used quantity
*** example: I first received 10 cases, then I used 5 cases. On the Addition, I should not be able to reduce less than 5 cases
** check Physical Inventory 
*** if Inventory Addition date is less than most recent Physical Inventory ‘Counted On’ date, do not allow items from that Physical Inventory to be adjusted

# NXT-3780 — Create Date selection

## Description
The user can either enter a date or select a date from a calendar.

## Acceptance Criteria
The typed date format is MM/DD/YYYY or MMDDYYYY or MM/DD/YY

The typed date can be MM

The typed date can be MM/DD

The calendar selector will allow the user to select any date in the past and the current date.

When dates are selected they are treated as an AND with other filters.

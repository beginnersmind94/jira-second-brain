# NXT-53405 — MP- Publish Menus- Remove page and add function to Menu List Page

## Description
As a menu Planner, I would like to be able to see all my menus on my list page and then quickly have a button that will give me the ability to publish my menus without having to go to another page. Provide me with a slideout that can quickly give me the tools to publish.

## Acceptance Criteria
* Provide a “Publish” button next to the New on the Menu List page
** This will enable a slideout (1/3 page) that will give the user the same functionality that is currently on the publish menu page
*** Stack functionality with date range selector and apply button, then show all menu details, then confirm publish at the bottom
** Instead of having a start date and number of days, provide a date range selector that the users can pick for what days that would like to publish for

# NXT-8152 — Orders - ExpressPoint - Order Fulfillment

## Description
As a user I need to be able to deliver and indicate in the patrons account that the ordered items were delivered or picked up. Both single orders and Bundled(multiple orders) can be viewed, selected and fulfilled.

*Requirements:*

* From ExpressPoint Orders the Order Fulfillment is started
* The user can select the orders for a site/date/date-range/meal type/fulfillment type
* Both Single orders and bundled orders are loaded.
* The user can search for a specific person or order number
** Search is on the preselected data only.
* All orders displayed can be selected to be fulfilled at one time
** if a select was performed then only those orders are selected for fulfillment.
* Individual orders can be selected and fulfilled
* Bundled orders can be selected and all individual orders in a bundle are fulfilled.
** Individual orders in a bundle can be selected

## Acceptance Criteria
From the ExpressPoint Orders from the Menus
This page is launched by selecting Orders

# NXT-70671 — GC - SC - Item Catalog : Enable multi selection of Vendors / Manufacturers/ Brands / Item categories/States for searching items from catalog.

## Description
As a user, I want to select multiple Vendors / Manufacturers/ Brands/Item categories and States  in the Item Catalog search filters so that I can view items from more than one Vendor / Manufacturer/ Brand/Item category and State at the same time without running multiple searches.

* Users can: Select multiple Vendors / Manufacturers/ Brands/Item categories and States via checkbox dropdown along with check all checkbox to check or uncheck all selections
* When multiple options are selected for a filter : The system should return items that match ANY of the selected manufacturers (OR logic).
* When more than one filter is selected - its going to be current behavior - use AND condition between all filters.
* If more than 1 selected: Show {Selected Name}“+X more” 
* Existing button “Clear Filters” should reset all filters - current behavior.
* Selected Vendors / manufacturers/ Brands and Item categories/States remain selected: After search execution and When navigating between result pages.

# NXT-5207 — Application Details - Applicant Contact Details 

## Description
*User Story* 

As an app processor, i can see the provided contact information for the applicant so i can reach out if there are any questions or discrepancies.

*Mock Ups* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=87%3A2599&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=87%3A2599&scaling=min-zoom]

*Requirements* 

* Applicant Contact Details (section) should match what’s in the application and guided based on settings and conditionals 
** Display Data that was completed in the application: Max fields - 
*** Name
*** Phone 
*** Email 
*** Address 
* It’s possible this section was left blank in the application submission for manual entry bc none of the fields are required 
** If no data display text > _Contact information was not indicated in the application._

## Acceptance Criteria
- Guardian Info tab is now Applicant Contact Details
- data above is included in this section
- Multi-language, ADA compliant.

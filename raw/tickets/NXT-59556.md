## Acceptance Criteria
Navigation: Settings- System Settings - Global Catalog

Currently there is no option to restrict the usage of SSO 



Changes Needed:

Introduce a new INTERNAL setting as “Enable GDSN SSO Login”, Default Toggle Value: No, User can change to Yes if needed



If the toggle value is Yes, then SHOW Log-On GDSN Connect Button

If the toggle value is No, then HIDE the Logon - GDSN Connect button in the UI

This option should be accessible only if primary setting of “Enable Global Catalog is enabled. If not show them in disabled state

# NXT-4672 — AcctMgt - Homeroom Drop-downs - Association between Homeroom and Site

## Description
*User Story*

As a district admin I may have a student enrolled at Site A, but their homeroom may be in Site B. I need a way to accomplish this.

*Mockup*

N/A

*Requirements*

* While adding a Student, and while Editing the General Info panel…
** Change the way the drop-down is fueled:
*** At present, it’s tied to the selected Site.
*** We instead want to show all active homerooms across the district.
** The drop-down should show all active homerooms at the district, with the homerooms at the selected EnrollmentSite (if any) at the top of the list. (This can be deferred if too much effort)
*** The rest should be sorted by Site, then Alphabetical by Homeroom Name.
** The drop-down should show the name of the homeroom, and also the site code + site name that the homeroom is tied to.
*** Example: 'Mrs. Brown 5-8' is at site 301 - Barker Elementary. We should show that information under the name of the homeroom.

Discussion 8/18 during planning

* With the number of homeroom options in the dropdown, having search functionality will be very handy here.
** *New requirement: User should be able to search the homeroom dropdown.*

## Acceptance Criteria
User can select a homeroom for a patron that isn't at the enrolled site.
User can see all active homerooms, and can clearly see what site the homeroom is located at.
User can search the homeroom dropdown.

# NXT-64519 — Acc - CACFP Reports - Revenue

## Description
As Dana the Director I want to run the Revenue report for my CACFP program so I can get an overview on generated revenue from the program.

Currently the Revenue report has support for CACFP, but does not appear to be putting Supper together with Lunch.

*Requirements*

* Update the note on the page:
** {noformat}Note: For SFSP and CACFP, all snacks are collected under the Snacks column, and Supper is included with Lunch.{noformat}
* User can see CACFP revenue for all meal types in the Revenue report.
** Put Supper together with Lunch.
** Make sure all Snacks are together under Snack
*** AM, PM, Evening Snack
** Update column labels on the report if Program is set to SFSP or CACFP:
*** Lunch → Lunch/Supper
*** Snack → Snacks

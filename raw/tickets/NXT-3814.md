# NXT-3814 — Account Management - Advanced Search - Site Section - Site code subsection

## Description
The site codes defined in the district will be listed here. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Site Code values. Site codes are numerical values assigned to a site. Each site has a unique site code within the district.

!Screen Shot 2020-04-28 at 9.24.31 AM.png|width=305,height=368!

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Site Type section they are 'or'ed together.

When a filter value is selected the Site Type Subsection count is increased by 1 and the Site Section count is increased by 1.

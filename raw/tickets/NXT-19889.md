# NXT-19889 — IM- Recipe- Menu Item creation prepopulate recipe name

## Description
As a user, when I create or have a recipe and I am ready to set it as a menu item, I want the system to use to prepopulate the menu name with the recipe name.

The user still has the option to change it if they would like but it gives them a chance to use the name from the recipe which happens to usually be the same name.

## Acceptance Criteria
Use recipe name as menu name when enabling recipe as a menu item. Make this field available but allow user to make changes if they would like before saving.

# NXT-11261 — Routing Navigation - Account Management - Workspace and Modules Launcher (Changes)

## Description
As a Perseus user I want to navigate to the Account Management module from the workspace landing page and/or the module sidebar so that I can quickly go to the module for managing accounts.

Currently, in the module shortcuts section in the Workspace, we’re using the ‘Financials’ icon as the shortcut to Account Management. 

# Instead, rename ‘Financials’ to ‘Account Management’. (Since Financials will not be included in the initial release, it can be ignored for now.)
# Do the same in the ‘Spaces’ drawer that allows users to navigate between modules.
# For now, use the same module icon as the “Program Management” module (holding hands). We will update the module icon at a later time with improved branding.

## Acceptance Criteria
User should be able to see Account Management within the available modules of the Workspace if assigned role permissions

User should be able to see Account Management within the available modules of the Module/Spaces sidebar if assigned role permissions

Routing should still work as expected

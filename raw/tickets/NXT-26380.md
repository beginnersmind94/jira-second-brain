# NXT-26380 — INV - Vendors - Contract Items - Add filter for items missing mandatory fields

## Description
As a District Admin I want the ability to quickly find contract items missing VI# or Price so I can fill those fields and make the items orderable.

*Requirements*

* Add a new designation for items which are on the contract but still lacking their required fields
* Provide additional Status Dropdown - Orderable/Unorderable/All
** default: Orderable
* Add grid header [toggles/switch|https://mui.com/components/switches/]  
** ‘Items missing mandatory attributes’
*** Vendor Item #
*** Price (at least one)
** If enabled, show *only* items missing VI# and/or a _current_ purchase price
* Show an alert icon for any items missing mandatory attributes
** Tooltip: 
*** Item is missing mandatory attributes (use this one if we only use one tooltip)
*** Item is missing Vendor Item #
*** Item is missing Price
** Highlight the attribute missing info in the side panel using red text
* Modify the Item list grid
** Remove the sub-grid (same on [Add Item screen story|https://primeroedgenext.atlassian.net/browse/NXT-25491])
** Update columns
*** Item #
*** Item Description
*** Unit Type
*** Item Category
*** Current Price 
*** **optional columns (unchecked by default):
**** Storage Category
**** Brand Name
**** Manufacturer Name
**** GTIN
**** Standard Price
**** Weight
***** *Add the weight unit\! (lb, oz, etc.)
* -Side-panel: Fix alignment issues with label and buttons (see image below)- *(DONE)*
* Future enhancement: provide a way to jump to the Contract Items screen for bulk updates

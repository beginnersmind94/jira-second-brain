# NXT-54826 — Workspace - Accountability Dashboard Page - Focus View

## Description
As Dana the Director I want a way to view all sites for Front of House to see their progress with a focus on highlighting areas of concern to streamline my workflow with the product.

*requirements*

* User has a new tab on the Workspace Dashboard
** Label: FOH
* User can select site types (multi-select)
* User can select sites (multi-select)
* User can select a date, or a date range
** Restriction: User cannot select more than 31 days
** User still needs to be able to pick a custom date range, however we want to reduce the number of clicks. 
*** Keeping the dates visible and letting the user pick with them may solve for that.
* User has quick date options:
** Today (Today)
** This Week (Sunday → Saturday)
** Last Week (Last Sunday → Saturday)
** This Month (Current Month Day 1 → LastDay)
** Last Month (Last Month Day 1 → LastDay)

## Acceptance Criteria
* New page added
* Role permission for the page added
* User can select multiple site types
* User can select multiple sites
* User can select a date, or establish a date range
* User has “quick select” date options for:
** Today
** This Week
** Last Week
** This Month
** Last Month
* No preloading - user has to search

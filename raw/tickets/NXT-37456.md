# NXT-37456 — Student Import – History Log – Show Exceptions Count

## Description
As a SchoolCafé Admin I want to quickly see if an import had exceptions and how many so that I can know if I need to dig deeper.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* Click on View Log button
* Click on History Log tab
* View the count of exceptions

*Notes:*

* Note that in Import Monitor, we also need to show the count of exceptions also, so you might be able to reuse that code

*Requirements*:

* In the Summary section for the History tab, do the following:
* 
* Add a count of exceptions
** Label: Exceptions
** Value: the count of exceptions

## Acceptance Criteria
* Count is added to the summary section

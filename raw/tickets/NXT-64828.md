# NXT-64828 — WS - Benchmark - Audit Log

## Description
As a Workspace user, I want to be able to view the history of changes for benchmark changes, so that I can track what changes were made.

*Requirements*

* This story is for the benchmark configuration page
* Add a button on the top right:
** Label: History
** On click, open the History drawer
*  
* Add a History drawer
** Use the new design, same as was done for License Management
*  
* In the code, log into history all changes on the benchmark page the same way you do for history log in DataExchange
** Note: there are no Add or Delete activities, just Edit. So first save can be the Add/Create action
* If there are any numbers auto inserted/created by default, add that to history log also

# NXT-71516 — SC - PBIRD - K12 Insights – Power BI Integration

## Description
As a Director, I want Power BI reports to be available in the K12 Insights mobile app so that I can access my distributed and starred reports on mobile.

----

h2. *Description*

Ensure that Power BI reports are fully integrated into the *K12 Insights mobile app*, allowing users to access distributed reports and starred reports.

This integration must follow the same behavior and functionality currently implemented for *Managed View Reports* and *Custom Reports*.

----

h2. *User Flow*

# User opens the *K12 Insights mobile app*
# User navigates to:
#* Starred Reports
#* Distributed Reports
# User views available Power BI reports
# User selects a report
# User views or downloads the report

----

h2. *Out of Scope*

* Mobile UI redesign
* Report creation or editing from mobile
* Notification system changes

----

h2. *Requirements*

h3. *Starred Reports Integration*

* Power BI reports marked as *favorite (starred)* must:
** Appear in the *Starred Reports* section of the K12 Insights app
* Behavior must match:
** Managed View Reports
** Custom Reports

----

h3. *Distributed Reports Integration*

* Power BI reports distributed to the user must:
** Appear in the *Distributed Reports* section of the K12 Insights app
* Applies to all distribution types:
** Scheduled
** Ad hoc (Download, Email, etc.)

----

h3. *Report Interaction*

* Users must be able to:
** View report
** Download report
* Behavior must match existing mobile functionality for:
** Managed View Reports
** Custom Reports

----

h3. *Consistency & Reuse*

* Implementation must:
** Reuse existing mobile app components and patterns
** Follow existing behavior for report handling in K12 Insights
** Maintain consistency across report types

## Acceptance Criteria
* Given a Power BI report is marked as favorite, when the user opens the K12 Insights app, then the report appears in the Starred Reports section
* Given a Power BI report is distributed to a user, when the user opens the app, then the report appears in the Distributed Reports section
* Given a report is visible in the app, when the user selects it, then the user can view or download the report
* Given Power BI reports are integrated, when compared to Managed View and Custom Reports, then behavior and functionality are consistent

# NXT-60313 — Digitize AI - Production Record - Actions

## Description
As a school manager I want the ability to discard digitized documents/images in case the information is not acceptable or is not useful.

* Discard option button will prompt the user “Are you sure you want to discard digitized documents? These documents will be removed and will be unable to be recovered”. Provide Yes and No options.

As a school manager I want the ability to accept and update my production record based on the digitized documents/images.

* Update Production Record option button case 1: If the production record has not yet been updated, it will simply update the information fields on the PR and close the popup.
* Update Production Record option button case 2: If the production record has information already populated, we should prompt the user “This action will override counts for captured items. Items that are not part of the digitized information will remain. Are you sure you want to continue?” Provide Yes and No options.

!image-20250324-144355.png|width=1426,height=711,alt="image-20250324-144355.png"!

The digitized summary section with details expander can be the same as we are doing on the document upload preview control.

Close option simply closes the popup.

## Acceptance Criteria
* Provide ability within the document preview to Discard documents/images
* Provide ability within the document preview to Close
* Provide ability to update production record with information from the scan
* Ensure update action only overrides information that is associated with digitized items

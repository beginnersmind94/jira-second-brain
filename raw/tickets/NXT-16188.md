# NXT-16188 — INV - Orders - Order Detail - Receipts Tab - Open Receipt Details (view-only) + Header

## Description
As an inventory user I want a View Receipt button on the Receipt tab so I can open individual receipts to see their details or make adjustments.

*Requirements*

* Open the Receipt Details screen when user clicks the View Receipt button
* Receipt Details screen will be in view-only state
** fields will not be editable by default
** user has to click Edit to make changes to the receipt ([NXT-15379|https://primeroedgenext.atlassian.net/browse/NXT-15379])
* Provide a version of the Receipt Details screen (Create Receipt, Edit Receipt) that is not editable by default, showing the saved values as labels.
* The right-side receipt specific header values (Date/Time, Invoice #, etc.) will be displayed as labels not in editable state
* Item list is covered in [https://primeroedgenext.atlassian.net/browse/NXT-16826|https://primeroedgenext.atlassian.net/browse/NXT-16826|smart-link]

Note: do not implement Edit button in this story

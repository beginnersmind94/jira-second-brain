# NXT-4654 — ItemMgmt - Copy Item - Product Info

## Description
*User Story*

As a Item Management User, I copy an item from an existing item with Product Information

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=1]

*Requirements*

* Use the same logic from Copy Item - [https://primeroedgenext.atlassian.net/browse/NXT-4099|https://primeroedgenext.atlassian.net/browse/NXT-4099|smart-link] 
* Add the Product Information as a checkbox
** Checkboxes for the information they want to copy, text to use below 
*** Default them all to unchecked
**** If no information is available for the sections, then they would not be clickable to copy (ie, if there are no allergens, then it would be greyed out)
** After the user clicks ‘Copy and Save’ the item is created and the user is brought to the view page NXT-3811

## Acceptance Criteria
* User must be able to click ‘Copy Item’ button on an item that is already created
* Not allowed to make a duplicate item
* Allow the user to copy Product Information over

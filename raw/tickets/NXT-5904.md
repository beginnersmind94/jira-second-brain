# NXT-5904 — Applications - Sessions - Application Approval List

## Description
*Story* 

As an application processor, i would like to print a work log of the applications i reviewed during my session so i can get a signature of the work completed.

*Mock Ups*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421061%3A0&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421061%3A0&scaling=scale-down-width|smart-link] 

!image-20210212-001805.png|width=1439,height=684!

*Requirements* 

* User has an action off of the Session Summary where they can print out an Application Approval List
* User is shown the approval list and can export / print.
* The canned report contains:
** Determining Official (the processor’s fname + lname)
** Grid containing:
*** Application #
*** Income Amount
*** Income Frequency
*** Household Size
*** SNAP/TANF Case #
*** Eligibility
*** Basis
*** Processed Date
** Signature lines + date lines for the following to be filled in:
*** Reviewing Official’s Signature
*** Confirming Official’s Signature
*** Follow-up Official’s Signature



*Discussion 3/12/21*

* If the application was approved for Categorical, the Household Size, Income Amount and Income Frequency cells should be blank.
** This is due to it being information that had 0 impact on the approval.
* Categorical will also need to surface the categorical type - Foster, Homeless, SNAP, etc.

## Acceptance Criteria
User can preview the application approval list (canned report)

Report shows the contents of the processing session.
Report has signature lines available.

User can export / print

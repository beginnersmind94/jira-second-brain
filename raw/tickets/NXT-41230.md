## Acceptance Criteria
* Provide a version of the Item Import which matches the onboarding tool sections and data
** this should replace the existing item import (eventually)
** excludes all Pack Size fields (GTIN, FMV)
** this will create only items which are ‘inventory ineligible’ by default
* Provide a ‘Use a File' button which will open a slideout
** in this slideout provide the upload and download options
* Download section will have two options
** Download File w/ Data
*** we will also show the last time a file with data was downloaded
** Download Blank Template
* Upload section will have a simple Upload button
** Parsing the file will be done in the slideout
** Results from the file will be displayed there
*** invalid data records (wrong data type)
*** overwritten data records (items with changed data compared against the session)
*** incomplete data records (missing/incomplete data, like Nutrients without Fat values)
*** ignored records (items with no changes)
** Exceptions file can be downloaded
** Updates for acceptable records will be done in the slideout
* Update Vitamin A from *UI* to *mcg* in the file

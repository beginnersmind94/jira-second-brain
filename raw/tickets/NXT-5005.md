# NXT-5005 — Data Exchange - Data Review Screen

## Description
*User Story*

As a PrimeroEdge user, I have a screen that allows me to review data that was imported once the import is complete.

*Mockup*

To be built

*Requirments*

* After import is complete, user is taken to the Data Review screen
* User is presented with a summary of the import
** Total Items Processed
** Number Added
** Number Edited
** Number of Errors
* List of added entries is present
** Each entry links to the item that was added
*** E.g. For a student added, user can click a link on any entry to go to that student’s page in account management
* There is a search function that allows the user to search for particular entries

## Acceptance Criteria
* Data Review screen is present after an import
* Summary displays correct information
* Data entries are properly displayed and linked
* User can search data entries

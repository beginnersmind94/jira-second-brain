# NXT-3678 — Edit Inspections - Re-Inspections

## Description
*User Story*
As an Inspector conducting a Re-Inspection, I have a view of the Inspection based on what the supervisor selects when marking for Re-Inspection.

*Mockup*
Needs Mockup

*Requirements*
* Edit Inspections is adjusted based on how the Supervisor responds when sending back for Re-Inspection
* If "All Questions" was selected, user is required to re-run the Inspection from its original state
** There is an action available where user can view the previous results
*** Method TBD
** Inspection follows the rules set forth by the original Template
*** Required questions, scoring, etc.
* If the Supervisor selects specific questions for the Re-Inspection, only those questions are available
** Selected questions can be edited
*** All selected questions are marked as required, even if they aren't in the Template
*** User cannot submit unless all selected questions are answered
** Non-Selected questions are displayed by default, with their responses present
*** Those questions cannot be edited
*** Those questions are visibly different from the selected questions to indicate that they cannot be edited
** Scoring is updated based on the responses to the selected questions
*** Non-selected responses are still figured into the score
** There is a toggle available for showing all questions that is different from the normal Show All Questions toggle
*** If enabled, the user sees both selected and non-selected questions
*** If disabled, only selected questions are displayed

## Acceptance Criteria
* Re-Inspections where an all questions selection is made can be run properly
* Re-Inspections where only particular questions are selected can be run properly

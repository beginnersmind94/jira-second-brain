# NXT-5227 — Application Processing - (Edit) Contact Info

## Description
*User Story* 

As a processor, I can edit applicant info on the contact info page.

*Mock Ups* 

*Requirements* 

* Contact Info section > Edit Pencil > Takes user to the Contact Info page
** Same functionality as the guided process 
*** Ability to contact details 
* Save and Cancel options 
** Save > Saves changes made and brings them back to the processing page
** Cancel > Discards changes And brings them back to the processing page
* -all saved changes are tracked in the application details \{history} section-

!image-20201201-192724.png|width=553,height=683!

## Acceptance Criteria
- Multi-language, ADA compliant
- edit pencil takes the admin back to the page to make changes 
- admin can cancel changes to discard or save to update

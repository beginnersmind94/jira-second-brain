# NXT-14261 — IM - Reports - Recipe Report - Recipe as an Ingredient

## Description
As a user, I need to be able to run a report with all my recipes from a specific datasource.

Requirements

* Need to be able to click on 'All Recipe Reports' on the Recipes homepage
* A slideout will come out to select the datasource, default to Local, but allow shared / global datasources if available
* Tags are not needed at this time
* User can hit CANCEL or GENERATE REPORT to view the report

## Acceptance Criteria
User must be able to pull a list of all their recipe reports per datasource

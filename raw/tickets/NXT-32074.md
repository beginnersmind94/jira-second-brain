# NXT-32074 — INV - Configuration - Ordering Conditions - Update for Pack Size Config

## Description
As a District Admin I want the  Item Ordering Conditions configuration to reflect the pack size units of my items so I can configure sites to order the correct item unit.

*Note:* this story is for the dedicated page under the Configuration category. See related [https://primeroedgenext.atlassian.net/browse/NXT-32073|https://primeroedgenext.atlassian.net/browse/NXT-32073|smart-link]  for per-Vendor page

*Requirements*

* Update the Vendor’s Ordering Conditions tab to consider item pack size configuration
* Display each unit on its own row, with a checkbox available for each unit
** Show item units on individual rows but indicate items which are linked together
*** Replace Unit Type with Unit Description
* Display Item # only once for the highest unit
* Show conversion (sub) quantity 
* Add Vendor Item # (missing in mockup)
* +Business Rules+
** Ordering Conditions are still defined for each Unit 
** Multiple vendors cannot be assigned to different items for the same site
*** Restrict the selection of a different vendor on a sub-unit. Vendors must be the same for multiple units
*** Users can still select or deselect for ordering

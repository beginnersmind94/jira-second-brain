# NXT-9780 — Data Exchange - File Import Setup: Field Mapping Enhancements

## Description
As a PrimeroEdge user, I would like to have an intuitive screen with *helpers* that allows me to easily associate fields in PrimeroEdge with the fields in my uploaded file, so that I am not overwhelmed by the amount of information on the screen.

*Summary*

# In the Data Exchange wizard, add a PrimeroEdge mapped field column (tentative name) that displays all of the PrimeroEdge fields related to the selected Import Type.
# Data Exchange → Field Mapping screen, add a *PrimeroEdge mapped field* (tentative name) section.
# Within this section, display all of the fields related to the selected Import Type.
#* Field Mapping screen Legends are visible for user to understand and navigate this screen
#* Tooltips are missing on DRAG, DROP and FILE PREVIEW columns.
#*  User should be able to use tooltip and legends to understand the mapping helpers
#** File Data Column tooltip: “Fields listed below are from your uploaded data file”
#** DROP column tooltip: “Field Configurations & Mapping placeholders to drag from PrimeroEdge fields on the right column”
#** PRIMEROEdge Column tooltip: “Draggable PrimeroEdge fields available to map with your Import confugurations”

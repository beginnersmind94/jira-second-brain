# NXT-18644 — INV - Orders - Manage Orders - Quick Receipts - Cancel Quick Receipt

## Description
As a site manager I want the ability to cancel a Quick Receipt so I can completely remove an in-progress Quick Receipt.

*Requirements*

* Provide a Cancel button on the Quick Receipt Details screen (Receive step)
* Cancel button will be top right
* Prompt user on click of Cancel: “Are you sure you want to cancel this Quick Receipt?” Yes/No
* If user cancels Quick Receipt:
** Order Status will be ‘Cancelled'
** Display success message
*** Confirm: “This Quick Receipt has been Cancelled.”
** Keep user on this screen
** Remove all buttons
** Change screen to view-only mode
*** all fields are saved as labels / not editable
** Provide a new button ‘Return to Receipt List’ 
*** this will navigate to the Receipts tab of the Manage Orders page

Mockup Note: Cancel button will replace ‘Add Item’ button

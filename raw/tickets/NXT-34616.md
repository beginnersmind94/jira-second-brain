# NXT-34616 — Users - Disable Ability To Activate User Until Email Verification Has Been Completed

## Description
Users shall not be able to activated via the Users page unless they have already responded to their “Welcome email” and their email verification is set to true.

!image-20220922-170910.png|width=823,height=561!

## Acceptance Criteria
* A user’s status shall remain “Inactive” and the “Active” radio button on the User page shall remain disabled for a new user until the user responds to their “Welcome email” and their email verification is set to true.

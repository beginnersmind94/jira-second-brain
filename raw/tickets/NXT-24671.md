# NXT-24671 — (SPIKE) MP > Menu > Menu Editing/Creating Page > Drag/Drop Menu Items

## Description
There are occasions when a user needs to shift a menu item from one menu item category to another while working on their menus. This might occur in a breakfast menu, where MMA or WGR can cross over... it could also occur with beans or hummus that might sometimes be used as a vegetable, and other times be used as an MMA. 

Users need the ability to drag and drop their menu items from one menu item category to another while working within a menu.

# NXT-56377 — IM - Item Onboarding - Update Nutrients - Carbs, Sugar, Protein

## Description
As an onboarding district admin I want a step to review nutrients so I can ensure the correct nutritional values for my bulk imported items.

h4. Update Nutrients - Carbs, Sugars & Protein

!IO Nutrients - Carbs-20241102-190403.png|width=83.33333333333334%,alt="IO Nutrients - Carbs-20241102-190403.png"!

## Acceptance Criteria
* Provide an Update mode in which the values for the (macro) nutrients can be entered or modified 
* Display Serving Size in read only for context of nutrient values 
* Split the 10 major nutrients into 2 separate views
** use a radio option (or something similar) in the grid corner options
** default to the first view ([https://primeroedgenext.atlassian.net/browse/NXT-56268|https://primeroedgenext.atlassian.net/browse/NXT-56268|smart-link] )
* include Calories plus the *_second 5_* nutrients in the second view
** titled ‘Enter Fats, Cholesterol & Sodium’
* Display completion percentage indicators for each column
** Show the percentage in font color ‘alert unfilled orange’ (D14200) for the mandatory columns
* highlight missing values for mandatory columns 
** only once the user has progressed through the field (don’t show red alert if they haven’t even got to it yet)
** apply ‘unfilled orange’ color to weight unit label and text field
* Display an alert icon in the Item # column for missing values
** red (EB0000) if any mandatory values are missing
** orange (F08B46) if the optional values are missing
* Allow items to be saved with missing values for any and all fields
** show confirmation prompt if any items are missing mandatory values
** items will be indicated as missing values on the [review screen|https://primeroedgenext.atlassian.net/browse/NXT-41655], where they will also not be eligible to be published

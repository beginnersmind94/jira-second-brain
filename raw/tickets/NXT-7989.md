# NXT-7989 — INV - Manage Orders - Order Detail - History tab

## Description
As an Inventory user I want a History tab on the Order Details so I can view past version transactions of the order.

*Requirements*

* Add a History tab on the Order Detail
* History tab will display a ‘Transaction History’ table
** this table will list an entry for each time the order was updated/changed
* This history table will have the following columns
** Order Transaction #
*** this needs to be some sort of unique number to indicate the specific transaction
** Transaction Details 
*** this will explain the details of the change for that transaction
** Status
*** Order Status at the time the order was changed
** Modified Date
*** this is the date and time that the order was updated
** Comments
*** this will display any Order Comments that were saved at the time that the order was changed
** Modified By
*** the user that saved the updated order
** Action column
*** this will contain a View icon that will load the historical Order Detail page for the selected transaction ([NXT-18122|https://primeroedgenext.atlassian.net/browse/NXT-18122])

* Order Adjusted/Updated 
** quantities changed

* Order Merged

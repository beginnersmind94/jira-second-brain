# NXT-33974 — INV - Additions - Confirmation + Details - Consolidate units

## Description
As a Site Manager I want the Additions Details page to consolidate units if there aren’t different prices or expiry dates so I can see items as I do on other transactions like Withdrawals. 

*Requirements*

* Update both the Addition Details and Confirmation stepper screen 
* Display a single row per item on Additions if
** there is only one unit 
** the sub-unit price is perfectly scalable to the higher unit
** all units have the same Bin Lin info
* Create batches according to same rules above
** Single batch if all units have the same per-unit price and the same Bin Line
** Different batches for each unique
*** Price (per unit)
*** Date on Product

## Acceptance Criteria
* Display a single row per item for single-unit items or units which would all be in the same batch

# NXT-71700 — GC - PE -Item Catalog - Enhance Review and Import items functionality- Meal Credits popup

## Description
GC - PE -Item Catalog - Enhance Review and Import items functionality- Meal credits popup

# Popup opens when clicked on View/Edit… button under meal credits column for an item
# Below is the rough mock up for displaying the meal credits information.
## serving size - allows numbers , decimals up to 3 digits and fractions with denominator 2,3,4,8
## serving unit : display the unit from catalog for meal contribution basis and allows selection of other units given for nutrient serving size unit (if its different from meal contribution basis) and standard weights (gram, Ounce, Pound), - importing/exporting this item will import with updated serving size
### for example nutrient serving size is defined for 1 cup  and meal contribution is defined for 1 each then on this popup serving size unit will show the options cup, each, ounce, gram, and pound
### changing the serving size amount and unit  should re calculate the weight for meal contribution basis.
# we have this UI available in menu planning module and on ingredient detail page > nutrients tab 
# save button will save changes to allergens for that district’s and user session but not update the catalog item
# Save to catalog button will save the data and update the item allergens in our catalog.
# we may need to add close button to close the popup.

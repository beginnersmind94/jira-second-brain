# NXT-5344 — System - Configuration - Eligibility Guidelines - Display Guidelines [Implemented in System]

## Description
As a district admin, I would like to be able to view and, if necessary, edit the federal eligibility guidelines, so that my organization can reference this information when working with parents and students.

+Summary+:

Allow users to view and edit Eligibility Guidelines by Academic Year and State Category.

+Details+:

* In the System Configuration category, add the *Eligibility Guidelines* page.
* Basic design can mimic Classic:
** Within the page, display filters for Academic Year and State Category.
*** Requirements for both can duplicate Program Configuration - Reimbursement Rates page tabs: 
**** Current academic year should be selected by default and drop-down menu should display last 10 years.
**** State Category should display Contiguous States (default option), Alaska, and Hawaii in that order. 
** Display the current academic year and contiguous states' eligibility guidelines for the user by default when they land on the page.
** Display an Edit button/pencil. When selected, allow the user to edit the data within the table and commit it to the database.
* Get Guidelines and Generate Report button functionalities can be covered in a separate story, so those buttons should show as disabled for now.
* Error prone tab will be covered in a separate story.

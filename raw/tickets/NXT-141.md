# NXT-141 — [TaggedReport] - Accountability - Report - Edit Check Worksheet

## Description
*Story*

As a user I need to be able to generate a report that shows a summary or detail list of all the edit check entries. As a district admin I have a set report that I need to keep on file for audit purposes. It’s called the Edit Check Worksheet.

*Requirements:*

* I can select to see the Edit Checks by Day/Month/Date Range
* There are display options that can be set to filter what data is displayed
* The Edit Check Worksheet can be created..
** It shows each day of the selected month.
** Breaks down by meal type.
** Breaks down further by eligibility (free/reduced/paid)
*** Columns:
**** Served (meal count for that meal for that eligibility for that date)
**** Eligible (how many students are approved for that eligibility for that date)
**** Eligibility X AF (eligible number above multiplied by the attendance factor for that month)
**** It also shows totals.
***** Served total
***** Site total
***** District total
** Report also shows the attendance factor which is used for a calculation above.
** It also shows the total enrollment count - the highest number of students active for that time period.

* Summary and Detail views are available (Detail view is data by date)
* Not all Display options can be selected together
** 1. Exceptions Only,Exclude Non Serving Days, and Include Site Name can be selected with any other selection
** Include Satellite Sites on Parent Site can only be with Item 1
** Include CEP Details for Non CEP sites can only be combined with Item 1
** Provision Sites Only can only be combined with item 1
* List Displayed can be Printed or Exported (When Print/Export functionality is available)

## Acceptance Criteria
Serving Site and Exceptions Only are default.
Edit check data is selected by Area/Site Type/SIte Code/Site/Programs/Meal Type
The list can be shown in Summary view or Detailed view
Counts that exceed the Eligibility Count or Attendance factor are highlighted
The date is a day/month/ or date range. 
The date range is upto one year
The list has the paging and scrolling
The list can be organized by grouping headers
The list can be sorted
The list can be filtered
Headers can be added or removed
There are sections for each Eligibility Type Free/Reduced/Paid with a Total and Attendance Factor percentage.
Each Section Free/Paid/Reduced/Total has sub headers for Approved/Eligible (V/A)/AA Eligible/Count/Percent
Apprv = Approved students enrolled at site. 
AF = Attendance Factor
Elig (V/A) = Students Eligible to be served. V = Students served visiting from other sites. A = Students served away at other sites. Calculation = (Apprv + V - A)
AA Elig = Attendance Adjusted Eligible. The number of meals allowed before the Attendance Factor is exceeded. Calculation = (Elig (V/A) X AF)

# NXT-12699 — Platform - Security - Edit a Resource

## Description
As a primero edge admin, I should be able to access resource catalog to EDIT/Update a resource available for any district so that the resource can be accessed by any user with the assigned role for that resource.

## Acceptance Criteria
# User shall be able to click on EDIT button from resource catalog page
# User shall be able to update resource details and map to a/multiple role from drawer pop-up
# User shall be able to view the updated resource in resource table
# User shall NOT be able to create Resource when same resource name is provided with Module & Type combination (check for duplicates)

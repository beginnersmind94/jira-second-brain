# NXT-17016 — Platform - Licensing - Implement Product Active Toggle

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Site

*Description*:

As a Operational site admin, I should be able to view list of products we are enrolled and be able to enable/disable any product for available site/district

*Mockup:*

[https://www.figma.com/proto/PK64OPHjiNeQP1Lm1jz9eu/Summary_Licensing?node-id=3%3A1550&scaling=min-zoom&page-id=0%3A1&starting-point-node-id=3%3A1550|https://www.figma.com/proto/PK64OPHjiNeQP1Lm1jz9eu/Summary_Licensing?node-id=3%3A1550&scaling=min-zoom&page-id=0%3A1&starting-point-node-id=3%3A1550|smart-link] 

*Requirements*:

* Site/District admin logs into System/Licensing Summary page for list of products displayed in “My Products” section with ON/OFF toggle button
** For District Type product license, when user clicks on ON/OFF toggle, 
*** It should display all the sites under which the highlighted product is enabled/disabled
*** Each site row should display “active/inactive” toggle.
*** User should be able to toggle any of the site
*** User action for “ON/OFF” should trigger a confirmation prompt.
** For Site Type license,
*** Each site should display “active/inactive” toggle.
*** User action for “ON/OFF” should trigger a confirmation prompt.
*** When user confirms the toggle action, central office & user must be notified via Workspace task



*Assumptions:*

* License *usage* is directly mapped to Activate toggle of the product per site/district
** _Once insights are in place, site activity per product should populate/activate the licenses._

## Acceptance Criteria
* All sites for both District and Site types shall have “ON/OFF” toggle buttons enabled with current status
* District Type product license must render all sites under that district with toggle status for each site
* Notifications shall be sent to central office team for any toggle updates via workspace tasks
* Toggle confirmations are enabled for each action and parent toggle confirmation for enables/disables all sub-items

# NXT-3817 — Account Management - Advanced Search - Eligibility Status Section 

## Description
The Eligibility Status sections has two sections. They can be treated as subsections. The first section will show the eligibility statuses available from the DB. The current statuses are Free, Reduced, Paid, and not available. The second section shows the reason the status was assigned.

## Acceptance Criteria
The filter values are populated from the DB table

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Eligibility Status section they are 'or'ed together.

When a filter value is selected the Eligibility Status Subsection count is increased by 1 and the Site Section count is increased by 1.

# NXT-66306 — API Change - Populate Curated Include Item Category, Storage Category, Valuation Category

## Description
_(none)_

## Acceptance Criteria
*Navigation: DB Enabler*

*Changes Required:* Applicable to Search API and View API for Cybersoft Data Source Only

The search and view API should reflect the group name against Item Category, Storage Category, Valuation Category

*Changes in Search API:* 

Find the *Item Category* string value against Item Category Master and return the Group Name (If Not NULL) as Item Category in the API. If no match found in the master, then return the original value as is

Find the *Storage Category* string value against Storage Category Master and return the Group Name  (If Not NULL) as Storage Category in the API. If no match found in the master, then return the original value as is


Find the *Valuation* string value against Valuation Group Master  (If Not NULL) and return the Group Name as Valuation Group in the API. If no match found in the master, then return the original value as is

*Changes Items Batch Creation API (Sc):*

Include the revised group name against corresponding fields such as Item Category, Storage Category and Valuation Group. 


*Changes in Translated and Exception Download:*
Include the revised group name against corresponding fields such as Item Category, Storage Category and Valuation Group.

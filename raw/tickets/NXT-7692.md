# NXT-7692 — INV - Physical Inventory - Review Screen - Modify Inventory (Pending Reconciliation status)

## Description
As an inventory user I want an action bar on my physical inventory summary/review screen to complete the necessary actions on my inventory.

This action bar should be accessible from the Summary tab/section on the Review Screen.

Provide Access to:

* Actions (permission based)
** *Modify Physical Inventory*
*** prompt user to enter comments in order to reopen inventory
*** this will reopen the inventory for re-editing
*** this puts the inventory back in ‘Physical Incomplete’ status
*** navigate user back to Main Screen (physical incomplete) on Summary tab
** Cancel Physical Inventory
** Reconcile Inventory

Rule for modifying:

# modifying the reconciled physical inventory is not allowed if there is another physical inventory completed after this physical inventory.
# modifying the physical inventory will insert new record for it by cancelling the existing physical inventory, all item counts will be copied over to the new physical inventory recorded.
# We don't allow cancelling the physical inventory if its status is reconciled.
# Comments added while modifying the inventory will be overridden by the comments if that physical inventory is cancelled later.

# NXT-30895 — PII - Protect Sensitive Fields on Import Preview UI

## Description
When a user imports files, the final step of the import process shows the user a preview of data from the import file (See attachment).  The PII data within this preview needs to be protected.

!image-20220517-123920.png|width=1616,height=539!

## Acceptance Criteria
* PII data within the import preview shall be visible only to those who are a member of a role with PII visibility enabled.
* The UX treatment for the protected fields shall be according to the UX design.

# NXT-17808 — MP- Add Menu Criteria for Menu Planner - UI Only

## Description
As a user, I need to be able to designate who and for what I am creating the new menu in Menu Planner for.

Mock up 

!image-2021-08-16-14-26-29-259.png|width=200,height=183!

 

There needs to be three fields added to start the process of using the Menu Planner in this order.
1) Meal Pattern
2) Meal Type
3) Site Group
4) Serving Group

A user needs to only be able to select 1 serving group. User must also be able to “apply” this criteria and the system process it before given the option to use the Menu Planner. Menu planner functionality should not work until the criteria is selected and applied. 

We also need the option to clear all filters with pop up displaying that if user clears filters, they will lose all information they have worked on. 

Display pop up wording 

“Are you sure you want to clear filters? This will cause all information to be lost.”

## Acceptance Criteria
All three criteria to select need to be available in this order. 
1) Meal Type
2) Meal Pattern
3) Site Group
4) Serving Group
User must be able to select the information to use functionality of menu Planner. User can only select 1 serving group as this will be needed for analysis purpose. 

The functionality currently in the “Add Menu” page will be acceptable on this page.

(The apply button will impact functionality later in Release 4.0)



We also need the option to clear all filters with pop up displaying that if user clears filters, they will lose all information they have worked on. 

Display pop up wording 

“Are you sure you want to clear filters? This will cause all information to be lost.”

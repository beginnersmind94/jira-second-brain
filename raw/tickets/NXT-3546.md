# NXT-3546 — Training Events - Edit

## Description
At the Central and District Realm, administrators can edit training events that were created in their realm. (editing training events is permission based)

* District can edit district events 
* State can edit state events (Central Realm) 
* The district cannot edit events created by the state (and offered to districts - checkbox)

* Since we are tying each subtopic to a main topic, there will be no floating/overlapping of subtopics. Therefore, when creating/editing training events, the user must select the topic level to have access to the subtopics below it.

## Acceptance Criteria
District can edit district events 
State can edit state events (Central Realm) 
The district cannot edit events created by the state (and offered to districts - checkbox)

# NXT-64726 — WS - Filters - Date Range All Previous Years - SC

## Description
As a Workspace user, I want to be able to view the data for all previous years, so that I can track what I used to vs what I am doing now.

*Requirements*

* This story is for the filters on the main page
* It applies only to SC Districts
* In the drop down list for the date range, show all previous years that we have data for that district
** Don’t show years older than when the district has data

## Acceptance Criteria
* Users are able to see all their previous academic years in the drop down
* drop down doesn’t show years where there is no data

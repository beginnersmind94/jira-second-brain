# NXT-6681 — MP > Configuration > Menu Line - allow deletion of Menu Lines not currently configured for a Site

I want to be able to remove (deactivate) a Menu Line
so that I can keep only relevant data visible

confirmation message should read: “Please confirm to delete Menu Line aaa.” (where aaa is the name of the Menu Line)

# NXT-10958 — Dashboard (Insights) - Accountability - Reimbursement Widget

## Description
As a central office user or cafeteria manager I want the ability to see the the number of meals sold for reimbursable and non-reimbursable meals. I also want to be able to see that same data over a selected period of time so I can identify any trends. 

* The initial presentation is a graph or chart
* Selecting the graph/chart will allow the user to see the raw data.
* Data shown for reimbursable meals
** Breakdown of Free, Reduced, Paid, and Total Reimbursable Count
* Data shown for non-reimbursable meals
** Breakdown of 2nd Meals, Visitor, Staff, Prog Adults, and Total Non-Reimbursable Count

If date range selector is available, display this information based on the selection. If the district/site selector is available, display the information based on the context selected. 

If date range selector is not available, display the information for the current month of operation.

If district/site selector is not available, display the information for the district (if a district level user) or the site (if a site level user).

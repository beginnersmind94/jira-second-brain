# NXT-7264 — INV - View Perpetual Inventory - Search/Filter section

## Description
As an Inventory User I want a Perpetual Inventory search screen so I can filter my perpetual inventory list.

Selections (header):

* Site selection (use new user control)
** Area
** Site 
*** use one dropdown for both code and name
*** for users with more than one site, default to ‘All’
** Add an ‘All’ option (for district users)
*** this would show total of items across the whole district in the Items List
**** All option is still filtered by Area selection. If ‘North’ area is selected, ‘all’ sites would be all sites in the North area
*** All is only available to district users (permission based)
* Item radio button
** Show All Inventory Items
*** reference site-item record for the selected site
** Show Only On Hand Inventory Items
*** reference perpetual
* Item filters
** Storage Category dropdown
*** filtered by site selection
** Item Category dropdown
*** filtered by site selection
*** filtered by Storage Category selection
* Apply button
* Reset button
** resets the user selections

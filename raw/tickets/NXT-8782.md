*Requirements*

* When the Session is a Summary Sale session the Transaction Grid should have a row per Person Type.
* Each row will have a Meal Count, the Meal Price, Tax if applicable, and the total amount for the Person Type

## Acceptance Criteria
User has an area where the transactions are shown by Person Type.
List shows Person Type, Meal Count, Meal Price, Tax and Total.
Selecting a person type opens it for editing.
The Meal Count sales field can be edited.
Selecting a transaction opens it for viewing or editing.

# NXT-7546 — INV - Physical Inventory (Incomplete) Resources Permissions (R&D)

## Description
As an Inventory supervisor I want the ability to control whether or not certain users can see item perpetual quantity or perform physical inventory actions.

*Note:* this will start with *R&D*; effort may be higher/lower depending on understanding.
*QA* effort may increase.

Map page-level permissions:

* Perpetual Inventory
** provides access/visibility to the Perpetual Inventory page
* Physical Inventory
** provides access/visibility to the Physical Inventory page
* Physical Inventory Status
** provides access/visibility to the Physical Inventory Status page

Create function level resources:

* New Physical Inventory - allows users to start a new physical inventory
** this will enable the visibility of the ‘New’ tab on the Physical Inventory page
* Save Physical Inventory as Incomplete
** allows users to save physical inventories (for the sites to which they have access) as Incomplete
* Save Physical Inventory as Complete
** allows users to save physical inventories (for the sites to which they have access) as Complete
* Add Item to Physical Inventory
** enables the Add Item button on Incomplete physical inventories

*Note:* the actual name of these resources may need to be formatted according to [Platform standards|https://primeroedgenext.atlassian.net/wiki/spaces/NEXT/pages/2170617925/Resource+Management]

* *Example for Function: ModuleName_CategoryName_FunctionsName*

## Acceptance Criteria
Add permissions for Physical and Perpetual Inventory

# NXT-62569 — IM - Item Onboarding - Publish To Local

## Description
As a district admin using the item onboarding tool, I need a way to be able to at any time publish items that are in my session that are ready to go. I do not want to have to have all my items ready before I can publish.

!image-20250612-144802.png|width=503,height=639,alt="image-20250612-144802.png"!

## Acceptance Criteria
* Allow Items that are ready to be published to be published to the local Items
** Clicking the publish Items button will insert all publishable items into the local item list
* Check for Item Description Uniqueness against Local Items
** This applies to both Item and Menu item (If applicable)
* User can publish items and they will be removed from the overview list
* Provide a Button (Published Items) on the overview that opens a slide out that shows all published items 
** Item #, Item Description and Published Date
** Display items in order from most recently published sorted Alpha per publish date 
* Once any items are published, do not allow users to add any additional items to the existing session

# NXT-10414 — POS - Multiple Meals - Meal Service

## Description
As a user I need to be able to select the meals served to the patrons.

!m.png|width=1378,height=641!

!n.png|width=1378,height=641!



*Requirements:*

* All of a meal type can be selected with 1 selection
* individual meals for a day can be selected
* A session for each day a meal is selected is opened if not previously opened.
* Only one session per day is opened.
* up to 14 sessions can be open at one time
* All meal types can be recorded to one session.

## Acceptance Criteria
The patrons name/id/enrolled site/ grade is displayed.
Each date possible from the service context is displayed, skipping the weekend if weekend is not selected.
If a Reimbursable meal has been served for the meal type and day it is checked and can't be selected.
You can select all days for a meal type with one selection.
You can select or deselect a single meal type for a single day
Multiple meal types for multiple days is allowed.
A count of all meals selected to serve is displayed
Clicking 'serve' saves all the transactions into the appropriate session.
A message 'Successfully saved the Multiple Meal selections'

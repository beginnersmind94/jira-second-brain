# NXT-47343 — Update SC Item Details-End Point to write item details from SC end to GDSNConnect_SC Table

## Description
A new end point is required to get the response from SC once the items are created

## Acceptance Criteria
The end point should have the following fields when it is called:

# Import Item ID
# SchoolCafe Item ID
# SchoolCafe Import Status
# SchoolCafe Exception Remarks



The method shall update the following fields in GDSNConnect_SC Table (addiotnal fields to be created as needed)

  1. SchoolCafe Item ID (Ex: LI_12345)

# SchoolCafe Import Status (Success or Fail)
# SchoolCafe Exception Remarks (Ex: Invalid Serving Size)

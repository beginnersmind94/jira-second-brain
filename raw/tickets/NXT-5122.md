# NXT-5122 — Data Exchange - One Roster APIs Research

## Description
Evaluate the work needed to integrate One Roster into PE and create POC as needed.

*Requirements*: 

* Import student information from an API, instead of a flat file done in classic.
* Follow the specification provided by [http://www.imsglobal.org/oneroster-v11-final-specification#_Toc480452019|http://www.imsglobal.org/oneroster-v11-final-specification#_Toc480452019] 
* These specifications are followed by all the API vendors. 
* Skywards is the vendor for Cybersoft, who will be providing the API 
* The current scope is to have student import only. 
* The import is a regular activity, can be scheduled and can be triggered on demand.

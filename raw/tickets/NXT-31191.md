# NXT-31191 — Custom Views - Make current view name visible for user

## Description
With the current implementation of “Custom Views”, the user is not able to easily identify which view he/she is looking at.  The objective of this epic and story is to make the view name visible for the user somewhere on the UI without them having to click on and/or hover over a UI control.

## Acceptance Criteria
* As a user, I would like to see the name of the current view somewhere on the UI so that I know which view I am working on.
** I don’t want to have to click on and/or hover over a UI control to find this information.

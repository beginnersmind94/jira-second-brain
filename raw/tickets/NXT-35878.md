# NXT-35878 — INV - Config - Ordering Conditions - CRUD permissions

## Description
As a District Admin I want permissions for configuration pages so I can control who has access to change important settings and config.

*Requirements*

* Assign the View permission for the Ordering Conditions page
** controls page access and visibility in navigation
** also controls access to the slide-out
* Edit - provides access to the checkboxes and Save button on the slide-out
** Create a view-only version of the slide-out for users without Edit permission
* Add - controls +Add button access
** Add is a sub-page, but does not have permissions itself
* No Delete permission
** Disable on Resource

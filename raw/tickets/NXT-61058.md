# NXT-61058 — AI Onboarding - Digital Recipe - Previous Files

## Description
As a customer support person, I would like to use AI to digitize a recipe document so that I can speed up onboarding

*Notes:*

* This story covers resuming editing a previously uploaded file

*User Flow:*

* User browses to Item Management and then Recipes
* User clicks on the AI button and it opens the AI Recipe drawer
* User clicks on the Previous chip
* User selects which previously uploaded file they want to work on
* System loads the previous file into the File Analysis section

*Requirements:*

* On the Report AI screen, do the following:
* In the upload section, update the Previous chip as follows:
** If click Previous chip, show the Previous content as per below and hide the New content
* 
* Add an instruction text per the Bolt design
* Add a list of previous uploads as cards
* In each card, include:
** file name
** format
** status
** recipes found
** Who uploaded it
** When it was uploaded
* Make the card clickable:
** On click: 
*** load that recipe in the File Analysis section
*** Contract the upload section
*

# NXT-4970 — AggtMgt - Add Patrons - Adult - Wizard Step 2 - Contact Info

## Description
*User Story*

As a district admin I want to fill out general information for an adult patron.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1&view=default]

*Requirements*

* Populate Step 2 of the wizard added via NXT-4969.
* The majority of the information (fields, validations) is identical to a Student’s contact information, with some changes.
** No additional contacts can be added. 
** The contact they are adding is the adult themselves, so the following fields are not necessary:
*** Prefix
*** First Name
*** Middle Name
*** Last Name
*** Suffix
*** Relationship
* 'SAVE & CLOSE' button this step, which will save any information but close out of the wizard.
** User should receive a success confirmation toast about the completed task.
* ‘NEXT’ button available which will save, and take the user to Step 3.
* ‘CANCEL’ button should provide a confirmation prompt for the user.
** ‘Are you sure you want to cancel? Changes from this step will be discarded.’
* ‘BACK’ button is available, acts as a Save, and back to Step 1.
* Stepper (top right component) can be clicked to allow navigation to a future step.
** This would be the same as the user clicking on BACK/NEXT. Validations + Save must happen.

## Acceptance Criteria
Contact information can be entered.
Information is retained while navigating the wizard.
Multi-language support enabled (LANGUAGENAME_ENGTEXT ie: FR_Students, FR_ADULTS)
Accessibility / ADA Compliance programmed appropriately (full KB nav possible, etc.)

# NXT-69853 — ACC - Credit Card Terminals > Square > Reminders

## Description
[https://developer.squareup.com/docs/oauth-api/overview|https://developer.squareup.com/docs/oauth-api/overview|smart-link] 
To make sure we offer the most help to the users to remember to refresh their authorization, we will implement a reminder feature to avoid our customers being surprised with non functioning terminals. 

This feature consists of the Reminder  section on the main page as well as the setup and functionality of sending alerts and emails to users.

## Acceptance Criteria
Display a static in-page Warning Message for when token is going to expire with status for notifications.
Add a tooltip for the Warning.
Allow user to configure which roles to send the notification to in a slide out.
Show that the reminders are set if any single role is selected else show “Not Set”



Token renewal happens automatically by default.
Display renewal information section over normal reminder section.
Add a tooltip for the auto renew section.
If the system detects that auto renewal did not go through:

* send email and notification to configured roles then fall back to normal notification logic.
* display red warning message on Credit Cards terminal page.

# NXT-17793 — MP- Drag and Drop from Accordions

## Description
As a user, I want to be able to quickly drag and drop menu items from my "Recently Searched", "Frequently Used", and "Favorites". Seeing the information only does not help my process.

No Mock up needed.

## Acceptance Criteria
User should be able to drag and drop menu items from Accordions just like they are able to drag and drop from the search bar when menu items populate.

# NXT-23003 — INV - Physical Inventory (Incomplete) - Add Item - Show validations and messages

## Description
As a Site Manager I want validations and messages when adding an item to a physical inventory so I can confirm I am adding the correct items.

*Requirements*

* Display asterisks ( * ) on mandatory fields
* Display validation messages if incorrect data is entered
* Display a success message once the item is successfully added to the inventory

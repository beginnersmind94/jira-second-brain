# NXT-63651 — INV - Distribution - Assign & Pick - Adding items from perpetual

## Description
As a Distribution Manager I want the Substitution functionality to intelligently place added items into existing shipping groups so I can easily add items that are not on the contract.

## Acceptance Criteria
* Update the Add/Substitutions functionality to consider the distribution site’s perpetual inventory
** from [https://primeroedgenext.atlassian.net/browse/NXT-66209|https://primeroedgenext.atlassian.net/browse/NXT-66209|smart-link] 
* When non-contract items are added, assign a Shipping Group
** if there is only one SG, use that for the added items
** otherwise, use the Default for any added non-contract items
* If a contract item is being added, just use its configured SG (current behavior)

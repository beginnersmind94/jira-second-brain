# NXT-5387 — Acct Mgmt -Transactions - Comments

## Description
*User Story:*

As the manager or director I want to be able to add and trace comments about transactions. They need to log who and when the comment was entered.

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=7122%3A103101&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=7122%3A103101&scaling=min-zoom]

*Requirements:*

* Each transaction can have one or more comments.
* Each comment will have a time and date stamp as well as who entered the comment. 
* Comments are saved and never deleted or changed. [https://primeroedgenext.atlassian.net/browse/NXT-5388|https://primeroedgenext.atlassian.net/browse/NXT-5388|smart-link]
* There are two sources for comments, ExpressPoint (Future functionality) and Transaction adjustments through the web.

## Acceptance Criteria
A transaction can have 1 or more  comments
Comments cannot be deleted
Comments cannot be changed
Comments have a timestamp and who entered the comment
A comment MUST be entered if a transaction is changed.

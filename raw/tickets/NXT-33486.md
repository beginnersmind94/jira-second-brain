# NXT-33486 — PR- Prep Site Distribution tab- Received status on individual sites

## Description
As a prep site manager, I would like to be able to go back to my prep site distribution tab and see sites already marked their items are received and how much they received.

## Acceptance Criteria
* Show what items have been already marked as received from the receiving site
* Display “Received: (with the count that was received), this only displays on the prep site distribution tab if the record is complete on the receiving site, until then we do not display that information
* Disable the ability to change count for that menu item only for particular site and allow all other sites counts to still be changed
* If all sites have marked menu item as received before the status changes to Shipped, push the status to shipped from Shipping in Progress and disable the ability to change counts for that menu item for all sites
** IF receiving sites do not all enter their counts on the day of service, and complete their record on a different day, still pull that received information whenever available

# NXT-5242 — Direct Certification - File Search

## Description
As a district admin I want to search all entries from DC files, year to date, so that I can match a student that isn’t showing up elsewhere.

*requirements*

* User needs a way to search the contents of every entry in DC files for the selected academic year.
** Add a File Search tab to the Unmatched page.
* User can search by student first name, last name, DOB, guardian, ID numbers (student ID/state ID/SSN)
** At least one field must be filled out in order for any results to come back.
* User is shown a grid with search results that match their input.
** Given the sensitive nature - no like search results.
* Grid should contain the following:
** Direct Certification File Details
*** First Name
*** Last Name
*** DOB
*** ID #s
*** Case Number
*** DC Type
*** File Date
*** Status (Matched or Available)
*** Additional Columns
**** County
**** Gender
**** Address (street + city + state + zip)
**** Guardian (first + last)

## Acceptance Criteria
User has a File Search page available.
User can enter search parameters as listed above.

User sees correct search results in the grid, with columns as detailed above.
User has additional columns they can add.

Search results are ordered by relevance.

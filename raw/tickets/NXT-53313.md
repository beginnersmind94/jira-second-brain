# NXT-53313 — IM - List Pages - Resources & Permissions

## Description
As a District Admin I want permissions for Item Management Items so I can control various levels of access for viewing and changing items.

## Acceptance Criteria
* Define resource and permission structure for Items page and all sub-pages and sections

Use the following permissions structure:

*Item Management*

* +View Items+ (+ Details) - Page - *Edit*
** +Create Items+ - Action
** +Copy Items+ - Action
** +Deactivate Items+ - Action
* +View Recipes+ (+ Details) - Page - *Edit*
** +Create Recipes+ - Action
** +Copy Recipes+ - Action
** +Deactivate Recipes+ - Action
* +View Menu Items+ - Page
** +Create Variety Menu Items+ - Action
** +Variety Menu Item Details+ - Subpage - *Edit*
*** edit includes Deactivation
*** edit also includes POS sub-type deactivation
** +Create POS Only Items+ - Action
** +POS Only Item Details+ - Subpage - *Edit*
*** edit includes Deactivation

No *Add* or *Delete* operations required.

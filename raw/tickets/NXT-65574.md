# NXT-65574 — PR - Pre & Post Production Analysis - Combine into Single Page & ability to pick analysis type

## Description
As a user, I would like 1 page for my production analysis and the ability to run it pre or post as well as what type of analysis I would like to run against the records.

## Acceptance Criteria
* Combine Pre and Post Production Analysis Report Pages into 1
** Rename page Production Analysis
* Provide Radio Options 
** Pre Production Analysis (set as Default)
** Post Production (Offered)Analysis
* Provide Drop Down for Analysis Type
** USDA CACFP Meal Pattern
** USDA Meal Pattern SY 2014+ (set as Default)
** USDA SFSP Meal Pattern
* All remaining filters stay the same and logic to populate the analysis remains the same

## Acceptance Criteria
* Provide a create mode for menu, with all header information needed to move onto the next step
* Once user has saved the header information, provide the ability to add menu categories and serving groups to populate the menu
** Provide a menu preview that allows users to see what filling the counts does in building the menu
** 1 Serving is required to confirm
* Once user confirms the serving groups and categories, user now lands into edit mode of the menu (Only in the create flow)
** If user leaves the page without confirming the serving groups, land user back on the confirmation mode

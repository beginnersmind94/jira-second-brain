# NXT-7567 — INV - Physical Inventory - Review Screen - Summary/Details tab (Pending Reconciliation status)

## Description
As an inventory user I want a physical inventory summary/review screen header to quickly review the main inventory information saved on a complete inventory.

This screen is displayed once a physical inventory has been saved as complete.
This is also the screen that is displayed when a physical inventory in ‘Pending Reconciliation’ status is opened from the physical inventory list.

This story is for the creation of the Summary page and the content displayed in the ‘Physical Inventory Details’ section.
This story does not include the Item List, Action Bar, or any of the report tabs.

Main Physical Inventory (header) details:

* Counted By (user)
* Counted On (date)
* Started By (user)
* Started On (date)
* Last Modified By (user)
* Last Modified On (date)
* Comments
* Number of items

Physical Inventory Summary section:
As is the same on the Physical Incomplete screen, the Storage Category selections will be available

* Storage Category
** There will be an ‘All Storage Categories’ option, same as the other screen.
* Total Items (total number of items for that storage category)
* Zero Qty Items (number of items in that storage category with '0' as count)
* Select icon
** select will navigate to the Items tab for the selected category

Exclude the Print icon

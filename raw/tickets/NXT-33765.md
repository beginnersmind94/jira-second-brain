# NXT-33765 — RFP - Sites - Ability For Multiple Phone Numbers Per Building

## Description
As a SchoolCafe admin, I would like to be able to add multiple phone numbers for sites so that I can have additional information stored about it.

*Requirements:*

* These requirements are for Add New Site from the Site Configuration page:
* Add new fields:
* Grid:
** Title: Phone Numbers
** Do inline editing style
** Columns:
*** Number
**** Textbox
**** Numbers only
**** Put phone number mask on it if you can
**** Required
*** Extension
**** Small textbox
**** allow alpha, numeric and special characters
**** Optional
*** Type
**** Single select list
**** Values:
***** Office
***** Mobile
***** Fax
***** Other
**** Default select Office
**** Required
*** Description
**** Textbox
**** Short like Name textbox, not large like normally do for a description
**** Required
*** Save / Edit
*** Delete
* 
* These requirements are for Edit Site from the Site Configuration page:
* Add the same fields with the same requirements
* 
* Imports
** We do not need to account for phone numbers on Site imports at this time (can be added later in a separate story if needed)



Original message:

* Ability to add multiple phone numbers per building that display based on report type

## Acceptance Criteria
* User can add multiple phone numbers to a site

# NXT-48226 — IM/Platform - Manage Views - Need ability to associate a saved view with Basic or Advanced tab

## Description
As an Item Management user I want the Saved Views to be associated with the Basic or Advanced tab on the IM pages, so I can use my saved views on the filter section to which they correspond.

Prerequisite for enabling Manage Views on upcoming Advanced Filter stories.

## Acceptance Criteria
* For list screens which have both a ‘Basic’ and ‘Advanced’ filter mode, we would like the ability to indicate whether a particular saved view is associated with the Advanced filter view, so users will understand why they are being automatically switched to the Advanced tab

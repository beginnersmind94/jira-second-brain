# NXT-2452 — Inspections Reports - Inspectors Report

## Description
*User Story:*

As an Inspections user, I have a report that allows me to view Inspections completed by particular Inspectors based on sites, templates, and date range.

*Current Version:*

[https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports|https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports]

*Requirements:*

* Inspectors Report can be run using existing Inspections and available parameters
** User must have Reports role permission to run report
** Report can be created in PDF, Excel, or Emailed
* The following parameters are used:
** Site - Can select 1 or many
*** Optional
** Template - Can select 1 or many
*** Optional
** Inspector - Can select 1 or many
** Start Date
** End Date
* All Inspections that fit within the chosen parameters appear, regardless of status
* The following data is displayed on the report:
** Header
*** Report Name
*** Start Date
*** End Date
** Inspector
** Total Inspections
** Count of Inspections listed by individually by status
*** Scheduled
*** In Progress
*** Submitted
*** Re-Inspection
*** Approved

## Acceptance Criteria
* Inspectors Report can be created with available data across all parameters in all formats

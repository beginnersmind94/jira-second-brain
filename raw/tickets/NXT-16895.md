# NXT-16895 — INV - Orders - Review Orders - Notifications

## Description
As an operations or site manager, I want to receive notifications when an order goes into Pending Review status, is approved or rejected, etc.

*Requirements*

* Send action-based notifications based on the Order Review process
* Send a notification to users with the Review Orders permission (TBD) when an order goes into Pending Review status
** “Order #[12345|https://] has been created in Pending Review status and requires approval.”
*** link to the Order Detail
* Send a notification to the user who created the order when a Pending Review order is approved
** “Pending Review order #[12345|https://] has been approved.”
*** link to the Order Detail
* Send a notification to the user who created the order when a Pending Review order is rejected
** “Pending Review order #[12345|https://] has been rejected with the following comments:
'Please check the quantities for items DST 9721 and DST 18810. Resubmit this order when you believe the corrections have been made. -JM'”
*** link to the Order Detail

## Acceptance Criteria
* Notifications are sent through system notifications framework
* Notifications will be sent on actions related to order review/approval
* Notifications will be limited to the relevant users

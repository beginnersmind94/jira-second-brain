# NXT-47223 — PR- Reports- Print Production Recipe- UI enhancements and PowerBI Consolidated Tab

## Description
As a Production manager, I would like some more information about my planned recipes along with a way to quickly go to the recipes in IM.

## Acceptance Criteria
* Display Menu Item name along with Code 
** Column Name: Menu Item Name (MI#)
** Allow user to search by either Name or Code (Similar to add menu item function)
* Display Recipe Name along with Code
** Column Name: Recipe Name (MI#)
** Allow user to search by either Name or Code (Similar to add menu item function)
* Provide an Action Column with the ability to navigate to the Recipe in IM
** Provide Tool tip (Open Recipe) on icon
* IF there happens to be multiple serving sizes of the same , when user unchecks or checks one of the recipes, treat them as independent and not as the same recipe

!image-20240208-205608.png|width=1299,height=337!

* Update report to use the new PowerBI report that we are using on the plan and record page (Currently it is using the old report)
** Planned recipes should be a consolidated planned total of all sites 
* Display a warning sign on pending deactivation menu items or recipes
** Provide a tool tip on the same warning, “Pending Deactivation”
* Display a warning sign on inactive menu items or recipes
** Provide a tool tip on the same warning, “Menu Item is Inactive”

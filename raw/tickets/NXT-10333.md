# NXT-10333 — {Menu Cycle} Analysis - add Food Component Report 14+

## Description
Need to make sure on screen information is provided, but can also be generated in a report format.

Often used for administrative reviews.

As a menu planner, I need detailed information regarding the accumulation of my component servings for each week, for each meal pattern, for each serving group. This report should also indicate what the component standards are, so that I can view those next to my actual numbers, with the results displayed in color for the instances where I fail to meet these standards.

Because of the nature of this report, it will be a large amount of information, and would be best to have the full report open in a new tab.

See linked story for samples.

## Acceptance Criteria
On the menu cycle page, when viewing the Analysis, an option for the Food Component Report must be available to select.
This report must open in a new tab.
It must contain details indicating the component standards.
It must contain details of the daily and accumulated component information for the chosen week.
It must indicate with color if standards are not achieved or met.

# NXT-11824 — [TaggedNotification] - Acct Mgmt - Notifications - Reminders - Print Functionality

## Description
As a district admin, I would like to be able to view and send letters out to patrons in my district that may have a low or negative account balance, so that they are more likely to make payments toward their account.

+***Update 3/29*+*:  The page will include bolting in the common component for the Print Criteria Drawer. Additionally, we should* +*remove*+ *the Print & Email checkboxes/columns from the grid, with only one set of checkboxes and one column ('Notify'), where the individual rows can be selected and then the Notify buttons activate. The Notify selected/all buttons will still open the same component and then the user can indicate which they would like to Print or Email from there, rather than having to check different boxes in 2 different columns through the grid. See below.*

*Details*

* In the Account Management - *Notifications* page - *Reminders* tab, implement the email/print functionality:
** Print Criteria - Single vs. Multiple
*** When a user selects Single, each notification would be printed on a single full-page.
*** When Multiple is selected, notifications would be printed as 3 condensed notifications per sheet of paper. 
!Print Single Reminder.JPG|width=1913,height=1039!
*** !Print Multiple Reminders.JPG|width=1911,height=1034!
This is done to save paper. Each of the above has a separate letter template to follow.
** Email column
*** Only display a checkbox for a patron in this column if the patron in that row has a primary guardian with an email address saved
*** When a checkbox is selected, enable the Notify Selected button
** Print column
*** Display a checkbox for each patron in the grid.
*** When a checkbox is selected, enable the Notify Selected button
** Action(s) column
*** When user selects a button and opens the letter viewer, add a Print button that allows them to print that individual notification letter.
** _Notify Selected_ 
*** This button should trigger the email and print actions that are related to the checkboxes selected in the Search criteria grid. i.e. if the user has checked the email checkbox for 3 patrons and the print checkbox for 2, it should simultaneously email and print notifications for all 5 patrons.
*** This button should be disabled until one or more checkboxes have been selected in the Search Criteria.
** _Notify All_
*** This button will open a pop-up that allows the user to trigger the email and print actions for everyone in the grid.
** Notify pop-ups (display when either ‘Notify’ button is selected):
*** List of templates:
**** Show the list of Reminder templates that will be used to notify the current selection of students. 
***** To do this, get the primary language for each student. If there is a corresponding template in that language for the student, display the name of it in this list. Otherwise, use the active English template. 
***** i.e., if 2 students selected in the grid have a default language of Spanish, 2 have Chinese as their language, and 5 are English, then this list should show 3 different templates (as long as Chinese and Spanish are active). If an active template does not exist for Chinese, it should show 2 different templates--English and Spanish.
*** Letter Format
**** Individual
***** 1 notification is printed for every individual student in the grid selection
**** Household
***** If any of the students in the grid selection belong to the same household, combine them on the letter.
*** Email/Print Options - Display the total number of patrons in the grid
**** Email/Print
***** Show the total of each option, and when the user hits ‘Notify’, perform both the Email and Print jobs corresponding to the grid selection.
**** Email Only
***** Show the total of each option, and when the user hits ‘Notify’, perform both the Email and Print jobs corresponding to the grid selection.
**** Print All
***** Show the total of each option, and when the user hits ‘Notify’, perform both the Email and Print jobs corresponding to the grid selection.
**** Print Only
***** Show the total of each option, and when the user hits ‘Notify’, perform both the Email and Print jobs corresponding to the grid selection.

## Acceptance Criteria
This story is complete when each student in the grid can have their letters viewed or printed.

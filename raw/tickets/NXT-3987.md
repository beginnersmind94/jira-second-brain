# NXT-3987 — Platform-Level Permissions - Framework

## Description
*User Story*
As a PrimeroEdge Admin, I have the ability to assign permissions that are present across the Platform for the assigned user.

*Requirements*
* Platform-level permissions are applied to a user across the entire product
** All Modules are included in this
** Examples include admin permissions, access to PII, etc.
* Platform-level permissions can be set to an individual or through a Persona
* The framework is created and documented in a way that developers can easily add new ones as needed

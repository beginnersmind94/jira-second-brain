# NXT-47737 — PR- Print Production Recipes- Allow DMI to show as options to print recipes

## Description
As a production manager, even though I have marked an item as a DMI, there are still important details such as my HACCP or prep and cook time or even directions that I would like to print and show my staff.

## Acceptance Criteria
* Allow DMIs to show on print production recipe report
** Recipe report
* Allow DMIs to show on Planned Recipe report 
** Plan Page
** Record Page
** Consolidated Page
* Any DMI that is “Recipe Enabled” should show up to be able to print recipe
* Display N/A in the Recipe field
* Display Identifier (DMI) on menu item field (similiar to menu)
* DIsplay VMI Child Items if they are also DMI

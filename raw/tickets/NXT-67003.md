# NXT-67003 — Create financial periods as an option for period management

## Description
*As a* user of the financial oversight module,

*I want* to manage financial periods

*So* I can enforce audit rules.

## Acceptance Criteria
* Financial periods are now an option under System → Management → Periods in ‘Generate Periods’ and ‘Manage Periods’.
* These periods include all functionalities available for POS and Inventory Periods.

For any fiscal year, under ‘Generate Periods’, you will see:

* Period name
* Period type: ‘Financial’
* Start Date and End Date
* Option to Split
* Edit, Delete, and view history of edits showing user actions

For any fiscal year, under ‘Manage Periods’, you will see:

* Select Period Type: ‘Financial’, Site Type, and a list of sites available for that customer for filtering
* Upon selecting a financial period, you will see:
** Period Name
** Period Type: ‘Financial’
** Start Date, End Date
** Status (X Open, Z-X Closed for Z number of sites, responding to site type and selection; for individual sites, it will only show Open/Close)
** Action to ‘Close All’ or ‘Reopen All’ (the ‘Reopen All’ option is greyed out if any site has an open period)
** History of actions along with the user

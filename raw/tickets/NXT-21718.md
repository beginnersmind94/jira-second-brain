# NXT-21718 — Clone Model

## Description
As an administrator, I want to be able to clone a model configuration so that I can speed up the process of building models in the future by starting with one I already configured

*Application*: Face and Food Detection Website

*Flow:*

* Browse to the Food Detection Settings page and then the Model Management Page
* Click on the Clone button on one of model rows and be taken to the Create Model Page
* On the first tab, do the configuration and click next
* On the second tab, select the publishing information
* Select Create and it does everything it normally does

 

*Background:*

* We need a way to speed up model building in the future
* Example: Imagine a user has configured a particular model in a special way but now needs to update with the latest trained images.  They would have to go through all the effort of remaking the model.  But with this feature, they can autoload Create Model and not have to redo it

 

*Assumptions:*

* It is assumed you have done the stories for Add Tabs to Build Model, Add Publishing Models to Sites and Add Sites to Model Building

 

*Requirements for Model Management page:*

* On the rows, under the actions column, add another button action and title it: Clone
* The clone button, when clicked takes the user to the Create Model page
** It needs to send the Model ID to the page
** It needs to notify that page that it is the clone action

 

*Requirements for Create Model page:*

* You will not modify any of the existing fields or add any fields
*  
* You need to add in logic on page load to determine if the user clicked on Create Model button or Clone Model button
* If did Create Model, do everything like you normally do
* If did Clone Model, do the following:
** Get the Model ID of the model the user wants to clone
** Get the Model Configuration from the database using that ID
** Load all the fields on the page as from the values of the Model Configuration
*** Don’t try to set the Model ID with the old Model ID; you will generate a new Model ID for this new model
** Let the user make any changes they want and all the buttons continue to work as they already did
*  
* Security – already done

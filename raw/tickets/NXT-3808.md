# NXT-3808 — New Goal

## Description
*User Story*
As an Insights user, I can create a custom Goal using existing Measures.

*Mockup*
[https://qvv7ze.axshare.com/#id=fg7b0q&p=new_goal|https://qvv7ze.axshare.com/#id=fg7b0q&p=new_goal] (Mockup created before having visibility into existing 2.0 mockups)

*Requirements*

* Users must enter a Name into a text field
* Users must enter a Description into a text/paragraph field
* Users must select a “Measure” from a multi-level dropdown list.
** Only enabled Measures will show up
** Metrics
** Default KPIs
** Custom KPIs
* Users must add at least 1 Target with an associated Value, Operator, and Date.
** Users select a Target from a multi-level dropdown list. Target is “*who*” is being measured.
*** Entire District
*** Single Site
*** Single Employee
** Target multi-level dropdown is limited based on the available Level(s) of the Measure selected.
* Users enter the Value into a number field. Target Value is “*what*” value is to be met or exceeded.
* Users select an Operator from a dropdown list. Operator determines if the Goal is to be >= or <= to the Target Value
* Users enter a Date into this date field. Date is “*when*” a Target Value is expected to be met or exceeded.
* Users can create multiple Targets with associated Values, Operators, and Dates.
* Name, Description, and Measure must be selected to Save a Goal.  A Target, Operator, Value, and Date must be selected for each Target added in order to Save a Goal.
* Users are prompted when attempting to navigate away from this page with unsaved changes.

## Acceptance Criteria
New Goals can be created in Insights.

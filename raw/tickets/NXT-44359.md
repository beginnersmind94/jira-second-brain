# NXT-44359 — AcctMgt - Use Tax for Add/Edit Transaction

## Description
As Dora the Director I want to charge tax to adults and need the Add/Edit Transaction windows to properly add sales tax based on a percentage.

*requirements*

* Changes from [https://primeroedgenext.atlassian.net/browse/NXT-44341|https://primeroedgenext.atlassian.net/browse/NXT-44341|smart-link] replicated in Account Management.





Original story requirements:

----

* Charge tax on the Add/Edit Transaction windows if:
** the setting for tax is enabled
** the menu item is marked as taxable
** the account is an adult
* Tax field inside of Add/Edit Transaction is editable
** Add Transaction → Sale Amount → Tax
** Edit Transaction → New Sale Amount → Tax
** Tax calculation should take place, but the user has the option to edit the amount
** Special permission added for this
*** Name: Sales Tax - Editing

## Acceptance Criteria
* POS charges tax appropriately.
** Setting TAXMEAL must be On.
** Use the system setting TAXPERCENT to figure out the sales tax percentage.
** Example: TAXPERCENT is 5%. Item costs $1.00. Tax is 5% of 1 - 0.05c. Item $1.00, Tax $0.05, Total = $1.05
* User can see the tax amount as a separate figure.
* User can see the total amount which is the sales total + tax total.
* Tax amount is stored as a separate amount.
* Tax amount is included in transaction details at the POS.
* Over/under for a session needs to include the tax amount.
* Tax amount is editable for Add Transaction and the new sale amount in Edit Transaction

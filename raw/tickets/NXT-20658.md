# NXT-20658 — MP-Menu- Variety Menu Item Analysis

## Description
As a Menu Planner, I want to get a rough idea of the nutritional analysis based on all my menu items on my Variety Menu Item

## Acceptance Criteria
When performing the component analysis for the Variety Menu Item, the component specified for the Variety Menu Item will be used as my component contribution.

When performing the nutrient analysis, all of the Menu Items used in the creation of the Variety Menu Item will be used and averaged. (i.e. if there are 3 different cereals for a Variety Cereal Menu Item, and those calories are 90, 120, and 150 - the nutrient analysis will display 120 calories … 90 + 120 + 150 = 360 / 3 = 120)

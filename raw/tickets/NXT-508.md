# NXT-508 — Employee Details - Add New 

## Description
Admin can add new employee based on user permissions***. See fields from classic version.

Employee Details Sections:

* Personal
* Degrees/Certifications 
** Degrees will have a Title, Awarded By, Year Earned, Comments
** Certifications will have Title, Awarded By, Issue Date, Expiration Date
* Notes and Documents*** 
** Be consistent with System module, use the upload tool***.
** Add Notes
** Notes will have a time stamp/history
** Document history also shown in the tab***
* Duplicate entry validation/check upon Add/Save- look at ID and State ID if enabled.

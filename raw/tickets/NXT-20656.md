# NXT-20656 — IM- Menu Item Category Configuration Page

## Description
As a Menu Planner, I would like to be able to add or remove Menu Item Categories that would help me better organize my menus.

## Acceptance Criteria
Display all Menu Item Categories created
Schoolcafe created  categories can not be discontinue or edited
No limit on how many categories can be created
Categories can not be discontinue if there are items within that category.
Tool tip  message "Categories associated to an Items cannot be discontinued“

Show column (# of Items) with total count of items associated to menu item category

Display order duplicates are not allowed; error message should be shown to user as “Display Order Already in Use”

Allow user to associate a color with particular menu item category (very important for Menu Planner)

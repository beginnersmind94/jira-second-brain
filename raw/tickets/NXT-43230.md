# NXT-43230 — Custom Reports – Add New Report – Add Parameters to Configuration

## Description
As a SchoolCafé user, I would like to be able to use the report parameters of my custom reports so that I can have more customized reports.

*Flow:*

* Open SchoolCafé and browse to Settings > Dashboard > Custom Reports > Report Parameters
* Optionally add a value to one of the report parameters
* 
* Open SchoolCafé and browse to Settings > Dashboard > Custom Reports > + New
* Go to configuration tab
* Optionally use one of the time filters on a date/time field
* Optionally use one of the location filters

*Requirements*:

* These requirements are for the Configuration tab
* Dynamically, at run time, identify the columns that are of a date/time type
* For each one, use the Time report parameter instead of the textbox in the value column
** Ensure that you show the default values we add
** Ensure that you show the custom values the user added, if any
* 
* Show the Location parameter at the bottom of the screen, per Figma
** show the values per what the user added in the Report Parameters screen
*

## Acceptance Criteria
* All date / time fields use the Time parameter
* Values for Time parameter use both standard values and user created values
* Location parameter is on the screen and has all the user created values

# NXT-31649 — MP- Menu/ Menu Cycle Add additional Sodium Target (Target 1A) for Lunch 

## Description
Based on USDA regulations, They are introducing new Sodium Target to be affective July 1, 2023 but would like for the software to include it to start getting users used to it being there. 

We must add a new line on all analysis that includes Sodium Target 1 to add another row that is Sodium Target 1A

Sodium Target 1 will remain the same for all Breakfast Meal Grade Groups through 2023-2024 school year

Sodium Target 1 will remain the same for Lunch Meal Grade Groups through 2022-2023 school year

## Acceptance Criteria
Change Current Name of Sodium (mg) to Sodium Target 1 (mg) on all pages (Breakfast, Lunch) Except PK

!image-20220606-225330.png|width=1021,height=767!



Include New Sodium Target 1A (mg) on a new row for Lunch Meal Type

Targets: 

K-5 <=1,110 mg

K-8 <=1,110 mg

6-8 <=1,225 mg

9-12 <= 1,280 mg

Do not fail analysis if Target 1A is not met until July 1, 2023. Until then, mark as exceeded with color currently used for exceeded 

Changes/Additions to be made for menu:

*Menu Analysis Slideout*

*Nutrient Summary (Nutrition Report) and print options*

Changes/Additions to be made for menu Cycle:

*Menu Cycle Slideout*

*Menu Cycle Nutrient Summary (Nutrition Report) and print options*

*Nutrient Summary Report and print options*

On all pages mentioned, add Disclaimer

_Sodium Target 1A will be marked in orange if exceeded prior to the effective date of July 1, 2023 and does not impact pass/fail compliance prior to that date._

Add indication on Sodium Target 1A Row is being affected by this disclaimer

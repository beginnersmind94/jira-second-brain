# NXT-14064 — [Accountability] - Deposits - Add action button within grid to view details

## Description
As a user I want to clearly see how to view deposit details instead of clicking on the grid so that I am not confused.

*Requirements:*

* Implement the standard grid action for navigating to details by adding an additional column called ‘Action(s)’ located here 
* Within the column, add a details button that redirects the user to the Deposit Details (we can use the standard ‘Details’ icon).

## Acceptance Criteria
Verify the following has been added:

* Additional column: Action(s)
* Icon representation for ‘Details’, used to navigate the user to the deposit details
** Removal of grid row selection as the action to navigate to deposit details

# NXT-18465 — Production- Reports- Total Waste (in Pounds)- (UI and API Report)

## Description
This report will give the user information regarding their waste in pounds per day. 

Page Display needs to have this criteria:
Site Code
Site
Site Group
Meal Type
*From* and to *Date*

Criteria for Columns in report include:
Serving Date
Meal type
Menu Line
Waste (in pounds)

The reports needs to display the school with site code and site total for waste.

Way to calculate Waste:

Serving Size (weight) x amount of wasted serving (waste column)= Total waste in pounds

Example: 

serving size (2 oz)X 6 waste servings= 12 oz =0.75 LB

serving size ( 425 grams) x 6 wasted servings= 2550 grams= 5.62 LB

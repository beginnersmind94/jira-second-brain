# NXT-16355 — INV - Settings - Delivery Locations - View Locations

## Description
As a Food Service Director I want a Delivery Locations page so I can view the existing configuration or set up new delivery locations for all of my schools that have more than location.

Delivery Locations are an optional part of configuration. For schools that have more than one possible location to which an order would be delivered, they may choose to set up “delivery locations.” These locations are an extension of the school and if these are configured then it is an additional criteria required for placing orders.

*Requirements*

* Create a new page under the Settings category, titled ‘Delivery Locations’
** This page will be a list of sites with delivery locations configured
** There will also be a tool to create a new delivery location assignment
* Provide an Add button to create a new delivery location assignment ([NXT-17812|https://primeroedgenext.atlassian.net/browse/NXT-17812])
** This will open a new screen
* Provide a list of the current sites with delivery locations enabled
** provide ‘Field Search’ button
** use pagination
* List columns:
** Site
*** displayed as Code + Name
** Delivery Locations
*** this will list all *active* delivery locations created for the site
*** separate each location with a comma
*** if the number of locations exceeds the width of the column, provide the number of additional locations
**** Ex: “North Mall Kitchen, South Mall Kitchen, Warehouse Bay, and 3 more.”
** Action
*** View icon to open the site details ([NXT-17813|https://primeroedgenext.atlassian.net/browse/NXT-17813])

# NXT-49904 — IM - Recipes - Steps & Ingredients - Lock Measure..

## Description
As a menu planner or nutritionist I want the ability to lock a serving measure when added to a step so that it remains in that measure when I scale the recipe.

h3. Recipe Steps - Edit Step - Lock Measure

!Recipe Steps - Single Step Edit v2-20250911-141737.png|width=91.66666666666667%,alt="Recipe Steps - Single Step Edit v2-20250911-141737.png"!

## Acceptance Criteria
* Provide a lock icon for each ingredient to force the ingredient to scale only in that unit (no unit upconverting)
** Ex: locking ‘Tablespoon’ so it never upconverts to Cup
** Lock icon tooltip: ‘Lock measure for recipe scaling’
* Users can add secondary measures, this measure will be upscaled to the primary measure
** EX: Teaspoon is the secondary, will upscale to tablespoon if tablespoon measure is locked
* If ingredient is available on multiple steps, each step will have its own ability to lock
** If lock is not present on all steps, scale ingredient card to the highest unit (Current Logic)
** If lock is present on all steps, scale to the locked measure
* On report, each scaling will be determined by the individual lock per step

# NXT-26820 — IM - Recipe History (Recipes Yield Factors)

## Description
As Item Manager administrator I need an option where I can observe the history of a Recipes' s Yield Factors Details, for example updates made to the Recipes' s Yield Factors  by who and when so that have an audit process

Also fix the formatting error with First Name / Last Name

* When we create a Recipe it is showing Last name first and when we edit it is showing First name first.

## Acceptance Criteria
*Entry Point*

* Item Management → Items → Selected Recipe → Recipes' s Yield Factors Details -> History icon
* An option to view Recipes' s Yield Factors  Details history shall be available to the user \[ history on ]
* User shall be able to see all Recipes' s Yield Factors  Details property values that were updated
* System date and Time on which the update took place
* User name which performed the update (who performed the update)
* What fields that were updated, preview value(s) and new value(s)
* The information is for display purposes (Shall not be alter by the user)
* Applies to ADD and EDIT item

*General*

* Page pagination shall be present (as standard way)

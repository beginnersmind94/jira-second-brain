# NXT-54928 — MP- Menu Planner- Editing Serving groups on Menu planner on locked menu 

## Description
As a menu planner, if somebody is accessing a menu and it is on my menu cycle that I am working on in the menu planner, do not allow any changes to that menu, especially if I am changing serving groups

## Acceptance Criteria
* Do not effect any menus that are checked out when editing serving groups
** Any changes done to the serving groups whether adding or removing serving groups will only effect the menus that are available to be edited
* User is allowed to refresh the page and if the menu is now available then they can be included in the change but user will have to edit the serving groups again
* If editing serving groups has already been completed and then the menu is now available, the options for the user to add those serving groups to the menu/menus would be to either add from the standard OR user can remove from menu planner, save, then add back and it will effect all menus again

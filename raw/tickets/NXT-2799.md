# NXT-2799 — **Inspection Template - Text Question Type

## Description
*User Story*
As an Inspections user, I can add a Text question to a template when creating or editing the template.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=freetext_question&id=rrknu8&c=1|https://nqyc9v.axshare.com/start.html#p=freetext_question&id=rrknu8&c=1]

*Requirements*

* Text can be selected as a Question Type when adding a question in a Template
** Not available for Pass/Fail scoring type
** A read-only text box is present to display to the user how it will appear
* Character limit is defined by what the database can accept
* There are no scoring options for Text questions
* No validation for the purposes of this story

## Acceptance Criteria
* Text questions can be added to templates

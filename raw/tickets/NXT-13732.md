# NXT-13732 — IM - Recipes - HACCP Process shouldn't be default 

## Description
As a user i would like to have an ability to choose the HACCP process steps so that it’s easy to update the cooking time or process

## Acceptance Criteria
Given the User is on the Edit Recipe page 

When Navigated to the HACCP process block 

Then don’t default the HACCP process steps instead let the user make the selection between NoCook>SameDayService>ComplexFood 

And also Hide the Radio buttons “For Entire Recipe” and “For Each Step”

And enable the HACCP process steps upon any selection of  NoCook>SameDayService>ComplexFood and enable the Radio Buttons ““For Entire Recipe” and “For Each Step”

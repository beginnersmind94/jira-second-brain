# NXT-510 — Admin Dashboard - State

## Description
State realm interactive dashboard - rollup of district compliance data

* Data: Show % of CEUs training hours met grouped by CEU Requirement level across districts
** Filter for annual year and district – default current year and all districts
* Display: Bar graph
** Show % complete for each CEU level, but on hover show # of hours out of total for each level



Ability to narrow data down by district/site/CEU/jobtitle/employee

## Acceptance Criteria
Dashboard shows a rollup of all districts' data to ensure USDA prof standards compliance

# NXT-64844 — WS - Benchmark - Autofill PE & SC

## Description
As a Workspace user, on the Benchmark drawer, I want to be able to load previous year’s values rapidly in the grid, so that I can speed up the process and get more accurate data

*Use Case*

* Users will typically just want to use the previous year’s results as their benchmark for the next year, so we want a button click way of quickly setting it for them
* They might also want current year

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* Above the grid, add a button
** Use Icon button 
** Label: Autofill
** on click:
*** Show a confirmation popup:
**** Text: This will overwrite the existing data in the textboxes with the average values from the previous year. Are you sure?
**** If No, do nothing
**** If Yes, continue
* If confirmed, do the following:
** For each site and for each KPI:
*** If %: Calculate the average value for the previous year
*** If $ or #: Get the total for the previous year and divide by operating days to get the daily number
*** Round the number to a whole number
*** Insert the value in the relative textbox
** Note: you are doing for each site, not for the whole district, so the numbers should match the previous year for sites in the performance grid
** Note: be sure to test on a large site district and make sure it updates all sites
* Allow user to reset to what it was if you can
* If the district has less data available than the whole year, then calculate with the data you have
* If the district has no data for the previous year, hide the button
** Note: the drop down filter also looks for previous years, so you can tie into that same check
* If you have audit log enabled, be sure to record the change in the audit log
* 
* Above the grid, add a button
** Use Icon button so that you don’t have a huge button on it
** Tooltip: Load Current Year’s Data
** on click:
*** Show a confirmation popup:
**** Text: This will overwrite the existing data in the textboxes with the average values from the current year. Are you sure?
**** If No, do nothing
**** If Yes, continue
** Do everything else the same as you did for previous year, but for current year’s data

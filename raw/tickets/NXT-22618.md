# NXT-22618 — INV - Vendor Detail - Contacts Card - History

## Description
As an Inventory manager I want history for my vendor Contacts card so I can see how contacts have been added, removed or changed over time.

*Requirements*

* Use the new [standard history component|https://qasharedutilitiesservice.perseusedge.com/swagger/index.html] developed by Front Office
* Add History button/icons to the Contacts card
** one single history for the entire Contacts section
* Show history for all attribute changes
** Display ‘New Contact Added’ when a new contact is added
*** show all the attributes which were populated at the time the contact was created
*** old values would all be null; new value is what was entered
** Display ‘Contact Deleted’ when a contact is deleted
*** old value would be actual value; new value would be null

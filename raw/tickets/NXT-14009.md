# NXT-14009 — Item Management - District Switching

## Description
As an Item Management user, I want to be able to switch districts in the UI.

When switching the district, we need to have it go to the new district’s workspace page. It shouldn’t go back to the same page itself for data integrity. The reason to go back to the workspace homepage now is due to the new district may not have the same permissions and may not be able to access the same module they are in depending on the district.

If that is not possible, then just return the user to the module homepage.



*UPDATE:*
If module has no access, redirect to workspace
If list Pages, refresh data
If detail(view,edit,add), navigate to list pages)

## Acceptance Criteria
User must be brought to the new district to the home page.

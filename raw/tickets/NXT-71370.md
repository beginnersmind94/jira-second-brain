# NXT-71370 — GC - PE -Item Catalog - Enhance Review and Import items functionality - Convert Search Results into a tab and add new Review Items tab for reviewing the selected items.

## Description
# Convert the current search results into one of the tabs of tab container. - no changes to the functionality on this tab, all existing buttons should work. 
# Add new tab for Review Items.
# Remove the dots after Review items label in the “Review items…” button as its not a popup anymore. 
!image-20260406-190914.png|width=139,alt="image-20260406-190914.png"!
# Browse item catalog - search results are populated in the grid : 
## user selects a few items by checking the checkboxes : currently , its limited to 50 items, it helps if we can increase that limit let me know what is the maximum item count we can go up to w/o any performance issues.  
## Click on Review Items button 
## Review items tab will be populated with selected items. 
### GTIN column: GTIN will be displayed as a label if GTIN exists for an item from our catalog otherwise, it will be like an editable textbox to allow users to enter GTIN for an item. 
###* if user enters GTIN for an item which is missing in our catalog: 
###** Item will be imported to district with given GTIN if no other selected item has the same GTIN
###** update catalog Item’s GTIN value  - Do not accept the GTIN if it already exists for an item in our catalog. 
###* only GTIN will be assigned to items in our catalog , no other fields are assigned/updated to catalog items. 
### Ingredient name column: Editable textbox, populate the ingredient name from our catalog by default, user can edit and import the item with new name,
###* if user enters a different name for an item : Item will be imported to district with given ingredient name if no other item has the exact same name as ingredient name is unique in a data source.
### Manufacturer, Product code : Editable textboxes, populate the manufacturer /product code values from our catalog by default, user can edit and import the item with new data,
###* if user modified the value for an item : Item will be imported to district with given value, changes are not going to be applied to catalog item. 
### Serving Size : serving size column will be displayed as a label . no textbox.  if user needs to change this they can edit and save on View/Edit popup for  nutrients . 
### Nutrients, Allergens, Meal Credits, Stock Items columns: display a  View/Edit… button. we will have separate stories for these popups for each column.
### Reviewed column: a checkbox to allow users to mark an item as reviewed by checking the checkbox. 
###* default checkbox status is unchecked. 
###* checking the checkbox will save the item status as reviewed for that user and district session. 
### import column to give import button functionality to import single item by collecting all changes for all sections of the item. asks the user for confirmation before the importing the item. 
### Delete column : Trash can icon is provided for allow removing an item from review items tab if they selected an item by mistake. this is existing functionality on review items popup. 
## Allow multiple batches of items to be assigned to review items session as long as the item count not reached to max count. 
### for example: select 10 items and click on review items  - all selected 10 items are added to review items tab (please note that selected items will not be removed from search results grid)
###   select 10 more items and click on review items  - new batch of 10 items are available on review items tab. total 20 items are in review session 
### if 50 is max count then after selecting 50 items : system will not allow more items to be assigned to review session. 
## Cancel button to cancel the review items session, asks user for confirmation. 
## Save as incomplete : saves the review items session for an user and district combination along with all changes to all selected items in review items tab
## Import items button: to import all +*reviewed*+ items from review items tab (from all pages if pagination is used) along with all changes they made to all items,,
### display an error message if no items are marked as reviewed in the grid.

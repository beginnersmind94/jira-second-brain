# NXT-42856 — PR- Production Adjustment- Adding Variety Menu Item

## Description
As a district manager, I would like to be able to add Variety Menu items onto my production plans from the Production Adjustment page. Currently, it is adding the parent name but not the “child” Menu items.

## Acceptance Criteria
* Allow Variety Menu items to be added as part of production adjustment
** Currently, only the “Parent” menu item is being added

!image-20230919-125454.png|width=1802,height=785!



* Display all “Child” Menu Items with the VMI on Production plan/Record
* Toggle “child” menu items ON if menu item already exists individually
** IF VMI Menu item is added to a *Plan Incomplete* record, and “Child” Menu item is already available as an individual menu item, toggle on “child” Menu item and insert count if available and remove individual menu item
** IF VMI Menu item is added to a *Plan Complete* record, and “Child” Menu item is already available as an individual menu item, toggle on “child” Menu item and insert count and remove individual menu item
** IF VMI Menu item is added to a *Plan Complete* record, and “Child” Menu item is already available as an individual menu item, *DO not* toggle on “child” Menu item IF count is 0 and remove individual menu item
** IF VMI Menu item is added to Plan Incomplete or Plan Complete Record and “Child” Menu Item is already available as an individual menu item via Carryover, insert carryover data onto the “child” menu item and treat the “Child” menu item as a regular menu item
*** IF VMI Menu Item is then toggled on from the Record, show 0 planned count and allow the menu item to still be able to be produced.
*** DO NOT allow “parent” VMI to be deleted if any of the “Child” menu items contain carryover from (Inserted via carryover)
* Insert VMI based on categories available, but always default to the VMI menu Item category if available
* In the case of a individual menu item that is already on the plan or record having a carryover, continue using the serving size that the carryover has

# NXT-30101 — IM - Items - Create New Item - Food Item - Nutrients

## Description
As a District Admin I want a Nutrients Info section so I can specify nutritional information according to my serving measures.

*Requirements*

* Load the Nutrients Info section once the Serving Measures ([NXT-30099|https://primeroedgenext.atlassian.net/browse/NXT-30099]) is successfully saved along with the contribution section ([NXT-30102|https://primeroedgenext.atlassian.net/browse/NXT-30102])
* Once loaded, user has an option to enter in all nutrient information or “skip for now” 
** Skip for now will skip the nutrient section for them to enter at another time
* If user decides to enter information, all information must be entered or marked as missing
* Move ‘Missing Value’ checkbox to left side

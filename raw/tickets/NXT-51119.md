# NXT-51119 — POS - Dynamic Entrees - Option to change sorting method for items in the DE Category

## Description
As Connie the Cashier I want to change the sorting method for the buttons inside the Dynamic Entrees category so I can find what I’m looking for faster and save time on the line.

*requirements*

* User has an button/toggle at the POS to change the sorting method for the items inside the Dynamic Entrees category and on the menu grid.
** Toggle added to Device Information.
*** Only available if the DYNENTREE setting is Yes and the pos site setting is also set to yes.
** Text: Dynamic Entree Sorting
*** Options: Alphabetical, Production Count
*** Alpha is default.
*** If set to Production Count, items should be ordered by the Production Plan count descending.
* Example:
** Burrito - 100
** Pizza - 200
** Spaghetti - 300
* If set to Production Count, the order would be:
** Menu Button 1: Spaghetti
** Menu Button 2: Pizza
** Menu Button 3: Burrito
* If set to Alphabetical:
** Menu Button 1: Burrito
** Menu Button 2: Pizza
** Menu Button 3: Spaghetti

## Acceptance Criteria
* Button/toggle to change the sorting method added to Device Information
** Option only available of DYNENTREE system setting and the pos site setting are on
** Text: Dynamic Entree Sorting
** Options: Alphabetical, Production count
** Alphabetical is the default option.
* User can change the toggle and it is a persistent change on each POS load.
* If Production Count is selected, items inside the Dynamic Entrees category + menu grid should be ordered by the Production Plan count descending.

# NXT-57427 — IM - Item Onboarding - Format Excel Template according to session data

## Description
As an onboarding district I want an import template with formatting that restricts the Excel cells according to the data configured.

## Acceptance Criteria
* Format the Item Import excel file according to the data needs of items in the Bulk Onboarding tool
* When import templates are downloaded for sessions with existing data, read the data to format the file for business rules
* Disable all food related fields for non-food items
* Disable sections not enabled for the session
* Set the serving size weight/volume field according to fluid/non-fluid selections
** otherwise we use the gram or milliliter to _determine_ the fluid/non-fluid flag

# NXT-53193 — IM - Find & Replace - Include Discontinued CN items when 'Pending Deactivation Ingredients Only' is selected

## Description
As a menu planner I want the Find & Replace page to include Discontinued CN items when I select ‘Pending Deactivation Ingredients Only' so I can remove those ingredients from my recipes just like local ingredients.

## Acceptance Criteria
* Associate CN Discontinued status with local Pending Deactivation status for code mappinng
* Include any CN ingredients that are in Discontinued status when the ‘Pending Deactivation Ingredients Only’ 
* Do not show Discontinued CN items when using the Replace With search (neither Inactive obviously)
* Do not change CN statuses through F&R procedures

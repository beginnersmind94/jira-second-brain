# NXT-54084 — PR- Reports- Plan Worksheet- Add Menu line filter and allow multi site selection

## Description
As a production manager, I would like to be able to provide my managers with the plan worksheets per menu line and also the ability to give multiple sites this information in one go instead of having to select one site at a time.

## Acceptance Criteria
* Provide an all selection and multi select for site group/sites to able to generate reports for multi sites
* Provide an all selection or multi select for meal type
* Provide a new filter for menu lines with all and multi select
** Menu lines display should only display the menu lines available for that site/sites *within date range selected*
*** Remove Date range max of 60 days and allow for N number of days that the user prefers
*** Provide a message if the date range is too long (60 days or longer) or loading data is longer than 30 mil seconds (Check with IM/INV team as to the messaging they are providing)
** IF all or Multi select is selected for meal type, display (B), (L), (S), (Sup) for breakfast, lunch, snack, supper so users can know the difference IF menu lines are the same for multiple meal types that are selected
*** EX: Main line for both Breakfast and Lunch and if they only want to print/view one of them

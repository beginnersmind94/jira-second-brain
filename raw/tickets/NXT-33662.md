# NXT-33662 — INV - Reports - Price Details - Update for Pack Size changes

## Description
As a District Admin I want the Price Details report to reflect the way items were used on transactions and what pack size units are in those transactions.

*Requirements*

* Update the Price Details Report to include Pack Size details in the Quantity column
** -show the full quantity string in each place it is presented-
** Show each unit clumped together but on its own line
*** all units of an item should be under the same item heading
*** show the unit description for each transactional unit 
** Add unit description details to the quantity column
** Show conversion quantities (*Sub Qty)* somewhere (after Item Description?)
* Show the Price for each unit in the selected batch
* -Provide an average price for the Item Total, showing price for highest unit in the batch/transaction (just like Withdrawal Ticket)-
** -this is the only part of the report that consolidates the units in a single cell (i.e. 5 Case, 4 Bag)-
** Removing Average from the report for now, for ease of communicating average highest unit price (hierarchy complexity)
* Remove Transaction Total cell for Item Quantity column and Item Price column
** these are not logical to average together
* UI Corrections
** Remove unit in parentheses after Item Description
** Improve header alignment
*** values are too far right compared to header labels

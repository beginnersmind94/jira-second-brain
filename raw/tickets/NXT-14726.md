# NXT-14726 — IM - Catalog Integration (Item Catalog) Landing Page Search Result 

## Description
As an authorized User I would like to Search for items in Global Catalog so that I can see a List of Items, based on specified selection criteria, in the Search Results

## Acceptance Criteria
Given the User is an authorized user and the user is on the Item Catalog Page 

When the User Search for items based on selected criteria  

Then the list of items that matches the search criteria should be displayed in the Search results 

Example  - The user selects a manufacturer ( Pepsi-Cola US) and an item name (Dole)  the user  should bet able to see all Dole items for specified manufacturer in the Search results

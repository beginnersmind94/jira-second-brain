# NXT-3034 — * Template Hub - List View UI

## Description
*User Story*
As an Inspections user, I can view Templates on the Template Hub screen in List View and switch between List View and Card View without any loss of information.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1|https://nqyc9v.axshare.com/start.html#p=template&id=7r5wnv&g=1]

*Requirements*

* User can toggle between Card View and List View when viewing Templates
* Toggle is done through the buttons on the top right
* List View is the default
* List View
** Templates are stacked on top of each other in a list
** Default sorting is alphabetical by name, from most recently edited to least recently edited
** Sorting can be changed by clicking on the individual column headers
*** Can only be sorted by one column at a time - clicking a new one overrides the sort
* Default View
** Template Name
** Status  (Active/Inactive)
** Last Updated By
** Last Updated Date
** No. Of Questions
** Scoring
** Actions (….)
* Actions are available but are placeholder for the purposes of this story.
** Copy for Active and Inactive
** Edit for Inactive
* Default view is Active
* Default sorting is by Template Name

## Acceptance Criteria
* List view is present in a visual format
* Sorting is functioning in list view

Note: Using dummy data is acceptable to demonstrate completeness

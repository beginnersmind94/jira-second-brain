# NXT-56764 — PR- Plan/Record- Status Change based on changes on Menu from Find and Replace

## Description
As a production manager, if changes on the menu were made that warranted changes in the status on the plan  and record pages, the system should automatically update the status instead of me having to resave the plan.

## Acceptance Criteria
* Changes on the menu from find and replace should trigger status changes based on what is going on 
* IF Plan Incomplete and menu items were removed to warrant a plan complete status, change status to plan complete on next plan load
* IF Plan Complete and menu items were removed to warrant a record complete status, change status to record complete on next plan/record load

# NXT-59071 — INV - Vendor & Contract Items - Update Price - Option to use Standard Price

## Description
As a food service admin I want the Contract Items tools to have an option for assigning standard price as the vendor price, so I can do the configuration once and use it for both IM and Inventory.

h2. Contract Items - Update Price

* with ‘Use Standard Price’ option

!Contract Items - Update Price - Effective Date v2-20250318-132150.png|width=100%,alt="Contract Items - Update Price - Effective Date v2-20250318-132150.png"!

## Acceptance Criteria
*Provide Standard Price option*

* Update both the Vendor > Items page and the Contract Items page to support a new option to the ‘Update Price’ mode
* When in the Update Price more, provide an option to ‘Use Standard Price’
** When clicked, prompt the user ‘Do you wish to use the Standard Price as the Vendor Price?’ *Cancel/Confirm*
** Only populate the fields which are blank
* After confirmation, populate the Vendor Price field with the value of the current Standard Price
** do not auto save anything
** The user still has to provide the Effective Date

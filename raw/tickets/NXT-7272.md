# NXT-7272 — INV - Orders - Order Detail - Main - Order Header

## Description
As an Inventory user I want an Order Detail page so I can open an order and view the details, regardless of which status or state the order is in.

Order Detail page is the page that is accessed when opening a single order. There will be additional tabs on this page to handle the different states of the order (editing, receiving)

+*Views/Tabs*+:

* View (Main) - *this story*
** the default view of the screen
* Modify (edit) - [NXT-9343|https://primeroedgenext.atlassian.net/browse/NXT-9343]
** a view that changes the screen to allow new items to be added and quantities to be entered/changed
* Receipts - [NXT-9732|https://primeroedgenext.atlassian.net/browse/NXT-9732]
** a view that shows how many of each items is received or pending receipt

*Requirements*

* Display Order Number in page header

+Order Details Header:+

* Site
** Code + Name
** Ex: 123 - Primero High School
* Vendor
** display only Vendor Name (no vendor number needed)
* Order #
* Order Date
** Date and Time
* Delivery Date
** Date and Time
*** Date comes from order creation
*** Time comes from Vendor Contract delivery time (as part of Delivery Schedule)
*** This will be saved on the order record as part of Sprint 46 story
* Status
* Comments
* Entry Method
** Standard
** Par Level Order
** Production Order
** Automatic

# NXT-23257 — Production - Production Adjustment (Selection Criteria and Menu Item Info GET) (Step 1)

## Description
As a menu planner, I need to be able to make changes to multiple sites within my district at one time. This could be because multiple sites are having an event such as grandparents day, or because of product that is not getting delivered from the vendor, but they’ve offered a substitute item, which I now need to enter on the menus of sites that will be receiving that item. 

I need to be able to select a date, a meal,  and a site group, and assign a menu item to that selection… with the additional ability to narrow down to specific sites within the site group. 

Making the selections for Meal Type, Site Group, Menu Line, and the Date or Date Range that I want to impact along with what menu item I would like to add to my plans

## Acceptance Criteria
User will be able to select a date, a meal service, a menu line,  and a site group, and assign a menu item to that selection… with the additional ability to narrow down to specific sites within the site group. (Field Search on Plan Grid)

Field Search should already be available with text boxes without user having to click to field search, on the site grid.

Allow all criteria selection to be ALL as an option

Search field for Menu items *(All menu items should be available to be added)*

* Once menu item is selected, criteria is no longer editable, user must click cancel to clear the menu item selection
** Wording for pop up ( Are you sure you want to cancel your menu item selection?) Yes/No

Clear is also available on the smart search with cross (Does same functionality as cancel)

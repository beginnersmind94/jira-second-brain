# NXT-8627 — Data Exchange - Enhance Import UI Component with preview file option

## Description
Enhancement story for existing Upload file UI component to include preview of the uploaded file as defined in the UX mockup. 

Mockup: [https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=184633%3A191106&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=184633%3A191106&scaling=scale-down-width|smart-link]

## Acceptance Criteria
* Component available for use across the product for file uploads with preview options
* Allowable file extensions: .txt,.csv,.doc,.docx,.pdf,.jpg,.png,.bmp,.msg,.xls,.xlsx
* Max size of each file type is configurable - Set to 5MB for images, 250MB for documents
* File should have preview in the prefix icon next to the file name

# NXT-20650 — INV - Vendor Detail - Info tab - Vendor User Card

## Description
As an Inventory user I want a Vendor user card so I can manage the image and status of my vendors.

*Requirements*

* On the Info tab of the Vendor Detail page, show a left-side “user card” for the vendor
* Display the following in the card
** Vendor Image
*** this will show the uploaded image (in the future will be imported)
*** if there is no uploaded photo, display the first letter of the company name as a default logo
** Vendor Name - heading
** Phone Number - smaller text below the name
** Email Address - smaller text below number
** Status button/label 
*** Active/Inactive
**** use different color for each
** Edit icon
*** Clicking the button enables the edit mode
**** this displays a radio option for ‘Active’ or ‘Inactive’ 
**** this should also display a photo upload button for the Vendor Image (“Select Image”)
***** please follow standards for image upload control regarding image size, types, etc. (*may add details to the story)
**** also display save/cancel icons
*** Prompt on Inactive status 
**** “Are you sure you want to inactivate this vendor?” Yes/No
*** Alert for pending transactions
**** If there are pending orders/receipts for the vendor
**** “There are pending orders or receipts for this vendor. Are you sure you want to inactivate?” Yes/No
**** Inactivating a vendor should not cause problems for existing order/receipts for that vendor

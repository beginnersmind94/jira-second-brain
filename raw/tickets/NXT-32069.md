# NXT-32069 — INV - Orders - Order Details - Update for Pack Size Units

## Description
As a Site Manager I want to see all orderable items as separate rows on the Order Details screen so I can view items in the way they were ordered according to the contract.

*Requirements*

* Update the Order Details screen to reflect the new pack size configuration
** This should appear the same for Standard and Production orders
* This page will still only display items which were explicitly added to the order
** multiple item units will not be displayed unless those units are specifically on Contracts and Ordering Conditions
* Show item units on individual rows but indicate items which are linked together
** It does not need to be an actual TreeList grid, but if we can simply indicate when items are child units this will be sufficient
* Replace Unit Type with Unit Description
** could possibly remove this since it is displayed with Order Quantity
* Display Item # only once for the highest orderable unit
* Show conversion (sub) quantity 
* Ensure Add Item to Order function is updated for pack size as well
* +-UI Improvements:-+ (to be done in dedicated story)
** Reduce header height
** Move ‘Total Order Amount’ to a footer
*** Implement sticky footer (for Details tab; different functions on other tabs)
** Reevaluate banner approach
** Reposition buttons

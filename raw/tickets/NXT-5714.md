# NXT-5714 — INV - Create Orders - Order Review/Confirmation

## Description
*User Story*
As an Inventory user I want a review screen with my generated order information so I can review or make changes before saving/creating the order.

This page will list the individual orders that are generated from the Create Order Process. Orders will be generated according to Vendor and Delivery Date.

Splitting and creating orders is part of [https://primeroedgenext.atlassian.net/browse/NXT-15877|https://primeroedgenext.atlassian.net/browse/NXT-15877|smart-link]

*Requirements*

* Create a grid list containing the list of generated orders
* Within each order row display the following columns:
** Order # 
** Vendor (name)
** Delivery Date
** Delivery Location
*** only display this column if Delivery Locations are configured for the site
*** this will be a dropdown containing the list of configured delivery locations
*** whenever user makes a selection, automatically save/retain that selection
** Status
*** all orders will initially be ‘Incomplete’
** Action/alert column - does not require a label - will display icons
*** (!) icon will be displayed if the order is violating any contract purchase limits or item quantity limits
**** Tooltip messages:
***** if Order Total does not meet contract minimum purchase amount: “Increase Order Total by $XXX.XX to meet minimum purchase amount.”
***** if Order Total exceeds contract maximum purchase amount: “Decrease Order Total by $XXX.XX to meet maximum purchase amount.”
***** if one or more items on the order are violating minimum or maximum order quantities: “Adjust item order quantities to satisfy minimum or maximum order quantity limits.”
*** an Edit icon for editing an order ([NXT-9678|https://primeroedgenext.atlassian.net/browse/NXT-9678])
*** a Comments icon for adding or updating comments ([NXT-15887|https://primeroedgenext.atlassian.net/browse/NXT-15887])
** Order Total column
*** this will display the total price of all items on the order
* Within each order, display a sub-grid of the items on the order
** Action/Alert column - will display icons
*** (!) icon will be displayed if the item is violating any contract order quantity limits
**** Tooltip messages:
***** if quantity does not meet minimum: “Increase order quantity by X to meet minimum order quantity.”
***** if quantity exceeds maximum: “Decrease order quantity by X to meet maximum order quantity.”
*** Edit icon for editing an item ([NXT-15885|https://primeroedgenext.atlassian.net/browse/NXT-15885])
*** a Comments icon for adding or updating comments ([NXT-15887|https://primeroedgenext.atlassian.net/browse/NXT-15887])
** Item #
** Item Name
** Whole Units (single column)
*** Whole Unit Quantity + Whole Unit Description (ex: 12 cases)
** Broken Units (single column)
*** Broken Unit Quantity + Broken Unit Description (ex: 4 packets)
** Unit Price
** Total Price
* Display a footer with the ‘Combined Order Total’ label
** this will display the total dollar amount of all orders on the screen
** this should be a footer and will be visible when orders are collapsed
* Back button to return to the previous Enter Item Quantity screen ([NXT-16044|https://primeroedgenext.atlassian.net/browse/NXT-16044])
* Create button to create the eligible orders ([NXT-15883|https://primeroedgenext.atlassian.net/browse/NXT-15883])

+*Validation Alerts*+

* Orders meet minimum and maximum order amounts ($) per contract
* Items meet minimum and maximum order quantities per contract

# NXT-16335 — INV - Create view-only Comments popup and update icon

## Description
As an Inventory user I want a comments box without save buttons so I can see view-only comments.
As an Inventory user I want to see a different icon for Comments when some comments have already been entered, so I can easily see which orders/transfers/items/etc. currently have comments.

*Requirements*

* +Popup+
** Show a version of the Comments popup that does not have save or cancel buttons
*** Provide a ‘Close’ button
** Display the Comment text without a textbox
** This version of the popup will be displayed when orders/transfers/etc. are not editable anymore
* +Icon+ (check with Platform Team)
** Everywhere the Comments functionality is enabled should have two possible icons
*** One icon for when there are no comments currently entered
*** Another icon when some comments already exist 
*** If user has comments entered but removes them and saves, the icon should revert to the original icon
** Update every existing page that uses the comments icon to reflect this
*** Ensure that the same icons are used everywhere throughout the module

## Acceptance Criteria
* Create a Comments popup for view-only scenarios
* Popup will not have Save or Cancel buttons
* User can close popup
* Comments text will not be displayed in a textbox

* Comments icon will appear differently if comments are saved
* Comments icon will be consistent throughout Inventory

# NXT-17773 — INV - Settings - Site Par Levels - Set Par Levels

## Description
As a site ordering manager I want the ability to configure par levels for certain items so that the system can provide recommended order quantities to stay at the par levels defined.

This is a continuation of [https://primeroedgenext.atlassian.net/browse/NXT-1770|https://primeroedgenext.atlassian.net/browse/NXT-1770|smart-link] 
This story will capture the editable mode of the Site Par Levels page.

*Requirements*

* On the Site Par Levels page, when the user clicks Edit, the page will change to a mode in which users can enter or update par level quantities.
* When user clicks Edit, do the following:
** Reload the items list to contain all items with a site-item record
*** items that have been ordered/received/transferred/etc. by the site
*** OR items that are on an active vendor contract for that site
** Change the Par Level column to an editable state
*** this will be a textbox input field
*** this is where the user can enter the Par Level quantity for that item
*** this would display the currently saved value for any items that already have a par level
**** must be greater than zero
*** the Whole Unit description should be displayed as a label after the textbox
** Replace the Edit button with Save and Cancel buttons
*** this would allow the user to save the entered/update values for item par levels
* After the user has saved or cancelled the Edit mode, return to the view-only version ([NXT-1770|https://primeroedgenext.atlassian.net/browse/NXT-1770])

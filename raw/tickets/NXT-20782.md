# NXT-20782 — IM - Create Item As Single Item Recipe

## Description
As a user I need the option create a single item recipe from my Item page (Create from an existing Item)

* ex. If I have a rice Krispy treat,  and that it will be be my only item in the recipe there is no need for me to create a brand new recipe. Give me an option to create a single Item recipe

ACEPTANCE

* Display option to create item as recipe  on all items
* When the user click create item as recipe use current item name as default name but allow user to change/edit
* Use the item information to automatically populate Nutrition, Contribution and Allergen information for the single item recipe
* All serving sizes will be allowed in the recipe 
* The first serving size will be use as default unless specified by the user 
* Allow user to enter in information as normal recipe as seen below
* Do NOT allow for recipe Steps to be added since this is a single item recipe
* Mark the recipe generation as Single Item Recipe
* Do not display Yield factor  or recipe step or HCCP 
* Creation as recipe button will not be displayed  item when single item recipe
* when saved the system will display “Single Item Recipe Created”

Mandatory fields:

* Recipe Name
* Data Source
* Measure Size
* Measure Description
* Number of servings

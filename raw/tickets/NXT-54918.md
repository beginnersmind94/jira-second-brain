# NXT-54918 — MP- Menu Planner- Allowing change of serving groups on the create (UI ONLY)

## Description
As a menu planner, the system does not allow me to add or remove serving groups when creating or menu cycle

## Acceptance Criteria
* Allow users to edit serving groups on the create process of the menu planner, currently once the serving groups are selected the user is stuck until they save or save and close
* Allowing edit to serving groups should work the same as the edit scenario
** If menu items are already added to a particular serving group/s and another serving group is added, automatically add item to that serving group as well 
** User will need to remove any item from a particular serving group if not needed
** One serving group must always be present

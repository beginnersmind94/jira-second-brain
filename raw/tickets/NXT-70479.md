# NXT-70479 — MP - Menu Cycle - Menu Cycle List page filters

## Description
As a menu planner, I would like for a way to be able to filter menus and have just the appropriate columns needed 

!image-20260310-163322.png|width=1009,alt="image-20260310-163322.png"!

## Acceptance Criteria
* Default Columns : Name (Rename to Menu Cycle Description), Site Group, Meal Type, Meal Pattern and action Column
** Remove Columns On Calendar, Creation Method, Number of Weeks, Number of Days and Status
** Lock Menu Description and Action (Dock Action)
** Created Date and Updated date need to be left aligned
* Provide filters for Status, Meal Type, Meal Pattern, Site Group, Number of Weeks and Number of Days
** Default to only show Active menu Cycles
** Show icon for inactive
** Show Bubble for Number of weeks and days in menu cycle name
* Change the landing header to Menu Cycles in navigation and on the breadcrumbs

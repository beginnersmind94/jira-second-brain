# NXT-28516 — IM - POS (DM Reimbursable Meal)

## Description
As a Item Management Administrator or Nutritionist I need to be able to set a menu item as a POS Dynamic Reimbursable Meal so that the item can be added to as such to the POS sale' s grid

## Acceptance Criteria
*Entrance Point:*

* Item Management → Items → Selected Item (POS) (List/Split view)



* User shall be able to see flag DM Reimbursable Meal (Check Box)
* User shall be able set the flag on or off
* This applies on Item creation (POS), Item (POS) Edit 
* Applies to Items List and Split view modes
* Not a mandatory field
* Publish data to any other module that consumes this data

!image-20220317-083120.png|width=2372,height=1142!

*Validate:* When a DM Reimbursable Meal Flag is set there is not duplicate for combination of  \[Button Name] + \[Meal Type] + DM Reimbursable Meal Flag.

# NXT-45478 — AcctMgt/Acc - Preferred Name on Receipts

## Description
As Cali the California Director I want to see the use of preferred name on Receipts.

*requirements*

* If Preferred Name is filled out, use it on the receipts from both modules.
** Anywhere we use Firstname, go with Firstname (Preferred)
** Example: Smithers, Matthew (Matty)

## Acceptance Criteria
Website transaction receipts to use preferred name field if available.

Name format updated to always show Lastname, Firstname (Preferred)

* Smithers, Matthew (Matt)

If no preferred name has been set, do not show brackets.

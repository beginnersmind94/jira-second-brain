# NXT-57382 — IM - Item Onboarding - Update Contributions - Enter Values (Step 2)

## Description
As an onboarding district admin I want a step to enter Contribution values so I can finish setting up the food components enabled on the previous step.



!IO - Contributions Values-20241211-154916.png|width=91.66666666666666%,alt="IO - Contributions Values-20241211-154916.png"!

## Acceptance Criteria
* Provide an edit mode for ‘Enter Contribution Values’ which uses the food components selected on the other edit mode
* Dynamically size the column widths based on the component needs determined by the previous step
** For example, if GBD is not applicable to any Grains, the Grains column can be less width
** if there are no items with the Milk component, the Milk column can be very small (or hidden)
* Display smaller component icons to the left of the row by Item Description, for each food component selected on the other step
** (R&D) determine ideal location: Item # column, Description column, etc.?
** show the icon in opaque view if the value is missing
** color the icon in full when the value is provided for that component
* Display the corresponding Contribution value dropdown for each food component enabled on the previous step
* Display sub-identifiers like ‘Juice’ or ‘Non-WGR’ below the contribution value dropdown
* For Vegetables, insert a row for each sub-group selected
** Display a container box around these multiple listings
** Display Juice underneath the container box if it was enabled on the other edit mode
* Provide a Save button to go back to the review/view screen
* Provide a Back button to move back to the [Select Components step|https://primeroedgenext.atlassian.net/browse/NXT-57377]

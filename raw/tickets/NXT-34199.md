# NXT-34199 — MP- Menu/Menu Cycle- Display menu and menu cycles by recently updated order (API ONLY)

## Description
As a menu planner, when I land on the menu or menu cycle page, I would like the menus to show in the order of the last worked on menus or last updated.

## Acceptance Criteria
* Display Menus in order of most recently updated so that the most recently created menu or edited menu would show on the top. (DEFAULT VIEW)
* Display Menu cycles in order of most recently updated so that the most recently created menu cycle or edited menu cycle would show on the top. (DEFAULT VIEW)
* When searching using field search, display in the same order
** Once a user clears the search, display again the most recently updated (DEFAULT VIEW)

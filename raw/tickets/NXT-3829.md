# NXT-3829 — Account Management - Advanced Search - Demographics Section - Language Subsection

## Description
The Language Subsection will allow the user to filter the patrons list by their Language. The user can select one or more. As each specific value is selected it is ‘or’ed with previously selected Race values. The Language are listed in Alphabetical order. As languages are selected they are moved to the top of the list. Unselected languages are separated from selected languages and are shown in alphabetical order.

!Screen Shot 2020-04-29 at 2.57.26 PM.png|width=284,height=186!

## Acceptance Criteria
The filter values are populated from the DB table

There is a search field that allows the user to search for a name

The search field shows partial matches.

Filter values can be selected by clicking the value

There is count of the number of records matching the filter value.

The when there are multiple selections in the Language section they are 'or'ed together.

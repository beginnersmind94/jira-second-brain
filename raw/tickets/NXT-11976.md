# NXT-11976 — Data Exchange - File Export - SIS Export to CSV file for Pilot

## Description
*Summary*

As a Perseus 2.0 Pilot user,  I would like the ability to schedule an export of student information to occur nightly.

*Mockup*

In progress.

*Acceptance Criteria*

* A “File Export Dashboard” shall be available within the “Configuration” section of the menu above the “File Import Dashboard”
* The “File Export Dashboard” shall consist of a list of a single export entry consisting of the following columns:
** Name
** Added on
** Actions
* The list of jobs shall be sorted by “added on” date (newest to oldest)
* The list of jobs shall handle pagination with default display rows option
** An “Export” button shall exist as the only action
* Upon click of the “Export” button, the export shall create a file named "Student Eligibility Export.csv" file that is delimited by commas and qualified by double quotes. It shall contain the following fields:
** Student ID
** Eligibility Status Code
** Account Balance

# NXT-17504 — INV - Manage Orders - Order Detail - Orders in 'Resubmit' status

## Description
As an order reviewer I want orders in ‘Resubmit’ status to function differently on the Order Detail page so I can ensure these orders will be resubmitted for review again.

*Requirements*

* Once a Pending Review order has been rejected (covered in [NXT-16894|https://primeroedgenext.atlassian.net/browse/NXT-16894]) that order will go into ‘Resubmit’ status
** this is a new status which needs to be created
*** this new status will not be eligible for Merge Orders (only Open and Pending Review orders can be merged)
** this should be reflected as an option in the Manage Orders list as well
** this will only be used for orders that have been rejected (for now)
* On the Order Detail page, editing/resaving the order will immediately send that order back to ‘Pending Review’ status
** we do not need to check Order Review rules for this; orders in ‘Resubmit’ status will always advance to Pending Review
** the banner will be displayed as “Order #12345 is in Resubmit status. Please view comments, make updates and resubmit this order for review.”

## Acceptance Criteria
* Resubmit is a new order status which needs to appear on the Manage Orders page Status dropdown
* Resubmit orders will always go back into ‘Pending Review’ status after changes are saved to the order
* Resubmit orders will not be eligible for merging

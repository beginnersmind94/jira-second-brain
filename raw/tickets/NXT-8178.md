# NXT-8178 — INV - Physical Inventory - Supervisor View - Items tab - Valuation Group organization

## Description
As an Inventory District Supervisor I want a Valuation Group view on the Items list so I can view the item list according to Valuation Groups instead of Storage and Item Category.

On the Pending Reconciliation screen, as a supervisor, I want to be able to change the organization of my items from Storage & Item Category to instead be Valuation Group.

The list of items and the item details (physical count, perpetual count, discrepancy, etc.) would all remain the same, but the grouping and organization of the items would change.

*Requirements*

* Provide an option for viewing the Item list according to Valuation Groups
** only provide the option if valuation groups are configured
* Valuation Group organization would replace the Storage Category & Item Category grouping

May be better to create duplicate table organized by Valuation Groups rather than modify the same table used with Storage & Item Category.

Note about Valuation Price: until inventory is reconciled, the accurate valuation cost cannot be known (due to batches not yet being selected). The price used for this column needs to indicate to the user that it is not exact until reconciled.

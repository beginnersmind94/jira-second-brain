# NXT-25850 — UX - Share the saved view to other users -- Custom Views

## Description
As a user I need a way to share the save views I've created with users.

## Acceptance Criteria
Acceptance Criteria
Acceptance criteria will be met when:
1. When I click on share button of a particular saved view, I should see a popup containing a search bar where I can search for users that I would like to share my view with. 

2. After clicking share button the view should be shared with the other user.

# NXT-71124 — Eligibility - Configuration - FRE Application Image - Display Revelant Application Images in ELIG/FH

## Description
As a Cybersoft admin, I would like to be able to modify the HTML image template for my FRE apps, so that I may make them more closely aligned to my SchoolCafé (and paper) application.

*Summary*

* This is the ‘Part 2’ story of displaying an updated/customized FRE app image
* When users open an FRE application or view it in the Family Hub, display the Default image, with any changes made to it by a Cybersoft user 

*Details*

* When users pull up an Application (not a form) anywhere in Eligibility, display it according to the Default image and whatever changes have been made by the Cybersoft admin.
* Some places to check include:
** Application Details - Application Image
** Family Hub (Parent or Guest) - Submit step of Apply page
** Family Hub (Parent) - Eligibility Info page ‘View’ button
* Additional note:
** If FRENOSSN is set to 'Off', then do not show field content in the 2 SSN related sections of the image. They can be blank or just contain whatever plain text the user enters there.
*** If it is set to ‘Partial’, then it should behave (show or hide SSN content) dynamically based on any CEP students on the app.
** Be careful to accommodate the other existing eligibility settings as discussed.
** Also, when applicants in the Family Hub see this image (such as on the submission step of the Apply for Benefits page or Manage Applications page), the changes made to this image should be reflected in what those users see there as well.

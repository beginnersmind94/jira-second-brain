# NXT-5138 — Workspace - Display Broadcast Banner Messages

## Description
*User Story*

As a PrimeroEdge user, I can received broadcasted messages for important messages sent by either my administration or Cybersoft.

*Requirements*

* Users can receive a banner message that is broadcasted by either their administration or Cybersoft
** How these messages are sent is covered by another story
* A banner appears at the top of the screen when the message is posted.
** If the user is not logged in when the message is sent, it should appear the first time they log in afterwards
** Banner remains until the user dismisses it
** The banner can display the first 75 characters of the message.
*** If necessary (Message content >= 75 characters), a View More button should appear and display the full text of the banner when clicked.
**** This should include any links attached to the message.
**** A Show Less button should return the banner to its default state. 
* The banner can appear on every page and persists through navigation until the user dismisses it
** This persists through logins as well
* Banners appear one at a time
** Next one will appear once the one before is dismissed
* Banner only appears on or after its scheduled Post date.

## Acceptance Criteria
* Broadcast message banner appears and persists through screens and login sessions

* Broadcast message can be dismissed

* Banner should display first 75 chars by default

* A View More button should display for messages larger than 75 characters

* When selected, the View More button should:
-Display the rest of the message
-Display any hyperlinks
-Display a 'Show Less' button that hides the content/returns to the default view

* Banner only appears on scheduled post date

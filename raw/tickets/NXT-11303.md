# NXT-11303 — IM - Production Recipes - Recipe within a Recipe 

## Description
As a use, I need to be able to search and add a recipe into my recipe. 

Requirements:

* I need to be able to search by recipe name or recipe code in the same place I search for items in the recipe step
* I need to be able to select the recipe and the number of servings for that recipe by default. Can change to convertible measures if needed
* It will appear just like an ingredient with primary and secondary measures.
* User must be able to enter in directions for the recipe
* This would just use the same search
** Items over recipes, split half and half, so show ingredients first, then recipes first

## Acceptance Criteria
User needs to be able to search and select a recipe, as well as enter it in like a food item

# NXT-35969 — PR- Update Pack Size information on Multiple pages

## Description
As a user, I would like to be able to see my pack size information to reflect on a few pages that currently have this information. 

This story is going to update the food Required with the new pack size changes that were made in IM. 

Update 

*Quantity prepared on the Production Record page*

*Food Required on the Consolidated Plan Tab*

*Food Used on the Prep Site Distribution Tab*

*Food Required on the Distribution Worksheet*

*Amount needed on Grocery List UI and Report*

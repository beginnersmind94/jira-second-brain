# NXT-18498 — IM - GC - P-Item History

## Description
As an authorized  user I need to monitor and or trace the status of Items that have been imported from GC to IM 

Actions:

I want to be able to query or filter the data via:

* Quick filter
** Past 3 months
** Past 6 months
** Past year
** Date Range
*** Between a given start date and end date
* Date fields
** From    (Don't enable From Date,  if Date Range is not selected  from the quick Filters)
** To        (Don't enable To Date,  if Date Range is not selected  from the quick Filters)

I also need to be able to filter by 

* -Organization Level- 
* Role
* Status
* User Name
* User Firs Name
* User Last Name
* Data Source

When filters are applied the results should display the following information:

* Manufarer
* -Ingredient Changes-
* Ingredient Name
* Ingredient Code
* -Stock Item Changes (previews value - Accepted values)-
* -Stock Item Number- 
* Date on which changes were accepted
* Username of who accepted the change



* I should be able to filter by the fields from the result set.
* I also should be able to export the result set to Excel or PDF format file.
* Screen should not allow modifications (add/update/delete) to the data presented

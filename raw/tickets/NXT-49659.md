# NXT-49659 — 2FA - District - Notifications

## Description
As a Customer Support user, I want to be able to notify impacted users of adding 2FA to their methods so that they have a chance to update

*Notes*:

* This story is for notifying users about changes to 2FA authentication methods
* It covers for all authentication methods that use 2FA



*Requirements*:

* These requirements apply to the System > Security page > Authentication tab:
* 
* For every notification that goes out to users when they are impacted, ensure that you check for the second factor
* Right now, they only look at the first or only factor
* Use case:
** District admin adds SMS 2FA as the 2nd factor for SchoolCafe Authentication
** Admin then turns on SchoolCafe Authentication
** Currently, the system will look for who is impacted by SCA and stop
** But we need to have it look for those who are impacted by SMS 2FA as well and notify them too

## Acceptance Criteria
* User are notified when impacted by the 2nd factor of 2FA as well as the first.

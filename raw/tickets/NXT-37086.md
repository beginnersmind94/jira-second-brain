# NXT-37086 — IM - Items - Single Item Recipe - Summary & Steps

## Description
As a Nutrionist I want a Single Item Recipe tab on the Item screen so I can access all necessary recipe info on a sub-page.

*Requirements*

* On the Item details screen, change Single Item Recipe tab to be on a dedicated tab, not shared with Menu Item
* Display two major cards on the sub-page: Steps, Summary
* Once Serving Size is selected, first enable the Summary card so user can enter mandatory Recipe Description and optional Yield Factor
* Display the Summary info in a right-side card which takes the place of the Ingredients card
** Reorganize the data points to fit more vertically 
** Display the _recipe specific_ Yield Factor card on the right side below the Summary
* Display the Steps card as usual
** Update the Steps card to the updated form per [https://primeroedgenext.atlassian.net/browse/NXT-36491|https://primeroedgenext.atlassian.net/browse/NXT-36491|smart-link] 
** Change the Steps card to allow multiple steps for a single item recipe
*** the only constraint on SIR is that they have 1 ingredient
*** Only enable Directions on added Steps (no Ingredients needed)
** Allow CCPs to be added as usual



*Mockups* : 

* Show child serving size row for non-single serve recipes
* Include recipe Yield Factor card

## Acceptance Criteria
* Single Item Recipe information occupies an entire tab on the Item Details screen, not shared with Menu Item
* Steps card appears in the same location as with multi item recipes
* Single item recipes can have multiple recipe steps, like multi-item
* Recipe Summary card is present in place of the Ingredients card
* Recipe Yield Factor information is present alongside the Summary info
* Summary card is styled with updated format (Tags in header, Strict Batching label present, etc.)

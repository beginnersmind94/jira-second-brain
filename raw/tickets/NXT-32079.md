# NXT-32079 — INV - Create Transfer - Items (Step 2) - Update for Pack Size configuration

## Description
As a Site Manager I want the Transfers functionality to allow me to submit a transfer for any unit in an item’s pack size configuration so I can enter the exact quantities I intend to transfer.

*Requirements*

* Update the Transfer Items screen to reflect the updated item hierarchy configuration
** Items will be searched and managed at the Item # level throughout the Inventory category
*** Ex: I need to transfer 2 cases and 4 boxes of Item 100 - Orange Juice
* Display Quantity on Hand
** Present the full string
** Enable Alternate QOH popover ([NXT-26951|https://primeroedgenext.atlassian.net/browse/NXT-26951])
* +Step 2+
** Same item control used for Withdrawals
** Provide Item Category as the first filter
** Add Storage Category as a second filter
* +Update the UI to accommodate multiple unit types+
* Replace Unit Type with Unit Description
** remove the dedicated column, as Unit will be selected after Quantity
* Provide a Unit dropdown after the Transfer Quantity
** Display a label instead of the dropdown if there is only one unit in the pack size configuration
** By default, pick the highest unit for which there is quantity on hand
* Provide an icon to expand the pack size hierarchy
** Only if the item has multiple possible units
*** this adds rows below for each sub-unit in the pack size hierarchy
** Remove the dropdown and lock the row to the specific unit label
** Provide an icon to collapse the pack size hierarchy if the sub-units aren’t being used

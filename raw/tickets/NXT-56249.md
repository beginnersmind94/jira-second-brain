## Acceptance Criteria
* Provide a Bulk Apply button for most columns on the Summary Info screen
** everything except Product Code
* Ensure the column searches are still available
* Allow Bulk Apply from both view mode and edit mode of the Summary Info step
* Upon click of Bulk Apply for a column
** Disable the other columns and hide/disable their Bulk button
** Add checkboxes to the Item # column
** Provide a dropdown/text field in the column header
** Provide an ‘Apply to Selected’ button in the column header
** Change the sticky footer message to indicate the bulk update is being made for that column
** Provide a column-specific save button in the sticky footer
*** e.g. “Save Storage Categories”
*** Return to previous mode (either View mode or one of the 2 Update modes)

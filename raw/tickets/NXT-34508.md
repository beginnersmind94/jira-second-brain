# NXT-34508 — INV - Orders - Confirm Orders - Combine Orders with same info

## Description
As a Site Manager I want the Confirm Orders screen to combine orders which have the same Vendor, *Delivery Date* and Delivery Location so that I am not generating unnecessary quantity of orders to vendors.

*Requirements*

* Provide auto or manual way to combine orders with the same Delivery Date, Vendor, and Delivery Location if applicable
** Delivery Date change was added in R6
** Only orders on the same Confirm Orders screen 
* Check for combinable orders every time an order’s Delivery Date is changed
* Combine all order items together
** Combine order quantities for items on multiple orders
*** Note: this can’t happen on Standard Orders (yet) but can happen for Production orders

*UX Refinement Notes:*

* Provide an action to combine identical orders
** Use the merge icon on combinable orders
** Put the icon in the action column
** TBD: Show some indication or highlight to the combine icon
* Open Combine popup on click of merge icon
** Revised message on popup: “Would you like to combine the following orders into a single combined order with the same….”
** Show the Order # in the popup
** Show the Order Total ($ Amount) in the popup, instead of Delivery Date

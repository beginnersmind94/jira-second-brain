# NXT-26888 — IM - Recipe Report  Book

## Description
As a user I need to be select on or more recipes (list of recipes) recipes for printing  by recipe name, recipe code, tags. 

!image-20220215-202702.png|width=1304,height=1074!

[^Recipe Report Arroz Con Pollo (1).pdf]

## Acceptance Criteria
*Entry Point:*

Item Management → Reports →  Recipe Book Report

*Report Title*: Recipe Book

* The user shall be able to select more than one recipe or all recipes for printing
*  User shall be able to select/filter recipes by recipe name, recipe code \[Smart search, text box based on any of the 2 fields available for selection]
* The recipe (s) shall contain:
**  the recipe number,
** Recipe name,
** potion size, 
** HACCP,
** Yield of recipe (number of servings),
**  Ingredients and amounts of each ingredient,
** Preparation instructions,
** Nutrient analysis of recipe per serving . 
** Recipes in recipe book should be/contain same information of current recipe report 
* User Shall have an option to clear filters
* User shall have an option to return to previous window (recipe landing page)
* When GENERATE is press the report shall be directly created in PDF and printed

*Reports layouts (Nutrient information section) should the same as screen Nutrient information section of Recipe*

*Current Report Layout (Nutrition information) on : Recipe Report -* *_Entry Point Item Management → Recipes → Selected Recipe → More Options → Recipe Report_*

# NXT-6584 — Users - History

## Description
*User Story*

As a PrimeroEdge admin, I have an audit trail for changes made to individual users in the System Module.

*Requirements*

* When viewing a user, there is a tab available named History
** Main tab should be named “User Profile”
* The following columns are present:
** Date/Time
** User
** Update Made
* First entry should be user creation

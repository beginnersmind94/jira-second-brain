# NXT-113 — [TaggedReport] - Accountability - Reports - Menu Item Sales - Item Summary

## Description
As a Site Manager I want a report that shows me sales figures so I can make better purchases.

As a Central Office Admin I want a report that shows sales figures across the district to help make better inventory purchases.

Report Configuration from PE

!Screen Shot 2020-11-23 at 1.46.18 PM.png|width=1130,height=588!

!image-20201204-015019.png|width=1408,height=1039!

*Requirements*

* User can select from multiple options as part of this:
** Area
** Site: single, multi-select, ‘All’
** Grade: all, selection pruned if a site is selected
** Homeroom
** Terminal: any terminal at the district, selections pruned if a site is selected, multi select avail
** Date: use the dropdown
*** Today
*** Last 7 Days
*** Last 30 Days
*** Year to Date
*** Custom Date Range
**** User can pick a start and end date.
** Meal: any configured meal type for the district, multiple
** Menu Item: any menu item the district has configured
** Checkboxes for Meals Only and A la carte Only
** Grouping: District, Site Type, Grade, Terminal
* Report Types:
** Item Summary
** Daily Summary
* -Option to do sales comparisons - month over month, year over year, this time last year, this month last year.-
** -Let the user see how many sales of Item X they had during March 2019, and compare that against March 2018.-
* Option to do a day-by-day breakdown for sales. How many sales of X for each day in the date range?

Reporting:

* Implement the Item Summary version of the report with this story.

!image-20210810-151103.png|width=541,height=334!

## Acceptance Criteria
User can run a report to see their sales figures by menu item, grouped and ordered in different ways.

Item Summary report loading with data.

Daily Breakdown option available.

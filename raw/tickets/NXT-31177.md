# NXT-31177 — DataQuery Framework - Custom View  Change View on Save As

## Description
When a user creates a new custom view via the “Save As” function, the view being displayed needs to switch to the new view once the “Save As” completes.

## Acceptance Criteria
* When a user creates a new custom view via the “Save As” function, the view being displayed shall be switched to the new view once the “Save As” completes.
** This desired behavior is very common and occurs in most applications (Microsoft Word, Excel, …..)

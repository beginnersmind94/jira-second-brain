# NXT-9509 — INV - Transfers - Transfer Request - Site Quantities

## Description
As a site level Inventory user who has the Other Sites QOH permission, I want an additional page on the Transfer Request process so I can enter request quantities for specific From Sites.

This is a continuation of [NXT-9411|https://primeroedgenext.atlassian.net/browse/NXT-9411] for users who have the Other Sites QOH permission. This page will be used to enter request quantities from specific sites based on their quantity on hand.

After the user has added items to the Item Request List and clicked the ‘Next’ button, they are brought to a new page which will provide the details of which site has QOH of the items they selected.

+*Item Request Quantities*+

* Provide a list of the items which were added to the request list
* +Item Attributes:+
** Item #
** Item Name
** WU Description
** BU Description
** WU Request Qty
*** this will be calculated based on the user’s entries in the Site Sub-Grid
** BU Request Qty
*** this will be calculated based on the user’s entries in the Site Sub-Grid
* Provide a Site Sub-Grid which will list all the sites who have some QOH of the item
* +Site Attributes:+
* Site
** Code + Name
** QOH
*** display as a label
*** Ex: 10 cases, 12.00 packs
** WU Request Qty
*** this is where the user will enter how many whole units they are requesting from the site
*** do not allow the user to enter more than the available WU QOH
** WU Description - label
*** Ex: Cases
** BU Request Qty
*** this is where the user will enter how many broken units they are requesting from the site
*** do not allow the user to enter more than the available BU QOH
** BU Description - label
*** Ex: Packs
*** Ex: Each
** Comments
*** user can click to add comments for the site-specific request
* Next button for proceeding to Confirmation screen ([NXT-9408|https://primeroedgenext.atlassian.net/browse/NXT-9408])

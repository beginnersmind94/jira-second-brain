# NXT-24945 — PR- Pass or Fail for Pre- Production Component Report

## Description
As a user, I would like for my Food component Report to tell me whether I am passing or failing meeting my required food components based on USDA provided requirements.

## Acceptance Criteria
Change name from current Food Components to *Pre-Production - Food Components* 

Allow a Pass or Fail at the top of the page at the header with the Pass or Fail in the appropriate colors based on UX component (See Menu Cycle Analysis Food Component report)

Daily Pass/Fail will fail or pass the individual day based on components.

Header Pass or Fail will be overall for the whole week based on the weekly check. If any of the weekly checks are failing the whole food component report fails. 

Apply changes to the print option

Change current design of page to the new color border layout displayed below. (Reference Menu Cycle Food Component Report for Additional guidance)

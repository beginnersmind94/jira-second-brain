# NXT-62993 — INV - Consolidated Perpetual Inventory - Permissions & Tabs

## Description
As a Site Manager I want all my Inventory management functionality to be logically grouped or combined in such a way that it doesn’t clutter my navigation to perform the various transactions that affect my perpetual inventory.

*Goals*

* Aim to consolidate the the different “Inventory” category functions
** Idea: use the Perpetual Inventory page as a hub with action buttons for transfer, withdraw, etc.

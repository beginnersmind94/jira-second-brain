# NXT-36341 — MP- Menu- Fail on daily should apply to components only

## Description
Based on USDA standards, we should not be failing a menu based on them failing nutrients on a daily level. Currently, we are failing users if anything nutrients/ components are failing, however there are no daily standards for nutrients so a user should not fail. 

!image-20230130-183357.png|width=945,height=651!

## Acceptance Criteria
* Do not fail menu if only nutrients are failing for the day and components are passing
** This should apply to the slide out and the nutrition report
* Continue to mark nutrients as low or exceeded but will only be for information purposes only

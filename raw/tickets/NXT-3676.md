# NXT-3676 — View Inspections - Reports Dropdown

## Description
*User Story*
As an Inspections user, I have a method for running reports directly from the View Inspections screen.

*Mockup*
https://nqyc9v.axshare.com/start.html#p=inspections_approved_layout&id=f1misp&g=1

*Requirements*
* There is a dropdown menu that presents all available reports on the View Inspections screen
** Located in the upper righthand corner
** Clearly labeled as for reports
* All reports are available in the dropdown
** For the purposes of this story, reports are placeholder
* Selecting a report brings up a popup where user can select report parameters
** For the purposes of this story, popups should function but are placeholder

## Acceptance Criteria
* Report dropdown present on View Inspections screen
* Reports present
* Popup present
* Actual reports do not need to be able to be run

# NXT-33790 — MP- Menu- Lock access to multiple users when viewing or editing 

## Description
As a menu planner, I would like a way to either be notified that changes have been made on the edit menu page or on the menu planner page so that I do not end up either overriding somebody's work or waste time doing work somebody else already did or locking down menus to only be accessed by a single user at a time. It also would prevent from multiple changes from happening at the same time.

## Acceptance Criteria
* Allow multiple users to view the menu at the same time
* Allow users to view the menu while another user is editing at the same time 
* DO NOT ALLOW multiple users to edit the menu at the same time 
* As a user clicks on the edit and is on the editing view of the menu from the standard edit or menu planner edit, that user has now essentially “checked out” the menu
** Disable the edit option from being available to other users while User 1 is editing menu 
*** Display a tool tip hover on the disabled edit pencil “*This menu has been checked out by (Name of the User)*”
**** If user is already on the landing page with access to edit, and user 2 goes and starts editing a menu, do another check on user 1 entry to the edit menu page, display Warning message: “*This menu has been checked out by (Name of the User).* *Please access this menu in view mode*.”
*** Users can still view the menu
*** This applies to being able to edit menus from the menu cycle, disable the menus that are already checked out and display hover
** On production plan and record, there is a link to access the menu, default to allow the user to access the menu in edit mode, this will check out the menu and give no access to other users in edit mode.
***  However if the menu is checked out, display the menu in view mode and display banner at the top of the screen “*This menu has been checked out by (Name of the User). You are now in view only mode*”
*** If the user2 is still on the menu once the user 1 “checks in”, they can refresh the page and it will put them into edit mode now that no user has the menu checked out
**** This only applies to production plan and record menu link

* There is also a link to access the menu from the assign menu page with the link
** Apply same logic as the production scenario 
* User 1 “checks in” menu back in either being logged/timed out or if they save and continue and gets redirected to the menu landing page (Menu Search Page) by the menu planner mode
* User 1 checks in” menu back in either being logged/timed out or leaving the standard edit menu page
** Any user will now be allowed to edit the menu and the above process happens again

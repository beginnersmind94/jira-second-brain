# NXT-47955 — POS - Picture Mode - Sorting Options

## Description
As Carmen the Cashier I want to sort the cards in a way that makes the most sense for me, to perform my work more efficiently.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=1471%3A906&type=design&node-id=1471-1180&viewport=858%2C1476%2C0.5&t=553BkxmzmsfpvQ2n-1&scaling=scale-down-width|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=1471%3A906&type=design&node-id=1471-1180&viewport=858%2C1476%2C0.5&t=553BkxmzmsfpvQ2n-1&scaling=scale-down-width|smart-link] 

*requirements*

* User has an option to choose the sorting method used for the cards
** Add sorting options to the option menu added with [https://primeroedgenext.atlassian.net/browse/NXT-47956|https://primeroedgenext.atlassian.net/browse/NXT-47956|smart-link]  
** Option name: Sorting
** Options: Last Name, First Name, ID Number
*** User can only pick 1, and can pick if it’s ascending or descending
*** Last Name (ascending) is the default
* User’s option is saved and used as the default on load.
* Cards should update on save without the user having to back out.

## Acceptance Criteria
* Option to change sorting order added
** Last Name, First Name, ID Number
** Ascending and Descending option
* Option is saved as as a default
* Card order should update immediately after the option is saved

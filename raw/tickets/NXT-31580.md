# NXT-31580 — Create New Images Page

## Description
As a site administrator, I want to select an image to train it so that I can build up a large repository for my model to use when it is time to build.

*Application*: SchoolCafé

*Flow:*

* Browse to SchoolCafé > Accountability > Food Detection > New Images
* Optionally filter items
* Optionally click an image to go label it
* Optionally check an image(s) and delete

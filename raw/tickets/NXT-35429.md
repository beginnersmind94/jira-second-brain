# NXT-35429 — Custom Reports – Dashboard

## Description
As a SchoolCafé user, I would like to be able to see and action my saved custom reports so that I can use the information in my decision making and reviews.

*Notes:*

* This page already exists and has some useful functionality already available
* The screen currently is under Management > Reports with the name of Reports but will be moved
* We will be making modifications to it rather than creating a new one

*Flow:*

* Open SchoolCafé and browse to Settings > Dashboard > Custom Reports
* View list of reports
* Optionally add a new one
* Optionally edit an existing one
* Optionally delete an existing one
* Optionally clone one
* Optionally share one with others
* Optionally star a report
* Optionally view the custom report
* Optionally view report details
* Optionally view report history
* Optionally view recent reports
* Optionally manage orphaned reports
* Optionally download the list
* Optionally view report groups
* Optionally create a new Report Group

*Requirements*:

*Menu Changes:*

* Move the page to under System > Dashboard
* Change name from Reports to Custom Reports

 

*Custom Reports Screen:*

* All these requirements are for the Custom Reports screen
* 
* _Upper section:_
* Change title of page to Custom Reports from Reports
* Get rid of Reports shared with me tab
** What is on that tab will be combined with the My reports
** Keep tabs, though, since in a later story, we will add in another tab for Report Groups
* 
* There are two tabs on this page
** One for the Custom Reports and one for the Report Groups
** But for now, don’t add or hide the one for Report Groups

 

h3. Custom Reports tab:

* Get rid of the title for the grid, we don’t use this elsewhere, page title is sufficient
* Update the grid as follows:
* Columns:
* Name
** Keep as is
* Module
** Keep as is
* Data Source
** Keep as is
* Shared
** Values: Yes/No
** Comes from the “if they want to share with anyone” question on the sharing tab
* Scheduled
** Values: Yes/No
** Comes from the “if they want to schedule the report” question on the scheduling tab
* Distributed
** Values: No, Email, FTP, API
** Comes from the “if they want to distribute the report” question on the Distribution tab
* Owner
** Name of report owner
** Comes from the Owner field in the Selection tab
* 
*  Action buttons on rows:
** For now, do not add any action buttons
** The buttons will be added when we add in that feature
* 
* Grid has usual paging, sorting, filtering which is already there
* Grid has export to Excel and CSV which are already there
* 
* Security
** SchoolCafé admins can add/edit/delete any report
** Ensure Roles & Permissions security is applied to this page

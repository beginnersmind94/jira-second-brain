# NXT-27151 — Family Hub - Eligibility - Implement 'Apply as Guest' functionality

## Description
In Family Hub, users have the ability to both create an account and apply for benefits through that account, or sign in as a guest and apply for benefits without an account. However, only the first method has been implemented. The button to 'apply for benefits as a guest' is disabled, even though the setting for it exists. 

Video: https://www.loom.com/share/90a786936b6148858826fd5d178de808

In this story, complete the full integration of applying as guests, such that guest applications can be viewed and processed in SchoolCafé just like standard account applications.

Example as to how this already working in Classic, with the button being functional: https://test.schoolcafe.com/tcs44

## Acceptance Criteria
Verify that users can apply through the Family Hub district homepage using the 'Apply for Benefits (as a Guest)' button.

Verify that these applications (and surveys, where applicable) can be viewed and processed in SchoolCafé.

# NXT-19165 — Production- Dashboard- Meal Count View

## Description
As a user, on my dashboard I would like to have a view where I could see my average/total Meal Count per meal type

There will be different views for this depending on user permissions as well as what type of view they select. 
User permissions will determine what view user will have access to. For example, a site manager in most circumstances will not have access to the district or site group view. A regional manager will have access to site group and site view but will most likely not have access to the district view. 

*ALL DASHBOARD VIEWS WILL HAVE THE OPTION FOR YTD ( YEAR TO DATE), Last 30 DAYS, PAST 7 DAYS, TODAY*

District View (All): Selecting this view will allow user to view a meal count total per meal type for the district as well as display the average meal count per meal type based on site groups. Information should be displayed in a bar graph to make it easy for the user to interpret. 
EX: If there are 3 site groups Elementary, middle, high, display all 3 of those site groups in the bar graph
Information should be based on Actual Meal Count numbers from all production records. 

Site Group View: Selecting this view will allow user to view a meal count total per meal type for the site group as well as display average meal count per meal type based on all schools in a particular site group. Display the average for all schools but only display top 10 schools (by average) in the bar graph. 

Site View: Selecting this view will allow user to view Total and average meal count per meal type by site. 
Total and Average Meal Count 
*Day*: Display total meal count in *pie Chart*
*Last 7 Days*: Display total and average meal count over the past 7 days in *Bar graph* displaying each day total while giving the user an average on the graph and grand total. 
*Last 30 days*: Display total and average meal count over the past 30 days in *line graph *displaying each day total while giving the user the average on the graph and grand total.
*YTD*: Display total and average meal count from beginning of year to date in *Bar graph* displaying each month total while giving user the average on the graph and grand total.

# NXT-12064 — Accountability - Sessions - Filters - Implement Filter Panel

## Description
As a user I want the ability to filter/view my sessions in a way that is not cluttered so that I can save space on the page for easy viewing of information.

*Filter Order:*

# Area
# Site
# From
# To
# Status

*Default Filters:*

* Area - All
* Sites - All applicable to the authenticated user
* From date as today’s date
* To date as today’s date
* Status - All

!image-20210311-185124.png|width=1603,height=934!

This requirement is dependent on the advanced filter framework component that is linked as a blocker until completed.

## Acceptance Criteria
# User should be able to select the Filters option within the session grid header to show the filter options.
# User should be able to apply filter to update results of the sessions grid. Upon applying the filters panel should collapse.
# User should be able to reset the filters to the initial values (default filters).
# Validations for filters should be retained

# NXT-14467 — File Import-Export : Disable auto saving

## Description
In both File Import and File Export, during the editing of the configurations, the new configurations are saved and displayed in next page even when new configurations are not saved.

Refer to 13801 if needed.

## Acceptance Criteria
In both Import File and Export File, the new configurations must be saved and displayed only when the edited configurations are saved. 

Edited configurations when not saved, the Edit page must reflect the old original configurations and the same in next page.

# NXT-71622 — Cash Collection Report - Remove Cards from Report

## Description
Currently, As a left over from adopting PrimeroEdge ways, the report is including credit card sums with cash.
This causes issues for our customers when trying to make clear and accurate reporting of cash amounts.

Thus we will need to modify the Cash Collection Report to remove card transactions entirely, so only relevant payment types are displayed.

## Acceptance Criteria
All Credit Card sums are removed from all calculations used in this report.

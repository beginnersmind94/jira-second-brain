# NXT-6237 — System - Configuration - Add Attendance page - Yearly

## Description
As a Central Office Admin I want to set the attendance factor (AF) and average daily attendance (ADA) for each site, CACFP Approval Number (CACFP) for each month of the year.

* {color:#ff5630}The requirements below correspond to the Yearly Term selection only{color}
* In the Configuration category of System, add an *Attendance* page.
* See Classic screenshots. We can carry the old functionality over and give it a new look and feel. 
* Requirements:
** Various filters, including the Academic Year, can be set.
** User can set the Attendance Factor (AF) - Percentage
** User can set the Average Daily Attendance (ADA) - Count
** User can set the CACFP Approval Number - Count
** AF, ADA, CACFP can be set for individual months OR
AF, ADA, CACFP can be set for the year
** User can configure attendance factor district wide OR
User can configure attendance factor for each site
** User can review historical data. (for last 10 previous academic years)
* Validations:
** AF value range: 0.00 - 100.00. Fields are also *Optional*
** ADA value range: 0 - 1,000,000. Fields are also *Optional*
** CACFP value range: 0 - 1,000,000. Fields are also *Optional*. This type of field may show as disabled in the mock-up, but that is because it will be tied to a license (future development). For now, CACFP fields should show as enabled as well.
* User should be able to change fields based on selections/validation requirements above and commit the changes via saving.
* Page size/initial number of sites should be 25.
* When the user makes a selection in Yearly, populate that change across all months if they change to Monthly. 
* For switching to Monthly term from Yearly: 
** When user switches from Site to District, take the maximum value for each site as the value.
** When user switches from District to Site, take the district value and assign it to each site.

# NXT-33766 — RFP - Sites - Ability To Add Notes and Attach Files To Building

## Description
As a SchoolCafe admin, I would like to be able to add notes and upload files for sites so that I can have additional information stored about it.

*Requirements:*

* These requirements are for Add New Site from the Site Configuration page:
* Add new fields:
* Notes
** Multi-line textbox
** Optional
** All max characters
** If you could use the rich text box, that would be great
* Upload dialogue box
** Give users ability to upload multiple files as attachments to the site
* 
* These requirements are for Edit Site from the Site Configuration page:
* Add the same fields with the same requirements





Original Description:
Ability to add notes and attach files to each school/building record



*Technical Notes*

* Backend: 
** Add Notes column (nvarchar(max)) to PR_Site table
** Add new Table PR_SiteXAttachments 
** Modify CRUD operations for PR_Site to respect these new fields
** Upload files to blob and store fileurls in PR_SiteXAttachments
* Frontend
** Add Notes & File upload controls
** Modify the CRUD to respect new fields
* QA
** Need to verify CRUD with new fields

## Acceptance Criteria
* User is able to add notes
* User is able to upload files for sites

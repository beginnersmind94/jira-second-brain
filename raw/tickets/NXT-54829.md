# NXT-54829 — Workspace - MWD - Acc - Daily Stats per Site

## Description
As Dana the Director I want to drill down and view daily stats for the site I just selected.

*mockup*

[https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=8827%3A5889&node-id=8827-5890&viewport=465%2C491%2C0.13&t=kLhGb1UT9TUh6B9f-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=8853%3A175|https://www.figma.com/proto/MFg1OFhP2GtXhGajDnPKRC/Dec-2023-Systems-FO?page-id=8827%3A5889&node-id=8827-5890&viewport=465%2C491%2C0.13&t=kLhGb1UT9TUh6B9f-1&scaling=scale-down-width&content-scaling=fixed&starting-point-node-id=8853%3A175|smart-link] 

*requirements*

* User can click a site on the Summary grid and be shown a drawer containing a daily breakdown of information.
* User sees a grid with each date as a line, and then sees the daily stats:
** Meals Served, Payments, Revenue, Participation %, Tasks
** Tasks Action
*** Take the user to the End of Day page and load the Site + Date
*** Either a new tab, or needs a working Back button that takes the user right back to here.
* Grid should have standard sorting, filtering, pagination.
** 10 per page as the default.
** Default sort: Date desc

## Acceptance Criteria
* User can click into a site from the Summary grid
* User sees a daily breakdown of stats
* User can click into Tasks and be taken to the End of Day page
** Nice to have: Initiate the search for the site + date (click Apply for the user)

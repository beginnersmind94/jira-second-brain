# NXT-70461 — MP - Menu Versioning - Restore or Create Menu Option

## Description
As a menu planner, i would like an option to be able to restore an older version of a menu to be my current one or I would like to be able to use that version to just create a whole new menu.

## Acceptance Criteria
* The restore icon on the version slide out will provide two options to allow users to make a choice on what they would like to do
* Restore: will allow user to make the version they are restoring as now the current version
** Essentially we will just copy the information from that version and create a new one as the current
** That version will stay in the spot that it is in the version sequence and a whole new instance will be created
* New Menu: Create a new menu and add Copy - as part of the name EX: Copy - Menu Name V4
** This will act almost like a copy 
* If user does not want to do either action, simple reclick the restore icon and the prompt will go away

# NXT-17181 — Export Monitor-Dashboard

## Description
This story is to cover the task related to building Export Monitor .

Exports Monitor will be accessed through system-->ConfigurationExports Monitor 

Export Monitor should also be accessed from the tool bar.

This will be similar to that of Imports Monitor.

Exports Monitor must have filters for Export Type, Date range , status.

Exports Monitor must provide details like the Configuration Name, start time end time, duration, triggered by, status etc.

Exports Monitor should reflect the Status of the job (error, successful or running). Provision must be provided to provide detail logs if needed.

## Acceptance Criteria
# Verify and Validate that the user is able to access the Exports Monitor from System-->Configuration-->Exports Monitor.
# Verify and Validate that a short cut exists to access Exports Monitor from the tool bar.
# Verify and Validate that the filtering conditions exist for the Date range, Status and Export Type and they can be validated real time by refreshing the page.
# Verify and Validate that the the jobs triggered, we have details like given below similar to that of the Imports Monitor.
## Configuration Name
## Start Time
## End Time
## Duration
## Triggered by
## status 
## actions 
# Verify and Validate that additional logs (in depth) can be accessed from the actions menu (similar to that of Imports Monitor)
# Verify and Validate that the user can navigate through the pages of logs and they can be arranges in ascending or descending for all the menus possible.

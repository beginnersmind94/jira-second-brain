# NXT-9411 — INV - Transfers - Transfer Request (Site)

## Description
As a site level Inventory user I want a Transfer Request page so I can request product from other sites or from the district.

The view and function of this page is dependent on a permission for being able to see other site’s QOH/perpetual.

# Site *cannot* see other site QOH
## the request is made to the district who will later select which sites from which the item will be transferred
# Site *can* see other site QOH
## the request is made to specific sites who have some quantity of the item or items

This site-level permissions needs to be created:

* Permission name: Other Sites QOH
* Permission description: This permission allows a site user to see other sites quantity on hand when making transfer requests.

This page is used by a site to request one or many products from other sites.

# If the user *does not* have the ‘Other Sites QOH’ permission, they will have a space to enter the requested quantity for each item they add to the grid.
# If the user *does* have the ‘Other Sites QOH’ permission, they will see a second page where they can enter how much they are requesting from each site that has the item ([NXT-9509|https://primeroedgenext.atlassian.net/browse/NXT-9509])

The Request process will be in multiple steps

# Search items
# Select item
# Review Items / Request item quantities
# Request Site-specific Quantities ([NXT-9509|https://primeroedgenext.atlassian.net/browse/NXT-9509])
## this step only appears to users with the Other Sites permission 
# Confirmation ([NXT-9408|https://primeroedgenext.atlassian.net/browse/NXT-9408])

+*Item Search*+

* A search section will be provided to search items to be added to the Item Request List
* Only items which have some inventory site-item record should be searchable
* Search filters:
** Storage Category dropdown
** Item Category dropdown
** Item # search
** Item Name search
* Apply button
** the items will be loaded in the Item Selection grid

+*Item Selection*+

* The items returned by the search criteria will be populated in a below grid
* This section will be contained to a certain size and use pagination so it does not take up too much vertical space on the page
* The user will have checkboxes for each item they wish to select and add to the Request List
* Items will be listed with the following attributes:
** Item #
** Item Name 
** Storage Category
** Item Category
* Add button
** this will add the selected (checked) items to the Item Request List

+*Item Request List*+

* On the next step, the user will be provided with a list of the items they have specifically added to the Item Request List
** the Site will make some request quantity for each of these items
*** the request functionality will be different depending on whether or not the user has the Other Sites QOH permission
* +Item List:+
* Item# 
* Item Name
* Whole Unit 
** Item Whole Unit description / abbreviation
* Broken Unit
** Item Broken unit description / abbreviation
* +*Request Entry Field*+
* if the user *does not* have the Other Sites QOH permission, they will see two additional columns:
** WU Request Qty
*** numeric entry field for user to enter requested whole unit quantity of that item
** BU Request Qty
*** numeric entry field for user to enter requested broken unit quantity of that item
** Comments
*** the user can click an icon and enter transfer comments for that specific item
* if the user *does* have the Other Sites QOH permission, they will *not* be provided with the numeric request entry fields or the Comments field, and they will instead have a Next button to proceed to the Site Quantity Request screen ([NXT-9509|https://primeroedgenext.atlassian.net/browse/NXT-9509])
* Next button
** If user does not have Other Sites QOH permission and has entered their request quantities, the Next button will navigate to the Confirmation screen ([NXT-9408|https://primeroedgenext.atlassian.net/browse/NXT-9408])
** If the user does have the Other Sites QOH permission, the Next button will navigate to the Site Quantity Request screen ([NXT-9509|https://primeroedgenext.atlassian.net/browse/NXT-9509])

## Acceptance Criteria
Provide new screen for site users to make transfer requests
Allow user to search for items 
Allow user to select items to add to the request list
Allow user to enter request quantities (if they do not have Other Sites QOH permission)
Add Next button which will navigate the user to either the Site Request page or the Confirmation screen

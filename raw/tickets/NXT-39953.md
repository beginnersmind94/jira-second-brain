# NXT-39953 — Acc - Claims - Updates to use System Settings, Edit Check by Enrollment Site

## Description
As Dora the Director I want to submit my reimbursement claim via enrollment site so that I am in accordance with state guidelines and won’t get in trouble.

There’s a system setting we need to make use of during claim generation that we aren’t currently:

|Accountability|Reimbursements|ALLSCHMEAL|Claim - Meals Claimed at Enrollment Site (No = Claimed at Serving Site)|Meal claim location(Site Serving 0, Enrollment 1)|



*requirements*

* Make use of the ALLSCHMEAL system setting during claim generation.
** If it’s turned off - no change to how we’re doing things.
** If it’s turned on - meals need to be counted at the enrollment sites instead of the serving sites.
* Edit Check report for showing meals at enrollment site needs to be checked and fixed up
** This is not related to the system setting but we’ll fix it as part of this work.



I ran this on production at 3:37 pm 5/23 and all meal counts show as 0 for a date where meals exist, if the show meals at enrollment site option is used.

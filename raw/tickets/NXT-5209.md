# NXT-5209 — Eligibility - Eligibility Descriptions

## Description
*User Story*

As a district admin I want to configure the names of our assistance and categorical programs and see that name used across the module.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5359%3A613&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5359%3A613&scaling=scale-down-width]

!image-20201019-181959.png|width=1304,height=603!



*Requirements*

* Add a page called Eligibility Descriptions.
* Two grids on the page -
** Assistance Programs
*** Tracks the following for USDA Description: SNAP, TANF, FDPIR
** Other Source Categorically Eligible Programs
*** Tracks the following for USDA Description: Foster, Head Start, Homeless, Migrant, Runaway
** Columns: 
*** USDA Description
**** Varchar, 15 character limit.
*** Local Description
**** Varchar, 30 character limit.
*** Visible
**** Flag / bit / checkbox.
*** Last Updated
**** Date Time + User
**** 8/2/2020 2:34:31 PM (Matt Smithers)
* Edit option on the page
** User can change the following:
*** Local Description
*** Visible
** Save / Cancel options.

## Acceptance Criteria
Eligibility Descriptions page added, data shown.
Edit mode for the grids - values update appropriately, and the Last Updated also tracks properly.

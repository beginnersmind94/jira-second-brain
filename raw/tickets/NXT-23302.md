# NXT-23302 — Account Management - Accounts - Integrate 'Columns & Views'

## Description
As a product owner, I would like to provide users with the ability to save their current filters to a View, so that they can easily select and reference this view later on without having to manually select individual filter options each time.

*Summary*

In the Accounts Management *Accounts* page, add buttons to be able to save a view. Then, integrate the new columns and views button to allow them to select that view whenever reloading the page (in addition to having the default view).

*Details*

# Add Views, and Save As buttons (reference resource #1 below)
## Download button shouldn’t be needed since we have the Excel button already.
# Integrate these buttons to allow the user to save their current view (whatever is selected in the columns shown in the grid) with a custom name.
# Allow the user to select that or any other view whenever reloading the page, with the selection then populating the grid with those columns. 
# Allow the user to set a view as their default, so that this is the particular view that loads every time the page is reloaded.
# View titles should be unique and throw an error (like “View Name already exists”)

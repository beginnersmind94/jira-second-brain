# NXT-69503 — SCTV+ - Compact Card

## Description
As an admin publishing menus without images, when I use the compact card, I want to see a clean text-only list of menu item names so that the display remains readable and consistent without requiring galleries or metadata.

## Acceptance Criteria
* Compact Card shows only: Category Header + stacked Menu Item Names with Calories; no images and no other secondary details (components, ratings)
* Hide style controls for hidden elements so users don't expect to edit them
* Layout: Header centered and all-caps; Item Names left-aligned with shared left edge; Calories displayed [inline after name? / right-aligned? / below name?]
* Item Names wrap by word up to 2 lines max (40 character limit), then truncate with "…" — no mid-word breaks
** Clarified on {{2026-02-09}}:   Item names fill each line left-to-right up to 20 characters without breaking words; if the next full word would exceed the 20-character limit, move the entire word to the next line; allow up to 2 lines total, and if text remains after the second line, remove it and add “…”; if a single word itself is longer than 20 characters, split that word at 20 characters.
* Spacing: Equal vertical gaps between rows, consistent padding from edges, shared left alignment across all items
* Overflow: If items exceed available space, remaining items continue to next slide with same header and styling; no partial/cut-off rows

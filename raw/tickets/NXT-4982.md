# NXT-4982 — AcctMgt - Add Patrons - Students - Wizard

## Description
*User Story*

As a district admin I want to have a nice step-by-step wizard to lead me through adding a patron.

*Mock-up*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view_-_stepper&id=tj8d55&g=1]

*Requirements*

* We’re updating the Add > Student option into a multi-step wizard.
** This will be a 6-step wizard per the wireframes.
* User has Cancel, Back and Next buttons on each step to provide navigation.
** Information must be retained as the user paths forward and back.
** Step 1’s Next button should likely not be available the first time they load in. It should be available if the user navigates back to Step 1 however.
* User has a progress display on the top-right of the drawer that lets them know what step they’re on, and what steps are present in the wizard.
** User can navigate to prior steps by clicking on the progress display.
*** Check how the Item Management team is handling this as far as how the progress steps change. Looks like an edit pencil comes into play.
* Display a ‘Step X of 6’ to also let the user know how far along they are.
* When the user first opens the wizard, they need to be shown the information pop-up regarding the stepper itself. Screenshot below.

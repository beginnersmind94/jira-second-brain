# NXT-49526 — MVR - Dashboard - Recent Reports

## Description
As a SchoolCafé user, I would like to be able to view the list of of my recently run managed view reports so that I can view a report that was run.

*Notes:*

* Copy the Recently Run Reports from Custom Reports as much as possible and the requirements included here assume you will do that

*Requirements:*

* On the Edit button of the grid in Reports > Managed View Reports, do the following:
* Action column updates:
** Recently Run Reports 
*** If button not there yet, add it
*** On click, open the View Recently Run Managed View Reports drawer



*View Recently Run Managed View Reports drawer*

* Create a new drawer
** Name = Recently Run Managed View Reports
** Size = same as CR
** Use the sliding action when opening
** Copy from Custom Reports
* 
* Show a list of MVR recently run reports the same way as CR 
** Be able to open a previous report like in CR
** all the same rules and logic as in CR
* 
* Security:
** User must have view rights to Manage Views Report dashboard to use this drawer
** In R&P, don’t show this drawer

## Acceptance Criteria
* Users are able to view the recently run MVRs
* Only users with view rights can views the settings

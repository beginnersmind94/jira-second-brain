# NXT-26085 — UX Mock @ Code Look up- Allergen Imports 

## Description
This story is to cover the task related to UX mock up @ Code Look up.

During the mapping process when Student ID and Allergens (source fields) are mapped, option for code look up @ Allergens must be provided.

In the code look up, there will be options to Map the source fields to Target fields, which will be auto populated. Option to change the values in both source and Target must be provided (drop downs).

The columns will be Source Fields, Target Fields, Mapped (check box). This is similar to that of Code Look up @ Student Import.

For all the fields @ Source Fields, if any matching fields are not present @ target Fields, source fields must be in Red Color to reflect that there are no matching values on the target side.

Similarly, provision to add new rows must be provided.

At the bottom of the page above save option, below warning must be populated incase if there are no matching values auto populated.

"Missing Allergens can be added via Item Management Page".

## Acceptance Criteria
Verify and Validate that the Source and Target fields in the Code Mapping can be selected using drop downs style.

Verify and Validate that the Source and Target fields in the Code Mapping can added using the add option

Verify and Validate that the Source fields are in Red if no auto value is populated (Missing Target Values).

Verify and Validate that the disclaimer is present in Red incase of missing values @ Source Fields.

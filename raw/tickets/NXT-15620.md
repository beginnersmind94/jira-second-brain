# NXT-15620 — INV - Order Template List - Template Actions

## Description
As an Inventory Operations Manager I want an action column on the Order Templates list so I can open, rename, or deactivate existing order templates.

This is a continuation of [https://primeroedgenext.atlassian.net/browse/NXT-15619|https://primeroedgenext.atlassian.net/browse/NXT-15619|smart-link]

*Requirements*

* Within the Order Templates list grid, provide an Action column
* In the Action column, provide the following:
** View icon to open Template Details ([NXT-14810|https://primeroedgenext.atlassian.net/browse/NXT-14810])
** Edit icon (pencil) - to edit the Template Name in-line
*** provide save/cancel icons: green checkmark (save) and red X (cancel)
*** must enforce uniqueness validation on save
**** “Template Name must be unique.”
**** Unique across entire district, case insensitive 
*** this will also change Active column into checkbox selection
** Delete icon - to remove template
*** cannot delete a Template if it is has items assigned to it
**** disable Delete icon if the template is in use 
**** provide hover tooltip for disabled Delete icons: “Templates cannot be deleted if they are in use.”
*** Prompt user on delete attempt: “Are you sure you want to delete this template?”
**** No/Yes

## Acceptance Criteria
* Assign icons to the Action column
* Edit icon will enable the Template Name field for edits
** Edit icon will be replaced by the green checkmark icon (save) and red X icon (cancel) when in Edit state
* View icon will navigate to the Order Template Details page for the selected template
* Delete icon will prompt the user to delete a template
** Delete icon will be disabled if it has items assigned to it
*** Tooltip will be provided on disabled Delete icons

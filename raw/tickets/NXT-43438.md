# NXT-43438 — Broadcast - Message All Districts

## Description
As a Customer Support person, I want to broadcast a message to all district users at once so that I can inform them of system wide changes

*Notes*

* We already have broadcast ability, but it is targeted at users of a district
* In this story, we want to leverage that broadcast ability, but make it applicable to all users of all districts

*Requirements*

* On the Workspace > Broadcast > Add New Broadcast popup, do the following:
* When the user clicks the To button:
* -Add a tab next to Users:-
** -title: Districts-
* 
* -On the Districts tab, do the following:-
** -Add a multi-select list box-
** -Values: list of active districts-
** -Ensure has a Select All button-
** 
** -When click Save:-
*** -Add the districts / users to the recipients list-
*** -Not sure how you will handle the To list on the New Message screen; maybe you can just put in a value like “Multiple Districts” or something like that-
* Add a checkbox on the popup
** Title = Global Message
** If click yes, then show the banner message to everyone who logs in
** Don’t show based upon users, but show it to everyone for the duration
** 
* -User can use / edit New message screen the same as if sending to just an individual user-
* 
* Users can dismiss the message when they log on

## Acceptance Criteria
* Broadcast messages are viewable by all users for the duration specified

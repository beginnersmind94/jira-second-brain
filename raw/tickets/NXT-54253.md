# NXT-54253 — LMA - License Keys - Vending

## Description
As a Customer Support person, I would like to be able to apply a vending license to my sites so that we can sale this technology to customers

*Notes:*

* This story is to add vending as a license to School Cafe
* It is different in that it is not a module, but it should be able to fit into our current model

*Requirements:*

* Do the necessary in the backend to ensure Vending is a valid license
* Ensure it shows up in the list of licenses under license management on the summary and configuration tabs
* Ensure it can apply to both district and sites the same way as the other modules
* Ensure any changes are captured in the History log

## Acceptance Criteria
* Users are able to enable and disable vending licenses
* Users are able to apply to sites and districts
* Users are able to view the history

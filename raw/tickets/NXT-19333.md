# NXT-19333 — Production- Reports- Production Record (Production Plan Page Link)

## Description
This report will give the districts a way to print their production record from the day of service that they are currently on. This option should only display for the user once the production status is in *Record Complete*.

Report should display very similarly in format to the production record page with the following criteria
Display on report
Site code with Site Name
Serving Date
Meal Service
Menu Line
Preparation Site
Offer VS Serve (True or False)
Meal Pattern
A text box showing:
Serving Group
Planned Count 
Actual Count

There also needs to be a display of any plan or record comments that users have added on UI side.
Columns to display information are:
Locally Grown
Menu item (Recipe #) and a small size font underneath for serving size with (portioning tool)
Serving Group
Adult
Total Planned
Servings Produced
Total Served
Adult/ A la Carte
Reimbursable
Leftover
Comments

Items also need to display in same order of category as on the record.

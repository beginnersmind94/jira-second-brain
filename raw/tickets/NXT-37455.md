# NXT-37455 — Student Import – Show Exception Count in Import Monitor - Dev

## Description
As a SchoolCafé Admin I want to quickly see if an import had exceptions and how many so that I can know if I need to review the exception log.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* View an import row and see the count of exceptions it had

*Notes:*

* We already did the stories to create import exceptions, now we need to show them on Import Monitor
* Note that in History tab, we also need to show the count of exceptions, so you might be able to reuse that code

*Requirements*:

* On the Import Monitor grid, do the following:
* 
* Filters:
* Add a filter for Exceptions
** Values: All/Yes/No
** Default: All
** If select Yes, show only rows that have an exception count > 0
** If select No, show only rows that have an exception count = 0
* 
* Grid:
* Add a column called Exceptions
** Show the count of exceptions for that import

## Acceptance Criteria
* Filter exists and affects the grid per requirements
* Exceptions column added and each row has the exception count correct

# NXT-3675 — Inspections Dashboard - Inspections Status Widget 

## Description
*User Story*
As an Inspections user, I have a dashboard widget that allows me to see at a glance the number of Inspections in each status.

*Mockup*
https://nqyc9v.axshare.com/start.html#p=inspection_dashboard&id=0rb254&g=1

*Requirements*
* There is a dashboard widget that displays information on which Inspections are within each status
** A donut chart is present with all statuses except for the total
** All statuses are displayed numerically along the bottom
* The following statuses are displayed
** Scheduled
** In Progress
** Pending Approval
** Re-Inspection
** Approved
** Total
*** Note: Not displayed on chart
* Clicking on any status takes the user to the View Inspections screen, with the appropriate filter automatically set
** E.g. Clicking on Scheduled will show the user all relevant scheduled Inspections
** Any filtering applied to the dashboard (e.g. dates, Show Only Mine) is carried over
* Inspectors only see their own Inspection stats
* Supervisors by default see all Inspections they have access to
** Clicking the "Show Only Mine" toggle will switch to viewing only their own Inspections
* On the chart, each is represented by a different color
** Text color below matches the appropriate color
*** E.g. If Scheduled is blue in the chart, the text for Scheduled below is blue as well
** Clicking the section on the chart links to View Inspections in the same way clicking on the number below does

## Acceptance Criteria
* Status Widget present and showing proper data
* Linking and filtering functioning properly

# NXT-48380 — MP- Menu Cycle- Updates to menus in cycle and save changes

## Description
As a menu planner, when I change any menus that are associated with the menu cycle I am on, I would like for the menu cycle to ask me to save my changes and also update the updated date and who on the list page so that the latest menu cycle updated is showing on the top of the list page.

## Acceptance Criteria
* When any menu is added or changed on the menu cycle, allow for the close button to be save button to allow for user to confirm the changes
* Once user saves, update the updated date and updated by so it updates the list page with the latest menu cycle updated 
* This should also reflect on the assign menu page with the latest menu cycle showing at the top of the list
* Display unsaved changes pop up if user tries to the leave the menu cycle page without saving
* When user changing the menu cycle name on top save button is enabled, but if user is reverting back to original text still it showing save button enabled.  We need to revert from save to close button enable.
* When user changing the values in week of menu dropdown list we need to enable save button, but if user reverting the changes again revert from save to close button.
* If time permits make a floating banner for Cancel, Copy, Close/Save button.

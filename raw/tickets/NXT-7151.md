# NXT-7151 — Data Exchange - File Upload - Allow user to indicate delimiter, header and qualifier

## Description
As on the initial step of Classic, let’s allow the user to select these 3 options either before or after uploading their file. 

* There can be a separate story to validate these selections on the File Mapping step. 
* We can reuse the static options in the drop-downs that we had in Classic.
* These can be ignored for the Item Import type.

## Acceptance Criteria
Verify that when a user is on the File Upload step, we allow them to indicate their delimiter, header, and qualifier

Verify drop-down options are static

Verify controls are all hidden for Item Import type

Verify that these values save to the database as expected

There can be a separate story to validate these selections on the File Mapping step.

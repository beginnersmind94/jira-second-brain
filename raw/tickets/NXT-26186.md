# NXT-26186 — IM- Item and Recipe Copy (Image box)

## Description
Currently wen trying to copy an Item or Recipe the check box that enable images and tags to be copied is disabled. If the Item/recipe has an image(s)  I need the option to select if I want to copy images enabled.

!image-20220201-225618.png|width=1232,height=676!

!image-20220201-225302.png|width=978,height=542!

## Acceptance Criteria
*Entry Point*

* Item Management - > Items → Selected Item → Copy (Copy Icon)
* Item Management - > Recipes → Selected Recipes → Copy (Copy Icon)

*Actions*

* When the user clicks or navigates with keyboard to copy icon the user shall prompted with the copy pop-up window (see image above).
*  The Images and Attachments should be enable if the item/Recipe have any image or Attachments
* If the user select to copy the image(s) or attachments by checking the corresponding checkbox then the images or attachments shall be copied. 
* User shall be able to uncheck the box if desired, prior to confirming the copy & save action
* Enable Tags on recipe as well

# NXT-15511 — Tag APIs - Retrieve entities containing a tag

## Description
As a module developer, I would like to have an API available so that I can retrieve a list of module entities that have a given tag or list of tags assigned to them.

## Acceptance Criteria
* An API shall exist that will provide a list of entities that have a given tag or list of tags assigned to them.
** All entities regardless of module and entity type will be returned.  Each module consumer will be responsible for filtering out the entities for the relevant module or entity type that they need.
* The API shall return its list within 1 second.

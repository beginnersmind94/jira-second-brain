# NXT-5259 — Direct Certification - File Search - Show info for when preapproval data is matched to more than 1 account

## Description
*User Story*

As a district admin I want to be informed if DC File Data has been matched to another student already so I can investigate.

*Mockup*

tbd

*requirements*

On the File Search grid results:

* For data that has already been matched, show the user who was already matched.
** Data to show: ID #, Student Name, Match Date.
* If the user attempts to match already-matched data against another student, leading to the 1:many relationship, ask the user for confirmation on that.
** The system or another user has already matched a student against this data. Are you sure you want to proceed?

## Acceptance Criteria
User can see which student(s) have been matched against already-matched file data in File Search.

User is asked to confirm matching already-matched data against another student.

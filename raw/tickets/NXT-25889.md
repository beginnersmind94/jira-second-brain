# NXT-25889 — PR- Dynamic Menus API 

## Description
This story is primarily to create an API for the Front Office modules.
There needs to be an API created that will include
	Production Data such as
	Menu Line
	Menu Item Categories
	Planned Menu Items
	Menu Item information
	Contributions
	Planned Counts
	Carryover counts

## Acceptance Criteria
API must include Production Data such as

* Serving Date
* Site name
* Menu Line
* Menu Item Categories
* Planned Menu Items
* Menu Item information (Nutrients, item/recipe information)
* Contributions
* Planned Counts (Production Plans need to be complete to pull data)
* Carryover counts

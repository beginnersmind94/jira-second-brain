# NXT-69490 — WS - Insights - KPI - Inventory Value - Details Drawer

## Description
As a School Nutrition Manager I want to view the current inventory value broken down by food and non-food categories So that I can understand how much of my budget is tied up in stock, identify imbalances, and make informed purchasing decisions.

# Dashboard Access
#* Insights Dashboard shows  Inventory Value card 
#* When clicked, a slide-out panel opens with detailed value breakdown by storage category for food and non food item types
# Inventory Value Overview
#* Displays Inventory total value in dollars.
#* Displays food inventory value in dollars and % of total inventory value.
#* Displays non-food inventory value in dollars and % of total inventory value.
#* Displays Commodity Inventory value in dollars and % of total inventory value.
#* Displays Non-Commodity Inventory value in dollars and % of total inventory value.
# Food Inventory Breakdown
#* Shows the inventory value breakdown for each food storage category for selected school(s)
#*  shows Inventory Value ($) and % of Inventory for each food storage category
#* A Total Food Inventory row is shown.
# Non-Food Inventory Breakdown
#* Shows the inventory value breakdown for each non food storage category for selected school(s)
#*  shows Inventory Value ($) and % of Inventory for each non food storage category
#* A Total Non-Food Inventory row is shown.

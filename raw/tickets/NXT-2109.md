# NXT-2109 — API and QA-Production - Production Record - Add Menu Item

## Description
As a site manager, I need to be able to make changes to my menu sometimes. This could be because my site is having an event such as grandparents day, or because of product that is not getting delivered from the vendor, but they’ve offered a substitute item, which I now need to enter on the menu.

From within my production record page, I need to be able to select and assign a menu item for my site.

## Acceptance Criteria
There will be a method of “Add Menu Item” - which allows me to search for a menu item that is not on my current production plan. There should also be a way to 

1.) enter the quantity of servings that I need for that menu item

2.) Select the portion size 

3.) Save these details to the Production Record page.

# NXT-50288 — PR- Plan Summary- Prep site and receiving site items on report

## Description
As a prep site, when I am pulling the Plan Summary for all the items I need to prepare for all other sites, they are currently not showing.

## Acceptance Criteria
* All prepared menu items should show on plan summary on prep site report
** Display total for all menu item counts on number of servings
** Display icon next to menu item Code (PI) and tool tip for “Prepared Item” 
*** On report, show in text [Prepared Item] on top of the menu item code
** Display all additional non prepared items for prep site
** Materials needed should calculate based on total servings for all prep site and receiving sites
* On receiving site, *only* display non prepared items 
* THIS ONLY APPLIES IF PREP SITE AND RECEIVING SITE ARE DIFFERENT

# NXT-47445 — IM - Recipes - Version History

## Description
As a Menu Planner I want to see the history of changes to my recipes including specific versions, so I can see what the recipes looked like at the time of important menu assignments or production records.

!915060ee-7a3f-4105-a779-08221550ce51#media-blob-url=true&id=8c5b4d91-d0ab-4ff8-af13-a76c896ab588&contextId=64251&collection=|width=91.66666666666666%!

## Acceptance Criteria
* Provide a History view on the Recipe Details which consolidates all history sections into a single view
* Display important ‘Versions’ in the history which correspond to important dates when the recipe was assigned in a menu or added to a production record 
* Allow the versions to be selected so the historical recipe can be viewed (like Order history)

# NXT-20750 — Receive, Load & Store Edge Scan Results

## Description
As a Cashier, I want to receive the results from the Edge device scan so that the terminal can associate the food detection with the student and the POS transaction.

*Application*: POS

*Flow:*

* The following stories occur prior to these requirements:
** The POS transaction screen with Food Detection is enabled
** Food Detection event is triggered
** Edge scans food and returns results
* Then this story takes over:
** POS receives and process the results
** POS saves the results to the DB
** POS optionally loads the results into the transactions

*Assumptions:*

* Completed stories to enable Food Detection Transactions and is turned on
* Story for Food Detection Trigger completed
* Terminal connected to an Edge device

* This page exists in the classic ExpressPoint
* You can copy all the code and try to copy as-is
* Talk to Saketh if you have any questions

*Requirements*:

* Copy everything as-is from classic

** Detection Results

*** Show a red icon when doesn’t
*** Default is red

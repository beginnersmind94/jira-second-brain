# NXT-60905 — INV - Orders - Order Details - Receipts tab - Pre-populate Receipts from Distribution

## Description
As a site ordering manager I want my Receipts tab to pre-populate according to the Distribution shipments, so I can save time during receipt creation and I can more easily see item and quantity adjustments from the distribution site.

h3. Order Details - Pre-populated receipt

* 1 Receipt already created (from earlier shipment)
* 2nd receipt pre-created from Distribution shipping
* Create New Receipt button is removed

!Receipts Tab Redesign - Pre Populate-20250506-140448.png|width=91.66666666666667%,alt="Receipts Tab Redesign - Pre Populate-20250506-140448.png"!

h3. Internal Receipt

* Invoice # and Invoice Amount prepopulated
* Add Item removed
* Delivered By is populated with Driver Name (if provided)
* Icons for Substitutions (stretch goal)
* Shipped Quantity replaces Ordered Quantity
* Icons for increased/decrease quantities from order

!Create Internal Receipt-20250501-231525.png|width=91.66666666666666%,alt="Create Internal Receipt-20250501-231525.png"!

## Acceptance Criteria
* Create a placeholder receipt when the Distribution site ships an order
* Prepopulate the Invoice # with the Distribution ID + Order ID
* Prepopulate the Invoice Amount according to the shipped value of the products
* Prepopulate the ‘Delivered By’ field with the Driver Name from Distribution
* Replace Ordered Quantity with Shipped Quantity
* Update the receipt to reflect any changes made by the Distribution site
** quantity changes, added items, substitutions 
* Provide icons for items which were substituted, or items which had some quantity adjustment
** stretch goal

# NXT-74 — Reconciliation Bulk Adjustment

## Description
As a Site Manager I need a way to change the menu item sold inside of a session for multiple patrons at once.

## Acceptance Criteria
User can change the menu item sold for multiple patrons from 1 menu item to another.
User can select the criteria for selecting patrons based on Price Type - all Paid, all Free, all Reduced for example, combinations of price types.
Transaction totals are recalculated appropriately on the menu item change, based on eligibility of patrons for the transaction date.

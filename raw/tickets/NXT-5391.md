## Acceptance Criteria
Depending on the claim type the data for each site is shown in the required format for the claim data created for SNP.
There is a column for each type of program participation. 
There are 7 possible columns but only 5 are possible on for a claim
The 3 possible columns for NSLP, Less than 60%, 60% or More, Max Rate. Only one is valid for a claim.
The 2 possible columns for SBP are Severe Need, and Non-severe Need. both are possible on a claim, but only 1 is valid for a site.
The 2 possible columns for ASP are Area Eligible, and Non-area Eligible. both are possible on a claim, but only 1 is valid for a site.
If a site does not participate in a program the column entries are left blank.
Each site has
Meal counts for each Program the site participates in for each meal count.
Each meal count is multiplied times the appropriate rate resulting in a claim amount.
There are subtotals per program participation by site.
There are totals per eligibility type by site

There are grand totals per column.

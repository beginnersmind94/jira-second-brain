# NXT-3201 — Reconciliation - Reconcile Info - Summary

## Description
*User Story*

As a site manager I want to see at a glance if a session is balanced so that I know if any actions need to be taken.

*Mock-up*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1137%3A22760&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1137%3A22760&scaling=scale-down-width]

*Requirements*

* Add a Summary area to the Reconciliation Info / Detailed page.
* Show information about the session balance for cash, checks and cards:
** Display how much of each is in the closing balance. (Cashier Column)
** Display how much the transactions indicate should be tied to the session for each. (System Column)
** Display the over/under - Cashier - System = Over/Under.
** Display amount entered as Session Adjustment [https://primeroedgenext.atlassian.net/browse/NXT-3207|https://primeroedgenext.atlassian.net/browse/NXT-3207|smart-link]

## Acceptance Criteria
Cashier Cash = Closing Cash Total - Opening Cash Total
Cashier Check = Closing Checks Total
Cashier Card = Closing Card Total
System Cash = Transaction Cash Payment Total
System Check = Transaction Check Payment Total
System Card = Transaction Card Payment Total
Reconciliation Info shows totals for cash, cards and checks in the Closing Balance vs the Transactions.
Show an Over/Under what is an equation between the two. Cashier - System = Over/Under.

IF the session is not balanced display this text under the Graph.
THE SESSION IS NOT BALANCED
Check the opening or closing balance detail for incorrect denominations
Check the session transactions for incorrect transactions
Select "Adjust Session ..."

IF the session is not Closed display this text
THE SESSION IS NOT CLOSED
Wait for the cashier to close the session
-- or --
Correct all incorrect transactions
Close the session by selecting the icon next to the closing balance

IF the session is Closed and Balanced display this text
THE SESSION IS BALANCED AND READY TO RECONCILE

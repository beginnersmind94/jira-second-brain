# NXT-3448 — Configuration - Add Topic, Subtopic

## Description
With permission, ability to add Topics, subtopics, methods, sources, job titles

Ability for admin (with permission) to configure Topics, Subtopics, Training sources, Training methods 

* Adding new Topic, Subtopic populates in the in the field for next event entry
* User can add a topic without a subtopic
* Subtopic should align with the selected topic it comes under
* If topic doesnt have a number, then subtopic description should be validated for duplicate entry within that topic.
* If topic has a code, Duplicate check for subtopic sequence
* topics and subtopics dont have to have a code (USDA provided topics/subtopics will have codes 1000-4100 )
[https://nqyc9v.axshare.com/start.html#p=training_configuration&id=suie2f&g=4|https://nqyc9v.axshare.com/start.html#p=training_configuration&id=suie2f&g=4]

## Acceptance Criteria
Admin (with permission) can add new topic, subtopic
checks for duplicate codes at subtopic level if topic has code. 
checks for duplicate subtopic description if topic doesnt have code.

# NXT-47450 — PR- Production Plan- Autofill- Based on Menu Planning Counts

## Description
As a production manager, there are plenty of times that I repeat the same menus over and over throughout the course of the school year. I would like to quickly fill in my production plan counts based on menu planning counts. The options could be to copy the menu item counts straight from the menu along with the meal counts or to scale the counts based on the meal counts entered on the production side.

## Acceptance Criteria
* Provide a drop down option that allows user to select *Menu Planning Counts*
* Display Menu Name and Meal Counts for each serving groups on the menu 
* 2 options should display for grabbing data and autofilling counts
** Copy Meal and Menu Item Counts from Menu
*** This will be a straight copy from the menu and all the counts that exist on the menu and the menu items
*** Insert all counts and IF there are any counts that are blank, leave those blank on the production plan (DO NOT ENTER 0 for a blank count)
** Adjust Menu Item Counts to Meal Counts for this Site
*** This will adjust the meal counts based on the meal counts entered by the user on the production plan or the grid given on the slideout
**** Prefill the meal counts given on the production plan if user has already entered them, user can still change on the slideout grid and it will update the production plan
**** IF not, user must enter in meal counts in the meal count grid on the slideout 
***** Provide Error Message on slideout “Meal Counts must be entered"
* Display an option for user to override existing planned counts
** Default to be selected
** If not selected, only insert counts for menu items that have not had a count entered for
!image-20240227-155150.png|width=1222,height=550!
* Disregard any VMI counts as they do not apply to the child items 
* Apply Auxiliary serving groups to display if Setting is on, this should not effect the menu item counts as we do not track menu item counts based on auxiliary groups
* Not required to show the ‘Serving Group & Planned Count’ Grid for this option as we are showing the data in Menu Name and Meal Type fields.

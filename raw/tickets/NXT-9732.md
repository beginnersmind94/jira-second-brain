# NXT-9732 — INV - Orders - Order Detail - Receipts tab - Summary

## Description
As an inventory user I want a Receipts view on the Order Detail page so I can view my list of items in terms of how many have been received.

This would be the 2nd tab of the Order Detail page. This is where Receipts would be displayed

In this view, the Item List would have additional or different columns to show the Receipt Quantity.

+Story Scope:+

* Create the Receipt tab, show the Summary section, Action buttons 
** ‘Close Order’ and ‘Keep Order Open’ will not be functional for this story. The buttons are placeholders.
** Charges & Discounts ([NXT-16824|https://primeroedgenext.atlassian.net/browse/NXT-16824])

*Requirements*

* add a new tab to the Order Detail screen titled Receipts
* the Receipts tab will contain a tab-based section to display a summary of the order’s receives and details of the individual receipts
** there will be a ‘Summary’ tab/view for all orders
*** there will be an additional tab for each specific receipt (NXT-????)
*** we will not use the tab control. instead we will use buttons/links that appear similar to tabs
** for all tabs, there will be a banner with details of the receipt status
** for all tabs, the list of columns will be the same:
*** Item #
*** Vendor Item #
*** Item Name
*** Quantity Ordered
*** Quantity Received
**** total quantity received 
*** Quantity Due
**** difference between Quantity Ordered and Quantity Received
* Add a “tab” button for each individual receipt ([NXT-16823|https://primeroedgenext.atlassian.net/browse/NXT-16823]

+*Actions:*+

* Create New Receipt button ([NXT-7256|https://primeroedgenext.atlassian.net/browse/NXT-7256])
* Close Order button
** This changes the order status to ‘Closed’ and means the order is no longer editable or eligible for additional receipts
* Keep Order Open button (maybe)
** this is used in conjunction with settings that determined whether an order is automatically closed when a receipt is created

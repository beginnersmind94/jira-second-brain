# NXT-4211 — System - Management - Letter Templates - Add SFAs data (Officials tab)

## Description
As a district administrator, I would like to be able to manage my district's list of officials, so that fields on my notification templates will always automatically have the right information.

*Summary*:

* Within the Letter Templates page, add the option to update the address and list of officials within a district.
* This data is from the SFAs (School Food Authority) page of classic, but relocated to where the data is being used
** Other tabs from that page (Sites, Users, Contract Details, Grace Period) can be ignored for now
* All fields for the section are _optional_ except first and last name which should be required.
* All fields should be less than 50 characters each. No need to validate zip, phone number, or email address. Data can be saved as-is.
* User should be able to save information to the database
* Goal: Data will correspond to fields within Notification Templates (testing not really necessary since we won’t have the means to send out notifications until Account Management is further developed anyway)

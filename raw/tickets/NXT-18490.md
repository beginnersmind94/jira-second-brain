# NXT-18490 — Production- Reports- Plan Summary (UI)

## Description
This report will give the user a way to see all their plans item servings and what materials/ food is needed to make these items based on the servings. 

Criteria needed for report page:
Organizational Level/ Site
*From* and *To* date
Dates default to show a week in advance from the after the report is being generated. 
For Example: Trying to generate report on 8/27/2021, the default dates when arriving on the page will show *From Date: 8/28/202*1 and *To date: 9/4/2021*

Report needs to also be able to generate an excel with the information as well as a regular report that can export to excel, Word, PDF and CSV like all other reports. 

Criteria in Columns needed for Report :
Recipe Code
Menu Item
Serving Size
Meal Contribution
Servings (This will be total servings needed of an item over the date range)
Materials needed (Item name with Stock/Item number

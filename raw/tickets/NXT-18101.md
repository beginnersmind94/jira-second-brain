# NXT-18101 — Production - Reports - Production Print Recipes (Single)

## Description
This report will have to different ways of pulling information. 

There will be a single and multiple option.

The *single* option will give the user an option to enter the recipe code or the recipe name and generate a printable recipe. The user does not have to enter in either of the information and can click apply for all items to appear in a drop down.

Current Primeroedge View: 

!image-2021-08-23-13-44-36-647.png|width=200,height=183!

 

Once a user clicks apply, the recipe will appear and give the user an option to scale recipe by batch size if applicable or by a custom scale recipe as well. The page will display what a batch serving size is and give the user the option to select how many batches they would like to scale if available. 

Current Primeroedge View once click apply:

!image-2021-08-23-14-02-57-848.png|width=200,height=183!

 

User will have the option to select between just a scaled recipe for either batch or custom scale recipe or they can generate a scale production recipe report for either option as well.

## Acceptance Criteria
The *single* option will give the user an option to enter the recipe code or the recipe name and generate a printable recipe.

User does not have to enter a menu item or a recipe code, clicking apply without either will show all menu items available in the entire software defaulting to the first numeric name 1-9 then first alpha A-Z

EX for List:

* 1% Milk
* 2% Milk 
* Apple
* Banana
* Carrot


Once a user clicks apply, the recipe will appear and give the user an option to scale recipe (Report Available in IM) by batch size if applicable or by a custom scale recipe as well. The page will display what a 1 batch serving size is all the way to 10 batches.

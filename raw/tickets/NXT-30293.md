# NXT-30293 — IM - Menu Items - Deconstructed MI Flag

## Description
As a District Admin I would like the ability to define an item as a ‘Deconstructed Menu Item’ so that it can be planned/recorded for each sub ingredient rather than at the Menu Item level.

*Requirements*

* Add a new flag on the Menu Item for multi-item recipes titled ‘Deconstructed Menu Item’
** This flag will be considered within Production to determine whether a Menu Item is planned at the MI level, or if it can be planned/recorded for each individual sub ingredient.

## Acceptance Criteria
* Users can specify a Menu Item as a ‘Deconstructed Menu Item’
* This flag will only be available for Menu Items linked to a multi-item recipe

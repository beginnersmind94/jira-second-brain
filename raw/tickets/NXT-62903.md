# NXT-62903 — FamPortal - Admin - Reports - Add 'Auto-Pay Report'

## Description
As a district admin, I would like to be able to view a report of all grade 12 students who are on accounts where autopay is enabled, so that I do not accidentally refund them and have autopay immediately replenish the funds.


Several customers have requested a report that they can run at the end of the school year which gives them a list of users with students (typically seniors) that have auto-pay enabled, so that they can either disable auto-pay first or tell the user to do so. This is very helpful for them because they often make refunds which immediately trigger the auto-pay and thus require another refund.



*Summary*
Add a new district admin report (accessible to Super Admins as well) that gives users the list of parents and staff with connected students who have auto-pay enabled. 



*Details*

* In Admin → Reports, add a new report: *Auto-Pay Enabled*

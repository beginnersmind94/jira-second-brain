# NXT-45813 — GDSNConnect - UAT Observation - Review Tab - Documents count should be shown as 1 when there is an update

## Description
Currently user cannot see the image count on pending review screen on GDSN Connect source

## Acceptance Criteria
Steps:

User upload the data with more than 1 images at an ingredient level

In pending updates page, the system should show the count of the images (before the user accepts the image). The count should be dynamic based on number of images comes at source

Example:

Documents (2) - this represents there are 2 images

Scenarios:

In case of partial updates - reduce the count by showing the number of images that are not taken yet

If all the images are taken show the image count as zero on documents

Regression- the images updated shows in documents tab of the ingredient (as-is)

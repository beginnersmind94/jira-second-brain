# NXT-43544 — MP- Menu Planner- Apply serving size exceptions

## Description
As a menu planner, when I designate a serving size exception in IM, I am purposely specifying that a particular serving group gets a specific serving size or a serving group does not get served a particular serving size.

## Acceptance Criteria
* Apply serving size exceptions based on configuration in IM 

!image-20231003-071710.png|width=1607,height=733!



* Toggle menu items that are marked as Not Served for a particular serving group
* If serving group is the only serving group available on the menu and the serving exception is “Not served”, insert menu item with *default serving size*
* User still has the ability to change the serving sizes if they would like it is only just defaulted to whatever exceptions are present

# NXT-31389 — IM - Items - Item Details - Recipe/Menu Item tab - View & Edit

## Description
As a District Admin I want a Menu Item card so I can specify the required Menu Item attributes at the same time I am creating the corresponding Item.

*Requirements*

* Display the Single Item Recipe card if item is flagged as a Single Item Recipe
* Display the Menu Item Info card if item is flagged as Menu Item
* Provide the toggles until the tab is successfully saved
* Stretch the width of the card if only one is enabled

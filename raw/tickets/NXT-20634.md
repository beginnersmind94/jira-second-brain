# NXT-20634 — Platform - BO - Onboarding Modules Details Page

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view system modules detailed page with its overall progress and take necessary actions

*Mockup:*

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|smart-link]

*Requirements*:

* View Details button should bring *License Management* screen *“Summary”* page with option to close and return to Onboarding dashboard
* No need to % calculation for this widget/panel
* Clicking on VIEW should open that module configs for “License Management” screen
* Each of this row should bring the module view from license management

## Acceptance Criteria
* User shall be able to see the Modules widget in Onboarding dashboard
* User shall be able to see VIEW DETAILS button to open Modules/Products licenses summary page
* User shall be able to VIEW each module to review the module licensing details per district/site
* User shall be able to close the drawer by clicking on CLOSE button and return to onboarding dashboard
* System shall save changes to the licensing details upon user interactions

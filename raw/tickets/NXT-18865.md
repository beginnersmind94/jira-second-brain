# NXT-18865 — System - Sites - POS Settings

## Description
As Vera the Director I want to change POS Settings for multiple sites at one time to save time and propagate changes to all sites faster.

*Mockup:* 

[https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=556%3A7669&scaling=scale-down-width&page-id=555%3A6980&starting-point-node-id=556%3A7669|https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=556%3A7669&scaling=scale-down-width&page-id=555%3A6980&starting-point-node-id=556%3A7669|smart-link] 

*Requirements:*

Add a tab to the Site Configuration page called POS Configuration

* All sites with a POS license will be available.
* Page should operate similar to the Program Configuration page:
** User can see all POS settings listed below.
** User can see the current value at a District level.
** User can open a drawer for each setting to see the sites underneath and what the current value is at a site level.
** User can enter an Edit mode to adjust the settings.
*** User can do this at a District level by adjusting the setting row.
*** User can do this at a Site level by adjusting just that site’s row.
** User can filter by Site Type or Site Name
** Nesting is as follows:
*** POS Setting
**** Site Code + Site Name (sorted by site code) → current setting value

POS Settings List

* Principal Account Available on ExpressPoint (default=On)
* Display Message for visiting students (default=On)
* Check Server for visiting students 2nd meal (default=Off)
* Use custom meal type settings for this site (default=Off)
* Deposit change due (default=Off)
* Serve Inactive Students (default=Off)
* Change focus to PIN Input (default=On)
* Allow Ala-Carte sales to Principal Account (default=Off)
* Allow charging of visitors to patron accounts (default=On)
* Meal Service Buttons
** Student (default=On)
Visitor (default=On)
Staff (default=On)
Program Adult (default=On)
* Search Students by PIN (default=On)
* Print All Transactions (default=Off)

## Acceptance Criteria
* POS Settings tab added.
* User can see all licensed sites.
* User can filter by site type or site name.
* Page organized by:
** POS Setting
*** Site Code + Site Name
* All POS Settings listed in the description are available.
* User can enter an Edit mode to adjust the settings.
** User can adjust all sites at once or all site types at once by using the action in the Setting or Site Type row respectively.
** User can adjust a single site by using the action on that site’s row.
* Page added as part of the Resource Catalog.
** Page access.
** Edit button access.

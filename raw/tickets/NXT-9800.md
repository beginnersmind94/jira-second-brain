# NXT-9800 — AcctMgt - Account Details - Eligibility - Active and Source indicators

## Description
As a central office user I want to quickly see the active (applied) eligibility records and their source of the eligibility for a student so that I can quickly see a reference and link/navigate to the source page (application details or DC file details) for more information.

Summary

Currently, there is no easy way to see which of these is the active eligibility, nor a way to quickly get to that student’s application or DC file. Let’s add a toggle to show the active record only as well as hyperlink shortcuts to get to the Eligibility module pages for the student.

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=422986%3A134&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=422986%3A134&scaling=min-zoom|smart-link] 

*Requirements*

* Add a visual indication for inactive rows so that the user immediately recognizes the benefits that are ‘Active’ or ‘Inactive’
** For records that are inactive, display the Eligibility with an * in front. Also, display the row in italics.
* Populate the Source entry for applications (and direct cert files as well if we can do this) with hyperlink shortcuts.
** Should provide the Application #, and be a link that takes the user to the Application Details.
** For DC - provide the File #, and it’s a link that leads the user to that file.
* Add a toggle to show or hide active benefits:
** Toggled off     
*** --selected by default
*** Shows both active and inactive records
** Toggled on
*** Shows Active records only

## Acceptance Criteria
This story is complete when, if a user navigates to the Eligibility / Benefits section of an Account Info page, they can view records with a Source column populated with application or DC file # hyperlinks that redirect them to the related application or DC file, they can see an indication that shows whether each record is Active or Inactive, and they can use a toggle that allows them to filter which of those records are shown in the grid.

The toggle should display in the edit view as well, as needed to display all records.

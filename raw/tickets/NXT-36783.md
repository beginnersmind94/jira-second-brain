# NXT-36783 — MP- Menu/Menu Cycle- Default Data Source to Local and Hide (UI ONLY)

## Description
When a user is creating a menu from the standard menu, since we do not have the concept of data sources at the moment, there is no reason for the user to have to select that every time they are creating a menu or menu cycle.

## Acceptance Criteria
* Default all menus/menu cycles created to local and hide the selection of data source
* This should effect all previously created menus/menu cycles
* This should be hidden on the copy pop up  for menus or menu cycles

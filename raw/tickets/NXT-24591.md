# NXT-24591 — MP > Menu Cycle > Search > Active vs. Inactive

## Description
When using Menu Planning in the software, and creating menu cycles, those menu cycles are automatically marked as "Active" in the system.

Only "Active" menu cycles should show up on the Menu Cycle page to search.

When I change a menu cycle to "Inactive" - I do not want it to always show up in the list of Menu Cycles on the Menu Cycles page.  However, I need to have an "option" that will allow me to view the Inactive menu cycles, should I want to.

## Acceptance Criteria
When searching for Menu Cycles, only Active menu cycles will be displayed in the results. I will have an option to include "Inactive" menu cycles, should need access to one of those that I may have previously marked as "Not Active"

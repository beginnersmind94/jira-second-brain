# NXT-64498 — Acc - CACFP Reports - Activity

## Description
As Dana the Director I want to run the Activity report for my CACFP program so I can get an overview on meal counts, sales, and an estimate on reimbursement.

Currently the Activity report does not provide support for CACFP.

*Requirements*

* User can select CACFP from the Programs dropdown.
* User can run the Activity report for CACFP and see the typical information, but focused on the CACFP meal types.
** Nice to have: Only show the configured meals (from Program Configuration) for the selected site(s) + date(s).
*** Reason: There are 6 different CACFP Meals, and most districts only run ~2 meals on average. Lot of blank space if we show all 6.
*** Alternative: Add a Meal Type dropdown to the report page when CACFP is selected, and let the user select which meal types to show/not show.
** Reimbursement column will need to check the Site Config to determine what rate(s) to pull for calculations. See the table below.
*** If not set, default to using the Centers reim. rates.
*** If more than one selected, use the first that applies.
*** Do not run reimbursement calculation if set to a Facility Type that we’re ignoring.

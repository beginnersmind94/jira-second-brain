# NXT-66719 — FO - Accountability - reports - Site Balancing(maybe)

## Description
To Appease the Finance Office, it is common for Districts to need several key points of data to send off to typically a separate department. These metrics typically revolve around a sites balance and the details that make it up which the financial office takes to determine how funds are dispersed in the district.
While we can technically get all of this data from existing reports, doing so is quite cumbersome potentially requiring up to half a days work. We can solve this friction point by providing a neat report containing all of the usual required data points all providing great value our customers.

# NXT-40632 — IM - Items - Update top search bar logic

## Description
As a IM user I would like the header bar Item search to have sorting logic so I can see the correct items prioritized.

!image-20230621-182452.png|width=91.66666666666666%!

## Acceptance Criteria
* Exact matches should be top priority in results
** e.g. “1001” should return the CN item 1001
* Local Items should always be prioritized over CN items
* GTIN results should not be searched unless 8+ characters are being entered
* CN Items should have a CN identifier in the search results to differentiate them
** try to use CN icon from Recipe screen
** otherwise use (CN) prefix
* Only Active CN items should be returned in the results
** Discontinued and (especially) Inactive should not be in the results

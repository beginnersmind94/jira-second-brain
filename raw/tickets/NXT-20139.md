# NXT-20139 — INV - Move (Order) Templates to Settings

## Description
As a user I want a single ‘Templates’ page so I can use it for both Order Templates and Withdrawal Templates.

*Requirements*

* Move the ‘Order Templates’ page from the Orders category to the Settings category
* Rename the page to simply ‘Templates’
* Add a radio button next to the Template Name field
** Radio options: Order, Withdrawal
** Default to ‘Order’
* Add a ‘Type’ column to the Templates list
** this will display either ‘Order’ or ‘Withdrawal’
* Template Name should be unique regardless of Type

# NXT-3753 — Edit Inspections - Add Media

## Description
*User Story*
As an Inspections user conducting an Inspection, I have the ability to add files to any question that allows for it.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=inspections_approved_layout&id=f1misp&g=1|https://nqyc9v.axshare.com/start.html#p=inspections_approved_layout&id=f1misp&g=1]

*Requirements*

* Users are able to add files to questions that allow media
** Allow Media covered by [NXT-3631]
* The following media is allowed:
** Images: [https://www.iana.org/assignments/media-types/media-types.xhtml#image|https://www.iana.org/assignments/media-types/media-types.xhtml#image]
** Videos: MP4, M4V, OGV, OGG, FLV, 3GP, WEBM
** Audio: MP3, MP4, WEBM, WAV, AAC, OGG, OGA, M4A, MPEG
** File: DOC, DOCX, XLS, XLSX, PPS, TXT, RTF, PDF, MSG, EML, GIF, RAR, ZIP, DWG, BMP, PNG, JPG
* Users can upload multiple files
* User can view any file while conducting the Inspection
** Files can be deleted by the user
** Any delete files are permanently removed
* A thumbnail appears at the bottom of the screen
** When clicked, can be viewed in a popup or downloaded
* Files are included when the Inspection is submitted
** Anyone with access to the Inspection can view the files
** Once submitted, files cannot be deleted

## Acceptance Criteria
* Files can be added to Inspections
* Files can be viewed while conducting an Inspection
* Files can be viewed once an Inspection has been submitted

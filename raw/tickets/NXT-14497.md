# NXT-14497 — Site Import - Step 1 of 2 - Data Exchange piece

## Description
This Story is to cover the Sites @ Imports. 

Site Import has all the info needed to set up a site and is usually done once a year or once the new site is created. 



Site import file is CSV and has no parameters or filters.

This is a straight forward import with mappings mapped, saved and triggered.

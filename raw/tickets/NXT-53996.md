# NXT-53996 — Acc - Sessions - Block Delete for Online Payments

## Description
As Dana the Director I just made a mistake and deleted online payments and now my staff is getting bombarded with calls from parents. I wish the system had blocked me from making this mistake.

*requirements*

* For sessions with online payments - block the Delete Session feature.
** Inform the user: This session cannot be deleted. It contains transactions or online payments from Family Hub. Please contact Support if you need assistance.
* Add a note to the confirmation prompt to tell the user what they’re about to remove.
** Example: *This session contains 403 transactions and $130.56 in payments.*

## Acceptance Criteria
* Online Payment sessions cannot be deleted.
** User is informed why.
* Delete Session Confirmation prompt updated to tell the user how many transactions and $$ payments are in the session.

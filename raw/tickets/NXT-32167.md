# NXT-32167 — Settings Page - Config Type, Module, and Category should Correspond

## Description
The values of the Config Type, Module, and Category dropdowns should dependent upon and driven by the election of each other.



!image-20220623-204352.png|width=759,height=191!

## Acceptance Criteria
* If the user selects a value in the “Configuration Type” dropdown, the module dropdown shall contain “All” and only those modules that have a setting defined for the selected configuration type.
* If the user selects a value in the “Module” dropdown, the “Configuration Type” dropdown and the “Category” dropdown shall each contain “All” and only those configuration types and categories that have a setting defined for the selected module.
* If the user selects a value in the “Category” dropdown, the “Configuration Type” dropdown and the “Module” dropdown shall each contain “All” and only those configuration types and modules that have a setting defined for the selected category.

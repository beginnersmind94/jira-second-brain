# NXT-34147 — Team Hub - Add Messaging to Home Screen

## Description
As a school admin, I want to be able to view current messages on the home screen so that I can stay up to date with the app and coworkers.

*Flow:*

·         After logging in, land on this page

·         View Messages

·         Optionally click on View All Messages

 

*Assumptions:*

·         User has logged in

·         To complete Messages control, you will have to have completed the push / pull messages stories

 

*Requirements*:

·         Fields:

·         Messages – List Box

o   Shows list of messages for the user

o   Show From, date / time, subject and beginning text of message like Outlook and Gmail apps do

§  Text is concatenated to fit on the screen

o   Can scroll through

o   Indicates if read or not read by some kind of visual indicator using State field (Read or Unread)

o   Can click on the message and see entire message by being redirected to the Message Details screen which then loads the selected message

·         View All – Image button

o   Icon inserted in same row / space as the Messages label

o   On click, redirect the user to the Messages screen

·          

·         Messages Control updated by the Messages Timer

o   Either the control checks the timer or the timer pushes to the control, either way the Messages list box needs to be updated periodically when the messages timer is triggered

o   When the timer runs, if there are any new messages, they need to be reflected in the messages control

·          

·         Security – already covered in previous stories

## Acceptance Criteria
·         Messages

o   Shows list of messages for the user

o   Text is concatenated to fit on the screen

o   Can scroll through

o   Indicates if read or not read

o   Can click on the message and see entire message

·         User can’t see messages for another user

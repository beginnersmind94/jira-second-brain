# NXT-35324 — INV - Add supplementary Pack Size info for every Item display that requires it

## Description
As an Inventory user I need the pack size conversion information for every item so I can know how to understand the quantities across multiple pack sizes.

*Requirements*

* Add the pack size units and sub-quantities somewhere near Item Description on every page that requires this information
** This is often *not* required where QOH is displayed because we have the alternate QOH popup
** This is especially necessary for reports
** Display the conversion info inside brackets
*** Ex: [Case = 12 Bag]
** Display to the right (at the end) of Item Description in reports
*** unless insufficient space
* Implement on the following pages/reports
** Back Orders 
** Batch Detail 
** Expected Receipts
** Valuation Report
** Additions Report (on page)
** Order Detail Report
** Item History Report
** Withdrawals Summary Reports
*** Item Summary
*** Reason Summary
** Physical Inventory
*** change parentheses to brackets
** Site Ordering Guide (maybe)
** Price Details
*** change to brackets, not parentheses
** Any other reports TBD

## Acceptance Criteria
* Add Pack Size sub quantities everywhere with Item Description that needs it 
* Display in brackets after Item Description

# NXT-23481 — System- Import Dashboard - Hide view log button 

## Description
When an import is in progress and the user want to monitor the import via import monitoring page the view log button is showing even the when the import is in queued state. This button need to be hidden if the import is in queued status

## Acceptance Criteria
AC1 : View log button should be hidden when the import is in queued status
AC2: The remaining status functionality shouldn't be affected with this change

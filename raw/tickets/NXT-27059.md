# NXT-27059 — IM - Add Fields Columns & Views with fields (Recipe)

## Description
As I user I should be to filter recipes  from a complete list of recipe property fields so I can reduce my scope of search. 

!image-20220222-005028.png|width=235,height=32!

## Acceptance Criteria
*Entry Point*

* Item Management → Recipes → Columns and & Views. User should be able to select columns to be displayed from a complete of available fields by clicking on “Columns & View button”. 
** Default fields in view are:
*** Recipe Code
*** Recipe Name
*** Data Source
** User should not be able to de-select/uncheck default fields
** An option to Select ALL fields should be available to the user
** While on selection of fields to view, the user should be able to reset to the default fields
** View a count of the number of fields selected
** The user should have an option to close the card and return to the Recipe list view
** The user should be able to see in the results grid the fields and values as per field selection
** add fields as per attached excel file to Columns and views
** Display Serving Size & Measure as one field

*Generic Requirement:*

* The check box parameters are deselected by default when the screen is loaded for 1st time
* When user apply more than one parameters, it works based on AND condition
* Standard pagination should be maintained  
* Standard Grid functionality should remain (for example most recent record should be at the top, sorting columns)

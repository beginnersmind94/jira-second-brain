# NXT-11030 — IM - Items - View - Item Side Bar

## Description
As an Item Management user, I want to be able to view my items on the left sidebar

*Mockups*

 !image-2021-02-14-18-22-22-706.png|thumbnail! 

*Requirements*

* Item Name
* Item Code
* Missing Information Label
* Make it expandable
** GTIN
** Product Code
** Datasource

Allow the user to expand and collapse the item on the left bar
Keep the thumbnail image on the left if there is an image
* Should be no bigger than the top of the Item Name field and the bottom of the Item Code field
* If no image, then move the Item Name and Item Code to the left to keep it 'centered'
If 'Mandatory' fields, then show them always. If no data in the GTIN / Product Code fields, then don't show that field.

## Acceptance Criteria
User will be able to see and click on the item on the left hand side
User can expand and collapse the item
User will be able to see the picture if applicable
If no data is present in the fields, the fields and data won't show for non mandatory fields

# NXT-27638 — IM - POS Item Copy

## Description
As Item Management administrator I need  to able to edit the POS button when copying an item that is a POS item in order to reduced the risk of creating POS items with duplicate POS button names.

## Acceptance Criteria
*Entry Point*

* Item Management → Items → Selected Item → copy Icon
* Before the POS section of item is saved prompt the user to change the POS button name (English & Spanish)
* Do not validate Spanish button for duplicate at this time.

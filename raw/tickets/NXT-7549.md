# NXT-7549 — INV - Physical Inventory - Supervisor View - Count History

## Description
As an Inventory user I want a Item Count History so I can see the history of item counts over multiple physical inventories.
Based on user permissions.

*Requirements*

Provide a link on the Items list of the Pending Reconciliation screen that allows a user to select an item and view the history of that item on previous physical inventories.

* Provide a history icon next to Item Number
* Provide a hover tooltip on the icon which says ‘Count History’ so user knows what link is
* Upon clicking icon, display a popup containing count history grid
** Count History information provided:
*** Physical Inventory Date
*** Physical Inventory Count
*** Discrepancy (and/or Perpetual)
*** Counted By user
** Provide the last 6 physical inventory counts.

|7/1/2020|15 cases, 1 packs|7 cases, 2 packs|Justin|
|8/1/2020|12 boxes, 2 packs|1 box, 2 packs|Justin|

## Acceptance Criteria
History icon will provide Count History popup
Hover tooltip will be displayed to indicate the purpose of the count history icon link
Count History popup will display last 6 Physical Inventory counts

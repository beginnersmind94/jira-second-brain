# NXT-19190 — UX - IM - Added Ingredient / Recipe Report

## Description
As a Nutritionist I need a report that tells me what Ingredient or recipes have been added given a date range so I can trace recipe changes

## Acceptance Criteria
This story is done when:

I have a selection criteria that allows me to select by

* Data source 
* Date range
* Recipe or Item
* The result data grid display fields:
** Data Source,
** Code,
**  Name
** Manufacturer (Only apply to Items. For Recipes the Value should be left blank)
** Added By
** Added On.
* I can filter by  any of the fields on the data grid
* Data displayed on data grid is as per criteria selection and/or filter(s)
* I can print the page(s)
* I can export the page(s) to excel or PDF 
* For Items created manually thru the system

# NXT-5208 — Other Benefits - List

## Description
*User Story*

As a district admin I want to see the available Other Benefits configured for the academic year.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5798%3A25&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=5798%3A25&scaling=min-zoom]

*Business Information*

* Districts offer additional benefits/programs to households, like assistance paying for textbooks, or SAT testing.
* Benefits are made available, and households can put their hand up to partake while applying for free/reduced meals.
* The benefit information is recorded against a student directly if they’re partaking in it.
* This information needs to be collected and available for export at any time.
* Benefit information runs by academic year and needs to be configured each year as programs may start/finish.

*Requirements*

* Add a ‘Other Benefits’ page under configuration.
* Information is sorted by Academic Year
** Academic Year selector, default to current academic year on page load.
** Changing the year updates the page immediately to show info from that year.
* List of Departments available with the following information:
** Name (Varchar, 30 characters)
** Code (Varchar, 15 characters)
** Active (bit, 1=active, 0=inactive)
*** Shown as a toggle in the grid.
**** Cannot be toggled in the grid.
* List of Benefits available with the following information:
** Name (Varchar, 50 characters)
** Description (Varchar, 150 characters)
*** A description should exist for any active Language the district is using.
*** Some districts may activate upwards of 20 languages - we might want to bury anything beyond the English description in an expanding section, ‘View More’ or a chevron or similar.
** Department
*** Selection from existing departments.

## Acceptance Criteria
Other Benefits page added, showing information on departments and benefits.
Description for benefits shows all active languages.
Academic year available, data refreshes accordingly on year change.

# NXT-2451 — Inspections Reports - Inspection Summary Report

## Description
*User Story:*

As an Inspections user, I have a report that provides a summary of Inspection results based on parameters of sites, templates, and date range.

*Current Version:*

[https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports|https://inspectionsrelease.primeroedge.co/#/app/inspections/Reports]

*Requirements:*

* Inspection Summary Report can be run using existing Inspections and available parameters
** User must have Reports role permission to run report
** Report can be created in PDF, Excel, or Emailed
* The following parameters are used (all are required):
** Site - Can select 1 or many
** Template - Can select 1 or many
** Start Date
** End Date
* All Inspections that fit within the chosen parameters appear, regardless of status
* The following data is displayed on the report:
** Header
*** Report Name
*** Start Date
*** End Date
** Site
** Template
** Inspection Name
** Date
** Inspector
** Score
** Total count for each template
** Total count for each site

## Acceptance Criteria
* Inspection Summary Report can be created with available data across all parameters in all formats

## Description
_(none)_

## Acceptance Criteria
Navigation: Insights → Command Centre

User should be able to see 2 tabs named as Front Office and Back Office

Upon clicking Front Office all the KPIs built based FO data shall be visible to users

Upon Clicking Back Office, the following KPIs shall be listed at region level

Production Accuracy %

Serving Percentage %



Upon clicking on percentage, the following data should be displayed in slide out:

Planned Quantity

Produced

Total Served

Coverage Pie Chart + Region Level cards

The threshold bar should reflect in UI based on the values in defined threshold settings 

Export - user should be able to export the data in Excel

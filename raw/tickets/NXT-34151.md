# NXT-34151 — Team Hub - Reply or Forward a Message

## Description
As a school admin, I want to reply or forward a specific message so that I properly respond to a message.

*Flow:*

·         After logging in, user redirected to Home Screen

·         On Home Screen, click the View All Messages Icon and redirected to Messages Screen and on that screen click on a message, or

·         On Home Screen, click on a message in the messages control

·         This screen opens up and user views automatically loaded message details of selected message

·         Admin click on Reply, Reply All or Forward button

·         Admin redirected to Send Message screen with the proper fields completed

 

*Assumptions:*

·         User has logged in

·         To code this, you will have to have completed the push / pull messages stories and one of the source triggers (Home Screen or Messages Screen)

·         To code this, you will have to set up the Message Details screen covered in another story

 

*Requirements*:

·         Fields: No additional fields, just updating existing fields

o   If these fields were not added to the screen, then add them now

·          

·         Forward – Image button

o   Make look like traditional email Forward icon

o   Works like traditional forward button

o   On click, redirect the user to the Send Message screen

§  Do not preload that screen’s To

§  Preload that screen’s From with the Admin’s account

·         Reply – Image button

o   Make look like traditional email Reply icon

o   Works like traditional reply button

o   On click, redirect the user to the Send Message screen

§  Preload that screen’s To with the From in this screen

§  Preload that screen’s From with the Admin’s account

·         Reply All – Image button

o   Make look like traditional email Reply All icon

o   Works like traditional reply all button

o   On click, redirect the user to the Send Message screen

§  Preload that screen’s To with all the emails in the From and To in this screen except the Admin’s Account

§  Preload that screen’s From with the Admin’s account

·         Compose – Image button

o   Use a traditional writing, composing icon

o   On click, redirect the user to the Send Message screen with no auto-populated fields

·          

·         Security – no additional requirements necessary

*Requirements:*

* The Daily Restrictions set the Maximum number of AlaCarte items that can be sold by Site Ty

## Acceptance Criteria
Default Restrictions is launched from the Configuration from the left panel
Default Restrictions is reached through the tabs of the Configuration page.
Daily Restrictions
    Selecting Edit Button allows the user to change the value for the Site Type selected.
   If the changed data is not saved it returns to the previous value
   The default value is 0.
   Maximum value is 9999
   There is a note (Note: If set to zero, there will be no restrictions on the sale of Non Reimbursable menu items)
IF cancel is selected any edited values revert to the previously saved value.
IF save is selected all edited values are saved.

Save Success Message "Daily restrictions have been saved"
If no changes have been made Save button is not enabled
If changes and Cancel button is selected. display "You have unsaved changes, are you sure you want to cancel?"

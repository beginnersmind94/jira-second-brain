# NXT-2812 — Inspection Template - Global Template Creation

## Description
*User Story*
As a Cybersoft Employee, I have settings for creating a Global Template using the create template screen.

*Mockup*
[https://m5uloa.axshare.com/#id=usu4ze&p=add_new_questionnaire&view=default&g=1|https://m5uloa.axshare.com/#id=usu4ze&p=add_new_questionnaire&view=default&g=1]

*Requirements*

* Global templates can be created using the Create Template screen
** Setting is only available to users with a Cybersoft Only role permission
* There is a checkbox that allows creation of a global template
** Checking the box sets the template as a global template
* Global Template settings are available under Settings
** These settings are only available if the user has the Global Template checkbox checked
** Settings
*** Access - Determines whether template is available to all or to specific customers only
*** Tags - Allows tags that can be applied to the template to provide specific categories for the template
*** Description - An explanation of the Global Template, including any sources
* Once the template is saved, it is not set live to users outside of Cybersoft until it is manually done
** This will be covered in a separate story

## Acceptance Criteria
* Global template settings are present and can be used to create global templates

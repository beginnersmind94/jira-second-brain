# NXT-3552 — Configuration - Soft Delete Method, Source

## Description
With permission, ability to delete method, source

* Deleting method, source removes the entry (for future events) but doesnt affect past events that have the method assigned. 
* Users cannot delete (we wont show trashcan) the methods and sources preloaded in the system and provided to them by primeroedge. 
* Cant only delete if it’s made inactive first. trashcan shows when status is inactive.

## Acceptance Criteria
Admin (with permission) can delete method, source

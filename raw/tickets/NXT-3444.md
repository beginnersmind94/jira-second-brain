# NXT-3444 — System - Management - System Settings - Create System Settings page

## Description
As a district or Cybersoft administrator, I would like to be able to easily manage settings related to the entire website in one location, so that I can quickly make the changes I want for PrimeroEdge to function as needed.



* In the _Configuration_ category, add a *System Settings* page
* In this page, display +*only*+ the settings related to the overall performance found in the system. These are located in the “System” tab of the page in PrimeroEdge Classic. 
* We currently show tool-tips when hovering over a setting and can retain that as well (if not hovering, then with an icon)
* Internal system settings should be displayable via a toggle for those with the permission to view them (currently a checkbox. Will be a toggle in 2.0)
* Allow the user to view default setting value related to that setting
* Settings should be of the same type (i.e. bool, string, drop-down, calendar) even though we can upgrade the controls themselves, i.e. using a toggle instead of Yes/No radio buttons.
* Tool Tip needs to be added to every setting

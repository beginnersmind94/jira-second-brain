# NXT-23829 — Production- Production Plan- Menu items/serving groups added or removed to MP after assignment show on Production Plan

## Description
As a user, when I add additional menu items/serving groups to a menu once assigned, update that information on the production plan.

## Acceptance Criteria
When menu item is added or removed on the menu planning side, display menu item/serving groups on production plan
If production plan is in plan complete, change status to production incomplete only if menu item is added since the counts will need to be entered.
IF production record is in record complete with a particular menu, do not allow user to add/remove menu items or serving groups to the menu on menu planning.







THIS IS FOR MENU PLANNING (Display message "One or more Production Records have been completed for this Menu assigned to (next assigned date). Only changes to the counts are allowed. You may however, copy the menu and reassign with new changes")

# NXT-26258 — Acc - Display Dynamic Menu Reimbursable Meal and Transaction Details in Sessions

## Description
The transaction details for dynamic menus needs to be displayed when editing or showing a transaction for an individual in a session.

When a reimbursable meal is formed the meal is shown with the qty of 1, the price set to the eligibility price of the reimbursable meal and the extended price.

The items that makeup the reimbursable meal are listed below and indented from the Reimbursable meal entry with the correct Qty, the price of the item based on it being sold individually, with the extended price being $0.00 or not shown

Additional a-la-carte items are shown with the appropriate qty, price and extended price.

## Acceptance Criteria
The transaction detail shows the Reimbursable meal formed by DM and the items included to create the meal.

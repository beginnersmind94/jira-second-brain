# NXT-22845 — Front Office Reports - Add Download Option + Move Report to Overlay

## Description
As Vera the Director I want to easily view a report in a larger space as well as download a copy of that report, so that I can send it to print, or email it off.

*Mockup (NOTE: This only shows Accountability reports, but the same changes need to be made to the Eligibility/Verification/Processing reports as well\!)*
[https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=5071%3A48667&scaling=scale-down-width&page-id=3981%3A36238&starting-point-node-id=3981%3A36243|https://www.figma.com/proto/Mr8UK5bhrIQCTrAgRmVPoO/Account-Management-4.0?node-id=5071%3A48667&scaling=scale-down-width&page-id=3981%3A36238&starting-point-node-id=3981%3A36243|smart-link] 

Generate each report in the style of the mockup above - note the overlay + the PDF button.

*Requirements*

* For both the +Eligibility → Reports and Accountability → Reports pages (but+ +*also*+ +including Verification-Collection Worksheet/Application Processing reports)+:
** Every report needs to be updated to load into an overlay rather than embedded on the page.
** Every report needs a PDF button added which will send the HTML report to PDF and allow the user download it.
** Add a ‘Reset’ button to each report, which when selected allows the user to reset all of their filter selections to the default.
** All reports should have their content/behavior otherwise unchanged.
** Filename convention: \{Report}_MM-DD-YYYY.pdf

## Acceptance Criteria
* Reports moved from embedded to overlays, per the mockup.
* Download button exports the report to PDF.
* Reset button added to each report, which when selected allows the user to reset all of their filter selections to the default.
* The above should be applicable for both the Eligibility and Accountability reports (and *also* the Verification-Collection Worksheet/Application Processing reports).
* Behavior/content of reports should be otherwise unchanged

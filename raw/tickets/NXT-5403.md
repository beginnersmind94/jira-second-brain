# NXT-5403 — Item Management - Serving Measures used in Recipes

## Description
*User Story*

As a user of Item Managment, I need to make sure I can’t remove a serving measure from an item that is used in a recipe

*Mockups*

N/A

*Requirements*

* If an item and it’s serving measure are present in a recipe, the user CANNOT remove or edit the serving measure from that item.
** The user can use standard weights in the recipe, in which case the measurements could be edited on the item itself
* User cannot edit the serving size and the serving measurement if it’s being used in a recipe
* User can edit and change the weight
* Error Message to popup - “This serving measure is being used in a recipe. Please remove this item from the recipe to delete the serving measure”
** Click OK
** Updated messaging based off 10-20-20 comment:
*** “Serving measure is currently used in an items contribution section”
“Serving measure is currently being used in an items nutrient information section”
“Serving measure is currently being used by a recipe”

## Acceptance Criteria
User cannot edit or remove a serving measurement from an item if it's used in a recipe
The user can change the weight of the serving measure

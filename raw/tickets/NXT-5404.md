# NXT-5404 — Item Management - Recipe - (Re)calculate Nutrient Information

## Description
User Story

As a user, I want to be able to ‘(Re)calculate’ my Nutrients

Mockups

!image-20201013-131855.png|width=100%!

Requirements

* Add a Outline button that says ‘(Re)calculate’ to the bar for Nutrient Information
** !image-20201013-131949.png|width=1876,height=364!

* This needs to come after [https://primeroedgenext.atlassian.net/browse/NXT-6216|https://primeroedgenext.atlassian.net/browse/NXT-6216|smart-link] and [https://primeroedgenext.atlassian.net/browse/NXT-3884|https://primeroedgenext.atlassian.net/browse/NXT-3884|smart-link] which will allow the Edit nutrient information  and auto calculate
* When clicking this button, it will function the same as NXT-3884, just not directly on save of the recipe
* This button will be allowed to be selected at any time
* Warning message WILL NOT display if the user has NOT made any manual changes
* Warning message to display if the user has made any manual changes with a confirmation dialog
** By clicking (re)calculate, you will lose all your manual changes below. Do you want to continue?
** ‘CANCEL' and ‘CONFIRM’
* This information will not be stored in the database until the user hits ‘SAVE’

## Acceptance Criteria
User can autocalculate the nutrient information for a recipe

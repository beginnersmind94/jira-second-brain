# NXT-20890 — DataQuery Framework - Developer's Guide

## Description
A “Developer’s Guide” shall be provided for a module developer so that they know how to incorporate the DataQuery framework into their web pages.

## Acceptance Criteria
* As a module developer, I would like to have a developer’s guide available so that I can easily and intuitively incorporate the DataQuery framework into my web pages.  The following shall be made available in the guide:
** A high-level overview of the components involved in incorporating the DataQuery framework
** Code examples
** A responsibility matrix of what the Platform team is responsible for and what I, as a module developer, is responsible for

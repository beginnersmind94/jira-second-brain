# NXT-30224 — IM - Items - Item Details - Edit Product Info

## Description
As a District Admin I want an Edit function for the Product Info card so I can update my category and valuation attributes.

*Requirements*

* Update the Create mode
** -Change ‘Smallest’ to ‘Lowest’-
** Switch ‘Lowest’ to be the first radio option, and selected by default
** If ‘Highest’ is selected, automatically add a second row
** -Add a delete icon for the active row-
* Provide an Edit button for the Product Info card
* Enable the 3 dropdown values to be edited
** Item Category, Storage Category, Valuation Group
* Allow the Pack Size Configuration to be edited in some ways
** The Descriptions can be updated
** GTINs can be added/updated
*** future enhancement: prevent if item is linked
** Fair Market can be adjusted on the highest unit
** Weight can be adjusted on the smallest unit
*** Possible warning: “This may affect the conversion of Serving Measures to items.”
* Do *not* allow the Sub-Quantity values to be changed
* Do *not* allow units to be removed, added (one exception), etc.
* Provide a Save button
** all 3 dropdown attributes are required
** Minimum 1 unit with Description, Weight, FMV
* Provide a Cancel button

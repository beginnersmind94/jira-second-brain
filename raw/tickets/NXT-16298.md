# NXT-16298 — INV - Create Orders - Confirm Screen - Merge Order Popup

## Description
As an inventory manager I want a Merge Order prompt when creating a new order so I am able to merge my new order with an existing order if they have the same Site, Vendor, Delivery Date and Delivery Location.

*Requirements:*

* Provide a merge icon if an order can be merged into an existing order ([NXT-9730|https://primeroedgenext.atlassian.net/browse/NXT-9730])
* Open a popup when user clicks the Merge icon
** Popup will display the current order on the left with the following attributes displayed:
*** Order #
*** Site
*** Vendor
*** Delivery Date
*** Delivery Location
*** Number of Items
** The other order, to which this order can be merged, will be displayed on the right
*** Display the same attributes
*** This will include a View icon to open the corresponding Order Detail page
**** this should open in a new tab
** Cancel button
*** this will close the popup without merging orders
** Merge button ([NXT-16300|https://primeroedgenext.atlassian.net/browse/NXT-16300])
*** this will merge the order into the existing one (shown on the right)
*** this will close the popup
** Validation section
*** Show an alert icon (!) for any order validation issues that may occur if the orders are merged
*** Provide an explanation for the alert icon
**** (!) “Merging these orders would result in exceeding the maximum purchase amount for this vendor.”
**** (!) “Merging these orders would result in one or more items exceeding the maximum order quantity.”
**** Disable the Merge icon if any of these validation issues occur
* If there is more than 1 potential orders to merge into, use the most recently created order
* Once an order has been merged, change its status to ‘Merged into Order #12345’
* Remove the Merge icon for any orders that were merged

# NXT-8110 — Production - Production Record - Add Menu Item

## Description
As a site manager, who has completed service for the day… there are occasionally times when things do not go according to plan, and I will need to add a menu item to my end of day details that was not originally on the menu. This could happen because of product being expired or spoiled, it could be due to an event that I was not informed about in a timely enough manner to notify my central office, product that did not arrive in time for service, or any number of other occurrences. 

When this occurs, I need to be able to select and add a different menu item to the end of day details.

## Acceptance Criteria
The user will be able to select a menu item from the software to add to their production record.

The user will be able to add the quantity produced, and other end of day details to the item (carryover, waste, etc) 

The user will be able to also remove that item from the page, in the event that they select an incorrect item to add. (Note that the user will not be able to remove any of the items originally assigned to their menu.)

The food required and withdrawal details will also come through with that item, as the user enters the details.

# NXT-37094 — PR- Production Status- Allow user to refresh the data based on the criteria entered

## Description
As a Production Manager, I would like to be able to refresh my display of the production status view with the latest information. Since the menu planner can continue making additional assignments, there is no way for me to know unless I refresh the view to see the new changes.

## Acceptance Criteria
* Provide a Refresh button next to the criteria of the page
* The refresh will use the same criteria has preset and redisplay the information with any new changes if applicable
** If user changes the criteria, keep the criteria they change and refresh based on that

!image-20230227-231810.png|width=1324,height=483!

* If user does not refresh and they select a particular plan that is no longer available, do not lead them to a non existent plan/record page, refresh the production status page
** Ex: The menu planner reassigns the menu or deletes the menu and the plan is still showing up, this would be the scenario 
** Display a message at the top when the page is refreshed “Plan/Record selected is no longer available”

# NXT-48998 — AM - Reminders - Scheduler Setup

## Description
As Dana the Director I want a way to setup an automated balance reminder to be emailed out at a regular cadence to save time and encourage account payments.

*requirements*

* User can save their search criteria on the Reminders page as a recurring send.
* User can set the frequency for recurrence while saving it.
* User can provide a name the the recurrence schedule while saving it.



Letter Format mentioned in acceptance criteria is from the print options dialog.

## Acceptance Criteria
* User can save their search criteria on the Reminders page as a recurring send.
* User is prompted to set the frequency for recurrence while saving it.
** Start Date
** End Date
*** No end date is required
** Start Time
** -End Time-
** Frequency
*** -Hourly-
*** Daily
*** Weekly
*** Monthly
** Days
*** Sunday → Saturday
* User is prompted to name the schedule while saving it + provide a description.
** Name: Text field, 100 character limit. Mandatory.
** Description: Text field, 150 character limit. Optional.
* User is prompted to pick a letter format (Individual or Household) and a Letter Template.
** We’ll always use Reminder Letter (Single) for this.
** Letter Template dropdown should contain all available templates - so for students, all templates (Active and Inactive) for Reminder Letter (Single) - Student.
*** Default selection is the Active template.
* Recurrence is saved and used for the automatic scheduler.
* Text strings:
** Save success message: Recurring Notice saved.  
** Text on the scheduler drawer near the Template Selector button:
*** The system defaults to active letter templates. Use the Template Selector button to choose different templates.

# NXT-65486 — Income Statement Pt.5 Meal Counts

## Description
*As a* district administrator
*I want* the Meal Counts section of the financial report to be automatically sourced from existing POS meal transaction data
*So that* I can view accurate counts for each meal category for the current period, prior period, year-to-date totals, and related percentages without manual calculations.

!image-20250811-223427.png|width=677,height=302,alt="image-20250811-223427.png"!

## Acceptance Criteria
* Create a table with the following columns:
** Current period
** Prior period %
** % change from prior period
** Year-to-date
** % of total meals
* Use the same source data as the Meal Count report for the specified filters and dates.
* Maintain the breakdowns shown in that report (program types, adult, a la carte, etc.).
* Ensure totals in the financial report match those in the source report.

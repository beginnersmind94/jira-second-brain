# NXT-5210 — Application Details - Comments

## Description
*User Story* 

As an applications processor, i can see the comments attached to the application with the option to add more as needed. 

*Mock Ups* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=87%3A2599&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=87%3A2599&scaling=min-zoom]

*Requirements* 

* Add Comment > ability to Add comments to an application
** Any comments added when submitting via manual entry or processing applications will be collected and shown here.
* Date, Timestamp, and user (firstname + lastname) are displayed with each comment 
* Longer comments - show the first 35 characters, do an ellipsis after that. Chevron to show the full comment.
* cannot edit or delete comments at this time

## Acceptance Criteria
- Multi-language, ADA compliant
- shows comments with date, timestamp, and user info 
- user can add a comment up to 500 characters long

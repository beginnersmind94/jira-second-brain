# NXT-22677 — MP- Menu Item Help Button- Displayed information needs to be hyperlinked

## Description
As a user, when I click on the Menu item Help Button, in addition to the information currently displayed, I also need to see the Recipe Code for the menu item. This can be next to or above the Menu Item Name.

The Recipe Code and the Ingredient Codes should be hyperlinks, so that I can click on them to access the recipe or ingredient pages. (These should open in a new tab, as they do when clicking on the recipe code from the “Add Menu Item” screen on the menu. )

“Stock Item #” field to be changed to state “Item #”

Additional Item # behind the pipe delimiter on the right hand side can be removed, as it is redundant in SchoolCafe. (While being necessary in classic, due to a “pairing” of Ingredient to stock item - this is no longer needed with Item Management handling the full instance)

Can the icon of question mark (?) be changed to something indicating information? (i) or something similar? (I am not sure if there’s precedence anywhere else in the platform for this?)

# NXT-7563 — INV - Physical Inventory - Main Screen - Summary tab (Physical Incomplete status)

## Description
As an Inventory user I want a screen for all my physical inventory functions so I can count my items, generate reports, reconcile quantities, etc.

[Figma|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=89559%3A150108&viewport=325%2C-4288%2C0.08251962065696716&scaling=scale-down]

This is the first tab of the the screen that contains the different components of Physical Inventory

Summary tab - first tab that appears when page loads

* Summary section, to display information per Storage Category
** Storage Category
** Total Items (for that category)
** Zero Quantity Items
** ‘Select this Category’ button -  [https://primeroedgenext.atlassian.net/browse/NXT-7856|https://primeroedgenext.atlassian.net/browse/NXT-7856|smart-link]
*** the Item list will be filtered based on which Storage Category is selected
*** All option - this will open all items from all storage categories
* Action buttons:
** Save as Incomplete
*** keep the inventory in ‘Physical Incomplete’ status
*** saves work for returning later and continuing.
** Save as Complete [https://primeroedgenext.atlassian.net/browse/NXT-7833|https://primeroedgenext.atlassian.net/browse/NXT-7833|smart-link]
*** when this is clicked, provide a prompt (popup, slide out, etc.) that requires the user to enter
**** Counted By (user)
**** Comments
**** Counted Date/Time 
***** only required if the physical inventory is by Date and not Period
**** display Period
***** loaded from the inventory
** *Cancel* (same behavior as [https://primeroedgenext.atlassian.net/browse/NXT-7902|https://primeroedgenext.atlassian.net/browse/NXT-7902|smart-link])
*** this will prompt the user to cancel the inventory and invalidate all of the item counts
*** this changes the status to Cancelled
*** if user clicks Cancel, prompt user to enter comments
**** user is provided a popup to enter comments 
**** comments are required to cancel an inventory
*** user remains on the same page but header changes to Cancelled information

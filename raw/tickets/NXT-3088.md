# NXT-3088 — **System - Site Types Card View

## Description
As an administrator, I would like to be able to create a list of site types to which all of my sites belong, so that I can properly organize my sites according to the grades they support as well as as other non-school functions.

Classic PrimeroEdge path: _System > Sites and Users > Site Types_

!image-20200121-221050.png|width=1043,height=559!



Functionality-wise, this can be a straight port of the existing application. The main changes should be in terms of UI/UX improvements.

## Acceptance Criteria
1. User should be able to view existing site types
2. User should be able to expand a site type to view list of grades

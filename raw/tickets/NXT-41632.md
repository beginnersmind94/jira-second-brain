# NXT-41632 — Adult & Service Employee Import – Gender Auto-Mapping

## Description
As a SchoolCafé Admin I want the system to auto-map the Gender field so that I can import files faster.

*Notes:*

* This story corresponds to how we do gender auto-mapping when importing in Student Import process
* We want to do the exact same thing for Adult Import and for the Service Employee Import

*Requirements*:

* When importing an Adult Import, do the following to the Gender field:
* 
* Create default mapping that is automatically available when the user wants to import files
* Gender:
** Female = F
** Male = M
** Blank = N
** Not Provided = N
** Not Specified = U
* Do not auto-map the F, M, N, U values if found in the file
* This is the same as Student Import
* This should be for both manual and automatic imports
* Do all the same warnings, edit icon color changes, etc. as Student Import

## Acceptance Criteria
* Gender field auto-maps these fields from the file
* User can modify the mapping or delete it
* User can continue to add / edit new or existing mappings
* System will automatically convert the import text based upon mappings

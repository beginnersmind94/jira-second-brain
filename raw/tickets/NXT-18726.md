# NXT-18726 — INV - Manage Orders - Receipt Detail - Move receipt adjustments from Receipts tab to History tab

## Description
As an Inventory user I want to see receipt adjustment transactions as part of the receipt history, so I am not seeing a receipt tab for every adjustment transaction.

*Requirements*

* Do not display a tab on the Receipts tab of Order Detail for any adjustment transactions
* The receipts displayed on the Receipts tab will be only the current state of the individual receipt
** only the most recent version of that receipt
* Adjustment transactions will be displayed on the history tab for the specific receipt
* Cancelled receipts will be displayed as a tab with strikethrough
** Only the most recent, final cancelled version will appear as a tab with the original Receipt #
** Transactions prior to the cancellation will be displayed as history under the cancelled receipt
** Cancelled Receipts will have the negation transaction as the current transaction version

## Acceptance Criteria
* Historical receipt transactions will appear under the history of a specific receipt
* Historical receipt transactions will not appear as tabs/chips on the Receipts tab
* Cancelled receipts will remain as a tab/chip but with strikethrough

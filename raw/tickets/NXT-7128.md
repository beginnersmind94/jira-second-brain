# NXT-7128 — Menu Grid Layout

## Description
*Story*

As a user I need to be able to create menu grids for use on ExpressPoint. There are two types of menu grids, static which have menu items that don’t change and dynamic which has items that change based on menu planning. This story handles Static menu grids. 

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=29545%3A1331&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=29545%3A1331&scaling=scale-down-width]

*Requirements*

* Menu Items can be selected and dragged to the Menu Grid
* Categories can be selected and dragged to the Menu Grid
* A resolution can be set for grid display
* A Menu Grid can be saved as a Template [https://primeroedgenext.atlassian.net/browse/NXT-5903|https://primeroedgenext.atlassian.net/browse/NXT-5903|smart-link] (Ver 2)
* The Menu Grid can Use a predefined Template [https://primeroedgenext.atlassian.net/browse/NXT-5903|https://primeroedgenext.atlassian.net/browse/NXT-5903|smart-link] (Ver 2)
* The menu grid size can be defined.
* Buttons can be resized on the grid [https://primeroedgenext.atlassian.net/browse/NXT-7132|https://primeroedgenext.atlassian.net/browse/NXT-7132|smart-link] 
* Button Details can be viewed and edited [https://primeroedgenext.atlassian.net/browse/NXT-7197|https://primeroedgenext.atlassian.net/browse/NXT-7197|smart-link] 
* Buttons can be moved/repositioned on a grid [https://primeroedgenext.atlassian.net/browse/NXT-7132|https://primeroedgenext.atlassian.net/browse/NXT-7132|smart-link] 
* A Button can be removed from a grid.
* Buttons can show an image if provided
* Grid resolutions are shown based on device (Ver 2)

## Acceptance Criteria
Menu Grid layout can use a saved template NXT-5903 (Version 2)
Menu Grid Rows and Columns can be set. 
Rows Max = 16, Min = 1
Column Max = 15, Min = 1
Default Device Resolution is POS (1080 x 1024) 
Any Grid changes can be Saved or Canceled
Canceled reverts to the previously saved data
A history of edit changes is recorded to allow UNDO and REDO is unlimited.
If no changes have been made UNDO/REDO are not enabled.
When the grid is saved the UNDO/REDO history is reset
Undo removes the previous change 
Redo adds the previous change removed by Undo
Menu Items or Categories can be dragged onto the Grid

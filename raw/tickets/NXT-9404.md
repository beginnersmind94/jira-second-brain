# NXT-9404 — Item Management - POS Menu Item - Enable Sale when Price is set

## Description
When a price is entered for All Prices, A Row, or a column the Enable Sale is toggled to true.

## Acceptance Criteria
When set all prices is selected, all Enable Sale is set to True for all price types.
When a price is set for a price type the Enable Sale is set to True for that Price Type
When a Site type or Grade group price is set the Enable Sale for all price types is set to true.

# NXT-20657 — MP-Menu- Variety Menu Item

## Description
As a Menu Planner, I want to be able to add my variety Menu Item to a Menu. Allow for the Variety Menu Item to be a "placeholder" while all menu items within it are displayed in a collapsable or tree like format.

API is dependent on IM story 

[https://primeroedgenext.atlassian.net/browse/NXT-20655|https://primeroedgenext.atlassian.net/browse/NXT-20655|smart-link]

## Acceptance Criteria
When Variety Menu Item is added to menu, only display the Variety Menu Item Name and Contributions and option to enter total Projected Meal Count
Allow for collapse button with tree grid to open and display the assortment of items within Variety Menu Item.
Projected meal count will represent the average of all menu items in the variety menu item (THIS WILL BE HANDLE IN SEPARATE STORY)

User can add variety menu item from Add Menu Items button on the MP screen. (Currently not showing in the add menu item list)

Show variety menu item bubble collapsed. User must click on “Show Variety Menu items” to see all the items. 

No strict batching for variety menu item.

Exclude menu item help button for Variety Menu items.

Disregard Variety Menu item when it comes to menu costing/menu cycle costing if no cost is available or there is a null value

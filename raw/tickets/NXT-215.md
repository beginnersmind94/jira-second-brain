# NXT-215 — Meal Exceptions - View/Export Duplicate Meals

## Description
As a user I want to see the list of meal exceptions. and be able to resolve them. There are 2 types of meal exceptions. 

# 2nd Meals - when two or more reimbursable meals are served for the same meal type and same day as being reimbursable.
# When meals are recorded at the wrong eligibility type. [https://primeroedgenext.atlassian.net/browse/NXT-10082|https://primeroedgenext.atlassian.net/browse/NXT-10082|smart-link]

*Requirements:*

* A list of patrons with duplicate meals is shown
* When a patron is selected the list of meals is shown.

## Acceptance Criteria
A list of patrons with more than one reimbursable meal for the same day and meal type
Each patron can be expanded to show a list of duplicate meals
Each meal has the option to Keep, Delete, or Change to 2nd meal and charge account.
Default is Keep

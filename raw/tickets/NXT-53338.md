# NXT-53338 — CC - High Level Stats

## Description
As Andy the Area Manager I want to see stats for how sites have performed for the dates selected. This will help quickly determine if any sites are showing abnormal numbers.

*requirements*

On the Focus View…

* User can see totals for the following for the selected sites + dates:
** Meals Served
** Payments
** Revenue
** Meal Participation
* User has a section available where these stats are broken down by site and by date.

## Acceptance Criteria
* Site dropdown added
** Multi-select
** Format: Sitecode - Sitename
* Selected sites saved by user - either a feature to Save the default, or a cookie to store them locally
* Date options added
** Default: Today
** User can pick other single dates
** User can pick a date range
** User can pick a whole week
** User can pick a whole month

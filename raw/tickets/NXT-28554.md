# NXT-28554 — Adult & Service Employee Import – Exceptions Tab

## Description
As a SchoolCafé Admin I want view the exceptions for Adult Import and for the Service Employee Import so that I can quickly find any import failures.

*Flow:*

* User logs in and browses to System > Imports & Exports > Import Monitor
* User clicks view log of and Adult Import and taken to view log screen
* User clicks the Exceptions tab
* User views exceptions

*Notes:*

* We are doing this exactly the way we did exceptions for Student Import, so copy everything from there that you can
* Make changes to both the Adult Import and for the Service Employee Import

*Requirements*:

* For the import monitor tab and rows that are Adult Import and for the Service Employee Import, do the following for the View Log:
* Add a tab
** Title: Exceptions
* 
* Make it just like Student Import Exceptions tab
** Same everything, just pulling the Adult exceptions instead of Student exceptions
* 
* Ensure that the Exceptions Count has correct count in the Summary section at top
* 
* Ensure grid has sorting (doesn’t work in Student Import)
* Ensure grid has export to excel (missing in Student Import)
* For pagination, add 50 and 100 as options (missing in Student Import)

## Acceptance Criteria
* Tab exists with right title and UX
* Grid shows all the exceptions and nothing else with the required columns
* Grid has pagination, sorting and export to excel
* Exceptions at top has correct count

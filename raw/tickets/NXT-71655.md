# NXT-71655 — IM - Configuration - Multi-Tenant Settings

## Description
As a central level user I want a configuration page for multi-tenancy settings so I can manage the data source we use and how local users will interact with them.

## Acceptance Criteria
* Create settings to manage how data sources will be used
** ex: local users *can* / *cannot* create items in the local data source
** ex: local users must use state-level Item Categories

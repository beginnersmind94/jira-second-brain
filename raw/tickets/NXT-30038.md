# NXT-30038 — IM - Items - Create New Item - Images (Optional Step 2) - UI Only

## Description
As a District Admin I want the option to provide general Product Info about an item I am creating so I can populate important attributes.

*Requirements*

* Upon click of Save/Next on [Initial Info|https://primeroedgenext.atlassian.net/browse/NXT-30036], load the image control to allow user to add images, along with the additional information from step two from story [https://primeroedgenext.atlassian.net/browse/NXT-30037|https://primeroedgenext.atlassian.net/browse/NXT-30037|smart-link]

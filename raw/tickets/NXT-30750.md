# NXT-30750 — User Import - Update Existing Users

## Description
The User Import process needs to account for updating users if the import file contains users that already exist.

## Acceptance Criteria
* When a user that is imported already exists within SchoolCafe, the SchoolCafe user record shall be updated with the data from the import file.

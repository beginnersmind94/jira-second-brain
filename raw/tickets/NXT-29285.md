# NXT-29285 — INV - Transfers - Transfer Detail - History tab

## Description
As an Inventory user I want a History tab on my Transfer Detail pages so I can view transfer history just like Additions, Withdrawals, etc.

*Requirements*

* Add the History tab to Transfer Detail page
* Show historical transactions
** including ‘Transfer Approved', etc.
*** Transfer Approved, for example, would not have a View icon
* Allow viewing historical transaction details for relevant transactions
** Transfer Adjusted would have a view icon to see specific changes

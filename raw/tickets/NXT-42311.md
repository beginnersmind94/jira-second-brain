# NXT-42311 — IM - Recipes & Recipe Details - Header bar search

## Description
As a menu planner I want a header bar search within the Recipes section so I can quickly navigate to a specific recipe without using the filter list page.

## Acceptance Criteria
* Provide a header bar search for Recipes, in the same way currently provided for Items
* Search should be accessible on the navigable Recipes list page
* Search should also be accessible on the Recipe Details page
* Search results should return both multi-item and single item recipes
* Single item recipes should open on the Item Details page (*revisit!*)

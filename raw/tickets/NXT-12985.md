# NXT-12985 — INV - Order Template List - Add New Template

## Description
As an Inventory Operations Manager I want an Add feature on the Order Template List so I can type a new name and create a new Order Template.

This is a continuation of [https://primeroedgenext.atlassian.net/browse/NXT-15619|https://primeroedgenext.atlassian.net/browse/NXT-15619|smart-link]

*Requirements*

* Provide a header section on the Order Templates List
* Header section will contain:
** a text field to type a new Template Name
*** Uniqueness validation: “Template Name is already in use. Please enter a different Template Name.”
*** Unique across entire district, case insensitive 
** an Add button to add (create) the new template
*** immediately navigate the user to the Order Template Details ([NXT-14810|https://primeroedgenext.atlassian.net/browse/NXT-14810]) once template is successfully created

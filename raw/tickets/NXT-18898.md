# NXT-18898 — INV - Manage Orders - Receipts tab - Enhancements

## Description
As an Inventory user I want a more flexible and accurate Receipts list so I can view only the receipts I need.

*Requirements*

Make the following changes on the Receipts tab

+Header+:

* Change Vendor dropdown
** Make this multi-select dropdown the same style as Site dropdown (show checkboxes)
** Change the width of this dropdown to be the same as others
* Change ‘Show Partial Receipts’ checkbox
** Change to a dropdown with a label of ‘Display’
** Dropdown options:
*** All Orders (default)
*** Only fully received orders
*** Only partially received orders
** Display this dropdown after Vendor dropdown
* Add a ‘Show Quick Receipts’ checkbox (replace Partial Receipts checkbox location)
** Default to checked
** This will filter according to the value for ‘Entry Method’
*** if this is unchecked it will not include orders/receipts with ‘Quick Receipt’ entry method

+In the list:+

* Change ‘Order Code’ to ‘Order #’
* Remove time from date columns
** Receipt Date
** Delivery Date
** Resize column width accordingly
** Show full date and time in hover tooltip
* Display Discrepancy value as absolute value (always positive)

## Acceptance Criteria
* Make changes to the Manage Orders > Receipts tab

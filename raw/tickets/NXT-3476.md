# NXT-3476 — **Inspection Template - Multiple Choice Question Type - Dropdown

## Description
*User Story*
As an Inspections user, I can add a Multiple Choice question to a template when creating or editing the template.

*Mockup*
Single Choice – Dropdown - [https://nqyc9v.axshare.com/start.html#p=single_choice_-_dropdown&id=kfb0rt|https://nqyc9v.axshare.com/start.html#p=single_choice_-_dropdown&id=kfb0rt]

*Requirements*

* Multiple Choice can be selected as a Question Type when adding a question in a Template
** Not available for Pass/Fail scoring type
* There is an option for type of display
** -Radio Buttons (One Choice Allowed)-
** Dropdown List (One Choice Allowed)
** -Checkboxes (Multiple Choices Allowed)-
* When selected, users can add options to the question
** There is no hard limit to the number of options available
** -Options can have media added to them-
** -Options can be dragged and dropped within the question to reorder-
** Options can be deleted
* -When Custom Scoring is selected, users can add point values to each option-
** -Point values must be whole numbers greater than or equal to zero-

## Acceptance Criteria
* Multiple Choice questions of all three types can be added to templates and saved

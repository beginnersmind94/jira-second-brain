# NXT-22665 — IM - Data Exchange - Upload Image API for Item

## Description
Need to create a new API in FileStorageController and the API URL is *api/FileStorage/Upload/ItemPhoto*
API needs to be created to upload more than one image.
It will upload the original image as well as the resized image of the original image uploaded.
The resized image will be of size 200x200 px (taking into account the Aspect Ratio).
Azure Blob Container name will be *itemphoto*.
UI needs to pass *ItemId* in the API header.

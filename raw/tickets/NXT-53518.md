# NXT-53518 — VND - Family Hub - Editing of Security PIN

## Description
As Anna the Family Hub Account Holder I want to see and edit the security PIN for the accounts in my household.

*Requirements*

* Add two District Settings:
** Name: Allow viewing of Security PIN
*** Default: Off
*** Description: This will show the Security PIN in the Student/Account section of the Dashboard. This PIN is typically used for Vending.
** Name: Allow editing of Security PIN
*** Default: Off
*** Description: This will allow the Security PIN to be edited by the household.
* User can view the security pin tied to an account if the first setting is on.
* User can edit the security pin tied to an account if the second setting is on.
** This should update in Account Management.

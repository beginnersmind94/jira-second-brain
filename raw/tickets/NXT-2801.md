# NXT-2801 — Inspection Template - Scale Question Type

## Description
*User Story*
As an Inspections user, I can add a question to a template allowing the user to give a rating over a given range of values.

*Mockup*
[https://nqyc9v.axshare.com/start.html#p=rating_scale&id=fa7d0e|https://nqyc9v.axshare.com/start.html#p=rating_scale&id=fa7d0e]
Click Add Question

*Requirements*

* Scale can be selected as a Question Type when creating or editing a template
** Not available for Pass/Fail scoring type
* User can set the range for the Scale
** 0 is the minimum
*** 1 is the default minimum
** 10 is the maximum
* User can set whether to have labels at each end of the scale
* If scoring type is Custom Scoring, user can choose to have the scale to apply to the score
** The rating given by the Inspector is added to the score of the inspection, with the highest rating possible corresponding to the highest score possible.

## Acceptance Criteria
* Rating questions can be added to templates

# NXT-45502 — 1WS Item Import Script (Pack Size Changes)

## Description
_(none)_

## Acceptance Criteria
Pack Size Criteria:

*Single Tier* - GTIN which does not have Parent or Child (Ex: Case or Each) - regression

*2 Tier*- a GTIN which has either Parent or Child (Ex: Case - Each or Pack - Each, or Case - Pack, or Pallet-Case)- 

* If given GTIN has a child and if Base Unit = True, then child GTIN should be treated as lowest UNIT
* If given GTIN has a parent and if given GTIN is base unit, the given GTIN should be treated as lowest unit and parent should be treated as Unit 2 

*3 Tier*- a GTIN which has both Parent or Child (Ex: Case - Pack - Each or Pallet - Case - Each)

* Identify the lowest GTIN by comparing the given GTIN, child GTIN and Parent GTIN where Is Base Unit=True and populate the lowest unit GTIN, Description, unit weight and unit weight unit of measure  
* Identify the Parent of the Lowest GTIN and treat as Unit 2 and populate Unit Description, GTIN and Sub Quantity
* Identify the Parent GTIN of Unit 2 GTIN as Unit 3 populate Unit Description, GTIN and Sub Quantity

# NXT-3851 — AcctMgt - Homerooms - Add

## Description
*User Story*

As a district administrator I want to add a homeroom.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=homeroom&id=spdi2j&c=1|https://nqyc9v.axshare.com/start.html#p=homeroom&id=spdi2j&c=1]

*Requirements*

* Add an ‘Add +’ button to the Homerooms split view.
* Clicking this will load a sidebar where the user can set the Homeroom Name, Description, and Site.
** Blank text fields for Homeroom Name and Homeroom Description.
*** Description is optional, 250 character limit.
** Site is a drop-down of active sites at the district.
** Homeroom Name must be validated - each homeroom name is unique per site.
** Save / Cancel buttons.
*** Success message if saved successfully.
*** Search results, if any, should be removed so that the newly created homeroom is visible in the search results.
*** The new homeroom will be the top result.
*** The new homeroom will have a little chip/tag at the top which says ‘NEW’ per the wireframe.
**** This chip/tag should not display on any other homeroom, only the freshly created one.
*** Right-side of the split view should load with the freshly created homeroom.



Discussion Notes (7/20/20)

* Team discussed scenarios where the user may have searched prior to creating a new homeroom.
** User may have searched for ‘Add’ and sees all homerooms in the results containing ‘add’.
** User then adds a homeroom called ‘Subtract.’
** User should not see the new homeroom in the list view, as it is still filtered to homerooms containing ‘add’.

Discussion (8/18/20)

* Adding Grade -in as an option- for homerooms, with a 1:many approach.
** Example: Grade is allocated to Site A. Site A has grades PK-5. User can multi-select from: PK, KG, 1, 2, 3, 4, 5.
** Required field, not optional.

## Acceptance Criteria
Add button available.
User can add a homeroom.
Homeroom names must be unique per site.
Description field is optional.
Grade is required.
Search results clear when a new homeroom is added.
New homeroom has a 'new' tag/chip on it and shows as the 1st result.
Multi-language support added.

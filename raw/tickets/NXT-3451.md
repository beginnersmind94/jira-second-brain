# NXT-3451 — System - Notification Templates - Add Letter Template History as a tab

## Description
As a district admin, I would like to view and restore previous versions of my letter templates, so that I can easily make corrections if needed.

Summary:

* Allow users to select a History button related to each template which, when selected, allows them to view previous versions of the template and overwrite the current version with an older one.

Details:

* In the *Templates* page, add a History button above any template that is selected.
* When this button is selected, display the last 5 versions of the template with summary details, sorted in descending order from most recent to oldest.
* When a version is selected, populate the template editor with the details of that previous version.
* The user should then be able to save/save as the template with the details of the previous version.
* The Template History should then update as well.
* However, if the user does not explicitly click Save, no template/history details should be updated.

## Acceptance Criteria
Verify that a History button is available for every template

Verify that, when selected, the button shows the last 5 versions of the template, sorted in descending order from most recent. Add a 'Show More' button to display the older templates.

Verify that when selected, the details of the previous version populate the template editor window

Verify that when saved, the template details and history are now updated and the process can be repeated

Verify that if the user does not explicitly click Save, then the template and history are not updated

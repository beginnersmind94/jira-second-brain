# NXT-35810 — Onboarding - Add grades to Site Types seed data

## Description
As a Cybersoft administrator, I would like to automatically set default grades along with the default site types we create for a district, so that users do not have to go in and manually add grades (which will probably be the same as ours the majority of the time). 

*Mockups*

* N/A *(No UI required)*

*Requirements:*

Since we send Site Types as part of our seed data but leave Grades blank, please add the following default grades for the default School Site Types:

# Elementary School: PK, KG, 1, 2, 3, 4, 5
# Middle School: 6, 7, 8
# High School: 9, 10, 11, 12

!Screen Shot 2023-01-06 at 12.33.16 PM.png|width=1103,height=483!

Also, create the following default Grade Groups and send that as seed data as well:

# *PK-5:* PK, KG, 1, 2, 3, 4, 5
# *6-8:* 6, 7, 8
# *9-12:* 9, 10, 11, 12

!Screen Shot 2023-01-06 at 12.32.29 PM.png|width=1104,height=307!

When this is complete, when a new district is created and the user goes through the Create Sites task in Onboarding, they will see these default site type grades and grade groups. Then, the user will be able to quickly verify and make changes rather than adding them from scratch.

## Acceptance Criteria
Verify that these seed-related changes have been made so that the Site Configuration page shows these default grades and grade groups during the Onboarding process for a new district.

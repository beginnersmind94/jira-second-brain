# NXT-58549 — Elig - Reviews - Statistical Sample Generator Config

## Description
As Dana the Director I want to setup parameters for my Administrative Review with regards to the review of FRE Applications. I need a way to setup those parameters and trigger application tagging.



As part of the audit, the district sends a list of students (an Eligibility Roster) to the state, who respond with whether they’re doing a 95% or 99% sample, along with a starting point and a review interval.

We want to let the district/auditor enter this information so we can generate an eligibility list, and a selected list, along with granting the auditor access to review the application/DC information.

## Acceptance Criteria
* User has a config screen, can store values by academic year:
** Benefits/Roster Report Date
** Number of Free/Reduced-Price Eligible Students
** 95% or 99% Confidence
** Number of Eligible Students' Determinations that must be reviewed
** Random starting point
** Review interval

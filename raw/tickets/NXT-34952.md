# NXT-34952 — Manage Views Component - Empty View Name should not be Allowed

## Description
The “Manage Views” component allows the user to save a view name as all spaces.

## Acceptance Criteria
* Names with spaces shall not be allowed [No leading spaces, no trailing space. No spaces between words.]
* Only alphanumeric characters and a hyphen (“-”) can be typed by the user and used in a view name.
* The maximum length of a view name should be 50 characters

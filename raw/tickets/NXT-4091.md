# NXT-4091 — Item Management - Split View - Display card details

## Description
*User Story*

As a Item Management User, I should have access to view the display cards on the left panel of the split view.

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_super_user&id=n0hwa7&g=1]

!image-20200526-125646.png|width=384,height=312!



*Requirements*

Display card format:

*Item Name |* *-Short Name-* *| Item ID# | GTIN# | Product Code# | Datasource*

If the data is not available, then just display the information as a ' - ' to keep consistent

*Item Name | Item ID | Datasource*

When selecting the item, it needs to be highlighted / targeted and show the information

## Acceptance Criteria
Item information from creation needs to show up here.

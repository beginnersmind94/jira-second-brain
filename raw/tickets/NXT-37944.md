# NXT-37944 — System Settings – Update Modules and Settings

## Description
We need to go through system settings and remove unwanted settings so that the end users are not confused.

*Notes:*

* After River Road onboarding, we received this feedback:
* Dallas: take a look at system settings with your dev leads and see if we can remove all of those legacy/unused settings. there's been some confusion as we've onboarded some customers about what settings are active/used vs not.

 

*Requirements*:

* Do the following for the page: System > Configuration
* *Add Modules*
* Add 2 new Modules to the modules list in the Module list
** Values:
** Item Management
** POS
* 
* *Remove Modules*
* Remove from the modules list the following values:
** Bid Analysis
** Financials
** Food Distribution
** Inspections
** Program Management
** Reviews
** Team Work
** Thermo Track
* 
* *Remove Records*
* Remove from the list of settings all the records that have a Module value in one of the following:
** Bid Analysis
** Financials
** Food Distribution
** Program Management
** Reviews
** Team Work
** Thermo Track
* Example: Bid Analysis has 2 settings (Codes MNURTNGS and VENDORRESP). You will remove these 2 records so that they do not show up in the list
* 
* *Other*
* For module list, sort alphabetically

## Acceptance Criteria
* Unused Two new modules are added
* Listed Modules are removed along with all their settings/records

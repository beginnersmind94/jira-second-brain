# NXT-1921 — MP - Menu Costing

## Description
As a Food Service Director, one of my concerns is the potential cost of food that I’m using in my district.

There should be a method of accessing and viewing this information on the Menu page, as well as the ability to print a report with the information, should the user desire.

This should give me a list of the menu items that are contained in my menu, the planned quantities, the per serving cost, and the total cost. This information will be based on the product/information costing data from the Item Management Details page.

You will see that the report gives the cost for the individual serving, then multiplies it by the number of serving that have been entered on the menu page. 

(If no costing information is available or “compatible” for a menu item, then it’s expected that $0.00 would be the result, and it is up to the district to remedy those occurrences by entering or correcting data.)

“Standard Price” will be the pricing criteria for the information pulling from Item Management in regards to costing.

The pricing information comes from Item Management - using the recipe calculations, then how many servings of each individual menu item, add together the total cost for each menu item category, then also divide by total meal count to get the “Food Cost per Meal” for each section, then the Total Food Cost at the bottom of the page.

If there are multiple serving sizes, those will be displayed on separate lines with their appropriate costing details.

## Acceptance Criteria
As a Food Service Director, I will be able to select a button or icon on the menu page, and get the full costing information based on the criteria I’ve entered on my items and menu.

Gray out the ($) button for cost if counts have not been saved. Only allow for the button to be enabled if all information on the page is current and saved. 

Display 2 or more separate rows if there are multiple serving sizes. Each serving size will have its own weight and cost association as well as count. 

The pricing information comes from Item Management - using the recipe calculations, then how many servings of each individual menu item, add together the total cost for each menu item category, then also divide by total meal count to get the “Food Cost per Meal” for each section, then the Total Food Cost at the bottom of the page.

(Note: Break Costing down to Cost per Unit based on gram weight)

Add Meal Count (projected meal count)  on the top of the report with the other criteria

Grand Total Row= All category totals for Unit Cost column. Sum all purchase cost for all items  for Purchase cost column 

Disregard Variety Menu item when it comes to menu costing/menu cycle costing if no cost is available or there is a null value

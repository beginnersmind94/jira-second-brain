# NXT-5356 — Workspace - Top Used Dashboards and Reports

## Description
*User Story*
As a PrimeroEdge User, I can see my top 5 most used Dashboards and Reports from the Insights module under the “Top used Dashboards & Reports” header.

*Mockup*
[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down-width&hide-ui=1|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=724%3A21152&scaling=scale-down-width&hide-ui=1]

*Requirements*

* There are two tabs, one for Dashboards and one for Reports
* Listed by name
** Up to 10 are displayed for each one
** If Dashboard or Report applies to a particular module, the module icon is displayed by it
* Clicking on the listed item will navigate to the Dashboard or Report in the Insights module within the same browser window.

## Acceptance Criteria
- Top Used Dashboards & Reports section is present in the workspace
- Dashboards and reports are appearing properly
- Functionality present as described in requirements

# NXT-156 — Accountability - Reports - Meal Participation

## Description
As a district admin I want to see what our average participation rate is for sites across the district, and the district as a whole.

PrimeroEdge report:

!image-20201204-000143.png|width=1723,height=910!

!image-20201204-000252.png|width=1708,height=1414!

Requirements

* User can select the following inputs:
** Site: single, multiple, entire district ('All')
** Site type: Elementary, Middle, High (this is likely configured, not hard coded)
** Dates: single date, range
** Meal: any configured meal for the district, do not restrict to just breakfast / lunch / snack like PrimeroEdge does.

* User is shown the the average daily participation (meal counts / percentages) broken down by:
** Site
** Site Type (Elementary, Middle, High, etc. for comparison)
** Date
** Meal
** Patron Type (adult or student)
** Eligibility (Free, Reduced, Paid, Employee, Other)
* User is also shown totals for these respective counts.

## Acceptance Criteria
User can make selections per the requirements.
User has a report that shows them the selected criteria.
User can see the attendance factor and enrollment counts for the selected date(s), typically going by the highest count for each eligibility (free/reduced/paid)
User can select whether they want to see a summary of data or drill down to a detailed analysis date over date.
User can see the meal counts and the percentages broken out accordingly.

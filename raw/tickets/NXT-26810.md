# NXT-26810 — IM - Item History (Contributions)

## Description
As Item Manager administrator I need an option where I can observe the history of a item' s  Contribution Details, for example updates made to the item by who and when so that have an audit process

## Acceptance Criteria
*Entry Point*

* Visible at the Item detail history
* An option to view item’s Contribution Details  history shall be available to the user at item  history icon 
* User shall be able to see tall item’s Contribution Details  property values that were updated
* System date and Time on which the update took place
* User name which performed the update (who performed the update)
* What fields that were updated, preview value(s) and new value(s)
* The information is for display purposes (Shall not be alter by the user)
* Applies to ADD and EDIT item

*General*

* Page pagination shall be present (as standard way)

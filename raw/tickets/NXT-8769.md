# NXT-8769 — Claims - Claim Detail - CACFP Detail View - [RTG]

## Description
*Story*

As a user I want to see the reimbursable meal counts by site for each of the claims generated.

*Mockup*

[^CACFP DistrictClaimBySite (3).pdf]

*Requirements:*

* Format the display to show the programs, meal counts and amounts for reimbursement by site
* Subtotal the sites
* Provide totals for all sites

## Acceptance Criteria
Depending on the claim type the data for each site is shown in the required format for the claim data created for CACFP.
There is a column for each type of program participation. 
There are 6 possible columns but only 5 are possible for a claim.
The 6 possible columns for CACFP are Lunch, Breakfast, Snack (AM), Snack (PM), Snack Night, and Supper. 
If a site does not participate in a program the column entries are left blank.
Each site has
Meal counts for each Program the site participates in for each meal count.
Each meal count is multiplied times the appropriate rate resulting in a claim amount.
There are subtotals per program participation by site.
There are totals per eligibility type by site

There are grand totals per column.

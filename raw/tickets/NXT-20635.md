# NXT-20635 — Platform - BO - Onboarding System District Operations Details Page

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view District Operations detailed page with its overall progress and take necessary actions

+*Mockup:*+

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=scale-down-width&page-id=548%3A8289&starting-point-node-id=637%3A9929|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=scale-down-width&page-id=548%3A8289&starting-point-node-id=637%3A9929|smart-link] 

+*Requirements*+:

*Progress Calculation*

·        Each row is 20% aggregates to 100%

*Behavior:*

* District Operations should display table with *Columns* displayed as:
o   Activities
o   Status
o   No of Completion
o   Actions
* District Operations table has available actions listed below
** Remind Me
** View
** *Activities* should include
o   Sites Configuration
o   Program Configuration
o   Roles Configuration
o   Users Configuration
** *Each* Activity must be follow the precedence
*** Without sites, program configuration should be greyed out and vice versa for roles, users
** Once Program configuration is partially complete (atleast one program) Roles should be enabled for configuration
** VIEW should open that config page for the respective activity in a pop-up “*Configuration*” screen of school café
** Program Configuration screen when the user clicks on VIEW button of Program Setup row
** Each pop-up for the VIEW should contain CLOSE button for user to return to Onboarding dashboard
** System provides “Remind Me” action to notify user on scheduled time in a pop-up with data/time

## Acceptance Criteria
* User shall be able to view list of pending activities under District Operations table
* User shall be able to click on VIEW to bring up respective configuration screen
** ex: when user clicks on Program Configuration, system shall bring program configuration screen in SchoolCafe
* User shall be able to close the details page and return to Onboarding dashboard by clicking on CLOSE button
* User shall be able to use “Remind Me” action to notify user on scheduled time.

# NXT-6025 — Application Entry - Guided Process (4. Contact Info)

## Description
*User Story* 

Admin is guided to enter their contact information.

*Mock Ups* 

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1306%3A27072&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=1306%3A27072&scaling=min-zoom]

*Requirements* 

* Applicant Contact Information 
** Entry Fields: First Name, Last Name, Email, Phone, Street Address, City, State, Zip Code
*** -State field - prepopulate with district’s state for convenience as default (-removed, as we dont want the recurring possibility of only the state appearing in the contact info section) 
** ***Contact information is not required for the applicant/application so it’s possible no information is provided during this step 
*** can go to the next step without entering anything in the fields
* Ability to go to previous step and next step

## Acceptance Criteria
- applicant is guided to next step 
- applicant has option to go to previous step 
- applicant can add contact info
- applicant can edit contact info from this step as needed
- Multi-language, ADA compliant.

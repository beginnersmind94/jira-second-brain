# NXT-21405 — INV - Vendor Detail - Info tab - Edit 

## Description
As an Inventory manager I want an Edit ability on the Vendor Info tab so I can make changes to the main vendor attributes.

*Requirements*

* On the new Vendor Info tab ([NXT-6795|https://primeroedgenext.atlassian.net/browse/NXT-6795]), on click of Edit icon:
** Enable all editable fields in place of labels
** Replace Edit icon with Save and Cancel buttons
* On click of Save or Cancel:
** Turn everything back to labels
** Display updated values
** Replace Save/Cancel buttons with the Edit icon
* Show the labels in this order, and as the following types during edit mode
** Vendor Name - text field - *mandatory, unique*
** Vendor # - text field - *mandatory, unique*
** Data Source - not editable (but would be a dropdown)
** Vendor Type - dropdown - *mandatory*
** Preferred Ordering Mode - dropdown
** Vendor Lead Time - numeric text field - *mandatory*
** Include Weekends for Lead Time - checkbox
** Production Order Lead Time - numeric text field - *mandatory*
** Comments - text field/area
** **Note*: hide or remove the ‘Strategic Partner’ field
* Show in-line validation messages next to the field (on the right):
** Duplicate vendor name
** duplicate vendor #
** ….

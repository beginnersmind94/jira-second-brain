# NXT-54660 — PEI - Auth - Add Insights to Roles Permissions in PE

## Description
As a PrimeroEdge admin, I would like to restrict access to which user have access to Insights so that I don’t have users looking at information they shouldn’t

*Requirements:*

* Add Insights as a module to the Roles Permissions grid with these values;
** Module: Insights
** Category: Front Office
** Checkbox: Command Center
* The default is unchecked for all roles except Director
** _Not sure if should be added to other roles_
* Ensure that roles can click or unclick these permissions
* Ensure that PE is able to query if a user in a role that has been granted access to this resource

* It can be applied and unapplied to new and existing roles
* PE code can query if a user is in a role with that permission enabled

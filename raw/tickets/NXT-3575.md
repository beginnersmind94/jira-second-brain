# NXT-3575 — System - Management - System Settings - Allow user to save setting

## Description
As a district or Cybersoft administrator, I would like to be able to easily manage settings related to the entire website in one location, so that I can quickly make the changes I want for PrimeroEdge to function as needed.

* In addition to viewing, allow the user to make changes (save setting)
* Save changes to the database
* Display a snacker message that setting has been updated successfully

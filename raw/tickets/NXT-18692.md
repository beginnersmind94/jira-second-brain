# NXT-18692 — Nutrition Report > Monthly - DEV

## Description
Once the options have been selected on the page, there are 3 different versions of the Monthly Nutrients Report. 

The different “versions” are based on the options chosen, and are impacted by the selection of “Expanded,” “Simplified,” and “All”

ALL of the views below should contain the name of the menu on each date, and a column for the “Planned Quantity” number that comes from the menu that was assigned to the calendar. 

Projected Meal Count must be listed for each Menu on the report.

If I select “All” - then ALL of the nutritional values available should be displayed.

If I select “Simplified,” then the following nutritional values should be displayed: Calories, Sat Fat, Trans Fat, Sodium

If I select “Expanded,” then the following nutritional values should be displayed: Calories, Total Fat, Sat Fat, Trans Fat, Sodium, Carb, Protein

## Acceptance Criteria
I’m expecting to be able to pull this report with specific parameters, and receive the information based on those selections.

ALL of the views will contain the name of the menu on each date, and a column for the “Planned Quantity” number that comes from the menu that was assigned to the calendar. 

Projected Meal Count must be listed for each Menu on the report.

If I select “All” - then ALL of the nutritional values available will be displayed next to the Menu Item Name and Planned Count

If I select “Simplified,” then the following nutritional values will be displayed next to the Menu Item Name and Planned Count: Calories, Sat Fat, Trans Fat, Sodium 

If I select “Expanded,” then the following nutritional values will be displayed next to the Menu Item Name and Planned Count: Calories, Total Fat, Sat Fat, Trans Fat, Sodium, Carb, Protein

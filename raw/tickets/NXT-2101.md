# NXT-2101 — Production - Reports - Multiple Recipe Report

## Description
As a site manager, I would would like to be able to access a way of printing out my recipes for multiple days at one time. I know there’s access to my recipes from within my production plan, but that only allows me to print them out for the single date that I’m on. I would like to access multiple days of recipes at one time, so that maybe I can get some of the prep-work done ahead of service dates.

## Acceptance Criteria
As a site manager, I would be able to access a way of printing out my recipes for multiple days at one time. I know there’s access to my recipes from within my production plan, but that only allows me to print them out for the single date that I’m on. I would like to access multiple days of recipes at one time, so that maybe I can get some of the prep-work done ahead of service dates. 

On the current page - I can select a date range, and the software will provide me with a list of the recipes that I have on each of those dates, as well as the planned counts that I have associated with each of those dates. It also allows me to filter based on menu item, recipe code, or category… thus allowing me to narrow down a grouping of recipes that I want. The checkboxes next to each menu item allow me to choose the recipes that I want, and I don’t have to pull the recipes that are not needed, such as cartons of milk, or individually packaged cereal, etc.

Once I select the “Generate Report” button, the recipes will all appear with their associated ingredients, instructions, etc.

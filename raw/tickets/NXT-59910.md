# NXT-59910 — INV - Vendors/Contract Items - Change default sort order

## Description
As a food service admin I want the Vendor and Contract Items page to show the most recently added/updated items at the top of the grid so I can see the items requiring additional data input.

Raju’s comment on predecessor story: [https://primeroedgenext.atlassian.net/browse/NXT-59009?focusedCommentId=168918|https://primeroedgenext.atlassian.net/browse/NXT-59009?focusedCommentId=168918|smart-link] 

{quote}Previously, we were sorting based on the Item Description, which is the same method we are using now. Users have the option to enable "Items missing mandatory attributes" to filter only items with missing attributes. 

As we discussed, We can implement a default sort based on the updated date / created date. So the items which are added or updated will always on the top{quote}

## Acceptance Criteria
* Modify default sort order of Vendor Items and Contract Items to show most recently added/updated at the top of the list
* Users can still change sort order; we are just changing the default

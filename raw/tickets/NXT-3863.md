# NXT-3863 — Add Item - Ingredient - Add Tags

## Description
*User Story*

As a Item Management Super User/Menu Planner/Food Service Director/Nutritionist, I can add tags while adding a new item

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1&g=1|https://nqyc9v.axshare.com/start.html#p=itemmanagement_-_ingredient&id=00ehm1&g=1]

*Requirements*

* Tags
** Optional field
** Allow user to input a free form text entry
*** After adding the text and clicking off of the record or hitting enter, the text is ‘saved’
*** When the text is saved, the tag will auto populate other tags for that instance
*** No default tags at this time
** No limits at this time
* Tags will be tied to the datasource
** PE (PrimeroEdge Tags)
** Global Tags (Global datasources)
** Shared Tags (Shared / Regional datasources)
** Local tags (Local datasources)

## Acceptance Criteria
* After the tag is ‘created’, must be able to click 'x' to delete the tag from the item
* No duplicate entries
* Common across all ingredients per datasource
* Provide auto suggestions
* Tags won't be saved in the database until 'saved'
* Alphanumeric with special characters allowed
* Max character limit as is today in the database

# NXT-3572 — Edit Inspections - Date/Time Question Type

## Description
*User Story*
As an Inspections user, I can see Date/Time Questions on the Edit Inspections screen and give responses, as well as save and edit any responses.

*Requirements*

* If a Template has a Date/Time Question Type as part of it, the question is present when an Inspection is created using the Template
* Question information is displayed correctly
** Question
** Question Help
** Date/Time Fields
** Notes
** Media Addition (Non-Functioning)
** Whether question is required
* Depending on the question’s settings, user has a date and/or time field
** Date fields are a calendar selector
** Time fields are manually entry that is numbers only
*** It is indicated whether it’s 12 or 24 hour
*** AM/PM selection available if it’s 12 hour
*** Time is validated to ensure it’s valid based on 12 or 24 hour selection
* Time and Date can both be saved
** If question is required, Inspection cannot be submitted if selection is empty
* Time and Date can both be edited'
* No scoring is associated with a Time/Date question

## Acceptance Criteria
* Time/Date question appears in Inspections
* Time and Date can be entered, saved, and edited

# NXT-35385 — Accountability - Reports- Account Balance - First Name & Last Name shows in the wrong order

## Description
Accountability → Reports → Account Balance : 

First and Last name are showing in the wrong order. Shows as “First, Last” when it should be “Last, First” (or “First Last” without the comma, but this is less preferable). Sorting is affected by using the first name instead of the expected last name. 

We don’t show name as this way anywhere else. PatronName should either be [FirstName LastName] or [LastName, FirstName].

# NXT-4640 — AcctMgt - ID Cards - Patron General Info

## Description
*User Story*

As a District Admin I want an easy way to generate an ID Card for a single patron via their account.

*Mock-up*

TBD

*Requirements*

* Populate the Quick Interactions dropdown (vertical three-dot box) with an option for ID Card.
** Selecting this option should show a preview of the patron’s ID card based on the active ID Cards template.
** There should be an option to print this card out that will indeed send it to a printer.

## Acceptance Criteria
Quick Interaction bar dropdown shows an ID Card option.
Clicking it shows a preview of the card based on the active template.
Fields populate data appropriately.
User can print the card.
Multi-language support enabled for the drop-down.

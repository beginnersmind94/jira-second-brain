# NXT-24011 — IM - Edible Yield Factor

## Description
As a Nutrition Dept. user I need to know what the edible yield factor of a food item is in order to more accurately calculate recipe cost.

!image-20211220-174347.png|width=1369,height=786!

## Acceptance Criteria
*Entry Point:*

* Item Detail Screen (List and split view)
* Applies to Food Items only

*Actions:*

ADD (Applies to List and Split View)

* When creating a new food item I can add the Edible Yield Factor  (EY%) to a Food Item
* This field is a % value
* This not a mandatory field
* Default value is 100%
* Acceptable values 1 to 100 %
* The field is a text box

EDIT(Applies to List and Split View)

* When editing  food item I can modify the Edible Yield Factor  (EY%) of a Food Item
* This field is a % value
* This not a mandatory field
* Default value is 100%
* Acceptable values 1 to 100 %
* Field has to have a number 
* Is not mandatory
* If user move away from the field and the field is blank the system should auto-populated with 100%

General

* This field should not be visible for none-food items.
* It does not affect recipe weight

# NXT-28467 — INV - Management - Contract Items - Show Commodity Contracts

## Description
As a Food Service Director I want Commodity Contracts to appear on Contract Items page so I can make adjustments and updates without going to the Vendors page. 

*Requirements*

* Add Commodity Contracts on the Contract Items screen
* Display extra columns for Commodity Contracts
** Current QOH
** Open Order Qty
** Available Quantity
* Add Adjust function 
** Enable the manual adjustment the same as the Vendors page
** Hide the Update button during Adjust mode
* Add the Report button for Commodity Contracts
** same behavior as dedicated Vendor page 
** allow in Adjust mode as well

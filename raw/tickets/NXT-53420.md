# NXT-53420 — DMF - Refresh grid every 30 seconds

## Description
!image-20240730-221451.png|width=1236,height=733,alt="image-20240730-221451.png"!

The refresh button will auto-refresh every 30 seconds when the grid page index is 1 and when the migration is in Queued or In-Progress (Running) status.
The tooltip text will be as follows and must be in 2 lines as shown.
'This page will auto-refresh every 
30 seconds when migration is running'
Accordion needs to be expanded only on the first page (PageIndex=1) of the grid when the migration is in Queued or In-Progress (Running) status.
On the rest of the grid pages, it need to be in collapsed state.

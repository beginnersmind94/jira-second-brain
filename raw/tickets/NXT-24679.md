# NXT-24679 — INV - Management - Contract Items - Update Vendor Item #

## Description
As a District Admin I want to be able to provide Vendor Item # for many items at once so I am not required to enter it each time I add an item to a contract.

*Requirements*

* On the Update Price screen ([NXT-23956|https://primeroedgenext.atlassian.net/browse/NXT-23956]) of the new Contract Items page 
** add another update method for providing/editing the Vendor Item #
** Redesign according to revised UX mockup
* Go into Edit mode for Vendor Item # 
** Enable the Vendor Item # field for all items in the list
*** move Vendor Item # column to the right side for editing
** Allow individual saves
** Provide a Save All function
** Provide an option to show *only* items missing Vendor Item #
* Enforce existing uniqueness validations
** show error messages *(use existing messages)*

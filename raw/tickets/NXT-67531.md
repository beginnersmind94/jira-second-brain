# NXT-67531 — IM - Bulk Item - Start Update Session

## Description
As a District Manager, once I have selected my local items to add to update, I need to be able to get those items and start a session and start the update process.

!image-20251028-151614.png|width=278,alt="image-20251028-151614.png"!

!image-20251028-151637.png|width=278,alt="image-20251028-151637.png"!

## Acceptance Criteria
* Once local items have been added to the repository to start a session, the button “Preview Session” along with how many items have been selected will be available
** IF there are still items that have been selected that have not been added to be included in the session, the “Preview Session” button will be disabled and the “Add Items” button will be enabled for the user to finish adding the items 
* Preview Session will open a slideout that will allow for last minute review of all items selected
** Users can delete items from the list and also cancel the entire session 
*** Provide Message “Are you sure you want to delete “Item Name”? (Single Delete)
*** Provide Message “Are you sure you want to cancel and remove all items?” (Session Cancel)
* Land items on the overview page with all sections enabled 
* At this point, the session will behave like an onboarding session

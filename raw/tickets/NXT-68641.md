# NXT-68641 — Fin - Journal Report – Location Selection, Export Generation, and Recent Journal Reports

## Description
As a *District Finance User*, I need to generate a Journal export for one or more locations (All, District-level, or specific sites). All locations in case of multi-selection will be within one worksheet in the Excel file.

## Acceptance Criteria
* Tooltip “You’ll be notified once your file is ready and can download it from the Document Center.”
* Standard Multi-selection dropdown with search feature 
* From (Posted Date): Standard date picker labeled “From (Posted Date)”.
* To (Posted Date): Standard date picker labeled “To (Posted Date)”.
* Export info tooltip: “We’ll notify you when your document is ready. You’ll find it in the Document Center.”
* Multi-location “+X more” tooltip: On hover over “+X more”, show the remaining locations in a simple list, one per line (e.g., Adams Elementary / Jefferson HS / Central Office / Sunrise MS).
* Tooltip to indicate that all selected locations will be within one worksheet in that workbook.
* If no records are available, indicate in export that “No records were found”

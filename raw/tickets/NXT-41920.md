# NXT-41920 — DataExchange Import – Fixed Width – Configure Database

## Description
As a SchoolCafé Admin I want to import a fixed width file so that I can use my state’s existing format.

*Notes:*

* This is the backend story to setup the database / backend to handle storing fixed width configurations
* It is needed for the DC Import front end stories

*Requirements*:

* Add to the database a way to store fixed width file formats
* 
* At a minimum, you will need to store a name for the file format and ensure it is linked to an Import 
* You will also need a way to store a list of column names and their length, character type, order and notes and link this to the file format
* 
* This data will be used to convert the file into delimited

## Acceptance Criteria
* We are able to store fixed width file formats to the database
* The fixed width file formats can be used by the import routine to convert files to delimited files

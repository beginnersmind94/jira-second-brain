# NXT-15379 — INV - Orders - Order Detail - Receipts tab - Edit Receipt Details - Load Details + Header Validations

## Description
As an operations manager I want the ability to edit a receipt header so I can make any changes or updates to the header info (Invoice #, Invoice Amount, etc.) of a previously created receipt.

This is for only loading the editable header section of a receipt. There are separate stories for editing items ([NXT-16997|https://primeroedgenext.atlassian.net/browse/NXT-16997]), Charges & Discounts ([NXT-16998|https://primeroedgenext.atlassian.net/browse/NXT-16998]) and saving the receipt ([NXT-17418|https://primeroedgenext.atlassian.net/browse/NXT-17418])

Note: this functionality is dependent on the user having the ‘Edit Receipt’ permission (separate story)

**Note:* this story includes the API to load *all* receipt details

*Requirements*

* On the Receipt Details screen, when the user edits a receipt, enable the editable fields of the receipt header
* When user clicks the Edit button, the following receipt header fields will be editable
** Order Delivered By
** Invoice #
*** mandatory
*** must be unique within district/region (for now)
**** validate uniqueness on Save (as part of NXT-17418)
*** restrict to 20 characters or fewer
** Invoice Amount
*** mandatory
** Comments
*** add character limit indicator
*** keep text area fixed size
** Note: Transaction Date/Time is not editable
*** the date that the edit is being made will be saved but not will not replace this field
*** this field will display the original date/time the receipt was first created

Mockup Notes:

* do not show Transaction Date/Time as editable

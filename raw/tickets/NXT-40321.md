# NXT-40321 — eOrders – View eOrder File

## Description
As a SchoolCafé user, I would like to view the eOrder file I generated so that I can verify my data.

*Notes:*

* This is a continuation of the eOrders – Dashboard story

*Flow:*

* Open SchoolCafé and browse to Inventory > Orders > eOrders
* View list of eOrders
* User clicks View File button and original file open for them

*Requirements*:

* These requirements are for the eOrders Dashboard page
* 
* In the grid there are rows with an action column
* If not already there, add the View File button to the action column
* 
* Link the file that was generated by the eOrder Framework to the button
* On click, open the file
* If no file, show the user an error message
* 
* Security - No additional requirements

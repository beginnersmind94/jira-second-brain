# NXT-10157 — Orders - Label Configuration Label Content

## Description
As a user I need to be able to format the content of a label or ticket.

see [www.labelgrid.net|http://www.labelgrid.net]

*Mockup:*

Labelgrid Example: using 7x2 standard label template

!Screen Shot 2021-01-20 at 2.09.26 PM.png|width=407,height=306!

!Screen Shot 2021-01-20 at 2.26.07 PM.png|width=665,height=263!



*Requirements:*

* The user will be able to format the content of a label using the page template.
* Label fields can be selected and placed on the label representation.

## Acceptance Criteria
Under Extra a function Order Label Configuration can be selected.
The user can select a template
The user can select label content from the list.
An edit window allows the label content fonts and sizes to be selected.

The label content can be 
Allergens, Delivery Contact, Delivery Location, Delivery Method, Delivery Service Provider, Delivery Time, Items, Menu Item ID, Note, Ordered Date/Time,
Order Number/Barcode, Patron ID/Barcode, Patron Name, Prepared Date/Time, Price, QR Code, Enrollment Site, Fulfillment Site, Serving Location, Serving Time, Serving Zone, Serving Period, Pickup Date, Pickup Site, Pickup Window, Pickup Instructions, Classroom Name, Homeroom Name, Bundle Number/Barcode, Fulfill on Date, Service For.

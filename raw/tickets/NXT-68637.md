# NXT-68637 — Fin - General Ledger Report – Account Selection, Export Generation

## Description
As a District Finance User,  I need to select a group of ledger accounts, choose a posted date range, and generate an Excel export containing account-level worksheets.

## Acceptance Criteria
* Tooltip says “You’ll be notified once your file is ready and can download it from the Document Center.”
* Account selection uses a standard multi-select with a search bar.
* Search filters accounts by code or name after 2+ characters.
* Selected accounts appear as removable chips (code + name + “×”), with a “Clear All” action and a count label like “SELECTED (4)”.
* Search results are shown as a list
* Date range fields “From (Posted Date)” and “To (Posted Date)” use the standard date picker and are required.
* Always show the processing notice: “We’ll notify you when your document is ready. You’ll find it in the Document Center.”
* On generating, give an informative note: -“Each selected account will be exported to its own worksheet within a single Excel workbook.”-
** The success message should be “Each selected account will be exported as a separate Excel file.”
* If no records are available, indicate in export that “No records were found” 

Updates:

Put the text that says “each selected account…” above the accounts drop down. once the report is generated, display the text in the green text box you showed for consistency.

# NXT-53337 — CC - Site and Date Selections

## Description
As Andy the Area Manager I have 6 sites to manage and want to see data on the Command Center filtered just for those sites and not the rest. This will save me time and allow me to be more efficient. 

*requirements*

* User has a multi-select dropdown for sites (Sitecode - Sitename) 
** User can see how many sites they have selected 
* User can save the selected sites as their default selection
** Alternative: Can we store the selected sites as a cookie so they’re automatically selected when the user returns to the page? 
* User can set their date ranges
** Default: Today
** Can pick by week
** Can pick by month
* User has a toggle between a Focus View (List) and a Calendar View - default is Focus View

## Acceptance Criteria
* Site dropdown added
** Multi-select
** Format: Sitecode - Sitename
* Selected sites saved by user - either a feature to Save the default, or a cookie to store them locally
* Date options added
** Default: Today
** User can pick other single dates
** User can pick a date range
** User can pick a whole week
** User can pick a whole month

# NXT-1724 — INV - E-Orders - Download E-Order file

## Description
As an Inventory manager I want to be able to download the file for any generated E-Orders, in case I need to manually send the file or download for my own records.

!image-20240425-140715.png|width=91.66666666666667%,alt="image-20240425-140715.png"!

## Acceptance Criteria
* Use the download icon on the E-Orders page to download the generated file for a selected E-Order
* Manage E-Order Status accordingly based on platform status
** If it will take time for file to be submitted and sent to vendor, show *Processing…* status 
** Once file is confirmed as successfully sent to vendor, show *Sent*
** If file fails to send successfully…
*** or show *Retrying…* if System is retrying to send file
**** if Platform team can enable this
*** show {color:#ff5630}*Failed*{color} if the file cannot be sent to vendor

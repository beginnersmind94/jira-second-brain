# NXT-15924 — INV - Order Template Details - Items tab - Add Items to Template

## Description
As an Operations Manager I want select checkboxes for items along with an Add button so I can select items and add them to a specific Order Template.

This is a continuation of [https://primeroedgenext.atlassian.net/browse/NXT-14811|https://primeroedgenext.atlassian.net/browse/NXT-14811|smart-link] 

*Requirements*

* Within the item list of the Add Items screen, provide a selection column for all items
** Items can be selected via the checkbox next to each row
** a Select All checkbox will be available in the grid header
** In the right corner of the grid header, display a ‘Number of Items Selected' label
* Provide an ‘Add Items’ button
** Add Items button will only be enabled if one or more items are selected (checked)
** Provide success message when items are added to the template
*** Message: “Selected items successfully added to template.”
** Remove items from the list once they are added
* Back button, to return to the Items tab of the Order Template Details screen
** Prompt user with “unsaved changes” prompt if there are items selected when user clicks Back button

## Acceptance Criteria
* Items can be selected via the checkbox column
* Add button will add the selected items to the specific template
* Template items will be autosaved as items are added
* Items will be removed from the list upon being successfully added to the template
* Back button will return the user to the Template Details screen

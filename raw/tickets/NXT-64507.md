# NXT-64507 — Acc - CACFP Reports - Edit Check

## Description
As Dana the Director I want to run the Edit Check report for my CACFP program so I can get an overview on meal counts, sales, and an estimate on reimbursement.

Currently the Edit Check reports do not provide support for CACFP.

*Requirements*

* SNP Update
** Only flag CEP/P2 sites on the Total columns.
* CACFP Updates
** User can select CACFP from the Programs dropdown.
** Meal Type dropdown updates for CACFP meal types.
** User can run the Edit Check reports for CACFP and see the typical information, but focused on the CACFP meal types.
*** Need to use the new setting for AF being calculated or the Monthly AF value from Attendance.
*** Need to show the ADA or Attendance counts.
**** Add a new column for this - ‘Atd’
**** Position on Edit Check: After AF
**** Position on Edit Check Worksheet: After Day of Month
**** Add to Report Legend: Atd - Attendance
** When flagging Edit Checks, only flag the Total columns.
** We do no need to consider CEP/P2 for CACFP.

# NXT-20633 — Platform - BO - Onboarding Settings Detail Page

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view system modules summary widgets with its overall progress and take necessary actions

*Mockup:*

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289&starting-point-node-id=637%3A9929|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289&starting-point-node-id=637%3A9929|smart-link] 

*Requirements*:

* Site admin logs into Onboarding Summary Dashboard page for Settings widgets displayed in “Dashboard” tab
* *Settings* tab displays all Settings that has been configured for given district
* Clicking on VIEW DETAILS should bring up the Settings page from schoolcafe
* VIEW should include a drop-down to Select *Category* and list of settings are displayed in the settings table similar to current schoolcafe configs
* Each Setting pre module is populated based on the category selection by the user in the popup

*Progress Calculation*

* Each module required/default config is 10%

## Acceptance Criteria
* User shall be able to see the Settings widget with category, setting name, status & actions column
* User shall be able to see VIEW ALL button to open Settings page from SchoolCafe with Module & Category selection
* User shall be able to click on VIEW to open that setting configs in a pop-up “*Settings*” with Category selection as required drop-down
* User shall be able to see number Settings configured for each module
* User shall be able to see the default & value column for each setting row

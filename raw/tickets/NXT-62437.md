# NXT-62437 — IM - Item Onboarding - Behavior Change Autosave

## Description
As a district admin using the onboarding tool, I would for a way that the system does not always refresh the page on me when it autosaves.

## Acceptance Criteria
* All update Screens
** Do not lose the cursor place on the page when autosaving
** Do not refresh the whole page on autosave
** Allow longer autosave timing (can be discussed) 
** Display autosave timer on footer to give user an idea of how long before autosave
*** 1 Min 
**** Trigger on any unsaved changes 
**** Continue validation checks to ensure no bad data is saved

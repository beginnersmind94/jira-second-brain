# NXT-20028 — Acc. - Claims - History

## Description
As Vera the Director I want the system to hold onto all claims that I’ve created, and have access to them, so that I can see how numbers are changing.

*requirements*

Add the ability to view / download historical claims.

* Claim History
** Provide an option to see all claims that have been created for a particular period, and access to re-download the file.
** Info to show for the view:
*** Generation Date
*** Generated By
*** Program
**** Options here are the following:
***** SNP, SFSP, or CACFP
*** Amount
** Action:
*** Download

## Acceptance Criteria
User can see all claims that have been created for the period, info on the claim, and an option to re-download.

# NXT-52115 — PR- Grocery List- Display of multiple vendor, contract, Ingredient Code and spacing on report (API ONLY)

## Description
As a production manager, I do not need to see the contract number for the vendor on the grocery list report. We are also not providing appropriate spacing after the comma to ensure people understand its a different vendor.

## Acceptance Criteria
* Display only Vendor name WITHOUT contract number
** IF there happens to be the same item on the same vendor with multiple contracts, still only show it one time



!image-20240625-210903.png|width=1501,height=877,alt="image-20240625-210903.png"!



* IF the vendor has multiple orderable pack sizes, it is still one item under that one vendor so only display it once, but display multiple vendor item numbers if they have for each pack size orderable
!image-20240625-210921.png|width=1532,height=839,alt="image-20240625-210921.png"!



!image-20240625-210939.png|width=1768,height=670,alt="image-20240625-210939.png"!

* Provide appropriate spacing after comma on both vendor item and vendor item #\
* When showing items that are both an item and a DMI, currently we are showing the menu item code, but we need to show the ingredient code as the item that we are showing is the ingredient and not the finished product (Menu Item)

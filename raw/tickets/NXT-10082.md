# NXT-10082 — Meal Exceptions - View/Export Eligibility Variance

## Description
As a user I want to see the list of meal exceptions. and be able to resolve them. When meals are recorded at a different eligibility type.

*Requirements:*

* The user can create a list of patrons whose meals are recorded at a different eligibility price than their current eligibility.
* From the list the user can select what transactions to change the pricing to the current eligibility status.
* The user can select multiple items from the list to change, then select save to apply the changes.

## Acceptance Criteria
The user can select the list of patrons whose meals are recorded at a different eligibility than their current eligibility status.
The user can select the meals to change the meal status and record the changes when save is selected.
Multiple updates are applied with one save.
When complete the selections are displayed.
There is an option to print or export the selections made

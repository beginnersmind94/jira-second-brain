# NXT-20653 — IM- Menu Item Management Page

## Description
As a Menu Planner, I would like to have a page where I can view all my menu items with some details related to them. The menu items should be clickable to be able to get more details. Columns to display should be: Item Category, Menu Item Name, Recipe Code, Serving Size, Meal Contribution, Allergens, Datasource, Status

## Acceptance Criteria
Columns to display should be: Item Category, Menu Item Name, Recipe Code, Serving Size, Meal Contribution, Allergens, Data Source, Status 

If Menu Item does not have any contribution values then the pop-up should show as missing
Allow Menu Items to have a link to get more details.
Allow all column fields to be Field Searched

Filter Default to Local Data Source
Show in List view. 

The records listed in the page should consist of Recipes (Single Recipe Items + Duel Entry recipes) which has Menu flag enabled + Menu Items Created by clicking “Add” button

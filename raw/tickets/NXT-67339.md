# NXT-67339 —  GC>Item upload in SC> Viewing/exporting/importing matched items after searching for items by uploading the file with preferred name

## Description
Viewing/exporting/importing  matched items after searching by uploading the file: display the ingredient name same as the preferred ingredient name. 

SC- 2

Ingredient name field should display the preferred ingredient name from the uploaded file in the results grid and in the popup in PE /slide out in SC

Review items popup will display the preferred name from uploaded file - this is not applicable for SC, please let me know if i am missing anything. 

Export excel icon will show the given name from file- both PE and SC

Importing items will import the items with the given ingredient name.

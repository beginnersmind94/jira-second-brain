# NXT-66826 — GC Portal - Data Filters - UI Changes

## Description
_(none)_

## Acceptance Criteria
Navigation: GC Portal - Search - Search Results Grid 

Changes Required:

Include a filter icon, upon clicking the icon, the system should show the filter options 

Once the search results are populated, the user should be able to apply data filter and be able to perform custom filter options

Upon using any value on the custom filter , the system should show corresponding values for the user

Upon clicking reset the filter should be cleared, the user can start over if needed

# NXT-50899 — IM- Recipes- Considering Sub Recipes as part of recommendations on a recipe. 

## Description
As a Nutritionist, when I am adding a sub recipe (Ground beef) to my taco salad, I need the system to pull the sub recipe contribution information to allow me to apply the recommendations along with my other ingredients. 



h4. This Recommendation should include sub-recipes as part of the “ingredients”

!image-20240528-144658.png|width=66.66666666666666%,alt="image-20240528-144658.png"!

h4. Sub-recipes are not currently included in the recommendation calculation

!image-20240528-144745.png|width=83.33333333333333%,alt="image-20240528-144745.png"!

## Acceptance Criteria
* Pull contributions for all sub recipes as we do ingredients for recipes
** Since there is only one serving size for recipe, take the contribution of the serving size
* IF user adds a weight based measure (standard measure) for the sub recipe on the recipe, scale the contributions accordingly
** Same rules apply as far as scaling down

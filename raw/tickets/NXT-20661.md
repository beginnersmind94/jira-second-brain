# NXT-20661 — Production- Pre/ Post Nutrition Summary Production Analysis- Variety Menu Item 

## Description
As a Production Manager, when I pick my menu items from my variety menu item list, my pre- and post production nutritional analysis must capture all information as if it was a normal item and do not include my “parent” menu item on my Analysis. 

Currently shows on the nutrition Summary Report:

!image-20220528-052258.png|width=1384,height=479!

## Acceptance Criteria
Display analysis based on planned count for menu item.
Item analysis will behave as a normal item with a category.
Use nutritional and meal contribution based on the items selected and not the variety menu item. 
All other analytical behavior should be exactly how the analysis currently functions. 

DO NOT Display “Parent” Variety Menu Item

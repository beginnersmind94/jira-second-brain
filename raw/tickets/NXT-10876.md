# NXT-10876 — Acct Mgmt - Notifications - Reminders

## Description
As a district admin, I would like to be able to view and send letters out to patrons in my district that may have a low or negative account balance, so that they are more likely to make payments toward their account and purchase more food in the cafeteria.

*Summary*

* Create a Notifications page, with a Reminders tab. Then fill out the tab with basic functionality (no printing or emailing yet), which mimics Classic

*Mockup*: [Link|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=421269%3A9&scaling=scale-down-width]

*Details*

* In the Account Management *Notifications* page, add a *Reminders* sub-title + section that displays the following:
** Filters for searching for one or more patrons (same as Classic, with improved UX. See mock-up):
*** Patron Criteria
**** Fields to filter list of patrons in Search Criteria. 
**** Each should be cascading and multi-select.
*** Balance Criteria
**** Toggle button group to further filter out patrons who are within a certain balance threshold.
**** Also should still include checkboxes to filter out ‘no charge’ patrons and those whose balance = 0.00.
*** Search Criteria. A grid displaying the results of those filters which includes:
**** Patron information (same fields as Classic--see mockup)
***** For the Balance column, display balances in green if they are >0.00, in black if they are = 0.00, and in red if they are <0.00.
**** Additional columns that allow the user to *check boxes* to email OR print Reminders for one, more, or all patrons in that list.
***** Email column
****** Only display a checkbox for a patron in this column if the patron in that row has a primary guardian with an email address saved
***** Print column
****** Display a checkbox for each patron in the grid.
**** Actions(s) column
***** The ability to *view* a Reminder, using the active Reminder letter template from the Templates page (for one, more, or all patrons in that list). This should open a pop-up letter viewer window, or open it in a new page. 
****** {color:#6554c0}These buttons can be disabled for now. Functionality will be covered in a separate story (NXT-11608){color}
*** Notify buttons that correspond to the Search Criteria grid:
**** _Notify Selected_ 
**** _Notify All_
***** {color:#6554c0}These buttons can be disabled for now. Functionality will be covered in a separate story (NXT-11608){color}
* This story is complete when a user can navigate to the Notifications page - Reminders tab, sees a set of filters that they can use to accurately locate one or more patrons, and then sees a grid with accurate results where they can select/deselect checkboxes  for any patron in the grid.

## Acceptance Criteria

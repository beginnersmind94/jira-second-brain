## Description

* Remove the ‘Is Template’ toggle from inside Menu Grid Layout
* Add a ‘Copy Menu Grid’ action from the main grid

** Copy the menu grid under the new name

**** Meal Type, Enabled, Site Type, Days Allowed
**** Rows, Columns
**** Buttons, button positions

# NXT-71802 — PE-SC-Insights-School Performance-Dashboard-Grid- Sort Columns

## Description
We need to add in the ability to sort the columns in the performance grid

----

h2. *Notes*

* This story applies to both PE and SC
* -Client side sorting is ok for this list-

----

h2. *Requirements*

h3. *1. Dashboard screen Updates*

* For the Dashboard Performance Grid, do the following:
** Add sorting to all column headers
** Like elsewhere, do no show a default up/down arrow on columns that are not sorted
** Only show the up or down arrow of what is actually sorted
** Default sort by site name ascending
** See Bolt code for example



For PE:
Use the isActive flag for School Performance Grid Sorting V2 to build the sorting functionality.
If the flag is false, then the grid should work as it is currently, without grid header sorting. (Backward Compatible)
To implement sorting, use the V2 APIs as shown in the screenshot.

!image-20260409-200758.png|width=511,alt="image-20260409-200758.png"!

!image-20260409-195148.png|width=495,alt="image-20260409-195148.png"!

For SC:
No need to provide a backward compatibility feature.
Both APIs are modified to handle sorting:

!image-20260409-200246.png|width=423,alt="image-20260409-200246.png"!

For sorting, pass 2 new additional parameters. sortColumn and sortDirection

!image-20260409-200211.png|width=515,alt="image-20260409-200211.png"!

schoolname

# NXT-12289 — Direct Certification - File Details - Documents

## Description
*User Story*

As a district admin I want to add the supporting / reference documents that show why I’m manually importing a list of students for direct certification.

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=47124%3A0&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=47124%3A0&scaling=scale-down-width|smart-link] 

!image-20201113-175222.png|width=1908,height=703!

*Requirements*

* Add a Documents section to the DC File Details page.
** Section should show any uploaded documents and associated comments.
** Who uploaded it, when, file name, file size.
** User can download documents.
* User needs a way to remove existing documents.
* User needs a way to upload new documents.
** User can add a comment to the document while doing so.

## Acceptance Criteria
User can see documents in the DC File Details
User can remove existing documents.
User can add additional documents.

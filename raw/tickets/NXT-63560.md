## Acceptance Criteria
If a session date has been changed to a date other than the current date display a warning message before opening the session.

”Session is not set to current date. You are about to process transactions for (Selected Date). Please double-check you wish to ring up transactions for this date. Are you sure you want to continue with back/postdating transactions?”

For Detailed and Bulk Sales, highlight the selected date (border + text color change) if the date is a past date.

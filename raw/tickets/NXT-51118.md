# NXT-51118 — POS - Dynamic Entrees - Option to automatically load the Dynamic Entree category on account load

## Description
As Connie the Cashier I want the Dynamic Entree category to open automatically after an account loads to save time on the line.

*mockup*

*requirements*

* User has an button/toggle at the top of the POS screen to enable/disable automatic loading of the dynamic entrees category.
** Text: Auto-Open Entrees
** Disabled by default.
** Button/toggle is only available if the menu grid has the Dynamic Entrees category on it.
** If the user enables it, it should be persistent - enabled the next time they open into POS.
** If enabled, after loading an account, the Dynamic Entrees category should open automatically.
** If disabled, no change, current POS.

## Acceptance Criteria
* Button/toggle to Auto-Open Entrees added.
* Button/toggle is only available if the menu grid has the Dynamic Entrees category.
* Button/toggle is Off by default.
* User can toggle it on or off.
* Toggle is persistent - next time they load into POS, it should be in the same state.
* If enabled, DE Category should open automatically after an account is loaded.

# NXT-27904 — MP- Menu Planner- Disclaimers Meal pattern 14+ (K-12) Snack

## Description
As a user, I would like to know what all the disclaimers are based on my particular criteria that I select when using the Menu planner

## Acceptance Criteria
Display disclaimers based on click event.

Gray out the ability to click disclaimers until criteria is selected. 

Disclaimers: Meal Pattern 14+ (K-12) Snack

All Menus are defaulted to 100 as the serving count, which are divided evenly between menu items per category

The weighted nutrient analysis is based on counts of 100 in each category divided evenly between menu items within the category

There are no nutrient standards for K-12 Snack, all nutrients over 0 will be considered as pass including sodium

All component colors are turned on based on the daily nutrient standards

Milk component color will turn on only if the combination fluid milk and Unflavored Low Fat or Skim Milk Varieties are met

Any carbohydrate amount GREATER than 0 will show as passing as there are no nutrient standards for carbohydrate

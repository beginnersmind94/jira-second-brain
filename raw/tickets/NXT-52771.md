# NXT-52771 — INV - Physical Inventory Review - Select items for Recount

## Description
As an inventory supervisor I want the ability to select specific items to be recounted and create a new inventory containing just those items.

## Acceptance Criteria
* Provide a selection tool on the Review Completed Inventories page to select specific items
* Provide a method to create a new inventory containing those items
** append ‘Recount’ to newly created inventory
* On the original inventories, disable the items which have been added to the Recount inventory

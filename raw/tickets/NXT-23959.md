# NXT-23959 — INV - Orders - Production Ordering - View Items - UI Only

## Description
As an Ordering Manager I want a Production Orders page so I can select a date range for production records and begin ordering.

This is the first step in a multi-step process to place Production Orders. Production Orders are orders that are generated based on quantities of items used as part of serving. The quantities used on production records will drive the required order quantity.

*Note:* dependent on completion of [https://primeroedgenext.atlassian.net/browse/NXT-25684|https://primeroedgenext.atlassian.net/browse/NXT-25684|smart-link]

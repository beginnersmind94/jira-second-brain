# NXT-14126 — Platform - SSO - Update Login Page with SSO options for user

## Description
*Persona*: Perseus User/admin

*Access level:* School District/Site

*Description*:

As a Perseus user, I should be able to login to Perseus using my registered account using my ISD’s domain configuration

*Requirements*:

* User launches Perseus login page to login using registered email account
* When user clicks on “Next” user is taken to domain of the registered account
* User accounts with respective domains must be pre-configured
** Onboarding process to include Google/MS 365 online domains
* User logs into their account and validates the session

## Acceptance Criteria
* User shall be prompted for registered email address during login
* Registered accounts are updated in Perseus DB
* User redirected to respective email domains configured at site/school level
* Perseus login page is displayed when ISD do not have domain/SSO setup

# NXT-48610 — VND - Device Management - Add/Edit Vending Terminals

## Description
As Ian the IT Manager I want to roll out vending machines and need a way to register their existence in SchoolCafe as a terminal for reporting / session needs.

*Requirements*

* User can add a Vending Terminal
** User enters a Terminal Name
** User selects a Site
** Vending ID number is given after saving 
*** unique terminal id needs to be assigned
* User an update a Vending terminal
** Change status from active/inactive
** Edit site
** Edit terminal name
** History tracking for edits
* -License Integration-
** -There must be an available Vending License for this to succeed-
** -If they are out of licenses, inform the user they need to purchase additional licenses or deactivate an existing vending terminal-

## Acceptance Criteria
If district has vending licenses:

* User can add new vending terminals
** Fields: Terminal Name, Site
*** Terminal Name should be unique for the district, no duplicates. If the user attempts to use the same name again give them an error message.
**** Terminal Name already exists. Please rename the existing terminal if you want to reuse this name.
** TerminalId is given automatically, as always a UID across the entire installation
** This is a new Terminal Type - Vending
* User can edit existing vending terminals
** Editable fields: Terminal Name, Site
** Status: Active/Inactive
* -If district does not have enough vending licenses available, they should be informed when adding.-
** -Every active Vending TerminalType counts against their number of available licenses-
* History tracking on add/edits

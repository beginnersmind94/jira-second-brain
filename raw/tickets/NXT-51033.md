# NXT-51033 — INV - Orders - Order Details - Show banner when order has been included on a submitted E-Order

## Description
As a district admin I want a warning/informational message on the Order Details page when that order has been successfully submitted on an E-Order, so that my site managers do not make changes to orders after it’s too late.

## Acceptance Criteria
* Create a flag to know/track when an order has been submitted via (successful) E-Order
** *important note:* for now this is just an informational message but in the future we may use this flag to prevent the order from being edited
* Show a message in the banner of the Order Details page 
** “This order is in Open status and has been submitted as part of E-Order #12345.”
*** display most recent E-Order #
* Remove message once further status banner is reached
** existing ‘Receipts have been created’ banner, etc.

# NXT-13348 — Platform - AI/ML - Device Management UX

## Description
This EPIC handles all the efforts required for UX to enable interfaces integrated with training application to be used by Operational Readiness (training center) team to setup the recognition devices (EdgeBox - Onlogix, Cameras) which are provisioned and configured using these services. 

*Requirements:*

* Update Edit & Add screen fields to be same and aligned properly
* Mandatory fields should have “*” asterisk symbol to identify
* Device Management must have an icon associated for the catalog page
* Remove (delete) device must prompt for confirmation
* Status is a static field instead of a drop-down

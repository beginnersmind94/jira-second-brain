# NXT-28580 — INV - Physical Inventory (Complete) - Use static grouping UI Only

## Description
As a Warehouse Manager I want the ability to custom group the Physical Inventory item list so I can enter my counts in the order I counted the items.

*Requirements*

* -Allow users to change item sort order on Physical Inventory as needed- *(Future story)*
** -Use the custom grouping grid option to enable this-
** -groupable columns:-
*** -Storage Category-
*** -Item Category-
*** -Physical Quantity-
*** -Perpetual Quantity-
*** -Discrepancy-
**** -we may need to make these separate columns, or independently selectable-
*** -Physical Valuation-
*** -Perpetual Valuation-
* Set default grouping:
** by Storage Category first
** by Item Category second
* Consider future Data Query functionality as possible partial solution

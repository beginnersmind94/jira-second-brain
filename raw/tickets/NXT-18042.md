# NXT-18042 — MP- Search by Menu Item Category on Menu Planner

## Description
As user, I would like to search my menu items on the Menu Planner by Item Category to narrow my search.

Default will be set to All.

## Acceptance Criteria
Display an option to select a menu item category to search through. Default to ALL.

If I select Entrees, and type in Garbanzo Beans, it will give me the ones that are configured as an entree, and not the menu item configured as a vegetable.

# NXT-8033 — Platform - Create multi-language support - UI (Workspace UI Only)

## Description
As a non-native English speaker using PrimeroEdge, I would like the option to be able to translate the application into another language, so that I can maximize my use and understanding of PrimeroEdge by reading it in my native language.

*Summary:*

Add language drop-downs *for all Workspace pages only*, that when selected allows users to translate any of those pages into Spanish or any new languages being supported in the future. ({color:#ff5630}UI Only{color})

*Details:*

* A *Language/Preferred Language* drop-down/icon should be added to the top navigation (within the preferences menu along with Theme)
* When selected, the user should be able to choose between either English or Spanish (English being the default)
** In the future, we will take the logged-in user’s preferred language (from their User Profile) and make that the default language upon logging in.
* When Spanish is selected, translate all non-proper nouns on the page (i.e. Names, School Names, Districts, etc.) into Spanish.
** No functionality should be impacted.
* When English is selected, revert all translations back to English. 
* This is for every Workspace-related page +only+ for now.

## Acceptance Criteria
Verify that on the top navigation, a language selection icon/drop-down is available

For the Workspace pages only:
Verify that the menu allows a user to switch the language selected between English and Spanish, with English being the default selection

Verify that when Spanish is selected, all content on the site that is not a proper noun gets translated

Verify that proper nouns remain untranslated

Verify that when English is selected, we translate the content back to English

# NXT-25491 — INV - Vendors - Contract - Add Items to Contract - Allow items to be added without any required fields

## Description
As a District Admin I want the option to add items to a contract without specifying any new fields so I can quickly setup my contract.

*Requirements*

* Change Vendor > Contract Items to allow for no required fields
** Not required for adding but still required to actually use the item
* +Add Items to Contract screen:+
** checkbox for each item
** do not show side panel when selecting multiple items
*** *Note: deciding interactions between having the side panel open for single item vs. closing side panel for multi-select
**** We will try to preserve both, but may remove single item
** Add Items button somewhere
* Update the Items list
** Remove the sub-grid
** Add the ‘Columns’ option to the table
** Grid columns:
*** Item #
*** Item Description
*** Unit Type
*** Item Category
*** Brand Name
*** Manufacturer Name
*** **optional columns (unchecked by default):
**** Storage Category
**** GTIN
**** Standard Price
**** Weight
***** *Add the weight unit\! (lb, oz, etc.)
* -Set ‘Orderable’ flag to ‘No’ for any items which are added without mandatory fields-
** We will allow items to be added as orderable, since without VI# and Price they will be filtered out of the Create Order screen anyway

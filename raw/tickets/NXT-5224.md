# NXT-5224 — Application Processing - (Edit) Students

## Description
*User Story* 

Processing application data, I am able to click the edit pencil and navigate to the student section to make any edits as needed.

*Mock Ups* 

*Requirements* 

* Student section > Edit Pencil > Takes user to the student page
** Same functionality as the guided process 
*** Add a Student(s)
**** Find a Student 
**** Add Pending Student 
*** Edit Student(s) 
*** Remove student(s) 
* Save and Cancel options 
** Save > Saves changes made and brings them back to the processing page
** Cancel > Discards changes And brings them back to the processing page

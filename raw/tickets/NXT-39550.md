# NXT-39550 — IM - Item Details - Use 'More Actions' icon on cards

## Description
As an IM user I want a ‘More Actions’ icon or button wherever multiple card level actions exist so that my UI is not cluttered with individual actions.

*Requirements*

* *Fix* Nutrient Details card styling
** data is not stretched to the boundaries of the card area, per the mockup
* In instances where a card has multiple action icons, replace them with the more actions icon
* Provide individual actions in more actions dropdown
* -Consider applying logic which will show individual icons in the instances where user does not have permissions to multiple actions-
** Ex: user does not have Edit permissions, so History is the only available action. We should not need to click the more actions in this scenario
** *Future story with Permissions*

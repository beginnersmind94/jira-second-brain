# NXT-9901 — MP > Menus > Assign Menus - Assign and Save Menus

## Description
As a Menu Planner
I need to be able to assign Menus
so that I can let my site staff know what to serve.

Assign Menus allows selection of a single Menu to assign and puts it on the calendar.

* Only allow 1 Menu per menu line per day
* If no values are in place for the calendar days based off the cycles or the menus, then automatically save
* If there are values, then show a warning
** “\[Date] has a \[‘Menu' Menu Name'] already. Do you want to overwrite?]
*** Ok / Discard

User will be able so search and select the menus on the left of the screen

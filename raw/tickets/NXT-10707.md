# NXT-10707 — Applications - Notify on process complete (Auto Notify Enabled)

## Description
As a central office user I want the ability to set my notification mode as automatic when processing applications so that notifications are automatically sent to the family when a determination is saved.

Provide the ability to turn on/off the automatic notification method. Within the validate/processing window, display an “Auto-Notify” toggle that allows the user to automatically generate a notification when the application is processed.

We will start with automatic notification of applications with an email on file. Meaning, based on the students that are added to the application, there is a valid email address attached to the student account. The email can also be applied to the application instance. This could be used as well and is often preferred as the first email to use, then use the email address for the youngest child on the application. 

Those who do not have an email on file, we will give an indicator (some type of icon visual next to the language) for the user that it will have to be a print notification.

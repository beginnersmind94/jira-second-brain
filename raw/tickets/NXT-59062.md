# NXT-59062 — PEI - Auth - SSO from PE

## Description
As a PrimeroEdge Insights user, I would like to be able to login to PrimerEdge Insights (PEI) FO Command Center using my PE login credentials so that I don’t have to login again with a new password

*Notes:*

* The PE part of SSO was done in this story: [https://primeroedgenext.atlassian.net/browse/NXT-54593|https://primeroedgenext.atlassian.net/browse/NXT-54593|smart-link] . See Lalyta for details.

*Requirements:*

* Write the code to enable the user to click on the Insights button in PE and be redirected to PEI.
* The code should properly set the authentication token.
* The solution must accommodate users logging in manually or from SchoolCafé, without hardcoding a single login method.
* Ensure that the necessary license, roles, and profile information are retrieved for license checks and role-based security.
* Write the code to enable SSO from PE to the new Insights website.
* If the existing code for SSO is not available, it should be created.
* The code must pass the following information:
** Authentication token and necessary info for user authentication.
** District information.
** Profile information (e.g., name).
** Confirmation that the district has the Insights license.
** Resources the user has access to in Insights, ensuring security trimming based on user roles.

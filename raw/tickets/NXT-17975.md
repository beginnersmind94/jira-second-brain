# NXT-17975 — INV - Settings - Delivery Locations - Site Delivery Location Details - Delivery Instructions

## Description
As a Food Service Director I want a Delivery Instructions field to enter instruction text and associate it with a particular location, so that the instructions can be included on orders.

Note: these instructions will later be added to Order reports.

*Requirements*

* +On the Site Delivery Location Details screen+ 
* Provide an additional column in the Delivery Locations list, titled ‘Delivery Instructions’
** This will be a text column which displays what the user entered
* Provide a Comments icon within this column, so user can enter delivery instructions
** User can enter up to 250 characters
** On the popup, provide Cancel and Save buttons
* Once user has entered instructions, saved and closed the popup
** refresh the page to display the instructions in the Delivery Instructions column
** display ellipses “….” if the instructions exceed the width of the column

# NXT-6234 — AcctMgt - Accounts - Filters - Available selections for Grade

## Description
*User Story*

As a district admin, there are times when I want to search the entire student population based on a certain grade. Right now, I can’t do that.

*Requirements*

* If no sites are selected, the grade dropdown should contain all grades for the district.
** Do this on the Home page, and the Patrons (Split/Table) search page.
* Allow for all, single, and/or multiple grade selections
* -Provide a “Select All” option for quickly selecting all grades in the list. If the check box is deselected, the expected functionality is to uncheck all grades.-
* Provide a “Not Specified” selection for accounts not assigned a grade. Should always be the bottom option in the grade list.
* By default, ALL grades should be selected.
* Note: Grades are not applicable to adult accounts

*UX/UI:*

* We need a common component that we can use in other places for grade selection in a similar fashion. Make sure that we add to the shared component library (storybook).

## Acceptance Criteria
Grade dropdown populates if no site is selected
Grade is populated by sites selected and their available grades
User is able to select all grades easily
User is able to deselect all grades easily
User is able to select multiple grades
User is able to select a single grade
If no grades are selected, consider as if all are selected
If no grade is assigned to accounts, provide a "Not Specified" option for viewing accounts not assigned to a grade

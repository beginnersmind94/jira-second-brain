# NXT-59008 — INV - Vendors - Add Vendor - Internal Vendor - Link Site (System dependency)

## Description
As a FSD I want internal vendors to be linked to a site so the site’s perpetual inventory can be used to fulfill distributions.

*Platform dependencies with Site Configuration and potentially Licensing*

# District purchases Inventory *Distribution* licenses
# Support/Implementation configures License page according to customer agreements
# Support/Implementation configures Sites with Distribution Site Type
## Distribution Site Type is limited to # of Distribution licenses
# _Inventory Admin configures Internal Vendor_
## _internal vendor is linked to Distribution Sites_

_Italicized point #4_ is the scope of this story.

## Acceptance Criteria
* Add an option in the Vendor Type row when a vendor is configured with Internal vendor type
* For each internal vendor, provide a site link option
* Use an edit icon to enable an in-line dropdown menu to select the Distribution site
* In the dropdown provide the sites with Distribution site type
** *Note:* the ‘Warehouse’ site type will be changed to ‘Distribution’ by the Platform team
* Sites cannot be linked to multiple internal vendors
** in the unlikely scenario there are multiple Distribution sites, be sure to *remove* any sites which are already linked to another internal vendor
* Once user selects the site, link it to the vendor and display next to the Vendor Type
* This option cannot be changed once that vendor-site link is permanent
** once there are orders placed to that vendor

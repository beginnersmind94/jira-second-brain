# NXT-3793 — AcctMgt - Patrons - Search Results - Split View

## Description
*User Story*

As a district administrator I want my quick search to show me results in a card view. 

*Mockup*

[https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5|https://nqyc9v.axshare.com/start.html#p=category_based_on_patron_view&id=jytwv5]

*Requirements*

* Add a toggle near the results table that allows the user to flip between table view and a split view.
* Split view should show little cards for each result.
** Cards should show basic information about each patron.
*** Patron ID
*** Last Name, First Name
*** Site, Grade
** Photo / Circle with Initials will be added later in another story.
* Cards should show in a column.
* We’re only implementing the left side of the mockups.

## Acceptance Criteria
Toggle added to change between split and table view.
Cards show appropriate patron information.
Slider / splitter that can be expanded to maximize view.

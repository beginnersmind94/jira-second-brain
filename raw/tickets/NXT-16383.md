# NXT-16383 — INV - Orders - Order Detail - Receipt tab - Receipt Details - Addable Items

## Description
As an operations manager I want the ability to add items to a receipt so I can add any substituted items to the receipt.

Scope:

* Add Items button on Receipt screen
* Loading items that can be added
* Loading/populating new Add Item to Receipt screen and items

*Requirements*

* On the Receipt Details screen (Create Receipt), provide an Add Items button on the grid header of the items list
* When the user clicks Add Items, save (in the session) any current progress and navigate to a new Add Items screen
** this will require a different screen, but it would be ideal if we could in some way keep it within the receipt details (house it in a second tab on the Items list maybe?)
* On the Add Items screen, provide a searchable list of items 
* Provide a search header with the following
** Storage Category dropdown
** Item Category dropdown
** Data Source dropdown
** Reset button
** Apply button
* In a grid below the search header, provide a list of items returned by the search results
** this should be filtered to only display items provided by the vendor associated with the order 
*** items that the site can order
*** do not include items which already exist on the receipt
** use Field Search button on this grid
** use pagination on this grid
* Item List columns:
** Select column
*** this will provide a checkbox for each item
** Item #
** Item Description
** Vendor Item #
** Storage Category
** Item Category
* Selection of items and adding to receipt covered in [https://primeroedgenext.atlassian.net/browse/NXT-18073|https://primeroedgenext.atlassian.net/browse/NXT-18073|smart-link]

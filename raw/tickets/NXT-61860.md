# NXT-61860 — IM - Item Onboarding - Publishing to Local Items 

## Description
As a district admin using the onboarding tool to add my items into the system, Do not allow items that do not have proper data to be added into the system and alert me to fix those items before they can be published.

!image-20250610-154641.png|width=1188,height=764,alt="image-20250610-154641.png"!

## Acceptance Criteria
* Provide a way for user to check items that are ready to publish by providing a button “Check Publishing”
** This will allow for the system to run validations against what is considered a “good enough” item to be published to local items
* All validations can or will be relaxed once Local updates are available to allow for users to continue adding additional information
** Allow Items to be published with partial Summary Info
** DO NOT allow items to be published if no serving or partial serving info is provided (Food Item ONLY)
** Allow Items to be published IF mandatory nutrients have been entered
*** Mark as Missing for all Remaining if not entered
** Allow items to be published IF either all contribution information has been entered for an item or NONE (No Partial)
** Allow items to be published with allergen or no allergen information
** Allow items to be published with ingredient or no ingredient info
** DO NOT allow items to published if all mandatory information is missing for Menu item Info
** DO NOT allow items to be published with partial Pack Size Info or Partial Inventory Info
*** Any items with any Inventory units indicated must be entered
*** Pack size descriptions, sub qty and base unit to serving relationship must be present
*** Cost
*** Categories

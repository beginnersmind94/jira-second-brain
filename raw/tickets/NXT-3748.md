# NXT-3748 — CEU Levels API 

## Description
Create an API to provide the list of standard CEU requirements so it is easier for adding new job title. Currently only API that returns is all the job titles “/api/Configurations/ViewCEUJobTitle”.

But it would be better if we had two APIs, one to fetch all the available job titles and relevant data and another API that will fetch the 5 mandatory CEU levels.

## Acceptance Criteria
CEU Levels API created

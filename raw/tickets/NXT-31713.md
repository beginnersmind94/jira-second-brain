# NXT-31713 — IM - Items - Update Copy function for CNDB Items

## Description
As a District Admin I want to be able to copy CNDB items so I can use them in my local district and convert them into possible POS items, menu items or single item recipes.

*Update 6/16:* We will leverage the existing Copy popup to accommodate this. We will attempt to add the missing card sections as checkbox options on the copy popup.

** -Start on the global header info section with all possible fields already populated-
*** Display an indicator for CNDB Linked item

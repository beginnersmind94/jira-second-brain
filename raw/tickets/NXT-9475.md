# NXT-9475 — INV - Contract Items - Ordering Conditions - Configure Items (multiple)

## Description
As a District Supervisor I want a detailed item ordering conditions slide-out view on the Contract Items page so I can configure ordering conditions for multiple items at once.

_Inventory > Vendors > Contracts_

*Requirements*

On the Contract Items Ordering Conditions tab ([NXT-5648|https://primeroedgenext.atlassian.net/browse/NXT-5648]) provide a ‘Configure’ button so user can select multiple items, click the button and view the slideout to configure ordering conditions.

When the user clicks the button, a right-side slide-out will appear displaying the list of sites and which are configured to order the item. The slide-out will apply to all of the selected items.

+Slide-out Header:+

* in a section at the top, list the items which are currently selected, so the user knows which items will be affected by the new ordering conditions. 

+Site List:+

* sorted by Site Type
** a checkbox for each Site Type which will allow the user to bulk select all the Sites within that Site Type
*** All checkboxes will be unchecked by default (as if user is creating new conditions)
** Site Types are collapsible/expandable 
*** Site Types are all collapsed by default
*** only one Site Type at a time can be expanded
* Site Code - Site Name
** a checkbox for each Site, which will allow the user to select or deselect that Site for the selected item
*** All checkboxes will be unchecked by default (as if user is creating new conditions)
** checking the Site checkbox will indicate that the Site will be ordering the item from the selected contract vendor
* a Save button to save the changes made to the sites.
** provide an alert so user is aware that all the selected items will inherit the configuration
*** “The selected site ordering conditions will be applied to all of the selected items. This may overwrite existing ordering conditions.”
**** OK/Cancel
* a Cancel button to discard changes made

## Acceptance Criteria
Allow user to select item checkboxes
Enable 'Configure' button if items are selected
Configure button will open slide-out
Slide-out configuration will apply to all selected items
Provide user prompt when saving
Save updated configuration, close slide-out and refresh page

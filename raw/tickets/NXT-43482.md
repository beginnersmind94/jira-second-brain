# NXT-43482 — PR-Pre and Post Production Analysis- Remove Sodium Target 1A disclaimer (API ONLY)

## Description
Since July 1, 2023 has now passed, we need to remove the sodium target 1A disclaimer as now we are going to need to flag Pre and Post Production Analysis as fail if they are exceeding the sodium for target 1A.

# NXT-66550 — Initial Fund Balance Setup

## Description
*As a* school nutrition director
*I want to* enter my starting fund balance for the fiscal year
*So that* I can track my current fund balance and ensure compliance with the 3-month operating expense limit

## Acceptance Criteria
*Given* I am setting up my fiscal year in the Financials module
*When* I navigate to the fiscal year configuration page
*Then* I should see a field labeled "Starting Fund Balance" with a dollar input box

*Given* I enter my starting fund balance amount (e.g., $250,000)
*When* I save the configuration
*Then* the system should store this value and use it for all fund balance calculations

*Given* I have entered a starting fund balance and have YTD revenue/expenses
*When* I view any financial report or dashboard
*Then* my current fund balance should display as: Starting Fund Balance + YTD (Revenue - Expenses)

*Given* I have P&L data for the current fiscal year
*When* I generate compliance reports
*Then* the system should calculate months of operating expenses using the current fund balance

*Technical Notes:*

* Field should accept positive or negative values (some districts may start with deficit)
* Should be editable only by users with fiscal year setup permissions
* Value persists for entire fiscal year until next year's setup
* Used in fund balance calculations across all reports and dashboards

*Where do we get this data from?*

!image-20221011-211346.png|width=2455,height=231!

Image above outlines it but we’re mostly pulling from site configuration, the Attendance page, the Edit Check report, and the Activity report. *The numbers need to match.*

* Site Code = Reimbursement Claim Export Code on the Site Config, or the Site Code if ‘same as site code’ is selected.
* Site Name = Site Name on Site Config
* Approved counts with Visiting/Away = Edit Check
** This should be the highest day for the claiming period - highest free (v/a), highest reduced (v/a), paid would be total - (highest free + highest reduced)
* Attendance Factor = Attendance page for that month
* ADA = (Total Approved with Visiting/Away * Attendance Factor) / 100
* Meal Counts = Activity Report
* Student Meals Not Reimbursable = Second Meals on the Activity Report
* Adult Meals = Staff + Program Adult meals on the Activity Report

## Acceptance Criteria

** Numbers must match for the month if the Standard Export, Activity, and Edit Check are all run at the same time.
* Export option available to dump the file to Excel.
* Format for Excel should be the same as the first screenshot

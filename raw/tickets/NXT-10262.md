# NXT-10262 — UI - Platform - Notifications - Templates - Email Subject Line

## Description
As a district admin I want to specify the email subject line for a particular letter template so that it provides clarity to the guardian as to the purpose of the message.

*mockup*

We need to Subject as a new text field in the existing screen.

## Acceptance Criteria
* As part of the letter template, user should be able to enter Subject Line.
* Save the Subject Line as part of the letter template.
* View the subject line that saved as part of the letter template.

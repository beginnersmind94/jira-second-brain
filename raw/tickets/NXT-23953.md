# NXT-23953 — INV - Vendors - Add Contract - Delivery Schedule (Step 2)

## Description
As a district admin I want the second step of Add Contract to be the Delivery Schedule so I have a dedicated screen for setting site delivery days.

*Requirements*

* As part of [https://primeroedgenext.atlassian.net/browse/NXT-23807|https://primeroedgenext.atlassian.net/browse/NXT-23807|smart-link], we are changing the Add Contract stepper
* As a result, Step 2 will be entirely for the Delivery Schedule
* Display the Vendor-level Delivery Schedule and Time fields
** Keep the ‘Apply to All Sites’ button (find a better spot)
*** ensure that the alert regarding overwriting site configuration is still displayed 
* Remove the ‘Site Level Configuration’ toggle
** instead we will provide a dedicated site list with filters/search
* Provide a Site List 
** do not load sites by default
** provide Site Type and individual Site dropdowns
*** both default to ‘All’ 
*** Apply button to load the list of sites according to selections
** Group sites by Site Type
** Save button to save individual site configurations
*** Example use case: User selects ‘High School’ from Site Type dropdown, selects Mon, Wed, and Fri for Vendor level schedule, clicks the ‘Apply to All Sites’ button to load the same schedule for all high schools, then Save. Then they select Elementary school and repeat the process with different schedules.
* Complete button to finish setup
** Delivery Schedule is not mandatory, but alert if no days are selected
*** ‘Proceed without configuring Delivery Schedule?’

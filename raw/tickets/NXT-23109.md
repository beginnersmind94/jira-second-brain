# NXT-23109 — Data Exchange - Photo Import - Error and data integrity handling - Display Exception Logs-API

## Description
Data Exchange - Photo Import - Error and data integrity handling - Display Exception Logs-API.

As a central office administrator or technical team member I want to see a log of photo import errors so that I can determine why particular records in my file did not process correctly.

## Acceptance Criteria
Data Exchange - Photo Import - Error and data integrity handling - Display Exception Logs-API
When the user clicks on Import Monitor-View Log, show the Exception tab next to the Activity Monitor tab.
The Exception tab should have a Field Search button where the user can search a particular AccountId which is error-prone.
It will also have a grid to display all the error-prone data.
The grid will have 2 columns, Account ID and Exception.
The grid should have pagination and the API should provide paginated data.

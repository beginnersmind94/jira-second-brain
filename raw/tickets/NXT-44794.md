# NXT-44794 — Local - Stock Item - Create Stock Item using GTIN

## Description
_(none)_

## Acceptance Criteria
Navigation: Menu Planning - Stock Items

When the user click on the link, the system should show both the ingredient and stock item tabs together so that user knows that they can create stock item.

* Name - Populate the item description against corresponding GTIN
* Broken Unit - Capture Next Lower Level Item Quantity against Case GTIN
* Broken Unit Description - Capture the the product type description of base unit
* Populate fair market value as Zero

If there is no match found, then system shall show these validations as-is

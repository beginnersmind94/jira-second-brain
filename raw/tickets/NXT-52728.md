# NXT-52728 — MP - Menu Cycle - Full history of all changes made on Menu Cycle from Menu Planner

## Description
As a menu Planner, I would like to be able to see a history of all the changes that have been made on the menu cycle that I am working on or any menu in general. This will give me the ability to see why there are any out of the ordinary things or changes I do not recognize.

## Acceptance Criteria
* Capture history of create or edit changes from Menu Planner to display on the history on the menu cycle standard history display
** IF we can capture whether changes were made from standard or menu planner that would be a plus
* History should display:
** All new data in the create process 
*** Meal Pattern
*** Meal Type
*** Site Group
*** Menu Cycle Name (This is the only one that can be updated so show the updated values as well if there are any)
*** Number of weeks 
*** Days per Week
* Track All Menus added to menu cycle
** Track what day has what menu (This is on create process since there is no way to change menus on the menu planner at the moment)
** If multiple menus are added at the same time, track it as one entry
* Any changes done to the menus should be tracked on the menus history and not on the menu cycle history especially from the menu planner functionality

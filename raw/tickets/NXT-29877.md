# NXT-29877 — PR- Add Menu items from Production Plan or Record Comment Display

## Description
As a production Manager, When I add an item to my production plan or record, I would like to be able to see on the record, who added the item in the comment for that particular item.

Currently when a user adds an item, there is no indication on who added it and when it was added to the plan or record.

## Acceptance Criteria
Display in the record comment box who added the menu item, on what date and what time. 

EX: Added from the Plan Page by Veronica Pineda on 03/10/2022 08:00:49 AM

EX: Added from the Record Page by Veronica Pineda on 03/10/2022 08:00:49 AM

Do not allow users to remove this information from comment box, but they may add additional information

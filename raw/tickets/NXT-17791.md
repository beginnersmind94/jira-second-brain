# NXT-17791 — IM- Mark Menu Item as Favorite

## Description
As a user, I need to be able to mark a menu Item as a favorite so that it can show up on my menu planner as a quick option. 

On  Recipe Screen where you can create and also edit Menu Item. 

Refer to Muck-up screen attached for fields. 



!image-2021-08-16-10-33-15-735.png|width=200,height=183!

## Acceptance Criteria
When Creating or editing a Menu Item on the recipe screen

*  Allow user to flag or mark as FAVORITE the Menu Item.
* The Favorite flag should remain set after saving
* The user should also be able to mark a Menu Item as not favorite and the remains after saving
* The default value for this flag should be set False (Not favorite)

# NXT-3839 — Notifications

## Description
*User Story*
As an Insights user, I can create notifications to be sent when a Measure reaches a certain threshold.

*Mockup*


*Requirements*

* When creating a notification, users select a Measure that they want to be notified regarding from a multi-level dropdown list.
** Default KPIs (enabled only)
** Custom KPIs (enabled only)
** Metrics (enabled only)
** Goals
* When creating a Notification, Users establish a Green, Yellow, and Red threshold.
** Green
*** User chooses operator (>= or <=)
**** If Red operator is chosen first, this operator will be automatically selected to the opposite operator.
*** User enters number
** Red
*** User chooses operator (>= or <=)
**** If Green operator is chosen first, this operator will be automatically selected to the opposite operator.
*** User enters number
** Yellow
*** System automatically calculates the range between the Green and Red thresholds.
** Thresholds are validated by the system to ensure that the ranges are mathematically correct
*** Notification can not be saved if there is an error in threshold validation.
* When creating a Notification, users need to define a frequency. (How often are they notified that a Measure is in the Red? Daily? Weekly? Monthly?)

# NXT-22438 — INV - Withdrawals - Updates for displaying Transactions

## Description
As a District Admin I want the Withdrawal and Addition transactions to reflect their most recent state, including most recent comments.

*Note:* Additions in separate story ([NXT-24670|https://primeroedgenext.atlassian.net/browse/NXT-24760])

*Requirements*

* Add a new static ID number
** this would be the ID for the transaction for the user
** in the background, individual transactions will still be used and tied to the main ID
** Exact descriptions TBD
* Cancelled transactions in the list should show all cancellation details
** Comments, Date, UpdatedBy (DB only), UpdatedOn (DB only), etc.
* Most recent transaction should always be displayed on the Details page
** for Cancelled, show some indication the user is viewing a cancelled transaction

* Always show most recent transaction state on list page

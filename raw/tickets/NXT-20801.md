# NXT-20801 — INV - Physical Inventory (Incomplete) - Autosave should not reset the grid (R&D)

## Description
As an Inventory user I want the Physical Inventory auto-save to not affect the list of items, so that my filters and searches do not get reset every 30 seconds.

This is first a *Research story.* Will discuss again pending results.

*Notes:* 

* need R&D to evaluate the way the items are binded to the auto-save
* may need help from Platform Team
* evaluate viability of future user-configurable setting that determines auto-save frequency

## Acceptance Criteria
* User selections and changes should not be reset by the autosave

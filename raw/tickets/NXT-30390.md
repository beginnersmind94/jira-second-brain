# NXT-30390 — ADA Compliance - Keyboard access for Import Code Mapping

## Description
This story is to cover the tasks related to importing the photos in bulk without any limitations.

## Acceptance Criteria
* As a user with a disabilities, I would like the ability to access “Code Mapping” within the “Mapping” screen while importing.
** I shall have the ability to use the “TAB” key to move to the “Code Mapping” icon so that it has scope.
** When the “Code Mapping” icon has scope, I shall be able to press the “Enter” key to open the code mapping window.
* Visual treatment of the “Selected Source” shall be as it is in Figma.
* Visual treatment of the drop zone shall be as it is in Figma.
* Help text for ADA compliance keyboard instructions shall be present and as it is in Figma
* The UX shall be according to this Figma: [https://www.figma.com/file/OAZ7qTRgdG5bFFGuhDmkoZ/Item-Management-6.0?node-id=479%3A16596|https://www.figma.com/file/OAZ7qTRgdG5bFFGuhDmkoZ/Item-Management-6.0?node-id=479%3A16596|smart-link]

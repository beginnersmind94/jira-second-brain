# NXT-50221 — LMA - Subscription - Profile

## Description
As a SchoolCafe user, I would like to manage my License Management event subscriptions so that I can control how I am notified.

*Notes:*

* You should do the subscription stories for License Keys and District Licenses first

*Flow:*

* User logs in and clicks on Profile in the Workspace sub menu: Workspace > Profile
* User clicks on the Preferences tab
* User makes any modifications they want
* System uses these settings when sending notifications

*Requirements:*

* On Workspace > Profile > Preference tab do the following:
* Ensure the License Key Events show up in the list for users that are subscribed
* Ensure the District Licenses Events show up in the list for users that are subscribed
* For both, ensure each editable field has the correct values per their values created in the original stories

## Acceptance Criteria
* Users can subscribe to an event selecting their criteria and when they want to be notified
* Event list is dynamic, not hard coded
* Users can edit their own values

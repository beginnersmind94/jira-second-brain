# NXT-3338 — System - Reimbursement Rates - Add Standard tab

## Description
As a district or central office admin, I would like to be able to view the Federal & State reimbursement rates pertaining to my organization, so that I can rightly plan our financial goals.

+Summary+:

Allow a user to view their Federal and State reimbursement rates, by program, for a specific academic year and state category.

+Details/Use Cases+:

* This page will be for viewing data. For now, no updates will take place here.
* In the Standard Tab of the Program Configuration page, allow the user to select an academic year and state category filter.
** Type: Federal or State. Previously in Classic these were on separate tabs, but they will be on the same tab here. Federal option can be selected by default.
** Academic Year: Current year should be selected by default, and previous years should be sorted in descending order. It can display up to the last 10 years.
** State Category: Contains ‘Contiguous States’, ‘Alaska’, and ‘Hawaii/Puerto Rico’ (in that order). Contiguous States should be the selected option by default.
* Beneath the filters, display a list of programs in alphabetical order. List of programs should match those from Classic, along with the tooltips displaying next to certain programs.
** For each program, display the Classification options nested underneath (should match what is shown for that program on Programs tab, if available. Otherwise, get list from Classic Reimbursement Rates tabs).
** Programs can be collapsed by default
* For CACFP, there will be additional options related to Centers, Day Care Homes, and At-Risk After School Meals. These can be nested beneath CACFP as toggle buttons.
** When each button is selected, display the same fields as in Classic for that respective category.
* For each program, display associated reimbursement rates for Free, Reduced, and Paid meals. This data will come from the government each academic year but can be copied from Classic and stored in the database for now.
* When the user switches between filters or CACFP tabs, the data should be reflected to the user immediately.
* A search field should be added that allows the user to find any text on the page.

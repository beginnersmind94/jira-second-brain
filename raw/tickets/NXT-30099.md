# NXT-30099 — IM - Items - Create New Item - Food Item - Serving Measures

## Description
As a District Admin I want the system to provide a Food Item option so I can configure items according to their respective types.

*Requirements*

* If user specifies the item as a food item, load the Food Info tab 
** add red ‘Incomplete’ tab label
** [https://primeroedgenext.atlassian.net/browse/NXT-30128|https://primeroedgenext.atlassian.net/browse/NXT-30128|smart-link] 
* Display two cards to load the Serving Measures section (this story) and the 'Ingredient Info' section ([NXT-30100|https://primeroedgenext.atlassian.net/browse/NXT-30100])
** Set the card height equivalent to the Ingredient Info card
** Use an in-line vertical scroll if the number of serving measures exceeds available card height
** Add a Cancel icon for each in-line edit
* Remove the card level Save/Edit
** all editing will be done on the individual measures
** also remove the ‘Clear’ button 
* Change the flag for ‘Main Serving Size’ and add one for ‘System Calculated’
** Main Serving Measure - the default measures from which the others are calculated
*** display this serving size as (Main) next to size description
*** change the (i)  to be the new ‘Main’ flag; update the wording (TBD)
** Allow System Calculated sizes to be deleted
* Provide a selection option to indicate which size is used for all sections (measures, nutrients, contributions)
* Upon saving at least one Serving Measure, load the Nutrient Details section ([NXT-30101|https://primeroedgenext.atlassian.net/browse/NXT-30101]) and Contributions ([NXT-30102|https://primeroedgenext.atlassian.net/browse/NXT-30102])
** change the tab color to yellow to indicate that some info has been entered but not all

# NXT-22615 — IM - UI Improvements - Recipe - List View Changes

## Description
!image-20211122-233314.png|width=2557,height=1267!

## Acceptance Criteria
The user should be able to search a recipe using following search parameters:

*Quick Filter* (Default Value: All), other values should list as shown below. The system should show the results based what user selected

* All- show all the transactions, other values are listed below
* !image-20211122-233525.png|width=271,height=212!
* Past 3 Months - show all the transaction created or updated in the last 3 months (system date - 90 days)
* Past 6 Months - show all the transaction created or updated in the last 6 months (system date - 180 days)

* Past Year - show all the transaction created or updated in the past year (Ex: 1st Jan 2020 until 31st Dec 2020)
* Date Range - enable the date range so that user can select

Date Range Values should show following error- in case user enter invalid dates:

!image-20211122-233607.png|width=531,height=86!



*Generation Type*:

Default Value = All, other values: Dual Entry, Single Item Recipe



*Contains an Item:*

Default = Blank, when the user types an item description - it should list the item as shown below (Item Description | Item Code - similar to Item Search in Recipe Steps). The user can choose one or more items as needed

When value is defined, the system should list the recipes which has either of the item value selected (OR condition)

!image-20211122-234851.png|width=285,height=252!



*Recipe Category:*

Default: All; other values based on Recipe Category Dropdown in Recipe Screen (as shown below)

!image-20211122-234240.png|width=2517,height=649!



*Generic Requirement:*

 When the system loads the page for the 1st time, it should list all the transactions (Quick Filter = All, Generation Type= All, Brand = All, Contains Item = All, Recipe Category= All)

When user apply more than one parameters, it works based on AND condition (except “Item Contains” which should be OR condition when one or more value listed)

# NXT-45475 — IM - Items - Deactivate Inventory Item

## Description
As a district admin I want the ability to make an Item inactive if it is not being used anywhere in the system, so I can delete any unused or accidentally created/imported items.

## Acceptance Criteria
* Provide the Deactivate tool for the Inventory Item sub-type in the Item Type panel
* After clicking the deactivate icon for Inventory Item, prompt the user with the message:
** “Do you wish to make this inventory item inactive and remove from all active contracts?”
** If the user clicks Yes, Deactivate, disable the item on all active contracts
*** Disable means forcibly make it Unorderable, and prevent it from being made orderable again
*** On Vendor Items and Contract Items, the item should be made unorderable but will remain on the UI with an alert icon
*** Ordering Conditions will only indicate the item with an alert
**** overlap with [Config > Ordering Conditions|https://primeroedgenext.atlassian.net/browse/NXT-45552]
* After disabling on the contracts, check item for presence on Open or Pending Review orders
** If the item is present on Open/Pending Review orders, move the status to Pending Deactivation
** if the item is not on any orders, move directly to Inactive status

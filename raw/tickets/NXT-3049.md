# NXT-3049 — **System - Organization - View User Details

## Description
As an administrator I want to view the details related to each user in my school district, so that I can quickly view and make changes to their profile and password as needed.

The users list page contains a table of all users that are authorized to use PrimeroEdge within a school district.

PrimeroEdge Classic Reference: System > Sites and Users > Sites > Users > Select hyperlink in ‘Username’ column

Page should be the landing page of the administrator selecting a user from the User page grid. This story is only for the User Profile and Password sections of the user’s details. Organization and Roles will be in a separate story.

We propose a cleaner card the user to reduce the amount of scrolling required if possible. Page fields themselves should be duplicated from the legacy application. Authentication rules are addressed in separate stories. This story is mainly to view and modify the fields.

## Acceptance Criteria
1. Users with permission to select user details can navigate to this page and see the user's profile data and password settings
2. Users with edit permission can edit fields and save the new values as well

# NXT-20796 — INV - Withdrawals - Create Withdrawal - Items List (Step 2)

## Description
As a Site Manager I want an items list so I can view the items I am withdrawing from inventory and provide additional input.

Continued from [https://primeroedgenext.atlassian.net/browse/NXT-1675|https://primeroedgenext.atlassian.net/browse/NXT-1675|smart-link] 

*Requirements*

* As items are added to the list via the Add Items tool they will load in the Item List
* There are 2 main sections on this screen:
** Add Item control ([NXT-1675|https://primeroedgenext.atlassian.net/browse/NXT-1675])
*** the tool which is used to find and select items to add them to the list
** Item list (this story)
*** the list that contains the items which have been added to the list
* Item List section
** This will be the lower section on the page
** Display a standard grid which is unpopulated by default
** Display the following columns:
*** Item #
*** Item Description
*** Quantity on Hand
**** display the current QOH value for the selected item
*** Quantity
**** editable
**** contains the value which was entered when the item was added
**** the value here cannot exceed the available QOH 
***** “Withdrawal Quantity cannot exceed available QOH.”
***** show under the Withdrawal Quantity field
*** Action
**** Comments icon + popup
**** Delete icon
***** prompt on delete: “Are you sure you want to remove this item?”
** Display the stepper Next button once at least one item is added

**Mockup Note:* Quantity on Hand is not an editable column

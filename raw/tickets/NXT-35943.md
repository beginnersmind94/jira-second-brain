# NXT-35943 — Acc - Menu Item Sales Report - District Summary option to sum all individual sites

## Description
As Laina the Director I want to see a true total of items sold across the entire district without having to add each site together so I can send that off to 3rd parties for future decisions. This will save time and lower frustration.

!image-20230112-180942.png|width=1564,height=694!

!image-20230112-180818.png|width=2299,height=793!

The current ‘District’ Item Summary report still splits everything out by site, so let’s rename that and add a proper District Item Summary report.



For Menu Item Sales report → Item Summary option:

* Rename the current Group Type option: District → Site
* Add a new Group Type: District
* Add a new version of the Item Summary report for this District option
** This version should sum items across all sites.
** Do not list by site - show the true total across the district

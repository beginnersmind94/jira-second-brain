# NXT-4737 — AcctMgt - Households - Updates

## Description
*User Story*

As a district admin I am more concerned with actions that will have a large impact on a student - not all confirmation boxes are necessary.

*Requirements*

* Couple of updates for the scenario of adding a student to a household via the Household panel.
** Display the confirmation prompt only if the student to-be-added is in a household that has a size greater than 1.
** Example:
*** Student A is in a household of 1.
*** Student B is in a household of 3.
*** If we add Student A to Student B, there would be no confirmation prompt.
*** If we add Student B to Student A, there would be a confirmation prompt.
** Update the text on the confirmation:
*** Text: Adding Matt Smithers to this household will remove them from a household of 3 students.
Are you sure you want to add them to this household?
*** Show the name of the student, and the number of patrons in their existing household.
* Users should be able to click on students in the grid as a form of navigation.
** If Student B is showing in the grid, user should be able to select them and load that account.

## Acceptance Criteria
Users receive a confirmation prompt only if the patron they're adding is in a household of size 2 or greater.
Users can navigate from 1 member of a household to another via the Households grid.

# NXT-62442 — INV - Config - Ordering Conditions - Support ordering from multiple vendors

## Description
As an ordering manager I want to be able to configure multiple vendors as orderable for site ordering conditions, so I can order items from specific vendors based on real world conditions.

## Acceptance Criteria
* Update the Ordering Conditions page to support multiple vendors being orderable for an item-site condition
* Allow a vendor priority to be established

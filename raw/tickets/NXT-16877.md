# NXT-16877 — [TaggedReport] - Reports - Export to PDF/Excel

## Description
As a district admin I want to export reports out to manipulate in Excel, or to send to others / print for reference.

*Requirements*

* Existing reports need an Export option.
** Should be able to export to PDF or Excel.
** (Should be able to Print also, but users can print from either Excel or PDF.)
* We need to add some more information to the reports to make them more useful once exported - 
** Add all selected parameters.
*** Show the sites and selected date(s) in the header, along with the report name.
*** Show the selected report options in the footer so any user can re-create the exported report.
** Add generation date and user (Firstname + Lastname) to the header.

## Acceptance Criteria
User can export a report to PDF or Excel.

Reports contain parameters/options and are properly dated/named.

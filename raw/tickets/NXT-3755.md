# NXT-3755 — Employee Details - Additional Sites 

## Description
* Tied to system integration 
* not critical for TW, original understanding was that the info was already provided by system module. 
* If user is an employee and has additional sites, we can populate this field 
* goal of the additional sites field was to show secondary sites the employee may work (I.e. Kitchen director- responsible for the elementary and middle school) 



To be evaluated with system integration.

## Acceptance Criteria
Employees that operate our of more than one site can use the additional site field to add a secondary site. (i.e. primary - elementary, secondary - middle school)

# NXT-19993 — MP- Menu Planner- Check for unsaved changes 

## Description
As a user, when using the menu planner, I would like a pop up to appear to warn me that I am about to leave the page and if I am sure I would like to leave the page with unsaved changes. 

This will also be needed on the Menu page (when building a single menu), and on the Menu Cycle page (when putting together a cycle in the traditional method.)

## Acceptance Criteria
Pop to display for user to ask if they are sure they want to navigate away from page if any changes are made on the page that have not been saved. 



Message to Display: Are you sure you want to leave this page? Any unsaved changes will be lost. (Yes) (No)

Yes= navigate to new page user has picked

No= stay on page

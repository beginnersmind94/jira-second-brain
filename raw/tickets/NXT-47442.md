# NXT-47442 — IM - Configuration - Menu Item Categories - Usability Overhaul

## Description
As a menu planner or POS sales manager I want the Menu Item Categories page to be more streamlined so I can create new categories without breaking any core category experience.

## Acceptance Criteria
* Change the Menu Item Categories page to an updated grid design
* Change colors to be removed from text background and displayed in icon or square
* Allow sub-categories to be created
* Move POS-only categories to be separately managed or added on top of existing categories (sub-categories)

# NXT-7581 — Platform - Create multi-language support (Back-end/Framework)

## Description
As a non-native English speaker using PrimeroEdge, I would like the option to be able to translate the application into another language, so that I can maximize my use and understanding of PrimeroEdge by reading it in my native language.

*Summary:*

Add language drop-downs site-wide, that when selected allows users to translate any given page into Spanish or any new languages being supported in the future. ({color:#ff5630}Back-end Only{color})

*Details:*

* Add the back-end framework that will be able to support the following:
** For every page, a *Language* drop-down/icon should be added to the top navigation (next to Theme, User Profile, etc.)
** When selected, the user should be able to choose between either English or Spanish (English being the default)
*** In the future, we will take the logged-in user’s preferred language (from their User Profile) and make that the default language upon logging in.
** When Spanish is selected, translate all non-proper nouns on the page (i.e. Names, School Names, Districts, etc.) into Spanish.
*** No functionality should be impacted.
** When English is selected, revert all translations back to English.

## Acceptance Criteria
Verify multi-language support framework is added and usable for any page throughout the application

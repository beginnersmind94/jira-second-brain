## Description
_(none)_

## Acceptance Criteria
The user should be able to run a monthly KPI report based on the following parameters:

# Site Name - All Sites + Individual Site Name an option
# Meal Type - All Meal Types + Individual Meal Type as an option
# Menu Item Category - All menu Item Category + Individual  Menu Item category as an option 
# Frequency - KPI Report - Detailed
# From Date and To Date

Report Title: Back Office KPI Report - Detailed

Sorting On report should be based on Site Name, followed by Serving date, followed by Menu Description  



The RDL Template should have following sections:

Report Header - as shown in sample report attached

Criteria Selected - as shown in sample report attached

 Column Headers in Brand Color

Report Legend - as shown in sample report attached

Footer as Shown

Coverage - Design of RDL Template, and Populate the Details (as given in Excel)

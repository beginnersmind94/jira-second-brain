# NXT-3832 — Edit Goal

## Description
*User Story*
As an Insights user, I can edit an existing custom Goal using existing Measures.

*Mockup*
[https://qvv7ze.axshare.com/#id=5v5eav&p=goal_1|https://qvv7ze.axshare.com/#id=5v5eav&p=goal_1] (Mockup created before having visibility into existing 2.0 mockups)

*Requirements*

* Edit page auto-populates all values related to the existing Goal.
** Name
** Description
** Measure
** Targets
*** Target
*** Operator
*** Value
*** Date
* Users can edit the values of any field on this page.
* Only enabled Measures will show up
* Users can add multiple Targets with associated Values, Operators, and Dates.
** Target multi-level dropdown is limited based on the available Level(s) of the Measure selected.
* Name, Description, and Measure must be selected to Save a Goal.  A Target, Operator, Value, and Date must be selected for each Target added in order to Save a Goal.
* Users are prompted when attempting to navigate away from this page with unsaved changes.

## Acceptance Criteria
Users can edit and save the values of all fields on the existing Goal.

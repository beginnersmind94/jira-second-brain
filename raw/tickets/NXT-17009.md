# NXT-17009 — INV - Orders - Orders List - View Receipts - Open Receipt Details

## Description
As an Inventory user I want a Receipt List on the Receipts tab of the Manage Orders page so I can view a list of all receipts, view relevant receipt info, and open specific receipts.

This story is for opening the receipts tab of a specific order via the Receipts List, navigating to the specific tab of that Receipt #.

*Requirements*

* On click of View icon on Receipts list
** Navigate to the selected Order Detail
** Load the Order Detail on the Receipts tab
** If possible, pre-select the tab for the specific receipt that was selected

No API work needed.

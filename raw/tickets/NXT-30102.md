# NXT-30102 — IM - Items - Create New Item - Food Item - Contributions

## Description
As a District Admin I want a Contributions Info section so I can specify my contributions.

*Requirements*

* Load the Contributions section once the Serving Measures ([NXT-30099|https://primeroedgenext.atlassian.net/browse/NXT-30099]) is successfully saved
** Put the section in the allocated 1/3rd width to the right of the Nutrients card
** The height of this card is dynamic (The Allergens card is displayed below)
* Add/Edit Contributions in the slideout
** Reference Recipes for visual example
* Can create or enter contributions without having nutrient information entered
* Allow user to “skip for now” if  they do not want to enter the information at the time
* Display Contributions in ‘card’ format, rather than list

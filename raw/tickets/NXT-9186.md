# NXT-9186 — INV - Add New Ordering Conditions - Create Conditions (Step 4)

## Description
As a District Supervisor I want a Create action on the Add New Ordering Conditions tool so I can officially create ordering conditions and return to the Summary page.

This is the final step of adding new ordering conditions where user can click Create button and save newly configured conditions.

*Requirements*

Provide action buttons at the bottom right of page to allow user to complete adding new conditions process.

+After Site selection+

* Create button to officially save the configured Ordering Conditions
** require the selection of at least one site before enabling Create button
** when the user saves the ordering conditions, the Ordering Conditions will be saved
** after clicking Create, the user will be navigated to the Summary page ([NXT-5706|https://primeroedgenext.atlassian.net/browse/NXT-5706])

!image-20201224-213925.png|width=83.33333333333333%!



APIs:

* one to insert Ordering Conditions

## Acceptance Criteria
User can save/create new ordering conditions (if at least one site is selected)
User will be returned to Summary page once conditions have been created

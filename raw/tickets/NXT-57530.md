# NXT-57530 — Imports - Explain Why Cancelled - FD (269127) - Spike

## Description
As a imports user, I would like to see why imports were cancelled for Student, adult and employee Imports so that I can save time debugging the issues

*Notes*

* Do this the same way we do for Student Import

*Requirements*

* Do this for Student, Adult and Employee imports
* On the View History screen when clicking on a row in Import Monitor, do the following:
* Show an error message explaining why the import was cancelled
* Put the message as an error information text at the bottom of the Summary section
* Only show this message when the status is Cancelled for those import types
* Show for these scenarios:
** Canceled because the withdrawal threshold was exceeded
*** Text: Import cancelled because the maximum withdrawal threshold of n% was exceeded
*** Note: get the n% from the threshold settings
** Canceled because the user clicked cancel on a long running task
*** Text: Import cancelled manually because it was a long running process
** Canceled because there was an ADF error
*** Text: Import cancelled by ADF because of an error

## Acceptance Criteria
* User is able to see why the import was cancelled

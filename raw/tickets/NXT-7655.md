# NXT-7655 — INV - Physical Inventory - Start new Physical Inventory - Existing Inventory

## Description
As an inventory user I want to be alerted when there is an existing open physical inventory for the site and period I am attempting to start so I can open that existing inventory.

[Figma|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=80104%3A148411&viewport=325%2C-4288%2C0.08251962065696716&scaling=scale-down]

Related to [https://primeroedgenext.atlassian.net/browse/NXT-7258|https://primeroedgenext.atlassian.net/browse/NXT-7258|smart-link]

If there is an existing Physical Inventory for that period, do not allow the user to start a new inventory, and instead provide a link to open the existing one.

Label:

“There is a pending physical inventory. Click here to view.”

User can click the link to open the existing inventory. 

* if the existing inventory is in ‘Physical Incomplete’ status, this will navigate to existing “main screen” for the inventory.
* if the existing inventory is in ‘Pending Reconciliation’ status, this will navigate to existing “review screen” for the inventory.

Do not show the message for Cancelled inventories.

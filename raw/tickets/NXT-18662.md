# NXT-18662 — Platform - BO - Enable District Account Setup

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* Central Office/District Admin

*Description*:

As a Biz admin, I should be able to create new district account and add modules/products that was purchased for specific school district

*Requirements*:

* Central biz admin logs into System/Onboarding page for list of district accounts onboarded in “District Accounts” section
** +District Accounts:+
*** District Name
*** Number of Modules Licensed
* “NEW DISTRICT ACCOUNT” button should be enabled
** Clicking on New District Account function should bring up onboarding screen with available modules
** User shall be able to select the modules and add them to the district for later site assignment.
* Clicking on each district on the “VIEW” icon should bring up district account summary with licensing summary view.

## Acceptance Criteria
* List of District accounts to be displayed under Onboarding page
* NEW DISTIRCT ACCOUNT button is enabled for district account setup and modules addition
* List of available licenses are displayed under District Account
* User shall be able to click on “view” icon to view the district account summary with all products that has been procured with licenses (License summary)
* Number of products enabled/licensed shall be displayed on the district accounts page

# NXT-43944 — PR- Plan/Record- Consider Prep site flag with statuses to recalculate food required/food needed - API Only

## Description
This story is to consume production statuses on the menu item so that we can update the flag on the production plan and records accordingly.

## Acceptance Criteria
* Production plan/record Menu items need to consider prep site flag
* DO NOT update record complete plans
* DO NOT update plan complete plans if they are associated with a consolidated plan that is already completed
* IF flag is off, calculate food required as normal
* IF flag is on but prep and receiving site is the same, calculate food required as normal
* IF flag is on but prep and receiving site are different, remove food required on plan page and show on consolidated page
**  IF flag is turned off but prep and receiving site are different, remove from consolidated page *ONLY* if consolidated plan is not complete

# NXT-49776 — Notifications - Bell Icon

## Description
As a SchoolCafé user, I would like to see a count of new inbox notifications in a bell on top so that I can know immediately if there is a new message.

*Flow*

* User logs in and browses to any page
* System or someone sends the user a new message
* System adds to the count of new messages in the bell icon on top
* User can click on the bell to see the list of recent messages
* User can click on a message and redirect to view the message

*Requirements*

* Add a bell icon to the top right of the page
* Bell should show on any page in the website
* On the bell, show the count of new messages
* If there are no new messages, don’t show a number
* Anytime the user gets a new message, add to the count of new messages
* On click:
** Show a list of most recent messages
** Show all unread messages
** Be able to click on a message and be redirected to the message in their inbox
** No delete, edit, paging, sorting, filtering for list
*

## Acceptance Criteria
* Bell shows at top
* User can see how many new messages they have
* User can click on bell to see list of new messages
* User can click on message to go see it in entirety in Inbox

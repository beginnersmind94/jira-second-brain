# NXT-36073 — IM - Recipe Rework - Split View Search changes

## Description
As an Item Management user I want the split view search to be more lightweight so I can simply use it to quickly find what I’m searching for in a condensed format.

*Requirements*

* Update the Split View search component
* Make the width smaller or at least more controllable
** we may want it wider for later applications, but it’s too wide for Recipes
** Update the icons and search to fit this more compact design
** Retain the alert icon that indicates when a Recipe is missing critical information 
** Remove Recipe Category (per [NXT-36068|https://primeroedgenext.atlassian.net/browse/NXT-36068]) 
* Keep the expandable filter area
* Remove the expand sub-area where Serving Size and Allergens currently are shown
** Remove this interaction altogether

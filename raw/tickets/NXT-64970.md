# NXT-64970 — WS - Benchmark - Grid Previous Actuals

## Description
As a Workspace user, on the Benchmark drawer, I want to be able to load previous year’s actual values in the grid next to the benchmark, so that I can avoid having to go back and try and remember

*Use Case*

* Users will typically just want to use the previous year’s results as their benchmark for the next year, so we can show them what that was next to the benchmark so that they don’t have to put effort in remembering or recording it

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* On the grid next to each benchmark, add the actual value for that site and KPI from the previous year
* If they don’t have an actual value, leave it blank
* If the actual value is less than the benchmark, visually indicate that (ex: make it red)
** Note: this means that the user is setting a benchmark higher than actual performance, so we want to warn them because if they couldn’t achieve that performance last year, it might be even harder this year
**

## Acceptance Criteria
* Users are able to see the previous actual values in the grid if they exist next to the benchmark

## Description
+Add Single Item Recipe+

* The user will have to answer these questions with a YES or NO answer.
** Is the item: 
*** Food Item - *Answer is YES*
*** Is it a Single Item Recipe *Answer is YES*
*** Is it a Menu Item *Answer is NO*
*** Is it a POS Menu Item *Answer is NO*

## Acceptance Criteria
* Display sections for user enter to enter information
** Item
** Nutrient
** Contributions
** Allergens
** Recipe




* Use current rules that apply to above mentioned sections
* Use current item name as the default name for single item recipe but allow user to change/edit
* Use the item information to automatically populate Nutrition, Contribution and Allergen information for the single item recipe
* Allow user to enter in information as normal recipe
* Item should be mark as Food-Item
* Set Generation Type to “Single Item Recipe” 
* Serving size and weight should be the same at Serving Measure, Nutrition and contribution Information sections

* *Mandatory fields*:
** *Item Name*
** *Data Source*
** *Recipe Name*
** *Serving Size*
** *Serving Measure*
** *Serving Weight*
** *No. Of Servings*

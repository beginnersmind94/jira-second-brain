# NXT-30306 — Contextual Help Widget - Configurable Settings and Context Data

## Description
As someone who manages the content and behavior of the “Contextual Help Tool”, I would like the ability to define the data needed by the tool in a configurable manner so that I can change the content and behavior during run-time.

## Acceptance Criteria
* The settings that the “Contextual Help Tool” require to function shall be defined in a setting that is accessible by an administrator during runtime.
* The data that provides the context for the “Contextual Help Tool” to determine which content to display shall be defined in a setting that is accessible by an administrator during runtime.

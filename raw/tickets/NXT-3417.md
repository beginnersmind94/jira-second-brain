# NXT-3417 — Template Hub - Actions - Email

## Description
*User Story*
As an Inspections user, I can email a copy of a template to a person or group of people through an action.

*Requirements*
* There is an action for emailing a template
** Action exists on all templates, including Global
* When clicked, a pop up appears
** User is able to select existing groups or manually enter email addresses
* Once sent, an automatically generated email is sent to all addresses with a pdf of the template attached

## Acceptance Criteria
* Email template is present and emails can be sent with templates attached

# NXT-65301 — ACC - Labor Hours Feature - Benchmark Config (target entry per site per month / yearly)

## Description
When the KPI Benchmarks button is clicked a side menu is displayed
showing a table of all sites a user has access to.

Splash text:
Set a Meals Per Labor hour benchmark for reference when viewing Meals Per Labor hour reporting.


Main Pop up:
-Allows the user to define a simple float number per month segmented by academic year.
-A feature to apply an entered number for all months is also available to make configuration easy for users.
-Provide a save and cancel button to save values or exit the side menu without saving.
-Support side scrolling and pagination after 10 sites results.





Special Role Permission added for access to the KPI Benchmarks button
Permission Name: Labor Hours - KPI Benchmarks Access
When permission - User can click the button on the Labor Hours page and can edit table freely only for site the user has access to.
If the user does not have the role permission, disable the button on the Labor hours screen.

Tooltip for disabled button: Access is linked to the Labor Hours - KPI Benchmarks role permission.

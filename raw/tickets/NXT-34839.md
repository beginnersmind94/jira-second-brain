# NXT-34839 — Manage Views - Improve User Experience for Sharing Views

## Description
The Material UI “Autocomplete” component does not provide a good experience as a UI control for sharing views.

The objective of this story is to find a different control or improve the existing control so that it is easier to use.

## Acceptance Criteria
* The Material UI “Autocomplete” component will be assess to determine if there is a different component and/or behavior to make it more user friendly.

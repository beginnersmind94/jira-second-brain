# NXT-20679 — INV - Inventory - Additions - Create Addition - Confirmation

## Description
As a Site Manager I want to remain on the Confirm screen when doing Inventory Additions so I can view the success message.

*Requirements*

* After the user has confirmed the Addition via the Confirm step ([NXT-20678|https://primeroedgenext.atlassian.net/browse/NXT-20678])
** Provide a success message
*** “Addition transaction #123456 has been saved successfully.”
** Display the header in view-only
*** All fields should display as labels
** Remove the Previous and Confirm buttons
** Display a ‘Back to Additions List’ button to navigate back to the main Additions page
* Add quantity to perpetual inventory on click of Confirm
** Create batches based on Lot Number / Date on Product info

# NXT-3331 — Inspection Template - Drag and Drop Functionality

## Description
*User Story*
As an Inspections user editing a template, I can drag and drop questions and categories to reorder them.

*Mockup*
https://nqyc9v.axshare.com/start.html#p=add_new_questionnaire&id=usu4ze&g=1

*Requirements*
* Users can drag and drop questions to reorder or to assign to different categories
** Behavior should be similar to Google Forms (forms.google.com)
** Question is not placed until user releases it
** Display numbers are updated dynamically once the question is placed
* Users can drag and drop categories to reorder them
** When a user is dragging a category, all questions are temporarily minimized so the user only sees categories
** Category is not placed until the user releases it
** Display numbers are updated dynamically once the category is placed

## Acceptance Criteria
* Categories and questions can be reordered using drag and drop

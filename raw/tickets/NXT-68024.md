# NXT-68024 — FO - Reports > Editcheck Worksheet > necessary changes for release

## Description
With the implementation of the Edit check worksheet and new columns we will need to ensure things are working in a way that is useful to the end user.

As such

Change existing checkbox text label to be “Use entered Attendance Counts”

On the report:
For the Attendance Factor column
-Remove value for Sub Total, Grand Total

* Site total will become the average off the highest count for Serving days across all meal types.
Example. Breakfast has 3 serving days Weds - Fri, Lunch has 4 Serving Days M-Thrs - Highest of the serving days = 5, Site total = Average of the Attendance Factor for M-Fri.

This is what the Calculated Attendance Factor should be For the site for the selected date range.
This number is important across 1 month and should be able to be easily referenced for the Claim.

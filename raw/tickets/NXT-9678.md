# NXT-9678 — INV - Create Orders - Edit Order 

## Description
As an inventory user creating an order I need an edit screen so I can modify the orders that have been generated through the Create Orders process.

Edit icon is provided for each generated order on the Summary screen. Clicking the Edit icon would take the user to an order screen that is specific to the Vendor and Delivery Date. (This screen is similar to classic screen.)

Unlike the Summary screen, the Edit screen would be for a single order instance. This means the edit screen would list only the items for a specific vendor and delivery date.

*Requirements*

On the Confirm screen, next to each generated order row, there will be an Edit button next to the Create button.

+Edit Order logic:+

If the user clicks the Edit icon for a particular order, the following will happen:

* the quantity fields (both Whole Unit and Broken Unit) will switch from being labels into being editable fields
* a delete icon will be presented next to each item
* a ‘Add Items’ button/icon/link will be provided ([NXT-5711|https://primeroedgenext.atlassian.net/browse/NXT-5711])
* the Edit button will change into a Save button
** once the user clicks the Save button, the order will revert to the non-editable view
* a Delete icon will appear for the entire order
** if user attempts to delete an order, show prompt:
** “This action will be remove the entire selected order and all items associated with it. Do you wish to continue?”

Only allow user to edit one order at a time:

* when the Edit icon for an order is clicked, hide the Edit icon for all other orders
** this includes hiding the Edit icon for all the items on the other orders
* when the user saves their edits for an order, re-enable/show the Edit icon for other orders

+Tooltips (for Order level):+

* Order Amount validation:
** “Order does not meet minimum purchase amount of $500.00”
** “Order exceeds maximum purchase amount of $2000.00”

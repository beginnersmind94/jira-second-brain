# NXT-50277 —   GDSN Connect - Image Updates - UI Integration

## Description
_(none)_

## Acceptance Criteria
Global Catalog - Pending Updates -(Images Tab)

The system should show the available images for update against the local images

Rules:

The functionality will cover for items that are newly created from GDSN source without any images on the local source

Up to 6 images are allowed at source, while target local source shall take more than 6 images

The available images are selected by default

Upon clicking an Image, user can view the preview of the image available at source (public image directly rendered in UI

The images that are already taken is no longer listed under available updates

Updated images are available in local item (the target update API shall create thumbnail and actual image)

Once all the images are taken, the images & docs tab can be disabled

# NXT-43940 — IM - Images - Migrate old images into new Image & Docs

## Description
As an Item Management user I want the existing images I have for items, menu items, etc. to be migrated so I can view them in the new Images & Documents tool.

May require a utility which downloads existing images, and then reupload and save those images into the new component.

## Acceptance Criteria
* Migrate the existing images assigned to Items, Recipes and Menu Items into the new universal Images & Documents tool 
* Set the Image Option flags according to existing image type (Menu Image > ‘Preferred Menu Image')

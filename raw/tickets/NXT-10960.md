# NXT-10960 — Dashboard (Insights) - Framework - Central vs Site View

## Description
As a site manager I want to see the dashboard display information for all of the widgets based on the data for my site only.

All widgets should have the ability to narrow down to the user’s site. If the user has access to multiple sites, we should give a site selector so that it will modify the information based on the site that is chosen.

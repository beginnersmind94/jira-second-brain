## Description
h1.

## Acceptance Criteria
Manu Navigation:

PE: MP- GC - Pending Upload Screen

Currently system do not filter out the data source IDs for comparison



Change:

Upon refreshing the data in Pending Upload Screen, the system shall compare the selected data source against the records of same data source (Ex: Selecting Local will only compare the local records in GDSN Connect table)

Only Selected data source shall be populated when the user clicks on refresh screen 

“Available Changes” UI Label Change - the number of records scanned will reflect the number of record of the selected data source, and show only number of available changes for the same corresponding data source selected

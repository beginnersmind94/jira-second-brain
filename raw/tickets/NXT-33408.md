# NXT-33408 — Photo Import - Make Exception Grid Pretty

## Description
The exception grid that shows failed photo imports needs to look more professional.  This story is to seek input from the UX team for the UI treatment.

## Acceptance Criteria
* The exception grid shall be per the UX team input.
* In the scenario where there are some errors and the import process continues, please display “Not all files were imported.  Please see the table below for more details.”
* In the scenario where the import is aborted, please display “The import process was aborted due to the severity of issues encountered.”

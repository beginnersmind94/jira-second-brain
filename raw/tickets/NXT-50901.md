# NXT-50901 — Login - SubStory - Favourites & Messages Bar

## Description
This story is a child story for the parent [https://primeroedgenext.atlassian.net/browse/NXT-50530|https://primeroedgenext.atlassian.net/browse/NXT-50530|smart-link]

Below are the items that needs to be implemented as part of this story

* Messages
** Display the number of pending messages
** Redirect the user to the dedicated Inbox page (/workspace/inbox)
* Tasks
** Display the number of pending tasks (combined My + Recommended tasks)
** Redirect the user to the dedicated Tasks page (/workspace/tasks)
* Starred Reports
** Only display this if the user has permission to access Custom Reports or view distributed reports
** Display the number of pending reports (combined Starred + Distributed reports)
** Redirect the user to the dedicated Custom Reports page (/reports/Custom-Reports)



* Your Favorites
** If development on this feature is complete, then display the user’s top-3 favorites (in any/random order) here, which they can click to be redirected. If not, then hide this section.
** Display a button to view more. This can open a popover, drawer, or other component which allows users to view their other favorite pages and click into them to be redirected.

## Acceptance Criteria
Able to display messages strip and favourite strip

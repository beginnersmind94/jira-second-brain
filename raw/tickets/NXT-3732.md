# NXT-3732 — Inspection Template - Scoring Change Rules

## Description
*User Story*
As an Inspections user creating or editing a Template, I have well defined rules for scenarios where I make a change to how the Template scoring is, and have appropriate behaviors and warning messages as needed while attempting to change scoring.

*Requirements*

* User has the ability to change Scoring setting when creating a new Template
** Once the Template has been created and saved, the user can no longer change the scoring
* If no questions have been created, user can change as many times as they want with no message or consequence
* None to Custom Scoring
** If user has created questions with scoring set to None and changes to Custom Scoring, a message appears
*** "You are switching to Custom Scoring. Scoring must be set for individual questions to apply. Are you sure you want to proceed?"
*** Options:
**** Confirm - Scoring setting is changed and users can continue adding questions or set scores for existing questions
**** Cancel - Scoring setting remains on None, no changes made
** No question settings or content is changed
* Custom Scoring to None
** If user has created questions with Custom Scoring and changes to None, a message appears
*** "You are switching from Custom Scoring to None. Please note that any scores you entered for questions will be permanently deleted. Are you sure you want to proceed?"
*** Options:
**** Confirm - Scoring setting is changed and users can continue editing the template. All score data is deleted
**** Cancel - Scoring setting remains on Custom Scoring, no changes made to questions
** All question fields other than score remain the same
* Pass/Fail to None
** If user has created questions with Pass/Fail and changes to None, a message appears
*** "You are switching from Pass/Fail Scoring to None. Your Inspections will no longer have a score applied and other question types can now be added. Any score settings will be permanently deleted. Are you sure you want to proceed?"
*** Options:
**** Confirm - Scoring setting is changed and users can continue editing the template. All score data is deleted. Questions are carried over from Pass/Fail
**** Cancel - Scoring setting remains on Pass/Fail Scoring, no changes made to questions
** Pass/Fail questions are all maintained and other question types can now be added
* Pass/Fail to Custom Scoring
** If user has created questions with Pass/Fail and changes to Custom Scoring, a message appears
*** “You are switching from Pass/Fail scoring to Custom Scoring. Scoring must be set for individual questions to apply. Are you sure you want to proceed?”
*** Options:
**** Confirm - Scoring setting is changed and users can continue editing the template. Questions are carried over from Pass/Fail
**** Cancel - Scoring setting remains on Pass/Fail Scoring, no changes made to questions
** Pass/Fail questions are all maintained and other question types can now be added
* Custom Scoring to Pass/Fail
** If user has created questions with Custom Scoring and changes to Pass/Fail, a message appears
*** “You are switching from Custom Scoring to Pass/Fail. Any questions that are not Pass/Fail question type and any assigned score settings will be permanently deleted. Are you sure you want to proceed?”
*** Options:
**** Confirm - Score setting is changed to Pass/Fail and users can continue editing the template. Any Pass/Fail questions are carried over. Score settings and non Pass/Fail questions are deleted.
**** Cancel - Score setting remains on Custom Scoring, no changes made to questions
* None to Pass/Fail
** If user has created questions with None Scoring and changes to Pass/Fail, a message appears
*** “You are switching from None to Pass/Fail. Any questions that are not Pass/Fail question type will be permanently deleted. Are you sure you want to proceed?”
*** Options:
**** Confirm - Score setting is changed to Pass/Fail and users can continue editing the template. Any Pass/Fail questions are carried over. Non Pass/Fail questions are deleted.
**** Cancel - Score setting remains on None, no changes made to questions

## Acceptance Criteria
* Score settings are properly changed according to the rules set forth in the requirements
* Score setting cannot be changed once a template has been created

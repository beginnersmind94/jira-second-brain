# NXT-3885 — Item Management - Recipe - Save Menu Item

## Description
*User Story*

As a user, I want to be able to save my menu item information and have it validate and save

*Mockup*

!image-20200929-024405.png|width=100%!



*Requirements*

* Menu Item name (Required)
** Validation Message
*** Menu Item Name is required. Please….
* Marketing Name (Optional)
* Marketing Description (Optional)
* Menu Item Category (Default - Unspecified) - Required
* Leftover Category (Default- Unspecified)
** Carryover
** Waste
** Return to Stock
* Maximum Carryover Days (Required - Only when ‘Carryover’ or ‘Unspecified’ is selected in Leftover Category)

Upon save, if the user doesn’t have Menu Item Name or Maximum Carry over days if the toggle is selected, then throw an error. If the toggle is unselected, then those values will not be saved.

## Acceptance Criteria
User can save the Menu Item information correctly on the main recipe when editing or adding a recipe

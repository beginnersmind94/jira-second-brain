# NXT-212 — System - Configuration - Special Assistance Programs - CEP Student Analysis

## Description
Configure special assistance for the Community Eligibility Provision. Allows for determination of sites that can offer meals at no cost to students. Claims are calculated by percentage for Free or Paid meals.

The user will be stepped through the process and can either Import the data (ver 2 dependent on platform), or enter by hand.

*Requirements:*

* From the System>Settings>Management the Special Assistance Programs can be launched
* The CEP option can be selected and the user stepped though the process.
* They can select the Configuration level District, Site, or Site/CEP Group
* They can select Annual
* They can select the Meal Types
* They can set the Student Count Date
* They can enter the Identified Students and Enrolled Students (Per Site) the Identified Student Percentage (ISP) is calculated,
* The Qualified %, Free %, and Paid % are calculated based on the Configuration Level. if excepted they can be set and saved.
* The Start and Stop year can be set.

## Acceptance Criteria
CEP is a configuration option for Special Assistance Programs
CEP can be configured by district
CEP can be configured by site
CEP can be configured by site and CEP Group(s)
CEP can be configured by Annual data
CEP can be configured for SNP - Breakfast, Lunch, Snack
CEP can be configured for SFSP/SSO - Summer Breakfast, Summer Lunch, Summer Snack(AM), Summer Snack(PM), Summer Supper
Identified students are entered for each site, (Even when District)
Enrolled students are entered for each site

The ISP % =  Identified Students/Enrolled Students * 100
*Then, carry the calculation to four decimal places using standard rounding (four or less, round down; five or more, round up). Then, multiply the result by 100 to calculate the percentage to two decimal places.
* For example, a school has 550 identified students and 700 enrolled students. 550/700 =
 0.78571429, which rounds to 0.7857. Multiply by 100 to calculate an ISP of 78.57 percent.
 The ISP, which may not be rounded to fewer than two decimal places, must be at least 40.00
percent for the entity to be eligible. A percentage of 39.98 percent does NOT meet the threshold.

District Qualified ISP% = total of Identified Students/Total Enrolled Students
Site Qualified ISP% = site Identified Students/site Enrolled Students
CEP Group Qualified ISP% = total of sites in group Identified Students/total of sites in group Enrolled Students

Qualified % is assigned per site.
Free% = Qualified % * 1.6  (Cap at 100)
Paid% = 100 - Free%
Free% should never be greater than 100. 
User can select if site is included in CEP program
Each Site can be saved.
Summary is displayed and Start/End dates can be selected

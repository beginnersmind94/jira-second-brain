# NXT-22704 — INV - Vendors - Validations

## Description
As a FSD I want validations when managing my vendors so I can properly make changes.

*Requirements*

* Implement rules regarding the relation of vendor status to contract status, open orders, etc.
* Rules
** A vendor cannot be made Inactive with any Active contracts. All contracts must be in a different status
*** “This vendor has active contracts. Please inactivate all contracts to inactivate the vendor.”
* Alerts
** alert when making a vendor inactive if there are open orders for that vendor
*** “There are open orders for this vendor. Are you sure you want to inactivate?”
* Fix existing issues:
** Create Order - items from inactive vendors appear in the grid if you select all vendors
*** probably Quick Receipt as well
* Change appearance of Vendor User Card buttons to resemble the layout on the Info card
** Save/Cancel both on right side

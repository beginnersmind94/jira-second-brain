# NXT-40334 — eOrders – Generate Part 2

## Description
This is a continuation of the eOrders – Generate Part 1 and covers the Selection Tab

*Selection Tab*

* If the tab hasn’t been created already, then create it
* The tab is used for selecting which existing orders to include in the eOrder
* Tab Title = Selection
* 
* Filters section:
* All are optional
* Order #
** Textbox
** No default value
** Unique ID of Orders
* Vendor*
** Single select list
** Required
** Default to none
** Values: list of active vendors
** Note: they must select a Vendor or nothing else will work
** On selection, load the Sites that are linked to the Vendor in the eOrder Configuration
* Site
** Multi-select list
** Default no values in the list until the user selects a Vendor
** Once they select a vendor then load the sites
** Values: list of sites linked to the vendor in the eOrder Configuration
* Order Status
** Multi-select list
** Default to Open
** Values: list of Order Status values [same list as Status filter on View Orders page]
* From*
** From Delivery Date
** Date picker
** Required
** Default to today minus 14
* To*
** To Delivery Date
** Date picker
** Required
** Default to today plus 14
* Reset
** On click, clear the filters back to their defaults and regenerate the grid
* Apply
** Load the grid with those filters applied
* 
* Below the filters is a grid:
** Place a note as a tooltip or small text above it: Select orders to include in the eOrder
* Columns:
* Order #
* Site
* Delivery Date
* Total
** Dollar amount of the order
* Last eOrder Sent
** Date and time the last time the order was sent as part of an eOrder
* Select All
** Need the ability to click on a Select All button that when clicked, selects all the select checkboxes in the grid
* Each row has a select checkbox
* 
* At the bottom of the tab, have these buttons:
* Previous
** On click, close the screen and go back to the dashboard
** Don’t ask them if they are sure, just do it
* Next
** Validate the following:
*** They must have selected a Vendor
*** They must have selected at least one Site
*** They must have selected at least one row in the grid
** If fail validation, show the user an error message and show the user which fields need correction
** Open the next tab
** Save state on the screen in case they come back from the next tab

# NXT-64956 — WS - Benchmark - Bulk Update

## Description
As a Workspace user, on the Benchmark drawer, I want to be able to update all values on the grid simultaneously, so that I can speed up the process and get more accurate data

*Use Case*

* Users might load the previous year’s results as their benchmark for the next year and then just want to add a small incremental increase to set as the new goal
** For example, maybe they had 75% Breakfast participation last year and so want to set the benchmark at 76%, so they can go into the bulk update and put a 1 for breakfast and update all the sites by one

*Requirements*

* This story is for the benchmark configuration page
* It applies to both SchoolCafé and PrimeroEdge Districts
* Above the grid, add a button
** Use Icon button so that you don’t have a huge button on it
** Label: Bulk Update
** on click:
*** Show a modal popup:
**** Get fields and text from the Bolt screen
*** Allow users to enter a whole number
*** Can be positive or negative
*** Can leave blank which means ignore
* If entered a value and clicked Submit:
** For each site for each KPI with a value:
*** Add or subtract that value to the value that is present
*** If the subtracting more than what is in the text, make it zero (no negative numbers
* Note: user will still have to click Save on the screen to commit the changes

## Acceptance Criteria
* Users are able to make incremental bulk changes to the grid values

# NXT-17424 — AI Data Insights - Application AI capability for Eligibility and Participation

## Description
As a system, AI Insights would require AI capability application to achieve end-to-end AI implementation.


*Description:*
App AI capability design and process allows to integrate AI model intelligence into the application. This design and process integrates interfaces required for district admins/school admins and parents. 

*Application interfaces*

# AI actionable item interface – for District and School
# Online F & R Meal application - for Parents
# Online Income Data verify – for Parents
# Online F & R application status verify – for District, School and Parents



* *AI actionable item interface(s)*
Application action items for District/School admins using predictive insights. AI/ML model recommendation will evaluate and provide potential forecast metrics & data for Districts and Schools 
For District/School admins
a.	Email campaign to parents 
b.	Physical email to parents 
c.	Reminders and Alerts 
d.	Other



* *Online F & R Meal application*
Application action item for Parents. Complete online application for eligibility
* *Online Income Data verify*
Application action item for Parents. Complete online income data entry and verify eligibility
* *Online F & R application status verify*
Parents can verify application status online. Central office and School’s view application and approve.

## Acceptance Criteria
•	The system shall automate end-to-end workflow Integrating AI/ML services and AI capable application modules
•	The system shall integrate AI/ML forecast based metrics and data output
•	The system shall outreach district/schools based on AI/ML participation forecast rate
•	The system shall create relevant actionable events for districts/schools based on AI/ML recommendation for Eligibility and Participation
•	The system shall outreach parents based on AI/ML eligibility recommendation
•	The system shall generate quantitative feedback scores for each actionable event
•	The system shall collect feedback data KPIs and Metrics for data driven analysis, measure performance and evaluate success standard rate

# NXT-9935 — INV - Transfers - Transfer Request - Selecting Items - Step 3

## Description
+*Item Selection*+

* The items returned by the search criteria will be populated in a below grid
* This section will be contained to a certain size and use pagination so it does not take up too much vertical space on the page
* The user will have checkboxes for each item they wish to select and add to the Request List
* Items will be listed with the following attributes:
** Item #
** Item Name 
** Item Description
** Storage Category
** Item Category
* Add button
** this will add the selected (checked) items to the Item Request List

+*Item Request List*+

* On the next step, the user will be provided with a list of the items they have specifically added to the Item Request List
** the Site will make some request quantity for each of these items
*** the request functionality will be different depending on whether or not the user has the Other Sites QOH permission
* +Item List:+
* Item# 
* Item Name
* Whole Unit 
** Item Whole Unit description / abbreviation
* Broken Unit
** Item Broken unit description / abbreviation
* +*Request Entry Field*+
* if the user *does not* have the Other Sites QOH permission, they will see two additional columns:
** WU Request Qty
*** numeric entry field for user to enter requested whole unit quantity of that item
** WU Desc
*** Pulls from Item Management. If empty, then show blank
** BU Request Qty
*** numeric entry field for user to enter requested broken unit quantity of that item
** BU Desc
*** Pulls from Item Management. If empty, then show blank
** Comments
*** Handled in [https://primeroedgenext.atlassian.net/browse/NXT-9936|https://primeroedgenext.atlassian.net/browse/NXT-9936|smart-link] 
** Delete
*** Trash can icon by the icon
*** Allow the user to click the button. This will remove the item from the request. When clicking trash, ask for confirmation
**** Are you sure you want to delete this item?
***** Yes
***** No
* Next button
** If user does not have Other Sites QOH permission and has entered their request quantities, the Next button will navigate to the Confirmation screen ([NXT-9408|https://primeroedgenext.atlassian.net/browse/NXT-9408]) - *Default behavior as of now. No permissions in place*
** If user has not entered a quantity, display error
*** “Please enter quantity.”
*** Display in red and in the top right. See 3rd mockup

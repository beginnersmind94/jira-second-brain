# NXT-31743 — IM - CNDB Items - Manage Discontinued CN items - only DB Script (6.2 only)

## Description
As a District Admin I want to be able to update all my recipes if one or more of the ingredients are CNDB items which were discontinued in a CN update, so I can review value changes before updating the recipe.

*Note:* this does *not* need to be checked into 4.1 like the other CN25 stories. This only needs to be done in 6.2+ (current version).

*Requirements*

* Update CNDB Items to consider discontinued items in the [CN25 Update |https://www.fns.usda.gov/tn/child-nutrition-database]
** Update all relevant CN items as ‘Discontinued’
** Identify all recipes using Discontinued CN items
** Create a copy of each discontinued CN item in the Local data source
** Replace the CN item on all affected Recipes with the new local copy
** Notify on the Recipe that ingredients were replaced

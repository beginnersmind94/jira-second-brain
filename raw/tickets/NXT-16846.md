# NXT-16846 — Copy Jobs

## Description
This story is to cover the tasks related to Copy Jobs. 

This feature will provide the user to copy any of the existing configuration.

Copy Job feature is to save time to the end user by copying and creating the existing job (duplicate of the copied job parameters) upon selection to save time and eliminate user mistakes.

## Acceptance Criteria
# Verify and Validate that the user can copy/duplicate an existing job.
# Verify and Validate that the copied job is immediately created as the next job at the bottom of the list on the landing page.
# Verify and Validate that the newly created job is tagged with copy at the end
## Incase the job is copied twice, the next one will be tagged as copy 2 and next one as copy 3
# Verify and Validate that the newly copied job is always in disabled mode.
# Verify and Validate that the newly copied job name can be edited as per user requirement.
# Verify and Validate that the newly copied job parameters can be altered as needed.
# Verify and Validate that the old job is not altered when the newly copied job is modified.

# NXT-43075 — INV - Orders - Receipt Details - Move Charges & Discounts to slide-out

## Description
As a site manager I want the Fee and NOI fields to be managed in a slide-out so they do not interfere with the main receipt screen, since item level charges and discounts are not commonly used.

## Acceptance Criteria
* Move the Charges & Discounts to a slideout for both the main receipt and for individual items
* Open the slideout by clicking the new $ icon
** ensure correct icon is being used
* Maintain a view-only and edit version of the slideout
* Display the Net Charge/Discount
** calculate and display the total +/- impact of the charges & discounts
* Display the item level version of the slide-out for individual items
** Fields should be disabled by default
** Future permissions will allow users to edit

# NXT-9329 — Add the Menus with respective to Meal Pattern and Meal Types -UI

## Description
while adding the Menus for USDA Meal Pattern SY 2014+ Meal Pattern, Serving groups should show as per the meal types selected.

## Acceptance Criteria
while adding the Menus for USDA Meal Pattern SY 2014+ Meal Pattern, Serving groups should show as per the meal types selected.

While adding the Menus for USDA Meal Pattern SY 2014+, follow the below conditions.

1) PK should been shown for all Meal types [ Breakfast, Lunch, Supper, Snack ]
2) K-5, K-8, 6-8, 9-2, Adult and Ala Carte serving groups should be shown for following Meal Types [ Breakfast, Lunch & Supper ]
3) K-12 Serving group should be shown for only Breakfast & Snack Meal Types

While adding the Menus for USDA CACFP & USDA SFSP Meal Patterns, serving groups should be shown for all the meal types [ Breakfast, Lunch, Supper & Snack ]

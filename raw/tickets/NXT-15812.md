# NXT-15812 — IM - Display Serving size under Nutrient and Contribution Info 

## Description
As a User i would like to have an ability to View the Serving Measures and Serving Sizes of an item

## Acceptance Criteria
Given the User is on items landing page 

When an item is selected from the Search result 

Then Display the Serving Measures in Nutrient Info as below 

* Serving Measures 
** Serving size 
** Weight in Grams 
** Preferred Measurement 
** Possible Measurement

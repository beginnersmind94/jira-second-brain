# NXT-14472 — [AcctMgt] - Special Instructions - Increase input and display length from 250 to 500 chars

## Description
SchoolCafe now allows for input of 500 characters. We need to update Account Management configuration and POS to accept and display larger information. Story can be split for each module if needed.

Account Management > Accounts > Account Details > Restrictions (section) > Special Instructions (field). When editing, we need to increase from 250 to 500 to allow for SchoolCafe family portal information to not be trimmed.

!image-20210510-171711.png|width=1495,height=809!

## Acceptance Criteria
* Account Management > Accounts > Account Details > Restrictions > Special Instructions should accept input of 500 characters
* Account Management > Accounts > Account Details > Restrictions > Special Instructions should display values entered from SchoolCafé up to 500 characters

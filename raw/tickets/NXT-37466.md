# NXT-37466 — Student Export – Add Additional Fields - Dev

## Description
As a SchoolCafé Admin I want to be able to pick and choose which fields I will export on Student Export so that I can have more control over my export.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Exports
* Go through the Pre-export and get to the Field Mapping screen
* Pick which fields you will export and then finish the export

*Notes:*

* We already have selected some predefined fields on the Student Export, but want to change it so it is flexible and users can pick for themselves which ones they want

*Requirements*:

* When in Student Export on the Field Mapping Screen, do the following:
* Ensure these fields are in the list of available fields for export
** Order: StudentId (already there), StateId, PIN, SSN, FirstName, LastName, MiddleName, BirthDate, Gender, SchoolCode (already there), GradeCode (already there), Ethnicity, Race, Language, AddressLine1, AddressLine2, Apartment, City, State, Zip, HomeroomTeacher, HomeroomNumber, GuardianFirstName, GuardianLastName, GuardianHomePhoneNumber, GuardianWorkPhoneNumber, GuardianCellPhoneNumber, Email, GuardianEmail, Notes, Eligibility (already there), Reason (already there), AccountBalance (already there), EligibilityEffectiveDate (already there), AccountBalanceDate (already there)
* Auto-map all fields like currently do
* 
* When exporting:
* Do like you currently do, no changes
*

## Acceptance Criteria
* Additional fields designated above are added to the list of available fields
* When exported, only the enabled fields are exported

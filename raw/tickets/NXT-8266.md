# NXT-8266 — INV - Physical Inventory - changes to Reconciled screen - UI

## Description
As an inventory user I want the Reconciled inventory screen to have a dedicated Items tab so it is more similar to the other inventory status screens.

*Requirements*

Reconciled screen should be identical to the Pending Reconciliation screen, as explained in 8115 & 8116

* Storage Category selection should be available
* Items should be on a dedicated Items tab
* unlike Pending Reconciliation, the buttons will be unavailable once the inventory is Reconciled
* Message is displayed at top of the screen:
** “This physical inventory is reconciled. Changes cannot be made.”

## Acceptance Criteria
Reconciled screen should reflect the changes made on Pending Reconciliation screen
Summary tab will have the details section
Summary tab will have the Select Storage Category section

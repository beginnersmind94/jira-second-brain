# NXT-35697 — IM - Items/Recipes - Split View Component R&D

## Description
As an IM user I want the split view search to be docked/floating so I can use the side panel regardless of where I am within the Item/Recipe.

This is an R&D task, which will assist with the upcoming Item Type Consolidation epic.

Reference locations:

* IM > Recipes
* MP > Menu Planner (different component but similar style of docked side panel)

*Requirements*

* Research necessary changes to improve the split view search side panel
* Retain existing functionalities with respect to expandable additional filters, item previews, item selection, etc.
* The search panel should be docked/floating, so that it does not scroll with the whole page
* The search panel should be collapsible so user can transition into full screen view for Item/Recipe Details

# NXT-5342 — System - Setting Settings - Add setting descriptions

## Description
As a district admin, I would like to be able to see what each system setting does, so that I can make an informed decision as to whether I want to update a setting’s value or not.

+Summary:+

Display the setting description along with each setting on the System Settings page.

+Details+:

* In the System Settings page, next to each setting, display the setting description.
** Descriptions themselves can be carried over from Classic.
** Description can be displayed in tooltip form, like in the earliest mock-up.

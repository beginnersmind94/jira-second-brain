# NXT-69606 — SchoolCafe Testing & Utilities Tool

## Description
Develop a comprehensive web-based testing and utilities tool for SchoolCafe that provides:

* *HTTP Performance Testing* - Test API endpoints with detailed metrics (response time, TTFB, throughput, P95/P99 latencies, concurrent requests)
* *SQL Performance Testing* - Analyze database query performance with execution metrics, warmup runs, and optimization tips
* *AI-Powered PR Review* - Automated Pull Request analysis against React/TypeScript coding standards with Azure DevOps integration
* *React Code Review* - AI-powered code analysis with best practices checking and suggested refactored code
* *Dashboard* - Central overview of all available tools with feature summaries
* *Azure App Service Deployment* - Production deployment to Azure

*Tech Stack:*

* React 18 + Vite (Frontend)
* Node.js/Express (Backend proxy server)
* OpenAI APIs (AI analysis)
* MSSQL support (Database testing)
* Azure App Service (Hosting)

## Acceptance Criteria
*HTTP Performance Test:*

* User can test API endpoints via URL input or cURL command paste
* Supports GET, POST, PUT, DELETE, PATCH methods with custom headers/body
* Displays response time metrics: min, max, avg, median, P95, P99
* Shows TTFB, throughput, and success rate
* Supports configurable concurrent requests and iterations
* Provides AI-powered performance recommendations

*SQL Performance Test:*

* User can connect to MSSQL database with server/database/credentials
* Executes queries with configurable iterations and warmup runs
* Displays execution time statistics and rows returned
* Shows throughput (queries/second) and consistency metrics
* Provides AI-powered query optimization suggestions

*PR Review (Azure DevOps):*

* User can input Azure DevOps PR URL and PAT token
* Fetches and analyzes code changes from PR
* Validates against 24 coding standards (critical/high/medium severity)
* Checks accessibility (WCAG 2.1 AA), error handling, performance patterns
* Returns recommendation: APPROVE, REQUEST_CHANGES, or COMMENT

*React Code Review:*

* User can paste React code for analysis
* Returns overall quality score (0-100)
* Identifies issues by severity (critical/high/medium/low)
* Checks hooks usage, accessibility, security, performance
* Provides suggested refactored code

*Dashboard & UI:*

* Dashboard displays all tools with descriptions and feature lists
* Responsive UI with side navigation
* Rate limit display for API usage tracking

*Azure App Service Deployment:*

* App Service created with Node.js 20 LTS runtime
* Environment variables configured (ANTHROPIC_API_KEY, OPENAI_API_KEY, PORT)
* Production build deployed
* Application accessible via App Service URL
* CORS configured for production domain

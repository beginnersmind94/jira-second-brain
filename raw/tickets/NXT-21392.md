# NXT-21392 — POS - Allow users to view/print Receipts

## Description
As a cashier or manager, I need the ability to print out receipts via the POS  so that they have a financial record and paper trail for  any transaction.

*Requirements:*

* Allow users to print a receipt from the POS > Transactions > Transaction Details area.
** Print Preview.pdf shows the Edge version of a receipt - this is fine to re-use.
* The width of the text being sent to print should not exceed 58mm to allow for receipt printers to be used.
* If the POS Setting ‘Print All Transactions’ is on, then the full transaction (items + payments) is printed.
** Otherwise, only Payments are printable.
* If the system setting ‘Print Payment Receipts Automatically’ is on, send a receipt to print after a payment is successfully processed.

## Acceptance Criteria
Print button available in Transaction Details, sends a receipt to print.

Receipts can be printed on receipt printers. (Hanu has one in the office.)

System setting lets payment receipts print automatically if turned on.

POS setting pulls in all transactions if turned on.

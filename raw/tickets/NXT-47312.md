# NXT-47312 — Files – View Config Details

## Description
As a SchoolCafé admin, I would like to be able to view the configuration details in one step so that I don’t have to click through the configuration every time.

*Note:*

* We already have this type of functionality in other system pages, so this should be consistent with that
* We can leverage what we do on the Review tab to make this quicker

*Requirements*:

* For all rows on the Files tab’s grid, do the following:
* In the ellipsis, add a button to the actions:
** Title: View Configuration
** On click, open the configuration drawer
* 
* Create a new drawer
** Title: File Import Configuration
** Size: same as in other system screens
** This is a rea-only screen, no edit
* 
* Use the component we created on the Review tab and place it in this screen
* Instead of a textbox for name, just show the name (no editing
* On page load, show the configuration of the File import

## Acceptance Criteria
* User can see the configuration of the File import by clicking the View Configuration button on a row
* All the config data that is shown in the Review screen is shown on this page in read-only format

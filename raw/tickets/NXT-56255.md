# NXT-56255 — DMF - Rerun Source Migration when Destination Migration fails.

## Description
When Destination DB migration fails then, we need to manually run SQL scripts to update the Source DB Migration to failed status so that we can re-run Source DB migration.
We need to re-run the Source DB migration to incorporate all the fixes that caused the Destination DB to fail.
Show a re-run icon near the Done button for the Source DB Migration.
For example, when the Eligibility Destination DB fails, then show the re-run button for the Source DB of Eligibility only.
Need to do the same for System and Item Management.



!image-20241028-224231.png|width=1558,height=321,alt="image-20241028-224231.png"!

When the user clicks the Source DB Rerun, the Source DB Rerun will be in the In-Progress state and the Destination DB will be in Not Started state. (This functionality already exists)

# NXT-15018 — Acct.Mgmt - Outreach - Campaigns - Manage existing campaigns

## Description
As a central office admin, I would like to be able to view, update, or delete any active outreach campaign within my district, so that I can easily make changes if the details of my campaign need to be modified.

*Summary*

* In the Campaigns tab of the Outreach Center, show all active and completed campaigns, allowing the user to modify their details and set them as active or inactive. 

*Details*

* In Account Management → Outreach → Campaigns, add a grid that shows:
** Campaign Name
** Template Name
** Start Date
** End Date
** Frequency
** Action(s)
*** Edit
* Allow the Edit button to open an Edit Campaign drawer that allow the user to: 
** update any of the details listed in the grid, 
** update any of the filters that are shown in the Broadcast Message grid:
*** Template Name
*** Patron Type
*** Eligibility
*** etc. (Display all of the filters)
*** Make the Start Date field disabled unless it is at a future date and date.
** use a toggle button to set the campaign as active or inactive, 
*** We may add a Status column and show all campaigns later later
** save the changes with a Save button.
* When the user saves the changes, update the campaign so that future campaign broadcasts go out according to saved filter/template details.

## Acceptance Criteria
Verify that the Campaigns tab displays a list of active and inactive campaigns for the district

Verify that the campaign details can be edited and saved

Verify that subsequent messages sent out (if the campaign is active) are done so with the latest filters

Verify that campaigns made inactive, no longer send out messages

# NXT-60360 — IM- Item Onboarding- Pack Size Info- GTIN/UPC

## Description
As a district admin using the bulk onboarding tool, I would love a way to bulk add my GTIN/UPC codes for all the packs on all the items I am trying to create.

!image-20250429-071916.png|width=1363,height=735,alt="image-20250429-071916.png"!

## Acceptance Criteria
* The “GTIN/UPC” button from the Review Pack Size info screen will lead to this step
* Provide a text box for all the pack units that are available per item to enter GTIN/UPC code
** Each Pack can have their own code
** Provide Ghost text with the description 
** Show an identifier on the pack overview on the pack that has GTIN code added
* Only show items that have full pack size information
* This step is completely optional and will not trigger any alerts
* Once user has entered information, validate number of digits to ensure a proper UPC/GTIN format

# NXT-14652 — POS - Menu Grid - Support for expanded grid size (16x15)

## Description
As a user I want the ability for larger size menu configuration within the POS so that I can fit more items.

The goal was to previously support 16x15 grid. However, the library chosen for the POS did not support this capability. We need to research what it takes to support the layout within the POS after it has been configured within Accountability > Settings > Menu Grids.

We should explore using a simple calculation and percentage logic to size the button and their placement.

## Acceptance Criteria
* Grid should support up to 16 (columns) x 15 (rows) within the menu grid area without horizontal or vertical scrolling
* The size and positioning should correlate exactly to the look and location of the menu grid configuration done within Accountability > Settings > Menu Grids.

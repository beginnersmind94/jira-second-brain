# NXT-5716 — INV - Orders - Review Order Settings - List 

## Description
As an Inventory operations manager I want a list of currently configured Review Order Settings so I can see my current rules for order reviews.

*Requirements*

* Create a new page titled ‘Review Order Settings’
** this will contain the list of existing settings and will also provide a way to add new settings
* Provide an ‘Add’ button that will navigate to the rule creation stepper
* Provide a ‘Rules List’ grid that lists all currently configured Review Order rules
** Columns:
*** Site
**** display the site name if only a single site was specified for the the rule
**** display ‘All’ if no site was specified in the rule creation
**** display the number of sites if multiple sites, but not all sites, were specified in the rule
***** provide a popup displaying the list of sites
*** User
**** display the name of the user if there is only a single user assigned
**** display a user icon if more than one user was assigned to the rule
***** clicking the icon would display the user (First Name, Last Name, Email) in a popup
*** Vendor
**** display the vendor name if only a single vendor was specified in the rule creation
**** display ‘All’ if no site was specified in the rule creation
**** display the number of vendors if multiple vendors, but not all vendors, were specified in the rule
***** provide a popup displaying the list of vendors
*** Purchase Amount
**** display N/A if no Purchase Amount was added to the rule
**** if only a minimum Purchase Amount was specified, display ‘Greater than $500' 
**** if only a maximum Purchase Amount was specified, display ‘Less than $500’
**** if a minimum and maximum were specified, display the range as ‘Between $200 - $400’
*** Items
**** display the Item Category if only a single item category was specified
**** display the number of items if multiple items were specified
***** provide a popup to view the list of items 
**** display the Item Number + Description if a single item was provided
*** Date Range
**** always display the Start Date
***** only the date; no time
**** if an End Date was provided, show the date range: 8/1/2021 - 6/30/2022
**** if no End Date was provided, display ‘From 8/1/2021’ (with 8/1/2021 being the Start Date)
*** Action
**** View icon, to open the stepper for that rule ([NXT-16910|https://primeroedgenext.atlassian.net/browse/NXT-16910])
**** Delete icon, to delete the rule
***** Prompt: “Are you sure you want to remove this review order rule?” Yes/Cancel

## Acceptance Criteria
* Provide an Add Rule button to add a new rule
* Display all configured rules in the list
* Display single values in the cell (i.e. ‘Kampar High Schools’)
* Display multiple values as a number (i.e. ‘4 Sites’)
* Display all values as “All”
* Provide a popup for:
** User - if one or more are assigned to the rule
** Site - if two or more are assigned
** Vendor - if two or more are assigned
** Items - if two or more are assigned
* Provide a view icon to open the rule
* Provide a delete icon to remove a rule

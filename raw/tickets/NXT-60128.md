# NXT-60128 — IM- Item Onboarding- Update Menu Item Options (Step 2)

## Description
As a district admin using the item onboarding tool, I need a way to be able to add all my options for menu items such as categories, leftover information etc.

!image-20250317-203436.png|width=1547,height=911,alt="image-20250317-203436.png"!

## Acceptance Criteria
* Provide an edit mode for “Update Menu Item Options” that includes some of the columns from the review
** Menu Item Name (View Only)
** Menu Item Category (Editable and Mandatory)
** Leftover Category (Editable and Mandatory) default to *Unspecified*
** Max Carryover days (Editable and Mandatory) only appears if Unspecified or Carryover is selected
** Prep Site Item (Editable Check box)
** Show in Summary (Editable Check box)
** Tags (Editable)
* Do not show Items that have “No” selected for “Is this a Direct Menu Item?”
* Do not show Menu items that have selected “Yes” for “Is this a Direct Menu Item?” but have not entered a Menu item Name
* Provide an alert next to item # on a row if user goes through the row and does not enter ALL mandatory information
** Tool tip on alert “Mandatory Information Missing”
* User is allowed to save this page at any point in time whether mandatory information is entered or not, highlight textboxes that have not been entered

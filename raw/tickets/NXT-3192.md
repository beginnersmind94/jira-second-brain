# NXT-3192 — Accounts - Account Details - Balance Adjustment

## Description
*User Story*

As a user I need to make account balance adjustments when a balance is determined to be incorrect.

*Requirements*

* Within More Actions, provide an option for Balance Adjustment.
* Balance Adjustment selection activates the action slide out panel.
* Fields
** Adjustment type group buttons provided are “ADD” and “DEDUCT”
*** Add applies an addition to the balance
*** Deduct applies a subtraction from the balance 
*** By default, do not select an option button, but require a user to make a selection
* Adjustment Amount
** Text box for dollar amount
* Previous Balance
** Label for displaying the current account balance before adjustment
* Adjustment Amount
** Label for displaying the amount entered by the user. If adjustment type is “DEDUCT”, we should display a negative symbol ( - ) before the amount to indicate the amount will be subtracted.
* New Balance
** Previous Balance + Adjustment Amount
* Comment is required is for all adjustments
* Save button will create the transaction and update the balance to the new balance
* Cancel button and Close icon allow the user to cancel the adjustment and close the action panel
* Upon save, the action panel closes and the balance on the account details should be updated and visible to the user in a lazy loading fashion (should note require refresh of the page)
* An uneditable transaction is entered in the account with a transaction type and description of “*Account Adjustment*”.
* A session is created and the transaction is entered into the session (need the story to create or use open session).

## Acceptance Criteria
User should be able to get to the Adjust Balance page through the 'More Actions' button on the Student Info page of an account

Then, the user can select to enter an adjustment.
The user can enter an "ADD" to increase the balance
The user can enter a "DEDUCT" to decrease the balance
The transaction date is the current date
Comment must be entered when a adjustment is made
The transaction type is Account Adjustment
The transaction is added to a session

User can back out of a transaction by using Close/Cancel buttons.

# NXT-56403 — Photos - Serial Imports

## Description
We need to upload photos in serial instead of parallel because the parallel is causing failure

*Requirements*

* Change the photo imports to import in serial instead of parallel everywhere we do photo imports
* On the Add New drawer:
* When the user select Photos import:
** Capture that it is for photos
** Check if there is already a photos import running at that moment for the district
** If yes, show an error message:
*** Message: There is currently a photos import running. Only one photo import can run at a time. Please wait a few minutes and try again.
*** Do not allow the user to proceed in the stepper
** If no, allow the user to continue

## Acceptance Criteria
* Photos are imported in serial
* Existing imports continue to work

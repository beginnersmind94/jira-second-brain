# NXT-17428 — AI Data Insights - Dashboards and Visualization 

## Description
As a system, AI insights needs Reports, Dashboards and Viz to achieve pre analysis, post-AI model implementation analysis and actionable events data driven analytics. 

*KPIs for dashboards:* 
1.	Student Eligibility Rate (SER)
2.	Average Daily Participation (ADP)
3.	Paid Not applied (PNA)
4.	Goal Completion Rate (GCR)
5.	Parent Effort Score (PES)
6.	Participation Increase Target Level (PITL)
7.	Other non-standard KPIs

* Eligibility forecast rate (EFR)
* Participation forecast rate (PFR)
* Revenue forecast rate (RFR)

----

*Requirement:*

+*1. Current value analytics*+

* Current eligibility metrics with filters for all sites, multi sites and single site - Dashboard
* Current participation metrics with filters for all sites, multi sites and single site – Dashboard

# NXT-35642 — MP- Menu Cycle Analysis- Revert Change for Fruit and Veg juice based on Latest USDA communication

## Description
Based on latest USDA communication, they would like for us to continue using the MAX range when calculating Fruit offering and Veg offering. We made the change to include all offering for fruit and veg and they disagreed with that approach. This was discussed before our final resubmission. 

This should effect the Analysis Slide out, nutrition report, and food component report

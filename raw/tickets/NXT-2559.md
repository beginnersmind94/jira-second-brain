# NXT-2559 — Inspections Reports - Failed Questions Report

## Description
*User Story:*
As an Inspections supervisor, I have a report that shows me failed questions by Site, Template, and Date

*Current Version:*

[https://inspectionsrelease.primeroedge.co/#/app/inspections/failedQuestions|https://inspectionsrelease.primeroedge.co/#/app/inspections/failedQuestions]

*Requirements:*

* Failed Questions Report can be run using existing Inspections and available parameters
** User must have Reports role permission to run report
** Report can be created in PDF, Excel, or Emailed
* The following parameters are used (all are required):
** Site - Can select 1 or many
** Template - Can select 1 or many
** Start Date
** End Date
* All Inspections  that fit within the chosen parameters appear in the report, regardless of status
* The following data is displayed on the report:
** Header with Report Name and District
** Each Individual Inspection is separated out into its own section
*** Site
*** Inspection Name
*** Date and Time
*** Template
*** Notes
*** Site Rep
*** Failed Questions with answers and notes listed by Category

## Acceptance Criteria
* Failed Questions Report can be created with available data across all parameters in all formats

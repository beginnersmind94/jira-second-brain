# NXT-19094 — Income Surveys - Add New Survey - Summary

## Description
As a district admin, I would like the ability to quickly and easily add a manual income survey on behalf of a household, so that their income data is entered into our system for reference at any time.

*Summary*

Within the ‘Add New Survey’ function, add a step to help add a new survey record for a set of students and then see the summary of their submission, which they can then export or print if desired, as well as start another survey.

*Requirements*

* When the survey has been submitted successfully, display a *Summary* step
* Display a success toast message: _Survey \{\{surveyNumber}} was submitted successfully\!_
* If possible, display an image of the completed survey that can be printed/exported to PDF (negotiable)
** Display all sections from Review step without edit pencil
** Allow user to export/print a page that shows all of those sections
** Title: Survey \{surveyNumber}
* Display 2 buttons:
** Add another survey
*** Refresh/clear the survey selections
*** Return the user to the Students step
** I’m done for now
*** Return the user to the main Income Surveys grid/page

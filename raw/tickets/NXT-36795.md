# NXT-36795 — IM - Items/Recipes - Enhanced Images card

## Description
As an Item Management user a better Images card so I can manage all the different ways in which I want to use photos for items, recipes, or menu items.

*Requirements*

* Update the UI/UX of the standard Images component for better use cases within Item Management
* Redesign the backend structure of Images to leverage the new ‘Item Master’ concept and house images at that level
* Find better ways to manage common actions
** Add Image
** Delete image
* Find a way to incorporate new options and actions
** Set as Ingredient image
** Set a Menu Item image
** Set as Recipe image

# NXT-34969 — INV - Orders - Standard Ordering - Update Par Level popup for Pack Sizes

## Description
As a Site Manager I want the Site Par Levels page to have a Unit Description dropdown for each item so that I can set the par level quantity in terms of whichever pack size unit I would like to use.

h4. +Popup+ (-will be changed into a general popup/report-)

* Display the Par Level *popup* only on the highest orderable item
* Include all units within the Available Quantity calculations
* Add ‘Required Quantity’ field
** this will always reflect the +exact+ quantity required to get to Par Level
** this quantity will be calculated irrespective of unit orderability
* Update Suggested Order Quantity
** this will take the orderability into consideration and suggest an order quantity which would meet the Required Quantity (basically round up to the nearest orderable unit)

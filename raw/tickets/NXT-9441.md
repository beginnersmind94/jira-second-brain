# NXT-9441 — INV - Transfers - Transfer Detail (Site-to-Site transfer)

## Description
As an Inventory user I want a Transfer Detail page so I can open a transfer from the Transfer List and view the detailed information.

This is the screen that will be displayed for existing transfers and the page that will be accessed when the user opens a transfer from the Transfer List.

The user will click on the Transfer # which will show up as a hyperlink field here.

This Transfer Detail screen will look different depending on:

* Type of Transfer
** Site-to-Site transfer
** Transfer Request 
** District-generated Transfer
* User access/permission
** District user
** Site User
** Receiving Site
*** Approver
** Sending Site

+*Action buttons*+

* Transfer Ticket (report button)
* Price Details (report button)

+*Header Information*+

* Transfer Number
* From Site
* To Site
* Transaction Date/Time
* Status
* Comments

+*Transfer Item List*+

* Item #
* Item Description
* Whole Unit Request Qty
* WU Description
* Broken Unit Request Qty
* BU Description
* From Site
** Site-to-Site transfer: this will always appear
** Transfer Request: this will only appear if the the From Sites have been assigned
* Comments

Mockup differences:

* Header
** Include Transfer #
** Include Status

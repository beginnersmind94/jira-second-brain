# NXT-73303 — FIN — Reverse Import: Opening Balance Import

## Description
Adds Reverse import support for **Opening Balance Import** on the Import & Export Log page (NXT-73286).

OB Import follows the ledger-posting import reverse pattern. One OB Import is active per FY — the latest. Prior imports overwritten by the Restatement Gate cannot be reversed.

**Gates:** Year-End Close (hard block), overwrite (hard block), closed period (prerequisite — reopen to unblock).

**Post-reverse state:** FY has no opening balances. Trial Balance shows $0.00 with no-OB indicator per NXT-71240 sec 7.5. Restatement Gate does not fire on subsequent OB imports for that FY. Deliberately overrides NXT-71240 sec 4.1 (read-only constraint).

All UI patterns (confirmation dialog, mandatory comment, Cancel / Reverse import buttons, Status = Reversed, audit trail) follow NXT-73286.

## Acceptance Criteria
**AC1 — Reverse available on OB Import.** Given the Import & Export Log page (NXT-73286) is built, Reverse import appears on the current active OB Import for a given FY when Status = Done.

**AC2 — One active per FY.** Only the current active OB Import for a given FY shows Reverse. Prior imports overwritten via the Restatement Gate are blocked.

**AC3 — Closed period prerequisite.** If any period affected by the OB Import is closed, Reverse is disabled with tooltip directing the user to reopen the period(s).

**AC4 — Year-End Close hard block.** If Year-End Close has been run for the affected FY, Reverse is permanently disabled.

**AC5 — Overwrite hard block.** If the OB Import was overwritten by a later import via the Restatement Gate, Reverse is blocked.

**AC6 — Confirmation dialog.** Dialog shows import name, entry count, line count, affected FY, total debits/credits. Warning per NXT-73286 sec 4.4 ledger import pattern. Mandatory comment. Buttons: Cancel (default) / Reverse import (destructive).

**AC7 — Hard delete.** On confirm, BL permanently removes all OB journal entry lines, headers, and batch record. Atomic. DE status updated to Reversed.

**AC8 — Post-reverse state.** After reversal, FY has no opening balances. Trial Balance shows $0.00 with no-OB indicator per NXT-71240 sec 7.5. Restatement Gate does not fire on subsequent OB imports for that FY.

**AC9 — Override of NXT-71240 sec 4.1.** This story deliberately supersedes the read-only constraint in NXT-71240 sec 4.1 for the Reverse action. NXT-71240 should be updated.

**AC10 — Rollups recalculated.** After deletion, rollups and balances for affected fiscal periods are re-run.

**AC11 — Duplicate detection.** After reversal, subsequent OB imports for the same FY do not trigger the Restatement Gate.

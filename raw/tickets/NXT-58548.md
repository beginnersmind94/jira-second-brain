# NXT-58548 — Elig - Reviews > Student Data - Page, Role Access

## Description
As Dana the Director I want to setup parameters for my Administrative Review with regards to the review of FRE Applications. I need an area to do this, and to let an auditor have access as well if required.

As part of the audit, the district sends a list of students (an Eligibility Roster) to the state, who respond with whether they’re doing a 95% or 99% sample, along with a starting point and a review interval.

We want to let the district/auditor enter this information so we can generate an eligibility list, and a selected list, along with granting the auditor access to review the application/DC information.

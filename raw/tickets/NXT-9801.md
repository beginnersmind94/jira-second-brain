# NXT-9801 — Data Exchange - File Import Dashboard for First Time Users

## Description
As a PrimeroEdge user, when I access “File Import Dashboard” page under configuration module, I would like to view a welcome dashboard with a “quick start guide” steps to easily follow so that I can setup new file import configuration, Import data file or schedule a data file import easily. 

*Mockup:*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=393561%3A873&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=393561%3A873&scaling=scale-down-width|smart-link] 

*Requirements:*

* When the user has clicks on “*File Import Dashboard*” under configuration, file import dashboard is loaded to create, schedule and import configurations.
* Header is displayed for the user to clearly to highlight the *Dashboard* with a welcome message “Welcome to File Import Dashboard” in welcome section.
* *Quick Start Guide* steps are listed in the *welcome table*
** Header should display “*Please follow these steps to get started”*
*** Click on “NEW FILE IMPORT SETUP” (clickable button) to save file configurations
*** Once you have configured the fields then you can come back to this dashboard for next steps
*** You will see the options to either IMPORT (clickable button) a file or “SCHEDULE” (Clickable button) it for later.
* When user clicks on “*NEW FILE IMPORT SETUP*” button on top or in quick start guide instructions, *Pre-upload page* should be loaded for user to *create new import configuration*
* When user clicks on “*IMPORT*” button on top or in quick start guide instructions, “File Import/upload” page should be loaded for user to import new data file for a specific configuration. 
** [https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A239056&scaling=scale-down-width|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=388758%3A239056&scaling=scale-down-width|smart-link] 
** If there are no configurations exists. it should load *Pre-upload page*
* Each file import setup row has expanded actions with DELETE, EDIT options
* SCHEDULE is out of scope for release-1

## Acceptance Criteria
* Verify that when the user has clicks on “File Import Dashboard” under configuration, loads the dashboard page

* Verify that File Import Dashboard is loaded with Welcome Message and Quick Start Guide instructions as listed in requirements

* Verify when user clicks on "New File Import Setup" button available on Top-RIGHT and within Welcome table loads the Pre-upload screen to create new file import configuration

* Verify that instructions include 2 buttons with  IMPORT & SCHEDULE enabled

* Verify that this screen is NOT loaded when the user belongs/has access to certain district/s CONTAINS existing file import setup configuration files.

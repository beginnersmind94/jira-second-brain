# NXT-24944 — PR- Display Food Required On Production Plan

## Description
As a Production Manager, I would like to be able to see all the food required to produce each menu item on my production plan based on my planned counts.

## Acceptance Criteria
Display Food Required on the Production Plan based on the items that are used within a menu item (Similar to Grocery List)

Display the total units, and weight that are needed per each item as well as display the recipe/ item Name and Code for each portion of the menu item.

IE: 25 Each OR 1 pound 3.25 ounces (Peeled Cucumber- LI123456)

Calculate the total amount needed based on the total counts for each serving group.

If there are multiple serving groups with multiple serving sizes, take into account each serving size with that count and add the total weight. Then add the total weight for the other serving size and combine to get a grand total weight and units needed. 

When user clicks save with a different count, recalculate the food required and update the new amount needed.

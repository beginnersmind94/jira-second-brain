# NXT-39788 — Student Import – Exceptions Log – Designate if Mandatory

## Description
As a SchoolCafé Admin I want to quickly filter the grid to see only exceptions for mandatory records so that I can know if I need to review the exception log.

*Flow:*

* Open SchoolCafé and browse to Settings > Imports & Exports > Import Monitor
* Click on a Student Import record to view the import details
* Go to Exceptions tab, view the grid and filter on the Required column

*Requirements*:

* *Database:*
* Update the SPROCs when inserting an exception to designate if it is a Mandatory or Optional field
* Update the database table to record if is a Mandatory or Optional field
* *API:*
* In the API that feeds the Exceptions Grid, Add the Required field to the dataset
* *Front End:*
* On the exceptions tab of a student import, add a column to the grid there:
** Title: Required
** Values: Mandatory, Optional
** Filter the results based on the selection

## Acceptance Criteria
* The grid has the column with the designated filter values and filter the data correctly

# NXT-32959 — INV - Withdrawal Details - Update for Pack Size configuration

## Description
As a Site Manager I want the Withdrawal Details page to reflect updated Pack Size and QOH logic so I can know exactly which items and units I’ve withdrawn from my perpetual inventory.

*Requirements*

* Update the Withdrawal Details screen to reflect the updated item hierarchy configuration
** Items and their pack size units should be displayed according to how they were entered on the Withdrawal ([NXT-29044|https://primeroedgenext.atlassian.net/browse/NXT-29044])
* Remove Quantity on Hand as a column
** this is removed from Details because QOH is a constantly changing field and should not be shown historically
** Retain appropriate QOH validations and error messages
* Display Withdrawal Quantity as a string in view mode
* Enable Pack Size expansion in Edit mode
** Same behavior as Create

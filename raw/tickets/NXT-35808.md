# NXT-35808 — Onboarding - Feature Selection - Show date & timestamp/'last updated by' info for completed tasks

## Description
As a Cybersoft or SchoolCafé administrator, I would like to see the audit info for users who have modified or completed any onboarding tasks, so that we can determine when/why a change was made during the onboarding process in the event that multiple people are completing the process at once. 

*Requirements:*

* Update the existing Feature selection page UI:
*# First, whenever a user completes a task (when adding info and/or selecting ‘Return to Onboarding’), log that information in the database: 
*## Modified By (FirstName + ' ' + LastName)
*## Modified On (MM/DD/YYYY hh/mm AM/PM)
*# Then, back on the Feature selection page, display this info beneath each task that has been completed using the following format (in italics): “_Updated by {ModifiedBy} on {modifiedOn}”_
*# For tasks that are still incomplete, no change is required.
*# It would be nice to do this in a way that is efficient with the space. The goal is to have all of the sections (System, Front Office, Back Office) to look uniform once we’re done. So add appropriate/dynamic spacing so that System, FO, and BO heights are still the same whether one task is complete or all tasks are complete.

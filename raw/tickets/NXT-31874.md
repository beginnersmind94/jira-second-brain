# NXT-31874 — Contextual Help Widget - Enable Widget for All Modules

## Description
As an end-user, I would like the ability to easily access contextual help while I am using SchoolCafe.

## Acceptance Criteria
* As a content team member, I would like the ability to have a unique widget per module that is centrally implemented (See below list)
** Workspace
*** The widget for Workspace will be shared with Insights
** Accountability
** Eligibility
** Account Management
** Item Management
** Inventory
** Menu Planning
** Production
** Insights
*** The widget for Insights will be shared with Workspace
** System
* As a SchoolCafe Admin, I would like to have a single true/false setting to control all widgets and whether they are enabled/disabled
** Setting Name: *Enable Contextual Help*
** Default value: True
** Code: HELPWIDGET
** Category: Configuration
** Module: System
** Type: Internal
* The code snippet for each widget for the above modules can be found in the attached spreadsheet.
* The content team will manage where the widget is placed on the UI via the Freshdesk Admin console.

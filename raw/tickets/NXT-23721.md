# NXT-23721 — Platform - BO - Multi-District Access User Enhancements

## Description
As a district new user logging into SchoolCafe could have multiple districts and I should be able to select one of the district which has not started onboarding so that I can setup onboarding.

* Once customer logs into SchoolCafe should presented with list of districts that are not onboarded
* Customer selects the district to get onboarded
* System loads the welcome questionnaire page
* Customer selects the activities specific for that district and starts onboarding

## Acceptance Criteria
Customer shall be able to:

* Customer logging into SchoolCafe could have multiple districts selection post-onboarding permissions check
* Once customer logs into SchoolCafe should presented with list of districts that are not onboarded
* Customer shall be able to select the district to get onboarded
* System loads the welcome questionnaire page based on the district selection & purchased licenses
* Selects the activities specific for that district and starts onboarding
* select a district from multi-district drop down with a flag to show which district was not onboarded and which of them has been already onboarded

*Content for UX/UI:*

* Not Onboarded: This school district is ready for onboarding, please select to start onboarding
* In-Progress: District onboarding is in progress, select to continue onboarding steps
* Onboarded: This district has been already onboarded. Please select to go to dashboard

# NXT-54197 — Acc - Reports - Meal Count - By Grade option (FD 256623)

## Description
[https://primeroedge.freshdesk.com/a/tickets/256623|https://primeroedge.freshdesk.com/a/tickets/256623]

As Sally the Site Manager I need to view meal counts by grade in order to fill in my production records quickly.

*requirements*

Meal Count Report page

* Add a Grade dropdown - multi-select, all selected by default.
** Need to include No Grade Assigned (students with no grade)
*** This would be aimed at the New Student account which has no grade
* Add a new Meal Count By option: Grade
* Add a report option:
** Include Adults (checkbox)
*** If enabled, show the Adults column and the meal count by meal type for adult patrons.
* Existing reports will need to be updated to account for the Grade dropdown.

Meal Count by Grade Report

* Summary and Detailed versions
* Columns are the selected grades from the new grade dropdown.
** Would be great if we can only show the selected grade columns
* “Others” column to include students with no grade assigned (New Student account)
* Legend section
* Selection Criteria section for what options the user has selected so they can replicate the report only

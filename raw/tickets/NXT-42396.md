# NXT-42396 — IM - Items - Inventory Info - Allow custom Vendor Item entry

## Description
As a production manager I want the ability to manually configure a vendor and vendor item number so I can see that info on reports with purchase quantity without having to use the Inventory module.

!Inventory tab - Custom Entry.png|width=83.33333333333334%!

!image-20230906-140629.png|width=83.33333333333334%!

## Acceptance Criteria
* On the Inventory Info tab Vendor Contracts card, add two button options
** Custom Entry (secondary)
** Vendors (primary)
* The Vendors button should simply open the Vendors page in a new tab
* The Custom Entry button should allow users to insert a custom record 
* Only allow one Custom Entry record
* The custom record will have only Vendor and Vendor Item as editable

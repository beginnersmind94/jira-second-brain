# NXT-43070 — INV - Enhance Multi Select feature

## Description
Alterations needed for the Autocomplete Multi-Select (multiSelect) custom component:

When a value is inputted into the autocomplete field, causing a filtered list of items to appear, the "Select All" action should apply only to the filtered options, not the entire list of options, upon clicking.

# NXT-9537 — Data Exchange - File Export Screen

## Description
*Summary*

As a PrimeroEdge user,  from Data Exchange scree, I should be able to select  or create new export configuration from export screen. 

*Mockup*

[https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=363194%3A238893&scaling=min-zoom|https://www.figma.com/proto/yrkJeGfd3f9hWjqm0qvfir/Edge2.0_V1?node-id=363194%3A238893&scaling=min-zoom|smart-link] 

*Requirements*

* Export Tab is displayed on the configuration tab under Data Exchange
* List of existing jobs listed in the export configurations table
* Export configuration table headers should contain:
** Export Configuration Name (Export Name)
** Module
** Export Type
** Data Category
** Schedule
** Last Run Time
** Last run Status
** Added on
** Actions
* Export config table should sort by “added on” date
* Export config table should handle pagination with default display rows option
* Exports screen should display  “New Export” button to bring up new export page
* User should be able to click on any export job which should bring up export configuration job

## Acceptance Criteria
* Verify Export configuration page is added to Data Exchange configuration
* Verify Export configuration jobs are displayed in a table
*Verify new export configuration job button is added to Export screen
*Verify all headers in the configuration table are displayed and sorted by default
*Verify clicking on any export job opens Export configuration details
*Verify actions are available for export jobs displayed in the table
* There shall be one export job pre-configured to perform a "Student Information System (SIS) Export"
** Title of job shall be "Student Information System (SIS) Export"
** The export shall be a ".csv" file containing the columns: Student ID, Eligibility Status Code, and Account Balance
*** The ".csv" shall be delimited by commas and qualified by double quotes.
** The job shall be disabled by default, but scheduled to run at 9pm.

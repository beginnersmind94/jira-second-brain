# NXT-27764 — IM - Single Recipe to Menu Item

## Description
As a Nutritionist/Chef  when i am creating a single item recipe from an item I shall be able to to create a Menu Item in the process similarly to as when I am create an item (food Item, Single Item Recipe, Menu Item, POS) so I can quickly create Menu Item and POS item in one process if needed

## Acceptance Criteria
*Entry Point* 

* IM → Items → Selected Item (Single Item Recipe button)



* User Shall be able to create a menu item detail in the process same as when creating an item (food Item, Single Item Recipe, Menu Item, POS).

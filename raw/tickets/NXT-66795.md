# NXT-66795 — GL Account File Upload and Basic Validation

## Description
As a district finance administrator, I want to upload my Chart of Accounts file and see validation results, so that I can understand whether my account codes match the expected format before importing.

## Acceptance Criteria
* The *Import* button is available on the _Import GL Accounts_ tab
* When the user clicks *Import*, a file explorer opens so they can select and upload their Chart of Accounts file
* A *progress bar* is displayed while the file is uploading, showing the status in real time
* After the upload completes, the system automatically validates the file against the configured account code structure
* The user sees a *validation results summary* showing total accounts processed, passed validation, and failed validation counts

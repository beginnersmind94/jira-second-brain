# NXT-24939 — PR- Preparation Site Distribution Tab

## Description
As a Prep Site Manager, I would like to have a way to indicate once I have finished prepping menu items, how much I am shipping to the receiving sites and when

## Acceptance Criteria
Enable The Preparation Site Distribution Tab once the consolidated Plan has been saved. 

Columns to Display:

* Shipping Date (default to actual record date for shipping) user can change if needed (Date Range to select from Prep Date- Record Date)
* Menu Item (with Menu Item Help Button)
* Serving Size
* Prep Site Name (with “Preparation Site” as an indicator if there Prep Site has the same Menu Line)
* Site Name ( Receiving Site/ Sites) (Each site will be in its own column)

Total (Planned Counts for Each item for all sites)

Next To each Planned count per site and on the total column, add an additional text box that will be the shipping quantity.

*Prep Site Column will use the verbiage of Planned/ Prepped* 

User has the opportunity to type each shipping/ prep quantity and then submit

* User also has the opportunity to use Ship Planned Counts (it will populate the planned counts in the shipped or prepped fields)
** User can come in and reuse the ship planned counts toggle again, it will not effect any menu items that are marked as shipped
* There will be a status column to indicate where the process is in the shipping (Default to Shipping Counts Not Complete)
** Shipping Date can not be enabled until all counts have been entered
** *Not Shipped*- They have put shipping counts but the shipping date has not happened yet (6/22) shipping on (6/24)
** *Shipping Counts not Complete*- if not all counts have been added to a particular menu item for all sites
** *Shipping in Process*- Shipping counts have been entered and the date is the shipping date is current date (6/24)
*** Until the day is done, user can still make changes to shipping count
** *Shipped*- Shipping counts have been entered and the date is a  past date (6/25)
*** User can not make changes to shipped counts
* Display Food Used (This will be all the food used based on all shipping counts) This will populate once all shipping counts have been added
* *Non Prepared Items will only show menu items for the Prep Site that are not considered Prepared Items. It will give the user the option to add the prepped quantities* 
** Do not Show all other sites for Non prepared Items as the Prep site is not responsible for any non Prepared items from other sites
* Display Distribution Worksheet available to access ([NXT-2086|https://primeroedgenext.atlassian.net/browse/NXT-2086])
** Update the distribution worksheet to show the updated numbers once user clicks submit
* Once user clicks submit, they are allowed to come back and change the counts or the shipping date.
** If shipping date has passed, user can not change the shipping date or counts for that menu item

# NXT-66854 — SC Insights - Insights System Setting

## Description
We need a way to turn Workspace Insight on/off easily so that the navigation button doesn’t show in the main menu.


*Requirements*

* Do the following in System Settings:
* In System Settings, add a new property
** Setting Name: Enable Workspace Insights
** Type: Internal
** Code: Insights
** Value: Yes/No
** Default: No
** Category: Configuration
** Module: System
*  
* Ensure that if No, no district user can view the Insights icon in the left nav so that they cannot browse to it
* 
* *Security*
** Only Customer Support roles can make this change

## Acceptance Criteria
* Users cannot browse to WS Insights if this setting is disabled

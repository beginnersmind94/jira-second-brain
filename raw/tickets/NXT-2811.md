# NXT-2811 — Inspection Template - Temperature Grid Question Type

## Description
*User Story*
As an Inspections user, I can add a Temperature Grid question type when creating or editing a template

*Mockup*
https://m5uloa.axshare.com/#id=usu4ze&p=add_new_questionnaire&view=default&g=1

*Requirements*
* Temperature Grid can be selected as a question type when adding a question to a template
** Not available for Pass/Fail scoring type
* Temperature Grid allows users to enter temperatures in a table
** Users can edit the header of the columns
** Users can determine whether temperature should be in F or C
** Users can set the number of temperatures that must be taken

## Acceptance Criteria
* Temperature Grid questions can be added to templates

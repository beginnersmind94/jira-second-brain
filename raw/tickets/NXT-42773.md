# NXT-42773 — Acc - Reports - Meal Count (by Site) - Updates for State Eligibility

## Description
As Anna the Application Processor I want to see State Eligibility on reports so I can verify information and counts as needed.

*requirements*

Meal Count report (By Site)

* If the state eligibility setting is turned on, we’ll show additional counts in the Meal Count report.

Current format:

!image-20240129-234713.png|width=594,height=324!

* Show State-Free and State-Reduced columns for each MealType
** Only show the columns if they are both configured - for example, if a district does not put an Income Guidelines percentage in State-Reduced, do not show it
** Use the local descriptions if possible
** Put the new columns after the Total column for each meal type
* Show the number of meals sold to students who have State Eligibility
* Add note to the bottom of the report:
** For the Total column, State Eligibility meal counts are reflected as part of the Paid count. They are shown separately for clarity and additional reporting.

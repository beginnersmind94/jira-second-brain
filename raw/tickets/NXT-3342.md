# NXT-3342 — Employees - Multi-Chip Search

## Description
The search tool/format will be consistent across entire system. MULTI CHIP

* count for number of records (grid/card)
** count is updated with search results
* no limit of chips
* search the fields on the page (card/grid)

((NOT INCLUDING PREDICTIVE SEARCH- 3439)

## Acceptance Criteria
search field narrows down results 
search looks at data in the card/grid 
the result count is accurate/updated with each search

# NXT-5230 — [SchoolcafeTesting] - Applications - Automation - Categorical Apps

## Description
*User Story* 

As a district admin I want finer control on whether categorical applications are automatically approved.

*Mock Ups* 

n/a

*Requirements* 

* User needs the ability to exclude foster, head start, migrant, homeless and runaway students from the automatic approval process.
* In System > Settings > Eligibility > Benefits (Common):
** *ADD NEW SETTING* > “Automatic approval for Other Source Categorical Status applications.”
*** Yes - student is automatically approved for benefits (free)
*** No - Look at other factors that determine eligibility (income, benefit programs)
* This setting should only be considered while submitting an app online, not re-processing by any means.

----

{color:#00b8d9}*_Attached PDF is an FYI does not include internal (red settings) and applies to all SE, not just applications._*{color}

## Acceptance Criteria
User has a system setting that lets them control whether or not categorical applications are automatically approved when submitted online.

Note: If the standard 'automatically approve applications' system setting is off, this shouldn't fire also.

System behaves appropriately.

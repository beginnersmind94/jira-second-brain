# NXT-46180 — MP- Menu Planner- Inactive and Pending Deactivation Menu Items

## Description
As a Menu Planner, I would like to be able to see which menu items on my menus are either inactive or pending deactivation and should be paid attention to possibly removing them from the menus they are on.

## Acceptance Criteria
* Capture menu items that are in active status or pending deactivation status
* Display a warning sign on pending deactivation menu items
* Provide a tool tip on the same warning, “Menu Item is Marked for Deactivation”

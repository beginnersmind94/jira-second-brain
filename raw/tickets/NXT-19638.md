# NXT-19638 — Platform - BO - Onboarding Summary Dashboard

## Description
*Persona*: CyberSoft/ISD Administrator

*Access level:* School District/Central Admin

*Description*:

As a school admin, I should be able to view the onboarding summary widgets with its overall progress and next remaining steps to complete System onboarding

+*Mockup:*+

[https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|https://www.figma.com/proto/PLmXuylGMphY0Bz7WBDCOp/Systems-4.0?node-id=637%3A9929&scaling=min-zoom&page-id=548%3A8289|smart-link] 

*Requirements*:

* Site admin logs into Onboarding Summary Dashboard page for summary widgets displayed in “Dashboard” tab 
** +Widgets to be displayed+
*** System Modules Widget - Activation
*** Imports Widget
*** Settings Widget
*** Roles & Users widget
*** Sites Widget
** Each widget has actions available to *VIEW DETAILS*
* System Modules Widget
** View Details button is enabled with option to open System Modules tab
** ADD NEW button is enabled with option to add new module to be license section
* Imports Widget
** View Details button is enabled with option to open Imports tab
** Actions included in this widget are START and continue
* Settings Widget
** View Details button is enabled with option to open Settings tab
** ADD NEW button is enabled with option to add new setting pop-up window
* Roles & Users widget
** View Details button is enabled with option to open Roles & Users tab

## Acceptance Criteria
* User shall be able to see onboarding dashboard with all widgets enabled
* User shall be able to see the onboarding status widget with progress %
* User shall be able to see the summary of onboarding progress for each section
* User shall be able to start or resume the onboarding process from Onboarding status widget which retrieves the last step/activity performed by the user
* User shall be able to see VIEW DETAILS and ADD buttons for each section/widget available in the onboarding page

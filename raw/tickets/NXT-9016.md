# NXT-9016 — INV - Add New Ordering Conditions - Search Items (Step 1)

## Description
As a District Supervisor I want an Add Ordering Conditions page so I can search for items and begin to assign Site Ordering Conditions.

This page will allow users to search for items and populate them in the list below ([NXT-5705|https://primeroedgenext.atlassian.net/browse/NXT-5705]).

*Requirements*

Provide a page that allows for the selection of items (one or many)

* Storage Category dropdown
** select specific Storage Category (no ‘All’ option)
* Item Category dropdown
** disabled until Storage Category is selected
** select specific Item Category (no ‘All’ option)
** Item Categories will be limited based on Storage Category selection
* Data Source dropdown
** disabled until Storage Category and Item Category are selected
** select specific Data Source (no ‘All’ option)
* Search button
** must have specific selection for each dropdown before user can search

## Acceptance Criteria
User selects dropdown values (no 'All' option) to load list of items
Dropdowns are dependent on previous selection

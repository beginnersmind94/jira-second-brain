# Complete a Production Record Quick Guide

This guide will cover how to complete a Production Record. Features include entering Actual Counts, entering Leftovers, adding Menu Items, saving the Production Record, and printing reports.
To complete a Production Record, navigate to Production > Calendar.

## ENTER ACTUAL COUNTS

## Process / Descriptions Images
On the Calendar page
1. Choose a Site
2. Select a month

Use the key at the top of the page to view the status of a menu line.

3. Select a menu line on the desired day to edit

4. Check the Offer vs Serve checkbox for each

## Serving Group as needed
5. Enter the Actual Count for each Serving
   Group

6. Enter the Produced count

## ENTER LEFTOVERS
The Produced Items that were not served should be entered into the appropriate leftover columns. There are three types of leftovers: Carryover, Return to Stock, and Waste.
If no leftovers need to be entered, proceed to the Add Menu Item section.
1. Click the Carryover Date
   The Carryover Calendar displays.

Review the following information:
a. Menu Item – The selected Item displays
b. Carryover From – The date the Menu
Item was produced
c. Max Carryover Days – The number of days the Item can be served after it has been produced
d. Menu Item Planned – The dates available to carryover the Item and the Menu Item is also planned for
e. Can Serve – Dates available to carryover the Item
2. Select the Meal Type
3. Select the Menu Line
4. Enter a Comment

5. Select the appropriate date

6. Click the SAVE button

## The Carryover Date, Meal Type, and Line will
display in the Carryover Date.

7. Enter the Return to Stock count

8. Enter the Waste count

To verify the Reimbursable count (Produced – Total Leftover = Reimbursable).

## ADD MENU ITEM(S)
1. Click ADD MENU ITEM
   The ADD MENU ITEM page displays.

2. Select the Item(s) to add
   A checkmark will display on the selected Item(s) row.
3. Select the Category
4. Select the Serving Size
5. Enter the Planned Count
6. Click the ADD ITEM button
   The user is returned to the Plan page.

The added Menu Item is now displayed under the selected Category.
7. Enter the Produced count
   To delete the Menu Item, click the Trash Can icon. Only added Menu Items can be deleted. All Menu Items initially assigned cannot be deleted.

## SAVE THE PRODUCTION RECORD
After all the data has been entered, two options are available to save the completed Production Record: Save Record or Save.

## SAVE RECORD OR SAVE
Selecting the SAVE RECORD or SAVE button will change the Plan Status to Record Complete.
If any produced counts were not entered, a red outline would display for all missing counts.
The Plan Status will continue to show Plan
Complete until all counts are entered.

## The Plan Status has changed to Record
Complete.

## PRINT REPORTS
After the Production Record is in a Record Complete Status, the following reports are available.
These reports can be accessed at any time during the school year.
1. Click REPORTS
   The following report options appear.
   Click the report name to download and print.
   The Plan Worksheet is a printable version of the Production Plan for users to manually enter data before electronically inputting it into the system.

The Planned Recipes Report scales the recipe to the amount of servings the user plans to serve.
This is a printable version of the Production
Record with the ability to see multiple records simultaneously.
The Production Form is the uncompleted
Production Record. This can be printed out and used for pen and paper Production Records.
The Menu Costing Report breaks down the cost of all the planned Menu Items as well as costs per meal based on all planned counts.

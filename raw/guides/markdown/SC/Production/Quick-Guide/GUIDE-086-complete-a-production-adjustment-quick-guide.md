# Complete a Production Adjustment Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-086`.

This guide will cover how to add Menu Items to Menus that have already been assigned to the Production Calendar.
To complete a Production Adjustment, navigate to Production > Production Adjustment.
On the Production Adjustment page
1. Select the Meal Type
2. Select the Site Group
3. Select the Menu Line
4. Enter the From Date
5. Enter the To Date
6. Select the Menu Item

7. Click the SELECT PLANS button

## The Meal Patterns and Select Sites tables
appear.
8. Select the Serving Size for each Meal Pattern as needed
9. Select the Sites
   By default, all sites are selected.
   Click the header Checkbox to deselect all sites.
10. Select the Category for each selected site as needed
   Click the View Plan icon to view the Production Plan for each site as needed.

11. Click the SUBMIT ADJUSTMENT(S) button
   A success message displays.

# Complete a Production Plan Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-084`.

This guide will cover how to complete a Production Plan. Features include entering Planned Counts, adding Menu Items, saving Planned Counts, and printing the Plan Worksheet.
To complete a Production Plan, navigate to Production > Calendar.

## ENTER PLANNED COUNTS

## Process / Descriptions Images
On the Calendar page
1. Choose a Site
2. Select a month

Use the key at the top of the page to view the status of a menu line.

3. Select a menu line on the desired day to edit
4. Check the Offer vs Serve checkbox for each

## Serving Group as needed
5. Enter the Planned Count for each Serving
   Group

6. Enter the planned count for each Menu Item
   If zero is entered for a Menu Item, it will not appear on the RECORD tab.

## ADD MENU ITEM(S)
If a Menu Item needs to be added to the Production Plan, follow the steps below.
If no Menu Items need to be added, proceed to the next step to Save the Planned Counts.
1. Click ADD MENU ITEM
   The ADD MENU ITEM page displays.
2. Select the Item(s) to add
   A checkmark will display on the selected Item(s) row.
3. Select the Category
4. Select the Serving Size
5. Enter the Planned Count
6. Click the ADD ITEM button
   The user is returned to the Plan page.

The added Menu Item is now displayed under the selected Category.
To delete the Menu Item, click the Trash Can icon. Only added Menu Items can be deleted. All Menu Items initially assigned cannot be deleted.

## SAVE THE PLANNED COUNTS
The last step in completing a Production Plan is to save the planned counts. There are two options for saving a Production Plan: Save or Save as Plan Complete.
SAVE OR SAVE AS PLAN COMPLETE
Selecting the SAVE button will change the Plan
Status to Plan Complete if the planned count is entered for all Menu Items.
If any counts were not entered, a red outline would display for all missing counts.  The Plan Status will continue to show Plan Incomplete until all counts are entered.

## Selecting SAVE AS PLAN COMPLETE will force
the Production Plan into a Plan Complete
Status.
A zero will be automatically populated for any
Menu Items with no counts entered.

The Plan Status has changed to Plan Complete.

PRINT THE PLAN WORKSHEET
After the Production Plan is in a Plan Complete Status, it is recommended to print the Plan Worksheet to document all produced Menu Items and Leftovers.
A best practice is to print the Plan Worksheet on the same day of service to show Carryovers from previous serving days.
1. Click REPORTS
   A pop-up appears.

2. Click the PLAN WORKSHEET button to download and print

# Add Applications Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-036`.

The Applications function allows users to
- Manually add a new application

## Add Manual Applications
1. Click the NEW button in the top right corner of the page
   The Application Entry page displays.
   The Application Entry page has a process wizard that turns purple as you complete each section.
   Click on a purple icon in the process wizard to return to that section.
   Students
1. Enter a student’s Name or ID into the search field
   Options display based on the information entered.
2. Click the SELECT STUDENT button for the desired student
   The Add a Student slide deck displays.
3. Verify the student’s information
4. Answer the Categorical Status question (Yes or No) If the answer is Yes and the confirmed Categorical Status is unchecked, the student will receive an eligibility status based on the application and not their
   Categorical Status.
   If Yes is selected, a Categories dropdown displays. Select the Category.
   If a warning icon displays next to the student, hover over it to see the student's current eligibility status, eligibility end date, and the application number and outcome for any other application the student has been on this year.
5. Click the SAVE button in the bottom right corner of the page
   You are returned to the Application Entry page.
6. Click the NEXT button to proceed

## Add New Student
You can use the ADD NEW STUDENT button to add a student if the student does not automatically populate.
1. Click the ADD NEW STUDENT button
   The Add a Student slide deck displays.
2. Enter the required information

## Answer the Categorical Status question (Yes or No)
If Yes is selected, a Categories dropdown displays. Select the Category.
3. Click the SAVE button in the bottom right corner of the page
   You are returned to the Application Entry page.
4. Click the NEXT button to proceed

Benefits
Answer the following questions regarding the applicant and household benefits.
1. Do any household members receive benefits from FDPIR, FS/TANF Case

## Number, SNAP or TANF? (Yes or No)
- If Yes, enter the optional Recipient Name
- Select the Case Type
- Enter the required Case Number
2. Has the applicant Refused benefits? (Yes or No)
3. Click the NEXT button to proceed

## Household Members
1. Click the Edit (Pencil) icon to enter the TOTAL CHILD INCOME if applicable
- Enter the Total Child Income amount
- Select the Frequency
- Click the SAVE button
2. Click the ADD HOUSEHOLD MEMBER button to add additional household members and their income to the application
- Enter the First Name and Last Name of the household member
- Select Yes if this member receives income
- Enter the Income amount(s) based on source
- Select the Frequency of the income(s)
- Click the SAVE button
Add additional household members and income as needed.
3. Enter the total HOUSEHOLD SIZE based on the application
4. Click the NEXT button to proceed

## Contact Info
1. Enter the applicant’s contact information as necessary
2. Click the NEXT button to proceed
   Details
1. Enter the available demographic data from the application, such as Ethnicity,

## Racial Identity, and Other Benefits
2. Click the NEXT button to proceed
   Review
1. Click the Edit (Pencil) icon for the section you wish to update
2. Click any of the process wizard icons atop the page to navigate directly to that section to review or edit the information
3. Click the NEXT button to proceed

Submit
1. Enter the last four digits of the applicant’s Social Security Number or click the
   Application has no SSN checkbox if it wasn’t provided
2. Click the Application signature confirmed checkbox to confirm the applicant’s signature
3. Select the Language Preference from the available options
4. Enter the Application Received Date
5. Review the Pending Reasons, if applicable, and address them as necessary
6. Enter optional Comments
7. View the Eligibility Determination
8. Click the SUBMIT button to complete the application process
   Click the PENDING button to mark the application as pending for later access.
9. View the application
10. Use the EXPORT or PRINT buttons to export this information if desired
   Click the ADD ANOTHER APPLICATION button to return to the beginning of the process.
   Click IM DONE FOR NOW button to return to the Applications page.
   Repeat this process to add additional applications as needed.

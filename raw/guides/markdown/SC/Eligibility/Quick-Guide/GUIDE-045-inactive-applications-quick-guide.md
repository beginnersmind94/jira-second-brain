# Inactive Applications Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-045`.

This report allows users to view details on applications that have fallen inactive during the verification timeframe. These are due to the students having moved on to another source of eligibility.

## View Inactive Applications
1. Select the INACTIVE APPLICATIONS tab
2. Select an Academic Year from the dropdown
3. Click the APPLY button
   Inactive Applications display in the table below.
   View the Results:
- Application Number
- Student ID Numbers
- Application Process Date
- Eligibility Start Date
- Inactive Date

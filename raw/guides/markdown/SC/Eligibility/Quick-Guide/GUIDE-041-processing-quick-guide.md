# Processing Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-041`.

The Processing function allows district users to review batches of applications processed per date range within each academic year. From this page, a log can be printed for review and sign-off.

## View Processed Applications
1. Select an Academic Year
2. Enter a Start Date and End Date to specify the date range
3. Click the APPLY button
   Results display below.

## Review Session Summary
1. Identify the Session you wish to review
2. Click the View icon for the desired Session
   The Session Summary page displays.
3. View the Session Information atop the page

## Generate Application Approval List
In the Processed Applications table, you can click the desired Application Number hyperlink to navigate to the Application Details page for that application. See the Application Processing guide to continue processing the application.
1. Click the GENERATE APPLICATION APPROVAL LIST button to view the approval details
   The Application Approval List page displays.
2. Use the EXPORT button to download a PDF file or use the PRINT button to print the approval list
   Click the BACK button to return to the Session Summary page.

# Collection Worksheet Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-047`.

The FNS-742 Collection Worksheet populates with automatically calculated totals based on live data. The district must submit these totals to the state annually, as each state asks for this report to be submitted after verification.

## View the Collection Worksheet
1. Select the COLLECTION WORKSHEET tab
   The current academic year Collection Worksheet automatically displays on the page.
2. View the Collection Worksheet details
   Various Document icons are available throughout the worksheet; you can click on them to view the Collection Report. This information can be downloaded.
   Click the PDF button to export the form. The export delivers all pages together as a single download.
   Specific dates are shown in column headers to indicate when the underlying information was pulled.
   Use the PREVIOUS PAGE and NEXT PAGE buttons to navigate between pages in the Collection Worksheet.
   Click the SAVE button to store a copy of the Collection Report in the system. Saved reports appear in a grid showing the Generation Date and User, with View (preview the PDF) and Download actions. Saving is important because numbers on the report can shift later as eligibility is backdated.

## Sections 3 and 4 — California-specific and Minnesota-specific behavior
- Section 3-3 does not include Medicaid Reduced students.
- For California, MP-1 and MP-2 show a second count for the number of benefit extensions. Click through the count to see the student details.
- When the system setting *Include Students from CEP Sites for Verification Reporting* is enabled, CEP-site students are counted in sections 3 and 4. The setting is at System > Configuration > Settings.
- For Minnesota, a Section 4.1 is available.

## Select a Different Academic Year
1. Select an Academic Year from the dropdown
2. Click the APPLY button
   The selected academic year’s Collection Worksheet displays.

# Tracking Forms Quick Guide

Tracking Forms allows district users to view individual tracking forms for each selected application detailing their progress through verification.

## View Tracking Forms
1. Ensure that the TRACKING FORMS tab is selected
2. Select an Academic Year from the dropdown
3. Click the APPLY button
   Tracking Forms display in the table below.
   View the Results:
- Application Number
- Student ID Numbers
4. Click the App Info icon for the desired application
   The Verification Tracking Form displays.
5. View the form details
   Click the PDF button to export the form.
   Click the X to return to the TRACKING FORMS tab.

## Print Tracking Forms
1. Click the Checkbox(es) for the application(s) you wish to print
   Click the header checkbox to select all applications.
   The checkbox(es) will illuminate purple for the selected application(s).
2. Click the PRINT button for the selected application(s)
   You can also click the PRINT ALL button to print all applications.
   The Verification Tracking Form appears.
3. Click the PRINT button in the top right corner
4. Follow the browser standard for printing
   You can click DOWNLOAD ALL on the Tracking Forms table to download all the applications.
   You can click the Download icon on the Tracking Forms table to export this information to Excel.

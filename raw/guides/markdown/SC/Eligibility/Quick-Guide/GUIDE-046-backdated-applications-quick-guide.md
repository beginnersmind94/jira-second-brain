# Backdated Applications Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-046`.

The Backdated Applications report allows users to identify applications where the approval date was moved back into the verification window. This can lead to a mismatch between the number of applications and students. It can also lead to the Collection Worksheet numbers changing over time.

## View Backdated Applications
1. Select the BACKDATED APPLICATIONS tab
2. Select an Academic Year from the dropdown
3. Click the APPLY button
   Backdated Applications display in the table below.
   The applications that display were processed after the sample date but backdated before the sample date.
   View the Results:
- Application Number
- Student ID Numbers
- Process Date
- Received Date
- Status

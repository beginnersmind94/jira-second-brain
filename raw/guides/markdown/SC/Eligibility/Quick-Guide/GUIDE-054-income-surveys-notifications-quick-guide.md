# Income Surveys Notifications Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-054`.

The INCOME SURVEYS tab within the Notifications function allows users to
- Search for students you want to send notifications to
- View notification letters
- Send Income Survey notifications to the selected student(s)

## Search for Students to Send Notifications for Income Surveys
1. Select the INCOME SURVEYS tab
2. Enter the Start Date and End Date from the dropdowns
3. Click the Exclude previously notified surveys checkbox as desired
4. Click the APPLY button
   Notifications for Income Surveys display in the table below.
   A Student ID column is available in the grid, with search filters on the column.
   A link to the Documents Center is available from the page, and a Back button returns you to the previous page. The grid uses server-side pagination. Use the Preview, Notify, and Notify All buttons to act on selected surveys. When a print batch contains more than 20 documents, the print runs in the background and is delivered to the Documents Center as a single file.

## View Notification Letter
1. Click the View Notification Letter (Eye) icon for the desired survey
   The Income Survey Notification letter displays.
   The letter is displayed in the student's selected language if an applicable language template is available.
   Click the X in the top right corner to return to the INCOME SURVEYS tab.

## Notify Students
1. Click the Checkbox(es) for the student(s) you wish to notify
   Clicking the header checkbox will select all listed students.
   The checkbox(es) will illuminate purple for the selected student(s).
2. Click the NOTIFY button
   The All Options slide deck displays.
3. Select from the Email/Print Options
   The number in the parentheses indicates the number of notifications that will be emailed or printed.
   Email notifications require a guardian email address in the student account details.
- Email/Print: Emails are sent and follow the browser standard for printing.
- Email Only: The notification is sent to the designated email and a success message will appear.
- Print All or Print Only: Follow the browser standard for printing.
4. In the Print Options section, select a Sort By method
5. Click the NOTIFY button and the system will execute notifications based on the selections
   A confirmation response appears atop the page.
   You are returned to the INCOME SURVEYS tab.

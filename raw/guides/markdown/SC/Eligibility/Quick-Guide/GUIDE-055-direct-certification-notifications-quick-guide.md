# Direct Certification Notifications Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-055`.

The DIRECT CERTIFICATION tab within the Notifications function allows users to
- Search for students you want to send notifications to
- View notification letters
- Send Direct Certification notifications to the selected student(s)

## Search for Students to Send Notifications for Direct Certification
1. Select the DIRECT CERTIFICATION tab
2. Enter the Start Date and End Date from the dropdowns
3. Click the Exclude students who already have Direct Certification notifications checkbox as needed
4. Click the Exclude students who were notified on the same eligibility status based on another approval method checkbox as needed
5. Click the APPLY button
   Notifications for Direct Certification display in the table below.
   A link to the Documents Center is available from the page, and a Back button returns you to the previous page. The grid uses server-side pagination. Use the Preview, Notify, and Notify All buttons to act on selected students. When a print batch contains more than 20 letters, the print runs in the background and is delivered to the Documents Center; the system notifies you when it is ready for collection.

## Recurring Notices tab
The Recurring Notices tab on the Notifications page contains the Automatic DC Notifications schedule. Set the scheduled task to Active to send notifications automatically. The tab shows a history of sends, who received them, and the letter that was sent. Districts that previously used the system-level "Automatically Notify Direct Certifications" setting should confirm the scheduled task is Active.
When the schedule is Active, an approval notice is sent automatically to the primary guardian after successful Direct Certification matches are processed. Potential Matches are included once approved. DC Extensions are not included. Automatic letters run at 6 PM daily to allow time for any mistaken approvals to be corrected.

## View Notification Letter
1. Click the View Notification Letter (Eye) icon for the desired student
   The NOTIFICATION OF APPROVAL FOR DIRECT CERTIFICATION MEAL BENEFITS letter displays.
   The letter is displayed in the student's selected language if an applicable language template is available.
   Click the X in the top right corner to return to the DIRECT CERTIFICATION tab.

## Notify Students
1. Click the Checkbox(es) for the student(s) you wish to notify
   Clicking the header checkbox will select all listed students.
   The checkbox(es) will illuminate purple for the selected student(s).
2. Click the NOTIFY button
   The All Options slide deck displays.
3. Select the Letter Format
   Individual – One notification per patron per address.
   Household – Consolidates multiple notifications into one by address.
4. Select from the Email/Print Options
   The number in the parentheses indicates the number of notifications that will be emailed or printed.
   Email notifications require a guardian email address in the student account details.
- Email/Print: Emails are sent and follow the browser standard for printing.
- Email Only: The notification is sent to the designated email and a success message will appear.
- Print All or Print Only: Follow the browser standard for printing.

5. In the Print Options section, select a Sort By method
6. Click the NOTIFY button and the system will execute notifications based on the selections
   A confirmation response appears atop the page.
   You are returned to the DIRECT CERTIFICATION tab.

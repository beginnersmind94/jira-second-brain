# Create a Quick Receipt Quick Guide

The Quick Receipt function is used when an Order was not created for Items that need to be added to the Perpetual Inventory.
This guide will cover how to create a Quick Receipt and the different ways to save.
To create a Quick Receipt, navigate to Inventory > Orders > Quick Receipt.

## CREATE A QUICK RECEIPT
On the Quick Receipt page
1. Select the Site
2. Enter the Order Date
3. Select the Vendor
4. Select the Templates as needed
5. Click the APPLY button
   The Items will display in the Item List below.

6. Enter the Item Quantity for each Item as needed
   Once the Item Quantity is entered, the Items Checkbox will be selected.
   The Checkbox(es) will illuminate purple for the selected Item(s).

7. Click the NEXT button
   The Create Quick Receipt page displays.

In the Create Quick Receipt section
8. Verify the Site, Vendor, Order #, Order Amount, and Receipt Amount
9. Enter the Transaction Date, Order Delivered By, Invoice #, Invoice Amount, and Comments Fields indicated with an asterisk are required.
   If the Order Amount and Invoice Amount are not equal, then a pop-up message will appear to verify.

In the Charges / Discounts Details section
10. Enter all Charge and Discount amounts as needed

In the Receipt Details List
11. Confirm or edit the Received Qty, Item Price, Delivery Fee, Processing Fee, and Net Off Invoice fields for each
   Item
12. Use the icons in the Action column to add the Date on Product or add Comments for an Item as needed

## SAVE A QUICK RECEIPT
Once all the required information is confirmed and entered, two options are available when saving the Quick Receipt:
Save as Incomplete or Confirm.

## SAVE AS INCOMPLETE

## Selecting the SAVE AS INCOMPLETE button will
change the Quick Receipt to an Incomplete
Status. Users can come back and complete the
Quick Receipt as needed.
Items will not be updated in the Perpetual
Inventory if this button is used.

A success message displays.

CONFIRM
Selecting the CONFIRM button will close the
Quick Receipt. All received Items will be added to the Perpetual Inventory.
A success message displays with an Order
Number.

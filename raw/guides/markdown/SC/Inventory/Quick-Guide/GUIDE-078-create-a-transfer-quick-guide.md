TRANSFERS
This guide will cover how to create an Inventory Transfer and options on the Transfer Details page.
 To create a Transfer, both sites must be able to order the product.
To create an Inventory Transfer, navigate to Inventory > Inventory > Transfers.

## CREATE AN INVENTORY TRANSFER

## Process / Descriptions Images
On the Transfers List page
1. Click the NEW button
   The Create Transfer page displays.
   In the Selections section
2. Select the Sending Site
3. Select the Receiving Site
4. Select the Transaction Date / Time
5. Enter Comments as needed

6. Click the NEXT button

In the Items section
7. Select the Item Category
8. Select the Storage Category
9. Select the Item
   Once the Item is selected, the Quantity on Hand appears next to the field.
10. Enter the Quantity

11. Click the ADD button
   The Item has been added to the Transfer
   Item List.

Click the + icon to transfer additional Pack Sizes.
Click the Comments icon to add comments.
Click the Trash Can icon to delete the Item.

12. Click the NEXT button

In the Confirm section
13. Click the CONFIRM button
   A success message displays.

TRANSFER DETAILS OPTIONS
The Transfer Details page allows users to print the Transfer Ticket, view the Price Details Report, edit, and cancel the Transfer.
The PRINT button allows users to view and print the Transfer Ticket.
The PRICE button allows users to view and print the Price Details Report.

The EDIT button allows users to edit the quantity transferred.
A comment is required for any changes.
The CANCEL button allows users to cancel the
Transfer.
A comment is required to cancel the Transfer.
Click the BACK TO TRANSFERS LIST button to return to the Transfers List page.

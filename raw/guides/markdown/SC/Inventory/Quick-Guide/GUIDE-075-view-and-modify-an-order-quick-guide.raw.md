# View and Modify an Order Quick Guide

## VIEW ORDERS
This guide will cover how to view and modify Orders.
To view or modify an Order, navigate to Inventory > Orders > View Orders.

## VIEW AND MODIFY ORDERS
The Order Details page allows users to view and manage the Details, Receipts, and History of an Order.
On the ORDERS tab
1. Select from the Filters as needed
2. Click the APPLY button
   The View Order List will change based on the filters selected.

3. Click the View icon for the desired Order
   The Order Details page displays.

## DETAILS TAB
On the DETAILS tab, users can view and modify the information as needed.
a. Click the ADD ITEM button to add an Item to the Order
See the ADD ITEM(S) section for more information.
b. Click the CANCEL ORDER button to cancel the Order
A confirmation response is required to cancel the Order.
c. Click the REPORT button to view and export the Order Detail Report
d. Click the Pencil icon to edit the Delivery Date
e. Click the Comments icon to add a comment for the Order

In the Order Details List
f. Click the Pencil icon to edit an Item in the Order
See the EDIT ITEM(S) section for more information.
g. Click the Comment icon to add a comment for an Item in the Order

## ADD ITEM(S)
On the DETAILS tab
1. Click the ADD ITEM button

The Add Item page displays.
2. Search and enter the Quantity for each Item as needed

3. Click the SAVE button
   The user is returned to the DETAILS tab,

## and the Item is added to the Order Details
List.

## EDIT ITEM(S)
In the Order Details List
1. Click the Pencil icon for the desired Item

2. Edit the Quantity as needed
   Click the Trash Can icon to delete the Item. A confirmation response is required.
   Click the X to cancel the edit.
   Click the Comments icon to add a comment for the Item.
3. Click the Checkmark to confirm the quantity

4. Click the SAVE button to retain changes
   A success message displays.

## RECEIPTS TAB
On the RECEIPTS tab, users can view and modify the information as needed.
a. Click the CLOSE ORDER button to close the Order
A confirmation response is required to close an Order.
b. Click the CREATE NEW RECEIPT button to create a new Receipt
See the Create a Receipt and Create a Quick Receipt guides for more information.
c. View the Order Receipts Summary
d. Click the BACK TO VIEW RECEIPTS button to navigate to the View Order Receipts page

## HISTORY TAB
On the HISTORY tab, users can view the information as needed.
a. View the Order Transaction History List
b. Click the View Modified Order Details icon to navigate to the DETAILS tab
The Item(s) information will be highlighted in green based on the Order Transaction Number selected.

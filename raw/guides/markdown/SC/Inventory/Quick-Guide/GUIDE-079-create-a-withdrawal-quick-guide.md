WITHDRAWALS
This guide will cover how to create a Withdrawal and options on the Withdrawal Details page.
To create a Withdrawal, navigate to Inventory > Inventory > Withdrawals.

## CREATE A WITHDRAWAL

## Process / Descriptions Images
On the Withdrawals page
1. Click the NEW button
   The Create Withdrawal page displays.
   In the Selections section
2. Select the Site
3. Select the Withdrawal Type
   If Production Withdrawal is selected, the
   Production Date field will display.
4. Enter the Transaction Date & Time
5. Select the Reason
6. Enter Comments as needed

7. Click the NEXT button

In the Items section
8. Select the Withdrawal Templates as needed
9. Select the Item Category
10. Select the Item
   Once the Item is selected, the Quantity on
   Hand appears next to the field.
11. Enter the Quantity

12. Click the ADD button
   The Item has been added to the
   Withdrawal Item List.

Click the + icon to withdraw additional Pack Sizes.
Click the Comments icon to add comments.
Click the Trash Can icon to delete the Item.

13. Click the NEXT button

In the Confirm section
14. Click the CONFIRM button
   A success message displays.

WITHDRAWAL DETAILS OPTIONS
The Withdrawal Details page allows users to print the Withdrawal Ticket, view the Price Details Report, edit, and cancel the Withdrawal.
The PRINT button allows users to view and print the Withdrawal Ticket.
The PRICE button allows users to view and print the Price Details Report.

The EDIT button allows users to edit the
Reason and the quantity withdrawn.
A comment is required for any changes.
The CANCEL button allows users to cancel the
Withdrawal.
A comment is required to cancel the
Withdrawal.

Click the BACK TO WITHDRAWAL LIST button to return to the Withdrawals page.

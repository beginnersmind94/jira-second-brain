# Menu Planner Quick Guide

The Menu Planner is an interactive tool for creating Menus or Menu Cycles using a drag and drop feature and on-the-fly nutrient and component analysis.
To access the Menu Planner, go to Menu Planning > Menus > Menu Planner.
The Create Menu Planner page automatically displays once the Menu Planner function is selected.
On the Create Menu Planner page
1. Select the Meal Pattern
2. Select the Meal Type
3. Select the Site Group
4. Select the Serving Group
5. Click the APPLY button
   The Menu Planner Weekly Grid and additional fields display.

After the Filters are selected, the user will select Menu Items to add to the desired day.  The following fields are available on the left-hand panel to quickly search for Menu Items.  Once the Menu Items are selected, the user will need to drag and drop those Items to the desired day.
a. Use the Categories dropdown to narrow down Menu Items
b. Use the Search Menu Items field to search

On the Menu Planner Weekly Grid
6. Identify the Menu Item to assign using the fields on the left-hand panel
7. Click on the Menu Item and hold the mouse button to drag and then release it to the desired day

8. Select the Serving Size for the Menu Item as needed
   This will update the Analysis on the fly.
   Click the X to delete a Menu Item from the Menu as needed.
9. Continue to drag and drop Menu Items for the desired day
   Users can drag and drop Menu Items from one day to another, but they can only do this with one Menu Item at a time.

On the Daily Nutrient and Component Card
The components will illuminate if Menu Items are added that meet the minimum contribution requirement. In some cases, multiple Menu Items are needed to meet the component requirement.
The Milk component will illuminate only if the combination of fluid milk and the variety of milk standards are met. Milk is the only component that has additional requirements.
When one milk is added, the component will be half-shaded. Hover over the component to see what is missing.
The nutrient calculations are color-coded for each day. As the user builds their Menu/Menu Cycle, the color indicates whether they are within the range of nutrients they need every day to pass for the week.
The PROJECTED MEAL COUNT is set to 100.

After all the information has been entered, two options are available to save the completed Menu/Menu Cycle: SAVE or SAVE & CLOSE.
A CANCEL button is also available that can be used to remove all information from the Menu Planner.
Selecting the SAVE button will save the information and put the user in an edit mode as a Menu or Menu Cycle.
The Create Menu Planner page will change to the Edit Menu Planner page.
Selecting the SAVE & CLOSE button will save the information and close automatically.
The user will be navigated to the Menu or Menu Cycle page.

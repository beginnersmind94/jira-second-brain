# Menu Item Quick List Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-081`.

Menu Item Quick Lists allow users to add lists of Menu Items to a Menu in one quick process.
To manage Menu Item Quick Lists, go to Menu Planning > Configuration > Menu Item Quick List.

## ADD MENU ITEM QUICK LIST
On the Menu Item Quick List page
1. Click the ADD button
   The Create Quick List page displays.
2. Enter the Quick List Name

3. Click the CONTINUE button

4. Click the ADD button to add Items to the Quick List
   The Search Item field is available to search for Items.
5. Once an Item is added, enter the Projected Serving count as needed
   Items can be removed from the Quick List by clicking the Trash Can icon.

6. Click the SAVE button to add the Quick List
   The page refreshes and the user is returned to the Menu Item Quick List page.
   The Menu Item Quick List is added. These can be edited or deleted as desired.

## EDIT MENU ITEM QUICK LIST
On the Menu Item Quick List page
1. Click the Edit icon for the desired Quick List
   The Edit Quick List page displays.

On the Edit Quick List page, the user can do the following:
a. Edit the Quick List Name as needed
b. Add Items to the Quick List as needed
The Search Item field is available to search for Items.
c. Edit or enter the Project Serving count for each Item as needed
d. Click the Trash Can icon to remove an Item as needed
Filters are available below each column header to search for Items.

2. Click the SAVE button to retain changes
   The page refreshes and the user is returned to the Menu Item Quick List page.
   The Menu Item Quick List is updated.

## DELETE MENU ITEM QUICK LIST
On the Menu Item Quick List page
1. Click the Delete icon for the desired Quick List
   A Notification pop-up message displays.

2. Click YES to confirm the deletion
   The page refreshes and a success message displays.

The Menu Item Quick List is deleted.

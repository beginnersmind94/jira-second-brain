# Create a Menu Cycle Using the Standard Method Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-082`.

The Menu Cycle function allows users to create Menu Cycles by combining Menus for rotation every week. The Menu Cycle Analysis and Costing Reports can be done on this page as the Menu Cycle is being created or modified to ensure that the overall weekly meal plan meets USDA standards.
To create a Menu Cycle using the Standard Method, go to Menu Planning > Menus > Menu Cycle.

## CREATE A MENU CYCLE USING THE STANDARD METHOD

## Process / Descriptions Images
On the Menu Cycle page
1. Click the NEW button
   The Create Menu Cycle page displays.
2. Enter the Menu Cycle Name
3. Select the Meal Pattern
4. Select the Meal Type
5. Select the Site Group
6. Select the Number of Weeks
7. Select the Days per Week
8. New Menu Cycles are set as Active by default
   Users can click the toggle switch to make the Menu Cycle Inactive.
   Fields indicated with an asterisk are required.

9. Click the SAVE button to create the Menu
   Cycle
   The page refreshes and a success message displays.
   The Edit Menu Cyle page displays.
   Once the Menu Cycle has been created, the
   Menu Cycle Details section will appear below.

a. Click the X icon to remove the Menu
b. Click the Edit Menu icon to edit the Menu

2. Click the SAVE button once all Menus have been chosen for each day and week displayed
   A success message displays.

## MENU CYCLE COSTING REPORTS
The following reports allow users to view and print Menu costing information specific to the Menu.
These reports contain a list of the Menu Items on Menus included in the Menu Cycle, the planned quantities, the per-serving cost, and the total cost.
This information is available for each week, as well as the entire Menu Cycle.
1. Click the Weekly Cost & Menu Cycle Cost icon for the desired week
   Options display.

## WEEKLY COST
The Weekly Cost Report allows users to view a cost report for each week.
1. Select Weekly Cost

## The Menu Cycle Costing – Week 1 Report
displays.

a. View information such as the Menu Cycle Name, Meal Pattern, Meal Type, and Site Group
b. Click the Printer icon to export the report
c. View the Week, Day, and Menu Item Category
d. View the Serving Size, Planned Qty, Unit Cost, and Total Cost for each Recipe
e. View the Category Food Cost Per Meal
f. View the Average / Total Meal Cost for each day
g. View the Average / Total Meal Cost for Week: 1 and the Average / Total Menu Cycle Cost

## MENU CYCLE COST
The Menu Cycle Cost Report allows users to view a cost report for the entire Menu Cycle. This displays cost information for all weeks and days.
1. Select Menu Cycle Cost
   The Menu Cycle Costing Report displays.

a. View information such as the Menu Cycle Name, Meal Pattern, Meal Type, and Site Group
b. Click the Printer icon to export the report
c. View the Week, Day, and Menu Item Category
d. View the Serving Size, Planned Qty, Unit Cost, and Total Cost for each Recipe
e. View the Category Food Cost Per Meal
f. View the Average / Total Meal Cost for each day
g. View the Average / Total Meal Cost for Week: 1 and the Average / Total Menu Cycle Cost

## VIEW MENU CYCLE ANALYSIS
The Analysis feature displays nutrient and meal contribution details for the Menu Cycle per serving group and week.
 Even though each Menu may individually pass on the Nutrition Analysis, it can still fail on the weekly Analysis for a Menu Cycle due to the overall combination of nutrients from Menus each week.
1. Click the Analysis icon for the desired week
   The Menu Cycle Analysis slide deck displays.

2. Click the Show Only Required toggle switch on or off as desired
   The nutrients and food components will display when enabled to reflect the USDA requirements.
   The Menu Cycle Analysis slide deck displays the nutrient and meal contribution details. Notice the three color-coding keys below the table. If any nutrients from ingredients in this Menu Cycle have Exceeded, are High / Fail, or are Low, the ACTUAL value column will highlight the color corresponding to the warning.

## VIEW THE NUTRITION ANALYSIS REPORT
Users can generate the Nutrition Analysis Report from the Menu Cycle Analysis slide deck.
On the Menu Cycle Analysis slide deck
1. Click the NUTRITION REPORT button to get a detailed printable report for the nutrition analysis

## The Nutrition Analysis Report displays in
another tab, showing the nutrient analysis.

a. Select a Nutrition Option from the dropdown to view ALL, Expanded, or Simplified versions of the report
b. Click the Printer icon to export the report
c. View information such as the Menu Cycle, Week, Result, Meal Pattern, Meal Type, Serving Group, and Site Group
Notice that the Result for this Menu Cycle will show the outcome of Pass or Fail.
d. View the Cycle Week Nutrient Summary
e. View the Cycle Week Food Component Summary

f. View each Menu Items nutrient information for each day
Use the scroll bar at the bottom of the table to view additional columns.

## VIEW THE FOOD COMPONENTS REPORT
Users can generate the Food Components Report from the Menu Cycle Analysis slide deck.
On the Menu Cycle Analysis slide deck
1. Click the FOOD COMPONENT REPORT button to get a detailed printable report for the food component analysis

## The Food Components Report appears in
another tab, showing the food component information.

a. Click the Printer icon to export the report
b. View information such as the Menu Cycle Name, Site Group, Serving Group, Meal Pattern, Meal Type, and Number of M/MA Subs
Notice that the result for this Menu Cycle will show the outcome of Pass or Fail.
c. View the Food Component information for each day in the Menu Cycle
d. View the Weekly Total, Weekly Requirement, and Weekly Check for each Food Component
Notice whether each day and week Passes or Fails based on the food component(s).

# Add Homerooms Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-064`.

The HOMEROOMS tab allows users to add a new homeroom for a specific site.
1. Select the HOMEROOMS tab atop the page
2. Click the NEW button in the top right corner of the page
   The New Homeroom slide deck displays.
3. Enter the required Homeroom Name, Site, Grades, and optional Description
4. Click the SAVE button to create the new homeroom
   A confirmation response appears atop the page, and the new homeroom will appear in the homerooms list.

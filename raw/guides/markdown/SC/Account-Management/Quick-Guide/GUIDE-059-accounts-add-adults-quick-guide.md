# Accounts - Add Adults Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-059`.

The Accounts function allows users to create an adult account when an adult is not included in the adult file import.
If the district uses an adult file import, the manually added adult must be added to the file to maintain their Active Status.
The information on the adult file import will overwrite a manually added adult.

## Add an Adult Account
1. Click the NEW button in the top right corner of the page, and options appear
2. Select Add Adult
   The Add Adult page displays.
   The Add Adult page has a process wizard on the left-hand side that illuminates purple, indicating which section you are working in.

## Adult Information
1. Enter the adult’s information as desired
2. Enter Meal Pricing Information as desired
   Fields indicated with an asterisk are required.
3. Click the NEXT button
   The adult account has been successfully created.
   The Adult Information section changes, allowing you to add a profile picture for the adult.
4. Use the Drag & Drop File Here option or click the SELECT FILE button to upload a profile picture of the adult is desired
   Uploading an adult profile picture is not required.
5. Click the NEXT button to proceed

## Contact Info
1. Enter the adult's contact information as desired
   Fields indicated with an asterisk are required.
   You can click the SAVE & CLOSE button to return to the Accounts page.
2. Click the NEXT button to proceed
   Restrictions
   The Restrictions section allows users to enter applicable account restrictions.
1. Enter optional restrictions into the fields or select them from the dropdowns
   Hover over an Exclamation Mark icon to view a description of the restriction.
2. Click the NEXT button to proceed

Allergens
The Allergens section allows users to add applicable food allergies that an adult has.
Allergen information is required.
If an adult has no food allergies, it must be indicated.
If no allergies are indicated
1. Click the No allergens indicated checkbox
   If there are allergies indicated
1. Click the dropdowns for Common Allergens or Custom Allergens as desired
2. Select allergens from the list of Common Allergens or Custom Allergens as desired
   An Allergen Feature Disclaimer is displayed at the bottom of the page for additional information.
   You can click the SAVE & CLOSE button to save the information and return to the Accounts page.
3. Click the NEXT button to proceed

## Documents & Comments
This section is optional.
1. Use the Drag & Drop Files Here option or click the SELECT FILES button to upload documents
   Uploaded files populate to the right side of the page with options to download and delete.
   The Checkmark icon indicates that the document was correctly uploaded.
2. Enter Comments as desired
3. Click the SAVE & CLOSE button to save the adult account
   A confirmation response appears atop the page.
   You are returned to the Accounts page.
   Repeat this process to add additional adults as needed.

# Notifications - Household Letters Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-066`.

The HOUSEHOLD LETTERS tab allows users to view, print and/or email students’ household letters based on a specific site.
A valid email address in the Guardian & Contacts section of a patron’s Account Info page is required to send reminders electronically.

## Notify Students
1. Ensure that the HOUSEHOLD LETTERS tab is selected
2. Select the Site Code/Site Name from the dropdown
3. Select the Basis from the dropdown (Default, Carryover)
   This allows you to send letters to students who Default to a Paid Status or a Carryover Status.
   You can send letters to every household by leaving the Basis dropdown blank.
4. Select the Insights from the dropdown
   This allows you to focus on students who have not yet applied for benefits.
5. Click the APPLY button
   Notifications for household letters display in the table below.
   Click the MANAGE VIEWS button to edit the columns displayed in the table.

## View Household Letters
- Click the View Letter icon for the desired student
The Letter to Household displays.
The letter is displayed in the student’s selected language if applicable.
- Click the X in the top right to return to the HOUSEHOLD LETTERS tab
6. Click the Checkbox(es) for the student(s) you wish to notify
   Clicking the header checkbox will select all listed students.
   The checkbox will illuminate purple for a selected patron.
7. Click the NOTIFY button
   The All Options slide deck displays.
8. Select from the Email/Print Options
   The number in the parenthesis indicates the number of Household Letters that will be emailed or printed.
   Email notifications require a guardian email address in the student account details.
- Email/Print: Emails are sent and follow the browser standard for printing.
- Email Only: The notification is sent to the designated email and a success message will appear.
- Print All or Print Only: Follow the browser standard for printing.
9. In the Print Options section, select a Sort By method
10. Click the NOTIFY button, and the system will execute notifications based on the selections
   A confirmation response appears atop the page.
   You are returned to the HOUSEHOLD LETTERS tab.

# ID Cards Quick Guide

The ID Cards function allows users to view and print one or more ID cards for patrons within their district.

## Filter ID Cards
ID Cards can be filtered by Name, ID, Site, Grade, Homeroom, or Roster. Also, users can specify the Print Settings.
1. Under Generate Cards
- Enter the desired search criteria
A Site must be selected to select a Homeroom or Roster.
2. Under Print Settings
- Select the Cards Per Page
- Select the Template
- Select the Margin Guide
3. Click the APPLY button
   Patrons display in the table below.
   Click DOWNLOAD ALL to download all the patron IDs.
4. Select the patron(s)
   The checkbox(es) will illuminate purple for the selected patron(s).
   Click the header checkbox to select all patrons.
5. Click the PRINT button for the selected patron(s)
   You can click the PRINT ALL button to print all patron IDs.
   Alternatively, you can click the Print icon to print an individual patron ID.
6. Follow the browser standard for printing

# View and Modify Default Menu Grids Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-032`.

The Default Menu Grids function allows users to view, edit, or delete meal types, menus, and days of use.

## Edit Default Menu Grids
You can edit the Default Menu Grid details, including the Site Type, Site, Device, Meal Type, Name, and Days For Use.
Menu Items and Menu Grids must be configured before setting up Default Menu Grids.
1. Identify the Default Menu Grid you wish to change
2. Click the Edit icon for a desired Default Menu Grid
   The Set Menu Grids pop-up window displays.
3. Modify the desired information
4. Click the SAVE button to save your changes
   A confirmation response appears atop the page, and the Default Menu Grid is changed.
   Repeat this process to edit additional Default Menu Grids as needed.

## Delete Default Menu Grids
1. Click the Delete icon for a desired Default Menu Grid
   The Confirmation pop-up window displays.
2. Click DELETE to confirm your selection
   A confirmation response appears atop the page, and the account is deleted.
   Repeat this process to delete additional Default Menu Grids as needed.

# Transactions - Account History Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-016`.

The ACCOUNT HISTORY tab within the Transactions function allows users to search and view specific patrons’ detailed transaction information for a specified date range. Two views of the information are available, including the Summary and Detailed Views.

## View Account History
1. Select the ACCOUNT HISTORY tab atop the page
2. Enter the Patron Name / ID Number into the field
3. Select the Site Code/Site Name and Grade from the dropdowns
4. Click the SEARCH button
   Patrons’ information displays in the table below.
5. Click the Arrow left of the patron’s name to expand their information
6. View information such as the patron’s Address, Guardian Name, Eligibility, and Status
7. Click on the row of the desired patron
   The Filters atop the page no longer display.

## Summary View
If the Summary View option is selected, follow the following steps.
The patron’s transaction information will not automatically expand.
8. Select the Summary View radio button
9. Select the Date Range (Daily, Monthly, or Date Range)
10. Enter the Dates into the field(s) based on the selected Date Range
11. Click the APPLY button
   Transaction information displays in the table below.
12. Click the Arrow left of the patron’s ID to expand their transaction information
13. View the patron’s transaction information, such as the Meal Type, Menu Item, Transaction Amount, and Totals and starting Balance as of the specific date
   View the Transaction Receipt
14. Click the Receipt icon for the desired transaction
   The Receipt slide deck displays.
15. View information such as when the receipt was Generated On, Patron Details,

## Items Served, Payment Details, and Current Balance
16. Click the PRINT button at the bottom right corner of the page to print the receipt

## Detail View
If the Detail View option is selected, follow the following steps.
The patron’s transaction information will automatically expand.
8. Select the Detail View radio button

9. Select the Date Range (Daily, Monthly, or Date Range)

11. Click the APPLY button

13. Click the Receipt icon for the desired transaction

# Transactions - Search Quick Guide

> Seeded from the raw extraction. Edit freely. When the raw drifts, review the diff and update this file, then mark it reviewed with `python scripts/mark_guide_reviewed.py GUIDE-015`.

The SEARCH tab within the Transactions function allows users to search, view, and edit transactions associated with patrons. Users can display every Transaction, Exception, Suspicious Transaction, and Reimbursable Meal.
Suspicious Transactions are view only.

## View Transaction History
1. Ensure that the SEARCH tab is selected
2. Select the optional Filters as needed
3. Select the Date from the dropdown
4. Select from the Display options (Everything, Exceptions, Suspicious Transaction, or Reimbursable Meals)
5. Click the APPLY button
   Transaction information displays in the table below.

## Edit Transactions
1. Click the Edit icon for the desired transaction
   The Edit Transaction slide deck displays.
2. In the Sales Details section, you can add Items, edit the sales Quantity, and remove Items
3. In the Payment area, you can edit the Payment Amount (Amount Tendered, Deposit

## Amount, or Change Returned) or Payment Method
4. Once these changes are made, enter the required comments
   Comment History is available at the bottom of the slide deck.
5. Click the SAVE button at the bottom right to retain changes

## View Transaction Receipt
1. Click the Receipt icon for the desired transaction
   The Receipt slide deck displays.
2. View information such as when the receipt was Generated On, Patron Details, Items

## Served, Payment Details, and Current Balance
3. Click the PRINT button at the bottom right corner of the page to print the receipt

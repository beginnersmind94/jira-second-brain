# End of Day Quick Guide

The End of Day function is a self-management tool for site managers, allowing them to review and complete Daily Tasks. These Daily Tasks include Sessions, Marked for Review, Suspicious Transactions, Edit Checks, Serving Exceptions, and Deposit Slip. Also, the history for the individual task and the site and date can be viewed as needed. Reports can also be generated.

## View Daily Tasks
1. Select the Site Code/Site Name from the dropdown
2. Select the Date from the dropdown
   The date defaults to today’s date.
3. Select the Status from the dropdown
4. Click the APPLY button
   Daily tasks display in the table below.
   A progress bar below the Filters indicates how many Daily Tasks are completed.
   The REFRESH button is available to update the data on demand for the selected site and date. A tooltip for the REFRESH button displays the last time the background job was run for the selected site and date.
   The SETTINGS button can turn Daily Tasks on or off as needed. If a Daily Task is turned on, it will become visible in the table.

## Sign Off on Daily Tasks
The Status for each Daily Task will update automatically; the site manager will check whether everything is complete for each task and mark it as done.
1. If a task is not complete, you can click on the Daily Task name, and the system will navigate you to the corresponding function to complete the task
   Click the BACK TO END OF DAY button in the top right corner to return to the End of Day function.
2. If the status of a task is complete, the site manager can sign off directly on that task by clicking the Checkmark in the designated row
   The task changes from Not Done to Done.
3. Click the DAY FINISHED button once all tasks are accounted for
   A pop-up window displays.
4. Enter a comment into the Add Comment field
   The site manager can enter comments for each task or day.
5. Click the SAVE button to retain information
   If you cannot complete each task for the day, you can use the INCOMPLETE toggle switch to roll back the statuses of the Daily Tasks and make the day incomplete again.

## View the History of a Task
1. Once the Daily Tasks are generated, click the History icon for a specific task
   The History slide deck displays.
2. View information for the selected task, such as the Daily Task, When/Who, and Status

## Generate Reports
You can generate the following reports from the End of Day function: Activity, Cash Collection, Edit Check, Meal Count, Menu Item Sales, Payment, Participation, and Revenue.
1. Once the Daily Tasks are generated, click REPORTS at the bottom of the page
   A pop-up displays the available reports.
2. Select the report name as needed
   The selected report page will display.
3. Enter the report criteria and click the GENERATE button to view the report results
   Repeat this process to view any of the available reports as needed.

## View History for the Site and Date
1. Once the Daily Tasks are generated, click HISTORY at the bottom of the page
   The History slide deck displays.
2. View information for the selected site and date, such as the Status, When/Who, and Comments

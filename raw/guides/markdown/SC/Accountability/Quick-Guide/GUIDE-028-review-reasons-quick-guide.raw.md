# Review Reasons Quick Guide

Review Reasons allow users to edit the review reason names. Review Reasons enable cashiers to note in the POS application that a correction must be made to a patron’s transaction. Within the POS application, these entered review reasons can be accessed via the Mark for Review button.

## Edit Review Reasons
1. Select the REVIEW REASONS tab atop the page
   You can click the RESET TO DEFAULT button to change all review reasons to the default text.
2. Click the Edit icon for the desired Review Reason
3. Modify the Review Reason name as desired
4. Click the Checkmark to save the entered name or click X to cancel
   A confirmation response appears atop the page, and the name is changed.
   Repeat this process to change additional Review Reasons as needed.

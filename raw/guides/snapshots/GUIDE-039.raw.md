# Add Income Surveys Quick Guide

Income Surveys are typically used by districts engaging in a Special Provision Program.
Rather than applying for free and reduced benefits, they submit household income data in compliance with state funding programs. This function allows you to enter survey information submitted by households manually.
1. Click the NEW button in the top right corner of the page
   The Add New Survey page displays.
   The Add New Survey page has a process wizard on the top of the page that illuminates purple, indicating which section you are working in.
   Students
1. Enter the student’s Name or ID to generate the results
2. Click the ADD button for the desired student
   If potential students exist in the same household, the system will suggest that you add them to the survey. If so, click the ADD button to add the student(s).
   The student is added to the table and can be deleted.

## Add a New Student
1. Click the ADD NEW button
   The Add Student slide deck displays.
2. Enter the student’s First Name and Last Name
   All other fields are optional.
3. Click the ADD button
   The student is added to the table with a PENDING Status and can be edited or deleted.
3. Click the NEXT button to proceed
   Household
1. Select a Response Type

## Special Assistance Response Type
Select the Special Assistance radio button if a household member receives special assistance from the appropriate case types.
2. Select a Case Type
3. Enter the Case Number
   The Case Number must match the established format for the selected Case Type.

## Income Response Type
Select the Income radio button if no household member receives special assistance.
2. Enter the Household Size

3. Select a Frequency from the dropdown
4. Select an Income Range from the Dropdown
2. Click the NEXT button to proceed
   Details
   This section is optional.
1. Select an Ethnicity
2. Select a Racial Identity
3. Select from the Other Benefits
4. Click the Checkbox below to confirm the information
5. Click the NEXT button to proceed

## Contact Info
This section is optional.
1. Enter the contact information as needed
2. Click the NEXT button to proceed
   Comments
   This section is optional.
1. Select the Language Preference from the dropdown
2. Enter the Survey Accepted Date
3. Add comments into the Add Comment field
4. Click the Signature Confirmed checkbox to confirm the signature
5. In the Documents section, use the Drag & Drop Files Here option or click the SELECT
   FILES button to upload an image or documentation associated with the Income Survey if desired
6. Click the NEXT button to proceed
   Review
1. Review the sections and make edits as necessary
2. Click the SUBMIT button to submit the Income Survey
   Alternatively, you can click the MARK AS PENDING button to save the information entered and review it later.
   A confirmation response appears atop the page, and the survey was submitted successfully.
   Repeat this process to submit additional surveys as needed.
   Summary
1. Click the PDF button to export the survey information
2. Review the sections
3. Click the ADD ANOTHER SURVEY button to continue entering Income Surveys

4. Click the I’M DONE FOR NOW button to return to the Income Surveys page

How to set it up — five steps

Enable Forms for your district

Back Office → Settings → Forms Licensing. Toggle the forms you plan to offer (Summer EBT, Income Survey, custom).

Build the form template

Eligibility → Form Configuration. On the Forms tab, click + Add Form, label it, add fields, save.

Attach an image template

Same page, Form Image Templates tab → upload a PDF or image that renders the paper equivalent for auditors and parents who prefer a printed copy.

Set qualification logic (optional)

In Form Configuration, choose which responses trigger a Qualified result. Once set, submitted forms auto-populate the Result column in the Other Forms grid.

Publish & monitor

Form appears in Family Hub for eligible households. Track submissions under Eligibility → Other Forms. District admin can also add submissions manually via Add New Form Application.

Frequently asked questions

Do I need Cybersoft support to add a new form?

No. Once Forms Licensing is enabled for your district, form creation is fully self-serve under Eligibility → Form Configuration. Support is only required for the initial licensing enablement.

Can we use the same form across multiple districts?

Yes. Forms set up at the state or multi-tenant level can be inherited by member districts. Template reuse is supported; qualification rules can be district-specific.

Where do submitted applications appear?

Eligibility → Other Forms shows the grid of all submitted Form Applications with status, result, applicant, and submission date. The grid supports filters, custom views, and export.

Do submissions integrate with existing Applications (F/R meal)?

Yes. Application Result is tied into the Applications pipeline, so Qualified forms feed into the same downstream eligibility workflows your team already uses.

Can parents submit without re-authenticating?

Yes. If the parent is already signed into the SchoolCafe Family Hub, forms appear in their dashboard and submit with a single tap-through.

Is there a paper fallback for households without online access?

Yes. Image Templates attached to each form render as printable PDFs. District admins can also enter submissions manually via the Add New Form Application button on the Other Forms page.

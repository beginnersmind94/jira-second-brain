### Page 1

Collection Worksheet
The FNS-742 Collection Worksheet populates with automatically calculated totals based
on live data. The district must submit these totals to the state annually, as each state
asks for this report to be submitted after verification.
View the Collection Worksheet
1. Select the COLLECTION WORKSHEET tab
The current academic year Collection Worksheet automatically displays on the page.
2. View the Collection Worksheet details
Various Document icons are available throughout the worksheet; you can click on
them to view the Collection Report. This information can be downloaded.
Click the SAVE button to store a PDF copy of the worksheet. Saved copies appear in a grid below the worksheet with Generation Date, User, and Actions (View, Download), so you can retrieve previously saved versions later.
Use the PREVIOUS PAGE and NEXT PAGE buttons to navigate between pages in the
Collection Worksheet.
Select a Different Academic Year
1. Select an Academic Year from the dropdown
2. Click the APPLY button
The selected academic year’s Collection Worksheet displays.

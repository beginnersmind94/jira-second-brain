PERPETUAL INVENTORY 
The Perpetual Inventory function allows users to see what items they have in their Inventory, based on the items that 
they input into the system. Perpetual inventory is what the system thinks a user has at their site.  
For Perpetual Inventory, go to Inventory > Inventory > Perpetual Inventory. 
VIEW PERPETUAL INVENTORY INFORMATION  
 
On the Perpetual Inventory page  
1. Select a Site and from the dropdown 
Your Site displays; if you have access to more than one Site, select the desired Site. 
2. Select Show All Inventory Items or Show Only On Hand Inventory Items 
3. Select an Item Category from the dropdown 
4. Enter an Item Description and Item 
5. Click Apply

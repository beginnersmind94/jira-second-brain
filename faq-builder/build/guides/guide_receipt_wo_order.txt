RECEIPTS W/O ORDERS 
The Receipts w/o Orders function is used when Items were not ordered via PrimeroEdge but was received by the site. 
However, the Item will still need to be received in the system to update the Perpetual Inventory. An example of this 
usage is when there is a scheduled delivery of items. The items are not ordered via PrimeroEdge since it’s automatically 
delivered to the site, but will still need to be accounted for in the system.  
 The Vendors and Items must be appropriately configured before using the Receipts w/o Orders function. To 
configure the Vendors and Items, go to Inventory > Configuration > Vendors and Inventory > Configuration > Items. 
For Receipts w/o Orders, go to Inventory > Orders > Receipts w/o Orders. 
RECEIPT HEADER 
 
1. Select a Receiving Site from the dropdown 
The listed Receiving Site is dependent on the users access. 
2. Select a Vendor from the dropdown 
3. Enter or select the Transaction Date and Time 
4. Enter an Invoice # and Invoice Amount 
 The Invoice # and Invoice Amount are required fields. The Invoice Amount must equal the Receipt Amount. The 
Receipt Amount is calculated based on the items entered under the Received Items section. 
5. Enter Delivered By 
6. Enter Comments 
7. Users can toggle between List View and Quick Entry View for the Received Items section

RECEIVED ITEMS 
The orderable Items are listed in two views:  
• Quick Entry View – This view allows the user to search for specific items to add in the Received Items section. 
• List View – This view lists all orderable Items for the Vendor chosen under the Receipt Header section. 
QUICK ENTRY VIEW 
1. If in the List View, click Switch to Quick 
Entry View from the Receipt Header 
section  
2. Enter an Item # into the field, if known, or 
click the Three-Dot Look Up button to 
search for an Item 
 
The Search Item pop-up appears. 
3. Select an Item Category, Storage Category 
from the dropdown, enter Item Description 
like* or enter Item# like* 
4. Click Get Items 
Click Get Items without filtering to list all 
items.  
Items that meet the filter criteria will be listed 
under the Item List section. 
5. Click the Item # link to select the Item

CONFIRM RECEIPT 
 
A new page with the Receipt details appears. 
1. Click Edit Receipt to go back to the previous page and make modifications to the Receipt 
2. Click Save as Incomplete to save the changes but not confirm the Receipt 
3. Click Confirm Receipt to finish the process
